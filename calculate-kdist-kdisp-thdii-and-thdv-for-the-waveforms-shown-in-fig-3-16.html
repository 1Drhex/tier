Guest Question.

Calculate \(\ K_{dist} , K_{disp} , THD_{ii} and THD_{v} \) for the waveforms shown in Fig. 3.16.

Admin
Solution

Since the voltage is purely sinusoidal, then \(\ THD_{v} \)=0%. To obtain the current \(\ THD_{i} \) , ﬁrst we obtain the rms value of \(\ i_{s} \left(t\right) \) and its fundamental component:

\(\ I^{2}_{s , rms} = \frac{1}{2\pi } \int_{-\theta }^{2\pi -\theta }{i^{2}_{s} d\omega t } =\frac{1}{2\pi }\left[\int_{-\theta }^{2\pi -\theta }{I^{2}_{0} d\omega t } + \int_{\pi -\theta }^{2\pi -\theta }{\left(-I^{2}_{0} \right) d\omega t } \right] \) \(\ =I^{2}_{0} \)

The fundamental component of \(\ i_{s} \left(t\right) \) is given by:

\(\ I_{s1} = I^{\shortmid }_{s1} \sin \omega t + I^{’’}_{s1} \cos t \)

where

\(\ I^{\shortmid }_{s1} = \frac{1}{2\pi } \left[\int_{-\theta }^{2\pi -\theta }{I_{0} \sin\omega t d\omega t } + \int_{\pi -\theta }^{2\pi -\theta }{- I_{0} \sin \omega t d\omega t } \right] \)

If we let \(\ \omega \tau =\omega t+\theta \) , the above integral becomes:

\(\ I^{\shortmid }_{s1} = \frac{1}{2\pi } \left[\int_{\theta }^{\pi }{I_{0} \sin\left(\omega \tau -\theta \right) d\omega t } – \int_{\pi }^{2\pi }{ I_{0} \sin \left(\omega \tau -\theta \right) \omega t d\omega t } \right] \)

 

\(\ = \frac{ I_{0}}{\pi } \left[ -\cos \left(\pi -\theta \right) + \cos \left(-\theta \right) + \cos \left(2\pi -\theta \right) + \cos \left(-\theta \right) \right] \)

 

\(\ =\frac{4I_{0}}{\pi } \cos \theta \)

 

Similarly, I00 \(\ I^{\shortmid \shortmid }_{s1} = \left(4I_{0}/ \pi \right) \sin \theta \) . Therefore, the peak fundamental component is given by:

\(\ I_{s1} \sqrt{\left( I^{\shortmid }_{s1}\right) ^{2} + \left( I^{\shortmid \shortmid }_{s1}\right) ^{2} } = \frac{4I_{0}}{\pi } \)

Hence, the fundamental component of \(\ i_{s} \left(t\right) \) is given by:

\(\ i_{s1} \left(t\right) = I_{s1} \sin \left(\omega t +\theta \right) \)

and the rms of \(\ I_{s1} \) is given by:

\(\ I_{s1 , rms} = \frac{4}{\sqrt{2} } \frac{I_{0}}{\pi} = \frac{2\sqrt{2} }{\pi } I_{0} \)

\(\ K_{dist} , K_{disp} and THD \) are given by the following expressions:

 

\(\ K_{dist} =\frac{2\sqrt{2} }{\pi } \)

 

\(\ K_{disp}= \cos\theta \)

 

\(\ THD =\left(\sqrt{\frac{\pi ^{2} }{8} -1} \right) \times 100 \) %

=48.34 %
