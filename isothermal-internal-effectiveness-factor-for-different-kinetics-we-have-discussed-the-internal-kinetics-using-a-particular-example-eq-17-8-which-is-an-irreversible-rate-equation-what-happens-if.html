Guest Question.

Isothermal internal effectiveness factor for different kinetics
We have discussed the internal kinetics using a particular example, Eq. (17.8) \(\eta_{e}=\frac{-r_{A}^{\prime} S}{-r_{A b}^{\prime}}=\frac{\left(-r_{A S}^{\prime \prime}\right) S_{p}}{\left(-r_{A b}^{\prime \prime}\right) S_{p}}=\frac{-r_{A S}^{\prime \prime}}{-r_{A b}^{\prime \prime}}\) , which is an irreversible rate equation. What happens if different kinetics is observed? The least we can be sure is that the procedure we have learned can be applied to any kinetics. In some cases, the results can even be directly used. Here is an example: reaction

\(A \leftrightarrows B\) (E17.4.1)

occurs on a single site solid-catalyst. LHHW kinetics is given by

\(-r_{A}^{\prime}=\frac{k^{\prime}\left(C_{A}-C_{B} / K_{C}\right)}{1+K_{A}^{\prime} C_{A}+K_{B}^{\prime} C_{B}}\) (E17.4.2)

which is identical to Eq. (8.105) \(r=k \frac{C_{A}-C_{B} / K_{e q}}{1+K_{A} C_{A}+K_{B} C_{B}}\) except we have used primes here for the kinetic constants. In the reaction mixture (bulk fluid phase), the concentration of A and B together are \(C_{0}=C_{A}+C_{B}\). Derive the effectiveness factor for this reaction system.

 

Admin
Solution

This is a reversible kinetics case. From stoichiometry, we obtain

\(C_{A}+C_{B}=C_{0}=\text { constant }\) (E17.4.3)

and this is valid anywhere inside the catalyst when pseudosteady state has established. Thus Eq. (E17.4.2) can be written in terms of \(C_{A}\) only by making use of this stoichiometry relation, Eq. (E17.4.3). That is

\(-r_{A}^{\prime}=\frac{k^{\prime}\left[C_{A}-\left(C_{0}-C_{A}\right) / K_{C}\right]}{1+K_{A}^{\prime} C_{A}+K_{B}^{\prime}\left(C_{0}-C_{A}\right)}\) (E17.4.4)

Let

\(C_{A}^{\prime}=C_{A}-\frac{C_{0}}{1+K_{C}}\) (E17.4.5)

Eq. (E17.4.4) can be further reduced to

\(-r_{A}^{\prime}=\left\lgroup 1+\frac{1}{K_{C}} \right\rgroup \frac{k^{\prime}}{K_{A}^{\prime}-K_{B}^{\prime}} \frac{C_{A}^{\prime}}{\frac{1+\left(K_{A}^{\prime}+K_{C} K_{B}^{\prime}\right) C_{0} /\left(1+K_{C}\right)}{K_{A}^{\prime}-K_{B}^{\prime}}+C_{A}^{\prime}}\) (E17.4.6)

which can be reduced to

\(-r_{A}^{\prime}=\frac{r_{\max }^{\prime} C_{A}^{\prime}}{K_{A}+C_{A}^{\prime}}\) (E17.4.7)

where

\(r_{\max }^{\prime}=\left\lgroup 1+\frac{1}{K_{C}} \right\rgroup \frac{k^{\prime}}{K_{A}^{\prime}-K_{B}^{\prime}}\) (E17.4.8a)

\(K_{A}=\frac{1+\left(K_{A}^{\prime}+K_{C} K_{B}^{\prime}\right) C_{0} /\left(1+K_{C}\right)}{K_{A}^{\prime}-K_{B}^{\prime}}\) (E17.4.8b)

Therefore this reversible kinetics is reduced to the same form as the irreversible kinetics given by Eq. (17.8) with the exception that the concentration of A is replaced by the difference of concentration of A from the equilibrium concentration of A as given by Eq. (E17.4.5). Since the equilibrium concentration is constant, and from Eq. (17.42)

\(\frac{d}{d X} D_{e A} S \frac{d C_{A}}{d x}+r_{A}^{\prime} S=0\), (17.42)

we obtain the governing equation inside the porous catalyst as

\(\frac{d}{d x} D_{ e A} S \frac{d C_{A}^{\prime}}{d x}+r_{A}^{\prime} S=0\). (17.4.9)

That is the solution we obtained so far is directly applicable since the derivative of a constant is zero. By definition, the generalized Thiele modulus is given by

\(\phi=\frac{-r_{A S}^{\prime}}{a_{p}}\left[2 \int_{C_{A e, 0}}^{C_{A S}}-r_{A}^{\prime} D_{e A} d C_{A}\right]^{-\frac{1}{2}}\) (17.49)

Noting that

\(C_{A e, 0}^{\prime}=C_{A e, 0}-\frac{C_{0}}{1+K_{C}}\) (E17.4.10a)

\(C_{A e, 0}=\frac{C_{0}}{1+K_{C}}\) (E17.4.10b)

Eq. (17.49) is integrated out to give

\(\phi=\frac{1}{a_{p}} \sqrt{\frac{r_{\max }^{\prime}}{2 D_{e A} C_{A S}^{\prime}}} \frac{\left[1-K_{\beta} \ln \left(1+K_{\beta}^{-1}\right]^{-\frac{1}{2}}\right.}{1+K_{\beta}}\) (E17.4.11)

with \(K_{\beta}\) given by Eq. (17.50) \(K_{\beta}=\frac{K_{A}}{C_{A S}}\) , \(r_{\max } \text { and } K_{A}\) given by Eq. (E17.4.8), and

\(C_{A S}^{\prime}=C_{A S}-\frac{C_{0}}{1+K_{C}}\). (E17.4.12)

Indeed, the equation for the Thiele modulus is directly applicable, Eq. (E17.4.11) is identical to (17.51) \(\phi=\frac{1}{a_{p}} \sqrt{\frac{I_{m a x}^{\prime}}{2 D_{e A} C_{A S}}} \frac{\left[1-K_{\beta} \ln \left(1+K_{\beta}^{-1}\right]^{-\frac{1}{2}}\right.}{1+K_{\beta}}\) .

This last example shows that when different kinetics is observed, the same approach to effectiveness factor can be applied. The key to the analysis is to convert all the dependent variables (i.e., concentrations) into one single dependent variable (concentration). Although the final forms of equations may or may not the same, one can solve the equations in the same manner. We also learned in this section that nonisothermal reaction systems can be treated in the same way, with one additional (energy balance) equation.
