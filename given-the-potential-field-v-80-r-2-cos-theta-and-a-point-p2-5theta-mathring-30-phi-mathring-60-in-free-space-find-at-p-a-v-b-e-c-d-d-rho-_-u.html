Guest Question.

Given the potential field V = \(80{ r }^{ 2 }\cos { \theta }\) and a point P\((2.5,\theta =\mathring { 30 } ,\phi =\mathring { 60 } )\) in free space, find at P: (a) V; (b) E; (c) D; (d) \({ \rho }_{ \upsilon }\); (e) dV/dN; (f) \({ a }_{ N }\).

Admin
Solution

a) Substitute the coordinates into the function and ﬁnd \({ V }_{ P }=80{ \left( 2.5 \right) }^{ 2 }\cos { \left( 30 \right) } =433V.\)
b)
E= \(-\triangledown V=-\frac { \partial V }{ \partial r } { a }_{ r }-\frac { 1 }{ r } \frac { \partial V }{ \partial \theta } { a }_{ \theta }=-160r\cos { \theta } { a }_{ r }+80r\sin { \theta } { a }_{ \theta }V/m\)
Evaluating this at P yields \({ E }_{ P } = -346{ a }_{ r } + 100{ a }_{ \theta } V/m.\)
c) In free space , \({ D }_{ P } = { \epsilon }_{ 0 }{ E }_{ P } = (-346{ a }_{ r } + 100{ a }_{ \theta }){ \epsilon }_{ 0 } = -3.07{ a }_{ r } + 0.885{ a }_{ \theta } nC/{ m }^{ 2 }.\)
d) \({ \rho }_{ \upsilon }=\triangledown .D = { \epsilon }_{ 0 }\triangledown . E = { \epsilon }_{ 0 }\left[ \frac { 1 }{ { r }^{ 2 } } \frac { \partial }{ \partial r } \left( { r }^{ 2 }{ E }_{ r } \right) +\frac { 1 }{ { r }^{ 2 }\sin { \theta } } \frac { \partial }{ \partial \theta } \left( { E }_{ \theta }\sin { \theta } \right) \right] \)

Substituting the components of E, we ﬁnd
\({ \rho }_{ \upsilon } = \left[ \frac { 160\cos { \theta } }{ { r }^{ 2 } } 3{ r }^{ 2 }+\frac { 1 }{ { r }\sin { \theta } } 80r\left( 2\sin { \theta \cos { \theta } } \right) \right] { \epsilon }_{ 0 }=-320{ \epsilon }_{ 0 }\cos { \theta } =-2.45nC/{ m }^{ 3 }\)
with \(\theta =\mathring { 30 }\) .
e) This will be just \(\left| { E }\right| \)evaluated at P , which is
\({ \frac { dV }{ dN } }{ | }_{ P }=\left| -346{ a }_{ r }+100{ a }_{ \theta } \right| =\sqrt { { \left( 346 \right) }^{ 2 }+{ \left( 100 \right) }^{ 2 } } =360V/m\)
f) This will be
\({ a }_{ N }=-\frac { { E }_{ P } }{ \left| { E }_{ P } \right| } =-\left[ \frac { -346{ a }_{ r }+100{ a }_{ \theta } }{ \sqrt { { \left( 346 \right) }^{ 2 }+{ \left( 100 \right) }^{ 2 } } } \right] = 0.961{ a }_{ r } – 0.278{ a }_{ \theta }\)
