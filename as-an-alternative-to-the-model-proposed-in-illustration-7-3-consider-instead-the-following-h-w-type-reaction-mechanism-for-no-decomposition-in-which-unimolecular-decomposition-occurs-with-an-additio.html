Guest Question.

As an alternative to the model proposed in Illustration 7.3, consider instead the following H-W-type reaction mechanism for NO decomposition, in which unimolecular decomposition occurs with an additional active site, *,and \(N_{2}\) desorption is the rate determining step:

(1) \(2 \left[NO + * \xrightleftharpoons[]{K_{NO} } NO *\right] \)

 

(2) \(2 \left[NO *+ * \xrightleftharpoons[]{K_{1} } N*+ O *\right] \)

 

(3) \(2 O* \xrightleftharpoons[]{1/K_{O_{2} } } O_{2}+ 2*\)

 

(4) \(2 \ N* \xrightarrow[]{K_{2} } N_{2}+ 2* \)

—————————————————————

\(2 NO \Longrightarrow N_{2}+ O_{2}\)

The stoichiometric number for steps 1 and 2 in the catalytic cycle is 2, as indicated.
(a) From this sequence of elementary steps, derive the most general form of the rate expression. What rate law is obtained when each of the following assumptions ismade:
(b) If [NO *] is assumed to be very low compared to the other surface species,
(c) If [NO *] is assumed to be the MARI,
(d) If [O *] is assumed to be very small compared to the other surface species,
(e) If [O *] is assumed to be the MARI,
(f) If [N *] is assumed to be the MARI,
(g) If [N *] is not only the MARI, but the surface is also nearly saturated
with N atoms.
Over a \(La_{2}O_{3}\) catalyst at 923 K, this reaction exhibited a reaction order on NO of about 1.2 with no \(O_{2}\) in the feed and near 1.5 with \(O_{2}\) in the feed stream, and the rate decreased significantly with \(O_{2}\) present [58]. Based on a cursory examination of the rate expressions derived in (b) â€“ (g), can any of them be rejected? Why?

Solution

(a) The rate, defined by \(N_{2}\) formation, is \(r= \frac{d \left[N_{2}\right] }{dt}= \frac{-1}{2} \frac{ d \left[NO\right] }{dt }= K_{2} \left[N*\right] ^{2} \)

From the 3 quasi-equilibrated steps:

\(\left[NO*\right]= K_{NO} P_{NO} \left[*\right] , \left[N*\right]\left[O*\right]= K_{1}\left[NO*\right]\left[*\right] \) and \(\left[O*\right]= K^{1/2}_{O_{2} } P^{1/2}_{O_{2} } \left[*\right]\)

\(\left[N*\right]= \frac{K_{1} \left[NO*\right]\left[*\right] }{\left[O*\right]} = \frac{K_{1} K_{NO} P_{NO} \left[*\right] ^{2} }{\left[O*\right]} = \frac{K_{1} K_{NO} P_{NO} \left[*\right] ^{2}}{K^{1/2}_{O_{2} } P^{1/2}_{O_{2} } \left[*\right] } \)

 

Site balance gives: \(L= \left[*\right] + \left[NO*\right]+ \left[N*\right]+ \left[O*\right] \ \ \ \ \Rightarrow \)

 

\(L= \left[*\right]+ K_{NO} P_{NO}\left[*\right] + \frac{K_{1} K_{NO} P_{NO}}{K^{1/2}_{O_{2} } P^{1/2}_{O_{2} }}\left[*\right]+ K^{1/2}_{O_{2} } P^{1/2}_{O_{2} }\left[*\right] \) so

\(\left[*\right]= L/\left(1+ K_{NO} P_{NO} + K^{1/2}_{O_{2} } P^{1/2}_{O_{2} }+ K_{1} K_{NO} P_{NO}/ K^{1/2}_{O_{2} } P^{1/2}_{O_{2} }\right) ,\)

for site pairs: \(\frac{Z}{2 L} \) is the probability factor, so

\(r=\frac{Z L}{2} K_{2} K^{2}_{1} K^{2}_{NO} P^{2}_{NO} \left[*\right] ^{2} /K_{O_{2} } P_{O_{2} } = LK^{\prime }_{2} K^{2}_{1} K^{2}_{NO} P^{2}_{NO}/ K_{O_{2} } P_{O_{2} } \left(1+ K_{NO}P_{NO}+ K^{1/2}_{O_{2} } P^{1/2}_{O_{2} }+ K_{1} K_{NO}P_{NO}/ K^{1/2}_{O_{2} } P^{1/2}_{O_{2}} \right) ^{2} \)

 

or \(r= K P_{NO}/\left(K^{1/2}_{O_{2} } P^{1/2}_{O_{2} }+ K^{1/2}_{O_{2} } K_{NO} P^{1/2}_{O_{2} } P_{NO}+ K_{O_{2} } P_{O_{2} } + K_{1}K_{NO} P_{NO} \right) ^{2} \)

 

(b) \(\left[NO*\right] \approx O \Rightarrow r= K P^{2}_{NO} /\left(K^{1/2}_{O_{2} } P^{1/2}_{O_{2} }+ K_{O_{2} } P_{O_{2} }+ K_{1}K_{NO} P_{NO} \right) ^{2} \)

 

(c) \(\left[NO*\right] \) is MARI \(\Rightarrow r=K P^{2}_{NO} /\left(K^{1/2}_{O_{2} } P^{1/2}_{O_{2} }+ K^{1/2}_{O_{2} } K_{NO} P^{1/2}_{O_{2} } P_{NO} \right) ^{2} \)

 

(d) \(\left[O*\right]\approx O \Rightarrow r= K P^{2}_{NO} / \left(K^{1/2}_{O_{2} } P^{1/2}_{O_{2} }+ K^{1/2}_{O_{2} } K_{NO} P^{1/2}_{O_{2} } P_{NO} + K_{1}K_{NO} P_{NO} \right) ^{2} \)

 

(e) \(\left[O*\right]\) is MARI \( \Rightarrow r=K P^{2}_{NO} /\left(K^{1/2}_{O_{2} } P^{1/2}_{O_{2} }+ K_{O_{2} } P_{O_{2} }\right) ^{2} \)

 

(f) \(\left[N*\right] \) is MARI \( \Rightarrow r=K P^{2}_{NO} /\left(K^{1/2}_{O_{2} } P^{1/2}_{O_{2} }+ K_{1}K_{NO} P_{NO}\right) ^{2}\)

 

(g) \(\left[N*\right] \) near saturation \(\left(>> \left[*\right]\right) \Rightarrow \left[N*\right] \cong L\Rightarrow r= \frac{Z}{2 L} K_{2}L^{2}= L K^{\prime }_{2} \)

Reject: (e) – Cannot give reaction order on \( P_{NO}\) below 2
(g) – Observed reaction order is not zero order
