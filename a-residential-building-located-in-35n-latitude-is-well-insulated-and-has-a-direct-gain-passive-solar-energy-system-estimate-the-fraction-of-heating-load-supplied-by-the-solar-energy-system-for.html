Guest Question.

A residential building located in 35°N latitude is well insulated and has a direct gain passive solar energy system. Estimate the fraction of heating load supplied by the solar energy system for December and the auxiliary energy required, given the following information:

Window area = 10 m2.

Effective heat capacity of the building, Cb = 60 MJ/°C.

Allowable temperature swing = 7°C.

Low set point temperature = 18.3°C.

U value of window without night insulation = 5.23 W/m2-°C.

Building UA, excluding direct gain window = 145 W/°C.

Degree days for December, estimated at a base temperature of 18.3°C = 928 °C-days

Mean ambient temperature, \(\overline{T}_{a}\) = 11.1°C.

Monthly average daily total radiation \(\overline{H}\) = 9.1 MJ/m2.

Monthly average \((\overline{\tau \alpha })\)= 0.76.

Admin
Solution

The thermal load needs to be calculated first. The UA for the building, including the direct gain window, is

\((UA)_{h}=145+(10\times 5.23)=197.3\) W/°C

From Eq. (6.24),

\(L=(UA)_{h}(DD)_{h}=197.3\times 928\times 24\times 2600 = 15.82\) GJ

Because the latitude of the building under consideration is the same as the one used in , rn = 0.171, rdn = 0.159.

\(\overline{H}_{0}\) = 16.8 MJ/m2, which from Eq. (2.82)

\(\overline{K}_{T} =\frac{\overline{H}}{\overline{H}_{o} } \)

gives \(\overline{K}_{T}\) = 0.54. For December, hss = 72.7°; from Eq. (11.32a),

\(\frac{H_{D}}{H}=\begin{cases}1.0 – 0.2727K_{T}+2.4495K^2_{T}-11.9514K^3_{T}+9.3879K^4_{T}&forK_{T}\lt 0.715 \\0.143& for K_{T}\geq 0.715\end{cases}\)

HD/H = 0.483; from Eq. (2.105c),

\(\frac{\overline{H}_{D}}{\overline{H}} =1.391-3.560\overline{K}_{T} +4.189\overline{K}^{2}_{T}-2.137 \overline{K}^{3}_{T}\)

\(\overline{H}_{D}/\overline{H}\)=0.35; and from Eq. (2.108),

\(\overline{R}_{B} =\frac{\cos (L-\beta )\cos (\delta )\sin (h^{′}_{ss})+(\pi /180)h^{′}_{ss}\sin (L-\beta )\sin (\delta )}{\cos (L)\cos (\delta )\sin (h_{ss})+(\pi /180)h_{ss}\sin (L)\sin (\delta )}\)

\(\overline{R}_{B} \)=2.095. Using β = 90° (vertical surface) and assuming a ground reflectance of 0.2, \({R}_{B,n}\)= 1.603, \({R}_{n}\)= 1.208, \(\overline{R} \) = 1.637.

From Eq. (11.66),

\(X=\frac{N\overline{S}A_{r} }{L} =\frac{N\overline{H}\overline{R}(\overline{\tau \alpha } )A_{r} }{L} =\frac{31\times 9.1\times 10^6\times 1.637\times 0.76\times 10}{15.82\times 10^9}=0.222 \)

From Eq. (11.63),

\(I_{tc}=\frac{(UA)_{h}(T_{b}-\overline{T}_{a} )}{(\overline{\tau \alpha } )A_{r}} =\frac{197.3(18.3-11.1)}{0.76\times 10} =186.9 W/m^2\)

The next parameter that we need to calculate is\(\overline{\Phi } \) . From Eq. (11.33),

\(\overline{X}_{c} =\frac{I_{tc}}{r_{n}R_{n}\overline{H} } =\frac{186.9\times 3600}{0.171\times 1.208\times 9.1\times 10^6} =0.358\)

From Eq. (11.34),

\(A=2.943-9.271\overline{K}_{T} +4.031\overline{K}^2_{T} \)

\(=2.943-9.271\times 0.54+4.031(0.54)^2\)

= –0.888

\(B= –4.345+8.853\overline{K}_{T} –3.602\overline{K}^2_{T} \)

\(=–4.345+8.853\times 0.54–3.602(0.54)^2\)

= –0.615

\(C= –0.170–0.306\overline{K}_{T} +2.936\overline{K}^2_{T} \)

\(=–0.170–0.306\times 0.54+2.936(0.54)^2\)

= 0.521

\(\overline{\Phi } =exp\left\{\left[A+B\left\lgroup\frac{R_{n}}{\overline{R} } \right\rgroup \right]\left[\overline{X}_{c}+C\overline{X^2}_{c} \right] \right\}\)

\(=exp\left\{\left[-0.888-0.615\left\lgroup\frac{1.208}{1.637} \right\rgroup \right]\left[0.358+0.521(0.358)^2\right] \right\}\)

=0.566

From Eq. (11.70),

\(Y=\frac{C_{b}\Delta T_{b}}{\overline{H}_{t}(\overline{\tau \alpha } )A_{r}\overline{\Phi } } =\frac{C_{b}\Delta T_{b}}{\overline{H}\overline{R}(\overline{\tau \alpha } )A_{r}\overline{\Phi } } =\frac{60\times 7}{9.1\times 1.637\times 0.76\times 10\times 0.566} =6.55\)

From Eq. (11.71b),

\(P=[1-exp(-0.294Y)]^{0.652}=[1-exp(-0.294\times 6.55)]^{0.652}=0.902\)

From Eq. (11.71a),

\(f=min\left\{P{X}+(1-P)(3.082-3.142\overline{\Phi } )[1-exp(-0.329{X})],1\right\} \)

\(=min\left\{0.902\times 0.222+(1-0.902)(3.082-3.142\times 0.566 )[1-exp(-0.329\times 0.222)],1\right\} \)

\(=min\left\{0.21,1\right\}\)

=0.21

Finally, from Eq. (11.82),

\(Q_{aux}=(1-f)L=(1-0.21)\times 15.82=12.50\)GJ
