Heat Input Calculations Using Different Data Sources

Consider heating 2 moles of steam from 200ºC and 1 MPa to 500ºC and 1 MPa. Calculate the heat input required using the following sources for data:

(a) Heat capacity

(b) Steam tables

Admin
Solution

(a) Since this process occurs at constant pressure, the system will expand as T increases. In accordance with the discussion above, enthalpy is the appropriate property to calculate the heat input. The extensive version of Equation (2.28) can be written as:

 

\(\Delta h=q \quad \text { closed system, const } P\) (2.28)

 

\(Q=n \Delta h\)

 

If we assume water is an ideal gas, we use the values of heat capacity given in Appendix A.2 to calculate \(\Delta h\):

 

\(\frac{c_{P}^{\text {ideal gas }}}{R}=A+B T+D T^{-2}=3.470+1.450 \times 10^{-3} T+\frac{0.121 \times 10^{5}}{T^{2}}\)

 

Using the defi nition of heat capacity, we get the following integral expression:

 

\(\Delta h=\int_{T_{1}}^{T_{2}} c_{P} d T=R \int_{473}^{773}\left[A+B T+D T^{-2}\right] d T\)

 

Integrating,

 

\(\Delta h=R\left[A T+\frac{B}{2} T^{2}-\frac{D}{T}\right]_{473}^{773}\)

 

Substituting in values:

 

\(\Delta h=8.314\left[3.470(300)+0.725 \times 10^{-3}\left(773^{2}-473^{2}\right)\right.\)

 

\(\left.-0.121 \times 10^{5}\left(\frac{1}{773}-\frac{1}{473}\right)\right]=10,991\left[\frac{ J }{ mol }\right]\)

 

and, \(Q=n \Delta h=21,981[ J ]\)

 

(b) From the steam tables, Appendix B.2:

 

\(\hat{h}_{1}\left(\text { at } 1 MPa , 200^{\circ} C \right)=2827.9[ kJ / kg ]\)

 

\(\hat{h}_{2}\left(\text { at } 1 MPa , 500^{\circ} C \right)=3478.4[ kJ / kg ]\)

 

So, \(\Delta \hat{h}=\hat{h}_{2}-\hat{h}_{1}=650.5[ kJ / kg ]\)

 

Since the steam tables give us the specifi c enthalpy, we must multiply by mass. Thus, we must use the molecular weight for water, \(M W_{ H _{2} O }=0.018[ kg / mol ]\):

 

\(Q=m \Delta \hat{h}=(2[ mol ])(0.018[ kg / mol ])(650.5[ kJ / kg ])\left(10^{3}[ J / kJ ]\right)=23,418[ J ]\)
