Guest Question.

Modelling non-radiative energy transfer

Consider the following mode of a non-radiative transition. Let the initial state be \(\left|i\right\rangle \), and let there be a uniform ladder of states \(\left|v\right\rangle \) of spacing e that acts as a thermal reservoir (Fig. 11.19). Take the matrix elements of the perturbation that mixes the states of the two systems to be real and equal to a for all values of v. Calculate the probability using first-order perturbation theory that the system has undergone energy transfer from the initial state of energy \(E_{i}\) to the thermal reservoir.

Method First-order perturbation theory (Section 6.2) tells us that the probability amplitude for finding the system in the state \(\left|v \right\rangle \) of the reservoir is \(a/(E_{i}-E_{v } ) \) (see eqn 6.26). The probability is the square of this amplitude, and the total probability is the sum over all v. Set \(E_{v } =E_{f} +v \varepsilon \) with \(v =0,\pm 1,\pm 2,...\) (see Fig. 11.19). We shall write \(\Delta E=E_{i} -E_{f} \).

\(c_{k} =\frac{H^{(1)}_{k0} }{E^{(0)}_{0}-E^{(0)}_{k} } \) (6.26)

Solution

It follows from eqn 6.26 that the total probability is

\(P=\sum\limits_{v }{\frac{a^{2} }{(E_{i}-E_{v } )^{2} } } =\sum\limits_{\nu }{\frac{a^{2} }{(E_{i}-E_{f } -v \varepsilon )^{2} } }\)

 

\(=\sum\limits_{v }{\frac{a^{2} }{ \varepsilon ^{2} \left\{(E_{i}-E_{f } )/\varepsilon -v\right\}^{2} } } =\left(\frac{a}{\varepsilon } \right)^{2} \sum\limits_{v }{\frac{1 }{\left\{(\Delta E/\varepsilon )-v\right\}^{2} } }\)

 

\(=\left(\frac{a\pi }{\varepsilon } \right) ^{2} cosec ^{2} \left(\frac{\pi \Delta E}{\varepsilon } \right) \)

(For the sum, see M. Abramowitz and I.A. Stegun, Handbook of mathematical functions, Dover, New York (1965), eqn 4.3.92, or use mathematical software.) The variation of P with the parameters is illustrated in Fig. 11.20.

Comment The model is a greatly simplified version of the Bixon–Jortner theory of radiationless transitions; see M. Bixon and J. Jortner, J. Chem. Phys., 3284, 50 (1969). The quantity ρ = 1/ε is the density of states in the reservoir, so an alternative version of the result is

\(P=(a\pi \rho )^{2} cosec ^{2} (\pi \rho \Delta E)\)

If \(E_{i}\) lies halfway between the \(v = 0\) and \(v = 1\) levels, then \(\Delta E=\frac{1}{2}\varepsilon \), and \(P=(a\pi \rho )^{2} \). Although P appears to be infinite for ΔE = 0, that is an artefact of first-order perturbation theory.
