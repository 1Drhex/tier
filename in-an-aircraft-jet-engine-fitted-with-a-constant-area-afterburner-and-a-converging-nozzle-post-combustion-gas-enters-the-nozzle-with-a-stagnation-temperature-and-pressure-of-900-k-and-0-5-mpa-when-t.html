Guest Question.

In an aircraft jet engine fitted with a constant area afterburner and a converging nozzle, post-combustion gas enters the nozzle with a stagnation temperature and pressure of 900 K and 0.5 MPa, when the afterburner is not lit. With the afterburner lit, the stagnation temperature increases to 1900 K with a 15 percent loss in stagnation pressure at the nozzle inlet. If the mass flow rate has to bemaintained at 90 kg/s, determine the required nozzle exit area in both cases. Also, determine the thrust augmentation with afterburner operation, assuming that the engine is on a static test stand at sea level (ambient pressure 0.1MPa). Assume isentropic process for the nozzle. For the gas, take \(\gamma =4/ 3\) and \(C_{p} =1.148\) kJ/kg.K

Admin
Solution

When the afterburner is not lit, the stagnation pressure at entry to the nozzle is \(P_{0} =0.5 MPa.\) The critical pressure corresponding to this stagnation pressure is

\(P^{\ast }= P_{0} \left(\frac{2}{\gamma + 2} \right) ^{\frac{\gamma }{\gamma -1} }=0.27 MPa.\)

Since the ambient pressure \(P_{\infty } =0.1MPa\) is less than \(P^{\ast } \), the nozzle is choked and so the exit static pressure \(P_{e} =P^{\ast } =0.27 MPa.\) The mass flow rate is thus given by

\(\dot{m} =\frac{P_{0}A_{throat} }{\sqrt{T_{0} } } \sqrt{\frac{\gamma }{R}\left(\frac{2}{\gamma + 1} \right) ^{\left(\gamma + 1\right)/ \left(\gamma -1\right) } }\)

When the afterburner is not lit, \(P_{0} =0.5 MPa \) and \(T_{0} =900 K.\) Substituting these values, we get

\(A_{throat} =0.1359 m^{2}\)

Also, the exit static temperature can be evaluated as

\(T_{e} =T^{\ast } =T_{0}\left(\frac{2}{\gamma + 1} \right) =771K\)

and the exit velocity is thus

\(u_{e} =\sqrt{\gamma RT_{e} } =543 m/ s.\)

When the afterburner is lit, \(P_{0} =0.5\times 0.85=0.425 MPa\) and the critical pressure corresponding to this stagnation pressure is

\(P^{\ast } =P_{0} \left(\frac{2}{\gamma + 1} \right) ^{\frac{\gamma }{\gamma -1} }=0.229 MPa .\)

Since the ambient pressure \(P_{\infty } =0.1 MPa\) is less than \(P^{\ast } ,\) the nozzle is choked in this case also and so the exit static pressure \(P_{e} =P^{\ast } =0.229\)

MPa. With \(T_{0} =1900 K,\) we can get

\(A_{throat} =0.1359 \left(\frac{0.5}{0.425} \right)\sqrt{\frac{1900}{900} }0.3817 m^{2}\)

The exit static temperature \(T_{e} \) and the exit velocity \(u_{e} \) can be calculated to be 1629 K and 789 m/s in the same manner as before.

From Eqn. 6.4, thrust developed is given as

\(\tau =\dot{m} \left(u_{e} – u_{\infty } \right) + \left(P_{e}- P_{\infty } \right) A_{e}.\)

 

\(\tau =\dot{m} u_{e} + \left(P_{e}- P_{\infty } \right) A_{e},\)

where we have set \(u_{\infty }=0,\) since the engine is on a static thrust stand. Therefore,

Thrust (no afterburner)=48.87 kN + 23.103 kN = 71.973 kN

and Thrust (with afterburner) = 71 kN + 30 kN = 101 kN

Thrust augmentation due to afterburner is about 40 percent. Note that
the pressure thrust in both the cases is non-zero due to under-expanded
operation of the nozzle.
