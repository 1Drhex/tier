Guest Question.

Pure gaseous reactant \(A\left(C_{A 0}=100\right.\) millimol/liter) is fed at a steady rate into a mixed flow reactor \((V=0.1\) liter \()\) where it dimerizes \((2 \mathrm{~A} \rightarrow \mathrm{R}) .\) For different gas feed rates the following data are obtained:

\(\begin{array}{r|cccc}\text { Run number } & 1 & 2 & 3 & 4 \\v_{0} \text { , liter/hr } & 10.0 & 3.0 & 1.2 & 0.5 \\C_{\text {Af }} \text { , millimol/liter } & 85.7 & 66.7 & 50 & 33.4\end{array}\)

Find a rate equation for this reaction.

Admin
Solution

For this stoichiometry, \(2 \mathrm{~A} \rightarrow \mathrm{R}\), the expansion factor is

\(\varepsilon_{\mathrm{A}}=\frac{1-2}{2}=-\frac{1}{2}\)

and the corresponding relation between concentration and conversion is

\(\frac{C_{\mathrm{A}}}{C_{\mathrm{A} 0}}=\frac{1-X_{\mathrm{A}}}{1+\varepsilon_{\mathrm{A}} X_{\mathrm{A}}}=\frac{1-X_{\mathrm{A}}}{1-\frac{1}{2} X_{\mathrm{A}}}\)

or

\(X_{\mathrm{A}}=\frac{1-C_{\mathrm{A}} / C_{\mathrm{A} 0}}{1+\varepsilon_{\mathrm{A}} C_{\mathrm{A}} / C_{\mathrm{A} 0}}=\frac{1-C_{\mathrm{A}} / C_{\mathrm{A} 0}}{1-C_{\mathrm{A}} / 2 C_{\mathrm{A} 0}}\)

The conversion for each run is then calculated and tabulated in column 4 of Table E5.2.

From the performance equation, Eq. 11, the rate of reaction for each run is given by

\(\left(-r_{\mathrm{A}}\right)=\frac{v_{0} C_{\mathrm{A} 0} X_{\mathrm{A}}}{V}, \quad\left[\frac{\text { millimol }}{\text { liter } \cdot \mathrm{hr}}\right]\)

These values are tabulated in column 5 of Table E5.2. Having paired values of \(r_{\mathrm{A}}\) and \(C_{\mathrm{A}}\) (see Table E5.2) we are ready to test various kinetic expressions. Instead of separately testing for first-order (plot \(r_{\mathrm{A}}\) vs. \(\left.C_{\mathrm{A}}\right),\) second-order \(\left(\right.\) plot \(r_{\mathrm{A}}\) vs. \(\left.C_{\mathrm{A}}^{2}\right),\) etc., let us test directly for \(n\) th-order kinetics. For this take logarithms of \(-r_{\mathrm{A}}=k C_{\mathrm{A}}^{n},\) giving

\(\log \left(-r_{\mathrm{A}}\right)=\log k+n \log C_{\mathrm{A}}\)

For \(n\) th-order kinetics this data should give a straight line on a log \(\left(-r_{\mathrm{A}}\right)\) vs. log \(C_{\mathrm{A}}\) plot. From columns 6 and 7 of Table E5.2 and as shown in Fig. E5.2, the

four data points are reasonably represented by a straight line of slope \(2,\) so the rate equation for this dimerization is

\(-r_{\mathrm{A}}=\left(0.36 \frac{\text { liter }}{\mathrm{hr} \cdot \text { millimol }}\right) C_{\mathrm{A}}^{2}, \quad\left[\frac{\text { millimol }}{\text { liter } \cdot \mathrm{hr}}\right]\)

Comment. If we ignore the density change in our analysis (or put \(\varepsilon_{\mathrm{A}}=0\) and use \(\left.C_{\mathrm{A}} / C_{\mathrm{A} 0}=1-X_{\mathrm{A}}\right)\) we end up with an incorrect rate equation (reaction order \(n \cong 1.6)\) which when used in design would give wrong performance predictions
