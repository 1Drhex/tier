Guest Question.

(a) A sinusoidal voltage with \( V_{ eff }=10 V \) is connected across \( Z_{1}=1+j \) as shown in Fig. 10-7(a). Find \( i_{1}, I _{1, \text { eff }}, p_{1}(t), P_{1}, Q_{1}, \text { power factor } pf _{1} \text {, and } S _{1} \). (b) Repeat part (a) replacing the load \( Z_{1} \) in (a) by \( Z_{2}=1-j \), as shown in Fig. 10-7(b). (c) Repeat part (a) after connecting in parallel \( Z_{1} \text { in }(a) \text { and } Z_{2} \text { in (b) } \) as shown in Fig. 10-7(c).

Admin
Solution

Let \( v=10 \sqrt{2} \cos \omega t \).

(a) See Fig. 10-7(a).

\(\begin{aligned}&Z_{1}=\sqrt{2} \angle 45^{\circ} \\&i_{1}=10 \cos \left(\omega t-45^{\circ}\right) \\& I _{1, \text { eff }}=5 \sqrt{2} \angle-45^{\circ} \\&p_{1}(t)=(100 \sqrt{2}) \cos \omega t \cos \left(\omega t-45^{\circ}\right) \\&\quad=50+(50 \sqrt{2}) \cos \left(2 \omega t-45^{\circ}\right) W \\&P_{1}=V_{ eff } I_{1, eff } \cos 45^{\circ}=50 W \\&Q_{1}=V_{ eff } I_{1, eff } \sin 45^{\circ}=50 var \\& S _{1}=P_{1}+j Q_{1}=50+j 50 \\&S_{1}=\left| S _{1}\right|=50 \sqrt{2}=70.7 VA \\& pf _{1}=0.707 \text { lagging }\end{aligned}\)

 

(b) See Fig. 10-7(b)

 

\(\begin{aligned}&Z_{2}=\sqrt{2} \angle-45^{\circ} \\&i_{2}=10 \cos \left(\omega t+45^{\circ}\right) \\& I _{2, \text { eff }}=5 \sqrt{2} \angle 45^{\circ} \\&p_{2}(t)=(100 \sqrt{2}) \cos \omega t \cos \left(\omega t+45^{\circ}\right) \\&\quad=50+(50 \sqrt{2}) \cos \left(2 \omega t+45^{\circ}\right) W \\&P_{2}=V_{\text {eff }} I_{2, \text { eff }} \cos 45^{\circ}=50 W \\&Q_{2}=-V_{\text {eff }} I_{2, \text { eff }} \sin 45^{\circ}=-50 var \\& S _{2}=P_{2}+j Q_{2}=50-j 50 \\&S_{2}=\left| S _{2}\right|=50 \sqrt{2}=70.7 VA \\& pf _{2}=0.707 \text { leading }\end{aligned}\)

 

(c) See Fig. 10-7(c).

 

\(\begin{aligned}&Z=Z_{1} \| Z_{2}=\frac{(1+j)(1-j)}{(1+j)+(1-j)}=1 \\&i=10 \sqrt{2} \cos \omega t \\& I _{ eff }=10 \\&p(t)=200 \cos ^{2} \omega t=100+100 \cos 2 \omega t W \\&P=V_{ eff } I_{ eff }=100 W \\&Q=0 \\& S =P=100 \\&S=| S |=100 VA \\& pf =1\end{aligned}\)

 

The results of part (c) may also be derived from the observation that for the \( Z_{1} \| Z_{2} \text { combination, } i=i_{1}+i_{2} \) and, consequently,

\(\begin{aligned}p(t) &=p_{1}(t)+p_{2}(t) \\&=\left[50+(50 \sqrt{2}) \cos \left(2 \omega t-45^{\circ}\right)\right]+\left[50+(50 \sqrt{2}) \cos \left(2 \omega t+45^{\circ}\right)\right] \\&=100+100 \cos 2 \omega t W \\P=& P_{1}+P_{2}=50+50=100 W \\Q=& Q_{1}+Q_{2}=50-50=0 \\S=& 100<S_{1}+S_{2}\end{aligned}\)

 

The power triangles are shown in Figs. 10-7(a), (b), and (c). Figure 10-7(d) shows the plots of v, i, and p for the three loads.
