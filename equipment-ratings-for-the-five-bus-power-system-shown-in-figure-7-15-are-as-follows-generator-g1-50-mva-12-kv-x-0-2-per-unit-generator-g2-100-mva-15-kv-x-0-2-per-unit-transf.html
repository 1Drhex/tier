Guest Question.

Equipment ratings for the five-bus power system shown in Figure 7.15 are as follows:

Generator G1: \(50 MVA , 12 kV , X ^{\prime \prime}=0.2 \text { per unit }\)
Generator G2: \(100 MVA , 15 kV , X ^{\prime \prime}=0.2 \text { per unit }\)
Transformer T1: \(50 MVA , 10 kV Y / 138 kV Y , X =0.10 \text { per unit }\)
Transformer T2: \(100 MVA , 15 kV \Delta / 138 kV Y , X =0.10 \text { per unit }\)
Each 138-kV line: \(X _{1}=40 \Omega\).

A three-phase short circuit occurs at bus 5, where the prefault voltage is 15 kV. Prefault load current is neglected. (a) Draw the positive-sequence reactance diagram in per unit on a 100-MVA, 15-kV base in the zone of generator G2. Determine (b) the Thévenin equivalent at the fault, (c) the subtransient fault current in per unit and in kA rms, and (d) contributions to the fault from generator G2 and from transformer T2.

Admin
Solution

(a)

\(\begin{aligned}&X_{g 1}^{\prime \prime}=(0.20)\left(\frac{12}{10}\right)^{2}\left(\frac{100}{50}\right)=0.576 \text { per unit } \\&X_{g 2}^{\prime \prime}=0.20 \text { per unit } \\&X_{T 1}=(0.10)\left(\frac{100}{50}\right)=0.20 \text { per unit } \\&X_{T 2}=0.10 \text { per unit } \quad X_{\text {Line }}=40 / 190.44=0.21 \text { per unit }\end{aligned}\)

(b)

\(\begin{aligned}X_{T h} &=(0.20) / /(0.576+0.20+0.21+0.21+0.10) \\&=0.20 / / 1.296=\underline{\underline{0.1733} }\text { per unit } \\V_{F} &=\underline{\underline{1.0}} \text { per unit }\end{aligned}\)

(c)

\(\begin{aligned}\bar{I}_{F}^{\prime \prime} &=\frac{\bar{V}_{F}}{\bar{Z}_{T h}}=\frac{1.0 \angle 0^{\circ}}{j 0.1733}=\underline{\underline{-j 5.772}} \text { per unit } \\I_{b a s e 2} &=\frac{100}{15 \sqrt{3}}=3.849 kA \\\bar{I}_{F}^{\prime \prime} &=(-j 5.772)(3.849)=\underline{\underline{-j 22.21}} kA\end{aligned}\)

(d)

\(\begin{aligned}&\bar{I}_{g 2}^{\prime \prime}=\frac{1.0 \angle 0^{\circ}}{j 0.20}=\underline{\underline{-j 5.0}} \text { per unit }=(-j 5.0)(3.849)=\underline{\underline{-j 19.245} }kA\\&\bar{I}_{T 2}^{\prime \prime}=\frac{1.0 \angle 0^{\circ}}{j 1.296}=\underline{\underline{-j 0.7716}} \text { per unit }=(-j 0.7716)(3.849)=\underline{\underline{-j 2.970}} kA\end{aligned}\)
