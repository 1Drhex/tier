Guest Question.

The equation of state for one mole of a van der Waals fluid is

\((p + a/v^{2})(v − b) = RT\),

where p is the pressure, v is the volume per mole, T is the temperature, and a and b are constants. Calculate the following quantities and show that they agree with the results for an ideal gas in the limit a = b = 0:

(a) The isothermal compressibility, \(κ_{T} = −(1/v)(∂v/∂p)_{T}\)
(b) The isobaric coefficient of thermal expansion, \(α = (1/v)(∂v/∂T)_{p}\)
(c) The molar heat capacity difference, \((C_{p} − C_{V} )/N\)
(d) Show directly that \((∂p/∂T)_{v} = α/κ_{T}\) . Why is this true?

Admin
Solution

We first take the differential of the given equation to obtain

\(dp \left(v-b\right) + \left(p – \frac{a}{v^{2}} + \frac{2ab}{v^{3} } \right)dv = RdT.\)

(a)

\(k_{T} = -\frac{1}{v} \left(\frac{\partial v}{\partial p} \right) _{T} = \frac{1}{v} \left(v -b\right) \left(p-\frac{a}{v^{2} } + \frac{2ab}{v^{3} } \right)^{-1} \rightarrow \frac{1}{p}\) for an ideal gas,

(b)

\(\alpha =\frac{1}{v}\left(\frac{\partial v}{\partial T} \right)_{p} = \frac{R}{v} \left(p-\frac{a}{v^{2} } + \frac{2ab}{v^{3} } \right)^{-1} \rightarrow \frac{R}{pv} = \frac{1}{T}\) for an ideal gas,

(c)

\(\left(C_{p} – C_{V} \right)/ N = \frac{Tv\alpha ^{2} }{k_{T} } = \frac{R^{2}T }{\left(v-b\right) } \left(p – \frac{a}{v^{2} } + \frac{2ab}{v^{3} } \right)^{-1} \rightarrow \frac{R^{2}T }{pv} = R\) for an ideal gas,

(d)

\(\left(\frac{\partial P}{\partial T} \right)_{v} = \frac{R}{v-b} = -\left(\frac{\partial v}{\partial T} \right)_{p} / \left(\frac{\partial v}{\partial p} \right)_{T} = \frac{\alpha }{k_{T} }.\)

 

This relation is generally true, not just true for the van der Waals fluid, as can be seen from the differential

\(dv = \left(\frac{\partial v}{\partial T} \right)_{p} dT + \left(\frac{\partial v}{\partial p} \right)_{T} dp.\)
