Guest Question.

For the network in Fig. 21.36 with \(f_p\) provided:
a. Determine \(Q_l\).
b. Determine \(R_p\).
c. Calculate \(Z_{T_p}\).
d. Find C at resonance.
e. Find \(Q_p\).
f. Calculate the BW and cutoff frequencies.

Admin
Solution

a. \(Q_l=\frac{X_L}{R_l}=\frac{2\pi f_pL}{R_l}=\frac{2\pi(0.04 MHz)(1 mH)}{10 Ω} =25.12 \)

 

b. \(Q_l\geq 10.\enspace Therefore,\\[0.5cm] R_p ≅ Q_l^ 2 R_l = (25.12)^2 (10 Ω) = 6.31 kΩ\)

 

c. \(Z_{Tp} = R_s \parallel R_p = 40 kΩ \parallel 6.31 kΩ = 5.45 kΩ\)

 

d. \(Q_l\geq 10.\enspace Therefore,\\[0.5cm]f_p\cong \frac{1}{2\pi\sqrt{LC} } \\[0.5cm] and\hspace{30 pt} C=\frac{1}{4\pi^2f^2L}=\frac{1}{4\pi^2(0.04MHz)^2(1mH)}=15.83nF \)

 

e. \(Q_l\geq 10.\enspace Therefore,\\[0.5cm]Q_p=\frac{Z_{T_p}}{X_L} =\frac{R_s\parallel Q^2_lR_l}{2\pi f_pL}=\frac{5.45 kΩ}{2\pi(0.04 MHz)(1 mH)}=21.68 \)

 

f. \(BW=\frac{f_p}{Q_p}=\frac{0.04 MHz}{21.68}=1.85 kHz\\[0.5cm] f_1=\frac{1}{4\pi C}\left[-\frac{1}{R}+\sqrt{\frac{1}{R^2}+\frac{4C}{L} } \right]\\[0.5cm] =\frac{1}{4\pi (15.9nF)}\left[-\frac{1}{5.45 kΩ}+\sqrt{\frac{1}{(5.45 kΩ)^2}+\frac{4(15.9 nF)}{1mH} } \right]\\[0.5cm] =5.005\times10^{6}\left[-183.486\times10^{-6}+7.977\times10^{-3}\right]\\[0.5cm] =5.005\times10^6 \left[7.794\times10^{-3}\right]=39 kHz\\[0.5cm] f_2=\frac{1}{4\pi C}\left[\frac{1}{R}+\sqrt{\frac{1}{R^2}+\frac{4C}{L} } \right] \\[0.5cm] =5.005\times10^6\left[183.486\times10^{-6}+7.977\times10^{-3}\right]\\[0.5cm] =5.005\times10^6\left[8.160\times10^{-3}\right] =40.84 kHz\)

 

Note that \(f_2 – f_1 = 40.84kHz – 39kHz = 1.84kHz\), confirming our solution for the bandwidth above. Note also that the bandwidth is not symmetrical about the resonant frequency, with 1 kHz below and 840 Hz above.
