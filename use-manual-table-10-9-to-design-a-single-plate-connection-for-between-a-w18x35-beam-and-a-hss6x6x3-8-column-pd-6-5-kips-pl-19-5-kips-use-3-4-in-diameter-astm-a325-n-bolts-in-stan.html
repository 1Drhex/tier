Guest Question.

Use Manual Table 10-9 to design a single-plate connection for between a W18×35 beam and a HSS6×6×3/8 column.
\(\begin{array}{l}P_{D}=6.5 \text { kips } \\P_{L}=19.5 \text { kips }\end{array}\)
Use 3/4-in. diameter ASTM A325-N bolts in standard holes and 70 ksi weld electrode.

Admin
Solution

Material Properties:

Beam ASTM A992 \(F_{y}=50 ksi \quad F_{u}=65 ksi\)

Column ASTM A500 Gr. B \(F_{y}=46 ksi \quad F_{u}=58 ksi\)

Plate ASTM A36 \(F_{y}=36 ksi \quad F_{u}=58 ksi\)

Manual
Tables 2-3
and 2-4

Geometric Properties:

Beam W18×35 \(d=17.7 \text { in. } \quad t_{w}=0.300 \text { in. } \quad T=15 1 / 2 in .\)

Column HSS6×6×3/8 B = H = 6.00 in. t = 0.349 in. b/t = 14.2

Manual
Tables 1-1
and 1-12

Determine applicability of Specification Section K1

Strength: \(F_{y}=46 ksi <52 ksi\) o.k.

Ductility: \(\frac{F_{y}}{F_{u}}=\frac{46 ksi }{58 ksi }=0.79<0.80\) o.k.

Section K1.2

Determine if a single plate connection is suitable (the HSS wall is not slender)

Slenderness: \(\lambda=1.40 \sqrt{\frac{E}{F_{y}}}=1.40 \sqrt{\frac{29,000 ksi }{46 ksi }}=35.2>14.2\) o.k.

Table B4.1
Case 12

Calculate the required strength

LRFD ASD
\(\begin{aligned}R_{u} &=1.2(6.50 kips )+1.6(19.5 kips ) \\&=39.0 kips\end{aligned}\) \(\begin{aligned}R_{a} &=6.50 \text { kips }+19.5 \text { kips } \\&=26.0 \text { kips }\end{aligned}\)

Calculate maximum single-plate thickness

\(t_{p \max }=\frac{F_{u} t}{F_{y p}}=\frac{58 ksi (0.349 in. )}{36 ksi }=0.562 in \text { . }\)

Eqn. K1-10

Note: Limiting the single-plate thickness precludes a shear yielding failure of the HSS wall.

Section K1.5

Design the single-plate connection

Try 3 bolts and a 5/16-in. plate thickness with 1/4-in. fillet welds.

\(t_{p}=5 / 16 \text { in. }<0.562 \text { in. }\) o.k.

Obtain the available single plate connection strength from Manual Table 10-9

LRFD ASD
\(\phi R_{n}=47.7 \text { kips }>39.0 \text { kips }\) o.k. \(R_{n} / \Omega=31.8 \text { kips }>26.0 \text { kips }\) o.k.

Manual
Table 10-9

\(\text { Use a } PL 5 / 16 \times 4 1 / 2 \times 0^{\prime}-9 "\)

Check HSS shear rupture strength at welds

\(t_{\min }=\frac{6.19 D}{F_{u}}=\frac{6.19(4)}{58 ksi }=0.427 in. >t=0.349 in\) n.g.

Manual
Section 9

Since \(t<t_{\min }\), the shear rupture strength of the HSS is less than the weld strength. Consider reducing the weld size to 3/16 in.

\(t_{\min }=\frac{6.19 D}{F_{u}}=\frac{6.19(3)}{58 ksi }=0.320 \text { in. }<t=0.349 \text { in. }\) o.k.

Determine the weld shear strength

This is the special case where the load is not in the plane of the weld group. Use k = 0.
