Guest Question.

Consider two long parallel plates of width w as shown in Fig. 6-16. Both plates are isothermal at the (same) temperature T, and both have a gray, diffuse emittance of ε. The reflectance of the material is partly diffuse, partly specular, so that ε = 1 – \(ρ^{s} - ρ^{d}\) . The plates are separated by a distance h and are placed in a large, cold environment. Determine the local radiative heat fluxes along the plate using numerical quadrature.

Admin
Solution

From equation (6.18) \(E_{b}(r)-\int_{A}^{}{\left[1-\rho ^{s}(r^{\prime } ) \right] } E_{b}(r^{\prime } ) dF^{s}_{dA-dA^{\prime } } = \frac{q(r)}{\epsilon (r)}- \int_{A}^{}{\frac{\rho ^{d} (r^{\prime } )}{\epsilon (r^{\prime } )} } q(r^{\prime } ) dF^{s}_{dA-dA^{\prime } } + H^{s}_{o}(r)\) we find, for location xi on the lower plate,

\(E_{b} – (1- \rho ^{s} )E_{b} \left[\int_{0}^{w}{dF^{s}_{dx_{1} -dx_{1} ^{\prime } } } + \int_{0}^{w}{dF^{s}_{dx_{1} -dx_{2} ^{\prime } } } \right] = \frac{q(x_{1} )}{\epsilon } – \frac{\rho ^{d} }{\epsilon } \left[\int_{0}^{w} q (x^{\prime }_{1} ){dF^{s}_{dx_{1} -dx_{1} ^{\prime } } }+ \int_{0}^{w} q (x^{\prime }_{2} ){dF^{s}_{dx_{1} -dx_{2} ^{\prime } } }\right]\)

The necessary specular view factors are readily found from

\(dF^{s}_{dx_{1} -dx_{1} ^{\prime } } = \rho ^{s} dF_{dx_{1(2)} -dx_{1} ^{\prime } } + (\rho ^{s})^{3} dF_{dx_{1(212)} -dx_{1} ^{\prime } }+ …,\)
\(dF^{s}_{dx_{1} -dx_{2} ^{\prime } } = dF_{dx_{1} -dx_{2} ^{\prime } } + (\rho ^{s})^{2} dF_{dx_{1(21)} -dx_{2} ^{\prime } }+ … .\)

The view factor between two infinitely long parallel strips of infinitesimal width and separated by a distance kh(k- 1,2,…) is given by as

\(\frac{1}{2}\frac{(kh)^{2} dx^{\prime } }{\left[(kh)^{2} + ( x-x^{\prime } )^{2} \right]^{3/2} }\)

Thus,

\(dF^{s}_{dx_{1} -dx_{1} ^{\prime } } + dF^{s}_{dx_{1} -dx_{2} ^{\prime } }= dF_{dx_{1} -dx_{2} ^{\prime } }+ \rho ^{s} dF_{dx_{1(2)} -dx_{1} ^{\prime } } + (\rho ^{s})^{2} dF_{dx_{1(21)} -dx_{2} ^{\prime } }+ …\)
\(=\frac{1}{2}\sum\limits_{k=1}^{\infty }{(\rho ^{s} )^{k-1} }\frac{(kh)^{2} dx^{\prime } }{{\left[(kh)^{2} + ( x_{1} -x^{\prime } )^{2} \right]^{3/2} }}\),

where we have made use of \(x^{\prime }_{1} =x^{\prime }_{2} = x^{\prime }\) . This expression may be substituted into the governing integral equation. Realizing that, by symmetry, \(q(x^{\prime }_{1}) =q(x^{\prime }_{2}) = q(x^{\prime })\) and nondimensionalizing with ξ= x/h , W = w/h and ψ = q(ξ)/\(E_{b}\), lead to

\(1-(1-\rho ^{s} )\int_{0}^{W}{\frac{1}{2} }\sum\limits_{k=1}^{\infty }{\frac{(\rho ^{s})^{k-1} k^{2}d\xi ^{\prime } }{\left[k^{2} + (\xi -\xi ^{\prime } )^{2} \right]^{3/2} } } = \frac{\Psi (\xi )}{\epsilon } – \frac{\rho ^{d} }{\epsilon } \left[\int_{W}^{0}{\Psi (\xi ^{\prime } ) \frac{1}{2}\sum\limits_{k=1}^{\infty }{\frac{(\rho ^{s})^{k-1} k^{2}d\xi ^{\prime } }{\left[k^{2} + (\xi -\xi ^{\prime } )^{2} \right]^{3/2} }} } \right] .\)

As in this equation may be solved by numerical quadrature as

\(\Psi _{i} -\rho ^{d} W\sum\limits_{j=1}^{J}{c_{j}\Psi _{j}f_{ij} } = \epsilon \left[1-(1-\rho ^{s} )W \sum\limits_{j=1}^{J}{c_{j}f_{ij} }\right]\) ,

where \(\Psi _{i}\) , is evaluated at J nodal positions\(\xi _{i}\), i = 1,2,…, J, and the \(c_{j}\) are weight coefficients for the numerical integration. The \(f_{ij}\) are an abbreviation for the integration kernel,

\(f_{ij}= \frac{1}{2} \sum\limits_{k=1}^{\infty }{\frac{k^{2}(\rho ^{s})^{k-1} }{\left[ k^{2}+ (\xi_{i} -\xi_{i} ^{\prime } )^{2} \right]^{3/2} } }\).

They must be evaluated by summing as many terms as necessary (decreasing as \((\rho ^{s})^{k-1}\)/k for large k). Results for the same simple J = 5 quadrature of are given in Fig. 6-17, together with “exact” solutions (high-order quadrature). The results show that, for W = w/h = 1, the heat loss from the plates decreases if reflection is specular: Specular reflection traps emitted radiation somewhat more through repeated reflections between the plates.
Note that, if both surfaces are purely specular, the heat flux may be calculated directly (i.e., no solution of an integral equation is necessary). This calculation was first done for the parallelplate case by Eckert and Sparrow [4], In general, equation (6.18) is actually easier to solve than its diffuse-reflection counterpart if some or all of the surfaces are purely specular. However, the necessary specular view factors are generally much more diflficult—if not impossible—^to evaluate. Such a case arises, for example, for curved surfaces with multiple specular reflections.
Since the specular view factors for such problems are most easily found from statistical methods, such as the Monte Carlo method (Chapter 20), it is usually best to solve the entire heat transfer problem using the Monte Carlo method.
