Guest Question.

Use the WKB approximation to find the allowed energies of the general power-law potential:

\( V(x)=\alpha|x|^{\nu} \),

where ν is a positive number. Check your result for the case v=2 Answer :

\( E_{n}=\alpha\left[(n-1 / 2) \hbar \sqrt{\frac{\pi}{2 m \alpha}} \frac{\Gamma\left(\frac{1}{v}+\frac{3}{2}\right)}{\Gamma\left(\frac{1}{v}+1\right)}\right]^{\left(\frac{2 v}{v+2}\right)} \). (9.54).

Admin
Solution

Equation 9.52 ⇒ \( \left(n-\frac{1}{2}\right) \pi \hbar=2 \int_{0}^{x_{2}} \sqrt{2 m\left(E-\alpha x^{\nu}\right)} d x=2 \sqrt{2 m E} \int_{0}^{x_{2}} \sqrt{1-\frac{\alpha}{E} x^{\nu}} d x ; \quad E=\alpha x_{2}^{\nu} . \quad \text { Let } \).

\( \int_{x_{1}}^{x_{2}} p(x) d x=\left(n-\frac{1}{2}\right) \pi \hbar, \quad \text { with } \quad n=1,2,3, \ldots \) (9.52).

\( z \equiv \frac{\alpha}{E} x^{\nu}, \text { so } x=\left(\frac{z E}{\alpha}\right)^{1 / \nu} ; \quad d x=\left(\frac{E}{\alpha}\right)^{1 / \nu} \frac{1}{\nu} z^{\frac{1}{\nu}-1} d z \). Then

\( \left(n-\frac{1}{2}\right) \pi \hbar=2 \sqrt{2 m E}\left(\frac{E}{\alpha}\right)^{1 / \nu} \frac{1}{\nu} \int_{0}^{1} z^{\frac{1}{\nu}-1} \sqrt{1-z} d z=2 \sqrt{2 m E}\left(\frac{E}{\alpha}\right)^{1 / \nu} \frac{1}{\nu} \frac{\Gamma(1 / \nu) \Gamma(3 / 2)}{\Gamma\left(\frac{1}{\nu}+\frac{3}{2}\right)} \).

\( =2 \sqrt{2 m E}\left(\frac{E}{\alpha}\right)^{1 / \nu} \frac{\Gamma\left(\frac{1}{\nu}+1\right) \frac{1}{2} \sqrt{\pi}}{\Gamma\left(\frac{1}{\nu}+\frac{3}{2}\right)}=\sqrt{2 \pi m E}\left(\frac{E}{\alpha}\right)^{1 / \nu} \frac{\Gamma\left(\frac{1}{\nu}+1\right)}{\Gamma\left(\frac{1}{\nu}+\frac{3}{2}\right)} \).

\( E^{\frac{1}{\nu}+\frac{1}{2}}=\frac{\left(n-\frac{1}{2}\right) \pi \hbar}{\sqrt{2 \pi m}} \alpha^{1 / \nu} \frac{\Gamma\left(\frac{1}{\nu}+\frac{3}{2}\right)}{\Gamma\left(\frac{1}{\nu}+1\right)} \); \( E_{n}=\left[\left(n-\frac{1}{2}\right) \hbar \sqrt{\frac{\pi}{2 m \alpha}} \frac{\Gamma\left(\frac{1}{\nu}+\frac{3}{2}\right)}{\Gamma\left(\frac{1}{\nu}+1\right)}\right]^{\left(\frac{2 \nu}{\nu+2}\right)} \).

\( \text { For } \nu=2: \quad E_{n}=\left[\left(n-\frac{1}{2}\right) \hbar \sqrt{\frac{\pi}{2 m \alpha}} \frac{\Gamma(2)}{\Gamma(3 / 2)}\right] \alpha=\left(n-\frac{1}{2}\right) \hbar \sqrt{\frac{2 \alpha}{m}} \).

For a harmonic oscillator, with \( \alpha=\frac{1}{2} m \omega^{2}, \quad E_{n}=\left(n-\frac{1}{2}\right) \hbar \omega(n=1,2,3, \ldots) \).
