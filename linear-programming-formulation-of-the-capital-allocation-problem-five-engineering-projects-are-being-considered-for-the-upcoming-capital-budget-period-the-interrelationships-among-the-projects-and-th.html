Solution

The objective function and constraints for this problem are written as follows:
Maximize

Net PW = 13.4 \(X_{B1}\) + 8.0 \(X_{B2} \) − 1.3 \(X_{C1}\) + 0.9 \(X_{C2}\) + 0.9 \(X_{D}\)

subject to
50 \(X_{B1} \) + 30\( X_{B2} \) + 14 \(X_{C1}\) + 15 \(X_{C2}\) + 10 \(X_{D}\) ≤ 48;

(constraint on investment funds)

\(X_{B1} + X_{B2}\) ≤ 1 ;

(B1 and B2 are mutually exclusive)

\(X_{C1} + X_{C1}\) ≤ \( X_{B2}\) ;

(C1 or C2 is contingent on B2)

\(X_{D} \) ≤ \( X_{C1}\) ;

(D is contingent on C1)

\(X_{j}\) = 0 or 1 for j = B1, B2, C1, C2, D.
(No fractional projects are allowed.)

A problem such as this could be solved readily by using the simplex method of linear programming if the last constraint (\(X_{j} \) = 0 or 1) were not present.With that constraint included, the problem is classified as a linear integer programming problem. (Many computer programs are available for solving large linear integer programming problems.)
