Guest Question.

Find the shaft and hole dimensions for a loose running fit with a 34-mm basic size.

Admin

, we find that tolerance grade IT11 is 0.160 mm. The symbol 34H11/c11 therefore says that \(\Delta D=\Delta d=0.160\) mm.
Using Eq. (7–36) \(\Delta_{max}=D=\Delta D \ D_{min}=D\)
for the hole, we get
\(D_{max}=D+\Delta D=34+0.160=34.160 \ mm \\ D_{min}=D=34.000 \ mm\)

, the fundamental deviation is \(\delta_F= −0.120\) mm.
Using Eq. (7–37),
\(d_{max}=d+\delta_F \ \ d_{min}=d+\delta_F-\Delta d\)
we get for the shaft dimensions
\(d_{max}=d+ \delta_f = 34+ \left(-0.120\right)=33.880 \ mm \\ d_{min}=d+ \delta_F- \Delta d = 34 + \left(-0.120\right)-0.160=33.720\) mm
