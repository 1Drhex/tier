Guest Question.

A WT8 × 25 structural steel section (see Appendix B for cross-sectional properties) is used for a 20-ft column. Assume pinned connections at each end of the column. Determine:
(a) the slenderness ratio.
(b) the Euler buckling load. Use E = 29,000 ksi for the steel.
(c) the axial stress in the column when the Euler load is applied.

Admin
Solution

The following section properties for a standard steel WT8 × 25 shape are given in Appendix B:

\(A=7.37 \text { in. }^{2}, I_{x}=42.3 in .^{4}, r_{x}=2.40 \text { in., } I_{y}=18.6 \text { in. }^{4}, r_{y}=1.59 \text { in. }\)

(a) Slenderness ratio:

\(\frac{L}{r}=\frac{(20 ft )(12 in . / ft )}{1.59 in .}=150.9\)

(b) Euler buckling load:

\(P_{c r}=\frac{\pi^{2} E I}{L^{2}}=\frac{\pi^{2}(29,000 ksi )\left(18.6 in. ^{4}\right)}{[(20 ft )(12 in. / ft )]^{2}}=92.425 kips =92.4 kips\)

(c) Axial stress at \(P _{ c r }\):

\(\sigma=\frac{92.425 kips }{7.37 in .^{2}}=12.54 ksi\)
