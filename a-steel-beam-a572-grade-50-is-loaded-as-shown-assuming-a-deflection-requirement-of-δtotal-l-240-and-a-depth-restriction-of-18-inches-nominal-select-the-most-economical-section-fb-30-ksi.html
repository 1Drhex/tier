Guest Question.

A steel beam (A572 Grade 50) is loaded as shown. Assuming a deflection requirement of \( \Delta_{\text {total }} \) = L/240 and a depth restriction of 18 inches nominal, select the most economical section.

\(F_{b}\) = 30 ksi; \(F_{v}\) = 20 ksi; E = 30 × 10³ ksi

Admin
Solution

\(V_{\max }\)= 24 k

\(M_{\max }\)=238 k-ft.

Steel beams are usually designed for bending. Once a trial section has been selected, shear and deflection are checked.

Bending

\(S_{\text {req }^{\prime} d}=\frac{M}{F_{b}}=\frac{238 \text { k-ft. } \times 12 \text { in. } / ft. }{30 ksi }\)=95.2 in.³

Try W18 ×55.

\(\left(S_{x}=98.3 in .^{3} ; I_{x}=890 in .^{4} ; t_{ w }=0.39^{\prime \prime} ; d=18.11^{\prime \prime}\right) \)

 

\(M_{\text {add. }}=\frac{\omega_{\text {beam }} L^{2}}{8}=\frac{55 \# / ft.\left(28^{\prime}\right)^{2}}{8}=5,390 \#- ft.\)

 

\(S_{\text {add. }}=\frac{M_{\text {add. }}}{F_{ b }}=\frac{5.39 k \text { -ft. } \times 12 \text { in. } / ft .}{30 ksi }=2.2 in .{ }^{3}\)

 

\(S_{\text {total }}\) = 95.2 in.³ + 2.2 in.³ = 97.4 in.³ \(< \) 98.3 in.³ \(\therefore OK \)

Shear Check

\(V_{\text {add. }}=\frac{\omega_{\text {beam }} L}{2}=\frac{55 \# / ft.\left(28^{\prime}\right)}{2}=770 \#\)

 

\(f_{ v _{ average }}=\frac{V_{\max }}{t_{ w } d}=\frac{24,000 \#+770 \#}{\left(0.39^{\prime \prime}\right)\left(18.11^{\prime \prime}\right)}=3,510 psi\)

 

\(f_{ v _{\text {average }}}=3,510 \text { psi }<20,000 \text { psi } \therefore \text { OK }\)

Deflection Check

\(\Delta_{\text {allow }}=\frac{L}{240}=\frac{28^{\prime} \times 12 \text { in. } / ft .}{240}=1.4^{\prime \prime}\)

 

\(\Delta_{\text {actual }}=\frac{P L^{3}}{48 E I}+\frac{5 \omega L^{4}}{384 E I}\)

 

\(\Delta_{\text {actual }}=\frac{20 k \left(28^{\prime}\right)^{3} 1,728}{48\left(30 \times 10^{3}\right)(890)}+\frac{5(1.06 k / ft .)\left(28^{\prime}\right)^{4} 1,728}{(384)\left(30 \times 10^{3}\right)(890)}\)

 

\(\Delta_{\text {actual }}=0.59^{\prime \prime}+0.55^{\prime \prime}=1.14^{\prime \prime}<1.4^{\prime \prime} \therefore OK\)
