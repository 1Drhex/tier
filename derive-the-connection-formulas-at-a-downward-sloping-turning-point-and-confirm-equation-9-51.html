Guest Question.

Derive the connection formulas at a downward-sloping turning point, and confirm Equation 9.51.

\( \psi(x) \approx \begin{cases}\frac{D^{\prime}}{\sqrt{|p(x)|}} \exp \left[-\frac{1}{\hbar} \int_{x}^{x_{1}}\left|p\left(x^{\prime}\right)\right| d x^{\prime}\right], & x<x_{1} \\ \frac{2 D^{\prime}}{\sqrt{p(x)}} \sin \left[\frac{1}{\hbar} \int_{x_{1}}^{x} p\left(x^{\prime}\right) d x^{\prime}+\frac{\pi}{4}\right], & x>x_{1}\end{cases} \) (9.51).

Admin
Solution

Shift origin to the turning point.

\( \psi_{ WKB }= \begin{cases}\frac{1}{\sqrt{|p(x)|}} D e^{-\frac{1}{\hbar} \int_{x}^{0}\left|p\left(x^{\prime}\right)\right| d x^{\prime}} \quad(x<0) \\ \frac{1}{\sqrt{p(x)}}\left[B e^{\frac{1}{\hbar} \int_{0}^{x} p\left(x^{\prime}\right) d x^{\prime}}+C e^{-\frac{1}{\hbar} \int_{0}^{x} p\left(x^{\prime}\right) d x^{\prime}}\right] & (x>0)\end{cases} \).

Linearized potential in the patching region:

\( V(x) \approx E+V^{\prime}(0) x \text {. Note : } V^{\prime}(0) \text { is negative. } \quad \frac{d^{2} \psi_{p}}{d x^{2}}=\frac{2 m V^{\prime}(0)}{\hbar^{2}} x \psi_{p}=-\alpha^{3} x \psi_{p}, \text { where } \alpha \equiv\left(\frac{2 m\left|V^{\prime}(0)\right|}{\hbar^{2}}\right)^{1 / 3} \).

\( \psi_{p}(x)=a A i(-\alpha x)+b B i(-\alpha x) \). (Note change of sign, as compared with Eq. 9.38).

\( \psi_{p}(x)=a \operatorname{Ai}(\alpha x)+b \operatorname{Bi}(\alpha x) \) (9.38).

\( p(x)=\sqrt{2 m\left[E-E-V^{\prime}(0) x\right]}=\sqrt{-2 m V^{\prime}(0) x}=\sqrt{2 m\left|V^{\prime}(0)\right| x}=\sqrt{\alpha^{3} \hbar^{2} x}=\hbar \alpha^{3 / 2} \sqrt{x} \).

Overlap region 1 (x < 0):

\( \int_{x}^{0}\left|p\left(x^{\prime}\right)\right| d x^{\prime}=\hbar \alpha^{3 / 2} \int_{x}^{0} \sqrt{-x^{\prime}} d x^{\prime}=\left.\hbar \alpha^{3 / 2}\left(-\frac{2}{3}\left(-x^{\prime}\right)^{3 / 2}\right)\right|_{x} ^{0}=\frac{2}{3} \hbar \alpha^{3 / 2}(-x)^{3 / 2}=\frac{2}{3} \hbar(-\alpha x)^{3 / 2} \).

\( \psi_{ WKB } \approx \frac{1}{\hbar^{1 / 2} \alpha^{3 / 4}(-x)^{1 / 4}} D e^{-\frac{2}{3}(-\alpha x)^{3 / 2}} . \text { For large positive argument }(-\alpha x \gg 1): \)

\( \psi_{p} \approx a \frac{1}{2 \sqrt{\pi}(-\alpha x)^{1 / 4}} e^{-\frac{2}{3}(-\alpha x)^{3 / 2}}+b \frac{1}{\sqrt{\pi}(-\alpha x)^{1 / 4}} e^{\frac{2}{3}(-\alpha x)^{3 / 2}} . \quad \text { Comparing } \Rightarrow a=2 D \sqrt{\frac{\pi}{\alpha \hbar}} ; \quad b=0 \).

Overlap region 2 (x > 0):

\( \int_{0}^{x} p\left(x^{\prime}\right) d x^{\prime}=\hbar \alpha^{3 / 2} \int_{0}^{x} \sqrt{x^{\prime}} d x^{\prime}=\left.\hbar \alpha^{3 / 2}\left[\frac{2}{3}\left(x^{\prime}\right)^{3 / 2}\right]\right|_{0} ^{x}=\frac{2}{3} \hbar(\alpha x)^{3 / 2} \).

\( \psi_{ WKB } \approx \frac{1}{\hbar^{1 / 2} \alpha^{3 / 4} x^{1 / 4}}\left[B e^{i \frac{2}{3}(\alpha x)^{3 / 2}}+C e^{-i \frac{2}{3}(\alpha x)^{3 / 2}}\right] \). For large negative argument \( (-\alpha x \ll-1) \):

\( \psi_{p}(x) \approx a \frac{1}{\sqrt{\pi}(\alpha x)^{1 / 4}} \sin \left[\frac{2}{3}(\alpha x)^{3 / 2}+\frac{\pi}{4}\right]=\frac{a}{\sqrt{\pi}(\alpha x)^{1 / 4}} \frac{1}{2 i}\left[e^{i \pi / 4} e^{i \frac{2}{3}(\alpha x)^{3 / 2}}-e^{-i \pi / 4} e^{-i \frac{2}{3}(\alpha x)^{3 / 2}}\right](\text { remember }: b=0) \).

Comparing the two: \( B=\frac{a}{2 i} \sqrt{\frac{\alpha \hbar}{\pi}} e^{i \pi / 4}, C=-\frac{a}{2 i} \sqrt{\frac{\alpha \hbar}{\pi}} e^{-i \pi / 4} \).

Inserting the expression for a from overlap region \( 1: B=-i e^{i \pi / 4} D ; C=i e^{-i \pi / 4} D . \quad \text { For } x>0 \), then,

\( \psi_{ WKB }=\frac{-i D}{\sqrt{p(x)}}\left[e^{\frac{i}{\hbar} \int_{0}^{x} p\left(x^{\prime}\right) d x^{\prime}+i \frac{\pi}{4}}-e^{-\frac{i}{\hbar} \int_{0}^{x} p\left(x^{\prime}\right) d x^{\prime}-i \frac{\pi}{4}}\right]=\frac{2 D}{\sqrt{p(x)}} \sin \left[\frac{1}{\hbar} \int_{0}^{x} p\left(x^{\prime}\right) d x^{\prime}+\frac{\pi}{4}\right] \).

Finally, switching the origin back to\(x_1\):

\( \psi_{ WKB }(x)=\left\{\begin{array}{ll}<br /> \frac{D}{\sqrt{|p(x)|}} e^{-\frac{1}{\hbar} \int_{x}^{x_{1}}\left|p\left(x^{\prime}\right)\right| d x^{\prime}}, & \left(x<x_{1}\right) \\ \frac{2 D}{\sqrt{p(x)}} \sin \left[\frac{1}{\hbar} \int_{x_{1}}^{x} p\left(x^{\prime}\right) d x^{\prime}+\frac{\pi}{4}\right], & \left(x>x_{1}\right) \end{array}\right\} \).
