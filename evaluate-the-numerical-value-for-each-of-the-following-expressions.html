Guest Question.

Evaluate the numerical value for each of the following expressions.

  1. \(\int_{1}^{4}{ \frac{dx }{x} } \)
  2. \(\int_{0}^{\infty }{ e^{-2x}dx }\)
  3. \(\int_{0}^{\pi /3}{(3\cos\theta^{2}-1 )\sin \theta d\theta }\)

 

Admin
Solution

hese can be solved by substitution of the expressions in Table A.5.

TABLE A.5 Solutions to selected integrals. In these equations, a and b are constants, n is a
whole number, and C is the constant of integration.
\(\int{x^{n}dx }\) \(= \frac{1}{n+1}x^{n+1}+C \) \(\int{adx }\) = a(x + C)
\(\int{\frac{1}{x} dx } \) = ln x + C \(\int{e^{x} dx }\) \(= e^{x}+C\)
\(\int{\ln xdx }\) = x ln x – x + C \(\int{\frac{dx}{x(a+bx)} }\) \(= \frac{1}{a}\ln (\frac{a+bx}{x} )+C\)
\(\int{\sin xdx }\) = -cos x + C \( \int{\cos xdx }\) = sin x + C
\(\int{\sin ^{2}(ax)dx }\) \(= \frac{x}{2} -\frac{\sin (2ax)}{4a} +C\) \(\int{\cos ^{2}(ax) dx }\) \(= \frac{x}{2} +\frac{\sin (2ax)}{4a} +C\)
\(\int{[f(x)+g(x)]dx }\) \(\int{f(x)+g(x)dx }\) \(\int_{a}^{b}{dx}\) \(= x \mid ^{b}_{a}=b-a\)
\(\int_{0}^{\infty }{x^{n}e^{-ax^{2} }dx } \) \(= \frac{n!}{a^{n+1} } \) \(\int_{0}^{\infty }{e^{-ax^{2} } dx}\) \(= \frac{1}{2} (\frac{\pi}{a} )^{1/2 } \)
\(\int_{0}^{\infty }{xe^{-ax^{2} }dx } \) \(= \frac{1}{2a}\) \(\int_{0}^{\infty }{x^{2} e^{-ax^{2} } dx}\) \(= \frac{1}{4} (\frac{\pi}{a^{3} } )^{1/2 } \)
\(\int_{0}^{\infty }{x^{2n+1}e^{-ax^{2} }dx } \) \(= \frac{n!}{2a^{n+1} }\) \(\int_{0}^{\infty }{x^{2n} e^{-ax^{2} } dx}\) \(= \frac{[1\cdot 3\cdot 5 …(2n-1)]\sqrt{\pi } }{2^{n+1}a^{n+(1/2)} } \)
\(\int_{0}^{5}{x^{n}e^{-ax^{2} }dx }\) \(= \frac{n!}{a^{n+1} } – e^{-as}\sum\limits_{i=0}^{n}{\frac{n!s^{n-i} }{a^{i+1}(n-1)! } } \)

(a) \(\int_{1}^{4}{ \frac{dx }{x} } = \ln x \mid ^{4}_{1}= \ln 4- \ln 1 = 1.386 – 0 = 1.386\)

(b) \(\int_{0}^{\infty }{ e^{-2x}dx }= -\frac{1}{2} e^{-2x} \mid ^{\infty }_{0} = -\frac{1}{2}(e^{-\infty }- e^{0} )= -\frac{1}{2}(0-1) = \frac{1}{2}\)

(c) \(\int_{0}^{\pi /3 }{ (3x^{2}-1 ) \sin \theta d\theta}\)

\(= [-\cos ^{3}\theta +\cos \theta ] \mid ^{\pi /3}_{0} = [-(\frac{1}{2})^{3}+(\frac{1}{2}) ]-[-(1)^{3}+ (1) ]= \frac{3}{8}\)
