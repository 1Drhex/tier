(b) \(\mathbb{Z}^{+} \subseteq \mathbb{Z} \subseteq \mathbb{R}\)

(c) Let \(A\) be the set of all positive integers whose last two digits are 24 and let \(B\) be the set of all positive integers that are evenly divisible by 4 . Then if \(n\) is in \(A\), it is of the form

\(n=100 k+24\)
Admin
Solution

for some integer \(k\). (For example, \(36,524=100 \cdot 365+24\).) But then

\(n=100 k+24=4(25 k+6)\)

so \(n / 4=25 k+6\), which is an integer. Hence, \(n\) is evenly divisible by 4 , so it is in \(B\). Therefore, \(A \subseteq B\).
