Guest Question.

An undamped9 system is driven by the function,

\(\ddot{x}+16 x=\frac{F(t)}{m}=\frac{1}{m} \begin{cases}\cos 4 t & \text { if } 0 \leq t<\pi \\ 0 & \text { elsewhere }\end{cases}\)

The initial conditions are \(x(0)=0 m \text { and } \dot{x}(0)=1 m / s\) Solve for the response.

Admin
Solution

We cannot use the usual harmonic solution approach because the loading function does not act over all time. The term “harmonic function” implicitly means harmonic over all time. We can solve this problem using
Laplace transforms. Two shifting theorems for Laplace transforms are useful. They are provided here without proof:

\(\begin{aligned}L \left[F(t) e^{a t}\right] &= F (s-a) \\L [F(t-a) u(t-a)] &=e^{-a \sigma} F (s)\end{aligned}\)

To solve the equation above using Laplace transforms, we need to use the second shifting theorem. We can rewrite the loading as

\(F(t)=\cos 4 t-\cos 4 t u(t-\pi)=\cos 4 t-\cos 4(t-\pi) u(t-\pi)\)

where u(t) is the unit step function, and cos 4t = cos(4t− 4π). Taking the Laplace transform of each side of the governing equation, we obtain

\(\begin{aligned}&s^{2} X(s)-s x(0)-\dot{x}(0)+16 X(s) \\&=\frac{1}{m}\left[\frac{s}{s^{2}+16}-\frac{\varepsilon}{s^{2}+16} e^{-\pi o}\right]\end{aligned}\)

Substituting the initial conditions and solving for X(s), we find

\(X(s)=\frac{1}{s^{2}+16}+\frac{1}{m}\left[\frac{s}{\left(s^{2}+16\right)^{2}}-\frac{s}{\left(s^{2}+16\right)^{2}} e^{-\pi s}\right]\)

from which

\(\begin{aligned}x(t) &=\frac{1}{4} L ^{-1}\left[\frac{4}{s^{2}+16}\right]+\frac{1}{8 m} L ^{-1}\left[\frac{8 s}{\left(s^{2}+16\right)^{2}}\right] \\&-\frac{1}{8 m} L ^{-1}\left[\frac{8 s}{\left(s^{2}+16\right)^{2}} e^{-\pi s}\right] \\&=\frac{1}{4} \sin 4 t+\frac{1}{8 m} t \sin 4 t \\&-\frac{1}{8 m}(t-\pi) \sin 4(t-\pi) u(t-\pi) .\end{aligned}\)

This solution can be written more clearly as

\(x(t)= \begin{cases}\frac{1}{4} \sin 4 t+\frac{1}{8 m} t \sin 4 t, & 0 \leq t<\pi \\ \frac{1}{4} \sin 4 t+\frac{\pi}{8 m} \sin 4 t, & t \geq \pi\end{cases}\)

The plot of this response is shown in Figure 4.6 for the case \(m\) = 1 kg.
Note that the response grows during the loading phase, and then oscillates in steady-state after the load disappears. Because there is no damping, the oscillation continues without end.
