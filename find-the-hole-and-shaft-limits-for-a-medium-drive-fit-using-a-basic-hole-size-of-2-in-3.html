Guest Question.

Find the hole and shaft limits for a medium drive fit using a basic hole size of 2 in.

Admin
Solution

The IT6 tolerance for the shaft is Δd = 0.0006 in. Also, from Table A–14, the fundamental deviation is \(δ_{F}\) = 0.0017 in. Using Eq. (7–38), we get for the shaft that

\(d_{min} = d + δ_{F} d_{max} = d + δ_{F} +Δd\) (7–38)

\(d_{min} = d + δ_{F}= 2 + 0.0017 = 2.0017 in\)

\( d_{max} = d + δ_{F} +Δd= 2 + 0.0017 + 0.0006 = 2.0023 in\)
