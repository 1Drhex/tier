Guest Question.

Designing a CMOS inverter Design a CMOS inverter, as shown in Fig. 15.26(a), to operate at a transition voltage \( V_{ MD }=2.5 V \). The threshold voltages are \( V_{ tP }=-1 V , V_{ tN }=1 V , \text { and } K_{ p }=20 \mu A / V ^{2} \) . The supply voltage is \( V_{ DD }=5 V \text {. Assume } V_{ IH }=V_{ OH }=5 V \text {, load capacitance } C_{ L }=0.5 pF \text {, and frequency } f_{ clk }=5 MHz \).

(a) Find the design parameter \( K_{ R }=K_{ n } / K_{ p } \), neglecting the body effect.

(b) Calculate \( NM _{ L } \text { and } NM _{ H } \).
(c) Calculate the propagation delay \( t_{ pd } \).
(d) Calculate the delay-power product (DP).
(e) Use PSpice/SPICE to verify your results.

Admin
Solution

(a) Solving Eq. (15.68) for \( K_{ R } \), we have

\( V_{ I (\operatorname{tran} 1)}=V_{ MD }=\frac{V_{ DD }+V_{ tP }+V_{ tN } \sqrt{K_{ n } / K_{ p }}}{1+\sqrt{K_{ n } / K_{ p }}}=\frac{V_{ DD }+V_{ tP }+V_{ tN } \sqrt{K_{ R }}}{1+\sqrt{K_{ R }}} \) (15.68)

 

\( K_{ R }=\frac{(W / L)_{ N }}{(W / L)_{ P }}=\left[\frac{V_{ DD }+\left|V_{ tP }\right|-V_{ M }}{V_{ M }-V_{ tN }}\right]^{2} \) (15.71)

 

which, for \( V_{ MD }=2.5 V \text {, gives } K_{ R }=1 \text {. Thus, } K_{ n }=K_{ p }=20 \mu A / V ^{2} \).

(b) \( V_{ OH }=V_{ DD }=5 V \). Then

\(\begin{aligned}&V_{ O (\operatorname{tran} 1)}=V_{ MD }-V_{ tP }=2.5+1=3.5 V \\&V_{ O (\operatorname{tran} 2)}=V_{ MD }-V_{ tN }=2.5-1=1.5 V\end{aligned}\)

 

Substituting numerical values of \( V_{ DD }, V_{ tN }, V_{ t P }, K_{ n } \text { and } K_{ p } \) in Eq. (15.67), we get

\( K_{ n }\left(v_{ I }-V_{ tN }\right)^{2}=K_{ p }\left[2\left(V_{ DD }-v_{ I }+V_{ tP }\right)\left(V_{ DD }-v_{ O }\right)-\left(V_{ DD }-v_{ O }\right)^{2}\right] \) (15.67)

 

\( v_{ I }^{2}=14-8 v_{ I }+2 v_{ O }+2 v_{ I } v_{ O }-v_{ O }^{2} \)

which, at \( d v_{ O } / d v_{ I }=-1 \text {, gives } v_{ O }=v_{ I }+2.5 \). Solving these two equations, we get \( V_{ IL }=v_{ I }=2.13 V \) at \( v_{ O }=4.63 V \).

Substituting numerical values of \( V_{ DD }, V_{ tN }, V_{ tP }, K_{ n } \text { and } K_{ p } \) in Eq. (15.70), we get

\( K_{ n }\left[2\left(v_{ I }-V_{ t N }\right) v_{ O }-v_{ O }^{2}\right]=K_{ p }\left(V_{ DD }-v_{ I }+V_{ t P }\right)^{2} \) (15.70)

 

\( 16-8 v_{ I }+v_{ I }^{2}=2 v_{ I } v_{ O }-2 v_{ O }-v_{ O }^{2} \)

which, at \( d v_{ O } / d v_{ I }=-1 \text {, gives } v_{ I }=v_{ O }+2.5 \). Solving these two equations, we get \( V_{ IH }=v_{ I }=2.88 V \) at \( v_{ O }=0.38 V \).

At \( v_{ I }=V_{ DD }+V_{ tP }=5-1=4, M _{ P } \text { turns off and } V_{ OL }=v_{ O }=0 . \) Thus,

 

\(\begin{aligned}& NM _{ L }=V_{ IL }-V_{ OL }=2.13-0=2.13 V \\& NM _{ H }=V_{ OH }-V_{ IH }=5-2.88=2.12 V\end{aligned}\)

(c) The inverter with a load capacitor \( C_{L} \) has an arrangement similar to that shown in Fig. 15.19(a). As \( v_{ I } \) goes low to \( V_{ OL }, M _{ N } \) turns off immediately, and capacitor \( C_{L} \) is charged up by the drain current \( i_{ DP } \). Since \( M _{ P } \) is in saturation and \( v_{ I }=V_{ OL }=0 \), the output voltage is related to the charging current by

\( C_{ L } \frac{d v_{ O }}{d t}=i_{ DP }=K_{ p }\left(V_{ DD }-v_{ I }+V_{ t P }\right)^{2} \)

which, after integration between \( V_{ DD } \text { and } V_{ DD } / 2, \text { gives } t_{ pLH } \) as

\(\begin{aligned}t_{ pLH } &=\frac{C_{ L } V_{ DD }}{2 K_{ p }\left(V_{ DD }-\left|V_{ tP }\right|\right)^{2}} \\&=\frac{0.5 pF \times 5 V }{2 \times 20 \mu A / V ^{2} \times(5 V -1 V )^{2}}=3.91 ns\end{aligned}\) (15.72)

Because of the topology of the CMOS inverter, \( t_{ pHL } \) will have the same value as \( t_{ pLH } \). Thus, the propagation time \( t_{ pd } \approx t_{ pLH }=3.91 ns \). In practice, a symmetric CMOS does not occupy the minimum chip area, and thus not all CMOS designs are symmetric.

(d) A CMOS inverter draws a negligible current (on the order of nanoamperes) from the power supply in both high and low states. Hence, the static power dissipation is almost zero: \( P_{\text {static }}=0 \). This is a distinct advantage for portable CMOS equipment because standby operation of the equipment will not discharge the battery. We have

\( P_{\text {dynamic }}=f_{\text {clk }} C_{ L } V_{ DD }^{2}=5 MHz \times 0.5 pF \times 5^{2}=62.5 \mu W \)

 

so \( P_{ D }=P_{\text {static }}+P_{\text {dynamic }}=0+62.5 \mu W =62.5 \mu W \)

Therefore, the delay-power product is

\( DP =P_{ D } \times t_{ pd }=62.5 \mu W \times 3.91 ns =0.244 pJ \)

 

(e) The PSpice schematic is shown in Fig.15.27. The PSpice plot of the VTC is shown in Fig. 15.28(a), which gives \(V_{ IL }=2.1245 V\) (expected value is 2.13 V), \(V_{ IH }=2.8755 V\) (expected value is 2.88 V), \( V_{ OL }=0\) (expected value is 0) at \( V_{ OH }=5 V , \text { and } V_{ I (\operatorname{tranl})}=2.5 \) (expected value is 2.5 V). The VTC values are very close to the hand calculations, as expected. The transient response is shown in Fig. 15.28(b), which gives \( t_{ pHL }=4.18 ns \text { and } t_{ pLH }=4.255 ns \) (for L = 50 µm and W = 100 µm). The transient performance, however, will depend on the values of length (L) and width (W). A typical value of the transconductance parameter for the NMOS process is \( \mu_{ n } \epsilon / t_{ ox }=40 \mu A / V ^{2} \) . [Note: We can find \( d v_{ O } / d v_{ I }=-1 \) points by plotting dv(2) in Probe.]
