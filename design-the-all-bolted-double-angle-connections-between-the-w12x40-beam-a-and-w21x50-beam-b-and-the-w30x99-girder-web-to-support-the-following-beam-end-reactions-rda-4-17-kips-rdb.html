Supporting Girder

Check the supporting girder web

The required bearing strength per bolt is greatest for the bolts that are loaded by both connections. Thus, for the design of these 4 critical bolts, the required strength is determined as

LRFD ASD
From Beam A, each bolt must support onefourth of 25 kips or 6.25 kips/bolt.

 

From Beam B, each bolt must support onetenth of 110 kips or 11.0 kips/bolt.

 

Thus,

 

\(\begin{aligned} R_{u} &=6.25 \text { kips } / \text { bolt }+11.0 \text { kips } / \text { bolt } \\&=17.3 \text { kips } / \text { bolt }\end{aligned}\)

 

The design bearing strength per bolt is

 

\(\phi r_{u}=(87.8 kips / in )(0.520 in .)\)

 

= 45.7 kips/bolt > 17.3 kips/bolt o.k

 From Beam A, each bolt must support onefourth of 16.7 kips or 4.18 kips/bolt.

 

From Beam B, each bolt must support onetenth of 73.3 kips or 7.33 kips/bolt.

 

Thus,

 

\(\begin{aligned}R_{a} &=4.18 \text { kips } / \text { bolt }+7.33 \text { kips } / \text { bolt } \\&=11.5 \text { kips } / \text { bolt }\end{aligned}\)

 

The allowable bearing strength per bolt is

 

\(r_{a} / \Omega=(58.5 kips / in .)(0.520 in .)\)

 

= 30.4 kips/bolt > 11.5 kips/bolt o.k.
