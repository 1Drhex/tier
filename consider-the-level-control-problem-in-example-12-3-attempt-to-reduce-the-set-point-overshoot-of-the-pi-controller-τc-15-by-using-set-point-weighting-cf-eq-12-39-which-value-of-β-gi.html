Guest Question.

Consider the level control problem in . Attempt to reduce the set-point overshoot of the PI controller \(\left(\tau_{c}=15\right)\) by using set-point weighting (cf. Eq. 12-39). Which value of \(\beta\) gives the best results? Does the value of \(\beta\) affect the disturbance response?

\(IE =\frac{\tau_{I}}{K_{c}} \quad(\text { when } e(t) \text { does not change signs })\) (12-39)

Admin
Solution

From Eq. 12-40 (with \(\gamma=0\) ):

\(\begin{aligned}u(t)=& K_{c}\left[y_{s p}(t)-y_{m}(t)\right]+K_{I} \int_{0}^{t}\left(y_{s p}\left(t^{*}\right)-y_{m}\left(t^{*}\right)\right) d t^{*} \\&+K_{D}\left(-\frac{d y_{m}}{d t}\right)\end{aligned}\) (12-40)

\(p(t)=\bar{p}+K_{c}\left[\beta y_{s p}(t)-y_{m}(t)\right]+K_{c}\left[\frac{1}{\tau_{I}} \int e\left(t^{*}\right) d t^{*}-\tau_{D} \frac{d y_{m}}{d t}\right]\)

Closed-loop responses are compared for four values of \(\beta: 1,0.7,0.5 \& 0.3 .\)

As shown in Figure \(S 12.8\), as \(\beta\) increases the set-point response becomes faster but exhibits more overshoot. The value of \(\beta=0.5\) seems to be a good choice. The disturbance response is independent of the value of \(\beta\).
