Guest Question.

Inside the cylindrical shell, 3 <\(\rho\) < 4m , the electric flux density is given as \(5\rho { \left( \rho -3 \right) }^{ 3 }{ a }_{ \rho } C/{ m }^{ 2 }\). (a) What is the volume charge density at \(\rho\) = 4m? (b) what is the electric flux density at \(\rho\) = 4m? (c) How much electric flux leaves the closed surface: \(3 <\rho < 4, 0 <\phi < 2\pi , -2.5 < z < 2.5?\) (d) How much charge is contained within the volume \(3 <\rho < 4, 0 <\phi < 2\pi , -2.5 < z < 2.5?\)

Admin
Solution

a)In this case we have

\({ \rho }_{ \upsilon }=\triangledown .D=\frac { 1 }{ \rho } \frac { d }{ d\rho } \left( \rho { D }_{ \rho } \right) = \frac { 1 }{ \rho } \frac { d }{ d\rho } \left[ 5\rho { \left( \rho -3 \right) }^{ 3 } \right] =\frac { 5{ \left( \rho -3 \right) }^{ 2 } }{ \rho } \left( 4\rho -3 \right) C/{ m }^{ 3 }\)
Evaluating this at \(\rho\) = 4m, we find \({ \rho }_{ \upsilon }(4) = 16.25 C/{ m }^{ 3 }\)
b) We evaluate the given D at this point to find D(4) = \(5 { a }_{ \rho } C/{ m }^{ 2 }\)
c) We note that D has only a radial component, and so ﬂux would leave only through the cylinder sides. Also, D does not vary with φ or z, so the ﬂux is found by a simple product of the side area and the ﬂux density . We further not that D = 0 at \(\rho\) = 3. so only the outer side ( at \(\rho\) = 4) will contribute . We use the result of part b, and write the ﬂux as
\(\Phi \)=\(\left[ 2.5-\left( -2.5 \right) \right] 2\pi \left( 4 \right) \left( 5 \right)\) = \(200\pi C\)

d) By Gauss’ law, this will be the same as the net outward ﬂux through that volume, or again, \(200\pi C\).
