Guest Question.

For the two-link planar arm of Fig. 7.4, find the dynamic model with the Lagrange formulation when the absolute angles with respect to the base frame are chosen as generalized coordinates. Discuss the result in view of a comparison with the model derived in (7.82).

\(\left(I_{l_{1} } + m_{l_{1} }l^{2}_{1} +k^{2}_{r1} I_{m_{1} } + I_{l_{2} } + m_{l_{2} } \left(a^{2}_{1} + l^{2}_{2} + 2 a_{1}l_{2}c_{2}\right) + I_{m_{2} } + m_{m_{2} } a^{2}_{1} \right) \ddot{\vartheta _{1} } + \left(I_{l_{2} } + m_{l_{2} }\left(l^{2}_{2} + a_{1}l_{2}c_{2}\right) + k_{r2} I_{m_{2} } \right)\ddot{\vartheta _{2} } - 2 m_{l_{2} }a_{1}l_{2}s_{2}\dot{\vartheta _{1} } \dot{\vartheta _{2} } - m_{l_{2} } a_{1}l_{2}s_{2}\dot{\vartheta }^{2}_{2} + \left(m_{l_{1} }l_{1} + m_{m_{2} }a_{1} + m_{l_{2} } a_{1}\right) gc_{1} + m_{l_{2} }l_{2}gc_{12} \) = \(\tau_{1}\)

\(\left(I_{l_{2} } + m_{l_{2} }\left(l^{2}_{2} + a_{1}l_{2}c_{2}\right)+ k_{r2} I_{m_{2} } \right)\ddot{\vartheta _{1} } + \left(I_{l_{2} } + m_{l_{2} }l^{2}_{2} + k^{2}_{r2} I_{m_{2} } \right)\ddot{\vartheta _{2} } + m_{l_{2} } a_{1}l_{2}s_{2}\dot{\vartheta }^{2}_{1} + m_{l_{2} }l_{2} g c_{12}\) = \(\tau_{2}\)

(7.82)

Admin
Solution

Let \(x_{1}\) and \(x_{2}\) denote the absolute angles of the two joints. The relationship between relative and absolute angles is

\(x_{1}\) = \(\vartheta _{1} \) \(x_{2}\) = \(\vartheta _{1} + \vartheta _{2}\)

This can be formally treated as a coordinate transformation from the joint space to the operational space of absolute angles. The analytical Jacobian characterizing such transformation at velocity level is then

\(J_{A}\) = \(\left[\begin{matrix} 1 & 0\\ 1 & 1 \end{matrix} \right] \)

which is always nonsingular, implying that the absolute angles constitute a set of generalized coordinates for the system.
Therefore, the terms of the operational space dynamic model can be computed according to (7.134)–(7.136). From (7.134), the inertia matrix is

\(B_{A} \) = \(J^{-T}_{A} B J^{-1}_{A} \) (7.134)

\(g_{A}\) = \( J^{-T}_{A} g (7.136)\)

\(B_{A}\) = \( \left[\begin{matrix} b_{A_{11}} & b_{A_{12}} \\ b_{A_{12}} & b_{A_{22}} \end{matrix} \right] \)

where, using the expressions of \(b_{ij}\) in Sect. 7.3.2, the elements in the new coordinates are

\( b_{A_{11}} \) = \( b_{11} + b_{22} – 2 b_{12} \) = \( I_{l_{1}} + m_{l_{1}} l^{2}_{1} + k^{2}_{r1} I_{m_{1}} + m_{l_{2}} a^{2}_{1} + \left(1 – k_{r_{2}} \right)^{2} I_{m_{2}} + m_{m_{2}} a^{2}_{1}\)

\( b_{A_{12}} \) = \( b_{12} – b_{22} \) = \( m_{l_{2}} a_{1} l_{2} \cos \left(x_{2} – x_{1}\right) + k_{r_{2}} \left(1 – k_{r_{2}}\right) I_{m_{2}} \)

\( b_{A_{22}} \) = \( b_{22} \) = \( I_{l_{2}} + m_{m_{2}} l^{2}_{2} + k^{2}_{r2} I_{m_{2}} \)

Notice that the diagonal elements are configuration-invariant; also, if \( k_{r_{2}} \) = 1 (direct drive), then the effect of motor 2 inertia on both \( b_{A_{12}} \) and \( b_{A_{22}}\) vanishes. Further, from (7.135), the velocity-dependent term is

\(C_{A} \dot{x}_{e}\) = \(J^{-T}_{A} C \dot{q} – B_{A} \dot{J}_{A} \dot{q}\) (7.135)

\( C_{A} \dot{x} \) = \(\left[\begin{matrix} c_{11} – c_{12} & c_{21} – c_{22} \\ c_{21} & c_{22} \end{matrix} \right]\)

which, using the expressions of \( c_{ij}\) in Sect. 4.3.2, in the new coordinates becomes

\( C_{A} \dot{x} \) = \(\left[\begin{matrix} h\dot{x }^{2}_{2} \\ -h\dot{x}^{2}_{1} \end{matrix} \right] \)

where h = \(– m_{l_{2}} a_{1} l_{2} \sin \left(x_{2} – x_{1}\right) \). Interestingly enough, there is no Coriolis effect; this can be explained by observing that rotation of Link 2 is represented with reference to Frame 0 and thus is independent of rotation of Link 1.

Finally, from (7.136), the gravity term is

\( g_{A} \) = \(\left[\begin{matrix} g_{1} – g_{2} \\ g_{2} \end{matrix} \right] \)

which, using the expressions of \( g_{i} \) in Sect. 4.3.2, in the new coordinates becomes

\( g_{A} \) = \(\left[\begin{matrix} \left( m_{l_{1}} l_{1} + m_{m_{2}} a_{1} + m_{l_{2}} a_{1}\right) g \cos x_{1} \\ m_{l_{2}} l_{2} g \cos x_{2}\end{matrix} \right] \)
