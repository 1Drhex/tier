Guest Question.

Representing a network by Thevenin’s equivalent circuit Represent the network shown in Fig. B.15 by Thevenin’s equivalent circuit. The circuit values are \( R_{ i }=1.5 k \Omega, R_{ C }=25 k \Omega, \beta_{ F }=50, h_{ r }=3 \times 10^{-4} \) , and \( V_{ s }=5 mV \).

(a) Calculate the parameters of Thevenin’s equivalent circuit.

(b) Use PSpice/SPICE to check your results.

Admin
Solution

(a) \( R_{ i }=1.5 k \Omega, R_{ C }=25 k \Omega, \beta_{ F }=50, h_{ r }=3 \times 10^{-4} \text {, and } V_{ s }=5 mV \text {. The output voltage } V_{ o } \) between terminals a and b is

\( V_{ o }=-I_{2} R_{ C }=-\beta_{ F } I_{ s } R_{ C } \) (B.18)

The input current \( I_{ S } \) can be found from loop I as

\( I_{ s }=\frac{V_{ s }-h_{ r } V_{ o }}{R_{ i }} \) (B.19)

Substituting \( I_{ S } \) from Eq. (B.19) into Eq. (B.18) gives the output voltage \( V_{0} \) as

\(\begin{aligned}V_{ Th } &=V_{ o }=\frac{-\beta_{ F } R_{ C }}{R_{ i }-\beta_{ F } h_{ r } R_{ C }} V_{ s } \\&=\frac{-50 \times 25 k \times 5 m }{1.5 k -50 \times 3 \times 10^{-4} \times 25 k }=-5.5556 V\end{aligned}\) (B.20)

Thevenin’s resistance \( R_{ Th } \) can be determined from the circuit in Fig. B.16, which is obtained by short-circuiting the independent source \( V_{ s} \text {. Considering } V_{ x } \text { and } I_{ x } \) as the test voltage and current, respectively, we get

\( I_{ s }=-\frac{h_{ r } V_{ x }}{R_{ i }} \) (B.21)

 

\( I_{ x }=\beta_{ F } I_{ s }+\frac{V_{ x }}{R_{ C }} \) (B.22)

 

Substituting \( I_{ S } \) from Eq. (B.21) into Eq. (B.22) gives

\( I_{ x }=-\frac{\beta_{ F } h_{ r } V_{ x }}{R_{ i }}+\frac{V_{ x }}{R_{ C }}=\frac{R_{ i }-\beta_{ F } h_{ r } R_{ C }}{R_{ i } R_{ C }} V_{ x } \)

which gives Thevenin’s resistance \( R_{ Th } \) as

\(\begin{aligned}R_{ Th } &=\frac{V_{ x }}{I_{ x }}=\frac{R_{ i } R_{ C }}{R_{ i }-\beta_{ F } h_{ r } R_{ C }} \\&=\frac{1.5 k \times 25 k }{1.5 k -50 \times 3 \times 10^{-4} \times 25 k }=33.33 k \Omega\end{aligned}\) (B.23)

 

(b) The circuit for PSpice simulation is shown in Fig. B.17, which is run by OrCAD PSpice A/D. The results of the PSpice simulation are shown below:

\(\begin{array}{llllllll}\text { NODE } & \text { VOLTAGE } & \text { NODE } & \text { VOLTAGE } & \text { NODE } & \text { VOLTAGE } & \text { NODE } & \text { VOLTAGE } \\(1) & .0050 & (2) & -.0017 & (3) & 0.0000 & (4) & -5.5556\end{array}\)

 

\( V_{ Th }=V_{ o }= V (4)=-5.5556 V \)

\(\begin{array}{ll}V (4) / VS =-1.111 E +03 & \text { Gain } A=V_{ o } / V_{ s }=-1111 \\\text { INPUT RESISTANCE AT VS=1.125E+03 } & R_{ in }=V_{ s } / I_{ s }=1.125 k \Omega \\\text { OUTPUT RESISTANCE AT V(4)=3.333E+04 } & R_{ Th }=33.33 k \Omega\end{array}\)
