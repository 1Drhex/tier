Guest Question.

For \(M_{1} → ∞\), find the attached oblique shock solution for the deflection angle, shock wave angle, downstream Mach number. Consider \(M_{1} = 10\) as approaching ∞. Find ratios of pressure and temperature across the shock wave for γ = 1.4.

Admin
Solution

The maximum shock wave angle \(\) can be directly determined without knowing δ, according to Eqn. (4.29) \(\sin ^{2}\theta _{\max } =\frac{1}{\gamma M^{2}_{1} } \left\{\frac{\gamma +1}{4} M^{2}_{1} -1+\left[\left(\gamma +1\right) \left(1+\frac{\gamma -1}{2} M^{2}_{1} +\frac{\gamma +1}{16} M^{4}_{1}\right) \right] ^{1/2} \right\} \):

\(\sin ^{2}\theta _{\max } =\underset{M_{1}\rightarrow \infty }{\lim } \frac{1}{\gamma M^{2}_{1} } \left\{\frac{\gamma +1}{4} M^{2}_{1} -1+\left[\left(\gamma +1\right) \left(1+\frac{\gamma -1}{2} M^{2}_{1} +\frac{\gamma +1}{16} M^{4}_{1}\right) \right] ^{1/2} \right\} =\frac{\gamma +1}{2\gamma }\)

 

\(\theta _{\max } =\sin ^{-1} \sqrt{\frac{\gamma +1}{2\gamma } } \)

Then

\(\tan \delta _{\max } =\underset{M_{1}\rightarrow \infty }{\lim } 2\cot \theta \frac{M^{2}_{1} \sin ^{2} \theta -1}{M^{2}_{1} \left(\gamma +\cos 2\theta \right) +2} =2\frac{\cos \theta _{\max } }{\sin \theta _{\max } } \frac{\sin ^{2} \theta _{\max } }{\gamma +\cos 2\theta _{\max } } =\frac{\sin 2\theta _{\max } }{\gamma +\cos 2\theta _{\max }} \)

for γ = 1.4,

\(\theta _{\max }=67.792°\)

 

\(\delta _{\max }=45.579°\)

These values are consistent with those than can be read from Fig. 4.12.

\(M^{2}_{2} \sin ^{2} \left(\theta -\delta \right) =\frac{M^{2}_{1}\sin ^{2}\theta +\frac{2}{\gamma -1} }{\frac{2\gamma }{\gamma -1} M^{2}_{1}\sin ^{2}\theta -1} =\frac{\gamma -1}{2\gamma }\) in the limit of \(M_{1} \rightarrow \infty\).

\(M_{2}=\sqrt{\frac{\gamma -1}{2\gamma \sin ^{2}\left(\theta -\delta \right) } } =\sqrt{\frac{0.4}{2\times 1.4\times \sin ^{2}\left(67.792° -45.579° \right) } } =0.9998\approx 1\)

For \(M_{1} = 10\),

\(M_{1n}= M_{1} \sin \theta \approx M_{1}\sqrt{\frac{\gamma +1}{2\gamma } } =9.26\)

\(\frac{p_{2} }{p_{1}} =\frac{2\gamma M^{2}_{1n}}{\gamma +1} -\frac{\gamma -1}{\gamma +1} \approx M^{2}_{1} =100\)( ≈ 100 from Appendix D)

\(\frac{T_{2} }{T_{1}}= \frac{2\gamma \left(\gamma -1\right) M^{2}_{1n}}{\left(\gamma +1 \right) ^{2}} =\frac{\left(\gamma -1\right) }{\left(\gamma +1\right) } M^{2}_{1}\approx 16.7\)( ≈ 17.5 from Appendix D)

Comments –

This near-unity value of the downstream Mach number is to be expected, as Eqns. (4.29) \(\sin ^{2}\theta _{\max } =\frac{1}{\gamma M^{2}_{1} } \left\{\frac{\gamma +1}{4} M^{2}_{1} -1+\left[\left(\gamma +1\right) \left(1+\frac{\gamma -1}{2} M^{2}_{1} +\frac{\gamma +1}{16} M^{4}_{1}\right) \right] ^{1/2} \right\}\) and (4.30) \(\sin ^{2}\theta _{s } =\frac{1}{\gamma M^{2}_{1} } \left\{\frac{\gamma +1}{4} M^{2}_{1} -\frac{3-\gamma }{4} +\left[\left(\gamma +1\right) \left(\frac{\gamma +9}{16}- \frac{3-\gamma }{8} M^{2}_{1} +\frac{\gamma +1}{16} M^{4}_{1}\right) \right] ^{1/2} \right\}\) for \(\theta _{\max }\) and \(\theta _{s}\) (shock wave angle for \(M_{2} = 1\)) are the same as \(M_{1} \rightarrow \infty\). This result can be seen in Fig. 4.12. From Appendix D, the pressure and temperature ratios are approximately 100 and 17.5, respectively.
