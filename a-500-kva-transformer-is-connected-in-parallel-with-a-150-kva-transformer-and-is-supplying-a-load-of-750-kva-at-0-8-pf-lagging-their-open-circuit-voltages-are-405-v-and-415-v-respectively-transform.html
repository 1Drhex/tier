Guest Question.

A 500 kVA transformer is connected in parallel with a 150 kVA transformer and is supplying a load of 750 kVA at 0.8 pf lagging. Their open circuit voltages are 405 V and 415 V, respectively. Transformer A has 1% resistance and 5% reactance and transformer B has 1.5% resistance and 4% reactance. Find (a) cross-current in the secondaries on no-load and (b) the load shared by each transformer.

Admin
Solution

It is more convenient to work with ohmic impedances and for that purpose, we will convert percentage values into Numerical values by assuming 400 volt as the terminal voltage (i.e., \(V_{1} = V_{2} = 400 V\), this value is arbitrary but this assumption will not introduce appreciable error).

Now, \(\frac{I_{1\left(fl\right)}R_{1}}{V_{1}} \times 100 = 1 or R_{1} = \frac{V_{1}}{I_{\left(fl\right)} \times 100} = \frac{1}{100} \times \frac{V_{1}}{I_{1\left(fl\right)}} \)

where, \(I_{1\left(fl\right)} = \frac{kVA_{1}\times 1000}{V_{1}} = \frac{500 \times 1000}{400} = 1250 A \)

\(\therefore R_{1} = \frac{1}{100} \times \frac{400}{1250} = 0.0032 \Omega \)

 

Similarly, \(X_{1} = \frac{5}{100} \times \frac{400}{1250} = 0.01600 \Omega \)

Now, for transformers – B, \(I_{2\left(fl\right)} = \frac{kVA_{2}\times 1000}{V_{2}} = \frac{150 \times 1000}{400} = 375 A \)

\(\therefore R_{2} = \frac{1.5}{100} \times \frac{400}{375} = 0.016 \Omega ; X_{2} = \frac{4}{100} \times \frac{400}{375} = 0.0427 \Omega \)

 

\(\therefore \overline{Z}_{1} = 0.0032 + j 0.016 = 0.0163 \angle 78.5^{\circ } \)

 

\(\overline{Z}_{2} = 0.016 + j 0.0427 = 0.0456 \angle 69.46^{\circ } \)

 

\(\overline{Z}_{1} + \overline{Z}_{2} =0.0032 + j 0.016 + 0.016 + j0.0427 = 0.0192 + j0.0587 = 0.0618 \angle 71.88^{\circ } \)

 

Next step is to calculate load impedance. Let \(Z_{L}\) be the load impedance and \(V_{2}\) be the terminal voltage which has been assumed as 400 V.

\(\therefore \left\lgroup\frac{V_{2}}{\overline{Z}_{L}} \right\rgroup \times 10^{-3} = 750 \angle -36.87^{\circ } \)

 

\(\therefore \overline{Z}_{L} = \frac{V^{2} \times 10^{-3}}{750 \angle -36.87^{\circ }} = \frac{400 \times 400 \times 10^{-3}}{750 \angle -36.83^{\circ }} \\[0.5cm] \qquad \; \,= 0.214\angle 36.87^{\circ } = \left(0.171 + j 0.128\right) \Omega \)

 

(a) \(I_{C} = \frac{E_{1} – E_{2}}{Z_{1} + Z_{2}} = \frac{\left(405 – 415 \right) }{0.0618\angle 71.88^{\circ}} = -161.8\angle -71.88^{\circ} \)

(b) \(\overline{I}_{1} = \frac{\overline{E}_{1} \overline{Z}_{2} + \left(\overline{E}_{1} – \overline{E}_{2}\right) \overline{Z}_{L}}{\overline{Z}_{1}\overline{Z}_{2} + \overline{Z}_{L}\left(\overline{Z}_{1} + \overline{Z}_{2}\right) } \\[0.5cm] \qquad \enspace = \frac{405 \times 0.0456\angle 69.46^{\circ} + \left(405 – 415 \right) \times 0.214\angle 36.87^{\circ}}{0.0163\angle 78.5^{\circ} \times 0.0456\angle 69.46^{\circ} + 0.214\angle 36.87^{\circ} \times 0.0618\angle 71.88^{\circ}} \\[0.5cm] \qquad \enspace = \frac{18.47\angle 69.46^{\circ} – 2.14\angle 36.87^{\circ}}{0.00074\angle 147.96^{\circ} + 0.01323\angle 108.75^{\circ}} \\[0.5cm] \qquad \enspace = \frac{6.48 + j17.29 – 1.712 – j1.284}{- 0.00063 + j 0.0004 – 0.00425 + j0.0125} \\[0.5cm] \qquad \enspace = \frac{4.768 + j16.01}{- 0.00488 + j0.0129} = \frac{16.705\angle 73.41^{\circ}}{0.0138 \angle -69.28^{\circ}} = 1210 \angle -37.3^{\circ} A\)

Load shared by transformer – A

\(S_{1} = 400 \times 1210 \times 10^{–3} \angle –37.3^{\circ } = \mathbf{484\angle –37.3^{\circ } kVA} \)

 

\(p.f. = \cos \phi_{1}= \cos 37.3^{\circ} = \mathbf{0.795 lag.} \)

 

\(I_{2} = \frac{\overline{E}_{2} \overline{Z}_{1} – \left(\overline{E}_{1} – \overline{E}_{2}\right) \overline{Z}_{L}}{\overline{Z}_{1}\overline{Z}_{2} + \overline{Z}_{L}\left(\overline{Z}_{1} \overline{Z}_{2}\right) } \\[0.5cm] \quad \, = \frac{415 \times 0.0163\angle 78.5^{\circ} – \left(405 – 415 \right) \times 0.214\angle 36.87^{\circ}}{0.0138\angle -69.28^{\circ}} \\[0.5cm] \quad \, = \frac{1.348 + j6.629 + 1.712 + j1.284}{0.0138\angle -69.28^{\circ}} = \frac{3.0607 + j7.913}{0.0138\angle -69.28^{\circ}} \\[0.5cm] \quad \, = \frac{8.4843 \angle 68.85^{\circ}}{0.0138 \angle -69.28^{\circ}} = 615 \angle -41.87^{\circ} A\)

 

Load shared by transformers – S,

\(S_{2} = 400 \times 615 \times 10^{–3} \angle –42.87^{\circ } = \mathbf{246\angle –42.87^{\circ } kVA} \)

 

\(p.f. = \cos \phi_{2}= \cos 42.87^{\circ} = \mathbf{0.744 lag.} \)
