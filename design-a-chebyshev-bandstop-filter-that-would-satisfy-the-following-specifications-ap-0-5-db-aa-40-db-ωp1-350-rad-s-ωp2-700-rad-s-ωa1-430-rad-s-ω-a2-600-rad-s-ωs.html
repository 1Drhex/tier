Guest Question.

Design a Chebyshev bandstop filter that would satisfy the following specifications:

\(A_{p}=0.5 dB \quad A_{a}=40 dB \quad \tilde{\Omega}_{p 1}=350 rad / s \quad \tilde{\Omega}_{p 2}=700 rad / s\) \(\tilde{\Omega}_{a 1}=430 rad / s \quad \tilde{\Omega}_{a 2}=600 rad / s \quad \omega_{s}=3000 rad / s\)

Admin
Solution

The sampling period is given by

 

\(T=\frac{2 \pi}{\omega_{s}}=\frac{2 \pi}{3000}\)

 

Hence

 

\(\begin{array}{ll}\frac{\tilde{\Omega}_{p 1} T}{2}=\frac{35 \pi}{300} & \frac{\tilde{\Omega}_{p 2} T}{2}=\frac{70 \pi}{300} \\\frac{\tilde{\Omega}_{a 1} T}{2}=\frac{43 \pi}{300} & \frac{\tilde{\Omega}_{a 2} T}{2}=\frac{60 \pi}{300}\end{array}\)

 

From Table 12.3, we have

 

\(K_{A}=\tan \frac{70 \pi}{300}-\tan \frac{35 \pi}{300}=5.165400 \times 10^{-1}\)

 

\(K_{B}=\tan \frac{35 \pi}{300} \tan \frac{70 \pi}{300}=3.456327 \times 10^{-1}\)

 

\(K_{C}=\tan \frac{43 \pi}{300} \tan \frac{60 \pi}{300}=3.512244 \times 10^{-1}\)

 

Since \(K_{C}>K_{B}\), according to Table 12.5, we need to compute \(K=1 / K_{2}\), that is,

 

\(K=\frac{1}{K_{2}}=\frac{\tan ^{2}\left(\tilde{\Omega}_{a 2} T / 2\right)-K_{B}}{K_{A} \tan \left(\tilde{\Omega}_{a 2} T / 2\right)}=4.855769 \times 10^{-1}\)

 

From Table 12.5,

 

\(D=\frac{10^{4.0}-1}{10^{0.05}-1}=8.194662 \times 10^{4}\)

 

\(n=\frac{\cosh ^{-1} \sqrt{D}}{\cosh ^{-1} 1 / K}=4.70 \rightarrow 5\)

 

\(\omega_{p}=1\)

 

Now from Table 12.3, the parameters of the LP-to-BS transformation can be obtained as

 

\(\omega_{0}=\frac{2 \sqrt{K_{B}}}{T}=5.614083 \times 10^{2}\)

 

\(B=\frac{2 K_{A} \omega_{p}}{T}=4.932594 \times 10^{2}\)

By obtaining the appropriate Chebyshev approximation (see Sec. 10.4.3) and then applying the LP-to-BS transformation followed by the bilinear transformation the transfer function is of the required digital filter can be obtained as

 

\(H_{D}(z)=H_{0} \prod_{j=1}^{5} \frac{a_{0 j}+a_{1 j} z+z^{2}}{b_{0 j}+b_{1 j} z+z^{2}}\)

 

where the coefficients \(a_{i j} \text { and } b_{i j}\) are given in Table 12.11. The loss characteristic of the filter is plotted in Fig. 12.6. The actual minimum stopband loss is 43.50 dB.
