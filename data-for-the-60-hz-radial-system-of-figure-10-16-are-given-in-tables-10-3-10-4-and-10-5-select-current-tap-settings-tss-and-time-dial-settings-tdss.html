Coordinating time-delay overcurrent relays in a radial system

Data for the 60-Hz radial system of Figure 10.16 are given in Tables 10.3, 10.4, and 10.5. Select current tap settings (TSs) and time-dial settings (TDSs)

Bus S MVA Lagging p.f.
1 11.0 0.95
2 4.0 0.95
3 6.0 0.95
Admin
Solution

First, select TSs such that the relays do not operate for maximum load currents. Starting at B3, the primary and secondary CT currents for maximum load L3 are

\(I_{L3}=\frac{S_{L3} }{V_{3}\sqrt{3} } =\frac{6\times 10^{6} }{\left(34.5\times 10^{3} \right)\sqrt{3} }=100.4\) A

\(I^{\prime } _{L3}=\frac{100.4}{\left(200/5\right) }=2.51 \) A

From Figure 10.12, relay a 3-A TS is selected for the B3, which is the lowest TS above 2.51 A.

Note that \(\mid S_{L2} + S_{L3} \mid =\mid S_{L2} \mid +\mid S_{L3} \mid \) because the load power factors are identical. Thus, at B2, the primary and secondary CT currents for maximum load are

\(I_{L2}=\frac{S_{L2}+S_{L3} }{V_{2}\sqrt{3} }=\frac{\left(4+6\right)\times 10^{6} }{\left(34.5\times 10^{3} \right)\sqrt{3} } =167.3 \) A

\(I^{\prime } _{L2}=\frac{167.3}{\left(200/5\right) }=4.18 \) A

From Figure 10.12, select for the B2 relay a 5-A TS, the lowest TS above 4.18 A. At B1,

\(I_{L1}=\frac{S_{L1}+S_{L2}+S_{L3} }{V_{1}\sqrt{3} }=\frac{\left(11+4+6\right)\times 10^{6} }{\left(34.5\times 10^{3} \right)\sqrt{3} } =351.4\) A

\(I^{\prime } _{L1}=\frac{351.4}{\left(400/5\right) }=4.39\) A

Select a 5-A TS for the B1 relay.

Next select the TDSs. First, coordinate for the maximum fault currents in Table 10.4, checking coordination for minimum fault currents later. Starting at B3, the largest fault current through B3 is 2000 A, which occurs for the threephase fault at bus 2 (just to the right of B3). Neglecting CT saturation, the faultto- pickup current ratio at B3 for this fault is

\(\frac{I^{\prime }_{3Fault} }{TS3} =\frac{2000/\left(200/5\right) }{3} =16.7\)

To clear faults as rapidly as possible, select a 1/2 TDS for the B3 relay. Then, from the 1/2 TDS curve in Figure 10.12, the relay operating time is T3 = 0.05 seconds. Adding the breaker operating time (5 cycles 5 0.083 s), primary protection clears this fault in \(T3 + T_{breaker} = 0.05 + 0.083 = 0.133\) seconds.
For this same fault, the fault-to-pickup current ratio at B2 is

\(\frac{I^{\prime }_{2Fault} }{TS2} =\frac{2000/\left(200/5\right) }{5} =10.0\)

Adding the B3 relay operating time (T3 = 0.05 s), breaker operating time (0.083 s), and 0.3 s coordination time interval, a B2 relay operating time is desirable;

\(T2=T3+T_{breaker}+T_{coordination} =0.05 + 0.083 + 0.3\thickapprox 0.43\) s
