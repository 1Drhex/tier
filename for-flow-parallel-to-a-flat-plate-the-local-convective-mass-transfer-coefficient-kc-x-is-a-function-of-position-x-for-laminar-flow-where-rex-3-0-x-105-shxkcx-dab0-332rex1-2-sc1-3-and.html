Guest Question.

For flow parallel to a flat plate, the local convective mass-transfer coefficient \( \left(k_{c, x}\right) \) is a function of position x. For laminar flow, where \( \operatorname{Re}_{x} < 2.0 \times 10^{5} \)

 

\( Sh _{x}=\frac{k_{c} x}{D_{A B}} = 0.332 Re _{x}^{1 / 2} Sc ^{1 / 3}\)

 

and for fully developed turbulent flow, where \( \operatorname{Re}_{x} > 3.0 \times 10^{5} \). the experimentally validated correlation is

 

\( Sh _{x} = \frac{k_{c} x}{D_{A B}} = 0.0292 Re _{x}^{4 / 5} Sc ^{1 / 3}\)

where

\( Re _{x} = \frac{ V _{\infty} x}{\nu} \)

 

Develop an expression to estimate the average mass-transfer coefficient \( \left(k_{c}\right) \) in the flow regime between laminar and fully developed turbulent flow, where \( 2.0 \times 10^{5} < \operatorname{Re}_{x} < 3.0 \times 10^{6} \). and then plot out the average Sh vs. Re for \( Sc = 1.0 \).

Admin
Solution

The transition from laminar to turbulent flow over a flat plate is visualized in Figure. To begin, we note that the location on the flat plate where the flow is no longer fully laminar \( \left(x = L_{t}\right) \) is bounded by Re:

 

\( \operatorname{Re}_{t} = 2.0 \times 10^{5} = \frac{ V _{\infty} L_{t}}{\nu} \)

 

or

\( L_{t} = 2.0 \times 10^{5} \frac{\nu}{ V _{\infty}} \)

 

The average mass-transfer coefficient is obtained by integration of the local mass-transfer coefficient \( \left(k_{c, x}\right) \) over position x. The integral is broken up into a laminar component from x = 0 to \( x = L_{t} \), and a turbulent component, from \( x = L_{t} \) to \( x = L : \)

 

\( k_{c} = \frac{\int_{0}^{L} k_{c, x}(x) d x}{\int_{0}^{L} d x}= \frac{\int_{0}^{L} k_{c, 1 a m}(x) d x + \int_{L_{t}}^{L} k_{c, t u r b}(x) d x}{L} \)

 

with

\( k_{c, \operatorname{lam}}(x) = 0.332 \frac{D_{A B}}{x}\left(\operatorname{Re}_{x}\right)^{1 / 2} Sc ^{1 / 3} \)

 

\( k_{c, t u r b}(x) = 0.0292 \frac{D_{A B}}{x}\left(\operatorname{Re}_{x}\right)^{4 / 5} Sc ^{1 / 3} \)

 

Substitution of these two equations into equation (\( k_{c}=\frac{\int_{0}^{L} k_{c, x}(x) d x}{\int_{0}^{L} d x} = \frac{\int_{0}^{L} k_{c, 1 a m}(x) d x + \int_{L_{t}}^{L} k_{c, t u r b}(x) d x}{L} \) ) gives

 

\( k_{c} = \frac{\int_{0}^{L} \frac{0.332 D_{A B}\left(\operatorname{Re}_{x}\right)^{1 / 2}}{x} S c^{1 / 3} d x + \int_{L_{t}}^{L} \frac{0.0292 D_{A B}\left(\operatorname{Re}_{x}\right)^{4 / 5}}{x} S c^{1 / 3} d x}{L} \)

 

\( k_{c} = \frac{0.332 D_{A B}\left(\frac{ V _{\infty}}{\nu}\right)^{1 / 2} Sc ^{1 / 3} \int_{0}^{L_{4}} x^{-1 / 2} d x + 0.0292 D_{A B}\left(\frac{ V _{\infty}}{\nu}\right)^{4 / 5} Sc ^{1 / 3} \int_{L_{t}}^{L} x^{-1 / 5} d x}{L} \)

 

\( k_{c} = \frac{0.664 D_{A B}\left(\frac{ V _{\infty}}{\nu}\right)^{1 / 2} Sc ^{1 / 3} L_{t}^{1 / 2} + 0.0365 D_{A B}\left(\frac{ V _{\infty}}{\nu}\right)^{4 / 5} Sc ^{1 / 3}\left[(L)^{4 / 5} – \left(L_{t}\right)^{4 / 5}\right]}{L} \)

 

\( k_{c} = \frac{0.664 D_{A B}\left(\operatorname{Re}_{t}\right)^{1 / 2} Sc ^{1 / 3} + 0.0365 D_{A B} Sc ^{1 / 3}\left[\left(\operatorname{Re}_{L}\right)^{4 / 5} – \left(\operatorname{Re}_{t}\right)^{4 / 5}\right]}{L} \)

 

or

\( \frac{k_{c} L}{D_{A B}} = Sh _{L} = 0.664\left( Re _{t}\right)^{1 / 2} Sc ^{1 / 3}+ 0.0365 Sc ^{1 / 3}\left[\left(\operatorname{Re}_{L}\right)^{4 / 5} – \left( Re _{t}\right)^{4 / 5}\right] \)

 

with

\( \operatorname{Re}_{L}=\frac{ V _{\infty} L}{\nu} \)

 

which is valid for \( \operatorname{Re}_{L} > 2.0 \times 10^{5} \), For laminar flow where \( \operatorname{Re}_{L} \leq 2.0 \times 10^{5},\) the average mass-transfer coefficient is

 

\( k_{c} = \frac{\int_{0}^{L} k_{c, l a m}(x) d x}{\int_{0}^{L} d x} = 0.664 \frac{D_{A B}}{L} \operatorname{Re}_{L}^{1 / 2} Sc ^{1 / 3}\)

 

or

\( Sh _{L} = 0.664 \operatorname{Re}_{L}^{1 / 2} Sc ^{1 / 3} \)

 

In fully developed turbulent flow, values of \( Sh _{L} \) from equation (\( Sh _{L} = 0.0365 Re _{L}^{4 / 5} Sc ^{1 / 3} \)) approach equation (\( \frac{k_{c} L}{D_{A B}} = Sh _{L} = 0.664\left( Re _{t}\right)^{1 / 2} Sc ^{1 / 3} + 0.0365 Sc ^{1 / 3}\left[\left(\operatorname{Re}_{L}\right)^{4 / 5} – \left( Re _{t}\right)^{4 / 5}\right] \)), and the laminar contribution to the convective mass-transfer process can be neglected. Representative plots of \( Sh _{L} \) vs \( \operatorname{Re}_{L} \) for \( Sc = 1.0 \) are provided in Figure.
