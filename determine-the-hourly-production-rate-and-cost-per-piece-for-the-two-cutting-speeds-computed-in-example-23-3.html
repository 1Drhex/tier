Guest Question.

Determine the hourly production rate and cost per piece for the two cutting speeds computed in .

Admin
Solution

(a) For the cutting speed for maximum production, \(v_{max}\) = 50 m/min, machining time per piece and tool life are calculated as follows:

Machining time \(T_{m} = \frac{\pi (0.5)(0.1)}{(0.25)(10^{-3})(50)}\) = 12.57 min/pc

Tool life \(T = \left(\frac{70}{50}\right)^{8}\) = 14.76 min/cutting edge

The number of pieces per tool \(n_{p}\) = 14.76/12.57 = 1.17. Use \(n_{p}\) = 1. From Equation (23.3) \(T_{c} = T{m} + T_{h} + \frac{T_{t}}{n_{p}} \), average production cycle time for the operation is

\(T_{c}\)= 12.57 + 5.0 + 2.0/1 = 19.57 min/pc

Corresponding hourly production rate \(R _{p}\) = 60/19.57 = 3.1 pc/hr. From Equation (23.11) \(C_{c} = C_{o}T{m} + C_{o}T_{h} + \frac{C_{o}T_{t}}{n_{p}} + \frac{C_{t}}{n_{p}} \), average cost per piece for the operation is

\(C_{c}\) = 0.50(12.57) + 0.50(5.0) + 0.50(2.0)/1 + 3.00/1 = $12.79/pc

(b) For the cutting speed for minimum production cost per piece, \(v_{min }\) = 42 m/min, the machining time per piece and tool life are calculated as follows:

Machining time \(T_{m} = \frac{\pi (0.5)(0.1)}{(0.25)(10^{-3})(42)}\) = 14.96 min/pc

Tool life \(T = \left(\frac{70}{42}\right)^{8}\) = 59.54 min/cutting edge

The number of pieces per tool \(n_{p}\) = 59.54/14.96 = 3.98. Use \(n_{p}\) = 3 to avoid failure during the fourth workpiece. Average production cycle time for the operation is

\(T_{c}\)= 14.96 + 5.0 + 2.0/3 = 20.63 min/pc

Corresponding hourly production rate \(R _{p}\) = 60/20.63 = 2.9 pc/hr. Average cost per piece for the operation is

\(C_{c}\) = 0.50(14.96) + 0.50(5.0) + 0.50(2.0)/3 + 3.00/3 = $11.32/pc

Note that production rate is greater for \(v_{max}\) and cost per piece is lower for \(v_{min }\) .
