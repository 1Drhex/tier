Guest Question.

A long thin radiating wire is to be employed as an infrared light source. To maximize the output of infrared energy into the desired direction, the wire is fitted with an insulated, highly reflective sheath as shown in Fig. 5-16. The sheath is cylindrical with radius R (which is much larger than the diameter of the wire), and has a cutout of half-angle (f to let the concentrated infrared light escape. Assuming that the wire is heated with a power of \(Q^{\prime }\) W/m length of wire, and that the sheath can lose heat only by radiation and only from its inside surface, determine the temperature distribution across the sheath.

Admin
Solution

From an energy balance on a surface element dA it follows from equation (5.20) \(q(r)=\epsilon (r)E_{b}(r) – \alpha (r)H(r) = J(r)-H(r)\) that, with q(θ) = 0,

\(\sigma T^{4}(\theta ) = J(\theta ) = H(\theta )\)

and

\(H(\theta ) = \int_{A}^{}{J(\theta^{\prime }) dF_{dA-dA^{\prime }}+H_{0} (\theta ) }\).

We may treat the energy emitted from the wire as external radiation (neglecting absorption by the wire since it is so small). Since the total released energy will spread equally into all directioas, we find

\(H_{0} (\theta )=Q^{\prime }/2\pi R = const\)

The view factor \(dF_{dA-dA^{\prime }}\) between two infinitely long strips on the cylinder surface is given by Configuration 1 in Appendix D as

\(dF_{dA-dA^{\prime }} = \frac{1}{2} \cos \beta d\beta\)

where the angle β is indicated in Fig. 5-16 and may be related to θ through

\(2\beta + \left|\theta^{\prime }-\theta \right| =\pi\)

Differentiating β with respect to θ’ we obtain dβ = ±dθ’/2 jl, depending on whether θ’ is larger or less than θ , Substituting for β in the view factor, this becomes

\(dF_{dA-dA^{\prime }} = \frac{1}{2} \cos \left(\frac{\pi }{2} – \left|\frac{\theta^{\prime }-\theta }{2} \right| \right) \frac{1}{2} d\theta ^{\prime } =\frac{1}{4} \sin \left|\frac{\theta^{\prime }-\theta }{2}\right| d\theta ^{\prime }\)

where the ± has been omitted since the view factor is always positive (i.e., \(\left|d\beta \right|\) is to be used). Substituting this into the above relationship for radiosity we obtain

\(J(\theta )= \frac{1}{4}\int_{-\pi +\varphi }^{\pi -\varphi }{J(\theta ^{\prime }) \sin \left|\frac{\theta^{\prime }-\theta }{2}\right| d\theta ^{\prime }} +H_{0}\)

\(= \frac{1}{4}\int_{-\pi +\varphi }^{\theta } {J(\theta^{\prime }) \sin \frac{\theta^{\prime }-\theta }{2} d\theta^{\prime }} + \frac{1}{4}\int_{\theta }^{\pi -\varphi }{J(\theta ^{\prime }) \sin \frac{\theta-\theta^{\prime }}{2} d\theta ^{\prime }} +H_{0}\)

Since the view factor in the integrand has repetitive derivatives we may convert this integral equation into a second-order differential equation, as was done in the kernel approximation method. Differentiating twice, we have

\(\frac{dJ}{d\theta } = \frac{1}{8}\int_{-\pi +\varphi }^{\theta }{J(\theta ^{\prime }) \cos \frac{\theta-\theta^{\prime }}{2} d\theta '} – \frac{1}{8}\int_{\theta }^{\pi -\varphi }{J(\theta ^{\prime }) \cos \frac{\theta^{\prime }-\theta }{2} d\theta ^{\prime }}\)

\(\frac{d^{2}J}{d\theta ^{2} } =\frac{1}{8} J (\theta ) -\frac{1}{16}\int_{-\pi +\varphi }^{\theta }{J(\theta ^{\prime }) \sin \frac{\theta-\theta^{\prime }}{2} d\theta^{\prime }} + \frac{1}{8} J(\theta ) -\frac{1}{16} \int_{\theta }^{\pi -\varphi }{J(\theta ^{\prime })\sin \frac{\theta^{\prime }-\theta }{2} d\theta ^{\prime }}\)

Comparing this result with the above integral equation for J{θ) we find

\(\frac{d^{2}J}{d\theta ^{2} } =\frac{1}{4} J (\theta ) -\frac{1}{4} \left[J(\theta ) – H_{0} \right] =\frac{1}{4} H_{0}\)

This equation is readily solved as

\(J(\theta )= \frac{1}{8} H_{0}\theta ^{2} + C_{1}\theta +C_{2}\)

The two integration constants must now be determined by substituting the solution back into the goveming integral equation. However, \(C_{1}\) may be determined from symmetry since, for this problem, j(θ) = j{-θ) and \(C_{1}\) = 0. To determine \(C_{2}\) we evaluate J at θ=0 :

\(J(0)= C_{2} = \frac{1}{4}\int_{-\pi +\varphi }^{0}{J(\theta ')\sin \left(-\frac{\theta ^{\prime }}{2} \right) d\theta ^{\prime }} + \frac{1}{4}\int_{0}^{\pi -\varphi }{J(\theta ^{\prime }) \sin \frac{\theta^{\prime }}{2} d\theta^{\prime } } + H_{0}\)

\(= \frac{1}{2}\int_{0}^{\pi -\varphi }{J(\theta ^{\prime }) \sin \frac{\theta ^{\prime }}{2} d\theta ^{\prime } } + H_{0}\)

\(= \frac{1}{2}\int_{0}^{\pi -\varphi }{\left(C_{2} +\frac{H_{0} }{8}\theta ^{2} \right) \sin \frac{\theta^{\prime }}{2} d\theta ^{\prime }} + H_{0}\)

Integrating twice by parts we obtain

\(C_{2}=H_{0} – \left(C_{2} +\frac{H_{0} }{8}\theta ^{2} \right) \cos \frac{\theta^{\prime }}{2} |^{\pi -\varphi }_{0} + \frac{H_{0} }{4} \int_{0}^{\pi -\varphi }{ \theta ^{\prime } \cos \frac{\theta ^{\prime }}{2} d\theta ^{\prime }}\)

\(=H_{0} – \left[(C_{2} +\frac{H_{0} }{8} (\pi -\varphi )^{2}\right] \cos \left(\frac{\pi }{2} -\frac{\varphi }{2} \right) + C_{2} + \frac{H_{0} }{2} \left(\theta ^{\prime } \sin \frac{\theta ^{\prime }}{2} |^{\pi -\varphi }_{0} – \int_{0}^{\pi -\varphi }{ \sin \frac{\theta ^{\prime }}{2} d\theta ^{\prime }} \right)\)

\(= H_{0} + C_{2} – \left[(C_{2} +\frac{H_{0} }{8} (\pi -\varphi )^{2}\right] \sin \frac{\varphi }{2} + \frac{H_{0} }{2} \left((\pi -\varphi ) \sin \left(\frac{\pi }{2} -\frac{\varphi }{2} \right)+2\cos \frac{\theta ^{\prime }}{2} |^{\pi -\varphi }_{0} \right)\)

\(= H_{0} + C_{2} – \left[(C_{2} +\frac{H_{0} }{8} (\pi -\varphi )^{2}\right] \sin \frac{\varphi }{2} + \frac{H_{0} }{2} (\pi -\varphi ) \cos \frac{\varphi }{2} +H_{0} \sin \frac{\varphi }{2} -H_{0}\)

Solving this equation for Ci we get

\(C_{2}=H_{0}\left[1+ \frac{\pi -\varphi }{2}\cos \frac{\varphi }{2}-\frac{1}{8}(\pi -\varphi )^{2} \right]\)

Therefore,

\(T^{4} (\theta )= \frac{J}{\sigma } = \frac{Q'}{2\pi R\sigma } \left\{1+ \frac{\pi -\varphi }{2} \cos \frac{\varphi }{2} -\frac{1}{8} \left[(\pi -\varphi )^{2}-\theta ^{2} \right] \right\}\)

We find that the temperature has a minimum at θ = 0, since around that location the view factor to the opening is maximum, resulting in a maximum of escaping energy. The temperature level increases as φ decreases (since less energy can escape) and reaches \(T\rightarrow \infty\) as φ = 0 (since this produces an insulated closed enclosure with internal heat production).

The fact that long cylindrical surfaces lend themselves to exact analysis was apparently first recognized by Sparrow [25]. The preceding two examples have shown that exact solutions may be found for a number of special geometries, namely, (i) enclosures whose sxirfaces all He on a single sphere, and (ii) enclosures for which view factors between surface elements have repetitive derivatives. For other still fairly simple geometries an approximate analytical solution may be determined fi-om the kernel approximation method. However, the vast majority of radiative heat transfer problems in enclosures without a participating medium must be solved by numerical methods. A large majority of these are solved using the net radiation method described in the first few sections of this chapter. If greater accuracy or better numerical efficiency is desired, one of the numerical methods briefly described in this section needs to be used, such as numerical quadrature leading to a set of linear algebraic equations (as in the net radiation method).
