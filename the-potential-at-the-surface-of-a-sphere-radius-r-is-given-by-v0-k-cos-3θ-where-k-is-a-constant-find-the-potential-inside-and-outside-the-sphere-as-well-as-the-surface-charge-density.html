Guest Question.

The potential at the surface of a sphere (radius R) is given by

\(V_{0}=k \cos 3 \theta\) ,

where k is a constant. Find the potential inside and outside the sphere, as well as the surface charge density σ (θ) on the sphere. (Assume there’s no charge inside or outside the sphere.)

Admin
Solution

\(V_{0}(\theta)=k \cos (3 \theta)=k\left[4 \cos ^{3} \theta-3 \cos \theta\right]=k\left[\alpha P_{3}(\cos \theta)+\beta P_{1}(\cos \theta)\right]\) .

(I know that any \(3^{ rd }\) order polynomial can be expressed as a linear combination of the first four Legendre polynomials; in this case, since the polynomial is odd, I only need \(P_{1} \text { and } P_{3}\).)

\(4 \cos ^{3} \theta-3 \cos \theta=\alpha\left[\frac{1}{2}\left(5 \cos ^{3} \theta-3 \cos \theta\right)\right]+\beta \cos \theta=\frac{5 \alpha}{2} \cos ^{3} \theta+\left(\beta-\frac{3}{2} \alpha\right) \cos \theta\) ,

so

\(4=\frac{5 \alpha}{2} \Rightarrow \alpha=\frac{8}{5} ; \quad-3=\beta-\frac{3}{2} \alpha=\beta-\frac{3}{2} \cdot \frac{8}{5}=\beta-\frac{12}{5} \Rightarrow \beta=\frac{12}{5}-3=-\frac{3}{5}\) .

Therefore

\(V_{0}(\theta)=\frac{k}{5}\left[8 P_{3}(\cos \theta)-3 P_{1}(\cos \theta)\right]\) .

Now

\(V(r, \theta)=\left\{\begin{array}{l}\sum\limits_{l=0}^{\infty }{} A_{l} r^{l} P_{l}(\cos \theta), \text { for } r \leq R(\text { Eq. } 3.66) \\\sum\limits_{l=0}^{\infty }{} \frac{B_{l}}{r^{l+1}} P_{l}(\cos \theta), \text { for } r \geq R(\text { Eq. } 3.72)\end{array}\right\}\) ,

where

\(A_{l}=\frac{(2 l+1)}{2 R^{l}} \int_{0}^{\pi} V_{0}(\theta) P_{l}(\cos \theta) \sin \theta d \theta\) (Eq. 3.69)

\(=\frac{(2 l+1)}{2 R^{l}} \frac{k}{5}\left\{8 \int_{0}^{\pi} P_{3}(\cos \theta) P_{l}(\cos \theta) \sin \theta d \theta-3 \int_{0}^{\pi} P_{1}(\cos \theta) P_{l}(\cos \theta) \sin \theta d \theta\right\}\)

 

\(=\frac{k}{5} \frac{(2 l+1)}{2 R^{l}}\left\{8 \frac{2}{(2 l+1)} \delta_{l 3}-3 \frac{2}{(2 l+1)} \delta_{l 1}\right\}=\frac{k}{5} \frac{1}{R^{l}}\left[8 \delta_{l 3}-3 \delta_{l 1}\right]\)

 

\(=\left\{\begin{array}{l}8 k / 5 R^{3}, \text { if } l=3 \\-3 k / 5 R, \text { if } l=1\end{array}\right\} \text { (zero otherwise) }\) .

Therefore

\(V(r, \theta)=-\frac{3 k}{5 R} r P_{1}(\cos \theta)+\frac{8 k}{5 R^{3}} r^{3} P_{3}(\cos \theta)=\frac{k}{5}\left[8\left(\frac{r}{R}\right)^{3} P_{3}(\cos \theta)-3\left(\frac{r}{R}\right) P_{1}(\cos \theta)\right]\) ,

or

\(\frac{k}{5}\left\{8\left(\frac{r}{R}\right)^{3} \frac{1}{2}\left[5 \cos ^{3} \theta-3 \cos \theta\right]-3\left(\frac{r}{R}\right) \cos \theta\right\} \Rightarrow V(r, \theta)=\frac{k}{5} \frac{r}{R} \cos \theta\left\{4\left(\frac{r}{R}\right)^{2}\left[5 \cos ^{2} \theta-3\right]-3\right\} \)

 

\((\text { for } r \leq R)\) . Meanwhile, \(B_{l}=A_{l} R^{2 l+1}\) (Eq. 3.81—this follows from the continuity of V at R). Therefore

\(B_{l}=\left\{\begin{array}{ll}8 k R^{4} / 5, & \text { if } l=3 \\-3 k R^{2} / 5, & \text { if } l=1\end{array}\right\}\) ( zero otherwise) .

So

\(V(r, \theta)=\frac{-3 k R^{2}}{5} \frac{1}{r^{2}} P_{1}(\cos \theta)+\frac{8 k R^{4}}{5} \frac{1}{r^{4}} P_{3}(\cos \theta)=\frac{k}{5}\left[8\left(\frac{R}{r}\right)^{4} P_{3}(\cos \theta)-3\left(\frac{R}{r}\right)^{2} P_{1}(\cos \theta)\right]\) ,

or

\(V(r, \theta)=\frac{k}{5}\left(\frac{R}{r}\right)^{2} \cos \theta\left\{4\left(\frac{R}{r}\right)^{2}\left[5 \cos ^{2} \theta-3\right]-3\right\} \)

(for r ≥ R). Finally, using Eq. 3.83:

\(\sum_{l=0}^{\infty}(2 l+1) A_{l} R^{l-1} P_{l}(\cos \theta)=\frac{1}{\epsilon_{0}} \sigma_{0}(\theta) \) (3.83)

\(\sigma(\theta)=\epsilon_{0} \sum_{l=0}^{\infty}(2 l+1) A_{l} R^{l-1} P_{l}(\cos \theta)=\epsilon_{0}\left[3 A_{1} P_{1}+7 A_{3} R^{2} P_{3}\right]\)

 

\(=\epsilon_{0}\left[3\left(-\frac{3 k}{5 R}\right) P_{1}+7\left(\frac{8 k}{5 R^{3}}\right) R^{2} P_{3}\right]=\frac{\epsilon_{0} k}{5 R}\left[-9 P_{1}(\cos \theta)+56 P_{3}(\cos \theta)\right]\)

 

\(=\frac{\epsilon_{0} k}{5 R}\left[-9 \cos \theta+\frac{56}{2}\left(5 \cos ^{3} \theta-3 \cos \theta\right)\right]=\frac{\epsilon_{0} k}{5 R} \cos \theta\left[-9+28 \cdot 5 \cos ^{2} \theta-28 \cdot 3\right]\)

 

\(=\frac{\epsilon_{0} k}{5 R} \cos \theta\left[140 \cos ^{2} \theta-93\right]\) .
