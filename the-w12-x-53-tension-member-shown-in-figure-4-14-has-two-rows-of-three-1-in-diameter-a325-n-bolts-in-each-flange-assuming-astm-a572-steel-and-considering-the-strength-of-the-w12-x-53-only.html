Guest Question.

The \(W12 × 53\) tension member shown in Figure 4-14 has two rows of three \(1-in.\)-diameter A325 N bolts in each flange. Assuming ASTM A572 steel and considering the strength of the \(W12 × 53\) only,

1. Determine the design tension strength of the \(W12 × 53,\)

2. Determine the service dead load that can be supported if there is no live load, and

3. If a service dead load, \(P_{D} =100 kips\) is applied, what is the maximum service live load, \(P_{L}\), that can be supported?

Admin
Solution

1. For A572 steel, \(F_{y} = 50 ksi\), \(F_{u} = 65 ksi\) (AISCM Table 2-3).

For 1-in.-diameter bolts, \(d_{\mathrm{hole}}=1 \mathrm{in.}+1 / 8 \mathrm{in.}=1.125 \mathrm{in}\)

From part 1 of the AISCM, for \(W12 53,\)

\(\begin{aligned} A_{g} &=15.6 \mathrm{in}^{2} \\ b_{f} &=10 \mathrm{in} . \\ t_{f} &=0.575 \mathrm{in} . \\ d &=12.1 \mathrm{in} . \end{aligned}\)

 

\(\bar{x}=1.02 \text { in. }\) (AISCM Table 1-8, use value for \(WT\) shape)

\(\begin{aligned} A_{n} &=A_{g}-\Sigma A \text { holes } \\ &=15.6-(4 \text { holes })(1.125)(0.575)=13.0 \text { in. }^{2} \end{aligned}\)

 

From Table 4-1, we obtain \(U = 0.90\), since \(b_{f}>\frac{2}{3} d ; 10 \text { in. }>\frac{2}{3}(12.1)=8.07 \text { in. } \)

Using the calculation method,

\(U=1-\frac{\bar{x}}{\ell}=1-\frac{1.02}{6}=0.83 \rightarrow\) larger value of \(U =0.90\) may be used.

\(ι=\) centerline distance between outer bolts \(= 3 in. + 3 in. = 6 in.\)

\(A_{e}=U A_{n}=(0.9)(13.0)=11.7 \mathrm{in}^{2}\)

 

a. Yielding failure mode: \(\phi P_{n}=0.9 A_{g} F_{y}=(0.9)(50)(15.6)=702 \text { kips }\)

b. Fracture failure mode: \(\phi P_{n}=0.75 F_{u} A_{e}=(0.75)(65)(11.7)=570 \mathrm{kips}\)

c. Block shear failure mode:

\(\begin{aligned} A_{g v} &=(4)(2 \mathrm{in} .+3 \mathrm{in} .+3 \text { in. })(0.575 \mathrm{in} .)=18.4 \mathrm{in} .^{2} \\ A_{g t} &=(4)(2.25 \mathrm{in} .)(0.575 \mathrm{in} .)=5.18 \mathrm{in} .^{2} \\ A_{n v} &=A_{g v}-\Sigma A_{\mathrm{holes}} \\ &=18.4-(4)(2.5)(1.125 \mathrm{in} .)(0.575 \mathrm{in} .)=11.9 \mathrm{in} .^{2} \\ A_{n t} &=A_{g t}-\Sigma A_{\mathrm{holes}} \\ &=5.18-(4)\left(\frac{1}{2}\right)(1.125 \mathrm{in} .)(0.575 \mathrm{in} .)=3.88 \mathrm{in} .^{2} \end{aligned}\)

 

The available block shear strength is found from equation (4-11) \(\left(U_{b s}=1.0\right):\)

\(\begin{aligned} &\begin{aligned} \phi P_{n}=& \phi\left(0.60 F_{u} A_{n v}+U_{b s} F_{u} A_{n t}\right) \leq \phi\left(0.60 F_{y} A_{g v}+U_{b s} F_{u} A_{n t}\right) \\ \phi P_{n}=&(0.75)[(0.60)(65)(11.9)+(1.0)(65)(3.88)] \\ \leq &(0.75)[(0.60)(50)(18.4)+(1.0)(65)(3.88)] \\ & 537 \text { kips }<603 \text { kips } \end{aligned}\\ &\phi P_{n}=537 \text { kips (block shear capacity) } \end{aligned}\)

 

Summary:

Yielding failure mode: \(\phi P_{n}= 702 kips\)

Fracture failure mode: \(\phi P_{n}= 570 kips\)

Block shear failure mode: \(\phi P_{n}= 537 kips\)

The smallest of the three values governs \(\therefore\) Design strength, \(\phi P_{n}= 537 kips \)

2. \(P_{L}= 0\); the two load combinations to be considered will be

\(\begin{array}{l} P_{u}=1.4 P_{D} \quad \text { Governs } \\ P_{u}=1.2 P_{D}+1.6 P_{L}=1.2 P_{D} \end{array}\)

 

Setting the design strength equal to the applied loads yields

\(\begin{array}{l} P_{u}=1.4 P_{D} \leq \phi P_{n}=537 \text { kips } \\ 1.4 P_{D} \leq 537 \text { kips } \end{array}\)

 

Solving for \(P_{D}= 383 kips\) (maximum unfactored dead load that can be supported)

3. Given \(P_{D} = 100 kips\), determine the live load, \(P_{L}\), that can be safely supported:

\(\begin{array}{l} P_{u}=1.2 P_{D}+1.6 P_{L} \\ P_{u}=1.2 P_{D}+1.6 P_{L} \leq \phi P_{n}=537 \text { kips } \\ (1.2)(100)+1.6 P_{L} \leq 537 \text { kips } \end{array}\)

 

Solving for \(P_{L}= 260 kips\) (maximum unfactored live load that can be supported)

\(\phi P_{n}=\phi\left(0.60 F_{u} A_{n v}+U_{b s} F_{u} A_{n t}\right) \leq \phi\left(0.60 F_{y} A_{g v}+U_{b s} F_{u} A_{n t}\right) \quad\quad (4-11)\)
