Guest Question.

In a study of formaldehyde oxidation over Ag catalysts, the power rate law over Ag powder was \(r= kP_{O_{2} }\) and over supported Ag it was \(r=k P^{0.3}_{O_{2} } P^{0.3}_{H_{2}CO } \) [62]. The following sequence of steps was proposed:

\(O_{2 \ \left(g\right) } + 2 \ S \xrightarrow[]{k_{1} } 2 \ O-S \)

 

\(H_{2}CO + 2 \ O-S \xrightleftharpoons[]{K_{2}} HCO_{2}-S + OH-S \)

 

\(HCO_{2}-S+ O-S \xrightarrow[]{k_{2} } CO_{2}+ OH -S + S\)

 

\(2 \ OH-S \xrightarrow[]{1/K_{H_{2}O } } H_{2}O_{\left(g\right) } + O-S + S\)

————————————————————

\(H_{2}CO + O_{2} \Longrightarrow CO_{2}+ H_{2}O\)

Derive a rate expression based on this model. You may assume the OH-S concentration is very small. Can this rate law be further simplified to give either of the two power rate law expressions above? If so, what assumptions are involved in each case?

Admin
Solution

\(\frac{-d \left\lfloor O_{2 \left(g\right) } \right\rfloor }{dt}= r= 2 K_{1}P_{O_{2} } \left[S\right] ^{2} = K_{3} \left[HCO_{2} – S \right] \left[O-S \right]= K P_{O_{2} } \left[S\right]^{2} \) \(L= \left[S\right] + \left[HCO_{2} – S \right] + \left[O-S \right]\)

A. \(\left[HCO_{2} – S \right] = K_{2} P_{H_{2}CO } \left[O-S \right]^{2} / \left[OH – S\right]\)

 

B. \(\left[OH – S\right] = K^{1/2}_{H_{2}O} P^{1/2}_{H_{2}O} \left[O-S \right]^{1/2} \left[ S\right]^{1/2}\)

 

C. \(\left[HCO_{2} – S \right] = \frac{ K_{2} P_{H_{2}CO} \left[O – S \right]^{2} }{ K^{1/2}_{H_{2}O} P^{1/2}_{H_{2}O} \left[O-S \right]^{1/2} \left[ S\right]^{1/2} } = \frac{K_{2} P_{H_{2}CO} \left[O – S \right]^{3/2} \left[ S\right]^{-1/2} }{ K^{1/2}_{H_{2}O} P^{1/2}_{H_{2}O} } \)

 

D. At steady-state: \( \left[O-S \right] = \frac{ K_{1} P_{O_{2} } \left[S \right]^{2} }{ K_{3} \left[ HCO_{2}-S \right] } \) (Balance on O-S species)

 

E. Substitute D in C: \(\left[ HCO_{2}-S \right] = \frac{ K_{2} P_{H_{2}CO} K^{3/2}_{1} P^{3/2}_{O_{2}} \left[S \right]^{5/2} }{ K^{3/2}_{3} \left[ HCO_{2}-S \right]^{3/2} K^{1/2}_{H_{2}} P^{1/2}_{H_{2}} } \Rightarrow \)

 

\(F. \left[\mathrm{HCO}_{2}-\mathrm{S}\right]=\frac{\mathrm{K}_{2}^{2 /5}\mathrm{P}_{\mathrm{H}_{2} \mathrm{CO}}^{2 / 5} \mathrm{k}_{1}^{3 / 5}\mathrm{P}_{\mathrm{O}_{2}}^{3 / 5}[\mathrm{~S}]}{\mathrm{k}_{3}^{3 / 5}\mathrm{~K}_{\mathrm{H}_{2} \mathrm{O}}^{1 / 5} \mathrm{P}_{\mathrm{H}_{2} \mathrm{O}}^{1 /5}}\)

 

\(G. [\mathrm{O}-\mathrm{S}]=\frac{\mathrm{k}_{1} \mathrm{P}_{\mathrm{O}_{2}}^{2 / 5}[\mathrm{~S}] \mathrm{K}_{\mathrm{H}_{2} \mathrm{O}}^{1 / 5} \mathrm{P}_{\mathrm{H}_{2} \mathrm{O}}^{1 / 5}}{\mathrm{k}_{3}^{2 / 5} \mathrm{~K}_{2}^{2 / 5} \mathrm{k}_{1}{ }^{3 / 5} \mathrm{P}_{\mathrm{H}_{2} \mathrm{CO}}^{2 / 5}}\)

 

\(H. \mathrm{L}=[\mathrm{S}]+\frac{\mathrm{K}_{2}^{2 / 5} \mathrm{k}_{1}^{3 / 5} \mathrm{P}_{\mathrm{H}_{2} \mathrm{CO}}^{2 / 5} \mathrm{P}_{\mathrm{O}_{2}}^{3 / 5}[\mathrm{~S}]}{\mathrm{k}_{3}^{3 / 5} \mathrm{~K}_{\mathrm{H}_{2} \mathrm{O}}^{1 / \mathrm{O}} \mathrm{P}_{\mathrm{H}_{2} \mathrm{O}}^{1 / 5}}+\frac{\mathrm{k}_{1}^{2 / 5} \mathrm{P}_{\mathrm{O}_{2}}^{2 / 5} \mathrm{~K}_{\mathrm{H}_{2} \mathrm{O}}^{1 / 5} \mathrm{P}_{\mathrm{H}_{2} \mathrm{O}}^{1 / 5}[\mathrm{~S}]}{\mathrm{k}_{3}^{2 / 5} \mathrm{~K}_{2}^{2 / 5} \mathrm{P}_{\mathrm{H}_{2} \mathrm{CO}}^{2 / 5}}\)

 

\(I. [\mathrm{S}]=\mathrm{L} /\left(1+\mathrm{K}^{\prime} \mathrm{P}_{\mathrm{H}_{2} \mathrm{CO}}^{2 / 5} \mathrm{P}_{\mathrm{O}_{2}}^{3 / 5} \mathrm{P}_{\mathrm{H}_{2} \mathrm{O}}^{-1 / 5}+\mathrm{K}^{\prime \prime} \mathrm{P}_{\mathrm{O}_{2}}^{2 / 5} \mathrm{P}_{\mathrm{H}_{2} \mathrm{C} O}^{-2 / \mathrm{C}} \mathrm{P}_{\mathrm{H}_{2} \mathrm{O}}^{1 / 5}\right)^{2}\)

Assume PH20 is constant ,plus power of 0.2 is low

Then , \(r=LK_{1} P_{O_{2} } / \left(1+ K^{\prime \prime \prime } P^{0.4}_{H_{2}CO } P^{0.6}_{O_{2} } + K^{\prime \prime \prime \prime } P^{0.4}_{H_{2}CO } P^{- 0.4}_{H_{2}CO } \right)^{2} \)

If \(\theta _{HCO_{2} } << 1 \) and \(\theta _{O} << 1\) , then \(r= KP_{O_{2} } \)

If \(\theta _{O} \) is MASI, i.e.,\(\theta _{O} >> \theta _{HCO_{2} }\) , then

\(r=\frac{LK_{1} P_{O_{2} } }{ \left(1+ K^{\prime \prime \prime \prime} P^{0.4}_{O_{2} }/ P^{0.4}_{H_{2}CO } \right)^{2} } = \frac{K^{\prime} P_{O_{2} } P^{0.8}_{H_{2}CO }}{ \left( P^{0.4}_{H_{2}CO }+ K^{\prime \prime \prime \prime} P^{0.4}_{O_{2} } \right)^{2} }\)

(Note typographical error in power dependence on \(H_{2}O\) in ref. 62)
