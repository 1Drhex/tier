Guest Question.

For the linear transformation given in the above , use the matrix \(\mathrm{A}\) to find \(T(\mathbf{u})\) where \(\mathbf{u}=\left(\begin{array}{lll}1 & 0 & 2\end{array}\right)^{T}\) with respect to the same bases

\(B=\left\{\mathbf{v}_{1}=\left(\begin{array}{l}1 \\1 \\1\end{array}\right), \mathbf{v}_{2}=\left(\begin{array}{r}-1 \\0 \\1\end{array}\right), \mathbf{v}_{3}=\left(\begin{array}{l}0 \\0 \\1\end{array}\right)\right\} \text { and } C=\left\{\mathbf{w}_{1}=\left(\begin{array}{l}1 \\2\end{array}\right), \mathbf{w}_{2}=\left(\begin{array}{r}-1 \\1\end{array}\right)\right\}\)
Admin
Solution

We need to find \([T(\mathbf{u})]_{C}=\mathbf{A}[\mathbf{u}]_{B}\). First we determine \([\mathbf{u}]_{B}\).

What does \([\mathbf{u}]_{B}\) represent?

Remember, the entries in the given vector \(\mathbf{u}=\left(\begin{array}{lll}1 & 0 & 2\end{array}\right)^{T}\) are the coordinates with respect to the standard basis (xyz axes). However, we need to write \(\mathrm{u}\) with respect to the given basis \(B\) because \([\mathbf{u}]_{B}\) is the coordinates of the vector \(\mathbf{u}\) with respect to the basis (axes) vectors \(B\).

\( \mathbf{u}=\left(\begin{array}{l} 1 \\ 0 \\ 2 \end{array}\right)=a \mathbf{v}_{1}+b \mathbf{v}_{2}+c \mathbf{v}_{3}=a\left(\begin{array}{l} 1 \\ 1 \\ 1 \end{array}\right)+b\left(\begin{array}{r} -1 \\ 0 \\ 1 \end{array}\right)+c\left(\begin{array}{l} 0 \\ 0 \\ 1 \end{array}\right) (*)\)

These \(a, b\) and \(c\) are not the same values as for .

How can we find these?

Solve the simultaneous equations by expanding \(\left({ }^{*}\right)\) :

\(\left.\begin{array}{l}a-b \quad=1 \\a \quad=0 \\a+b+c=2\end{array}\right\} \text { gives } a=0, b=-1 \text { and } c=3\)

What are the coordinates of \(u\) with respect to the basis (axes) \(B,[\mathbf{u}]_{B}\), equal to?

\( [\mathbf{u}]_{B}=\left(\begin{array}{l} a \\ b \\ c \end{array}\right)=\left(\begin{array}{r} 0 \\ -1 \\ 3 \end{array}\right) \)

Note that \(\mathbf{u}=\left(\begin{array}{lll}1 & 0 & 2\end{array}\right)^{T}\) gives the coordinates of \(\mathbf{u}\) with respect to our standard basis \((x y z\) axes) but \([\mathbf{u}]_{B}=\left(\begin{array}{lll}0 & -1 & 3\end{array}\right)^{T}\) gives the coordinates of \(\mathbf{u}\) with respect to the given basis \(B=\left\{\mathbf{v}_{1}, \mathbf{v}_{2}, \mathbf{v}_{3}\right\}\)

From the above Proposition \((5.31):[T(\mathbf{u})]_{C}=\mathbf{A}[\mathbf{u}]_{B}\)

We found matrix \(\mathrm{A}\) in the previous example, so using this A we have

\( [T(\mathbf{u})]_{C}=\mathbf{A}[\mathbf{u}]_{B}=\left(\begin{array}{crr} 2 / 3 & -4 / 3 & 0 \\ -13 / 3 & 2 / 3 & -1 \end{array}\right)[\mathbf{u}]_{B} \)

and evaluating this right hand side gives

\( [T(\mathbf{u})]_{C}=\left(\begin{array}{rrr} 2 / 3 & -4 / 3 & 0 \\ -13 / 3 & 2 / 3 & -1 \end{array}\right)\left(\begin{array}{r} 0 \\ -1 \\ 3 \end{array}\right)=\left(\begin{array}{c} 4 / 3 \\ -11 / 3 \end{array}\right) \)

We can illustrate the above process as shown in Fig. 5.32

The coordinates of \(T(\mathbf{u})\) where \(\mathbf{u}=\left(\begin{array}{lll}1 & 0 & 2\end{array}\right)^{T}\) with respect to the given basis \(C\) are \(\left(\begin{array}{c}4 / 3 \\ -11 / 3\end{array}\right)\). We have measured the vector \(T(\mathbf{u})\) against the new axes represented by \(\mathbf{w}_{1}\) and \(\mathrm{w}_{2}\) :

\(T(\mathbf{u})=\frac{4}{3} \mathbf{w}_{1}-\frac{11}{3} \mathbf{w}_{2}\)

This is illustrated in Fig. 5.33..

However, the coordinates of \(T(\mathbf{u})\) with respect to our standard basis ( \(x y\) axes) are given by:

\( \begin{aligned} T\left(\left(\begin{array}{l} 1 \\ 0 \\ 2 \end{array}\right)\right)=\frac{4}{3} \mathbf{w}_{1}-\frac{11}{3} \mathbf{w}_{2} &=\frac{4}{3}\left(\begin{array}{l} 1 \\ 2 \end{array}\right)-\frac{11}{3}\left(\begin{array}{r} -1 \\ 1 \end{array}\right)\left[\begin{array}{l} \text { because } \mathbf{w}_{1} \text { and } \mathbf{w}_{2} \\ \text { are vectors in basis } C \end{array}\right] \\ &=\left(\begin{array}{c} 4 / 3 \\ 8 / 3 \end{array}\right)-\left(\begin{array}{r} -11 / 3 \\ 11 / 3 \end{array}\right)=\left(\begin{array}{r} 5 \\ -1 \end{array}\right) \end{aligned} \)

You can check this \(T(\mathbf{u})=\left(\begin{array}{r}5 \\ -1\end{array}\right)\) by applying the given transformation directly with

\( \mathbf{u}=\left(\begin{array}{lll} 1 & 0 & 2 \end{array}\right)^{T} \)

Proposition (5.31). Let \(T: V \rightarrow W\) be a linear transformation and \(V\) and \(W\) be finite-dimensional vector spaces with basis \(B=\left\{\mathbf{v}_{1}, \mathbf{v}_{2}, \ldots, \mathbf{v}_{n}\right\}\) and \(C=\left\{\mathbf{w}_{1}, \mathbf{w}_{2}, \ldots, \mathbf{w}_{m}\right\}\) respectively. Then we can write the transformation as

\([T(\mathbf{u})]_{C}=\mathbf{A}[\mathbf{u}]_{B}\)

where \([T(\mathbf{u})]_{C}\) are the coordinates of \(T(\mathbf{u})\) with respect to the basis (axes) \(C,[\mathbf{u}]_{B}\) are the coordinates of \(\mathbf{u}\) with respect to the basis (axes) \(B\) and \(\mathbf{A}\) is a matrix given by

\(\mathbf{A}=\left(\left[\begin{array}{ll}\left.T\left(\mathbf{v}_{1}\right)\right]_{C} & {\left[T\left(\mathbf{v}_{2}\right)\right]_{C}} & \cdots & \left.\left[T\left(\mathbf{v}_{n}\right)\right]_{C}\right)\end{array}\right.\right.\)
