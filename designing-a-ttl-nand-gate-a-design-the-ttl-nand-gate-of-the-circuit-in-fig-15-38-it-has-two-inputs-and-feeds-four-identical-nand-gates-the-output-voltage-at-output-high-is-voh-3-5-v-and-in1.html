Designing a TTL NAND gate

(a) Design the TTL NAND gate of the circuit in Fig. 15.38. It has two inputs and feeds four identical NAND gates. The output voltage at output high is \( V_{ OH }=3.5 V \text {, and } I_{ N 1}=72.5 \mu \text { A. Assume } V_{ CC }=5 V , V_{ D }=0.7 V \), \( V_{ CE \text { (sat) }}=0.2 V , V_{ BE }=0.7 V , V_{ BC }=0.7 V , \beta_{ F \text { (forced) }}=10, \beta_{ R }=0.1, I_{ C 3 \text { (sat) }}=1 mA \text {, and } I_{ C 4( max )}=1 mA \). Capacitance of each load is, \( C_{ i }=5 pF \).

(b) Use PSpice/SPICE to check your design by plotting the transfer function. Determine \( NM _{ L } \text { and } NM _{ H } \). Model parameters for diodes are

RS=4 TT=0.1NS

and for transistors are

BF=10 BR=0.1 TF=0.1NS TR=10NS VJC=0.85 VAF=50

Admin
Solution

(a) The design can be carried out using the following steps:

Step 1. Using Eq. (15.89), calculate the value of \( R_{1}\):

\( I_{ N 1}=\beta_{ R }\left[\frac{V_{ CC }-V_{ BE 2 \text { (sat) }}-V_{ BE 3 \text { (sat) }}-V_{ BC 1}}{R_{1}}\right] \) (15.89)

 

\( R_{1}=\beta_{ R } \frac{V_{ CC }-2 V_{ BE ( sat )}-V_{ BC }}{I_{ N 1}}=\frac{0.1 \times[5-(2 \times 0.7)-0.7] V }{72.5 \mu A }=4 k \Omega \)

Step 2. Using Eq. (15.91) for \( v_{ O }=V_{ OH }, \text { find the value of } R_{2} \) to meet the output requirement:

\( N_{ H }=\frac{\left(1+\beta_{ F }\right)\left(V_{ CC }-V_{ BE 4}-V_{ D 1}-v_{ O }\right)}{R_{2} I_{ N 1}} \) (15.91)

 

\(\begin{aligned}R_{2} &=\frac{\left(1+\beta_{ F (\text { forced })}\right)\left(V_{ CC }-V_{ BE 4}-V_{ D 1}-V_{ OH }\right)}{N I_{ N 1}} \\&=\frac{(1+10) \times(5-0.7-0.7-3.5) V }{4 \times 72.5 \mu A }=3.79 k \Omega\end{aligned}\)

Step 3. Using Eq. (15.86), find the value of \( I_{ B 1} \) at output high:

\( I_{ B 1}=\frac{V_{ CC }-V_{ BE 2 \text { (sat) }}-V_{ BE \text { (sat) }}-V_{ BC 1}}{R_{1}} \) (15.86)

 

\( I_{ B 1}=\frac{(5-0.7-0.7-0.7) V }{4 k \Omega}=725 \mu A \)

Step 4. Using Eq. (15.88), calculate the low-state load current:

\(I_{ N 0}=\beta_{ R }\left[\frac{V_{ CC }-V_{ CE 3 \text { (sat) }}-V_{ BE 1( sat )}}{R_{1}}\right]\) (15.88)

 

\( I_{ N 0}=\beta_{ R }\left[\frac{V_{ CC }-V_{ CE 3( sat )}-V_{ BEl ( sat )}}{R_{1}}\right]=0.1\left[\frac{(5-0.2-0.7) V }{4 k \Omega}\right]=102.5 \mu A \)

 

The total load current at low state for N = 4 is

\( I_{ L 0}=4 \times 102.5 \mu A =410 \mu A \)

Step 5. Calculate the high-state \( I_{ B 2}, I_{ C 2}, \text { and } I_{ E 2} . \text { The high-state } I_{ B 2} \) is

\( I_{ B 2}=-I_{ C 1}=\left(1+\beta_{ R }\right) I_{ B 1}=(1+0.1) \times 725 \mu A =798 \mu A \)

The high-state \( I_{ C 2} \) is

\( I_{ C 2}=\frac{V_{ CC }-V_{ BE 3( sat )}-V_{ CE 2( sat )}}{R_{2}}=\frac{(5-0.7-0.2) V }{3.79 k \Omega}=1082 \mu A \)

The high-state \( I_{ E 2} \) is

\( I_{ E 2}=I_{ B 2}+I_{ C 2}=798 \mu A +1082 \mu A =1.88 mA \)

Step 6. Assume that most of \( I_{ E 2} \) flows through the base of \( Q _{3} \) rather than through \( R_{ B } . \text { Let } I_{ R }=0.4 I_{ E 2}=0.4 \times 1.88 m =0.75 mA \). Then calculate the value of \( R_{ B } \):

\( R_{ B }=\frac{V_{ BE 3 \text { (sat) }}}{I_{ R }}=\frac{0.7 V }{0.75 mA }=933 \Omega \)

Choose \( R_{ B }=1 k \Omega .\)

Step 7. Calculate the maximum permissible collector low-state current to drive \( Q _{3} \) in saturation:

\(\begin{aligned}&I_{ B 3}=I_{ E 2}-I_{ R }=1.88 m -0.75 m =1.13 mA \\&I_{ C 3(\max )}=\beta_{ F } I_{ B 3}=10 \times 1.13 mA =11.3 mA\end{aligned}\)

 

which is higher than \( I_{ L 0}=410 \mu A \), from step 4. Thus, the design should be satisfactory. (Otherwise steps 1 through 7 should be repeated with a lower value of N, a higher value of \( \beta_{ F \text { (sat) }} \text {, or a lower value of } I_{ N 1} \).) The overdrive factor is

\( k_{ ODF }=\frac{I_{ C 3(\max )}}{I_{ L 0}}=\frac{11.3 m }{410 \mu}=27.6 \)

Step 8. Calculate the base current of \( Q_{4} \) at output high:

\( l_{ B 4}=\frac{N I_{ N 1}}{1+\beta_{ F }}=\frac{4 \times 72.5 \mu A }{1+10}=26.4 \mu A \)

Step 9. Using Eq. (15.92), calculate the value of \( R_{4} \text { for } v_{ O }=V_{ OH } \):

\( I_{ C 4(\max )}=\frac{V_{ CC }-v_{ O }-V_{ CE 4( sat )}-V_{ D 1}}{R_{4}} \) (15.92)

 

\( R_{4}=\frac{V_{ CC }-V_{ OH }-V_{ CE 4( sat )}-V_{ D 1}}{I_{ C 4(\max )}}=\frac{(5-3.5-0.2-0.7) V }{1 mA }=600 \Omega \)

The power rating of \( R_{4} \text { is } P_{ R 4}=(1 mA )^{2} R_{4}=(1 mA )^{2} \times 600=0.6 mW \).

Step 10. Calculate the power rating of \( R_{2} \text { due to } I_{ C 2} \text { or } I_{ B 2} \), whichever has the higher value. Since \( I_{ C 2}=1082 \mu A \) and \( I_{ B 2}=798 \mu A , \text { use } I_{ C 2} \).

\( P_{ R 2}=(1.082 mA )^{2} R_{2}=(1.082 mA )^{2} \times 3.79 k \Omega=4.44 mW \)

Step 11. Calculate the power rating of \( R_{1} \text { due to } I_{ B 1} \text { at output high or } I_{ B 1} \) at output low, whichever has the higher value. Since the base current of \( Q _{1} \) at output high is

\( I_{ B 1}=\frac{V_{ CC }-V_{ BE 2( sat )}}{R_{1}}=\frac{(5-0.7) V }{4 k \Omega}=1.075 mA \)

and \( I_{ B 1} \) at output low is 725 µA (from step 3), use \( I_{ B 1} \) at output high.

\( P_{ R 1}=(1.075 mA )^{2} R_{2}=(1.075 mA )^{2} \times 4 k \Omega=4.6 mW \)

(b) Each transistor has junction capacitances that will affect the switching speed. To examine the propagation time, we will assume that a load resistance of

\( R_{ L }=\frac{V_{ OH }}{N I_{ N 1}}=\frac{3.5 V }{4 \times 72.5 \mu A }=12 k \Omega \)

and an equivalent capacitance of

\( C_{ L }=4 \times 5 pF =20 pF \)

is connected to the output.

The PSpice schematic is shown in Fig. 15.41. The PSpice plots of the characteristics, which are shown in Fig. 15.42, give \( V_{ OH }=3.53 V \text { (expected value is } 3.5 V \text { ), } V_{ IL }=0.6 V \text { at } V_{ O }=3.49 V , V_{ OL }=42 mV \text { at } V_{ O }=5 V \), and \( V_{ IH }=1.5 V \text { at } V_{ O }=62 mV \).

\(\begin{gathered}NM _{ L }=V_{ IL }-V_{ OL }=0.6-0.042 V \approx 0.56 V \\NM _{ H }=V_{ OH }-V_{ IH }=3.53-1.5=2.03 V \\t_{ pHL }=4.53 ns , t_{ pLH }=6.68 ns , \text { and } t_{ pd }=5.61 ns\end{gathered}\)
