Guest Question.

The plate shown in Fig. 5.14 has a variety of boundary conditions. The left surface is isothermal at 500 C. The bottom surface is isothermal at 100 C. The top and right surfaces are insulated, and the inside corner on the left has convection to a fluid. The convective coefficient is 100 \(W / m ^{2} C\), and the fluid temperature is 30 C. The plate is steel and has a conductivity of 43 W/m C. The grid is square with \(\Delta x=\Delta y=8 cm\).

(a) If there is no internal heat generation, determine the nodal temperatures for steady state.

(b) If the plate has internal heat generation at a strength of \(10^{6} W / m ^{3}\), determine the nodal temperatures for steady state.

Admin
Solution

Let us start the solution by deriving the nodal equations. For a particular node, we assume that all heat flows go into the node and we sum up the heat flows to zero. Again, we will use â€œb” as the thickness of the plate. As Part (b) has heat generation, we will include the heat generation term in the nodal equations.

Insulated nodes (2, 3, 4, 8, 10):

Each node gets heat flow by conduction from its adjacent nodes. For node 2, we have conduction from nodes 1, 3, and 6. The nodal equation is

 

\(\frac{k b(\Delta y / 2)}{\Delta x}\left(T_{1}-T_{2}\right)+\frac{k b(\Delta y / 2)}{\Delta x}\left(T_{3}-T_{2}\right)+\frac{k b \Delta x}{\Delta y}\left(T_{6}-T_{2}\right)+q_{g e n} b \frac{\Delta y}{2} \Delta x=0\) (5.74)

 

For node 3, we have conduction from nodes 2, 4, and 7, and the nodal equation is

 

\(\frac{k b(\Delta y / 2)}{\Delta x}\left(T_{2}-T_{3}\right)+\frac{k b(\Delta y / 2)}{\Delta x}\left(T_{4}-T_{3}\right)+\frac{k b \Delta x}{\Delta y}\left(T_{7}-T_{3}\right)+q_{g e n} b \frac{\Delta y}{2} \Delta x=0\) (5.75)

 

Similarly for nodes 4, 8, and 10:

Node 4

 

\(\frac{k b(\Delta y / 2)}{\Delta x}\left(T_{3}-T_{4}\right)+\frac{k b(\Delta x / 2)}{\Delta y}\left(T_{8}-T_{4}\right)+q_{g e n} b \frac{\Delta y}{2} \frac{\Delta x}{2}=0\) (5.76)

 

Node 8

 

\(\frac{k b(\Delta x / 2)}{\Delta y}\left(T_{4}-T_{8}\right)+\frac{k b(\Delta x / 2)}{\Delta y}\left(T_{10}-T_{8}\right)+\frac{k b \Delta y}{\Delta x}\left(T_{7}-T_{8}\right)+q_{g e n} b \frac{\Delta x}{2} \Delta y=0\) (5.77)

 

Node 10

 

\(\frac{k b(\Delta x / 2)}{\Delta y}\left(T_{8}-T_{10}\right)+\frac{k b(\Delta x / 2)}{\Delta y}\left(T_{12}-T_{10}\right)+\frac{k b \Delta y}{\Delta x}\left(T_{9}-T_{10}\right)+q_{g e n} b \frac{\Delta x}{2} \Delta y=0\) (5.78)

 

Convection nodes (6, 7, 9):

These nodes get conduction from adjacent nodes and convection from the adjacent fluid.

For node 6, we have conduction from nodes 2, 5, and 7 and convection from the fluid. The nodal equation is

 

\(\frac{k b(\Delta y / 2)}{\Delta x}\left(T_{5}-T_{6}\right)+\frac{k b(\Delta y / 2)}{\Delta x}\left(T_{7}-T_{6}\right)+\frac{k b \Delta x}{\Delta y}\left(T_{2}-T_{6}\right)+h b \Delta x\left(T_{\infty}-T_{6}\right)+q_{g e n} b \frac{\Delta y}{2} \Delta x=0\) (5.79)

 

Similarly for nodes 7 and 9:

Node 7

 

\(\begin{aligned}&\frac{k b(\Delta y / 2)}{\Delta x}\left(T_{6}-T_{7}\right)+\frac{k b(\Delta x / 2)}{\Delta y}\left(T_{9}-T_{7}\right)+\frac{k b \Delta y}{\Delta x}\left(T_{8}-T_{7}\right)+\frac{k b \Delta x}{\Delta y}\left(T_{3}-T_{7}\right) \\&+h b((\Delta x / 2)+(\Delta y / 2))\left(T_{\infty}-T_{7}\right)+q_{g e n} b\left(\frac{3}{4} \Delta x \Delta y\right)=0\end{aligned}\) (5.80)

 

Node 9

 

\(\frac{k b(\Delta x / 2)}{\Delta y}\left(T_{7}-T_{9}\right)+\frac{k b(\Delta x / 2)}{\Delta y}\left(T_{11}-T_{9}\right)+\frac{k b \Delta y}{\Delta x}\left(T_{10}-T_{9}\right)+h b \Delta y\left(T_{\infty}-T_{9}\right)+q_{g e n} b \frac{\Delta x}{2} \Delta y=0\) (5.81)

 

Isothermal nodes (1, 5, 11, 12):

 

\(T_{1}=T_{5}=500 C \text { and } T_{11}=T_{12}=100 C\)

 

We will solve the nodal equations by matrix inversion. We have a square grid \((\Delta x=\Delta y)\), which simplifies the equations considerably. To use matrix inversion, we need to rearrange the terms in Eqs (5.74)e(5.82) to facilitate creation of a coefficients matrix and a constants vector. Doing this, and incorporating the square matrix feature, the equations become

 

\(\text { For Node } 1: T_{1}=500\) (5.83)

 

\(\text { For Node } 2:-\frac{1}{2} T_{1}+2 T_{2}-\frac{1}{2} T_{3}-T_{6}=\frac{q_{g e n}(\Delta x)^{2}}{2 k}\) (5.84)

 

\(\text { For Node } 3:-\frac{1}{2} T_{2}+2 T_{3}-\frac{1}{2} T_{4}-T_{7}=\frac{q_{g e n}(\Delta x)^{2}}{2 k}\) (5.85)

 

\(\text { For Node 4: }-\frac{1}{2} T_{3}+T_{4}-\frac{1}{2} T_{8}=\frac{q_{g e n}(\Delta x)^{2}}{4 k}\) (5.86)

 

\(\text { For Node } 5: T_{5}=500\) (5.87)

 

\(\text { For Node } 6:-T_{2}-\frac{1}{2} T_{5}+\left(2+\frac{h \Delta x}{k}\right) T_{6}-\frac{1}{2} T_{7}=\frac{q_{g e n}(\Delta x)^{2}}{2 k}+\frac{h \Delta x}{k} T_{\infty}\) (5.88)

 

\(\text { For Node 7: }-T_{3}-\frac{1}{2} T_{6}+\left(3+\frac{h \Delta x}{k}\right) T_{7}-T_{8}-\frac{1}{2} T_{9}=\frac{3}{4} \frac{q_{g e n}(\Delta x)^{2}}{k}+\frac{h \Delta x}{k} T_{\infty}\) (5.89)

 

\(\text { For Node 8: }-\frac{1}{2} T_{4}-T_{7}+2 T_{8}-\frac{1}{2} T_{10}=\frac{q_{g e n}(\Delta x)^{2}}{2 k}\) (5.90)

 

\(\text { For Node 9: }-\frac{1}{2} T_{7}+\left(2+\frac{h \Delta x}{k}\right) T_{9}-T_{10}-\frac{1}{2} T_{11}=\frac{q_{g e n}(\Delta x)^{2}}{2 k}+\frac{h \Delta x}{k} T_{\infty}\) (5.91)

 

\(\text { For Node 10: }-\frac{1}{2} T_{8}-T_{9}+2 T_{10}-\frac{1}{2} T_{12}=\frac{q_{g e n}(\Delta x)^{2}}{2 k}\) (5.92)

 

\(\text { For Node } 11: T_{11}=100\) (5.93)

 

\(\text { For Node } 12: T_{12}=100\) (5.94)

 

For our problem, \(k=43 W / mC , h=100 W / m ^{2} C , T_{\infty}=30 C , \text { and } \Delta x=8 cm =0.08 m\).

We put these values into Eqs. (5.83)e(5.94) and created the coefficients matrix and the constants vector. For Part (a) of the problem, \(q_{\text {gen }}=0 . \text { For Part (b) of the problem, } q_{\text {gen }}=10^{6} W / m ^{3}\). The problem was solved using matrix inversion and Matlab, with the following results:
