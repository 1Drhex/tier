Solution

Given: Financial statements given in Tables 2.4 and 2.5, and other financial data provided in , tax rate = 28.77%.
Find: Compute the financial ratios.
(a) The amount of working capital:

Working capital = Current assets – Current liabilities
= $77,400 – $28,000
= 49,400

(b) The debt ratio:

\(Debt ratio = \frac{Total debt}{Total assets} = \frac{\$58,000}{\$161,400} = 35.94 \%\)

(c) The time-interest-earned ratio:

\(Times-interest-earned ratio = \frac{EBIT}{Interest expense} = \frac{\$33,280}{\$5,200} = 6.40 times\)

(d) The current ratio:

\(Current ratio = \frac{Current assets}{Current labilities} = \frac{\$77,400}{\$28,000} = 2.76 times\)

(e) The quick ratio:

\(Quick ratio =\frac{Current assets – Inventory}{Current labilities} = \frac{\$77,400 – \$37,700}{\$28,000}\)

 

\(= 1.42 times\)

(f) The inventory-turnover ratio:

\(Inventory Turnover =\frac{Sales}{Average Inventory} = \frac{\$300,000}{\left(\$39,800 + \$37,700\right)/2 } = 7.74 times\)

The typical item sits in inventory almost 1.55 months (12 months/7.74) or 46.51 days before being sold.

(g) The days-sales-outstanding ratio:

\(DSO = \frac{A/R}{Average sales per day} = \frac{\$23,700}{\$300,000/365} = 28.84 days\)

(h) The total asset turnover ratio:

\(Total Asset Turnover = \frac{Sales}{Total Assets}=\frac{\$300,000}{\$161,400} = 1.86 times\)

(i) The net profit margin:

\(Net Margin Ratio = \frac{Net Income (\$)}{Net Sales}=\frac{\$20,000}{\$300,000} = 6.67\%\)

(j) The return on total assets:

\(Return on Total Assets = \frac{Net Income + interest expenses (1 – tax rate)}{Average total assets}\)

 

\(= \frac{\$20,000 + \$5,200 (1 – 0.2877)}{(\$161,400 + \$169,900)/2} = 14.31 \%\)

(k) The return on common equity.

\(Return on Equity = \frac{Net Income – Cash Dividend to Preferred Stockholders}{Average Common Equity}\)

 

\(=\frac{\$20,000 – \$600}{(\$93,400 + \$83,400)/2} = 21.95 \%\)
