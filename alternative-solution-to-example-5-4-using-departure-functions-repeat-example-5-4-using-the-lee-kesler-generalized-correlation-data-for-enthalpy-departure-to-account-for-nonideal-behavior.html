Guest Question.

Alternative Solution to Using Departure Functions

Repeat using the Lee–Kesler generalized correlation data for enthalpy departure to account for nonideal behavior.

Admin
Solution

The schematic of the process is drawn in Figure E5.4A. To find the heat, we need to calculate the enthalpy difference between the outlet (state 2) and the inlet (state 1). From Equation (5.47), we get:

 

\(h_{2}-h_{1}=-\Delta h_{T_{1}, P_{1}}^{ dep }+\left[\int_{T_{1}}^{T_{2}} c_{P} d T+0\right]+\Delta h_{T_{ g }, P_{2}}^{ dep }\) (5.47)

 

\(q+w_{s}=h_{2}-h_{1}=-\Delta h_{T_{1}, P_{1}}^{ dep }+\left[\int_{T_{1}}^{T_{2}} c_{P} d T+0\right]+\Delta h_{T_{2}, P_{2}}^{ dep }\) (E5.7A)

 

To find values of enthalpy departure, we can use the Lee–Kesler tables. They have the form:

 

\(\frac{\Delta h_{T_{r}, P_{r}}^{\text {dep }}}{R T_{c}}=\frac{h_{T_{r}, P_{r}}-h_{T_{r}, P_{r}}^{\text {ideal gas }}}{R T_{c}}=\left[\frac{h_{T_{r}, P_{r}}-h_{T_{r}, P_{r}}^{\text {ideal gas }}}{R T_{c}}\right]^{(0)}+\omega\left[\frac{h_{T_{r} ,P_{r}}-h_{T_{r} P_{r}}^{\text {ideal gas }}}{R T_{c}}\right]^{(1)}\)

 

Looking up the critical properties and acentric factor for n-butane from Appendix A.1 gives:

 

\(T_{c}=425.2[ K ]\)

 

\(P_{c}=37.9[\text { bar }]\)

 

\(\omega=0.199\)

 

Thus, the reduced coordinates for state 1 and state 2 are:

 

\(T_{1, r}=\frac{T_{1}}{T_{c}}=\frac{353.15[ K ]}{425.2[ K ]}=0.83 \quad \text { and } \quad P_{1, r}=\frac{P_{1}}{P_{c}}=\frac{9.47[ bar ]}{38.0[ bar ]}=0.20\)

 

\(T_{2, r}=\frac{T_{2}}{T_{c}}=\frac{393.15[ K ]}{425.2[ K ]}=0.925 \quad \text { and } \quad P_{2, r}=\frac{P_{2}}{P_{c}}=\frac{18.9[ bar ]}{37.9[ bar ]}=0.50\)

 

We can fi nd the values for the enthalpy departure terms in Tables C.3 and C.4 in Appendix C. For state 1, by interpolation:

 

\(\left[\frac{h_{T_{1}, r, P_{1}, r}-h_{T_{1}, r, P_{1}, r}^{\text {ideal gas }}}{R T_{c}}\right]^{(0)}=-0.413\left[\frac{h_{T_{1}, r, P_{1}, r}-h_{T_{1}, r, P_{1}, r}^{\text {ideal gas }}}{R T_{c}}\right]^{(1)}=-0.622\)

 

so,

 

\(\frac{h_{T_{1}, r, P_{1}, r}-h_{T_{1}, r, P_{1}, r}^{\text {ideal gas }}}{R T_{c}}=\left[\frac{h_{T_{1}, r, P_{1}, r}-h_{T_{1}, r, P_{1}, r}^{\text {ideal gas }}}{R T_{c}}\right]^{(0)}+\omega\left[\frac{h_{T_{1}, r, P_{1}, r}-h_{T_{1}, r, P_{1}, r}^{\text {ideal gas }}}{R T_{c}}\right]^{(1)}=-0.536\) (E5.7B)

 

and for state 2:

 

\(\left[\frac{h_{T_{2}, r, P_{2}, r}-h_{T_{2}, r, P_{2}, r}^{\text {ideal gas }}}{R T_{c}}\right]^{(0)}=-0.771 \text { and }\left[\frac{h_{T_{2}, r, P_{2}, r}-h_{T_{2}, r, P_{2}, r}^{\text {ideal gas }}}{R T_{c}}\right]^{(1)}=-0.994\)

 

so,

 

\(\frac{h_{T_{2}, r, P_{2}, r}-h_{T_{2}, r, P_{2}, r}{R T_{c}}^{\text {ideal gas }}}{R T_{c}}=\left[\frac{h_{T_{2}, r, P_{2}, r}-h_{T_{2}, r, P_{2}, r}^{\text {ideal gas }}}{R T_{c}}\right]^{(0)}+\omega\left[\frac{h_{T_{2}, r, P_{2}, r}-h_{T_{2}, r, P_{2}, r}^{\text {ideal gas }}}{R T_{c}}\right]^{(1)}=-0.969\) (E5.7C)

 

The value for the integral of the heat capacity is the same as in :

 

\(\Delta h_{T_{1} \rightarrow T_{2}}^{\text {ideal gas }}=\int_{T_{1}}^{T_{2}} c_{P} d T=R \int_{353}^{393}\left[1.935+36.915 \times 10^{-3} T-11.402 \times 10^{-6} T^{2}\right] d T=4,696[ J / mol ]\) (E5.7D)

 

Plugging in the values from Equations (E5.7B), (E5.7C), and (E5.7D) into Equation (E5.7A) gives:

 

\(h_{2}-h_{1}=0.536 R T_{c}+4,696-0.969 R T_{c}=3,167[ J / mol ]\)

 

The value calculated using the Redlich–Kwong equation of state differs from this value by 11.0%. What value do you think is more accurate? Solving for heat, we get:

 

\(q=\Delta h-w_{s}=1067[ J / mol ]\)
