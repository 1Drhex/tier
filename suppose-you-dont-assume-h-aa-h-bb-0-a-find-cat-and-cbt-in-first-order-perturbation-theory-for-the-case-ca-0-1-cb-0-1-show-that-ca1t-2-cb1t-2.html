Guest Question.

Suppose you don’t assume \( H_{a a}^{\prime}=H_{b b}^{\prime}=0 \) .

(a) Find \( c_{a}(t) \text { and } c_{b}(t) \) in first-order perturbation theory, for the case \( c_{a}(0)=1, c_{b}(0)=0 \text {. Show that }\left|c_{a}^{(1)}(t)\right|^{2}+\left|c_{b}^{(1)}(t)\right|^{2}=1 \) , to first order in \( \hat{H}^{\prime} \).

(b) There is a nicer way to handle this problem. Let

\( d_{a} \equiv e^{\frac{i}{\hbar} \int_{0}^{t} H_{a a}^{\prime}\left(t^{\prime}\right) d t^{\prime}} c_{a}, \quad d_{b} \equiv e^{\frac{i}{\sqrt{2}} \int_{0}^{t} H_{b b}^{\prime}\left(t^{\prime}\right) d t^{\prime}} c_{b} \) (11.26).

Show that

\( \dot{d}_{a}=-\frac{i}{\hbar} e^{i \phi} H_{a b}^{\prime} e^{-i \omega_{0} t} d_{b} ; \quad \dot{d}_{b}=-\frac{i}{\hbar} e^{-i \phi} H_{b a}^{\prime} e^{i \omega_{0} t} d_{a} \). (11.27).

where

\( \phi(t) \equiv \frac{1}{\hbar} \int_{0}^{t}\left[H_{a a}^{\prime}\left(t^{\prime}\right)-H_{b b}^{\prime}\left(t^{\prime}\right)\right] d t^{\prime} \) (11.28).

So the equations for \( d_{a} \text { and } d_{b} \) are identical in structure to Equation 11.17 (with an extra factor \( \left.e^{i \phi} \text { tacked onto } \hat{H}^{\prime}\right) \text {. } \)

\( \dot{c}_{a}=-\frac{i}{\hbar} H_{a b}^{\prime} e^{-i \omega_{0} t} c_{b}, \quad \dot{c}_{b}=-\frac{i}{\hbar} H_{b a}^{\prime} e^{i \omega_{0} t} c_{a} \) (11.17).

(c) Use the method in part (b) to obtain \( c_{a}(t) \text { and } c_{b}(t) \) in first-order
perturbation theory, and compare your answer to (a). Comment on any discrepancies.

Admin
Solution

(a)

\( \left.\begin{array}{l} \text { Eq. } 111.14 \Longrightarrow \dot{c}_{a}=-\frac{i}{\hbar}\left[c_{a} H_{a a}^{\prime}+c_{b} H_{a b}^{\prime} e^{-i \omega_{0} t}\right] \\ \text { Eq. } 11.17 \Longrightarrow \dot{c}_{b}=-\frac{i}{\hbar}\left[c_{b} H_{b b}^{\prime}+c_{a} H_{b a}^{\prime} e^{i \omega_{0} t}\right] \end{array}\right\} \text { (these are exact, and replace Eq. 11.15). } \).

\( \dot{c}_{a}=-\frac{i}{\hbar}\left[c_{a} H_{a a}^{\prime}+c_{b} H_{a b}^{\prime} e^{-i\left(E_{b}-E_{a}\right) t / \hbar}\right] \) (11.14).

\( \dot{c}_{a}=-\frac{i}{\hbar} H_{a b}^{\prime} e^{-i \omega_{0} t} c_{b}, \quad \dot{c}_{b}=-\frac{i}{\hbar} H_{b a}^{\prime} e^{i \omega_{0} t} c_{a} \) (11.17).

\( \dot{c}_{b}=-\frac{i}{\hbar}\left[c_{b} H_{b b}^{\prime}+c_{a} H_{b a}^{\prime} e^{i\left(E_{b}-E_{a}\right) t / \hbar}\right] \) (11.15).

Initial conditions: \( c_{a}(0)=1, \quad c_{b}(0)=0 \),

Zeroth order: \( c_{a}(t)=1, \quad c_{b}(t)=0 \).

First order: \( \left\{\begin{array}{l} \dot{c}_{a}=-\frac{i}{\hbar} H_{a a}^{\prime} \quad \Longrightarrow c_{a}(t)=1-\frac{i}{\hbar} \int_{0}^{t} H_{a a}^{\prime}\left(t^{\prime}\right) d t^{\prime}\\ \dot{c}_{b}=-\frac{i}{\hbar} H_{b a}^{\prime} e^{i \omega_{0} t} \Longrightarrow c_{b}(t)=-\frac{i}{\hbar} \int_{0}^{t} H_{b a}^{\prime}\left(t^{\prime}\right) e^{i \omega_{0} t^{\prime}} d t^{\prime} \end{array}\right. \).

\( \left.\left|c_{a}\right|^{2}=\left[1-\frac{i}{\hbar} \int_{0}^{t} H_{a a}^{\prime}\left(t^{\prime}\right) d t^{\prime}\right]\left[1+\frac{i}{\hbar} \int_{0}^{t} H_{a a}^{\prime}\left(t^{\prime}\right) d t^{\prime}\right]=1+\left[\frac{1}{\hbar} \int_{0}^{t} H_{a a}^{\prime}\left(t^{\prime}\right) d t^{\prime}\right]^{2}=1 \text { (to first order in } H^{\prime}\right) . \).

\( \left.\left|c_{b}\right|^{2}=\left[-\frac{i}{\hbar} \int_{0}^{t} H_{b a}^{\prime}\left(t^{\prime}\right) e^{i \omega_{0} t^{\prime}} d t^{\prime}\right]\left[\frac{i}{\hbar} \int_{0}^{t} H_{a b}^{\prime}\left(t^{\prime}\right) e^{-i \omega_{0} t^{\prime}} d t^{\prime}\right]=0 \text { (to first order in } H^{\prime}\right) \).

\( \text { So }\left|c_{a}\right|^{2}+\left|c_{b}\right|^{2}=1 \text { (to first order) } \).

(b)

\( \dot{d}_{a}=e^{\frac{i}{\hbar} \int_{0}^{t} H_{a a}^{\prime}\left(t^{\prime}\right) d t^{\prime}}\left(\frac{i}{\hbar} H_{a a}^{\prime}\right) c_{a}+e^{\frac{i}{\hbar} \int_{0}^{t} H_{a a}^{\prime}\left(t^{\prime}\right) d t^{\prime}} \dot{c}_{a} . \quad \text { But } \dot{c}_{a}=-\frac{i}{\hbar}\left[c_{a} H_{a a}^{\prime}+c_{b} H_{a b}^{\prime} e^{-i \omega_{0} t}\right] \).

Two terms cancel, leaving

\( \dot{d}_{a}=-\frac{i}{\hbar} e^{\frac{i}{\hbar} \int_{0}^{t} H_{a a}^{\prime}\left(t^{\prime}\right) d t^{\prime}} c_{b} H_{a b}^{\prime} e^{-i \omega_{0} t} . \quad \text { But } c_{b}=e^{-\frac{i}{\hbar} \int_{0}^{t} H_{b b}^{\prime}\left(t^{\prime}\right) d t^{\prime}} d_{b} \).

\( =-\frac{i}{\hbar} e^{\frac{i}{\hbar} \int_{0}^{t}\left[H_{a a}^{\prime}\left(t^{\prime}\right)-H_{b b}^{\prime}\left(t^{\prime}\right)\right] d t^{\prime}} H_{a b}^{\prime} e^{-i \omega_{0} t} d_{b}, \quad \text { or } \quad \dot{d}_{a}=-\frac{i}{\hbar} e^{i \phi} H_{a b}^{\prime} e^{-i \omega_{0} t} d_{b} \).

Similarly,

\( \dot{d}_{b}=e^{\frac{i}{\hbar} \int_{0}^{t} H_{b b}^{\prime}\left(t^{\prime}\right) d t^{\prime}}\left(\frac{i}{\hbar} H_{b b}^{\prime}\right) c_{b}+e^{\frac{i}{\hbar} \int_{0}^{t} H_{b b}^{\prime}\left(t^{\prime}\right) d t^{\prime}} \dot{c}_{b} . \quad \text { But } \dot{c}_{b}=-\frac{i}{\hbar}\left[c_{b} H_{b b}^{\prime}+c_{a} H_{b a}^{\prime} e^{i \omega_{0} t}\right] \).

\( =-\frac{i}{\hbar} e^{\frac{1}{\hbar} \int_{0}^{t} H_{b b}^{\prime}\left(t^{\prime}\right) d t^{\prime}} c_{a} H_{b a}^{\prime} e^{i \omega_{0} t} . \quad \text { But } c_{a}=e^{-\frac{i}{\hbar} \int_{0}^{t} H_{a a}^{\prime}\left(t^{\prime}\right) d t^{\prime}} d_{a} \).

\( =-\frac{i}{\hbar} e^{\frac{i}{\hbar} \int_{0}^{t}\left[H_{b b}^{\prime}\left(t^{\prime}\right)-H_{a a}^{\prime}\left(t^{\prime}\right)\right] d t^{\prime}} H_{b a}^{\prime} e^{i \omega_{0} t} d_{a}=-\frac{i}{\hbar} e^{-i \phi} H_{b a}^{\prime} e^{i \omega_{0} t} d_{a} \). QED

(c)

Initial conditions: \( c_{a}(0)=1 \Longrightarrow d_{a}(0)=1 ; \quad c_{b}(0)=0 \Longrightarrow d_{b}(0)=0 \).

Zeroth order: \( d_{a}(t)=1, \quad d_{b}(t)=0 \).

First order: \( \dot{d}_{a}=0 \Longrightarrow d_{a}(t)=1 \Longrightarrow c_{a}(t)=e^{-\frac{i}{\hbar} \int_{0}^{t} H_{a a}^{\prime}\left(t^{\prime}\right) d t^{\prime}} \).

\( \dot{d}_{b}=-\frac{i}{\hbar} e^{-i \phi} H_{b a}^{\prime} e^{i \omega_{0} t} \Longrightarrow d_{b}=-\frac{i}{\hbar} \int_{0}^{t} e^{-i \phi\left(t^{\prime}\right)} H_{b a}^{\prime}\left(t^{\prime}\right) e^{i \omega_{0} t^{\prime}} d t^{\prime} \Longrightarrow \)

\( c_{b}(t)=-\frac{i}{\hbar} e^{-\frac{i}{\hbar} \int_{0}^{t} H_{b b}^{\prime}\left(t^{\prime}\right) d t^{\prime}} \int_{0}^{t} e^{-i \phi\left(t^{\prime}\right)} H_{b a}^{\prime}\left(t^{\prime}\right) e^{i \omega_{0} t^{\prime}} d t^{\prime} \).

These don’t look much like the results in (a), but remember, we’re only working to first order in \( H^{\prime} \) so \( c_{a}(t) \approx 1-\frac{i}{\hbar} \int_{0}^{t} H_{a a}^{\prime}\left(t^{\prime}\right) d t^{\prime} \text { (to this order), while for } c_{b}, \text { the factor } H_{b a}^{\prime} \) in the integral means it is already first order and hence both the exponential factor in front and \( e^{-i \phi} \) should be replaced by 1. Then \( c_{b}(t) \approx-\frac{i}{\hbar} \int_{0}^{t} H_{b a}^{\prime}\left(t^{\prime}\right) e^{i \omega_{0} t^{\prime}} d t^{\prime} \), and we recover the results in (a).
