Guest Question.

Consider the following sequence of steps to describe the high-temperature kinetics of NO reduction by \(CH_{4}\)on \(La_{2} O_{3} \)and Sr/\(La_{2} O_{3} \) in the absence of \(O_{2}\):

(a) Provide any necessary stoichiometric coefficient for each step to balance the overall reaction, i.e., no active centers are contained in the overall stoichiometry;

(b) Derive a kinetic rate expression from this sequence assuming NO* and \(CH_{2}\)* are the only two significant surface species;
(c) Does the final rate expression have the capability of fitting the rate data reported in the paper by Vannice et al. [4], which gave reaction ordersof 0.19–0.26 for \(CH_{4}\)and 0.73–0.98 for NO? Why?

Admin
Solution

Write balanced sequence:

(1) \(4 \left[NO + * \xrightleftharpoons[]{K_{NO} } NO * \right] \)

(2) \(NO * + * \xrightleftharpoons[]{K_{1} } N*+ O*\)

(3) \(CH_{4}+ O* \xrightleftharpoons[]{K_{2} } CH_{2}* + H_{2}O\)

(4) \(CH_{2}* + NO* \xrightarrow[]{K} CHNO * + H* \)

(5) \(H* + CHNO* + 2 \ NO * \xrightleftharpoons[]{K_{3} } CO_{2}+ N_{2}+ H_{2}O + N* + 3*\)

(6) \(2 N* \xrightleftharpoons[]{1/K_{N_{2} } } N_{2}+ 2*\)

———————————————————

\(4 NO + CH_{4} \Longrightarrow 2 N_{2}+CO_{2}+ 2 H_{2}O\)

Step 4 is RDS \(\therefore r= K \left[CH_{2} * \right] \left[NO * \right]\)

 

\(\left[\mathrm{CH}_{2} *\right]=\frac{\mathrm{K}_{2} \mathrm{P}_{\mathrm{CH}_{4}}[O *]}{\mathrm{P}_{\mathrm{H}_{2} \mathrm{O}}}, \quad[\mathrm{O} *]=\frac{\mathrm{K}_{1}[\mathrm{NO} *][*]}{[\mathrm{N} *]},[\mathrm{NO} *]=\mathrm{K}_{\mathrm{NO}} \mathrm{P}_{\mathrm{NO}}[*]\)

 

\(\left[O * \right] = \frac{K_{1} K_{NO} P_{NO} \left[*\right] ^{2} }{ \left[N*\right]} \) , \(\left[N*\right] ^{2}= K_{N_{2} } P_{N_{2} } \left[*\right] ^{2}\)

 

\(\left[O * \right]= \frac{K_{1} K_{NO} P_{NO} \left[*\right]}{K^{1/2}_{N_{2} } P^{1/2}_{N_{2} } } \) , \(\left[CH_{2}*\right]= \frac{K_{2} P_{CH_{4} } K_{1} K_{NO} \left[*\right]}{ K^{1/2}_{N_{2} } P^{1/2}_{N_{2} } P_{H_{2}O }} \)

 

\(r=\frac{K K_{1}K_{2} K^{2}_{NO } P_{CH_{4} } P^{2}_{NO } \left[*\right] ^{2} }{ K^{1/2}_{N_{2} } P^{1/2}_{N_{2} } P_{H_{2}O } } \)

 

\(L= \left[*\right]+ \left[NO*\right]+ \left[ CH_{2}*\right]= \left[*\right] + K_{NO} P_{NO} \left[*\right]+ \frac{K_{1}K_{2} K_{NO } P_{CH_{4} } P_{NO }\left[*\right] }{ K^{1/2}_{N_{2} } P^{1/2}_{N_{2} } P_{H_{2}O } } \)

 

\(\left[*\right] = \frac{L}{ \left\lgroup 1+ K_{NO } P_{NO } + \frac{K P_{CH_{4} } P_{NO } }{ P^{1/2}_{N_{2} } P_{H_{2}O } } \right\rgroup } \) where \(K= \frac{K_{1}K_{2} K_{NO } }{ K^{1/2}_{N_{2} } }\)

 

and \(r= \frac{L K K_{1}K_{2} K^{2}_{NO} P_{CH_{4} } P^{2}_{NO} }{ K^{1/2}_{N_{2} } P^{1/2}_{N_{2} } P_{H_{2}O } \left\lgroup 1+ K_{NO } P_{NO } + \frac{K P_{CH_{4} } P_{NO } }{P^{1/2}_{N_{2} } P_{H_{2}O } } \right\rgroup ^{2} } \)
