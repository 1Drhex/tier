Guest Question.

The coordinates of event A are \(\left(x_{A}, 0,0\right), t_{A}\), and the coordinates of event B are \(\left(x_{B}, 0,0\right), t_{B}\). Assuming the displacement between them is spacelike, find the velocity of the system in which they are simultaneous.

Admin
Solution
\(\text { Using Eq. } 12.18 \text { (iv): } \Delta \bar{t}=\gamma\left(\Delta t-\frac{v}{c^{2}} \Delta x\right)=0 \Rightarrow \Delta t=\frac{v}{c^{2}} \Delta x, \text { or } v=\frac{\Delta t}{\Delta x} c^{2}=\frac{t_{B}-t_{A}}{x_{B}-x_{A}} c^{2}\)

 

\(\begin{matrix} \text { (i) } \bar{x}=\gamma(x-v t) \text {, } \\ \text { (ii) } \bar{y}=y \text {, } \\ \text { (iii) } \bar{z}=z \text {, } \\ \text { (iv) } \bar{t}=\gamma\left(t-\frac{v}{c^{2}} x\right) \text {. } \end{matrix}\) (12.18)
