Solution

Geometric Properties:

W18×50 \(r_{t s}=1.98 \text { in. } \quad S_{x}=88.9 \text { in. }{ }^{3} \quad J=1.24 \text { in. }{ }^{4} \quad h_{o}=17.4 \text { in }\)

Required strength from Example F.1-3a

LRFD ASD
\(M_{u}=266 kip – ft\) \(M_{a}=184 \text { kip-ft }\)

Calculate the nominal flexural strength, \(M_{n}\)

\(\text { Calculate } C_{b}\) \(C_{b}=\frac{12.5 M_{\max }}{2.5 M_{\max }+3 M_{A}+4 M_{B}+3 M_{C}} R_{m} \leq 3.0\)

Eqn. F1-1

The required moments for Equation F1-1 can be calculated as a percentage of the maximum midspan moment as: \(M _{\max }=1.00, M _{A}=0.438, M _{B}=0.750, \text { and } M _{C}=0.938\)

\(R_{m}\) = 1.0 for doubly-symmetric members

\(C_{b}=\frac{12.5(1.00)}{2.5(1.00)+3(0.438)+4(0.750)+3(0.938)}(1.0)=1.30\)

 

\(\begin{array}{l}L_{p}=5.83 ft \\L_{r}=17.0 ft\end{array}\)

Manual
Table 3-6

For a compact beam with an unbraced length \(L_{b}>L_{r}\), the limit state of elastic lateral-torsional buckling applies.

Calculate \(F_{c r} \text { with } L_{b}=17.5 ft\)

 

\(F_{c r}=\frac{C_{b} \pi^{2} E}{\left(\frac{L_{b}}{r_{t s}}\right)^{2}} \sqrt{1+0.078 \frac{J c}{S_{x} h_{o}}\left(\frac{L_{b}}{r_{t s}}\right)^{2}}\) where c = 1.0 for doubly symmetric I-shapes

Eqn. F2-4

 

\(F_{c r}=\frac{1.30 \pi^{2}(29,000 ksi )}{\left(\frac{17.5 ft (12 in / ft )}{1.98 in .}\right)^{2}} \sqrt{1+0.078 \frac{\left(1.24 in .^{4}\right) 1.0}{\left(88.9 in. ^{3}\right)(17.4 in .)}\left(\frac{17.5 ft (12 in / ft )}{1.98 in .}\right)^{2}}=43.2 ksi\)

 

\(M_{n}=F_{c r} S_{x} \leq M_{p}\)

Eqn. F2-3

 

\(M_{n}=43.2 ksi \left(88.9 in .^{3}\right)=3840 \text { kip-in. }<5050 \text { kip-in. }\)

 

\(M_{n}=3840 \text { kip-in or } 320 \text { kip-ft }\)

Calculate the available flexural strength

LRFD ASD
\(\phi_{b}=0.90\) \(\Omega_{b}=1.67\)
\(\phi_{b} M_{n}=0.90(320 kip – ft )=288 kip – ft\) \(\frac{M_{n}}{\Omega_{b}}=\frac{320 kip – ft }{1.67}=192 kip – ft\)
288 kip-ft > 266 kip-ft o.k. 192 kip-ft > 184 kip-ft o.k.
