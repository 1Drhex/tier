Guest Question.

A new business jet design concept has a clean (gear and flaps retracted) subsonic (below \(M_{\text {crit }}\) = 0.78) drag polar of \(C_{D}=0.02+0.06 C_{L}^{2}, \text { a } C_{L_{\max }}\) for takeoff and landing of 1.6, a \(C_{D_{0}}\) for takeoff and landing of 0.04, and a design point of \(T_{ SL } / W_{ TO }=0.35 \text { and } W_{ TO } / S=65 lb / ft ^{2}\). Its design mission consists of takeoff at sea level, acceleration to M = 0.5, climb to h = 40,000 ft, cruise at \(M=M_{\text {crit }}\) for 2000 n miles, then descent and landing with enough fuel to loiter ath = 10,000 for 20 min. What is the fuel fraction for this design concept and this mission? Assume \(c_{t_{\text {sea level }}}=0.8 / h\).

 

Admin
Solution

For the takeoff \(\rho=0.002377 slug / ft ^{3}, a=1116.2 ft / s , \alpha=1, \beta=1 , \text { and } \mu=0.03\) ,

\(\begin{gathered}V_{ TO }=1.2 \sqrt{\frac{2 W_{ TO }}{\rho S C_{L_{\max }}}}=1.2 \sqrt{\frac{2\left(65 lb / ft ^{2}\right)}{0.002377 slug / ft ^{3}(1.6)}}=221.8 ft / s \\0.7 V_{ TO }=0.7(221.8)=155.3 ft / s\end{gathered}\)

 

\(\begin{aligned}&q \text { at } 0.7 V_{ TO }=\frac{1}{2} \rho V^{2}=\frac{1}{2}\left(0.002377 slug / ft ^{2}\right)(155.3 ft / s )^{2}=28.67 lb / ft ^{2} \\&t_{ TO }=V_{ TO } / \frac{ d V}{ d t}=221.8 ft / s ^{2} /\left\{g\left[\left(\frac{\alpha}{\beta} \frac{T_{ SL }}{W_{ TO }}\right)-\frac{q}{\beta}\left(\frac{C_{D_{0}}}{W_{ TO } / S}\right)-\mu\right]\right\}\end{aligned}\)

 

\(\begin{gathered}t_{ TO }=221.8 ft / s ^{2} /\left\{32.2 ft / s ^{2}\left[(0.35)-\left(28.67 lb / ft ^{2}\right)\left(\frac{0.04}{65 lb / ft ^{2}}\right)-0.03\right]\right\} \\t_{ TO }=22.78 s\end{gathered}\)

 

\(\frac{W_{f_{ TO }}}{W_{ TO }}=\alpha \frac{T_{ SL }}{W_{ TO }} c_{t} t_{ TO }=(0.35)\left(\frac{0.8}{ h }\right)(22.78 s )\left(\frac{1 h }{3600 s }\right)=0.00177\)

 

For the acceleration,

\(\begin{gathered}\rho=0.002377 slug / ft ^{3} \quad a=1116.2 ft / s \quad \alpha=1 \\\beta=1-0.00177=0.9982\end{gathered}\)

and the acceleration is basically from \(V_{\text {TO to }} M_{\infty}=0.5 \text { or }\)

\(V_{\infty}=M_{\infty} a=0.5(1116.2 ft / s )=558.1 ft / s\)

The average velocity during the acceleration is

\(\frac{558.1 ft / s +221.8 ft / s }{2}=390 ft / s\)

and so the average q is

\(\begin{aligned}q=\frac{1}{2} \rho V_{\infty}^{2} &=\frac{1}{2}\left(0.002377 slug / ft ^{2}\right)(390 ft / s )^{2}=180.7 lb / ft ^{2} \\\frac{ d V}{ d t}=g &\left[\left(\frac{\alpha}{\beta} \frac{T_{ SL }}{W_{ TO }}\right)-\frac{q}{\beta}\left(\frac{C_{D_{0}}}{W_{ TO } / S}\right)-\frac{n^{2} \beta}{q} k_{1}\left(\frac{W_{ TO }}{S}\right)\right] \\=& 32.2 ft / s ^{2}\left[\left(\frac{0.35}{.9985}\right)-\frac{180.7 lb / ft ^{2}}{0.9985}\left(\frac{0.02}{65 lb / ft ^{2}}\right)\right.\\&\left.-\frac{0.9985}{180.7 lb / ft ^{2}}(0.06)\left(65 lb / ft ^{2}\right)\right]=8.8 ft / s ^{2}\end{aligned}\)

 

The approximate time to accelerate, using the average value method, is

\(t_{\text {accel }}=\frac{(558.1 ft / s -221.8 ft / s )}{8.8 ft / s ^{2}}=38.2 s\)

and the fuel fraction used to accelerate is

\(\frac{W_{f_{\text {accel }}}}{W_{\text {TO }}}=0.35(0.8 / h )(38.2 s )\left(\frac{1 h }{3600 s }\right)=0.0029\)

For the climb, the average altitude is 20,000 ft, where \(\rho=0.001267 \text { slug/ft }^{3} \text { and } a=1036.9 ft / s \text {, so that }\)

\(\begin{gathered}V_{\infty}=M_{\infty} a=0.5(1036.9 ft / s )=518.5 ft / s \\\beta=0.9982-0.0029=0.9953 \\n=1\end{gathered}\)

 

\(\begin{gathered} \alpha=\left(\rho / \rho_{ SL }\right)=\left(0.001267 slug / ft ^{3} / 0.002377 slug / ft ^{3}\right)=0.533 \\c_{T}=c_{\text {Tsea level }}\left(a / a_{ SL }\right)=0.8 / h (1036.9 ft / s ) /(1116.2 ft / s )=0.743 / h \\q=\frac{1}{2} \rho V_{\infty}^{2}=\frac{1}{2}\left(0.001267 slug / ft ^{2}\right)(518.5 ft / s )^{2}=170.3 lb / ft ^{2}\end{gathered}\)

 

\(\begin{aligned}\frac{ d h}{ d t}=& V\left[\left(\frac{\alpha}{\beta} \frac{T_{ SL }}{W_{ TO }}\right)-\frac{q}{\beta}\left(\frac{C_{D_{0}}}{W_{ TO } / S}\right)-\frac{n^{2} \beta}{q} k_{1}\left(\frac{W_{ TO }}{S}\right)\right] \\=& 518.5 ft / s \left\{\left[\frac{0.533}{0.9956}(0.35)\right]-\frac{170.3 lb / ft ^{2}}{0.9956}\left(\frac{0.02}{65 lb / ft ^{2}}\right)\right.\\&\left.-\frac{0.9953}{170.3 lb / ft ^{2}} 0.06\left(65 lb / ft ^{2}\right)\right\}=69.1 ft / s\end{aligned}\)

 

\(\begin{gathered}t_{\text {climb }}=\frac{\left(h_{\text {final }}-h_{\text {initial }}\right)}{ d h / d t}=\frac{(40,000 ft -0)}{69.1 ft / s }=579 s \\\frac{W_{f_{\text {climb }}}}{W_{\text {TO }}}=\alpha \frac{T_{ SL }}{W_{\text {TO }}} c_{T} t_{\text {climb }}=(0.533)(0.35)(0.743 / h )(579 s )\left(\frac{1 h }{3600 s }\right)=0.022\end{gathered}\)

 

For the cruise, the altitude is 40,000 ft, where \(\rho=0.000587 \text { slug/ft }^{3} \text { and } a=968.1 ft/s, \text {so that}\)

\(\begin{gathered}V_{\infty}=M_{\infty} a=0.78(968.1 ft / s )=755.1 ft / s =446.8 kn \\\beta=0.9953-0.022=0.9733 \quad n=1\end{gathered}\)

 

\(\begin{gathered}c_{T}=c_{T \text { sea level }}\left(a / a_{ SL }\right)=0.8 / h (968.1 ft / s ) /(1116.2 ft / s )=0.694 / h \\q=\frac{1}{2} \rho V_{\infty}^{2}=\frac{1}{2}\left(0.000587 slug / ft ^{2}\right)(755.1 ft / s )^{2}=167.4 lb / ft ^{2}\end{gathered}\)

 

\(\begin{gathered}C_{L}=\frac{n W}{q S}=\frac{n \beta}{q} \frac{W_{ TO }}{S}=\frac{0.9733}{167.4 lb / ft ^{2}}\left(65 lb / ft ^{2}\right)=0.378 \\C_{D}=C_{D_{0}}+k C_{L}^{2}=0.02+0.06(0.378)^{2}=0.0286\end{gathered}\)

 

The time to travel 2000 n miles is \(R / V_{\infty}=2000 n \text { miles } / 446.8 n \text { miles } / h = 4.48 h\) so that

\(\begin{aligned}\frac{W_{f}}{W_{ TO }} &=\beta\left(1-\exp \left\{-\left[c_{t} \Delta t /\left(\frac{C_{L}}{C_{D}}\right)\right]\right\}\right) \\&=0.9953\left(1-\exp \left\{-\left[(0.694)(4.48) /\left(\frac{0.378}{0.0286}\right)\right]\right\}\right)=0.209\end{aligned}\)

Finally, for the loiter (because descents and landings are not modeled) the altitude is 10,000 ft, where

\(\begin{aligned}\rho=0.001756 slug / ft ^{3} \quad a=1077.4 ft / s \quad n=1 \quad \beta &=0.9733-0.209 \\&=0.764\end{aligned}\)

 

\(c_{T}=c_{T \text { sea level }}\left(a / a_{ SL }\right)=0.8 / h (1077.4 ft / s ) /(1116.2 ft / s )=0.772 / h\)

 

The loiter would be flown at maximum endurance airspeed, where

\(\left(\frac{C_{L}}{C_{D}}\right)_{\max }=\frac{1}{2 \sqrt{k C_{D_{0}}}}=\frac{1}{2 \sqrt{0.06(0.02)}}=14.4\)

The loiter time is 20 min = 0.33 h, and so

\(\begin{aligned}\frac{W_{f}}{W_{ TO }} &=\beta\left(1-\exp \left\{-\left[c_{t} \Delta t /\left(\frac{C_{L}}{C_{D}}\right)\right]\right\}\right) \\&=0.764(1-\exp \{-[(0.772)(0.33) / 14,4]\})=0.013\end{aligned}\)

 

\(\begin{aligned}\frac{W_{f_{\text {mission }}}}{W_{\text {TO }}} &=\frac{W_{f_{\text {takeoff }}}}{W_{\text {TO }}}+\frac{W_{f_{\text {accel }}}}{W_{\text {TO }}}+\frac{W_{f_{\text {climb }}}}{W_{\text {TO }}}+\frac{W_{f_{\text {cruise }}}}{W_{\text {TO }}}+\frac{W_{f_{\text {loiter }}}}{W_{\text {TO }}} \\&=0.0018+0.0029+0.022+0.209+0.013=0.248\end{aligned}\)
