\(\psi(x)=\lambda\int _{ a }^{ b }{ K(x,y)\psi(y) } dy \)
has the form
\(K(x,y)=\sum_{ n=0 }^{ \infty }{ { h }_{ n }(x){ g }_{ n }(y) }\)
where the \({h}_{n}(x)\) form a complete orthonormal set of functions over the interval [a, b].
(a) Show that the eigenvalues \({λ}_{i}\) are given by
\(|M − {λ}^{-1}I| = 0\),
where M is the matrix with elements
\({ M }_{ kj }=\int_{ a }^{ b }{ { g }_{ k }(u){ h }_{ j }(u) } du\).
If the corresponding solutions are \({ \psi }^{ (i) }(x)=\sum_{ n=0 }^{ \infty }{ { a }_{ n }^{ (i) }{ h }_{ n }(x) }\), find an expression for \({ a }_{ n }^{ (i) }\).
(b) Obtain the eigenvalues and eigenfunctions over the interval [0, 2π] if
\(K(x,y)=\sum _{ n=1 }^{ \infty }{ \frac { 1 }{ n } \cos { nx } \cos { ny } }\).
