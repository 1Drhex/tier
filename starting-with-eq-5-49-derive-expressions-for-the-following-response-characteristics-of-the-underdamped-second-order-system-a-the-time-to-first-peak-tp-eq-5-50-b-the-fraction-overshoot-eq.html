Guest Question.

Starting with Eq. 5-49, derive expressions for the following response characteristics of the underdamped second-order system.

(a) The time to first peak \(t_{p}\) (Eq. 5-50).

(b) The fraction overshoot (Eq. 5-51).

(c) The decay ratio (Eq. 5-52).

(d) The settling time \(\left(t_{s}\right.\), defined in Fig. 5.9). Can a single expression be used for \(t_{s}\) over the full range of \(\zeta, 0<\zeta<1\) ?

\(\begin{gathered}y(t)=K M\left\{1-e^{-\zeta t / \tau}\left[\cos \left(\frac{\sqrt{1-\zeta^{2}}}{\tau} t\right)\right.\right. \\\left.\left.+\frac{\zeta}{\sqrt{1-\zeta^{2}}} \sin \left(\frac{\sqrt{1-\zeta^{2}}}{\tau} t\right)\right]\right\}\end{gathered}\) (5-49)

\(\text { Time to first peak: } \quad t_{p}=\pi \tau / \sqrt{1-\zeta^{2}}\) (5-50)

\( \text {Overshoot: } \quad OS =\exp \left(-\pi \zeta / \sqrt{1-\zeta^{2}}\right)\) (5-51)

\(\text { Decay ratio: } \quad DR =( OS )^{2}=\exp \left(-2 \pi \zeta / \sqrt{1-\zeta^{2}}\right)\) (5-52)

Admin
Solution

For underdamped responses,

\(y(t)=K M\left\{1-e^{-\zeta t / \tau}\left[\cos \left(\frac{\sqrt{1-\zeta^{2}}}{\tau} t\right)+\frac{\zeta}{\sqrt{1-\zeta^{2}}} \sin \left(\frac{\sqrt{1-\zeta^{2}}}{\tau} t\right)\right]\right\}\) (5-51)

a) At the response peaks,

\(\begin{gathered}\frac{d y}{d t}=K M\left\{\frac{\zeta}{\tau} e^{-\zeta t / \tau}\left[\cos \left(\frac{\sqrt{1-\zeta^{2}}}{\tau} t\right)+\frac{\zeta}{\sqrt{1-\zeta^{2}}} \sin \left(\frac{\sqrt{1-\zeta^{2}}}{\tau} t\right)\right]\right. \\\left.-e^{-\zeta t / \tau}\left[-\frac{\sqrt{1-\zeta^{2}}}{\tau} \sin \left(\frac{\sqrt{1-\zeta^{2}}}{\tau} t\right)+\frac{\zeta}{\tau} \cos \left(\frac{\sqrt{1-\zeta^{2}}}{\tau} t\right)\right]\right\}=0\end{gathered}\)

\(\text { Since } K M \neq 0 \text { and } e^{-\zeta t / \tau} \neq 0\)

 

\(0=\left(\frac{\zeta}{\tau}-\frac{\zeta}{\tau}\right) \cos \left(\frac{\sqrt{1-\zeta^{2}}}{\tau} t\right)+\left(\frac{\zeta^{2}}{\tau \sqrt{1-\zeta^{2}}}+\frac{\sqrt{1-\zeta^{2}}}{\tau}\right) \sin \left(\frac{\sqrt{1-\zeta^{2}}}{\tau} t\right)\)

 

\(0=\sin \left(\frac{\sqrt{1-\zeta^{2}}}{\tau} t\right)=\sin n \pi \quad, \quad t=n \frac{\pi \tau}{\sqrt{1-\zeta^{2}}}\)

where n is the number of the peak.

\(\text { Time to the first peak, } \quad t_{p}=\frac{\pi \tau}{\sqrt{1-\zeta^{2}}}\)

b)

\(\text { Overshoot, OS }=\frac{y\left(t_{p}\right)-K M}{K M}\)

 

\(\begin{aligned}OS &=-\exp \left(\frac{-\zeta t}{\tau}\right)\left[\cos (\pi)+\frac{\zeta}{\sqrt{1-\zeta^{2}}} \sin (\pi)\right] \\&=\exp \left[\frac{-\zeta \tau \pi}{\tau \sqrt{1-\zeta^{2}}}\right]=\exp \left[\frac{-\pi \zeta}{\sqrt{1-\zeta^{2}}}\right]\end{aligned}\)

c)

\(\text { Decay ratio, } DR =\frac{y\left(t_{3 p}\right)-K M}{y\left(t_{p}\right)-K M}\)

 

\(\text { where } y\left(t_{3 p}\right)=\frac{3 \pi \tau}{\sqrt{1-\zeta^{2}}} \text { is the time to the third peak. }\)

 

\(\begin{aligned}DR =\frac{K M e^{-\zeta t_{3 p} / \tau}}{K M e^{-\zeta t_{p} / \tau}} &=\exp \left[-\frac{\zeta}{\tau}\left(t_{3 p}-t_{p}\right)\right]=\exp \left[-\frac{\zeta}{\tau}\left(\frac{2 \pi \tau}{\sqrt{1-\zeta^{2}}}\right)\right] \\&=\exp \left[\frac{-2 \pi \zeta}{\sqrt{1-\zeta^{2}}}\right]=( OS )^{2}\end{aligned}\)

d) Consider the trigonometric identity

sin (A+B) = sin A cos B + cos A sin B

\(\text { Let } B =\left(\frac{\sqrt{1-\zeta^{2}}}{\tau} t\right), \quad \sin A =\sqrt{1-\zeta^{2}}, \quad \cos A =\zeta\)

 

\(\begin{aligned}y(t) &=K M\left\{1-e^{-\zeta t / \tau} \frac{1}{\sqrt{1-\zeta^{2}}}\left[\sqrt{1-\zeta^{2}} \cos B+\zeta \sin B\right]\right\} \\&=K M\left\{1-\frac{e^{-\zeta t / \tau}}{\sqrt{1-\zeta^{2}}} \sin (A+B)\right\}\end{aligned}\)

 

\(\text { Hence for } t \geq t_{s} \text {, the settling time, }\)

 

\(\left|\frac{e^{-\zeta t / \tau}}{\sqrt{1-\zeta^{2}}}\right| \leq 0.05 \quad \text {, or } \quad t \geq-\ln \left(0.05 \sqrt{1-\zeta^{2}}\right) \frac{\tau}{\zeta}\)

 

\(\text { Therefore, } \quad t_{s} \geq \frac{\tau}{\zeta} \ln \left(\frac{20}{\sqrt{1-\zeta^{2}}}\right)\)
