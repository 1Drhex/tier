Guest Question.

Evaluate the THD factor of the input current and the power factor of the single phase
full-bridge fully controlled rectifier when the output current is considered
to be pure dc of \(\bar{I}_{0} \) value.

Admin
Solution

Solution

As shown in Fig. 5.7 the input current is a square-wave waveform with\(\overline{ I }_{ o }\) amplitudes. Therefore, the input current can be represented by the following Fourier series:

\(i _{ i }=\sum_{ n =1,2,3,}^{\infty}\left(\widehat{ I }_{ i 1, n } \cos n \omega t +\widehat{ I }_{ i 2, n } \sin \omega \omega t\right)=\sum_{ n =1,2,3,}^{\infty} \widehat{ I }_{ i , n } \cos \left( n \omega t -\theta_{ n }\right)\)

where

\(\begin{aligned} &\widehat{ I }_{ i 1, n }=\frac{1}{\pi} \int_{\alpha}^{\alpha+2 \pi} i _{ i }(\omega t ) \cos ( n \omega t ) d \omega t =-\frac{4 \overline{ I }_{ o }}{ n \pi} \sin ( n \alpha) \\ &\widehat{ I }_{ i 2, n }=\frac{1}{\pi} \int_{\alpha}^{\alpha+2 \pi} i _{ i }(\omega t ) \sin ( n \omega t ) d \omega t =\frac{4 \overline{ I }_{ o }}{ n \pi} \cos n \alpha \\ &\theta_{ n }=\tan ^{-1}\left(\frac{\widehat{ I }_{ i 2, n }}{\widehat{ I }_{ i 1, n }}\right) \\ & n =1,3,5 \cdots \cdots \end{aligned} \)

 

The rms input current nth harmonic component is:

 

\(\widetilde{ I }_{ i , n }=\frac{1}{\sqrt{2}} \sqrt{\hat{ I }_{ i 1, n }^{2}+\hat{ I }_{ i 2, n }^{2}}=\frac{4 \overline{ I }_{ o }}{\sqrt{2} n \pi}\)

 

Also, the rms values of the fundamental, third, fifth, and seventh harmonic components are:

 

\(\widetilde{ I }_{ i , 1}=\frac{4 \overline{ I }_{ o }}{\sqrt{2} \pi} \widetilde{ I }_{ i , 3}=\frac{4 \overline{ I }_{ o }}{3 \sqrt{2} \pi}=\frac{1}{3} \tilde{ I }_{ i , 1} \quad \widetilde{ I }_{ i , 5}=\frac{1}{5} \tilde{ I }_{ i , 1} \quad \widetilde{ I }_{ i , 7}=\frac{1}{7} \tilde{ I }_{ i , 1}\)

 

Since the input current is a square wave with amplitudes \(\pm \bar{I}_{0}\), its rms value is:

 

\(\begin{aligned} \widetilde{I}_{i} &=\left[\frac{1}{\pi} \int_{\alpha}^{\alpha+\pi}\left(\bar{I}_{o}^{2}\right) d (\omega t)\right]^{1 / 2} \\ &=\left[\left.\frac{1}{\pi}\left(\bar{I}_{o}^{2}\right) \omega t\right|_{\alpha} ^{\alpha+\pi}\right]^{1 / 2}=\bar{I}_{0} \end{aligned}\)

 

Also,

\(S _{ i }= input apparent power=\widetilde{ V }_{ i } \widetilde{ I }_{ i }=\widetilde{ V }_{ i } \overline{ I }_{ o }\)

 

\(P _{ i }=input active power =\widetilde{ V }_{ i } \widetilde{ I }_{ i , 1} \cos \varphi_{1}\)

Therefore, taking into consideration the above results the total harmonic distortion of the input current is given by:

 

\(\begin{aligned} THD _{ i } \% &=\frac{\sqrt{ I _{ i , 3}^{2}+\widetilde{ I }_{ i , 5}^{2}+\widetilde{ I }_{ i , 7}^{2}+\ldots}}{\widetilde{ I }_{ i , 1}} \times 100 \approx \frac{\sqrt{ I _{ i , 3}^{2}+\widetilde{ I }_{ i , 5}^{2}}}{\widetilde{ I }_{ i , 1}} \times 100 \\ &=\sqrt{\left(\frac{1}{3}\right)^{2}+\left(\frac{1}{5}\right)^{2}} \times 100=38.7 \% \end{aligned}\)

 

Finally, knowing that the output current is pure dc of \(\overline{ I }_{0}\) value and the firing angle, α = \(\varphi_{1}\), then the power factor of the rectifier is given by

 

\(\lambda=\frac{ P _{ i }}{ S _{ i }}\)

 

 

\(=\frac{\widetilde{ V }_{ i } \widetilde{ I }_{ i , 1} \cos \varphi_{1}}{\widetilde{ V }_{ i } \widetilde{ I }_{ i }}\)

 

 

\(=\frac{\frac{4 \overline{ I }_{ o }}{\sqrt{2} \pi} \cos (-\alpha)}{\overline{ I }_{ o }}=0.9 \cos \alpha\)
