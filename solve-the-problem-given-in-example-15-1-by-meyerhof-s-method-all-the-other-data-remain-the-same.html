Guest Question.

Solve the problem given in by Meyerhof ‘s method. All the other data remain the same.

Admin
Solution

Per Table 9.3, the sand in-situ may be considered in a loose state for \(\phi=30^{\circ}\). The corrected SPT value \(N_{\text {cor }}=10\).

 

Point bearing capacity

From Eq. (15.20)

 

\(Q_{b}=q_{o}^{\prime} N_{q} A_{b} \leq q_{b l} A_{b}\) (15.20)

 

\(q_{b}=q_{0}^{\prime} N_{q} \leq q_{b l}\)

 

From Eq. (15.19 b)

 

for loose sand: \(q_{b l}=25 N_{q} \tan \phi kN / m ^{2}\) (15.19b)

 

\(q_{b l}=25 N_{q} \tan \phi kN / m ^{2}\)

 

Now From Fig. 15. 10 \(N_{q}=60 \text { for } \phi=30^{\circ}\)

 

\(\begin{aligned}q_{0}^{\prime} &=\gamma L =17.5 \times 15=262.5 kN / m ^{2} \\q_{b} &=262.5 \times 60=15,750 kN / m ^{2} \\q_{b l} &=25 \times 60 \times \tan 30^{\circ}=866 kN / m ^{2}\end{aligned}\)

 

Hence the limiting value for \(q_{b}=866 kN / m ^{2}\)

 

Now, \(Q_{b}=A_{b} q_{b}=A_{b} q_{b}=\frac{3.14}{4} \times(0.45)^{2} \times 866=138 kN\)

 

Frictional resistance

Per Section 15.17, the unit skin resistance \(f_{s}\) is assumed to increase from 0 at ground level to a limiting value of \(f_{s l} \text { at } L_{c}=20 d \text { where } L_{c}=\) critical depth and d = diameter. Therefore \(L_{c}=20 \times 0.45=9\) m

 

Now \(f_{s l}=q_{ o }^{\prime} \bar{K}_{s} \tan \delta=\gamma L_{c} \bar{K}_{s} \tan \delta\)

 

Given: \(\gamma=17.5 kN / m ^{3}, L_{c}=9 m , \quad \bar{K}_{s}=1.0 \text { and } \delta=22.5^{\circ}\) m.

 

Substituting and simplifying we have

 

\(f_{s l}=17.5 \times 9 \times 1.0 \times \tan 22.5=65 kN / m ^{2}\)

 

The skin load \(Q_{f}=Q_{f l}+Q_{f 2}=\frac{1}{2} f_{s l} P L_{c}+P f_{s l}\left(L-L_{c}\right)\)

 

Substituting \(Q_{f}=\frac{1}{2} \times 65 \times 3.14 \times 0.45 \times 9+3.14 \times 0.45 \times 65(15-9)\)

 

\(=413+551=964 kN\)

 

The failure load \(Q_{u}\) is

 

\(Q_{u}=Q_{b}+Q_{f}=138+964=1,102 kN\)

 

with \(F_{s}=2.5\),

 

\(Q_{a}=\frac{1102}{2.5}=440 kN\)
