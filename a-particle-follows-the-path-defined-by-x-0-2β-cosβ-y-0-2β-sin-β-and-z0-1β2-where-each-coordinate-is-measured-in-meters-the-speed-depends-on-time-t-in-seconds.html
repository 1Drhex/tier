Guest Question.

A particle follows the path defined by x = 0.2β cos(β), y = 0.2β sin (β), and \(z = 0.1β^{2}\) , where each coordinate is measured in meters. The speed depends on time t (in seconds) according to v = 20t m/s, and the particle was at the origin when t = 0. Determine the velocity and acceleration of the particle when t = 0.5 s. Also describe the path.

Admin
Solution

This problem is quite intricate. It will serve to illustrate the application of most of the relevant relations and also to highlight the thought process needed to implement these relations. We begin by observing that the path is specified in parametric form, with β as the parameter. Furthermore, the value of β for the position of interest is not specified. To determine this value we need to follow an inverse process, in which we use the given function v (t) to determine s at t = 0.5 s, then solve the relation for s in terms of β to determine the corresponding value of the parameter. After that, the remainder of the work reduces to straightforward calculations. The speed increases at a constant rate and s = 0 at t = 0, so we have

\( v = 20t =⇒ s = 10t^{2} , \dot{v} = 20.\)

Setting t = 0.5 s shows that the values at the position of interest are

\( s = 2.5 m, v = 10 m/s, \dot{v} = 20 m/s^{2}\) .

The derivatives of the coordinate functions are

\(\large\begin{array}{c} x^{\prime}=0.2[\cos (\beta)-\beta \sin (\beta)], \quad x^{\prime \prime}=0.2[-2 \sin (\beta)-\beta \cos (\beta)] , \\\\ y^{\prime}=0.2[\sin (\beta)+\beta \cos (\beta)], \quad y^{\prime \prime}=0.2[2 \cos (\beta)-\beta \sin (\beta)] ,\\\\ z^{\prime}=0.2 \beta, \quad z^{\prime \prime}=0.2 . \end{array}\)

The first derivatives are the components of \(\bar{r}^{\prime}\) , so

\(\large\begin{aligned} s^{\prime} &=\left[\left(x^{\prime}\right)^{2}+\left(y^{\prime}\right)^{2}+\left(z^{\prime}\right)^{2}\right]^{1 / 2} \\\\ &=0.2\left(2 \beta^{2}+1\right)^{1 / 2} .\end{aligned} \)

We find s as a function of β by integrating the preceding. Setting x = y = z = 0 for the starting position reveals that β = 0 there, so that is the lower limit for a definite integral. Thus,

\(\large s\left(\beta_{P}\right)=\int_{0}^{\beta_{P}} 0.2\left(2 \beta^{2}+1\right)^{1 / 2} d \beta=0.1 \beta\left(2 \beta_{P}^{2}+1\right)^{1 / 2}+0.05 \sqrt{2} \sinh ^{-1}\left(\sqrt{2} \beta_{P}\right)\) .

The root-finding function of our mathematical software can be used to find the value of \(β_{P}\) for which s = 2.5 m. To do so we need a starting value of\(β_{P}\), which we can obtain by graphing s for a range of\(β_{P}\). From the graph, we see that a good starting value is \(β_{P} = 4\). The root finder then yields \(β_{P} = 4.026\) . (It should be mentioned that if one encounters a situation for which it is not possible to integrate \(s^{\prime} dβ\) analytically, numerical integration can be used to determine s at a set of β values.)

The values of \( \bar{r}^{\prime} \) and \(\bar{r}^{\prime \prime}\) corresponding to this value of \(β_{P}\) are

\(\large \begin{array}{l} \bar{r}^{\prime}=x^{\prime} \bar{i}+y^{\prime} \bar{j}+z^{\prime} \bar{k}=0.4962 \bar{i}-0.6650 \bar{j}+0.8052 \bar{k} , \\\\ \bar{r}^{\prime \prime}=x^{\prime \prime} \bar{i}+y^{\prime \prime} \bar{j}+z^{\prime \prime} \bar{k}=0.8197 \bar{i}+0.3995\bar{j}+0.20 \bar{k} ,\end{array}\)

And \( s^{\prime}\left(\beta_{P}\right)=1.1489\) Substitution of these values into Eqs. (2.1.18), (2.1.29), and (2.1.28) yields

\( \Large\bar{e}_{t}=\frac{d \bar{r}}{d s}=\frac{d \alpha}{d s} \frac{d \bar{r}}{d \alpha} \equiv \frac{\bar{r}^{\prime}}{s^{\prime}}\) (2.1.18)

 

\(\Large \bar{e}_{n}=\frac{\bar{r}^{\prime \prime}\left(s^{\prime}\right)^{2}-\bar{r}^{\prime}\left(\bar{r}^{\prime} \cdot \bar{r}^{\prime \prime}\right)}{\left(s^{\prime}\right)\left[\left(\bar{r}^{\prime \prime} \cdot \bar{r}^{\prime \prime}\right)\left(s^{\prime}\right)^{2}-\left(\bar{r}^{\prime} \cdot \bar{r}^{\prime \prime}\right)^{2}\right]^{1 / 2}}\) (2.1.29)

 

\( \Large\rho=\frac{\left(s^{\prime}\right)^{3}}{\left[\left(\bar{r}^{\prime \prime} \cdot \bar{r}^{\prime \prime}\right)\left(s^{\prime}\right)^{2}-\left(\bar{r}^{\prime} \cdot \bar{r}^{\prime \prime}\right)^{2}\right]^{1 / 2}}\) (2.1.28)

 

\( \large\begin{array}{c} \bar{e}_{t}=0.4292 \bar{i}-0.5751 \bar{j}+0.6965 \bar{k}, \\\\ \bar{e}_{n}=0.7975 \bar{i}+0.6033 \bar{j}+0.0068 \bar{k}, \\\\ \rho=1.5226 \mathrm{~m} . \end{array}\)

The basic kinematical formulas, Eqs. (2.1.10), correspondingly give

\( \large\begin{array}{l} \bar{v}=v \bar{e}_{t}, \quad v=\dot{s} , \\\\ \bar{a}=\dot{v} \bar{e}_{t}+\frac{v^{2}}{\rho} \bar{e}_{n} .\end{array}\)

 

\(\large\begin{array}{c} \bar{v}=4.292 \bar{i}-5.751 \bar{j}+6.965 \bar{k} \mathrm{~m} / \mathrm{s}, \\\\ \bar{a}=60.96 \bar{i}+28.123 \bar{j}+14.38 \bar{k} \mathrm{~m} / \mathrm{s}^{2}. \end{array}\)

The last task is to describe the path. We observe that x and y are respectively proportional to cos(β) and sin (β), and recall that \( \left(x^{2}+y^{2}\right)^{1 / 2}\) is the distance to the z axis. This suggests that we consider polar coordinates, with the polar angle defined by tan θ = y/x = tan β, so we take θ = β. Thus the radial distance \(R =\left(x^{2}+y^{2}\right)^{1 / 2}= 0.2θ \) , which means that the projection of the path onto the xy plane is a linearly increasing spiral. The axial distance becomes \(z = 0.1θ^{2} = 2.5R^{2}\) , which tells us that the distance along the z axis increases as the square of the perpendicular distance from the z axis. In other words, the path seems to be a rapidly expanding helical-type path that wraps around the z axis.
