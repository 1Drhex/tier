Solution

\(O _{2} \Leftrightarrow 2 O\) ; From A.11: K(3200) = exp(-3.069) = 0.046467

Look at initially 1 mol Oxygen and shift reaction with x

\(n _{ O 2}=1- x ; \quad n _{ O }=2 x ; \quad n _{\text {tot }}=1+ x ; \quad y _{ i }= n _{ i } / n _{ tot }\)

 

\(K =\frac{ y _{ O }^{2}}{ y _{0_{2}}}\left(\frac{ P }{ P _{ o }}\right)^{2-1}=\frac{4 x ^{2}}{(1+ x )^{2}} \frac{1+ x }{1- x } 2=\frac{8 x ^{2}}{1- x ^{2}}\)

 

\(x ^{2}=\frac{ K / 8}{1+ K / 8} \quad \Rightarrow \quad x =0.07599 \text { ; }\)

 

\(y _{0_{2}}=\frac{1- x }{1+ x }= 0 . 8 5 9 ; \quad y _{0}=1- y _{0_{2}}= 0 . 1 4 1\)
