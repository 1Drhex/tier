Guest Question.

Prying Action

Determine the required angle size for the moment connection shown below. The bolts are 3⁄4-in.-diameter A490 in standard holes, and the steel is ASTM A36.

Admin
Solution

A trial size of L7 × 4 × 3⁄4 will be selected. The tension force due to the applied moment is found first:

\(T_{u}=\frac{M}{d}=\frac{(60)(12)}{17.9 in.}= 40.2\) kips.

Since there are two bolts to resist the tension, the force in each bolt is 20.1 kips.

Using the geometry from Figure 9-28, the minimum angle thickness can be found using the simplified approach (eq. (9-26)):

\(t_{min}=\sqrt{\frac{4.44Tb′}{PF_{u} } }\) (9-26)

a = 1.5 in.

b = 4 in. – 1.5 in. – (0.5)(0.75 in.) = 2.12 in.

\(b^′=b -\frac{d_{b} }{2}= 2.12 in – \frac{0.75 in.}{2} = 1.75\) in.

p = 4 in.

\(t_{min}=\sqrt{\frac{4.44Tb^′}{PF_{u} } }=\sqrt{\frac{(4.44)(20.1)(1.75)}{(4)(58)} }=0.82\) in.

The minimum angle thickness is found to be greater plan the thickness of the selected angle, but recall that this approach is con-sevative. We will also consider a less conservative approacal for comparison purposes.

Using the preliminary hanger selection tables (AISC, Table 15-1a), the required strength is

\(\frac{T_{u} }{P}= \frac{20.1}{4}=5.03\) kips/in.

With b = 2.12 in., we find that the required angle thickness is approximately 5⁄8 in. Based on this thickness, the angle is adequate. We will proceed with the in-depth analysis to compare the results.

From equation (9-28), the initial angle thickness is determined:

\(t=\sqrt{\frac{2.22Tb}{pF_{u} } }\) (9-28)

\(t=\sqrt{\frac{2.22Tb}{pF_{u} } }=\sqrt{\frac{(2.22)(20.1)(2.12)}{(4)(58)} }=0.64 \) in.

The minimum angle thickness, from equation (9-29), is determined as follows:

\(t_{min}=\sqrt{\frac{4.44Tb^′}{pF_{u}(1+\delta \alpha^′ )} }\) (9-29)

\(\acute{a} =\left(a+ \frac{d_{b} }{2} \right)\leq \left(1.25b+\frac{d_{b} }{2} \right)\)

 

\( =\left(1.5+ \frac{0.75 }{2} \right)\leq \left(\left(1.25\right) (2.12)+\frac{0.75 }{2} \right)\)

 

=1.88 in. < 3.03 in.

=1.88 in.

\(\rho =\frac{b^′ }{a^′}=\frac{1.75}{1.88} = 0.933\)

\(B=\phi r_{nt}=\phi F_{nt}A_{b}= (0.75)(113)(0.442)= 37.4\) kips

(agrees with AISCM, Table 7-2)

\(\beta =\frac{1}{\rho } \left(\frac{B}{T}-1 \right)= \frac{1}{0.933}\left(\frac{37.4}{20.1}-1 \right)=0.922\)

 

\(\delta =1-\frac{d^′}{p}= 1-\frac{13/16 in.}{4 in.}=0797\)

 

\(\alpha^′= \frac{1}{\delta }\left(\frac{\beta }{1-\beta } \right)\lt 1.0 for \beta \lt 1\)

 

\(\alpha^′= \frac{1}{0.797 }\left(\frac{0.922 }{1-0.922 } \right)\lt 1.0\)

 

\(\alpha^′=14.8\gt 1.0\)

 

\(\therefore\alpha^′=1.0\)

 

\(t_{min}=\sqrt{\frac{4.44Tb′}{pF_{u}(1+\delta \alpha ′) } } =\sqrt{\frac{(4.44)(20.1)(1.75)}{(4)(58)[1+(0.797)(1.0)]} }\)

 

= 0.612 in. < t = 0.64 in. OK

The L7 × 4 × 3⁄4 angle size is adequate.
