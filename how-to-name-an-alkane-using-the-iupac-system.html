How To Name an Alkane Using the IUPAC System

Admin
Solution

Step [1] Find the parent carbon chain and add the suffix.
• Find the longest continuous carbon chain, and name the molecule using the parent name for that number of carbons, given in Table 10.6. To the name of the parent, add the suffix -ane for an alkane. Each functional group has its own suffix.

• The longest chain may not be written horizontally across the page. Remember that it does not matter if the chain is straight or has bends. All of the following representations are equivalent.

Step [2] Number the atoms in the carbon chain to give the first substituent the lower number.

Step [3] Name and number the substituents.
• Name the substituents as alkyl groups, and use the numbers from step [2] to designate their location.

• Every carbon belongs to either the longest chain or a substituent, but not both.
• Each substituent needs its own number.
• If two or more identical substituents are bonded to the longest chain, use prefi xes to indicate how many: di- for two groups,tri- for three groups, tetra- for four groups, and so forth. The following compound has two methyl groups so its name contains the prefi x di- before methyl → dimethyl.

Step [4] Combine substituent names and numbers + parent + suffi x.
• Precede the name of the parent by the names of the substituents. Alphabetize the names of the substituents, ignoring any prefi xes like di-. For example, triethyl precedes dimethyl because the e of ethyl comes before the m of methylin the alphabet.
• Precede the name of each substituent by the number that indicates its location. There must be one number for each substituent.
• Separate numbers by commas and separate numbers from letters by dashes.
