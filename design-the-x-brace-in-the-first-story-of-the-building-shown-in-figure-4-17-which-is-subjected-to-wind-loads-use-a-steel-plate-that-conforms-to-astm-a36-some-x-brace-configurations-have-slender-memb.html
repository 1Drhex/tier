Guest Question.

Design the X-brace in the first story of the building shown in Figure 4-17, which is subjected to wind loads. Use a steel plate that conforms to ASTM A36. Some X-brace configurations have slender members such that they can only support loads in tension. In this figure, all of the members are assumed to be too slender to support compression loads. Only the shaded members support lateral loads in the assigned direction of the lateral loads.

The lateral loads shown below are the loads acting on each X-braced frame. The wind loads acting on the entire building must be distributed to the various braced frames in the building in the direction of the lateral load. If the diaphragm is assumed to be rigid, the lateral load is distributed in proportion to the stiffness of each braced frame. If the diaphragm is flexible, the lateral load is distributed in proportion to the tributary area of each braced frame.

Admin
Solution

The maximum load factor for wind is \(1.6\) (from the ASCE 7 load combinations).

Loads to Each Level:

Service Loads:\(\quad\quad\quad\) Factored Loads:

\(P_{r}=11.4 \text { kips } \quad \quad\quad (1.6)(11.4)=18.2 \mathrm{kips}.\).

\(P_{3}=17.9 \text { kips } \quad\quad\quad (1.6)(17.9)=28.6 \text { kips }.\).

\(P_{2}=23.6 \text { kips } \quad\quad\quad (1.6)(23.6)=37.7 \mathrm{kips}.\).

 

\(\begin{array}{l} V_{u} \text { (base shear) }=18.2+28.6+37.8=84.6 \text { kips } \\ \theta=\tan ^{-1}\left(\frac{12 \mathrm{ft} .}{32 \mathrm{ft}}\right)=20.6^{\circ} \\ T_{A B}=\frac{V_{u}}{\cos \theta}=\frac{84.6}{\cos 20.6}=90.4 \mathrm{kips} \end{array}\)

 

We will cover bolts and welds in Chapters 9 and 10, but for now, assume the shear strength of \(A325\) \(N\) bolts in single shear to be as follows:

\(ΦR_{n} = 15.9 kips 3/4-in\).-diameter bolt

\(ΦR_{n} =21.6 kips 7/8-in\).-diameter bolt

  1. \(A_{g} \geq \frac{P_{u}}{0.9 F_{y}}\) (allow for added \(5\)% to \(10\)% pretension \(\therefore\) use \(7.5\)% pretension)

 

\(A_{g} \geq \frac{(90.4)(1.075)}{0.9(36)}=2.99 \mathrm{in.}^{2}\)

 

\(\text { Try } 1 / 2 \text { -in. } \times 6 \text { -in. plate, } A_{g}=(0.5)(6)=3.0 \text { in. }^{2}\)

 

2. Number of bolts:

Five \(3⁄4-in\).-diameter bolt \(\rightarrow \quad \phi R_{n}=(5)(15.9)=79 \text { kips }<90.4 \text { kips }\)

Five \(7⁄8-in\).-diameter bolt \(\rightarrow \quad\phi R_{n}=(5)(21.6)=108 \mathrm{kips}>90.4 \text { kips }\)

Five \(3⁄4-in\).-diameter bolt \(\rightarrow \quad \phi R_{n}=(6)(15.9)=95 \mathrm{kips}>90.4 \mathrm{kips}\)

Use five \(7⁄8-in\). diameter bolts in a single line.

The shear lag factor, \(U\), is \(1.0\) for plates connected with bolts (from Table 4-1).

\(A_{g} \geq \frac{P_{u}}{0.75 F_{u} U}+\Sigma A_{\text {holes }}\)

 

\(A_{g} \geq \frac{(90.4)(1.075)}{(0.75)(58)(1.0)}+(1 \text { hole })\left(\frac{7}{8}+\frac{1}{8}\right)(0.5)=2.73 \mathrm{in} .^{2}<3.0 \mathrm{in} .^{2} \mathrm{OK}\)

 

3. From step 1, \(A_{g} =2.99 in.^{2}\) and from step 2, \(A_{g} =2.73 in.^{2}\); both are less than the gross area of the trial member size for \(3.0 in.^{2}.\)

 

4. Check slenderness ratio:

\(\begin{aligned} L &=\sqrt{(12)^{2}+(32)^{2}}=34.2 \mathrm{ft} \\ I_{\min } &=\frac{b h^{3}}{12}=\frac{(6)(0.5)^{3}}{12}=0.0625 \mathrm{in}^{2} \\ r_{\min } &=\sqrt{\frac{I_{\min }}{A_{g}}}>\frac{L}{300} \\ r_{\min } &=\sqrt{\frac{0.0625}{3}}>\frac{(34.2)(12)}{300} ; 0.144<1.37 \rightarrow \text { Slenderness limit is exceeded } \end{aligned}\)

 

The assumption to pretension the X-brace is justified.

5. Check block shear strength.

The bolt spacing and configuration shown in Figure 4-18 will be used.

\(\begin{aligned} A_{g v} &=(13.5)(0.5)=6.75 \mathrm{in} .^{2} \\ A_{n v} &=A_{g v}-\Sigma A_{\mathrm{holes}} \\ &=6.75-(4.5 \text { holes })\left(\frac{7}{8}+\frac{1}{8}\right)(0.5)=4.5 \mathrm{in} .^{2} \\ A_{g t} &=(3)(0.5)=1.5 \mathrm{in} .^{2} \\ A_{n t} &=A_{g t}-\Sigma A_{\mathrm{holes}} \\ &=1.5-(0.5 \text { holes })\left(\frac{7}{8}+\frac{1}{8}\right)(0.5)=1.25 \mathrm{in.}^{2} \end{aligned}\)

 

\(\begin{aligned} \phi P_{n} &=\phi\left(0.60 F_{u} A_{n v}+U_{b s} F_{u} A_{n t}\right) \leq \phi\left(0.60 F_{y} A_{g v}+U_{b s} F_{u} A_{n t}\right) \\ \phi P_{n} &=0.75[(0.60)(58)(4.5)+(1.0)(58)(1.25)] \\ & \leq 0.75[(0.60)(36)(6.75)+(1.0)(58)(1.25)] \\ & 171 \text { kips }>163 \text { kips } \end{aligned}\)

 

\(\phi P_{n}= 163 kips\) (block shear capacity) \(> T_{u}= 90.4 kips \quad OK\)

Use a \(6-in\). by \(1⁄2-in\). plate with five \(7⁄8-in\).-diameter \(A325\) \(N\) bolts.
