Guest Question.

The “bouncing ball” revisited. Consider the quantum mechanical analog to the classical problem of a ball (mass m) bouncing elastically on the floor.

(a) What is the potential energy, as a function of height x above the floor? (For negative x, the potential is infinite—the ball can’t get there at all.)

(b) Solve the Schrödinger equation for this potential, expressing your answer in terms of the appropriate Airy function (note that \( Bi_z \) blows up for large z, and must therefore be rejected). Don’t bother to normalize ψ(x).

(c) Using \( g=9.80 m / s ^{2} \text { and } m=0.100 \) kg, find the first four allowed energies, in joules, correct to three significant digits. Hint: See Milton Abramowitz and Irene A. Stegun, Handbook of Mathematical Functions, Dover, New York (1970), page 478; the notation is defined on page 450.

(d) What is the ground state energy, in eV, of an electron in this gravitational field? How high off the ground is this electron, on the average? Hint: Use the virial theorem to determine 〈 x 〉.

Admin
Solution

(a) \( V(x)=m g x \).

(b)

\( -\frac{\hbar^{2}}{2 m} \frac{d^{2} \psi}{d x^{2}}+m g x \psi=E \psi \Longrightarrow \frac{d^{2} \psi}{d x^{2}}=\frac{2 m^{2} g}{\hbar^{2}}\left(x-\frac{E}{m g}\right) . \text { Let } y \equiv x-\frac{E}{m g}, \text { and } \alpha \equiv\left(\frac{2 m^{2} g}{\hbar^{2}}\right)^{1 / 3} \).

Then \( \frac{d^{2} \psi}{d y^{2}}=\alpha^{3} y \psi . \text { Let } z \equiv \alpha y=\alpha\left(x-\frac{E}{m g}\right), \text { so } \frac{d^{2} \psi}{d z^{2}}=z \psi \). This is the Airy equation (Eq. 9.37), and the general solution is \( \psi=a \operatorname{Ai}(z)+b B i(z) \). However, Bi(z) blows up for large z, so b = 0 (to make ψ normalizable). Hence \( \psi(x)=a A i\left[\alpha\left(x-\frac{E}{m g}\right)\right] \).

\( \frac{d^{2} \psi_{p}}{d z^{2}}=z \psi_{p} \) (9.37).

(c) Since \( V(x)=\infty \text { for } x<0, \text { we require } \psi(0)=0 \); hence \( A i[\alpha(-E / m g)]=0 \). Now, the zeros of Ai are \( a_{n}(n=1,2,3, \ldots) \). Abramowitz and Stegun list \( a_{1}=-2.338, a_{2}=-4.088, a_{3}=-5.521, a_{4}=-6.787 \), etc . Here \( -\frac{\alpha E_{n}}{m g}=a_{n}, \text { or } E_{n}=-\frac{m g}{\alpha} a_{n}=-m g\left(\frac{\hbar^{2}}{2 m^{2} g}\right)^{1 / 3} \), \( a_{n}, \text { or } E_{n}=-\left(\frac{1}{2} m g^{2} \hbar^{2}\right)^{1 / 3} a_{n} \) In this case \( \frac{1}{2} m g^{2} \hbar^{2}=\frac{1}{2}(0.1 kg )\left(9.8 m / s ^{2}\right)^{2}\left(1.055 \times 10^{-34} J \cdot s \right)^{2}=5.34 \times 10^{-68} J ^{3} ; \quad\left(\frac{1}{2} m g^{2} \hbar^{2}\right)^{1 / 3}=3.77 \times 10^{-23} J \).

\( E_{1}=8.81 \times 10^{-23} J , \quad E_{2}=1.54 \times 10^{-22} J , \quad E_{3}=2.08 \times 10^{-22} J , \quad E_{4}=2.56 \times 10^{-22} J . \)

(d)

\( 2\langle T\rangle=\left\langle x \frac{d V}{d X}\right\rangle \text { (Eq. 3.113); here } \frac{d V}{d x}=m g, \quad \text { so } \quad\left\langle x \frac{d V}{d x}\right\rangle=\langle m g x\rangle=\langle V\rangle, \quad \text { so } \quad\langle T\rangle=\frac{1}{2}\langle V\rangle \).

\( 2\langle T\rangle=\left\langle x \frac{d V}{d x}\right\rangle \) (3.113).

\( \text { But }\langle T\rangle+\langle V\rangle=\langle H\rangle=E_{n}, \quad \text { so } \quad \frac{3}{2}\langle V\rangle=E_{n}, \quad \text { or } \quad\langle V\rangle=\frac{2}{3} E_{n} . \quad \text { But }\langle V\rangle=m g\langle x\rangle, \quad \text { so }\langle x\rangle=\frac{2 E_{n}}{3 m g} \).

For the electron; \( \left(\frac{1}{2} m g^{2} \hbar^{2}\right)^{1 / 3}=\left[\frac{1}{2}\left(9.11 \times 10^{-31}\right)(9.8)^{2}\left(1.055 \times 10^{-34}\right)^{2}\right]^{1 / 3}=7.87 \times 10^{-33} J \).

\( E_{1}=1.84 \times 10^{-32} J =1.15 \times 10^{-13} eV \).

\( \langle x\rangle=\frac{2\left(1.84 \times 10^{-32}\right)}{3\left(9.11 \times 10^{-31}\right)(9.8)}=1.37 \times 10^{-3}=1.37 mm \).
