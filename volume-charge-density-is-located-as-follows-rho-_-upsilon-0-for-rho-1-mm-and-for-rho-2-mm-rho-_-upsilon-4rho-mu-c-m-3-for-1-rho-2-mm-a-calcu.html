Guest Question.

Volume charge density is located as follows : \({ \rho }_{ \upsilon }\) = 0 for \(\rho\) < 1 mm and for \(\rho\) > 2 mm, \({ \rho }_{ \upsilon } = 4\rho \mu C/{ m }^{ 3 }\) for 1 < \(\rho\) < 2 mm. (a) Calculate the total charge in the region 0 < \(\rho < { \rho }_{ 1 }\), 0 < z < L, where 1 < \({ \rho }_{ 1 }\) < 2 mm. (b) Use Gauss’s law to determine \({ D }_{ \rho }\) at \(\rho = { \rho }_{ 1 }\) . (c) Evaluate \({ D }_{ \rho }\) at \(\rho \) = 0.8 mm, 1.6 mm, and 2.4 mm.

Admin
Solution

a) We find
Q = \(\int _{ 0 }^{ L } \int _{ 0 }^{ 2\pi } \int _{ .001 }^{ { \rho }_{ 1 } }{ 4\rho d\rho d\phi dz=\frac { 8\pi L }{ 3 } \left[ { { \rho }_{ 1 } }^{ 3 }-{ 10 }^{ -9 } \right] \mu C }\)
where \({ \rho }_{ 1 }\) is in meters
b) Gauss’s Law states that \(2\pi { \rho }_{ 1 }{ LD }_{ \rho }\) = Q, where Q is the result of part a . Thus
\({ D }_{ \rho }\left( { \rho }_{ 1 } \right)\) = \(\frac { 4\left( { { \rho }_{ 1 } }^{ 3 }-{ 10 }^{ -9 } \right) }{ 3{ \rho }_{ 1 } } \mu C/{ m }^{ 2 }\)
where \({ \rho }_{ 1 }\) is in meters.
c) At \(\rho\) = 0.8 mm, no charge is enclosed by a cylindrical gaussian surface of that radius, so \({ D }_{ \rho }( 0.8mm)\) = 0. At \(\rho\) = 1.6 mm, we evaluate the part b result at \({ \rho }_{ 1 }\) = 1.6 to obtain :
\({ D }_{ \rho }( 1.6 mm)\) = \(\frac { 4\left[ { (.0016) }^{ 3 }-({ .0010) }^{ 3 } \right] }{ 3(.0016) }\) = \(3.6 \times { 10 }^{ -6 } \mu C/{ m }^{ 2 }\)
AT \(\rho\) = 2.4, we evaluate the charge integral of part a from .001 to.002, and gauss’ law is written as
\(2\pi { \rho }{ LD }_{ \rho }\) = \(\frac { 8\pi L }{ 3 } \left[ { (.002) }^{ 2 }-{ (.001) }^{ 2 } \right] \mu C\)
from which \({ D }_{ \rho }( 2.4 mm)\) = \(3.9 \times { 10 }^{ -6 } \mu C/{ m }^{ 2 }\).
