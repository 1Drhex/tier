Guest Question.

Let us consider a concrete model of phase coexistence, defined here for a liquid solution containing two substances at a given pressure. Let NA be the number of moles of substance A and NB the number of moles of substance B. The concentration of substance A is defined as \(c = N_A/ (N_A + N_B)\) where 0 ≤ c ≤ 1. The Gibbs free energy of this model is given by,

\(G(T,N_A,N_B) = N_A RT\ln \Bigl(\frac{N_A}{N_A+N_B} \Bigr)+N_B R\ln \Bigl(\frac{N_B}{N_A+N_B} \Bigr)+\frac{N_AN_B}{N_A+N_B}ΔU\).

where ΔU > 0 is an energy term representing the interaction between the substances. The first two terms will be derived in the framework of Chapter 8 [63]. The global condition of stability requires that the Gibbs free energy of the system \(G(T,N_A,N_B)\) is a convex function of the extensive variables \(N_A\) and \(N_B\).

1. Analyse the behaviour of the dimensionless function \(g (β, c) = G(T,N_A,N_B) / (RT(N_A + N_B))\) with respect to the dimensionless parameters c and β = ΔU/RT > 0.
2. Draw a graph of the function g (β, c) where \(2 < β ≤ 4\ln (2)\) is constant.
3. Show that if \(2 < β ≤ 4 \ln (2)\), there is a domain of concentrations c in which the system splits into two phases. Determine the proportions \(r_1\) and \(r_2\) of phases 1 and 2 as
a function of the concentration c, in terms of the concentrations \(c_0\) and \(1 − c_0\) at the minima of the function g (β, c).

Admin
Solution

1. The dimensionless function g (β, c) writes,

\(g (β, c) = c \ln (c) + (1 − c) \ln (1 − c) + β c (1 − c)\).

where the concentration 0 ≤ c ≤ 1. The function g (β, c) is symmetric with respect to c = 1/2 ∀ β. This can be verified analytically by showing that the function g (β, c) is invariant under the exchange of c and 1− c. The function g (β, c) vanishes when c → 0 and c → 1,

\(\lim _{c\longrightarrow 0}g(\beta ,c)=\lim _{c\longrightarrow 0}\ln \Bigl(c^c(1-c)^{1-c}\Bigr)=0\).

\(\lim _{c\longrightarrow 1}g(\beta ,c)=\lim _{c\longrightarrow 1}\ln \Bigl(c^c(1-c)^{1-c}\Bigr)=0\).

The first derivative of function g (β, c) with respect to c is given by,

\(\frac{dg}{dc} (β, c) = \ln(c) − \ln (1 − c) + β (1 − 2 c)\).

This implies that,

\(\lim _{c\longrightarrow 1/2}\frac{dg}{dc} (\beta ,c)=0\).

Therefore, function g (β, c) has an extremum if c = 1/2 ∀ β. The second derivative of
function g (β, c) with respect to c is given by,

\(\frac{d^2g}{dc^2} (\beta ,c)=\frac{1}{c} +\frac{1}{1-c} -2\beta\).

which implies that,

\(\lim _{c\longrightarrow 1/2}\frac{d^2g}{dc^2} (\beta ,c)=2 (2 − β)\).

Therefore, if 0 < β < 2, the point g (β, 1/2) is a minimum and if β > 2, the point
g (β, 1/2) is a maximum. Furthermore,

\(\lim _{c\longrightarrow 1/2}g(β, c)=\ln \Bigl(\frac{1}{2} \Bigr)+\frac{\beta }{4} \).

Thus, if \(β = 4 \ln (2)\), then g(β, 1/2) = 0. Therefore, if \(2 < β ≤ 4 \ln (2)\), then function g (β, c) ≤ 0 ∀ 0 ≤ c ≤ 1. Furthermore, this function has a maximum at c = 1/2. So, there has to be two minima, located symmetrically with respect to c = 1/2 (Fig 6.14).
We will label the concentrations of these minima \(c_0\) and \(1 − c_0\) where \(c_0 < 1/2\). These minima are function of β. The locus of all the minima, obtained by varying β, constitutes the saturation curve.

2. The graph of g (β, c) for 0 ≤ c ≤ 1 is drawn for β = 2.5 in Fig. 6.14.

3. The global condition of stability for the Gibbs free energy requires that g (β, c) is a convex function of c. At a concentration \(c ≤ c_0\) where \(c ≥ 1 − c_0\), the model is stable. In this case, the solution is made up of one phase which contains both substances A and B. At a concentration \(c_0 < c < 1 − c_0\), the model is unstable. This instability manifests itself by the fact that the actual minimum is obtained for a solution consisting of two phases, each containing both substances A and B. We speak of phase segregation to express the fact that they are located in different spatial regions. At concentrations \(c ≤ c_0\) only phase 1 exists, containing solute A in solvent B, and at concentrations \(c ≥ 1 − c_0\) only phase 2 exists, containing solute B in solvent A. At concentrations \(c_0 < c < 1 − c_0\), the proportion \(r_1\) of phase 1 and the proportion \(r_2 = 1 − r_1\) of phase 2 vary linearly between 0 and 1 as c varies between the concentrations at both minima, i.e.

\(r_1=\frac{(1-c_0)-c}{1-2c_0}\) and \(r_2=\frac{c-c_0}{1-2c_0}\).

This linear variation is known as the lever rule.
