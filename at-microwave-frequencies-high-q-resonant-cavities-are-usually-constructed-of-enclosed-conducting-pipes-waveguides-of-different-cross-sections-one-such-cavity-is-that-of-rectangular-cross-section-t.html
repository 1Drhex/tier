Guest Question.

At microwave frequencies, high Q resonant cavities are usually constructed of enclosed conducting pipes (waveguides) of different cross sections. One such cavity is that of rectangular cross section that is enclosed on all six sides, as shown in Figure P1-46. One set of complex fields that can exist inside such a source-free cavity filled with free space is given by

\(\textbf{E} = \hat{\textbf{a}}_y E_0 \sin (\frac \pi a x) \sin (\frac \pi c z)\)

such that

\(\omega = \omega_r = \frac 1 { \sqrt{ \mu_0 \varepsilon_0 } } \sqrt { (\frac \pi a)^2 + (\frac \pi c)^2 }\)

where \({ E }_{ 0 }\) is a constant and \({ \omega }_{ r }\) is referred to as the resonant radian frequency. Within the cavity, determine the:

(a) Corresponding magnetic field.

(b) Supplied complex power.

(c) Dissipated real power.

(d) Time-average magnetic energy.

(e) Time-average electric energy.

Ultimately verify that the conservation of energy equation in integral form is satisfied for this set of fields inside this resonant cavity.

Admin
Solution
\(\underline{E} = \hat{a}_y E_0 \sin (\frac \pi a x) \sin (\frac \pi c z), \quad \omega_r = \omega = \frac 1 { \sqrt { \mu_0 \varepsilon_0 } } \sqrt{ (\frac \pi a)^2 + (\frac \pi c)^2 }\)

a.

\(\\ \underline{H} = -\frac 1 { j\omega \mu_0 } \nabla \times \underline{E} = -\frac 1 { j\omega \mu_0 } [-\hat{a}_x \frac { ə E_y } { ət } + \hat{a}_z \frac { ə E_y } { əx }] \\ = \hat{a}_x \frac { E_0 } { j\omega \mu_0 } (\frac \pi c) \sin (\frac \pi a x) \cos (\frac \pi c z) – \hat{a}_z \frac { E_0 } { j\omega \mu_0 } (\frac \pi a) \cos (\frac \pi a x) \sin (\frac \pi c z)\)

b.

\(\\ P_s = -\frac 1 2 \underset {V} { \int\int\int } [\underline{H}^{ * } \cdot \underline{M}_i + \underline{E} \cdot \underline{J}^*_i] dv = 0\)

c.

\(P_e = \underset { S_a } { \oiint } (\frac 1 2 \underline{E} \times \underline{H}^*) \cdot d\underline{s} = \underset { S_a } { \oiint } \underline{ S } \cdot d \underline{ s }, \quad\quad \underline{ S } = \frac 1 2 \underline{ E } \times \underline { H }^* \\ \underline{ S } = \frac 1 2 \underline{E} \times \underline{H}^* = \frac 1 2 \hat{a}_y E_y \times (\hat{a}_x H_x^* – \hat{a}_z H_z^*) = \frac 1 2 (-\hat{a}_z E_y H_x^* – \hat{a}_x E_y H_z^*) \\ \frac 1 2 [\hat{a}_x \frac { E_0 } { j\omega \mu_0 } (\frac \pi a) \sin (\frac \pi a x) \cos (\frac \pi a x) \sin^2 (\frac \pi c z) + \hat{a}_z \frac { |E_0|^2 } { j\omega\mu_0 } (\frac \pi c) \sin^2 (\frac \pi a x) \sin (\frac \pi c z) \omega s (\frac \pi z z)]\)

Contributions to \(P_e\) from the different walls:

Left and right walls:

\(\int_0^c \int_o^b (\hat{a}_x S_x)_{ x = 0 } \cdot (-\hat{a}_x dydz) = 0; \quad\quad \int_0^c \int_0^b (\hat{a}_x S_x)_{ x = 0 } \cdot (\hat{a}_x dydz) = 0\)

Back and front walls:

\(\int_0^b \int_0^a (\hat{a}_z S_z) \cdot (-\hat{a}_z dxdy) = 0_j \quad\quad \int_0^b \int_0^a (\hat{a}_z S_z)_{ z = 0 } \cdot (\hat{a}_z dxdy) = 0\)

Top and bottom walls:

Since there are no y components of the power density, there are no contributions from the top and bottom walls.

Therefore \(P_e = 0 + 0 + 0 + 0 + 0 = 0\)

d. \(P_d = \frac 1 2 \underset { V } { \int\int\int } \sigma |\underline { E }|^2 dv = 0\)

e.

\(\overline{ W }_m = \frac { \mu_0 } 4 \underset { V } { \int\int\int } |\underline H|^2 dv = |E_0|^2 \frac { \mu_0 } 4 \frac 1 { (\omega \mu_0)^2 } [(\frac \pi c)^2 \int_0^c \int_0^b \int_0^a \sin^2 (\frac \pi a x) \cos^2 (\frac \pi c z) dxdyz] \\ + (\frac \pi a)^2 \int_0^c \int_0^b \int_0^a \cos^2 (\frac \pi a x) \sin^2 (\frac \pi c z) dxdydz] \\ \overline{W}_m = |E_0|^2 \frac { \mu_0 } 4 \frac 1 { (\omega \mu_0)^2 } [(\frac \pi c)^2 \frac { abc } 4 + (\frac { \pi } a)^2 \frac { abc } 4] = |E_0|^2 \frac { abc } { 16 } \frac { \varepsilon_0 } { \omega^2 \mu_0 \varepsilon_0 } [(\frac \pi a)^2 + (\frac \pi c)^2] = |E_0|^2 \frac { abc } { 16 } \varepsilon_0\)

f.

\(\overline{W}_e = \frac { \varepsilon_0 } 4 \underset { V } { \int\int\int } |\underline{E}|^2 dv = |E_0|^2 \frac { \varepsilon_0 } 4 \int_0^c \int_0^b \int_0^a \sin^2 (\frac \pi a x) \sin^2 (\frac \pi c z) dxdyxz = |E_0|^2 \frac { \varepsilon_0 } 4 \frac { abc } 4 = |E_0|^2 \frac { abc } { 16 } \varepsilon_0\)

Utimately

\(P_s = P_e + P_d + j2\omega (\overline{W}_m – \overline{W}_e) \\ 0 = 0 + 0 + j2\omega (|E_0|^2 \frac { abc } { 16 } \varepsilon_0 – |E_\varepsilon|^2 \frac { abc } { 16 } \varepsilon_0) \\ 0 = 0\)
