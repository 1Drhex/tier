Guest Question.

The discrete-time systems S1 and S2 are described in the following figures.

a. Calculate the discrete impulse response of the first system.

b. Compute the coefficients K1, K2, K3 and K4 of the second system, so as to have the same impulse response as the first.
c. Give other forms of realization and compare each form with the others.
d. Are the systems stable?

Admin
Solution

a. For S1 (implemented in series), the total transfer function is:

\(H(z)=H_{1}(z)H_{2}(z)H_{3}(z)=\frac{1-z^{-1}}{1-\frac{1}{2}z^{-1} }\frac{1-z^{-1}}{1-\frac{1}{8}z^{-1}}\frac{1-z^{-1}}{1}=\frac{(1-z^{-1})^{3}}{(1-\frac{1}{2}z^{-1} )(1-\frac{1}{8}z^{-1} )}\) (1)
\(\frac{H(z)}{z}\overset{(1)}{=}\frac{(z-1)^{3}}{z^{2}(z-\frac{1}{2} )(z-\frac{1}{8} )}=\frac{A}{z^{2}}+\frac{B}{z}+\frac{C}{z-\frac{1}{2} }+\frac{D}{z-\frac{1}{8} }\) (2)
\(A=\underset{z\rightarrow 0}{\lim }\frac{(z-1)^{3}}{(z-\frac{1}{2} )(z-\frac{1}{8} )}=-16, B=\underset{z\rightarrow 0}{\lim }\frac{d}{dz}(\frac{(z-1)^{3}}{(z-\frac{1}{2} )(z-\frac{1}{8} )} )=-112 \)
\(C=\underset{z\rightarrow \frac{1}{2} }{\lim }\frac{(z-1)^{3}}{z^{2}(z-\frac{1}{8} )}=-\frac{4}{3},D=\underset{z\rightarrow \frac{1}{8} }{\lim }(\frac{(z-1)^{3}}{z^{2}(z-\frac{1}{2} )})=\frac{343}{3}\)

Thus,

\(H(z)\overset{(2)}{=}-\frac{16}{z}-112-\frac{\frac{4}{3} z}{z-\frac{1}{2} }+\frac{\frac{343}{3}z }{z-\frac{1}{8} } \) (3)

The discrete impulse response of the first system is given by relation (4).

\(h[n]=IZT[H(z)]\overset{(3)}{=}-112\delta [n]-16\delta [n-1]-\frac{4}{3}(\frac{1}{2} )^{n}u[n]+\frac{343}{3}(\frac{1}{8} )^{n}u[n]\) (4)

b. For S2 (implemented in parallel), we can see that:

\(\frac{Y(z)}{X(z)}=k_{1}+k_{2}z^{-1}+k_{3}\frac{1}{1-\frac{1}{2} z^{-1}} +k_{4}\frac{1}{1-\frac{1}{8} z^{-1}}\) (5)

Thus \(k_{1} = –112, k_{2} = –16, k_{3} = –4/3, k_{3} = –343/3\)

c. S1: Implementation in direct form II (Canonical form)

\(H(z)=\frac{1-3z^{-1}+3z^{-2}-z^{-3}}{1-\frac{5}{8}z^{-1}+\frac{1}{16}z^{-2} }=\frac{Y(z)}{X(z)}\) (6)

Based on the relation (6), the implementation of the discrete system in canonical form is presented in the following figure:

\((6)\Rightarrow (1-\frac{5}{8}z^{-1}+\frac{1}{16} z^{-2} )Y(z)=(1-3z^{-1}+3z^{-2}-z^{-3})X(z)\Rightarrow\)
\(y[n]=x[n]-3x[n-1]+3x[n-2]-x[n-3]+\frac{5}{8}y[n-1]-\frac{1}{16}y[n-2]\) (7)
