Guest Question.

Use Manual Tables 10-8 and 10-14 to design a stiffened seated connection for a W21×68 beam to a HSS12×12×1/2 column.
\(\begin{array}{l}P_{D}=20 \text { kips } \\P_{L}=60 \text { kips }\end{array}\)
Use 3/4-in. diameter ASTM A325-N bolts in standard holes to connect the beam to the seat plate.
Use 70 ksi electrode welds to connect the stiffener, seat plate and top angle to the HSS.

 

Admin
Solution

Material Properties:

Beam ASTM A992 \(F_{y}=50 ksi \quad F_{u}=65 ksi\)

Column ASTM A500 Gr. B \(F_{y}=46 ksi \quad F_{u}=58 ksi\)

Angles and Plates ASTM A36 \(F_{y}=36 \text { ksi } \quad F_{u}=58 ksi\)

Manual
Table 2-3

Geometric Properties:

Beam W21×68 \(t_{w}=0.430 \text { in. } \quad d=21.1 \text { in. } \quad k=1.19 \text { in } .\)

Column HSS12×12×1/2 t = 0.465 in. B = 12.0 in.

Manual
Tables 1-1
and 1-12

Check limits of applicability for Specification Section K1

Strength: \(F_{y}=46 ksi <52 ksi\) o.k.

Section K1.2

Ductility: \(\frac{F_{y}}{F_{u}}=\frac{46 ksi }{58 ksi }=0.79<0.80\) o.k.

Calculate the required strength

LRFD ASD
\(\begin{aligned}P_{u} &=1.2(20.0 kips )+1.6(60.0 kips ) \\&=120 kips\end{aligned}\) \(\begin{aligned}P_{a} &=20.0 \text { kips }+60.0 \text { kips } \\&=80.0 \text { kips }\end{aligned}\)

Determine stiffener width W required for web crippling and local web yielding

For web crippling, assume N/d > 0.2 and use constants \(R_{5} \text { and } R_{6}\) from Manual Table 9-4. Assume a 3/4-in. setback.

Manual
Section 10

LRFD ASD
\(\begin{aligned}W_{\min } &=\frac{R_{u}-\phi R_{5}}{\phi R_{6}}+\text { setback } \geq k+\text { setback } \\&=\frac{120 \text { kips }-75.9 \text { kips }}{7.94 kips / in .}+3 / 4 in. \\&=6.31 in .\end{aligned}\) \(\begin{aligned}W_{\min } &=\frac{R_{a}-R_{5} / \Omega}{R_{6} / \Omega}+\text { setback } \geq k+\text { setback } \\&=\frac{80.0 kips -50.6 kips }{5.29 kips / in .}+3 / 4 in. \\&=6.34 in .\end{aligned}\)

Manual
Table 9-4

For local web yielding, use constants \(R_{l} \text { and } R_{2}\) from Manual Table 9-4. Assume a 3/4-in. setback.

LRFD ASD
\(\begin{aligned}W_{\min } &=\frac{R_{u}-\phi R_{1}}{\phi R_{2}}+\text { setback } \\&=\frac{126 \text { kips }-63.7 \text { kips }}{21.5 kips / in .}+3 / 4 \\&=3.65 in .\end{aligned}\) \(\begin{aligned}W_{\min } &=\frac{R_{a}-R_{1} / \Omega}{R_{2} / \Omega}+\text { setback } \\&=\frac{84.0 kips -42.5 kips }{14.3 kips / in. }+3 / 4 in . \\&=3.65 in .\end{aligned}\)

Manual
Table 9-4

The minimum stiffener width, W, for web crippling controls. Use W = 7.00 in.

Check bearing width assumption

\(N=7.00 \text { in. }-3 / 4 \text { in. }=61 / 4 \text { in. }\)

 

\(N / d=\frac{61 / 4 \text { in. }}{21.1 \text { in. }}=0.30>0.2 \text { , as assumed. }\)

Determine the weld strength requirements for the seat plate

Try L = 24 in. with 5/16 in. fillet welds. Enter Manual Table 10-8 using W = 7 in. as determined above.

LRFD ASD
\(\phi R_{n}=293 \text { kips }>126 \text { kips }\) o.k. \(R_{n} / \Omega=195 \text { kips }>84.0 \text { kips }\) o.k.

Manual
Table 10-8

Since t of the HSS is greater than \(t_{\min }\) for the 5/16-in. weld, no reduction in the weld strength to account for shear in the HSS is required.

The minimum length of the seat-plate-to-HSS weld on each side of the stiffener is 0.2L = 4.8 in. This establishes the minimum weld between the seat plate and stiffener; use 5 in. of 5/16-in. weld on each side of the stiffener.

Manual
Section 10

Determine the stiffener plate thickness

To develop the stiffener-to-seat plate welds, the minimum stiffener thickness is

\(t_{p \min }=2 w=2(5 / 16 \text { in. })=5 / 8 \text { in. }\)

 

For a stiffener with \(F_{y}=36\) ksi and a beam with \(F_{y}=50\) ksi, the minimum stiffener thickness,

 

\(t_{p \min }=\left(\frac{F_{y \text { beam }}}{F_{y \text { stiffener }}}\right) t_{w}=\left(\frac{50 ksi }{36 ksi }\right)(0.430 in. )=0.597 \text { in. }\)

For a stiffener with \(F_{y}=36\) ksi and a column with \(F_{u}=58\) ksi, the maximum stiffener thickness is

 

\(t_{p \max }=\frac{F_{u} t}{F_{y p}}=\frac{58 ksi (0.465 in .)}{36 ksi }=0.749 in\)

Eqn. K1-10

Use stiffener thickness of 5/8 in.

Determine the stiffener length using Manual Table 10-14

LRFD ASD
\(\begin{aligned}\left(\frac{R_{u} W}{t^{2}}\right)_{r e q} &=\frac{120 kips (7.00 in .)}{(0.465 in .)^{2}} \\&=4100 kips / in .\end{aligned}\)

 

To satisfy the minimum above, select astiffener L = 24 in. from Table 10-14

 

\(\frac{R_{u} W}{t^{2}}=4310 kips / \text { in. }>4100 kips / in .\) o.k.

 \(\begin{aligned}\left(\frac{R_{a} W}{t^{2}}\right)_{r e q} &=\frac{80.0 kips (7.00 in. )}{(0.465 in .)^{2}} \\&=2720 kips / in .\end{aligned}\)

 

To satisfy the minimum above, select a stiffener L = 24 in. from Table 10-14

 

\(\frac{R_{a} W}{t^{2}}=2870 kips / in. >2720 kips / in .\) o.k.

Manual
Table 10-14

Use PL5/8×7×2’-0 for the stiffener.

Check the HSS width

The minimum width is \(0.4 L+t_{p}+3 t\)

\(B=12.0 \text { in. }>0.4(24.0 \text { in. })+5 / 8 \text { in. }+3(0.465 \text { in. })=11.6 \text { in. }\)

Determine the seat plate dimensions

To accommodate two 3/4-in. diameter ASTM A325-N bolts on a 5 1/2 in. gage connecting the beam flange to the seat plate, a width of 8 in. is required. To accommodate the seat-plate-toHSS weld, the required width is:

\(0.4(24 in .)+0.625 in .=10.2 in\)

Manual
Part 10

Note: To allow room to start and stop welds, an 11.5 in. width is used.

Use PL3/8×7×0’-11 1/2 for the seat plate.

Select the top angle, bolts and welds

The minimum weld size for the HSS thickness according to Specification Table J2.4 is 3/16 in.

The angle thickness should be 1/16 in. larger.

Use L4×4×1/4 with 3/16-in. fillet welds along the toes of the angle to the beam flange and HSS. Alternatively two 3/4-in. diameter ASTM A325-N bolts may be used to connect the beam leg of the angle to the beam flange.
