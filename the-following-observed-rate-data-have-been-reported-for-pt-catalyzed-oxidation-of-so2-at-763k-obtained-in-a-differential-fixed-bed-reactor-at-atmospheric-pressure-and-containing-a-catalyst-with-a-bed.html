Guest Question.

The following observed rate data have been reported for Pt-catalyzed oxidation of \(SO_{2}\) at 763K obtained in a differential fixed-bed reactor at atmospheric pressure and containing a catalyst with a bed density of 0:8 g \(cm^{-3} \) [2].
The catalyst pellets were 3.2 by 3.2 mm cylinders, and the Pt was superficially deposited upon the external surface. If the void volume of the catalystbed is 1/2, the density of the catalyst itself will be 1.6 g \(cm^{-3} \).

a) What error prevails if, assuming linear (\(1^{st}\)-order) kinetics in \(SO_{2}\) and isothermal operation, external concentration gradients are ignored, i.e.what are the effectiveness factors?
b) Assume isothermal conditions and compute the concentration decreases to the surface, then determine the minimum and maximum external concentration gradients.
c) Assume no concentration gradient exists and compute the external temperature gradients.
d) If the reaction activation energy is 30 kcal \(mole^{-1}\) , assuming no concentration gradient, what error in rate measurement exists if an external \(\Delta T\) is neglected , i.e., what are the nonisothermal effectiveness factors?

Because of the change in the Reynolds number, calculated values of \(k_{g}\) change; however, because of the relationship between two other dimensionless groups, the Prandtl and Schmidt numbers, assume the \(k_{g}/h\) ratio is constant at \(9.0\times 10^{3} cm^{3} K/cal\)

Solution

a)For isothermal,\(1^{st} \)-order reaction:

\(\overline{\pmb{\eta}} = C_{s}/C_{o}= 1- \overline{\pmb{\eta}} Da_{o}\) (Eq. 4.54)

 

\(\overline{\pmb{\eta}} Da_{o}=\Re / k_{g} a C_{o}\) (Eq. 4.50)

\(a=\frac{A}{V}= \frac{2\pi r^{2} + 2\pi rL}{L \pi r^{2}} = 18.75 cm^{-1} \) for cylinder

(For a sphere, \(a=\frac{A}{V}=\frac{6}{2r}= 18.75 cm^{-1} \))

\(C_{o_{SO_{2}}} = \frac{P}{RT}=\frac{0.06\ atm}{\left\lgroup\frac{82.06\ atm \cdot cm^{3}}{g\ mole\cdot K} \right\rgroup\left(763.2\ K\right)}=9.58\times 10^{-7} \frac{mole\ SO_{2} }{cm^{3}} \)

See Table 1 for \(\overline{\pmb{\eta}} Da_{o}\) and \(\overline{\pmb{\eta}}\) values.

b) The concentration drop can be determined from Eq. 4.52: \(C_{s} /C_{o}= 1- \frac{\Re }{k_{g}a C_{o}} = 1- \overline{\pmb{\eta}} Da_{o}\),

i.e.,\(C_{s} /C_{o}= 1- \overline{\pmb{\eta}} Da_{o}\) so \(C_{o}-C_{s}=\Re/k_{g}a =\Delta C\)

The \(\Delta C\) values are given in Table 1. The concentration gradients through the film thickness , \(\pmb{\delta} \), will be \(\Delta C/\pmb{\delta}\) .

From Eq. 4.44 : \(k_{g}= D/ \pmb{\delta} \) , so \(\pmb{\delta} = D/k_{g}\)

From Eq. 4.75 : \(D_{b} = \frac{\overline{v} \lambda _{g} }{3}\),

the bulk diffusivity is: \(D_{b}= \frac{\overline{v} \lambda }{3} \) The mean-free path in the gas 3 phase is (See Illustration 4.6):

\(\lambda =\frac{1}{\sqrt{2} \pi \sigma ^{2} \left(N/V\right) } = \frac{RT}{\sqrt{2} \pi \sigma ^{2} P }\)

for an ideal gas where \(\sigma\) = is the molecular cross-section area, so

\(\lambda =\frac{\left(\frac{82.06 \ atm \ cm^{3}}{g \ mole \cdot K} \right) \left(763.2 K\right) \left\lgroup\frac{1 mole }{6.02 \times 10^{23} molecule} \right\rgroup }{\sqrt{2} \pi \left [\frac{\left(4\times 10^{-8} cm\right)^{2} } {molecule} \right] \left(1 atm\right) } = 1.5 \times 10^{-5} cm \)

 

\(\overline{v}= \left\lgroup \frac{8 k_{B} T}{\pi m} \right\rgroup ^{1/2} \) (Eq. 4.76)

 

\(\overline{v}=\left[\frac{8\left(1.38 \times 10^{-16} erg/K \right)\left(763.2 K\right)}{\pi \left(64 \ amu\right) \left(1.66 \times 10^{-24} \ g/amu \right) } \right] ^{1/2} = 5.0 \times 10^{4} cm/s\)

 

\(D_{b}= 1/3\left(1.5 \times 10^{-5} cm \right) \left(5.0 \times 10^{4} cm/s\right) = 0.25 \ cm^{2}/s \)

so at high \(SV: \delta = D/k_{g} = \frac{0.25 cm^{2}/s}{\left(4.5 \times 10^{4} cm/h \right)\left(1 \ h /3600s\right) } = 0.020 cm\)

and at low \(SV: \delta=\frac{0.25 cm^{2}/s}{\left(2.8 \times 10^{4} cm/h \right)\left(1 \ h /3600s\right) } = 0.032 cm\)

at high \(SV : \Delta C/\delta = \frac{2.6 \times 10^{-7} mole/cm^{3} }{0.020 cm} = 1.3 \times 10^{-5} mole/cm^{4}\)

at low \(SV : \Delta C/\delta = \frac{2.9 \times 10^{-7} mole/cm^{3} }{0.032 cm} = 9.0 \times 10^{-6} mole/cm^{4}\)

c) Nonisothermal situation, assume \(C_{s} \cong C_{o}\)

From Eq 4.51 \(\overline{\eta } = \frac{K_{s} }{K_{o} } \left(\frac{C_{S} }{C_{O} } \right) ^{n}= \frac{K_{s} }{K_{o} } \)

and \(\frac{K_{s} }{K_{o} }= e^{-E/RT\left(1/t-1\right) } \)

also \(t=T_{s}/T_{o}= 1+ \overline{\beta } \cdot \overline{\eta } Da_{o} \)

where \(\overline{\beta } = \frac{- \Delta H^{o}_{R} C_{o} k_{g} }{T_{o} h } \)

\(\Delta H^{o}_{R} = -23.6 kcal/mole SO_{2}\) (CRC Handbook)

\(\overline{\beta } = – \frac{\left\lgroup\frac{- 23600 cal}{mole SO_{2} } \right\rgroup \left\lgroup\frac{9.58 \times 10^{-7} mole SO_{2}}{cm^{3} } \right\rgroup \left\lgroup\frac{9 \times 10^{3} cm^{3} K}{cal} \right\rgroup }{763.2 K} = 0.27 \)

Then \(\Delta T= T_{s} – T_{o} = \overline{\beta } \cdot \overline{\eta } Da_{o} T_{o}\) and the \(\overline{\beta } \cdot \overline{\eta } Da_{o}\) values are listed in Table 1.

d) The non-isothermal effectiveness factor is

\(\overline{\eta}^{'}= \frac{k_{s} }{k_{o} } = \frac{\Re _{s} }{\Re _{o} } = e^{-E/RT_{o}\left(1/t-1\right) } = \left(1- \overline{\eta }Da_{o} \right) e^{-E/RT_{o} } \left(-\overline{\beta }\cdot \overline{\eta } Da_{o} \right)\left(1+\bar{\beta }\cdot\bar{\eta}Da_{o}\right) \)

where \(E/RT_{o}= \frac{-3000 cal/mole}{\left(1.987 cal/mole\cdot K\right) \left(763.2 K\right) } = -19.78\)

The \(\overline{\eta}^{'}\) values are also given in Table 1.
