Guest Question.

The missile shown in Figure 13.14(a) can be aerodynamically controlled by torques created by the deflection of control surfaces on the missile’s body. The commands to deflect these control surfaces come from a computer that uses tracking data along with programmed guidance equations to determine whether the missile is on track. The information from the guidance equations is used to develop flight control commands for the missile. A simplified model is shown in Figure 13.14(b). Here the computer performs the function of controller by using tracking information to develop input commands to the missile .An accelerometer in the missile detects the actual acceleration, which is fed back to the computer. Find the closed-loop digital transfer function for this system and determine if the system is stable for K = 20 and K = 100 with a sampling interval of T = 0.1 second.

Admin
Solution

The input to the control system is an acceleration command developed by the computer. The computer can be modeled by a sample-and-hold. The s-plane model is shown in Figure 13.14(c). The first step in finding the z-plane model is to find G(z), the forward-path transfer function. From Figure 13.14(c) or ( d),

\(G\left ( s \right )=\frac{1-e^{-Ts}}{s}\frac{Ka}{s\left ( s+a \right )}\) (13:48)

where a = 27. The z-transform, G(z), is \(\left ( 1-z^{-1} \right )z\left \{ Ka/\left [ s^{2}\left ( s+a \right ) \right ] \right \}\) .

The term \(Ka/\left [ s^{2}\left ( s+2 \right ) \right ]\) is first expanded by partial fractions, after which we find the z-transform of each term from Table 13.1. Hence,

\(z\left \{ \frac{Ka}{s^{2}\left ( s+a \right )} \right \} = Kz\left \{ \frac{Ka}{s^{2}\left ( s+a \right )} \right \} = Kz\left \{ \frac{1}{s^{2}}- \frac{1/a}{s}+\frac{1/a}{s+a} \right \}\)

 

\(=K\left \{ \frac{Tz}{\left ( z-1 \right )^{2}}-\frac{z/a}{z-1} +\frac{z/a}{z-e^{-aT}}\right \}\)

 

\(= K \left \{ \frac{Tz}{\left ( z-1 \right )^{2}}-\frac{(1-e^{-aT})z}{a(z-1)(z-e^{-aT})}\right \}\) (13:49)

Thus,

\(G\left ( z \right )= K \left \{ \frac{T(z-e^{-aT})-\left ( z-1 \right )\left ( \frac{1-e^{-aT}}{a} \right )}{\left ( z-1 \right )(z-e^{-aT})}\right \} \) (13:50)

Letting T = 0.1 and a = 27, we have

\(G\left ( z \right ) = \frac{K\left ( 0.0655z +0.02783 \right )}{\left ( z-1 \right )\left ( z-0.0672 \right )}\) (13:51)

Finally, we find the closed-loop transfer function, T(z) ,for a unity feedback system:

\(T\left ( z \right ) =\frac{G\left ( z \right )}{1+ G\left ( z \right )} = \frac{K\left ( 0.0655z + 0.02783 \right )}{z^{2} +\left ( 0.0655K + 0.02783 \right )z +\left ( 0.02783K + 0.0672 \right )}\) (13:52)

The stability of the system is found by finding the roots of the denominator. For K = 20 ,the roots of the denominator are \(0.12 \pm j0.78\).The system is thus stable for K = 20, since the poles are inside the unit circle. For K = 100, the poles are at – 0.58 and – 4.9. Since one of the poles is outside the unit circle, the system is unstable for K = 100.
