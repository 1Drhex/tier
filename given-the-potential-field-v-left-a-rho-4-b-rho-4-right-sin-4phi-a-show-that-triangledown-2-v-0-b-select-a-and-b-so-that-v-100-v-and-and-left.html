Guest Question.

Given the potential field V = \(\left( A{ \rho }^{ 4 }+B{ \rho }^{ -4 } \right) \sin { 4\phi }\) : (a) show that \({ \triangledown }^{ 2 }\) V = 0; (b) select A and B so that V = 100 V and and \(\left| E \right|\) = 500 V/m at P\((\rho = 1, \phi = \mathring { 22.5 } , z = 2)\).

Admin
Solution

a)In cylindrical coordinates,
\({ \triangledown }^{ 2 }V=\frac { 1 }{ \rho } \frac { \partial }{ \partial \rho } \left( \rho \frac { \partial V }{ \partial \rho } \right) +\frac { 1 }{ { \rho }^{ 2 } } \frac { { \partial }^{ 2 }V }{ \partial { \phi }^{ 2 } } \\ =\frac { 1 }{ \rho } \frac { \partial }{ \partial \rho } \left( \rho \left( 4A{ \rho }^{ 3 }-4B{ \rho }^{ -5 } \right) \right) \sin { 4\phi } -\frac { 1 }{ { \rho }^{ 2 } } 16\left( A{ \rho }^{ 4 }+B{ \rho }^{ -4 } \right) \sin { 4\phi } \\ =\frac { 16 }{ \rho } \left( A{ \rho }^{ 3 }+B{ \rho }^{ -5 } \right) \sin { 4\phi } -\frac { 16 }{ { \rho }^{ 2 } } \left( A{ \rho }^{ 4 }+B{ \rho }^{ -4 } \right) \sin { 4\phi } =0\)
b) First,
\(E=-\triangledown V=-\frac { \partial V }{ \partial \rho } { a }_{ \rho }-\frac { 1 }{ \rho } \frac { \partial V }{ \partial \phi } { a }_{ \phi }\\ =-4\left[ \left( A{ \rho }^{ 3 }-B{ \rho }^{ -5 } \right) \sin { 4\phi } { a }_{ \rho }+\left( A{ \rho }^{ 3 }+B{ \rho }^{ -5 } \right) \cos { 4\phi } { a }_{ \phi } \right] \)
and at P , \({ E }_{ P } = -4(A – B){ a }_{ \rho }\). Thus \(\left| { E }_{ P } \right| = \pm 4 (A – B)\). Also, \({ V }_{ P }\) = A + B. Our two equations are:
4(A- B) = \(\pm 500\)
and
A + B = 100
We thus have two pairs of values for A and B:
A = 112.5, B = -12.5 or A = -12.5, B = 112.5
