Guest Question.

An uncharged conducting sphere of radius a is coated with a thick insulating shell (dielectric constant \(\epsilon_{r}\)) out to radius b. This object is now placed in an otherwise uniform electric field \(E _{0}\). Find the electric field in the insulator.

Admin
Solution

Potentials:

\(\left\{\begin{array}{lr}V_{\text {out }}(r, \theta) & =-E_{0} r \cos \theta+\sum \frac{B_{l}}{r^{l+1}} P_{l}(\cos \theta), & (r>b); \\V_{\text {med }}(r, \theta) & =\sum\left(A_{l} r^{l}+\frac{\bar{B}_{l}}{r^{l+1}}\right) P_{l}(\cos \theta), & (a<r<b); \\V_{\text {in }}(r, \theta) & =0, & (r<a).\end{array}\right.\)

Boundary Conditions:

\(\begin{cases}\text { (i) } V_{\text {out }}=V_{\operatorname{med},} & (r=b); \\ \text { (ii) } \epsilon \frac{\partial V_{\text {med }}}{\partial r}=\epsilon_{0} \frac{\partial V_{\text {out }}}{\partial r}, & (r=b); \\ \text { (iii) } V_{\text {med }}=0, \quad & (r=a).\end{cases}\)

\(\text { (i) } \Rightarrow-E_{0} b \cos \theta+\sum \frac{B_{l}}{b^{l+1}} P_{l}(\cos \theta)=\sum\left(A_{l} b^{l}+\frac{\bar{B}_{l}}{b^{l+1}}\right) P_{l}(\cos \theta)\) ;

 

\(\text { (ii) } \Rightarrow \epsilon_{r} \sum\left[l A_{l} b^{l-1}-(l+1) \frac{\bar{B}_{l}}{b^{l+2}}\right] P_{l}(\cos \theta)=-E_{0} \cos \theta-\sum(l+1) \frac{B_{l}}{b^{l+2}} P_{l}(\cos \theta)\) ;

 

\(\text { (iii) } \Rightarrow A_{l} a^{l}+\frac{\bar{B}_{l}}{a^{l+1}}=0 \Rightarrow \bar{B}_{l}=-a^{2 l+1} A_{l}\) ;

For l â‰  1 :

\(\text { (i) } \frac{B_{l}}{b^{l+1}}=\left(A_{l} b^{l}-\frac{a^{2 l+1} A_{l}}{b^{l+1}}\right) \Rightarrow B_{l}=A_{l}\left(b^{2 l+1}-a^{2 l+1}\right)\) ;

 

\(\text { (ii) } \epsilon_{r}\left[l A_{l} b^{l-1}+(l+1) \frac{a^{2 l+1} A_{l}}{b^{l+2}}\right]=-(l+1) \frac{B_{l}}{b^{l+2}} \Rightarrow B_{l}=-\epsilon_{r} A_{l}\left[\left(\frac{l}{l+1}\right) b^{2 l+1}+a^{2 l+1}\right] \Rightarrow A_{l}=B_{l}=0\) .

For l = 1 :

 

\(\text { (i) }-E_{0} b+\frac{B_{1}}{b^{2}}=A_{1} b-\frac{a^{3} A_{1}}{b^{2}} \Rightarrow B_{1}-E_{0} b^{3}=A_{1}\left(b^{3}-a^{3}\right)\) ;

 

\(\text { (ii) } \epsilon_{r}\left(A_{1}+2 \frac{a^{3} A_{1}}{b^{3}}\right)=-E_{0}-2 \frac{B_{1}}{b^{3}} \Rightarrow-2 B_{1}-E_{0} b^{3}=\epsilon_{r} A_{1}\left(b^{3}+2 a^{3}\right)\) .

 

\(\text { So }-3 E_{0} b^{3}=A_{1}\left[2\left(b^{3}-a^{3}\right)+\epsilon_{r}\left(b^{3}+2 a^{3}\right)\right] ; \quad A_{1}=\frac{-3 E_{0}}{2\left[1-(a / b)^{3}\right]+\epsilon_{r}\left[1+2(a / b)^{3}\right]}\) .

 

\(V_{ med }(r, \theta)=\frac{-3 E_{0}}{2\left[1-(a / b)^{3}\right]+\epsilon_{r}\left[1+2(a / b)^{3}\right]}\left(r-\frac{a^{3}}{r^{2}}\right) \cos \theta\) ,

 

\(E (r, \theta)=-\nabla V_{ med }=\frac{3 E_{0}}{2\left[1-(a / b)^{3}\right]+\epsilon_{r}\left[1+2(a / b)^{3}\right]}\left\{\left(1+\frac{2 a^{3}}{r^{3}}\right) \cos \theta \hat{ r }-\left(1-\frac{a^{3}}{r^{3}}\right) \sin \theta \hat{ \theta }\right\}\) .
