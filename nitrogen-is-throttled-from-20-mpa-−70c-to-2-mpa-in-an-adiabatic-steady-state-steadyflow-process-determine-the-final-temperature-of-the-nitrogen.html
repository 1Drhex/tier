Guest Question.

Nitrogen is throttled from 20 MPa, −70°C, to 2 MPa in an adiabatic, steady-state, steadyflow process. Determine the final temperature of the nitrogen.
Control volume: Throttling valve.
Inlet state: \(P_{1}, T_{1}\) known; state fixed.
Exit state: \(P_{2}\) known.
Process: Steady-state, throttling process.
Diagram: Figure 12.8.
Model: Generalized charts, Fig. D.2.

Energy equation:

\(h_{1}=h_{2}\)

Admin
Solution

Using values from Table A.2, we have

\(P_{1}=20 MPa \quad P_{r 1}=\frac{20}{3.39}=5.9\)

\(T_{1}=203.2 K \quad T_{r 1}=\frac{203.2}{126.2}=1.61\)

\(P_{2}=2 MPa \quad P_{r 2}=\frac{2}{3.39}=0.59\)

From the generalized charts, Fig. D.2, for the change in enthalpy at constant temperature, we have

\(\frac{h_{1}^{*}-h_{1}}{R T_{c}}=2.1\)

\(h_{1}^{*}-h_{1}=2.1 \times 0.2968 \times 126.2=78.7 kJ / kg\)

It is now necessary to assume a final temperature and to check whether the net change in enthalpy for the process is zero. Let us assume that \(T_{2}=146 K\). Then the change in enthalpy between \(1^{*}\) and \(2^{*}\) can be found from the zero-pressure, specific-heat data.

\(h_{1}^{*}-h_{2}^{*}=C_{p 0}\left(T_{1}^{*}-T_{2}^{*}\right)=1.0416(203.2-146)=+59.6 kJ / kg\)

(The variation in \(C_{p 0}\) with temperature can be taken into account when necessary.) We now find the enthalpy change between \(2^{*}\) and 2.

\(T_{r 2}=\frac{146}{126.2}=1.157 \quad P_{r 2}=0.59\)

Therefore, from the enthalpy departure chart, Fig. D.2, at this state

\(\frac{h_{2}^{*}-h_{2}}{R T_{c}}=0.5\)

\(h_{2}^{*}-h_{2}=0.5 \times 0.2968 \times 126.2=19.5 kJ / kg\)

We now check to see whether the net change in enthalpy for the process is zero.

\(h_{1}-h_{2}=0=-\left(h_{1}^{*}-h_{1}\right)+\left(h_{1}^{*}-h_{2}^{*}\right)+\left(h_{2}^{*}-h_{2}\right)\)

\(=-78.7+59.6+19.5 \approx 0\)

It essentially checks. We conclude that the final temperature is approximately 146 K. It is interesting that the thermodynamic tables for nitrogen, Table B.6, give essentially this same value for the final temperature.
