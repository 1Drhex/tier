Guest Question.

Designing a narrow-band-pass filter

(a) Design a narrow-band-pass filter as in Fig. 12.31(b) such that \(f_{ C }=1 kHz , Q=4, \text { and } K_{ PB }=8\).

(b) Calculate the value of RB required to change the center frequency from 1 kHz to 1.5 kHz.

(c) Use PSpice/SPICE to plot the frequency response of the narrow-band-pass filter designed in part (a) from 100 Hz to 1 MHz.

Admin
Solution

(a) \( f_{ C }=1 kHz \text { and } Q=4 \text {. Let } C_{1}=C_{2}=C=0.0047 \mu F \). Check that the condition in Eq. (12.98) is satisfied; that is, \( 2 Q^{2}=2 \times 4^{2}=32 \), which is greater than \( K_{ PB }=8 \). Thus, we must use \( R_{ B } \) in Fig. 12.32. Using Eqs. (12.94), (12.95), (12.96), and (12.97), we get

\( R_{1}=\frac{Q}{2 \pi f_{ C } C K_{ PB }} \) (12.94)

 

\(R_{2}=\frac{Q}{\pi f_{ C } C}\) (12.95)

 

\(K_{ PB }=\frac{R_{2}}{2 R_{1}}=2 Q^{2}\) (12.96)

 

\(R_{ B }=\frac{R_{1} K_{ PB }}{2 Q^{2}-K_{ PB }}=\frac{Q}{2 \pi f_{ C } C\left(2 Q^{2}-K_{ PB }\right)}\) (12.97)

 

\( K_{ PB }<2 Q^{2} \) (12.98)

 

\(\begin{aligned}&R_{1}=\frac{Q}{2 \pi f_{ C } C K_{ PB }}=\frac{4}{2 \pi \times 1 kHz \times 0.0047 \mu F \times 8}=16.93 k \Omega \\&R_{2}=\frac{Q}{\pi f_{ C } C}=\frac{4}{\pi \times 1 kHz \times 0.0047 \mu F }=270.9 k \Omega \\&K_{ PB }=\frac{R_{2}}{2 R_{1}}=\frac{270.9 k \Omega}{2 \times 16.93 k \Omega}=8 \\&R_{ B }=\frac{Q}{2 \pi f_{ C } C\left(2 Q^{2}-K_{ PB }\right)}=\frac{4}{2 \pi \times 1 kHz \times 0.0047 \mu F \times\left(2 \times 4^{2}-8\right)}=5.64 k \Omega\end{aligned}\)

 

(b) From Eq. (12.99), we find that the new value of \( R_{ B }^{\prime} \) is

\(R_{ B }^{\prime}=R_{ B }\left(\frac{f_{ C }}{f_{ C }^{\prime}}\right)^{2}\) (12.99)

 

\( R_{ B }^{\prime}=R_{ B }\left(\frac{f_{ C }}{f_{ C }}\right)^{2}=5.64 k \Omega\left(\frac{1 kHz }{1.5 kHz }\right)^{2}=2.51 k \Omega \)

 

NOTE: The exact numbers of the resistances are used to demonstrate the validity of the calculated values with the PSpice simulation results. In a practical design, the commercially available standard values will be used (i.e., 17 kΩ instead of 16.93 kΩ and 270 kΩ instead of 270.9 kΩ ; see Appendix E).

(c) The narrow-band-pass filter with the designed values is shown in Fig. 12.32.
The frequency response is shown in Fig. 12.33, which gives \( f_{ C } \approx 1 kHz \) (expected value is 1 kHz) and \( K_{ PB }=8 \).
