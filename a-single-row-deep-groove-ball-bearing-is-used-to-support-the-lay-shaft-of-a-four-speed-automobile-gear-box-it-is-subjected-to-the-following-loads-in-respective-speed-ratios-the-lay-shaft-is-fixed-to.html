A single-row deep groove ball bearing is used to support the lay shaft of a four speed automobile gear box. It is subjected to the following loads in respective speed ratios:

load (N)		 Axial load

1% 4000 3250 First gear
3% 2750 500 Second gear
21% 2750 50 Third gear
75% Nil Nil Fourth gear

Solution

\( \text { Given } \quad n=1750 rpm \quad C_{o}=11600 N \).

\( C=17600 N \quad L_{h}=4000 h \).

Step I Equivalent load for complete work cycle
Considering the work cycle of one minute duration,

\( N_{1}=\frac{1}{100}(1750)=17.50 rev \).

\( N_{2}=\frac{3}{100}(1750)=52.50 rev \).

\( N_{3}=\frac{21}{100}(1750)=367.50 rev \).

\( N_{4}=\frac{75}{100}(1750)=1312.50 rev \).

\( \left(N_{1}+N_{2}+N_{3}+N_{4}\right)=1750 rev \).

First gear

\( \left(\frac{F_{a}}{F_{r}}\right)=\left(\frac{3250}{4000}\right)=0.8125 \).

and \( \left(\frac{F_{a}}{C_{o}}\right)=\left(\frac{3250}{11600}\right)=0.28 \).

Step II Bearing life \( L_{10} \text { and } L \)
From Eq. (15.6),

\( L_{10}=\left(\frac{C}{P}\right)^{3} \) (15.6).

\( L_{10}=\left(\frac{C}{P}\right)^{3}=\left(\frac{17600}{1932.67}\right)^{3}=755.2 \text { million rev. } \).

From Eq. (15.9),

\( L=\frac{60 n L_{ h }}{10^{6}} \) (15.9).

\( L=\frac{60 n L_{ h }}{10^{6}}=\frac{60(1750)(4000)}{10^{6}}=420 \text { million rev } \).

Step III Reliability of bearing
From Eq. (15.17),

\( \left(\frac{L}{L_{10}}\right)=\left[\frac{\log _{e}\left(\frac{1}{R}\right)}{\log _{e}\left(\frac{1}{R_{90}}\right)}\right]^{1 / b} \) (15.17).

\( \left(\frac{L}{L_{10}}\right)=\left[\frac{\log _{e}\left(\frac{1}{R}\right)}{\log _{e}\left(\frac{1}{R_{90}}\right)}\right]^{1 / b} \).

or \( \left(\frac{L}{L_{10}}\right)^{b}=\left[\frac{\log _{e}\left(\frac{1}{R}\right)}{\log _{e}\left(\frac{1}{R_{90}}\right)}\right] \).

Substituting the following values,

\( L=420 \text { million rev. } L_{10}=755.2 \text { million rev. } \)

\( R_{90}=0.90 \quad b=1.17 \).

we get,

\( \left(\frac{420}{755.2}\right)^{1.17}=\left[\frac{\log _{e}\left(\frac{1}{R}\right)}{\log _{e}\left(\frac{1}{0.90}\right)}\right] \).

\( \therefore \quad R=0.9483 \text { or } 94.83 \% \).
