Guest Question.

A three-phase half-wave diode rectifier has the following specifications: Input voltage 127 V 50 Hz, load resistance 1 Ω, and load inductance 100 mH.
a) Calculate the average output voltage and current.
b) If the input source has an inductance of 1 mH per phase and the average output current
is 129 A, calculate the commutation overlap angle and the average output voltage.

Admin
Solution

a)Using Eq. (4.89) the average output voltage is:

 

\(\begin{aligned}\overline{ V }_{ o } &=\frac{\text { Area of one pulse }}{\text { Interval of one pulse }}=\frac{ A }{\frac{2 \pi}{3}}=\frac{\int_{-\frac{\pi}{3}}^{\frac{\pi}{3}} \sqrt{2} \widetilde{ V }_{ i } \cos (\omega t ) d \omega t }{\frac{2 \pi}{3}} \\&=\left.\frac{3 \sqrt{2} \widetilde{ V }_{ i }}{2 \pi}(\sin \omega t )\right|_{-\frac{\pi}{3}} ^{\frac{\pi}{3}}=\frac{3 \sqrt{2} \widetilde{ V }_{ i }}{2 \pi}\left(\sin \frac{\pi}{3}-\sin \left(-\frac{\pi}{3}\right)\right) \\&=\frac{3 \sqrt{6} \widetilde{ V }_{ i }}{2 \pi}=1.17 \widetilde{ V }_{ i }\end{aligned}\) (4.89)

 

\(\overline{ V }_{ o }=1.17 \widetilde{ V }_{ i }=1.17 \times 127=148.6 V\)

 

Therefore, the average output current is \(\overline{ I }_{ o }=\frac{\overline{ V }_{o}}{ R }=\frac{148}{1}=148.6 A\) Since \(\omega L \gg R \text {, }\) the output current is considered to be pure dc of 148 A.

b) In case there is an input source inductance \(L _{ S }\) in the power source, the commutation overlap angle μ is given by:

 

\(\begin{aligned}\mu &=\cos ^{-1}\left(1-\frac{2 \overline{ I }_{ o } \omega L _{ s }}{\sqrt{6} \widetilde{ V }_{ i }}\right)=\cos ^{-1}\left(1-\frac{2 \times 129 \times 2 \pi \times 50 \times 0.001}{\sqrt{6}(127)}\right) \\&=\cos ^{-1}(0.739)=42.35^{\circ}\end{aligned}\)

 

The reduction of the average output voltage due to the commutation overlap phenomenon is:

 

\(\overline{ V }_{ o ( loss )}=0.58 \widetilde{ V }_{ i }(1-\cos \mu)=0.58 \times 127\left(1-\cos 42.35^{\circ}\right)=19.22 V\)

 

Therefore, the average output voltage is \(\overline{ V }_{ o }=148.6-19.22=129.39 V\) Due to the commutation overlap the average output voltage is reduced by 12.93%.
