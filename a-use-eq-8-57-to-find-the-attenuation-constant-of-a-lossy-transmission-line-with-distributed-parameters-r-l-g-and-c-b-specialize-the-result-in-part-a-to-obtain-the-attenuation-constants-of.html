Guest Question.

\(a)\) Use Eq. (8-57) to find the attenuation constant of a lossy transmission line with distributed parameters \(R, L, G\), and \(C\).

\(b)\) Specialize the result in part \((a)\) to obtain the attenuation constants of a low-loss line and of a distortionless line.

Admin
Solution

\(a)\) For a lossy transmission line the time-average power loss per unit length is

 

\({ P }_{ z }\left( z \right) =\frac { 1 }{ 2 } \left[ { \left| I\left( z \right) \right| }^{ 2 }R+{ \left| V\left( z \right) \right| }^{ 2 }G \right]\)

 

\(\quad =\frac { { V }^{ 2 } }{ 2{ \left| { Z }_{ 0 } \right| }^{ 2 } } \left( R+G{ \left| V\left( z \right) \right| }^{ 2 } \right) { e }^{ -2\alpha z }\quad \quad \left( 8-58 \right)\)

 

Substitution of Eq . (8-56) and (8-58) in Eq. ( -57) gives

 

\(\alpha =\frac { 1 }{ 2{ R }_{ 0 } } \left( R+G{ \left| { Z }_{ 0 } \right| }^{ 2 } \right) \quad \left( Np/m \right) \quad \quad \left( 8-59 \right)\)

 

\(b)\) For a low-loss line, \({ Z }_{ 0 }\cong { R }_{ 0 }=\sqrt { L/C }\), Eq. (8-59) becomes

 

\(\alpha \cong \frac { 1 }{ 2 } \left( \frac { R }{ { R }_{ 0 } } +G{ R }_{ 0 } \right)\)

 

\(\alpha \cong \frac { 1 }{ 2 } \left( \frac { R }{ { R }_{ 0 } } +G{ R }_{ 0 } \right) \quad \quad \left( 8-60 \right) \)

 

For a distortionles line, (Eq. (8-52) gives \({ Z }_{ 0 }={ R }_{ 0 }=\sqrt { L/C }\) .Thu q. (8-60) applies and yields

 

\(\alpha =\frac { 1 }{ 2 } R\sqrt { \frac { C }{ L } } \left( 1+\frac { G }{ R } \frac { L }{ C } \right) \)

 

which, in view of the condition in Eq. (8-46), reduces to

 

\(\alpha =R\sqrt { \frac { C }{ L } }\)

 

\(\alpha =\frac { { P }_{ L }\left( z \right) }{ 2P\left( z \right) } \quad \quad \left( Np/m \right) \quad \quad \left( 8-57 \right) \)

 

\(P\left( z \right) =\frac { { V }_{ 0 }^{ 2 } }{ 2{ \left| { Z }_{ 0 } \right| }^{ 2 } } { R }_{ 0 }{ e }^{ -2\alpha z }\quad \quad \quad \left( 8-57 \right)\)
