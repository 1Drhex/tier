Guest Question.

A rectangular bar having width w= 6.00 in. and thickness t= 1.50 in. is subjected to a tension load Pas shown in Figure P1.42/43. The normal and shear stresses on plane AB must not exceed 16 ksi and 8 ksi, respectively. Determine the maximum load P that can be applied without exceeding either stress limit.

Admin
Solution

The general equations for normal and shear stresses on an inclined plane in terms of the angle \(\theta \) are
\({ \sigma }_{ n }=\frac { P }{ 2A } \left( 1+cos\quad 2\theta \right) \)
and
\({ \tau }_{ nt }=\frac { P }{ 2A } sin{ 2 }\theta \)
The angle\(\theta \) for inclined plane AB is calculated from
\(\tan { \theta =\frac { 3 }{ 1 } =3 } \quad \quad \quad \quad \quad \quad \therefore \theta =71.5651°\)
The cross-sectional area of the bar is\(A=wxt=\left( 6.00㏌. \right) \left( 1.50㏌. \right) =9.0{ ㏌. }^{ 2 }\)
The normal stress on plane AB is limited to 16 ksi; therefore, the maximum load Pcan be found from Eq. (a):
\(P\le \frac { 2A{ \sigma }_{ n } }{ 1+cos2\theta } =\frac { 2{ \left( 9.2㏌. \right) }^{ 2 }\left( 16\quad ksi \right) }{ 1+cos2\left( 71.5651° \right) } =1,440\quad ksi\)
The shear stress on plane AB is limited to 8 ksi. From Eq. (b), the maximum load Pbased the shear stress limit is
\(P\le \frac { 2A{ \tau }_{ nt } }{ sin2\theta } =\frac { 2{ \left( 9.2㏌. \right) }^{ 2 }\left( 8\quad ksi \right) }{ sin2\left( 71.5651° \right) } =\quad 240\quad kips\)
Thus, the maximum load that can be supported by the bar is
\({ P }_{ max\quad }=\quad 240\quad kips\)
