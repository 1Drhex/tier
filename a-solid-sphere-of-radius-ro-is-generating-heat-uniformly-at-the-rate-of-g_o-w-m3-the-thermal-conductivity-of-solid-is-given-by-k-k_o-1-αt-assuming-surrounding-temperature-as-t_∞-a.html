Guest Question.

A solid sphere of radius ro is generating heat uniformly at the rate of \(g_o W/m^3.\) The thermal conductivity of solid is given by

\(k = k_o (1 + αT)\)
Assuming surrounding temperature as T_∞ and heat transfer coefficient as h, prove that the temperature distribution in sphere is given by
\(T =-\frac{1}{\alpha } \pm \sqrt{\frac{g_o r_{o}^{2}}{3\alpha k_o}\left [ 1 - \left ( \frac{r}{r_o} \right )^{2} \right ]+\left (\frac{g_o r_{o}}{3h} + T_{\infty } + \frac{1}{\alpha } \right )^{2}}\)

Admin
Solution

Given : (i) The relation for thermal conductivity of solid sphere
\(k = k_o (1 + αT)\)
(ii) Heat generation rate = \(g_o\)
(iii) Heat transfer coefficient = h
(iv) Ambient temperature = \(T_∞.\)
Boundary conditions :
At r = 0,\(\frac{dT}{dr} = 0\)
At r = \(r_o, – k \left ( \frac{dT}{dr} \right ) = h[T_{r = r_o} – T_{∞}].\)
Analysis : The governing differential equation with variable thermal conductivity and constant heat generation rate go can be written as

\(\frac{1}{r^{2}}\frac{d}{dr}\left ( r^{2} k \frac{dT}{dr} \right )+ g_o =0\)
or \(\frac{d}{dr}\left ( r^{2} k \frac{dT}{dr} \right )= – g_o r^{2}\)
Integrating with respect to r, we get
\( r^{2} k \frac{dT}{dr} = – \frac{g_o r^{3}}{3}+C_1\)
Applying boundary condition at the centre i.e.,
At r = 0 ; \(\frac{dT}{dr}=0\)
It gives, \(C_1 =0\)
with using C_1 = 0, we get
\(r^{2} k \frac{dT}{dr} = – \frac{g_o r^{3}}{3}\)
or \(k \frac{dT}{dr} = – \frac{g_o r}{3} …(i)\)
Using \(k = k_o(1 + αT)\)
\(k_o(1 + αT)\frac{dT}{dr} = – \frac{g_o r}{3}\)

Rearranging \(k_o(1 + αT)dT = – \frac{g_o rdr}{3} \)

Integrating both sides, we get
\(k_o \left ( T + \frac{\alpha T^2}{2} \right ) = – \frac{g_o r^{2}}{6} +C_2 …(ii)\)
Now using the boundary condition
At \(r = r_o ; – k \left ( \frac{dT}{dr} \right ) = h(T_{r = r_o} – T_{∞})\)

Using eqn. (i) we get \(\frac{g_o r_o}{3} = h(T_{r = r_o} – T_{∞})\)
or \(T_{r = r_o} =\frac{g_o r_o}{3h} +T_{∞}\) …(iii)
Substituting the value of \(T_{ r =r_o} \) in eqn. (ii), at \(r = r_o\)
\(k_o\left \{ \left ( \frac{g_o r_o}{3h} +T_{∞} \right ) +\frac{\alpha }{2} \left (\frac{g_o r_o}{3h} +T_{∞} \right )^{2}\right \}=-\frac{g_o r_{o}^2}{6} + C_2 \)
or \(C_2 = k_ok_o\left \{ \left ( \frac{g_o r_o}{3h} +T_{∞} \right ) +\frac{\alpha }{2} \left (\frac{g_o r_o}{3h} +T_{∞} \right )^{2}\right \}+\frac{g_o r_{o}^2}{6} …(iv)\)
Substituting the value of \(C_2\) in eqn. (ii)

\(k_o \left \{ T + \frac{\alpha T^2}{2} \right \}= -\frac{g_o r^2}{6}+\frac{g_o r_{o}^2}{6} + k_o\left \{ \left ( \frac{g_o r_o}{3h} +T_{∞} \right ) +\frac{\alpha }{2} \left (\frac{g_o r_o}{3h} +T_{∞} \right )^{2}\right \}\)

or \(T + \frac{\alpha T^2}{2} = \frac{g_o r_{o}^2}{6k_o}\left \{ 1 – \left ( \frac{r}{r_o} \right )^{2} \right \} +\left \{ \left ( \frac{g_o r_o}{3h} +T_{∞} \right ) +\frac{\alpha }{2} \left (\frac{g_o r_o}{3h} +T_{∞} \right )^{2}\right \}\)

or \(T^{2} + \frac{ 2T}{\alpha} = \frac{g_o r_{o}^2}{3k_o\alpha } \left [1 – \left ( \frac{r}{r_o} \right )^{2} \right ] +\left \{\frac{2}{\alpha } \left ( \frac{g_o r_o}{3h} +T_{∞} \right ) + \left (\frac{g_o r_o}{3h} +T_{∞} \right )^{2} +\frac{1}{\alpha ^{2}} – \frac{1}{\alpha ^{2}}\right \}\)

\(T^{2} + \frac{ 2T}{\alpha} = \frac{g_o r_{o}^2}{3k_o\alpha } \left\{ 1 – \left ( \frac{r}{r_o} \right )^{2} \right \} +\left \{ \left ( \frac{g_o r_o}{3h} +T_{∞} +\frac{1}{\alpha ^{2}}\right ) ^{2} – \frac{1}{\alpha ^{2}}\right \}\) \( T^{2} + \frac{ 2T}{\alpha} – \frac{g_o r_{o}^2}{3k_o\alpha } \left\{ 1 – \left ( \frac{r}{r_o} \right )^{2} \right \} -\left \{ \left ( \frac{g_o r_o}{3h} +T_{∞} +\frac{1}{\alpha ^{2}}\right ) ^{2} + \frac{1}{\alpha ^{2}}\right \} =0\)

It is quadratic equation and its root for T are

\(T =\frac{-\frac{2}{\alpha }\pm \sqrt{\frac{4T^{2}}{\alpha ^{2}} + \frac{4g_o r_{o}^{2}}{3\alpha k_o}\left \{ 1 – \left ( \frac{r}{r_o} \right )^{2} \right \}-\frac{4}{\alpha ^{2}} + 4 \left ( \frac{g_o r_o}{3h} +T_{\infty } +\frac{1}{\alpha } \right )^{2}}}{2 \times 1}\)

 

\(T =-\frac{1}{\alpha }\pm \sqrt{\frac{g_o r_{o}^{2}}{3\alpha k_o}\left \{ 1 – \left ( \frac{r}{r_o} \right )^{2} \right \} + \left ( \frac{g_o r_o}{3h} +T_{\infty } +\frac{1}{\alpha^{2} } \right )^{2}} Proved.\)
