Guest Question.

Air at 30°C and 200 kPa travels at Mach 4.0. (a) Find the speed, temperature, pressure, shock strength, entropy increase, and pressure loss after the air passing through a normal shock. (b) For comparison, if the air flow experiences an oblique shock at an angle θ = 30°, find the same parameters downstream of this oblique shock. (c) In case of oblique shock, also find the angle through which the air flow is deflected. Assume for air \(c_{p} = 1.004 kJ/kg ⋅ K\).

Admin
Solution

(a) For the normal shock,

\(M^{2}_{2} =\frac{M^{2}_{1}+\frac{2}{\gamma -1} }{\frac{2\gamma }{\gamma -1}M^{2}_{1}-1} =\frac{16+\frac{2}{0.4} }{\frac{2.8}{0.4}\times 16-1 } =0.1892\); \(M_{2}=0.4350\)

 

\(\frac{T_{2} }{T_{1}}=\frac{\left(1+\frac{\gamma -1}{2} M^{2}_{1}\right)\left(\frac{2\gamma }{\gamma -1} M^{2}_{1}-1\right) }{\left[\frac{\left(\gamma +1\right)^{2} }{2\left(\gamma -1\right) } \right]M^{2}_{1} } =\frac{\left(1+\frac{0.4}{2}\times 16 \right)\left(\frac{2\times 1.4}{0.4} \times 16-1\right) }{\left[\frac{2.4^{2} }{2\left(1.4-1\right) } \right]\times 16 } =4.0469\);

\(T_{2}=1,226.2 K\)

 

\(V_{2}=M_{2} \sqrt{\gamma RT_{2} } =0.4350\times \sqrt{1.4\times 287\times 1,226.2} =305.3 m/s\)

 

\(\frac{p_{2} }{p_{1}}=\frac{2\gamma M^{2}_{1}}{\gamma +1} -\frac{\gamma -1}{\gamma +1} =\frac{2\times 1.4\times 16}{1.4+1} -\frac{1.4-1}{1.4+1} =18.50\); \(p_{2}=3,700 kPa\)

\(\frac{\Delta p}{p_{1} } =\frac{p_{2}}{p_{1}} -1=17.50\) (shock strength)

\(s_{2}-s_{1} =c_{p}\ln \frac{T_{2}}{T_{1}} -R\ln \frac{p_{2}}{p_{1}}=\left(1.004 kJ/kg\cdot K\right) \ln \left(4.0469\right) -\left(0.287 kJ/kg\cdot K\right) \ln \left(18.5\right) =0.5661 kJ/kg\cdot K\)

 

\(\frac{p_{t2}}{p_{t1}}=\left[\frac{\frac{\gamma +1}{2} M^{2}_{1}}{1+\frac{\gamma -1}{2}M^{2}_{1} } \right] ^{\gamma /\left(\gamma -1\right) } \left[\frac{1}{\frac{2\gamma }{\gamma +1} M^{2}_{1}-\frac{\gamma -1}{\gamma +1} } \right] ^{1/\left(\gamma -1\right) } =\left[\frac{\frac{1.4+1}{2}\times 16 }{1+\frac{1.4-1}{2}\times 16 } \right] ^{1.4/0.4} \left[\frac{1}{\frac{2\times 1.4}{1.4+1}\times 16 }-\frac{1.4-1}{1.4+1} \right] ^{1/0.4} =0.1388\)

 

\(\frac{p_{t1}}{p_{1}}=\left(1+\frac{\gamma -1}{2} M^{2}_{1}\right) ^{\gamma /\left(\gamma -1\right) } =\left(1+\frac{1.4-1}{2}\times 16 \right) ^{1.4/0.4} =151.52\)

 

\(\Delta p_{t} =\left(1-\frac{p_{t2}}{p_{t1}}\right)\frac{p_{t1}}{p_{1}} p_{1}=\left(1-0.1388\right) \times 151.52\times 200 kPa=26.098 MPa\)

(b) For the oblique shock, the first parameter to know is \(M_{1n}\).

\(M_{1n}= M_{1} \sin \theta =4 \sin 30° =2\)

 

\(M^{2}_{2n}=\frac{M^{2}_{1n}+\frac{2}{\gamma -1} }{\frac{2\gamma }{\gamma -1} M^{2}_{1n}-1} =\frac{4+\frac{2}{0.4} }{\frac{2.8}{0.4} \times 4-1} =0.3333\); \(M_{2n}=0.5774\)

 

\(M_{2t}=M_{1t}\sqrt{\frac{T_{1} }{T_{2} } }\); \(\frac{T_{2} }{T_{1} }=\frac{\left(1+\frac{\gamma -1}{2} M^{2}_{1n}\right) \left(\frac{2\gamma }{\gamma -1} M^{2}_{1n}-1\right) }{\left[\frac{\left(\gamma +1\right)^{2} }{2\left(\gamma -1\right) } \right] M^{2}_{1n}} =\frac{\left(1+\frac{0.4}{2}\times 4 \right)\left(\frac{2\times 1.4}{0.4} \times 4-1\right) }{\left[\frac{2.4^{2} }{2\left(1.4-1\right) } \right] \times 4} =1.6875\); \(T_{2}=511.3 K\)

 

\(M_{2t}=M_{1t}\sqrt{\frac{T_{1} }{T_{2} } } =M_{1}\cos 30°\sqrt{\frac{T_{1} }{T_{2} } }=4\cos 30°\sqrt{\frac{1}{1.6875} } =2.6667\)

 

\(M_{2}=\sqrt{M^{2}_{2n}+M^{2}_{2t}} =\sqrt{\left(0.5774\right)^{2} +\left(2.6667\right) ^{2} } =2.7285\)

It is noted that in this case the contribution of \(M_{2n}\) to \(M_{2}\) is relatively small.

\(V_{2} =M_{2} \sqrt{\gamma RT_{2} } =2.7285\times \sqrt{1.4\times 287\times 511.3} =1,236.7 m/s\)

 

\(\frac{p_{2} }{p_{1}}=\frac{2\gamma M^{2}_{1n} }{\gamma +1} -\frac{\gamma -1}{\gamma +1} =\frac{2\times 1.4\times 4}{1.4+1}-\frac{1.4-1}{1.4+1} =4.50 \); \(p_{2}=900 kPa\)

 

\(\frac{\Delta p}{p_{1}} =\frac{p_{2}}{p_{1}}-1=3.50\)

(This shock strength is five times smaller than that across a normal shock with the same upstream Mach number.)

\(s_{2}-s_{1} =c_{p}\ln \frac{T_{2}}{T_{1}} -R\ln \frac{p_{2}}{p_{1}}=\left(1.004 kJ/kg\cdot K\right) \ln \left(1.6875\right) -\left(0.287 kJ/kg\cdot K\right) \ln \left(4.5\right) =0.0936 kJ/kg\cdot K\)

 

\(\frac{p_{t2}}{p_{t1}}=\left[\frac{\frac{\gamma +1}{2} M^{2}_{1n}}{1+\frac{\gamma -1}{2}M^{2}_{1n} } \right] ^{\gamma /\left(\gamma -1\right) } \left[\frac{1}{\frac{2\gamma }{\gamma +1} M^{2}_{1n}-\frac{\gamma -1}{\gamma +1} } \right] ^{1/\left(\gamma -1\right) } =\left[\frac{\frac{1.4+1}{2}\times 4 }{1+\frac{1.4-1}{2}\times 4 } \right] ^{1.4/0.4} \left[\frac{1}{\frac{2\times 1.4}{1.4+1}\times 4 }-\frac{1.4-1}{1.4+1} \right] ^{1/0.4} =0.7209\)

 

\(\Delta p_{t} =\left(1-\frac{p_{t2}}{p_{t1}} \right) \frac{p_{t1}}{p_{1}}p_{1}=\left(1-0.7209\right) \times 151.52\times 200 kPa=8.458 MPa\)

(c) The flow deflection angle can be found using either Eqn. (4.24c) or Eqn.(4.24d).

\(\frac{M_{2n}}{M_{2}} =\frac{0.5774}{2.7285} =\sin \left(\theta -\delta \right) =\sin \left(30° -\delta \right)\); \(\delta =17.78° \)

Comments –

As discussed in the text, when the back pressure is insufficient to cause a normal shock (900 kPa < 3,700 kPa needed for a normal shock), an oblique shock may result. Both the pressure and temperature rises across an oblique shock are smaller than those across a normal shock, as an oblique shock is a weaker compression wave than a normal shock; it also causes a smaller entropy increase and thus a smaller pressure loss. The flow deflection angle can also be a result of a two-dimensional wedge (or surface) at an angle of 17.78° measured from the flow direction.

The downstream properties can also be found after \(M_{2}\) is calculated. As an illustration, for \(M_{2}=2.7285\), \(\frac{p_{t2}}{p_{2}}=\left(1+\frac{\gamma -1}{2} M^{2}_{2} \right) ^{\gamma /\left(\gamma -1\right) } =\left(1+\frac{1.4-1}{2}\times 7.4445 \right) ^{1.4/0.4}=24.32\)

\(p_{2}=\frac{p_{t2}}{24.32} =\frac{0.7209p_{t1}}{24.32} =\frac{0.7209\times 151.52\times 200 kPa}{24.32} =898.1 kPa\approx 900 kPa\)
