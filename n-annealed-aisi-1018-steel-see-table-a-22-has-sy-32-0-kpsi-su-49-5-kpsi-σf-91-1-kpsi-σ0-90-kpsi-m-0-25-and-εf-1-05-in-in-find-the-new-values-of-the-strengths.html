Solution

From Eq. (2–12), we find the true strain corresponding to the ultimate strength to be

\(m = ε_{u}\) (2–12)

\(ε_{u}=m=0.25 \)

The ratio\( A_{0}/A_{i}\) is, from Eq. (2–9),

\(W =\frac {A_{0} − A′_{i}}{A_{0}} ≈\frac {A_{0} − A_{i}}{A_{0}}\) (2–9)

\(\frac {A_{0}}{A_{i}} =\frac {1}{1 − W} =\frac {1}{1 − 0.15} = 1.176\)

The true strain corresponding to 15 percent cold work is obtained from Eq. (2–13). Thus

\(ε = ln \frac {l}{l_{0}} = ln \frac {A_{0}}{A}\) (2–13)

\(ε_{i} = ln \frac {A_{0}}{A_{i}} = ln 1.176 = 0.1625\)

Since \(ε_{i} < ε_{u}\) , Eqs. (2–14) and (2–15) apply. Therefore,

\(S′_{y} =\frac {P_{i}}{A′_{i}} = σ_{0}ε^{m}_{i} P_{i} ≤ P_{u}\) (2–14)

\(S′_{u} =\frac {S_{u} A_{0}}{A_{0}(1 − W)} =\frac {S_{u}}{1 − W} ε_{i} ≤ ε_{u}\) (2–15)

\(S′_{y} = σ_{0}ε^{m}_{i} = 90(0.1625)^{0.25} = 57.1 kpsi\)

\(S′_{u}=\frac {S_{u}}{1 − W} =\frac {49.5}{1 − 0.15} = 58.2 kpsi\)
