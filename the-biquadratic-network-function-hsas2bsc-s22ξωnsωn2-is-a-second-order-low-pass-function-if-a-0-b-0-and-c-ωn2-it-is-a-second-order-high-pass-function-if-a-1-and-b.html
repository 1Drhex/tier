Guest Question.

The biquadratic network function \(H(s)=\frac{a s^{2}+b s+c}{s^{2}+2 \xi \omega_{n} s+\omega_{n}^{2}}\) is a second-order low-pass function if \(a = 0\), \(b = 0\) and \(c = \omega_{n}^{2}\) It is a second-order high-pass function if \(a = 1\) and \(b = c = 0\). It is a band-pass function if \(a = c = 0\) and \(b = 2ξω_{n}\) and it is a band-reject function if \(a = 1\), \(b = 0\) and \(c =\omega_{n}^{2}\) The frequency response function for low-pass, high-pass and band-pass second-order functions were studied in detail in Section 11.11 in Chapter 11 in the context of frequency response of Series \(RLC\) circuit. Consider the band-pass function and band-reject function and sketch their frequency response plots for \(ξ << 1.\)

Admin
Solution

The poles of the function are at \(s=-\xi \omega_{n} \pm j \sqrt{1-\xi^{2}} \omega_{n}\) Let us consider the band-pass function first.

\(H(s)=\frac{2 \xi \omega_{n} s}{s^{2}+2 \xi \omega_{n} s+\omega_{n}^{2}}\)

 

The zero of this function is at \(s = 0.\)

The distance of the pole at \(s=-\xi \omega_{n}+j \sqrt{1-\xi^{2}} \omega_{n}\) to the excitation frequency point \(j\omega\) is denoted by \(d_{1}\) and the distance of the pole at \(s=-\xi \omega_{n}-j \sqrt{1-\xi^{2}} \omega_{n} \text { to } j \omega \text { is } d_{2}\)

The distance of zero at \(s = 0\) to \(j\omega\) is \(\omega\) itself. The pole angles \(q_{1}\) and \(q_{2}\) are also shown in the pole-zero diagram in Fig. 13.13-3.

The magnitude function then is \(2 x ω_{n}\) \(ω/(d_{1} d_{2})\) and the phase function is \((π/2)-(q_{1} + q_{2}).\)

The distances \(d_{1}\) and \(d_{2}\) are equal to \(ω_{n}\) at \(ω = 0\) and the sum of the angles \(θ_{1}\) and \(θ_{2}\) at that frequency is \(360°\). Therefore, the gain at zero frequency is \(0\) (due to the zero-distance of zero) and angle is \(90°.\) As \(ω → ∞\), all the three distances go to \(∞\) and hence magnitude goes to zero. \(θ_{1}\) and \(θ_{2}\) go to \(90°\) as \(ω → ∞\) and hence the phase angle of frequency response function goes to \(–90°\) as \(ω → ∞.\)

As \(\omega\) increases from \(0\), the distance \(d_{1}\) decreases and the distances \(d_{2}\) and \(d_{3}\) increase. Consider a pair of \(ω\) values equal to \(\left[\sqrt{\left(1-\xi^{2}\right)}-\xi\right] \omega_{n} \text { and }\left[\sqrt{\left(1-\xi^{2}\right)}+\xi\right] \omega_{n}\) two \(\omega\) values separated by \(±\) real part of the pole from the imaginary part of the pole.

The distance \(d_{1}\) undergoes a variation from \(\sqrt{2} \xi \omega_{n} \text { to } \xi \omega_{n}\) and again to \(\sqrt{2} \xi \omega_{n} \text { as } \omega\) varies from \(\left[\sqrt{\left(1-\xi^{2}\right)}-\xi\right] \omega_{n} \text { to }\left[\sqrt{\left(1-\xi^{2}\right)}+\xi\right] \omega_{n}\) passing through the point \(\left[\sqrt{\left(1-\xi^{2}\right)}\right] \omega_{n}\) The distances \(d_{2}\) and \(d_{3}\) also vary. However, if \(\xi << 1\), the variation in these two quantities will be negligible over this frequency range and approximation \(d_{2} \approx 2 \sqrt{\left(1-\xi^{2}\right)} \omega_{n} \text { and } d_{3} \approx \omega_{n}\) will be satisfactory.

Therefore, the magnitude of frequency response function will vary from \(2 \xi \omega_{n} \omega_{n} \div 2 \sqrt{2} \xi \omega_{n} \sqrt{\left(1-\xi^{2}\right)} \omega_{n} \approx 1 / \sqrt{2} \text { to } 2 \xi \omega_{n} \omega_{n} \div \xi \omega_{n} \times 2 \sqrt{\left(1-\xi^{2}\right)} \omega_{n} \approx 1\) and again to \(1 / \sqrt{2}\) as \(\omega\) varies from \(\left[\sqrt{\left(1-\xi^{2}\right)}-\xi\right] \omega_{n} \text { to }\left[\sqrt{\left(1-\xi^{2}\right)}+\xi\right] \omega_{n}\) passing through the point \(\left[\sqrt{\left(1-\xi^{2}\right)}\right] \omega_{n} \) The
imaginary part of poles can be taken as approximately \(\omega_{n}\) itself for \(\xi << 1\). Therefore, the maximum gain is \(1\) at \(\omega = \omega_{n}\) and gain goes through \(1 / \sqrt{2} \text { at } \omega_{1,2}=(1 \mp \xi) \omega_{n}\) The phase angle at \(\omega = \omega_{n}\) is zero. See Fig. 13.13-3.

The center frequency of the narrow band-pass function is seen to be \(≈ \omega_{n}\) and the bandwidth is \(≈ 2\xi\omega_{n}\). Thus the ratio of center frequency to bandwidth of a narrow band-pass second-order network function is \(1/2\xi\) or \(Q\) of the denominator polynomial.

Let us consider the band-reject function now.

\(H(s)=\frac{s^{2}+\omega_{n}^{2}}{s^{2}+2 \xi \omega_{n} s+\omega_{n}^{2}}\)

The poles are at \(s=-\xi \omega_{n} \pm j \sqrt{1-\xi^{2}} \omega_{n}\) and zeros are at \(s j n = ± \omega\) . The distance of the pole at \(s=-\xi \omega_{n}+j \sqrt{1-\xi^{2}} \omega_{n} \) to the excitation frequency point \(j\omega\) is denoted by \(d_{1}\) and the distance of the pole at \(s=-\xi \omega_{n}-j \sqrt{1-\xi^{2}} \omega_{n} \text { to } j \omega \text { is } d_{2}\) The distance of zero at \(s j n = ω\) to \(jω\) is \(d_{3}\) and the distance of zero at \(s j n = − \omega\) to \(j\omega\) is \(d_{4}\). The pole angles \(q_{1}\) and \(q_{2}\) are also shown in the pole-zero diagram in Fig. 13.13-4. The zero-angles are \(–90°\) and \(90°\) for all \(ω < ω_{n}\) and \(90°\) and \(90°\) for all \(ω > ω_{n}\). The gain is given by \(d_{3} d_{4}/d_{1}d_{2}\) and starts at \(1\) at \(ω = 0\) since \(d_{1} = d_{2}\) and \(d_{3} = d_{4}\). The gain goes to \(1\) as \(ω → ∞\) since \(d_{1} ≈ d_{2}\) and \(d_{3} ≈ d_{4}\) under that condition. The gain is zero at \(ω = ω_{n}\) since \(d_{3}\) is zero under that condition. Therefore, it is a band-reject function.

The pole-zero plots and frequency response plots are shown in Fig. 13.13-4. For \(ξ << 1\), it may be shown that the gain crosses \(1/√2\) level at \(ω_{1,2} = (1± ξ )ω_{n}\) and that the phase angles at those frequencies are \(–45°\) and \(45°\). The centre frequency of the narrow band-reject function is seen to be \(≈ ω_{n}\) and the bandwidth is \(≈ 2ξωn\). Thus the ratio of centre frequency to bandwidth of a narrow band-reject second order network function is \(1/2ξ\) or \(Q\) of the denominator polynomial.

The frequency response of higher order network functions can similarly be sketched. A higher order \(H(jω)\) can be expressed as the product of first-order factors and biquadratic factors.

The magnitude response for first-order factors and biquadratic factors may be sketched separately first and then multiplied together to get the magnitude response curve of \(H(jω).\) Phase curves will have to be added.

Poles on negative real axis contribute a monotonically decreasing magnitude response. Poles close to \(jω-\)axis render resonant peaks in magnitude response and zeros on \(jω-\)axis produce zero gain response at the excitation frequencies equal to the zero locations.

Thus graphical interpretation of frequency response function is a valuable aid to a circuit designer who wants to locate poles and zeros of a network function to tailor the frequency response function to meet design specifications.
