Guest Question.

A control system consists of a process having a transfer function \(G_{p}\), a measuring element H and a controller \(G_{c}\).
If \(G_{p}=(3s+1)^{-1}\exp (-0.5s)\) and H=4.8\((1.5s+1)^{-1}\) , determine, using the method of Ziegler and Nichols, the controller settings for P, PI, PID controllers.

Admin
Solution

The relevant block-diagram is shown in Figure 7n.

To use the Ziegler-Nichols rules, it is necessary to plot the open-loop Bode diagram without the controller. All other transfer functions are assumed to be unity.
\(G_{p}\): This consists of a DV lag of \(e^{-0.5s}\) and a first order system, 1/(3s + 1) With the AR plot, only the first-order system contributes:

LFA: AR = 1

HFA: The slope = -1, passing through AR = 1, \(\omega _{c}=1/\tau \) = 0.33 radians/minute.

With the \(\psi \) plot, for the first order part:

\(\psi =\tan^{-1}(-\omega \tau )=\tan^{-1}(-3\omega )\)

which gives:

\(\omega \) \(\psi \)
0 \(0^{\circ}\)
0.1 \(-17^{\circ}\)
0.2 \(-31^{\circ}\)
0.33 \(-48^{\circ}\)
0.8 \(-67^{\circ}\)
2 \(-81^{\circ}\)

With the DV lag plot:

\(\psi _{DV}=-\omega \tau _{DV}=-0.5\omega \) radians.

which gives:

\(\omega \) \(\psi \)
0.5 \(-14^{\circ}\)
1 \(-29^{\circ}\)
2 \(-57^{\circ}\)
3 \(-86^{\circ}\)
5 \(-143^{\circ}\)
8 \(-229^{\circ}\)

H (measuring element):

This is first order with a time constant of 1.5 min. and a steady-state gain of 4.8.

With the AR plot,

LFA: AR = 1

HFA: The slope is -1 and \(\omega_{c} \) = (1/1.5) = 0.67

With the \(\psi \) plot,

\(\psi=\tan^{-1}(-\omega \tau ) =\tan^{-1}(-1.5\omega )\)

which gives:

\(\omega \) \(\psi \)
0 \(-0^{\circ}\)
0.2 \(-17^{\circ}\)
0.5 \(-37^{\circ}\)
0.67 \(-45^{\circ}\)
1 \(-56^{\circ}\)
2 \(-72^{\circ}\)
\(\infty \) \(-90^{\circ}\)

The Bode diagrams are plotted for these in Figure 7o. The asymptotes on the AR plots are summed and the sums on the \(\psi \) plots are obtained by linear measurement.

From the overall \(\psi \) plots, at \(\psi =-180^{\circ}\) , \(\omega _{co}\) = 1.3 radians/min and the corresponding value of AR = 0.123. By the Ziegler–Nichols procedure, this means that:

\(K_{u}=\frac{1}{(0.123)}=8.13\)

Taking into account the steady-state gain of the measuring elements, \(K_{u}\) will be reduced by a factor of 4.8 as both together will give the total gain of the loop.

Hence: \(K_{u}=\left ( \frac{8.13}{4.8} \right )\approx 1.7\)

and: \(T_{u}=\left ( \frac{2\pi }{\omega _{co}} \right )=\frac{2\pi }{1.3}\approx 4.8\) min.

From the Ziegler–Nichols settings:

for P controller \(K_{c}\) = 0.5\(K_{u}\) = 0.85

PI \(K_{c}\) = 0.77 and \(\tau _{I}\) = 4 min.

PID \(K_{c}\) = 1.0, \(\tau _{I}\) = 2.4 min, \(\tau _{s}\) = 0.6 min
