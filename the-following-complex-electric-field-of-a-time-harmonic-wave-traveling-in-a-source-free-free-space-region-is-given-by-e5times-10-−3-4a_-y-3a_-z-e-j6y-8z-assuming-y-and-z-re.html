The following complex electric field of a time-harmonic wave traveling in a source free, free-space region is given by

\(E=5\times 10^{ −3 }(4â_{ y }+3â_{ z })e^{ j(6y-8z) }\)

Assuming y and z represent their respective distances in meters, determine the:

(a) Angle of wave travel (relative to the zaxis).

(b) Three phase constants of the wave along its oblique direction of travel, they axis, and the z axis (in radians permeter).

(c) Three wave lengths of the wave alongits oblique direction of travel, the y axis, and the z axis (in meters).

(d) Three phase velocities of the wave along the oblique direction of travel,the y axis, and the z axis (in metersper second).

(e) Three energy velocities of the wavealong the oblique direction of travel,the y axis, and the z axis (in metersper second).

(f) Frequency of the wave.

(g) Associated magnetic field.

Solution
\(\underline { E } =5\times 10^{ −3 }(4â_{ y }+3â_{ z })e^{ j(6y-8z) }\\ \underline { E } =5\times 10^{ −3 }(5)(\frac { 4 }{ 5 } â_{ y }+\frac { 3 }{ 5 } â_{ z })e^{ j10(0.6y-0.8z) }=25\times 10^{ −3 }(0.8â_{ y }+0.6â_{ z })e^{ j10(0.6y-0.8z) }\)

(a)

\(\underline { E } =25\times 10^{ −3 }(\cos { \theta } â_{ y }+\sin { \theta } â_{ z })e^{ j{ \beta }_{ 0 }(\sin { \theta } y-\cos { \theta } z) }\\ \cos { \theta } =0.8\Rightarrow \theta =\cos ^{ -1 }{ (0.8) } =36.87^{ \circ }\\ \sin { \theta } =0.6\Rightarrow \theta =\sin ^{ -1 }{ (0.6) } =36.87^{ \circ }\)

\(\underline { E } =25\times 10^{ −3 }(\cos { \theta } â_{ y }+\sin { \theta } â_{ z })e^{ j({ \beta }_{ y }y-{ \beta }_{ z }z) }\\ { \beta }_{ r }=10\quad rad/m\\ { \beta }_{ y }=6\quad rad/m\\ { \beta }_{ z }=8\quad rad/m\)

\({ \lambda }_{ r }=\frac { 2\pi }{ { \beta }_{ r } } =\frac { 2\pi }{ 10 } =0.6283m\quad \quad ;{ \lambda }_{ y }=\frac { 2\pi }{ { \beta }_{ y } } =\frac { 2\pi }{ 6 } =1.047m\quad \quad ;{ \lambda }_{ z }=\frac { 2\pi }{ { \beta }_{ z } } =\frac { 2\pi }{ 8 } =0.7854m\)
