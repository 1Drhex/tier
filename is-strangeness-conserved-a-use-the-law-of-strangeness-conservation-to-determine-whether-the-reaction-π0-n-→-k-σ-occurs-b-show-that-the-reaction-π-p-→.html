Guest Question.

Is Strangeness Conserved?

(A) \(\text {Use the law of strangeness conservation to determine whether the reaction } \pi^{0}+ n \rightarrow K ^{+}+\Sigma^{-} \text {occurs. }\)

(B) \(\text { Show that the reaction } \pi^{-}+ p \rightarrow \pi^{-}+\Sigma^{+} \text {does not conserve strangeness. }\)

Admin
Solution

Conceptualize We recognize that there are strange particles appearing in this reaction, so we see that we will need to investigate conservation of strangeness.

Categorize We use a conservation law developed in this section, so we categorize this example as a substitution problem.

Evaluate the strangeness for the left side of the reaction using Table 46.2:

S = 0 + 0 = 0

Evaluate the strangeness for the right side of the reaction:

S = +1 – 1 = 0

Therefore, strangeness is conserved and the reaction is allowed.

Evaluate the strangeness for the left side of the reaction:

S = 0 + 0 = 0

Evaluate the strangeness for the right side of the reaction:

S = 0 + (-1) = -1

Therefore, strangeness is not conserved.
