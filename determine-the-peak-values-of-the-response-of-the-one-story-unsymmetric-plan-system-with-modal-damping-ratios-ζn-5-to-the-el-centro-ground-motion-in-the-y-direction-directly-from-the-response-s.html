Guest Question.

Determine the peak values of the response of the one-story unsymmetric-plan system with modal damping ratios \(\zeta _{n} = 5\)% to the El Centro ground motion in the y-direction, directly from the response spectrum for this ground motion.

Admin
Solution

_Corresponding to the known \(T_{n}\) and \(\zeta _{n} = 5\)%, gives the ordinates \(D_{n}\) and \(A_{n}\). For\( T_{1} = 1.069 sec: D_{1} = 4.256 in. = 0.3547 ft\) and \(A_{1}/g = 0.381. For T_{2} = 0.9248 sec: D_{2}= 4.161 in. = 0.3468 ft \)and\( A_{2}/g = 0.497.\)

_ The peak values of roof displacement and rotation are obtained by specializing Eq. (13.9.1a) for the one-story system:
\(u_{yn}= \Gamma _{n} \phi _{yn} D_{n}\)
\(u_{\theta n} = \Gamma _{n} \phi _{\theta n} D_{n}\)(a)

Substituting numerical values for \(\Gamma _{n} , \phi _{yn}\), and \(\phi _{\theta n}\) in Eq. (a) with n = 1 gives the first-mode peak responses:

\(u_{y1} = (−0.974)(−0.5228)(0.3547) = 0.1806 ft = 2.168 in.\)

\(\frac {b}{2}u_{\theta 1} = \frac {30}{2} (−0.974)(0.0493)(0.3547) = −0.2555 ft = −3.065 in.\) (b)

where\( (b/2)u_{\theta 1} \)represents the lateral displacement at the edge of the plan due to floor rotation. Similarly, the second-mode peak responses are
\(u_{y2} = (−0.956)(−0.5131)(0.3468) = 0.1701 ft = 2.042 in.\)

\(\frac {b}{2}u_{\theta 2} =\frac {30}{2} (−0.956)(−0.0502)(0.3468) = 0.2497 ft = 2.999 in\). (c)

_ The peak values of \(f _{yn}\) and \(f_{\theta n}\), the lateral force and torque, are obtained by specializing Eq. (13.9.3) for this one-story frame:

\(f_{yn} = \Gamma _{n} m \phi _{yn} A_{n}\)
\(f_{\theta n} = \Gamma _{n} r^{2} m \phi _{\theta n} A_{n}\) (d)

By statics, the base shear and base torque are

\(V_{bn} = f_{yn} T_{bn} = f_{\theta n}\) (e)

Alternatively, Eq. (13.7.1) can be used for computing the peak modal response. For example, the modal static responses \(V^{st}_{bn}\) and \(M^{st}_{bn}\) are available . Substituting \(V^{st}_{b1} = 0.509m, T^{st}_{b1} = −5.203m, m = 60 kips/g,\) and \(A_{1} = 0.381g\) in Eq. (13.7.1) gives

\(V_{b1} = [0.509(60/g)]0.381g = 11.63 kips\)

\(T_{b1} = [−5.203(60/g)](0.381g) = −118.8 kip-ft\) (f)

Substituting \(V^{st}_{b2} = 0.491m, T^{st}_{b2} = 5.203m, and A_{2 }= 0.497g \)in Eq. (13.7.1) gives

\(V_{b2} = [0.491(60/g)]0.497g = 14.64 kips\)

\(T_{b2} = [5.203(60/g)](0.497g) = 155.2 kip-ft\) (g)

_ The peak lateral force for frame A is given by Eq. (13.9.4b) specialized for a one-story frame:

\(f_{An} = k_{A} \Gamma _{n} (\phi _{y n} + x_{A}\phi _{\theta n} )D_{n}\) (h)

Substituting \(k_{A} = 75 kips/ft, x_{a} = 1.5 ft,\) and numerical values for \(\Gamma _{n}, \phi _{y n} , \phi _{\theta n}\), and \(D_{n}\) gives

\(f_{A1} = 75(−0.974)[−0.5228 + 1.5(0.0493)]0.3547 = 11.63 kips\)

 

\(f_{A2} = 75(−0.956)[−0.5131 + 1.5(−0.0502)]0.3468 = 14.64 kips\)

The base shear in a one-story frame is equal to the lateral force; thus \(V_{bA1} = 11.63 kips V_{bA2} = 14.64 kips\) (i)
The lateral force for frame B is given by Eq. (13.9.4a) specialized for a one-story frame:

\(f_{Bn}= k_{B} \Gamma _{n} (-y _{B}\phi _{\theta n} )D_{n}\) ( j)

Substituting \(k_{B} = 40 kips/ft, y_{B}= 10 ft,\) and numerical values for \(\Gamma _{n} , \phi _{\theta n}\), and \(D_{n} \)gives

\(f_{B1} = 40(−0.974)[−10(0.0493)]0.3547 = 6.814 kips\) \(f_{B2} = 40(−0.956)[−10(−0.0502)]0.3468 = −6.662 kips\)

The corresponding base shears are
\(V_{bB1} = 6.814 kips\)

\(V_{bB2}= −6.662 kips\) (k)

_For this system with two modes, the ABSSUM, SRSS, and CQC rules, Eqs. (13.7.2)–(13.7.4), specialize to

\(r \leq\left|r_{1}\right|+\left|r_{2}\right| \quad r \simeq\left(r_{1}^{2}+r_{2}^{2}\right)^{1 / 2} \quad r \simeq\left(r_{1}^{2}+r_{2}^{2}+2 \rho_{12} r_{1} r_{2}\right)^{1 / 2} \)(l)

For this system,

\(\beta 12 = \omega _{1}/\omega _{2} = 5.878/6.794 = 0.865.\)

For this value of\( \beta 12\) and \(\zeta = 0.05,\) Eq. (13.7.10) gives \(\rho 12 = 0.322.\)
The results from Eq. (l) are summarized in Table E13.12b, wherein the peak values of total responses determined by RHA are also included. These were computed using the results , where \(D_{n} (t)\) and\( A_{n}(t)\) were computed by dynamic analysis of the nth-mode SDF system. As expected, the ABSSUM estimate is always larger than the RHA value. The SRSS estimate is better, but the CQC estimate is the best because it accounts for the cross-correlation term in the modal combination, which is significant in this example because the natural frequencies are close, a situation common for unsymmetric-plan systems.
