Guest Question.

Designing a biasing circuit for an NMOS amplifier

(a) Design the biasing circuit shown in Fig. 7.25(a) for an NMOS. The DC supply voltage is \( V_{ DD }=15 V \). The NMOS parameters are \( V_{ t }=1 V , K_{ n }=3.25 mA / V ^{2}, I_{ D (\max )}=10 mA , \text { and }\left|V_{ M }\right|=1 / \lambda=100 V \).

(b) Calculate the small-signal parameters of the NMOS: \( g_{ m } \text { and } r_{0} \).

(c) Use SPICE to verify your design. Assume \( K_{ p }=2 K_{ n } \text { for } W=L=1 \mu m \).

Admin
Solution

(a) Let us assume that \( v_{ DS }=V_{ DD } / 3=15 / 5=5 V \text {, and } i_{ D }=I_{ D (\max ) / 3}=10 mA / 3=3.33 mA \cdot R_{ D }=R_{ SR }= v_{ DS } / i_{ D }=5 /\left(3.33 \times 10^{-3}\right)=1.5 k \Omega \).
Substituting \( i_{ D }=3.33 mA \text { and } K_{ n }=3.25 mA / V ^{2} \text { into Eq. (7.10), } i_{ D }=K_{ n }\left(v_{ GS }-V_{ t }\right)^{2} \text {, gives } v_{ GS }=2.013 \) V or -0.013 V. For the NMOS, \( v_{ GS } \) must be greater than 1 V. Thus, the acceptable value is \( v_{ GS }=2.013 V \).

 

\(\begin{array}{rlr}i_{ D } & =K_{ n }\left[2\left(v_{ GS }-V_{ t }\right) v_{ DS }-v_{ DS }^{2}\right] & \text { for } v_{ GS }>V_{ t } \text { and } v_{ DS }<\left(v_{ GS }-V_{ t }\right) \\& =K_{ n }\left(v_{ GS }-V_{ t }\right)^{2} & \text { for } v_{ GS }>V_{ t } \text { and } v_{ DS } \geq\left(v_{ GS }-V_{ t }\right)\end{array}\)

(7.10)

Using KVL around the gate-to-source loop,

\( v_{ G }=v_{ SR }+v_{ GS }=5+2.013=7.013 V \)

 

\( \frac{R_{1}}{R_{2}}=\left(\frac{V_{ DD }}{v_{ G }}\right)-1=\frac{15}{7.013}-1=1.139 \)

 

Letting \( R_{2}=100 k \Omega, \text { we get } R_{1}=1.139 R_{2}=1.139 \times 100 k =113.9 k \Omega \).

 

(b) From Eq. (7.27), \( g_{ m }=2 K_{ n }\left(v_{ GS }-V_{ t }\right)=2 \times 3.25 m \times(2.013-1)=6.583 mA / V .\) From Eq. (7.26), \( r_{ o }=1 /\left(\lambda i_{ D }\right)=\mid V_{ M } \mid / i_{ D }=100 / 3.33 mA =30 k \Omega .\)

 

\( \frac{1}{r_{ o }}=\frac{d i_{ D }}{d v_{ DS }}=\frac{I_{ D }}{\left|V_{ M }\right|}=\lambda I_{ D } \quad \text { for all MOSFETs } \) (7.26)

 

\( g_{ m }=\frac{\delta i_{ D }}{\delta v_{ GS }}=2 K_{ n }\left(V_{ GS }-V_{ t }\right) \quad \text { for enhancement MOSFETs } \)

(7.27)

 

(c) The details of the DC bias calculations by the PSpice simulation are given here:

 

\(\begin{array}{ll}ID =3.35 E -03(3.33 mA ) & VGS =1.99 E +00(2 V ) & \text { VDS }=4.96 E +00(5 V ) \\GM =6.76 E -03(6.583 mA / V ) & GDS =3.19 E -05\left(1 / r_{ o }=1 / 30 k =33.3 \mu A / V \right)\end{array}\)

 

NOTE: The values obtained by hand calculations are shown in parentheses.
