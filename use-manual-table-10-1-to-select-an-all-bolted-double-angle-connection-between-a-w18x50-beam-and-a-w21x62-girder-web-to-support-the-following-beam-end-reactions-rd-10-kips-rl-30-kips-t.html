Use Manual Table 10-1 to select an all-bolted double-angle connection between a W18×50 beam and a W21×62 girder web to support the following beam end reactions:
\(\begin{array}{l}R_{D}=10 \text { kips } \\R_{L}=30 \text { kips }\end{array}\)
The beam top flange is coped 2-in. deep by 4-in. long, \(L_{e v}=11 / 4 \text { in., } L_{e h}=13 / 4 \text { in }\). (assumed to be 1 1/2 in. for calculation purposes to account for possible underrun in beam length)
Use 3/4-in. diameter ASTM A325-N bolts in standard holes.

Beam W18×50 ASTM A992 \(F_{y}=50 ksi \quad F_{u}=65 ksi\)
Girder W21×62 ASTM A992 \(F_{y}=50 ksi \quad F_{u}=65 ksi\)
