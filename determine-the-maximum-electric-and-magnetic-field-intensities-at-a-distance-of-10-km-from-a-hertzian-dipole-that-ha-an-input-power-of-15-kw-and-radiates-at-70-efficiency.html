Guest Question.

Determine the maximum electric and magnetic field intensities at a distance of \(10 (km)\) from a Hertzian dipole that ha an input power of \(15 (kW)\) and radiates at \(70\)% efficiency.

Admin
Solution

From Eqs (10-12) and (10-14):

 

\({ G }_{ D }\left( \theta ,\phi \right) =\frac { 4\pi { R }^{ 2 }{ \wp }_{ av } }{ { P }_{ r } }\)

 

Maximum \({ G }_{ D }\) at \({ \wp }_{ av }\) occur at \(\theta =\pi /2\)

 

\({ \wp }_{ av }=\frac { D{ P }_{ r } }{ 4\pi { R }^{ 2 } } =\frac { { E }_{ 0 }^{ 2 } }{ 2{ \eta }_{ 0 } }\)

 

\({ E }_{ 0 }^{ 2 }=\frac { { \eta }_{ 0 }D{ P }_{ r } }{ 2\pi { R }^{ 2 } } ;\quad D=1.5,\quad { P }_{ r }=0.70\times 15\times { 10 }^{ 3 }\left( W \right)\)

 

\(\rightarrow { E }_{ 0 }=0.0972\left( V/m \right) =97.2\left( mA/m \right)\)

 

\(\quad { H }_{ 0 }=\frac { { E }_{ 0 } }{ { \eta }_{ 0 } } =0.258\left( mA/m \right) \)

 

\(U={ R }^{ 2 }{ \wp }_{ av }\quad \left( W/sr \right) \quad \quad\quad\quad \quad \left( 10-12 \right)\)

 

\({ G }_{ D }\left( \theta ,\phi \right) =\frac { U\left( \theta ,\phi \right) }{ { P }_{ r }/4\pi } =\frac { 4\pi U\left( \theta ,\phi \right) }{ \oint { U } d\Omega } \quad \quad \left( 10-14 \right)\)
