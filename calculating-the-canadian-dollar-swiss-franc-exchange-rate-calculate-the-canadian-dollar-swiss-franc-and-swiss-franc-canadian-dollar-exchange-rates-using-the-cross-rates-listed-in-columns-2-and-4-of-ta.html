Calculating the Canadian Dollar/Swiss Franc Exchange Rate


Calculate the Canadian dollar/Swiss franc and Swiss Franc/Canadian dollar exchange rates using the cross rates listed in columns 2 and 4 of Table 16-3.

Solution

STEP 1: Formulate a Solution Strategy

Columns 2 and 4 of Table 16-3 provide the Canadian dollar/euro and euro/Swiss Franc rates, which can be used to determine the Canadian dollar/Swiss franc and Swiss franc/Canadian dollar exchange rates as follows:

 

\(\frac{\text { Canadian } \$}{€} \times \frac{€}{\text { Swiss franc }}=\frac{\text { Candian } \$}{\text { Swiss franc }}\)

 

Since the Swiss franc/Canadian dollar is the reciprocal of the Canadian dollar/Swiss franc shown above, it can then be calculated as the reciprocal of the direct quote.

 

STEP 2: Crunch the Numbers


Substituting into the equation, we compute the euro/pound exchange rate:

 

\(\frac{\text { Canadian } \$}{€} \times \frac{€}{\text { Swiss franc }}=1.4013 \times 0.9549=\frac{1.3381 \text { Candian } \$}{\text { Swiss franc }}\)

 

Thus, the Swiss Franc/Canadian dollar exchange rate is:

 

\(\frac{1}{1.3381}=\frac{0.7473 \text { Swiss francs }}{\text { Canadian } \$}\)

 

STEP 3: Analyze Your Results

The Canadian dollar/Swiss franc exchange rate is 1.3381 Canadian dollars per Swiss franc, and the Swiss franc/Canadian dollar exchange rate is 0.7473 Swiss francs per Canadian dollar. You’ll notice that these rates are the same exchange rates as those given in Table 16-3.
