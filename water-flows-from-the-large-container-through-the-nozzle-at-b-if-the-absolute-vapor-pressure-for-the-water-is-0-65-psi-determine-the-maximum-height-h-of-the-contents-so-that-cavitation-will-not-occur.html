Guest Question.

Water flows from the large container through the nozzle at B. If the absolute vapor pressure for the water is 0.65 psi, determine the maximum height h of the contents so that cavitation will not occur at B.

Admin
Solution

Bernoulli Equation. Since the water is discharged through B from a large source, \(V_A \cong 0. \) Here, A and C are open to the atmosphere,\( P_A = P_C = 0 \). Also, at B, the pressure is required to be equal to the vapor pressure (cavitation). Then,

\((P_B)_{abs} = (P_B)_g + P_{atm} \)

 

\(0.65 psi = (P_B)_g + 14.7 psi \)

 

\((P_B)_g = (-14.05 \frac{Ib}{in^2}) (\frac{12 in}{1 ft})^2 = -2023.2\frac{Ib}{ft^2} \)

If we set the datum at C, \(z_C = 0, z_B = 0.5 ft, \) and \( z_A = 0.5 ft + h \). From A to B,

\(\frac{P_A}{\rho } + \frac{V_A^2}{2} + gz_A = \frac{P_B}{\rho } + \frac{V_B^2}{2} + gz_B \)

 

\(0 + 0 + (32.2 ft/s^2)(0.5 ft + h) = \frac{-2023.2\frac{Ib}{ft^2}}{\frac{62.4 lb/ft^3}{32.4 lb/ft^3}} + \frac{V_B^2}{2} + (32.2 ft/s^2)(0.5 ft) \)

 

\(V_B^2 = 64.4 h + 2088.05 \)(1)

From A to C,

\(\frac{P_A}{\rho } + \frac{V_A^2}{2} + gz_A = \frac{P_C}{\rho } + \frac{V_C^2}{2} + gz_C \)

 

\(0 + 0 + (32.2 ft/s^2)(0.5 ft + h) = 0 + \frac{V_C^2}{2} + 0 \)

 

\(V_C^2 = 64.4 h + 32.2 \) (2)

Continuity Equation. Consider the water within the container of the control volume.

\(\frac{\partial }{\partial t} \int_{cv} \rho dV + \int_{cs} \rho V \cdot dA =0 \)

 

\(0 – V_BA_B + V_CA_C = 0\)

 

\(-VB[\pi (0.1 ft)^2] + V_C[\pi (0.15 ft)^2] = 0 \)

 

\(V_B = 2.25 V_C \) (3)

Solving Eqs. (1) through (3) yields

\(V_B = 50.6 ft/s \)

 

\(V_C = 22.5 ft/s \)

h = 7.36 ft
