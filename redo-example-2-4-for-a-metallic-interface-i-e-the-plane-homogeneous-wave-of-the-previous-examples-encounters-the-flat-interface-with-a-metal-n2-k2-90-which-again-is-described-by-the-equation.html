Guest Question.

Redo for a metallic interface, i.e., the plane homogeneous wave of the previous examples encounters the flat interface with a metal \((n_{2} = k_{2} = 90)\), which again is described by the equation z = 0. Calculate the incidence, reflection and refraction angles. What fraction of energy of the wave is reflected, how much is transmitted?

Admin
Solution

If \(n_{2}\) and \(K_{2}\) are much larger than \(n_{1}\) it follows from equations

\(P^{2} = \left(\frac{w^{\prime }_{t} \cos \theta _{2} }{\eta _{0} } \right) ^{2} =\frac{1}{2} \left[ \sqrt{(n^{2}_{2}-k^{2}_{2} -n^{2}_{1} \sin ^{2}\theta _{1} )^{2}+4 n^{2}_{2}k^{2}_{2}} +(n^{2}_{2}-k^{2}_{2} -n^{2}_{1} \sin ^{2}\theta _{1}) \right]\) ,

\(q^{2} =\left(\frac{w^{\prime \prime }_{t} }{\eta _{0} } \right) ^{2} = \frac{1}{2} \left[ \sqrt{(n^{2}_{2}-k^{2}_{2} -n^{2}_{1} \sin ^{2}\theta _{1} )^{2}+4 n^{2}_{2}k^{2}_{2}} -(n^{2}_{2}-k^{2}_{2} -n^{2}_{1} \sin ^{2}\theta _{1}) \right]\) ,

that \(P\approx n_{2}\) and \(q\approx K_{2}\) and, from equation

\(w^{\prime }_{i} \sin \theta _{1} =\eta _{0} n_{1} \sin \theta _{1} = w^{\prime }_{t} \sin \theta _{2}\).

\(n_{1} \sin \theta _{1}\approx n_{2} \tan \theta _{2} \approx n_{2}\sin \theta _{2}\)

(i.e., as long as \(n_{1}\gg n_{2}\),Snell’s law between dielectrics holds) and it follows that \(\theta _{2}=1.02^{\circ }\) . With \(n_{2} =K_{2}\) equations

\(\rho _{\shortparallel } =\widetilde{r_{\shortparallel } } \widetilde{r_{\shortparallel } } ^{*} =\frac{(p-n_{1}\sin \theta _{1}\tan \theta _{1} )^{2}+q^{2} }{(p+n_{1}\sin \theta _{1}\tan \theta _{1} )^{2}+q^{2}}\rho _{\bot }\) ,

\(\rho _{\bot } =\widetilde{r_{\bot } } \widetilde{r_{\bot } } ^{*} =\frac{(n_{1} \cos \theta _{1}-p )^{2} +q^{2} }{(n_{1} \cos \theta _{1}+p )^{2} +q^{2}}\) .

reduce to

\(\rho _{\bot } =\frac{(n_{1} \cos \theta _{1}- n_{2})^{2}+n_{2}^{2} }{(n_{1} \cos \theta _{1}+ n_{2})^{2}+n_{2}^{2} } =\frac{(1.2-90)^{2} +90^{2} }{(1.2+90)^{2} +90^{2}} =0.9737\) ,

\(\rho _{\shortparallel } = \frac{(n_{2}-n_{1} \sin \theta _{1}\tan \theta _{1})^{2} +n_{2}^{2}}{(n_{2}+n_{1} \sin \theta _{1}\tan \theta _{1})^{2} +n_{2}^{2}} \rho _{\bot } =\frac{(90-2\times 0.8^{2}/0.6 )^{2}+90^{2} }{(90+2\times 0.8^{2}/0.6 )^{2}+90^{2}} \times 0.9737=0.9286\),

and the total reflectivity is again evaluated from equation

\(\rho =\frac{E_{i\shortparallel } E_{i\shortparallel }^{*} \rho _{\shortparallel }+E_{i\bot } E_{i\bot }^{*}\rho _{\bot } }{E_{i\shortparallel } E_{i\shortparallel }^{*}+E_{i\bot } E_{i\bot }^{*}}\).

as

\(\rho =\frac{E_{i\shortparallel } E_{i\shortparallel }^{*} \rho _{\shortparallel }+E_{i\bot } E_{i\bot }^{*}\rho _{\bot } }{E_{i\shortparallel } E_{i\shortparallel }^{*}+E_{i\bot } E_{i\bot }^{*}}=\frac{125\times 0.9286+29\times 0.9737}{154} =0.9371\) .

Thus, nearly 94% of the radiation is being reflected (and even more would have been reflected if the metal was surrounded by air with \(n\approx 1\)), and only 6% is transmitted into the metal, where it undergoes total attenuation after a very short distance because of the large value of \(K_{2}\): Equation

\(\overline{S} =\frac{1}{2\nu \mu } \Re \left\{E_{c}\times (W^{*}\times E_{c}^{*} ) \right\} = \frac{1}{2\nu \mu } \Re \left\{W^{*}(E_{c} .E_{c}^{*} ) \right\}=\frac{n}{2 c_{0} \mu } \left|E_{0} \right| ^{2} e^{-4\pi \eta _{0} kz} \widehat{s}\) .

shows that the transmission reaches its 1/e value at

\(4\pi \eta _{0}k_{2} z=1\) , or \(z=1/(4\pi \times 2500\times 90)=3.5\times 10^{-7} cm =0.0035 \mu m\) .
