Guest Question.

For the simply supported beam subjected to the loading shown,
(a) Derive equations for the shear force V and the bending moment M for any location in the beam. (Place the origin at point A.)
(b) Plot the shear-force and bending-moment diagrams for the beam using the derived functions.
(c) Report the maximum positive bending moment, the maximum negative bending moment, and their respective locations.

Admin
Solution

Beam equilibrium:

\(\begin{gathered}\Sigma M_{A}=250 kip – ft -(7 kips / ft )(25 ft )(12.5 ft ) \\+C_{y}(17 ft )=0 \\\therefore C_{y}=113.97 kips \\\Sigma F_{y}=A_{y}+C_{y}-(7 kips / ft )(25 ft ) \\=A_{y}+113.97 kips -(7 kips / ft )(25 ft )=0 \\\therefore A_{y}=61.03 \text { kips }\end{gathered}\)

 

Section a-a:
For the interval \(0 \leq x<13 ft\):

\(\begin{aligned}\Sigma F_{y}=A_{y} &-(7 kips / ft ) x-V=61.03 kips -(7 kips / ft ) x-V=0 \\& \therefore V=-7 x+61.03 kips\end{aligned}\)

 

\(\begin{aligned}\Sigma M_{a-a} &=-A_{y} x+(7 kips / ft ) x\left(\frac{x}{2}\right)+M \\&=(-61.03 kips ) x+(7 kips / ft ) x\left(\frac{x}{2}\right)+M=0 \\& \therefore M=-3.5 x^{2}+61.03 x kip – ft\end{aligned}\)

 

Section b-b:
For the interval \(13 ft \leq x<17 ft\):

\(\begin{aligned}\Sigma F_{y}=& A_{y}-(7 kips / ft ) x-V=61.03 kips -(7 kips / ft ) x-V=0 \\& \therefore V=-7 x+61.03 kips\end{aligned}\)

 

\(\begin{aligned}\Sigma M_{b-b} &=-A_{y} x+(7 kips / ft ) x\left(\frac{x}{2}\right)+250 kip – ft +M \\&=(-61.03 kips ) x+(7 kips / ft ) x\left(\frac{x}{2}\right)+250 kip – ft +M=0 \\& \therefore M=-3.5 x^{2}+61.03 x-250 kip – ft\end{aligned}\)

Section c-c:
For the interval \(17 ft \leq x<25 ft\):

\(\begin{aligned}\Sigma F_{y}=& A_{y}+C_{y}-(7 kips / ft ) x-V \\=& 61.03 kips +113.97 kips -(7 \text { kips } / ft ) x-V=0 \\& \therefore V=-7 x+175 kips\end{aligned}\)

 

\(\begin{aligned}&\Sigma M_{c-c}=-A_{y} x-C_{y}(x-17 ft )+(7 kips / ft ) x\left(\frac{x}{2}\right) \\&\quad+250 kip – ft +M \\&=-(175 kips ) x+(113.97 kips )(17 ft )+(7 kips / ft ) x\left(\frac{x}{2}\right) \\&\quad+250 kip – ft +M=0 \\&\therefore M=-3.5 x^{2}+175 x-2,187.5 kip – ft\end{aligned}\)
