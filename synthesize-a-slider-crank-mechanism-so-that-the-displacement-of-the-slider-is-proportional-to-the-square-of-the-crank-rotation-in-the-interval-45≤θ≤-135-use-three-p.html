Guest Question.

Synthesize a slider crank mechanism so that the displacement of the slider is proportional to the square of the crank rotation in the interval \(45^{\circ} \leq \theta \leq 135^{\circ}\). Use three precision points with Chebyshev’s spacing

Admin
Solution

Given. \(\theta_{ S }=45^{\circ} ; \theta_{ F }=135^{\circ}\)

 

First of all, let us find the three precision points \(\text { (i.e. } \left.x_{1}, x_{2} \text { and } x_{3}\right)\). We know that

 

\(x_{j}=\frac{1}{2}\left(x_{ S }+x_{ F }\right)-\frac{1}{2}\left(x_{ F }-x_{ S }\right) \cos \left[\frac{\pi(2 j-1)}{2 n}\right]\) ; where j = 1, 2 and 3

Assuming the starting displacement of the slider \(\left(s_{ S }\right)=100\) mm and final displacement of the slider \(\left(s_{ F }\right)=30 mm\). It may be noted that for the crank rotating in anticlockwise direction, the final displacement will be less than the starting displacement.

\(\therefore \) \(x_{1}=\frac{1}{2}(100+30)-\frac{1}{2}(30-100) \cos \left[\frac{\pi(2 \times 1-1)}{2 \times 3}\right]=95.3 mm\) \(\ldots\left(\because \quad x_{ S }=s_{ S } ; x_{ F }=s_{ F } \text { and } n=3\right)\)

 

\(x_{2}=\frac{1}{2}(100+30)-\frac{1}{2}(30-100) \cos \left[\frac{\pi(2 \times 2-1)}{2 \times 3}\right]=65 mm\)

 

and \(x_{3}=\frac{1}{2}(100+30)-\frac{1}{2}(30-100) \cos \left[\frac{\pi(2 \times 3-1)}{2 \times 3}\right]=34.7 mm\)

The corresponding three values of θ are given by

\(\theta_{j}=\theta_{ S }+\frac{\theta_{ F }-\theta_{ S }}{x_{ F }-x_{ S }}\left(x_{j}-x_{ S }\right) ; j=1,2, \text { and } 3\)

 

\(\therefore \) \(\theta_{1}=45+\frac{135-45}{30-100}(95.3-100)=51.04^{\circ}\)

 

\(\theta_{2}=45+\frac{135-45}{30-100}(65-100)=90^{\circ}\)

 

and \(\theta_{3}=45+\frac{135-45}{30-100}(34.7-100)=128.96^{\circ}\)

Since it is given that the displacement of the slider (s) is proportional to the square of the crank rotation \((\theta)\), therefore, for the displacement from initial position \(\left(s_{ S }\right)\) to s when crank rotates from initial position \(\left(\theta_{ S }\right) \text { to } \theta\) , we have

 

\(s-s_{ S }=C\left(\theta-\theta_{ S }\right)^{2}\) \(\ldots(\theta \text { is expressed in degrees) }\)

 

\(\therefore \) \(C=\frac{s-s_{ S }}{\left(\theta-\theta_{ S }\right)^{2}}=\frac{30-100}{(135-45)^{2}}=\frac{-7}{810}\) \(\text { (Taking } s=s_{ F } ; \text { and } \theta=\theta_{ F } \text { ) }\)

 

Now the three positions for the slider displacement (s) corresponding to the three positions of the crank angle \((\theta)\) are given by

\(s_{1}=s_{ S }+C\left(\theta_{1}-\theta_{ S }\right)^{2}=100-\frac{7}{810}(51.4-45)^{2}=99.7 mm\)

 

\(s_{2}=s_{ S }+C\left(\theta_{2}-\theta_{ S }\right)^{2}=100-\frac{7}{810}(90-45)^{2}=82.5 mm\)

 

\(s_{3}=s_{ S }+C\left(\theta_{3}-\theta_{ S }\right)^{2}=100-\frac{7}{810}(128.96-45)^{2}=39.08 mm\)

 

Now the three equations relating the \(\left(\theta_{1}, s_{1}\right),\left(\theta_{2}, s_{2}\right) \text { and }\left(\theta_{3}, s_{3}\right)\) are written as

 

\(k_{1} \times 99.7 \cos 51.04^{\circ}+k_{2} \sin 51.04^{\circ}-k_{3}=(99.7)^{2}\)

 

or \(62.7 k_{1}+0.7776 k_{2}-k_{3}=9940\) …(i)

 

Similarly, \(k_{1} \times 82.5 \cos 90^{\circ}+k_{2} \sin 90^{\circ}-k_{3}=(82.5)^{2}\)

 

or \(k_{2}-k_{3}=6806\) …(ii)

 

and \(k_{1} \times 39.08 \cos 128.96^{\circ}+k_{2} \sin 128.96^{\circ}-k_{3}=(39.08)^{2}\)

 

or \(-24.57 k_{1}+0.776 k_{2}-k_{3}=1527\) …(iii)

 

The equations (i), (ii) and (iii) are three simultaneous equations in three unknowns \(k_{1}, k_{2}\) and \(k_{3}\). On solving, we get

 

\(k_{1}=96.4 ; \quad k_{2}=13084 ; \text { and } k_{3}=6278\)

 

We know that \(k_{1}=2 a \text { , or } \quad a=k_{1} / 2=96.4 / 2=48.2 mm\)

 

\(k_{2}=2 a . c \text { or } c=k_{2} / 2 a=13084 / 2 \times 48.2=135.7 mm\)

 

and \(k_{3}=a^{2}-b^{2}+c^{2}\)

 

\(b^{2}=a^{2}+c^{2}-k_{3}=(48.2)^{2}+(135.7)^{2}-6278=14460\)

 

or b = 120.2 mm
