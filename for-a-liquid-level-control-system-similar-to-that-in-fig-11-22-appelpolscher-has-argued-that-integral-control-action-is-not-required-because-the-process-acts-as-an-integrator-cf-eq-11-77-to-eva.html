Guest Question.

For a liquid-level control system similar to that in Fig. 11.22, Appelpolscher has argued that integral control action is not required because the process acts as an integrator (cf. Eq. 11-77). To evaluate his assertion, determine whether proportional-only control will eliminate offset for step changes in (a) set point and (b) disturbance variable.

\(\frac{H^{\prime}(s)}{Q_{i}^{\prime}(s)}=\frac{1}{A s}\) (11-77)

Admin
Solution

For proportional controller, \(G_{c}(s)=K_{c}\)

Assume that the level transmitter and the control valve have negligible dynamics. Then,

\(\begin{aligned}&G_{m}(s)=K_{m} \\&G_{v}(s)=K_{v}\end{aligned}\)

The block diagram for this control system is the same as in Fig.11.8. Hence Eqs. 11-26 and 11-29 can be used for closed-loop responses to setpoint and load changes, respectively.

\(\frac{Y}{Y_{s p}}=\frac{K_{m} G_{c} G_{v} G_{p}}{1+G_{c} G_{v} G_{p} G_{m}}\) (11-26)

\(\frac{Y}{D}=\frac{G_{d}}{1+G_{c} G_{v} G_{p} G_{m}}\) (11-29)

The transfer functions \(G_{p}(s)\) and \(G_{d}(s)\) are as given in Eqs. 11-66 and \(11-67\), respectively.

\(\frac{H^{\prime}(s)}{Q_{3}^{\prime}(s)}=G_{p}(s)=-\frac{1}{A s}\) (11-66)

\(\frac{H^{\prime}(s)}{Q_{1}^{\prime}(s)}=G_{d}(s)=\frac{1}{A s}\) (11-67)

a) Substituting for \(G_{c}, G_{m}, G_{v}\), and \(G_{p}\) into Eq. 11-26 gives

\(\frac{Y}{Y_{s p}}=\frac{K_{m} K_{c} K_{v}\left(-\frac{1}{A s}\right)}{1+K_{c} K_{v}\left(-\frac{1}{A s}\right) K_{m}}=\frac{1}{\tau s+1}\)

where \(\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\tau=-\frac{A}{K_{c} K_{v} K_{m}}\) (1)

For a step change in the setpoint, \(Y_{s p}(s)=M / s\)

\(Y(t \rightarrow \infty)=\underset{s\rightarrow 0}{\lim } s Y(s)=\underset{s\rightarrow 0}{\lim } s\left[\frac{M / s}{\tau s+1}\right]=M\)

Offset \(=Y_{s p}(t \rightarrow \infty)-Y(t \rightarrow \infty)=M-M=0\)

b) Substituting for \(G_{c}, G_{m}, G_{v}, G_{p}\), and \(G_{d}\) into \((11-29)\) gives

\(\frac{Y(s)}{D(s)}=\frac{\left(\frac{1}{A s}\right)}{1+K_{c} K_{v}\left(-\frac{1}{A s}\right) K_{m}}=\frac{\left(\frac{-1}{K_{c} K_{v} K_{m}}\right)}{\tau s+1}\)

where \(\tau\) is given by Eq. \(1 .\)

For a step change in the disturbance, \(D(s)=M / s\)

\(\begin{aligned}&Y(t \rightarrow \infty)=\underset{s\rightarrow 0}{\lim } s Y(s)=\lim _{s \rightarrow 0} s\left[\frac{-M /\left(K_{c} K_{v} K_{m}\right)}{s(\tau s+1)}\right]=\frac{-M}{K_{c} K_{v} K_{m}} \\&\text { Offset }=Y_{s p}(t \rightarrow \infty)-Y(t \rightarrow \infty)=0-\left(\frac{-M}{K_{c} K_{v} K_{m}}\right) \neq 0\end{aligned}\)

Hence, offset is not eliminated for a step change in disturbance.
