Guest Question.

The level in a tank responds as a first-order system to changes in its inlet flow. The data shown below were gathered after the inlet flow was increased quickly from \(1.5\) to \(4.8 gal / min\).

(a) Determine the transfer function by estimating the time constant using one of the methods of Section 7.2. Be sure to use deviation variables and include units for the model parameters.

(b) Repeat part (a) using nonlinear regression (e.g., Excel) and the liquid level data.

(c) Graphically compare the two model responses with the data. Which model is superior? (Justify your answer)

Solution

(a) \(63 \%\) response method

From inspection of the data, it is obvious that there is no time delay in the system \((\theta=0)\).

Time constant \(\tau\) is estimated by the \(63 \%\) response method:

\(\begin{aligned}&h^{\prime}(\tau)=0.63\left(h_{s s}^{\prime}-h_{0}^{\prime}\right)=0.63^{*} 20.3=12.78 f t \\&h(\tau)=12.78+10.4=23.18 f t\end{aligned}\)

From inspection at the data, \(\tau \approx 270 min\).

The process gain is calculated as:

\(K=\frac{h_{s s}^{\prime}-h_{0}^{\prime}}{q-q_{0}}=\frac{20.3-0}{(4.8-1.5) \times 0.1337}=\frac{20.3}{0.4412}=46 min / ft ^{2}\)

The estimated process model is:

\(\frac{H^{\prime}(s)}{Q^{\prime}(s)}=\frac{K}{\tau s+1}=\frac{46}{2.70 s+1}\)

(b) Nonlinear regression

By using deviation variables, the first order tank can be expressed as

\(\frac{H^{\prime}(s)}{Q^{\prime}(s)}=\frac{K}{\tau s+1}\)

The inlet flow rate is quickly changed from \(1.5\) gallon/min to \(4.8\) gallon/min so it is a step change, \(Q\) ‘(s) \(3.3 / s\) :

\(H^{\prime}(s)=\frac{K}{\tau s+1} Q^{\prime}(s)=\frac{K}{\tau s+1} \frac{(4.8-1.5) \times 0.1337}{s}=\frac{K}{\tau s+1} \frac{0.4412}{s}\)

Apply the inverse Laplace transform:

\(h^{\prime}(t)=0.4412 K\left(1-e^{-t / \tau}\right)\)

By using EXCEL, the estimated model is:

\(\frac{H^{\prime}(s)}{Q^{\prime}(s)}=\frac{K}{\tau s+1}=\frac{46.31}{2.65 s+1}\)

A comparison of the data and the two models is shown in Fig.S7.8.

The Sum of Squared Errors for the two models are:

\(\operatorname{SSE}(63.2 \%)=0.75\)

\(\text { SSE }( NR )=0.43\)

As indicated in Fig.S7.8, both methods fit the data well. The NR model is preferred due to its smaller SSE value.
