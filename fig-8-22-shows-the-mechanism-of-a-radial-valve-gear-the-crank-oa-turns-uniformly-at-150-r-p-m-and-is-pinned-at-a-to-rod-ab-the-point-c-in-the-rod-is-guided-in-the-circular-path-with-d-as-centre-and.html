Guest Question.

Fig. 8.22 shows the mechanism of a radial valve gear. The crank OA turns uniformly at 150 r.p.m and is pinned at A to rod AB. The point C in the rod is guided in the circular path with D as centre and DC as radius. The dimensions of various links are:
OA = 150 mm ; AB = 550 mm ; AC = 450 mm ; DC = 500 mm ; BE = 350 mm.
Determine velocity and acceleration of the ram E for the given position of the mechanism.

Admin
Solution

Given : \(N_{ AO }=150 \text { r.p.m. or } \omega_{ AO }=2 \pi \times 150 / 60=15.71 rad / s\) ; OA = 150 mm = 0.15 m; A B = 550 mm = 0.55 m ; AC = 450 mm = 0.45 m ; DC = 500 mm = 0.5 m ; BE = 350 mm = 0.35 m

We know that linear velocity of A with respect to O or velocity of A,

\(v_{ AO }=v_{ A }=\omega_{ AO } \times O A=15.71 \times 0.15=2.36 m / s\) …(Perpendicular to OA)

 

Velocity of the ram E

First of all draw the space diagram, as shown in Fig. (a), to some suitable scale. Now the velocity diagram, as shown in Fig. (b), is drawn as discussed below:

1. Since O and D are fixed points, therefore these points are marked as one point in the velocity diagram. Draw vector oa perpendicular to OA, to some suitable scale, to represent the velocity of A with respect to O or simply velocity of A, such that

\(\text { vector } o a=v_{ AO }=v_{ A }=2.36 m / s\)

2. From point a, draw vector ac perpendicular to AC to represent the velocity of C with respect to A (i.e. \(v_{ CA }\)), and from point d draw vector dc perpendicular to DC to represent the velocity of C with respect to D or simply velocity of C (i.e. \(v_{ CD } \text { or } v_{ C }\)). The vectors ac and dc intersect at c.

3. Since the point B lies on AC produced, therefore divide vector ac at b in the same ratio as B divides AC in the space diagram. In other words ac:cb = AC:CB. Join ob. The vector ob represents the velocity of B (i.e. \(v_{ B }\))

4. From point b, draw vector be perpendicular to be to represent the velocity of E with respect to B (i.e. \(v_{ EB }\)), and from point o draw vector oe parallel to the path of motion of the ram E (which is horizontal) to represent the velocity of the ram E. The vectors be and oe intersect at e.

By measurement, we find that velocity of C with respect to A,

\(v_{ CA }=\text { vector } a c=0.53 m / s\)

Velocity of C with respect to D,

\(v_{ CD }=v_{ C }=\text { vector } d c=1.7 m / s\)

Velocity of E with respect to B,

\(v_{ EB }=\text { vector } b e=1.93 m / s\)

and velocity of the ram E, \(v_{ E }=\text { vector } o e=1.05 m / s\)

 

Acceleration of the ram E

We know that the radial component of the acceleration of A with respect to O or the acceleration of A,

\(a_{ AO }^{r}=a_{ A }=\frac{v_{ AO }^{2}}{O A}=\frac{(2.36)^{2}}{0.15}=37.13 m / s ^{2}\)

Radial component of the acceleration of C with respect to A,

\(a_{ CA }^{r}=\frac{v_{ CA }^{2}}{O A}=\frac{(0.53)^{2}}{0.45}=0.624 m / s ^{2}\)

Radial component of the acceleration of C with respect to D,

\(a_{ CD }^{r}=\frac{v_{ CD }^{2}}{D C}=\frac{(1.7)^{2}}{0.5}=5.78 m / s ^{2}\)

Radial component of the acceleration of E with respect to B,

\(a_{ EB }^{r}=\frac{v_{ EB }^{2}}{B E}=\frac{(1.93)^{2}}{0.35}=10.64 m / s ^{2}\)

The acceleration diagram, as shown in Fig. (c), is drawn as discussed below:

1. Since O and D are fixed points, therefore these points are marked as one point in the acceleration diagram. Draw vector \(o^{\prime} a^{\prime}\) parallel to OA, to some suitable scale, to represent the radial component of the acceleration of A with respect to O or simply the acceleration of A, such that

\(\text { vector } o^{\prime} a^{\prime}=a_{ AO }^{r}=a_{ A }=37.13 m / s ^{2}\)

2. From point d’, draw vector d’x parallel to DC to represent the radial component of the acceleration of C with respect to D, such that

\(\text { vector } d^{\prime} x=a_{ CD }^{r}=5.78 m / s ^{2}\)

3. From point x, draw vector xc’ perpendicular to DC to represent the tangential component of the acceleration of C with respect to D (i.e. \( a_{ CD }^{t}\) ) whose magnitude is yet unknown.

4. Now from point a’, draw vector a’y parallel to AC to represent the radial component of the acceleration of C with respect to A, such that

\(\text { vector } a^{\prime} y=a_{ CA }^{r}=0.624 m / s ^{2}\)

5. From point y, draw vector yc’ perpendicular to AC to represent the tangential component of acceleration of C with respect to A (i.e \(a_{ CA }^{t}\)) . The vectors xc’ and yc’ intersect at c’.

6. Join \(a^{\prime} c^{\prime}\). The vector \(a^{\prime} c^{\prime}\) represents the acceleration of C with respect to A (i.e. \(a_{ CA }\)).

7. Since the point B lies on A C produced, therefore divide vector \(a^{\prime} c^{\prime}\) at \(b^{\prime}\) in the same ratio as B divides A C in the space diagram. In other words,\(a^{\prime} c^{\prime}: c^{\prime} b^{\prime}\) = AC : CB.

8. From point b’, draw vector b’ z parallel to BE to represent the radial component of the acceleration of E with respect to B, such that

\(\text { vector } b^{\prime} z=a_{ EB }^{r}=10.64 m / s ^{2}\)

9. From point z, draw vector ze’ perpendicular to BE to represent the tangential component of the acceleration of E with respect to B (i.e.\(a_{EB}^{t}\) ) whose magnitude is yet unknown.

10. From point o’, draw vector o’e’ parallel to the path of motion of E (which is horizontal) to represent the acceleration of the ram E. The vectors ze’ and o’e’ intersect at e’.

By measurement, we find that the acceleration of the ram E,

\(a_{E}\) = vector 3.1 m/\(s^{2}\)
