Guest Question.

A surge tank system is to be installed as part of a pilot plant facility. The initial proposal calls for the configuration shown in Fig. 4.4. Each tank is \(5 ft\) high and \(3 ft\) in diameter. The design flow rate is \(q_{i}=100 gal / min\). It has been suggested that an improved design will result if the two-tank system is replaced by a single tank that is \(4 ft\) in diameter and has the same total volume (i.e., \(\left.V=V_{1}+V_{2}\right)\).

(a) Which surge system (original or modified) can handle larger step disturbances in \(q_{i}\) ? Justify your answer.

(b) Which system provides the best damping of step disturbances in \(q_{i}\) ? (Justify your answer).

In your analysis you may assume that:

(i) The valves on the exit lines act as linear resistances.

(ii) The valves are adjusted so that each tank is half full at the nominal design condition of \(q_{i}=100 gal / min\).

(c) Use computer simulation to verify your answers in (a) and (b).

Admin
Solution

a) For the original system,

\(\begin{aligned}&A_{1} \frac{d h_{1}}{d t}=C q_{i}-\frac{h_{1}}{R_{1}} \\&A_{2} \frac{d h_{2}}{d t}=\frac{h_{1}}{R_{1}}-\frac{h_{2}}{R_{2}}\end{aligned}\)

\(\text { where } A_{1}=A_{2}=\pi(3)^{2} / 4=7.07 ft ^{2}\)

\(\begin{aligned}&C=0.1337 \frac{ ft ^{3} / min }{ gpm } \\&R_{1}=R_{2}=\frac{\bar{h}_{1}}{C q_{i}}=\frac{2.5}{0.1337 \times 100}=0.187 \frac{ ft }{ ft ^{3} / min }\end{aligned}\)

Using deviation variables and taking Laplace transforms,

\(\begin{aligned}&\frac{H_{1}^{\prime}(s)}{Q_{i}^{\prime}(s)}=\frac{C}{A_{1} s+\frac{1}{R_{1}}}=\frac{C R_{1}}{A_{1} R_{1} s+1}=\frac{0.025}{1.32 s+1} \\&\frac{H_{2}^{\prime}(s)}{H_{1}^{\prime}(s)}=\frac{1 / R_{1}}{A_{2} s+\frac{1}{R_{2}}}=\frac{R_{2} / R_{1}}{A_{2} R_{2} s+1}=\frac{1}{1.32 s+1} \\&\frac{H_{2}^{\prime}(s)}{Q_{i}^{\prime}(s)}=\frac{0.025}{(1.32 s+1)^{2}}\end{aligned}\)

For step change in \(q_{i}\) of magnitude \(M\),

\(\begin{aligned}h_{1 \max }^{\prime}&=0.025 M \\h_{2 \max }^{\prime}&=0.025 M \text { since the second-order transfer function }\\&\frac{0.025}{(1.32 s+1)^{2}} \text { is critically damped }(\zeta=1), \text { not underdamped }\\\text { Hence } &M_{\max }=\frac{2.5 f t}{0.025 ft / gpm }=100 gpm\end{aligned}\)

For the modified system,

\(\begin{aligned}&A \frac{d h}{d t}=C q_{i}-\frac{h}{R} \\&A=\pi(4)^{2} / 4=12.6 ft ^{2} \\&V=V_{1}+V_{2}=2 \times 7.07 ft ^{2} \times 5 ft =70.7 ft ^{3} \\&h_{\max }=V / A=5.62 ft \\&R=\frac{\bar{h}}{C \bar{q}_{i}}=\frac{0.5 \times 5.62}{0.1337 \times 100}=0.21 \frac{ ft }{ ft ^{3} / min }\\&\frac{H^{\prime}(s)}{Q_{i}^{\prime}(s)}=\frac{C}{A s+\frac{1}{R}}=\frac{C R}{A R s+1}=\frac{0.0281}{2.64 s+1}\\&h_{\max }^{\prime}=0.0281 M\\&M_{\max }=\frac{2.81 ft }{0.0281 ft / gpm }=100 gpm\end{aligned}\)

Hence, both systems can handle the same maximum step disturbance in \(q_{i}\).

b)

For step change of magnitude M, \(Q_{i}^{\prime}(s)=\frac{M}{s}\)

For original system,

\(\begin{aligned}Q_{2}^{\prime}(s)=& \frac{1}{R_{2}} H_{2}^{\prime}(s)=\frac{1}{0.187} \frac{0.025}{(1.32 s+1)^{2}} \frac{M}{s} \\&=0.134 M\left[\frac{1}{s}-\frac{1.32}{(1.32 s+1)}-\frac{1.32}{(1.32 s+1)^{2}}\right] \\q_{2}^{\prime}(t) &=0.134 M\left[1-\left(1+\frac{t}{1.32}\right) e^{-t / 1.32}\right]\end{aligned}\)

For modified system,

\(\begin{aligned}&Q^{\prime}(s)=\frac{1}{R} H^{\prime}(s)=\frac{1}{0.21} \frac{0.0281}{(2.64 s+1)} \frac{M}{s}=0.134 M\left[\frac{1}{s}-\frac{2.64}{2.64 s+1}\right\rfloor \\&q^{\prime}(t)=0.134 M\left[1-e^{-t / 2.64}\right]\end{aligned}\)

Original system provides better damping since \(q_{2}^{\prime}(t)<q^{\prime}(t) \text { for } t<3.4\).
