Guest Question.

A steel gabled frame (assembled as a three-hinged arch) is subjected to wind forces as shown in Figure 3.75.

Determine the support reactions at \(A\) and \(B\) and the internal pin forces at \(C\).

Admin
Solution

Determine the equivalent wind forces \(F_{1}\), \(F_{2}\), and \(F_{3}\).

\(F_{1}=250 \space \# /ft. (20^{\prime } )=5,000\# \)

 

\(F_{2}=100 \space \# /ft. (25.3^{\prime } )=2,530\# \)

 

\(F_{3}=100 \space \# /ft. (25.3^{\prime } )=2,530\# \)

 

Resolve forces \(F_{2}\) and \(F_{3}\) into x and y component forces:

\(F_{2_{x}}= F_{3_{x}} = \frac{8}{25.3} (2.53 \space k ) =0.8 \space k\)

 

\(F_{2_{y}}= F_{3_{y}} -\frac{24}{25.3} (2.53 \space k )=2.4 \space k\)

Solve for the vertical support reactions \(A_{y}\) and \(B_{y}\).

\(\left[\sum{} M_{A}=0\right]-F_{1} (10 ^{\prime } )+F_{2_{x}} (24 ^{\prime } ) +F_{2_{y}} (12^{\prime } )-F_{3_{x}} (24^{\prime } )+F_{3_{y}}(36 ^{\prime } )-B_{y}(48 ^{\prime } ) =0\)

 

\(-5 \space k (10 ^{\prime } )+.8 \space k (24 ^{\prime } ) + 2.4 \space k (12^{\prime } )- .8\space k (24^{\prime } )+2.4 \space k (36 ^{\prime } )-B_{y}(48 ^{\prime } ) =0\)

 

\(\therefore B_{y}=+1.36 \space k \left( \downarrow \right) \)

 

\(\left[\sum{F_{y}} =0\right]-A_{y} + 2.4 \space k +2.4 \space k -1.36 \space k =0\)

 

\(\therefore A_{y}=3.44 \space k \left( \downarrow \right) \)

Separate the frame at the crown joint \(C\), and draw an \(FBD\) of the left or right half.

Using \(FBD\) (c),

\(\left[\sum{} M_{C}=0\right]-2.4 \space k (12 ^{\prime } )-0.8 \space k (4 ^{\prime } ) +5 \space k (18^{\prime } )+3.44 \space k (24^{\prime } ) -A_{x}(28 ^{\prime } ) =0\)

 

\(28 A_{x}=140.56\)

 

\( \therefore A_{x}=5.02 \space k \left( \longleftarrow \right) \)

For the internal pin forces at \(C\),

\( \left[\sum{F_{x}} =0\right]+5 \space k -5.02 \space k -0.8 \space k + C_{x} =0 \)

 

\(\therefore C_{x}=+0.82 \space k \)

 

\( \left[\sum{F_{y}} =0\right]-3.44 \space k +2.4 \space k + C_{y} =0\)

 

\(\therefore C_{y}=+1.04 \space k \)

Going back to \(FBD\) (b) of the entire frame, solve for \(B_{x}\):

\( \left[\sum{F_{x}} =0\right]+5 \space k -5.02 \space k -0.8\space k +0.8 \space k -B_{x} =0\)

 

\(\therefore B_{x}= -0.02 \space k \)

The negative sign in the result for \(B_{x}\) indicates that the original direction assumed in the \(FBD\) was incorrect.

\( \therefore B_{x}= 0.02 \space k \left(\longrightarrow \right) \)
