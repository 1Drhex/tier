Guest Question.

An application requires a normalized lowpass inverse-Chebyshev filter that would satisfy the following specifications:

• Passband edge \(\omega_{p}: 0.6\) rad/s

• Stopband edge \(\omega_{a}: 1.0\) rad/s

• Maximum passband loss \(A_{p}: 1.0\) dB

• Minimum stopband loss \(A_{a}: 35.0\) dB

(a) Find the minimum filter order. (b) Obtain the required transfer function. (c) Calculate the actual maximum passband loss.

Admin
Solution

(a) From Eq. (10.57b)

 

\(D=\frac{10^{0.1 A_{a}}-1}{10^{0.1 A_{p}}-1}\) (10.57b)

 

\(D=\frac{10^{0.1 \times 35.0}-1}{10^{0.1 \times 1.0}-1}=1.2209 \times 10^{4}\)

 

Hence, Eq. (10.57a) yields

 

\(n \geq \frac{\cosh ^{-1} \sqrt{D}}{\cosh ^{-1}\left(1 / \omega_{p}\right)}\) (10.57a)

 

\(n \geq \frac{\cosh ^{-1} \sqrt{1.2209 \times 10^{4}}}{\cosh ^{-1}\left(\frac{1}{0.6}\right)}=\frac{5.3981}{1.0986}=4.9136 \rightarrow 5\)

 

(b) From Eqs. (10.48d) and (10.48e), we have

 

\(r= \begin{cases}\frac{n-1}{2} & \text { for odd } n \\ \frac{n}{2} & \text { for even } n\end{cases}\) (10.48d)

 

\(D_{0}(s)= \begin{cases}s-\frac{1}{p_{0}} & \text { for odd } n \\ 1 & \text { for even } n\end{cases}\) (10.48e)

 

\(r=(n-1) / 2=2 \quad \text { and } \quad D_{0}(s)=s-1 / p_{0}\)

 

and from Eqs. (10.49)–(10.55), we get

 

\(\delta=\frac{1}{\sqrt{10^{0.1 A_{a}}-1}}\) (10.49)

 

\(H_{0}= \begin{cases}\frac{1}{-p_{0}} \prod_{i=1}^{r} \frac{\left|z_{i}\right|^{2}}{\left|p_{i}\right|^{2}} & \text { for odd } n \\ \prod_{i=1}^{r} \frac{\left|z_{i}\right|^{2}}{\left|p_{i}\right|^{2}} & \text { for even } n\end{cases}\) (10.55)

 

\(\delta=\frac{1}{\sqrt{10^{0.1 \times 35.0}-1}}=0.017786\)

 

\(\sigma_{3}=-\sinh \left(\frac{1}{5} \sinh ^{-1} \frac{1}{0.017786}\right)=-1.091354\)

 

\(z_{i}=j \cos \frac{(2 i-1) \pi}{10}\)

 

\(\sigma_{i}=-\sinh \left(\frac{1}{5} \sinh ^{-1} \frac{1}{0.017786}\right) \sin \frac{(2 i-1) \pi}{10}\)

 

\(=-1.091354 \sin \frac{(2 i-1) \pi}{10}\)

 

\(\omega_{i}=\cosh \left(\frac{1}{5} \sinh ^{-1} \frac{1}{0.017786}\right) \cos \frac{(2 i-1) \pi}{2 n}\)

 

\(=1.480221 \cos \frac{(2 i-1) \pi}{10}\)

 

Hence,

 

\(p_{0}=\sigma_{3}=-1.091354\)

 

\(z_{1}=j \cos \frac{\pi}{10}=j 0.951057\)

 

\(z_{2}=j \cos \frac{3 \pi}{10}=j 0.587785\)

 

\(p_{1}=-1.091354 \sin \frac{\pi}{10}+j 1.480221 \cos \frac{\pi}{10}\)

 

\(=-0.337247+j 1.407774\)

 

\(p_{2}=-1.091354 \sin \frac{3 \pi}{10}+j 1.480221 \cos \frac{3 \pi}{10}\)

 

\(=-0.882924+j 0.870052\)

 

Therefore, the transfer function in Eq. (10.48c) assumes the form

 

\(=\frac{H_{0}}{D_{0}(s)} \prod_{i=1}^{r} \frac{s^{2}+\frac{1}{\left|z_{i}\right|^{2}}}{s^{2}-2 \Re e\left(\frac{1}{p_{i}}\right) s+\frac{1}{\left|p_{i}\right|^{2}}}\) (10.48c)

 

\(H_{N}(s)=\frac{H_{0}\left(s^{2}+\frac{1}{\left|z_{1}\right|^{2}}\right)\left(s^{2}+\frac{1}{\left|z_{2}\right|^{2}}\right)}{\left(s-\frac{1}{\sigma_{3}}\right)\left[s^{2}-2 \Re e\left(\frac{1}{p_{1}}\right) s+\frac{1}{\left|p_{1}\right|^{2}}\right]\left[s^{2}-2 \Re e\left(\frac{1}{p_{2}}\right) s+\frac{1}{\left|p_{2}\right|^{2}}\right]}\)

 

\(=\frac{H_{0}\left(s^{2}+a_{01}\right)\left(s^{2}+a_{02}\right)}{\left(s+b_{00}\right)\left(s^{2}+b_{11} s+b_{01}\right)\left(s^{2}+b_{12} s+b_{02}\right)}\)

 

where

 

\(\begin{array}{ll}a_{01}=\frac{1}{\left|z_{1}\right|^{2}}=1.105573 \quad a_{02}=\frac{1}{\left|z_{2}\right|^{2}}=2.894427 \\b_{00}=-\frac{1}{\sigma_{3}}=0.916293 \\b_{01}=\frac{1}{\left|p_{1}\right|^{2}}=0.477199 \quad b_{11}=-2 \Re e\left(\frac{1}{p_{1}}\right)=0.321868 \\b_{02}=\frac{1}{\left|p_{2}\right|^{2}}=0.650811 \quad b_{12}=-2 \Re e\left(\frac{1}{p_{2}}\right)=1.149232\end{array}\)

 

\(H_{0}=\frac{1}{-p_{0}} \prod_{i=1}^{2} \frac{\left|z_{i}\right|^{2}}{\left|p_{i}\right|^{2}}\)

 

= 0.088928

 

(c) From Eq. (10.56), the maximum passband loss can be determined by evaluating the loss at the passband edge as

 

\(\begin{aligned}A\left(\omega_{p}\right) &=10 \log \left[1+\frac{1}{\delta^{2} T_{n}^{2}\left(1 / \omega_{p}\right)}\right] \\&=10 \log \left[1+\frac{1}{\delta^{2}\left[\cosh \left(n \cosh ^{-1} 1 / \omega_{p}\right)\right]^{2}}\right]\end{aligned}\) (10.56)

 

\(A\left(\omega_{p}\right)=10 \log \left\{1+\frac{1}{0.017786^{2}\left[\cosh \left(5 \cosh ^{-1} \frac{1}{0.6}\right)\right]^{2}}\right\}\)

 

= 0.8427 dB
