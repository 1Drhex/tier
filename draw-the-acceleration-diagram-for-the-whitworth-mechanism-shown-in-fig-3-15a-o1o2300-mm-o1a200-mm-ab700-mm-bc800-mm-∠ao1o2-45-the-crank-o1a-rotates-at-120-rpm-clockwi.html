Guest Question.

Draw the acceleration diagram for the Whitworth mechanism shown in Fig.3.15(a).

\( O_{1} O_{2}=300 mm , O_{1} A=200 mm \),

AB=700 mm, BC=800 mm,

\( \angle A O_{1} O_{2}=45^{\circ} \).

The crank O1A rotates at 120 rpm clockwise. Determine (a) Velocity of ram C, (b) Coriolis acceleration, (c) angular acceleration of AB, (d) angular acceleration of BC, and (e) acceleration of ram C.

Admin
Solution

\( \omega=2 \pi \times \frac{120}{60}=12.57 rad / s \).

\( v_{a}=12.57 \times 0.2=2.51 m / s \).

Velocity diagram

1. Draw the configuration diagram to a scale of 1 cm=100 mm, as shown in Fig.3.15(a). By measurement, \( O_{2} P=460 mm \). Point P is the coincident point of A on the link AB, whereas point A is on the crank \( O_{1} A \).

2. Draw the velocity diagram to a scale of 1 cm=0.5 m/s, as shown in Fig.3.15(b), by adopting the following steps.

\( \text { 3. Draw } o_{1} a=v_{a}=5.02 cm \perp O_{1} A \).

4. Draw a line at ‘a’ parallel to PB and another line at \( o_{2} \) perpendicular to PB to meet at point \( p \cdot v_{a p}=p a \) represents the velocity of sliding between A and P.

\( \text { 5. By measurement, } o_{2} p=4.5 cm \).

\( p b=o_{2} p \times \frac{P B}{O_{2} P}=\frac{4.5 \times 700}{460}=6.85 cm \).

\( \text { 6. Extend line } p o_{2} \text { to } b \text { such that } p b=6.85 cm \).

7. Draw a line at b perpendicular to BC and another line parallel to the path of the ram at C to meet at point c. Then

\( v_{a p}=p a=2.3 cm =1.15 m / s \).

\( v_{p o 2}=o_{2} p=4.5 cm =2.25 m / s \).

\( v_{c b}=b c=0.8 cm =0.4 m / s \).

\( \text { Velocity of ram } C, v c=o_{1} c=1.9 cm =0.85 m / s \).

\( \text { Angular velocity of } \operatorname{link} O_{2} P, \omega_{o 2 p}=\frac{v_{p o 2}}{O_{2} P}=\frac{2.25}{0.46}=4.89 rad / s cw \). (see Fig.3.15(d)).

\( \text { Angular velocity of } B C, \omega_{b c}=\frac{v_{b c}}{B C}=\frac{0.4}{0.8}=0.5 rad / s \).

Acceleration diagram

\( \text { Coriolis acceleration, } f_{a p}^{c r}=2 v_{a p} \omega_{o 2 p}=2 \times 1.15 \times 4.89=11.247 m / s \).

Refer to figure for direction of Coriolis acceleration.

\( f_{a}^{n}=\frac{v_{a}^{2}}{O_{1} A}=\frac{(2.51)^{2}}{0.2}=31.5 m / s ^{2} \).

\( f_{p o 2}^{n}=\frac{v_{p o 2}^{2}}{O_{2} P}=\frac{(2.25)^{2}}{0.46}=11 m / s ^{2} \).

\( f_{c b}^{n}=\frac{v_{c b}^{2}}{B C}=\frac{(0.4)^{2}}{0.8}=0.2 m / s ^{2} \).

1. Draw the acceleration diagram as shown in Fig.3.15(c) to a scale of 1 cm2 m/s², by adopting the following procedure:

\( \text { 2. Draw } o_{1} a=f_{a}^{n}=15.75 cm \text { parallel to } O_{1} A \).

\( \text { 3. At point ' } a \text { ' draw a line } a p^{\prime \prime}=f_{a p}^{c r}=5.6 cm \text { perpendicular to } P B \text {. At point } p^{\prime} \) draw another line perpendicular to PB representing the acceleration of sliding \( f_{a p}^{s} \text { between } A \text { and } P . f_{a p}=p a \) is the total acceleration of A relative to P.

\( \text { 4. Draw } o_{2} p^{\prime}=f_{p o 2}^{n}=5.5 cm \text { parallel to } O_{2} P \cdot \text { At } p^{\prime} \text { draw another line perpendicular to } O_{2} P \) representing \( f_{o 2 p}^{t} \text { to meet the line } f_{a p}^{s} \text { at point } p \text {. Join } a p \text { and } o_{2} p \). Then

\( f_{a p}=p a \text { and } f_{p o 2}=o_{2} p \).

\( \text { 6. Now } p b=\frac{p o_{2} \times O_{2} B}{O_{2} P}=\frac{5.7 \times 700}{460}=8.67 cm\).

\( \text { Tangential acceleration of } O_{2} P, f_{p o 2}^{t}=p^{\prime} p=1.2 cm =4.4 m / s ^{2} \).

\( \text { Angular acceleration of } O_{2} P=\frac{f_{p o 2}^{t}}{O_{2} P}=\frac{2.4}{0.46}=5.21 rad / s ^{2} \).
