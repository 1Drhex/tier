Guest Question.

A capacitor \(C\) is initially charged to \(500 V\) and is left open for \(120 s\). The voltage across the capacitor at the end of this time interval is seen to be \(400 V\). A resistor of \(100\) \(k立\) is connected across this capacitor at \(120 s\). The voltage across the capacitor is found to reach \(100 V\) in \(216.4 s\) after this connection has been made. Find the value of \(C\) and its leakage resistance.

Admin
Solution

Let \(R\) be the leakage resistance of the capacitor. Then, using Eqn. 11.2-2 with suitable values for \(V_{o}\) and \(v_{C} (t),\)

\(\begin{array}{l} 400=500 e^{-120 / \tau_{1}} \Rightarrow \tau_{1}=\frac{120}{\ln 1.25}=537.8 \mathrm{~s} \text { and } \\ 100=400 e^{-216.4 / \tau_{2}} \Rightarrow \tau_{2}=\frac{216.4}{\ln 4}=156.1 \mathrm{~s} \end{array}\).

\(\text { where } \tau_{1}=R C \text { and } \tau_{2}=[R / / 100 k \Omega] C\).

\(\begin{aligned} &\therefore \frac{\tau_{1}}{\tau_{2}}=\frac{537.8 \mathrm{~s}}{156.1 \mathrm{~s}}=3.445=\frac{R(R+100)}{R \times 100}=\frac{(R+100)}{100}=0.01 R+1 ; \text { with } R \text { in } \mathrm{k} \Omega\\ &\therefore R=244.5 \mathrm{k} \Omega \text { and since } R C=537.8 \mathrm{~s}, C=\frac{537.8 \mathrm{~s}}{244.5 \mathrm{k} \Omega}=2.2 \mathrm{mF}=2200 \mu \mathrm{F} \end{aligned}\).

\(v_{\mathrm{C}}(t)=V_{\mathrm{o}} e^{-t / \tau} \mathrm{V} \text { for } t \geq 0^{+} \quad\quad (11.2-2)\)
