Design a bolted flange-plated FR moment connection between a W18×50 beam and a W14×99 column flange to transfer the following forces:
\(\begin{array}{ll}R_{D}=7 \text { kips } & M_{D}=42 \text { kip-ft } \\R_{L}=21 \text { kips } & M_{L}=126 \text { kip-ft }\end{array}\)
Use 7/8-in. diameter ASTM A325-N bolts in standard holes and E70 electrodes.

Material Properties:
Beam W18×50 ASTM A992 \(F_{y}=50 ksi \quad F_{u}=65 ksi\)
Column W14×99 ASTM A992 \(F_{y}=50 ksi \quad F_{u}=65 ksi\)
Plate ASTM A36 \(F_{y}=36 ksi \quad F_{u}=58 ksi\)

LRFD ASD
\(\begin{aligned}R_{u} &=1.2(7 kips )+1.6(21 kips ) \\&=42 kips\end{aligned}\)

 

\(\begin{aligned}M_{u} &=1.2(42 kip – ft )+1.6(126 kip – ft ) \\&=252 kip – ft\end{aligned}\)

 \(\begin{aligned}R_{a} &=7 \text { kips }+21 \text { kips } \\&=28 \text { kips }\end{aligned}\)

Check the beam available flexural strength

Assume two rows of bolts in standard holes.

\(A_{f g}=b_{f} t_{f}=(7.50 \text { in. })(0.570 \text { in. })=4.28 \text { in. }{ }^{2}\)

Section
F13.1

\(A_{f n}=A_{f g}-2\left(d_{b}+1 / 8 \text { in. }\right) t_{f}=4.28 \text { in. }^{2}-2(7 / 8 \text { in. }+1 / 8 \text { in. })(0.570 \text { in. })=3.14 \text { in. }{ }^{2}\)

 

\(\frac{F_{y}}{F_{u}}=\frac{50 ksi }{65 ksi }=0.769<0.8, \text { therefore } Y_{t}=1.0\)

 

\(F_{u} A_{f n}=(65 ksi )\left(3.14 in .^{2}\right)=204 kips\)

 

\(Y_{t} F_{y} A_{f g}=(1.0)(50 ksi )\left(4.28 in. ^{2}\right)=214 kips >204 kips\)

 

Therefore the nominal flexural strength, \(M_{n}\), at the location of the holes in the tension flange is as follows:

 

\(M_{n}=\frac{F_{u} A_{f n} S_{x}}{A_{f g}}=\frac{(65 ksi )\left(3.14 in. ^{2}\right)\left(88.9 in .^{3}\right)}{4.28 in .^{2}}=4240 kip \text { -in. or } 353 kip – ft\)

Check flange plate block shear rupture

There are two cases for which block shear rupture must be checked. The first case involves the tearout of the two blocks outside the two rows of bolt holes in the flange plate; for this case \(L_{e h}\) = 1 1/2 in. and \(L_{e v}\) = 1 1/2 in. The second case involves the tearout of the block between the two rows of the holes in the flange plate. Manual Tables 9-3a, 9-3b, and 9-3c may be adapted for this calculation by considering the 4 in. width to be comprised of two, 2-in. wide blocks where \(L_{e h}=2 \text { in. and } L_{e v}\) = 1 1/2 in. Thus, the former case is the more critical.
