Guest Question.

The kinetics of acetone hydrogenation on Pt catalysts have been studied by Sen and Vannice () [35]. To determine which bonding mode is more probable, the heat of adsorption for an on-top \(\eta ^{1} \mu _{1} \) adsorbed acetone species was compared to that for a di-\(\sigma \)-bonded \(\eta ^{2} \mu _{2} \) species which had a C atom and the O atom interacting with a close-packed Pt(111) surface. What are these two values? Which species is favored?

Admin
Solution

For acetone,\(\left(CH_{3} \right)_2 CO \left\lgroup or \overset {\overset{CH_{3}}{\diagdown } }{\underset{ \underset{CH_{3}}{\diagup } }{ \ \ \ \ \ \ \ \ \ \ C=O}} \right\rgroup \) , the enthalpy offormation is -217.3 kJ/mole or -51.9

kcal/mole [25]; therefore, \(D_{Total} = D_{\left({CH_{3}}\right) _{2}CO } = \Sigma \Delta H^{o}_{f_{i}}\left(atoms\right) – \Delta H^{o}_{f } \) so, based on (see figur) :

 

and from reference [25]:

\(3\left(C\rightarrow C\cdot \right)= 3 (171.3 \ kcal/mole)\)

 

\(3\left(H_{2}\rightarrow 2 \ H\cdot \right) = (3) (2) (51.2 \ kcal/mole)\)

 

\(1/2 \left(O_{2\rightarrow 2}\ O\cdot \right) = (0.5) (119.2 \ kcal/mole)\)

 

\(\Delta H^{o}_{f} = -(-51.9 \ kcal/mole)\)

\(D_{Total} = 938.0 kcal/mole\)

From Table 6.5, on \(Q_{C}=150 \ kcal/mole\) and \(Q_{O}= 85 \ kcal/mole\)

(a) For on-top adsorption \(\left(\eta ^{1} \mu _{1} \right) \) use equation 6.35: \(Q_{A}= Q\left(n\right)= Q_{oA}\left(2-1/n\right)\) with n = 3 for Pt(III). Molecule willbond O-end down; so

\(Q_{oO} \)= (3/5) (85) and from Table 6.7b, the bond energy associated with each\(C-CH_{3}\) group is 376 kcaljmole, so

\(D_{AB}= D_{C=O}= 938.0 – 2 (376) = 186 \ kcal/mole\)

Then, for “weak” chemisorption, use equation 6.41

\(Q_{AB}= Q^{2}_{oA} / \left(\frac{Q_{oA}}{\acute{n} } + D_{AB} \right) \left(for \ D_{AB}> \frac{\acute{n} -1 }{\acute{n}} Q_{oA}\right) \):

\(Q_{\left(CH_{3} \right)_{2} CO} = \frac{Q^{2}_{oO} }{Q_{oO}+ Q_{\left(CH_{3} \right)_{2} CO} } = \frac{\left(0.6\right) ^{2} \left(85 \right)^{2}}{\left(0.6\right) \left(85 \right)+ 186 }= 11.0 \ kcal /mole\)

 

(b) For di-\(\sigma\)-adsorption, i.e., \(Pt \xrightarrow[]{ \overset{ \overset{}{\underset{ \ C – O} {\overset {CH_{3}\ \ CH_{3}}{\diagdown \diagup } \ \overset{}{ } }}}{ \ \diagup \ \ \ \ \ \diagdown }} Pt\) \(\left(\eta ^{2} \mu _{2} \right) \)

equation 6.42(\(Q_{AB} = \frac{ab\left(a+b\right)+ D_{AB}\left(a-b\right) ^{2} }{ab + D_{AB}\left(a+b\right) } \)) must be used, keeping in mind that atoms A and B can have other groups attached to alter the A-B bond energy,thus from part (a) \( D_{AB}= D_{C=O}= 186 \ kcal/mole\)

From Eq. 6.43(\( a=Q^{2}_{oA}\left(Q_{oA}+ 2 Q_{oB} \right)/ \left(Q_{oA}+ Q_{oB} \right)^{2} \)), \( a=\frac{Q^{2}_{oC'}\left(Q_{oC'} + 2 \ Q_{oO}\right) }{\left(Q_{oC'} + \ Q_{oO}\right) ^{2} } \) where \(Q_{oC'}\) ‘ represents the heat of chemisorption of the quasi-atomic \(\left(CH_{3} \right)_{2}C \)· fragment. For this relatively large species, a reasonable way to calculate its heat of chemisorption is to again use eq. 6.41 (\(Q_{AB}= Q^{2}_{oA}/ \left(\frac{Q_{oA}}{n'} + D_{AB} \right) \left(for \ D_{AB} > \frac{n'-1 }{n'} Q_{oA} \right) \))where \(D_{AB}\)
now represents the total bond energy of all the bonds formed by the secondary C atom,
thus fromTable6.7b:

\(D_{AB}= 2 \left\lfloor D_{C-CH_{3}}- D_{CH_{3}} \right\rfloor = 2\left\lfloor 376 – 293\right\rfloor= 166 \ kcal/mole\)

so \(Q_{oC'}= \frac{\left(\left(0.6\right) ^{2} \left(\left(150\right) \right) ^{2} \right) }{\left(0.6\right)\left(150\right)+166}= 31.6 \ kcal/mole\)

\(a=\frac{\left(31.6\right) ^{2}\left[31.6 + 2\left(51.0\right) \right] }{\left(31.6+ 51.0\right) ^{2}} = \frac{1.33\times 10^{5} }{6.82\times 10^{3} }= 19.6 \ kcal/mole\) , and

from Eq, 6.44(\(b=Q^{2}_{oB}\left(Q_{oB}+ 2 Q_{oA} \right)/ \left(Q_{oA}+ Q_{oB} \right)^{2} \)),\(b=\frac{\left(51.0\right) ^{2}\left[51.0 + 2\left(31.6\right) \right] }{\left(51.0 + 31.6\right) ^{2}} = \frac{2.97 \times 10^{5}}{6.82\times 10^{3}} = 43.5 \ kcal/mole\)

thus \(Q_{\left(CH_{3} \right)_{2}CO } = \frac{ab\left(a+b\right)+ D_{AB}\left(a-b\right)^{2} }{ ab + D_{AB}\left(a+b\right)} = \frac{(19.6) (43.5) (19.6 + 43.5) + (186) \left(19.6 – 43.5\right)^{2} }{(19.6) (43.5) + (186) (19.6 + 43.5)} = \frac{1.60 \times 10^{5}}{1.26 \times 10^{4}} = 12.7 \cong 13 \ kcal/mole\)
