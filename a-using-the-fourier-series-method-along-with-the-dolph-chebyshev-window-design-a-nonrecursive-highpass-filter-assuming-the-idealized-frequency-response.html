Guest Question.

(a) Using the Fourier-series method along with the Dolph-Chebyshev window, design a nonrecursive highpass filter assuming the idealized frequency response

\(H\left(e^{j \omega T}\right)= \begin{cases}1 & \text { for }-\omega_{s} / 2<\omega<-\omega_{c} \\ 1 & \text { for } \omega_{c}<\omega<\omega_{s} / 2 \\ 0 & \text { otherwise }\end{cases}\)

The required filter parameters are as follows:

• Ripple ratio: −20 dB

• \(\omega_{c}: 6.0\) rad/s

• \(\omega_{c}: 20\) rad/s

• N = 21

(b) Assuming that the passband of the filter extends from 6.8 to 10 rad/s, normalize the design obtained in part (a) so as to achieve an amplitude response that oscillates about unity. (c) Find the passband peak-to-peak ripple \(A_{p}\) in dB. (d) Assuming that the stopband extends from 0 to 5.5 rad/s, find the minimum stopband attenuation \(A_{a}\) in dB.

Admin
Solution

(a) From Eq. (9.11), we have

 

\(h(n T)=\frac{1}{\omega_{S}} \int_{-\omega_{s} / 2}^{\omega_{s} / 2} H\left(e^{j \omega T}\right) e^{j \omega n T} d \omega\) (9.11)

 

\(h(n T)=\frac{1}{\omega_{s}}\left[\int_{-\omega_{s} / 2}^{-\omega_{c}} e^{j \omega n T} d \omega+\int_{\omega_{c}}^{\omega_{s} / 2} e^{j \omega n T} d \omega\right]\)

 

\(=\frac{1}{\omega_{s}}\left\{\left[\frac{e^{j \omega n T}}{j n T}\right]_{-\omega_{s} / 2}^{-\omega_{c}}+\left[\frac{e^{j \omega n T}}{j n T}\right]_{\omega_{c}}^{\omega_{s} / 2}\right\}\)

 

\(=\frac{1}{n \pi}\left[\sin \omega_{s} n T / 2-\sin \omega_{c} n T\right]\)

 

\(=\frac{1}{n \pi}\left[\sin n \pi-\sin \omega_{c} n T\right]\)

 

We note that \((\sin n \pi) / n \pi\) is always zero except for n = 0 and hence we get

 

\(h(n T)= \begin{cases}1-\frac{1}{n \pi} \sin \omega_{c} n T & \text { for } n=0 \\ -\frac{1}{n \pi} \sin \omega_{c} n T & \text { otherwise }\end{cases}\) (9.24)

 

The ripple ratio in dB is given by 20 log r and hence

 

\(20 \log r=-20 \quad \text { or } \quad r=10^{-1}=0.1\)

 

Onusing Eqs. (9.22) and (9.24), the design of Table 9.3 where \(h_{w}(n T)=w_{D C}(n T) h(n T)\) can be obtained.

 

\(w_{D C}(n T)=\frac{1}{N}\left[\frac{1}{r}+2 \sum_{i=1}^{(N-1) / 2} T_{N-1}\left(x_{0} \cos \frac{i \pi}{N}\right) \cos \frac{2 n \pi i}{N}\right]\) (9.22)

 

(b) The amplitude response can be computed by using the formula for a symmetrical impulse response of odd length in Table 9.1. Through a simple MATLAB m-file the maximum and minimum values of the passband amplitude response can be

 

obtained as

 

\(M_{\max }=0.6031 \quad \text { and } \quad M_{\min }=0.5737\)

 

Hence the required scaling factor to normalize the passband amplitude response to unity is obtained from Eq. (9.23) as \(M_{ AV }=0.5884\). The amplitude response of the filter is plotted in Fig. 9.9.

 

\(h_{w}^{\prime}(n T)=\frac{h_{w}(n T)}{M_{ AV }} \quad \text { where } M_{ AV }=\frac{1}{2}\left(M_{\max }+M_{\min }\right)\) (9.23)

 

(c) The peak-to-peak passband ripple in dB can be obtained as

 

\(A_{p}=20 \log M_{\max }-20 \log M_{\min }=20 \log \frac{M_{\max }}{M_{\min }}=0.43 dB\)

 

(d) The minimum stopband attenuation \(A_{a}\) is defined as the negative of the maximum stopband gain and it can be computed as 0.01549. Hence we have

 

\(A_{a}=-20 \log 0.01549=20 \log \frac{1}{0.01549}=36.2 dB\)
