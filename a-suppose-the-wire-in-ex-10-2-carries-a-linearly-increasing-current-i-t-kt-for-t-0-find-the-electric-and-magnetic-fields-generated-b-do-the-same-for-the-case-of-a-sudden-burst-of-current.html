Guest Question.

(a) Suppose the wire in carries a linearly increasing current

I (t) = kt,

for t > 0. Find the electric and magnetic fields generated.

(b) Do the same for the case of a sudden burst of current:

\(I(t)=q_{0} \delta(t)\).

Admin
Solution

(a) As in , for t < s/c, A = 0; for t > s/c,

\(A (s, t)=\left(\frac{\mu_{0}}{4 \pi} \hat{ z }\right) 2 \int_{0}^{\sqrt{(c t)^{2}-s^{2}}} \frac{k\left(t-\sqrt{s^{2}+z^{2}} / c\right)}{\sqrt{s^{2}+z^{2}}} d z=\frac{\mu_{0} k}{2 \pi} \hat{ z }\left\{t \int_{0}^{\sqrt{(c t)^{2}-s^{2}}} \frac{d z}{\sqrt{s^{2}+z^{2}}}-\frac{1}{c} \int_{0}^{\sqrt{(c t)^{2}-s^{2}}} d z\right\}\)

 

\(=\left(\frac{\mu_{0} k}{2 \pi} \hat{ z }\right)\left[t \ln \left(\frac{c t+\sqrt{(c t)^{2}-s^{2}}}{s}\right)-\frac{1}{c} \sqrt{(c t)^{2}-s^{2}}\right]\) . Accordingly,

\(E (s, t)=-\frac{\partial A }{\partial t}=-\frac{\mu_{0} k}{2 \pi} \hat{ z }\left\{\ln \left(\frac{c t+\sqrt{(c t)^{2}-s^{2}}}{s}\right)+\right.\)

 

\(\left.t\left(\frac{s}{c t+\sqrt{(c t)^{2}-s^{2}}}\right)\left(\frac{1}{s}\right)\left(c+\frac{1}{2} \frac{2 c^{2} t}{\sqrt{(c t)^{2}-s^{2}}}\right)-\frac{1}{2 c} \frac{2 c^{2} t}{\sqrt{(c t)^{2}-s^{2}}}\right\}\)

 

\(=-\frac{\mu_{0} k}{2 \pi} \hat{ z }\left\{\ln \left(\frac{c t+\sqrt{(c t)^{2}-s^{2}}}{s}\right)+\frac{c t}{\sqrt{(c t)^{2}-s^{2}}}-\frac{c t}{\sqrt{(c t)^{2}-s^{2}}}\right\}\)

 

\(=-\frac{\mu_{0} k}{2 \pi} \ln \left(\frac{c t+\sqrt{(c t)^{2}-s^{2}}}{s}\right) \hat{ z }\) (or zero, for t < s/c).

\(B (s, t)=-\frac{\partial A_{z}}{\partial s} \hat{ \phi }\)

 

\(=-\frac{\mu_{0} k}{2 \pi}\left\{t\left(\frac{s}{c t+\sqrt{(c t)^{2}-s^{2}}}\right) \frac{\left[s \frac{1}{2} \frac{(-2 s)}{\sqrt{(c t)^{2}-s^{2}}}-c t-\sqrt{(c t)^{2}-s^{2}}\right]}{s^{2}}-\frac{1}{2 c} \frac{(-2 s)}{\sqrt{(c t)^{2}-s^{2}}}\right\} \hat{\phi}\)

 

\(=-\frac{\mu_{0} k}{2 \pi}\left\{\frac{-c t^{2}}{s \sqrt{(c t)^{2}-s^{2}}}+\frac{s}{c \sqrt{(c t)^{2}-s^{2}}}\right\} \hat{\phi}=-\frac{\mu_{0} k}{2 \pi} \frac{\left(-c^{2} t^{2}+s^{2}\right)}{s c \sqrt{(c t)^{2}-s^{2}}} \hat{\phi}=\frac{\mu_{0} k}{2 \pi s c} \sqrt{(c t)^{2}-s^{2}} \hat{\phi}\).

\(\text { (b) } A (s, t)=\frac{\mu_{0}}{4 \pi} \hat{ z } \int_{-\infty}^{\infty} \frac{q_{0} \delta(t-ᴫ / c)}{ᴫ} d z . \text { But } ᴫ=\sqrt{s^{2}+z^{2}}\) , so the integrand is even in z:

\(A (s, t)=\left(\frac{\mu_{0} q_{0}}{4 \pi} \hat{ z }\right) 2 \int_{0}^{\infty} \frac{\delta(t-ᴫ / c)}{ᴫ} d z\).

\(\text { Now } z=\sqrt{ᴫ^{2}-s^{2}} \Rightarrow d z=\frac{1}{2} \frac{2 ᴫ d ᴫ}{\sqrt{ᴫ^{2}-s^{2}}}=\frac{ᴫ d ᴫ}{\sqrt{ᴫ^{2}-s^{2}}}\) , and \(z=0 \Rightarrow ᴫ=s, z=\infty \Rightarrow ᴫ=\infty \) . So:

\(A (s, t)=\frac{\mu_{0} q_{0}}{2 \pi} \hat{ z } \int_{s}^{\infty} \frac{1}{ᴫ} \delta\left(t-\frac{ᴫ}{c}\right) \frac{ᴫ d ᴫ}{\sqrt{ᴫ^{2}-s^{2}}}\).

Now \(\delta(t-ᴫ / c)=c \delta(ᴫ-c t)\) (); therefore \(A =\frac{\mu_{0} q_{0}}{2 \pi} \hat{ z } c \int_{s}^{\infty} \frac{\delta( ᴫ -c t)}{\sqrt{ᴫ^{2}-s^{2}}} d ᴫ\).

\(A (s, t)=\frac{\mu_{0} q_{0} c}{2 \pi} \frac{1}{\sqrt{(c t)^{2}-s^{2}}} \hat{ z }\) (or zero, if ct < s);

\(E (s, t)=-\frac{\partial A }{\partial t}=-\frac{\mu_{0} q_{0} c}{2 \pi}\left(-\frac{1}{2}\right) \frac{2 c^{2} t}{\left[(c t)^{2}-s^{2}\right]^{3 / 2}} \hat{ z }=\frac{\mu_{0} q_{0} c^{3} t}{2 \pi\left[(c t)^{2}-s^{2}\right]^{3 / 2}} \hat{ z }\) (or zero, for t < s/c);

\(B (s, t)=-\frac{\partial A _{z}}{\partial t} \hat{\phi}=-\frac{\mu_{0} q_{0} c}{2 \pi}\left(-\frac{1}{2}\right) \frac{-2 s}{\left[(c t)^{2}-s^{2}\right]^{3 / 2}} \hat{\phi}=\frac{-\mu_{0} q_{0} c s}{2 \pi\left[(c t)^{2}-s^{2}\right]^{3 / 2}} \hat{\phi}\) (or zero, for t < s/c).
