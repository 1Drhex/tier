Guest Question.

Verify the strength of a C9×20 ASTM A36 channel with end shears of 5 kips from dead load and 15 kips from live load in the weak direction.

Admin
Solution

Material Properties:

C9×20 ASTM A36 \(F_{y}=36 ksi \quad F_{u}=58 ksi\)

Manual
Table 2-3

Geometric Properties:

C9×20 \(b_{f}=2.65 \text { in. } \quad t_{f}=0.413 \text { in. }\)

Manual
Table 1-5

Note: There are no Manual tables for this, but the strength can be calculated as follows:

Calculate the required shear strength

LRFD ASD
\(V_{u}=1.2(5.0 kips )+1.6(15.0 kips )\)

= 30.0 kips

 \(V_{a}=5.0\text { kips }+15.0 \text { kips }\)

= 20.0 kips

For weak axis shear, use Equation G2-1 and Section G2.1(b) with \(A_{w}=b_{f} t_{f}\) for each flange and \(k_{v}=1.2\).

Section G7

Calculate \(A_{w}\) (multiply by 2 for both shear resisting elements)

 

\(A_{w}=2 b_{f} t_{f}=2(2.65 in .)(0.413 in .)=2.19 in .{ }^{2}\)

Section G7

 

\(\text { Calculate } C_{v}\) Section G2.1b

 

\(b_{f} / t_{f}=2.65 \text { in. } / 0.413 \text { in. }=6.42\)

 

\(1.10 \sqrt{k_{v} E / F_{y}}=1.10 \sqrt{1.2(29,000 ksi / 36 ksi )}=34.2>6.42 \text { Therefore, } C_{v}=1.0\)

Eqn. G2-3

\(\text { Calculate } V_{n}\)

 

\(V_{n}=0.6 F_{y} A_{w} C_{v}=0.6(36 ksi )\left(2.19 in. ^{2}\right)(1.0)=47.3 kips\)

Eqn. G2-1

Calculate the available shear strength

LRFD ASD
\(\phi_{v}=0.90\) \(\Omega_{v}=1.67\)
\(\phi_{v} V_{n}=0.90(47.3 kips )=42.6 \text { kips }\) \(V_{n} / \Omega_{v}=47.3\text { kips } / 1.67=28.3 \text { kips }\)
42.5 kips > 30.0 kips o.k. 28.3 kips > 20.0 kips o.k.
