Guest Question.

Consider a dc-dc converter unit system that uses the Weinberg converter with an input dc equal to 125 V and an output of 48 V. The design has the following speciﬁcations:

Output power \(\ P_{o} = 2 kW \)

Switching frequency f=40 kHz

Output ripple voltage \(\ \Delta V_{o} / V_{o} \) =0.2%

Determine the following:

(a) The value of n where D=0.7 and m=0.5

(b) The critical inductor value

(c) The maximum and minimum inductor current

(d) The average input and output current

(e) The capacitor current value

Admin
Solution

(a) From the gain relation, we have:

\(\ \frac{V_{o} }{V_{in} } = \frac{D}{\frac{D}{m}+ \frac{1-D}{n} } \)

\(\ \frac{48}{125} =\frac{0.7}{\frac{0.7}{0.5}+ \frac{1-0.7}{n} } \)

\(\ n\thickapprox 0.7 \)

(b) The minimum inductor value is given by:

\(\ L_{crit} \gt \frac{RDV_{in}}{2nV_{o}}\left(1-D\right) T\)

\(\ R= V^{2}_{o} / P = 1.152\Omega , and T_{s} = 1/ f=25 \mu s\) ; therefore:

\(\ L_{crit} =\frac{\left(1.125\right)\left(0.7\right) \left(125\right) }{2\left(48\right)\left(0.7\right) }\left(1-0.7\right) \left(25\mu s\right) \)

\(\ L_{crit} =11.25 \mu H\)

Select L=30 μH.

(c) The maximum inductor current is:

\(\ I_{L_{max} } = \frac{V^{2}_{o}}{RDV_{in}} +\frac{V_{o}}{2nL} \left(1-D\right) T_{s} \)

\(\ I_{L_{max} } =\frac{\left(48\right) ^{2} }{\left(1.125\right)\left(0.7\right) \left(125\right) } +\frac{48}{2\left(0.7\right)\left(30 \mu H\right) } \left(1-0.7\right) \left(25\mu s\right) \)

\(\ I_{L_{max} } =31.4 A \)

And the minimum value is given by:

\(\ I_{L_{max} } =\frac{V^{2}_{o}}{RDV_{in}} -\frac{V_{o}}{2nL} \left(1-D\right) T_{s} \)

\(\ I_{L_{max} } =14.3 A \)

(d) From Eq. (5.34), we have:

\(\ i_{L} \left(t\right) =\frac{V_{in} }{L} \left(t – t_{2} \right) + I_{L} \left(t_{2} \right) \) (5.34)

\(\ I_{o,ave} = \frac{\left(I_{L_{max} }+I_{L_{max} }\right) }{2} \left(\frac{D}{m} +\frac{1-D}{n} \right) \)

\(\ I_{o,ave} =\frac{\left(31.4 +14.3\right) }{2} \left(\frac{0.7}{0.5} +\frac{1-0.7}{0.7} \right) \)

\(\ I_{o,ave} = 41.78 A \)

And from Eq. (5.35) we have:

\(\ i_{L} \left(t_{3} \right) =\frac{V_{in} }{L} \left(t_{3} – t_{2} \right) + I_{L} \left(t_{2} \right) \) (5.35)

\(\ I_{in,ave} =\frac{\left(I_{L_{max} }+I_{L_{max} }\right) D}{2} \)

\(\ I_{in,ave} =\frac{\left(31.4+I_14.3\right) }{2} 0.7 \)

\(\ I_{in,ave} = 16 A \)

(e) The times at which the capacitor current becomes zero are given by:

\(\ t_{x} = \frac{I_{o} – I_{L_{min} }/ m}{\frac{1}{L} \left(\frac{V_{in} }{m^{2} } -\frac{V_{o}}{m} \right) } \)

\(\ t_{x} = \frac{41.78 – 14.3/ 0.5}{\frac{1}{30\mu H} \left(\frac{125 }{0.5} -\frac{48}{0.5^{2}} \right) } \)

\(\ t_{x} = 6.83 \) μs

and

\(\ t_{y} =\frac{nL}{V_{o}} \left(I_{L_{max} } -n I_{o}\right) +DT_{s} \)

\(\ t_{y} =\frac{\left(0.7\right)\left(30\mu H\right) }{48} \left(31.4 -0.7 \times 41.78\right) +\left(0.7\right)\left(25\mu s\right) \)

\(\ t_{y} = 18.44\mu s \)

Recall, \(\ \tau _{n} =\frac{\tau }{T_{s} } = \frac{L/ R}{T_{s}} and \tau _{nRC} =\frac{RC}{T_{s}} \)

Therefore

\(\ \Delta V_{nc} =\frac{n^{2}\tau _{n} }{2\tau _{nRC}} \left\{\left(\frac{I_{n.max} }{n} -1\right) ^{2} +\left(\frac{mD}{n\left(1-D\right) } \right) \left(\frac{1-I_{n.min} }{m}\right) ^{2} \right\} \)

The value for the capacitor is obtained as:

C = 476:6 μF
