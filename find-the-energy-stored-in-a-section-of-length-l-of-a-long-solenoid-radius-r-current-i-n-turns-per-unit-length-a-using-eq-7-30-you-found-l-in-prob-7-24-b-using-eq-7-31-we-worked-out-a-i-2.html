Guest Question.

Find the energy stored in a section of length l of a long solenoid (radius R, current I, n turns per unit length), (a) using Eq. 7.30 (you found L in ); (b) using Eq. 7.31 (we worked out A in Ex. ); (c) using Eq. 7.35; (d) using Eq. 7.34 (take as your volume the cylindrical tube from radius a < R out to radius b > R).

\(W=\frac{1}{2} L I^{2}\) (7.30)

\(W=\frac{1}{2} I \oint A \cdot d l =\frac{1}{2} \oint( A \cdot I ) d l\) (7.31)

\(W=\frac{1}{2 \mu_{0}} \int_{\text {all space }} B^{2} d \tau\) (7.35)

\(W=\frac{1}{2 \mu_{0}}\left[\int B^{2} d \tau-\int \nabla \cdot( A \times B ) d \tau\right]=\frac{1}{2 \mu_{0}}\left[\int_{\nu } B^{2} d \tau-\oint_{ S }( A \times B ) \cdot d a \right]\) (7.34)

Admin
Solution

(a) \(W=\frac{1}{2} L I^{2} . L=\mu_{0} n^{2} \pi R^{2} l\) () \(W=\frac{1}{2} \mu_{0} n^{2} \pi R^{2} l I^{2}\) .

(b) \(W=\frac{1}{2} \oint( A \cdot I ) d l . A =\left(\mu_{0} n I / 2\right) R \hat{\phi}\) , at the surface (Eq. 5.72 or 5.73). So \( W_{1}=\frac{1}{2} \frac{\mu_{0} n I}{2} R I \cdot 2 \pi R\) , for oneturn. There are nl such turns in length l, so \(W=\frac{1}{2} \mu_{0} n^{2} \pi R^{2} l I^{2}\) .

\(A =\frac{\mu_{0} n I}{2} s \hat{ \phi }, \quad \text { for } s \leq R \) (5.72)

\(A =\frac{\mu_{0} n I}{2} \frac{R^{2}}{s} \hat{ \phi }, \quad \text { for } s \geq R\) (5.73)

(c) \(W=\frac{1}{2 \mu_{0}} \int B^{2} d \tau . \quad B=\mu_{0} n I\) , inside, and zero outside; \(\int d \tau=\pi R^{2} l, \text { so } W=\frac{1}{2 \mu_{0}} \mu_{0}^{2} n^{2} I^{2} \pi R^{2} l= \frac{1}{2} \mu_{0} n^{2} \pi R^{2} l I^{2}\) .

(d) \(W=\frac{1}{2 \mu_{0}}\left[\int B^{2} d \tau-\oint( A \times B ) \cdot d a \right] . \text { This time } \int B^{2} d \tau=\mu_{0}^{2} n^{2} I^{2} \pi\left(R^{2}-a^{2}\right) l . \text { Meanwhile }\) ,

\(A \times B = 0 \text { outside }(\text { at } s=b) \text {. Inside, } A =\frac{\mu_{0} n I}{2} a \hat{\phi}(\text { at } s=a) \text {, while } B =\mu_{0} n I \hat{ z }\) .

\(A \times B =\frac{1}{2} \mu_{0}^{2} n^{2} I^{2} a(\underbrace{\hat{ \phi } \times \hat{ z }}_{\hat{ s }})\)

 

\( \overset{\text { points inward ("out" of the volume) }}{\swarrow } \)

\(\oint( A \times B ) \cdot d a =\int\left(\frac{1}{2} \mu_{0}^{2} n^{2} I^{2} a \hat{ s }\right) \cdot[a d \phi d z(-\hat{ s })]=-\frac{1}{2} \mu_{0}^{2} n^{2} I^{2} a^{2} 2 \pi l\).

\(W=\frac{1}{2 \mu_{0}}\left[\mu_{0}^{2} n^{2} I^{2} \pi\left(R^{2}-a^{2}\right) l+\mu_{0}^{2} n^{2} I^{2} \pi a^{2} l\right]=\frac{1}{2} \mu_{0} n^{2} I^{2} R^{2} \pi l\) .
