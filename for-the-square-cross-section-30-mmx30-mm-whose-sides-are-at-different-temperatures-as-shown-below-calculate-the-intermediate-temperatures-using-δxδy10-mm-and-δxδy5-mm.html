Guest Question.

For the square cross-section (30 mm×30 mm) whose sides are at different temperatures as shown below, calculate the intermediate temperatures using Δx=Δy=10 mm and Δx=Δy=5 mm.

Admin
Solution

For Δx=Δy=10 mm, the grid is shown:
Since all the four nodes 1, 2, 3, and 4 are interior nodes, one can apply Equation
\(T_{m+1,n}+T_{m-1,n}+T_{m,n+1}+T_{m,n-1}-4T_{m,n}=0\)
to each one of them as:
\(Node 1: T_{2}+T_{4}+100+100-4T_{1} =0\)
\(Node 2: T_{3}+T_{1}+100+200-4T_{2} =0\)
\(Node 3: T_{4}+T_{2}+500+200-4T_{3} =0\)
\(Node 4: T_{3}+T_{1}+100+500-4T_{4} =0\)
The solution of these four equations gives:
\(T_{1}=162.5^{\circ } C ,T_{2}=187.5^{\circ } C ,T_{3}=287.5^{\circ } C, T_{4}=262.5^{\circ } C \)
For the case of Δx = Δy = 5mm, the grid is as shown:
One can now apply Equation
\(T_{m+1,n}+T_{m-1,n}+T_{m,n+1}+T_{m,n-1}-4T_{m,n}=0\)
to nodes 1–25 resulting in the following 25 algebraic equations:
\(T_{2}+T_{10}+100+100-4T_{1} =0\)
\(T_{1}+T_{9}+T_{3}+100-4T_{2} =0\)
\(T_{2}+T_{8}+T_{4}+100-4T_{3} =0\)
\(T_{3}+T_{7}+T_{5}+100-4T_{4} =0\)
\(T_{4}+T_{6}+100+200-4T_{5} =0\)
\(T_{5}+T_{7}+T_{15}+200-4T_{6} =0\)
\(T_{6}+T_{4}+T_{8}+T_{14}-4T_{7} =0\)
\(T_{7}+T_{3}+T_{13}+T_{9}-4T_{8} =0\)
\(T_{8}+T_{2}+T_{10}+T_{12}-4T_{9} =0\)
\(T_{11}+T_{9}+T_{1}+100-4T_{10} =0\)
\(T_{10}+T_{20}+T_{12}+100-4T_{11} =0\)
\(T_{11}+T_{19}+T_{9}+T_{13}-4T_{12} =0\)
\(T_{12}+T_{18}+T_{8}+T_{14}-4T_{13} =0\)
\(T_{13}+T_{17}+T_{7}+T_{15}-4T_{14} =0\)
\(T_{14}+T_{16}+T_{6}+200-4T_{15} =0\)
\(T_{15}+T_{25}+T_{17}+200-4T_{16} =0\)
\(T_{16}+T_{24}+T_{14}+T_{18}-4T_{17} =0\)
\(T_{17}+T_{23}+T_{13}+T_{19}-4T_{18} =0\)
\(T_{20}+T_{22}+T_{12}+T_{18}-4T_{19} =0\)
\(T_{19}+T_{21}+T_{11}+100-4T_{20} =0\)
\(T_{20}+500+100+T_{22}-4T_{21} =0\)
\(T_{21}+500+T_{19}+T_{23}-4T_{22} =0\)
\(T_{22}+500+T_{24}+T_{18}-4T_{23} =0\)
\(T_{23}+500+T_{17}+T_{25}-4T_{24} =0\)
\(T_{24}+T_{16}+500+200-4T_{25} =0\)
