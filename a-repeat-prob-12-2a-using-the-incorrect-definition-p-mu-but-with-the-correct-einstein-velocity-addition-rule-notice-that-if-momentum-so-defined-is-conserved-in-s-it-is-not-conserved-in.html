Guest Question.

(a) Repeat using the (incorrect) definition p = mu, but with the (correct) Einstein velocity addition rule. Notice that if momentum (so defined) is conserved in S, it is not conserved in \(\overline{ S }\). Assume all motion is along the x axis.

(b) Now do the same using the correct definition, p = mη. Notice that if momentum (so defined) is conserved in S, it is automatically also conserved in \(\overline{ S }\). [Hint: Use Eq. 12.43 to transform the proper velocity.] What must you assume about relativistic energy?

\(\left. \begin{matrix} \bar{\eta}^{0}=\gamma\left(\eta^{0}-\beta \eta^{1}\right), \\ \bar{\eta}^{1}=\gamma\left(\eta^{1}-\beta \eta^{0}\right), \\ \bar{\eta}^{2}=\eta^{2}, \\ \bar{\eta}^{3}=\eta^{3}. \end{matrix} \right\} \) (12.43)

Admin
Solution

\(\text { (a) } m_{A} u_{A}+m_{B} u_{B}=m_{C} u_{C}+m_{D} u_{D} ; \quad u_{i}=\frac{u_{i}+v}{1+\left(\bar{u}_{i} v / c^{2}\right)}\).

\(m_{A} \frac{\bar{u}_{A}+v}{1+\left(\bar{u}_{A} v / c^{2}\right)}+m_{B} \frac{\bar{u}_{B}+v}{1+\left(\bar{u}_{B} v / c^{2}\right)}=m_{C} \frac{\bar{u}_{C}+v}{1+\left(\bar{u}_{C} v / c^{2}\right)}+m_{D} \frac{\bar{u}_{D}+v}{1+\left(\bar{u}_{D} v / c^{2}\right)}\).

This time, because the denominators are all different, we cannot conclude that

\(m_{A} \bar{u}_{A}+m_{B} \bar{u}_{B}=m_{C} \bar{u}_{C}+m_{D} \bar{u}_{D}\).

As an explicit counterexample, suppose all the masses are equal, and \(u_{A}=-u_{B}=v, u_{C}=u_{D}=0\) . This is a symmetric “completely inelastic” collision in S, and momentum is clearly conserved (0=0). But the Einstein velocity addition rule gives \(\bar{u}_{A}=0, \bar{u}_{B}=-2 v /\left(1+v^{2} / c^{2}\right), \bar{u}_{C}=\bar{u}_{D}=-v, \text { so in } \overline{ S }\) the (incorrectly defined) momentum is not conserved:

\(m\left(\frac{-2 v}{1+v^{2} / c^{2}}\right) \neq-2 m v\).

\(\text { (b) } m_{A} \eta_{A}+m_{B} \eta_{B}=m_{C} \eta_{C}+m_{D} \eta_{D} ; \quad \eta_{i}=\gamma\left(\bar{\eta}_{i}+\beta \bar{\eta}_{i}^{0}\right)\) . (The inverse Lorentz transformation.)

\(m_{A} \gamma\left(\bar{\eta}_{A}+\beta \bar{\eta}_{A}^{0}\right)+m_{B} \gamma\left(\bar{\eta}_{B}+\beta \bar{\eta}_{B}^{0}\right)=m_{C} \gamma\left(\bar{\eta}_{C}+\beta \bar{\eta}_{C}^{0}\right)+m_{D} \gamma\left(\bar{\eta}_{D}+\beta \bar{\eta}_{D}^{0}\right)\) . The gamma’s cancel:

\(m_{A} \bar{\eta}_{A}+m_{B} \bar{\eta}_{B}+\beta\left(m_{A} \bar{\eta}_{A}^{0}+m_{B} \bar{\eta}_{B}^{0}\right)=m_{C} \bar{\eta}_{C}+m_{D} \bar{\eta}_{D}+\beta\left(m_{C} \bar{\eta}_{C}^{0}+m_{D} \bar{\eta}_{D}^{0}\right)\).

\(\text { But } m_{i} \eta_{i}^{0}=p_{i}^{0}=E_{i} / c, \text { so if } \text { energy is conserved in } \overline{ S }\left(\bar{E}_{A}+\bar{E}_{B}=\bar{E}_{C}+\bar{E}_{D}\right)\) , then so too is the momentum (correctly defined):

\(m_{A} \bar{\eta}_{A}+m_{B} \bar{\eta}_{B}=m_{C} \bar{\eta}_{C}+m_{D} \bar{\eta}_{D} \cdot \quad \text { qed }\)
