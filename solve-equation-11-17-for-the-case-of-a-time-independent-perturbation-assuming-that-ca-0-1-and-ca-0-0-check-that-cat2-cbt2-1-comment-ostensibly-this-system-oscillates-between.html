Guest Question.

Solve Equation 11.17 for the case of a time-independent perturbation, assuming that \( c_{a}(0)=1 \quad \text { and } \quad c_{b}(0)=0 \) Check that \( \left|c_{a}(t)\right|^{2}+\left|c_{b}(t)\right|^{2}=1 \) . Comment: Ostensibly, this system oscillates between “pure \( \psi_{a} " \text { and "some } \psi_{b} \) ” Doesn’t this contradict my general assertion that no transitions occur for time-independent perturbations? No, but the reason is rather subtle: In this case \( \psi_{a} \text { and } \psi_{b} \) are not, and never were, eigenstates of the Hamiltonian—a measurement of the energy never yields \( E_{a} \text { or } E_{b} \). In time-dependent perturbation theory we typically contemplate turning on the perturbation for a while, and then turning it off again, in order to examine the system. At the beginning, and at the end, \( \psi_{a} \text { and } \psi_{b} \) are eigenstates of the exact Hamiltonian, and only in this context does it make sense to say that the system underwent a transition from one to the other. For the present problem, then, assume that the perturbation was turned on at time t = 0, and off again at time T —this doesn’t affect the calculations, but it allows for a more sensible interpretation of the result.

\( \dot{c}_{a}=-\frac{i}{\hbar} H_{a b}^{\prime} e^{-i \omega_{0} t} c_{b}, \quad \dot{c}_{b}=-\frac{i}{\hbar} H_{b a}^{\prime} e^{i \omega_{0} t} c_{a} \) (11.17).

Admin
Solution

\( \dot{c}_{a}=-\frac{i}{\hbar} H_{a b}^{\prime} e^{-i \omega_{0} t} c_{b} ; \quad \dot{c}_{b}=-\frac{i}{\hbar} H_{b a}^{\prime} e^{i \omega_{0} t} c_{a} \). Differentiating with respect to t :

\( \ddot{c}_{b}=-\frac{i}{\hbar} H_{b a}^{\prime}\left[i \omega_{0} e^{i \omega_{0} t} c_{a}+e^{i \omega_{0} t} \dot{c}_{a}\right]=i \omega_{0}\left[-\frac{i}{\hbar} H_{b a}^{\prime} e^{i \omega_{0} t} c_{a}\right]-\frac{i}{\hbar} H_{b a}^{\prime} e^{i \omega_{o} t}\left[-\frac{i}{\hbar} H_{a b}^{\prime} e^{-i \omega_{0} t} c_{b}\right] \), or

\( \ddot{c}_{b}=i \omega_{0} \dot{c}_{b}-\frac{1}{\hbar^{2}}\left|H_{a b}^{\prime}\right|^{2} c_{b} . \quad \text { Let } \alpha^{2} \equiv \frac{1}{\hbar^{2}}\left|H_{a b}^{\prime}\right|^{2} . \text { Then } \ddot{c}_{b}-i \omega_{0} \dot{c}_{b}+\alpha^{2} c_{b}=0 . \)

This is a linear differential equation with constant coefficients, so it can be solved by a function of the form \( c_{b}=e^{\lambda t} \):

\( \lambda^{2}-i \omega_{0} \lambda+\alpha^{2}=0 \Longrightarrow \lambda=\frac{1}{2}\left[i \omega_{0} \pm \sqrt{-\omega_{0}^{2}-4 \alpha^{2}}\right]=\frac{i}{2}\left(\omega_{0} \pm \omega\right), \text { where } \omega \equiv \sqrt{\omega_{0}^{2}+4 \alpha^{2}} \).

The general solution is therefore

\( c_{b}(t)=A e^{i\left(\omega_{0}+\omega\right) t / 2}+B e^{i\left(\omega_{0}-\omega\right) t / 2}=e^{i \omega_{0} t / 2}\left(A e^{i \omega t / 2}+B e^{-i \omega t / 2}\right) \), or

\( c_{b}(t)=e^{i \omega_{0} t / 2}[C \cos (\omega t / 2)+D \sin (\omega t / 2)] \). \( \text { But } c_{b}(0)=0, \quad \text { so } C= 0\), and hence

\( c_{b}(t)=D e^{i \omega_{0} t / 2} \sin (\omega t / 2) \). Then

\( \dot{c}_{b}=D\left[\frac{i \omega_{0}}{2} e^{i \omega_{0} t / 2} \sin (\omega t / 2)+\frac{\omega}{2} e^{i \omega_{0} t / 2} \cos (\omega t / 2)\right]=\frac{\omega}{2} D e^{i \omega_{0} t / 2}\left[\cos (\omega t / 2)+i \frac{\omega_{0}}{\omega} \sin (\omega t / 2)\right]=-\frac{i}{\hbar} H_{b a}^{\prime} e^{i \omega_{0} t} c_{a} \).

\( c_{a}=\frac{i \hbar}{H_{b a}^{\prime}} \frac{\omega}{2} e^{-i \omega_{0} t / 2} D\left[\cos (\omega t / 2)+i \frac{\omega_{0}}{\omega} \sin (\omega t / 2)\right] . \quad \text { But } c_{a}(0)=1, \quad \text { so } \quad \frac{i \hbar}{H_{b a}^{\prime}} \frac{\omega}{2} D=1 \). Conclusion:

\( c_{a}(t)=e^{-i \omega_{0} t / 2}\left[\cos (\omega t / 2)+i \frac{\omega_{0}}{\omega} \sin (\omega t / 2)\right], \)

\( c_{b}(t)=\frac{2 H_{b a}^{\prime}}{i \hbar \omega} e^{i \omega_{0} t / 2} \sin (\omega t / 2) \),

where

\( \omega \equiv \sqrt{\omega_{0}^{2}+4 \frac{\left|H_{a b}^{\prime}\right|^{2}}{\hbar^{2}}} \).

\( \left|c_{a}\right|^{2}+\left|c_{b}\right|^{2}=\cos ^{2}(\omega t / 2)+\frac{\omega_{0}^{2}}{\omega^{2}} \sin ^{2}(\omega t / 2)+\frac{4\left|H_{a b}^{\prime}\right|^{2}}{\hbar^{2} \omega^{2}} \sin ^{2}(\omega t / 2) \).

\( =\cos ^{2}(\omega t / 2)+\frac{1}{\omega^{2}}\left(\omega_{0}^{2}+4 \frac{\left|H_{a b}^{\prime}\right|^{2}}{\hbar^{2}}\right) \sin ^{2}(\omega t / 2)=\cos ^{2}(\omega t / 2)+\sin ^{2}(\omega t / 2)=1 \).

[In light of the Comment you might question the initial conditions. If the perturbation includes a factor θ(t) , are we sure this doesn’t alter \( c_{a}(0) \text { and } c_{b}(0) \)? That is, are we sure \( c_{a}(t) \text { and } c_{b}(t) \) are continuous at a step function potential? The answer is “yes”, for if we integrate Eq. 11.17 from \( -\epsilon \operatorname{to} \epsilon \),

\( \dot{c}_{a}=-\frac{i}{\hbar} H_{a b}^{\prime} e^{-i \omega_{0} t} c_{b}, \quad \dot{c}_{b}=-\frac{i}{\hbar} H_{b a}^{\prime} e^{i \omega_{0} t} c_{a} \) (11.17).

\( c_{a}(\epsilon)-c_{a}(-\epsilon)=-\frac{i}{\hbar} H_{a b}^{\prime} \int_{0}^{\epsilon} e^{-i \omega_{0} t} c_{b}(t) d t \) \( \text { But }\left|c_{b}(t)\right| \leq 1 \text {, so the integral goes to zero as } \epsilon \rightarrow 0 \text {, and hence } c_{a}(-\epsilon)=c_{a}(\epsilon) \text {. The same goes for } c_{b} \text {, of course.] } \)
