Guest Question.

Solve the second-order wave equation in characteristic coordinates,

\(u_{\xi n}\) = 0

subject to initial data

\( u\left(0,\eta \right) \) = \(\varphi \left(\eta \right) \) \( u_{\xi }\left(0,\eta \right)\) = \( \psi \left(\eta \right)\)

Admin
Solution

The characteristics of the governing PDE are defined by ξ = constant and η = constant. In this case, the initial data are prescribed along a characteristic.
Suppose we attempt to write a Taylor-series expansion in ξ to obtain a solution for u in the neighborhood of the initial data surface ξ = 0. Our solution must be in the form

\(u\left(\xi ,\eta \right) \) = \( u\left(0,\eta \right) + \xi u_{\xi } \left(0,\eta \right) + \frac{\xi^{2} }{2} u_{\xi \xi } \left(0,\eta \right) +\square \)

From the given initial data, u(0, η) and \( u_{\xi }\) (0, η) are known. It remains to determine \( u_{\xi \xi }\)(0, η). The governing differential equation requires

\( u_{\xi n}\) (0, η) = 0

However, we already have the condition that

\( u_{\xi n}\) (0,η)= \(\psi^{\prime } \left(\eta \right) \) = 0

Therefore,

\(\psi (η)\) = constant = \( c_{1}\)

We may also write

\(\frac{\partial u_{\xi \eta } }{\partial \xi } \) = \(\frac{\partial u_{\xi \xi } }{\partial \eta } \) = 0

Integration of this equation yields

\(u_{\xi \xi } \) = f (\( \xi \))

In view of the given initial data, we conclude that

\( u_{\xi \xi }\) (0, η) = constant = \( c_{2}\)

and

\(u\left(\xi,\eta\right) \) = \( \varphi \left(\eta \right) + \xi c_{1} + \frac{\xi ^{2} }{2} c_{2}\)

or

\(u\left(\xi,\eta\right) \) = \( \varphi \left(\eta \right) + g\left(\xi\right) \)

We are unable to uniquely determine the function g(ξ) when the initial data are given along the characteristic ξ = 0.
