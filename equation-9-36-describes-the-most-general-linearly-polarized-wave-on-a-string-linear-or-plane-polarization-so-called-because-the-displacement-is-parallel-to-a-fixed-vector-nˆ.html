Guest Question.

Equation 9.36 describes the most general linearly polarized wave on a string. Linear (or “plane”) polarization (so called because the displacement is parallel to a fixed vector \(\hat{ n }\)) results from the combination of horizontally and vertically polarized waves of the same phase (Eq. 9.39). If the two components are of equal amplitude, but out of phase by \(90^{\circ}\left(\text { say }, \delta_{v}=0, \delta_{h}=90^{\circ}\right)\) , the result is a circularly polarized wave. In that case:

\(\tilde{ f }(z, t)=\tilde{A} e^{i(k z-\omega t)} \hat{ n }\) (9.36)

\(\tilde{ f }(z, t)=(\tilde{A} \cos \theta) e^{i(k z-\omega t)} \hat{ x }+(\tilde{A} \sin \theta) e^{i(k z-\omega t)} \hat{ y }\) (9.39)

(a) At a fixed point z, show that the string moves in a circle about the z axis. Does it go clockwise or counterclockwise, as you look down the axis toward the origin? How would you construct a wave circling the other way? (In optics, the clockwise case is called right circular polarization, and the counterclockwise, left circular polarization.\()^{3}\)

(b) Sketch the string at time t = 0.

(c) How would you shake the string in order to produce a circularly polarized wave?

Admin
Solution

\(\text { (a) } f _{v}(z, t)=A \cos (k z-\omega t) \hat{ x } ; f _{h}(z, t)=A \cos \left(k z-\omega t+90^{\circ}\right) \hat{ y }=-A \sin (k z-\omega t) \hat{ y } \text {. Since } f_{v}^{2}+f_{h}^{2}=A^{2}\) , the vector sum \(f = f _{v}+ f _{h}\) lies on a circle of radius A. At time \(t=0, f =A \cos (k z) \hat{ x }-A \sin (k z) \hat{ y }\) . At time \(t=\pi / 2 \omega, f =A \cos \left(k z-90^{\circ}\right) \hat{ x }-A \sin \left(k z-90^{\circ}\right) \hat{ y }=A \sin (k z) \hat{ x }+A \cos (k z) \hat{ y }\) . Evidently it circles counterclockwise . To make a wave circling the other way, use \(\delta_{h}=-90^{\circ}\) .

(c) Shake it around in a circle, instead of up and down.
