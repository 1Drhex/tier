Guest Question.

An HSS 7 \(\times\) 0.125 with an effective length of 13 feet is filled with normal-weight concrete and used as a column. No longitudinal reinforcement is used. The concrete strength is \(f'_{c}= 5 ksi.\) Compute the available compressive strength. \(F_y = 42 ksi.\)

 

Admin
Solution

The following dimensions and properties from Part 1 of the Manual will be needed:
design wall thickness = 0.116 in., \(A_s 5 2.51 in.^{2}, D/t = 60.3 in.^{4}, and I_s = 14.9 in.^{4}.\)
The following values will also be required:

\(A_c = \frac {\pi d_{inside}^{2}}{4}= \frac {\pi(7.000 – 2 \times 0.116)^{2}}{4} = 35.98 in.^{2}\)

 

\(E_c =w^{1.5}\sqrt{f'_{c}}=(145)^{1.5}\sqrt{5}= 3904 ksi\)

 

\(I_c = \frac {\pi d_{inside}^{4}}{64} =\frac {\pi(7.000 – 2 \times 0.116)^{4}}{64} = in.^{4}\)
Determine the classification of the section:

\(\lambda =\frac{D}{t}=60.3\)

\(\lambda _{p} = \frac{0.15E}{F_y}=\frac{ 0.15(29,000)}{42} = 103.6\)
Since \(\lambda < \lambda _{p}, \)the section is compact, and from AISC Equation I2-9b,
\(P_{no} = P_p = F_{y}A_{s} + C_2 f'_{c}\left ( A_c + A_{sr} \frac{E_s}{E_c}\right )\)

 

\(= 42(2.51) + 0.95(5)(35.98 + 0) = 276.3 kips\)
Next, compute \((EI)_{eff} \)and \(P_e. \)From AISC Equation I2-13,

 

\(C_3 = 0.45 + 3\left (\frac {A_s + A_{sr}}{Ag } \right )\leq 0.9\)

 

\(= 0.45 + 3\left (\frac {2.51 + 0}{\pi (7)^{2}/4 } \right )= 0.6457 < 0.9\)
and from AISC Equation I2-12,

 

\((EI)_{eff} = E_{s}I_{s} + E_{s}I_{sr} + C_{3}E_{c} I_{c}\)

 

\(= 29,000(14.9) + 0 + 0.6457(3904)(103.0) = 6.917 3 105 kip-in.^{2}\)

From AISC Equation I2-5,
\(P_e =\frac{\pi^{2} (EI)_{eff}}{(L_c)^{2}} = \frac{\pi^{2} (6.917 \times 10^{5})}{(13 \times 12)^{2}}= 280.5 kips\)
Determine \(P_n:\)
\(\frac{P_{no}}{P_e}=\frac{276.3}{280.5} = 0.9850 < 2.25\) ∴ Use AISC Equation I2-2.

 

\(P_n = P_{no}\left [ 0.658^{\left ( \frac{P_{no}}{P_e} \right )} \right ]=276.3 [0.658^{0.9850}]= 183.0 kips\)
For LRFD, the design strength is \(\phi _{c}P_n = 0.75(183.0) = 137 kips.\)
For ASD, the allowable strength is \(P_{n}/\Omega _{c} =\) 183.0/2.00 = 91.5 kips.
By comparison, the strengths for an unfilled HSS7 \(\times\) 0.125 are
\(\phi _{c}P_n= 73.7 kips \)and
\(P_{n}/\Omega _{c} = 49.0 kips\)
Although these values are less than those for the filled shape, this will not always be the case.
