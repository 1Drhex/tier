Guest Question.

Consider the fixed-ended beam illustrated in Figure 5.21 of and calculate the support reactions induced by temperature changes such as those described in Figure 5.29b and in Equations 5.45 and 5.46. Assume the beam is of uniform cross-section with area A, second moment of area I, elastic modulus E and coefficient of thermal expansion α.

\( ε_T = ε_{T,r} − y_{κ_T} \) (5.45).

\( \varepsilon _{T.r} =\biggl[T_1 + (T_2 - T_1)\frac{y_t}{h}\biggr] \) \( κ_T = α \frac{T_2 - T_1}{h} \) (5.46a,b).

Solution

The reactions are calculated by solving the system of differential equations (Equations 5.41) and enforcing equilibrium of the free-body diagrams of the two member end nodes.

\( EIν^{IV} − w = 0 \) (5.41a).

\( EAu^{\prime \prime } + n = 0 \) (5.41b).

(1) Boundary conditions: u(x = 0) = u(x = L) = 0.

v (x = 0) = v(x = L) = 0 , \( v^{\prime }(x = 0) = v^{\prime }(x = L) = 0 \) .

(2) Determine constants of integration and expressions for displacements:

\( v^{IV} = 0; v^{\prime \prime \prime } = C_{v1} \); \( v^{\prime \prime } = C_{v1}x + C_{v2} \).

\( v^{\prime } = C_{v1} \frac{x^2}{2} + C_{v2}x + C_{v3} \);

\( v = C_{v1} \frac{x^3}{6} + C_{v2}\frac{x^2}{2} + C_{v3} x + C_{v4} \);

\( u^{\prime \prime} = 0 \); \( u^{\prime } = C_{u1} \);

\( u = C_{u1}x + C_{u2} \).

After applying the relevant boundary conditions:

\( v(x = 0) = C_{v4} = 0 \) ∴ \( C_{v4} = 0 \) \( v^{\prime } (x = 0) = C_{v3}= 0 \) ∴ \( C_{v3} = 0 \).

\( v(x = L) = C_{v1} \frac{L^3}{6} + C_{v2} \frac{L^2}{2} = 0 \) ∴ \( C_{v1} = 0 \).

\( v^{\prime }(x = L) = C_{v1} \frac{L^2}{2} + C_{v2} L = 0 \) ∴ \( C_{v1} = 0 \).

\( u(x = 0) = C_{u2} = 0 \) ∴\( C_{u2} = 0 u(x = L) = C_{u1}L + C_{u2} = 0 \) ∴ \( C_{u1} = 0 \).

we get the following expressions for u and v: u = 0 and v = 0.
Based on these, the variations of N and M become (Equations 5.52a and 5.53a)

\( N = EA u^{\prime } − EAε_{T,r} \). (5.52,a).

\( M = EI v^{\prime \prime} − EIε_{T} \). (5.53,a).

\( N = −EAε_{T,r} \); \( M = −EIκ_T \) (5.57a,b)

(3) The reactions are calculated from equilibrium at the two end nodes (Figures 5.33a and b) and are summarised in Figure 5.33c.

\( H_A = −N(x = 0) = −(−EAε_{T,r}) = EAε_{T,r} \) \( V_A = −S(x = 0) = EIv^{\prime \prime \prime} (x = 0) = 0 \).

\( M_A = −M(x = 0) = −(−EIκ_T) = EIκ_T \) \( H_B = N(x = L) = −EAε_{T,r} \).

\( V_B = S(x = L) = −EIv^{\prime \prime \prime}(x = L) = 0 \) \( M_B = M (x = L) = −EIκ_T \) .
