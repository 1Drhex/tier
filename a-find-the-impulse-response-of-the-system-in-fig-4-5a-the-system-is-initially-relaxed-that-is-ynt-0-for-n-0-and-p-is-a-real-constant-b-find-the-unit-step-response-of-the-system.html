Guest Question.

(a) Find the impulse response of the system in Fig. 4.5a. The system is initially relaxed, that is, y(nT) = 0 for n < 0, and p is a real constant. (b) Find the unit-step response of the system.

Admin
Solution

(a) From (a), the system is characterized by the difference equation

 

\(y(n T)=x(n T)+p y(n T-T)\) (4.16)

 

With \(x(n T)=\delta(n T)\), we can write

 

\(\begin{gathered}y(0)=1+p y(-T)=1 \\y(T)=0+p y(0)=p \\y(2 T)=0+p y(T)=p^{2} \\\ldots \ldots \ldots \ldots \ldots \ldots \ldots . \\y(n T)=p^{n}\end{gathered}\)

 

and since \(y(n T)=0 \text { for } n \leq 0\), we have

 

\(y(n T)=u(n T) p^{n}\) (4.17)

 

The impulse response is plotted in Fig. 4.10 for \(p<1, p=1, \text { and } p>1\). We note that the impulse response diverges if \(p>1\).

 

(b) With \(x(n T)=u(n T)\), we get

 

\(\begin{aligned}y(0) &=1+p y(-T)=1 \\y(T) &=1+p y(0)=1+p \\y(2 T) &=1+p y(T)=1+p+p^{2} \\\ldots \ldots & \ldots \ldots \ldots \ldots \ldots \ldots \ldots \ldots \\y(n T) &=u(n T) \sum_{k=0}^{n} p^{k}\end{aligned}\)

 

This is a geometric series with common ratio p and hence we can write

 

\(y(n T)-p y(n T)=u(n T)\left(1-p^{(n+1)}\right)\)

 

or

 

\(y(n T)=u(n T) \frac{1-p^{(n+1)}}{1-p}\) (4.18)

 

For \(p<1, \lim _{n \rightarrow \infty} p^{(n+1)} \rightarrow 0\) and hence the steady-state value of the response is obtained as

 

\(\lim _{n \rightarrow \infty} y(n T)=\frac{1}{1-p}\)

 

For p = 1, Eq. (4.18) gives y(nT) = 0/0 but if we apply l’Hˆopital’s rule, we obtain

 

\(y(n T)=\lim _{p \rightarrow 1} \frac{d\left(1-p^{(n+1)}\right) / d p}{d(1-p) / d p}=n+1\)

 

Thus \(y(n T) \rightarrow \infty \text { as } n \rightarrow \infty\).

 

\(\lim _{n \rightarrow \infty} y(n T) \approx \frac{p^{n}}{p-1} \rightarrow \infty\)

 

The unit-step response for the three values of p is illustrated in Fig. 4.11. Evidently, the response converges if p < 1 and diverges if p ≥ 1.
