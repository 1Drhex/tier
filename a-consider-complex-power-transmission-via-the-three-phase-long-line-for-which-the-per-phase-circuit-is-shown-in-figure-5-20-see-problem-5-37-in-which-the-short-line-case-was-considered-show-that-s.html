Guest Question.

(a) Consider complex power transmission via the three-phase long line for which the per-phase circuit is shown in Figure 5.20. See in which the short-line case was considered. Show that

sending­end power \(=S_{12}=\frac{Y^{\prime *}}{2} V _{1}^{2}+\frac{ V _{1}^{2}}{Z^{\prime *}}-\frac{ V _{1} V _{2}}{Z^{\prime *}} e^{j \theta_{12}}\)

received power \(=-S_{21}=-\frac{Y^{\prime *}}{2} V _{2}^{2}-\frac{ V _{2}^{2}}{Z^{\prime *}}+\frac{ V _{1} V _{2}}{Z^{\prime *}} e^{-j \theta_{12}}\)

where \(θ_{12}= θ_1-θ_2\).

(b) For a lossless line with equal voltage magnitudes at each end, show that

\(P _{12}=- P _{21}=\frac{ V _{1}^{2} \sin \theta_{12}}{ Z _{ c } \sin \beta \ell}= P _{ SIL } \frac{\sin \theta_{12}}{\sin \beta \ell}\)

(c) For \(\theta_{12}=45^{\circ} \text { and } \beta=0.002\) rad/km, find \(\left( P _{12} / P _{ SIL }\right)\) as a function of line length in km, and sketch it.

(d) If a thermal limit of \(\left( P _{12} / P _{ SIL }\right)=2\) is set, which limit governs for short lines and long lines?

Solution

(a) See (a) solution: Eqs. (1) and (2) with the substitution of \(\bar{Z}^{\prime} \text { for } \bar{Z}\) , adding the contribution of the complex power consumed by \(\bar{Y}^{\prime} / 2\) , using Eq. (1) of (a) solution, one gets

\(\bar{S}_{12}=\frac{\bar{Y}^{\prime *}}{2} V_{1}^{2}+\frac{V_{1}^{2}}{\bar{Z}^{*}}-\frac{V_{1} V_{2}}{\bar{Z}^{*}} e^{j \theta_{12}} \leftarrow\)

Similarly, subtracting the complex power consumed in \(\frac{\bar{Y}^{\prime}}{2}\) (on the right-hand side in Fig. 5.17),

For the received power, one has

\(-\bar{S}_{21}=-\frac{\bar{Y}^{\prime *}}{2} V_{2}^{2}-\frac{V_{2}^{2}}{\bar{Z}^{\prime *}}+\frac{V_{1} V_{2}}{\bar{Z}^{\prime *}} e^{-j \theta_{12}} \leftarrow\)

Except for the additional constant terms, the equations have the same form as those in .

(b) For a lossless line, \(Z_{C}=\sqrt{L / C}\) is purely real and \(\bar{\gamma}=j \beta\) is purely imaginary. Also

\(\bar{Y}^{\prime}=\bar{Y} \frac{\tanh (\bar{\gamma} l / 2)}{\bar{\gamma} l / 2}=j \omega c \frac{\tan (\beta l / 2)}{\beta l / 2} \text { and } \bar{Z}^{\prime}=\bar{Z}_{C} \sinh (\bar{\gamma} l)\)

which becomes \(j Z_{C} \sinh (\beta l)\).

Note: \(\bar{Y}^{\prime}\) is now the admittance of a pure capacitance;

\(\bar{Z}^{\prime}\) is now the impedance of a pure inductance.

Active power transmitted, \(P_{12}=-P_{21}\)

And \(P_{12}=\frac{V_{1}^{2} \sin \theta_{12}}{Z_{C} \sin (\beta l)} \leftarrow\)

Using Eq. (5.4.21) of the text for SIL

\(P_{12}=P_{S L L} \frac{\sin \theta_{12}}{\sin (\beta l)} \leftarrow\)

(c) For \(\beta l=0.002 l \text { radians }=(0.1146 l)^{\circ}, \text { and } \theta_{12}=45^{\circ} \text { , }\)

Applying the result of part (b), one gets

\(\frac{P_{12}}{P_{S I L}}=0.707 \frac{1}{\sin (0.1146 l)^{\circ}} \leftarrow\)

(d)

\(\left.\begin{array}{l}\text{Thermal limit governs the short lines;}\\ \text{Stability limit prevails for long lines.}\end{array}\right\} \leftarrow\)
