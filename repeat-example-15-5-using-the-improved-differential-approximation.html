Guest Question.

Repeat using the improved differential approximation.

Admin
Solution

For radiative equilibrium in a gray medium we have \(G^{*} = 4\pi I^{*}_{b}=4\sigma T^{*4}\) and, therefore, from equation (15.76) \(S^{*}(r,\overset{\wedge }{s} )= (1-w)I^{*}_{b}(r)+\frac{w}{4\pi }[G^{*}(r)+A_{1}q^{*}(r)\cdot \overset{\wedge }{s} ]\) for an isotropically scattering medium \(S^{*} = I^{*}_{b}=\sigma T^{*4}/\pi\), The solution for the ODA has already been found in Chapter 14, Example 14.5, as

\(G^{*}(\tau )=C-3q^{*}\tau =4\pi S^{*}; c=4\sigma T^{4}_{1}-2q^{*}\)

In this particular case, since \(S^{*}\) is linear, equations (15.75)\(I_{m}(r, \overset{\wedge }{s}) \simeq \int_{0}^{\tau _{s}}{}S^{*}(r^{\prime }+ s^{\prime } \overset{\wedge }{s}, \overset{\wedge }{s}) e^{-(\tau _s- \tau ^{\prime }_{s} ) }d\tau ^{\prime }_{s},\) , (15.78) \(\int_{0}^{\tau _{s}}{} S^{*}(r^{\prime }+ s^{\prime } \overset{\wedge }{s}, \overset{\wedge }{s}) e^{-(\tau _{s}-\tau ^{\prime }_{s})}d\tau ^{\prime }_{s} = S^{*}(r, \overset{\wedge }{s})(1-e^{\tau _{s}})-\frac{dS^{*}}{d\tau _{s}} (r, \overset{\wedge }{s}) [1-(1+\tau _{s})e^{-\tau _{s}}]\) , and (15.79) \(\int_{0}^{\tau _{s}}{} S^{*}(r^{\prime }+ s^{\prime } \overset{\wedge }{s}, \overset{\wedge }{s}) e^{-(\tau _{s}-\tau ^{\prime }_{s})}d\tau ^{\prime }_{s}\backsimeq S^{*}(r-s_{0}\overset{\wedge }{s},\overset{\wedge }{s})(1-e^{-\tau _{s}})\) all give the same result; and it is actually a little simpler to evaluate equation (15.78) rather than use equation (15.79). From Fig. 15-8, with \(J_{i} =\sigma T^{4}_{i}/\pi\), and replacing \(\tau _{s}\) (optical distance from point under consideration to point at wall) by \(\tau /\mu (for 0\lt \mu =\cos \theta \lt 1)\), or intensities traveling upward) and \(-(\tau _{L}-\tau ) /\mu (for -1\lt \mu \lt 0 \), or intensities traveling downward) gives

\(I(\tau ,\mu )=\frac{\sigma T^{4}_{1} }{\pi } e^{-\tau /\mu } + S^{*}(1-e^{r/\mu })-\mu \frac{dS^{*}}{d\tau } \left[1-\left(1+\frac{\tau }{\mu } \right) e^{-\tau /\mu }\right]\)
\(=S^{*}+\left[\frac{\sigma T^{4}_{1} }{\pi } – \frac{\sigma T^{4}_{1} }{\pi } + \frac{q^{*}}{2\pi }\left(1+\frac{3}{2}\tau \right) \right] e^{-\tau /\mu } + \frac{3}{4\pi }q^{*}[\mu -(\mu +\tau )e^{-\tau /\mu }] , 0\lt\mu \lt 1;\)
\(= \frac{\sigma T^{4}_{2} }{\pi } e^{(\tau _L-\tau )/\mu }+ S^{*}(1-e^{(\tau _L-\tau )/\mu })-\mu \frac{dS^{*}}{d\tau } \left[1-(1-\frac{\tau _L-\tau}{\mu } )e^{(\tau _L-\tau)/\mu }\right]\)
\(=S^{*}+ \left[\frac{\sigma T^{4}_{2} }{\pi } – \frac{\sigma T^{4}_{1} }{\pi } + \frac{q^{*}}{2\pi }\left(1+\frac{3}{2}\tau \right) \right] e^{(\tau_{L}-\tau ) /\mu } + \frac{3}{4\pi }q^{*}[\mu -(\mu-\tau _{L} +\tau )e^{(\tau _{L}-\tau) /\mu }] , -1\lt\mu \lt 0;\)

The heat flux is then determined as

\(\Psi (\tau )=\frac{q(\tau )}{\sigma (T^{4}_{1}-T^{4}_{2}) }=2\pi \int_{-1}^{1}{}I(\tau ,\mu )\mu d\mu /\sigma (T^{4}_{1}-T^{4}_{2})\)
\(=2E_{3}(\tau _{L}-\tau ) + \Psi ^{*}(1+\frac{3}{2}\tau )[E_{3}(\tau )-E_{3}(\tau _{L}-\tau )]\)
\(+\frac{3}{2} \Psi ^{*} \left[\frac{3}{2}- E_{4}(\tau ) – \tau E_{3}(\tau )- E_{4}(\tau _{L}-\tau ) – (\tau _{L}-\tau )E_{3}(\tau _{L}-\tau )\right] ,\)

which, with \(\Psi ^{*}=q^{*}/\sigma (T^{4}_{1}-T^{4}_{2})=1/(1+\frac{3}{4}\tau _{L} )\) ,reduces to

\(\Psi (\tau )= \frac{1+ E_{3}(\tau )+ E_{3}(\tau _{L}-\tau )-\frac{3}{2}[E_{4}(\tau )+ E_{4}(\tau _{L}-\tau )] }{1+\frac{3}{4}\tau _{L} }\)

This result is also included in Fig. 15-9 and compared with other solution methods. The accuracy of the IDA is very good, and of the same order as the MDA, but the most severe weakness of the IDA is displayed in Fig. 15-9. In this method radiative equilibrium is not satisfied exactly, i.e., the approximate heat flux fluctuates slightly around the exact (constant) value.
