Guest Question.

A particle of mass m is placed in a finite spherical well:

\( V(r)= \begin{cases}-V_{0}, & r \leq a \\ 0, & r>a\end{cases} \).

Find the ground state, by solving the radial equation with \( \ell=0 \). Show that there is no bound state if \( V_{0} a^{2}<\pi^{2} \hbar^{2} / 8 m \).

Admin
Solution

\( \text { For } r \leq a, u(r)=A \sin (k r), \text { with } k \equiv \sqrt{2 m\left(E+V_{0}\right)} / \hbar \text {. For } r \geq a \text {, Eq. } 4.37 \text { with } \ell=0, V=0 \) 0, and (for a bound state) E < 0 ⇒ :

\( -\frac{\hbar^{2}}{2 m} \frac{d^{2} u}{d r^{2}}+\left[V+\frac{\hbar^{2}}{2 m} \frac{\ell(\ell+1)}{r^{2}}\right] u=E u \) (4.37).

\( \frac{d^{2} u}{d r^{2}}=-\frac{2 m}{\hbar^{2}} E u=\kappa^{2} u, \text { with } \kappa \equiv \sqrt{-2 m E} / \hbar \Rightarrow u(r)=C e^{\kappa r}+D e^{-\kappa r} \).

\( \text { But the } C e^{\kappa r} \text { term blows up as } r \rightarrow \infty \text {, so } u(r)=D e^{-\kappa r} \).

\( \left.\begin{array}{l} \text { Continuity of } u \text { at } \quad r=a: A \sin (k a)=D e^{-\kappa a} \\ \text { Continuity of } u^{\prime} \text { at } \quad r=a: A k \cos (k a)=-D \kappa e^{-\kappa a} \end{array}\right\} \text { divide: } \frac{1}{k} \tan (k a)=-\frac{1}{\kappa}, \text { or }-\cot k a=\frac{k}{k} \).

Let \( k a \equiv z ; \quad \frac{\kappa}{k}=\frac{\sqrt{2 m V_{0} a^{2} / \hbar^{2}-z^{2}}}{z} \). Let \( z_{0} \equiv \frac{\sqrt{2 m V_{0}}}{\hbar} a \). \( -\cot z=\sqrt{\left(z_{0} / z\right)^{2}-1} \). This is exactly the same transcendental equation we encountered in -see graph there. There is no solution if \( z_{0}<\pi / 2 \), which is to say, if \( 2 m V_{0} a^{2} / \hbar^{2}<\pi^{2} / 4, \text { or } V_{0} a^{2}<\pi^{2} \hbar^{2} / 8 m \). Otherwise, the ground state energy occurs somewhere between \( z=\pi / 2 \text { and } z=\pi \):

\( E+V_{0}=\frac{\hbar^{2} k^{2} a^{2}}{2 m a^{2}}=\frac{\hbar^{2}}{2 m a^{2}} z^{2} \), so \( \frac{\hbar^{2} \pi^{2}}{8 m a^{2}}<\left(E_{0}+V_{0}\right)<\frac{\hbar^{2} \pi^{2}}{2 m a^{2}} \). (precise value depends on \( V_0 \)).
