Guest Question.

Design an unstiffened seated connection between a W21×62 beam and a W14×61 column flange, to support the following beam end reactions:
\(\begin{array}{l}R_{D}=9 \text { kips } \\R_{L}=27.5 \text { kips }\end{array}\)
Use 3/4-in. diameter ASTM A325-N bolts in standard holes to connect the supported beam to the seat and top angles. Use 70 ksi electrode welds to connect the seat and top angles to the column flange.
Note: For calculation purposes, assume setback is equal to ¾ in. to account for possible beam underrun

Material Properties:
Beam W21×62 ASTM A992 \(F_{v}=50 ksi \quad F_{u}=65 ksi\)
Column W14×61 ASTM A992 \(F_{v}=50 ksi \quad F_{u}=65 ksi\)
Angles ASTM A36 \(F_{y}=36 ksi \quad F_{u}=58 ksi\)

Solution
LRFD ASD
\(R_{u}=1.2(9 kips )+1.6(27.5 kips )=55 kips\)

 

Check the strength of the beam web

 

For local web yielding

 

\(\begin{aligned}N_{\min } &=\frac{R_{u}-\phi R_{1}}{\phi R_{2}} \geq k \\&=\frac{55 kips -55.8 kips }{20 kips } \geq 1.12 in .\end{aligned}\)

 

 

\(N_{\min }=1.12 \text { in. }\)

 

For web crippling,

 

\(\begin{array}{l}\left(\frac{N}{d}\right)_{\max }=\frac{3.25}{21} \\=0.16<0.2 \\\text { When } \frac{N}{d} \leq 0.2 \\N_{\min }=\frac{R_{u}-\phi R_{3}}{\phi R_{4}} \\=\frac{55 \text { kips }-71.7 \text { kips }}{5.37 kips }\end{array}\)

 

which results in a negative quantity.

 

\(\text { Therefore, } N_{\min }=1.12 in .\)

 

Check shear yielding and flexural yielding ofangle. Check local yielding and crippling of beam web

 

Try an 8 in. angle length with a 5/8-in. thickness and a 3 1/2 in. minimum outstanding leg.

 

\(\begin{array}{l}N_{r e q}>11 / 8 \text { in. } \\\text { For } N=11 / 8 \text { in. }\end{array}\)

 

\(\phi R_{n}=81.0 \text { kips }>55 \text { kips }\) o.k.

 

 

 

Try an L8×4×5/8 (4 in. OSL), 8 in. long with 5/16 in. fillet welds

 

\(\phi R_{n}=66.7 \text { kips }>55 \text { kips }\) o.k.

 

Use two 3/4-in. diameter ASTM A325-N bolts to connect the beam to the seat angle.

 \(R_{a}=9 \text { kips }+27.5 \text { kips }=36.5 \text { kips }\)

 

Check the strength of the beam web

 

For local web yielding

 

\(\begin{aligned}N_{\min } &=\frac{R_{a}-R_{1} / \Omega}{R_{2} / \Omega} \geq k \\&=\frac{36.5 kips -37.2 kips }{13.3 kips } \geq 1.12 in\end{aligned}\)
