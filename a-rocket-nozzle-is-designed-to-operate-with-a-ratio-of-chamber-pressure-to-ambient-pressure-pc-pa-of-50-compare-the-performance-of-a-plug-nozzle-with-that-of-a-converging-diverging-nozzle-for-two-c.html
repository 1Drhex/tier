Guest Question.

A rocket nozzle is designed to operate with a ratio of chamber pressure to ambient pressure \(\left( p _{ c } / p _{ a }\right)\) of 50. Compare the performance of a plug nozzle with that of a converging-diverging nozzle for two cases where the nozzle is operating overexpanded; \(p _{ c } / p _{ a }=40 \text { and } p _{ c } / p _{ a }=20\). Compare on the basis of thrust coefficient; \(C _{ T }= T /\left( p _{ c } A _{ th }\right)\), where T is the thrust and \(A_{\text {th }}\) is the area of throat. Assume Î³ = 1.3 and in both cases neglect the effect of nonaxial exit velocity components.

Admin
Solution

For the design case,

From \(p _{ e } / p _{ o }= p _{ a } / p _{ c }=1 / 50=0.02\), since in the design case the flow is isentropic, we can determine the Mach number at the exit, i.e., \(M _{ e }=3.1267\) (see Eq. (3.15)), and therefore \(T _{ e } / T _{ o }= T _{ e } / T _{ c }=0.4054\). Now from the definition of the thrust coefficient

 

\(p_{o}=p\left(1+\frac{\gamma-1}{2} M^{2}\right)^{\gamma /(\gamma-1)}\) (3.15)

 

\(C _{ T }=\frac{\left(\dot{ m }_{ th } V _{ e }\right)}{ p _{ c } A _{ th }}=\frac{\left(\rho_{ th } A _{ th } V _{ th }\right) V _{ e }}{ p _{ c } A _{ th }}=\left(\frac{ p _{ th }}{ RT _{ th }}\right) \frac{ V _{ th } V _{ e }}{ p _{ c }}=\left(\frac{ p _{ th }}{ p _{ c }}\right)\left(\frac{ p _{ c }}{ RT _{ o }}\right)\left(\frac{ T _{ o }}{ T _{ th }}\right) \frac{\left( M _{ th } a _{ th }\right)\left( M _{ e } a _{ e }\right)}{ p _{ c }}\)

 

Because the nozzle is choked, \(M_{\text {th }}=1 \text { and for } \gamma=1.3\),

 

\(\frac{ p _{ th }}{ p _{ c }}=\left(\frac{\gamma+1}{2}\right)^{\frac{\gamma}{1-\gamma}}=0.5457\)

 

\(\frac{ T _{ th }}{ T _{ c }}=\frac{2}{\gamma+1}=0.8696\)

 

Using these values and the values at the exit, we get

 

\(\left( C _{ T }\right)_{\text {design }}=\left[\frac{0.5457 p _{ c }}{ R \left(0.8696 T _{ c }\right)}\right]\left[\frac{\sqrt{(1.3)( R )(0.8696) T _{ c }}}{ p _{ c }}\right](3.1267) \sqrt{(1.3)( R )(0.4054) T _{ c }}\)

 

= 1.5145

 

Note, \(R , p _{ c } \text { and } T _{ c }\) drop out of the above expression.

 

For the converging-diverging nozzle operating off design,

 

\(C _{ T }=\left( C _{ T }\right)_{\text {design }}+\frac{ A _{ e }\left( p _{ e }- p _{ a }\right)}{ A _{ th } p _{ c }}=1.5145+\frac{ A _{ e }}{ A _{ th }}\left(\frac{ p _{ e }}{ p _{ c }}-\frac{ p _{ a }}{ p _{ c }}\right)\)

 

where at \(M _{ e }=3.1267, A _{ e } / A _{ th }= A _{ e } / A ^{*}=5.9590 . \text { So for } p _{ d } / p _{ a }=40\),

 

\(C_{T}=1.5145+5.9590\left(\frac{1}{50}-\frac{1}{40}\right)=1.4847\)

 

And for \(p _{ c } / p _{ a }=20\),

 

\(C _{ T }=1.5145+5.9590(0.02-0.05)=1.3357\)

 

For the plug nozzle,

Flow in the plug nozzle does not continue to expand below ambient pressure, so there is no pressure term in the expression for thrust.

 

Now at \(\frac{ p _{ c }}{ p _{ a }}=40, \quad M _{ e }=2.9918, \quad \text { and } \quad \frac{ T _{ e }}{ T _{ c }}=0.4269\)

 

\(C _{ T }=\frac{\dot{ m }_{ th } V _{ e }}{ p _{ c } A _{ th }}=\left[\frac{0.5457 p _{ c }}{ R \left(0.8696 T _{ c }\right)}\right]\left[\frac{ A _{ th } \sqrt{1.3( R ) 0.8696 T _{ c }}}{ p _{ c } A _{ th }}\right] 2.9918 \sqrt{1.3 R \left(0.4269 T _{ c }\right)}\)

 

= 1.4871

 

Whereas for \(p _{ c } / p _{ a }=20, \quad M _{ e }=2.5773 \text { and } \frac{ T _{ e }}{ T _{ c }}=0.5009\)

 

\(C _{ T }=1.3876\)

 

When these results are compared to those of Example 8.4, it is seen that the effect of changing Î³ from 1.4 to 1.3 is relatively small.
