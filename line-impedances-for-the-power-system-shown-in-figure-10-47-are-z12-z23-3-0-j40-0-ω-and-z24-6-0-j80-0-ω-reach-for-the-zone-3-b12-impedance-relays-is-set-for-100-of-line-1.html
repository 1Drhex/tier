Guest Question.

Line impedances for the power system shown in Figure 10.47 are \(Z _{12}=Z_{23}=3.0+j 40.0 \Omega, \text { and } Z _{24}=6.0+j 80.0 \Omega \text {. }\). Reach for the zone 3 B12 impedance relays is set for 100% of line 1–2 plus 120% of line 2–4. (a) For a bolted three-phase fault at bus 4, show that the apparent primary impedance “seen” by the B12 relays is

\(Z_{\text {apparent }}=Z_{12}+Z_{24}+\left( I _{32} / I _{12}\right) Z_{24}\)

where \(\left(I_{32} / I_{12}\right)\) is the line 2–3 to line 1–2 fault current ratio. (b) If \(\left|I_{32}\right| I_{12} \mid>\) 0.20, does the B12 relay see the fault at bus 4?
Note: This problem illustrates the “infeed effect.” Fault currents from line 2–3 can cause the zone 3 B12 relay to underreach. As such, remote backup of line 2–4 at B12 is ineffective.

Admin
Solution

(a) For the bolted three-phase fault at bus 4, the apparent primary impedance seen by the B12 relay is:

\(\bar{Z}_{a p p a r e n t}=\frac{\bar{V}_{1}}{\bar{I}_{12}}=\frac{\bar{V}_{1}-\bar{V}_{2}+\bar{V}_{2}}{\bar{I}_{12}}\) = \(\frac{\left(\bar{V}_{1}-\bar{V}_{2}\right)}{\underbrace{\bar{I}_{12}}_{\bar{Z}_{12}}}+\frac{\bar{V}_{2}}{\bar{I}_{12}}\)

 

\(\bar{Z}_{\text {apparent }}=\bar{Z}_{12}+\frac{\bar{V}_{2}}{\bar{I}_{12}}\)

 

Using \(\bar{V}_{2}=\bar{Z}_{24} \bar{I}_{24} \text { and } \bar{I}_{24}=\bar{I}_{12}+\bar{I}_{32}:\)

\(\bar{Z}_{a p p a r e n t}=\bar{Z}_{12}+\frac{\bar{Z}_{24}\left(\bar{I}_{12}+\bar{I}_{32}\right)}{\bar{I}_{12}}=\bar{Z}_{12}+\bar{Z}_{24}+\left(\frac{\bar{I}_{32}}{\bar{I}_{12}}\right) \bar{Z}_{24}\) which is the desired result.

 

(b) The apparent secondary impedance seen by the B12 relay. For the bolted three-phase fault at bus 4 is:

\(\bar{Z}_{\text {apparent }}^{\prime}=\frac{\bar{Z}_{\text {apparent }}}{\left(n_{V} / n_{I}\right)}=\frac{(3+j 40)+(6+j 80)+\left(\frac{\bar{I}_{32}}{\bar{I}_{12}}\right)(6+j 80)}{\left(n_{V} / n_{I}\right)}\)

 

\(\bar{Z}_{\text {apparent }}^{\prime}=\frac{\left[9+6\left(\frac{\bar{I}_{32}}{I_{12}}\right)\right]+j\left[120 .+80\left(\frac{\bar{I}_{32}}{\bar{I}_{12}}\right)\right]}{\left(n_{V} / n_{I}\right)} \Omega\)

 

where \(n_{V}\) is the VT ratio and nI is the CT ratio. Also, the B12 zone 3 relay is set with a secondary impedance:

\(\bar{Z}_{r 3}=\frac{(3+j 40)+1.2(6+j 80)}{\left(n_{V} / n_{I}\right)}=\frac{10.2+j 136}{\left(n_{V} / n_{I}\right)} \Omega \text { secondary }\)

Comparing \(\bar{Z}_{r 3}\) with \(\bar{Z}_{\text {apparent }}^{\prime}, \bar{Z}_{\text {apparent }}^{\prime}\) exceeds \(\bar{Z}_{r 3}\) when \(\left(\bar{I}_{32} / \bar{I}_{12}\right)>0.2\) . Hence \(\bar{Z}_{\text {apparent }}^{\prime}\) lies outside the trip region for the three-phase fault at bus 4 when \(\left(\bar{I}_{32} / \bar{I}_{12}\right)>0.2\) ; remote backup of line 2–4 at B12 is then ineffective.
