Guest Question.

The n = 2 Level of Hydrogen

GOAL Count and tabulate distinct quantum states and determine their energy based on atomic energy level.

PROBLEM (a) Determine the number of states with a unique set of values for \(\ell, m_{\ell}, \text { and } m_{s}\) in the hydrogen atom for n = 2. (b) Tabulate the distinct possible quantum states, including spin. (c) Calculate the energies of these states in the absence of a magnetic field, disregarding small differences caused by spin.

STRATEGY This problem is a matter of counting, following the quantum rules for \(n, \ell, m_{\ell}, \text { and } m_{s}\). “Unique” means that no other quantum state has the same set of numbers. The energies—disregarding spin or Zeeman splitting in magnetic fields—are all the same because all states have the same principal quantum number, n = 2.

Admin
Solution

(a) Determine the number of states with a unique set of values for \(\ell \text { and } m_{\ell}\), in the hydrogen atom for n = 2.

Determine the different possible values of \(\ell\) for n = 2:

\(0 \leq \ell \leq n-1 \text {, so for } n=2,0 \leq \ell \leq 1 \text { and } \ell=0 \text { or } 1\)

 

Find the different possible values of \(m_{\ell} \text { for } \ell\) = 0:

\(-\ell \leq m_{\ell} \leq \ell, \text { so }-0 \leq m_{\ell} \leq 0 \text { implies that } m_{\ell}=0\)

 

List the distinct pairs of \(\left(\ell, m_{\ell}\right) \text { for } \ell=0\):

\(\text { There is only one: }\left(\ell, m_{\ell}\right)=(0,0) \text {. }\)

 

Find the different possible values of \(m_{\ell} \text { for } \ell=1\):

\(-\ell \leq m_{\ell} \leq \ell \text {, so }-1 \leq m_{\ell} \leq 1 \text { implies that } m_{\ell}=-1,0 \text {, or } 1\)

 

List the distinct pairs of \(\left(\ell, m_{\ell}\right) \text { for } \ell=1\):

\(\text { There are three: }\left(\ell, m_{\ell}\right)=(1,-1),(1,0) \text {, and }(1,1) \text {. }\)

 

Sum the results for \(\ell=0 \text { and } \ell=1\) and multiply by 2 to account for the two possible spins of each state:

\(\text { Number of states }=2(1+3)=8\)

 

(b) Tabulate the different possible sets of quantum numbers.

Use the results of part (a) and recall that the spin
quantum number is always \(+\frac{1}{2} or -\frac{1}{2}\) .

 

n \(\ell\) \(m_{\ell}\) \(m_{s}\)
2 1 -1 \(-\frac{1}{2}\)
2 1 -1 \(\frac{1}{2}\)
2 1 0 \(-\frac{1}{2}\)
2 1 0 \(\frac{1}{2}\)
2 1 1 \(-\frac{1}{2}\)
2 1 1 \(\frac{1}{2}\)
2 0 0 \(-\frac{1}{2}\)
2 0 0 \(\frac{1}{2}\)

 

(c) Calculate the energies of these states.

The common energy of all the states, disregarding Zeeman splitting and spin, can be found with Equation 28.14:

\(E_{n}=-\frac{13.6 eV }{n^{2}} \rightarrow E_{2}=-\frac{13.6 eV }{2^{2}}=-3.40 eV\)

 

\(E_{n}=-\frac{13.6}{n^{2}} eV\) [28.14]

 

REMARKS Although these states normally have the same energy, the application of a magnetic field causes them to take slightly different energies centered around the energy corresponding to n = 2. In addition, the slight difference in energy due to spin state was neglected.
