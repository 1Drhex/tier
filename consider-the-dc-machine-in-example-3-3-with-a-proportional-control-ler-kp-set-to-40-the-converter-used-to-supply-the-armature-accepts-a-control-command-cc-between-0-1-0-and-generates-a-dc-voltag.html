Guest Question.

Consider the dc machine in with a proportional control-ler \(K_{p}\) set to 40. The converter used to supply the armature accepts a control command (cc) between [0, 1.0] and generates a dc voltage over [0, 400 V] in a linear manner. The converter generates 400 V in response to cc higher than 1.0. Obtain the response of the motor speed to a step reference speed of 200 rad/s with and without enforced limits on the cc. Assume no active load torque on the shaft. Rated armature voltage is 220 V.

Admin
Solution

\(\omega _{m}(s)=\frac{K}{(R_{a}+sL_{a})(B+sJ)+K^{2}}\upsilon _{a}(s)=\frac{0.8}{(0.25+s0.02)(0.05+s2)+0.8^{2}}\upsilon _{a}(s)\)
