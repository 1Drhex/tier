Guest Question.

A roof system consists of trusses of the type shown in Figure 5.51 spaced 15 feet apart. Purlins are to be placed at the joints and at the midpoint of each top-chord member. Sag rods will be located at the center of each purlin. The total gravity load, including an estimated purlin weight, is 42 psf of roof surface, with a live-load– to–dead-load ratio of 1.0. Assuming that this is the critical loading condition, use A36 steel and select a channel shape for the purlins.

Admin
Solution

For the given loading condition, dead load plus a roof live load with no wind or snow, load combination 3 will control:

\(w_{u} = 1.2w_{D} + 1.6L_{r} = 1.2(21) + 1.6(21) = 58.80\) psf

The width of roof surface tributary to each purlin is

\(\frac{15}{2}\frac{\sqrt{10} }{3}=7.906\) psf

Then

Purlin load = \(58.80(7.906) = 464.9\) Ib/ft

Normal component = \(\frac{3 }{\sqrt{10}}(464.9)=441.0\) Ib/ft

Parallel component =\(\frac{1}{\sqrt{10}}(464.9)=147.0\) Ib/ft

and

\(M_{ux}= \frac{1}{8}(0.4410)(15)^{2}=12.40\) ft-kips

With sag rods placed at the midpoint of each purlin, the purlins are two-span continuous beams with respect to weak axis bending. From Table 3-22c, “Continuous Beams,” the maximum moment in a two-span continuous beam with equal spans is at the interior support and is given by

\(M = 0.125w\ell ^{2} \)

where

w = uniform load intensity
\(\ell \) = span length

The maximum moment about the y axis is therefore

\(M_{uy} = 0.125(0.1470)(15/2)^{2} = 1.034\) ft-kips

To select a trial shape, use the beam design charts and choose a shape with a relatively large margin of strength with respect to major axis bending. For an unbraced length of 15⁄2 = 7.5 ft, try a C10 × 15.3.

For \(C_{b} = 1.0\), \(\phi _{b}M_{nx} = 33.0\) ft-kips. From Figure 5.15b, \(C_{b}\) is 1.30 for the load and lateral support conditions of this beam. Therefore,

\( \phi _{b}M_{nx} = 1.30(33.0) = 42.90\) ft-kips

From the uniform load table for C shapes,

\(\phi _{b}M_{px} = 42.9\) ft-kips \(∴ \) Use \(\phi _{b}M_{nx} = 42.9\) ft-kips.

This shape is compact (no footnote in the uniform load tables), so

\(\phi _{b}M_{ny} = \phi _{b}M_{py} = \phi _{b}F_{y}Z_{y} = 0.90(36)(2.34) = 75.82\) in.-kips = 6.318 ft-kips

But

\(\frac{Z_{y} }{S_{y} } =\frac{2.34}{1.15}=2.03\gt 1.6\)

\(\therefore \phi _{b} M_{ny} =\phi _{b}(1.6F_{y}S_{y} )=0.90(1.6)(36)(1.15)\)

=59.62ft-kips =4.968 ft-kips

Because the load is applied to the top flange, use only half this capacity to account for the torsional effects. From Equation 5.22,

\(\frac{M_{ux}}{\phi _{b} M_{nx}} +\frac{M_{uy}}{\phi _{b} M_{ny}}=\frac{12.40}{42.9}+\frac{1.034}{4.968/2}=0.705\lt 1.0 \) (OK)

The shear is

\(V_{u}=\frac{0.4410(15)}{2}=3.31\) kips

From the uniform load tables,

\(\phi _{v} V_{n}=46.7\) kips \(\gt 3.31\) kips (OK)

Use a C10 × 15.3.

For dead load plus a roof live load, load combination 3 will control:

\(q_{a} = q_{D} + q_{Lr} = 42\) psf

The width of roof surface tributary to each purlin is

\(\frac{15}{2}\frac{\sqrt{10} }{3}=7.906\) psf

Then

Purlin load = \(42(7.906) = 332.1\) Ib/ft

Normal component = \(\frac{3 }{\sqrt{10}}(332.1)=315.1\) Ib/ft

Parallel component =\(\frac{1}{\sqrt{10}}(332.1)=105.0\) Ib/ft

and

\(M_{ax}= \frac{1}{8}w L^{2}=\frac{1}{8}(0.3151)(15)^{2}=8.862\) ft-kips

With sag rods placed at the midpoint of each purlin, the purlins are two-span continuous beams with respect to weak axis bending. From Table 3-22c, “Continuous Beams,” the maximum moment in a two-span continuous beam with equal spans is at the interior support and is given by

\(M = 0.125 w\ell ^{2} \)

where

w = uniform load intensity
\(\ell\) = span length

The maximum moment about the y axis is therefore

\(M_{ay} = 0.125(0.1050)(15/2)^{2} = 0.7383\) ft-kips

To select a trial shape, use the beam design charts and choose a shape with a relatively large margin of strength with respect to major axis bending. For an unbraced length of 15/2 = 7.5 ft, try a C10 × 15.3.

For \(C_{b} = 1.0, M_{nx}/Ω_{b} = 22.0\) ft-kips. From Figure 5.15b, \(C_{b} = 1.30\) for the load and lateral support conditions of this beam. Therefore,

\(M_{nx}/Ω_{b} = 1.30(22.0) = 28.60\) ft-kips

From the uniform load tables,

\(M_{px}/Ω_{b} = 28.6\) ft-kips
\(∴\) Use \(M_{nx}/Ω_{b} = 28.6\) ft-kips.

This shape is compact (no footnote in the uniform load tables), so

\(M_{ny}/Ω_{b} = M_{py}/Ω_{b} = F_{y}Z_{y}/Ω_{b} = 36(2.34)/1.67 = 50.44\) in.-kips = 4.203 ft-kips

But

\(\frac{Z_{y} }{S_{y} } =\frac{2.34}{1.15}=2.03\gt 1.6\)

\(∴\) \(M_{ny}/Ω_{b} =1.6F_{y}S_{y}/Ω_{b}=( 1.6)(36)(1.15)/1.67 =39.66\) in.-kips = 3.300 ft-kips

Because the load is applied to the top flange, use only half of this capacity to account for the torsional effects. From Equation 5.23,

\(\frac{M_{ax}}{M_{nx}/Ω _{b} } +\frac{M_{ay}}{M_{ny}/Ω _{b} }=\frac{8.862}{28.6}+\frac{0.7383}{3.300/2}=0.757\lt 1.0 \) (OK)

The maximum shear is

\(V_{a}=\frac{0.315(15)}{2}=2.36\) kips

From the uniform load tables,

\(\frac{V_{n}}{\phi _{v} }=31.0\) kips \(\gt 2.36\) kips (OK)

Use a C10 × 15.3.
