Guest Question.

An additive process model is depicted in the figure below.

For \(G_{1}=\frac{1}{s}, G_{2}=\frac{4}{2 s+1}, G_{3}=\frac{-3}{s+1}, U(s)=1\) (unit impulse)

(a) Derive the response \(Y(s)\) and describe \(y(t)\) quantitatively.

(b) Simulate the response and identify its major characteristics.

Admin
Solution

a)
From block algebra,

\(Y(s)=G_{1}(s) U(s)+G_{2}(s) U(s)+G_{3}(s) U(s)\)

\(\text{or}\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad Y(s)=\left[G_{1}(s)+G_{2}(s)+G_{3}(s)\right] U(s)\)

After some simple operations, and by account that \(U(s)=1\), then

\(Y(s)=\left[\frac{1}{s}+\frac{4}{2 s+1}+\frac{-3}{s+1}\right] U(s)=\frac{4 s+1}{s(2 s+1)(s+1)}\)

\(\text{or}\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad Y(s)=\frac{4 s+1}{(2 s+1)(s+1)} \times \frac{1}{s}=\frac{4 s+1}{2 s^{2}+3 s+1} \times \frac{1}{s}\)

Notice that this system is equivalent to a step input response of an overdamped \((\zeta=1.06)\) second-order transfer function with numerator dynamics (see in your textbook).

For this example, \(\tau_{ a }>\tau_{1}\) (e.g., \(4>2\) ), so the response will exhibit some overshoot.

The system poles \((-0.5,-1)\) lie in the LHP, so \(y(t)\) will be bounded.

Finally,

\(y(0)=\underset{s\rightarrow \infty }{\lim } s Y(s)=0 \quad\quad\quad\quad\quad y(\infty)=\underset{s\rightarrow 0 }{\lim } s Y(s)=1\)
