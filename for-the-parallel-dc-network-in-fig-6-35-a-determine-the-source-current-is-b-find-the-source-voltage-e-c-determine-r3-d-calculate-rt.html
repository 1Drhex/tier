Guest Question.

For the parallel dc network in Fig. 6.35:
a. Determine the source current \(I_s\).
b. Find the source voltage E.
c. Determine \(R_3\).
d. Calculate \(R_T\).

Admin
Solution

a. First apply Eq. (6.13) at node a. Although node a in Fig. 6.35 may not initially appear as a single junction, it can be redrawn as shown in Fig. 6.36, where it is clearly a common point for all the branches.

\(\sum{I_i}=\sum{I_o}\) (6.13)

The result is

\(\sum{I_i}=\sum{I_o}\\[0.5cm] I_s=I_1+I_2+I_3\)

Substituting values: \(I_s=8mA+10mA+2mA=20mA\)

Note in this solution that you do not need to know the resistor values or the voltage applied. The solution is determined solely by the current levels.

b. Applying Ohm’s law gives

\(E=V_1=I_1R_1=(8mA)(2k\Omega )=16V\)

c. Applying Ohm’s law in a different form gives

\(R_3=\frac{V_3}{I_3}=\frac{E}{I_3}=\frac{16V}{2mA}=8k\Omega\)

d. Applying Ohm’s law again gives

\(R_T=\frac{E}{I_s}=\frac{16V}{20mA}=0.8k\Omega\)
