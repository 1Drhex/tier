Guest Question.

Given A\(( r = 20,\theta = \mathring { 30}, \phi = \mathring { 45 } )\) and B\(( r = 30,\theta = \mathring { 115} , \phi = \mathring { 160} )\), find: (a) \(\left|{ R }_{ AB } \right|\) ; (b) \(\left|{ R }_{ AC } \right|\), given C\(( r = 20,\theta = \mathring { 90} , \phi = \mathring { 45} )\); (c) the distance from A to C on a great circle path.

Admin
Solution

a) \(\left|{ R }_{ AB } \right| \): First convert A and B to Cartesian: Have\( { x}_{ A }\) =\( 20\sin { \mathring { 30}}\cos { \mathring { 45} }\) = 7.07, \( { y}_{ A }\) =\( 20\sin { \mathring { 30}}\cos { \mathring { 45} }\) = 7.07 and \({ z}_{ A }\) = \(20\sin { \mathring { 30}} \)=\(17.3. { x}_{ B}\) = \(30\sin { \mathring { 115}}\cos { \mathring { 160} }\) = -25.6. \({ y}_{ B}\) =\( 30\sin { \mathring { 115}}\sin { \mathring { 160} }\) = 9.3. and \({z}_{ B}\) =\( 30\sin { \mathring { 115}}\) = -12.7 . Now\( { R }_{ AB }\) =\( { R }_{ B } – { R }_{ A } \)= (-32.6, 2.2, -30.0), and So \(\left|{ R }_{ AB } \right| \)= 44.4.
b) Again converting C to Cartesian, obtain \({ x }_{ c }\) =\( 20\sin { \mathring { 90}}\cos { \mathring { 45} } \)= 14.14, \( { y}_{ C } \)= \(20\sin { \mathring { 90}}\sin { \mathring { 45} }\) = 14.14, and \({ z}_{ C } \)= \(20\cos { \mathring { 90}}\) = 0. \(So { R }_{ AC }\) = \({ R }_{ C } – { R }_{ A }\) = (7.07, 7.07, -17.3), and \( \left|{ R }_{ AC } \right|\) = 20.0.
c) Note that A and C share the same r and \(\phi\) coordinates; thus moving from A to C involves only a change in \(\theta\) of \(\mathring { 60 }\). the requested are length is then
distance = 20 \(\times \left[60 (\frac { 2\pi }{ 360 }) \right] \)= 20.9
