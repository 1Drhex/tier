Guest Question.

A 50 kW , 250 V compound motor takes a current of 9 A while running on no-load at rated voltage and speed. The shunt field current is 5 A. The resistances of the windings when hot are: Armature \( 0.1 \Omega\), series field \( 0.07 \Omega\) and interpole \( 0.03 \Omega . \) The brush drop is 2 V . Determine motor output and the efficiency when the motor intake is 155 A .

 

 

 

Admin
Solution
\(\begin{array}{l}\begin{array}{l}\text { At No-load; } I_{L 0}=9 A ; I_{s h}=5 A \\\qquad \text { Input at no-load }=I_{L 0} \times V=9 \times 250=2250 W \\\text { Armature current, } I_{\alpha 0}=I_{L 0}-I_{s h}=9-5=4 A\end{array}\\\text { Considering the machine as long-shunt; }\end{array}\)

 

\(\begin{aligned}\text { Variable losses } &=\text { Copper losses in armature circuit and at brushes } \\&=I_{a 0}^{2}\left(R_{a}+R_{s e}+R_{i}\right)+I_{a 0} \times \text { brush drop } \\&=4^{2}(0.1+0.07+0.03)+4 \times 2=11.2 W\end{aligned}\)

 

\( \text { Constant losses, } P_{C} =\text { Input at no-load }-\text { variable losses } \)

 

=2250-11.2=2238.8 W

 

At full-load; Line current, \( I_{L f}=155 A\) Power input \( =I_{L 1} \times V=155 \times 250=38750 W\)

 

\( \text { Armature current, } I_{a f} =I_{L f}-I_{s h}=155-5=150 A \)

 

\( \text { Variable losses } =I_{a f}^{2}\left(R_{a}+R_{s e}+R_{1}\right)+I_{a f} \times \text { brush drop } \)

 

\( =150^{2}(0.1+0.07+0.03)+150 \times 2=4800 W \)

 

\( \text { Total losses } =P_{C}+\text { variable losses }=2238.8+4800=7038.8 W \)

 

\( \text { Motor output, } P_{\text {out }} =P_{\text {in }}-\text { total losses }\)

 

\( =38750-7038.8=31711.2 W \text { or } 31.7 kW \)

 

\( \text { Motor efficiency, } \eta =\frac{P_{\text {out }}}{P_{\text {in }}} \times 100=\frac{31711.2}{38750} \times 100= 8 1 . 8 4 \% \)
