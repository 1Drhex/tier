Guest Question.

The magnetic field outside a long straight wire carrying a steadycurrent I is

\(B =\frac{\mu_{0}}{2 \pi} \frac{I}{s} \hat{\phi}\) .

The electric field inside the wire is uniform:

\(E =\frac{I \rho}{\pi a^{2}} \hat{ z }\) ,

where ρ is the resistivity and a is the radius (see Exs. and). Question: What is the electric field outside the wire\(?^{29}\) The answer depends on how you complete the circuit. Suppose the current returns along a perfectly conducting grounded coaxial cylinder of radius b (Fig. 7.52). In the region a < s < b, the potential V (s, z) satisfies Laplace’s equation, with the boundary conditions

\(\text { (i) } V(a, z)=-\frac{I \rho z}{\pi a^{2}}\) ; \( \text { (ii) } V(b, z)=0\) .

This does not suffice to determine the answer—we still need to specify boundary conditions at the two ends (though for a long wire it shouldn’t matter much). In the literature, it is customary to sweep this ambiguity under the rug by simply stipulating that V (s, z) is proportional to z: V (s, z) = z f (s). On this assumption:

(a) Determine f (s).

(b) Find E(s, z).

(c) Calculate the surface charge density σ(z) on the wire.

\(\text { [Answer: } V=\left(-I z \rho / \pi a^{2}\right)[\ln (s / b) / \ln (a / b)] \) This is a peculiar result, since \(E_{s}\) and σ(z) are not independent of z—as one would certainly expect for a truly infinite wire.]

Admin
Solution
\(\text { (a) } \nabla^{2} V=\frac{1}{s} \frac{\partial}{\partial s}\left(s \frac{\partial(z f)}{\partial s}\right)+\frac{\partial^{2}(z f)}{\partial z^{2}}=\frac{z}{s} \frac{d}{d s}\left(s \frac{d f}{d s}\right)=0 \Rightarrow \frac{d}{d s}\left(s \frac{d f}{d s}\right)=0 \Rightarrow s \frac{d f}{d s}=A(\text { a constant }) \Rightarrow\)

\(A \frac{d s}{s}=d f \Rightarrow f=A \ln \left(s / s_{0}\right) \quad\left(s_{0} \text { another constant). But (ii) } \Rightarrow f(b)=0, \text { so } \ln \left(b / s_{0}\right)=0, \text { so } s_{0}=b\right.\) , and

\(V(s, z)=A z \ln (s / b) . \text { But (i) } \Rightarrow A z \ln (a / b)=-(I \rho z) /\left(\pi a^{2}\right), \text { so } A=-\frac{I \rho}{\pi a^{2}} \frac{1}{\ln (a / b)} ; \quad V(s, z)=-\frac{I \rho z}{\pi a^{2}} \frac{\ln (s / b)}{\ln (a / b)}\).

\(\text { (b) } E =-\nabla V=-\frac{\partial V}{\partial s} \hat{ s }-\frac{\partial V}{\partial z} \hat{ z }=\frac{I \rho z}{\pi a^{2}} \frac{1}{s \ln (a / b)} \hat{ s }+\frac{I \rho}{\pi a^{2}} \frac{\ln (s / b)}{\ln (a / b)} \hat{ z }=\frac{I \rho}{\pi a^{2} \ln (a / b)}\left(\frac{z}{s} \hat{ s }+\ln \left(\frac{s}{b}\right) \hat{ z }\right)\) .

\(\text { (c) } \sigma(z)=\epsilon_{0}\left[E_{s}\left(a^{+}\right)-E_{s}\left(a^{-}\right)\right]=\epsilon_{0}\left[\frac{I \rho}{\pi a^{2} \ln (a / b)}\left(\frac{z}{a}\right)-0\right]=\frac{\epsilon_{0} I \rho z}{\pi a^{3} \ln (a / b)}\) .
