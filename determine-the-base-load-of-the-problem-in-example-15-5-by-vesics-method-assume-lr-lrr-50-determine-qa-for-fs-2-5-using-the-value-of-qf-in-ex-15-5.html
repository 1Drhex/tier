Guest Question.

Determine the base load of the problem in by Vesic’s method. Assume \(I_{r}=I_{r r}=50\). Determine \(Q_{a} \text { for } F_{s}=2.5\) using the value of \(Q_{f}\) in .

Admin
Solution

From Eq. (15.25) for c = 0 we have

 

\(q_{b}=c N_{c}^{*}+\sigma_{m} N_{\sigma}^{*}\) (15.25)

 

\(q_{b}=\sigma_{m} N_{\sigma}^{*}\)

 

From Eq. (15.24)

 

\(\sigma_{m}=\frac{1+2 K_{o}}{3} q_{o}^{\prime}\) (15.24)

 

\(\sigma_{m}=\frac{1+2 K_{0}}{3} q_{0}^{\prime}=\frac{1+2(1-\sin \phi)}{3} q_{0}^{\prime}\)

 

\(\sigma_{m}=\frac{1+2 K_{0}}{3} q_{0}^{\prime}=\frac{1+2(1-\sin \phi)}{3} q_{0}^{\prime}\)

 

\(q_{0}^{\prime}=15 \times 17.5=262.5 kN / m ^{2}\)

 

\(\sigma_{m}=\left[\frac{1+2\left(1-\sin 30^{0}\right)}{3}\right] \times 262.5=175 kN / m ^{2}\)

 

From Fig. 15.11a, \(N_{\sigma}^{*}=36 \text { for } \phi=30 \text { and } I_{r}=50\)

Substituting

 

\(q_{b}=175 \times 36=6300 kN / m ^{2}\)

 

\(Q_{b}=A_{b} q_{b}=\frac{3.14}{4} \times(0.45)^{2} \times 6300=1001 kN\)

 

\(Q_{b}=A_{b} q_{b}=\frac{3.14}{4} \times(0.45)^{2} \times 6300=1001 kN\)

 

\(Q_{u}=Q_{b}+Q_{f}=1001+964=1,965 kN\)

 

\(Q_{a}=\frac{1965}{2.5}=786 kN\)
