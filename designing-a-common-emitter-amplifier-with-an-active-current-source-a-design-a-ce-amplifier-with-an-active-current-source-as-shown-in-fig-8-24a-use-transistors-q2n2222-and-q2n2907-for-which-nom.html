Guest Question.

Designing a common-emitter amplifier with an active current source

(a) Design a CE amplifier with an active current source, as shown in Fig. 8.24(a). Use transistors Q2N2222 and Q2N2907, for which nominal \( \beta_{ F }=\beta_{ f }=173, I_{ S }=3.295 \times 10^{-14} A , \text { and } V_{ A }=100 V \). The operating collector current is set at \( I_{ C }=10 mA \). The DC power supply is \( V_{ CC }=15 V \text {. Assume } V_{ BE }=0.7 V \).

(b) Use PSpice/SPICE to verify your results in part (a).

Admin
Solution

(a) Step 1. Design the biasing current source. \( I_{ C }=10 mA , \text { and } V_{ CC }=15 V \). From Eq. (8.59), we can find the value of R in order to set the biasing current to \( I_{ ref }=10 mA \).

 

\( I_{ C 3} \approx I_{ ref }=\frac{V_{ CC }-V_{ BE 3}}{R}=I_{ C 2} \) (8.59)

 

\( R=\frac{V_{ CC }-V_{ BE 3}}{I_{ ref }}=\frac{15-0.7}{10 mA }=1.43 k \Omega\)

 

Step 2. Find the small-signal parameters of the transistors. From Eq. (8.40),

 

\( \frac{1}{r_{\pi}}=\frac{i_{ b }}{v_{ be }}=\left.\frac{d i_{ B }}{d v_{ BE }}\right|_{\text {at } Q \text {-point }}=\frac{I_{ B }}{V_{ T }}=\frac{I_{ B }}{25.8 mV } \) (8.40)

 

\( r_{\pi 1}=r_{\pi 2}=\frac{25.8 mV }{I_{ B }}=\frac{\beta_{ F } \times 25.8 mV }{I_{ C }}=\frac{173 \times 25.8 mV }{10 mA }=446 \Omega \)

 

From Eq. (8.42),

 

\( g_{ m }=\frac{i_{ c }}{v_{ be }}=\left.\frac{\beta_{ F } d i_{ B }}{d v_{ BE }}\right|_{\text {at } Q \text {-point }}=\frac{\beta_{ f } I_{ B }}{V_{ T }}=\frac{I_{ C }}{V_{ T }}=\frac{\beta_{ f }}{r_{\pi}} \) (8.42)

 

\( g_{ m 1}=g_{ m 2}=\frac{I_{ C }}{V_{ T }}=\frac{10 mA }{25.8 mV }=387.6 mA / V \)

 

From Eq. (8.43),

 

\( \frac{1}{r_{ o }}=\frac{i_{ c }}{v_{ ce }}=\left.\frac{d i_{ C }}{d v_{ CE }}\right|_{\text {at } Q \text {-point }}=\frac{I_{ C }}{ V _{ A }+V_{ CE }}=\frac{I_{ C }}{V_{ A }} \quad \text { for } V_{ A }>>V_{ CE } \) (8.43)

 

\( r_{ o1 }=r_{ o 2}=\frac{V_{ A }}{I_{ C }}=\frac{100 V }{10 mA }=10 k \Omega \)

 

Step 3. Evaluate the values of the input resistance \( R_{ i }\left(=v_{ b } / i_{ b }\right) \), the open-circuit voltage gain \( A_{\text {vo }} \), and the output resistance \( R_{0}\). We know that

 

\( R_{ i }=r_{\pi 1}=446 \Omega \)

 

From Eq. (8.81),

 

\( R_{ o }=r_{ o 2} \| r_{ o1 } \) (8.81)

 

\( R_{ o }=r_{ o 2}\left\|r_{ o1}=10 k \right\| 10 k =5 k \Omega \)

 

From Eq. (8.82),

 

\( A_{ vo }=\frac{v_{ o }}{v_{ b }}=-g_{ m 1}\left(r_{ o 2} \| r_{ o 1}\right) \) (8.82)

 

\( A_{ vo }=-g_{ m 1}\left(r_{ o 2} \| r_{ o1}\right)=-387.6 mA / V \times 5 k \Omega=-1938 \)

 

(b) From Eq. (8.1), we can find the value of \( v_{ BE 1} \) needed to give \( I_{ C }=10 mA \text { for } I_{ S }=3.295 \times 10^{-14} A \):

 

\( i_{ C }=I_{ C , n }=I_{ E , n }-I_{ B , n }=I_{ S } e^{v_{ B } / V_{ T }} \) (8.1)

 

\(10 mA =3.295 \times 10^{-14} \times \exp \left(\frac{v_{ BE 1}}{25.8 mV }\right)\)

 

which gives \(v_{ BE1 }=0.682 V\)

 

Using Eq. (8.61), we can find the biasing resistor \( R_{ B } \):

 

\(i_{ C }=\beta_{ F } i_{ B }=\frac{\beta_{ F }\left(V_{ CC }-v_{ BE }\right)}{R_{ B }}\) (8.61)

 

\(R_{ B }=\frac{\beta_{ F }\left(V_{ CC }-V_{ BE }\right)}{I_{ C }}=\frac{173 \times(15-0.682)}{10 \times 10^{-3}}=247.7 k \Omega \quad(\text { Use } 250 k \Omega .)\)

 

The CE amplifier with an active load for PSpice simulation is shown in Fig. 8.25.

The PSpice plot of the transfer function \(v_{ o }[\equiv V ( Q 1: C )] \text { against } v_{ S }\) is shown in Fig. 8.26. Notice that the operating range of the input voltage is very small—that is, 8.563 mV (709.863 mV – 701.30 mV). Thus, the small-signal gain becomes -15 V ⁄ 8.563 mV = -1752. Details of the .TF analysis are given below. The values obtained from hand calculations are shown in parentheses. The v alue of \(v_{ S }\left(=v_{ BE 1}\right)\) was adjusted to 0.705 V instead of 0.682 V in order to operate in the linear range of the amplifier and to illustrate the benefit of using an active load. Since the voltage gain is very large, any small change in \( v_{ S } \) could drive the amplifier into saturation. Thus, if we build the amplifier and test it in the laboratory with a value of \( v_{ S }=0.682 V \), it might not work; we need to adjust \( v_{ S } \). Using transistor circuit model, we make the range of input voltage from 681 mV to 685 mV.
