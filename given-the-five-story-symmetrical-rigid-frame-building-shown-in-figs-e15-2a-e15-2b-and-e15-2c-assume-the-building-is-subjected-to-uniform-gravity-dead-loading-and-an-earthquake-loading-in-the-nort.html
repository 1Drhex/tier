Guest Question.

Given: The five-story symmetrical rigid frame building shown in Figs. E15.2a, E15.2b, and E15.2c. Assume the building is subjected to uniform gravity dead loading and an earthquake loading in the north–south direction. Consider the floor load to be transmitted to all sides (two-way action). Assume the floors are rigid with respect to lateral motion.

Determine: The maximum forces in the columns and beams for a typical interior bay and the lateral displacement of the floors using computer software. Assume all the beams to be the same size and all the columns to be the same size.

The corresponding cross-sectional properties are specified for two cases:

The second case corresponds to doubling the column inertias for case one

I shape beams \(\left\{\begin{array}{l}I_{z}=445,146,750 mm ^{4} \\I_{ y }=22,798,170 mm ^{4} \\I_{ x }=1,135,750 mm ^{4} \\A=12,320 mm ^{2}\end{array}\right.\)

Hollow square columns \(\left\{\begin{array}{l}\text { case(1) }\left\{\begin{array}{l}I_{ z }=I_{ y }=148,520,925 mm ^{4} \\I_{ x }=233,390,025 mm ^{4} \\A=10,677 mm ^{2}\end{array}\right. \\\operatorname{case}(2)\left\{\begin{array}{l}I_{ z }=I_{ y }=309,106,575 mm ^{4} \\I_{ x }=486,749,250 mm ^{4} \\A=15,867 mm ^{2}\end{array}\right.\end{array}\right.\)

Admin
Solution

The floor loading is uniformly applied to the floor slab. Using the concept of tributary areas, we convert this loading to line loadings w on the perimeter floor beams. Note that the N–S and E–W loading are identical because of the geometry.

 

\(W_{\text {floor total }}=2000\) KN

 

\(\frac{2000}{(27)(27)}=2.744 kN / m ^{2}\)

 

\(\Downarrow\)

 

\(w_{1 \text { gravity }}=2.744(4.5)=12.35\) KN/m

 

\(w_{2 \text { gravity }}=2.744(9)=24.7\) KN/m

 

The gravity line loading patterns for the perimeter floor beams are listed below.

 

Using computer software, we analyze a 2D model of the rigid frame for gravity and earthquake loading. This approach is possible because the geometry and stiffness properties are symmetrical.

 

The critical values for the column forces (axial, shear, moment) occur in the first story. Results for the beams, columns, and lateral displacement corresponding to the two choices for column properties are listed and plotted below.

 

Note that there is only a small difference in the force magnitudes when the column inertia values are doubled. The main effect is on the lateral displacement which is to be expected.
