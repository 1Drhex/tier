Guest Question.

Two electrical signals represented by \(A = 101101\) and \(B = 110101\) are applied to a \(2\)-input \(AND\) gate. Sketch the output signal and the binary number it represents.

Admin
Solution

The pulse trains corresponding to \(A\) and \(B\) are shown in \(Fig. 70.49\) .

Remember that in an \(AND\) gate, \(C\) is \(1\) only when both \(A\) and \(B\) are \(1\). It is \(an all-or-nothing gate\). The output can be found in different time intervals as under :

1. 1st interval : \(1 + 1 = 1\)
2. 2nd interval : \(0 + 1 = 0\)
3. 3rd interval : \(1 + 0 = 0\)
4. 4th interval : \(1 + 1 = 1\)
5. 5th interval : \(0 + 0 = 0\)
6. 6th interval : \(1 + 1 = 1\)

Hence, ouput of the \(AND\) gate is \(100101_{2}\). It is sketched in \(Fig. 70.49\).
