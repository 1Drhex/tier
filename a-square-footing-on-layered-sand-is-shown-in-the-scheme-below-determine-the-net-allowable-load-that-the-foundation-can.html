Guest Question.

A square footing on layered sand is shown in the scheme below. Determine the net allowable load that the foundation can support, considering shear failure only. Use the Meyerhof bearing capacity and shape factors. The safety factor against bearing capacity failure should not be less than four.

Admin
Solution

This is a case of layered soils where Stronger sand \( \left ( c_{1} = 0 \right ) \) is overlying weaker sand \( \left ( c_{2} = 0 \right ) \).

Equation :

\( q_{ult} = \left [ {y}'_{1} \left ( D_{f} + H \right ) N_{q\left ( 2 \right )} F_{qs\left ( 2 \right )} + \frac{1}{2} {y}'_{2}B N_{y\left ( 2 \right )}F_{ys\left ( 2 \right )} + {y}'_{1}H^{2} \left ( 1 + \frac{B}{L} \right )\left ( 1 + \frac{2 D_{f}}{H} \right )\left ( \frac{K_{s} \tan {\varnothing }'_{1}}{B} \right ) – {y}'_{1} H \right ] \)

 

\( \leq \left ( {y}'_{1} D_{f} \right )N_{q\left ( 1 \right )}F_{qs\left ( 1 \right )} + \frac{1}{2} {y}'_{1} B N_{y\left ( 1 \right )}F_{ys\left ( 1 \right )} \)

 

\( H = 1.0 m; D_{f} = 1.5 m; B = L = 1.5 m \)
Table 4.4:

 

\( {\varnothing}'_{1} = 40^{\circ}; N_{q\left ( 1 \right)} = 64.1; N_{y\left ( 1 \right)} = 93.69 \)

 

\( {\varnothing}'_{2} = 32^{\circ}; N_{q\left ( 2 \right)} = 23.2; N_{y\left ( 2 \right)} = 22.02 \)

 

\( F_{qs\left ( 1 \right )} = F_{ys\left ( 1 \right )} = 1 + 0.1 \frac{B}{L} \tan^{2} \left ( 45 + \frac{{\varnothing }'_{1}}{2} \right ) = 1 + 0.1 \times \frac{1.5}{1.5} \times 4.6 = 1.5 \)

 

\( F_{qs\left ( 2 \right )} = F_{ys\left ( 2 \right )} = 1 + 0.1 \frac{B}{L} \tan^{2} \left ( 45 + \frac{{\varnothing }'_{2}}{2} \right ) = 1 + 0.1 \times \frac{1.5}{1.5} \times 3.3 = 1.3 \)

 

Figure : for \( \frac{q_{2}}{q_{1}} = \frac{{y}'_{2}B N_{y\left ( 2 \right )} }{{y}'_{1} B N_{y\left ( 1 \right )}} = \frac{16.7 \times 22.02}{18 \times 93.69} = 0.22 \) and \( {\varnothing}'_{1} = 40^{\circ}, \)

 

find \( K_{s} \cong 4.5 \)

 

\( q_{ult} =\left [ \left (18 \right )\left ( 1.5 + 1 \right)\left ( 23.2 \right)\left ( 1.3 \right) + \left ( 0.5 \right)\left ( 16.7 \right )\left ( 1.5 \right )\left ( 22.02 \right )\left ( 1.3\right ) + \left ( 18 \right )\left ( 1 \right )\left ( 1 + 1 \right )\left ( 1 + \frac{2 \times 1.5}{1} \right )\left ( \frac{4.5 \times 0.84}{1.5}\right ) – 18 \times 1 \right ] \)

 

\( = 1357.2 + 358.5 + 362.9 – 18 = 2060.6 kPa \)

 

As a check, we have: Equation :

Equation : \( q_{ult,t} = \left ( {y}'_{1} D_{f} \right ) N_{q\left ( 1 \right )} F_{qs\left ( 1 \right )} + \frac{1}{2} {y}'_{1} B N_{y\left ( 1 \right )} F_{ys\left ( 1 \right )} \)

 

\( = \left ( 18 \times 1.5 \right )\left ( 64.1 \times 1.5 \right ) + 0.5 \times 18 \times 1.5 \times 93.69 \times 1.5 \)

 

\( = 2596.1 + 1897.2 = 4493.3 kPa \gg q_{ult} \)

 

Use \( gross q_{ult} = 2060.6 kPa \)

 

Equation : \( net q_{ult} = gross q_{ult} – {\sigma}'_{o} \)

\( net q_{ult} = 2060.6 – 18 \times 1.5 = 2033.6 kPa \)

 

Considering shear failure only, \( net q_{a} = \frac{net q_{ult}}{SF} = \frac{2033.6}{4} = 508.4 kPa \)

 

The net allowable load is

\( net Q_{a} = 508.4 \left ( 1.5 \right )^{2} = 1144 kN \)
