Guest Question.

At a machined shaft shoulder the small diameter d is 1.100 in, the large diameter D is 1.65 in, and the fillet radius is 0.11 in. The bending moment is 1260 lbf . in and the steady torsion moment is 1100 lbf . in. The heat-treated steel shaft has an ultimate strength of \(S_{u t}=105 kpsi\) and a yield strength of \(S_{y}=82 kpsi\). The reliability goal for the endurance limit is 0.99.
(a) Determine the fatigue factor of safety of the design using each of the fatigue failure criteria described in this section.
(b) Determine the yielding factor of safety.

Admin
Solution

\(\text { (a) } D / d=1.65 / 1.100=1.50, r / d=0.11 / 1.100=0.10, K_{t}=1.68 \text { (Fig. A-15-9), }\)\(K_{t s}=1.42 \text { (Fig. A-15-8), } q=0.85 \text { (Fig. 6-20), } q_{\text {shear }}=0.88 \text { (Fig. 6-21). }\)

From Eq. (6–32),

 

\(K_{f}=1+q\left(K_{t}-1\right) \quad \text { or } \quad K_{f s}=1+q_{\text {shear }}\left(K_{t s}-1\right)\) (6–32)

 

\(K_{f}=1+0.85(1.68-1)=1.58\)

 

\(K_{f s}=1+0.88(1.42-1)=1.37\)

 

\(\text { Eq. (6-8): } \quad S_{e}^{\prime}=0.5(105)=52.5 kpsi\)

 

\(\begin{array}{ll}\text { Eq. (6-19): } &k_{a}=2.70(105)^{-0.265}=0.787\end{array}\)

 

Eq. (6–20):
\(k_{b}=\left(\frac{1.100}{0.30}\right)^{-0.107}=0.870\)

 

\(k_{c}=k_{d}=k_{f}=1\)

 

Table 6–6: \(k_{e}=0.814\)

 

\(S_{e}=0.787(0.870) 0.814(52.5)=29.3 kpsi\)

 

For a rotating shaft, the constant bending moment will create a completely reversed bending stress.

 

\(M_{a}=1260 lbf \cdot \text { in } \quad T_{m}=1100 lbf \cdot \text { in } \quad M_{m}=T_{a}=0\)

 

Applying Eq. (7–7) for the DE-Goodman criteria gives

 

\(\frac{1}{n}=\frac{16}{\pi d^{3}}\left\{\frac{1}{S_{e}}\left[4\left(K_{f} M_{a}\right)^{2}+3\left(K_{f s} T_{a}\right)^{2}\right]^{1 /2}+\frac{1}{S_{ut}}\left[4\left(K_{f} M_{m}\right)^{2}+3\left(K_{f s} T_{m}\right)^{2}\right]^{1 /2}\right\}\) (7–7)

 

\(\frac{1}{n}=\frac{16}{\pi(1.1)^{3}}\left\{\frac{\left[4(1.58 \cdot 1260)^{2}\right]^{1 / 2}}{29300}+\frac{\left[3(1.37 \cdot 1100)^{2}\right]^{1 / 2}}{105000}\right\}=0.615\)

 

n = 1.63 DE-Goodman


Similarly, applying Eqs. (7–9), (7–11), and (7–13) for the other failure criteria,

 

\(\frac{1}{n}=\frac{8 A}{\pi d^{3} S_{e}}\left\{1+\left[1+\left(\frac{2 B S_{e}}{A S_{u t}}\right)^{2}\right]^{1 / 2}\right\}\) (7–9)

 

\(\frac{1}{n}=\frac{16}{\pi d^{3}}\left[4\left(\frac{K_{f} M_{a}}{S_{e}}\right)^{2}+3\left(\frac{K_{f s} T_{a}}{S_{e}}\right)^{2}+4\left(\frac{K_{f} M_{m}}{S_{y}}\right)^{2}+3\left(\frac{K_{f s} T_{m}}{S_{y}}\right)^{2}\right]^{1 / 2}\) (7–11)

 

 

\(\frac{1}{n}=\frac{16}{\pi d^{3}}\left\{\frac{1}{S_{e}}\left[4\left(K_{f} M_{a}\right)^{2}+3\left(K_{f s} T_{a}\right)^{2}\right]^{1 / 2}+\frac{1}{S_{y}}\left[4\left(K_{f} M_{m}\right)^{2}+3\left(K_{f s} T_{m}\right)^{2}\right]^{1 / 2}\right\}\) (7–13)

 

n = 1.87 DE-Gerber

n = 1.88 DE-ASME Elliptic

n = 1.56 DE-Soderberg

 

For comparison, consider an equivalent approach of calculating the stresses and applying the fatigue failure criteria directly. From Eqs. (7–5) and (7–6),

 

\(\sigma_{a}^{\prime}=\left(\sigma_{a}^{2}+3 \tau_{a}^{2}\right)^{1 / 2}=\left[\left(\frac{32 K_{f} M_{a}}{\pi d^{3}}\right)^{2}+3\left(\frac{16 K_{f s} T_{a}}{\pi d^{3}}\right)^{2}\right]^{1 / 2}\) (7–5)

 

\(\sigma_{m}^{\prime}=\left(\sigma_{m}^{2}+3 \tau_{m}^{2}\right)^{1 / 2}=\left[\left(\frac{32 K_{f} M_{m}}{\pi d^{3}}\right)^{2}+3\left(\frac{16 K_{f s} T_{m}}{\pi d^{3}}\right)^{2}\right]^{1 / 2}\) (7–6)

 

\(\sigma_{a}^{\prime}=\left[\left(\frac{32 \cdot 1.58 \cdot 1260}{\pi(1.1)^{3}}\right)^{2}\right]^{1 / 2}=15235 psi\)

 

\(\sigma_{m}^{\prime}=\left[3\left(\frac{16 \cdot 1.37 \cdot 1100}{\pi(1.1)^{3}}\right)^{2}\right]^{1 / 2}=9988 psi\)

 

Taking, for example, the Goodman failure critera, application of Eq. (6–46) gives

 

\(\text { mod-Goodman } \frac{\sigma_{a}}{S_{e}}+\frac{\sigma_{m}}{S_{u t}}=\frac{1}{n}\) (6–46)

 

\(\frac{1}{n}=\frac{\sigma_{a}^{\prime}}{S_{e}}+\frac{\sigma_{m}^{\prime}}{S_{u t}}=\frac{15235}{29300}+\frac{9988}{105000}=0.615\)

 

n = 1.63

 

which is identical with the previous result. The same process could be used for the other failure criteria.

 

(b) For the yielding factor of safety, determine an equivalent von Mises maximum stress using Eq. (7–15).

 

\(\begin{aligned}\sigma_{\max }^{\prime} &=\left[\left(\sigma_{m}+\sigma_{a}\right)^{2}+3\left(\tau_{m}+\tau_{a}\right)^{2}\right]^{1 / 2} \\&=\left[\left(\frac{32 K_{f}\left(M_{m}+M_{a}\right)}{\pi d^{3}}\right)^{2}+3\left(\frac{16 K_{f s}\left(T_{m}+T_{a}\right)}{\pi d^{3}}\right)^{2}\right]^{1 / 2}\end{aligned}\) (7–15)

 

\(\sigma_{\max }^{\prime}=\left[\left(\frac{32(1.58)(1260)}{\pi(1.1)^{3}}\right)^{2}+3\left(\frac{16(1.37)(1100)}{\pi(1.1)^{3}}\right)^{2}\right]^{1 / 2}=18220 psi\)

 

\(n_{y}=\frac{S_{y}}{\sigma_{\max }^{\prime}}=\frac{82000}{18220}=4.50\)

 

For comparison, a quick and very conservative check on yielding can be obtained by replacing \(\sigma_{\max }^{\prime} \text { with } \sigma_{a}^{\prime}+\sigma_{m}^{\prime}\) This just saves the extra time of calculating \(\sigma_{\max }^{\prime}\) if \(\sigma_{a}^{\prime} \text { and } \sigma_{m}^{\prime}\) have already been determined. For this example,

 

\(n_{y}=\frac{S_{y}}{\sigma_{a}^{\prime}+\sigma_{m}^{\prime}}=\frac{82000}{15235+9988}=3.25\)

 

which is quite conservative compared with \(n_{y}=4.50\)
