Guest Question.

Select a rectangular HSS12×8 compression member with a length of 30 ft, to support an axial dead load of 26 kips and live load of 77 kips.
The base is fixed, the top is pinned.
A column with slender elements has been selected to demonstrate the design of such member.

Admin
Solution

Material Properties:

ASTM A500 Gr. B \(F_{y}=46 ksi \quad F_{u}=58 ksi\)

Manual
Table 2-3

Calculate the required strength

LRFD ASD
\(P_{u}=1.2(26.0 kips )+1.6(77.0 kips )=154 \text { kips }\) \(P_{a}=26.0 \text { kips }+77.0 \text { kips }=103 \text { kips }\)

Table Solution:

K = 0.8 Therefore \((K L)_{x}=(K L)_{y}=0.8(30.0 ft )=24.0 ft\)

Commentary
Table C-C2.2

Enter Manual Table 4-3, in the HSS12×8 section and proceed across the page until the lightest section is found with an available strength that is equal to or greater than the required strength, in this case a HSS 12×8×3/16.

Determine the available strength in axial compression

LRFD ASD
\(\phi_{c} P_{n}=155 \text { kips }>154 \text { kips }\) o.k. \(P_{n} / \Omega_{c}=103 \text { kips } \geq 103 \text { kips }\) o.k.

Manual
Table 4-3

The available strength can be easily determined by using the tables of the Manual. Available strength values can be verified by hand calculations, as shown below, including adjustments for slender elements.

Calculation Solution:

Geometric Properties:

HSS12×8×3/16 \(A_{g}=6.76 \text { in. }^{2} \quad r_{x}=4.56 \text { in. } \quad r_{y}=3.35 \text { in. } \quad \frac{b}{t}=43.0\)

 

\(\frac{h}{t}=66.0 \quad t_{\text {des }}=0.174\)

Calculate the limiting width-thickness ratios

\(\lambda_{r}=1.40 \sqrt{\frac{E}{F_{y}}}=1.40 \sqrt{\frac{29,000 ksi }{46 ksi }}=35.2<43.0\) therefore both the 10-in. and 12-in. walls are slender elements.

Case 12

Note that for determining the width-thickness ratio, b is taken as the outside dimension minus three times the design wall thickness.

For the selected shape

b = 8.00 in. – 3(0.174 in.) = 7.48 in.

Specification
Section
B4.2d

For an HSS member with slender elements, the nominal compressive strength, \(P_{n}\), shall be determined based upon the limit states of flexural buckling. Torsional buckling will not govern unless the torsional unbraced length greatly exceeds the controlling flexural unbraced length.

Compute effective area, \(A_{\text {eff }}\)

Section E7

\(Q_{a}=\frac{A_{e f f}}{A}\) Eqn. E7-16

where

\(A_{\text {eff }}\) = summation of the effective areas of the cross section based on the reduced effective widths \(b_{e}\).

For flanges of square and rectangular slender element section of uniform thickness,

\(b_{e}=1.92 t \sqrt{\frac{E}{f}}\left[1-\frac{0.38}{(b / t)} \sqrt{\frac{E}{f}}\right] \leq b\) Eqn. E7-18

 

where \(f=P_{n} / A_{e f f}\), but can conservatively be taken as \(F_{y}\)

For the 8-in. walls,

\(b_{e}=1.92 t \sqrt{\frac{E}{F_{y}}}\left[1-\frac{0.38}{(b / t)} \sqrt{\frac{E}{F_{y}}}\right]\) Eqn. E7-18

 

\(=1.92(0.174 \text { in. }) \sqrt{\frac{29,000 ksi }{46 ksi }}\left[1-\frac{0.38}{(43.0)} \sqrt{\frac{29,000 ksi }{46 ksi }}\right]=6.53 \text { in. } \leq 7.48 in\)

 

Length that can not be used = \(b-b_{e}=7.48 \text { in. }-6.53 \text { in. }=0.950 \text { in. }\)

For the 12 inch walls.

\(b_{e}=1.92 t \sqrt{\frac{E}{F_{y}}}\left[1-\frac{0.38}{(b / t)} \sqrt{\frac{E}{F_{y}}}\right]\)

 

\(=1.92(0.174 \text { in. }) \sqrt{\frac{29,000 ksi }{46 ksi }}\left[1-\frac{0.38}{(66.0)} \sqrt{\frac{29,000 ksi }{46 ksi }}\right]=7.18 \text { in. } \leq 11.5 in\)

Eqn. E-16

Length that can not be used = 11.5 in. – 7.18 in. = 4.32 in.

Therefore \(A_{e f f}=6.76 in .^{2}-2(0.174 in .)(0.950 in .)-2(0.174 in .)(4.32 in .)=4.93 in .^{2}\)

 

\(\text { Therefore } Q=Q_{a}=\frac{A_{e f f}}{A}=\frac{4.93 in .^{2}}{6.76 in ^{2}}=0.729\)

Determine the appropriate equation for \(F_{c r}\)

\(\frac{K_{y} L}{r_{y}}=\frac{0.8(30.0 ft )(12 in / ft )}{3.35 in .}=86.0<4.71 \sqrt{\frac{E}{Q F_{y}}}=4.71 \sqrt{\frac{29,000 ksi }{0.729(46 ksi )}}=139\)

 

\(\text { Therefore } F_{c r}=Q\left[0.658^{\frac{Q F_{y}}{F_{e}}}\right] F_{y}\) Eqn. E7-2

 

\(F_{e}=\frac{\pi^{2} E}{\left(\frac{K L}{r}\right)^{2}}=\frac{\pi^{2}(29,000 ksi )}{86.0^{2}}=38.7 ksi\) Eqn. E3-4

 

\(F_{c r}=Q\left[0.658^{\frac{Q F_{y}}{F_{e}}}\right] F_{y}=0.729\left[0.658^{\frac{0.729(46 k ki )}{38.7 ksi }}\right] 46 ksi =23.3 ksi .\)

Eqn. E7-2

Note: A conservative initial assumption \(\left(f=F_{y}\right)\) was made in applying Specification Equation E7-18. A more exact solution is obtained by iterating from the Compute effective area, \(A_{\text {eff }}\) step using \(f=P_{n} / A_{e f f}\) until the value of f converges.
