Guest Question.

In Example 5.1 and (b) we ignored spin (or, if you prefer, we assumed the particles are in the same spin state).

(a) Do it now for particles of spin 1/2. Construct the four lowest-energy configurations, and specify their energies and degeneracies. Suggestion: Use the notation \( \psi_{n_{1} n_{2}}|s m\rangle, \text { where } \psi_{n_{1} n_{2}} \) is defined in Example 5.1 and \( |s m\rangle \) in Section 4.4.3.

(b) Do the same for spin 1. Hint: First work out the spin-1 analogs to the spin-1/2 singlet and triplet configurations, using the Clebsch–Gordan coefficients; note which of them are symmetric and which antisymmetric.

Admin
Solution

(a) Distinguishable:

ground state: \( \psi_{11}|s m\rangle, \text { where }|s m\rangle \text { could be }|00\rangle,|11\rangle,|10\rangle \text {, or }|1-1\rangle \). E = 2K, d = 4.

1st excited: \( \psi_{12}|s m\rangle \text { or } \psi_{21}|s m\rangle \text {, where }|s m\rangle \) could be any of the same four spin states. E = 5K, d = 8.

2nd excited: \( \psi_{22}|s m\rangle, \text { where }|s m\rangle \text { could be }|00\rangle,|11\rangle,|10\rangle \text {, or }|1-1\rangle \). E = 8K, d = 4.

3rd excited: \( \psi_{13}|s m\rangle \text { or } \psi_{31}|s m\rangle \text {, where }|s m\rangle \). could be any of the same four spin states. E = 10K, d = 8.

Identical:

ground state: \( \psi_{11}|00\rangle \). E = 2K, d = 1.

1st excited: \( \frac{1}{\sqrt{2}}\left(\psi_{12}+\psi_{21}\right)|00\rangle \text { or } \frac{1}{\sqrt{2}}\left(\psi_{12}-\psi_{21}\right)|s m\rangle \text {, where }|s m\rangle \text { could be }|11\rangle,|10\rangle \text {, or }|1-1\rangle \). E = 5K, d = 4.

2nd excited: \( \psi_{22}|00\rangle \). E = 8K, d = 1.

3rd excited: \( \frac{1}{\sqrt{2}}\left(\psi_{13}+\psi_{31}\right)|00\rangle \text { or } \frac{1}{\sqrt{2}}\left(\psi_{13}-\psi_{31}\right)|s m\rangle \text {, where }|s m\rangle \text { could be }|11\rangle,|10\rangle \text {, or }|1-1\rangle \). E = 10K, d = 4.

(b) Two particles of spin 1 can combine to make spin 0, spin 1, or spin 2. From the 1 × 1 C-G table:

\( \text { singlet: }\left\{|00\rangle=\frac{1}{\sqrt{3}}|111-1\rangle-\frac{1}{\sqrt{3}}|1100\rangle+\frac{1}{\sqrt{3}}|11-11\rangle\right. \), (symmetric).

\( \text { triplet: } \begin{cases}|11\rangle=\frac{1}{\sqrt{2}}|1110\rangle-\frac{1}{\sqrt{2}}|1101\rangle, & \text { (antisymmetric), } \\ |10\rangle=\frac{1}{\sqrt{2}}|111-1\rangle-\frac{1}{\sqrt{2}}|11-11\rangle, & \text { (antisymmetric), } \\ |1-1\rangle=\frac{1}{\sqrt{2}}|110-1\rangle-\frac{1}{\sqrt{2}}|11-10\rangle, & \text { (antisymmetric). }\end{cases} \).

\( \text { quintuplet: } \begin{cases}|22\rangle=|1111\rangle, & \text { (symmetric), } \\ |21\rangle=\frac{1}{\sqrt{2}}|1110\rangle+\frac{1}{\sqrt{2}}|1101\rangle, & \text { (symmetric), } \\ |20\rangle=\frac{1}{\sqrt{6}}|111-1\rangle+\sqrt{\frac{2}{3}}|1100\rangle+\frac{1}{\sqrt{6}}|11-11\rangle, & \text { (symmetric), } \\ |2-1\rangle=\frac{1}{\sqrt{2}}|110-1\rangle+\frac{1}{\sqrt{2}}|11-10\rangle, & \text { (symmetric), } \\ |2-2\rangle=|11-1-1\rangle, & \text { (symmetric). }\end{cases} \).

Distinguishable:

ground state: \( \psi_{11}|s m\rangle \text {, where }|s m\rangle \) could be any of the nine spin states. E = 2K, d = 9.

1st excited: \( \psi_{12}|s m\rangle \text { or } \psi_{21}|s m\rangle \text {, where }|s m\rangle \) . could be any of the nine spin states. E = 5K, d = 18.

2nd excited: \( \psi_{22}|s m\rangle, \text { where }|s m\rangle \) could be any of the nine spin states. E = 8K, d = 9.

3rd excited: \( \psi_{13}|s m\rangle \text { or } \psi_{31}|s m\rangle, \text { where }|s m\rangle \) could be any of the nine spin states. E = 10K, d = 18.

Identical:

ground state: \( \psi_{11}|s m\rangle, \text { where }|s m\rangle \) is any of the singlet or quintuplet spin states. E = 2K,d = 6.

1st excited: \( \frac{1}{\sqrt{2}}\left(\psi_{12}+\psi_{21}\right)|s m\rangle, \text { where }|s m\rangle \) is any of the singlet or quintuplet spin states,

\( \text { or } \frac{1}{\sqrt{2}}\left(\psi_{12}-\psi_{21}\right)|s m\rangle \text {, where }|s m\rangle \) is any of the triplet spin states. E = 5K, d = 9.

2nd excited: \( \psi_{22}|s m\rangle \text {, where }|s m\rangle \) is any of the singlet or quintuplet spin states. E = 8K, d = 6.

3rd excited: \( \frac{1}{\sqrt{2}}\left(\psi_{13}+\psi_{31}\right)|s m\rangle, \text { where }|s m\rangle \) is any of the singlet or quintuplet spin states,

\( \text { or } \frac{1}{\sqrt{2}}\left(\psi_{13}-\psi_{31}\right)|s m\rangle, \text { where }|s m\rangle \) is any of the triplet spin states. E = 10K, d = 9.
