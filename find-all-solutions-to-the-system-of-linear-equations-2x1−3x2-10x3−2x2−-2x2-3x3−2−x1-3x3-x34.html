Guest Question.

Find all solutions to the system of linear equations

2×1−3×2 + 10×3=−2×2− 2×2+ 3×3=−2−x1+ 3×3+ x3=4

Admin
Solution

We begin by converting the system to an augmented matrix.

[21−1−3−2310 3 1|−2−24]

• Interchange rows (focus on column 1). We focus on the first column of the matrix, which contains the coefficients of \(x_{1}\). Although this step is not required, exchanging Row 1 and Row 2 will move a 1 into the upper left position and avoid the early introduction of fractions.

[21−1−3−231031|−2−24] ∼ [ 12−1−2−333101|−2−24]

– As with linear systems, we use the symbol ~ to indicate that two matrices are equivalent

– The compact notation for this operation is R1↔R2.

• Add a multiple of one row to another (focus on column 1). To transform the system to echelon form, we need to introduce zeros in the first column below Row 1. This requires two operations. Focusing first on Row 2, since (− 2)(1) + 2 = 0, we add −2 times Row 1 to Row 2 and replace Row 2 with the result.

[12−1−2−333101|−2−24] ∼ [ 10−1−213341|−224]

– The compact notation for this operation is − 2 \( R_{1} + R_{2} → R_{2}\).

Focusing now on Row 3, since (1)(1) +(−1) = 0 we add 1 times Row 1 to Row 3 and replace Row 3 with the result.

[10−1−213341|−224] ∼ [100−211344|−222]

– The compact notation for this operation is \( R_{2} + R_{3} → R_{3}\).

• Add a multiple of one row to another (focus on column 2). With the first column complete, we move down to the second row and to the right to the second column. Since (−1)(1) + (1) = 0, we add −1 times Row 2 to Row 3 and replace Row 3 with the result.

[100−211344|−222] ∼ [100−210340|−220]

– The compact notation for this operation is − \( R_{2} + R_{3} → R_{3}\)

We now extract the transformed system of equations from the matrix. The row of zeros indicates that one of the equations in the transformed system is 0 = 0. Since any choice of values for the variables will satisfy 0 = 0, this equation contributes no information about the solution set so can be ignored. The new equivalent system is therefore

x1−2×2+3×3=−2×2+4×3=2

Back substitution can be used to show that the general solution is

x1=2−11s1x2=2−4s1x3=s1

where \(s_{1}\) can be any real number. We can substitute into the original system to verify our solution.

(2−11s1)−2(2−4s1)+3s1=2−11s1−4+8s1+3s1=−2,−(2−11s1)+3(2−4

s1)+s1=−2+11s1+6−12s1+s1=4
