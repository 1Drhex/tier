Guest Question.

The single-line diagram of a three-phase power system is shown in Figure 9.17. Equipment ratings are given as follows:

Synchronous generators:

\(\begin{array}{llll}\text { G1 } & 1000 MVA & 15 kV & X _{d}^{\prime \prime}= X _{2}=0.18, X _{0}=0.07 \text { per unit } \\\text { G2 } & 1000 MVA & 15 kV & X _{d}^{\prime \prime}= X _{2}=0.20, X _{0}=0.10 \text { per unit } \\\text { G3 } & 500 MVA & 13.8 kV & X _{d}^{\prime \prime}= X _{2}=0.15, X _{0}=0.05 \text { per unit } \\\text { G4 } & 750 MVA & 13.8 kV & X _{d}^{\prime \prime}=0.30, X _{2}=0.40, X _{0}=0.10 \text { per unit }\end{array}\)

Transformers:

T1 1000 MVA 15 kV Δ /765 kV Y X = 0.10 per unit
T2 1000 MVA 15 kV Δ/765 kV Y X = 0.10 per unit
T3 500 MVA 15 kV Y/765 kV Y X = 0.12 per unit
T4 750 MVA 15 kV Y/765 kV Y X = 0.11 per unit

Transmission lines:

\(\begin{array}{lll}1-2 & 765 kV & X _{1}=50 \Omega, \quad X _{0}=150 \Omega \\1-3 & 765 kV & X _{1}=40 \Omega, \quad X _{0}=100 \Omega \\2-3 & 765 kV & X _{1}=40 \Omega, \quad X _{0}=100 \Omega\end{array}\)

The inductor connected to Generator 3 neutral has a reactance of 0.05 per unit using generator 3 ratings as a base. Draw the zero-, positive-, and negative-sequence reactance diagrams using a 1000-MVA, 765-kV base in the zone of line 1–2. Neglect the Δ–Y transformer phase shifts.

 

Admin
Solution

Calculation of per unit reactances

Synchronous generators:

G1 \(X_{1}=X_{d}^{\prime \prime}=0.18 \quad X_{2}=X_{d}^{\prime \prime}=0.18 \quad X_{0}=0.07\)

G2 \(X_{1}=X_{d}^{\prime \prime}=0.20 \quad X_{2}=X_{d}^{\prime \prime}=0.20 \quad X_{0}=0.10\)

G3

\(\begin{aligned}X_{1} &=X_{d}^{\prime \prime}=0.15\left(\frac{13.8}{15}\right)^{2}\left(\frac{1000}{500}\right) \\&=0.2539 \\X_{2} &=X_{d}^{\prime \prime}=0.2539\end{aligned}\) \(\begin{aligned}X_{0} &=0.05\left(\frac{13.8}{15}\right)^{2}\left(\frac{1000}{500}\right) \\&=0.08464 \\3 X_{n} &=3 X_{0}=0.2539\end{aligned}\)

G4

\(\begin{aligned}X_{1} &=X_{d}^{\prime \prime}=0.30\left(\frac{13.8}{15}\right)^{2}\left(\frac{1000}{750}\right) \\&=0.3386\end{aligned}\)

\(\begin{aligned}X_{2} &=0.40\left(\frac{13.8}{15}\right)^{2}\left(\frac{1000}{750}\right) \\&=0.4514\end{aligned}\) \(\begin{aligned}&X_{0}=0.10\left(\frac{13.8}{15}\right)^{2}\left(\frac{1000}{750}\right) \\&=0.1129\end{aligned}\)

Transformers:

\(\begin{aligned}&X_{T 1}=0.10 \quad X_{T 2}=0.10 \quad X_{T 3}=0.12\left(\frac{1000}{500}\right) \\&X_{T 4}=0.11\left(\frac{1000}{750}\right)=0.1467 \quad=0.24\end{aligned}\)

Transmission Lines:

\(Z_{\text {base } H}=\frac{(765)^{2}}{1000}=585.23 \Omega\)

\(\begin{aligned}&\text { Positive/Negative Sequence }\\&X_{12}=\frac{50}{585.23}=0.08544\\&X_{13}=X_{23}=\frac{40}{585.23}\\&=0.06835\end{aligned}\) \(\begin{aligned}&\text { Zero Sequence }\\&\begin{aligned}X_{12} &=\frac{150}{585.23} \\&=0.2563 \\X_{13} &=X_{23}=\frac{100}{585.23} \\&=0.1709\end{aligned}\end{aligned}\)
