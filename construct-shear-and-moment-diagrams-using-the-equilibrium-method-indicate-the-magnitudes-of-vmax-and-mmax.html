Guest Question.

Construct shear and moment diagrams using the equilibrium method. Indicate the magnitudes of Vmax and Mmax.

Admin
Solution

Secction a-a: @ x= 0 to x = 10’
∑Fy = 0: +20k – ωx – V = 0
V = 20k – 2x
(as x increases, V decreases)
∑Ma-a = 0: -20(x) +(ωx)(x/2) + M = 0
M = 20x – x2
(M increases with x; 2nd degree curve)

Section b-b: @ x = 10’ to x = 20’
∑Fy = 0: +20k – 2(k/ft)(10’) – V = 0
V = 0
(No shear between x = 10′ to x = 20’)
∑Mb-b = 0: -20x + 2(k/ft)(10’)(x – 5’) – M = 0
M = -100k-ft.
(assumed direction of M on the FBD is incorrect; it should be counter-clock)

Section c-c: @ x = 20’ to x = 30’
∑Fy = 0: +20k – 2(k/ft)(10’) – 2(km)(x – 20’) + V = 0
V = 2x – 40k (V increases with x)
∑M = 0: -20(x) + 2(10)(x-5’) + 2(x-20′)(x – 20′)/2
+ M = 0
M = -x2 + 40x -300
(M is a function of x ….2nd degree)

“V” Diagram:
@ x= 10’; V = 0 (V = 20k – 2x )
Between x = 10’ to x = 20’; V = 0
Between x = 20’ to x = 30’;
V goes from 0 to -20k
‘M’ Diagram:
@ x = 0 M = 0
(no moment at the hinge)
Betweenx = 0 tox = 10′, M is increasing.
@ x = 10’, M = 100 k-ft.
(positive bending)
Between x = 20’ to x = 30’ M is deer easing.
@ x = 30’, M = 0.
