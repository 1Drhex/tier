Guest Question.

Determine the resistance of the following four round rods of material, each 1 m long and 2 mm in diameter: copper, brass, stainless steel, and graphite. Also figure out how much power is lost to heating if a current of 0.2 A flows through each one.

Admin
Solution

Using Eq. (2.5), \(\rho \equiv R\frac{A}{L}\)

\(R = \rho\frac{L}{A} = \rho\frac{L}{\pi r^{2} } = \rho\frac{1 m}{\pi \left(0.001 m\right)^{2} } = \rho\frac{1 m}{3.14 \times 10^{-6}m^{2} } = \rho\left(3.18\times10^{5}m^{-1} \right)\)

Using Table 2.2:

\(\rho _{copper} = 1.72 \times 10^{-8} \Omega m, \rho _{brass} = 7.0 \times 10^{-8} \Omega m, \rho _{steel} = 7.2 \times 10^{-7} \Omega m, \rho _{graphite} = 3.5 \times 10^{-5} \Omega m\)

Substituting this into the resistance expression, we get:

\(R_{ copper} = 5.48 \times 10^{-3} \Omega, R_{brass} = 2.23 \times 10^{-2} \Omega, R_{ steel} = 2.31 \times 10^{-1} \Omega, R_{ graphite} = 11.1 \Omega\)

The power loss we get by using Eq. 2.10: P = VI = (IR)I = I²R = (0.2 A)²R = (0.04 A²)R:

\(P_{copper} = 2.2 \times 10^{-4} W, P_{brass} = 8.9 \times 10^{-4} W, P_{steel} = 9.2 \times 10^{-3} W, P_{graphite} = 0.44 W\)
