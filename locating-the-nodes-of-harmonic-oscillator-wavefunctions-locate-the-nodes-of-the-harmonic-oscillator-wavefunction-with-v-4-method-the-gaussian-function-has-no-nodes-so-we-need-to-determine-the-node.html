Locating the nodes of harmonic oscillator wavefunctions
Locate the nodes of the harmonic oscillator wavefunction with v = 4.
Method The Gaussian function has no nodes, so we need to determine the nodes of the Hermite polynomials by determining the values of x at which they pass through zero. The polynomials are listed in Table 2.1. We will need the solutions of a quadratic equation:

\(ax^{2}+bx+c=0\) \(x=\frac{-b\pm (b^{2}-4ac )^{1/2} }{2a}\)

Table 2.1 Hermite polynomials

v \(H_{v}(z)\)
0 1
1 \(2z\)
2 \(4z^{2}-2\)
3 \(8z^{3}-12z\)
4 \(16z^{4}-48z^{2}+12\)
5 \(32z^{5}-160z^{3}+120z\)
6 \(64z^{6}-480z^{4}+720z^{2}-120\)
7 \(128z^{7}-1344z^{5}+3360z^{3}-1680z\)
8 \(256z^{8}-3584z^{6}+13440z^{4}+1680\)
Differential equation: \(H^{\prime \prime }_{v} -2zH^{\prime }_{v}+2vH_{v}=0 \)
Recursion relation: \(H_{v+1}=2zH_{v}-2vH_{v-1}\)
Orthogonality: \(\int_{-\infty }^{+\infty }{H_{v}(z)H_{v^{\prime } }(z)e^{-z^{2} } }dz=0\) for \(v\neq v^{\prime }\)
Normalization: \(\int_{-\infty }^{+\infty }{H_{v}(z)^{2} e^{-z^{2} } }dz=\pi ^{1/2}2^{v} v!\)
Admin
Solution

Because \(H_{4}(\alpha x)=16(\alpha x)^{4}-48(\alpha x)^{2} +12\), we need to solve

\(16(\alpha x)^{4}-48(\alpha x)^{2} +12 =0 \)

This is a quadratic equation in \(y=(\alpha x)^{2}\),

\(16y^{2}-48y +12 =0 \)

with roots

\(y=\frac{48\pm \left\{48^{2}-4\times 16\times 12\right\}^{1/2} }{2\times 16}=2.7247\) and \(0.2753\)

The nodes are therefore at \(x=\pm 1.6507/\alpha\) and \(\pm 0.5246/\alpha\) (see Fig. 2.27).

Comment For more complicated polynomials it is best and sometimes essential to use numerical methods (the root extracting program of a mathematics package). Figure 2.28 shows the pattern of nodes: note how they spread away from the origin but become more uniformly distributed as v increases.
