Guest Question.

(a) In practice, one deals with the inductive reactance of the line per phase per mile and use the logarithm to the base 10. Show that Eq. (4.5.9) of the text can be rewritten as

\(\begin{aligned}&x=k \log \frac{ D }{r^{\prime}} \text { ohms per mile per phase }\\&=x_{d}+x_{a}\end{aligned}\)

where \(x_{d}=k \log D\) is the inductive reactance spacing factor in ohms per mile
\(x_{a}=k \log \frac{1}{r^{\prime}}\) is the inductive reactance at 1-ft spacing in ohms per mile
\(k=4.657 \times 10^{-3} f=0.2794 \text { at } 60 Hz\)

(b) Determine the inductive reactance per mile per phase at 60 Hz for a single-phase line with phase separation of 10 ft and conductor radius of 0.06677 ft. If the spacing is doubled, how does the reactance change?

__________________________

Eq. (4.5.9) : \(L _{a}=\frac{\lambda_{a}}{I_{a}}=2 \times 10^{-7} \ln \frac{ D }{r^{\prime}} \quad H / m \text { per phase }\)

Solution

(a) Eq. (4.5.9): \(L=2 \times 10^{-7} \ln \frac{D}{r^{\prime}} H / m \text { Per Phase }\)

\(\begin{aligned}X &=\omega L=4 \pi f \times 10^{-7} \ln \left(D / r^{\prime}\right) \Omega / m / \text { Phase } \\&=f . \Delta \pi \times 10^{-7}(1609.34) \ln \left(D / r^{\prime}\right) \Omega / MILE / PH .\\&=f .4 \pi(1609.34)(2.3026) 10^{-7} \log \left(D / r^{\prime}\right) \Omega / mi / ph . \\&=4.657 \times 10^{-3} f \log \left(D / r^{\prime}\right) \Omega / mi / ph .\\ &=0.2794 \log \left(D / r^{\prime}\right) \Omega / mi / ph \text { . at } f=60 HZ .\end{aligned}\)

âˆ´ \(X=k \log \left(\frac{D}{r^{\prime}}\right)=k \log D+k \log \left(\frac{1}{r^{\prime}}\right), \text { Where } k=4.657 \times 10^{-3} f \leftarrow\)

 

(b) \(r^{\prime}=r \cdot e^{-1 / 4}=0.06677(0.7788)=0.052 ft \text { . }\)

\(\begin{aligned}X_{a} &=k \log \frac{1}{r^{\prime}}=0.2794 \log \left(\frac{1}{0.052}\right)=0.35875 \\X_{d} &=k \log D=0.2794 \log (10)=0.2794 \\X &=X_{a}+X_{d}=0.63815 \Omega / mi / ph . \leftarrow\end{aligned}\)

When spacing is doubled, \(X_{d}=0.36351 \text { and } X=0.72226 \Omega / mi / ph . \leftarrow\)
