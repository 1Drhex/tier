Guest Question.

For vector field \( B = \hat{z} cos \phi /r,\) verify Stokes’s theorem for a segment of a cylindrical surface defined by \( r = 2, π/3 ≤ \phi ≤ π/2, and 0 ≤ z ≤ 3\) (Fig. 3-24).

Admin
Solution

Stokes’s theorem states that

\(\int_{S}^{}{(\nabla \times B) . ds} = \oint_{C}^{}{B . dl} \)

Left-hand side: With B having only a component \( B_z = cos φ/r,\) use of the expression for \(\nabla \times B \) in cylindrical coordinates from the inside back cover of the book gives

\( \nabla \times B = \hat{r} (\frac{1}{r}\frac{\delta b_z}{\delta \phi } – \frac{\delta b_\phi }{\delta Z} ) + \hat{\phi } (\frac{\delta b_r}{\delta z } – \frac{\delta B_z}{\delta r} ) + \hat{z } \frac{1}{r} (\frac{\delta}{\delta r}(rB_\phi )-\frac{\delta B_r}{\delta \phi } ) \)

 

\(= \hat{r} \frac{1}{r} \frac{\delta}{\delta \phi }(\frac{\cos \phi }{r} )-\hat{\phi }\frac{\delta}{\delta r}(\frac{\cos \phi }{r} ) \)

 

\( = -\hat{r}\frac{\phi }{r^2} + \hat{\phi } \frac{\cos \phi }{r^2} \)

The integral of \(\nabla \times B \) over the specified surface S is

\( \int_{S}^{}{(\nabla \times B) . ds}\)

 

\(=\int_{z=0}^{3}{\int_{\phi =\pi /3}^{\pi /2}{(-\hat{r}\frac{\sin \phi }{r^2}+ \hat{\phi }\frac{\cos \phi }{r^2}).\hat{r} r d\phi dz} } \)

 

\( =\int_{0}^{3}\int_{\pi /3}^{\pi /2} – \frac{\sin \phi }{r} d\phi dz = -\frac{3}{2r} = -\frac{3}{4} \)

 

Right-hand side: The surface S is bounded by contour C = abcd shown in Fig. 3-24. The direction of C is chosen so that it is compatible with the surface normal \( \hat{r}\) by the right-hand rule. Hence,

\(\oint_{C}^{}{B . dl} = \int_{a}^{b}{B_{ab}} .dl+\int_{b}^{c}{B_{bc}} .dl +\int_{c}^{d}{B_{c}} .dl +\int_{d}^{a}{B_{da}} .dl \)

 

where \( B_{ab}\), \(B_{bc} \), \(B_{cd} \), and \(B_{da} \) are the field B along segments ab, bc, cd, and da, respectively. Over segment ab, the dot product of \(B_{ab} = \hat{z} (cos \phi ) /2 and dl = \hat{\phi } r d\phi \) is zero, and the same is true for segment cd. Over segment bc, \(\phi =\pi /2\); hence, \(B_{bc} = \hat{z}(cos π/2)/2 = 0\). For the last segment, \( B_{da} = \hat{z}(cos π/3)/2 = \hat{z}/4 and dl = \hat{z} dz\). Hence,

\(\oint_{C}^{}{B . dl} = \int_{a}^{d}{(\hat{z}\frac{1}{4} ). \hat{z} dz} \)

 

\( = \int_{0}^{}{\frac{1}{4} dz} = – \frac{3}{4} \)
