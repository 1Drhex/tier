Solution

[∑MA = 0] – 3k(12′) – 7k(32′) – 2k(42′) + Ey(60′) = 0

Ey = 5.73k (25.5kN)

[∑Fy = 0] + Ay – 3k – 7k – 2k + 5.73k = 0

Ay = 6.27k (27.9kN)

 

[∑MC = 0] – 2k(10′) + 5.73k(28′) – Ex(12′) = 0

Ex = 11.7k (52.1kN); E = ED = 13.03k (58.0kN)

From the original FBD:

[∑Fx = 0 ] Ax = 11.7k (52.1kN)

∴ A = AB = 13.27k (59.1kN)

FBD(a)

[∑Fy =0] + CBy – 7k – 2k + 5.73k = 0

∴ CD = 12.28k (54.5kN)
