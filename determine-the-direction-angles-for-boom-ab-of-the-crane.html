Guest Question.

Determine the direction angles for boom AB of the crane.

Admin
Solution

Road Map We denote the position vector of the crane’s boom from point A to point B by \(\vec{r} _{AB} \). The \(35^{\circ }\) and \(73^{\circ }\) angles reported in the figure are indeed direction angles: the \(35^{\circ }\) angle is measured from the positive y direction to the direction of \(\vec{r} _{AB} \) , and the \(73^{\circ }\) angle is measured from the positive z direction to the direction of \(\vec{r} _{AB} \) . Thus, \(\theta _{y} = 35^{\circ } \) and \(\theta _{z} = 73^{\circ } \). The remaining direction angle \(\theta _{x} \) must satisfy Eq. (2.26).

\(\cos ^{2} \theta _{x} + \cos ^{2} \theta _{y} + \cos ^{2} \theta _{z} = 1\) (2.26)

Governing Equations & Computation Knowing that \(\theta _{y} = 35^{\circ } \) and \(\theta _{z} = 73^{\circ } \) , we solve Eq. (2.26) to obtain

\(\cos ^{2} \theta _{x} = 1 – \cos ^{2} \theta _{y} – \cos ^{2} \theta _{z}\) (1)
\(\cos \theta _{x}= \pm \sqrt{1 – \cos ^{2} \theta _{y} – \cos ^{2} \theta _{z}}\) (2)
\(\theta _{x} = \cos ^{-1} \left(\pm \sqrt{1 – \cos ^{2} \theta _{y} – \cos ^{2} \theta _{z}} \right)\)

\(= \cos ^{-1} \left(\pm \sqrt{1 – \cos ^{2} 35^{\circ } – \cos ^{2} 73^{\circ }} \right)\)
\(= \cos ^{-1} \left(\pm 0\cdot 493\right)\) (3)

Using the inverse cosine function on an electronic calculator provides the solutions \(\theta _{x}=\cos ^{-1} \left(+ 0\cdot 493 \right) =6 0\cdot 4^{\circ }\) and \(\theta _{x}=\cos ^{-1} \left(- 0\cdot 493 \right) =119\cdot 6^{\circ }\) . In fact, there are an infinite number of additional solutions, but the only two solutions in the interval \(0^{\circ } \leq \theta _{x} \leq 180^{\circ } \) are \(60\cdot 4^{\circ }\) and \(119\cdot 6^{\circ }\) . Inspection will usually allow us to easily determine which of the solutions for the direction angle is correct for our physical problem, and you should examine Fig. 1 now to see if you can make this determination. If there is uncertainty, then an unequivocal answer can be obtained by computing the unit vectors corresponding to the two sets of direction angles to see which of the two vectors points in the proper direction. To show this, we calculate these unit vectors as

for \(\theta _{x} = 60\cdot 4^{\circ }: \hat{u} _{1} = \cos \left(60\cdot 4^{\circ }\right) \hat{\imath} + \cos \left(35\cdot 0 ^{\circ } \right) \hat{\jmath} + \cos \left(73\cdot 0 ^{\circ } \right) \hat{k}\)

\(=0\cdot 493 \hat{\imath} + 0\cdot 819 \hat{\jmath} + 0\cdot292 \hat{k}\) (4)

for \(\theta _{x} = 119\cdot 6^{\circ }: \hat{u} _{2} = \cos \left(119\cdot 6^{\circ }\right) \hat{\imath} + \cos \left(35\cdot 0 ^{\circ } \right) \hat{\jmath} + \cos \left(73\cdot 0 ^{\circ } \right) \hat{k}\)

\(=-0\cdot 493 \hat{\imath} + 0\cdot 819 \hat{\jmath} + 0\cdot292 \hat{k}\) (5)

Notice that the only difference between \(\hat{u} _{1}\) and \(\hat{u} _{2}\) in Eqs. (4) and (5) is the sign of the x component. Figure 1 clearly shows \(\vec{r} _{AB} \) has a negative x component. So the solution for \(\theta _{x}\) and \(\hat{u} _{1}\) in Eq. (4) is physically meaningless and must be discarded. Thus, the direction angles for boom AB are

\(\theta _{x} = 119\cdot 6^{\circ } , \theta _{y} = 35\cdot 0 ^{\circ } \) and \(\theta _{z} = 73\cdot 0^{\circ }\) (6)

Discussion & Verification As a quick partial check of solution accuracy, you should verify that the values for \(\theta _{x}\), \(\theta _{y}\) and \(\theta _{z}\) in Eq. (6) satisfy the equation \(\cos ^{2} \theta _{x} + \cos ^{2} \theta _{y} + \cos ^{2} \theta _{z} = 1\) ; if this equation is not satisfied, then at least one of the angles is not a direction angle and an error has been made.
