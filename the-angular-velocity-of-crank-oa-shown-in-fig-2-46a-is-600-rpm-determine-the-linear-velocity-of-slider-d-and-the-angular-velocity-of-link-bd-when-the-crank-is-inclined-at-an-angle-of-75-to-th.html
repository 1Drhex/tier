Guest Question.

The angular velocity of crank OA shown in Fig.2.46(a) is 600 rpm. Determine the linear velocity of slider D and the angular velocity of link BD when the crank is inclined at an angle of 75째 to the vertical. The dimensions of the various links are: OA = 28 mm, AB = 44 mm, BC = 49 mm, and BD =46 mm. The centre distance between the centres of rotation O and C is 65 mm. The path of travel of the slider is 11 mm below the fixed point C. The slider moves along a horizontal path and OC is vertical.

Admin
Solution

\( \text { Angular velocity of } O A, \omega=\frac{2 \pi N}{60}=\frac{2 \pi \times 600}{60}=62.83 rad / s \).

\( \text { Linear velocity of } A, v_{a}=\omega \times O A=62.83 \times 28 \times 10^{-3}=1.76 m / s \).

1. Draw configuration diagram as shown in Fig.2.46(a).
2. Draw velocity diagram as shown in Fig.2.46(b).

\( \text { 3. Draw oa perpendicular to } O A \text { such that } o a=v_{a}=3.52 cm \).

\( \text { 4. Draw } a b \perp A B \text { and } o b \perp B C \text { to intersect at } b \).

\( \text { 5. Draw } b d \perp B D \text { and od parallel to the path of slider } D \text { to intersect at } d \).

\( \text { Linear velocity of slider } D, v_{d}=o d=2.8 cm =1.4 m / s \).

\( \text { Linear velocity of } B D, v_{d b}=3.2 cm =1.6 m / s \).

\( \text { Angular velocity of } B D, \omega_{b d}=\frac{v_{d b}}{B D}=\frac{1.6}{0.046}=34.78 rad / s \text { (clockwise about } B \text { ) } \).
