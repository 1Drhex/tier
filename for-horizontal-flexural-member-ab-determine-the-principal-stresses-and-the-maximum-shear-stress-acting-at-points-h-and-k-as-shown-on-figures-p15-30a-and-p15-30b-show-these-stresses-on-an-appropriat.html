Guest Question.

For horizontal flexural member AB, determine the principal stresses and the maximum shear stress acting at points H and K, as shown on Figures P15.30a and P15.30b. Show these stresses on an appropriate sketch for each point.

Admin
Solution

Moment of inertia about the z axis:

\(\begin{array}{lcccccc}{\text { Shape }} & \begin{array}{c}\text { Width } b \\\text { (in.) }\end{array} & \begin{array}{c}\text { Height } h \\\text { (in.) }\end{array} & \begin{array}{c}I_{C} \\\left( in .^{4}\right)\end{array} & \begin{array}{c}d=y_{i}-\bar{y} \\( in .)\end{array} & \begin{array}{c}d^{2} A \\\left( in .^{4}\right)\end{array} & \begin{array}{c}I_{C}+d^{2} A \\\left( in .^{4}\right)\end{array} \\\hline \text { flange } & 6.75 & 0.455 & 0.0530 & 6.7725 & 140.8683 & 140.9213 \\\text { web } & 0.285 & 13.090 & 53.2700 & 0.0000 & 0.0000 & 53.2700 \\\text { flange } & 6.75 & 0.455 & 0.0530 & -6.7725 & 140.8683 & 140.9213 \\\hline \text { Moment of inertia about the } z \text { axis }\left( in .^{4}\right)= 335.1125\end{array}\)

Equilibrium:

\(\begin{aligned}\Sigma F_{x}=& A_{x}-F_{B C} \cos \left(35.311^{\circ}\right)=0 \\\Sigma F_{y}=& A_{y}+F_{B C} \sin \left(35.311^{\circ}\right)-25 kips -15 kips =0 \\\Sigma M_{A}=& F_{B C}(12 ft )(12 in . / ft ) \sin \left(35.311^{\circ}\right) \\&+F_{B C}(10 in .) \cos \left(35.311^{\circ}\right) \\&-(25 kips )(4 ft )(12 in. / ft ) \\&-(15 kips )(8 ft )(12 in . / ft )=0\end{aligned}\)

 

\(\begin{aligned}\therefore F_{B C} &=28.886 kips \\A_{x} &=23.571 kips \\A_{y} &=23.304 kips\end{aligned}\)

 

Internal forces at H:

\(\begin{array}{ll}\Sigma F_{x}=A_{x}+F_{H}=0 & \therefore F_{H}=-23.571 kips \\\Sigma F_{y}=A_{y}-V_{H}=0 & \therefore V_{H}=23.304 kips \\\Sigma M_{H}=M_{H}-A_{y}(2 ft )(12 in . / ft )=0 & \therefore M_{H}=559.286 kip \text {-in. }\end{array}\)

Axial stress at H:

\(\begin{aligned}&A=2(6.75 in .)(0.455 in .)+(0.285 in. )(13.090 in. )=9.873 in .^{2} \\&\sigma_{\text {axial }}=\frac{-23.571 kips }{9.873 in. ^{2}}=2.387 ksi ( C )\end{aligned}\)

Shear stress at H:

\(\begin{aligned}&Q_{H}=(6.75 in. )(0.455 in .)(7 in .-0.2275 in .)+(0.285 in .)(3.545 in .)(3 in .+1.7725 in .)=25.622 in .{ }^{3} \\&\tau_{H}=\frac{(23.304 kips )\left(25.622 in .{ }^{3}\right)}{\left(335.1125 in. ^{4}\right)(0.285 in .)}=6.252 ksi\end{aligned}\)

Bending stress at H:

\(\sigma_{\text {bend }}=-\frac{(559.286 \text { kip-in. })(-3 \text { in. })}{335.1125 \text { in. }{ }^{4}}=5.007 ksi ( T ) \quad \text { (by inspection) }\)

Summary of stresses at H:

\(\begin{aligned}\sigma_{x} &=-2.387 ksi +5.007 ksi =2.619 ksi & \sigma_{y}=0 ksi \\\tau_{x y} &=-6.252 ksi\end{aligned}\)

Principal stress calculations:
The principal stress magnitudes can be computed from Eq. (12.12):

\(\begin{aligned}\sigma_{p 1, p 2} &=\frac{\sigma_{x}+\sigma_{y}}{2} \pm \sqrt{\left(\frac{\sigma_{x}-\sigma_{y}}{2}\right)^{2}+\tau_{x y}^{2}}=\frac{(2.619)+(0)}{2} \pm \sqrt{\left(\frac{(2.619)-(0)}{2}\right)^{2}+(-6.252)^{2}} \\&=1.310 \pm 6.387\end{aligned}\)

 

\(\begin{aligned}\sigma_{p 1} &=7.70 ksi \text { and } \quad \sigma_{p 2}=-5.08 ksi \\\tau_{\max } &=6.39 ksi \text{ (maximum in-plane shear str )}\\\sigma_{\text {avg }} &=1.310 ksi ( T ) \quad \text { (normal stress on planes of maximum in-plane shear stress) }\end{aligned}\)

 

\(\begin{aligned}\tan 2 \theta_{p} &=\frac{\tau_{x y}}{\left(\sigma_{x}-\sigma_{y}\right) / 2}=\frac{-6.252}{[(2.619)-(0)] / 2}=\frac{-6.252}{1.310}=-4.773333 \\&\left.\therefore \theta_{p}=-39.08^{\circ} \quad \text { (clockwise from the } x \text { axis to the direction of } \sigma_{p 1}\right)\end{aligned}\)

 

 

Internal forces at K:

\(\begin{aligned}\Sigma F_{x}=& A_{x}+F_{K}=0 \therefore F_{K}=-23.571\text { kips }\\\Sigma F_{y}=& A_{y}-V_{K}-25 \text { kips }-15 \text { kips }=0 \\& \therefore V_{K}=-16.696 \text { kips } \\\Sigma M_{K}=& M_{K}-A_{y}(10 ft )(12 in . / ft )+(25 kips )(6 ft )(12 \text { in. } / ft ) \\&+(15 kips )(2 ft )(12 in . / ft )=0 \\& \therefore M_{K}=636.428 kip – in .\end{aligned}\)

Axial stress at K:

\(\sigma_{\text {axial }}=\frac{-23.571 \text { kips }}{9.873 in .^{2}}=2.387 ksi ( C )\)

Shear stress at K:

\(\begin{aligned}&Q_{K}=(6.75 in .)(0.455 in .)(7 in .-0.2275 in .)+(0.285 in .)(4.545 in .)(2 in .+2.2725 \text { in. })=26.334 in .{ }^{3} \\&\tau_{K}=\frac{(16.696 kips )\left(26.334 in. ^{3}\right)}{\left(335.1125 in. ^{4}\right)(0.285 in .)}=4.604 ksi\end{aligned}\)

 

Bending stress at K:

\(\sigma_{\text {bend }}=-\frac{(636.428 \text { kip-in. })(2 \text { in. })}{335.1125 \text { in. }^{4}}=3.798 ksi ( C ) \quad \text { (by inspection) }\)

 

Summary of stresses at K:

\(\begin{aligned}\sigma_{x} &=-2.387 ksi -3.798 ksi =-6.186 ksi \quad \sigma_{y}=0 ksi \\\tau_{x y} &=4.604 ksi\end{aligned}\)

Principal stress calculations:
The principal stress magnitudes can be computed from Eq. (12.12):

\(\begin{aligned}\sigma_{p 1, p 2} &=\frac{\sigma_{x}+\sigma_{y}}{2} \pm \sqrt{\left(\frac{\sigma_{x}-\sigma_{y}}{2}\right)^{2}+\tau_{x y}^{2}}=\frac{(-6.186)+(0)}{2} \pm \sqrt{\left(\frac{(-6.186)-(0)}{2}\right)^{2}+(4.604)^{2}} \\&=-3.093 \pm 5.546\end{aligned}\)

 

\(\begin{aligned}\sigma_{p 1} &=2.45 ksi \quad \text { and } \quad \sigma_{p 2}=-8.64 ksi \\\tau_{\max } &=5.55 ksi \text{(maximum in-plane shear str ) } \\\sigma_{\text {avg }} &=3.09 ksi ( C ) \quad \text { (normal stress on planes of maximum in-plane shear stress) }\end{aligned}\)

 

\(\begin{aligned}\tan 2 \theta_{p} &=\frac{\tau_{x y}}{\left(\sigma_{x}-\sigma_{y}\right) / 2}=\frac{4.604}{[(-6.186)-(0)] / 2}=\frac{4.604}{-3.093}=-1.488502 \\&\left.\therefore \theta_{p}=-28.05^{\circ} \quad \text { (clockwise from the } x \text { axis to the direction of } \sigma_{p 2}\right)\end{aligned}\)
