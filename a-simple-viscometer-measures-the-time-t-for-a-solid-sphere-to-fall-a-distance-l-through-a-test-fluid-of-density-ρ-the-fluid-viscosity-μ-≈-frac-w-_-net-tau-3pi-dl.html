Guest Question.

A simple viscometer measures the time t for a solid sphere to fall a distance L through a test fluid of density ρ. The fluid viscosity

μ ≈ \(\frac { { W }_{ net }\tau }{ 3\pi DL } if \tau \ge \frac { 2\rho DL }{ \mu }\)

where D is the sphere diameter and W net is the sphere net weight in the fluid. (a) Show that both of these formulas are dimensionally homogeneous. (b) Suppose that a 2.5 mm diameter aluminum sphere

\( \left( density\quad 2700\quad k{ g }/{ { m }^{ 3 } } \right)\) falls in an oil of density \(875k{ g }/{ { m }^{ 3 } }\). If the time to fall 50 cm is 32 s, estimate the oil viscosity and verify that the inequality is valid.

Admin
Solution

(a) Test the dimensions of each term in the two equations:

\(\left\{ \mu \right\} \) = \( \left\{ \frac { \Mu }{ L\Tau } \right\} \) and \(\left\{ \frac { { W }_{ net }\tau }{ \left( 3\pi \right) DL } \right\} \) = \(\left\{ \frac { \left( \Mu { L }/{ { \Tau }^{ 2 } } \right) \left( \Tau \right) }{ \left( 1 \right) \left( L \right) \left( L \right) } \right\} \) = \( \left\{ \frac { \Mu }{ L\Tau } \right\} \) Yes dimensions OK.

\( \left\{ \tau \right\} \) = \(\left\{ \Tau \right\} \) and \(\left\{ \frac { 2\rho DL }{ \mu } \right\} \) = \(\left\{ \frac { \left( 1 \right) \left( { \Mu }/{ { L }^{ 3 } } \right) \left( L \right) \left( L \right) }{ { \Mu }/{ L\Tau } } \right\} \) = \(\left\{ \Tau \right\} \) Yes dimensions OK.

(b) Evaluate the two equations for the data. We need the net weight of the sphere in the fluid:

\( { W }_{ net }\) = \( \left( { \rho }_{ sphere }-{ \rho }_{ fluid } \right) g{ \left( Vol \right) }_{ fluid }\) = \(\left( 2700-875k{ g }/{ { m }^{ 3 } } \right) \left( 9.81{ m }/{ { s }^{ 2 } } \right) \left( { \pi }/{ 6 } \right) { \left( 0.0025 m \right) }^{ 3 } \) = 0.000146 N

Then μ = \(\frac { { W }_{ net }\tau }{ 3\pi DL } \) = \(\frac { \left( 0.000146 N \right) \left( 32 s \right) }{ 3\pi \left( 0.0025 m \right) \left( 0.5 m \right) } \) = \(0.40\frac { kg }{ m.s } \)

Check t = 32 s compared to \(\frac { 2\rho DL }{ \mu } \) = \( \frac { 2\left( 875 k{ g }/{ { m }^{ 3 } } \right) \left( 0.0025 m \right) \left( 0.5 m \right) }{ 0.40 k{ g }/{ m.s } } \) = 5.5 s OK, is greater
