Guest Question.

Designing a BJT biasing circuit

(a) Design a transistor biasing circuit as shown in Fig. 8.20(a). Use transistor Q2N2222, for which minimum \( \beta_{ F }=100, \text { nominal } \beta_{ F }=173, I_{ S }=3.295 \times 10^{-14} A , \text { and } V_{ A }=200 V \). The operating collector current is to be set at \( I_{ C }=10 mA \). The DC power supply is \( V_{ CC }=15 V \). Assume \( V_{ BE }=0.7 V \).

(b) Calculate the small-signal parameters \( r_{\pi}, g_{ m }, \text { and } r_{ o } \) of the transistor.

(c) Use PSpice/SPICE to verify your results in parts (a) and (b).

Admin
Solution

We will design for the worst-case value of \( \left.\beta_{ F } \text { (i.e., minimum } \beta_{ F }=100\right) . I_{ C }=10 mA , I_{ B }=10 mA / 100= 0.1 mA , \text { and } V_{ CC }=15 V \).

(a) Step 1. Calculate the values of \( \alpha_{ F } \text { and } I_{ E } \). From Eq. (8.78),

 

\( \alpha_{ F }=\frac{i_{ E }}{i_{ C }}=\frac{\beta_{ F }}{1+\beta_{ F }} \) (8.78)

 

\( \alpha_{ F }=\frac{\beta_{ F }}{\left(1+\beta_{ F }\right)}=\frac{100}{(1+100)}=0.99 \)

 

From Eq. (8.4),

 

\( \frac{i_{ C }}{i_{ E }} \equiv \alpha_{ F } \) (8.4)

 

\( I_{ E }=\frac{I_{ C }}{\alpha_{ F }}=\frac{I_{ C }}{0.99}=\frac{10 mA }{0.99}=10.1 mA \)

 

Step 2. Calculate the value of \( V_{ E } \). From Table 8.1 (Fig. 8.20),

 

\( V_{ E }=\frac{V_{ CC }}{3}=\frac{15}{3}=5 V \)

 

Step 3. Calculate the value of \( R_{ E } \) and its power rating:

 

\( R_{ E }=\frac{V_{ E }}{I_{ E }}=\frac{5 V }{10.1 mA }=495 \Omega \)

 

The power rating of \( R_{ E } \) is

 

\( P_{ RE }=I_{ E }^{2} R_{ E }=\left(10.1 \times 10^{-3}\right)^{2} \times 495=50.49 mW \)

 

Step 4. Calculate the value of \( V_{ CE } \). From Table 8.1 (Fig. 8.20),

 

\( V_{ CE }=\frac{V_{ CC }}{3}=\frac{15}{3}=5 V \)

 

Step 5. Calculate the value of \( R_{ C } \) and its power rating.

 

\(\begin{aligned}I_{ C } R_{ C } &=V_{ CC }-V_{ E }-V_{ CE }=15-5-5=5 V \\R_{ C } &=\frac{5}{I_{ C }}=\frac{5}{10 mA }=500 \Omega\end{aligned}\)

 

The power rating of \( R_{ C } \) is

\( P_{ RC }=I_{ C }^{2} R_{ C }=\left(10 \times 10^{-3}\right)^{2} \times 500=50 mW \)

 

Step 6. Calculate the values of \( R_{ Th } \text { and } V_{ Th } \). From Table 8.1 (Fig. 8.20),

 

\( R_{ Th }=\frac{\left(1+\beta_{ F }\right) R_{ E }}{10}=\frac{(1+100) \times 495}{10}=5 k \Omega \)

 

From Fig. 8.20(b),

\( V_{ Th }=V_{ E }+V_{ BE }+R_{ Th } I_{ B }=V_{ E }+0.7 V +5 k \Omega \times 0.1 mA =5+0.7+0.5=6.2 V \)

 

Step 7. Calculate the value of \( R_{1} \) and its power rating. From Eq. (8.71),

 

\( R_{1}=\frac{R_{ Th } V_{ CC }}{V_{ Th }} \) (8.71)

 

\( R_{1}=\frac{R_{ Th } V_{ CC }}{V_{ Th }}=\frac{5 k \times 15}{6.2}=12.1 k \Omega \)

 

The power rating of \( R_{1} \) is

 

\( P_{ R 1}=\frac{\left(V_{ CC }-V_{ Th }\right)^{2}}{R_{1}}=\frac{(15-6.2)^{2}}{12.1 k }=6.4 mW \)

 

Step 8. Calculate the value of \( R_{2} \) and its power rating. From Eq. (8.72),

 

\( R_{2}=\frac{R_{ Th } V_{ CC }}{V_{ CC }-V_{ Th }} \) (8.72)

 

\( R_{2}=\frac{R_{ Th } V_{ CC }}{V_{ CC }-V_{ Th }}=\frac{5 k \times 15}{15-6.2}=8.52 k \Omega \)

 

The power rating of \( R_{2} \) is

 

\( P_{ R 2}=\frac{V_{ Th }^{2}}{R_{2}}=\frac{6.2^{2}}{8.52 k }=4.51 mW \)

 

(b) From Eq. (8.40),

 

\( \frac{1}{r_{\pi}}=\frac{i_{ b }}{v_{ be }}=\left.\frac{d i_{ B }}{d v_{ BE }}\right|_{\text {at } Q \text {-point }}=\frac{I_{ B }}{V_{ T }}=\frac{I_{ B }}{25.8 mV } \) (8.40)

 

\( r_{\pi}=\frac{25.8 mV }{I_{ B }}=\frac{\beta_{ F } \times 25.8 mV }{I_{ C }}=\frac{100 \times 25.8 mV }{10 mA }=258 \Omega \)

 

From Eq. (8.42),

 

\( g_{ m }=\frac{i_{ c }}{v_{ be }}=\left.\frac{\beta_{ F } d i_{ B }}{d v_{ BE }}\right|_{\text {at } Q \text {-point }}=\frac{\beta_{ f } I_{ B }}{V_{ T }}=\frac{I_{ C }}{V_{ T }}=\frac{\beta_{ f }}{r_{\pi}} \) (8.42)

 

\( g_{ m }=\frac{I_{ C }}{V_{ T }}=\frac{10 mA }{25.8 mV }=387.6 mA / V \)

 

From Eq. (8.43),

\( \frac{1}{r_{ o }}=\frac{i_{ c }}{v_{ ce }}=\left.\frac{d i_{ C }}{d v_{ CE }}\right|_{\text {at } Q_{\text {-point }}}=\frac{I_{ C }}{ V _{ A }+V_{ CE }}=\frac{I_{ C }}{V_{ A }} \quad \text { for } V_{ A }>>V_{ CE } \) (8.43)

 

\( r_{ o }=\frac{V_{ CE }+V_{ A }}{I_{ C }}=\frac{(5+200) V }{10 mA }=20.5 k \Omega \)
