Guest Question.

Consider the frequency-dependent network in Fig. 8.41. Given the following circuit parameter values: L = 159 µH, C = 159 µF, and R = 10 Ω, let us demonstrate that this one network can be used to produce a low-pass, high-pass, or band-pass filter.

Admin
Solution

The voltage gain \(V_{R}\)/\(V_{s}\) is found by voltage division to be

\(\frac{V_{R}}{V_{S}}= \frac{R}{jwL+ R+1/(jwC)}= \frac{jw\left(\frac{R}{L} \right) }{(jw)^{2}+ jw \left(\frac{R}{L} \right) + \frac{1}{LC} } \)

 

\(\ \ \ \ \ \ = \frac{(62.9 \times 10^{3})jw}{-w^{2}+ (62.9 \times 10^{3})jw + 39.6 \times 10^{6}}\)

which is the transfer function for a band-pass filter. At resonance, \(w^{2}\) = 1/LC, and hence

\(\frac{V_{R}}{V_{S}}=1\)

Now consider the gain \(V_{L}/V_{S}\):

\(\frac{V_{L}}{V_{S}}=\frac{jwL}{jwL+ R+ 1/(jwC)}= \frac{-w^{2}}{(jw)^{2}+ jw \left(\frac{R}{L} \right) + \frac{1}{LC}}\)

 

\(\ \ \ \ \ \ = \frac{-w^{2}}{-w^{2}+ (62.9 \times 10^{3})jw + 39.6 \times 10^{6}}\)

which is a second-order high-pass filter transfer function. Again, at resonance,

\(\frac{V_{L}}{V_{S}}= \frac{jwL}{R}= jQ = j0.1\)

Similarly, the gain \(V_{C}/V_{S}\) is

\(\frac{V_{c}}{V_{s}}= \frac{1/(jwC)}{jwL + R+ 1/(jwC)}= \frac{\frac{1}{LC} }{(jw)^{2}+ jw\left(\frac{R}{L} \right)+\frac{1}{LC} } \)

 

\(\ \ \ \ \ \ = \frac{39.6 \times 10^{6}}{-w^{2}+ (62.9 \times 10^{3})jw + 39.6 \times 10^{6}}\)

which is a second-order low-pass filter transfer function. At resonant frequency,

\(\frac{V_{c}}{V_{s}}= \frac{1}{jwCR}= -jQ= -j0.1\)

Thus, one circuit produces three different filters depending on where the output is taken. This can be seen in the Bode plot for each of the three voltages in Fig. 8.42, where Vs is set to \(1\angle 0°\)V.

We know that Kirchhoff’s voltage law must be satisfied at all times. Note from the Bode plot that the \(V_{R}\)+ \(V_{C}\) + \(V_{L}\) also equals \(V_{S}\) at all frequencies! Finally, let us demonstrate KVL by adding \(V_{R}\), \( V_{L}\), and \(V_{c}\)·

\(V_{L}+ V_{R}+ V_{C}= \frac{\left((jw)^{2} + jw\left(\frac{R}{L} \right) + \frac{1}{\sqrt{LC} } \right) V_{S}}{(jw)^{2} + jw\left(\frac{R}{L} \right) + \frac{1}{\sqrt{LC} } } = V_{S}\)

Thus, even though \(V_{S}\) is distributed between the resistor, capacitor, and inductor based on frequency, the sum of the three voltages completely reconstructs \(V_{S}\) .
