Guest Question.

A rigid slab of weight 100 kN is supported on three columns each of height 4 m and cross-sectional area 300 mm\(^2\) arranged in line. The two outer columns are fabricated from material having a Young’s modulus of 80 000 N/mm\(^2\) and a coefficient of linear expansion of \( 1.85 \times 10^{-5} /{ }^{\circ} C \) the corresponding values for the inner column are 200 000 N/mm2 and \( 1.2 \times 10^{-5} /{ }^{\circ} C \). If the slab remains firmly attached to each column, determine the stress in each column and the displacement of the slab if the temperature is increased by 100°C.

Admin
Solution

The problem may be solved by determining separately the stresses and displacements produced by the applied load and the temperature rise; the two sets of results are then superimposed. Let subscripts o and i refer to the outer and inner columns, respectively. Using Eq. (7.38) we have

\(\sigma_{ S }=\frac{E_{ S }}{A_{ C } E_{ C }+A_{ S } E_{ S }} P \quad \sigma_{ C }=\frac{E_{ C }}{A_{ C } E_{ C }+A_{ S } E_{ S }} P\) (7.38)

 

\( \sigma_{ i }( load )=\frac{E_{ i }}{A_{ o } E_{ o }+A_{ i } E_{ i }} P \quad \sigma_{ o }( load )=\frac{E_{ o }}{A_{ o } E_{ o }+A_{ i } E_{ i }} P \) (i)

In Eq. (i)

\( A_{ o } E_{ o }+A_{ i } E_{ i }=2 \times 300 \times 80000+300 \times 200000=108.0 \times 10^{6} \)

Then

\(\begin{aligned}&\sigma_{ i }( load )=\frac{200000 \times 100 \times 10^{3}}{108.0 \times 10^{6}}=185.2 N / mm ^{2}(\text { compression }) \\&\sigma_{ o }( load )=\frac{80000 \times 100 \times 10^{3}}{108.0 \times 10^{6}}=74.1 N / mm ^{2}(\text { compression })\end{aligned}\)

Equation (7.49) give the values of \( \sigma_{ i } \text { (temp.) and } \sigma_{ o }(\text { temp. }) \) produced by the temperature rise, i.e.

\(\begin{aligned}&\sigma_{ C }=\frac{P_C}{A_S} =\frac{\Delta T\left(\alpha_{S}-\alpha_{C}\right) E_{C } A_{S } E_{ S }}{A_{ C } E_{ C }+A_{ S } E_{S }} \\&\sigma_{ S}=\frac{P_S}{A_S} =\frac{\Delta T\left(\alpha_{ S}-\alpha_{ C }\right) A_{ C } E_{ C } E_{ S }}{A_{ C } E_{ C }+A_{ S } E_{ S }}\end{aligned}\) (7.49)

\(\begin{aligned}&\sigma_{ o }(\text { temp. })=\frac{\Delta T\left(\alpha_{ i }-\alpha_{ o }\right) E_{ o } A_{ i } E_{ i }}{A_{ o } E_{ o }+A_{ i } E_{ i }} \\&\sigma_{ i }(\text { temp. })=\frac{\Delta T\left(\alpha_{ i }-\alpha_{ o }\right) A_{ o } E_{ o } E_{ i }}{A_{ o } E_{ o }+A_{ i } E_{ i }}\end{aligned}\) (ii)

In Eq. (ii) \( \alpha_{ o }>\alpha_{ i } \text { so that } \sigma_{ o } \text { (temp.) } \) is a compressive stress while \( \sigma_{ i }(\text { temp. }) \) is a tensile stress. Hence

\(\begin{aligned}\sigma_{ o }(\text { temp. }) &=\frac{100(1.2-1.85) \times 10^{-5} \times 80000 \times 300 \times 200000}{105.0 \times 10^{6}} \\&=-28.9 N / mm ^{2}(\text { i.e.compression }) \\\sigma_{ i }(\text { temp. }) &=\frac{100(1.2-1.85) \times 10^{-5} \times 2 \times 300 \times 80000 \times 200000}{108.0 \times 10^{6}} \\&=-57.8 N / mm ^{2}(\text { i.e.tension })\end{aligned}\)

Superimposing the sets of stresses, we obtain the final values of stress, \( \sigma_{ i } \text { and } \sigma_{ o } \), due to load and temperature change combined. Hence

\(\begin{aligned}&\sigma_{ i }=185.2-57.8=127.4 N / mm ^{2} \text { (compression) } \\&\sigma_{ o }=74.1+28.9=103.0 N / mm ^{2} \text { (compression) }\end{aligned}\)

The displacements due to the load and temperature change are found using Eqs (7.37) and (7.51), respectively. Hence

\(\delta=\frac{P L}{A_{C} E_{C}+A_{S} E_{S}}\) (7.37)

 

\(\delta=L_{0} \Delta T\left(\frac{\alpha_{ C } A_{ C } E_{ C }+\alpha_{ S } A_{ S } E_{ S }}{A_{ C } E_{ C }+A_{ S } E_{ S }}\right)\) (7.51)

 

\(\begin{aligned}\delta(\text { load })=& \frac{100 \times 10^{3} \times 4 \times 10^{3}}{108.0 \times 10^{6}}=3.7 mm (\text { contraction }) \\\delta(\text { temp. })=& 4 \times 10^{3} \times 100 \\& \times\left(\frac{1.85 \times 10^{-5} \times 2 \times 300 \times 80000+1.2 \times 10^{-5} \times 300 \times 200000}{108.0 \times 10^{6}}\right) \\=& 6.0 mm (\text { elongation })\end{aligned}\)

The final displacement of the slab involves an overall elongation of the columns of 6.0-3.7 = 2.3 mm.
