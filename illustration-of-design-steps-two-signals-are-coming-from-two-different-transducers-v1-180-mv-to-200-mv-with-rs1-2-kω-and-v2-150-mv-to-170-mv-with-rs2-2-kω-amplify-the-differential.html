Guest Question.

Illustration of design steps Two signals are coming from two different transducers: \( v_{1}=180 mV \) to 200 mV with \( R_{ s 1}=2 k \Omega \text { and } v_{2}=150 mV \text { to } 170 mV \text { with } R_{ s 2}=2 k \Omega \text {. } \) Amplify the differential voltage so that the output voltage is \( v_{0}=200\left(v_{1}-v_{2}\right) \). The gain variation should be less than ±3%. The load resistance is \( R_{ L }=5 k \Omega \). Determine the specifications of the amplifier.

Admin
Solution

Step 1. Study the design problem. \( v_{1}=180 mV \text { to } 200 mV \text { with } R_{ sl }=2 k \Omega \text {, and } v_{2}=150 mV \text { to } 170 mV \) with \( R_{ s 2}=2 k \Omega \).

Step 2. Identify the design specifications. \( A_{ v}=200 \pm 3 \%, R_{ L }=5 k \Omega \), and there is no bandwidth limit.

Step 3. Establish a design strategy, and find the functional block diagram solution. Since the input side will have two voltage signals whose difference is to be amplified, we need a voltage differential amplifier at the input stage. The output of this stage could be either voltage or current, which will be amplified by a gain stage, shown in Fig. 2.32(a).

Step 4. Find the circuit-level solution. We will use two identical transconductance amplifiers to give differential gain because it allows us to add (or subtract) two currents at a node. We will also use a transresistance amplifier at the output side to give the desired voltage gain and a low output resistance. This arrangement is shown in Fig. 2.32(b). Assuming ideal amplifiers of \( G_{ ms 1}=G_{ ms 2}=G_{ ms } \), the output voltage is given by

\( v_{ o }=\left(G_{ ms 1} v_{1}-G_{ ms 2} v_{2}\right) Z_{ mo }=Z_{ mo } G_{ ms }\left(v_{1}-v_{2}\right) \)

which gives \( A_{ vo }=Z_{ mo } G_{ ms } . \text { Assuming } G_{ ms }=20 mA / V \), we get

\( Z_{ mo }=\frac{A_{ vo }}{G_{ ms }}=\frac{200 V }{20 mA }=10 kV / A \)

Step 5. Evaluate your design. Let us take practical amplifiers with input and output resistances as shown in Fig. 2.32(c). Using Eq. (2.35), we can find the effective voltage gain \( A_{v} \) from

 

\( A_{ v }=\frac{v_{ o }}{v_{ s }}=\frac{i_{ o } R_{ L }}{v_{ s }}=\frac{v_{ o }}{v_{ i }} \times \frac{v_{ i }}{v_{ s }}=\frac{G_{ ms } R_{ o } R_{ L } R_{ i }}{\left(R_{ o }+R_{ L }\right)\left(R_{ i }+R_{ s }\right)}=\frac{G_{ ms } R_{ L }}{\left(1+R_{ L } / R_{ o }\right)\left(1+R_{ s } / R_{ i }\right)} \) (2.35)

 

\( A_{ v }=\frac{R_{ i 1}}{R_{ i 1}+R_{ s 1}} \times \frac{R_{ oc }}{R_{ oe }+R_{ i 2}} \times \frac{R_{ L }}{R_{ L }+R_{ o 2}} Z_{ mo } G_{ ms }\left(R_{ oe }=R_{ o1} \| R_{ o1 }^{\prime}\right) \)

 

Since \( A_{ v } \) will vary with variations in \( R_{ i 1}, R_{ i 1}^{\prime}, R_{ o 2}, Z_{ mo }, \text { and } G_{ ms } \), let us allow ±0.5% variation for each of them so that the overall variation is limited to ±2.5%. Assume \( R_{ s 1}, R_{ s 2} \text {, and } R_{ L } \) do not vary.

\( \frac{R_{ i 1}}{R_{ i 1}+R_{ s 1}}=\frac{R_{ i 1}}{R_{ i 1}+2 k }=0.995 \)

 

which gives \( R_{ i 1}=R_{ i 1}^{\prime} \geq 398 k \Omega \).

 

\( \frac{R_{ L }}{R_{ L }+R_{ o 2}}=\frac{5 k }{5 k +R_{ o 2}}=0.995 \)

 

which gives \( R_{ o 2} \leq 25.1 \Omega \).

 

Let us assume that \( R_{ o 1}=R_{ o1 }^{\prime} \geq 200 k \Omega . \text { Since } R_{ oe }=\left(R_{ o1} \| R_{ o1}^{\prime}\right)=100 k \Omega ,\)

\( \frac{R_{ o1 }^{\prime}}{R_{ o1 }^{\prime}+R_{ i 2}}=\frac{100 k }{100 k +R_{ i 2}}=0.995 \)

which gives \( R_{ i 2} \leq 502 \Omega \).

 

Step 6. Use PSpice/SPICE verification. The results of PSpice simulation for the circuit shown in Fig. 2.33 are as follows:

\(\begin{array}{lllllllll}\text { NODE } & \text { VOLTAGE } & \text { NODE } & \text { VOLTAGE } & \text { NODE } & \text { VOLTAGE } & \text { NODE } & \text { VOLTAGE } \\\text { ( 1) } & .2000 & (2) & .1990 & (3) & .1500 & (4) & .1493 \\(\text { 5) } & .4970 & (6) & .4970 & (7) & 9.9003 & (8) & 9.8508\end{array}\)

 

The output voltage is \( v_{ o }=9.8796 V , \text { and } A_{ v }=9.8508 V /(200 mV -150 mV )=197.02 \).

Step 7. Get a cost estimate.

Two identical transconductance amplifiers for the differential stage: \( G_{ ms }=20 mA / V \pm 0.5 \%, R_{ i1} \geq 398 k \Omega \), and \( R_{ o1} \geq 200 k \Omega \). Estimated cost is $1.50.
One transresistance amplifier for the gain stage: \( Z_{ mo }=10 kV / A \pm 0.5 \%, R_{ i 2} \leq 502 \Omega, \text { and } R_{ o 2} \leq 25.1 \Omega \).
Estimated cost is $1.
Two DC power supplies: \( V_{ CC }=V_{ EE }=12 V \).
