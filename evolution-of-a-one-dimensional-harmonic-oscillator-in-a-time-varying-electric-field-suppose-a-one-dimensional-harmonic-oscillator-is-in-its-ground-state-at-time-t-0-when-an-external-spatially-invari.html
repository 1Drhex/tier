Solution

The 1D harmonic oscillator is described by the Hamiltonian
\(H=-\frac{h2}{2m^{*}}\frac{d^{2}}{dz^{2}}+\frac{1}{2}m^{*}\omega ^{2}z^{2}.\)

In the presence of a spatially uniform but time-varying force F(t), the Hamiltonian becomes
\(H = H_{0} âˆ’ zF(t).\)

We get
\(H=h\omega \left ( a^{\dagger }a+\frac{1}{2} \right )-\left (a^{\dagger }+a \right )\sqrt{h/2m^{*}\omega }F\left ( t \right ).\)

We seek a solution \(\phi(t)\) to the time-dependent Schrodinger equation
\(ih\frac{\partial \phi \left ( t \right )}{\partial t}=H\phi \left ( t \right )=(H_{0}+H')\phi \left ( t \right ),\)

where
\(H_{0}=h\omega \left ( a^{\dagger} a+\frac{1}{2} \right ),\)
\(H'=-\left ( a^{\dagger}+ a \right )\sqrt{\frac{h}{2m^{*}\omega }}F\left ( t \right ).\)

Using the commutation relation
\([a,a^{\dagger }]=1,\)
we get
\([a,H]=h\omega a-\sqrt{\frac{h}{2m^{*}\omega }}F\left ( t \right ),\)
\([a^{\dagger },H]=-h\omega a^{\dagger }-\sqrt{\frac{h}{2m^{*}\omega }}F\left ( t \right )\)

Next, we calculate the time evolution of the average value of the annihilation operator \(A(t)=\left \langle \phi (t)|a|\phi (t) \right \rangle.\) Using Ehrenfestâ€™s theorem , we get
\(\frac{dA}{dt}=\frac{d\left ( a \right )}{dt}=-\frac{i}{h}\left \langle \phi [a,H]\phi \right \rangle=-i\omega A+i\left ( 2m^{*}h\omega \right )^{-\frac{1}{2}}F\left ( t \right ).\) (13.28)

The general solution of this first-order differential equation is
\(A\left ( t \right )=A\left ( -\infty \right )+i\left ( 2m^{*}h\omega \right )^{-\frac{1}{2}}\int_{-\infty }^{t}F\left ( t' \right )e^{-i\omega }\left ( t-t' \right )dt'\)
\(=A\left ( -\infty \right )+ie^{-i\omega t}\lambda \left ( t \right ),\)
where the following function was introduced:
\(\lambda \left ( t \right )=\left ( 2m^{*}h\omega \right )^{-\frac{1}{2}}\int_{-\infty }^{t}F\left ( t' \right )e^{i\omega t'}dt'.\)

The probability for the oscillator to be in its nth eigenstate at time t is
\(p_{n}=|\left \langle n|\phi \left ( t \right ) \right \rangle|^{2}.\) (13.31)

Since for the 1D harmonic oscillator we have
\(|n \rangle=\left ( n! \right )^{-1/2}\left ( a^{\dagger } \right )^{n}|o \rangle,\)

Equation (13.31) can be rewritten as
\(p_{n}=\frac{1}{n!}|\left \langle 0|a^{n}|\phi \left ( t \right ) \right \rangle|^{2}.\) (13.33)

Next, we consider the state
\(\chi \left ( t \right )=\left ( a-A\left ( t \right ) \right )\phi \left ( t \right ).\) (13.34)

Using Equations (13.28) and (13.34), we get

\(ih\frac{d\chi \left ( t \right )}{dt}=\left ( a-A\left ( t \right )\right ) H\phi \left ( t \right )-ih\frac{dA\left ( t \right )}{dt}\phi =|\left ( H+h\omega \right )\chi .\) (13.35)

Since \(H+h\omega \) is Hermitian, \(\left \langle \chi | \chi \right \rangle\) is a constant of motion. At time \(t=-\infty ,\phi =\left (-\infty \right )\) is the ground state of the harmonic oscillator. Therefore,\( A(-\infty )=0\). Hence, \(\chi (-\infty )=0\) From Equation (13.35), we then get \(\frac{d\chi }{dt}(-\infty )=0\). As a result,\( \chi \left ( t \right )=0\) for all time and therefore \(a\phi \left ( t \right ) = A\left ( t \right )\phi \left ( t \right ).\) (13.36)

Using Equations (13.33) and (13.36), we get
\(p_{n}=\frac{|A\left ( t \right )|^{2}}{n!}|\left \langle 0|\phi \left ( t \right ) \right \rangle|^{2}=\frac{|\lambda \left ( t \right )|^{2}}{n!}|\left \langle 0|\left ( t \right ) \right \rangle|^{2}.\)

Finally, we calculate \(\left \langle 0 | \phi \left ( t \right ) \right \rangle\)

\(ih\frac{d}{dt}\left \langle 0 | \phi \left ( t \right ) \right \rangle = \left \langle 0 | H |\phi \left ( t \right ) \right \rangle\) \(\frac{1}{2}h\omega \left \langle 0 | \phi \left ( t \right ) \right \rangle-\sqrt{\frac{h}{2m^{\ast }}}F\left ( t \right )\left \langle 0 |\left ( a^{\dagger }+ a \right ) | \phi \left ( t \right ) \right \rangle\)

This yields

\(\frac{d}{dt} \left \langle 0 | \phi \left ( t \right ) \right \rangle = \left [ -\frac{i}{2}\omega +\left ( 2m^{\ast h\omega } \right )^{-1/2} F\left ( t \right )A\left ( t \right )\right ]\)

Integrating from\( -\infty\) to t, and using the fact that \(\phi (-\infty ) = |0\rangle,\) we get

\(\left \langle 0 | \phi \left ( t \right ) \right \rangle =\exp \left ( -\frac{i}{2}\omega t+i\left ( 2m^{\ast }h\omega \right )^{-1/2}\int_{-\infty }^{+\infty }F\left ( t' \right )A\left ( t' \right )dt' \right )\)

Furthermore, since\( A(-\infty ) = 0\), we get

\(A\left ( t \right ) = ie^{-i\omega t}\lambda \left ( t \right )\)

Therefore,

\(P_{n} =\frac{\left | \lambda \left ( t \right ) \right |^{2}}{n!} e^{-\left | \lambda \left ( t \right ) \right |^{2}}\)
