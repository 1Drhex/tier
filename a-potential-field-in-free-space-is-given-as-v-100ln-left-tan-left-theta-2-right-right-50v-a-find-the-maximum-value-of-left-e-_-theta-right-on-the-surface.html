Guest Question.

A potential field in free space is given as V = 100\(\ln { \left[ \tan { \left( \theta /2 \right) } \right] } \)+50V. (a) Find the maximum value of \(\left| { E }_{ \theta } \right|\) on the surface \(\theta =\mathring { 40 }\) for 0.1 < r <0.8m,\(\mathring { 60 } <\phi <\mathring { 90 }\). (b) Describe the surface V = 80V.

Admin
Solution

a) First
\(E=-\frac { 1 }{ r } \frac { dV }{ d\theta } { a }_{ \theta }=-\frac { 100 }{ 2r\tan { (\theta /2)\cos ^{ 2 }{ (\theta /2) } } } { a }_{ \theta }=-\frac { 100 }{ 2r\sin { (\theta /2) } \cos { (\theta /2) } } { a }_{ \theta }=-\frac { 100 }{ r\sin { (\theta ) } } { a }_{ \theta }\)
This will maximize at the smallest value of r, or 0.1:
\({ E }_{ max }(\theta =\mathring { 40 } )=E(r=0.1,\theta =\mathring { 40 } )=-\frac { 100 }{ 0.1\sin { (40) } } { a }_{ \theta }=1.56{ a }_{ \theta }\)kV/m
b) Set 100 \(\ln { \tan { (\theta /2) } }\) +50 = 80 and solve for \(\theta\): obtain \(\ln { \tan { (\theta /2) } } = 0.3 \Longrightarrow \tan { \theta /2 } ={ e }^{ 3 }=1.35\Longrightarrow \theta =\mathring { 107 }\) ( the cone surface at \(\theta\) = 107 degrees).
