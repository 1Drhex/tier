Guest Question.

Figure 6.8 shows a pump and piping system that is to convey 600 gpm of propylene glycol from a tank to atmospheric conditions. Follow the suggested design procedure and make recommendations about the piping system.

Admin
Solution

We follow the design procedure in a step-by-step fashion.
1.Economic line size Table 6.4 lists the economic velocity range for propylene glycol as
\(4.5 ≤ V_{opt} ≤ 9.0\) ft/s
Values in the table were prepared using current and projected economic parameters to determine economic diameters. The calculated diameters were divided into flow rate to determine economic velocity. We can use any value inside the range, but the velocity we use will depends on what pipe sizes are appropriate for the job. We will make initial calculations for both 4.5 and 9.0 ft/s to determine the sizes available. For a volume flow rate of 600 gpm (= 1.34 \(ft^3/s\)), we calculate two flow areas:
\(A_{ \begin{matrix} upper \\ limit \end{matrix}}=\frac{Q}{V} =\frac{ 1.34}{4.5} = 0.298 ft^2\)

\(A_{ \begin{matrix} lower \\ limit \end{matrix}}= \frac{1.34}{9} = 0.150 ft^2\)
We now refer to a table of pipe sizes. The only standard size that falls within this range of flow areas is 6-nominal schedule 40, which is the size we use for the remainder of the calculations.

Economic line size = 6-nom sch 40 .

2.Pump power

In order to calculate pumping power, we must first obtain properties, and eventually substitute into the modified Bernoulli equation. We proceed in the usual way:
propylene glycol \(ρ = 0.968(1.94)\) slug/\(ft^3\) \(µ = 88 \times 10^{-5} lbf·s/ft^2\)
[Appendix Table B.1]
6-nominal schedule 40 \(ID = 0.5054 ft\quad A = 0.2006 ft^2\)
[Appendix Table D.1]
galvanized surface ε = 0.0005 ft (the average value)
[Table 3.1]
The modified Bernoulli equation with pump power is written as

\(\frac{p_1g_c}{ρg} +\frac{V^2_1}{2g} + z_1 = \frac{p_2g_c}{ρg} +\frac{V^2_2}{2g} + z_2 + ∑\frac{f L}{D_h}\frac{V^2}{2g}+∑K \frac{V^2}{2g}+\frac{g_c}{\dot{m}g}\frac{dW}{dt}\)

We define section 1 as the free surface of the propylene glycol in the tank and section 2 as the outlet of the pipe labeled “12.” Recall the convention that if we take the pressure at section 2 to be atmospheric and the velocity there to be zero, then we must include an exit loss in the minor-loss calculation. Evaluating properties, we have
\(p_1 =p_2 = p_{atm}\) \(V_1 = V_2 = 0\)
The velocity in the pipeline is
\(V=\frac{Q}{A} = \frac{1.34}{0.2006} = 6.68\) ft/s
Making height measurements from length #5, we write
\( z_1 = 3'+ 20'6" + 30'1" = 53.6\) ft
where the 3′ was added to account for the depth of liquid in the tank. We also have
\(z_2 = 40'3" + 10'6" sin 45° = 47.7\) ft
Also,
L= 1′ + 10’0″ + 30’1″ + 4’2″ + 4’2″ + 20’6″ + 62’4″ + 40’3″ + 10’6″
+ 40’9″ + 30’0″
or =L 253.8 ft
The minor-loss values from Table 3.3 include
\(ΣK =K_{inlet}+5K_{ \begin{matrix}long radius\\ 90 ells \end{matrix}} + K_{\begin{matrix}globe\\ valve\end{matrix}} +2K _{45°ells}+K_{ \begin{matrix}branch\\ T-joint\end{matrix}}+K_{exit}\)
and ΣK = 0.5 + 5(0.22) + 10 + 2(0.17) + 0.69 + 1.0
where we have assumed a square-edged inlet, because this is typically what is installed in a welded system. We have also assumed that the inlet to the pump from the tank is the same diameter as the outlet pipe. The minor losses then are
Σ K= 13.63
The Reynolds number is calculated as

\(Re=\frac{ρVD}{μg_c}=\frac{0.968(1.94)(6.68)(0.5054)}{88 \times 10^{-5}}\)

and

\(\left. \begin{matrix}Re = 7.21 \times 10^3 \\ \\ and\quad ε/D= \frac{0.0005}{0.5054} = 0.00099 \end{matrix} \right\}\) f= 0.036 (Moody diagram)
The modified Bernoulli equation becomes

\(z_1=z_2+\frac{fL}{D_h}\frac{V^2}{2g} +∑K\frac{V^2}{2g} + \frac{g_c}{\dot{m}g} \frac{dW}{dt}\)

From the pump power equation,

\(– \frac{g_c}{\dot{m}g}\frac{dW}{dt}= ∆H\)

Substituting into the modified Bernoulli equation, we get
\(∆H = z_2 – z_1 + \left\lgroup\frac{fL}{D_h} + ΣK\right\rgroup \frac{V^2}{2g}\) (i)
Substituting,
\(∆ H= (47.7 – 53.6) + \left[\frac{ 0.036(253.8)}{0.5054 }+ 13.63\right] \frac{6.68^2}{2(32.2)}\)
∆ H= –5.9 + 21.97
or ∆H = 16.0 ft @Q = 600 gpm

3.System Curve
The system curve in this case is a graph of power (∆H ) versus flow rate ( Q). We generate data for this curve by using Equation i above, written in terms of friction factor and velocity:
\(∆H = (47.7 – 53.6) + \left[\frac{f(253.8)}{0.5054 }+ 13.63\right] \frac {V^2}{2(32.2)} \)

or \( ∆H = –5.9 + (f7.8 + 0.212) V^2\)
