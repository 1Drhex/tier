Guest Question.

Let \(C[0,2 \pi]\) be the vector space of continuous functions on the interval \([0,2 \pi]\) and the inner product on this vector space be defined by

\(\left\langle\mathbf{f}_{n}, \mathbf{g}_{n}\right\rangle=\int_{0}^{2 \pi} f_{n}(t) g_{n}(t) \mathrm{d} t\)

Let \(S\) be the set \(\{1, \cos (t), \cos (2 t), \cos (3 t), \ldots, \sin (t), \sin (2 t), \sin (3 t), \ldots\}\). By letting \(\mathbf{f}_{0}=\cos (0)=1, \mathbf{f}_{n}=\cos (n t)\) and \(\mathbf{g}_{n}=\sin (n t)\) show that the set \(S\) is orthogonal and normalize these vectors to create an orthonormal set.

[Results: for any positive integers \(n\) and \(m\) we have

(i) \(\int_{0}^{2 \pi} \cos (n t) \sin (m t) \mathrm{d} t=0\),

(ii) \(\int_{0}^{2 \pi} \cos (n t) \mathrm{d} t=0\)

(iii) \(\int_{0}^{2 \pi} \sin (n t) \mathrm{d} t=0\)

(iv) \(\int_{0}^{2 \pi} \cos (n t) \cos (m t) \mathrm{d} t=0\) provided \(n \neq m\)

(v) \(\int_{0}^{2 \pi} \sin (n t) \sin (m t) \mathrm{d} t=0\) provided \(n \neq m\) and

(vi) \(\left.\int_{0}^{2 \pi} \cos ^{2}(n t) \mathrm{d} t=\int_{0}^{2 \pi} \sin ^{2}(n t) \mathrm{d} t=\pi .\right]\)

Admin
Solution

How do we show that the given set of vectors in \(S\) are orthogonal?

By using the integration results in the brackets above. First we test \(\mathrm{f}_{0}\) with the other vectors:

\(\begin{aligned}&\left\langle\mathbf{f}_{0}, \mathbf{f}_{n}\right\rangle=\int_{0}^{2 \pi}[1 \times \cos (n t)] \mathrm{d} t=\int_{0}^{2 \pi} \cos (n t) \mathrm{d} t \underbrace{=}_{\text { by result (ii) }} 0 \\&\left\langle\mathbf{f}_{0}, \mathbf{g}_{n}\right\rangle=\int_{0}^{2 \pi}[1 \times \sin (n t)] \mathrm{d} t=\int_{0}^{2 \pi} \sin (n t) \mathrm{d} t \underbrace{=}_{\text { by result (iii) }} 0\end{aligned}\)

Next we test \(\mathbf{f}_{n}\) with \(\mathrm{g}_{m}\) :

\(\left\langle\mathbf{f}_{n}, \mathbf{g}_{m}\right\rangle=\int_{0}^{2 \pi} \cos (n t) \sin (m t) \mathrm{d} t \underbrace{=}_{\text { by result (i) }} 0\)

Lastly we test \(\mathbf{f}_{n}\) with \(\mathbf{f}_{m}\) and \(\mathrm{g}_{n}\) with \(\mathrm{g}_{m}\) where \(m \neq n\) :

\(\begin{aligned}&\left\langle\mathbf{f}_{n}, \mathbf{f}_{m}\right\rangle=\int_{0}^{2 \pi} \cos (n t) \cos (m t) \mathrm{d} t \underbrace{=}_{\text { by result (iv) }} 0 \text { provided } m \neq n \\&\left\langle\mathbf{g}_{n}, \mathbf{g}_{m}\right\rangle=\int_{0}^{2 \pi} \sin (n t) \sin (m t) \mathrm{d} t \underbrace{=}_{\text { by result }(\mathrm{v})} 0 \text { provided } m \neq n\end{aligned}\)

Hence the inner product of all distinct vectors is zero, which means that the set \(\{1, \cos (t), \cos (2 t), \cos (3 t), \ldots, \sin (t), \sin (2 t), \sin (3 t), \ldots\}\) is orthogonal.

Normalizing:

\(\begin{aligned}&\text { First we determine }\left\|\mathbf{f}_{0}\right\|^{2},\left\|\mathbf{f}_{n}\right\|^{2} \text { and }\left\|\mathbf{g}_{n}\right\|^{2} \text { and then we take the square root: }\\&\left\|\mathrm{f}_{0}\right\|^{2}=\left\langle\mathbf{f}_{0}, \mathbf{f}_{0}\right\rangle\\&=\int_{0}^{2 \pi}(1 \times 1) \mathrm{d} t\\&\underbrace{=}_{\text { Ategrating }}[t]_{0}^{2 \pi}=2 \pi-0=2 \pi\\&\left\|\mathbf{f}_{n}\right\|^{2}=\left\langle\mathbf{f}_{n}, \mathbf{f}_{n}\right\rangle\\&=\int_{0}^{2 \pi} \cos (n t) \times \cos (n t) \mathrm{d} t=\int_{0}^{2 \pi} \cos ^{2}(n t) \mathrm{d} t \underbrace{=}_{\text { by result }(\mathrm{vi})} \pi\\&\left\|\mathrm{g}_{n}\right\|^{2}=\left\langle\mathrm{g}_{n}, \mathrm{~g}_{n}\right\rangle\\&=\int_{0}^{2 \pi} \sin (n t) \times \sin (n t) \mathrm{d} t=\int_{0}^{2 \pi} \sin ^{2}(n t) \mathrm{d} t \underbrace{=}_{\text { by result }(\mathrm{vi})} \pi\end{aligned}\)

Taking the square root of these results \(\left\|\mathbf{f}_{0}\right\|^{2}=2 \pi,\left\|\mathbf{f}_{n}\right\|^{2}=\pi\) and \(\left\|\mathrm{g}_{n}\right\|^{2}=\pi\) gives \(\left\|\mathbf{f}_{0}\right\|=\sqrt{2 \pi}\) \(\left\|\mathbf{f}_{n}\right\|=\sqrt{\pi}\) and \(\left\|\mathbf{g}_{n}\right\|=\sqrt{\pi}\) respectively.

How can we normalize these (convert into unit vectors) \(\mathrm{f}_{0}, \mathbf{f}_{n}\) and \(\mathrm{g}_{n}\) ?

By using Proposition (4.11) \(\mathbf{u}=\frac{\mathbf{w}}{\|\mathbf{w}\|}\) which means that we divide each vector by its norm. For all \(n \geq 1\) we have

\(\frac{\mathrm{f}_{0}}{\left\|\mathbf{f}_{0}\right\|}=\frac{1}{\sqrt{2 \pi}}, \quad \frac{\mathbf{f}_{n}}{\left\|\mathbf{f}_{n}\right\|}=\frac{\cos (n t)}{\sqrt{\pi}} \text { and } \frac{\mathrm{g}_{n}}{\left\|\mathrm{~g}_{n}\right\|}=\frac{\sin (n t)}{\sqrt{\pi}}\)

Hence the set \(\left\{\frac{1}{\sqrt{2 \pi}}, \frac{\cos (t)}{\sqrt{\pi}}, \frac{\cos (2 t)}{\sqrt{\pi}}, \ldots, \frac{\sin (t)}{\sqrt{\pi}}, \frac{\sin (2 t)}{\sqrt{\pi}}, \ldots\right\}\) is an orthonormal set of functions.

Proposition (4.11). Every non-zero vector w in an inner product space \(V\) can be normalized by setting \(\mathbf{u}=\frac{\mathrm{w}}{\|\mathbf{w}\|}\).
