Guest Question.

Given \(\Gamma_{T}=0.3 \angle 45^{\circ}\), what will be the reflection coefficient at the input end of a
transmission line 0.3 λ long with a one-way attenuation of 2 dB? If the characteristic impedance of the line is 50 ohms, what termination impedance caused the reflection coefficient and what is the input impedance of the line?

Admin
Solution

Solution: We need to evaluate \(\Gamma_{T}=0.3 \angle 45^{\circ} . \text { Since } e^{-\alpha L}\) tells how much a voltage
wave is attenuated in traveling one way on the line, we must say

 

\(-2 dB =20 \log _{10}\left(e^{-\alpha L}\right)=10 \log _{10}\left(e^{-2 \alpha L}\right) ; e^{-2 \alpha L}=10^{-0.2}=0.631\)

 

A more convenient way of expressing \(\beta \text { is } 2 \pi / \lambda\). Then

 

\(2 \beta L=2\left(\frac{2 \pi}{\lambda}\right) 0.3 \lambda=1.2 \pi=216^{\circ}\) \(\Gamma(L=0.3 \lambda)=0.3 \times 0.631 \angle\left(45^{\circ}-216^{\circ}\right)=0.189 \angle-171^{\circ}\)

 

The expression for Z(d) may be evaluated for any value of d, including zero. Hence,

 

\(Z_{t}=Z_{0} \frac{1+\Gamma_{T}}{1-\Gamma_{T}}=50 \frac{1+0.3 \angle 45^{\circ}}{1-0.3 \angle 45^{\circ}}=50 \frac{1+0.2121+j 0.2121}{1-0.2121-j 0.2121}\)

 

\(=50 \frac{1.2306 \angle 0.93^{\circ}}{0.8159 \angle-15.069^{\circ}}=75.41 \angle 25.00^{\circ}=(67.1+j 31.3) \Omega .\)

 

Similarly,

 

\(Z_{T}=Z_{0-} \frac{!+\Gamma(d)}{1-\Gamma(d)}=50 \frac{1+0.189 \angle-171^{\circ}}{1-0.189 \angle-171^{\circ}}=50 \frac{1-0.1870-j 0.0296}{1+0.01870+j 0.0296}\)

 

\(=34.26 \angle-3.51^{\circ}=34.20-j 2.10\)
