Guest Question.

Given H = \(\left( 3{ r }^{ 2 }/\sin { \theta } \right) { a }_{ \theta }+54r\cos { \theta } { a }_{ \phi }\) A/m in free space: (a) find the total current in the \({ A }_{ \theta }\) direction through the conical surface \(\theta = \mathring { 20 }, 0\le \phi \le 2\pi, 0 \le r \le 5\), by whichever side of Stokes’ theorem you like the best. (b) check the result by using the other side of Stokes’ theorem.

Admin
Solution

a) I chose the line integral side, where the integration path is the circular path in \(\phi \) around the top edge of the cone, at r = 5. The path direction is chosen to be clockwise looking down on the xy plane. This, by convention, leads to the normal from the cone surface that points in the positive \({ a }_{ \theta }\) direction (right hand rule). We ﬁnd
\(\oint { } H.dL=\int _{ 0 }^{ 2\pi }{ } { \left[ \left( 3{ r }^{ 2 }/\sin { \theta } \right) { a }_{ \theta }+54r\cos { \theta { a }_{ \phi } } \right] }_{ r=5,\theta =20 }\quad .5\sin { \left( \mathring { 20 } \right) } d\phi \left( -{ a }_{ \phi } \right) \\ =-2\pi \left( 54 \right) \left( 25 \right) \cos { \left( \mathring { 20 } \right) } \sin { \left( \mathring { 20 } \right) } =-2.73\times { 10 }^{ 3 }A\)

This result means that there is a component of current that enters the cone surface in the −\({ a }_{ \theta }\) direction, to which is associated a component of H in the positive \({ a }_{ \phi }\) direction.
b) We ﬁrst ﬁnd the current density through the curl of the magnetic ﬁeld, where three of the six terms in the spherical coordinate formula survive:
\(\triangledown \times H=\frac { 1 }{ r\sin { \theta } } \frac { \partial }{ \partial \theta } \left( 54r\cos { \theta } \sin { \theta } \right) { a }_{ r }-\frac { 1 }{ r } \frac { \partial }{ \partial r } \left( 54{ r }^{ 2 }\cos { \theta } \right) { a }_{ \theta }+\frac { 1 }{ r } \frac { \partial }{ \partial r } \left( \frac { 3{ r }^{ 3 } }{ \sin { \theta } } \right) { a }_{ \phi }=J\)
Thus
\(J=54\cot { \theta } { a }_{ r }-108\cos { \theta } { a }_{ \theta }+\frac { 9r }{ \sin { \theta } } { a }_{ \phi }\)

The calculation of the other side of Stokes’ theorem now involves integrating J over the surface of the cone, where the outward normal is positive \({ a }_{ \theta }\) , as deﬁned in part a:
\(\int _{ S }^{ }{ } \left( \triangledown \times H \right) .dS=\int _{ 0 }^{ 2\pi }{ } \int _{ 0 }^{ 5 }{ } { \left[ 54\cot { \theta } { a }_{ r }-108\cos { \theta } { a }_{ \theta }+\frac { 9r }{ \sin { \theta } } { a }_{ \phi } \right] }_{ \theta =\mathring { 20 } }\quad .{ a }_{ \theta }r\sin { \left( \mathring { 20 } \right) } drd\phi \\ =-\int _{ 0 }^{ 2\pi }{ } \int _{ 0 }^{ 5 }{ } 108\cos { \left( \mathring { 20 } \right) } \sin { \left( \mathring { 20 } \right) } rdrd\phi =-2\pi \left( 54 \right) \left( 25 \right) \cos { \left( \mathring { 20 } \right) } \sin { \left( \mathring { 20 } \right) } \\ =-2.73\times { 10 }^{ 3 }A\)
