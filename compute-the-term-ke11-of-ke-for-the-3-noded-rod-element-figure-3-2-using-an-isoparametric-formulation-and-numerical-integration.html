Guest Question.

Compute the term \(K_{11}^{(e)} \text { of } K ^{(e)}\) for the 3-noded rod element (Figure 3.2) using an isoparametric formulation and numerical integration.

Admin
Solution

The term \(K_{11}^{(e)}\) is obtained from Eq.(3.51) as

 

\(K_{i j}^{(e)}=\int_{-1}^{+1} \frac{1}{J^{(e)}} \frac{d N_{i}}{d \xi}(E A) \frac{d N_{j}}{d \xi} d \xi\) (3.51)

 

\(K_{11}^{(e)}=\int_{-1}^{+1} \frac{1}{J^{(e)}} \frac{d N_{1}}{d \xi}(E A) \frac{d N_{1}}{d \xi} d \xi\)

 

The expression of \(N_{1}\) for the 3-noded rod element is (Eq. (3:11))

 

\(\begin{aligned}&N_{1}=\frac{\left(\xi-\xi_{2}\right)\left(\xi-\xi_{3}\right)}{\left(\xi_{1}-\xi_{2}\right)}=\frac{1}{2} \xi(\xi-1) \\&N_{2}=\frac{\left(\xi-\xi_{1}\right)\left(\xi-\xi_{3}\right)}{\left(\xi_{2}-\xi_{1}\right)\left(\xi_{2}-\xi_{3}\right)}=\left(1-\xi^{2}\right) \\&N_{3}=\frac{\left(\xi-\xi_{1}\right)\left(\xi-\xi_{2}\right)}{\left(\xi_{3}-\xi_{1}\right)\left(\xi_{3}-\xi_{2}\right)}=\frac{1}{2} \xi(1+\xi)\end{aligned}\) (3.11)

 

\(N_{1}=\frac{1}{2}(\xi-1) \xi \quad y \quad \frac{d N_{1}}{d \xi}=\xi-\frac{1}{2}\)

 

Assuming that node 2 is centered in the element \(J^{(e)}=\frac{l^{(e)}}{2}\).

 

Substituting \(\frac{d N_{1}}{d \xi} \text { and } J^{(e)} \text { into } K_{11}^{(e)}\) we have

 

\(K_{11}^{(e)}=\int_{-1}^{+1} \frac{2 E A}{l^{(e)}}\left(\xi-\frac{1}{2}\right)^{2} d \xi\)

 

The integrand is a quadratic function and hence, the exact integral requires a Gauss quadrature of \(2^{ nd }\) order (q = 2). From Eq.(3:52) and Table 3.1 we obtain

 

\(\begin{gathered}K ^{(e)}=\sum_{r=1}^{q}\left[ B ^{T} D B J^{(e)}\right]_{r} W_{r}\quad (3.52a) \\K_{i j}^{(e)}=\sum_{r=1}^{q}\left[\frac{1}{J^{(e)}} \frac{d N_{i}}{d \xi}(E A) \frac{d N_{j}}{d \xi}\right]_{r} W_{r}\quad(3.52b)\end{gathered}\)

 

\(K_{11}^{(e)}=\sum_{r=1}^{2}\left[\frac{2 E A}{l^{(e)}}\left(\xi-\frac{1}{2}\right)^{2}\right]_{r} W_{r}=\left[\frac{2 E A}{l^{(e)}}\left(\xi-\frac{1}{2}\right)^{2}\right]_{\xi=-\frac{\sqrt{3}}{3}}+\)

 

\(+\left[\frac{2 E A}{l^{(e)}}\left(\xi-\frac{1}{2}\right)^{2}\right]_{\xi=\frac{\sqrt{3}}{3}}=\frac{7}{3}\left(\frac{E A}{l}\right)^{(e)}\)

 

The same procedure can be followed for computing the rest of terms of \(K ^{(e)}\).
