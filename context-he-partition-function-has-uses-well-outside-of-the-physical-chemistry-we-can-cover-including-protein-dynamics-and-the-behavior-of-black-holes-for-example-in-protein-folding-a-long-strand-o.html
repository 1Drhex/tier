context he partition function has uses well outside of the physical chemistry we can cover, including protein dynamics and the behavior of black holes. In these wide-ranging examples, the partition function can be said to accomplish one task—to obtain an efective count of the states available to the system, weighing in the likelihood of each possible state. he following example uses a simple model to show that—with an extreme energy distribution—the partition function does exactly that.

ProBLeM You discover a molecular system having the energy levels and degeneracies

\(\varepsilon = c(n-1)^{6}\); g = n; for n = 1,2,3,… and \(K_{B}T = 400 c \).

Evaluate the partition function, and calculate P(ε) for each of the four lowest energy levels.

Admin
Solution

The four lowest values of the energy are for n = 1 through n = 4, which yield values of 0, c, 64c, and 729c. he partition function (Eq. 2.33)is

\(q(T) = \sum\limits_{\varepsilon =o}^{\infty }{g(\varepsilon)e^{-\varepsilon/(K_{B}T)} }\) (2.33)

\(q(T)= e^{-0/(400c)}+ 2 e^{-c/(400c)}+ 3 e^{-64c/(400c)}+ 4e^{-729c/(400c)}+5 e^{-4096c/(400c)}+ …\)

 

\(= 1.000 + 1.995 + 2.556 + 0.646 + 2\cdot 10^{-4}= 6.20\)

and the probabilities (Eq. 2.32) are

\(P(\varepsilon)= \frac{g(\varepsilon)e^{-\varepsilon/(K_{B}T)} }{q(T)}\) (2.32)

P(n = 1) =\frac{1}{6.20} = 0.16 P(n = 2) = \frac{1.995}{6.20} = 0.32
P(n = 3) = \frac{2.556}{6.20} = 0.41 P(n = 4) =\frac{0.646}{6.20} = 0.10,
suggesting that the molecules are divided up among roughly the six lowest energy states, corresponding to n = 1 (g = 1) with 16% probability, n = 2 (g = 2) with 16%, and n = 3 (g = 3) with 25%. As the n = 4 level is occupied with only 2.5%, signiicantly lower than the others, it contributes little to the partition function. So the value of the partition function, 6.20, tells us that there are roughly six states with signiicant probability.
