Guest Question.

Consider a simply supported steel shaft as depicted in Fig. 7–14, with 1 in diameter and a 31-in span between bearings, carrying two gears weighing 35 and 55 lbf.
(a) Find the influence coefficients.
(b) Find \(\sum{wy} \ and \sum{wy^2} \) and the first critical speed using Rayleigh’s equation,Eq. (7–23).
(c) From the influence coefficients, find \(\omega _{11} \ and \omega _{22}\).
(d) Using Dunkerley’s equation, Eq. (7–32), estimate the first critical speed.
(e) Use superposition to estimate the first critical speed.
(f) Estimate the shaft’s intrinsic critical speed. Suggest a modification to Dunkerley’s equation to include the effect of the shaft’s mass on the first critical speed of the attachments.

 

Eq. (7–32)

\(w_{1 c}=w_{1} \frac{\delta_{11}}{\delta_{c c}}\)

Eq. (7–23).

\(\omega_{1}=\sqrt{\frac{g \sum w_{i} y_{i}}{\sum w_{i} y_{i}^{2}}}\)

Solution

(a) \(I=\frac{\pi d^4}{64} =\frac{\pi \left(1\right)^4 }{64} =0.04909 \ in^4\\6EIl=6\left(30\right)10^6 \left(0.04909\right) 31=0.2739\left(10^9\right) \ lbf.in^3\)

From Eq. set (7–24),

\(\delta _{ij}=\begin{cases}\frac{b_jx_i}{6EIl}\left(l^2-b^2_j-x^2_i\right) & \ x_i\leq a_i \\ \frac{a_j\left(l-x_i\right) }{6EIl}\left(2lx_i-a^2_j-x^2_i\right) & x_i\gt a_i \end{cases} \)

 

\(\delta _{11}=\frac{24\left(7\right)\left(31^2-24^2-7^2\right) }{0.2739\left(10^9\right) } =2.061\left(10^{-4}\right) \ in/lbf \\ \delta _{22}=\frac{11\left(20\right)\left(31^2-11^2-20^2\right) }{0.2739\left(10^9\right) } =3.534\left(10^{-4}\right) \ in/lbf \\ \delta _{12}= \delta _{12}=\frac{11\left(7\right)\left(31^2-11^2-7^2\right) }{0.2739\left(10^9\right) } =2.224\left(10^{-4}\right) \ in/lbf\)

\(y_1=w_1\delta _{11}+w_2\delta _{12}=35\left(2.061\right) 10^{-4}+55\left(2.224\right) 10^{-4}=0.01945 \ in \\ y_2=w_1\delta _{21}+w_2\delta _{22}=35\left(2.224\right) 10^{-4}+55\left(3.534\right) 10^{-4}=0.02722 \ in\)

(b) \(\sum{w_iy_i} =35\left(0.01945\right) +55\left(0.02722\right) =2.178 \ lbf.in \\ \sum{w_iy^2_i} =35\left(0.01945\right) ^2+55\left(0.02722\right) ^2=0.05399 \ lbf.in \\ \omega =\sqrt{\frac{386.1\left(2.178\right) }{0.05399} } =124.8 \ rad/s, \ or \ 1192 \ rev/min\)

(c) \(\frac{1}{\omega ^2_{11}} =\frac{w_1}{g} \delta _{11} \\ \omega _{11}=\sqrt{\frac{g}{w_1\delta _{11}} } =\sqrt{\frac{386.1}{35\left(2.061\right)10^{-4} } } =231.4 \ rad/s, \ or \ 2210 \ rev/min \\ \omega _{22}=\sqrt{\frac{g}{w_2}\delta _{22} } =\sqrt{\frac{3861.1}{55\left(3.534\right)10^{-4} } } =140.9 \ rad/s, \ or \ 1346 \ rev/min\)

 

(d) \(\frac{1}{\omega^2_1} \doteq \sum{\frac{1}{\omega _{ii}} } =\frac{1}{131.4^2} +\frac{1}{140.6^2} =6.905\left(10^{-5}\right) \\ \omega _1\doteq \sqrt{\frac{1}{6.905}\left(10^{-5}\right) } =120.3 \ rad/s, \ or \ 1149 \ rev/min\)

which is less than part b, as expected.

(e) From Eq. (7–24),

\(\delta _{cc}=\frac{b_{cc}x_{cc}\left(l^2-b^2_{cc}-x^2_{cc}\right) }{6EIl} =\frac{15.5\left(15.5\right)\left(31^2-15.5^2\right) }{0.2739\left(10^9\right) } \\ =4.215\left(10^{-4}\right) \ in/lbf \\ \)

From Eq. (7–33),

\(w_{1c}=w_1\frac{\delta _{11}}{\delta _{cc}} \) \(w_{1c}=w_1\frac{\delta _{11}}{\delta _{cc}} =35\frac{2.06\left(10^{-4}\right) }{4.21510^{-4}} =17.11 \ lbf \\ w_{2c}=w_2\frac{\delta _{22}}{\delta _{cc}} =55\frac{3.534\left(10^{-4}\right) }{4.215\left(10^{-4}\right) } =64.11 \ lbf\) \(\omega =\sqrt{\frac{g}{\delta _{cc}\sum{w_{ic}} } } =\sqrt{\frac{386.1}{4.215\left(10^{-4}\right)\left(17.11+46.11\right) } } =120.4 \ rad/s, \ or \ 1150 \ rev/min\)

which, except for rounding, agrees with part d, as expected.
(f) For the shaft, \( E = 30\left(10^6\right) \ psi, \gamma = 0.282 \ lbf/in^3 \ , \ and \ A = \pi {(1^2)}/{4} = 0.7854 \ in^2\).
Considering the shaft alone, the critical speed, from Eq. (7–22)

\(\omega _1=\left(\frac{\pi }{l} \right)^2\sqrt{\frac{EI}{m} } =\left(\frac{\pi }{l} \right)^2\sqrt{\frac{gEI}{A\gamma } }\)

, is

\(\omega _s=\left(\frac{\pi }{l} \right)^2\sqrt{\frac{gEI}{A\gamma } }=\left(\frac{\pi }{31} \right) \sqrt{\frac{386.1\left(30\right)10^6\left(0.04909\right) }{0.7854\left(0.282\right) } } = 520.4 \ rad/s, \ or \ 4970 \ rev/min\)

We can simply add \({1}/{\omega ^2_s}\) to the right side of Dunkerley’s equation, Eq. (1), to include the shaft’s contribution,

\(\frac{1}{\omega ^2_1}\doteq \frac{1}{520.4^2} +6.905\left(10^{-5}\right) =7.274\left(10^{-5}\right) \\ \omega _1\doteq 117.3 \ rad/s, \ 1120 \ rev/min\)

which is slightly less than part d, as expected.
The shaft’s first critical speed ωsis just one more single effect to add to Dunkerley’s equation. Since it does not fit into the summation, it is usually written up front.

\(\frac{1}{\omega ^2_1} \doteq \frac{1}{\omega ^2_s} =\sum\limits_{i=1}^{n}{\frac{1}{\omega ^2_{ii}} } \)

Common shafts are complicated by the stepped-cylinder geometry, which makes the influence-coefficient determination part of a numerical solution.
