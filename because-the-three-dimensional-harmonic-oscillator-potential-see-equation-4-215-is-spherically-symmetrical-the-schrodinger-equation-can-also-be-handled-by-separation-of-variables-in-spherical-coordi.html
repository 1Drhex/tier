Guest Question.

Because the three-dimensional harmonic oscillator potential (see Equation 4.215) is spherically symmetrical, the Schrödinger equation can also be handled by separation of variables in spherical coordinates. Use the power series method (as in Sections 2.3.2 and 4.2.1) to solve the radial equation. Find the recursion formula for the coefficients, and determine the allowed energies. (Check that your answer is consistent with Equation 4.216.) How is N related to n in this case? Draw the diagram analogous to Figures 4.3 and 4.6, and determine the degeneracy of nth energy level.

\( V(r)=\frac{1}{2} m \omega^{2} r^{2} \) (4.215).

\( E_{n}=\left(n+\frac{3}{2}\right) \hbar \omega \) (4.216).

Admin
Solution

Eq. 4.37: \( -\frac{\hbar^{2}}{2 m} \frac{d^{2} u}{d r^{2}}+\left[V+\frac{\hbar^{2}}{2 m} \frac{\ell(\ell+1)}{r^{2}}\right] u=E u \) (4.37).

\( -\frac{\hbar^{2}}{2 m} \frac{d^{2} u}{d r^{2}}+\left[\frac{1}{2} m \omega^{2} r^{2}+\frac{\hbar^{2}}{2 m} \frac{\ell(\ell+1)}{r^{2}}\right] u=E u \).

Following Eq. 2.72, let

\( \frac{d^{2} \psi}{d \xi^{2}}=\left(\xi^{2}-K\right) \psi \) (2.72).

\( \xi \equiv \sqrt{\frac{m \omega}{\hbar}} r . \text { Then }-\frac{\hbar^{2}}{2 m} \frac{m \omega}{\hbar} \frac{d^{2} u}{d \xi^{2}}+\left[\frac{1}{2} m \omega^{2} \frac{\hbar}{m \omega} \xi^{2}+\frac{\hbar^{2}}{2 m} \frac{m \omega}{\hbar} \frac{\ell(\ell+1)}{\xi^{2}}\right] u=E u \).

\( \text { or } \frac{d^{2} u}{d \xi^{2}}=\left[\xi^{2}+\frac{\ell(\ell+1)}{\xi^{2}}-K\right] u, \text { where } K \equiv \frac{2 E}{\hbar \omega} \), (as in Eq. 2.74):

\( K \equiv \frac{2 E}{\hbar \omega} \) (2.74).

At large \( \xi, \frac{d^{2} u}{d \xi^{2}} \approx \xi^{2} u, \text { and } u \sim() e^{-\xi^{2} / 2} \). (see Eq. 2.78):

\( \psi(\xi)=h(\xi) e^{-\xi^{2} / 2} \) (2.78).

At small \( \xi, \frac{d^{2} u}{d \xi^{2}} \approx \frac{\ell(\ell+1)}{\xi^{2}} u, \text { and } u \sim() \xi^{\ell+1} \). (see Eq. 4.59).

\( u(\rho) \sim C \rho^{\ell+1} \) (4.59).

So let \( u(\xi) \equiv \xi^{\ell+1} e^{-\xi^{2} / 2} v(\xi) \). [This defines the new function v(ξ).]

\( \frac{d u}{d \xi}=(\ell+1) \xi^{\ell} e^{-\xi^{2} / 2} v-\xi^{\ell+2} e^{-\xi^{2} / 2} v+\xi^{\ell+1} e^{-\xi^{2} / 2} v^{\prime} \).

\( \frac{d^{2} u}{d \xi^{2}}=\ell(\ell+1) \xi^{\ell-1} e^{-\xi^{2} / 2} v-(\ell+1) \xi^{\ell+1} e^{-\xi^{2} / 2} v+(\ell+1) \xi^{\ell} e^{-\xi^{2} / 2} v^{\prime}-(\ell+2) \xi^{\ell+1} e^{-\xi^{2} / 2} v \)

\( +\xi^{\ell+3} e^{-\xi^{2} / 2} v-\xi^{\ell+2} e^{-\xi^{2} / 2} v^{\prime}+(\ell+1) \xi^{\ell} e^{-\xi^{2} / 2} v^{\prime}-\xi^{\ell+2} e^{-\xi^{2} / 2} v^{\prime}+\xi^{\ell+1} e^{-\xi^{2} / 2} v^{\prime \prime} \).

\( = \cancel{{\ell}(\ell+1) \xi^{\ell-1} e^{-\xi^{2} / 2} v}-(2 \ell+3) \xi^{\ell+1} e^{-\xi^{2} / 2} v+\cancel{\xi^{\ell+3} e^{-\xi^{2} / 2} v}+2(\ell+1) \xi^{\ell} e^{-\xi^{2} / 2} v^{\prime} \)

\( -2 \xi^{\ell+2} e^{-\xi^{2} / 2} v^{\prime}+\xi^{\ell+1} e^{-\xi^{2} / 2} v^{\prime \prime}=\cancel{\xi^{\ell+3} e^{-\xi^{2} / 2} v}+\cancel{\ell(\ell + 1) \xi^{\ell-1} e^{-\xi^{2} / 2} v}-K \xi^{\ell+1} e^{-\xi^{2} / 2} v \).

Cancelling the indicated terms, and dividing off \( \xi^{\ell+1} e^{-\xi^{2} / 2} \), we have:

\( v^{\prime \prime}+2 v^{\prime}\left(\frac{\ell+1}{\xi}-\xi\right)+(K-2 \ell-3) v=0 \).

\( \text { Let } v(\xi) \equiv \sum_{j=0}^{\infty} a_{j} \xi^{j}, \text { so } v^{\prime}=\sum_{j=0}^{\infty} j a_{j} \xi^{j-1} ; \quad v^{\prime \prime}=\sum_{j=2}^{\infty} j(j-1) a_{j} \xi^{j-2} \). Then

\( \sum_{j=2}^{\infty} j(j-1) a_{j} \xi^{j-2}+2(\ell+1) \sum_{j=1}^{\infty} j a_{j} \xi^{j-2}-2 \sum_{j=1}^{\infty} j a_{j} \xi^{j}+(K-2 \ell-3) \sum_{j=0}^{\infty} a_{j} \xi^{j}=0 \).

In the first two sums, let \( j \rightarrow j+2 \) (rename the dummy index):

\( \sum_{j=0}^{\infty}(j+2)(j+1) a_{j+2} \xi^{j}+2(\ell+1) \sum_{j=0}^{\infty}(j+2) a_{j+2} \xi^{j}-2 \sum_{j=0}^{\infty} j a_{j} \xi^{j}+(K-2 \ell-3) \sum_{j=0}^{\infty} a_{j} \xi^{j}=0 \).

Note: the second sum should start at j = -1; to eliminate this term (there is no compensating one in \( \xi^{-1} \)) we must take \( a_1 = 0 \). Combining the terms:

\( \sum_{j=0}^{\infty}\left[(j+2)(j+2 \ell+3) a_{j+2}+(K-2 j-2 \ell-3) a_{j}\right]=0, \text { so } a_{j+2}=\frac{(2 j+2 \ell+3-K)}{(j+2)(j+2 \ell+3)} a_{j} \).

Since \( a_1 = 0 \), this gives us a single sequence: \( a_{0}, a_{2}, a_{4}, \ldots \) But the series must terminate (else we get the wrong behavior as \( \xi \rightarrow \infty \), so there occurs some even number \( j_{\max } \text { such that } a_{j_{\max }} \neq 0 \text { but } a_{j_{\max }+2}=0 \). Thus

\( K=2 j_{\max }+2 \ell+3 . \quad \text { But } E=\frac{1}{2} \hbar \omega K, \quad \text { so } E=\left(j_{\max }+\ell+\frac{3}{2}\right) \hbar \omega . \quad \text { Or, letting } j_{\max }+\ell \equiv n \),

\( E_{n}=\left(n+\frac{3}{2}\right) \hbar \omega \), and n can be any nonnegative integer; this agrees with Equation 4.216.

\( E_{n}=\left(n+\frac{3}{2}\right) \hbar \omega \) (4.216).

Since v(ξ) is a polynomial of degree \( j_{\max } \text { in } \xi \text {, or } j_{\max } / 2 \text { in } \xi^{2} \), the number of nodes (roots of v(ξ) is \( j_{\max } / 2= N-1, \text { so } n=j_{\max }+\ell \Rightarrow n=2 N+\ell-2 \). Suppose n is even; then (since \( j_{\max } \) is even) \( \ell=0,2,4, \ldots, n \) For \( \text { each } \ell \text { there are }(2 \ell+1) \text { values for } m \text {. So } \)

\( d(n)=\sum_{\ell=0,2,4, \ldots}^{n}(2 \ell+1) . \quad \text { Let } j=\ell / 2 ; \quad \text { then } d(n)=\sum_{j=0}^{n / 2}(4 j+1)=4 \sum_{j=0}^{n / 2} j+\sum_{j=0}^{n / 2} 1 \).

\( =4 \frac{\left(\frac{n}{2}\right)\left(\frac{n}{2}+1\right)}{2}+\left(\frac{n}{2}+1\right)=\left(\frac{n}{2}+1\right)(n+1)=\frac{(n+1)(n+2)}{2} \), as before ((b)).
