Guest Question.

A tension member consists of a double-angle section, 2L5 \(\times 3 \times \)1⁄2 LLBB (long legs placed back-to-back). The angles are attached to a 3⁄8-inch-thick gusset plate. All steel is A36. Design a welded connection, balanced to eliminate eccentricity, that will resist the full tensile capacity of the member.

 

Admin
Solution

The nominal strength of the member based on the gross section is
\(P_n = F_{y}A_{g} = 36(7.50) = 270.0 kips\)
The nominal strength based on the net section requires a value of U, but the length of the connection is not yet known, so U cannot be computed from Equation 3.1. We will use an estimated value of 0.80, and, if necessary, revise the solution after the connection length is known.
\(A_e = A_{g}U = 7.50(0.80) = 6.000 in.^{2}\)
\(P_{n} = F_{u}A_{e} = 58(6.000) = 348.0 kips\)
For A36 steel, the appropriate electrode is E70XX, and

Minimum weld\( size =\frac {3}{16} \) in. (AISC Table J2.4)
Maximum size \(= \frac {1}{2} – \frac {1}{16} =\frac {7}{16} in.\) (AISC J2.2b)
Compute the required design strength. For yielding of the gross section,
\(\phi _{t}P_n = 0.90(270.0) = 243.0 kips\)
For fracture of the net section,
\(\phi _{t}P_n = 0.75(348.0) = 261.0 kips\)
Yielding of the gross section controls. For one angle, the required design strength is
\(\frac {243.0}{2} = 121.5 kips\)
Try a 5⁄16-inch fillet weld:
Capacity per inch of length = \(\phi R_n\) = 1.392D = 1.392(5) = 6.960 kips/in.
where D is the weld size in sixteenths of an inch.
Check the base metal shear strength. The gusset plate is the thinner of the connected parts and controls. From Equation 7.35, the base metal shear yield strength per unit length is
\(\phi R_n =0.6F_{y}t = 0.6(36)\left ( \frac{3}{8} \right ) = 8.100 kips/in.\)

From Equation 7.36, the base metal shear rupture strength per unit length is
\(\phi R_n = 0.45F_{u}t = 0.45(58)\left ( \frac{3}{8} \right )= 9.788 kips/in.\)
The base metal shear strength is therefore 8.100 kips/in., and the weld strength of 6.960 kips/in. controls. Refer to Figure 8.22. Since there are both transverse and longitudinal welds, we will try both of the options given in AISC J2.4b(2). First, we will use the basic electrode strength of 0.6FEXX for both the longitudinal and transverse welds (this corresponds to \(\phi R_n= 6.960 kips/in.\)) The capacity of the weld across the end of the angle is
\(P_3 = 6.960(5) = 34.80 kips\)
Summing moments about an axis along the bottom, we get
\(\sum M_{L2} = 121.5(3.26) – 34.80\left ( \frac{5}{2} \right )- P_{1}(5) = 0, P_1 = 61.82 kips\)
\(\sum F = 121.5 – 61.82 – 34.80 – P_{2} = 0, P_2 = 24.88 kips\)
\(L_1 = \frac{P_1}{6.960} = \frac{61.82}{6.960}= 8.88 in. \therefore \)Use 9 in.
\(L_2 =\frac{P_2}{6.960} = \frac{24.88}{6.960}= 3.57 in. \therefore \)Use 4 in.
We will now examine the second option given in AISC J2.4b(2), in which we use 150% of the basic strength for the transverse weld and 85% of the basic strength for the longitudinal welds. For the transverse weld, use
\(\phi R_ n= 1.5 \times 6.960 = 10.44 kips/in.\)
and for the longitudinal welds,
\(\phi R_n =0.85 \times 6.960 = 5.916 kips/in.\)

The capacity of the weld across the end of the angle is
\(P_3 = 10.44(5) = 52.20 kips\)
Summing moments about an axis along the bottom, we get
\(\sum M_{L2} = 121.5(3.26) – 52.20\left ( \frac{5}{2} \right )- P_{1}(5) = 0, P_1 = 53.12 kips\)
\(\sum F = 121.5 – 53.12 – 52.20- P_{2} = 0, P_2 = 16.18 kips\)
\(L_1 = \frac{P_1}{5.916} = \frac{53.12}{5.916}= 8.98 in. \therefore\) Use 9 in.
\(L_2 =\frac{P_2}{5.916} = \frac{16.18}{5.916}= 2.73 in. \therefore\) Use 3 in.

The second option results in a slight savings and will be used.
Verify the assumption on U to be sure that the strength of the member is governed by yielding. From Equation 3.1,
\(U = 1 – \frac{\bar{x}}{ℓ}= 1 – \frac{0.746}{9} = 0.9171\)
This is larger than the value of 0.80 that was initially used, so the strength based on fracture will be larger than originally calculated, and the strength will be governed by yielding as assumed.
Use the weld shown in Figure 8.23.
Compute the required allowable strength. For yielding of the gross section,
\(\frac{P_n}{\Omega _{t}}= \frac{270.0}{1.67} = 161.7 kips\)
For fracture of the net section,
\(\frac{P_n}{\Omega _{t}}= \frac{348.0}{2.00} = 174.0 kips\)

Yielding of the gross section controls. For one angle, the required allowable strength is
\(\frac{161.7}{2} = 80.85 kips\)
Try a 5⁄16-inch fillet weld:
Capacity per inch of length \(= \frac{R_n}{\Omega } = 0.9279D = 0.9279(5) = 4.640 kips/in.\)
where D is the weld size in sixteenths of an inch.
Check the base metal shear strength. The gusset plate is the thinner of the connected parts and controls. From Equation 7.37, the base metal shear yield strength per unit length is
\(\frac{R_n}{\Omega }= 0.4F_{y}t = 0.4(36)\left ( \frac{3}{8} \right ) = 5.400 kips/in.\)
From Equation 7.38, the base metal shear rupture strength per unit length is
\(\frac{R_n}{\Omega }= 0.3F_{u}t = 0.3(58)\left ( \frac{3}{8} \right )= 6.525 kips/in.\)
The base metal shear strength is therefore 5.400 kips/in., and the weld strength of 4.640 kips/in. controls.
Refer to Figure 8.22. Since there are both transverse and longitudinal welds, we will try both of the options given in AISC J2.4b(2). First, we will use the basic electrode strength of 0.6FEXX for both the longitudinal and transverse welds (this corresponds to \(R_{n}/\Omega = 4.640 kips/in.\)) The capacity of the weld across the end of the angle is
\(P_3 = 4.640(5) = 23.20 kips\)
Summing moments about an axis along the bottom, we get
\(\sum M_{L2} = 80.85(3.26) – 23.20\left ( \frac{5}{2} \right ) – P_{1}(5) = 0, P_1 = 41.11 kips\)
\(\sum F = 80.85 – 41.11 – 23.20 – P_2 = 0, P_2 = 16.54 kips\)
\(L_1 = \frac{P_1}{4.640} = \frac{41.11}{4.640}= 8.86 in. \therefore\) Use 9 in.
\(L_2 = \frac{P_2}{4.640} = \frac{16.54}{4.640}= 3.56 in. \therefore\) Use 4 in.

We will now examine the second option given in AISC J2.4b(2), in which we use 150% of the basic strength for the transverse weld and 85% of the basic strength for the longitudinal welds. For the transverse weld, use
\(\frac{R_n}{\Omega }= 1.5 \times 4.640 = 6.960 kips/in.\)
and for the longitudinal welds,
\(\frac{R_n}{\Omega }= 0.85 \times 4.640 = 3.944 kips/in.\)
The capacity of the weld across the end of the angle is
\(P_3 = 6.960(5) = 34.80 kips\)
Summing moments about an axis along the bottom, we get
\(\sum M_{L2} = 80.85(3.26) – 2 34.80\left ( \frac{5}{2} \right ) – P_{1}(5) = 0, P_1 = 35.31 kips\)
\(\sum F = 80.85 – 35.31 – 34.80 – P_2 = 0, P_2 = 10.74 kips\)
\(L_1 = \frac{P_1}{3.944} = \frac{35.31}{3.944}= 8.95 in. \therefore\) Use 9 in.
\(L_2 = \frac{P_2}{3.944} = \frac{10.74}{3.944}= 2.72 in. \therefore Use \)3 in.

The second option results in a slight savings and will be used.
Verify the assumption on U to be sure that the strength of the member is governed by yielding. From Equation 3.1,
\(U = 1 – \frac{\bar{x}}{ℓ}= 1 – \frac{0.746}{9} = 0.9171\)

This is larger than the value of 0.80 that was initially used, so the strength based on fracture will be larger than originally calculated, and the strength will be governed by yielding as assumed.
