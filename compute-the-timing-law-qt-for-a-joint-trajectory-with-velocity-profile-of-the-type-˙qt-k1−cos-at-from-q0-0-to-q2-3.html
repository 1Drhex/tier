Guest Question.

Compute the timing law \(q\left(t\right)\) for a joint trajectory with velocity profile of the type \(\dot{q} \left(t\right)\) = \(k\left(1 - \cos\left(at\right)\right)\) from \(q\left(0\right) \) = 0 to \(q\left(2\right)\) = 3.

 

Admin
Solution

Imposing velocity constraints yields

\(\dot{q_{i} }\) = 0

\(\dot{q_{f} }\) = \(k \left(1 – \cos \left(at_{f}\right) \right)\)

The second equation implies that

0 ≤ \(\frac{\dot{q_{f} }} {k}\) ≤ 2

which is satisfied for all k as long as \(\dot{q_{f} }\) = 0. In that case, it is a = 2π/\(t_{f}\)
Then, integrating the given velocity profile gives

\(q\left(t\right)\) = \(k_{1} + k\left(t – \frac{ t_{f}}{2\pi } \sin \left(\frac{2\pi t} {t_{f}} \right) \right) \)

and imposing position constraints yields the two coefficients

\(k_{1}\) = \(q_{i} \)

k = \( \frac{q_{f} – q_{i} } {t_{f}}\)

With the given data, the files used to compute the two coefficients and to generate the time behaviour of position, velocity and acceleration can be found in Folder 4 – 2.

The coefficients are:

k = 1.5 \(k_{1}\) = 0
and the resulting trajectory is illustrated in Fig. S4.2
