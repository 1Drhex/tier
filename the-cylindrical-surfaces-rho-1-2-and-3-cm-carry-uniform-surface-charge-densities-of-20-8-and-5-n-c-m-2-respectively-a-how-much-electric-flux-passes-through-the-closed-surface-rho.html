Guest Question.

The cylindrical surfaces \(\rho\) = 1, 2, and 3 cm carry uniform surface charge densities of 20, -8 and 5 n C/\({ m }^{ 2 }\), respectively. (a) How much electric flux passes through the closed surface \(\rho\) = 5 cm, 0 < z < 1 m ? Find D at P(1 cm, 2cm, 3cm).

Admin
Solution

a) since the densities are uniform, the flux will be
\(\Phi\) = \(2\pi (a{ \rho }_{ s1 }+b{ \rho }_{ s2 }+c{ \rho }_{ s3 })(1)\) = \(2\pi \left[ (.01)(20)-(.02)(8)+(.03)(5) \right] \times { 10 }^{ -9 }\) = 1.2 nC
b) This point lies at radius \(\sqrt { 5}\) cm, and is thus inside the outermost charge layer. This layer , being of uniform density, will not contribute to D at P .We know that in cylindrical coordinates, the layers at 1 and 2 cm will produce the flux density :
D = \({ D }_{ \rho }{ a }_{ \rho }\) = \(\frac { a{ \rho }_{ s1 }+b{ \rho }_{ s2 } }{ \rho } { a }_{ \rho }\)
or
\({ D }_{ \rho }\) = \(\frac { (.01)(20)+(.02)(-8) }{ \sqrt { .05 } }\) = \(1.8 nC/{ m }^{ 2 }\)
At P, \(\phi\) = \(\tan ^{ -1 }{ (2/1)}\)= \(\mathring { 63.4 }\) . Thus \({ D }_{ x }\) = \(1.8\cos { \phi }\) = \(0.8 and { D }_{ y }\) = \(1.8\sin { \phi }\) = 1.6 Finally.
\({ D }_{ P }\) = \((0.8{ a }_{ x } + 1.6{ a }_{ y }) nC/{ m }^{ 2 }\)
