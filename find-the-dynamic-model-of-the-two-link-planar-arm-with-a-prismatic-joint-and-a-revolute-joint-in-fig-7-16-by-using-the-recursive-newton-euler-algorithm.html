Guest Question.

Find the dynamic model of the two-link planar arm with a prismatic joint and a revolute joint in Fig. 7.16 by using the recursive Newton–Euler algorithm.

Admin
Solution

Start by imposing the initial conditions for the velocities and accelerations:

\(\ddot{p}^{0}_{0} – g^{0}_{0}\) = \( \left[ g 0 0 \right]^{T}\) \( \omega^{0}_{0}\) = \(\ddot{\omega}^{0}_{0}\) = 0

All quantities shall be referred to the current link frame. With the choice of the frames as in Fig. S7.2, the following vectors are obtained:

\( r^{1}_{1,c_{1}} \) = \(\left[\begin{matrix} 0 \\ l_{c_{1}} \\ 0 \end{matrix} \right] \) \( r^{1}_{0,1} \) = \(\left[\begin{matrix} 0 \\ d_{1} \\ 0 \end{matrix} \right] \) \( r^{2}_{2,c_{2}} \) = \(\left[\begin{matrix} l_{c_{2}} \\ 0 \\ 0 \end{matrix} \right] \) \( r^{2}_{1,2} \) = \(\left[\begin{matrix} a_{2} \\ 0 \\ 0 \end{matrix} \right] \)

where \(l_{c_{1}}\) and \(l_{c_{2}}\) are both negative quantities. The rotation matrices needed for vector transformation from one frame to another are:

\( R^{o}_{1}\) = \(\left[\begin{matrix} -1 & 0 & 0 \\ 0 & 0 & 1 \\ 0 & 1 & 0 \end{matrix} \right] \) \( R^{1}_{2}\) = \(\left[\begin{matrix} c_{2} & -s_{2} & 0 \\ s_{2} & c_{2} & 0 \\ 0 & 0 & 1 \end{matrix} \right] \)

Further it is assumed that the axes of rotations of the rotors coincide with the respective joint axes, i.e. \( z^{i – 1}_{m_{i}}\) = \( z_{0}\) = \(\left[0 0 1 \right]^{T} \) for i=1,2.

According to (7.107)–(7.114), the Newton-Euler algorithm requires the execution of the following steps.

\(\tau_{i} \) = \(\begin{cases} f^{iT}_{i}R^{i – 1 T}_{i} z_{0} + k_{r_{i} } I_{m_{i} } \dot{\omega }^{i – 1T}_{m_{i}} z^{i – 1}_{m_{i}} + F_{\upsilon i} \dot{d_{i}} + F_{s i} sgn \left(\dot{d_{i}} \right) & for a prismatic joint \\ \mu^{iT}_{i} z_{0} + k_{r_{i} } I_{m_{i} } \dot{\omega }^{i – 1T}_{m_{i}} z^{i – 1}_{m_{i}} + F_{\upsilon i} \dot{\vartheta_{i} } + F_{s i} sgn \left(\dot{\vartheta_{i} } \right) & for a revolute joint \end{cases}\) (7.114)

\(\omega^{i }_{i } \) = \(\begin{cases} R^{i – 1 T}_{i} \omega^{i – 1}_{i – 1} & for a prismatic joint \\ R^{i – 1 T}_{i} \left(\omega^{i – 1}_{i – 1} + \dot{\vartheta }_{i} z_{0}\right) & for a revolute joint \end{cases}\) (7.107)

• Forward recursion: Link 1

\( \omega^{1}_{1}\) = 0

\(\dot{\omega }^{1}_{1} \) = 0

\(\ddot{p }^{1}_{1} \) = \(\left[\begin{matrix} -g \\ \ddot{d}_{1} \\ 0 \end{matrix} \right]\)

\(\ddot{p }^{1}_{c_{1}} \) = \(\left[\begin{matrix} -g \\ \ddot{d}_{1} \\ 0 \end{matrix} \right]\)

\(\dot{\omega }^{0}_{m_{1}}\) = \(\left[\begin{matrix} 0 \\ 0 \\ k_{r_{1}} \ddot{d}_{1} \end{matrix} \right]\)

• Forward recursion: Link 2

\(\omega^{2}_{2}\) = \(\left[\begin{matrix} 0 \\ 0 \\ \dot{\vartheta }_{2} \end{matrix} \right]\)

\(\dot{\omega }^{2}_{2}\) = \(\left[\begin{matrix} 0 \\ 0 \\ \ddot{\vartheta }_{2} \end{matrix} \right]\)

\(\ddot{p}^{2}_{2}\) = \(\left[\begin{matrix} s_{2}\ddot{d_{1} } – a_{2}\dot{\vartheta }^{2}_{2} – g c_{2} \\ c_{2}\ddot{d}_{1} + a_{2}\ddot{\vartheta }_{2} + g s_{2}\\ 0 \end{matrix} \right]\)

\(\ddot{p}^{2}_{c_{2}}\) = \(\left[\begin{matrix} s_{2}\ddot{d_{1} } – \left( l_{c_{2}} + a_{2}\right) \dot{\vartheta }^{2}_{2} – g c_{2} \\ c_{2}\ddot{d}_{1} + \left( l_{c_{2}} + a_{2}\right) \ddot{\vartheta }_{2} + g s_{2}\\ 0 \end{matrix} \right]\)

\( \dot{\omega}^{1}_{m_{2}}\) = \(\left[\begin{matrix} 0 \\ 0 \\ k_{r_{2} }\ddot{\vartheta }_{2} \end{matrix} \right]\)

• Backward recursion: Link 2

\( f^{2}_{2}\) = \(\left[\begin{matrix} m_{2}s_{2}\ddot{d}_{1} – m_{2}\left(l_{C_{2} } + a_{2} \right)\dot{\vartheta_{2} } – m_{2 }g C_{2} \\ m_{2}C_{2}\ddot{d}_{1} + m_{2}\left(l_{C_{2} } + a_{2} \right)\ddot{\vartheta_{2} } + m_{2}g s_{2} \\ 0 \end{matrix} \right]\)

\(\mu^{2}_{2} \) = \(\left[\begin{matrix}\ast \\ \ast \\ m_{2}\left(l_{C_{2} } + a_{2} \right)c_{2} \ddot{d}_{1} + \left(\bar{I}_{2zz} + m_{2}\left(l_{C_{2} } + a_{2} \right)^{2} \right) \ddot{\vartheta }_{2} + m_{2}\left(l_{C_{2} } + a_{2} \right) gs_{2}\end{matrix} \right] \)

\(\tau_{2}\) = \(m_{2}\left(l_{C_{2} } + a_{2} \right)c_{2} \ddot{d}_{1} + \left(\bar{I}_{2zz} + m_{2}\left(l_{C_{2} } + a_{2} \right)^{2} + k^{2}_{r2} l_{m_{2} }\right) \ddot{\vartheta }_{2} + m_{2}\left(l_{C_{2} } + a_{2} \right) gs_{2}\)

where the components marked by the symbol ‘∗’ have not been computed, since they are not related to the joint torque \(\tau_{2}\).

• Backward recursion: Link 1

\(f^{1}_{1}\) = \( \left[\begin{matrix} -m_{2}\left(l_{C_{2} } + a_{2} \right)s_{2}\ddot{\vartheta_{2} } – m_{2}\left(l_{C_{2} } + a_{2} \right)c_{2} \dot{\vartheta } ^{2}_{2} – \left(m_{1} + m_{2}\right)g \\ \left(m_{1} + m_{2}\right) \ddot{d}_{1} + m_{2}\left(l_{C_{2} } + a_{2} \right)c_{2}\ddot{\vartheta_{2} } – m_{2}\left(l_{C_{2} } + a_{2} \right)s_{2}\dot{\vartheta } ^{2}_{2}\\ 0 \end{matrix} \right] \)

\(\tau_{1}\) = \( \left(m_{1} + m_{2} + k^{2}_{r1} l_{m_{1} }\right)\ddot{d}_{1} + m_{2}\left(l_{C_{2} } + a_{2} \right)c_{2} \ddot{\vartheta }_{2} – m_{2} \left(l_{c_{2} } + a_{2} \right)s_{2} \dot{\vartheta } ^{2}_{2}\)

In view of the following equalities:

\(m_{1}\) = \(m_{l_{1} } + m_{m_{2} } \) \(m_{1}\) = \(m_{l_{2} } \) \(l_{2}\) = \(l_{C_{2} } + a_{2}\) \(\bar{I}_{2zz} \) = \(I_{l_{2} }\)

the above dynamic model coincides with the model derived in (S7.4), (S7.5),
with Lagrange formulation.

\(\left(m_{l_{1} } + m_{l_{2} } + m_{m_{2} } + k^{2}_{r1} I_{m_{1} } \right) \ddot{d}_{1} + m_{l_{2} } l_{2} c_{2} \ddot{\vartheta _{2} } – m_{l_{2} } l_{2} s_{2} \ddot{\vartheta } ^{2}_{2}\) = \(\tau_{1} \) (S7.4)

\(m_{l_{2} } l_{2} c_{2} \ddot{d _{1} } + \left(I_{l_{2} } + m_{l_{2} } l^{2}_{2} + k^{2}_{r2} I_{m_{2} }\right)\ddot{\vartheta _{2} } + m_{l_{2} }l_{2}g s_{2}\) = \(\tau_{2} \) (S7.5)
