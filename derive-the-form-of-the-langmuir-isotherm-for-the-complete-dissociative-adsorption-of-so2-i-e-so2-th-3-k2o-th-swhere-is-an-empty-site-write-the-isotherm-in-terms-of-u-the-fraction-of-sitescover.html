Guest Question.

Derive the form of the Langmuir isotherm for the complete dissociative adsorption of \(SO_{2}\), i.e.,
\(SO_{2} + 3 \ * \xrightleftharpoons[]{K} 2 \ O \ - * + S - *\)

where * is an empty site. Write the isotherm in terms of \(\theta\), the fraction of sites covered by any atom. What is the expression for the fraction of sites covered by sulfur atoms, \(\theta _{s} \)

Admin
Solution

\(K=\frac{\left[O-*\right]^{2}\left[S-*\right] }{P_{SO_{2} } \left[*\right] ^{3}} \) , \(L=\left[O-*\right]+\left[S-*\right]+\left[*\right] \) and \( \left[O-*\right] =2\left[S-*\right]\)

 

\(\left\lfloor*\right\rfloor = L-\left\lfloor O-*\right\rfloor – \left\lfloor S-* \right\rfloor =L -3 \left\lfloor S-*\right\rfloor\)

 

\(K= \frac{\left(2\left[S-*\right] \right)^{2} \left [S-*\right]}{P_{SO_{2}} \left[*\right] ^{3}} =\frac {4 \left[S-*\right] ^{3} }{P_{SO_{2}}\left(L-3\left[S-*\right] \right) ^{3} } \)

 

\(K\left(L-3\left[S-*\right]\right)^{3} = 4\left[S-*\right]^{3} /P_{SO_{2}}\Rightarrow K^{1/3} \left(L-3\left[S-*\right] \right)= \sqrt[3]{4} \left[S-*\right] /P^{1/3}_{SO_{2}} \)

 

\(LK^{1/3 } P^{1/3}_{SO_{2}} -3\left[S-*\right] K^{1/3 } P^{1/3}_{SO_{2}}= \sqrt[3]{4} \left[S-*\right] \)

 

\(LK^{1/3 } P^{1/3}_{SO_{2}}= \left[3 K^{1/3 } P^{1/3}_{SO_{2}} + \sqrt[3]{4} \right]\left[S-*\right] \)

 

\(\left[S-*\right]=\frac{LK^{1/3 } P^{1/3}_{SO_{2}}}{\sqrt[3]{4} + 3 K^{1/3 } P^{1/3}_{SO_{2}}} = \frac{L\left(K/4\right) ^{1/3} P^{1/3}_{SO_{2}}}{1+ 3 \left(K/4\right) ^{1/3} P^{1/3}_{SO_{2}}}= \frac{L\left(K^{1/3 } P^{1/3}_{SO_{2}}\right) }{1+3 K^{1/3 } P^{1/3}_{SO_{2}}} \)

 

\(\theta _{s} =\frac{\left[S-*\right]}{L} =\frac{K^{1/3 } P^{1/3}_{SO_{2}}}{1+3 K^{1/3 } P^{1/3}_{SO_{2}}} \) , \(\theta _{s \ max}=1/3\)

 

\(\theta =3 \ \theta _{s}= \frac{3 K^{1/3 } P^{1/3}_{SO_{2}}}{1+3 K^{1/3 } P^{1/3}_{SO_{2}}} \)
