Guest Question.

The cylindrical shell, 2 mm < \(\rho\) < 3 mm, carries a uniformly distributed total current of 8 A in the –\({ a }_{ z }\) direction, and a filament on the z axis carries 8 A in the \({ a }_{ z }\) direction. Find H everywhere.

Admin
Solution

We use Ampere’s circuital law, noting that from symmetry, H will be \({ a }_{ \phi }\) directed. Inside the shell \((\rho < 2mm)\),A circular integration path centered on the z axis encloses only the ﬁlament current along z: Therefore

\(H(\rho <2mm)=\frac { 8 }{ 2\pi \rho } { a }_{ \phi }=\frac { 4 }{ \pi \rho } { a }_{ \phi }A/m\) (\(\rho \) in m)

With the circular integration path within \((2 < \rho < 3mm)\), the enclosed current will consist of the ﬁlament plus that portion of the shell current that lies inside \(\rho\). Ampere’s circuital law applied to a loop of radius \(\rho\) is:

\(\oint { } H.dL={ I }_{ filament }+\int { } \int _{ shell area }^{ }{ } J.dS\)
where the current density is
\(J=-\frac { 8 }{ \pi { \left( 3\times { 10 }^{ -3 } \right) }^{ 2 }-\pi { \left( 2\times { 10 }^{ -3 } \right) }^{ 2 } } { a }_{ z }=-\frac { 8\times { 10 }^{ 6 } }{ 5\pi } { a }_{ z }A/{ m }^{ 2 }\)
So
\(2\pi \rho { H }_{ \phi }=8+\int _{ 0 }^{ 2\pi }{ } \int _{ 2\times { 10 }^{ -3 } }^{ \rho }{ \left( \frac { -8 }{ 5\pi } \times { 10 }^{ 6 } \right) } { a }_{ z }.{ a }_{ z }\acute { \rho } d\acute { \rho } d\phi =8-1.6\times { 10 }^{ 6 }{ \left( \acute { \rho } \right) }^{ 2 }|_{ 2\times { 10 }^{ -3 } }^{ \rho }\)
Solve for \({ H }_{ \phi }\) to find:
H\((2 < \rho < 3 mm) = \frac { 4 }{ \pi \rho } \left[ 1-\left( 2\times { 10 }^{ 5 } \right) \left( { \rho }^{ 2 }-4\times { 10 }^{ -6 } \right) \right] { a }_{ \phi }A/m\) (\(\rho \) in m)

Outside \((\rho > 3mm)\), the total enclosed current is zero, and so H\((\rho > 3mm)\)=0.
