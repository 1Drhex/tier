Guest Question.

Consider the full-bridge inverter whose equivalent circuit is represented in Fig. 9.19 with the four different loads shown in Fig. 9.20 with R=8 Ω, L=30 mH, C=147 μF, \(\ f_{o} \) =60 Hz, and \(\ V_{dc} \) =120 V.

(a) Determine the rms for \(\ i_{o} and v_{o} \) for the ﬁrst, third, ﬁfth, and seventh harmonics.

(b) Determine the total average power delivered to the load for each of the above harmonics.

(c) Determine the output current and voltage total harmonic distortion.

Admin
Solution

To determine the rms for \(\ i_{o} and v_{o} \) for the ﬁrst, third, ﬁfth, and seventh harmonics, we use the nth harmonic given by:

\(\ v_{n,o} = \frac{4V_{dc}}{n\pi } cos n \alpha sin n \omega t \)

\(\ I_{n,rms} =\frac{4V_{dc}}{\sqrt{2} n\pi \mid Z_{n}\mid } cos n \alpha \) (9.45)

i. For the resistive load:

\(\ V_{n,rms}= \frac{2\sqrt{2} V_{dc}}{n\pi } cos n\alpha I_{n,rms}=\frac{V_{n,rms}}{ \mid Z_{n}\mid^{,} } \mid Z_{n}\mid =R=8\Omega , \theta _{n} =0 , \alpha =\pi / 6 \)

 

\(\ V_{1,rms}= \frac{2\sqrt{2}. 120}{\pi } cos (\frac{\pi }{6} ) = 93.56 V I_{1,rms}=\frac{V_{1,rms}}{ \mid Z_{1}\mid }=\frac{V_{1,rms}}{R} =11.69 A \)

 

\(\ V_{3,rms}= \frac{2\sqrt{2}. 120}{3\pi } cos (\frac{\pi }{2} ) = 0 V I_{3,rms}=\frac{V_{3,rms}}{ \mid Z_{3}\mid }=\frac{V_{3,rms}}{R} =0 A \)

 

\(\ V_{5,rms}= \frac{2\sqrt{2}. 120}{5\pi } cos (\frac{5\pi }{6} ) = -18.71 V I_{5,rms}=\frac{V_{5,rms}}{ \mid Z_{5}\mid }=\frac{V_{5,rms}}{R} =-2.34 A \)

 

\(\ V_{7,rms}= \frac{2\sqrt{2}. 120}{7\pi } cos (\frac{7\pi }{6} ) =-13.37 V I_{7,rms}=\frac{V_{7,rms}}{ \mid Z_{7}\mid }=\frac{V_{7,rms}}{R} =-1.67 A \)

Using the ﬁrst four harmonics, the approximate rms values for \(\ v_{o} and i_{o} \) are given by:

\(\ V_{o,rms} =\sqrt{V^{2} _{1,rms}+V^{2} _{3,rms} +V^{2} _{5,rms} +V^{2} _{7,rms}} =96.34 V \)

\(\ I_{o,rms} =\sqrt{I^{2} _{1,rms}+I^{2} _{3,rms} +I^{2} _{5,rms} +I^{2} _{7,rms}} = 12.04 A \)

The exact rms values are given by:

\(\ V_{o,rms}=\sqrt{\frac{2}{\pi }\int_{\alpha }^{\pi }{V^{2}_{dc} d\omega t }=\sqrt{\frac{5}{6} } V_{dc} = 109.59 V } \)

\(\ I_{o,rms} =\frac{V_{o,rms}}{\mid Z_{n}\mid } =13.69 A \)

ii. For the series R-L load: The rms value for the output voltage is the same as the one above, since the output voltage is independent of the load.

\(\ I_{n,rms} =\frac{V_{n,rms}}{\mid Z_{n}\mid} , \mid Z_{n}\mid=\sqrt{R^{2} +(n \omega L)^{2} } \theta _{n} = tan^{-1} \left(\frac{n\omega L}{R} \right) \)

 

\(\ I_{1,rms} =\frac{V_{1,rms}}{\sqrt{R^{2} +(\omega L)^{2} } }= 6.75 A \)

 

\(\ I_{3,rms} =\frac{V_{3,rms}}{\sqrt{R^{2} +(3\omega L)^{2} } }= 0 A \)

 

\(\ I_{5,rms} =\frac{V_{5,rms}}{\sqrt{R^{2} +(5 \omega L)^{2} } } =-0.33 A \)

 

\(\ I_{7,rms} =\frac{V_{7,rms}}{\sqrt{R^{2} +(7\omega L)^{2} } }=-0.168 A \)

iii. For the series RLC load: The rms value for the output voltage is the same as in (i), since the output voltage is independent of the load.

\(\ \mid Z_{n} \mid = \sqrt{R^{2} + \left(n\omega L – \frac{1}{n\omega C} \right)^{2} , } \theta _{n} = tan^{-1} \left(\frac{n\omega L -\frac{1}{n\omega C} }{R} \right) \)

\(\ I_{1,rms} =\frac{V_{1,rms}}{\sqrt{R^{2} +\left(\omega L -\frac{1}{\omega C} \right)^{2} } } =8.95 A \)

\(\ I_{3,rms} =\frac{V_{3,rms}}{\sqrt{R^{2} +\left(3\omega L -\frac{1}{3\omega C} \right)^{2} } } =0 A \)

\(\ I_{5,rms} =\frac{V_{5,rms}}{\sqrt{R^{2} +\left(5\omega L -\frac{1}{5\omega C} \right)^{2} } } =-0.35 A \)

\(\ I_{7,rms} =\frac{V_{7,rms}}{\sqrt{R^{2} +\left(7\omega L -\frac{1}{7\omega C} \right)^{2} } } =-0.17 A \)

iv. For the parallel RLC load:

The rms value for the output voltage is the same as the one shown above, since the output voltage is independent of the load.

\(\ \mid Z_{n} \mid=\frac{1}{\sqrt{\left(\frac{1}{R} \right)^{2} + \left(\frac{n^{2}\omega ^{2} L C -1 }{n\omega L} \right) ^{2} } } , \theta _{n} = tan^{-1} \left(\frac{n^{2}\omega ^{2} L C -R }{n\omega L} \right) \)

\(\ I_{1,rms} = \frac{V_{1,rms}}{1/\sqrt{\left(\frac{1}{R} \right)^{2} + \left(\frac{1^{2}\omega ^{2} L C -1 }{\omega L} \right) ^{2} } } =12.1 A \)

\(\ I_{3,rms} = \frac{V_{3,rms}}{1/\sqrt{\left(\frac{1}{R} \right)^{2} + \left(\frac{3^{2}\omega ^{2} L C -1 }{\omega L} \right) ^{2} } } =0 A \)

\(\ I_{5,rms} = \frac{V_{1,rms}}{1/\sqrt{\left(\frac{1}{R} \right)^{2} + \left(\frac{5^{2}\omega ^{2} L C -1 }{\omega L} \right) ^{2} } } =-0.39 A \)

\(\ I_{7,rms} = \frac{V_{1,rms}}{1/\sqrt{\left(\frac{1}{R} \right)^{2} + \left(\frac{7^{2}\omega ^{2} L C -1 }{\omega L} \right) ^{2} } } =-5.29 A \)

(b) Power calculations are obtained from the following equation:

\(\ P_{on,ave} =V_{on,rms} I_{on,rms} cos \theta _{n} \)

i. For the resistive load:

\(\ cos \theta _{n} =1 For n= 1,3,5… \)

\(\ P_{o1,ave} =(93.56)(11.69) =1094.65 W \)

\(\ P_{o3,ave} =0 W \)

\(\ P_{o5,ave} =(-18.71)(-2.34)=43.78 W \)

\(\ P_{o7,ave}=(-13.37)(1.67)=22.33 W \)

\(\ P_{o} =\sum\limits_{n=1,3,5,…}^{\infty }{I_{on,rms} V_{on,rms} cos\theta _{n} } = 1160.76 W \)

ii. For the series R-L load:

\(\ P_{o1,ave} =(93.56)(11.69) cos (54.73^{\circ } )=364.67 W \)

\(\ P_{o3,ave} =0 W \)

\(\ P_{o5,ave} =(-18.71)(-2.34) cos(81.95^{\circ }) =0.865 W \)

\(\ P_{o7,ave}= (-13.37)(-1.67) cos (84.23^{\circ }) =0.226 W \)

\(\ P_{o,ave}=\sum\limits_{n=1,3,5,…}^{\infty }{I_{on,rms} V_{on,rms} cos\theta _{n} } = 365.75 W \)

iii. For the series RLC load:

\(\ P_{o1,ave} =(93.56)(11.69) cos (40.093^{\circ } )=640.85 W \)

\(\ P_{o3,ave} =0 W \)

\(\ P_{o5,ave} =(-18.71)(-2.34) cos(81.36^{\circ }) =0.98 W \)

\(\ P_{o7,ave}= (-13.37)(-1.67) cos (84.037^{\circ }) =0.24 W \)

\(\ P_{o,ave}=\sum\limits_{n=1,3,5,…}^{\infty }{I_{on,rms} V_{on,rms} cos\theta _{n} } = 641.8 W \)

iv. For the parallel RLC load:

\(\ P_{o1,ave} =(93.56)(11.69) cos (-14.76^{\circ } )=1094.2 W \)

\(\ P_{o3,ave} =0 W \)

\(\ P_{o5,ave} =(-18.71)(-2.34) cos(-64.274^{\circ }) =43.78 W \)

\(\ P_{o7,ave}= (-13.37)(-1.67) cos (-71.579^{\circ }) =22.35 W \)

\(\ P_{o,ave}=\sum\limits_{n=1,3,5,…}^{\infty }{I_{on,rms} V_{on,rms} cos\theta _{n} } = 1160.3 W \)

(c) The total harmonic distortion can be obtained from Eq. (9.43a) given by:

\(\ THD_{v} =\sqrt{\left(\frac{V_{o,rms}}{V_{o1,rms}} \right) ^{2} -1 } \)

\(\ = \frac{\pi }{2 cos \alpha } \sqrt{\frac{1}{2} -\frac{\alpha }{\pi } } \) (9.43a)

\(\ THD_{i} =\sqrt{\left(\frac{I_{o,rms}}{I_{o1,rms}} \right) ^{2} -1 } =0.31 \)
