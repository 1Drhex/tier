Guest Question.

Finding the voltage gain of a noninverting op-amp circuit For the noninverting amplifier in Fig. 3.9(a), the input voltage is \( v_{ S }=100 mV \) with a source resistance of \( R_{ s }=500 \Omega \). The circuit parameters are \( R_{ F }=395 k \Omega, R_{1}=5 k \Omega, \text { and } A_{ o }=2 \times 10^{5} \) . Calculate (a) the closed-loop gain \( A_{ f } \), (b) the output voltage \( v_{ O } \), and (c) the errors in the output voltage \( v_{ O } \) and the gain \( A_{ f } \text { if } A_{ o } \) tends to infinity.

Admin
Solution

Since the current drawn by the op-amp is zero, \( i_{1}=-i_{ f } \). That is,

\( \frac{v_{ x }}{R_{1}}=-\frac{v_{ x }-v_{ O }}{R_{ F }} \)

which gives

\( v_{ x }=\frac{R_{1}}{R_{1}+R_{ F }} v_{ O } \) (3.26)

 

The output voltage \( v_{ O } \) is

\( v_{ O }=A_{ o }\left(v_{ S }-v_{ x }\right)=A_{ o } v_{ d } \) (3.27)

The input voltage at the noninverting terminal is the sum of \( v_{ x } \text { and } v_{ d } \). That is,

\( v_{ S }=v_{ x }+v_{ d } \)

which, after substitution of \( v_{x} \text { from Eq. (3.26) and } v_{ d } \) from Eq. (3.27), becomes

\( v_{ S }=\frac{R_{1} v_{ O }}{R_{1}+R_{ F }}+\frac{v_{ O }}{A_{ o }}=v_{ O }\left(\frac{R_{1}}{R_{1}+R_{ F }}+\frac{1}{A_{ o }}\right) \)

Thus, the closed-loop voltage gain \( A_{ f } \) is given by

\( A_{ f }=\frac{v_{ O }}{v_{ S }}=\frac{A_{ o }\left(R_{1}+R_{ F }\right)}{A_{ O } R_{1}+R_{1}+R_{ F }}=\frac{1+R_{ F } / R_{1}}{1+\left(1+R_{ F } / R_{1}\right) \not A_{ o }}=\frac{1+R_{ F } / R_{1}}{1+x} \) (3.28)

 

where \( x=\frac{1}{A_{0}}\left(1+\frac{R_{ F }}{R_{1}}\right) \) (3.29)

 

For a small value of x, which is usually the case, \( (1+x)^{-1} \approx 1-x \), and Eq. (3.28) can be approximated by

\( A_{ f }=\left(1+\frac{R_{ F }}{R_{1}}\right)(1-x) \) (3.30)

Therefore, the error introduced for a finite value of gain \( A_{ o } \) is x.

(a) From Eq. (3.29),

\( x=\left(\frac{1+395 k / 5 k }{2 \times 10^{5} k }\right)=40 \times 10^{-5}=40 \times 10^{-3} \% \)

From Eq. (3.28),

\( A_{ f }=\left(\frac{1+395 / 5}{1+40 \times 10^{-5}}\right)=79.968 \)

(b) The output voltage \( v_{ O } \) is

\( v_{ O }=A_{ f } v_{ S }=79.968 \times 100 \times 10^{-3}=7.9968 V \)

(c) From Eq. (3.30), the error in the output voltage \( v_{ O } \) is

\( \Delta v_{ O }=-x\left(1+\frac{R_{ F }}{R_{1}}\right)=-40 \times 10^{-5} \times 80=-32 mV , \quad \text { or }-0.04 \% \)

The error in the gain \( A_{ f } \) is

\( \Delta A_{ f }=-x=-40 \times 10^{-5}=-0.04 \% \)

NOTE: To minimize the dependence of the closed-loop gain \( A_{ f } \) on the open-loop gain \( A_{ o } \), the value of x should be made very small. That is,

\( A_{0}>>\left(1+\frac{R_{ F }}{R_{1}}\right) \) (3.31)

This condition is often satisfied by making \( A_{ o } \) at least 10 times larger than \( \left(1+R_{ F } / R_{1}\right) \). That is,

\( \left(1+\frac{R_{ F }}{R_{1}}\right) \leq 0.1 A_{ o } \) (3.32)
