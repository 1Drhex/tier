Guest Question.

Determine the reaction forces and the driving torque for the four-bar linkage in the posture shown in Fig. 12.9. The input crank is operating at a constant angular velocity ω2 = 48 rad/s ccw, and the applied force acting at point C is \(F _{C}=-0.8 \hat{ j } kN\). The masses of links 3 and 4 are m3 = 1.5 kg and m4 = 5 kg, respectively, and the second moments of mass of links 2, 3, and 4 are \(I_{G_{2}}=0.025 kg \cdot m ^{2}, I_{G_{3}}=0.012 kg \cdot m ^{2} \text {, and } I_{G_{4}}=0.054 kg \cdot m ^{2}\), respectively.

ASSUMPTIONS

Gravity and friction effects are negligible.

Admin
Solution

The information given in the figure caption in vector form is:

\(\begin{aligned}R _{A O_{2}} &=60 mm \angle 115^{\circ}=-25.4 \hat{ i }+54.4 \hat{ j } mm \\R _{G_{3} A} &=90 mm \angle 48.7^{\circ}=59.4 \hat{ i }+67.6 \hat{ j } mm \\R _{B A} &=220 mm \angle 18.7^{\circ}=208.4 \hat{ i }+70.5 \hat{ j } mm \\R _{B O_{4}} &=150 mm \angle 61.5^{\circ}=71.6 \hat{ i }+131.8 \hat{ j } mm \\R _{G_{4} O_{4}} &=90 mm \angle 25.5^{\circ}=81.2 \hat{ i }+38.7 \hat{ j } mm \\R _{C O_{4}} &=120 mm \angle 5.1^{\circ}=119.5 \hat{ i }+10.7 \hat{ j } mm \\R _{O_{4} O_{2}} &=100 mm \angle 0^{\circ}=100.0 \hat{ i } mm .\end{aligned}\)

A kinematic analysis of the linkage gives: \( \alpha_{3}=-119 \hat{ k } rad / s ^{2}, \alpha_{4}=-625 \hat{ k } rad / s ^{2}, A _{G_{3}}= 162 m / s ^{2} \angle-73.2^{\circ}=46.8 \hat{ i }-155 \hat{ j } m / s ^{2}, \text { and } A _{G_{4}}=104 m / s ^{2} \angle-127^{\circ}=-62.6 \hat{ i }- 83.1 \hat{ j } m / s ^{2}\).

Next, we calculate the inertia forces and inertia moments using Eqs (12.12) and (12.13):

\(\sum F =m A _{G}\) (12.12)

 

\(\sum M _{G}=I_{G} \alpha\) (12.13)

 

\(\begin{aligned}-m_{2} A _{G_{2}} &= 0 \\-m_{3} A _{G_{3}} &=-(1.5 kg )\left(46.8 \hat{ i }-155 \hat{ j } m / s ^{2}\right)=-70.2 \hat{ i }+233 \hat{ j } N \\-m_{4} A _{G_{4}} &=-(5.0 kg )\left(-62.6 \hat{ i }-83.1 \hat{ j } m / s ^{2}\right)=313 \hat{ i }+415 \hat{ j } N \\-I_{G_{2}} \alpha _{2} &= 0 \\-I_{G_{3}} \alpha _{3} &=-\left(0.012 kg \cdot m ^{2}\right)\left(-119 \hat{ k } rad / s ^{2}\right)=1.43 \hat{ k } N \cdot m\end{aligned}\)

and

\(-I_{G_{4}} \alpha_{4}=-\left(0.054 kg \cdot m ^{2}\right)\left(-625 \hat{ k } rad / s ^{2}\right)=33.8 \hat{ k } N \cdot m\)

Since the solution here is analytic, we do not need to calculate offset distances using Eq. (12.18).

\(h=\frac{I_{G} \alpha_{3}}{m A_{G}}\) (12.18)

For our next step, we make sketches of the free-body diagrams of link 3 and link 4. From the two free-body diagrams, we note that there are a total of six unknown force components. Since there are a total of six scalar equations available, two components of the forces and one for the sum of the moments about the mass center of each link, then the problem is solvable. However, these six equations must be solved simultaneously to find the six unknowns. In an effort to decouple the equations, an alternative strategy is to take the sum of the moments about different points in the links.

Considering the free-body diagram of link 3, we formulate the summation of moments about point A:

\( \Sigma M _{A}= R _{G_{3} A} \times\left(-m_{3} A _{ G _{3}}\right)+\left(-I_{G_{3}} \alpha_{3}\right)+ R _{B A} \times F _{43}= 0 \) (1)

Also, considering the free-body diagram of link 4, we formulate the summation of moments about point O4:

\(\Sigma M _{O_{4}}= R _{G_{4} O_{4}} \times\left(-m_{4} A _{G_{4}}\right)+\left(-I_{G_{4}} \alpha_{4}\right)+ R _{C O_{4}} \times F _{C}+ F _{B O_{4}} \times F _{34}= 0\) (2)

Remembering that F43 = −F34, the in-plane components of F34 are the only two unknowns, and they are shared by these two equations.
The cross-product terms of these equations are

\(\begin{aligned}R _{G_{3} A} \times\left(-m_{3} A _{G_{3}}\right) &=(59.4 \hat{ i }+67.6 \hat{ j } mm ) \times(70.2 \hat{ i }+233 \hat{ j } N )=18.6 \hat{ k } N \cdot m \\R _{B A} \times F _{43} &=(208.4 \hat{ i }+70.5 \hat{ j } mm ) \times\left(-F_{34}^{x} \hat{ i }-F_{34}^{y} \hat{ j } N \right) \\&=\left(70.5 F_{34}^{x}-208.4 F_{34}^{y}\right) \hat{ k } N \cdot m\end{aligned}\)

and

\(\begin{aligned}R _{G_{4} O_{4}} \times\left(-m_{4} A _{G_{4}}\right) &=(81.2 \hat{ i }+38.7 \hat{ j } mm ) \times(313 \hat{ i }+415 \hat{ j } N )=21.6 \hat{ k } N \cdot m \\R _{C O_{4}} \times F _{C} &=(119.5 \hat{ i }+10.7 \hat{ j } mm ) \times(-800 \hat{ j } N )=-95.6 \hat{ k } N \cdot m \\R _{B O_{4}} \times F _{34} &=(71.6 \hat{ i }+131.8 \hat{ j } mm ) \times\left(F_{34}^{x} \hat{ i }+F_{34}^{y} \hat{ j } N \right) \\&=\left(-131.8 F_{34}^{x}+71.6 F_{34}^{y}\right) \hat{ k } N \cdot m\end{aligned}\)

Then, substituting these values into Eqs. (1) and (2) gives

\(\sum M _{A}=18.6 \hat{ k }+1.43 \hat{ k }+\left(70.5 F_{34}^{x}-208.4 F_{34}^{y}\right) \hat{ k }= 0\)

and

\(\sum M _{O_{4}}=21.6 \hat{ k }+33.8 \hat{ k }-95.6 \hat{ k }+\left(-131.8 F_{34}^{x}+71.6 F_{34}^{y}\right) \hat{ k }= 0\)

Rearranging terms gives two equations in two unknowns, that is,

\(70.5 F_{34}^{x}-208.4 F_{34}^{y}=-20.0 N \cdot m\)

and

\(-131.8 F_{34}^{x}+71.6 F_{34}^{y}=40.2 N \cdot m\)

Solving simultaneously, the two components of the force vector F34 are:

\(F_{34}^{x}=-0.31016 kN =-310.16 N \quad \text { and } \quad F_{34}^{y}=-0.00884 kN =-8.84 N\)

Therefore, the force vector is

\(F _{34}=-310.16 \hat{ i }-8.84 \hat{ j } N =310.29 N \angle-178.4^{\circ}\)

Next, summing forces on link 4 yields the equation

\(\Sigma F _{i 4}= F _{14}+ F _{34}+ F _{C}+\left(-m_{4} A _{G_{4}}\right)= 0\)

and, upon solving, we find the force vector

\(\begin{aligned}F _{14} &=- F _{34}- F _{C}-\left(-m_{4} A _{G_{4}}\right) \\&=-(-310.16 \hat{ i }-8.84 \hat{ j } N )-(-800 \hat{ j } N )-(313 \hat{ i }+415 \hat{ j } N ) \\&=-2.84 \hat{ i }+393.84 \hat{ j } N =393.85 N \angle 90.41^{\circ}\end{aligned}\)

Similarly, summing forces on link 3 yields

\(\Sigma F _{13}= F _{23}+ F _{43}+\left(-m_{3} A _{G_{3}}\right)= 0\)

and solving this, the force vector is

\(\begin{aligned}F _{23} &=-\left(- F _{34}\right)-\left(-m_{3} A _{G_{3}}\right) \\&=-(310.16 \hat{ i }+8.84 \hat{ j } N )-(-70.2 \hat{ i }+233 \hat{ j } N ) \\&=-239.96 \hat{ i }-241.84 \hat{ j } N =340.69 N \angle-134.78^{\circ}\end{aligned}\)

Now, for link 2, we have

\(\Sigma F _{i 2}= F _{12}+ F _{32}+\left(-m_{2} A _{G_{2}}\right)= 0\)

Rearranging this gives

\(F _{12}=- F _{32}= F _{23}=-239.96 \hat{ i }-241.84 \hat{ j } N =340.69 N \angle-134.78^{\circ}\)

Summing moments on link 2 about O2, gives

\(\sum M _{O_{2}}= R _{A O_{2}} \times F _{32}+ T _{12}+\left(-I_{G_{2}} \alpha _{2}\right)= 0\)

and solving for the driving torque gives

\(T _{12}=- R _{A O_{2}} \times F _{32}=-(-25.4 \hat{ i }+54.4 \hat{ j } mm ) \times(239.96 \hat{ i }+241.84 \hat{ j } N )=19.2 \hat{ k } N \cdot m\)
