Guest Question.

Designing a sawtooth-wave generator Design the sawtooth-wave generator shown in Fig. 16.40(a) so that \( f_{o}=4 kHz , V_{ th }=5 V \), and the circuit has a duty cycle of k = 0.25. Assume \( V_{\text {sat }}=\left|-V_{\text {sat }}\right|=14 V \).

Admin
Solution

The steps used to design the sawtooth-wave generator are as follows:
Step 1. Find the value of \( V_{\text {ref }} \) required to obtain the desired duty cycle k. From Eq. (16.47),

\(\begin{aligned}k &=\frac{t_{1}}{T}=\frac{2 R C V_{\text {th }}}{V_{\text {sat }}-V_{\text {ref }}} \times \frac{V_{\text {sat }}^{2}-V_{\text {ref }}^{2}}{4 R C V_{\text {th }} V_{\text {sat }}}=\frac{V_{\text {sat }}+V_{\text {ref }}}{2 V_{\text {sat }}} \\&=\frac{1}{2}\left[1+\frac{V_{\text {ref }}}{V_{\text {sat }}}\right]\end{aligned}\) (16.47)

 

\(\begin{aligned}V_{\text {ref }} &=(2 k-1) V_{\text {sat }} \\&=(2 \times 0.25-1) \times 14=-7 V\end{aligned}\) (16.48)

 

Step 2. Find the values of \( R_{1} \text { and } R_{ F } \). Equation (16.34) gives

\( V_{ th }=\frac{R_{1}}{R_{ F }}\left(+V_{ sat }\right) \) (16.34)

 

\( \frac{R_{1}}{R_{ F }}=\frac{V_{ th }}{V_{ sat }}=\frac{5}{14}=0.36 \)

Let \( R_{1}=10 k \Omega \); then

\( R_{ F }=\frac{R_{1}}{0.36}=\frac{10 k \Omega}{0.36}=28 k \Omega \quad \text { (use a } 30- k \Omega \text { potentiometer) } \)

Step 3. Choose a suitable value of C: Let C = 0.01 µF.
Step 4. Find the value of R. Equation (16.46) gives

\( f_{ o }=\frac{V_{ sat }^{2}-V_{ ref }^{2}}{4 R C V_{ th } V_{ sat }} \) (16.46)

 

\(\begin{aligned}R &=\frac{V_{\text {sat }}^{2}-V_{\text {ref }}^{2}}{4 f_{ o } C V_{\text {th }} V_{\text {sat }}} \\&=\frac{14^{2} V ^{2}-(-7)^{2} V ^{2}}{4 \times 4 kHz \times 0.01 \mu F \times 5 V \times 14 V }=13.1 k \Omega \quad \text { (use a 15-k } \Omega \text { potentiometer) }\end{aligned}\) (16.49)
