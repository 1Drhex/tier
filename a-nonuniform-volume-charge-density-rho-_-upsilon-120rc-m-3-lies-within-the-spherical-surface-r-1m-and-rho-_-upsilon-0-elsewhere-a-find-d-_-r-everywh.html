Guest Question.

A nonuniform volume charge density , \({ \rho }_{ \upsilon } = 120 r C/{ (m) }^{ 3 }\) , lies within the spherical Surface r = 1m, and \({ \rho }_{ \upsilon } = 0\) elsewhere. (a) Find \({ D }_{ r } \) everywhere. (b) What surface charge density \({ \rho }_{ S2 }\) should be on the surface r = 2 so that \({ D }_{ r|r={ 2 }^{ - } } = { D }_{ r|r={ 2 }^{ + } }\) ? (c) Make a sketch of \({ D }_{ r}\) vs r for 0 r < 5 with both distribution present .

Admin
Solution

a) For r < 1 m , we apply Gauss’ law to a spherical surface of radius r within this range to find
\(4\pi { r }^{ 2 }{ D }_{ r }\) = \(4\pi \int _{ 0 }^{ r }{ 120\acute { r } { (\acute { r }) }^{ 2 } } d\acute { r } = 120\pi { r }^{ 4 }\)
Thus \({ D }_{ r }\) = \((30{ r }^{ 2 })\) for r < 1 m. for r > 1 m, the gaussian surface lies outside the charge distribution. The set up is the same, except the upper limit of the above integral is 1 instead od r .
This results in \({ D }_{ r }\) = \((30/{ r }^{ 2 })\) for r > 1 m.

b) AT r = \({ 2 }^{ – }\), we have \( {D }_{ r,r={ 2 }^{ – } } \) = \(30/{ 2 }^{ 2 }\) = 15/2 , from part a. The flux density in the region r > 2 arising from a surface charge at r = 2 is found from Gauss’ law through
\(4\pi { r }^{ 2 }{ D }_{ rs }\) = \(4\pi { (2) }^{ 2 }{ \rho }_{ S2 } \Longrightarrow { D }_{ rs }\) = \(\frac { 4{ \rho }_{ S2 } }{ { r }^{ 2 } }\)

The total ﬂux density in the region r > 2 arising from the two distributions is
\({ D }_{ rT }=\frac { 30 }{ { r }^{ 2 } } +\frac { 4{ \rho }_{ s2 } }{ { r }^{ 2 } }\)
our requirement that \({ D }_{ r,r={ 2 }^{ – } }\) = \(2{ D }_{ r,r={ 2 }^{ + } }\) becomes
\(\frac { 30 }{ { (2) }^{ 2 } }\) = \(2\left( \frac { 30 }{ { (2) }^{ 2 } } +{ \rho }_{ S2 } \right) \Longrightarrow { \rho }_{ S2 }\) = \(-\frac { 15 }{ 4 } C/{ m }^{ 2 }\)
c) with both charges, \({ D }_{ r }\left( r<1 \right) \) = \(30{ r }^{ 2 }\), \({ D }_{ r }\left( 1<r<2 \right)\) = \(30{ /r }^{ 2 }\) , and \({ D }_{ r }\left( >2 \right)\) = \(15{ /r }^{ 2 }\) . These are plotted on the next page.
