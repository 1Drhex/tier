Guest Question.

Designing a clock circuit

(a) Design a clock circuit to produce a pulse output at a frequency of \( f_{ clk }=100 kHz \). The DC supply voltage is 5 V.

(b) Use PSpice/SPICE to check your design by plotting the output voltage.

Admin
Solution

(a) Given that \( T=1 / f_{ clk }=1 / 100 kHz =10 \mu s \), the design steps are carried out as follows:

Step 1. Choose the circuit topology. Figure 15.60(a) shows a circuit that uses inverters; NOR gates could be used instead.

Step 2. Choose the gate types. Although use of CMOS inverters would simplify the analysis, we will use bipolar inverters of type 7404. We will derive a general expression for the clock frequency that can be applied to CMOS inverters also. From the data sheet, the inverter parameters are \( V_{ CC }=5 V , V_{ OH }=3.4 V , V_{ OL }=0.5 V \), \( V_{ IH }=2 V , V_{ IL }=0.8 V , I_{ IL }=0.1 mA \text {, and } I_{ IH }=20 \mu A \).

Step 3. Analyze the circuit. Suppose that, at t = 0, \( v_{ O 1} \) (that is, the output of inverter \( U _{1} \)) is high at \( V_{ OH } \), \( v_{ O } \text { is low at } V_{ OL } \text {, and } v_{ I } \) (the input of inverter \( U _{1} \)) is at \( V_{ IH }-V_{ OH } \) (from the previous cycle). Capacitor C will charge exponentially, with a time constant of \( \tau=R C \). The equivalent circuit during charging is shown in Fig. 15.60(b). The input voltage \( v_{ I } \) will have a DC component and an exponentially decaying component.
Thus,

\( v_{ I }(t)-V_{ OH }=R I_{ IL }+\left[v_{ I }\left(0^{+}\right)-v_{ O }\left(0^{+}\right)\right] e^{-t / \tau}=R I_{ IL }+\left(V_{ IH }-2 V_{ OH }\right) e^{-t / \tau} \quad\left(\text { for } 0 \leq t \leq T_{2}\right) \)

At \( t=T_{2}, v_{ I } \text { rises to } V_{ IL } \text {, and we can find } T_{2} \text { from the condition } v_{ I }\left(t=T_{2}\right)=V_{ IL } \text { : } \)

 

\( T_{2}=R C \ln \left(\frac{V_{ IH }-2 V_{ OH }}{V_{ IL }-V_{ OH }-R I_{ IL }}\right) \) (15.101)

where IIL is the input current in the logic low state. When \( v_{ I } \text { reaches } V_{ IL } \text { at } t=T_{2}, v_{ O 1} \text { goes low to } V_{ OL } \text { and } v_{ O } \) rises to \( V_{ OH } \text { after a delay time of } t_{ pd } \). The capacitor feeds this change in \( \text { VO } \) back to the input side. As a result, \( v_{I} \text { jumps from } V_{I L} \text { to } V_{I L}+V_{O H} \).

Now, with \( v_{ I }=V_{ IL }+V_{ OH }, v_{ O 1}=V_{ OL }, \text { and } v_{ O }=V_{ OH } \), the capacitor C will discharge exponentially, with a time constant of \( \tau=R C \). The equivalent circuit during discharging is shown in Fig. 15.60(c). The input voltage \( v_{ I } \) will have a DC component of \( V_{ OL } \) and a decaying component. Redefining the time origin \( t=0 \text { (at } t=T_{2} \text { ) } \) gives the input voltage \( v_{ I } \):

\( v_{ I }(t)=V_{ OL }+\left[v_{ I }\left(0^{+}\right)-v_{ O }\left(0^{+}\right)\right] e^{-t / \tau}=V_{ OL }+\left(V_{ OH }+V_{ IL }-V_{ OL }\right) e^{-t / \tau} \quad\left(\text { for } 0 \leq t \leq T_{1}\right) \)

At \( t=T_{1}, v_{ I } \text { falls to } V_{ IH } \text {, and we can find } T_{1} \text { from the condition } v_{ I }\left(t=T_{1}\right)=V_{ IH } \):

\( T_{1}=R C \ln \left(\frac{V_{ OH }+V_{ IL }-V_{ OL }}{V_{ IH }-V_{ OL }}\right) \) (15.102)

When \( v_{ I } \text { falls to } V_{ IH } \text { at } t=T_{1}, v_{ O 1} \text { becomes high at } V_{ OH } \text { and } v_{ O } \text { falls to } V_{ OL } \text { after a delay time of } t_{ pd } \). Because the capacitor feeds this change in \( v_{ O } \text { back to the input side, } v_{ I } \text { falls from } V_{ IH } \text { to } V_{ IH }-V_{ OH } \), and the cycle is repeated. The voltage waveforms are shown in Fig. 15.60(d). The period T of the output voltage is given by

\( T=T_{1}+T_{2}=R C\left(\ln \frac{V_{ OH }+V_{ IL }-V_{ OL }}{V_{ IH }-V_{ OL }}+\ln \frac{V_{ IH }-2 V_{ OH }}{V_{ IL }-V_{ OH }-R I_{ IL }}\right) \) (15.103)

NOTE: For a CMOS gate, \( V_{ OL } \approx 0, V_{ OH }=V_{ DD }, I_{ IL } \equiv 0, \text { and } V_{ IL }=V_{ IH }=V_{ t } \) (threshold voltages of the NMOS). Thus, Eq. (15.103) becomes

\( T=T_{1}+T_{2}=R C\left(\ln \frac{V_{ DD }+V_{ IL }}{V_{ IH }}+\ln \frac{V_{ IH }-2 V_{ OH }}{V_{ IL }-V_{ OH }}\right) \) (15.104)

Step 4. Find the values of R and C. Using Eq. (15.103), we get

 

\( 10 \mu A =R C\left(\ln \frac{3.4+0.8-0.5}{2-0.5}+\ln \frac{2-2 \times 3.4}{0.8-3.4-R \times 0.1 mA }\right) \)

Choose a suitable value of C: Let \( C=1 nF \text {. Solving for } R \text { by iteration, we get } R \approx 8 k \Omega \text {. Then } \tau=R C=8 \mu s \),
\( T_{1}=7.22 \mu s \text {, and } T_{2}=2.76 \mu s \).

(b) The PSpice simulation is shown in Fig. 15.61. The PSpice plots shown in Fig. 15.62 give T = 7.53 µs, \( T_{1}=6.41 \mu s , \text { and } T_{2}=1.12 \mu s \). Thus, a higher value of R (say, R = 10 kΩ) will be needed.
