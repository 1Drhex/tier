Guest Question.

(a) Use \({ F }_{ 2 }={ \mu }_{ 0 }\frac { { I }_{ 1 }{ I }_{ 2 } }{ 4\pi } \oint { } \left[ d{ L }_{ 2 }\times \oint { } \frac { d{ L }_{ 1 }\times { a }_{ R12 } }{ { R }_{ 12 }^{ 2 } } \right] ={ \mu }_{ 0 }\frac { { I }_{ 1 }{ I }_{ 2 } }{ 4\pi } \oint { } \left[ \oint { } \frac { { a }_{ R12 }\times d{ L }_{ 1 } }{ { R }_{ 12 }^{ 2 } } \right] \times d{ L }_{ 2 }\) to show that the force of attraction per unit length between two ﬁlamentary conductors in free space with currents \({ I }_{ 1 }{ a }_{ z }\) at x = 0, y = d/2, and \({ I }_{ 2 }{ a }_{ z }\) at x = 0, y = -d/2, is \({ \mu }_{ 0 }{ I }_{ 1 }{ I }_{ 2 }/(2\pi d)\). (b) Show how a simpler method can be used to check your result.

Admin
Solution

The force on \({ I }_{ 2 }\) is given by
\({ F }_{ 2 }={ \mu }_{ 0 }\frac { { I }_{ 1 }{ I }_{ 2 } }{ 4\pi } \oint { } \left[ \oint { } \frac { { a }_{ R12 }\times d{ L }_{ 1 } }{ { R }_{ 12 }^{ 2 } } \right] \times d{ L }_{ 2 }\)

Let \({ z }_{ 1 }\) indicate the z coordinate along \({ I }_{ 1 }\), and \({ z }_{ 2 }\) indicate the z coordinate along \({ I }_{ 2 }\). We then have \({ R }_{ 12 }=\sqrt { { \left( { z }_{ 2 }-{ z }_{ 1 } \right) }^{ 2 }+{ d }^{ 2 } }\) and
\({ a }_{ R12 }=\frac { \left( { z }_{ 2 }-{ z }_{ 1 } \right) { a }_{ z }-d{ a }_{ y } }{ \sqrt { { \left( { z }_{ 2 }-{ z }_{ 1 } \right) }^{ 2 }+{ d }^{ 2 } } } \)
Also, \(d{ L }_{ 1 } = d{ z }_{ 1 } { a }_{ z } \)and \(d{ L }_{ 2 } = d{ z }_{ 2 } { a }_{ z }\) The “inside” integral becomes:
\(\oint { } \frac { { a }_{ R12 }\times d{ L }_{ 1 } }{ { R }_{ 12 }^{ 2 } } =\oint { } \frac { \left[ \left( { z }_{ 2 }-{ z }_{ 1 } \right) { a }_{ z }-d{ a }_{ y } \right] \times d{ z }_{ 1 }{ a }_{ z } }{ { \left[ { \left( { z }_{ 2 }-{ z }_{ 1 } \right) }^{ 2 }+{ d }^{ 2 } \right] }^{ 1.5 } } =\int _{ -\infty }^{ \infty }{ } \frac { -dd{ z }_{ 1 }{ a }_{ x } }{ { \left[ { \left( { z }_{ 2 }-{ z }_{ 1 } \right) }^{ 2 }+{ d }^{ 2 } \right] }^{ 1.5 } } \)
The force expression now becomes
\({ F }_{ 2 }={ \mu }_{ 0 }\frac { { I }_{ 1 }{ I }_{ 2 } }{ 4\pi } \oint { } \left[ \int _{ -\infty }^{ \infty }{ } \frac { -dd{ z }_{ 1 }{ a }_{ x } }{ { \left[ { \left( { z }_{ 2 }-{ z }_{ 1 } \right) }^{ 2 }+{ d }^{ 2 } \right] }^{ 1.5 } } \times d{ z }_{ 2 }{ a }_{ z } \right] ={ \mu }_{ 0 }\frac { { I }_{ 1 }{ I }_{ 2 } }{ 4\pi } \int _{ 0 }^{ 1 }{ } \int _{ -\infty }^{ \infty }{ } \frac { dd{ z }_{ 1 }d{ z }_{ 2 }{ a }_{ y } }{ { \left[ { \left( { z }_{ 2 }-{ z }_{ 1 } \right) }^{ 2 }+{ d }^{ 2 } \right] }^{ 1.5 } }\)
Note that the “outside” integral is taken over a unit length of current \({ I }_{ 2 }\). Evaluating, obtain,
\({ F }_{ 2 }={ \mu }_{ 0 }\frac { { I }_{ 1 }{ I }_{ 2 }d{ a }_{ y } }{ 4\pi { d }^{ 2 } } (2)\int _{ 0 }^{ 1 }{ } d{ z }_{ 2 }=\frac { { \mu }_{ 0 }{ I }_{ 1 }{ I }_{ 2 } }{ 2\pi { d } } { a }_{ y }N/m\)
as expected.

b) We use d\({ F }_{ 2 } = { I }_{ 2 } d { L }_{ 2 } \times { B }_{ 12 }\), where the field from current 1 at the location of current 2 is
\({ B }_{ 12 }=\frac { { \mu }_{ 0 }{ I }_{ 1 } }{ 2\pi d } { a }_{ x }T\).

so over a unit length of \({ I }_{ 2 }\) , we obtain

\({ F }_{ 2 }={ I }_{ 2 }{ a }_{ z }\times \frac { { \mu }_{ 0 }{ I }_{ 1 } }{ 2\pi d } { a }_{ x }={ \mu }_{ 0 }\frac { { I }_{ 1 }{ I }_{ 2 } }{ 2\pi d } { a }_{ y }\)N/m

This second method is really just the ﬁrst over again, since we recognize the inside integral of the ﬁrst method as the Biot-Savart law, used to ﬁnd the ﬁeld from current 1 at the current 2 location.
