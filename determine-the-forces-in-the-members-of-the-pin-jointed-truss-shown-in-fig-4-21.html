Guest Question.

Determine the forces in the members of the pin-jointed truss shown in Fig. 4.21.

Admin
Solution

The support reactions are first calculated and are as shown in Fig. 4.21.
The next step is to choose an xy axis system and then insert the joint coordinates in the diagram. In Fig. 4.21 we shall choose the support point A as the origin of axes although, in fact, any joint would suffice; the joint coordinates are then as shown.
Again, as in the method of joints, the solution can only begin at a joint where there are no more than two unknown member forces, in this case joints A and E. Theoretically it is immaterial at which of these joints the analysis begins but since A is the origin of axes we shall start at A. Note that it is unnecessary to insert arrows to indicate the directions of the member forces since the members are assumed to be in tension and the directions of the components of the member forces are automatically specified when written in terms of tension coefficients and joint coordinates (Eqs (4.3) and (4.4)).
The equations of equilibrium at joint A are

\(x \text { direction : } t_{ AB }\left(x_{ B }-x_{ A }\right)+t_{ AC }\left(x_{ C }-x_{ A }\right)-R_{ A , H }=0\) (i)

 

\(y \text { direction : } t_{ AB }\left(y_{ B }-y_{ A }\right)+t_{ AC }\left(y_{ C }-y_{ A }\right)-R_{ A , V }=0\) (ii)

Substituting the values of \(R_{ A , H }, R_{ A , V }\) and the joint coordinates in Eqs (i) and (ii) we obtain, from Eq. (i),

\(t_{ AB }(0-0)+t_{ AC }(1.5-0)-3=0\)

whence

\(t_{ AC }=+2.0\)

and from Eq. (ii)

\(t_{ AB }(1.5-0)+t_{ AC }(0-0)+1=0\)

so that

\(t_{ AB }=-0.67\)

We see from the derivation of Eqs (4.3) and (4.4) that the units of a tension coefficient are force/unit length, in this case kN/m. Generally, however, we shall omit the units.
We can now proceed to joint B at which, since \( t_{ BA }\left(=t_{ AB }\right) \) has been calculated, there are two unknowns

\(\text { parallel to the } x \text { axis : } t_{ AB }\left(x_{ B }-x_{ A }\right)\) (4.3)

 

\(\text { parallel to the } y \text { axis : } t_{ AB }\left(y_{ B }-y_{ A }\right)\) (4.4)

 

\(x \text { direction : } t_{ BA }\left(x_{ A }-x_{ B }\right)+t_{ BC }\left(x_{ C }-x_{ B }\right)+t_{ BD }\left(x_{ D }-x_{ B }\right)+3=0\) (iii)

 

\(y \text { direction : } t_{ BA }\left(y_{ A }-y_{ B }\right)+t_{ BC }\left(y_{ C }-y_{ B }\right)+t_{ BD }\left(y_{ D }-y_{ B }\right)=0\) (iv)

Substituting the values of the joint coordinates and \( t_{ BA } \) in Eqs (iii) and (iv) we have, from Eq. (iii)

\(-0.67(0-0)+t_{ BC }(1.5-0)+t_{ BD }(1.5-0)+3=0\)

which simplifies to

\(1.5 t_{ BC }+1.5 t_{ BD }+3=0\) (v)

and from Eq. (iv)

\( -0.67(0-1.5)+t_{ BC }(0-1.5)+t_{ BD }(1.5-1.5)=0 \)

whence

\(t_{ BC }=+0.67\)

Hence, from Eq. (v)

\(t_{ BD }=-2.67\)

There are now just two unknown member forces at joint D. Hence, at D

\(x \text { direction : } t_{ DB }\left(x_{ B }-x_{ D }\right)+t_{ DF }\left(x_{ F }-x_{ D }\right)+t_{ DC }\left(x_{ C }-x_{ D }\right)=0\) (vi)

 

\(y \text { direction : } t_{ DB }\left(y_{ B }-y_{ D }\right)+t_{ DF }\left(y_{ F }-y_{ D }\right)+t_{ DC }\left(y_{ C }-y_{ D }\right)-5=0\) (vii)

Substituting values of joint coordinates and the previously calculated value of \( t_{ DB }\left(=t_{ BD }\right) \) in Eqs (vi) and (vii) we obtain, from Eq. (vi)

\(-2.67(0-1.5)+t_{ DF }(3.0-1.5)+t_{ DC }(1.5-1.5)-5=0\)

so that

\(t_{ DF }=-2.67\)

and from Eq. (vii)

\( -2.67(1.5-1.5)+t_{ DF }(1.5-1.5)+t_{ DC }(0-1.5)=0\)

from which

\(t_{ DC }=-3.33\)

The solution then proceeds to joint C to obtain \( t_{ CF } \text { and } t_{ CE }\) or to joint F to determine \( t_{ FC } \text { and } t_{ FF } \); joint F would be preferable since fewer members meet at F than at C. Finally, the remaining unknown tension coefficient \( \left(t_{ EC } \text { or } t_{ EF }\right) \) is found by considering the equilibrium of joint E. Then

\(t_{ FC }=+2.67, \quad t_{ FE }=-2.67, \quad t_{ EC }=0\)

which the reader should verify.
The forces in the truss members are now calculated by multiplying the tension coefficients by the member lengths, i.e.

\(\begin{aligned}&T_{ AB }=t_{ AB } L_{ AB }=-0.67 \times 1.5=-1.0 kN (\text { compression }) \\&T_{ AC }=t_{ AC } L_{ AC }=+2.0 \times 1.5=+3.0 kN (\text { tension }) \\&T_{ BC }=t_{ BC } L_{ BC }\end{aligned}\)

in which

\(L_{ BC }=\sqrt{\left(x_{ B }-x_{ C }\right)^{2}+\left(y_{ B }-y_{ C }\right)^{2}}=\sqrt{(0-1.5)^{2}+(1.5-0)^{2}}=2.12 m\)

Then

\(T_{ BC }=+0.67 \times 2.12=+1.42 kN \text { (tension) }\)

Note that in the calculation of member lengths it is immaterial in which order the joint coordinates occur in the brackets since the brackets are squared. Also

\(T_{ BD }=t_{ BD } L_{ BD }=-2.67 \times 1.5=-4.0 kN (\text { compression })\)

Similarly

\(\begin{aligned}&T_{ DF }=-4.0 kN (\text { compression }) \\&T_{ DC }=-5.0 kN (\text { compression }) \\&T_{ FC }=+5.67 kN (\text { tension }) \\&T_{ FE }=-4.0 kN (\text { compression }) \\&T_{ EC }=0\end{aligned}\)
