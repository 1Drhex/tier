Guest Question.

An ideal gas undergoes four reversible processes known as the Diesel cycle (Fig. 7.23):
1 → 2 adiabatic compression,
2 → 3 isobaric decompression,

3 → 4 adiabatic decompression,
4 → 1 isobaric compression.
The pressure \(p_1\) and the volumes \(V_1, V_2\) and \(V_3\) are given.
1. Determine the pressures \(p_2\) and \(p_4\).
2. Determine the entropy variations \(ΔS_{12}, ΔS_{23}, ΔS_{34}\) and \(ΔS_{41}\) during all four processes and deduce the entropy variation ΔS per cycle.
3. Determine the internal energy variations \(ΔU_{12}, ΔU_{23}, ΔU_{34}\) and \(ΔU_{41}\) during all four processes and deduce the internal energy variation ΔU per cycle.

Admin
Solution

1. Taking into account the property (5.90) of an adiabatic process, the pressure \(p_2\) is written as,

\(p V^γ = const \).

\(p_1 V^γ_1= p_2 V^γ_2\) ⇒ \(P_2 = P_1 \Bigl(\frac{V_1}{V_2} \Bigr)^γ\)

Moreover, an isobaric process is characterised by equal pressures, i.e. \(p_2 = p_3\), and an isochoric process by equal volumes, i.e. \(V_4 = V_1\). Thus, the pressure \(p_4\) is written as,

\(p_3 V^γ_3= p_4 V^γ_4\) ⇒ \(P_4 = P_3\Bigl(\frac{V_3}{V_4} \Bigr)^γ = P_2 \Bigl(\frac{V_3}{V_1} \Bigr)^γ = P_1\Bigl(\frac{V_3}{V_2} \Bigr)^γ\).

2. Using definitions (5.68), (7.12), (7.20) and (7.23), the entropy variations \(ΔS_{12}, ΔS_{23},ΔS_{34}\) and \(ΔS_{41}\) during all four processes are given by,

\(\frac{C_p}{C_V} = \frac{K_p}{K_V} = \frac{c+1}{C} = γ > 1\).

\(\Delta S_{if} = \int_{S_i}^{S_f}{dS} =0\).

\(\Delta S_{if} = \int_{S_i}^{S_f}{dS} =c N R \int_{T_i}^{T_f}{\frac{dT}{T} } =c N R \ln \biggl(\frac{T_f}{T_i} \biggr)\).

\(\Delta S_{if} = \int_{S_i}^{S_f}{dS} =\int_{i}^{f}{\frac{dH}{T} } =(c+1)N R\int_{T_i}^{T_f}{\frac{dT}{T} } =(c+1) N R \ln \Bigl(\frac{T_f}{T_i} \Bigr)\).

\(ΔS_{12} = 0\).

\(ΔS_{23} = \int_{2}^{3}{\frac{dH}{T} } = (c+1) N R \int_{T_2}^{T_3}{\frac{dT}{T} } = (c+1) N R \ln \Bigl(\frac{T_3}{T_2} \Bigr) = (c+1) N R \ln \Bigl(\frac{V_3}{V_2} \Bigr)\).

\(ΔS_{34} = 0\).

\(ΔS_{41} = \int_{4}^{1}{\frac{dU}{T} } = c N R \int_{T_4}^{T_1}{\frac{dT}{T} } = c N R \ln \Bigl(\frac{T_1}{T_4} \Bigr) = c N R \ln \Bigl(\frac{p_1}{p_4} \Bigr)\).

This implies that the entropy variation ΔS per cycle vanishes,

\(ΔS = ΔS_{12} +ΔS_{23} +ΔS_{34} +ΔS_{41} = 0\).

which is a consequence of the fact that the entropy S is a state function.

3. Taking into account the ideal gas equation of state (5.47) and definition (7.10), the internal energy variation \(ΔU_{if}\) during a reversible process i → f is given by,

p V = NR T

\(ΔU_{if} = \int_{U_i}^{U_f}{dU } = c N R\int_{T_i}^{T_f}dT= c N R (T_f – T_i)\).

\(ΔU_{if} = c N R \int_{T_i}^{T_f}{dT } = c N R(T_f − T_i) = c (p_f V_f − p_i V_i)\).

Thus the internal energy variations \(ΔU_{12}, ΔU_{23}, ΔU_{34}\) and \(ΔU_{41}\) during all four processes are given by,

\(ΔU_{12} = c (p_2 V_2 − p_1 V_1) = c p_1 \biggl(\biggl(\frac{V_1}{V_2} \biggr)^\gamma V_2 -V_1\biggr)\).

\(ΔU_{23} = c (p_3 V_3 − p_2 V_2) = c p_1 \biggl(\frac{V_1}{V_2}\biggr)^\gamma (V_3-V_2)\).

\(ΔU_{34} = c (p_4 V_4 − p_3 V_3) = c p_1 \biggl(\biggl(\frac{V_3}{V_2}\biggr)^\gamma V_1 -\biggl(\frac{V_1}{V_2}\biggr)^\gamma V_3\biggr)\).

\(ΔU_{41} = c (p_1 V_1 − p_4 V_4) = c p_1 \biggl( V_1 -\biggl(\frac{V_3}{V_2}\biggr)^\gamma V_1\biggr)\).

This implies that the internal energy variation ΔU per cycle vanishes,

\(ΔU = ΔU_{12} +ΔU_{23} +ΔU_{34} +ΔU_{41} = 0\).

which is a consequence of the fact that the internal energy density U is a state function.
