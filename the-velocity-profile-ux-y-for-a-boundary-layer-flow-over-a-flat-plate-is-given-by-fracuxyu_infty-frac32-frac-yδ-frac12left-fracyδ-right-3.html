Guest Question.

The velocity profile u(x, y) for a boundary layer flow over a flat plate is given by \(\frac{u(x,y)}{u_{\infty }}=\frac{3}{2} \frac{ y}{δ} - \frac{1}{2}\left [ \frac{y}{δ} \right ]^{3}\)
where the boundary layer thickness δ(x) is the function of x and is given by \(δ(x) =\sqrt{\frac{280 vx}{13 u_{\infty }}}\)
(i) Develop an expression for local drag coefficient \(C_{fx}.\)
(ii) Develop an expression for average drag coefficient \(C_f\) over a distance x = L from the leading edge of the plate.

 

Admin
Solution

Given : The velocity profile for the boundary layer
\(\frac{u}{u_{\infty }}=\frac{3}{2} \frac{ y}{δ} – \frac{1}{2}\left ( \frac{y}{δ} \right )^{3}\)

and \(δ(x) =\sqrt{\frac{280 vx}{13 u_{\infty }}}\)

To find :
(i) An expression for local friction coefficient \(C_{fx},\)
(ii) An expression for average friction coefficient \(C_f.\)
Analysis : (i) The shear stress at the wall is expressed as
\(τ_s = µ \left.\begin{matrix} \frac{∂u}{∂y}\end{matrix}\right|_{y=0}…(i)\)
and shear stress in terms of local friction coefficient as
\(τ_s = \frac{C_{fx}}{2} ρ u_∞^2 …(ii)\)
Equating two equations for shear stress at the surface, we get
\(C_{fx} = \frac{2µ}{ ρ u_∞^2 }\left.\begin{matrix} \frac{∂u}{∂y}\end{matrix}\right|_{y=0}\)

For given velocity profile
\(\frac{∂u}{∂y}\left.\begin{matrix}\end{matrix}\right|_{y=0} = u_∞ \left [ \frac{3}{2} × \frac{ 1}{δ} – \frac{1}{2} \cdot \frac{3y^2}{δ^3} \right ]_{y=0} =\frac{3u_∞}{2 δ}\)

Then
\(C_{fx} = \frac{2 µ}{ρ u_∞^2} × \frac{3}{2} × \frac{ u_∞}{δ} = \frac{3v}{u_∞ δ }\)

Introducing the expression for δ(x), we get,
\(C_{fx} = \frac{3v}{u_∞ } × \sqrt{\frac{13 u_{\infty }}{280 vx}} = 0.646 \sqrt{\frac{v}{u_∞ x}}\)

\(= \frac{0.646}{\sqrt{Re_x}}\)

(ii) The average friction coefficient \(C_f\) is given by
\(C_f = \frac{1}{L} \int_{0}^{L} C_{fx} dx = \frac{1}{L} \int_{0}^{L} 0.646\sqrt{\frac{v}{u_∞ x}} dx \)

 

\(=\frac{0.646}{L}\sqrt{\frac{v}{u_∞ }} × \int_{0}^{L} x^{–1/2}dx\)

 

\(= \frac{0.646}{L} \sqrt{\frac{v}{u_∞ } }× \left ( \frac{x^{-1/2 +1}}{-1/2 +1} \right )_{0}^{L}\)

 

\(= 2 \left ( \frac{x^{-1/2 +1}}{-1/2 +1} \right )_{0}^{L}\times 0.646 \sqrt{\frac{v}{u_∞ L}}= \frac{2\times 0.646}{\sqrt{Re_L}}\)

 

\(= \left.\begin{matrix} 2C_{fx} \end{matrix}\right|_{x=L}\)
