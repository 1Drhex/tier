Guest Question.

Evaluating matrix elements of the angular momentum operators
Evaluate the matrix elements \((a)\left\langle j ,m_{j}+1\left|j_{x} \right| j,m_{j} \right\rangle ,(b)\left\langle j ,m_{j}+2\left|j_{x} \right| j,m_{j} \right\rangle\) , and \((c)\left\langle j ,m_{j}+2\left|j^{2}_{x} \right| j,m_{j} \right\rangle\).

Method Because we know the matrix elements of the shift operators, one approach is to express all the operators in the questions in terms of them and then to use eqns 4.24 and 4.29. Note that \(j^{2}_{x}=j_{x}j_{x} and \left\langle j^{\prime },m_{j^{\prime } }\mid j,m_{j} \right\rangle =\delta _{j^{\prime }j }\delta_{m_{j^{\prime } }m_{j^{\bullet } } } \)

Admin
Solution

\(\left\langle j,m_{j}+1\left|j_{+} \right|j,m_{j} \right\rangle =c_{+} (j,m_{j} )\hbar\) (4.24)

\(\left\langle j,m_{j}-1\left|j_{-} \right|j,m_{j} \right\rangle =c_{-} (j,m_{j} )\hbar\)

 

\(c_{+} (j,m_{j} )=\left\{j(j+1)-m_{j}(m_{j}+1 ) \right\} ^{1/2}\) (4.29a)

\(c_{-} (j,m_{j} )=\left\{j(j+1)-m_{j}(m_{j}-1 ) \right\} ^{1/2} \) (4.29b)

 

(a) \(\left\langle j,m_{j}+1\left|j_{x} \right| j,m_{j} \right\rangle =\frac{1}{2} \left\langle j,m_{j}+1\left|j_{+}+j_{-} \right| j,m_{j} \right\rangle\)

 

\(=\frac{1}{2} \left\langle j,m_{j}+1\left|j_{+} \right| j,m_{j} \right\rangle+\frac{1}{2} \left\langle j,m_{j}+1\left|j_{-} \right| j,m_{j} \right\rangle\)

 

\(=\frac{1}{2}c_{+}(j,m_{j} ) \hbar \)

because \( \left\langle j,m_{j}+1\left|j_{-} \right| j,m_{j} \right\rangle \propto \left\langle j,m_{j}+1\mid j,m_{j} -1\right\rangle=0\)

 

(b) \( \left\langle j,m_{j}+2\left|j_{x} \right| j,m_{j} \right\rangle =0\)

because \(j_{\pm }\) steps \(m_{j}\) only by one unit, and the resulting states are orthogonal to the state \(\left|j,m_{j}+2\right\rangle \).

(c) \( \left\langle j,m_{j}+2\left|j^{2} _{x} \right| j,m_{j} \right\rangle = \frac{1}{4} \left\langle j,m_{j}+2\left|j^{2} _{+}+j^{2} _{-} +j _{+}j _{-} +j _{-}j _{+}\right| j,m_{j} \right\rangle\)

 

\(= \frac{1}{4} \left\langle j,m_{j}+2\left|j^{2} _{+} \right| j,m_{j} \right\rangle = \frac{1}{4} c_{+}(j,m_{j}+1 )c_{+} (j,m_{j} )\hbar ^{2}\)

 

\(=\frac{1}{4} \left\{j(j+1)-(m_{j}+1 )(m_{j}+2 )\right\} ^{1/2} \left\{j(j+1)-m_{j}(m_{j}+1 ) \right\} ^{1/2} \hbar ^{2} \)

Comment Note that it is quite easy to spot short-cuts, as in (c), where it should be obvious that only \(j^{2}_{+}\) can contribute to the matrix element.
