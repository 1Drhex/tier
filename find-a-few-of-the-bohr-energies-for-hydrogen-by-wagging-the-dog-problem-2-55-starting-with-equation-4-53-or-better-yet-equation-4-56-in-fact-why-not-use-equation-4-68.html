Guest Question.

Find a few of the Bohr energies for hydrogen by “wagging the dog” (), starting with Equation 4.53—or, better yet, Equation 4.56; in fact, why not use Equation 4.68 to set \( ρ_0 = 2n \) , and tweak n? We know that the correct solutions occur when n is a positive integer, so you might start with n = 0.9 , 1.9 , 2.9 , etc. and increase it in small increments—the tail should wag when you pass 1, 2, 3, …. Find the lowest three ns, to four significant digits, first for \( \ell = 0 \), and then for \( \ell = 1 \) and \( \ell = 2 \) Warning: Mathematica doesn’t like to divide by zero, so you might change ρ to \( (\rho+0.000001) \) in the denominator. Note: \( u(0)=0 \text { in all cases, but } u^{\prime}(0)=0 \text { only for } \ell \geq 1 \) (Equation 4.59). So for \( \ell = 0 \) you can use \( u(0)=0, u^{\prime}(0)=1 . \text { For } \ell>0 \) you might be tempted to use \( u(0)=0 \text { and } u^{\prime}(0)=0 \), but Mathematica is lazy, and will go for the trivial solution \( u(\rho) \equiv 0 \); better, therefore, to use (say) \( u(1)=1 \text { and } u^{\prime}(0)=0 \).

\( -\frac{\hbar^{2}}{2 m_{e}} \frac{d^{2} u}{d r^{2}}+\left[-\frac{e^{2}}{4 \pi \epsilon_{0}} \frac{1}{r}+\frac{\hbar^{2}}{2 m_{e}} \frac{\ell(\ell+1)}{r^{2}}\right] u=E u \) (4.53).

\( \frac{d^{2} u}{d \rho^{2}}=\left[1-\frac{\rho_{0}}{\rho}+\frac{\ell(\ell+1)}{\rho^{2}}\right] u \) (4.56).

\( \rho_{0}=2 n \) (4.68).

\( u(\rho) \sim C \rho^{\ell+1} \) (4.59).
