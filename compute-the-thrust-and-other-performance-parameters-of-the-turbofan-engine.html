Guest Question.

compute the thrust and other performance parameters of the turbofan engine.

Admin
Solution

The following pertinent information is repeated here for convenience:
\(\dot{m}_{a}\) = 50 lbm/sec \(\dot{m'}_{a}\) = 150 lbm/sec
f = 0.0191 HV = 18,900 Btu/lbm
\(V_{0}\) = 882 ft/sec \(p_{0}\) = 546 psfa \(T_{0}=400^{\circ}R\)
\(V_{4^{'}}\) = 1113 ft/sec \(p_{4^{'}}\) = 1157 psfa \(T_{4^{'}}=516^{\circ}R\)
\(V_{6}\) = 1686 ft/sec \(p_{6}\) = 778 psfa \(T_{6}\) = \(1183^{\circ}R\)

We now compute the exit densities and areas.

\(\rho _{4^{'}}=\frac{p_{4^{'}}}{RT_{4^{'}}}\) = \(\frac{1157}{(53.3)(516)}\) = 0.0421 lbm/\(ft^{3}\)

\(A_{4^{'}}=\frac{\dot{m}_{a}}{\rho _{4^{'}}V_{4^{'}}}\) = \(\frac{150}{(0.0421)(1113)}\) = 3.20 \(ft^{2}\)

\(\rho _{6}=\frac{p_{6}}{RT_{6}}=\frac{778}{(53.3)(1183)}\) = 0.01234 lbm/\(ft^{3}\)

\(A_{6}=\frac{\dot{m}_{a}}{\rho _{6}V_{6}}=\frac{50}{(0.01234)(1686)}\) = 2.40 \(ft^{2}\)

Note that to calculate the net propulsive thrust, we must include contributions from both the primary jet and the fan.

\(F_{net}=\frac{\dot{m}_{a}V_{6}}{g_{c}}+A_{6}(p_{6}-p_{0})+A_{4^{'}}(p_{4^{'}}-p_{0})-(\dot{m}_{a}+\dot{m'}_{a})\frac{V_{0}}{g_{c}}\)

 

= \(\frac{(50)(1686)}{32.2}+(2.40)(778-546)+\frac{(150)(1113)}{32.2}+(3.20)(1157-546)-(50+150)\frac{882}{32.2}\)

\(F_{net}\) = 4840 lbf

The thrust horsepower is [by (12.47)]

\(P_{T}=F_{n}V_{0}=\frac{(4840)(882)}{550}\) = 7760 hp

The input horsepower is [by (12.45)]

\(P_{I}=\dot{m}_{f}(HV)=\dot{m}_{a}(f)(HV)\) = \(\frac{(50)(0.0191)(18,900)(778)}{550}\) = 25,530 hp

The overall efficiency is [by (12.52)]

\(\eta _{0}=\frac{P_{T}}{P}=\frac{7760}{25,530}\) = 30.4%

Thrust-specific fuel consumption is [by (12.62)]

\(tsfc=\frac{\dot{m}_{f}(3600)}{F_{n}}\) = \(\frac{(50)(0.0191)(3600)}{4840}\) = 0.71\(\frac{lbm}{lbf-hr}\)

This specific fuel consumption is slightly low, even for a fanjet engine. Had we simply changed to a higher value of specific heat in the hot sections (turbine and turbine nozzle), two effects would result:

1. The fuel–air ratio would increase because the enthalpy entering the turbine would increase.

2. The thrust would rise due to an increased exhaust velocity and exit pressure.

The increase in thrust would be small compared with the increase in fuel–air ratio, and the net effect would be to raise the tsfc to about 0.8.

 

equation (12.47):\(P_{T}=F_{net}V_{0}\)

equation (12.45):\(P_{I}=\dot{m}_{f}(HV)\)

equation (12.52):\(\eta _{0}\equiv \frac{P_{T}}{P_{I}}=\eta _{th}\eta _{p}\)

equation (12.62): \(tsfc=\frac{\dot{m}_{f}(3600)}{F_{net}}\)
