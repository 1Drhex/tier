Guest Question.

A double slider-crank mechanism is shown in Fig.3.22(a). Crank 2 rotates at constant angular speed \( \omega_{2}=10 rad / s \text {. Determine the velocity and acceleration of each slider. } O_{2} A \) = 1000 mm, AB = 200 mm, and AC = 200 mm.

Admin
Solution

Velocity diagram
1. Draw the configuration diagram shown in Fig.3.22(a) to a scale of 1 cm=25 mm.

\( \text { 2. } \omega_{2}=10 rad / s , v_{a}=\omega_{2} \times O_{2} A=10 \times 100=1000 mm / s \).

3. Draw the velocity diagram as shown in Fig.3.22(b) to a scale of 1 cm = 200 mm/s.

\( \text { 4. Draw } v_{a}=o_{2 a} \perp O_{2} A, o_{2 a}=5 cm \).

\( \text { 5. Draw } a b \perp A B \text { and } a c \perp A C, o_{2} c b \| o_{2} c \text { to meet } a b \text { and } a c \text { lines at } b \text { and } c \text { respectively, } \)

6. Measure

\( v_{c}=o_{2} c=3.1 cm =620 mm / s \).

\( v_{b}=o_{2} b=5.2 cm =1040 mm / s \).

\( v_{b a}=a b=2.2 cm =440 mm / s \).

\( v_{c a}=a c=2.8 cm =560 mm / s \).

Acceleration diagram

\( f_{o 2 a}^{n}=\frac{v_{a}^{2}}{O_{2} A}=\frac{1000^{2}}{100}=10,000 mm / s ^{2} \text { along } A O_{2} \).

\( f_{b a}^{n}=\frac{v_{b a}^{2}}{A B}=\frac{440}{200}=968 mm / s ^{2} \text { along } A B \).

\( f_{c a}^{n}=\frac{v_{c a}^{2}}{A C}=\frac{500^{2}}{200}=1568 mm / s ^{2} \text { along } A C \).

1. Draw the acceleration diagram as shown in Fig.3.22(c) to a scale of 1 cm =2000.

\( \text { 2. Draw } f_{o 2 a}^{n}=f_{a}^{n}=o_{2} a \| O_{2} A, o_{2} a=5 cm \).

\( f_{b a}^{n}=a b \| A B, a b^{\prime}=0.484 cm \).

\( b^{\prime} b \perp A B \text { and } f_{o 2 b}^{n}=o_{2} b \| O_{2} C \text { to meet } b^{\prime} b \text { at point } b \).

\( f_{b}=o_{2} b=1.4 \times 200=2800 mm / s ^{2} \).

\( \text { 3. Draw } f_{c a}=a c^{\prime} \| A C, a c^{\prime}=0.784 cm \).

\( c^{\prime} c \perp A C \text { and } o_{2} c \| O_{2} C \text { to meet at } c \).

\( \text { Then } f_{c}=o_{2} c=3.7 cm =7400 mm / s ^{2} \).
