Guest Question.

Finding the small-signal parameters of an NMOS amplifier The amplifier in Fig. 7.16(a) has \( V_{ GS }=2 V , V_{ DD }=15 V , \text { and } R_{ D }=3.5 k \Omega \). The NMOS parameters are \( V_{ t }=1 V , K_{ n }=3.25 mA / V ^{2} \) , and \( V_{ M }=1 / \lambda=100 V \).

(a) Find the DC biasing point \( V_{ GS }, I_{ D }, \text { and } V_{ DS } \).

(b) Find the small-signal transistor model parameters \( r_{ o } \text { and } g_{ m } .\)

(c) Find the small-signal amplifier parameters \( R_{ i }, R_{ o }, \text { and } A_{ vo }.\)

(d) Use PSpice to plot the small-signal AC output voltage for 1-mV sinusoidal input signal at 1 kHz. The NMOS parameters are KP = 6.5 M, VTO = 1 V, L = 1 U, W = 1 U, and LAMBDA 0.01. Note: PSpice uses \( K_{ p }=K_{ m }=2 K_{ n } .\)

Admin
Solution

(a) \( K_{ m }=K_{ p }=6.5 \times 10^{-3} \text { for } W=L \text {. } \) From Eq. (7.8),

 

\( i_{ D }=K_{ n }\left(v_{ GS }-V_{ t }\right)^{2} \) (7.8)

 

\(\begin{aligned}I_{ D } &=K_{ n }\left(V_{ GS }-V_{ t }\right)^{2}=3.25 \times 10^{-3}(2-1)=3.25 mA \\V_{ DS } &=V_{ DD }-R_{ D } I_{ D }=15-3.5 \times 10^{3} \times 3.25 \times 10^{-3}=3.625 V\end{aligned}\)

 

(b) \( r_{ o }=V_{ M } / I_{ D }=100 /\left(3.25 \times 10^{-3}\right)=30.77 K \Omega \)

 

\( g_{ m }=2 \times K_{ n }\left(V_{ GS }-V_{ t }\right)=2 \times 3.25 \times 10^{-3} \times(2-1)=6.5 mA / V \)

 

(c) \( R_{ o }=r_{ o }\left\|R_{ D }=30.77 k \right\| 3.5 k =3.143 k \Omega \)

 

\(\begin{aligned}G_{ mo } &=g_{ m }=6.5 m A / V \\A_{ vo } &=-g_{ m } \times R_{ o }=-6.5 mA / V \times 3.143 k \Omega=-20.426 V / V\end{aligned}\)

 

(d) Figure 7.21 shows the PSpice schematic and the PSpice plot for small-signal output voltage is shown in Fig. 7.22. The capacitor \( C _{2} \) blocks the DC and passes the small-signal output, which shows a voltage of -19.84; this is close to the calculated value of -20.42.

 

NOTE: All PSpice results given here are from running the simulation with the schematic (.SCH) files. If you run the simulation with the netlist circuit (.CIR) files, you may get different results because the student’s version of PSpice has a limited number of active devices and models.
