Guest Question.

The loading period for a building extended from Feb., 1957 to Feb., 1959. In Feb., 1962, the average measured settlement was found to be 117 mm. The ultimate settlement was expected to be 360 mm. Estimate the settlement in Feb., 1967, assuming double drainage to occur. What would be this result if the measured settlement in Feb., 1962 was 153 mm instead of 117 mm?

Admin
Solution

The reckoning of time is conventionally done from mid-way through the construction or loading period. In this case,

\(S_{4}=117 mm \text { when } t=4 \text { years. } S_{c}=360 mm .\)

The settlement is required at time t = 9 years.
Let us assume, in the first instance, that at t = 9 years,
U, the degree of consolidation is less than 50%.

\(\text { In such a case, } U=1.13 \sqrt{T_{ \nu }}, \text { where } T_{ \nu }\) is the Time-factor.

\(\frac{S_{4}}{S_{9}}=\frac{U_{4}}{U_{9}} \frac{U_{4}}{U_{9}}=\sqrt{\frac{T_{\nu _{4}}}{T_{\nu _{9}}}}\)

∴ \(\frac{S_{4}}{S_{9}}=\sqrt{\frac{t_{4}}{t_{9}}}, \text { since } \frac{C_{\nu }}{H^{2}} \text { is a constant }\).

∴ \(\frac{117}{S_{9}}=\sqrt{\frac{4}{9}}=2 / 3\)

∴ \(S_{9}=(3 / 2) \times 117=175.5 mm\)

Thus, \(T_{\nu _{9}}=\frac{S_{9}}{S_{c}}=\frac{175.5}{360.0}<50 \%\) .Hence the relationships used are valid.

\(\text { If } S_{4}=153 mm \text { at } t=4\) years,

\(U_{4}=153 / 360=42.5 \%\)

For double drainage,
this corresponds \(T_{ \nu }=0.142, \text { since } T_{ \nu }=(\pi / 4) U^{2}\)

Then \(\frac{C_{\upsilon } \times 4}{H^{2}}=0.142, \text { or } \frac{C_{\upsilon }}{H^{2}}=\frac{0.142}{4}=0.0355\)

For t = 9 years,\(T_{\nu _{9}}=0.0355 \times 9=0.3195 \approx 0.32\)

Correspondingly \(U_{9}=0.632\)

Hence the settlement in Feb., 1967 = 0.632 × 360 = 227.5 mm.
