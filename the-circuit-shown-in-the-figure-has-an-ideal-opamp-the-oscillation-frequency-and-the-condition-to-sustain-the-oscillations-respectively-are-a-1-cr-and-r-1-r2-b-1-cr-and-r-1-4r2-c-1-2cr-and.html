Guest Question.

The circuit shown in the figure has an ideal opamp. The oscillation frequency and the condition to sustain the oscillations, respectively are

(A) \(\frac{1}{C R} \text { and } R _{1}= R _{2}\)

(B) \(\frac{1}{C R} \text { and } R _{1}=4 R _{2}\)

(C) \(\frac{1}{2 C R} \text { and } R_{1}=R_{2}\)

(D) \(\frac{1}{2 C R} \text { and } R _{1}=4 R _{2}\)

Admin
Solution

The given circuit is Wein Bridge Oscillator

The gain of the op–amp is \(A=I+\frac{R_{f}}{R_{3}}\) and feedback

factor \(B=\frac{V_{f}}{V_{o}}\)

Let \(Z_{1}=R_{1}+1 / S C_{1} \quad \text { and } \quad Z_{2}=R_{2} \| 1 / S C_{2}\)

 

\(V_{f}=\frac{ Z _{2} \times V _{0}}{ Z _{1}+ Z _{2}} \quad \because B=\frac{Z_{2}}{Z_{1}+Z_{2}}\)

Sub all values mB we get finally

\(B=\frac{ j \omega R _{2} C _{1}}{1+ j \omega\left( R _{1} C _{1}+ R _{2} C _{2}+ R _{2} C _{1}\right)-\omega_{2} R_{1} R_{2} C_{1} C_{2}}\)

But B is real quantity so equate

\(1-\omega_{2} R_{1} R_{2} C_{1} C_{2}=0\)

Thus the frequency of oscillation,

\(\omega_{0}=\frac{1}{\sqrt{R_{1} R_{2} C_{1} C_{2}}} \omega_{0}=\frac{1}{2 R C}\)

From the given data

\(\begin{aligned}&R_{1}=2 R, R_{2}=R \\&C_{1}=C, C_{2}=2 C\end{aligned}\)

And \(B=\frac{R_{2} C_{1}}{R_{1} C_{1}+R_{2} C_{2}+R_{2} C_{1}}\)

Substitute all values

\(B=\frac{R C}{2 R C+2 R C+R C}=\frac{1}{5}\)

For sustained oscillation AB = 1

\(A=\frac{1}{B} \quad \therefore 1+\frac{R_{f}}{R_{3}}=5\)

But \(R_{f}=R_{1} \text { and } R_{3}=R_{2}\)

 

\(\therefore \quad R_{1}=4 R_{2}\)

Hence, the correct option is (D).
