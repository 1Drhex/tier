Guest Question.

Cracking of Methane

Steam reforming of methane can be used to make hydrogen gas. \(CO \text { and } CO _{2}\) are observed as by-products. Using a 4:1 \(H _{2} O \text { to } CH _{4}\) inlet ratio and a pressure of 1 bar, calculate the equilibrium composition obtained in the temperature range of 600–1100 K.

Admin
Solution

We first apply the phase rule to determine the number of independent reactions we must specify. For each calculation, we have five species, one phase, and a specific temperature and pressure. We have also specified the inlet feed ratio. Thus, the elements in the system form the stoichiometric constraint; the ratios must be identical to the ratios they have in the feed. The ratio of elemental O:H is constrained to 1:3; similarly, the ratio of C:H is constrained to 1:12. Hence, we have two additional stoichiometric constraints, s. The number of independent reactions is now given by:

 

\(R=m-\Im+2-\pi-s=5-2+2-1-2=2\)

 

For a set of two independent reactions, we can choose:

 

\(CH _{4}+ H _{2} O \rightleftarrows CO +3 H _{2} \quad \text { reaction } 1\) (E9.19A)

 

\(CH _{4}+2 H _{2} O \rightleftarrows CO _{2}+4 H _{2} \quad \text { reaction } 2\) (E9.19B)

 

Can you do this problem with a different reaction set?

According to Equations (9.41) through (9.43), we get:

 

\(n_{i}=n_{i}^{o}+\sum_{k=1}^{R} v_{k i} \xi_{k}\) (9.41)

 

\(y_{i}=\frac{n_{i}}{n^{v}}=\frac{n_{i}^{o}+\sum_{k=1}^{R} v_{k i} \xi_{k}}{n^{o}+\sum_{k=1}^{R} v_{k} \xi_{k}}\) (9.43)

 

\(\begin{aligned}n_{ CH _{4}} &=1-\xi_{1}-\xi_{2} \\n_{ H _{2} O } &=4-\xi_{1}-2 \xi_{2} \\n_{ H _{2}} &=3 \xi_{1}+4 \xi_{2} \\n_{ CO } &=\xi_{1} \\n_{ CO _{2}} &=\xi_{2} \\\hline n^{v} &=5+2 \xi_{1}+2 \xi_{2}\end{aligned}\) \(\begin{aligned}y_{ CH _{4}} &=\frac{1-\xi_{1}-\xi_{2}}{5+2 \xi_{1}+2 \xi_{2}} \\y_{ H _{2} O } &=\frac{4-\xi_{1}-2 \xi_{2}}{5+2 \xi_{1}-2 \xi_{2}} \\y_{ H _{2}} &=\frac{3 \xi_{1}+4 \xi_{2}}{5+2 \xi_{1}+2 \xi_{2}} \\y_{ CO } &=\frac{\xi_{1}}{5+2 \xi_{1}+2 \xi_{2}} \\y_{ CO _{2}} &=\frac{\xi_{2}}{5+2 \xi_{1}+2 \xi_{2}}\end{aligned}\)

 

At 1 bar, the equilibrium constants can be written assuming ideal gas conditions:

 

\(K_{1}=\frac{y_{ CO } y_{ H _{2}}^{3}}{y_{ CH _{4}} y _{ H _{2} O }} P^{2}=\frac{\left(\xi_{1}\right)\left(3 \xi_{1}+4 \xi_{2}\right)^{3}}{\left(5+2 \xi_{1}+2 \xi_{2}\right)^{2}\left(1-\xi_{1}-\xi_{2}\right)\left(4-\xi_{1}-2 \xi_{2}\right)} p^{2}\) (E9.19C)

 

\(K_{2}=\frac{y_{ CO _{2}} y_{ H _{2}}^{4}}{y_{ CH _{4}} y_{ H _{2} O }^{2}} P^{2}=\frac{\left(\xi_{2}\right)\left(3 \xi_{1}+4 \xi_{2}\right)^{4}}{\left(5+2 \xi_{1}+2 \xi_{2}\right)^{2}\left(1-\xi_{1}-\xi_{2}\right)\left(4-\xi_{1}-2 \xi_{2}\right)^{2}} p^{2}\) (E9.19D)

 

To solve for the equilibrium constants at the elevated temperatures, we need appropriate thermochemical data. Gibbs energy and enthalpy of reaction, as well as heat capacity data available in Appendices A.2 and A.3 are summarized in Table E9.19A. From these data, we can solve for \(K_{1} \text { and } K_{2}\) at different temperatures using Equation (9.24). This solution is conveniently done in a spreadsheet.

 

\(\ln \left(\frac{K_{T}}{K_{298}}\right)=\left\{\begin{array}{l}{\begin{array}{l}[-\frac{\Delta h_{ rxn , 298}^{o}}{R}+\Delta A(298)+\frac{\Delta B}{2}\left(298^{2}\right)+\frac{\Delta C}{3}\left(298^{3}\right) \\\left.-\frac{\Delta D}{298}+\frac{\Delta E}{4}\left(298^{4}\right)\right]\left(\frac{1}{T}-\frac{1}{298}\right) \\+\Delta A \ln \left(\frac{T}{298}\right)+\frac{\Delta B}{2}(T-298)+\frac{\Delta C}{6}\left(T^{2}-298^{2}\right) \\+\frac{\Delta D}{2}\left(\frac{1}{T^{2}}-\frac{1}{298^{2}}\right)+\frac{\Delta E}{12}\left(T^{3}-298^{3}\right)\end{array}}\end{array}\right\}\) (9.24)
