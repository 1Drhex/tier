Guest Question.

Let {g[n]} = {g[n]} = {5, 2, 4, −1, 0, 0} and h[n] = {−3, 4, 0, 2, −1, 2}, two sequences of finite length.

a. Compute the circular convolution \(y_{C}[n] = g_{e}[n] * _{6}h[n].\)
b. Compute the circular convolution using the DFT.

Admin
Solution

The circular convolution \(y_{C}[n] = g_{e}[n]*_{6}h[n]\) is: \(y_c[n]=\sum\limits_{k=0}^{5}{g_e[k]h[((n-k))_6]}\)

where {\(g_{e}[n]\)} = {5, 2, 4, −1, 0, 0}.
Thus,

\(y_{C}[0] = g_{e} [0]h[0] + g_{e}[1]h[5] + g_{e} [2]h[4] + g_{e} [3]h[3] ++g_{e}[4]h[2]+g_{e}[5]h[1]=g[0]h[0]+g[1]h[5]+g[2]h[4]+g[3]h[3]\)
\(=5\times (-3)+2\times 2+4\times (-1)+(-1)\times 2=-17\)
\(y_{C}[1]=g_{e}[0]h[1]+g_{e}[1]h[0]+g_{e}[2]h[5]+g_{e}[3]h[4]+g_{e}[4]h[3]+g_{e}[5]h[2]=g[0]h[1]+g[1]h[0]+g[2]h[5]+g[3]h[4]\)
\(=5\times 4+2\times (-3)+4\times 2+(-1)\times (-1)=23\)
\(y_{C}[2]=g_{e}[0]h[2]+g_{e}[1]h[1]+g_{e}[2]h[0]+g_{e}[3]h[5]+g_{e}[4]h[4]+g_{e}[5]h[3]=g[0]h[2]+g[1]h[1]+g[2]h[0]+g[3]h[5]\)
\(=5\times 0+2\times 4+4\times (-3)+(-1)\times 2=-6\)
\(y_{C}[3]=g_{e}[0]h[3]+g_{e}[1]h[2]+g_{e}[2]h[1]+g_{e}[3]h[0]+g_{e}[4]h[5]+g_{e}[5]h[4]=g[0]h[3]+g[1]h[2]+g[2]h[1]+g[3]h[0]\)
\(=5\times 2+2\times 0+4\times 4+(-1)\times (-3)=29\)
\(y_{C}[4]=g_{e}[0]h[4]+g_{e}[1]h[3]+g_{e}[2]h[2]+g_{e}[3]h[1]+g_{e}[4]h[0]+g_{e}[5]h[5]=g[0]h[4]+g[1]h[3]+g[2]h[2]+g[3]h[1]\)
\(=5\times (-1)+2\times2+4\times0+(-1)\times4=-5\)
\(y_{C}[5]=g_{e}[0]h[5]+g_{e}[1]h[4]+g_{e}[2]h[3]+g_{e}[3]h[2]+g_{e}[4]h[1]+g_{e}[5]h[0]=g[0]h[5]+g[1]h[4]+g[2]h[3]+g[3]h[2]\)
\(=5\times2+2\times(-1)+4\times2+(-1)\times0=16\)

b. Next, the two DFTs (length 6) are derived:

\(G_{e}[k]=\sum\limits_{n=0}^{5}{g_{e}[n]e^{-j\frac{2\pi kn}{6} }}\) and \(H[k]=\sum\limits_{n=0}^{5}{h[n]e^{-j\frac{2\pi kn}{6} }}\) resulting to:
{\(G_{e}[k]\)} = {10, 5 − j5.1962, 1 + − j1.7322, 8, 1 − j1.7322, 5 + j5.1962} and {H[k]} = {4,−1.5 − j2.5981,−3.5 − j0.866,−12,−3.5 + j0.866,−1.5 + j2.5981}

Thus,
{\(G_{e}[k]H[k]\)}={40,-21-j5.1962,-2-j6.9282,-96,-2+j6.9282,-21+j5.1962}
The circular convolution can be computed by the inverse DFT of the product above, i.e., \(y_{c}[n]=\frac{1}{6}\sum\limits_{k=0}^{5}{G_{e}[k]H[k]e^{j\frac{2\pi kn}{6}}}\) providing the following result:
{\(y_{C}[n]\)} = {−17, 23, −6, 29, −5, 16}, which is the same with the one estimated in (a).
