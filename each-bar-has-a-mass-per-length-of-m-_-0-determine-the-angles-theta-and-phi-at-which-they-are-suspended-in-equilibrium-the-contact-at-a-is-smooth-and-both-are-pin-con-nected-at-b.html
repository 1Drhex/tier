Guest Question.

Each bar has a mass per length of \({ m }_{ 0 }\). Determine the angles \(\theta\) and \(\phi\) at which they are suspended in equilibrium. The contact at \(A\) is smooth, and both are pin con-nected at \(B\).

Admin
Solution
\(\theta +\phi =a\tan { \left( \frac { 1 }{ 2 } \right) }\)

 

\(V=-\frac { 3l }{ 2 } { m }_{ 0 }\left( \frac { 3l }{ 4 } \right) \cos { \left( \theta \right) } -l{ m }_{ 0 }\left( \frac { l }{ 2 } \right) \cos { \left( \phi \right) } -\frac { 1 }{ 2 } { m }_{ 0 }\left( l\cos { \left( \phi \right) } +\frac { l }{ 4 } \sin { \left( \phi \right) } \right)\)

 

\(\frac { d }{ d\theta } V=\frac { 9{ m }_{ 0 }{ l }^{ 2 } }{ 8 } \sin { \left( \theta \right) } -{ m }_{ 0 }{ l }^{ 2 }\sin { \left( \phi \right) } +\frac { { m }_{ 0 }{ l }^{ 2 } }{ 8 } \cos { \left( \phi \right) } =0\)

 

Guess \(\quad \quad \theta =10deg\quad \quad \quad\quad\quad \phi =10deg\) Given \(\quad \quad \theta +\phi =a\tan { \left( \frac { 1 }{ 2 } \right) } \quad \quad \frac { 9 }{ 8 } \sin { \left( \theta \right) } -\sin { \left( \phi \right) } +\frac { 1 }{ 8 } \cos { \left( \phi \right) } \) \(\left( \begin{matrix} \theta \\ \phi \end{matrix} \right) =Find\left( \theta ,\phi \right) \quad \quad \quad\quad\quad \left( \begin{matrix} \theta \\ \phi \end{matrix} \right) =\left( \begin{matrix} 9.18 \\ 17.38 \end{matrix} \right) deg\)
