Guest Question.

Consider a transfer function:

\(\frac{Y(s)}{U(s)}=\frac{d}{b s+c}\)

(a) What is the steady-state gain?

(b) For a step change of magnitude M in the input, will the output response be bounded for all values of constants b, c and d? Briefly justify your answer.

Admin
Solution

\(\frac{Y(s)}{U(s)}=\frac{d}{b s+c}\)

a) Gain K can be obtained by setting s = 0

\(K=\frac{d}{c}\)

Alternatively, the transfer function can be placed in the standard gain/time constant form by dividing the numerator and denominator by c:

\(\frac{Y(s)}{U(s)}=\frac{K}{\tau s+1}, \text { where } K=\frac{d}{c} \text { and } \tau=\frac{b}{c} \text {. }\)

b) In order to determine the boundedness of the output response, consider a step input of magnitude M. Then use U(s) = M/s and

\(Y(s)=\frac{K}{\tau s+1} \frac{M}{s}\)

From Table 3.1, the step response is:

\(y(t)=K M\left(1- e ^{-t / \tau}\right)\)

By inspection, this response will be bounded only if τ > 0, or equivalently, only if b/c > 0.
