Guest Question.

Use Tables 10-1 and 10-2 to design a double-angle connection for a W36×231 beam to an HSS14×14×1/2 column. Use 3/4-in. diameter ASTM A325-N bolts in standard holes.
\(\begin{array}{l}P_{D}=37.5 \text { kips } \\P_{L}=113 \text { kips }\end{array}\)

Admin
Solution

Material Properties:

Beam ASTM A992 \(F_{y}=50 ksi \quad F_{u}=65 ksi\)

Column ASTM A500 Gr. \(F_{y}=46 ksi \quad F_{u}=58 ksi\)

Angles ASTM A36 \(F_{y}=36 ksi \quad F_{u}=58 ksi\)

Manual
Table 2-3

Geometric Properties:

Beam W36×231 \(t_{w}=0.760 \text { in. }\)

Column HSS14×14×1/2 t = 0.465 in. B = 14.0 in.

Manual
Tables 1-1
and 1-12

Compute the required strength

LRFD ASD
\(\begin{aligned}R_{u} &=1.2(37.5 kips )+1.6(113 kips ) \\&=225 kips\end{aligned}\) \(\begin{aligned}R_{a} &=37.5 kips +113 kips \\&=151 kips\end{aligned}\)

Design bolts and welds

Try 8 rows of bolts and 5/16-in. welds B

Obtain the bolt group and angle available strengths from Table 10-1

LRFD ASD
\(\phi R_{n}=254 \text { kips }>225 \text { kips o.k. }\) \(R_{n} / \Omega=170 \text { kips }>151 \text { kips } \quad \text { o.k. }\)

Manual
Table 10-1

Obtain the available weld strength from Table 10-2

LRFD ASD
\(\phi R_{n}=279 \text { kips }>225 \text { kips o.k. }\) \(R_{n} / \Omega=186 \text { kips }>151 \text { kips } \quad \text { o.k. }\)

Manual
Table 10-2

Check the minimum support thickness

Minimum support thickness = 0.238 in. < 0.465 in. o.k.

Manual
Table 10-2

Calculate the minimum angle thickness

\(t_{\min }=w+1 / 16=5 / 16+1 / 16=3 / 8 \text { in. }\)

Section
J2.2b

Use 3/8-in. angle thickness to accommodate the welded legs of the double angle connection.

\(\text { Use } 2 L 4 \times 31 / 2 \times 3 / 8 \times 1^{\prime}-11 1 / 2 " \text { . }\)

L = 23.5 in. > T/2 o.k.

B of the HSS is greater than 12 in., the minimum acceptable width for 4-in. OSL angles.

Calculate the available beam web strength

LRFD ASD
\(\begin{array}{l}\phi R_{n}=(702 kips / in .)(0.760 in. )=534 kips \\534 \text { kips }>225 \text { kips o.k. }\end{array}\) \(\begin{array}{l}R_{n} / \Omega=(468 \text { kips } / \text { in. })(0.760 in. )=356 kips \\356 \text { kips }>151 \text { kips } \quad \text { o.k. }\end{array}\)
