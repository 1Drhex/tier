Guest Question.

The steel step shaft shown in Fig. 4–7a is mounted in bearings at A and F. A pulley is centered at C where a total radial force of 600 lbf is applied. Using singularity functions evaluate the shaft displacements at \(\frac{1}{2}\)-in increments. Assume the shaft is simply supported.

Admin
Solution

The reactions are found to be \(R_{1}=360 lbf \text { and } R_{2}=240 lbf \text { . }\) Ignoring \(R_{2}\), using singularity functions, the moment equation is

 

\(M=360 x-600\langle x-8\rangle^{1}\) (1)

 

This is plotted in Fig. 4–7b.
For simplification, we will consider only the step at D. That is, we will assume section AB has the same diameter as BC and section EF has the same diameter as DE. Since these sections are short and at the supports, the size reduction will not add much to the deformation. We will examine this simplification later. The second area moments for BC and DE are

 

\(I_{B C}=\frac{\pi}{64} 1.5^{4}=0.2485 in ^{4} \quad I_{D E}=\frac{\pi}{64} 1.75^{4}=0.4604 in ^{4}\)

 

A plot of M/I is shown in Fig. 4–7c. The values at points b and c, and the step change are

 

\(\left(\frac{M}{I}\right)_{b}=\frac{2760}{0.2485}=11106.6 lbf / in ^{3} \quad\left(\frac{M}{I}\right)_{c}=\frac{2760}{0.4604}=5994.8 lbf / in ^{3}\)

 

\(\Delta\left(\frac{M}{I}\right)=5994.8-11106.6=-5111.8 lbf / in ^{3}\)

 

The slopes for ab and cd, and the change are

 

\(m_{a b}=\frac{2760-2880}{0.2485(0.5)}=-965.8 lbf / in ^{4} \quad m_{c d}=\frac{-5994.8}{11.5}=-521.3 lbf / in ^{4}\)

 

\(\Delta m=-521.3-(-965.8)=444.5 lbf / in ^{4}\)

 

Dividing Eq. (1) by \(I_{B C}\) and, at x = 8.5 in, adding a step of \(-5111.8 lbf / in ^{3}\) and a ramp of slope \(444.5 lbf / in ^{4}\), gives

 

\(\frac{M}{I}=1448.7 x-2414.5\langle x-8\rangle^{1}-5111.8\langle x-8.5\rangle^{0}+444.5\langle x-8.5\rangle^{1}\) (2)

 

Integration gives

 

\(E \frac{d y}{d x}=724.35 x^{2}-1207.3\langle x-8\rangle^{2}-5111.8\langle x-8.5\rangle^{1}+222.3\langle x-8.5\rangle^{2}+C_{1}\) (3)

 

Integrating again yields

 

\(E y=241.5 x^{3}-402.4\langle x-8\rangle^{3}-2555.9\langle x-8.5\rangle^{2}+74.08\langle x-8.5\rangle^{3}+C_{1} x+C_{2}\) (4)

 

At x = 0, y = 0. This gives \(C_{2}\) = 0 (remember, singularity functions do not exist until the argument is positive). At x = 20 in, y = 0, and

 

\(0=241.5(20)^{3}-402.4(20-8)^{3}-2555.9(20-8.5)^{2}+74.08(20-8.5)^{3}+C_{1}(20)\)

 

Solving, gives \(C_{1}=-50565 lbf / in ^{2}\). Thus, Eq. (4) becomes, with \(E=30(10)^{6} psi\),

 

\(\begin{aligned}y=& \frac{1}{30\left(10^{6}\right)}\left(241.5 x^{3}-402.4\langle x-8\rangle^{3}-2555.9\langle x-8.5\rangle^{2}\right.\\&\left.+74.08\langle x-8.5\rangle^{3}-50565 x\right)\end{aligned}\) (5)

 

When using a spreadsheet, program the following equations:

 

\(y=\frac{1}{30\left(10^{6}\right)}\left(241.5 x^{3}-50565 x\right) \quad 0 \leq x \leq 8 \text { in }\)

 

\(y=\frac{1}{30\left(10^{6}\right)}\left[241.5 x^{3}-402.4(x-8)^{3}-50565 x\right] \quad 8 \leq x \leq 8.5 \text { in }\)

 

\(\begin{aligned}y=& \frac{1}{30\left(10^{6}\right)}\left[241.5 x^{3}-402.4(x-8)^{3}-2555.9(x-8.5)^{2}\right.\\&\left.+74.08(x-8.5)^{3}-50565 x\right] & 8.5 \leq x \leq 20 \text { in }\end{aligned}\)

Substituting \(C_{1}\) into Eq. (3) the slopes at the supports are found to be \(\theta_{A}=1.686\left(10^{-3}\right)\) \(rad =0.09657 deg , \text { and } \theta_{F}=1.198\left(10^{-3}\right) rad =0.06864 deg .\) You might think these to be insignificant deflections, on shafts, they are not. A finite-element analysis was performed for the same model and resulted in

 

\(\left.y\right|_{x=8.5 \text { in }}=-0.009380 \text { in } \quad \theta_{A}=-0.09653^{\circ} \quad \theta_{F}=0.06868^{\circ}\)

 

Virtually the same answer save some round-off error in the equations.If the steps of the bearings were incorporated into the model, more equations result, but the process is the same. The solution to this model is

 

\(\left.y\right|_{x=8.5 \text { in }}=-0.009387 \text { in } \quad \theta_{A}=-0.09763^{\circ} \quad \theta_{F}=0.06973^{\circ}\)

 

The largest difference between the models is of the order of 1.5 percent. Thus the simplification was justified.
