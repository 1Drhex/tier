Guest Question.

Determine the magnetic flux density on the axis of a uniformly magnetized circular cylinder of a magnetic material. The cylinder has a radius \(b\), length \(L\), and axial magnetization \(M={ a }_{ z }{ M }_{ 0 }\).

Admin
Solution

In this problem concerning a cylindrical bar magnet, let the axis of the magnetized cylinder coincide with the \(z\)-axis of a cylindrical coordinate system, a shown in Fig. 5-10. Since the magnetization \(M\) is a constant within the magnet, \({ J }_{ m }={ \nabla }^{ \backprime }\times M=0\), and there is no equivalent volume current density. The equivalent magnetization surface current density on the side wall is

\({ J }_{ ms }=M\times { { a }_{ n } }^{ \backprime }=\left( { a }_{ z }{ M }_{ 0 } \right) \times { a }_{ r }\)

 

\(={ a }_{ \phi }{ M }_{ 0 }\)

 

The magnet is then like a cylindrical sheet with a lineal circumferential current density of \({ M }_{ 0 }(A/m)\). There is no surface current on the top and bottom faces. To find \(B\) at \(P(0, 0, z)\), we consider a differential length \(d{ z }^{ \backprime }\) with a current \({ a }_{ \phi }{ M }_{ 0 }d{ z }^{ \backprime }\) and use Eq. (5-37) to obtain

\(dB={ { a }_{ z } }\frac { \mu _{ 0 }{ M }_{ 0 }{ b }^{ 2 }d{ z }^{ \backprime } }{ 2{ \left[ { \left( z-{ z }^{ \backprime } \right) }2+{ b }^{ 2 } \right] }^{ 3/2 } } \)

 

and

\(B=\int { dB } ={ a }_{ z }\int _{ 0 }^{ L }{ \frac { \mu _{ 0 }{ M }_{ 0 }{ b }^{ 2 }d{ z }^{ \backprime } }{ 2{ \left[ { \left( z-{ z }^{ \backprime } \right) }2+{ b }^{ 2 } \right] }^{ 3/2 } } }\)

 

\(={ { a }_{ z } }\frac { \mu _{ 0 }{ M }_{ 0 } }{ 2 } \left[ \frac { z }{ \sqrt { { z }^{ 2 }+{ b }^{ 2 } } } -\frac { z-L }{ \sqrt { { \left( z-L \right) }^{ 2 }+{ b }^{ 2 } } } \right] \)
