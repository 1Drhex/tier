Guest Question.

Show that the magnetic field of an infinite solenoid runs parallel to the axis, regardless of the cross-sectional shape of the coil, as long as that shape is constant along the length of the solenoid. What is the magnitude of the field, inside and outside of such a coil? Show that the toroid field (Eq. 5.60) reduces to the solenoid field, when the radius of the donut is so large that a segment can be considered essentially straight.

\( B ( r )=\left\{\begin{array}{cc}\frac{\mu_{0} N I}{2 \pi s} \hat{ \phi }, & \text { for points inside the coil, } \\0 , & \text { for points outside the coil, } \end{array}\right.\) (5.60)

Admin
Solution

We might as well orient the axes so the field point r lies on the y axis: r = (0, y, 0). Consider a source point at \(\left(x^{\prime}, y^{\prime}, z^{\prime}\right)\) on loop #1:

\(ᴫ=-x^{\prime} \hat{ x }+\left(y-y^{\prime}\right) \hat{ y }-z^{\prime} \hat{ z } ; d l ^{\prime}=d x^{\prime} \hat{ x }+d y^{\prime} \hat{ y }\) ;

\(d l ^{\prime} \times ᴫ =\left|\begin{array}{ccc}\hat{ x } & \hat{ y } & \hat{ z } \\d x^{\prime} & d y^{\prime} & 0 \\-x^{\prime} & \left(y-y^{\prime}\right) & -z^{\prime}\end{array}\right|=\left(-z^{\prime} d y^{\prime}\right) \hat{ x }+\left(z^{\prime} d x^{\prime}\right) \hat{ y }+\left[\left(y-y^{\prime}\right) d x^{\prime}+x^{\prime} d y^{\prime}\right] \hat{ z }\) .

\(d B _{1}=\frac{\mu_{0} I}{4 \pi} \frac{d l ^{\prime} \times n }{ᴫ^{3}}=\frac{\mu_{0} I}{4 \pi} \frac{\left(-z^{\prime} d y^{\prime}\right) \hat{ x }+\left(z^{\prime} d x^{\prime}\right) \hat{ y }+\left[\left(y-y^{\prime}\right) d x^{\prime}+x^{\prime} d y^{\prime}\right] \hat{ z }}{\left[\left(x^{\prime}\right)^{2}+\left(y-y^{\prime}\right)^{2}+\left(z^{\prime}\right)^{2}\right]^{3 / 2}} \) .

Now consider the symmetrically placed source element on loop #2, at \(\left(x^{\prime}, y^{\prime},-z^{\prime}\right) \text {. Since } z^{\prime}\) changes sign, while everything else is the same, the \(\hat{ x } \text { and } \hat{ y }\) components from \( d B _{1}\) and \(d B _{2}\) cancel, leaving only a \( \hat{ z }\) component. qed

With this, \(\text { Ampére }\)’s law yields immediately:

\(B = \begin{cases}\mu_{0} n I \hat{ z }, & \text { inside the solenoid; } \\ 0, & \text { outside }\end{cases}\)

(the same as for a circular solenoid—).

For the toroid, N/2πs = n (the number of turns per unit length), so Eq. 5.60 yields \(B=\mu_{0} n I\) inside, and zero outside, consistent with the solenoid. [Note: N/2πs = n applies only if the toroid is large in circumference, so that s is essentially constant over the cross-section.]
