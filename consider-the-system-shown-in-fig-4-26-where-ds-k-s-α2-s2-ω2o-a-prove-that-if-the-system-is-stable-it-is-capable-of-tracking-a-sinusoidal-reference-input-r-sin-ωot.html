Guest Question.

Consider the system shown in Fig. 4.26, where
\(D(s) = K \frac{(s + \alpha)^{2}}{s^{2} + \omega^{2}_{o}}\).

(a) Prove that if the system is stable, it is capable of tracking a sinusoidal reference input \(r = \sin \omega_{o}t\) with zero steady-state error. (Look at the transfer function from R to E and consider the gain at \(\omega_{o}\).)
(b) Use Routh’s criterion to find the range of K such that the closed-loop system remains stable if \(\omega_{o} = 1 and \alpha = 0.25\).

Admin
Solution

(a)

\(D(s)G(s) = \frac{K(s + \alpha)^{2}}{(s^{2} + \omega^{2}_{o}) s (s + 1)}\)

 

\(\frac{E(s)}{R(s)} = \frac{1}{1 + DG} = \frac{s(s + 1) (s^{2} + \omega^{2}_{o})}{(s^{2} + \omega^{2}_{o}) s(s + 1) + K(s + \alpha)^{2}}\)

The gain of this transfer function is zero at \(s = \pm j \omega_{o}\) and we expect the error to be zero if R is a sinusoid at that frequency. More formally, let \(R(s) = \frac{\omega_{n}}{s^{2} + \omega^{2}_{n}}\) then

\(E(s) = \frac{s(s + 1) (s^{2} + \omega^{2}_{o})}{(s^{2} + \omega^{2}_{o}) s(s + 1) + K(s + \alpha)^{2}} \frac{\omega_{n}}{s^{2} + \omega^{2}_{n}}\)

Assuming the (closed-loop) system is stable, then if \(\omega_{n}\risingdotseq \omega_{o}\) Then E(s) has a pole on the imaginary axis and the FVT does not apply.
The final error will NOT be zero in this case. However, if \(\omega_{n} = \omega_{o}\) we can use the FVT and

\(e_{ss} = \underset{s \rightarrow 0}{\lim} s E(s) = 0\)

(b) To test for stability, the characteristic equation is,

\(s^{4} + (K + \omega^{2}_{o})s^{2} + s^{3} + (\omega^{2}_{o} + 2 \alpha K)s + K \alpha^{2} = 0\)

Using the Routh array

\(\begin{array}{r r c c c} \\ s^{4} & : & 1 & \omega^{2}_{o} + K & K \alpha^{2} \\ s^{3} & : & 1 & (\omega^{2}_{o} + 2 \alpha K) \\s^{2} & : & K(1 – 2 \alpha) & K \alpha^{2} \\ s^{1} & : & \omega^{2}_{o} + 2 \alpha K + \frac{\alpha^{2}}{(1 – 2 \alpha)} \\ s^{0} & : & K \alpha^{2} \end{array}\)

If \(\alpha = 0.25\), we must have \(K \gt 0 , and K \gt 0.25 – 2\omega^{2}_{o}.\)
