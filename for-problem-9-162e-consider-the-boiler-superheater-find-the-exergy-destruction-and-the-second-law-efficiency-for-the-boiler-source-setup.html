Guest Question.

For , consider the boiler/superheater. Find the exergy destruction and the second law efficiency for the boiler-source setup.

Admin
Solution

The boiler has flow in at state 4 and out at state 5 with the source providing a q at 350 F.

Assume state 4 is saturated liquid at \(T _{4}\) so \(h _{4}= h _{ f 4} \quad \Rightarrow \quad T _{4}=68.4 F\)

State 4: \(h _{4}=118.4 Btu / lbm = h _{ f 4}, s _{4}= s _{ f 4}=0.2494 Btu / lbm – R\)

State 5: \(h _{5}=744.1 Btu / lbm , \quad s _{5}=1.3062 Btu / lbm – R\)

Energy Eq. \(q _{ H }= h _{5}- h _{4}=744.1-118.4=625.7 Btu / lbm\)

Entropy Eq. \(s _{4}+ q _{ H } / T _{ H }+ s _{ gen }= s _{5} \Rightarrow s _{ gen }= s _{5}- s _{4}- q _{ H } / T _{ H }\)

\(\begin{aligned}\dot{ S }_{ gen } &=\dot{ m } s _{ gen }=10\left[1.3062-0.2494-\frac{625.7}{350+459.7}\right] \\&=10 \times 0.2841= 2 . 8 4 1 Btu / s – R\end{aligned}\)

The flow increase in exergy is:

\(\begin{gathered}\psi_{5}-\psi_{4}= h _{5}- h _{4}- T _{ o }\left( s _{5}- s _{4}\right)=625.7-536.7(1.3062-0.2494) \\=58.62 Btu / lbm\end{gathered}\)

The exergy provided by the source is:

\(\phi_{ H }=\left(1- T _{ o } / T _{ H }\right) q _{ H }=\left(1-\frac{536.7}{350+459.7}\right) 625.7=210.96 Btu / lbm\)

Second law efficiency is

\(\begin{aligned}&\eta_{I I}=\frac{\text { gain in exergy }}{\text { source exergy input }}=\frac{58.62}{210.96}= 0 . 2 6 5 \\&\left(=1- T _{ o } s _{ gen } / \phi_{ H }=1-536.7 \times 0.2841 / 210.96\right)\end{aligned}\)
