Solution

a) First transform the points to cartesian : \({ x }_{ A }=5\sin { { 110 }^{ o } } \cos { { 200 }^{ o } } =-4.42\) ,

\({ y }_{ A }=5\sin { { 110 }^{ o } } \sin { { 200 }^{ o } } =-1.61 ,and { z }_{ A }=5\cos { { 110 }^{ o } } =-1.71 ; { x }_{ B }=7\sin { { 30 }^{ o } } \cos { { 70 }^{ o } } =1.20 , { y }_{ B }=7\sin { { 30 }^{ o } } \sin { { 70 }^{ o } } =3.29 , and { z }_{ B }=7\cos { { 30 }^{ o } } =6.06\) . Now

\({ R }_{ AB }=B-A=5.62{ a }_{ x }+4.90{ a }_{ y }+7.77{ a }_{ z }\)

b)First , \({ R }_{ BQ }=Q-B=(-5,5,1)\) . Then at P , \(r=\sqrt { 4+9+16 } =5.39 , \theta =\cos ^{ -1 }{ (4/\sqrt { 29 } ) } ={ 42.0 }^{ o } , and \phi =\tan ^{ -1 }{ (-3/2) } =-{ 56.3 }^{ 3 }\) . Now

\({ R }_{ PQ }.{ a }_{ r }=-5\sin { ({ 42 }^{ o }) } \cos { { (-56.3 }^{ o }) } +5\sin { { (42 }^{ o }) } \sin { { (-56.3 }^{ o }) } +1\cos { { (42 }^{ o }) } =-3.90\) \({ R }_{ PQ }.{ a }_{ \theta }=-5\cos { ({ 42 }^{ o }) } \cos { { (-56.3 }^{ o }) } +5\cos { { (42 }^{ o }) } \sin { { (-56.3 }^{ o }) } -1\sin { { (42 }^{ o }) } =-5.82\)

So finally ,

\({ R }_{ PQ }=-3.90{ a }_{ r }-5.82{ a }_{ \theta }-1.39{ a }_{ \phi }\)

speciﬁed point. Use \({ a }_{ \rho }=({ a }_{ \rho }.{ a }_{ x }){ a }_{ x }+({ a }_{ \rho }.{ a }_{ y }){ a }_{ y } .At A(1,2,3) { \rho }=\sqrt { 5 } ,\phi =\tan ^{ -1 }{ (2) } ={ 63.4 }^{ o }\) ,

\(r=\sqrt { 14 } , and \theta =\cos ^{ -1 }{ (3/\sqrt { 14 } } )={ 36.7 }^{ o } .So { a }_{ \rho }=\cos { ({ 63.4 }^{ o }) } { a }_{ x }+\sin { ({ 63.4 }^{ o }) } { a }_{ y }=0.45{ a }_{ x }+0.89{ a }_{ y }\) . Then

\((5{ a }_{ r }-3{ a }_{ \theta }+4{ a }_{ \phi }).(0.45{ a }_{ x }+0.89{ a }_{ y })=5(0.45)\sin { \theta } \cos { \phi } +5(0.89)\sin { \theta } \sin { \phi } -3(0.45)\cos { \theta } \cos { \phi } -3(0.89)\cos { \theta } \sin { \phi } +4(0.45)(-\sin { \phi } )+4(0.89)\cos { \phi } =0.59\)
