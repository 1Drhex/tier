Guest Question.

Beam Coped at the Top Flange

For the beam shown in Figure 11-7, determine whether the beam is adequate for flexural rupture and flexural yielding at the beam cope. The steel is ASTM A992, grade 50.

Admin
Solution

From AISCM, Table 1-1,

d = 16.0 in. \( t_{w}\) = 0.305 in.

\( d_{f}\) = 7.0 in. \( t_{f}\) = 0.505 in.

\( h_{0}\) = d –\( d_{c}\) = 16.0 – 3 = 13.0 in.

Section properties at the critical section:

Web component is (13.0 in. – 0.505 in.) by 0.305 in.

Flange component is 7.0 in. by 0.505 in.

Table 11-1 Section properties at the beam cope

Element A y Ay I \(d=y -\overline{y}\) \(I + Ad^{2}\)

Flange 3.53 0.252 0.89 0.0751 -3.37 40.2
∑= 7.34 26.63 \(I_{net}\)=127.1

\(\overline{y}=\frac{\sum{Ay} }{\sum{A} }=\frac{26.6}{7.34}= 3.63\) in.

\(h_{1}=h_{0} -\overline{y} = 13.0 – 3.63 = 9.37\) in.

\(S_{net}=\frac{I_{net} }{h_{1}}=\frac{127.1}{9.37}= 13.6 in.^{3}\)

Alternatively, using AISCM, Table 9-2 with \(d_{c}\) =3 in., confirm that \(S_{net} =13.6 in.^{3}\) .

\(M_{u}=R_{u}e = (55 kips)(5 in.) = 275\) in.-kips

Flexural rupture strength:

\(\phi _{br}M_{n}= \phi _{br}F_{u}S_{net}\)

= (0.75)(65)(13.6) = 663 in.-kips > 275 in.-kips OK

c ≤ 2d → 4.5 in. < (2)(16.0 in.) = 32 in. OK

\(d_{c}\) ≤ 0.5d → 3 in. < (0.5)(16.0 in.) 8 in. OK

\(\frac{c}{d} =\frac{4.5}{16.0}= 0.281 \leq 1.0 \rightarrow f=\frac{2c}{d}=\frac{(2)(4.5)}{16.0}=0.563\) in.

\(\frac{c}{h_{1} } =\frac{4.5}{9.37}= 0.480 \leq 1.0 \rightarrow k= 2.2\left(\frac{h_{1} }{c} \right) ^{1.65}= 2.2\left(\frac{9.37 }{4.5} \right) ^{1.65} =7.38\)

 

\(F_{cr}= 26,210\left(\frac{t_{w} }{h_{1} } \right) ^{2}fk =26.210\left(\frac{0.305}{9.37} \right) ^{2}(0.563)(7.38) = 115.4\) ksi

\(\phi _{b}M_{n}= \phi _{b}F_{cr}S_{net} = (0.9)(115.4)(13.6) = 1412 in.-kips \gt 275\) in.-kips, OK

Shear strength (from Chapter 6):

\(\phi _{v}V_{n}=\phi _{v}0.6F_{y}A_{w}C_{v}\)

=(0.9)(0.6)(50)[(13.0)(0.305)](1.0)= 107 kips >\(V_{n}\)=55 kips OK

The coped connection is adequate in shear and in flexural rupture and flexural yielding.
