Guest Question.

Find the control voltage/output position and reaction torque/output position transfer functions for the scheme of Fig. 5.6.

Admin
Solution

From this block scheme, the following relations can be found:

\(\Theta _{m}\) = \(\frac{1}{s} \Omega _{m}\)

\(\Omega _{m}\) = \(\frac{C_{m} – C_{r}}{sI_{m} + F_{m}}\)

\(C_{m}\) = \( k_{t} P\)

\(P\) = \(\frac{1}{s k_{c} + k_{l}} \left(\frac{ k_{x}}{ k_{r}} \left(X – \frac{P}{k_{x}} \right) – k_{q} \Omega _{m}\right)\)

\(= \frac{k_{x} X – k_{r}k_{q}\Omega _{m}}{1 + k _{r} k _{l} + sk _{r}k _{c}}\)

\( X \) = \(\frac{G_{s}}{1 + sT_{s}} V_{c}\)

Substituting (S5.6) into (S5.6), and then (S5.6) into (S5.6), yield

\(\left(1 + \frac{k_{t}k_{r}k_{q}}{\left(sI_{m} + F_{m}\right) \left(1 + k_{r}k_{l} + s k_{r}k_{c}\right) } \right) \Omega _{m} \) = \( \frac{ k_{r}k_{x}}{\left(1 + k_{r}k_{l} + s k_{r}k_{c}\right)\left(sI_{m} + F_{m}\right) }X – \frac{1}{s I_{m} + F_{m}} C_{r}\)

Solving for \( \Omega _{m}\) , substituting it into (S5.6), and using (S5.6) leads to

\(\Theta _{m} \) = \(\frac{\frac{k_{x}G_{s}}{k_{r}k_{q}} }{s\left(1 + s T_{s}\right) \left(1 + \frac{F_{m}}{k_{t} k_{r}k_{q} } \left(1 + s \frac{I_{m}}{F_{m}} \right)\left(1 + k_{r}k_{l} + sk_{r}k_{c}\right) \right) } V_{c} \\[0.5cm] – \frac{\frac{1 + k_{r}k_{l} + sk_{r}k_{c}}{k_{t}k_{r} k_{q}} }{s \left(1 + \frac{F_{m}}{k_{t} k_{r}k_{q}} \left(1 + s\frac{I_{m}}{F_{m}} \right) \left(1 + k_{r}k_{l} + s k_{r}k_{c}\right) \right) } C_{r}.\)
