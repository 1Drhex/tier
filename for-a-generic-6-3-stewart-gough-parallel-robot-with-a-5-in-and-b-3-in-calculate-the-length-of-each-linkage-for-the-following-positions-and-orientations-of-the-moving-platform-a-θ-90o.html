Guest Question.

For a generic 6-3 Stewart-Gough parallel robot with a = 5 in. and b = 3 in., calculate the length of each linkage for the following positions and orientations of the moving platform:
\(a) θ =90^{\circ } , ϕ=0^{\circ } ,ψ =0^{\circ } , P_{x} =0, P_{y}=0, Pz_{z} = 12 in.\)

 

\(b) θ =0^{\circ } , ϕ=90^{\circ } ,ψ =0^{\circ } , P_{x} =2, P_{y} =2, P_{z} = 12 in.\)
Admin
Solution

For each case, we substitute the given values into Eqs. (4.10)–(4.16). For both cases, we get:

\(\bar{d_{i}}= \bar{P}+\bar{b_{i}}-\bar{a_{i}} for i=1…6\) (4.10)

 

\(\left|d_{i} \right|=\sqrt{(d_{ix})^{2}+(d_{iy})^{2}+(d_{iz})^{2} } for i=1…6\) (4.16)
