Guest Question.

Again find the work required to carry 2C from B to A in the same field, but this time use the straight-line path from B to A.

Admin
Solution

We start by determining the equations of the straight line. Any two of the following three equations for planes passing through the line are sufficient to define the line:

\(y-y_{B} =\frac{y_{A}-y_{B}}{x_{A}-x_{B}}(x-x_{B}) \)

 

\( z-z_{B} =\frac{z_{A}-z_{B}}{y_{A}-y_{B}}(y-y_{B})\)

 

\(x-x_{B} =\frac{x_{A}-x_{B}}{z_{A}-z_{B}}(z-z_{B})\)

From the first equation we have y = −3(x − 1)

and from the second we obtain z = 1

Thus,

\(W=-2 \int_{1}^{0.8}{y \space dx}-2 \int_{0}^{0.6}{x \space dy}-4\int_{1}^{1}{ \space dz}\)

 

\(=6 \int_{1}^{0.8}{(x-1) \space dx}-2 \int_{0}^{0.6}{(1-\frac{y}{3} ) \space dy}\)

 

\(=-0.96J\)

This is the same answer we found using the circular path between the same two points, and it again demonstrates the statement (unproved) that the work done is independent of the path taken in any electrostatic field.

It should be noted that the equations of the straight line showthat dy = −3 dx and
dx = −\(\frac{1}{3}\) dy. These substitutions may be made in the first two integrals, along with a change in limits, and the answer may be obtained by evaluating the new integrals. This method is often simpler if the integrand is a function of only one variable.

Note that the expressions for dL in our three coordinate systems use the differential
lengths obtained in Chapter 1 (rectangular in Section 1.3, cylindrical in Section 1.8, and spherical in Section 1.9):

\(dL=dx \space a_{x} + dy \space a_{y} +dz \space a_{z}\) (rectangular) (6)

 

\(dL=d\rho \space a_{\rho } +\rho \space d\phi \space a_{\phi } +dz \space a_{z}\) (cylindrical) (7)

 

\(dL=dr \space a_{r} + r \space d\theta \space a_{\theta } +r \space \sin \theta \space d\phi \space a_{\phi }\) (spherical) (8)

The interrelationships among the several variables in each expression are determined from the specific equations for the path.

As a final example illustrating the evaluation of the line integral, we investigate several paths that we might take near an infinite line charge. The field has been obtained several times and is entirely in the radial direction,

\(E=E_{\rho } a_{\rho } =\frac{\rho L}{2\pi \epsilon _{0}\rho } a_{\rho } \)

First we find the work done in carrying the positive charge Q about a circular path of radius \(\rho _{b}\) centered at the line charge, as illustrated in Figure 4.2a. Without lifting a pencil, we see that the work must be nil, for the path is always perpendicular to the electric field intensity, or the force on the charge is always exerted at right angles to the direction in which we are moving it. For practice, however, we will set up the integral and obtain the answer.

The differential element dL is chosen in cylindrical coordinates, and the circular
path selected demands that dρ and dz be zero, so \(dL=\rho _{1} \space d\phi \space a_{\phi }\). The work is then

\(W=-Q\int_{init}^{final}{\frac{\rho _{L} }{2\pi \epsilon _{0} \rho _{1} } a_{\rho \space }\cdot \rho _{1}\space d\phi \space a_{\phi } } \)

 

\(=-Q\int_{0}^{2\pi }{\frac{\rho _{L} }{2\pi \epsilon _{0} }\space d\phi \space a_{\rho \space } \cdot a_{\phi } } =0\)

We will now carry the charge from ρ = a to ρ = b along a radial path (Figure 4.2b). Here dL = dρ \(a_{\rho } \) and

\(W=-Q\int_{init}^{final}{\frac{\rho _{L} }{2\pi \epsilon _{0} \rho } a_{\rho \space }\cdot d\rho \space a_{\rho } }=-Q\int_{a}^{b}{\frac{\rho _{L} }{2\pi \epsilon _{0} } \frac{d \space \rho }{\rho } }\)

or

\(W= -\frac{Q\rho _{L} }{2\pi \epsilon _{0} }ln\frac{b}{a} \)

Because b is larger than a, ln(b/a) is positive, and the work done is negative, indicating that the external source that is moving the charge receives energy.

One of the pitfalls in evaluating line integrals is a tendency to use too many minus
signs when a charge is moved in the direction of a decreasing coordinate value. This is
taken care of completely by the limits on the integral, and no misguided attempt should
be made to change the sign of dL. Suppose we carry Q from b to a (Figure 4.2b).
We still have dL = \(d\rho \space a_{\rho }\) and show the different direction by recognizing ρ = b as the initial point and ρ = a as the final point,

\(W=-Q\int_{b}^{a}{\frac{\rho _{L} }{2\pi \epsilon _{0} } \frac{d \space \rho }{\rho } } = \frac{Q \rho _{L} }{2\pi \epsilon _{0} }ln \frac{b}{a} \)

This is the negative of the previous answer and is obviously correct.
