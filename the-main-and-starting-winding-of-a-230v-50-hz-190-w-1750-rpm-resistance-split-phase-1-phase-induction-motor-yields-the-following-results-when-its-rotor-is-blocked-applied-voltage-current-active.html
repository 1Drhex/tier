Guest Question.

The main and starting winding of a 230V, 50 Hz, 190 W, 1750 rpm, resistance split-phase (1-phase) induction motor yields the following results when its rotor is blocked:

Applied voltage Current Active Power

Main winding: 45 V 4 A 120 W

Starting winding: 45 V 1.5 A 60 W

Calculate (i) The phase angle between \(I_{m}\) and \(I_{S}\)

(ii) The locked-rotor current drawn from the lines at 230 V.

Admin
Solution

Here, \(V = 230 V; f = 50 Hz; N = 1750 rpm\)

At block rotor, for main winding, \(I_{m} = 4A; V_{sc} = 45V; P_{sc\left(m\right) } = 120 W\)

For starting winding, \(I_{S} = 4A; V_{sc} = 45V; P_{sc\left(s\right) } = 60 W\)

( i) Let us first calculate the phase angle between \(I_{m} \) and the applied voltage V for the main winding.

Apparent power, \(P_{a\left(m\right) } = V_{sc} I_{m}= 45 \times 4 = 180 VA\)

∴ Power factor, \(\cos \phi _{m}= P_{sc\left(m\right) } / P_{a\left(m\right) } = 120 / 180 = 0.667 \therefore \phi _{m}= 48.2^{\circ }\)

Therefore, \(I_{m} \) lags behind V by an angle of 48.2°

Now, we calculate the phase angle between \(I_{S} \) and V for the starting winding.

Apparent power, \(P_{a\left(s\right) } = V_{sc} I_{s}= 45 \times 1.5 = 67.5 VA\)

∴ Power factor, \(\cos \phi _{S}= P_{sc\left(s\right) } / P_{a\left(s\right) } = 60 / 67.5 = 0.89 \therefore \phi _{s}= 27.3^{\circ }\)

Therefore, \(I_{S} \) lags behind V by an angle of 27.3°.

The phase angle between \(I_{m} \) and \(I_{S} \), \(\theta = \phi _{m} – \phi _{s}= 8.2^{\circ } – 27.3^{\circ } = \mathbf{20.9^{\circ }} \)

(ii) Total active power, \(P = P_{m} + P_{s}= 120 + 60 = 180 W \)

Total reactive power, \(P_{r} = P_{rm} + P_{rs}= P_{m} \tan \phi _{m} + P_{s} \tan \phi _{s} \\[0.5cm] \hspace{30 pt} \hspace{30 pt} \qquad \quad \quad \, = 120 \tan 48.2^{\circ } + 60 \tan 27.3^{\circ } = 165.18 VAR\)

Total apparent power, \(P_{a} = \sqrt{P^{2} + P^{2}_{r}} = \sqrt{\left(180\right)^{2} + \left(165.18\right)^{2} } = 244.3 VA\)

Short circuit line current at \(45 V= \frac{P_{a}}{V_{sc}} = \frac{244.3}{45} = 5.43 A\)

Short circuit current at \(230 V = 5.43 \times \frac{230}{45} = \mathbf{27.75 A} \)
