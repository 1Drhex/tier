Guest Question.

A \(50-\left( \Omega \right)\) transmission line is connected to a load impedance \({ Z }_{ L }=35-j47.5\left( \Omega \right) \). Find the position and length of a short-circuited stub required to match the line.

Admin
Solution
\({ R }_{ 0 }=50\left( \Omega \right)\)

 

\({ Z }_{ L }=35-j47.5\left( \Omega \right)\)

 

\({ z }_{ L }={ Z }_{ L }/{ R }_{ 0 }=0.70-j0.95\)

 

\(1.\) Enter \({ z }_{ L }\) on the Smith chart as \({ P }_{ 1 }\) (Fig. 8- 13).

\(2.\) Draw a \(\left| \Gamma \right| \)-circle centered at \(O\) with radius \(\overline { O{ P }_{ 1 } } \).

\(3.\) Draw a straight line from \({ P }_{ 1 }\) through \(O\) to point \({ { P }_{ 2 } }^{ \backprime }\) on the perimeter, intersecting the \(\left| \Gamma \right| \)-circle at \({ P }_{ 1 } \), which represents \({ y }_{ L }\) Note \(0.109\) at \({ { P }_{ 2 } }^{ \backprime }\) on the “wavelengths toward generator” scale.

\(4.\) Note the two points of intersection of the \(\left| \Gamma \right|\) -circle with the \(g = 1\) circle.

 

At \({ P }_{ 3 }:\quad { y }_{ B1 }=1+j1.2=1+j{ b }_{ B1 };\)

 

At \({ P }_{ 4 }:\quad { y }_{ B2 }=1-j1.2=1+j{ b }_{ B2 };\)

 

\(5.\) Solutions for the position of the stub:

 

For \({ P }_{ 3 }(from{ { P }_{ 2 } }^{ \backprime }to{ { P }_{ 3 } }^{ \backprime }):\quad { d }_{ 1 }=\left( 0.168-0.109 \right) \lambda =0.059\lambda ;\)

 

For \({ P }_{ 4 }(from{ { P }_{ 2 } }^{ \backprime }to{ { P }_{ 4 } }^{ \backprime }):\quad { d }_{ 2 }=\left( 0.332-0.109 \right) \lambda =0.223\lambda ;\)

 

\(6.\) Solutions for the length of the short-circuited tub to provide \({ y }_{ s }=-j{ b }_{ B }:\)

For \({ P }_{ 3 }\) (from \({ P }_{ sc }\) on the extreme right of chart to \({ { P }_{ 3 } }^{ \backprime \backprime }\) which represents \(-j{ b }_{ B1 }=-j1.2\)):

 

\({ \ell }_{ 1 }=\left( 0.361-0.250 \right) \lambda =0.111\lambda ;\)

 

For \({ P }_{ 4 }\)(from \({ P }_{ sc }\) to \({ { P }_{ 4 } }^{ \backprime \backprime }\), which represents \(-j{ b }_{ 2 }=-j1.2):\)

 

\({ \ell }_{ 2 }=\left( 0.139-0.250 \right) \lambda =0.389\lambda ;\)

 

In general, the solution with the shorter lengths is preferred unless there are other practical constraints. The exact length, \(\ell\), of the hart-circuited tub may require fine adjustment in the actual matching procedure; hence the shorted matching sections are sometimes called stub tuners.
