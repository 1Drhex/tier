Guest Question.

Determine if a built-up, ASTM A572 grade 50 column with \(PL 3 / 8 in \times 101 / 2\) in. flanges and a \(PL ^{1 / 4} \text { in. } \times 7^{1} / 4\) in. web has sufficient available strength to carry a dead load of 40 kips and a live load of 120 kips in axial compression. The column unbraced length is 15 ft in both axes and the ends are pinned.

 

Admin
Solution

Material Properties: Manual Table 2-3

ASTM A572 Gr. 50 \(F_{y}=50 ksi \quad F_{u}=65 ksi\)

Geometric Properties:

Built-up Column \(d=8.00 \text { in. } \quad b_{f}=10.5 \text { in. } \quad t_{f}=0.375 \text { in. } \quad h=7.25 \text { in }\)

\(t_{w}=0.250 \text { in. }\)

Calculate the required strength

LRFD ASD
\(P_{u}=1.2(40.0 kips) + 1.6(120 kips)\)

= 240 kips

 \(P_{a}=40.0 \text { kips }+120 \text { kips }\)

= 160 kips

Calculate built-up section properties (ignoring fillet welds)

\(A=2(10.5 \text { in. })(0.375 in .)+(7.25 in .)(0.250 in .)=9.69 in .{ }^{2}\)

Since the unbraced length is the same for both axes, the weak axis will govern.

\(I_{y}=2\left[\frac{(0.375 in .)(10.5 in .)^{3}}{12}\right]+\frac{(7.25 in .)(0.250 in .)^{3}}{12}=72.4 in .{ }^{4}\)

 

\(r_{y}=\sqrt{\frac{I_{y}}{A}}=\sqrt{\frac{72.4 in ^{4}}{9.69 in ^{2}}}=2.73 in\)

 

\(I_{x}=2(10.5 in .)(0.375 in .)(3.81 in .)^{2}+\frac{(0.25 in .)(7.25 in .)^{3}}{12}+\frac{2(10.5 in .)(0.375 in .)^{3}}{12}\)

 

\(=122 in ^{4}\)

Check web slenderness

For a stiffened element (web) in a doubly symmetric I-shaped section, under uniform compression,

\(\lambda_{r}=1.49 \sqrt{\frac{E}{F_{y}}}=1.49 \sqrt{\frac{29,000 ksi }{50 ksi }}=35.9\) Table B4.1 Case 10

\(\frac{h}{t_{w}}=\frac{7.25 in .}{0.250 in .}=29.0<35.9\) Therefore, the web is not slender.

Note that the fillet welds are ignored in the calculation of h for built up sections.

Check flange slenderness

\(\text { Calculate } k_{c}\)

 

\(k_{c}=\frac{4}{\sqrt{\frac{h}{t_{w}}}}=\frac{4}{\sqrt{\frac{7.25 in .}{0.250 in }}}=0.743\) \(\text { where } 0.35 \leq k_{c} \leq 0.76 \text { o.k. }\) Table B4.1 Note [a]

 

\(\text { Use } k_{c}=0.743\)

For flanges of a built-up I-shaped section under uniform compression;

\(\lambda_{r}=0.64 \sqrt{\frac{k_{c} E}{F_{y}}}=0.64 \sqrt{\frac{0.743(29,000 ksi )}{50 ksi }}=13.3\) Table B4.1 Case 4

 

\(\frac{b}{t}=\frac{5.25 in .}{0.375 in }=14.0>13.3\) Therefore, the flanges are slender.

For compression members with slender elements, Section E7 of the Specification applies. The nominal compressive strength, \(P_{n}\), shall be determined based on the limit states of flexural, torsional, and flexural-torsional buckling. Depending on the slenderness of the column, Specification Equation E7-2 or E7-3 applies. \(F_{e}\) is used in both equations and is calculated as the lesser of Specification Equations E3-4 and E4-4. Section E7

For a pinned-pinned condition, K = 1.0

Since the unbraced length is the same for both axes, the weak axis will govern. Commentary Table C-C2.2

\(\frac{K_{y} L_{y}}{r_{y}}=\frac{1.0(15.0 ft )}{2.73 in .} \frac{12 in .}{ ft }=65.9\)

Calculate the elastic critical stress, \(F_{e}\), for flexural buckling

\(F_{e}=\frac{\pi^{2} E}{\left(\frac{K L}{r}\right)^{2}}=\frac{\pi^{2} 29,000 ksi }{(65.9)^{2}}=65.9 ksi\) Eqn. E3-4

Calculate the elastic critical stress, \(F_{e}\), for torsional buckling

Not likely to govern, but check for completeness

\(C_{w}=\frac{I_{y} h_{o}^{2}}{4}=\frac{72.4 in .^{4}(7.63 in .)^{2}}{4}=1050 in .\) Design Guide No. 3 Eqn 3-5

 

\(J=\sum \frac{b t^{3}}{3}=\frac{2(10.5 in .)(0.375 in .)^{3}+7.25 in .(0.25 in .)^{3}}{3}=0.407 in ^{4}\) Design Guide No. 3 Eqn 3-4

 

\(F_{e}=\left[\frac{\pi^{2} E C_{w}}{\left(K_{z} L\right)^{2}}+G J\right] \frac{1}{I_{x}+I_{y}}\)

 

\(=\left[\frac{\pi^{2}(29,000 ksi )\left(1050 in ^{6}\right)}{(180 in .)^{2}}+(11,200 ksi )\left(0.407 in ^{4}\right)\right] \frac{1}{122 in ^{4}+72.4 in ^{4}}\) Eqn. E4-4

 

= 71.2 ksi > 65.9 ksi

 

Therefore, use \(F_{e}\) = 65.9 ksi

Determine Q, the slenderness reduction factor

\(Q=Q_{s} Q_{a}, \text { where } Q_{a}=1.0\) because the web is not slender

Calculate \(Q_{s}\), the unstiffened element (flange) reduction factor Section E7-1

Determine the proper equation for \(Q_{s}\) by checking limits for Equations E7-7 to E7-9

\(\frac{b}{t}=14.0 \text { from above }\)

 

\(0.64 \sqrt{\frac{E k_{c}}{ F }}=0.64 \sqrt{\frac{29,000 ksi (0.743)}{50 ksi }}=13.3\) Section E7-1(b)

 

\(1.17 \sqrt{\frac{E k_{c}}{F_{y}}}=1.17 \sqrt{\frac{29,000 ksi (0.743)}{50 ksi }}=24.3\)

 

\(0.64 \sqrt{\frac{E k_{c}}{F_{y}}}<\frac{b}{t} \leq 1.17 \sqrt{\frac{E k_{c}}{F_{y}}}\) therefore, Equation E7-8 applies.

 

\(Q_{s}=1.415-0.65\left(\frac{b}{t}\right) \sqrt{\frac{F_{y}}{E k_{c}}}\)

 

\(=1.415-0.65(14.0) \sqrt{\frac{50 ksi }{(29,000 ksi )(0.743)}}=0.977\) Eqn. E7-8

 

\(Q=Q_{s} Q_{a}=(0.977)(1.0)=0.977\)

Calculate nominal compressive strength

\(4.71 \sqrt{\frac{E}{Q F_{y}}}=4.71 \sqrt{\frac{29,000 ksi }{0.977(50 ksi )}}=115>65.9\) therefore, Specification Eqn. E7-2 applies.

 

\(F_{c r}=Q\left[0.658^{\frac{Q F_{y}}{F_{e}}}\right] F_{y}=0.977\left[0.658^{\frac{0.977(50 ksi )}{65.9 ksi }}\right](50 ksi )=35.8 ksi\) Eqn. E7-2

 

\(P_{n}=F_{c r} A_{g}=(35.8 ksi )\left(9.69 in ^{2}\right)=347 \text { kips }\) Eqn. E7-1

 

LRFD ASD
\(\phi_{c}=0.90\) \(\Omega_{ c }=1.67\)
\(\phi_{c} P_{n}=0.90(347 kips )\)

= 312 kips > 240 kips o.k.

 \(P_{n} / \Omega_{c}=(347 kips) / 1.67\)

= 208 kips > 160 kips o.k.

Section E1

Note: Built-up sections are generally more expensive than standard rolled shapes; therefore, a standard compact shape, such as a W8×35 might be a better choice even if the weight is somewhat higher. This selection could be taken directly from Manual Table 4-1.
