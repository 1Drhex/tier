Guest Question.

An abdominal radiograph uses \(10^{13}\) photons to expose a film with an area of 0.15 \(m^{2}\) \(\left(1.5 \times 10^{-1} m^{2} or \ 1500 \ cm^{2}\right)\) during an exposure time of 0.1 second. All photons have an energy of 40 keV. Find the photon fluence \(\phi\) the photon flux \(\phi\), the energy fluence \(\Psi\), and the intensity I.

Admin
Solution

Photon fluence \(\Phi = \frac{N}{A}= \frac{1\times 10^{13} photons}{1.5 \times 10^{-1} m^{2}}= 6.7 \times 10^{13} \frac{photons}{m^{2}}\)

Photon flux \(\phi = \frac{N}{At}= \frac{\Phi }{t} = \frac{6.7 \times 10^{13} \frac{photons}{m^{2}} }{0.1 \ sec} = 6.7 \times 10^{14} \frac{photons}{m^{2} \cdot sec}\)

Energy fluence \(\Psi =\frac{NE}{A}= \Phi E= \left(6.7 \times 10^{13}\frac{photons}{m^{2}} \right) \left(40 \frac{KeV}{photons}\right)\)

 

\(\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ =2.68\times 10^{15} \frac{KeV}{m^{2}}\)

 

\( \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ =2.68\times 10^{12} \frac{MeV}{m^{2}} \)

 

\(Intensity \ I=\Psi =\frac{NE}{At}= \frac{\Psi }{t}= \frac{2.68\times 10^{12} \frac{MeV}{m^{2}} }{0.1 \ sec} = 2.68\times 10^{13} \frac{MeV}{m^{2}\cdot sec}\)

Note: A beam produced by an x-ray tube actually contains a spectrum of energies(Chapter 5), and a more accurate estimate of intensity would involve a weighted sum of the contributions of the various photon energies to the total intensity.
