Guest Question.

A long straight nonmagnetic conductor of 0.2-mm radius carries a uniformly distributed current of 2 A dc. (a) Find J within the conductor. (b) Use Ampe’re’s circuital law to find H and B within the conductor. (c) Show that \(\triangledown \times H\) = J within the conductor. (d) Find H and B within the conductor. (e) Show that \(\triangledown \times H\) = J outside the conductor.

Admin
Solution

a) Assuming the current is +z directed,
\(J=\frac { 2 }{ \pi { \left( 0.2\times { 10 }^{ -3 } \right) }^{ 2 } } { a }_{ z }=1.59\times { 10 }^{ 7 }{ a }_{ z }A/{ m }^{ 2 }\)
b) Inside, at radius \(\rho\), we have
\(2\pi \rho { H }_{ \phi }=\pi { \rho }^{ 2 }J\Longrightarrow H=\frac { \rho J }{ 2 } { a }_{ \phi }=7.96\times { 10 }^{ 6 }\rho { a }_{ \phi }A/m\)
Then \(B={ \mu }_{ 0 }H=\left( 4\pi \times { 10 }^{ -7 } \right) \left( 7.96\times { 10 }^{ 6 } \right) \rho { a }_{ \phi }=10\rho { a }_{ \phi }Wb/{ m }^{ 2 }\).
c) Using the result of part b, we ﬁnd
\(\triangledown \times H=\frac { 1 }{ \rho } \frac { d }{ d\rho } \left( \rho { H }_{ \phi } \right) { a }_{ z }=\frac { 1 }{ \rho } \frac { d }{ d\rho } \left( \frac { 1.59\times { 10 }^{ 7 }{ \rho }^{ 2 } }{ 2 } \right) { a }_{ z }=1.59\times { 10 }^{ 7 }{ a }_{ z }A/{ m }^{ 2 } = J\)
d) Outside, the entire current is enclosed by a closed path at radius ρ, and so
\(H=\frac { 1 }{ 2\pi \rho } { a }_{ \phi }=\frac { 1 }{ \pi \rho } { a }_{ \phi }A/m\)
Now B = \({ \mu }_{ 0 }H = { \mu }_{ 0 }/(\pi \rho ){ a }_{ \phi }Wb/{ m }^{ 2 }\).
e) Here we use H outside the conductor and write:
\(\triangledown \times H=\frac { 1 }{ \rho } \frac { d }{ d\rho } \left( \rho { H }_{ \phi } \right) { a }_{ z }=\frac { 1 }{ \rho } \frac { d }{ d\rho } \left( \rho \frac { 1 }{ \pi \rho } \right) { a }_{ z }=0\) ( as expected)
