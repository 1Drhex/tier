Guest Question.

Use the results of Section 3.5 to prove that the wave function for which\( Δx ΔP_{x} = \hslash /2 must\) be a Gaussian.

Admin
Solution

In Section 3.5 we established the general uncertainty relation Δ A Δ B≥ | 〈C〉 | /2 for observables A and B for which the corresponding operators satisfied the commutation relation \({\hat{A},\hat{B} }=i\hat{C}\) . The derivation started with the Schwarz inequality

\(\left\langle \alpha |\alpha \right\rangle\left\langle \beta |\beta \right\rangle\gneq |\left\langle \alpha |\beta \right\rangle |^{2}\)

with \(|\alpha 〉=(\hat{A}-\left\langle A\right\rangle )|\psi 〉\) and \(|\beta 〉=(\hat{B}-\left\langle B\right\rangle )|\psi\).ΔAΔB = |〈C〉|/2 to hold, there are two requirements. One is that|β〉 =c|α〉, where cis a constant, so that the Schwarz inequality becomes an equality and \(〈\psi |\hat{F}|\psi =0\) , where \(\hat{F}=\hat{O}+\hat{O} ^{\dagger }\) with \(\hat{O}=\left(\hat{A}-\left\langle A\right\rangle \right)\left(\hat{B}-\left\langle B\right\rangle \right)\).

We can simplify things a bit by positioning the origin so that 〈x〉 = 0 and choosing a frame of reference for which\(\left\langle P_{x} \right\rangle = 0\). Then the two requirements become

\( c\hat{x}|\psi〉=\hat{p_{x} }|\psi 〉\)

and

\(\left\langle \psi |(\hat{X}\hat{p_{x} }+\hat{p_{x} }\hat{X} )|\psi \right\rangle=0\)

Projecting the first equation into P\(\varrho\) position space, we see that

\(cx\left\langle x|\psi \right\rangle=\frac{\hslash }{i}\frac{\partial \left\langle x|\psi \right\rangle}{\partial x}\)

which has the solution

\(\left\langle x|\psi \right\rangle=x^{icx^{2}/2\hbar }\)

The first requirement also shows us that

\( 〈\psi |\hat{p_{x} }=c^{\ast }〈\psi |\hat{x}\)

Using these results in the second requirement, we see that

\(\left\langle \psi |(\hat{x}c\hat{x}+c^{\ast }\hat{x}\hat{x} )|\psi \right\rangle=0\)

or

\((c+c^{\ast } )\left\langle \psi |\hat{x } ^{2}|\psi \right\rangle=0\)

Therefore \(c=-c^{\ast }\), indicating that c is purely imaginary, or c=i|c|.Thus

\(\left\langle x|\psi \right\rangle=e^{-|c|x^{2}/2\hbar }\)
