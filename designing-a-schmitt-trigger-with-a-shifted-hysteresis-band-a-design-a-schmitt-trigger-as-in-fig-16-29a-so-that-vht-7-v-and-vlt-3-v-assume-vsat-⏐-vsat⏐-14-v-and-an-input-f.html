Guest Question.

Designing a Schmitt trigger with a shifted hysteresis band

(a) Design a Schmitt trigger as in Fig. 16.29(a) so that \( V_{ Ht }=7 V \text { and } V_{ Lt }=3 V \text {. Assume } V_{ sat }=\left|-V_{ sat }\right|=14 V \) and an input frequency of f = 400 Hz. Determine the values of \( R_{1}, R_{ F }, \text { and } V_{\text {ref }} \).

(b) Use PSpice/SPICE to plot the hysteresis characteristic for \( v_{ S }=10 \sin (800 \pi t) \). Use the macromodel of the µA741 op-amp.

Admin
Solution

(a) The Schmitt trigger can be designed using the following steps:

Step 1. Find the values of \( R_{1} \text { and } R_{ F } \). From Eqs. (16.21) and (16.22), we can find the input width of the hysteresis band (HB) as follows:

\( V_{ Ht }=V_{ st }+\frac{R_{1}}{R_{1}+R_{ F }}\left(+V_{ sat }\right) \) (16.21)

 

\( V_{ Lt }=V_{ st }+\frac{R_{1}}{R_{1}+R_{ F }}\left(-V_{ sat }\right) \) (16.22)

 

\( HB =V_{ Ht }-V_{ Lt }=\frac{2 R_{1}}{R_{1}+R_{ F }} V_{ sat } \)

which gives

\( 1+\frac{R_{ F }}{R_{1}}=\frac{2 V_{ sat }}{V_{ Ht }-V_{ Lt }}=\frac{2 \times 14}{7-3}=7 \)

 

Let \( R_{1}=10 k \Omega ; \text { then } R_{ F }=(7-1) \times R_{1}=60 k \Omega.\)

Step 2. Determine the value of reference voltage \( V_{\text {ref }} \).

\( V_{ st }=\frac{R_{ F }}{R_{1}+R_{ F }} V_{ ref }=\frac{V_{ Ht }+V_{ Lt }}{2}=\frac{7+3}{2}=5 V \)

which gives \( V_{ ref }=5.83 V.\)

(b) We will use op-amp µA741 rather than comparator LM111 to illustrate the advantage of a comparator in a Schmitt trigger. The circuit for PSpice simulation is shown in Fig. 16.32.

The transfer characteristic and the output voltage (using EX16-7.SCH) are shown in Fig. 16.33[(a) and (b)], which gives \( V_{ Ht }=7.03 V \) (expected value is 7 V), \( V_{ Lt }=3.03 V \) (expected value is 3 V), and \( V_{\text {sat }}=14.6 \) V (expected value is 14 V). The transition from low to high and vice versa is not very sharp, as a result of the low slew rate of the op-amp (compared to that of a comparator, shown in Fig. 16.27).
