Guest Question.

Taking air resistance into account, the acceleration of a free falling body can be described approximately by \(a\left(v\right)=g-\alpha v^{2} \) . Here g is the gravity acceleration and \(\alpha\) a constant.

Determine the velocity \(v\left(t\right) \) of the body that is released from rest.

Admin
Solution

According to the table we have for a given a(v)

Given Sought
a = 0 \(v=v_{0}=const \) , \(x=x_{0}+v_{0}t \) uniform motion
\(a=a_{0}=const \) \(v=v_{0}+a_{0}t \) , \(x=x_{0}+v_{0}t+\frac{1}{2}a_{0}t^{2} \) uniform acceleration
\(a=a\left(t\right) \) \(v=v_{0}+\int\limits_{t_{0} }^{t}{a\left(\bar{t} \right)d\bar{t} } \) , \(x=x_{0}+\int\limits_{t_{0} }^{t}{v\left(\bar{t} \right)d\bar{t} } \)
\(a=a\left(v\right) \) \(t=t_{0}+\int\limits_{v_{0} }^{v}{\frac{d\bar{v} }{a\left(\bar{v} \right) }=f\left(v\right) } \) , \(x=x_{0}+\int\limits_{t_{0} }^{t}{F\left(\bar{t} \right) d\bar{t} } \) with the inverse function\(v=F\left(t\right) \)
\(a=a\left(x\right) \) \(v^{2}=v^{2}_{0}+2\int\limits_{x_{0} }^{x}{a\left(\bar{x} \right) d\bar{x} } \) , \(t=t_{0}+\int\limits_{x_{0} }^{x}{\frac{d\bar{x} }{v\left(\bar{x} \right) } }=g\left(x\right) \) the inverse function of \(t=g\left(x\right) \) gives \(x=G\left(t\right) \)

\(t=t_{0}+\int_{v_{0} }^{v}{\frac{d\bar{v} }{g-\alpha \bar{v}^{2} } } \) .

If the motion starts at \(t_{0}=0\) we obtain with the initial condition \(v\left(t_{0} \right)=v_{0}=0\)

 

\(t=\int_{0}^{v}{\frac{d\bar{v} }{g-\alpha \bar{v}^{2} } }=\frac{1}{\alpha } \int_{0}^{v}{\frac{d\bar{v} }{\left(\sqrt{\frac{g}{\alpha }}-\bar{v} \right)\left(\sqrt{\frac{g}{\alpha } }+\bar{v} \right) } }\)

and after partial fraction decomposition

\(t=\frac{1}{\alpha }\frac{1}{2\sqrt{\frac{g}{\alpha } } }\int_{0}^{v}{\left(\frac{1}{\sqrt{\frac{g}{\alpha } }-\bar{v} }+\frac{1}{\sqrt{\frac{g}{\alpha } }+\bar{v} } \right) }d\bar{v} \)

 

\(=\frac{1}{2\sqrt{g\alpha } }\left[-\ln \left(\sqrt{\frac{g}{\alpha } } -\bar{v} \right)+\ln \left(\sqrt{\frac{g}{\alpha } } +\bar{v}\right) \right]^{v}_{0} =\frac{1}{2\sqrt{g\alpha } }\ln \frac{\sqrt{\frac{g}{\alpha } }+v }{\sqrt{\frac{g}{\alpha } } -v} \) .

Solving for \(v\) yields

\(e^{2\sqrt{g\alpha }t }=\frac{\sqrt{\frac{g}{\alpha } }+v }{\sqrt{\frac{g}{\alpha } } -v} \rightsquigarrow \underline{\underline{v=\sqrt{\frac{g}{a} }\frac{e^{2\sqrt{g\alpha }t }-1 }{e^{2\sqrt{g\alpha }t }+1 } } } \) .

With the hyperbolic function \(\tanh \varphi =\frac{e^{\varphi }-e^{-\varphi } }{e^{\varphi }+e^{-\varphi } }=\frac{e^{2\varphi } -1}{e^{2\varphi } +1} \) the result also can be written as

\(\underline{\underline{v=\sqrt{\frac{g}{\alpha } } \tanh \sqrt{g\alpha }t } } \) .

From this last representation the limit value

\(\underset{t\rightarrow \infty }{\lim }v\left(t\right) =\sqrt{g/\alpha } \) .

can be recognized, i.e. after sufficiently large time the body practically falls with constant speed \(\left(a=0 \rightsquigarrow v=\sqrt{g/\alpha } \right) \) .
