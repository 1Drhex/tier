Guest Question.

Design a Douglas fir–Larch No. 1 beam to support the load shown.

Fb = 1,300 psi

Fv = 85 psi

E = 1.6 × 106 psi

\(\Delta_{allow_{(LL)}} = L/360\)
Admin
Solution
\(M_{max} = \frac{\omega L^{2}}{8} = \frac{(700lb./ft.)(16^{'})^{2}}{8} = 22,400 lb.ft.\)

 

\(S_{req'd} = \frac{M}{F_{b}} = \frac{(22,400lb.-ft)(12in./ft.)}{1300lb./in.^{2}} = 207in.^{3}\)

 

\(V_{max} = \frac{\omega L}{2} = \frac{(700lb.)(16ft.)}{2} = 5,600lb.\)

 

\(A_{req'd} = \frac{1.5V}{F_{v}} = \frac{1.5(5,600lb.)}{85lb./in.^{2}} = 98.8in.^{2}\)

 

Try : 8 × 14 S4S

(A = 101.25 in.2, Sx = 227.8 in.3, Ix = 1538in.4, ω = 25.5lb./ft.*)

*Note: ω = 0.252 × A (for Douglas Fir and Southern Pine)

 

\(V_{add.} = \frac{\omega _{bm}L}{2} = \frac{(25.5lb./ft.)(16^{'})}{2} = 204lb.\)

 

\(A_{add.} = \frac{1.5V_{add.}}{F_{v}} = \frac{1.5(204lb.)}{85lb./in.^{2}} = 3.6in.^{2}\)

 

\(A_{total} = A_{req'd} + A_{add.} = 98.8 in.^{2} + 3.6in.^{2} = 102.4in.^{2} \gt 101.3in.^{3}\)

 

The beam is over stressed in shear.

Try : 8 × 16 S4S

(A = 116.25 in.2, Sx = 300.3 in.3, Ix = 2327in.4, ω = 29.3lb./ft.)

 

\(\Delta_{allow.} = L/360 = \frac{(16ft.)(12in./ft.)}{360} = 0.53^{"}\)

 

\(\Delta_{\underset{(LL)}{actual} } = \frac{5\omega L^{4}}{384EI} = \frac{5(400lb./ft.)(16ft.)^{4}(1728in.^{3}/ft.^{3})}{384(1.6\times 10^{6} lb./in.^{2})(2327in.^{4})} = 0.16^{"} \lt 0.53^{"}\)

 

∴ OK for deflection . Use: 8 × 16 S4S
