Guest Question.

Using the Born approximation
Calculate the differential cross-section for scattering from a ‘Yukawa potential’, a potential (actually a potential energy) of the form:

\(V(r)=V_{0} \frac{e^{-\alpha r} }{r} \)

where \(V_{0}\) and α are constants. The Yukawa potential is another example of a central potential; it was originally introduced to represent the interaction between fundamental particles.

Method The first step is to insert the Yukawa potential into the Born approximation for the scattering amplitude (eqn 14.85), and then to evaluate the integral. Then, with \(f_{k}\) determined, the differential scattering cross-section can be determined by using eqn 14.15.

\(ƒ_{k} (\theta ,\varphi )\approx -\frac{1}{4\pi } \int{e^{-ik\hat{r}\cdot r^{\prime } } U(r^{\prime } )\psi ^{(0)}(r^{\prime } )dr^{\prime } } =-\frac{1}{4\pi } \int{e^{-ik\hat{r}\cdot r^{\prime } } U(r^{\prime } ) e^{ikz^{\prime } } dr^{\prime } }\) (14.85)

\(\sigma (\theta ,\varphi )=\left|ƒ_{k}(\theta ,\varphi ) \right| ^{2} \) (14.15)

Admin
Solution

Within the Born approximation, we have

\(ƒ_{k}(\theta ,\varphi ) =-\frac{1}{4\pi }\left(\frac{2\mu V_{0} }{\hbar ^{2} } \right) \int{e^{-ik\hat{r}\cdot r^{\prime } }\frac{e^{-\alpha r^{\prime } } }{r^{\prime }} e^{ikz^{\prime } } dr^{\prime } }\)

Because

\(-k\hat{r} \cdot r^{\prime } +kz^{\prime } =-k\hat{r} \cdot r^{\prime }+k\cdot r^{\prime }=(k-k\hat{r} )\cdot r^{\prime }=\left|k-k\hat{r}\right| r^{\prime }\cos \theta ^{\prime}\)

and \(dr^{\prime }=r^{\prime 2}\sin\theta ^{\prime }dr^{\prime }d\theta ^{\prime }d\varphi ^{\prime }\), this expression is

\(ƒ_{k}(\theta ,\varphi )=-\frac{1}{4\pi }\left(\frac{2\mu V_{0}}{\hbar ^{2}} \right)\int_{0}^{\infty }{\int_{0}^{\pi }{\int_{0}^{2\pi }{\frac {e^{-\alpha r^{\prime }}}{r^{\prime }}e^{i\left|k-k\hat{r} \right|r^{\prime }\cos \theta ^{\prime } }} } } r^{\prime 2}\sin\theta ^{\prime }dr^{\prime }d\theta ^{\prime }d\varphi ^{\prime }\)

Integration over \(\varphi ^{\prime }\) gives a factor of \(2\pi\). The integral over \(\theta ^{\prime }\) is

\(\int_{0}^{\pi }{e^{i\left|k-k\hat{r} \right|r^{\prime }\cos \theta ^{\prime } }}\sin\theta ^{\prime }d\theta ^{\prime }=\int_{-1}^{1}{e^{i\left|k-k\hat{r} \right|r^{\prime }\cos \theta ^{\prime } }d\cos \theta ^{\prime }} =\frac{e^{i\left|k-k\hat{r} \right|r^{\prime }}-e^{-i\left|k-k\hat{r} \right|r^{\prime }}}{i\left|k-k\hat{r} \right|r^{\prime }} =\frac{2\sin (\left|k-k\hat{r} \right|r^{\prime })}{\left|k-k\hat{r} \right|r^{\prime }} \)

Therefore,

\(ƒ_{k} (\theta ,\varphi )=-\left(\frac{2\mu V_{0} }{\hbar ^{2} } \right) \int_{0}^{\infty }{\frac{e^{-\alpha r^{\prime }}\sin(\left|k-k\hat{r} \right|r^{\prime }) }{\left|k-k\hat{r} \right|r^{\prime }} } dr^{\prime }=-\left(\frac{2\mu V_{0} }{\hbar ^{2} } \right)\frac{1}{\alpha ^{2}+ \left|k-k\hat{r} \right|^{2} } \)

To obtain this result we have used the standard integral

\(\int_{0}^{\infty }{e^{-ax} \sin bx dx} =\frac{b}{a^{2}+b^{2}} \)

Because \(\theta\) is the angle between the incident wavevector k and the unit vector \(\hat{r} \) in the scattered direction (see Fig. 14.17), it follows that

\(\left|k-k\hat{r} \right| =2k\sin \frac{1}{2}\theta\)

and therefore that

\(ƒ_{k} (\theta ,\varphi )=-\frac{2\mu V_{0} /\hbar ^{2}}{\alpha ^{2}+4k^{2}\sin^{2}\frac{1}{2}\theta } \)

Note that the scattering amplitude (and consequently the differential cross-section) is independent of the angle \(\varphi \)(though we continue to use the notations \(ƒ_{k} (\theta ,\varphi )\) and \(\sigma (\theta ,\varphi )\) even if they are independent of \(\varphi \)). This independence is a general result for elastic scattering by a central potential. It follows from eqn 14.15 that the differential cross-section is

\(\sigma (\theta ,\varphi )=\frac{4\mu ^{2}V^{2}_{0}/\hbar ^{4} }{(\alpha ^{2}+4k^{2}\sin^{2}\frac{1}{2}\theta)^{2}}\)

Comment 1 The differential cross-section varies with k, so it also varies with energy E. In the limit of zero energy (k → 0),

\(\sigma (\theta ,\varphi )=\frac{4\mu ^{2}V^{2}_{0} }{\hbar ^{4} \alpha ^{4}} \)

and is independent of θ as well as of φ. Except at zero energy, σ peaks in the forward direction (θ = 0) and decreases monotonically as θ varies from 0 to π (Fig. 14.18). Note that, within the Born approximation, the differential crosssection is independent of the sign of \(V_{0}\), and gives the same result if the Yukawa potential is attractive \((V_{0}\lt 0)\) or repulsive \((V_{0}\gt 0)\).

Comment 2 If the value of α is set to zero, the Yukawa potential becomes the Coulomb potential and the Born approximation results in the expression

\(\sigma (\theta ,\varphi )=\frac{\mu ^{2}V^{2}_{0} }{4\hbar ^{4} k ^{4}\sin^{4}\frac{1}{2}\theta} \)

It turns out to be the case that this expression is the same as for the quantum mechanical scattering differential cross-section for the Coulomb potential, a result known as Rutherford’s formula.
