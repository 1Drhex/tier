Guest Question.

Assembly Process

Given: The plane frame shown in Fig. E12.4a.

Determine: The system stiffness and nodal force matrices.

Admin
Solution

We first number the members and nodes as indicated in Fig. E12.4a, and then generate the member–node incidence table listed below.

 

Member–node incidence table

 

Member m Negative node\(n_{-}\) Positive node \(n_{+}\)
(1) 1 2
(2) 2 3
(3) 3 4

 

 

Using the member–node incident table, we replace the member end force and displacement matrices with

 

\(\begin{array}{ll}U _{ B } \Rightarrow U _{n_{+}} & P _{ B } \Rightarrow P _{n_{+}} \\U _{ A } \Rightarrow U _{n_{-}} & P _{ A } \Rightarrow P _{n_{-}}\end{array}\)

 

In (12.15), the operation is carried out for each member. The resulting expressions for the end forces expressed in terms of the nodal displacements are

 

\(\begin{array}{l}P _{(m) B }= k _{(m) BB } U _{(m) B }+ k _{(m) BA } U _{(m) A }+ P _{(m) B }^{ F } \\P _{(m) A }= k _{(m) AB } U _{(m) B }+ k _{(m) AA } U _{(m) A }+ P _{(m) A }^{ F }\end{array}\) (12.15)

 

\(\begin{array}{l}P _{(1) 2}= k _{(1) BB } U _{2}+ k _{(1) BA } U _{1}+ P _{(1) B }^{ F } \\P _{(1) 1}= k _{(1) AB } U _{2}+ k _{(1) AA } U _{1}+ P _{(1) A }^{ F }\end{array}\)

 

\(\begin{array}{l}P _{(2) 3}= k _{(2) BB } U _{3}+ k _{(2) BA } U _{2}+ P _{(2) B }^{ F } \\P _{(2) 2}= k _{(2) AB } U _{3}+ k _{(2) AA } U _{2}+ P _{(2) A }^{ F }\end{array}\)

 

\(\begin{array}{l}P _{(3) 4}= k _{(3) BB } U _{4}+ k _{(3) BA } U _{3}+ P _{(3) B }^{ F } \\P _{(3) 3}= k _{(3) AB } U _{4}+ k _{(3) AA } U _{3}+ P _{(3) A }^{ F }\end{array}\)

 

Next, we equate the external nodal force to the sum of the member forces at each node. This step leads to the nodal force equilibrium equations (see (12.18)).

 

\(P _{ E j}+\sum_{n_{+}=j}\left(- P _{(m) B }\right)+\sum_{n_{-}=j}\left(- P _{(m) A }\right)=0\) (12.18)

 

\(P _{ E 1}= P _{(1) 1}\)

 

\(P _{ E 2}= P _{(2) 2}+ P _{(1) 2}\)

 

\(P _{ E 3}= P _{(2) 3}+ P _{(3) 3}\)

 

\(P _{ E 4}= P _{(3) 4}\)

 

Substituting for the end forces expressed in terms of the nodal displacements, these equations expand to

 

\(P _{ E 1}= k _{(1) AA } U _{1}+ k _{(1) AB } U _{2}+ P _{(1) A }^{ F }\)

 

\(P _{ E 2}= k _{(1) BA } U _{1}+\left( k _{(1) BB }+ k _{(2) AA }\right) U _{2}+ k _{(2) AB } U _{3}+ P _{(2) A }^{ F }+ P _{(1) B }^{ F }\)

 

\(P _{ E 3}= k _{(2) BA } U _{2}+\left( k _{(2) BB }+ k _{(3) AA }\right) U _{3}+ k _{(3) AB } U _{4}+ P _{(3) A }^{ F }+ P _{(2) B }^{ F }\)

 

\(P _{ E 4}= k _{(3) BA } U _{3}+ k _{(3) BB } U _{4}+ P _{(3) B }^{ F }\)

 

Lastly, we write these equations as a single equation in terms of “system” matrices [see (12.19)]. The forms of the system matrices are listed below.

 

\(P _{ E }= K U + P _{ I }\) (12.19)

 

\(U =\left\{\begin{array}{l}U _{1} \\U _{2} \\U _{3} \\U _{4}\end{array}\right\}\) \(P _{ E }=\left\{\begin{array}{l}P _{ E 1} \\P _{ E 2} \\P _{ E 3} \\P _{ E 4}\end{array}\right\}\) \(P _{ I }=\left\{\begin{array}{c}P _{(1) A }^{ F } \\\left( P _{(1) B }^{ F }+ P _{(2) A }^{ F }\right) \\\left( P _{(2) B }^{ F }+ P _{(3) A }^{ F }\right) \\P _{(3) B }^{ F }\end{array}\right\}\)

 

\(K =\left[\begin{array}{cccc}k _{(1) AA } & k _{(1) AB } & 0 & 0 \\k _{(1) BA } & \left( k _{(1) BB }+ k _{(2) AA }\right) & k _{(2) AB } & 0 \\0 & k _{(2) BA } & \left( k _{(2) BB }+ k _{(3) AA }\right) & k _{(3) AB } \\0 & 0 & k _{(3) BA } & k _{(3) BB }\end{array}\right]\)

 

There are four nodes in this example, so the partitioned form of K is \(4 \times 4\). The expanded size of K for this two-dimensional plane frame will be \(12 \times 12\) since there are three variables per node.

 

In this example, we chose to list all the equations first and then combine them in a single “system” equation. Normally, one would apply the algorithms defined by (12.20) and (12.21) and directly assemble the system matrices.

 

\(\begin{aligned}&k _{(m) AA } \text { in row } n_{-}, \text {column } n_{-}\\&k _{(m) AB } \text { in row } n_{-}, \text {column } n_{+}\\&k _{(m) BA } \text { in row } n_{+}, \text {column } n_{-}\\&k _{(m) B } \text { in row } n_{+}, \text {column } n_{+}\end{aligned}\) (12.20)

 

\(\begin{array}{l}P _{(m) A }^{ F } \text { in row } n_{-} \\P _{(m) B }^{ F } \text { in row } n_{+}\end{array}\) (12.21)
