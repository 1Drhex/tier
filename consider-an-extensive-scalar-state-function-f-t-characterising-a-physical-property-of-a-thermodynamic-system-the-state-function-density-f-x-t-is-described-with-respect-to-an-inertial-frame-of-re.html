Guest Question.

Consider an extensive scalar state function F (t) characterising a physical property of a thermodynamic system. The state function density f (x, t) is described with respect to an inertial frame of reference \(\Re\) . The local spatial coordinates are the components of the vector x and the time coordinate is t.

1. In case the system is moving with a uniform velocity \(v_0\) with respect to the inertial frame of reference \(\Re\), we can define a new inertial frame of reference \(\acute{\Re }\) where the system is at rest. The Galilean transformation relates the coordinates of the frames of reference \(\Re\)
and \(\acute{\Re }\),

\(\acute{x }=x-ν_0 t\).

\(\acute{t }=t\).

The local spatial coordinates expressed with respect to \(\acute{\Re }\) are the components of the vector \(\acute{x }\) and the time coordinate is \(\acute{t }\). Determine the transformation law between the time derivative of the state function density \(∂_t f (x, t)\) expressed with respect to the frame of reference \(\Re\) and the time derivative of the state function density \(∂_{\acute{t}} f (\acute{x }, \acute{t })\) expressed with respect to the frame of reference \(\acute{\Re }\). Determine the transformation law between the spatial derivative of the state function density∇f (x, t) expressed with respect to the frame of reference \(\Re\) and the spatial derivative of the state function density\(\acute{∇ }f (\acute{x },\acute{t })\) expressed with respect to the frame of reference \(\acute{\Re }\).
2. Show that the time derivative of the state function density \(\dot{f} (x, t)\) given by equation (10.18) is a Galilean invariant by performing a Galilean transformation of uniform velocity \(v_0\) between the inertial frames of reference \(\Re\) and \(\acute{\Re }\), where the frame of reference \(\acute{\Re }\) is an arbitrary inertial frame of reference.

\(\dot{f} (x, t) = ∂_t f (x, t) + (v (x, t) ·∇) f (x, t)\).

Admin
Solution

1. Taking into account relation (10.11) and the uniformity of the velocity \(v_0\), the time derivative of the extensive state function \(F (\acute{t} )\) in the inertial frame of reference \(\Re\) is written as,

\(\dot{F} (t) = \int_{V(t)}{dV(x)} \biggl[\partial _t f(x,t)+\nabla .\biggl(f(x,t)\nu (x,t)\biggr) \biggr]\).

\(\dot{F}(t)=\int_{V(t)}{dV(x)}\biggl(∂_t f (x, t)+(ν_0.\triangledown) .f (x, t) \biggr)\).

According to relation (10.11), the time derivative of the extensive state function \(F (\acute{t} )\) in the inertial frame of reference \(\acute{\Re} \) where the system is at rest is expressed as,

\(\dot{F} (t) = \int_{V(t)}{dV(x)} \biggl[\partial _t f(x,t)+\nabla .\biggl(f(x,t)\nu (x,t)\biggr) \biggr]\).

\(\dot{F} (\acute{t} )=\int_{V(\acute{t} )}{dV(\acute{x} )} \partial _{\acute{t} } f(\acute{x},\acute{t} ) \).

Since the time coordinate is invariant under a Galilean transformation, \(\dot{F}(t)= \dot{F}(\acute{t})\), which implies that the integrands of the two previous equations are equal. This equality gives rise to the transformation law,

\( ∂_{\acute{t}} f (\acute{x}, \acute{t}) = ∂_t f (x, t) + (ν_0 ·∇) f (x, t) \).

Since the inertial frame of reference \(\acute{\Re }\) moves at velocity \(v_0\) with respect to the frame of reference \(\Re\), the frame of reference \(\Re\) moves with velocity \(−v_0\) with respect to the
frame \(\acute{\Re }\). If the spatial coordinates are permuted, i.e. \(x ↔ \acute{x}\), and the time coordinates are permuted as well, i.e. \(t ↔ \acute{t}\), then \(v_0 →−v_0\), which implies that,

\(∂_t f (x, t) =∂_{ \acute{t}} f (\acute{x}, \acute{t}) −(v_0 ·\acute{∇}) f (\acute{x}, \acute{t})\).

Adding the two previous equations yields the transformation law,

\(\acute{∇}f (\acute{x}, \acute{t})= ∇ f (x, t)\).

2. The time derivative of the Galilean transformation of the spatial coordinates yields the Galilean transformation of the velocities,

\(\acute{ν}= v − v_0\).

Taking into account definition (10.18), the transformation law for velocities and the spatial and time derivatives, we have,

\(\dot{f} (x, t) = ∂_t f (x, t) + (v (x, t) ·∇) f (x, t)\).

\( ∂_{\acute{t}} f (\acute{x}, \acute{t})+(\acute{ν}.\acute{∇})f (\acute{x}, \acute{t})=∂_t f (x, t) + (v ·∇) f (x, t) \).

This establishes the Galilean invariance of the state function density \(\dot{f} (x, t)\),

\(\dot{f} (\acute{x}, \acute{t})=\dot{f}(x, t)\).
