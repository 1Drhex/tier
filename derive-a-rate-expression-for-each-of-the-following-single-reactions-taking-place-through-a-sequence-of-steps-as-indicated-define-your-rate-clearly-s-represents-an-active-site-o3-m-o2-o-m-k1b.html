Guest Question.

Derive a rate expression for each of the following single reactions taking place through a sequence of steps as indicated. Define your rate clearly. S represents an active site.

a.\(O_{3}+M \xrightarrow[]{k_{1} } O_{2}+O+M\)

 

\(O+O_{3} \xrightarrow[]{k_{2} } 2 \ O_{2} \)

————————————————————–

\(2 \ O_{3 } \Longrightarrow 3 \ O_{2}\)

 

b.\(N_{2}O + S \xrightarrow[]{k_{1} } N_{2}+S-O\)

 

\(CO + S-O \xrightarrow[]{k_{2} } CO_{2}+ S\)

————————————————————–

\(N_{2}O+ CO \Longrightarrow N_{2}+ CO_{2}\)

 

c. \(C_{2}H_{6 \ \ \left(g\right) } \xrightleftharpoons[]{K} C_{2}H_{2 \ \ \left(ad\right) } + 2 H_{2 \ \left(g\right) } \)

 

\(C_{2}H_{2 \ \ \left(ad\right) }+ H_{2 \ \left(g\right) } \xrightarrow[]{k_{1} } 2 CH_{2 \ \left(ad\right) }\)

 

\(2\left[ CH_{2 \ \left(ad\right) } + H_{2 \ \left(g\right) } \xrightarrow[]{k_{2} } CH_{4 \ \left(g\right) } \right] \)

————————————————————–

\(C_{2}H_{6} + H_{2} \Longrightarrow 2 \ CH_{4}\)

Assume single-site adsorption and that \(C_{2}H_{2 \ \ \left(ad\right) }\) is the MARI. This last sequence is shown as originally written in the literature and can be ambigu-ous. Rewrite it using defined active sites and derive a rate expression.

Admin
Solution

(a) SSA states that \(d\left[O\right]/dt = 0 \Rightarrow r_{1}=r_{2} \therefore r = – d \left[O_{3} \right] /dt\)

\(\frac{- d\left[O_{3} \right]}{dt} =r = 2 r_{1}= 2k_{1} \left[O_{3}\right]\left[M\right]\) or \(\frac{ d\left[O_{2} \right]}{dt}= 3 k_{1} \left[O_{3}\right] \left[M\right] \)

(b) Cannot assume Quasi-Equilibrated Adsorption (No Langmuir isotherm)

\(r= \frac{d \left[CO_{2} \right]}{dt} \) ; @ steady state, so \( r_{1}= r_{2}\) , Site Balance: \(L= \left[S-O\right]+ \left[S\right]\)

 

\(k_{1}\left[N_{2} O\right] \left[S\right] = k_{2} \left[CO\right] \left[S-O\right] \Rightarrow \left[S-O\right]= \left(k_{1}/k_{2} \right) \left[N_{2} O\right]\left[S\right]/\left[CO\right]\)

 

\(r=k_{2}\left[CO\right] \left[S-O\right]= Lk_{1}k_{2} \left[CO\right] \left[N_{2} O\right]/\left(k_{1} \left[N_{2} O\right] + k_{2} \left[CO\right] \right) \)

 

(c) \(r=d \left[CH_{4} \right]/dt= k_{2}\left[CH_{2 \ ad} \right] \left[H_{2} \right]\) & \(K=\frac{\left[C_{2} H_{2 \ ad} \right] \left[H_{2} \right]^{2} }{\left[C_{2} H_{6} \right] \left[S\right] } \)

 

I. Literature solution for reaction as written

If \(\left[C_{2} H_{2 \ ad} \right] \) is MARI , then \(L=\left[C_{2} H_{2 \ ad} \right]+ \left[S\right] \) & \(\left[C_{2} H_{2 \ ad} \right]=K \left[C_{2} H_{6} \right] \left[S\right]/ \left[H_{2} \right]^{2} = \frac{K \left[C_{2} H_{6} \right]}{\left[H_{2} \right]^{2}} \left(L-\left[ C_{2} H_{2 \ ad}\right] \right)\)

 

\(\left[C_{2} H_{2 \ ad} \right]\left(1+ K \left[C_{2} H_{6} \right]/ \left[H_{2} \right]^{2} \right) = \frac{L \ K \left[C_{2} H_{6} \right]}{\left[H_{2} \right]^{2}} \Rightarrow \left[C_{2} H_{2 \ ad} \right]= \frac{LK \left[C_{2} H_{6} \right]}{ \left[H_{2} \right]^{2} \left(1+ K \left[C_{2} H_{6} \right]\right)/ \left[H_{2} \right]^{2}} \)

 

@ Steady State \(\frac{d\left\lfloor CH_{2 \ ad} \right\rfloor }{dt} =0 : \frac{d \varepsilon }{dt} = \frac{1}{2}\frac{d \left\lfloor CH_{2 \ ad} \right\rfloor }{dt} = k_{1} \left[C_{2} H_{2 \ ad}\right] \left[H_{2} \right]\)

 

\(d\varepsilon /dt = \frac{d\left\lfloor CH_{2 \ ad} \right\rfloor }{dt}=k_{2} \left[ CH_{2 \ ad}\right] \left[H_{2} \right] \Rightarrow 2r_{1}= r_{2}\)

 

\(2 \ k_{1} \left[C_{2}H_{2 \ ad} \right] \left[H_{2} \right]= k_{2} \left[CH_{2 \ ad} \right]\left[H_{2} \right] \Rightarrow \left[CH_{2 \ ad} \right] =2 \left(k_{1}/k_{2} \right) \left[C_{2}H_{2} \right] _{ad} \)

 

\(d\left[CH_{4} \right]/dt = k_{2} \left[CH_{2 \ \left(ad\right) } \right] \left[H_{2} \right]= \frac{2L k_{1} K \left[ C_{2}H_{6} \right] \left[H_{2} \right] }{\left[H_{2} \right]^{2} + K \left[ C_{2}H_{6} \right] } = \frac{2L k_{1} K \left[ C_{2}H_{6} \right] }{\left[H_{2} \right] + K \frac{ \left[ C_{2}H_{6} \right]}{ \left[H_{2} \right]}}\)

 

II. Alternate solution for reaction written with well defined sites, S, i.e.,

(1) \(C_{2}H_{6 \ \left(g\right) } \xrightleftharpoons[]{K} C_{2}H_{2}-S+ 2 H_{2 \ \left(g\right)} \)

(2) \(C_{2}H_{2}-S+ H_{2 \ \left(g\right)} +S \xrightarrow[]{ k_{1}} 2 CH_{2}-S \)

(2) \(2 \left[ CH_{2}-S + H_{2 \ \left(g\right)} \xrightarrow[]{ k_{2}} 2 CH_{4} \right] \)

——————————————————–

\(C_{2}H_{6} + H_{2}\Longrightarrow 2 CH_{4}\)

 

Note that a \(CH_{2}\) group almost certainly requires a site for itself as shown in step 2.

\(r= \frac{d\left[CH_{4}\right] }{dt} = k_{2} \left[ C_{2}H_{2}-S \right]\left[ H_{2} \right] \) & \(K= \frac{ \left[ C_{2}H_{2}-S \right] \left[ H_{2} \right]^{2} }{ \left[ C_{2}H_{2}-S \right] \left[S \right] }\)

 

If \(\left[ C_{2}H_{2}-S \right]\) is the MARl, then \(L= \left[ C_{2}H_{2}-S \right] + \left[S \right]\) for the site balance, and

\(\left[ C_{2}H_{2}-S \right] = K \left[ C_{2} H_{6} \right] \left[S \right]/ \left[ H_{2} \right]^{2} = \left(K \left[ C_{2}H_{2} \right] / \left[ H_{2} \right]^{2} \right) \left(L-\left[ C_{2}H_{2}-S \right] \right) \Rightarrow \left [ C_{2}H_{2}-S \right]\left(1+K \left[ C_{2}H_{6} \right] /\left[ H_{2} \right]^{2}\right) = LK \left[ C_{2}H_{6} \right]/ \left[ H_{2} \right]^{2}\) & \( \left[ C_{2}H_{2}-S \right]= \frac{LK\left[ C_{2}H_{6} \right] }{ \left[ H_{2} \right]^{2} \left(1+K \left[ C_{2}H_{6} \right] / \left[ H_{2} \right]^{2} \right) } \)

 

@ Steady state: \(\frac{d \left[ CH_{2}-S \right]}{dt}=0\) & \(\frac{d\varepsilon }{dt}= \frac{1}{2 } \frac{d \left[ CH_{2}-S \right] }{dt}=k_{1} \left[ CH_{2}-S \right]\left[H_{2}\right]\left[S\right]\) & \(\frac{ d \varepsilon }{dt}= \frac{- d \left[ CH_{2}-S \right] }{dt}=k_{2} \left[ CH_{2}-S \right]\left[ H_{2} \right] \Rightarrow 2 r_{1} = r_{2}\) , so

\(2 k_{1} \left[ C_{2}H_{2}-S \right] \left[ H_{2} \right] \left[S\right]= k_{2} \left[ CH_{2}-S \right]\left[ H_{2} \right] \Rightarrow \left[ CH_{2}-S \right] = 2 \left(k_{1}/k_{2}\right)\left[ C_{2}H_{2}-S \right]\left[S\right] \)

 

\(\left[S\right]= L \left[ H_{2} \right]^{2} /\left( \left[ H_{2} \right]^{2} + K \left[ C_{2}H_{6} \right] \right) \) so

\(\frac{d \left[ CH_{4} \right]}{dt} = k_{2} \left[ CH_{2}-S \right]\left[ H_{2} \right]= \left\lgroup\frac{Z}{2L} \right\rgroup 2 L^{2} k_{1}K \left[ C_{2}H_{6} \right] \left[ H_{2} \right]^{3}/ \left( \left[ H_{2} \right]^{2} + K \left[ C_{2}H_{6} \right]\right) ^{2} = \frac{k^{\prime }\left[ C_{2}H_{6} \right] \left[ H_{2} \right]^{3} }{ \left( \left[ H_{2} \right]^{2} + K \left[ C_{2}H_{6} \right]\right) ^{2}} = \frac{k^{\prime }\left[ C_{2}H_{6} \right]}{ \left[ H_{2} \right] \left\lgroup 1+ \frac{K \left[ C_{2}H_{6} \right] }{\left[ H_{2} \right]^{2}} \right\rgroup ^{2} } =r\)
