Guest Question.

Equation 9.23 tells us the (approximate) transmission probability for tunneling through a barrier, when \( E>V_{\max } \) (again, a classically forbidden process). We’ll assume that V(x) is an even analytic function, that goes to zero as x → ± ∞ (Figure 9.15). Question: What is the analog to Equation 9.23?

\( T \approx e^{-2 \gamma}, \quad \text { with } \quad \gamma \equiv \frac{1}{\hbar} \int_{0}^{a}|p(x)| d x . \) (9.23).

(a) Try the obvious approach: assume the potential vanishes for \( |x| \geq a \) and
use the WKB approximation (Equation 9.13) in the scattering region:

\( \psi(x) \approx \frac{1}{\sqrt{p(x)}}\left[C_{+} e^{i \phi(x)}+C_{-} e^{-i \phi(x)}\right] \) (9.13)

\( \psi(x) \begin{cases}=A e^{i k x}+B e^{-i k x}, & x<a \\ \approx \frac{1}{\sqrt{p(x)}}\left[C_{+} e^{i \phi(x)}+C_{-} e^{-i \phi(x)}\right], & -a<x<a \\ =C e^{i k x}, & x>a\end{cases} \) (9.66).

Impose the usual boundary conditions at ± a ,and solve for the reflection probability, \( R=|B|^{2} /|A|^{2} \).

Unfortunately, the result (R=0) is uninformative. It’s true that the R is exponentially small (just as the transmission coefficient is, for \( \left.E<V_{\max }\right) \) , but we’ve thrown the baby out with the bath water—this approximation is simply too drastic. The correct formula is

\( R=e^{-2 \lambda}, \quad \text { where } \quad \lambda \equiv \frac{2}{\hbar} \int_{0}^{y_{0}} p(i y) d y \) (9.67).

\( \text { and } y_{0} \text { is defined by } p\left(i y_{0}\right)=0 \text {. Notice that } \lambda \text { (like } γ \) in Equation 9.23) goes like \( 1 / \hbar \); it is in fact the leading term in an expansion in powers of \(\hbar \): \( \lambda=c_{1} / \hbar+c_{2}+c_{3} \hbar+c_{4} \hbar^{2}+\cdots \text { In the classical limit }(\hbar \rightarrow 0) \text {, } \lambda \) and γ go to infinity, so R and T go to zero, as expected. It is not easy to derive Equation 9.67, but let’s look at some examples.

\( T \approx e^{-2 \gamma}, \quad \text { with } \quad \gamma \equiv \frac{1}{\hbar} \int_{0}^{a}|p(x)| d x . \) (9.23).

(b) Suppose \( V(x)=V_{0} \operatorname{sech}^{2}(x / a) \), for some positive constants \( V_0 \) and a. Plot \( V(x), \quad \text { plot } p(i y) \quad \text { for } \quad 0 \leq y \leq y_{0} \), and show that \( \lambda=(\pi a / \hbar)\left(\sqrt{2 m E}-\sqrt{2 m V_{0}}\right) \). Plot R as a function of E, for fixed \( V_0 \) .

(c) Suppose \( V(x)=V_{0} /\left[1+(x / a)^{2}\right] . \text { Plot } V(x), \text { and express } \lambda \) in terms of an elliptic integral. Plot R as a function of E.

Solution

(a) \( \phi(x)=\frac{1}{\hbar} \int_{0}^{x} p\left(x^{\prime}\right) d x^{\prime} ; \quad \phi^{\prime}(x)=\frac{1}{\hbar} p(x) ; \quad p(x)=\sqrt{2 m(E-V(x))} . \quad \text { But } V(x)=0 \text { for }|x| \geq a, \text { so } \),

\( p(\pm a)=\sqrt{2 m E}=\hbar k ; \quad \phi^{\prime}(\pm a)=k ; \quad p^{\prime}(\pm a)=0 \) (the latter follows because p(x) is constant for |x| ≥ a , so it’s derivative is zero in that region), and by assumption V (x) is smooth).

Boundary conditions at -a:

\( \left(A e^{-i k a}+B e^{i k a}\right)=\frac{1}{\sqrt{p(-a)}}\left[C_{+} e^{i \phi(-a)}+C_{-} e^{-i \phi(-a)}\right] \).

\( i k\left(A e^{-i k a}-B e^{i k a}\right)=\frac{i}{\sqrt{p(-a)}}\left[C_{+} e^{i \phi(-a)}-C_{-} e^{-i \phi(-a)}\right] \phi^{\prime}(-a) \).

(There’s another term in the second equation, where \( 1 / \sqrt{p(x)} \) is differentiated, but since \( \frac{d}{d x}\left(\frac{1}{\sqrt{p(x)}}\right)=-\frac{1}{2} \frac{1}{(p(x))^{3 / 2}} p^{\prime}(x), \text { and } p^{\prime}(-a)=0 \) it contributes nothing.) Meanwhile, using \( \phi^{\prime}(-a)=k \), the second equation simplifies to

\( \left(A e^{-i k a}-B e^{i k a}\right)=\frac{1}{\sqrt{p(-a)}}\left[C_{+} e^{i \phi(-a)}-C_{-} e^{-i \phi(-a)}\right] \);

Subtracting this from the first equation, we find:

\( B e^{i k a}=\frac{1}{\sqrt{p(-a)}} C_{-} e^{-i \phi(-a)} \).

Boundary conditions at a:

\( C e^{i k a}=\frac{1}{\sqrt{p(a)}}\left[C_{+} e^{i \phi(a)}+C_{-} e^{-i \phi(a)}\right] \).

\( i k C e^{i k a}=\frac{i}{\sqrt{p(a)}}\left[C_{+} e^{i \phi(a)}-C_{-} e^{-i \phi(a)}\right] \phi^{\prime}(a) \).

(Again, the other term in the second equation is zero.) Thus

\( C e^{i k a}=\frac{1}{\sqrt{p(a)}}\left[C_{+} e^{i \phi(a)}-C_{-} e^{-i \phi(a)}\right] \).

Subtracting this from the first equation yields

\( 0=\frac{1}{\sqrt{p(a)}} C_{-} e^{-i \phi(a)} \quad \Rightarrow \quad C_{-}=0 \).

Combining the results of the boundary conditions at -a and at +a, B = 0, and hence R = 0:

(b)

\( \operatorname{Plot}\left[(\operatorname{Sech}[ x ])^{2},\{ x ,-4,4\}, \text { PlotRange } \rightarrow\{0,1\}\right] \).

\( \operatorname{sech}(i z)=\frac{1}{\cos (z)}, \text { so } p(i y)=\sqrt{2 m\left(E-\frac{V_{0}}{\cos ^{2}(y / a)}\right)} \), Here’s the graph:

\( \text { Plot }\left[\sqrt{4-(1 / \operatorname{Cos}[x])^{\wedge} 2},\{x, 0,1.5\}, \text { PlotRange } \rightarrow\{0,2\}\right] \).

\( \lambda=\frac{2}{\hbar} \int_{0}^{y_{0}} p(i y) d y=\frac{2}{\hbar} \int_{0}^{y_{0}} \sqrt{2 m\left(E-\frac{V_{0}}{\cos ^{2}(y / a)}\right)} d y=\frac{2 a}{\hbar} \sqrt{2 m V_{0}} \int_{0}^{u_{0}} \sqrt{\beta^{2}-\frac{1}{\cos ^{2} u}} d u \).

\( \text { (where } \left.\beta \equiv \sqrt{E / V_{0}} \text { and } u \equiv y / a\right) \).

\( =\frac{2 a}{\hbar} \sqrt{2 m V_{0}} \int_{0}^{u_{0}} \frac{\sqrt{\beta^{2} \cos ^{2} u-1}}{\cos u} d u . \text { Now let } z \equiv \sin u, \text { so } d z=\cos u d u: \)

\( =\frac{2 a}{\hbar} \sqrt{2 m V_{0}} \int_{0}^{z_{0}} \frac{\sqrt{\beta^{2}\left(1-z^{2}\right)-1}}{1-z^{2}} d z=\frac{2 a}{\hbar} \sqrt{2 m V_{0}} \beta \int_{0}^{z_{0}} \frac{\sqrt{z_{0}^{2}-z^{2}}}{1-z^{2}} d z, \text { where } z_{0}^{2} \equiv 1-\left(1 / \beta^{2}\right) \).

Mathematica says this integral is \( \frac{\pi}{2}\left(1-\sqrt{1-z_{0}^{2}}\right)=\frac{\pi}{2}\left(1-\frac{1}{\beta}\right) \).

\( =\frac{2 a}{\hbar} \sqrt{2 m V_{0}} \frac{\pi}{2}(\beta-1)=\frac{\pi a}{\hbar} \sqrt{2 m V_{0}}\left(\sqrt{\frac{E}{V_{0}}}-1\right)=\frac{\pi a}{\hbar}\left(\sqrt{2 m E}-\sqrt{2 m V_{0}}\right) \).

\( \lambda=\frac{2}{\hbar} \int_{0}^{y_{0}} \sqrt{2 m\left(E-\frac{V_{0}}{1-(y / a)^{2}}\right)} d y=\frac{2 a}{\hbar} \sqrt{2 m V_{0}} \int_{0}^{u_{0}} \sqrt{\beta^{2}-\frac{1}{1-u^{2}}} d u \).

\( =\frac{2 a \beta}{\hbar} \sqrt{2 m V_{0}} \int_{0}^{u_{0}} \frac{\sqrt{u_{0}^{2}-u^{2}}}{\sqrt{1-u^{2}}} d u, \text { where } u_{0}^{2}=1-\left(1 / \beta^{2}\right) \).
