Guest Question.

Construct the Bode plot for the transfer function \(H(s)=100,000 s /\left[( s +1)\left(10,000+20 s + s ^{2}\right)\right]\).

Admin
Solution

Let’s consider the quadratic factor first and arrange it in a form such that we can see the value of ζ . We begin by dividing the second-order factor by its constant term, 10,000:

 

\(H ( s )=\frac{10 s }{(1+ s )\left(1+0.002 s +0.0001 s ^{2}\right)}\)

 

An inspection of the \(s ^{2}\) term next shows that \(\omega_{0}=\sqrt{1 / 0.0001}=100\).

Then the linear term of the quadratic is written to display the factor 2, the factor \(\left( s / \omega_{0}\right)\), and finally the factor ζ :

 

\(H ( s )=\frac{10 s }{(1+ s )\left[1+(2)(0.1)( s / 100)+( s / 100)^{2}\right]}\)

 

We see that ζ = 0.1.

The asymptotes of the magnitude-response curve are sketched in lightly in Fig. 16.31: 20 dB for the factor of 10, an infinite straight line through ω = 1 with a +20 dB/decade slope for the s factor, a corner at ω = 1 for the simple pole, and a corner at ω = 100 with a slope of −40 dB/decade for the second-order term in the denominator. Adding these four curves and supplying a correction of +14 dB for the quadratic factor lead to the heavy curve of Fig. 16.31.

The phase response contains three components: \(+90^{\circ}\) for the factor s; \(0^{\circ} \text { for } \omega<0.1,-90^{\circ} \text { for } \omega>10 \text { , and }-45^{\circ}\)/decade for the simple pole; and \(0^{\circ} \text { for } \omega<10,-180^{\circ} \text { for } \omega>1000 \text { , and }-90^{\circ}\) per decade for the quadratic factor. The addition of these three asymptotes plus some improvement for ζ = 0.1 is shown as the solid curve in Fig. 16.32.

PRACTICE

16.15 If \(H ( s )=1000 s ^{2} /\left( s ^{2}+5 s +100\right)\), sketch the Bode amplitude plot and calculate a value for (a) ω when \(H_{ dB }=0 ; \text { (b) } H_{ dB } \text { at } \omega=1; \text { (c) } H_{ dB } \text { as } \omega \rightarrow \infty\).

Ans: 0.316 rad/s; 20 dB; 60 dB.
