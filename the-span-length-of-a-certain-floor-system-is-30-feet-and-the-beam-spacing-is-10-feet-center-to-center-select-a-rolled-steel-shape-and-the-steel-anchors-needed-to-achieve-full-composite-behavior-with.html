Guest Question.

The span length of a certain floor system is 30 feet, and the beam spacing is 10 feet center-to-center. Select a rolled steel shape and the steel anchors needed to achieve full composite behavior with a 3.5-inch-thick reinforced concrete floor slab. Superimposed loading consists of a 20 psf partition load and a 100 psf live load. Concrete strength is \(f'_c = 4 ksi,\) and A992 steel is to be used. Assume that the beam has full lateral support during construction and that there is a 20 psf construction load.

Admin
Solution

Loads to be supported before the concrete cures are
Slab: (3.5/12)(150) = 43.75 psf
Weight per linear foot: 43.75(10) = 437.5 lb/ft
Construction load: 20(10) = 200 lb/ft

(The beam weight will be accounted for later.)
Loads to be supported after the concrete cures are
\(w_D = w_{slab} = 437.5 lb/ft\)
\(w_L = (100 + 20)(10) = 1200 lb/ft\)
where the 20 psf partition load is treated as a live load.
The composite section must resist a factored load and moment of
\(w_u = 1.2w_D + 1.6wL = 1.2(437.5) + 1.6(1200) = 2445 lb/ft\)
\(M_u = \frac{1}{8}w_uL^2 = \frac{1}{8}(2.445)(30)^2 = 275 ft-kips\)
Try a nominal depth of d = 16 inches: The internal moment arm is
\(y = \frac{d}{2} + t – \frac{a}{2} = \frac{16}{2} + 3.5 – 0.5 = 11 in.\)
From Equation 9.4, the estimated beam weight is
\(w = \frac{M_u}{y} = \frac{275}{11} = 25 lb/ft\)
Try a W16 \(\times \)26: Check the unshored steel shape for loads applied before the concrete cures (the weight of the slab, the weight of the beam, and the construction load).
\(w_u = 1.2(0.4375 + 0.026) + 1.6(0.200) = 0.8762 kips/ft\)
\(M_u = \frac{1}{8}(0.8762)(30)^2 = 98.6 ftÂ­kips\)
From the \(Z_x \)table,
\(\phi _{b}M_n = \phi _{b}M_p = 166 ft-kips > 98.6 ft-kips (OK)\)
After the concrete cures and composite behavior has been achieved,
\(w_D = w_{slab} + w_{beam} = 0.4375 + 0.026 = 0.4635 kips/ft\)
\(w_u = 1.2w_D + 1.6w_L = 1.2(0.4635) + 1.6(1.200) = 2.476 kips/ft\)
\(M_u = \frac{1}{8}w_uL^2 = \frac{1}{8}(2.476)(30)^2 = 279 ft-kips\)

Before computing the design strength of the composite section, we must first determine the effective slab width. For an interior beam, the effective width is the smaller of
\(\frac{Span}{4} = \frac{30(12)}{4} = 90\) in. or Beam spacing = 10(12) = 120 in.

Use b = 90 in. For full composite behavior, the compressive force in the concrete at ultimate (equal to the horizontal shear at the interface between the concrete and steel) will be the smaller of
\(A_sF_y = 7.68(50) = 384 kips \)
or
\(0.85f'_cA_c = 0.85(4)(90)(3.5) = 1071 kips\)
Use C = V’ = 384 kips. The depth of the compressive stress block in the slab is
\(a = \frac{C}{0.85f'_c b}= \frac{384}{0.85(4)(90)}=1.255 in.\)
and the moment arm of the internal resisting couple is
\(y =\frac{d}{2} + t – \frac{a}{2} = \frac{15.7}{2} + 3.5 – \frac{1.255}{2}= 10.72 in.\)
The design flexural strength is
\(\phi _{b}M_n = \phi _{b}(Cy) = 0.90(384)(10.72)\)
\(= 3705 in.-kips = 309 ft-kips > 279 ft-kips (OK)\)
Check the shear:
\(V_u = \frac{w_uL}{2} =\frac{ 2.476(30)}{2} = 37.1 kips\)
From the\( Z_x\) table,
\(\phi _{v}V_n = 106 kips > 37.1 kips (OK)\)
Use a W16 \(\times\) 26.
Design the steel anchors.
Maximum diameter = \(2.5t_f = 2.5(0.345) = 0.863 in. < 1 in.\)

Try 1â2 inch \(\times \)2 inch studs:
d = 1/2 in. < 0.863 in. (OK)
\(A_{sa} =\frac{\pi d^2}{4} = \frac{\pi (0.5)^2}{4} = 0.1963 in.^2\)
For normal weight concrete,
\(E_c = w^{1.5}\sqrt{f'_c}=(145)^{1.5}\sqrt{4}= 3492 ksi\)
From AISC Equation I3-3,
\(Q_n = 0.5A_{sa}\sqrt{f'_cE_c} \leq R_gR_pA_{sa}F_u\)
\(= 0.5(0.1963)\sqrt{4(3492)} = 11.60kips\)
\(R_gR_pA_{sa}F_u = 1.0(0.75)(0.1963)(65)\)
= 9.570 kips < 11.60 kips \(\therefore\) Use \(Q_n = 9.570 kips.\)
The number of studs required between the end of the beam and midspan is
\(N_1 = \frac{V'}{Q_n}= \frac{384}{9.570} = 40.1 \therefore \)Use 41 for half the beam, or 82 total.
and
Minimum longitudinal spacing is 4d = 4(0.5) = 2 in.
Minimum transverse spacing is 4d = 4(0.5) = 2 in.
Maximum longitudinal spacing is 8t = 8(3.5) = 28 in. (< 36 in. upper limit)
If one stud is used at each section, the approximate spacing will be
\(s =\frac{30(12)}{82} = 4.4 in.\)
This spacing is between the upper and lower limits and is therefore satisfactory. Use the design shown in Figure 9.13. The composite section must resist a service load and moment of
\(w_a = w_D + w_L = 437.5 + 1200 = 1638 lb/ft\)
\(M_a = \frac{1}{8}w_aL^2 = \frac{1}{8}(1.638)(30)^2 = 184.3 ft-kips\)

Try a nominal depth of d = 16 inches: The internal moment arm is
\(y = \frac{d}{2} + t – \frac{a}{2} = \frac{16}{2} + 3.5 – 0.5 = 11 in.\)
From Equation 9.5, the estimated beam weight is
\(w = \frac{1.4M_a}{y} = \frac{1.4(184.3)}{11} = 23 lb/ft\)
Try a W16 \(\times\) 26: Check the unshored steel shape for loads applied before the concrete cures (the weight of the slab, the weight of the beam, and the construction load).
\(w_a = w_{slab} + w_{beam} + w_{const} = 0.4375 + 0.026 + 0.200 = 0.6635 lb/ft\)
\(M_a = \frac{1}{8}w_aL^2 = \frac{1}{8}(0.6635)(30)^2 = 74.6 ft-kips\)
From the \(Z_x \)table,
\(\frac{M_n}{\Omega _{b}}= \frac{M_p}{\Omega _{b}}\)
= 110 ft-kips > 74.6 ft-kips (OK)
After the concrete cures and composite behavior has been achieved, the load and moment are
\(w_a = w_{slab} + w_{beam} + w_{part} + w_L = 0.4375 + 0.026 + 0.200 + 1.000 = 1.664 kips/ft\)
\(M_a = \frac{1}{8}w_aL^2 = \frac{1}{8}(1.664)(30)^2 = 187 ft-kips\)
The effective slab width for a typical interior beam is the smaller of
\(\frac{Span}{4} = \frac{30 \times 12}{4} = 90 in.\) or Beam spacing = 10 \(\times 12 = 120 in.\)

Use b = 90 in. For full composite behavior, the compressive force in the concrete at ultimate (equal to the horizontal shear at the interface between the concrete and steel) will be the smaller of
\(A_sF_y = 7.68(50) = 384 kips\)
or
\(0.85 f'_cA_c = 0.85(4) (3.5 \times 90) = 1071 kips\)
Use C = 384 kips.
\(a = \frac{C}{0.85f'_c b}= \frac{384}{0.85(4)(90)}= 1.255 in.\)
\(y = \frac{d}{2} + t – \frac{a}{2} = \frac{15.7}{2} + 3.5 – \frac{1.255}{2} = 10.72 in.\)
The allowable flexural strength is
\(\frac{M_n}{\Omega _{b}}= \frac{Cy}{\Omega _{b}}\)
\(=\frac{ 384(10.72)}{1.67} = 2465 in.-kips = 205 ft-kips > 187 ft-kips (OK)\)
Check shear:
\(V_a = \frac{w_aL}{2} = \frac{1.664(30)}{2}= 25.0 kips\)
From the \(Z_x \)table,
\(\frac{V_n}{\phi _{v}}= 70.5 kips > 25.0 kips (OK)\)
Use a W16 \(\times\) 26.
The ASD design of the steel anchors is identical to what was done for LRFD and will not be repeated here.
Use the design shown in Figure 9.13.
