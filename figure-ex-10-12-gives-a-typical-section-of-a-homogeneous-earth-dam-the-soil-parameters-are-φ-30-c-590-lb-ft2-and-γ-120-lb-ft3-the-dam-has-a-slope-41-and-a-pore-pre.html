Guest Question.

Figure Ex. 10.12 gives a typical section of a homogeneous earth dam. The soil parameters are: \(\phi^{\prime}=30^{\circ}, c^{\prime}=590 lb / ft ^{2}, \text { and } \gamma=120 lb / ft ^{3}\). The dam has a slope 4:1 and a pore pressure ratio \(r_{u}=0.5\). Estimate the factor of safety Fs by Bishop and Morgenstern method for a height of dam H = 140 ft.

Admin
Solution

Height of dam H= 140ft

 

\(\frac{c^{\prime}}{\gamma H}=\frac{590}{120 \times 140}=0.035\)

 

Given: \(\phi^{\prime}=30^{\circ}, \text { slope } 4: 1 \text { and } r_{u}=0.5\).

 

Since \(c^{\prime} / \gamma H=0.035, \text { and } n_{d}=1.43 \text { for } H=140 ft \text {, the } F_{s}\) for the dam lies between \(c^{\prime} / \gamma H=0.025\) and 0.05 and \(n_{d}\) between 1.0 and 1.5. The equation for \(F_{s}\) is

 

\(F_{s}=m-n r_{u}\)

 

Using the Tables in Appendix B, the following table can be prepared for the given values of \(c^{\prime} / \gamma H, \phi \text {, and } \beta\).

 

From Tables B2 and B3 for \(c^{\prime} / \gamma H=0.025\)
\(n_{d}\) m n \(F_{s}\)
1 2.873 2.622 1.562
1.25 2.953 2.806 1.55 Lowest
From Table B4, B5 and B6 for \(c^{\prime} / \gamma H=0.05\)
\(n_{d}\) m n \(F_{s}\)
1 3.261 2.693 1.915
1.25 3.221 2.819 1.812 Lowest
1.5 3.443 3.12 1.883
Hence \(n_{d}=1.25 \text { is the more critical depth factor. The value of } F_{s} \text { for } c^{\prime} / \gamma H=0.035\) lies between \(\left.1.55 \text { (for } c^{\prime} / \gamma H=0.025 \text { ) and } 1.812 \text { (for } c^{\prime} / \gamma H=0.05\right) \text {. By proportion } F_{s}=1.655\).
