Guest Question.

For a retaining wall with a cohesionless soil backfill, γ =15.5 kN/\(m^{3}\), \(\phi ^{\prime }\)= 30°,\(\delta ^{\prime} \)= 15°, θ= 0°, α= 0°, H = 4 m, \(k_{v}\) = 0, and \(k_{h}\) = 0.2. Determine \(P_{ae}\). Also determine the location of the resultant line of action of \(P_{ae}\)—that is,\(\bar{z}\)

Admin
Solution

From Eqs. (13.66) and (13.67),

\(\theta ^{*} =\theta +\bar{\beta }\)

 

\(\alpha ^{*} =\alpha +\bar{\beta }\)

 

\(\bar{\beta } =\tan^{-1} (\frac{k_{h} }{1-k_{v} } )=\tan^{-1} (\frac{0.2}{1-0} )=11.3°\)

so,

\(\theta ^{*}=0 + 11.3° = 11.3°\)

 

\(\alpha ^{*}=0 + 11.3° = 11.3°\)

From Eq. (13.70),

\( P_{ae}=[P_{a} (\theta ^{*},\alpha ^{*})](1-k_{v})(\frac{\cos^{2}\theta ^{*} }{\cos^{2}\theta \cos^{2} \bar{\beta } } )=[P_{a} (\theta ^{*},\theta ^{*})](1-k_{v}) [\frac{\cos^{2}(\theta +\bar{\beta } ) }{\cos\theta \cos^{2}\bar{\beta } }\)

 

\(P_{a} (\theta ^{*},\alpha ^{*})=\frac{1}{2} \gamma H^{2} K_{a}\)

where \(K_{a}\)is a function of \(\theta ^{*} and \alpha ^{*}\)

Since \(\delta ^{\prime} /\phi ^{\prime}\) =15/30 = 0.5, we will use Table 13.6. For \(\theta ^{*}=\alpha ^{*}=11.3°\) the value of \(K_{a}\approx 0.452\) .Thus,

\(P_{ae}=\frac{1}{2} (15.5)(4)^{2} (0.452)(1-0)[\frac{\cos^{2}(0+11.3) }{(\cos0)(\cos^{2}11.3 )} ]=56.05 kN/m\)

[Note:We can also get the same values of \(P_{ae}\) using Eq. (13.60) \(K^{\prime \prime }_{a} =\frac{\cos^{2}(\phi ^{\prime }-\theta -\bar{\beta } ) }{\cos^{2}\theta \cos\bar{\beta }\cos(\delta ^{\prime} +\theta +\bar{\beta } ) \left\{1+[\frac{\sin(\delta ^{\prime}+\phi ^{\prime} )\sin(\delta ^{\prime}-\theta -\bar{\beta } )}{\cos(\delta ^{\prime}+\theta +\bar{\beta } )\cos (\theta -\alpha )} ]^{2} \right\} ^{2} } \)

and \(K^{\prime}_{ae} \) from Table 13.8.] We now locate the resultant line of action. From Eq. (13.53),\(p_{a} =\frac{1}{2} \gamma H^{2} K_{a}\)

For \(\phi ^{\prime} =30° and \delta ^{\prime} =15°,K_{a} =0.3014\) (Table 13.6), so

\(P_{a} =\frac{1}{2} (0.3014)(15.5)(4)^{2} =37.37 KN/m\)

Hence, \(P_{ae}=56.05 – 37.37 = 18.68 kN/m.\) .From Eq. (13.77)

\(\bar{z} =\frac{P_{a}(\frac{H}{3} )+\Delta P_{ae}(0.6H) }{P_{ae} } =\frac{(37.37)(\frac{4}{3} )+(18.68)(2.4)}{56.05} =1.69 m\)
