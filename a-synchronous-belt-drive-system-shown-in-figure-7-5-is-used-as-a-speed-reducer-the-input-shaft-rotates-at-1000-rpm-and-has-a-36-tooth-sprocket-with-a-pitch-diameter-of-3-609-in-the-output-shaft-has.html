Guest Question.

A synchronous belt drive system shown in Figure 7-5 is used as a speed reducer. The input shaft rotates at 1000 rpm and has a 36-tooth sprocket with a pitch diameter of 3.609 in. The output shaft has a 72-tooth sprocket with a pitch diameter of 7.218 in. The center distance between the two shafts is 15.01 in.

Determine the following for the belt drive:
a. The velocity ratio
b. The angular velocity of the output shaft
c. The linear belt speed
d. The belt wrap on the input and output sprockets
e. The belt perimeter length
f. Optional: Draw the schematic of the belt drive system using CAD

Admin
Solution

Objective: Use the equations developed in this section to solve for the kinematics of the belt drive system.
Given:

\(\begin{array}{ll}N_{\text {driving }}=36 & P D_{\text {driving }}=3.609 \mathrm{in} \quad n_{\text {driving }}=1000 \mathrm{rpm} \\N_{\text {driven }}=72 & P D_{\text {driven }}=7.218 \text { in } \\C D=15.01 \text { in } &\end{array}\)
Results:

a. Use Equation (7-4) to calculate the velocity ratio.
\(\begin{aligned}&V R=\frac{\omega_{\text {driving }}}{\omega_{\text {driven }}}=\frac{PD_{\text {driven }}}{P D_{\text {driving }}}=\frac{N_{\text {driven }}}{N_{\text{driving }}} \\&V R=\frac{72}{36}=2.0\end{aligned}\)
This tells us the input sprocket turns twice as fast as the output sprocket and the belt drive is a speed reducer. The velocity ratio can be used to find the reduced speed of the output shaft.
b. Manipulating Equation (7-4), we can now solve for the driven speed of the output shaft.
\(\begin{aligned}&\omega_{\text {driven }}=\frac{\omega_{\text {driving }}}{V R} \\&\omega_{\text {driven }}=\frac{1000 \mathrm{rpm}}{2} \\&\omega_{\text {driven }}=500\mathrm{rpm}\end{aligned}\)

c. The belt linear velocity can be calculated using Equation (7-2)
\(v_{\text {belt }}=\frac{P D_{\text {driving }}}{2} \cdot \omega_{\text {driving }}=\frac{P D_{\text {driven }}}{2} \cdot \omega_{\text {driven }}\)
Using the input sprocket,
\(v_{\text {belt }}=\frac{3.609 \mathrm{in}}{2} \cdot 1000 \mathrm{rpm} \cdot \frac{2\cdot \pi \mathrm{rad}}{1 \mathrm{rev}} \cdot \frac{1 \mathrm{ft}}{12 \mathrm{in}}=944.83 \mathrm{fpm}\)
Using the output sprocket,
\(v_{b e n t}=\frac{7.218 \mathrm{in}}{2} \cdot 500 \mathrm{rpm} \cdot \frac{2 \cdot \pi\mathrm{rad}}{1 \mathrm{rev}} \cdot \frac{1 \mathrm{ft}}{12 \mathrm{in}}=944.83\mathrm{fpm}\)
d. In order to solve for the angle of the belt wrap on both sprockets, first we need to find the angle \(\varnothing\) using Equation (7-5)
\(\begin{aligned}\sin (\varnothing) &=\frac{\left(\frac{P D_{2}-P D_{1}}{2}\right)}{C D}\\\sin (\varnothing) &=\frac{\left(\frac{7.218 \text { in }-3.609}{2}\right)}{15.01 \text{in }}=0.120 \\\varnothing &=\sin ^{-1}(0.120) \\\varnothing &=6.905^{\circ}\end{aligned}\)

The angle of wrap on sprocket ( 1 ) is given by Equation (7-6)
\(\begin{aligned}&\theta_{1}=180^{\circ}-2 \cdot \varnothing \\&\theta_{1}=180^{\circ}+2\cdot 6.905^{\circ} \\&\theta_{1}=166.2^{\circ}\end{aligned}\)
The angle of wrap on sprocket (2) is given by Equation (7-7)
\(\begin{aligned}&\theta_{2}=180^{\circ}+2 \cdot \varnothing \\&\theta_{2}=180^{\circ}-2\cdot 6.905^{\circ} \\&\theta_{2}=193.8^{\circ}\end{aligned}\)
e. To find the belt perimeter length, the belt span distance and the arc lengths of belt wrap on both sprockets need to be found. Use Equation ( 7-10 ) to first calculate the belt span distance, \(d\),
\(\begin{aligned}&d=C D \cdot \cos \varnothing \\&d=15.01 \text { in } \cdot \cos\left(6.905^{\circ}\right) \\&d=14.90 \mathrm{in}\end{aligned}\)
Next solve for the arc length \(s_{1}\) using Equation (7-8),
\(\begin{aligned}s_{1} &=\frac{P D_{1}}{2} \cdot \theta_{1} \\s_{1} &=\frac{3.609\mathrm{in}}{2} \cdot 166.2^{\circ} \cdot \frac{2 \cdot \pi \mathrm{rad}}{360^{\circ}} \\s_{1} &=5.23 \mathrm{in}\end{aligned}\)

Now solve for the arc length \(s_{2}\) using Equation (7-9),
\(\begin{aligned}s_{2} &=\frac{P D_{2}}{2} \cdot \theta_{2} \\s_{2} &=\frac{7.218\mathrm{in}}{2} \cdot 193.8^{\circ} \cdot \frac{2 \cdot \pi \mathrm{rad}}{360^{\circ}} \\s_{2} &=12.21 \mathrm{in}\end{aligned}\)
Lastly, use Equation (7-11) to add the belt span distance and the two arc lengths to solve for the belt perimeter length,
\(\begin{aligned}&\text { Belt perimeter length }=2 \cdot d+s_{1}+s_{2} \\&\text { Beltperimeter length }=2 \cdot 14.90 \text { in }+5.23 \text { in }+12.21 \text { in } \\&\text { Belt perimeter length }=47.24 \text { in }\end{aligned}\)
f. The belt perimeter length would be used to purchase the correct belt length from the belt manufacturer. Since belt manufacturer make belts of specific lengths, the calculated center distance and belt perimeter length must correspond to the manufacturer’s belt specifications.
