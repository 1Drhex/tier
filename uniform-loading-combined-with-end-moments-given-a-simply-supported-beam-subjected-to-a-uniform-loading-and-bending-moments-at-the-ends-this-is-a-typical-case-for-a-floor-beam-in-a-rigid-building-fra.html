Uniform Loading Combined with End Moments

Given: A simply supported beam subjected to a uniform loading and bending moments at the ends. This is a typical case for a floor beam in a rigid building frame, i.e., where the beam-column connections apply moment to the ends of the beam (Fig. E3.12a).

Determine: The location and magnitude of the maximum moment.

Admin
Solution

We consider separate loadings and then superimpose the results. The solution due to the end moment is (Fig. E3.12b)

 

The uniform loading provides the following distribution (Fig. E3.12c):

 

Combining these solutions leads to the analytical solution

 

\(V(x) = -\frac{wL}{2}+wx+\frac{M^{\ast }_{A}+M^{\ast }_{B}}{L}\)

 

\(M(x)= M^{\ast }_{A}-\frac{M^{\ast }_{A}+M^{\ast }_{B}}{L}x + \frac{wL}{2}x-\frac{wx^{2}}{2}\)

 

These functions are plotted below (Figs. E3.12d and E3.12e).

 

The peak moment occurs where the shear is zero. Noting the plot of V(x), the shear is zero at \(x_{max}\).

 

\(V=0\)

 

\(\Downarrow\)

 

\(X_{max} = \frac{L}{2}- \frac{M^{\ast }_{A}+M^{\ast }_{B}}{wL}\)

 

The form of the solution suggests that we express the sum of the end moments as

 

\(M^{\ast }_{A} + M^{\ast }_{B}= \frac{\alpha }{2}wL^{2}\)

 

where \(\alpha\) is a dimensionless parameter. Substituting for this term, the equation simplifies to

 

\(X_{max}= \frac{L}{2}(1-\alpha )\)

 

Lastly, we determine \(M_{max}\) using this value for x.

 

\(M_{max}= M^{\ast }_{A}+\frac{wL^{2}}{8}(1-\alpha )^{2}\)

 

Given w and the end moments, one evaluates \(\alpha\) ,

 

\(\alpha = \frac{M^{\ast }_{A}+M^{\ast }_{B}}{\frac{wL^{2}}{2}}\)

 

and then \(M_{max}\).When \(\alpha = \pm 1\), the peak moment occurs at an end point and equals the applied end moment.

 

The case where the end moments are equal in magnitude but opposite in sense is of considerable interest. One sets \(M^{\ast }_{A}=-M^{\ast }_{B}=-M^{\ast }\) , and it follows that \(\alpha = 0\). The moment diagram is symmetrical with respect to the centerline. The peak negative values of moment occur at the end points; the peak positive moment occurs at the center point of the span (Fig. E3.12f).

 

When there is no end restraint, \(M^{\ast }= 0\). Then, \(M^{+}_{max} = wL^{2}/8\). The effect of end restraint is to reduce the positive moment and introduce a negative moment at the ends. This behavior is typical for rigid frames such as building frames subjected to gravity loading. We examine this behavior in more detail in Chap. 15.
