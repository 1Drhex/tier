Guest Question.

Steel Floor Plate

Determine the required thickness of a steel plate floor deck to support a live load of 125 psf. The plate conforms to ASTM A786 and the span between the supports is 3′-6″.

Admin
Solution

The dead load of the floor plate will have to be assumed. Assuming a \(\frac{3} {8}\) -in.-thick plate, the dead load is

\(\left(\frac{\frac{3}{8} }{12} \right)(490lb/ft^{3} ) =16\)psf.

The factored load is

(1.2)(16) + (1.6)(125) = 220 psf = 0.220 ksf

Check the strength using equations (6-44) and (6-45):

\(\phi _{b}M_{n}=\phi _{b}M_{p}=\phi _{b}F_{y} Z_{x},\) (6-44)

\(Z_{x}=\frac{Nt^{2} _{p} }{4}\) (plastic section modulus for a plate). (6-45)

\(\phi _{b}M_{p}=\phi _{b}F_{y} Z_{x}\)

Solving for \(Z_{x}\),

\(Z_{x} =\frac{\frac{(12)(0.220)(3.5)^{2} }{8} }{(0.9)(27)}=0.167in^{3}/ft\)

 

\(Z_{x} =\frac{bt^{2} }{4}=0.167=\frac{(12)t^{2} }{4}\)

\(t_{min}=0.236\)in = 1⁄4-in.-thick plate

Using AISCM, Table 3-18b,

Table 3-18 \(C_{t} \) values for various structural systems

Structural System \(C_{t} \) x
Steel Moment Resisting Frames 0.028 0.8
Eccentrically Braced Frames (EBF) 0.03 0.75
All other structural systems 0.02 0.75

we find that a 1⁄4-in.-thick plate has a factored load capacity of 245 psf, which also has less weight than the assumed value for a 3⁄8-in.-thick plate.

Check Deflection:

\(\Delta =\frac{5wL^{4} }{384EI^{’} }\) where \(\Delta=L/100\).

Solving for I (use service load for deflection),

\(I =\frac{(100)(5)(0.150/12)[(3.5)(12)]^{3} }{(384)(29,000)}=0.0346in^{4}/ft\)

 

\(I=\frac{bt^{3} }{12}=0.0346=\frac{(12)t^{3} }{12}\)

\(t_{min}=0.326\)in. →Select\(\frac{3}{8}\)-in. plate.

Using AISCM, Table 3-18a, we find that a 3/8-in.-thick plate has a service load capacity of 190 psf. Note that a 1⁄4-in.- thick plate is adequate for bending, but has a service load capacity of only 56.4 psf for deflection considerations.
