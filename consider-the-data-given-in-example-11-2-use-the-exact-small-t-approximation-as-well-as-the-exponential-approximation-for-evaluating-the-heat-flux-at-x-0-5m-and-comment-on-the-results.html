\(q^+_{L}=5.67\times 10^{-8}\left[2\times 400^4\times 0.416291+350^4(1-2\times 0.416291)\right]=1350.96 W/m^2\)

The small t approximation yields \(E_{3}(0.1)\approx \frac{1}{2}-0.1=0.4\). Using this value in Eq.11.27, the heat flux at x = L = 0.5m is

\(q^+_{L}=5.67\times 10^{-8}\left[2\times 400^4\times 0.4+350^4(1-2\times 0.4)\right]=1331.39W/m^2\)

The exponential approximation yields \(E_{3}(0.1)\approx \frac{1}{2}e^{-1.8\times 0.1}=0.417635\). Using this value in Eq. 11.27, the heat flux at x = L = 0.5m is

\(q^+_{L}=5.67\times 10^{-8}\left[2\times 400^4\times 0.417635+350^4(1-2\times 0.417635)\right]=1352.57W/m^2\)

Thus all three values are acceptable in this problem. The exact and the exponential approximations are very close to each other.
