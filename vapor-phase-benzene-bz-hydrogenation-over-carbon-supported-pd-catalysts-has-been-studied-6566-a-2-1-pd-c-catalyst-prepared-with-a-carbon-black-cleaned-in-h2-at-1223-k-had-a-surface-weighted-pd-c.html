Guest Question.

Vapor-phase benzene (Bz) hydrogenation over carbon-supported Pd catalysts has been studied [65,66]. A 2.1% Pd/C catalyst prepared with a carbon black cleaned in \(H_{2}\)at 1223 K had a surface-weighted Pd crystallite size of 21 nm, giving a Pd dispersion of 5%, based on TEM. The carbon itself had an average mesopore diameter of 25 nm, while the average micropore diameter was 0.9 nm; thus the majority of the Pd resided in the mesopores. The highest activity of this catalyst at 413 K and 50 Torr Bz (Total P= 1 atm,balance \(H_{2}\)) was 1.99\(\mu\) mole Bz \(s^{-1}g^{-1}\). The density of the catalyst was 0.60 g \(cm^{-3}\). The catalyst particle size distribution ranged from 10â€“500 microns (1 \(\mu\) = \(10^{-6}\) m). (a) Assuming all the Pd is in the mesopores, are any mass transfer limitations expected based on the W-P criterion? (b) If,instead, this catalyst had all the Pd in the micropores and it gave this performance, would mass transfer limitations exist? Why?

Admin
Solution
\(N_{W-P}=\Re R^{2}_{p}/ C_{s} D_{eff}\cong \Re R^{2}_{p}/CD_{kn}\)

 

\(\Re = \left\lgroup\frac{1.99 \ \mu mole \ B_{Z} }{s \ g \ cat} \right\rgroup \left\lgroup\frac{0.60 \ g \ cat }{cm^{3} } \right\rgroup = \frac{1.2 \ \mu \ mole \ B_{Z}}{s \ cm^{3}} \)

 

\(C_{s} =C_{o}= P/RT= \frac{\left(50 \ Torr \ B_{Z} \right) \left(1 \ atm /760 \ Torr\right) }{\left\lgroup\frac{82.06 \ atm \cdot cm^{3} }{g \ mole \ K} \right\rgroup \left(413 K\right) } = 1.9 \times 10^{-6} \frac{mole}{cm^{3}} \)

 

\(D_{eff}\cong D_{Kn}\) beeause \( \lambda \gg d_{p}\) (averagepore diameter)

\(D_{Kn}=1/3 \bar{v} d_{p} \) and let \(d_{p}= 250 \mathring{A}=25 nm\)

\(\bar{v}= \left(8 \ k_{B} T/\pi m \right)^{1/2}= \left[\frac{8\left(1.38 \times 10^{-16}erg/K \right) \left(413 \ K\right) }{\pi \left(78 \ amu\right)\left(1.66 \times 10^{-24}g/amu \right) } \right] ^{1/2} = 3.35\times 10^{4} \frac{cm}{s}\) (Eq.4.76)

\(D_{Kn}= 1/3 \left\lgroup\frac{3.35\times 10^{4}}{s} \right\rgroup \left(25 \ nm\right)\left\lgroup\frac{1 \ cm}{ 10^{7} nm} \right\rgroup = 2.8 \times 10^{-2} \frac{cm^{2} }{s} \) (Eq.4.78)

(a) With Pd in mesopores = 25 nm:

Worst case:\(R_{p}= 500 \ \mu ; N_{W-P}= \frac{\left(1.2\times 10^{-6} mole \ B_{Z} \ s\cdot cm^{3} \ \right) \left(500 \times 10^{-4} cm \right)^{2} }{\left(1.9\times 10^{-6} mole \ B_{Z} / cm^{3}\right) \left(2.8 \times 10^{-3} cm^{2}/s\right) } =0.056\)

Best ease:

\(R_{p}= 10 \ \mu ; N_{W-P}= \frac{\left(1.2\times 10^{-6} mole \ B_{Z} \ s\cdot cm^{3} \ \right) \left(10 \times 10^{-4} cm \right)^{2} }{\left(1.9\times 10^{-6} mole \ B_{Z} / cm^{3}\right) \left(2.8 \times 10^{-3} cm^{2}/s\right) } =2.3 \times 10^{-5} \)

 

(b) With Pd in mieropores = 0.9 nm:

\(D_{Kn}= 1/3 \left(3.35 \times 10^{4} cm/s\right)\left(0.9 nm\right)\left(1 \ cm /10^{7}nm\right)= 1.0 \times 10^{-3} cm^{2}/s \)

Worst case:\(R_{p}= 500 \ \mu ; N_{W-P}= \frac{\left(1.2\times 10^{-6} mole \ B_{Z} /s \cdot cm^{3} \ \right) \left(500 \times 10^{-4} cm \right)^{2} }{\left(1.9\times 10^{-6} mole \ B_{Z}/ cm^{3}\right) \left(1.0 \times 10^{-3} cm^{2} /s\right) } =1.6\)

Best ease:

\(R_{p}= 10 \ \mu ; N_{W-P}= \frac{\left(1.2\times 10^{-6} mole \ B_{Z} /s\cdot cm^{3} \ \right) \left(10 \times 10^{-4} cm \right)^{2} }{\left(1.9\times 10^{-6} mole \ B_{Z}/ cm^{3}\right) \left(1.0 \times 10^{-3} cm^{2}/s\right) } =6.3 \times 10^{-4} \)
