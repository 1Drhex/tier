Guest Question.

Figure 4.12 shows a planar 3-RPR robot with a 12 in. equilateral triangular base and a 3 in. platform. We would like to position the platform center at \(P_{x}= 4 in., P_{y} = 5 in\)., and \(θ =30^{\circ }\) . Determine the lengths of the three prismatic actuators that would place and orient the platform.

Admin
Solution

From Figure 4.12 we get the following values:

\(l_{1} = \left [ \begin{matrix} 0 \\ 0 \end{matrix} \right ] , l_{2} = \left [ \begin{matrix} 12 \\ 0 \end{matrix} \right ] , l_{3} = \left [ \begin{matrix} 6 \\ 10.4 \end{matrix} \right ]\)

 

At reset position,

\(C_{1} = \left [ \begin{matrix} -1.5 \\ -0.866 \end{matrix} \right ] ,C_{2} = \left [ \begin{matrix} 1.5 \\ -0.866 \end{matrix} \right ] , C_{3} = \left [ \begin{matrix} 0 \\ 1.73 \end{matrix} \right ]\)

 

Substituting into Eq. (4.3) for each actuator we find:

\(\left [ \begin{matrix} d_{_{ix} } \\ d_{_{iy} } \end{matrix} \right ] = \left [ \begin{matrix} P_{x} \\ P_{y} \end{matrix} \right ]+ \left [ \begin{matrix} C\theta _{c_{ix} }- S\theta _{c_{iy} } \\ S\theta _{c_{ix} }+ C\theta _{c_{iy} } \end{matrix} \right ]- \left [ \begin{matrix} l_{_{ix} } \\ l_{_{iy} } \end{matrix} \right ] for i=1,2,3\) (4.3)

 

\( \left [ \begin{matrix} d_{_{1x} } \\ d_{_{1y} } \end{matrix} \right ] = \left [ \begin{matrix} 4 \\ 5 \end{matrix} \right ]+ \left [ \begin{matrix} (0.866)(-1.5)-(0.5)(-0.866) \\ (0.5)(-1.5)+(0.866)(-0.866) \end{matrix} \right ]- \left [ \begin{matrix} 0 \\ 0 \end{matrix} \right ]= \left [ \begin{matrix} 3.134 \\ 3.5 \end{matrix} \right ] \Rightarrow d_{1} =4.7in\)

 

\(\left [ \begin{matrix} d_{_{2x} } \\ d_{_{2y} } \end{matrix} \right ] = \left [ \begin{matrix} 4 \\ 5 \end{matrix} \right ]+ \left [ \begin{matrix} (0.866)(1.5)-(0.5)(-0.866) \\ (0.5)(1.5)+(0.866)(-0.866) \end{matrix} \right ]- \left [ \begin{matrix} 12 \\ 0 \end{matrix} \right ]= \left [ \begin{matrix} -6.27 \\ 5 \end{matrix} \right ] \Rightarrow d_{2} =8.02in\)

 

\(\left [ \begin{matrix} d_{_{3x} } \\ d_{_{3y} } \end{matrix} \right ] = \left [ \begin{matrix} 4 \\ 5 \end{matrix} \right ]+ \left [ \begin{matrix} (0.866)(0)-(0.5)(1.73) \\ (0.5)(0)+(0.866)(1.73) \end{matrix} \right ]- \left [ \begin{matrix}6 \\ 10.4 \end{matrix} \right ]= \left [ \begin{matrix} -2.865 \\ -3.9 \end{matrix} \right ] \Rightarrow d_{3} =4.84in\)

 

Calculating the lengths of each actuator for any other position or orientation only requires substituting the given values into Eq. (4.3).
