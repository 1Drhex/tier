Guest Question.

Asynchronous capacitor is rated at 160 Mvar, 16 kV, 1200 r/min, 60 Hz. It has a synchronous reactance of 0.8 pu and is connected to a 16 kV line.

Calculate the value of \(E_{ o }\) so that the machine
a. Absorbs 160 Mvar
b. Delivers 120 Mvar

Admin
Solution

a. The nominal impedance of the machine is

 

\(\begin{aligned}Z_{ n } &=E_{ n }^{2} / S _{ n } \\&=16000^{2} /\left(160 \times 10^{6}\right) \\&=1.6 \Omega\end{aligned}\)

 

The synchronous reactance per phase is

 

\(\begin{aligned}X_{ s } &=X_{ S }( pu ) Z_{ n }=0.8 \times 1.6 \\&=1.28 \Omega\end{aligned}\)

 

The line current for a reactive load of 160 Mvar is

 

\(\begin{aligned}I_{ n } &=S_{ n } /\left(\sqrt{3} E_{ n }\right) \\&=\left(160 \times 10^{6}\right) /(1.73 \times 16000) \\&=5780 A\end{aligned}\)

 

The drop across the synchronous reactance is

 

\(\begin{aligned}E_{ x } &=I X_{ s }=5780 \times 1.28 \\&=7400 V\end{aligned}\)

 

The line-to-neutral voltage is

 

\(\begin{aligned}E &=E_{ L } / \sqrt{3}=16000 / 1.73 \\&=9250 V\end{aligned}\)

 

Selecting E as the reference phasor, we have

 

\(E=9250 \angle 0^{\circ}\)

 

The current I lags 90Â° behind E because the machine is absorbing reactive power; consequently,

 

\(I=5780 \angle-90^{\circ}\)

 

From Fig. 25a we can write

 

\(-E+j I X_{ s }+E_{ o }=0\)

 

hence

 

\(\begin{aligned}E_{ o } &=E-j I X_{ s } \\&=9250 \angle 0^{\circ}-5780 \times 1.28 \angle\left(90^{\circ}-90^{\circ}\right) \\&=1850 \angle 0^{\circ}\end{aligned}\)

 

Note that the excitation voltage (1850 V) is much less than the line voltage (9250 V).

 

b. The load current when the machine is delivering 120 Mvar is

 

\(\begin{aligned}I_{ n } &=Q /\left(\sqrt{3} E_{ n }\right) \\&=\left(120 \times 10^{6}\right) /(1.73 \times 16000) \\&=4335 A\end{aligned}\)

 

This time I leads E by 90Â° and so

 

\(I=4335 \angle 90^{\circ}\)

 

From Fig. 25b we can write

 

\(\begin{aligned}E_{ o } &=E-j I X_{ s } \\&=9250 \angle 0^{\circ}-4335 \times 1.28 \angle 180^{\circ} \\&=(9250+5550) \angle 0^{\circ} \\&=14800 \angle 0^{\circ}\end{aligned}\)

 

The excitation voltage (14 800 V) is now considerably greater than the line voltage (9250 V).
