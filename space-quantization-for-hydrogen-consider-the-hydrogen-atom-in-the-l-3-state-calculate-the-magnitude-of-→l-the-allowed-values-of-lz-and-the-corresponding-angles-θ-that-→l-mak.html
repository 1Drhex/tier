Guest Question.

Space Quantization for Hydrogen

Consider the hydrogen atom in the \(\ell=3\) state. Calculate the magnitude of \(\overrightarrow{ L }\), the allowed values of \(L_{z}\), and the corresponding angles \(\theta\) that \(\overrightarrow{ L }\) makes with the \(z\) axis.

Admin
Solution

Conceptualize Consider Figure \(42.13a\), which is a vector model for \(\ell=2 .\) Draw such a vector model for \(\ell=3\) to help with this problem.

Categorize We evaluate results using equations developed in this section, so we categorize this example as a substitution problem.

Calculate the magnitude of the orbital angular momentum using Equation 42.27:

\(L=\sqrt{\ell(\ell+1)} \hslash \quad \ell=0,1,2, \ldots, n-1\) (42.27)

\(L=\sqrt{\ell(\ell+1)} \hslash =\sqrt{3(3+1)} \hslash =2 \sqrt{3} \hslash \)

Calculate the allowed values of \(L_{z}\) using Equation \(42.28\) with \(m_{\ell}=-3,-2,-1,0,1,2\), and \(3:\)

\(L_{z}=m_{\ell} \hslash \) (42.28)

\(L_{z}=-3 \hslash ,-2 \hslash ,-\hslash , 0, \hslash , 2 \hslash , 3 \hslash \)

Calculate the allowed values of \(\cos \theta\) using Equation 42.29:

\(\cos \theta=\frac{L_{z}}{L}=\frac{m_{\ell}}{\sqrt{\ell(\ell+1)}}\) (42.29)

\(\begin{aligned}&\cos \theta=\frac{\pm 3}{2 \sqrt{3}}=\pm 0.866 \quad \cos \theta=\frac{\pm 2}{2 \sqrt{3}}=\pm 0.577 \\&\cos \theta=\frac{\pm 1}{2 \sqrt{3}}=\pm 0.289 \quad \cos \theta=\frac{0}{2 \sqrt{3}}=0\end{aligned}\)

Find the angles corresponding to these values of \(\cos \theta\) :

\(\theta=30.0^{\circ}, 54.7^{\circ}, 73.2^{\circ}, 90.0^{\circ}, 107^{\circ}, 125^{\circ}, 150^{\circ}\)
