Guest Question.

For the state equation and initial state vector shown in Eqs. (4.112), where u(t) is a unit step, find the state-transition matrix and then solve for \(\mathbf{x}(t)\).

\(\mathbf{\dot{x}} (t) = \begin{bmatrix} 0 & 1 \\ -8 & -6 \end{bmatrix} \mathbf{x}(t) + \begin{bmatrix} 0 \\ 1 \end{bmatrix}u(t)\) (4:112a)

\(\mathbf{x}(0) = \begin{bmatrix} 1 \\ 0 \end{bmatrix}\) (4:112b)

Admin
Solution

Since the state equation is in the form

\(\mathbf{\dot{x}} (t) = \mathbf{Ax} (t) + \mathbf{B}u(t)\) (4:113)

find the eigenvalues using \(det(s\mathbf{I} – \mathbf{A}) = 0\). Hence, \(s^{2}+ 6s + 8 = 0\), from which \(s_{1} =- 2\) and \(s_{2} = – 4\). Since each term of the state-transition matrix is the sum of responses generated by the poles (eigenvalues), we assume a state-transition matrix of the form

\(\mathbf{\Phi }(t) = \begin{bmatrix} (K_{1} e^{-2t}+ K_{2} e^{-4t}) & (K_{3} e^{-2t}+ K_{4} e^{-4t} ) \\ (K_{5} e^{-2t}+ K_{6} e^{-4t} ) & (K_{7} e^{-2t}+ K_{8} e^{-4t} ) \end{bmatrix}\) (4:114)

In order to find the values of the constants, we make use of the properties of the state-transition matrix located at www.wiley.com/ college/nise.

\(\mathbf{\Phi }(0) = \mathbf{I}\) (4:115)

\(K_{1} + K_{2} = 1\) (4:116a)

\(K_{3} + K_{4} = 0\) (4:116b)

\(K_{5} + K_{6} = 0\) (4:116c)

\(K_{7} + K_{8} = 1\) (4:116d)

and since

\(\mathbf{\dot{\Phi }}(0) =\mathbf{A}\) (4:117)

then

\(-2K_{1} – 4K_{2} = 0\) (4:118a)

\(-2K_{3} – 4K_{4} = 1\) (4:118b)

\(-2K_{5} – 4K_{6} = – 8\) (4:118c)

\(-2K_{7} – 4K_{8} = – 6\) (4:118d)

The constants are solved by taking two simultaneous equations four times. For example, Eq. (4.116a) can be solved simultaneously with Eq. (4.118a) to yield the values of \(K_{1}\) and \(K_{2}\). Proceeding similarly, all of the constants can be found. Therefore,

\(\mathbf{\Phi }(t) = \begin{bmatrix} (2e^{-2t}-e^{-4t}) & (\frac{1}{2}e^{-2t}- \frac{1}{2}e^{-4t} ) \\ (-4e^{-2t}+4 e^{-4t} ) & (-e^{-2t}+ 2e^{-4t} ) \end{bmatrix}\) (4:119)

Also,

\(\mathbf{\Phi }(t-\tau ) \mathbf{B} = \begin{bmatrix}(\frac{1}{2}e^{-2 (t-\tau )}- \frac{1}{2}e^{-4 (t-\tau )} ) \\(- e^{-2 (t-\tau )} +2 e^{-4 (t-\tau )}) \end{bmatrix}\) (4:120)

Hence, the first term of Eq. (\(\mathbf{x}\left ( t \right )=e^{\mathbf{At}}\mathbf{x}\left ( 0 \right )+\int_{0}^{t}e^{\mathbf{A(t-\tau }}\mathbf{Bu}\left ( \tau \right )d\tau = \mathbf{\Phi }\left ( t \right )\mathbf{x}\left ( 0 \right )+\int_{0}^{t} \Phi (t-\tau) \mathbf{Bu}\left ( \tau \right )d\tau)\) is

 

\(\mathbf{\Phi }(t) \mathbf{x}(0) = \begin{bmatrix}(2e^{-2t} – e^{-4t)}) \\(-4e^{-2t} +4 e^{-4t}) \end{bmatrix}\) (4:121)

The last term of Eq. (\(\mathbf{x}\left ( t \right )=e^{\mathbf{At}}\mathbf{x}\left ( 0 \right )+\int_{0}^{t}e^{\mathbf{A(t-\tau }}\mathbf{Bu}\left ( \tau \right )d\tau = \mathbf{\Phi }\left ( t \right )\mathbf{x}\left ( 0 \right )+\int_{0}^{t} \Phi (t-\tau) \mathbf{Bu}\left ( \tau \right )d\tau\)) is

\(\int_{0}^{t} \mathbf{\Phi }(t-\tau ) \mathbf{Bu} (\tau )d \tau = \begin{bmatrix} \frac{1}{2}e^{-2t} \int_{0}^{t} e^{2\tau }d \tau – \frac{1}{2}e^{-4t}\int_{0}^{t} e^{4\tau }d \tau \\ – e^{-2t} \int_{0}^{t} e^{2\tau }d \tau +2 e^{-4t}\int_{0}^{t} e^{4\tau }d \tau \end{bmatrix}\)

\(= \begin{bmatrix} \frac{1}{8} – \frac{1}{4} e^{-2t} + \frac{1}{8}e^{-4t} \\ \frac{1}{2}e^{-2t} -\frac{1}{2}e^{-4t} \end{bmatrix}\)
(4:122)

Notice, as promised, that Eq. (4.122), the zero-state response, contains not only the forced response, 1/8, but also terms of the form\( Ae^{-2t}\) and \(Be^{-4t}\) that are part of what we previously called the natural response. However, the coefficients, A and B, are not dependent on the initial conditions.

The final result is found by adding Eqs. (4.121) and (4.122). Hence,

\(\mathbf{x}(t)= \mathbf{\Phi }(t) \mathbf{x}(0) + \int_{0}^{t} \mathbf{\Phi }(t-\tau ) \mathbf{Bu} (\tau )d \tau = \begin{bmatrix} \frac{1}{8} + \frac{7}{4} e^{-2t} – \frac{7}{8}e^{-4t} \\ – \frac{7}{2}e^{-2t} + \frac{7}{2}e^{-4t} \end{bmatrix}\)
(4:123)
