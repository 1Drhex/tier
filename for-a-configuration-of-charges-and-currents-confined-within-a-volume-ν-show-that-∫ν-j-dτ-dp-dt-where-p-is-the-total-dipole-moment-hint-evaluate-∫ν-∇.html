Guest Question.

For a configuration of charges and currents confined within a volume ν, show that

\(\int_{ \nu } J d \tau=d p / d t\) ,

where p is the total dipole moment. [Hint: evaluate \(\int_{ \nu} \nabla \cdot(x J ) d \tau\) .]

Admin
Solution

\(\frac{d p }{d t}=\frac{d}{d t} \int_{ \nu} \rho r d \tau=\int\left(\frac{\partial \rho}{\partial t}\right) r d \tau=-\int( \nabla \cdot J ) r d \tau\) (by the continuity equation). Now product rule #5 says \(\nabla \cdot(x J )=x( \nabla \cdot J )+ J \cdot( \nabla x) . \text { But } \nabla x=\hat{ x }, \text { so } \nabla \cdot(x J )=x( \nabla \cdot J )+J_{x} . \text { Thus } \int_{ \nu}( \nabla \cdot J ) x d \tau=\int_{ \nu} \nabla \cdot(x J ) d \tau-\int_{ \nu } J_{x} d \tau\) . The first term is \( \int_{ S } x J \cdot d a\) (by the divergence theorem), and since J is entirely inside ν, it is zero on the surface S. Therefore \(\int_{ \nu}( \nabla \cdot J ) x d \tau=-\int_{ \nu } J_{x} d \tau\) , or, combining this with the y and z components, \(\int_{ \nu}( \nabla \cdot J ) r d \tau=-\int_{ \nu} J d \tau\) . Or, referring back to the first line, \(\frac{d p }{d t}=\int J d \tau\) . qed

Here’s a quicker method, if the distribution consists of a collection of point charges. Use Eqs. 5.30 and 3.100:

\(\sum_{i=1}^{n}() q_{i} v _{i} \sim \int_{\text {line }}() I d l \sim \int_{\text {surface }}() K d a \sim \int_{\text {volume }}() J d \tau \) (5.30)

\(p =\sum_{i=1}^{n} q_{i} r _{i}^{\prime}\) (3.100)

\(\int J d \tau=\sum q_{i} v _{i}=\frac{d}{d t} \sum q_{i} r _{i}=\frac{d p }{d t}\) .
