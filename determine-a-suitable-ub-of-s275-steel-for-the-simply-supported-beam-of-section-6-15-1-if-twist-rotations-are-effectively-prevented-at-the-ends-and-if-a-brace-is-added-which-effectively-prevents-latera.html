Guest Question.

Determine a suitable UB of S275 steel for the simply supported beam of if twist rotations are effectively prevented at the ends and if a brace is added which effectively prevents lateral deflection v and twist rotation Φ at mid-span.

Admin
Solution

Design bending moment.

Using , \(M_{Ed} = 433 kNm. \)

Selecting a trial section.

The central brace divides the beam into two identical segments, each of length L = 3750 mm, and eliminates the effect of the load height.

Guess \(f_{y} = 275 N/mm^2 and M_{b,Rd} /W_{y}f_{y} = 0.9. \)

Using equation 6.32,

\(W_{pl,y} ≥\frac{M_{Ed} / f_{y}}{M_{b,Rd} /(W_{y}f_{y})}\)

 

\(W_{pl,y} ≥ (433 × 10^6/275)/0.9 mm^3 = 1750 cm^3. \)

Try a 457 × 191 UB 82 with \(W_{pl,y} = 1831 cm^3 > 1750 cm^3\).

Section resistance.

As in , \(f_{y} = 275 N/mm^2, M_{c,Rd} = 503.5 kNm > 433 kNm = M_{Ed}\) and the section resistance is adequate.

Elastic buckling moment.

Using Figure 6.7, \(α_{m} = 1.75\).

Using equation 6.3, \(M_{zx} =\sqrt{}\left\{\left(\frac{π^2EI_{z}}{L^2} \right)\left(GI_{t} +\frac{π^2EI_{w}}{L^2} \right) \right\} \)

 

\(M_{zx} =\sqrt{}\left\{\frac{π^2\times 210 000 × 1871 × 10^4}{3750^2} \times \left(81 000 × 69.2 × 10^4 +\frac{π^2× 210 000 × 0.922 × 10^{12}}{3750^2} \right) \right\} Nmm= 727.5 kNm\)

 

Using equation 6.4, \(M_{cr} = α_{m}M_{zx}\)

 

\(M_{cr} = 1.75 × 727.5 = 1273 kNm\)

 

(Using the computer program PRFELB [18] leads to \(M_{cr} = 1345 kNm\).)

 

Member resistance.

Using equation 6.25, \(\bar{\lambda }_{LT}=\sqrt{(W_{y}f_{y}/M_{cr})}\)

 

\(\bar{\lambda }_{LT}=\sqrt{\left(503.5/1273\right) }= 0.629\)

 

\(h/b = 460.0/191.3 = 2.40 > 2\)

Using the EC3 simple general method with \(β = 1.0, \bar{\lambda }_{LT,0} = 0.2\) (Clause 6.3.2.2) and \(α_{LT} = 0.34\) (Tables 6.4 and 6.3), and equation 6.27,

 

\(Φ_{LT} = 0.5\left\{1 + α_{LT} (\bar{\lambda }_{LT} − \bar{\lambda }_{LT,0}) + β\bar{\lambda }^2_{LT}\right\},\)

 

\(Φ_{LT} = 0.5{\left\{1 + 0.34(0.629 − 0.2) + 1.0 × 0.629^2\right\} } = 0.771\)

 

Using equation 6.26,

\(\frac{M_{b,Rd}γ_{M1}}{W_{y}f_{y}}=\frac{1}{Φ_{LT} +\sqrt{Φ^2_{LT} − β\bar{\lambda }^2_{LT}} }\)

 

\(M_{b,Rd} = 503.5/{\left\{0.771 + √(0.771^2 − 0.629^2)\right\} } kNm = 414 kNm < 433 kNm = M_{Ed}\)

and the beam appears to be inadequate.

Using the EC3 less conservative method with \(β = 0.75, \bar{\lambda }_{LT,0} = 0.4\) (Clause 6.3.2.3) and \(α_{LT} = 0.49\) (Tables 6.5 and 6.3), and equation 6.27,

\(Φ_{LT} = 0.5\left\{1 + α_{LT} (\bar{\lambda }_{LT} − \bar{\lambda }_{LT,0}) + β\bar{\lambda }^2_{LT}\right\},\)

 

\(Φ_{LT} = 0.5{\left\{1 + 0.49(0.629 − 0.4) + 0.75 × 0.629^2\right\} } = 0.704\)

 

Using equation 6.26,

\(\frac{M_{b,Rd}γ_{M1}}{W_{y}f_{y}}=\frac{1}{Φ_{LT} +\sqrt{Φ^2_{LT} − β\bar{\lambda }^2_{LT}} }\)

 

\(M_{b,Rd} = 503.5/{\left\{0.704 + √(0.704^2 − 0.75 × 0.629^2)\right\} } kNm = 437 kNm > 433 kNm = M_{Ed}\)

and so the design moment resistance is adequate after all.
