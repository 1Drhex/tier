Guest Question.

Verify the strength of a \(2 L 4 \times 31 / 2 \times 3 / 8\) LLBB \(\text { (3/4-in. separation) }\) strut with a length of 8 ft and pinned ends carrying an axial dead load of 20 kips and live load of 60 kips. Also, calculate the required number of fully tightened or welded intermediate connectors required.

Admin
Solution

Material Properties:

ASTM A36 \(F_{y}=36 ksi \quad F_{u}=58 ksi\)

Manual Table 2-3

Geometric Properties:

\(2 L 4 \times 31 / 2 \times 3 / 8 LLBB\)

Manual Table 1-15 Table 1-7

\(\begin{aligned}&r_{z}=0.719 \text { in. (single angle) }\\&r_{x}=1.25 \text { in. }\end{aligned}\)

 

\(r_{y}=1.55 \text { in. for } 3 / 8 \text { inch separation }\)

 

\(r_{y}=1.69 \text { in. for } 3 / 4 \text { inch separation }\)

Calculate the required strength

LRFD ASD
\(P_{u}=1.2(20 kips) + 1.6(60 kips)\)

= 120 kips

 \(P_{a}=20 +60 \)

= 80.0 kips

Select a column using Manual Table 4-9

K = 1.0

Commentary Table C-C2.2

For \((K L)_{x}\) = 8 ft, the available strength in axial compression is taken from the upper (x-x) portion of the table as

LRFD ASD
\(\phi_{c} P_{n}=127 \text { kips }>120 \text { kips }\) o.k. \(P_{n} / \Omega_{ c }=84.2 \text { kips }>80.0 \text { kips }\) o.k.

Manual Table 4-9

For buckling about the y-y axis, the values are tabulated for a separation of \(3 / 8\) in.

 

To adjust to a spacing of \(3 / 4\) in., \((K L)_{y}\) is multiplied by the ratio of the \(r_{y}\) for a \(3 / 8\)-in. separation to the \(r_{y}\) for a \(3 / 4\)-in. separation. Thus,

 

\((K L)_{y}=1.0(8.00 ft )\left(\frac{1.55 in .}{1.69 in .}\right)=7.34 ft\)

The calculation of the equivalent \((K L)_{y}\) above is a simplified approximation of Specification Section E6.1. To ensure a conservative adjustment for a \(3 / 4\) in. separation, take \((K L)_{y}=8 ft\).

The available strength in axial compression is taken from the lower (y-y) portion of the table as

LRFD ASD
\(\phi_{c} P_{n}=130 \text { kips }>120 \text { kips }\) o.k. \(P_{n} / \Omega_{ c }=86.5 \text { kips }>80.0 \text { kips }\) o.k.

Manual Table 4-9

Therefore, x-x axis flexural buckling governs.

Determine the number of intermediate connectors required

Per Table 4-12, at least two welded or pretensioned bolted intermediate connectors are required. This can be verified as follows:

a = distance between connectors \(=\frac{(8.00 ft )(12 in . / ft )}{3 \text { spaces }}=32.0 in\)

Section E6.2

 

The effective slenderness ratio of the individual components of the built-up member based upon the distance between intermediate connectors, a, must not exceed three quarters of the controlling slenderness ratio of the overall built-up member.

 

Therefore: \(\frac{K a}{r_{i}} \leq \frac{3}{4}\left(\frac{K l}{r}\right)_{\max }\)

 

Solving for a gives, \(a \leq \frac{3 r_{i}\left(\frac{K L}{r}\right)_{\max }}{4 K}\)

 

\(\frac{K L}{r_{x}}=\frac{1.0(8.00 ft )(12 in . / ft )}{1.25 in .}=76.8\) controls

 

\(\frac{K L}{r_{y}}=\frac{1.0(8.00 ft )(12 in . / ft )}{1.69 in .}=56.8\)

 

Thus, \(a \leq \frac{3 r_{z}\left(\frac{K L}{r}\right)_{\max }}{4 K}=\frac{3(0.719 in .)(76.8)}{4(1.0)}=41.4 in .>32.0 in\) o.k.

The available strength can be easily determined by using the tables of the Manual. Available strength values can be verified by hand calculations, as shown below.
