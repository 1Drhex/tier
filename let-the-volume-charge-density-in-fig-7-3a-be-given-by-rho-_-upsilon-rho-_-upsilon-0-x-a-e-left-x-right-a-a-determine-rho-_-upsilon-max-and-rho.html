Guest Question.

Let the volume charge density in Fig. 7.3a be given by \({ \rho }_{ \upsilon } = { \rho }_{ \upsilon 0 }(x/a){ e }^{ -\left| x \right| a }\). (a) Determine \({ \rho }_{ \upsilon, max }\) and \({ \rho }_{ \upsilon, min }\) and their locations. (b) Find \({ E }_{ x }\) and V(x) If V(0) = 0 and \({ E }_{ x } \longrightarrow 0\) as x \(\longrightarrow \infty\). (c) Use a development similar to that of Sec. 7.4 to show that C = dQ/d\({ V }_{ 0} = { \epsilon }_{ 0 }S/(4\sqrt {2 } a)\).

Admin
Solution

a) Let \(\acute { x }\) =x/a. Then \({ \rho }_{ \upsilon }=\acute { x } { e }^{ -\left| \acute { x } \right| }\). Differentiate with respect to \(\acute { x }\) to obtain:
\(\frac { d{ \rho }_{ \upsilon } }{ d\acute { x } } ={ \rho }_{ \upsilon 0 }{ e }^{ -\left| \acute { x } \right| }(1-\left| \acute { x } \right| )\)
This derivative is zero at \( \acute { x } =\pm 1\), or the minimum and maximum occur at x = \(\pm\) a respectively. The value of \({ \rho }_{ \upsilon }\) at these points will be \({ \rho }_{ \upsilon, max } = { \rho }_{ \upsilon 0 }{ e }^{ -1 } = 0.368 { \rho }_{ \upsilon 0 }\), occuring at x = a. \({ \rho }_{ \upsilon, min} = -{ \rho }_{ \upsilon 0 }{ e }^{ -1 } = -0.368 { \rho }_{ \upsilon 0 }\), occuring at x = -a.
b) We use Poisson’s equation:
\({ \triangledown }^{ 2 }V=-\frac { { \rho }_{ \upsilon } }{ \epsilon } \Longrightarrow \frac { { d }^{ 2 }V }{ { dx }^{ 2 } } =-\frac { { \rho }_{ \upsilon 0 } }{ \epsilon } \left( \frac { x }{ a } \right) { e }^{ -\left| x \right| /a }\)
For x > 0, this becomes
\(\frac { { d }^{ 2 }V }{ { dx }^{ 2 } } =-\frac { { \rho }_{ \upsilon 0 } }{ \epsilon } \left( \frac { x }{ a } \right) { e }^{ -x /a }\)
Integrate once over x:
\(\frac { dV }{ dx } (x>0)=-\frac { { \rho }_{ \upsilon 0 } }{ \epsilon } \int { } \left( \frac { x }{ a } \right) { e }^{ -x/a }dx+{ C }_{ 1 }=\frac { { a\rho }_{ \upsilon 0 } }{ \epsilon } { e }^{ -x/a }\left( \frac { x }{ a } +1 \right) +{ C }_{ 1 }\)
Nothing that \({ E }_{ x }\) = – dV/dx, we use the first boundary condition, \({ E }_{ x }\longrightarrow 0\) as x \(\longrightarrow \infty\), to establish that \({ C }_{ 1 }\) = 0. Over the range x < 0, we have
\(\frac { dV }{ dx } (x<0)=-\frac { { \rho }_{ \upsilon 0 } }{ \epsilon } \int { } \left( \frac { x }{ a } \right) { e }^{ x/a }dx+\acute { { C }_{ 1 } } =\frac { { a\rho }_{ \upsilon 0 } }{ \epsilon } { e }^{ x/a }\left( \frac { -x }{ a } +1 \right) +\acute { { C }_{ 1 } } \)
where \(\acute { { C }_{ 1 } } \) = 0, since, by symmetry, \({ E }_{ x }\longrightarrow 0\) as x \(\longrightarrow -\infty\). These two equations can be unified to cover the entire range of x; the final expression for the electric field becomes:
\({ E }_{ x } = -\frac { dV }{ dx } =-\frac { { a\rho }_{ \upsilon 0 } }{ \epsilon } \left( \frac { \left| x \right| }{ a } +1 \right) { e }^{ -\left| x \right| /a }\)V/m

The potential function is now found by a second integration. For x > 0, this is
\(V(x) = \frac { { a\rho }_{ \upsilon 0 } }{ \epsilon } \int { } \left[ \left( \frac { x }{ a } \right) { e }^{ -x/a }+{ e }^{ -x/a } \right] dx+{ C }_{ 2 } =\frac { { { a }^{ 2 }\rho }_{ \upsilon 0 } }{ \epsilon } \left( \frac { -x }{ a } { e }^{ -x/a }-2{ e }^{ -x/a } \right) +{ C }_{ 2 } \)
We use the second boundary condition, V(0) = 0, from which \({ C }_{ 2 } = 2 { a }^{ 2 }{\rho }_{ \upsilon 0 } /\epsilon\) . Substituting this yields
\(V(x) (x > 0) = \frac { { { a }^{ 2 }\rho }_{ \upsilon 0 } }{ \epsilon } \left[ \frac { -x }{ a } { e }^{ -x/a }+2\left( 1-{ e }^{ -x/a } \right) \right] \)
We repeat the procedure for x < 0 to obtain
\(V(x) = \frac { { a\rho }_{ \upsilon 0 } }{ \epsilon } \int { } \left[ \left( \frac { x }{ a } \right) { e }^{ x/a }+{ e }^{ x/a } \right] dx+\acute { { C }_{ 2 } } =\frac { { { a }^{ 2 }\rho }_{ \upsilon 0 } }{ \epsilon } \left[ \frac { -x }{ a } { e }^{ x/a }-2{ e }^{ x/a } \right] +\acute { { C }_{ 2 } } \)
Again, with the V(0) = 0 boundary condition, we find \(\acute { { C }_{ 2 } } = -2 { a }^{ 2 }{\rho }_{ \upsilon 0 } /\epsilon\), which when substituted leads to
\(V(x) (x > 0) = \frac { { { a }^{ 2 }\rho }_{ \upsilon 0 } }{ \epsilon } \left[ \frac { -x }{ a } { e }^{ x/a }-2\left( 1-{ e }^{ x/a } \right) \right]\)

Combining the results for both ranges of x, we write
\(V(x) = \frac { { { -a }^{ 2 }\rho }_{ \upsilon 0 } }{ \epsilon } \left[ \left( \frac { x }{ a } \right) { e }^{ -\left| x \right| /a }-\frac { 2x }{ \left| x \right| } \left( 1-{ e }^{ -\left| x \right| /a } \right) \right] \)
c) First, the overall potential difference is
\({ V }_{ 0 }={ V }_{ x\longrightarrow \infty }-{ V }_{ x\longrightarrow -\infty }=2\times \frac { { { 2a }^{ 2 }\rho }_{ \upsilon 0 } }{ \epsilon } =\frac { { { 4a }^{ 2 }\rho }_{ \upsilon 0 } }{ \epsilon }\)
From this we find a = \(\sqrt { \left( \epsilon { V }_{ 0 } \right) /\left( { 4\rho }_{ \upsilon 0 } \right) }\) . Then the total charge on one side will be
\(Q=S\int _{ 0 }^{ \infty }{ } { \rho }_{ \upsilon 0 }\left( \frac { x }{ a } \right) { e }^{ -x/a }dx=S{ \rho }_{ \upsilon 0 }a{ e }^{ -x/a }\left[ \frac { -x }{ a } -1 \right] |_{ 0 }^{ \infty }=S{ \rho }_{ \upsilon 0 }a=\frac { 1 }{ 2 } S\sqrt { \epsilon { V }_{ 0 }{ \rho }_{ \upsilon 0 } }\)
Now
\(C = \frac { dQ }{ d{ V }_{ 0 } } =\frac { d }{ d{ V }_{ 0 } } \left( \frac { 1 }{ 2 } S\sqrt { \epsilon { V }_{ 0 }{ \rho }_{ \upsilon 0 } } \right) =\frac { S }{ 4 } \sqrt { \frac { { \rho }_{ \upsilon 0 }\epsilon }{ { V }_{ 0 } } }\)
But a = \(\sqrt { \left( \epsilon { V }_{ 0 } \right) /\left( { 4\rho }_{ \upsilon 0 } \right) } \) , from where \( ({ \rho }_{ \upsilon 0 }/{ V }_{ 0 }) = \epsilon /({ 4a }^{ 2 })\). Substituting this into the capacitance expression gives
\(C = \frac { S }{ 4 } \sqrt { \frac { { \epsilon }^{ 2 } }{ { 4a }^{ 2 } } } =\frac { \epsilon S }{ 8a } \)
