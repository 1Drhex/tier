Guest Question.

Using the variation theorem to find an optimized wavefunction
Find the optimum form of a trial function of the form \(e^{−kr}\) and the upper bound to the ground-state energy of a hydrogenic atom.
Method Begin by writing the hamiltonian for the problem and then evaluate the integrals that occur in the expression for the Rayleigh ratio. The ratio will be obtained as a function of the parameter k, so to find the minimum value of the ratio we need to find the value of k that corresponds to dE/dk = 0.

Admin
Solution

The hamiltonian for the hydrogenic atom is

\(H=-\frac{\hbar^{2} }{2\mu }\nabla^2 -\frac{Ze^{2} }{4\pi \varepsilon _{0}r }\)

However, because the trial function is independent of angle, we need consider only the radial derivatives in the laplacian (see eqn 3.28):

\(\nabla^2 =\frac{1}{r}\frac{\partial^{2} }{\partial r^{2} } r+\frac{1} {r^{2}} \Lambda ^{2} \) (3.28a)

\(\nabla^2 =\frac{1}{r^{2} }\frac{\partial }{\partial r} r^{2}\frac{\partial}{\partial r} +\frac{1} {r^{2}} \Lambda ^{2} \) (3.28b)

\(\nabla^2 =\frac{\partial^{2} }{\partial r^{2} } +\frac{2}{r} \frac{\partial}{\partial r} +\frac{1} {r^{2}} \Lambda ^{2} \) (3.28c)

\(\nabla^2\psi =\frac{1}{r} \frac{d^{2} }{dr^{2} } r\psi \)

The integrals we require are therefore

\(\int{\psi ^{\ast }_{trial} \psi _{trial} d\tau } =\overbrace{\int_{0}^{2\pi }{d\varphi } }^{2\pi } \overbrace{\int_{0}^{\pi }{\sin \theta d\theta } }^{2} \overbrace{\int_{0}^{\infty }{e^{-2kr}r^{2}dr } }^{1/(4k^{3} )} =\frac{\pi }{k^{3} }\)

 

\(\int{\psi ^{\ast }_{trial}\left(\frac{1}{r} \right) \psi _{trial} d\tau } =\int_{0}^{2\pi }{d\varphi } \int_{0}^{\pi }{\sin \theta d\theta } \overbrace{\int_{0}^{\infty }{e^{-2kr}rdr } }^{1/(4k^{2} )} =\frac{\pi }{k^{2} } \)

 

\(\int{\psi ^{\ast }_{trial}\nabla^2 \psi _{trial} d\tau } =\int{\psi ^{\ast }_{trial}\left(\frac{1}{r}\frac{d^{2} }{dr^{2} } \right) } re^{-kr} d\tau =\int\psi ^{\ast }_{trial}\left(k^{2}-\frac{2k}{r} \right)\psi _{trial}d\tau \)

 

\(=k^{2} \int{\psi ^{\ast }_{trial}}\psi _{trial}d\tau -2k\int{\psi ^{\ast }_{trial}\left(\frac{1}{r} \right) }\psi _{trial}d\tau \)

 

\(=\frac{\pi }{k} -\frac{2\pi }{k}=-\frac{\pi }{k} \)

Therefore,

\(\int{\psi ^{\ast }_{trial}H\psi _{trial}d\tau }=\frac{\pi \hbar ^{2} }{2\mu k}-\frac{Ze^{2} }{4\varepsilon _{0}k^{2} } \)

and the Rayleigh ratio is

\(E=\frac{(\pi \hbar ^{2}/2\mu k )-(Ze^{2}/4\varepsilon _{0}k^{2} )}{\pi /k^{3} } =\frac{k^{2}\hbar ^{2} }{2\mu } -\frac{Ze^{2}k }{4\pi \varepsilon _{0} }\)

This function is plotted in Fig. 6.10. To find its minimum value we differentiate with respect to k:

\(\frac{dE}{dk} =\frac{k\hbar ^{2} }{\mu } -\frac{Ze^{2} }{4\pi \varepsilon _{0} } =0\) when \(k=\frac{Ze^{2}\mu }{4\pi \varepsilon _{0}\hbar ^{2} }\)

The best value of E is therefore

\(E=-\frac{Z^{2} e^{4}\mu }{32\pi^{2} \varepsilon _{0}^{2} \hbar ^{2} }\)

and the optimum form of the wavefunction has the value of k given above.

Comment This optimum value of the Rayleigh ratio turns out to be the exact ground-state energy (see eqn 3.66) and the corresponding trial function is the true wavefunction for the atom (see Table 3.4). This special result follows from the fact that the trial function happens to include the exact wavefunction as a special case.

\(E_{n} =-\left( \frac{Z^{2}\mu e^{4} }{32\pi^{2} \varepsilon _{0}^{2} \hbar ^{2} }\right)\frac{1}{n^{2} }\) \(n=1,2,…\) (3.66)
