Guest Question.

To plan a commercial refrigeration system using R-123 we would like to know how much more volume saturated vapor R-123 occupies per kg at -30 °C compared to the saturated liquid state.

Admin
Solution

For R-123 there is no section B table printed. We will use compressibility chart.

From Table A.2 \(T _{ c }=456.9 K ; \quad P _{ c }=3.66 MPa ; \quad M =152.93\)

 

\(T _{ r }= T / T _{ c }\) = 243/456.9 = 0.53

 

\(R =\overline{ R } / M =\) = 8.31451 kJ/kmol-K / 152.93 kg/kmol = 0.0544 kJ/kgK

 

The value of \(T _{ r }\) is below the range in Fig. D.1 so use the table D.4

Table D.4, \(Z _{ g }=0.979 \quad Z _{ f }=0.00222\)

 

\(Z _{ fg }=0.979-0.0022=0.9768 ; \quad P _{ r }= P _{ r \text { sat }}=0.0116\)

 

\(P = P _{ r } \times P _{ c }=42.5 kPa\)

 

\(v _{ fg }= Z _{ fg } RT / P\) = 0.9768 × 0.0544 kJ/kgK × 243 K / 42.5 kPa

 

\(=0.304 m ^{3} / kg\)

Comment: If you check with the software the solution is off by a factor of 6. The linear interpolation is poor and so is the approximation for \(P _{ r } \text { sat }\) so the real saturation pressure should be 6.75 kPa. Also the very small value of \(Z _{ f }\) is inaccurate by itself, minute changes in the curve gives large relative variations.
