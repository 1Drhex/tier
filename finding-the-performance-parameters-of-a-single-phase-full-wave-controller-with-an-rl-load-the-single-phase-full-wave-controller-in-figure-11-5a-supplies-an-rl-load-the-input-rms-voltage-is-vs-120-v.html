Finding the Performance Parameters of a Single-Phase Full-Wave Controller with an RL Load

The single-phase full-wave controller in Figure 11.5a supplies an RL load. The input rms voltage is \(V_s = 120 V, 60 Hz\). The load is such that \(L = 6.5 mH\) and \(R = 2.5Ω\). The delay angles of thyristors are equal:\(α_1 =α_2 = π/2\). Determine (a) the conduction angle of thyristor \(T_1,δ\); (b) the rms output voltage \(V_o\), (c) the rms thyristor current \(I_R\); (d) the rms output current \(I_o\); (e) the average current of a thyristor \(I_A\); and (f) the input PF.

Admin
Solution

\(R = 2.5 Ω , L = 6.5 mH, f = 60 Hz, ω = 2π × 60 = 377 rad/s, V_s = 120 V, α = 90°, and θ = \tan^{-1}(L/R) = 44.43°\).

a. The extinction angle can be determined from the solution of Eq. (11.9)

\(\sin(\beta -\theta )= \sin(\alpha -\theta )e^{(R/L)(\alpha -\beta )/\omega }\) (11.9)

and an iterative solution yields \(β= 220.35°\). The conduction angle is \(δ= β – α= 220.35 – 90 = 130.35°\).

b. From Eq. (11.11),

\(V_o=V_s\left[\frac{1}{\pi }\left(\beta -\alpha +\frac{\sin 2\alpha }{2}-\frac{\sin 2\beta }{2} \right) \right]^{1/2}\) (11.11)

the rms output voltage is \(V_o = 68.09 V\).

c. Numerical integration of Eq. (11.12)

\(I_R=\frac{V_s}{Z}\left[\frac{1}{\pi }\int_{\alpha }^{\beta }{\left\{\sin(\omega t-\theta )-\sin(\alpha -\theta )e^{(R/L)(\alpha /\omega -t)}\right\}^2d(\omega t) } \right]^{1/2}\) (11.12)

between the limits \(ωt = α\) to gives the rms thyristor current as \(I_R = 15.07 A\).

d. From Eq. (11.13),

\(I_o=(I_R^2+I_R^2)^{1/2}=\sqrt{2}I_R\) (11.13)

\(I_o=\sqrt{2}\times 15.07=213 A\).

e. Numerical integration of Eq. (11.14)

\(I_A=\frac{\sqrt{2}V_s }{2\pi Z}\int_{\alpha }^{\beta }{[\sin(\omega t-\theta )-\sin(\alpha -\theta )e^{(R/L)(\alpha /\omega -t)}]d(\omega t)}\) (11.14)

yields the average thyristor current as \(I_A = 8.23 A\).

f. The output power \(P_o = 21.3^2 × 2.5 = 1134.2 W\), and the input VA rating is \(VA = 120 × 21.3 = 2556 W\); therefore,

\(PF=\frac{P_o}{VA}=\frac{1134.200}{2556}=0.444\) ( lagging)

Note: The switching action of thyristors makes the equations for currents nonlinear. A numerical method of solution for the thyristor conduction angle and currents is more efficient than classical techniques. A computer program is used to solve this example. Students are encouraged to verify the results of this example and to appreciate the usefulness of a numerical solution, especially in solving nonlinear equations of thyristor circuits.
