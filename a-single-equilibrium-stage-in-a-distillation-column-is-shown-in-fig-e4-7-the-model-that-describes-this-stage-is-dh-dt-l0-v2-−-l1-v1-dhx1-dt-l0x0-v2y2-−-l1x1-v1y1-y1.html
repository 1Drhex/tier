Guest Question.

A single equilibrium stage in a distillation column is shown in Fig. E4.7. The model that describes this stage is

\(\begin{aligned}\frac{d H}{d t} &=L_{0}+V_{2}-\left(L_{1}+V_{1}\right) \\\frac{d H x_{1}}{d t} &=L_{0} x_{0}+V_{2} y_{2}-\left(L_{1} x_{1}+V_{1} y_{1}\right) \\y_{1} &=a_{0}+a_{1} x_{1}+a_{2} x_{1}^{2}+a_{3} x_{1}^{3}\end{aligned}\)

Assume that the molar holdup \(H\) in the stage is constant and that constant molal overflow occurs. Thus, \(L_{1}=L_{0}\) and \(V_{2}=V_{1}\)

(a) Linearize the resulting model and introduce deviation variables.

(b) For constant liquid and vapor flow rates, derive the four transfer functions relating outputs \(x_{1}\) and \(y_{1}\) to inputs \(x_{0}\) and \(y_{2} .\) Place it in standard form.

Admin
Solution

The simplified stage concentration model becomes

\(H \frac{d x_{1}}{d t}=L\left(x_{0}-x_{1}\right)+V\left(y_{2}-y_{1}\right)\) (1)

\(y_{1}=a_{0}+a_{1} x_{1}+a_{2} x_{1}^{2}+a_{3} x_{1}^{3}\) (2)

\(\text { a) Let the right-hand side of Eq. } 1 \text { be denoted as } f\left(L, x_{0}, x_{1}, V, y_{1}, y_{2}\right)\)

\(\begin{aligned}&H \frac{d x_{1}}{d t}=f\left(L, x_{0}, x_{1}, V, y_{1}, y_{2}\right)=\left(\frac{\partial f}{\partial L}\right)_{s} L^{\prime}+\left(\frac{\partial f}{\partial x_{0}}\right)_{s} x_{0}^{\prime}+\left(\frac{\partial f}{\partial x_{1}}\right)_{s} x_{1}^{\prime} \\&+\left(\frac{\partial f}{\partial V}\right)_{s} V^{\prime}+\left(\frac{\partial f}{\partial y_{1}}\right)_{s} y_{1}^{\prime}+\left(\frac{\partial f}{\partial y_{2}}\right)_{s} y_{2}^{\prime}\end{aligned}\)

\(\text { Substituting for the partial derivatives and noting that } \frac{d x_{1}}{d t}=\frac{d x_{1}^{\prime}}{d t} \text { : }\)

\(H \frac{d x_{1}^{\prime}}{d t}=\left(\bar{x}_{0}-\bar{x}_{1}\right) L^{\prime}+\bar{L} x_{0}^{\prime}-\bar{L} x_{1}^{\prime}+\left(\bar{y}_{2}-\bar{y}_{1}\right) V^{\prime}+\bar{V} y_{2}^{\prime}-\bar{V} y_{1}^{\prime}\) (3)

Similarly,

\(y_{1}^{\prime}=g\left(x_{1}\right)=\left(\frac{\partial g}{\partial x_{1}}\right)_{s} x_{1}^{\prime}=\left(a_{1}+2 a_{2} \bar{x}_{1}+3 a_{3} \bar{x}_{1}^{2}\right) x_{1}^{\prime}\) (4)

\(\text { b) For constant liquid and vapor flow rates, } L^{\prime}=V^{\prime}=0\)

Taking Laplace transforms of Eqs. 3 and 4,

\(H s X_{1}^{\prime}(s)=\bar{L} X_{0}^{\prime}(s)-\bar{L} X_{1}^{\prime}(s)+\bar{V} Y_{2}^{\prime}(s)-\bar{V} Y_{1}^{\prime}(s)\) (5)

\(Y_{1}^{\prime}(s)=\left(a_{1}+2 a_{2} \bar{x}_{1}+3 a_{3} \bar{x}_{1}^{2}\right) X_{1}^{\prime}(s)\) (6)

From Eqs. 5 and 6, the desired transfer functions are:

\(\frac{X_{1}^{\prime}(s)}{X_{0}^{\prime}(s)}=\frac{\frac{\bar{L}}{H} \tau}{\tau s+1}\quad\text{,}\quad \frac{X_{1}^{\prime}(s)}{Y_{2}^{\prime}(s)}=\frac{\frac{\bar{V}}{H} \tau}{\tau_{s}+1}\)

\(\begin{aligned}&\frac{Y_{1}^{\prime}(s)}{X_{0}^{\prime}(s)}=\frac{\left(a_{1}+2 a_{2} \bar{x}_{1}+3 a_{3} \bar{x}_{1}^{2}\right) \frac{\bar{L}}{H} \tau}{\tau s+1} \\&\frac{Y_{1}^{\prime}(s)}{Y_{2}^{\prime}(s)}=\frac{\left(a_{1}+2 a_{2} \bar{x}_{1}+3 a_{3} \bar{x}_{1}^{2}\right) \frac{\bar{V}}{H} \tau}{\tau s+1}\end{aligned}\)

where:

\(\tau=\frac{H}{\bar{L}+\bar{V}\left(a_{1}+2 a_{2} \bar{x}_{1}+3 a_{3} \bar{x}_{1}^{2}\right)}\)
