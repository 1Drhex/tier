Guest Question.

Complete each of the following reactions by writing the structure of the organic product. If no reaction takes place, write “no reaction.” If there is more than one product, indicate which is the major and which is the minor product.

Admin
Solution

(a) Only one product is possible, because the reagent is symmetrical.

(b) Only one product is possible, because the alkene is symmetrical.

(c) Two products are possible, because both the reagent and the alkene are asymmetrical. The Markovnikov rule tells you which product is the major product.
