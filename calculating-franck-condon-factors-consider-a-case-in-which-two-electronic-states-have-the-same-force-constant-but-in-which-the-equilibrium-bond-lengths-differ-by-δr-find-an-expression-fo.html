Calculating Franck–Condon factors
Consider a case in which two electronic states have the same force constant but in which the equilibrium bond lengths differ by ΔR. Find an expression for the relative intensity of the 0–0 transition as a function of ΔR.

Method We need to evaluate the Franck–Condon factor \(\left|S(0,0)\right| ^{2} \). To do so we calculate the overlap integral \(S(0,0)\) using harmonic oscillator wavefunctions (Table 2.1), one centred on x = 0 and the other on x = ΔR. We shall need the following integral:

\(\int_{-\infty }^{+\infty }{e^{-ax^{2} } } dx=\left(\frac{\pi }{a} \right) ^{1/2} \)

 

Table 2.1 Hermite polynomials
\(\nu\) \(H_{\nu}(z)\)
0 1
1 2z
2 \(4z^{2}-2\)
3 \(8z^{3}-12z\)
4 \(16z^{4}-48z^{2}+12\)
5 \(32z^{5}-160z^{3}+120z\)
6 \(64z^{6}-480z^{4}+720z^{2}-120\)
7 \(128z^{7}-1344z^{5}+3360z^{3}-1680z\)
8 \(256z^{8}-3584z^{6}+13440z^{4}+1680\)
Differential equation: \(H^{\prime \prime }_{\nu } -2zH^{\prime }_{\nu } +2\nu H_{\nu } =0\)
Recursion relation: \(H_{\nu +1} =2zH_{\nu } -2\nu H_{\nu -1}\)
Orthogonality: \(\int_{-\infty }^{\infty }{H_{\nu }(z)H_{\nu^{\prime } } }(z)e^{-z^{2} } dz=0\) for \(\nu \neq \nu ^{\prime } \)
Normalization: \(\int_{-\infty }^{\infty }{H_{\nu }(z)^{2} }e^{-z^{2} } dz=\pi ^{1/2} 2^{\nu } \nu !\)
Admin
Solution

The wavefunctions for the two states are

\(\Psi _{0} =\left(\frac{\alpha }{\pi ^{1/2} } \right) ^{1/2} e^{-\alpha ^{2}x^{2}/2 } \) \(\Psi _{0^{\prime } } =\left(\frac{\alpha }{\pi ^{1/2} } \right) ^{1/2} e^{-\alpha ^{2}(x-\Delta R)^{2}/2 } \)

where \(\alpha =(mk_{f}/\hbar ^{2} )^{1/4} \) and the wavefunctions are normalized in the sense

\(\int_{-\infty }^{+\infty }{\left|\Psi _{0} \right|^{2} dx} =1\)

It then follows that

\(S(0,0)=\left(\frac{\alpha }{\pi ^{1/2} } \right) \int_{-\infty }^{+\infty }{e^{-\alpha ^{2}x^{2}/2-\alpha ^{2}(x-\Delta R)^{2}/2 }dx } =\left(\frac{\alpha }{\pi ^{1/2} } \right) \int_{-\infty }^{+\infty }{e^{-\alpha ^{2}x^{2}/2-\alpha ^{2}x^{2}/2+\alpha ^{2}x\Delta R-\alpha ^{2} \Delta R^{2}/2 }dx }\)

 

\(=\left(\frac{\alpha }{\pi ^{1/2} } \right)e^{-\alpha ^{2} \Delta R^{2}/4} \int_{-\infty }^{+\infty }{e^{-\alpha ^{2}x^{2}+\alpha ^{2}x\Delta R-\alpha ^{2} \Delta R^{2}/4 }dx }= \left(\frac{\alpha }{\pi ^{1/2} } \right)e^{-\alpha ^{2} (\Delta R/2)^{2} } \int_{-\infty }^{+\infty }{e^{-\alpha ^{2}(x-\Delta R/2)^{2} }dx } = e^{-\alpha ^{2}(\Delta R/2)^{2}}\)

The Franck–Condon factor for the transition is therefore

\(\left|S(0,0)\right|^{2} = e^{-\alpha ^{2}(\Delta R)^{2}/2}\)

This function is plotted in Fig. 11.11, and the strong dependence on ΔR should be noticed.
