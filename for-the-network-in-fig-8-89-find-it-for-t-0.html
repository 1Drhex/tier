Guest Question.

For the network in Fig. 8.89, find i(t) for t > 0 .

Admin
Solution

At t = 0-, the switch is open. i(0) = 0, and

v(0) = 5×100/(20 + 5 + 5) = 50/3

For t > 0, we have a series RLC circuit shown in Figure (a). After source transformation, it becomes that shown in Figure (b).

\(\begin{array}{l}\omega_{0}=1 / \sqrt{\mathrm{LC}}=1 / \sqrt{1 \times 1 / 25}=5 \\\\\alpha=\mathrm{R} /(2 \mathrm{L})=(4) /(2 \times 1)=2 \\\\\mathrm{s}_{1,2}=-\alpha \pm \sqrt{\alpha^{2}-\omega_{\mathrm{o}}^{2}}=-2 \pm \mathrm{j} 4.583 \\\\\text {Thus}, \mathrm{v}(\mathrm{t})=\mathrm{V}_{\mathrm{s}}+\left[\left(\mathrm{Acos} \omega_{\mathrm{d}} \mathrm{t}+\mathrm{B} \sin \omega_{\mathrm{d}} \mathrm{t}\right) \mathrm{e}^{-2 \mathrm{t}}\right]\end{array}\)

where \(\omega_{\mathrm{d}}=4.583\) and \(\mathrm{V}_{\mathrm{s}}=20\)

\(\begin{array}{l}\mathrm{v}(0)=50 / 3=20+\mathrm{A} \text { or } \mathrm{A}=-10 / 3 \\\\\mathrm{i}(\mathrm{t})=\mathrm{Cdv} / \mathrm{dt}=\mathrm{C}(-2)\left[\left(\mathrm{Acos} \omega_{\mathrm{d}} \mathrm{t}+\mathrm{B} \sin \omega_{\mathrm{d}} \mathrm{t}\right) \mathrm{e}^{-2 t}\right]+\mathrm{C} \omega_{\mathrm{d}}\left[\left(-\mathrm{A} \sin \omega_{\mathrm{d}} \mathrm{t}+\mathrm{B} \cos \omega_{\mathrm{d}} \mathrm{t}\right) \mathrm{e}^{-2 t}\right] \\\\\mathrm{i}(0)=0=-2 \mathrm{A}+\omega_{\mathrm{d}} \mathrm{B} \\\\\mathrm{B}=2 \mathrm{A} / \omega_{\mathrm{d}}=-20 /(3 \times 4.583)=-1.455 \\\\\left.\mathrm{i}(\mathrm{t})=\mathrm{C}\left\{\left[0 \cos \omega_{\mathrm{d}} \mathrm{t}+\left(-2 \mathrm{B}-\omega_{\mathrm{d}} \mathrm{A}\right) \sin \omega_{\mathrm{d}} \mathrm{t}\right)\right] \mathrm{e}^{-2 \mathrm{t}}\right\} \\\\\left.=(1 / 25)\left\{\left[(2.91+15.2767) \sin \omega_{\mathrm{d}} \mathrm{t}\right)\right] \mathrm{e}^{-2 \mathrm{t}}\right\} \\\\\mathrm{i}(\mathrm{t})=\left\{0.7275 \sin (4.583 \mathrm{t}) \mathrm{e}^{-2 \mathrm{t}}\right\} \mathrm{A}\end{array}\)
