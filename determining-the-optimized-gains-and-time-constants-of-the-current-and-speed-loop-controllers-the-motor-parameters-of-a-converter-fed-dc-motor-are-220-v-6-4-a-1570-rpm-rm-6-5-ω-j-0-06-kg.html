Determining the Optimized Gains and Time Constants of the Current and Speed Loop Controllers

The motor parameters of a converter-fed dc motor are \(220 V, 6.4 A, 6.4 A, 1570 rpm,R_m=6.5\Omega ,J=0.06 kg−m^2,L_m= 67 mH,B= 0.08N−rn/rad/s,K_b=1.24 V/rad/s.\). The converter is supplied from a Y-connected 230 V, three-phase ac source at 60 Hz. The converter can be assumed linear and its maximum control input voltage is \(V_{cm} = ±10 V\). The tachogenerator has the transfer function \(G_\omega (s)=0.074/(1+0.002s)\). The speed reference voltage has a maximum of 10 V. The maximum permissible motor current is 20 A. Determine (a) the gain \(K_r\) and time constant \(τ_r\) of the converter, (b) the current feedback gain \(H_c\) , (c) the motor time constant \(τ_1, τ_2\), and \(τ_m\), (d) the gain \(K_c\) and the time constant \(\tau _c\) of the current controller, (e) the gain \(K_i\) and the time constant \(\tau _i\) of the simplified current loop, and (f) the optimized gain \(K_s\) and the time constant \(τ_s\) of the speed controller.

Admin
Solution

\(V_{dc} = 220 V, I_{dc} = 6.4 A, N = 1570 rpm, R_m = 6.5 \Omega \), \(J = 0.06 kg−m^2, L_m = 67 mH, B = 0.087 N-m /rad/s, Kb = 1.24 V/rad/s, V_L = 220 V, f_s = 60 Hz, V_{cm }=± 10 V 20 A, I_{a(max)} = 20 A, K_ω = 0.074, t_ω= 0.002 s\).

a. The phase voltage is \(V_s=\frac{V_L}{\sqrt{3} }=\frac{220}{\sqrt{3} }=127.02 V\)

The maximum dc voltage is \(V_{dc(max)} = K_rV_{cm} = 29.71 × 10 = 297.09 V\)

The converter control voltage is \(V_c=\frac{V_{dc}}{V_{dc(max)}}V_{cm}=\frac{220\times 110}{297.09}=7.41 V\)

Using Eq. (14.88),

\(K_r=\frac{3\sqrt{3} V_m}{\pi V_{cm}} =\frac{3\sqrt{3} \sqrt{2} }{\pi V_{cm}}V_s =2.339\frac{V_s}{V_{cm}}\) (for three−phase converter) (14.88)

\(K_s=\frac{2339V_s}{V_{cm}}=\frac{2.339\times 127.02}{10}=29.71 V/V\)

Using Eq. (14.91a)

\(\tau _r=\frac{360/(2\times 6)}{360}T_s= \frac{1}{12}\times \frac{1}{f_s}\) (for three-phase converter) (14.91a)

\(\tau _r=\frac{1}{12f_s}=\frac{1}{12\times 60}=1.39 ms\)

b. The current feedback gain is \(H_c=\frac{V_c}{I_{a(max)}}=\frac{7.41}{20}=0.37 V/A\)

c. Using \(B_t = B\) and Eq. (14.99),\(K_m=\frac{B}{K_b^2+R_mB}=\frac{0.087}{1.24^2+6.5\times 0.087}=0.04\)

Using Eq. (14.100),

\(-\frac{1}{\tau _1};-\frac{1}{\tau _2}=\frac{-\left(\frac{R_m}{L_m}+\frac{B}{J} \right)\pm \sqrt{\left(\frac{R_m}{L_m}+\frac{B}{J} \right)^2 -4\left(\frac{R_mB+K_b^2}{JL_m} \right) } }{2}\) (14.100)

\(r_1 = -5.64\) ; \( \tau _1=\frac{-1}{r_1} =0.18 s\)

 

\(r_2 = -92.83\) ; \( \tau _2=\frac{-1}{r_2} =0.01 s\)

 

\(\tau _m=\frac{J}{B} =\frac{0.06}{0.087}=0.69 s\)

d. The time constant of the current controller is \(\tau _c=\tau _2=0.01 s\)

Using Eq. (14.110), \(K_c=\frac{\tau _1\tau _2}{2\tau _r}\left(\frac{1}{K_mK_rH_c\tau _m} \right)=\frac{0.18\times 0.01 }{0.04\times 29.71\times 0.37\times 0.69}=2.19\)

\(K_c=\frac{\tau _1\tau _2}{2\tau _r}\left(\frac{1}{K_mK_rH_c\tau _m} \right)\) (14.110)

Using Eqs (14.112) to (14.115),

\(\tau _i=\frac{\tau _3}{1+K_1}\) (14.112)

\(\tau _3=\tau _1+\tau _r\) (14.113)

\(K_i=\frac{1}{H_c}\left(\frac{K_1}{1+K_1} \right)\) (14.114)

\(K_1=\frac{K_mK_cK_rH_c\tau _m}{\tau _c}\) (14.115)

\(K_1=\frac{K_mK_cK_rH_c\tau _m}{\tau _c}=\frac{0.4 \times 2.19 \times 29.71 \times 0.37 \times 0.69}{0.01}=63.88\)

 

\(K_i=\frac{1}{H_c}\left(\frac{K_1}{1+K_1} \right)=\frac{1}{0.37}\left(\frac{63.88}{1+63.88} \right)=2.66\)

 

\(\tau _3=\tau _1+\tau _r=0.18 + 0.00139 = 0.18 s\)

 

\(\tau _i=\frac{\tau _3}{1+K_1}=\frac{0.18}{1+63.88}=2.76 ms\)

e. Using Eqs (14.118) and (14.119),

\(\tau _e=\tau _i+\tau _\omega \) (14.118)

\(K_\omega =\frac{KiK_bH_\omega }{B\tau _m} \) (14.119)

\(\tau _e=\tau _i+\tau _\omega =2.76 \times 10^{-3} + 2 \times 10^{-3} = 4.76 ms\)

 

\(K_\omega =\frac{KiK_bH_\omega }{B\tau _m}=\frac{2.66 \times 1.24 \times 0.074}{0.087 \times 0.69}=4.07\)

Using Eqs (14.121) and (14.122)

\(K_s=\frac{1}{2K_\omega \tau _e}\) (14.121)

\(\tau _s=4\tau _e\) (14.122)

\(K_s=\frac{1}{2K_\omega \tau _e}=\frac{1}{2 \times 4.07 \times 4.76 \times 10^{-3}}=25.85\)

 

\(\tau _s=4\tau _e=4 \times 4.76 \times 10^{-3} = 19.02 ms\)
