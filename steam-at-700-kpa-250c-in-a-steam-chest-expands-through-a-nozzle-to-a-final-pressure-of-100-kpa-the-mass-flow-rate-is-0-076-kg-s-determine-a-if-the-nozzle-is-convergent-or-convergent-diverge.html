Guest Question.

Steam at 700 kPa, 250°C in a steam chest expands through a nozzle to a final pressure of 100 kPa. The mass flow rate is 0.076 kg/s Determine (a) if the nozzle is convergent or convergent-divergent (b) the throat diameter (c)the exit diameter and (d) the dryness fraction at the exit. Assume the expansion process to be isentropic and in equilibrium throughout.°

Admin
Solution

From the steam table, it is easy to establish that for the given steam chest conditions of \(P_{0} = 700 kPa\) and \(T_{0}= 250°C\) the steam is initially superheated. Also, \(v_{0} = 0.336343 m^{3} / kg\) and \(s_{0}= 7.1062 kJ/kg.K.\)

(a) Since the steam is initially superheated, \(n=1.3\) from Eqn. 9.4. From Eqn. 9.10, we have \(P^{\ast } = 0.545\) \(P_{0} = 380 kPa.\) Since the flow is expanded to a final pressure of 100 kPa in the nozzle, it is clear that the nozzle is convergent-divergent.

\(n= \begin{cases} 1.3 & \quad if superheated \\ 1.035+ 0.1x_{1} & saturated mixture \end{cases}\) (9.4)

 

\(\frac{p^{\ast } }{P_{0} } =\left(\frac{2}{n+ 1} \right) ^{n/ \left(n-1\right) }\) (9.10)

(b) From Eqn. 9.3, we can get

\(P_{v^{n} } = constant \) (9.3)

 

\(v^{\ast } =v_{0} \left(\frac{P^{\ast } }{P_{0} } \right) ^{-1/ n}= 0.53648 m^{3} / kg\)

and

\(v^{\ast } =\sqrt{nP^{\ast }v^{\ast } }= 514.8m/s .\)

Since

\(A^{\ast } =\frac{\dot{m}v^{\ast } }{V^{\ast } } =7.92\times 10^{-5} m^{2} ,\)

the throat diameter\( D^{\ast } = 10 mm.\)

 

(c) Since the expansion process is isentropic, the process line crosses the
saturated vapor line at \(P_{g} = 212 kPa.\)With \(n= 1.3\)(as the flow is superheated until it crosses the saturated vapor line), we can get

\(h_{0} -h_{g} =\frac{n}{n-1} P_{0} v_{0} \left[1-\left(\frac{P_{g} }{P_{0} } \right) ^{\left(n-1/ n\right) } \right]=245.8 kJ/kg \)

and

\(v_{g} =v_{0} \left(\frac{P_{g} }{P_{0} } \right) ^{-1/ n}= 0.843 m^{3} / kg\)

It follows that \(V_{g}=\sqrt{2\left(h_{0} -h_{g} \right) } = 701.14 m/s.\) Once the flow crosses the saturated vapor line, \(n=1.135\) (this implies that the flow continues to be in equilibrium after crossing the saturated vapor line). Hence

\(h_{g} -h_{e} =\frac{n}{n-1} P_{g} v_{g} \left[1-\left(\frac{P_{e} }{P_{g} } \right) ^{\left(n-1/ n\right) } \right]=127.77 kJ/kg \)

and

\(v_{e} =v_{g} \left(\frac{P_{e} }{P_{g} } \right) ^{-1/ n}= 1.6344 m^{3} / kg\)

Therefore

\(V_{e} =\sqrt{V^{2}_{g}+ 2\left(h_{g} – h_{e} \right) } = 864m/s .\)

For the given mass flow rate, the exit area can be calculated as

\(A_{e} \frac{\dot{m}v_{e} }{V_{e} } =1.44\times 10^{-4} m^{2}\)

and the exit diameter can be calculated as \(D_{e} =13.5 mm.\)

(d) From the given value of the exit pressure \(P_{e} \) and the calculated value of \(v_{e}\), we can get the dryness fraction at the exit to be 0.96 using steam tables.

Alternatively, we can use the steam table (or Mollier diagram) to solve the problem. For the given steam chest condition, we can get \(h_{0} =2954kJ/kg\) and \(s_{0}= 7.1062 kJ/kg.K.\)

The pressure at the throat \(P^{\ast } = 0.545\) \(P_{0}= 380 kPa.\) Since the expansion is isentropic, \(s^{\ast } =s_{0}=7.1062 kJ/kg.K. \) Therefore, the fluid is superheated at the throat with \(v^{\ast } = 0.537 m^{3}/ kg\) and \(h^{\ast } = 2820 kJ/kg.\) It follows that

\(V^{\ast } =\sqrt{2\left(h_{0}- h^{\ast } \right) } = 518m/s\)

and

\(A^{\ast }=\frac{\dot{m}v^{\ast } }{V^{\ast } } =7.86\times 10^{-5} m^{2} .\)

Thus, the throat diameter \( D^{\ast } = 10 mm.\)

We have \(P_{e} = 100 kPa\) and \(s_{e} =s_{0} = 7.112 kJ/kg.K.\) t can be determined that the fluid at the exit is a saturated mixture. The dryness fraction \(x_{e}\) can be evaluated as 0.96. Also, \(h_{e} = 2581 kJ/kg\) and \(v_{e}= 1.623 m^{3}/kg.\) Hence

\(V_{e} =\sqrt{ 2\left(h_{0} – h_{e} \right) } = 864m/s .\)

and

\(A_{e} \frac{\dot{m}v_{e} }{V_{e} } =1.43\times 10^{-4} m^{2}\)

Thus, the exit diameter \(D_{e} = 13.5 mm.\)
