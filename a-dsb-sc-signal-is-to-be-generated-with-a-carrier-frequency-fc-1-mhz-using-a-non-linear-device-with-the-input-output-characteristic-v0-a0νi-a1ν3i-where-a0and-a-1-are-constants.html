Guest Question.

A DSB-SC signal is to be generated with a carrier frequency \(f_{ c }\) = 1 MHz using a non-linear device with the input–output characteristic \(V_{0}=a_{0} \nu _{ i }+a_{1} \nu _{ i }^{3}\) where \(a_{0}\) and \(a_{1}\) are constants. The output of the non-linear device can be filtered by an appropriate band-pass filter. Let \(V_{i}=A_{c}^{i} \cos \left(2 \quad f_{c}^{i} t\right)+m(t)\) where m(t) is the message signal. Then the value of \(f ^{i}{ }_{c}\) (in MHz) is

(a) 1.0 (b) 0.333

(c) 0.5 (d) 3.0

Admin
Solution

Given

\(\begin{aligned}V _{0}=& a_{0} \nu _{ i }+a_{1} \nu _{ i }^{3} \\=& a_{0}\left[A_{c}^{1} \cos w_{c}^{i} t\right]+a_{0} m(t)+a_{1} A_{c}^{c^{3}} \cos 3 w_{c}^{i} t \\&+a_{1} m^{3}(t)+3 a_{1} m^{2}(t) A_{c}^{i} \cos w_{c}^{i} t+3 a_{1} A_{c}^{i} \cos ^{2} w_{c}^{i} t\end{aligned}\)

For DSB SC we are concerned with only last term
\(m(t) \cos w_{c} t \rightarrow f_{c}=1 MHz\)
For \(\cos ^{2} \text { term } 2 w_{c}^{i}=2 \pi \times 1 MHz\)
Hence, the correct option is (c).
