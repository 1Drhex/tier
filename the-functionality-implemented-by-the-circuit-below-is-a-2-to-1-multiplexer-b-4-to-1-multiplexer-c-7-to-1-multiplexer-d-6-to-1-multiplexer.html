Guest Question.

The functionality implemented by the circuit below is
(A) 2 to 1 multiplexer
(B) 4 to 1 multiplexer
(C) 7 to 1 multiplexer
(D) 6 to 1 multiplexer

Admin
Solution

\(C_{1}C_{0}\) of decoder will activate \((=1)\), one of the output \(O_{0}\) to \(O_{3}\). So that corresponding tristate buffer will give output same as input, other tristate buffers will be disabled.

Y \(O_{3}\) \(O_{2}\) \(O_{1}\) \(O_{0}\) \(C_{0}\) \(O_{1}\)
P 0 0 0 1 0 0
Q 0 0 1 0 1 0
R 0 1 0 0 0 1
S 1 0 0 0 1 1

 

The two inputs of decoder are handling 4 outputs simultaneously disabling the unused buffers therefore; it will behave as multiplexer.
Hence, the correct option is (B).
