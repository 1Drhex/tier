Guest Question.

Suppose that \(n_{i} ≪ Δ\) so that Eq. (25.114) holds. Then n ∼ Δ and p ∼ 0. Show that μ approaches the edge of the conduction band \(ε_{c} as Δ \) approaches \(n^{∗}.\) Discuss the breakdown of the approximation that leads to the second form of Eq. (25.99). Evaluate μ under conditions for which Δ becomes sufficiently large that μ enters the conduction band. Then examine the same problem except for Δ negative but of large magnitude.

\(n \approx \left(Δ + \left|Δ\right| \right) / 2 + n^{2}_{i} / \left|Δ\right| ; p \approx \left(- Δ + \left|Δ\right| \right) / 2 + n^{2}_{i} / \left|Δ\right|,\) (25.114)

\(n = \int_{\varepsilon _{c} }^{\infty }{g_{c} (\varepsilon ) \frac{1}{e^{\beta (\varepsilon - \mu )} + 1} d\varepsilon } \approx \int_{\varepsilon _{c}}^{\infty }{g_{c} (\varepsilon ) e^{-\beta (\varepsilon - \mu )}} d\varepsilon ,\) (25.99)

Admin
Solution

From Eq. (25.105) with n = Δ we have

\(n = n^{*} e^{-\beta (\varepsilon_{c} – \mu )} ; p = p^{*} e^{-\beta (\mu – \varepsilon _{v})}, \) (25.105)

\(\varepsilon_{c} – \mu = k_{B}T \ln \left(n^{*} / Δ\right) .\) (25.128)

As Δ approaches \(n^{∗}\) from below, we see that μ appears to approach \(ε_{c}\) from below. Examination of Eq. (25.99) shows that the second form becomes invalid under these conditions because exp[β(ε − μ)] will no longer be large in the range of integration. Thus, Eq. (25.105) is no longer valid. From the first form of Eq. (25.99) and with \(g_{c}(ε)\) given by Eq. (25.104), we obtain

\(g_{c} (w + \varepsilon _{c}) = 2\frac{2}{\pi ^{1/2}} \left(\frac{m_{n}}{2\pi \hbar ^{2}} \right)^{3/2} w^{1/2} ; g_{v} ( \varepsilon _{v} – w) = 2 \frac{2}{\pi ^{1/2}} \left(\frac{m_{p}}{2\pi \hbar ^{2}} \right)^{3/2} w^{1/2}.\) (25.104)

\(n = n^{*} \frac{1}{\Gamma (3/2)} \int_{0}^{\infty }{\frac{u^{1/2}}{\lambda^{-1}_{C} e ^{u} + 1}} du = n^{*} f_{3/2} (\lambda _{C}),\) (25.129)

where \(λ_{c} = exp[β(μ − ε_{c})]\) and \(f_{3/2}\) is a fermion function defined in Chapter 23.

If \(λ_{c}\) were small, as it would be for μ near the middle of the band gap, one would have \(f_{3/2}(λ_{c}) = λ_{c}\) and Eq. (25.105) for n is recovered. But for heavy doping of donors, so that \(n ≈Δ\gg n^{∗}, λ_{c}\) will be large and we can use the asymptotic form \(f_{3/2}(λ_{c}) = [β(μ − ε_{c})]^{3/2} /Γ(5/2)\) to obtain

\(\mu – \varepsilon_{C} = \frac{\hbar ^{2}}{2m_{n}}\left(3\pi ^{2} Δ\right)^{2/3}.\) (25.130)

For Δ < 0 but \(|Δ| \gg p^{∗},\) one will have p ∼ |Δ| and n ∼ 0. Now (with \(β_{ε_{v}} ≈ \infty\) in the upper limit of the integral),

\(p = p^{*} \frac{1}{\Gamma (3/2)} \int_{0}^{\infty }{\frac{u^{1/2}}{\lambda ^{-1}_{V} e^{u }+ 1 } } du = p^{*} f_{3/2} (\lambda _{v}) \) (25.131)

with \(λ_{v} = exp[β(ε_{v} − μ.)] \) The chemical potential will move into the valence band and

\(\varepsilon_{v} – \mu = \frac{\hbar ^{2}}{2m_{p}} \left(3\pi ^{2} \left|Δ\right| \right)^{2/3} .\) (25.132)
