Guest Question.

A unity feedback control loop consists of a non-linear element N and a number of linear elements in series which together approximate to the transfer function:
\(G_{s}=\frac{3}{s(s+1)(s+2)}\)
Determine the range of values of the amplitude x0 of an input disturbance for which the system is stable where (a) N represents a dead-zone element for which the gradient k of the linear part is 4; (b) N is a saturating element for which k = 5; and (c) N is an onâ€“off device for which the total change in signal level is 20 units.

Admin
Solution

A block diagram for this problem is shown in Figure 7u.

The closed loop transfer function for a set-point change is given by:

\(\frac{\overline{C}}{\overline{R}}=\frac{NG(s)}{1+NG(s)}\), where G(s) =\(\frac{3}{s(s+1)(s+2)}\)

(a) When N represents a dead-zone element, then:

\(N=\frac{k}{\pi }(\pi -2\beta -\sin 2\beta )\) (equation 7.201)

where \(\beta =\sin^{-1}\delta /x_{0}\) and the dead-zone is \(2\delta \) .

Using the substitution rule:

\(G(i\omega )=\mathbf{\frac{3}{i\omega (i\omega +1)(i\omega +2)}}\) \(=\mathbf{\frac{-3}{\omega^{2}+i(\omega^{3}-2\omega )}=\frac{-3\left [ 3\omega ^{2}-i(\omega ^{3}-2\omega ) \right ]}{9\omega ^{4}+(\omega ^{3}-2\omega )}}\) \(=\mathbf{\frac{-9\omega ^{2}}{9\omega^{4}+(\omega^{3}-2\omega )^{2}}+i\cdot \frac{\omega ^{3}-2\omega }{9\omega ^{4}+(\omega ^{3}-2\omega )^{2}}}\)

the conditionally stable conditions are given when:

\(G(i\omega )=-\frac{1}{N}\)

\(-\frac{1}{N}\) lies on the real axis, that is where Im\([G(i\omega )]\)=0

or where: \(\omega ^{3}=2\omega \)

and \(\omega =0 \) or \(\omega =\sqrt{2}\)

At this point: \(\mathbf{G(i\omega )=Re[G(i\omega )]=\frac{-9\omega ^{2}}{9\omega ^{4}+(\omega ^{3}-2\omega )^{2}}}\)

When \(\omega =\sqrt{2}\) then:

\(G(i\omega )=\frac{-18}{36+\sqrt{}2(2-2)^{2}}=-0.5\)

That is: \(-\frac{1}{N}\) = -0.5 and N = 2

When k = 4, N/k = 0.5:

\(x_{0}/\delta \)=2.5

Hence, for k = 4, the system will be unstable if:

\(x_{0}> 2.5\delta \)

(b) For a saturating non-linear element with k = 5,

N/k = 2/5 = 0.4

the corresponding value of \(x_{0}/\delta \) is approximately 3.1.

Hence, in this case, the system will be unstable if:

\(x_{0}> 3.1\delta \)

(c) For an onâ€“off device having a total change in signal level of 20 units, \(Y_{0}\) = 10 units

Thus: \(N=4Y_{0}/\pi x_{0}\) (equation 7.198)

Hence: \(x_{0}=4Y_{0}/N\pi =40/2\pi =6.4\)

N is as in part (a) of the solution.

Thus, in this case, the system will be unstable if:

\(x_{0}> 6.4\) units.
