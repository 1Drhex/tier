Guest Question.

Perfectly conducting concentric spheres have radii of 2 and 6 cm. The region 2 < r < 3 cm is filled with a solid conducting material for which \(\sigma\) = 100 S/m, while the portion for which 3 < r < 6 cm has \(\sigma\) = 25 S/m. The inner sphere is held at 1 V while the outer is at V = 0. (a) Find E and J everywhere. (b) What resistance would be measured between the two spheres? (c) What is V at r = 3 cm?

Admin
Solution

a) From symmetry, E and J will be radially-directed and we note the fact that the current, I, must be constant at any cross-section; i.e., through any spherical surface at radius r between the spheres. Thus we require that in both regions,
\(J=\frac { 1 }{ 4\pi { r }^{ 2 } } { a }_{ r }\)
The fields will thus be
\({ E }_{ 1 }=\frac { 1 }{ 4\pi { \sigma }_{ 1 }{ r }^{ 2 } } { a }_{ r }(2<r<3)\) and \({ E }_{ 2}=\frac { 1 }{ 4\pi { \sigma }_{ 2 }{ r }^{ 2 } } { a }_{ r }(3<r<6)\)
where \({ \sigma }_{ 1 }\) = 100 S/m and \({ \sigma }_{ 2 }\) = 25 S/m. Since we know the voltage between spheres (1V). we can find the value of 1 through:
1V=\(-\int _{ .06 }^{ .03 }{ \frac { 1 }{ 4\pi { \sigma }_{ 2 }{ r }^{ 2 } } } { d }_{ r }-\int _{ .03 }^{ .02 }{ \frac { 1 }{ 4\pi { \sigma }_{ 1 }{ r }^{ 2 } } } { d }_{ r }=\frac { 1 }{ 0.24\pi } \left[ \frac { 1 }{ { \sigma }_{ 1 } } +\frac { 1 }{ { \sigma }_{ 2 } } \right]\)
and so
I=\(\frac { 0.24\pi }{ (1/{ \sigma }_{ 1 }+1/{ \sigma }_{ 2 }) }\) =15.08A
Then finally, with I = 15.08 A substituted into the field expression above, we find
\({ E }_{ 1 }=\frac { .012 }{ { r }^{ 2 } } { a }_{ r }\)V/m(2 < r < 3)
and
\({ E }_{ 2 }=\frac { .048 }{ { r }^{ 2 } } { a }_{ r }\)V/m(3 < r < 6)
The current density is now
\(J={ { \sigma }_{ 1 }E }_{ 1 }={ { \sigma }_{ 2 }E }_{ 2 } = \frac { 1.2 }{ { r }^{ 2 } } A/m(2<r<6)\)
b) We use
R=\(\frac { V }{ I } = \frac { 1V }{ 15.08A } =6.63\times { 10 }^{ -2 }\Omega \)
c) This we find through
V=\(-\int _{ .06 }^{ .03 }{ \frac { .048 }{ { r }^{ 2 } } } { d }_{ r }=.048\left( \frac { 1 }{ .03 } -\frac { 1 }{ .06 } \right) =0.8\)V
