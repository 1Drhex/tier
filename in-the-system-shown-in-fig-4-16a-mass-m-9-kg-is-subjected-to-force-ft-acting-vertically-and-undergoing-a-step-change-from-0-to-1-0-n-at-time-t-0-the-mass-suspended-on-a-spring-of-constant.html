Guest Question.

In the system shown in Fig. 4.16(a), mass m = 9 kg is subjected to force F(t) acting vertically and undergoing a step change from 0 to 1.0 N at time t = 0. The mass, suspended on a spring of constant k = 4.0 N/m, is moving inside an enclosure with a coefficient of friction between the surfaces of b = 4.0 N s/m. Using force F(t) as the input variable and the position of mass x(t) as the output variable, sketch an approximate step response of the system. If this response is oscillatory, determine the necessary modification to make the system critically damped.

Admin
Solution

The input–output equation of the system is

\(m\ddot{x} + b\dot{x} + kx=F\left(t\right) .\)

Substitution of the numerical values for the system parameters yields

\(9\ddot{x} + 4\dot{x} + 4x=F\left(t\right) .\)

From Eq. (4.66), \(\zeta =\frac{a_{1} }{2\sqrt{a_{0}a_{2} } } .\) the damping ratio is

\(\zeta =\frac{4 }{2\sqrt{\left(9\right) \left(4\right) } } =0.3333.\)

Because ζ< 1, the system step response will be oscillatory. Other step response specifications useful in sketching the step response curve can be determined from Eqs. (4.65) \(w_{n} =\sqrt{\frac{a_{0} }{a_{2} } } \) ,(4.74) \(w_{d} =w_{n} \sqrt{1- \zeta ^{2 } }\) ,(4.107) \(T_{d} =\frac{2\pi }{w_{d} }\) , and (4.113) \( M^{\%} _{p} =100e^{- \frac{\pi \zeta }{\sqrt{1- \zeta ^{2} } } } \) . as follows:

\(w_{n} =\sqrt{{4}/{9}} =0.6667 {rad}/{s} ,\)

 

\(w_{d} =0.6667\sqrt{1- 0.3333^{2} } =0.6286 {rad}/{s}\)

 

\(T_{d} =10.0s,\)

 

\(M^{\%} _{p} =100e^{- \frac{\pi 0.3333}{\sqrt{1- 0.3333^{2} } } } =32.94\%.\)

Given these parameters, the step response can be sketched as shown in Fig. 4.16(b). To make the system critically damped, another damper, \(b_{ad}\), is added, as shown in Fig. 4.17(a). The input–output equation of the modified system is

\(9\ddot{x} + \left(4+ b_{ad} \right) \dot{x} + 4x=F\left(t\right) .\)

The damping ratio is now given by

\(\zeta =\frac{\left(4+ b_{ad} \right) }{2\sqrt{4- 9} } .\)

For critical damping, the term on the right-hand side of the preceding equation must be equal to 1.0, which yields the value of additional damping, \(b_{ad} =8 N {s}/{m}\). The step response curve of the system with additional damping is shown in Fig. 4.17(b).
