Guest Question.

A piston/cylinder contains 0.1 kg water at 500°C, 1000 kPa. The piston has a stop at half the original volume, similar to Fig. 6.137. The water now cools to room temperature 25°C. Find the heat transfer and the entropy generation.

Admin
Solution

Energy Eq.: \(m \left( u _{2}- u _{1}\right)={ }_{1} Q _{2}-{ }_{1} W _{2}\)

Entropy Eq.: \(m \left( s _{2}- s _{1}\right)={ }_{1} Q _{2} / T _{\text {room }}+{ }_{1} S _{2, gen }\)

Process Eq: \(P = C \text { if } v > v _{\text {stop }} ; \quad V = C \text { if } P < P _{\text {float }}\)

State 1: \(v _{1}=0.35411 m ^{3} / kg , u _{1}=3124.34 kJ / kg\)

\(s _{1}=7.7621 kJ / kgK\)

State a: \(v _{ a }= v _{1} / 2=0.177055 m ^{3} / kg < v _{ g 1000 kPa }\) so \(T _{ a }= T _{\text {sat } 1000 kPa }=179.9^{\circ} C\)

The possible state 2 (P,V) combinations are shown. State “a” is (1000 kPa, \(v _{ a }\) ) so it is two-phase with \(T _{ a }=180^{\circ} C > T _{2}\)

\(P _{2}= P _{ sat 25 C }=3.169 kPa \quad \text { and } v _{2}= v _{ a }\)

 

\(\begin{aligned}x _{2} &=\left( v _{2}- v _{ f }\right) / v _{ fg }=(0.177-0.001003) / 43.358 \\&=0.0040604\end{aligned}\)

 

\(\begin{array}{l}u _{2}= u _{ f }+ x _{2} u _{ fg }=104.86+ x _{2} 2304.9=114.219 kJ / kg \\s _{2}= s _{ f }+ x _{2} s _{ fg }=0.3673+ x _{2} 8.1905=0.40056 kJ / kgK\end{array}\)

 

\(\begin{array}{l}{ }_{1} W _{2}= m \int P dv = m P _{1}\left( v _{2}- v _{1}\right) \quad[\text { see area below process curve in figure }]\\=0.1 kg \times 1000 kPa \times(0.177055-0.35411) m ^{3} / kg =-17.706 kJ\end{array}\)

 

\(\begin{aligned}{ }_{1} Q _{2} &= m \left( u _{2}- u _{1}\right)+{ }_{1} W _{2}=0.1 kg (114.219-3124.34) kJ / kg -17.706 kJ \\&=- 3 1 8 . 7 2 kJ\end{aligned}\)

 

\(\begin{aligned}{ }_{1} S _{2, gen } &= m \left( s _{2}- s _{1}\right)-{ }_{1} Q _{2} / T _{\text {room }} \\&=0.1 kg (0.40056-7.7621) kJ / kgK +318.72 kJ / 298.15 K \\&=-0.73615 kJ / K +1.06899 kJ / K = 0 . 3 3 3 kJ / K\end{aligned}\)
