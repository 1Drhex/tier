Guest Question.

Find poles and zeros of \( H ( s )=10 s /\left( s ^{2}+2 s +26\right) \). Place them in the s-domain and use the pole-zero plot to sketch H(jω).

Admin
Solution

H(s) has a zero at \( z_{1}=0 . \quad \text { Its poles } p _{1} \text { and } p _{2} \text { are found from } s ^{2}+2 s +26=0 \text { so that } p _{1}=-1+j 5 \) and \( p _{2}=-1-j 5 \). The pole-zero plot is shown in Fig. 12-14(a). The network function can then be written as

\( H ( s )=(10) \frac{ s – z _{1}}{\left( s – p _{1}\right)\left( s – p _{2}\right)} \)

 

For each value of s, the term \( \left( s – z _{1}\right) \) is a vector originating from the zero \( Z _{1} \) and ending at point s in the s-domain. Similarly, \( s – p _{1} \text { and } s – p _{2} \) are vectors drawn from poles \( p _{1} \text { and } p _{2} \), respectively, to the point s. Therefore, for any value of s, the network function may be expressed in terms of three vectors A, B, and C as follows:

\( H ( s )=(10) \frac{ A }{ B \times C } \quad \text { where } A =\left( s – z _{1}\right), B =\left( s – p _{1}\right), \text { and } C =\left( s – p _{2}\right) \)

The magnitude and phase angle of H(s) at any point on the s-plane may be found from:

\( | H ( s )|=(10) \frac{| A |}{| B | \times| C |} \) (5a)

 

\( \angle H ( s )=\angle A -\angle B -\angle C \) (5b)

By placing s on the jω axis [Fig. 12-14(a)], varying ω from 0 to ∞, and measuring the magnitudes and phase angles of vectors A, B, and C, we can use (5a) and (5b) to find the magnitude and phase angle plots. Figure 12-14(b) shows the magnitude plot.
