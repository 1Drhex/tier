Guest Question.

Calculate the wavelength, in centimeters, of the photon emitted under a hyperfine transition in the ground state ( n=1) of deuterium. Deuterium is “heavy” hydrogen, with an extra neutron in the nucleus; the proton and neutron bind together to form a deuteron, with spin 1 and magnetic moment

\( \mu_{d}=\frac{g_{d} e}{2 m_{d}} S _{d} \);

the deuteron g-factor is 1.71.

Admin
Solution

Equation 7.93 ⇒ \( E_{ hf }^{1}=\frac{\mu_{0} g_{d} e^{2}}{3 \pi m_{d} m_{e} a^{3}}\left\langle S _{d} \cdot S _{e}\right\rangle ; \quad \text { Eq. } 7.95 \Rightarrow S _{d} \cdot S _{e}=\frac{1}{2}\left(S^{2}-S_{e}^{2}-S_{d}^{2}\right) \).

\( E_{ hf }^{1}=\frac{\mu_{0} g_{p} e^{2}}{3 \pi m_{p} m_{e} a^{3}}\left( S _{p} \cdot S _{e}\right) \) (7.93).

\( S _{p} \cdot S _{e}=\frac{1}{2}\left(S^{2}-S_{e}^{2}-S_{p}^{2}\right) \) (7.95).

Electron has spin \( \frac{1}{2}, \text { so } S_{e}^{2}=\frac{1}{2}\left(\frac{3}{2}\right) \hbar^{2}=\frac{3}{4} \hbar^{2} ; \text { deuteron has spin } 1, \text { so } S_{d}^{2}=1(2) \hbar^{2}=2 \hbar^{2} \).

Total spin could be \( \frac{3}{2}\left[\text { in which case } S^{2}=\frac{3}{2}\left(\frac{5}{2}\right) \hbar^{2}=\frac{15}{4} \hbar^{2}\right] \text { or } \frac{1}{2}\left[\text { in which case } S^{2}=\frac{3}{4} \hbar^{2}\right. \text { ] } \). Thus

\( \left\langle S _{d} \cdot S _{e}\right\rangle=\left\{\begin{array}{ll} \frac{1}{2}\left(\frac{15}{4} \hbar^{2}-\frac{3}{4} \hbar^{2}-2 \hbar^{2}\right) & =\frac{1}{2} \hbar^{2} \\ \frac{1}{2}\left(\frac{3}{4} \hbar^{2}-\frac{3}{4} \hbar^{2}-2 \hbar^{2}\right) & =-\hbar^{2} \end{array}\right\} ; \text { the difference is } \frac{3}{2} \hbar^{2}, \text { so } \Delta E=\frac{\mu_{0} g_{d} e^{2} \hbar^{2}}{2 \pi m_{d} m_{e} a^{3}} \).

But \( \mu_{0} \epsilon_{0}=\frac{1}{c^{2}} \Rightarrow \mu_{0}=\frac{1}{\epsilon_{0} c^{2}}, \quad \text { so } \quad \Delta E=\frac{2 g_{d} e^{2} \hbar^{2}}{4 \pi \epsilon_{0} m_{d} m_{e} c^{2} a^{3}}=\frac{2 g_{d} \hbar^{4}}{m_{d} m_{e}^{2} c^{2} a^{4}}=\frac{3}{2} \frac{g_{d}}{g_{p}} \frac{m_{p}}{m_{d}} \Delta E_{\text {hydrogen }} \). (Eq. 7.97).

\( \Delta E=\frac{4 g_{p} \hbar^{4}}{3 m_{p} m_{e}^{2} c^{2} a^{4}}=5.88 \times 10^{-6} eV \) (7.97).

Now, \( \lambda=\frac{c}{\nu}=\frac{c h}{\Delta E}, \quad \text { so } \quad \lambda_{d}=\frac{2}{3} \frac{g_{p}}{g_{d}} \frac{m_{d}}{m_{p}} \lambda_{h}, \text { and since } m_{d}=2 m_{p}, \lambda_{d}=\frac{4}{3}\left(\frac{5.59}{1.71}\right)(21 cm )=92 cm . \)
