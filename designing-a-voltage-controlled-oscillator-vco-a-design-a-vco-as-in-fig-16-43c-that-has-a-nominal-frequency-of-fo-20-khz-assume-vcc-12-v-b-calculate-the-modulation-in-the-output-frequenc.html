Guest Question.

Designing a voltage-controlled oscillator (VCO)

(a) Design a VCO as in Fig. 16.43(c) that has a nominal frequency of \( f_{ o }=20 kHz \text {. Assume } V_{ CC }=12 V \).

(b) Calculate the modulation in the output frequencies if \( v_{ CN } \) is varied by ±10% because of the modulating signal \( v_{ cn } \).

Admin
Solution

(a) The steps used to design the VCO are as follows:
Step 1. Find the limiting values of \( V_{ CN } \text {. From Eq. (16.60), } 9 V \leq V_{ CN } \leq 12 V \).
Step 2. Choose a suitable value of \( C_{1}: \text { Let } C_{1}=0.001 \mu F \).
Step 3. Choose a value of \( R_{1} \text { between } 2 k \Omega \text { and } 20 k \Omega \text { : Let } R_{1}=10 k \Omega \).
Step 4. Find the value of \( v_{ CN } \text {. From Eq. (16.62), } v_{ CN } \) is given by

\( \frac{3}{4} V_{ CC } \leq V_{ CN } \leq V_{ CC } \) (16.60)

 

\( f_{ o } \approx \frac{2\left(V_{ CC }-v_{ CN }\right)}{R_{1} C_{1} V_{ CC }} \) (16.62)

 

\(\begin{aligned}v_{ CN } &=V_{ CC }\left(1-\frac{R_{1} C_{1} f_{ o }}{2}\right) \\&=12\left(1-\frac{10 k \Omega \times 0.001 \mu F \times 20 kHz }{2}\right)=12 \times 0.9=10.8 V\end{aligned}\) (16.63)

 

which falls within the range specified in step 1. If the calculated value falls outside of the specified range, choose a different value for \( R_{1} \text { and/or } C_{1} \text { and recalculate } v_{ CN } \).

Step 5. Find the values of \( R_{2} \text { and } R_{3} \text {. Letting } V_{ CN }=v_{ CN }=10.8 V \) from step 4, Eq. (16.59) gives

\( V_{ CN }=\frac{R_{3}}{R_{2}+R_{3}} V_{ CC } \) (16.59)

 

\(\begin{aligned}1+\frac{R_{2}}{R_{3}} &=\frac{V_{ CC }}{V_{ CN }} \\&=\frac{12}{10.8}=1.11\end{aligned}\) (16.64)

 

so \( R_{2} / R_{3}=1.11-1=0.11 \text {. Let } R_{3}=100 k \Omega \text {; } \) then

\( R_{2}=0.11 \times R_{3}=11 k \Omega \)

(b) For a 10% increase in \( v_{ CN } \),

\( v_{ CN }^{+}=V_{ CN }+v_{ cn }=1.1 \times v_{ CN }=1.1 \times 10.8=11.88 V \)

The corresponding value of output frequency can be calculated from Eq. (16.62):

\( f_{ ol } \approx \frac{2 \times(12-11.88) V }{10 k \Omega \times 0.001 \mu F \times 12 V }=2 kHz \)

For a 10% decrease in \( v_{ CN } \),

\( v_{ CN }^{-}=V_{ CN }-v_{ cn }=0.9 \times V_{ C }=0.9 \times 10.8=9.72 V \)

The output frequency is

\( f_{ o 2} \approx \frac{2 \times(12-9.72) V }{10 k \Omega \times 0.001 \mu F \times 12 V }=38 kHz \)

Thus, the change in the output frequency is

\( \Delta f_{ o }=f_{ o 2}-f_{ ol }=38 kHz -2 kHz =36 kHz \)

Using Eq. (16.62), we can find the V⁄ F transfer coefficient \( K_{ vf } \):

\(\begin{aligned}K_{ vf } &=\frac{d f_{ o }}{d v_{ CN }}=-\frac{2}{R_{1} C_{1} V_{ CC }} \\&=-\frac{2}{10 k \Omega \times 0.001 \mu F \times 12 V }=-16.67 kHz / V\end{aligned}\) (16.65)

 

Thus, for \( \Delta v_{ CN }=2 v_{ CN }=-2 \times 0.1 \times 10.8=-2.16 V ,\)

\( \Delta f_{ o }=f_{ o 2}-f_{ ol }=K_{ vf } \Delta v_{ CN }=-16.67 kHz / V \times(-2.16 V )=36.01 kHz \)

which is the same as the value obtained by calculating individual frequencies.

NOTE: If the modulating signal is a sine wave so that \( v_{ cn }=V_{ m } \sin \omega t \), then the control voltage becomes \( v_{ CN }=V_{ CN }+v_{ m } \sin \omega t \). During the positive half-cycle of the modulating signal, the control voltage will increase and the frequency \( f_{o} \) of the output voltage will decrease. However, during the negative half-cycle of the modulating signal, the control voltage will decrease and the frequency \( f_{o} \) of the output voltage will increase.
