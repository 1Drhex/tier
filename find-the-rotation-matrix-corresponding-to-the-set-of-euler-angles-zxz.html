Guest Question.

Find the rotation matrix corresponding to the set of Euler angles ZXZ

Admin
Solution

Composition of rotation matrices with respect to the current frame gives

\(R\left(\phi \right) \) = \( R_{z}\left(\varphi \right)R_{x ^{\prime} } \left(\vartheta \right) R_{z^{\prime \prime }\left(\psi \right) }\)

Using the expressions of elementary rotation matrices in (2.6) and (2.8):

\( R_{z} \left(\alpha \right) = \left[\begin{matrix} \cos \alpha & -\sin \alpha & 0 \\ \sin \alpha & \cos \alpha & 0 \\ 0 & 0 &1 \end{matrix} \right] \) (2.6)

\(R_{x} \left(\gamma \right) = \left[\begin{matrix} 1 & 0 & 0 \\ 0 & \cos \gamma & -sin \gamma \\ 0 & \sin \gamma & \cos \gamma \end{matrix}\right] \) (2.8)

\(R_{z}\left(\varphi \right) = \left[\begin{matrix} c_{\varphi } & -s_{\varphi } & 0 \\s_{\varphi } & c_{\varphi } & 0 \\ 0 &0 & 1 \end{matrix} \right]\)

 

\(R_{x ^{\prime} } \left(\vartheta \right) = \left[\begin{matrix} 1 & 0 & 0 \\0 & c_{\vartheta } & -s_{\vartheta } \\ 0 & s_{\vartheta } & c_{\vartheta } \end{matrix} \right]\)

 

\(R_{z^{\prime \prime }\left(\varphi \right) } = \left[\begin{matrix} c_{\psi } & -s_{\psi } & 0 \\s_{\psi } & c_{\psi } & 0 \\ 0 &0 & 1 \end{matrix} \right] \)

 

and taking the products gives

\(R\left(\phi \right) = \left[\begin{matrix} c_{\varphi }c_{\vartheta } – s_{\varphi }c_{\vartheta }s_{\psi } & – c_{\varphi }s_{\psi } – s_{\varphi }c_{\vartheta }c_{\psi } & s_{\varphi }s_{\vartheta } \\ s_{\varphi }c_{\psi } + c_{\varphi }c_{\vartheta } s_{\psi } & -s_{\varphi }s_{\psi } + c_{\varphi }c_{\vartheta } c_{\psi } & – c_{\varphi } s_{\vartheta } \\s_{\vartheta }s_{\psi } & s_{\vartheta }c_{\psi } & c_{\vartheta } \end{matrix}\right].\)

 

As for the inverse problem, given a rotation matrix

R =\(\left[\begin{matrix} r_{11} & r_{12} & r_{13} \\ r_{21} & r_{22} & r_{23} \\ r_{31} & r_{32} & r_{33} \end{matrix} \right],\)

the set of Euler angles ZXZ is given by

\(\varphi = A\tan 2\left(r_{13}, – r_{23} \right)\)

 

\(\vartheta = A\tan 2\left(\sqrt{r^{2} _{31} + r^{2} _{32} } , r_{33} \right)\)

 

\(\psi = A\tan 2 \left(r_{31}, r_{32} \right)\)

 

when ϑ ∈ (0, π). Otherwise, if ϑ ∈ (−π, 0) then the solution is

\(\varphi = A\tan 2\left( – r_{13}, r_{23} \right)\)

 

\(\vartheta = A\tan 2\left( -\sqrt{r^{2} _{31} + r^{2} _{32} } , r_{33} \right)\)

 

\(\psi \) = \(A\tan 2 \left( – r_{31}, – r_{32} \right).\)
