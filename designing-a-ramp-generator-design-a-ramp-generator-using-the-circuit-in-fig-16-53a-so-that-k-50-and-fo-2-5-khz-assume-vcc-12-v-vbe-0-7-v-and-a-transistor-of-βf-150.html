Guest Question.

Designing a ramp generator Design a ramp generator using the circuit in Fig. 16.53(a) so that k = 50% and \( f_{ o }=2.5 kHz \text {. Assume } V_{ CC }=12 V , V_{ BE }=0.7 V \text {, and a transistor of } \beta_{ F }=150 \).

Admin
Solution

\( k=50 \%=0.5, \text { and } T=1 / f_{ o }=1 / 2.5 kHz =400 \mu s \). The steps used to design the ramp generator are as follows:
Step 1. Find the charging time \( t_{ c } \) and the discharging time \( t_{ d } \):

\(\begin{aligned}t_{ c } &=t_{ d }=k T \\&=0.5 \times 400 \mu s =200 \mu s\end{aligned}\) (16.92)

 

Step 2. Choose a suitable value of C: Let C = 0.1 µF.
Step 3. Find the value of R. From Eq. (16.90),

\(\begin{aligned}f_{ o } &=\frac{1}{T}=\frac{3 k I_{ C }}{C V_{ CC }} \\&=\frac{3 k\left(V_{ CC }-V_{ BE }\right)}{C R V_{ CC }}\end{aligned}\) (16.90)

 

\(\begin{aligned}R &=\frac{3 k\left(V_{ CC }-V_{ BE }\right)}{V_{ CC } C f_{ o }} \\&=\frac{3 \times 0.5 \times(12-0.7) V }{12 \times 0.1 \mu F \times 2.5 kHz }=5.65 k \Omega\end{aligned}\) (16.93)

 

Step 4. Find the collector current \( I_{ C } \) of the transistor. From Eq. (16.86),

\(\begin{aligned}I_{ C } &=I_{ D }-I_{ B }=I_{2} \\&=\frac{V_{ B }}{R}=\frac{V_{ CC }-V_{ BE }}{R} \end{aligned}\) (16.86)

 

\( I_{ C }=\frac{V_{ CC }-V_{ BE }}{R}=\frac{(12-0.7) V }{5.65 k \Omega}=2 mA \)

 

Step 5. Find the current \( I_{ D } \) through the diode:

\( I_{ D }=I_{ E }=I_{ C }+I_{ B }=I_{ C } \frac{1+\beta_{ F }}{\beta_{ F }}=2 mA \times \frac{1+150}{150} \approx 2.01 mA \)
