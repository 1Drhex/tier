Guest Question.

A round cold-drawn 1018 steel rod has an 0.2 percent yield strength \( S_y= N \left(78.4, 5.90\right) \) kpsi and is to be subjected to a static axial load of \( P= N \left(50, 4.1\right) \) kip.

What value of the design factor \(\bar{n} \) corresponds to a reliability of 0.999 against yielding \( \left(z = −3.09\right) \)?
Determine the corresponding diameter of the rod.

Admin
Solution

\( C_s={5.90}/{78.4}=0.0753 \) , and

\(\sigma =\frac{P}{4} =\frac{4P}{\pi d^2} \)

Since the COV of the diameter is an order of magnitude less than the COV of the load
or strength, the diameter is treated deterministically:

\(C_\sigma =C_P=\frac{4.1}{50}=0.082\)

From Eq. (5–42),

\(\bar{n} =\frac{1+\sqrt{1-\left(1-z^2C^2_S \right)\left(1-z^2C^2_\sigma \right)} }{1-z^2C^2_S}\)

 

\(\bar{n} = \frac {1+\sqrt {1-\left[1-\left(-3.09\right)^2 \left(0.0753^2\right) \right] \left[1-\left(-3.09\right)^2 \left(0.082^2\right) \right]} } {1-\left(-3.09\right)^2 \left (0.0753^2\right)} = 1.416 \)

 

The diameter is found deterministically:

 

\(d=\sqrt{\frac{4\bar{P} }{\pi {\bar{S_y}}/{\bar{n} } } } =\sqrt{\frac{4\left(50000\right) }{{\pi \left(78400\right) }/{1.416}} } =1.072 \) in

\(S_{y}\) = N(78.4, 5.90) kpsi, P = N(50, 4.1) kip, and d = 1.072 in. Then

\(A=\frac{\pi d^2}{4} =\frac{\pi \left(1.72^2\right) }{4} =0.9026 \ in^2\)

\(\bar{\sigma } =\frac{\bar{P} }{A} =\frac{\left(50000\right) }{09026} =55400\) psi

 

\(C_P=C_\sigma =\frac{4.1}{50} =0.082\)

 

\(\hat{\sigma} _\sigma =C_\sigma \bar{\sigma } =0.082\left(55400\right) =4540\) psi

 

\(\hat{\sigma} _S=5.90\) kpsi

 

From Eq. (5–40)

 

\(z=\frac{m-\mu _m}{\hat{\sigma }_m } =\frac{0-\mu _m}{\hat{\sigma }_m } =-\frac{\mu _m}{\hat{\sigma }_m } =-\frac{\mu _S-\mu _\sigma }{\left(\hat{\sigma ^2_S}+\hat{\sigma ^2_\sigma } \right)^{{1}/{2}} } \)

 

\(z=-\frac{78.4-55.4}{\left(5.90^2+4.54^2\right)^{{1}/{2}} } =-3.09\)

From Appendix Table A–10,\(R=\Phi \left(-3.09\right) =0.999 \).
