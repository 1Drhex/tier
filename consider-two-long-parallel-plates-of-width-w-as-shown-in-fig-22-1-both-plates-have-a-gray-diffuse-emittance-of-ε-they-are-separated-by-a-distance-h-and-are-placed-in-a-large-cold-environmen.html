Guest Question.

Consider two long parallel plates of width w as shown in Fig. 22-1. Both plates have a gray, diffuse emittance of ε ; they are separated by a distance h and are placed in a large, cold environment.
The bottom plate is insulated, and it is desired to keep this plate at an isothermal temperature \(T^{*}\) across its width through radiative heating from the top plate. Determine the necessary temperature distribution \(T_{2}(x_{2}\)) to achieve this result.

Admin
Solution

Direct Solution. From equation (5.27) \(\int_{4\pi }^{}{}\widehat{s}d\Omega =\int_{0}^2\pi {}{}\int_{0}^{\pi }{}(\begin{matrix} \sin \theta &\cos \psi & \\\sin \theta & \sin \psi & \\ \cos \theta & & \end{matrix}) \sin \theta d\theta d\psi =\left\lgroup\begin{matrix} 0\\ 0 \\ 0 \end{matrix} \right\rgroup =0\) we find that \(J_{1}(x_{1})=\sigma T^{4}_{1}(x_{1})\) for an insulated surface. Thus, from equation (5.24) \(I(r,\theta ,\psi )=I_{0}^{0}+I^{0}_{1}\cos \theta +\frac{1}{\sqrt{2} }(I^{-1}_{1}-I_{1}^{1})\sin \theta \cos \psi -\frac{i}{\sqrt{2} } (I^{-1}_{1}+I_{1}^{1}) \sin \theta \sin \psi\) , with \(df_{di-di}=0\), we obtain

\(\sigma T^{4}_{1}(x_{1})=\int_{0}^{w}{} J_{2}(x_{2})dF_{d1-d2}\)

\(J_{2}(x_{2})=\epsilon \sigma T^{4}_{1}(x_{2})+(1-\epsilon )\int_{0}^{w}{} \sigma T^{4}_{1}dF_{d2-d1}\)

The necessary view factors have already been obtained in . Also, using similar nondimensionalization as in that example, i.e., \(W=w/h,\xi =x/h,\phi (x)=[T(x)/T^{*}]^{4}, J(\xi _{2})= J_{2}(x_{2})/\sigma T^{*4}\) , these equations become

\(\phi _{1}(\xi _{1})=\int_{0}^{W}{}J(\xi _{2})f(\xi _{1}-\xi _{2})d\xi _{2}\)

\(J({\xi _{2}})=\epsilon \phi _{2}(\xi _{2})+(1-\epsilon )\int_{0}^{W}{}\phi _{1}(\xi _{1}) (\xi _{1}-\xi _{2})d\xi _{1},\)

\(f(\xi _{1}-\xi _{2})=\frac{1}{2}[1+(\xi _{1}-\xi _{2})^{2}]^{-3/2} .\)

The desired result is \(\phi _{1}(\xi _{1})\) = 1 = const, for which the necessary \(\phi _{2}(\xi _{2})\) needs to be found. We will approximate the top plate’s temperature by an N-th order polynomial, or

\(\phi _{2}(\xi _{2})= \sum\limits_{n=0}^{N}{} a_{n} \xi _{2}^{n}\),

with optimum values of the parameters a„ to be determined (and, for illustrative purposes, without taking advantage of the obvious symmetry across x = w/2). We will solve the direct problem by using a simple Newton-Cotes quadrature as outlined in . Thus,

\(\phi _{1i}=W\sum\limits_{j=1}^{J}{c_{j}}J_{i}f_{ij}, i= 1,2,…, J,\)

\(J_{j}=\epsilon \sum\limits_{n=0}^{N}{}a_{n}\xi ^{m}_{j}+ (1-\epsilon )W\sum\limits_{k=1}^{J}{}c_{k}\phi _{1k}f_{jk}, j= 1,2,…,J\)

or

\(\phi _{1i}=\sum\limits_{N}^{m=0}{}a_{n}y_{in}+\sum\limits_{k=1}^{j}{z_{ik}}\phi _{1k}, i = 1,2,….,J,\) (22.20)

\(y_{in}= \epsilon W\sum\limits_{j=1}^{J}{c_{j}f_{ij}\delta ^{2}_{j}},\)

\(z_{ik}=(1-\epsilon )W^{2}c_{k}\sum\limits_{j=1}^{J}{c_{j}f_{ij}f_{kj}}\).

Inverse Problem. If we consider each of the quadrature points to be a design point where we want to achieve \(T_{1} = T^{*} (or \phi _{1}=1\) , the objective function becomes, assuming constant weights (\(\sigma _{i}=1\)),

\(S= \left|\phi _{1}-1\right|^{2}=\sum\limits_{i=1}^{J}{}(\phi _{1i}-1 )^{2}\)

The sensitivity matrix is obtained by differentiating equation (22.20) with respect to \(a_{n}\) , or

\(X_{in}=y_{in}+\sum\limits_{k=1}^{J}{z_{ik}}X_{kn}\) ,

which is solved by successive approximation. We note that the sensitivity matrix is independent of \(a_{n}\) .
Thus, the problem is linear and the \(\phi _{1i}\), are calculated from

\(\phi _{1i}=\sum\limits_{n=0}^{N}{}a_{n}X_{in}\) (22.21)

Levenberg-Marquardt Method If the Levenberg-Marquardt technique is used, updated values for p are found from equation (22.13) \(p^{k+1}=p^{k}+(X^{Tk}\cdot W\cdot X^{k}+\mu ^{k}\Omega ^{k})^{-1}\cdot X^{Tk}\cdot W\cdot (y-i^{k})\) , which requires the solution of N + 1 simultaneous, linear equations.
The iteration starts with a relatively large value for \(μ^{0}\) , such as \(μ^{0}\) = 0.001. The value for \(μ^{k}\) is decreased if the latest objective function, \(S^{k}\), has decreased (step has succeeded), or it is increased if \(S^{k}\) has grown (step failed), until convergence has been achieved. Here we have used the Numerical Recipes’ routine mrqmin which requires the arrays \(\xi _{j}, Y_{j}= 1 , \sigma _{j}=1\) as input as well as a subroutine that evaluates i(\(\phi _{1i}\)) and its derivative with respect to p(\(X_{in}\)) as a function of \(\xi\). Each call to mrqmin carries out one iteration, updating the solution vector p and recalculating the objective function S.
Conjugate Gradient Method In order to use the conjugate gradient method the \(\gamma ^{k} and \beta ^{k+1}\) coefficients need to be calculated from equations (22.17) \(\beta ^{k+1}=\frac{(X^{k}\cdot d^{k})\cdot W\cdot (i^{k}-y)}{(X^{k}\cdot d^{k})\cdot W\cdot (X^{k}\cdot d^{k})} , \beta ^{k+1}=\frac{\sum\limits_{i=1}^{I}{}\frac{I^{k}_{i}-Y_{i}}{\sigma ^{2}_{i}}\sum\limits_{j=1}^{J}{}X^{k}_{ij} d^{k}_{j} }{\sum\limits_{i=1}^{I}{}\frac{1}{\sigma ^{2}_{i}}\sum\limits_{j=1}^{J}{}(X^{k}_{ij}d^{k}_{j})^{2} }\) through (22.19) \(\left|\triangledown _{p}S^{k}\right|=\sum\limits_{j=1}^{J}{}\left\lgroup\sum\limits_{i=1}^{I}{} \frac{I_{i}^{k}-Y_{i}}{\sigma ^{2}_{i}}X^{k}_{ij} \right\rgroup ^{2}\) , i.e.,

\(\left|\triangledown _{p}S^{k}\right|^{2}=\sum\limits_{n=0}^{N}{}\left\lgroup\sum\limits_{i=1}^{J}{} (\phi ^{k}_{1i}-1)X_{in}\right\rgroup ^{2} ,\) (22.22)

\(\beta ^{k+1}=\sum\limits_{i=1}^{J}{} (\phi ^{k}_{1i}-1)\sum\limits_{n=0}^{N}{}X_{in}d^{k}_{n}/\sum\limits_{i=1}^{J}{}\sum\limits_{n=0}^{N}{} (X_{in}d^{k}_{n})^{2}\) (22.23)

The calculation proceeds as follows:
1. Since the problem is linear, the sensitivity matrix is precalculated once and for all.

2. An initial guess is made for the parameter vector (such as \(a_{n}\) = 0, all n), and the iteration counter is set to k = 0.
3. The direct solution \(\phi^{k}_{1}\) is found from equation (22.21), and the objective function \(S^{k}\) is calculated; if it meets certain stopping criteria, the iteration is terminated.
4. The gradient of \(S^{k}\) is found from equation (22.22); \(\gamma ^{k}\) is calculated by division with the previous value of \(\triangledown _{p}S\) (for the first iteration, the “old” value is set to a very large number to force \(\gamma ^{0}\) = 0). A new search direction \(d^{k}\) is set from equation (22.15) \(d^{k}=\triangledown _{p}S (p)+\gamma ^{k}d^{k-1}\).
5. The search step size is determined from equation (22.23), and the parameter vector is updated with equation (22.14) \(p^{k+1}=p^{k}-\beta ^{k}d^{k}\) . The calculation returns then to 3 above (alternatively, the step \(\beta^{k}\), or the change in the parameter vector can also be used as stopping criteria).

Sample results are shown in Fig. 22-2 for various w/h-ratios, using a fourth-order polynomial for \(T_{2}\)(N = 4), and a 40-point quadrature along the plates. If the gap between the plates is not too narrow (w/h ≤ 2) the design point of \(\phi_{1}\)= 1 = const is satisfied extremely well. For larger values of w/h undesirable oscillations in \(\phi_{1}\) become very noticeable, which is typical for ill-conditioned inverse problems.
A smoother, but less accurate solution can be obtained by aborting the iteration after partial convergence. Increasing the order of the polynomial approximating \(\phi_{2}\) , on the other hand, does not improve the smoothness of the solution.
