Guest Question.

Water is to be conveyed at a flow rate of 3.8 l/s in a commercial steel pipeline. Determine the optimum economic pipe size for the installation given that:
\(C_2 =\$0.04/(kW·hr) =\$0.04/(1 000 W·hr)\)
\(C_1 =\$700/m^{2.2}\)
t = 6 000 hr/yr a = 1/(7 yr)
F = 7.0 b = 0.01
n = 1.2 η = 75% = 0.75

Admin
Solution

In the last example, we worked with Equation 4.12 directly in order to find the optimum diameter \(D_{opt}\). The same procedure will be used here. The appropriate economic diameter selection graph will also be used in order to illustrate the method and to compare the results. We begin by obtaining properties from the appropriate tables:

water \(ρ = 1 000 kg/m^3\) \(µ = 0.89 \times 10^{-3} N·s/m^2\) [Appendix Table B.1]
commercial steel ε = 0.004 6 cm = 0.000 046 m [Table 3.1]

Next, we substitute all known parameters into Equation 4.12 to formulate the trial- and-error procedure. The volume flow rate is

\(Q = 3.8 l/s = 0.003 8 m^3/s\)

The mass flow rate is calculated as

\(\dot{m }= ρQ = 1 000(0.003 8) = 3.8\) kg/s

Equation 4.12 is

\(D_{opt} = \left[\frac{40f \dot{m}^3C_2t}{n(a + b)(1 + F) C_1ηπ^2ρ^2g_c }\right]^{\frac{1}{n+5}}\)

Substituting gives

\(D_{opt} = \left[\frac{40f(3.8)^3(0.04/1 000)(6 000)}{1.2(1/7 + 0.01)(1 + 7.0)(700)(0.75)π^2(1 000)^2}\right]^{1/6.2}\)
or \(D_{opt} = 0.070f^{0.161}\) (i)

The Reynolds number of the flow is

\(Re =\frac{ ρVD}{µg_c}=\frac{ 4ρQ}{πDµg_c}= \frac{4\dot{m}}{πDµg_c}\)

 

\(Re =\frac{ 4(3.8)}{πD(0.89 \times 10^{-3})}= \frac{5.44 \times 10^3}{D}\)

We start by assuming a diameter selected from a table of pipe sizes (such as Appendix Table D.1) or a diameter selected at random. Here we choose the latter.
1st trial: D = 4 cm; then
\(\left. \begin{matrix}Re= 5.44 \times 10^ 3/0.04=1.36\times 10^5\\ \\ \frac{ε}{D} = \frac{0.004 6}{4} = 0.001 15\end{matrix} \right\}\)f = 0.022 (Moody diagram)
Using Equation i of this example,

\(D_{opt} = 0.070(0.022)^{ 0.161}= 0.037 8\) m

2nd trial: D = 0.037 8 m; then

\(\left. \begin{matrix}Re= 5.44 \times 10^ 3/0.037 8=1.44\times 10^5\\ \\ \frac{ε}{D} = \frac{0.004 6}{3.78} = 0.001 2\end{matrix} \right\}\) f ≈ 0.022 (Moody Diagram)

which equals our assumed value. So

\(D_{opt} = 0.037 8\) m (close enough)

In order to avert trial and error, we can use Figure 4.5 to solve this problem.
We now work toward calculating ( \(f(Re)^{n+5})^{ 1/6} = ( f(Re)^{6.2})^{1/6}\) using Equation 4.14. Substituting into Equation 4.14 gives
\(( f(Re)^{n+5} )^{1/6} = \left[\frac{ 128}{5π^3g_c^4}\frac{\dot m^2}{µ^5} \left\lgroup \frac{4\dot{m}}{πµg_c}\right\rgroup^n \left\lgroup\frac{ n(a + b)(1 + F)C_1ηρ^2 }{C_2t}\right\rgroup \right]^{1/6}\) (4.14)
\(( f(Re)^{6.2})^ {1/6}= \left[\frac{ 128}{5π^3{(1)}^4} \frac{3.8^2}{(0.89 \times 10^{-3})^5} \left\lgroup \frac{4(3.8)}{π(0.89\times 10^{-3})(1)} \right\rgroup ^{1.2}\times \left\lgroup\frac{1.2(1/7+0.01)(1+7)+(700)(0.75)(1 000)^2}{(0.04/1 000)(6 000)}\right\rgroup \right]^{1/6}\)

Solving,

\(( f(Re)^{6.2})^{ 1/6} = 1.13 \times 10^5\)

Also, the roughness number is calculated to be

\(Ro = \frac{πεµg_c}{4\dot{m}}= \frac {π(0.000 046)(0.89 \times 10^{-3})(1)}{4(3.8)} = 8.46 \times 10^{-9}\)

With these values,

\(\left. \begin{matrix}( f(Re)^{6.2})^{ 1/6} = 1.13 \times 10^5\\ \\ Ro = 8.46 \times 10^{-9} \end{matrix} \right\}\) f ≈ 0.022 (Figure 4.5)

Therefore,

\(f(Re)^{6.2})^{ 1/6} = ( 0.022(Re)^{6.2})^ {1/6}= 1.13 \times 10^5\)

and the Reynolds number is calculated to be

\(Re = \left\lgroup\frac{(1.13 \times 10^5 )^6}{0.022}\right\rgroup^{1/6.2}= 1.44 \times 10^5\)

From the definition of Reynolds number,

\(Re =\frac{ 4\dot{m}}{πDµg_c}\)

we write

\(D =\frac{ 4\dot{m}}{πReµg_c}= \frac{4(3.8)}{π(1.44 \times 10^5)(0.89 \times 10^{-3})(1)}\)
or \(D = 3.78 \times 10^{-2 }m = 3.78\) cm

Using Equation 4.12 for \(D_{opt}\), we obtained 3.78 cm. Any discrepancy that might have resulted in these two results is due to roundoff errors and to errors in reading the graphs. Use of the dimensionless graphs has now been illustrated
