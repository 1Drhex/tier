Guest Question.

Evaluating Disney’s operating return on assets

Given the following financial information for the Walt Disney Company in 2014 (expressed in $ millions), evaluate the firm’s operating return on its assets.

 

Accounts receivable $ 8,319
Inventories 1,574
Sales 48,813
Credit sales 32,542
Cost of goods sold 26,420
Operating profits 11,400
Net fixed assets 23,332
Total assets 84,186

 

 

The industry norms are as follows:

 

Operating return on assets 9.24%
Operating profit margin 20.54%
Total asset turnover 0.45X
Accounts receivable turnover 5.80X
Inventory turnover 18.30X
Fixed-assets turnover 1.05X

 

Admin
Solution

step 1: Formulate a Decision strategy

The operating return on assets is determined by two factors: managing operations and managing assets. Recall equation (4-8a):

 

\(\begin{aligned}&\text { Operating return } \\&\text { on assets }\end{aligned}=\frac{\text { operating profits }}{\text { sales }} \times \frac{\text { sales }}{\text { total assets }}\) (Equation 4-8 a)

 

Substituting for the operating profit margin and total asset turnover (equation 4-8b):

 

\(\begin{gathered}\text { Operating } \\\text { return on assets }\end{gathered}=\frac{\text { operating profits }}{\text { sales }} \times \frac{\text { sales }}{\text { total assets }}\)

 

We can further analyze the total asset turnover ratio by looking at the efficiency of various components of total assets, including accounts receivable turnover, inventory turnover, and fixed-assets turnover.

 

step 2: Crunch the numbers

\(\begin{array}{ccc} & \text { Disney } & \text { Industry } \\\text { Operating return on assets } \frac{\$ 11,400 M}{\$ 84,186 M} & 13.54 \% & 9.24 \% \\\text { Operating profit margin } \frac{\$ 11,400 M}{\$ 48,813 M} & 23.35 \% & 20.54 \% \\\text { Total asset turnover } \frac{\$ 48,813 M}{\$ 84,186 M} & 0.58 X & 0.45 X \\\text { Accounts receivable turnover } \frac{\$ 32,542 M}{\$ 8,319 M} & 3.91 X & 5.80 X \\\text { Inventory turnover } \frac{\$ 26,420 M}{\$ 1,574 M} & 16.79 X & 18.30 X \\\text { Fixed-assets turnover } \frac{\$ 48,813 M}{\$ 23,332 M} & 2.09 X & 1.05 X\end{array}\)

 

step 3: analyze Your results

Disney generated a higher return on its assets than the average firm in the industry, 13.54 percent compared to 9.24 percent for the industry. Disney provided a higher operating return on its assets by (1) managing its operations better, achieving a higher operating profit margin (23.35 percent compared to 20.54 percent); and (2) making better use of its assets, as indicated by a higher total asset turnover (0.58X versus 0.45X). However, the higher total asset turnover is caused entirely by a much higher fixed asset turnover (2.09X compared to 1.05X), which more than offsets the lower accounts receivable turnover (3.91X as compared to 5.80X) and the less efficient management of inventories, as reflected by the lower inventory turnover (16.79X versus 18.30X).
