Guest Question.

A simply supported beam (B-2) supports a uniformly distributed load and is braced at six foot intervals (see Figure 8.48) in addition to a composite slab that is secured to the compression flange of the beam via shear studs. Assume that the beam is made of ASTM A992 grade with \(F_{ y }\) = 50 ksi.

Select the lightest W shape to safely support the following loads:

DL = 100 psf; LL = 75 psf

Beams are spaced at 12 feet o.c.

Admin
Solution

\(ω_{ DL }\) = 100 #/ft.² × 12′ = 1,200 #/ft.

\(ω_{ LL }\) = 75 #/ft.² × 12′ = 900 #/ft.

Because this example represents a floor load condition, the two factored load considerations that apply are

1.4D = 1.4 × 1,200 #/ft. = 1680 #/ft. (for DL only)

1.2D + 1.6L = 1.2(1,200 #/ft.) = 1.6(900 #/ft.)

= 2,880 #/ft.

The factored load of 2,880 #/ft. governs the design for the floor beams.

\(M_{ u }=\frac{ω L^{2}}{8}=\frac{(2.88 k – ft .)\left(24^{\prime}\right)^{2}}{8}=207.4 k – ft.\)

 

\(V_{ u }=\frac{ω L}{2}=\frac{(2.88 k – ft .)\left(24^{\prime}\right)}{2}=34.6 k\)

 

\(Z_{ x }=\frac{12 M_{ u }}{\phi_{ b } F_{ y }}=\frac{12(207.4 k \text { -ft. })}{(0.90)\left(50 k / in .^{2}\right)}=55.3 in .{ }^{3}\)

From in the Appendix, select the lightest trial size.

Try W18 × 35.

\(Z _{ x } \) = 66.5 in.³, d = 17.7 in., h = 15.5 in.

\(\left.b_{ f }=6.00 \text { in., } t_{ f }=0.425 \text { in., } t_{ w }=0.30 \text { in. }\right)\)

 

\(M_{ p }=F_{ y } Z_{ x }=\left(50 k -in.^{2}\right)\left(\frac{66.5 in .^{3}}{12 in . / ft }\right)=277 k – ft \)

 

\(\phi M_{ p }\)=(0.90)(277 k-ft.) = 249 k-ft.

This value can also be found directly in in the Appendix.

Next, check the compactness of the beam section.

\(\frac{b_{ f }}{2 t_{ f }}=\frac{6.00^{\prime \prime}}{2\left(0.425^{\prime \prime}\right)}=7.06<9.19 \) (the flange is compact)

\(\frac{h}{t_{ w }}=\frac{15.5^{\prime \prime}}{0.300^{\prime \prime}}=51.7<90.5\) (the web is compact)

Shear check:

\(\frac{h}{t_{ w }} \leq \frac{418}{\sqrt{F_{ y }}}\)

then use

\(V_{ u } \leq \phi_{ v } V_{ n }=(.90)\left(.6 F_{ y } t_{ w } d\right)\)

 

\(\frac{h}{t_{ w }}=\frac{15.5^{\prime \prime}}{0.300^{\prime \prime}}=51.7<\frac{418}{\sqrt{F_{ y }}}=\frac{418}{\sqrt{50}}=59\)

Therefore,

\(\phi_{ v } V_{ n }\)=(0.90)(0.6)(50)(0.30 × 17.7″) = 143.4 k

\(V_{ u }=34.6 k <\phi_{ v } V_{ n }=143.4 k\)

 

\(\therefore\) the beam is OK in shear

The W18 × 35 is adequate and the lightest section to use as a floor beam.
