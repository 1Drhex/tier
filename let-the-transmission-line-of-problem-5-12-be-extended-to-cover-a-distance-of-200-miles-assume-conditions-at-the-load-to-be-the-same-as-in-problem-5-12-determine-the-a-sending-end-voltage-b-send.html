Guest Question.

Let the transmission line of be extended to cover a distance of 200 miles. Assume conditions at the load to be the same as in . Determine the (a) sending-end voltage, (b) sending-end current, (c) sending-end real and reactive powers, and (d) percent voltage regulation.

Admin
Solution

The long line π-equivalent circuit is shown below:

\(\begin{aligned}&\bar{z}=(0.1826+j 0.784) \Omega / \text { mi per phase } \\&\bar{y}=\frac{1}{x_{c} \angle-90^{\circ}}=\frac{1}{185.5 \times 10^{3} \angle-90^{\circ}}=5.391 \times 10^{-6} \angle 90^{\circ} S / mi \text { per phase } \\&\bar{\gamma}=\sqrt{\bar{y} \bar{z}} ; \bar{Z}=\bar{z} l=160.99 \angle 76.89^{\circ} \Omega ; \bar{Y}=\bar{y} l=1.078 \times 10^{-3} \angle 90^{\circ} S \\&\bar{F}_{1}=(\sinh \bar{\gamma} l) / \bar{\gamma} l=0.972 \angle 0.37^{\circ} ; \bar{F}_{2}=\frac{\tanh (\bar{\gamma} l / 2)}{\bar{\gamma} l / 2}=1.0144 \angle-0.19^{\circ}\end{aligned}\)

 

\(\begin{aligned}&\therefore \bar{Z}^{\prime}=\bar{Z} \frac{\sinh \bar{\gamma} l}{\bar{\gamma} l}=156.48 \angle 77.26^{\circ} \Omega \\&\frac{\bar{Y}^{\prime}}{2}=\frac{\bar{Y}}{2}\left(\frac{\tanh (\bar{\gamma} l / 2)}{\bar{\gamma} l / 2}\right)=0.5476 \times 10^{-3} \angle 89.81^{\circ} S\end{aligned}\)

 

\(\begin{aligned}\bar{I}_{Z^{\prime}}=\bar{I}_{R}+\bar{V}_{R} \frac{\bar{Y}^{\prime}}{2} &=502 \angle 0^{\circ}+\left(132,800 \angle 0^{\circ}\right)\left(0.5476 \times 10^{-3} \angle 89.81^{\circ}\right) \\&=507.5 \angle 8.24^{\circ} A\end{aligned}\)

 

\(\begin{aligned}\bar{V}_{S}=\bar{V}_{R}+\bar{I}_{Z}, Z^{\prime} &=132,800 \angle 0^{\circ}+507.5 \angle 8.24^{\circ}\left(156.48 \angle 77.26^{\circ}\right) \\&=160,835 \angle 29.45^{\circ} V\end{aligned}\)

 

(a) Sending end line to line voltage magnitude \(=\sqrt{3} 160.835=278.6 kV\)

(b)

\(\begin{aligned}\bar{I}_{S}=\bar{I}_{Z^{\prime}}+\bar{V}_{S}\left(\frac{\bar{Y}^{\prime}}{2}\right) &=507.5 \angle 8.24^{\circ}+160.835(0.5476) \angle 29.45+89.81^{\circ} \\&=482.93 \angle 18.04^{\circ} A ; \quad I_{S}=482.93 A\end{aligned}\)

(c)

\(\begin{aligned}\bar{S}_{S(3 \phi)}=3 \bar{V}_{S} \bar{I}_{S}^{*}=& 3(160.835)(0.48293) \angle 29.45^{\circ}-18.04^{\circ} \\&=228.41 MW +j 46.1 M var\end{aligned}\)

(d) Percent voltage regulation \(=\frac{160.835-132.8}{132.8} \times 100=21.1 \%\)
