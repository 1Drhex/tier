Guest Question.

Designing a common-emitter BJT amplifier

(a) Design a CE amplifier as shown in Fig. 8.29 to give a voltage gain of \( \left|A_{ vo }\right|=v_{ o } / v_{ b } \geq 20 \). Use transistor Q2N2222, for which minimum\( \beta_{ F }=100, \text { nominal } \beta_{ F }=173, I_{ S }=3.295 \times 10^{-14} A , \text { and } V_{ A }=100 V \text {. } \) The operating DC collector current is to be set at \( I_{ C }=10 mA \). The DC power supply is \( V_{ CC }=15 V \text {. Assume } V_{ BE }=0.7 V\).

(b) Use PSpice/SPICE to verify your results in part (a).

Admin
Solution

(a) Step 1. Design the biasing circuit. The results of give \( R_{ C }=500 \Omega, R_{ E }=495 \Omega, R_{1}=12.1 k \Omega , \text { and } R_{2}=8.52 k \Omega \text {. } \)

 

Step 2. Find the small-signal parameters of the transistor. The results of give \( r_{\pi}=258 \Omega, g_{ m }= 387.6 mA / V , \beta_{ F }=100 \text {, and } r_{ o }=20 k \Omega \) (which can be ignored for hand calculations).

Step 3. Find the values of \( C_{1}, C_{2}, C_{ E }, \text { and } R_{ E 1} \). Let us choose \( C_{1}=C_{2}=C_{ E }=10 \mu F \). The worst-case maximum possible gain that we can obtain from transistor Q2N2222 operating at \( I_{ C }=10 mA \) can be found from Eq. (8.93):

 

\( A_{ vo ( max )}=-\frac{g_{ m } r_{\pi} R_{ C }}{r_{\pi}}=-g_{ m } R_{ C }=-\frac{\beta_{ f } R_{ C }}{r_{\pi}} \) (8.93)

 

\( \left|A_{ vo (\max )}\right|=\frac{\beta_{ f } R_{ C }}{r_{\pi}}=\frac{100 \times 500}{258}=193.8 V / V \)

 

The desired gain is less than the maximum possible value, so we can proceed with the design. Otherwise, we would need to choose another transistor with a higher value of \( \beta_{ f } \). The value of unbypassed emitter resistance \( R_{ E 1} \) in Fig. 8.29 can be found from Eq. (8.92). That is,

 

\( A_{ vo }=\frac{v_{ o }}{v_{ b }}=\frac{-g_{ m } r_{\pi} R_{ C }}{r_{\pi}+\left(1+g_{ m } r_{\pi}\right) R_{ E 1}}=\frac{-\beta_{ f } R_{ C }}{r_{\pi}+\left(1+\beta_{ f }\right) R_{ E 1}} \) (8.92)

 

\( r_{\pi}+\left(1+\beta_{ f }\right) R_{ E 1}=\frac{\beta_{ f } R_{ C }}{\left|A_{ vo }\right|}\) (8.94)

 

which, for \( \left|A_{ vo }\right|=20, \beta_{ f }=100, R_{ C }=500 \Omega, \text { and } r_{\pi}=258 \Omega, \text { gives } R_{ E 1}=22.2 \Omega \text { and } R_{ E 2}= R_{ E }-R_{ E 1}=495-22.2=472.8 \Omega .\)

 

(b) The transistor circuit for PSpice simulation is shown in Fig. 8.30. Note that a signal source resistance \( R_{ s }=0.1 \Omega \)is connected at the input side to make the PSpice circuit more general.

The results of .OP analysis obtained from the output file follow. (The values obtained from hand calculations are shown in parentheses.)

\(\begin{matrix} \text{IB} & 8.91\text{E-05} & I_B=89.1 \mu A & (100 \mu A) \\ \text{IC} & 9.15\text{E-03} & I_C=9.1 m A & (10 m A) \\ \text{VBE} & 6.81\text{E-01} & V_{BE}=0.681V & (0.7 V) \\ \text{VBC} & -5.18\text{E+00} & V_{BC}=5.15 V & \\ \text{VCE} & 5.86\text{E+00} & V_{CE}=5.86 V & (5 V) \\ \text{GM} & 3.54\text{E-01} & g_{m}=0.354 A/V & (0.3876 A/V) \\ \text{RPI} & 2.90\text{E+02} & r_{\pi }= 280 \Omega & (258 \Omega ) \\ \text{RO} & 2.24\text{E+04} & r_{o}= 22.4 k\Omega & (20 \Omega ) \end{matrix} \)

 

The PSpice plots of the base voltage \( v_{ B }= V ( C 1: 2) \), the collector voltage \( v_{ C }= V ( RC : 1) \), the input voltage \( v_{ s }= V ( vs :+) \), and the load voltage \( v_{ o }= V ( RL : 2) \) are shown in Fig. 8.31. Notice that \( v_{ B } \text { and } v_{ C } \) have a DC value with an AC signal superimposed on them such that \( v_{ B }=V_{ B }+v_{ s } \text { and } v_{ C }=V_{ C }+v_{ o } \). Capacitor \( C _{1} \text { superimposes } v_{ s } \) on \(V_{ B } \text {, whereas capacitor } C _{2}\) separates the amplified AC voltage \( \text { (i.e., the output voltage } v_{ o } \text { ) from } v_{ C } \). The voltage gain \(v_{ o } / v_{ s }\) is 16.77, which is less than the desired value of 20. Thus, the design calculation should be repeated until the desired gain is obtained. We might try reducing the value of \(R_{ E 1}\) and increasing the value of \( R_{ E 2}\) by the same amount so that the biasing point remains fixed. Even if we modify the design to satisfy the specifications, the results can be expected to differ from those that would be obtained in the laboratory, although not significantly due to the variable value of \(\beta_{ F }\).
