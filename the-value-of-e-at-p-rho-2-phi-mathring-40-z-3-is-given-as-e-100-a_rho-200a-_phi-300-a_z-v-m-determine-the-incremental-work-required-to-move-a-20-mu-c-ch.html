Guest Question.

The value of E at P\(( \rho =2, \phi =\mathring { 40 } , z = 3)\) is given as E = \(100 { a}_{\rho} - 200{a }_{\phi } + 300{ a}_{z }\). V/m. Determine the incremental work required to move a \(20-\mu C\) charge a distance of \(6 \mu m\) in the direction of : (a) \({ a}_{\rho}\); (b) \({a }_{\phi }\); (c) \({ a}_{z }\); (d) E; (e) \(G = 2{a }_{x } - 3{a }_{y } + 4{a }_{z }\).

Admin
Solution

a) The incremental work is given by dW = -q E . dL. where in this case, dL = \({ d }_{ \rho }{ a }_{ \rho }=6\times { 10 }^{ -6 }\). Thus
dW = \(-(20 \times { 10 }^{ -6 } C)(100 V/m)(6 \times { 10 }^{ -6 } m) = -12\times { 10 }^{ -9 } J = -12nJ\)
b) In this case dL = \(2{ d }_{ \phi }{ a }_{ \phi } = 6\times { 10 }^{ -6 }{ a }_{ \phi }\) , and so
dW = \(– (20 \times { 10 }^{ -6 })(-200)(6 \times { 10 }^{ -6 } ) = 2.4 \times { 10 }^{ -8 } J \)= 24 nJ
c) Here , dL = \({ d }_{z }{ a }_{ z } = 6\times { 10 }^{ -6 }{ a }_{ z }\) , and so
dW = \(– (20 \times { 10 }^{ -6 })(300)(6 \times { 10 }^{ -6 } ) = -3.6 \times { 10 }^{ -8 } J \)= -36 nJ
d) Here, dL = \(6 \times { 10 }^{ -6 } { a }_{ E }\), where
\({ a }_{ E } = \frac { 100{ a }_{ \rho }-200{ a }_{ \phi }+300{ a }_{ z } }{ { \left[ { 100 }^{ 2 }+{ 200 }^{ 2 }+{ 300 }^{ 2 } \right] }^{ 1/2 } } = 0.267 { a}_{\rho} – 0.535{a }_{\phi } + 0.802{ a}_{z }\)
Thus
dW = – \((20 \times { 10 }^{ -6 } )\left[ 100{ a }_{ \rho }-200{ a }_{ \phi }+300{ a }_{ z } \right] .\left[ 0.267{ a }_{ \rho }-0.535{ a }_{ \phi }+0.802{ a }_{ z } \right](6 \times { 10 }^{ -6 } ) \) = -44.9 nJ
e) In this case ,dL = \(6 \times { 10 }^{ -6 }{ a }_{ G }\) , where
\({ a }_{ G } = \frac { 2{ a }_{ x }-3{ a }_{ y }+4{ a }_{ z } }{ { \left[ { 2 }^{ 2 }+{ 3 }^{ 2 }+{ 4 }^{ 2 } \right] }^{ 1/2 } } = 0.371{ a }_{ x }- 0.557{ a }_{ y }+ 0.743{ a }_{ z }\)
so now
d w = – \((20 \times { 10 }^{ -6 } )\left[ 100{ a }_{ \rho }-200{ a }_{ \phi }+300{ a }_{ z } \right] . \left[ 0.371{ a }_{ x }-0.557{ a }_{ y }+0.743{ a }_{ z } \right] (6 \times { 10 }^{ -6 } ) = -(20 \times { 10 }^{ -6 } )\left[ 37.1({ a }_{ \rho }{ . }{ a }_{ x })-55.7({ a }_{ \rho }{ . }{ a }_{ y })-74.2({ a }_{ \phi }{ . }{ a }_{ z })+111.4({ a }_{ \phi }{ . }{ a }_{ y })+222.9 \right] (6 \times { 10 }^{ -6 } )\)
where, at P, \(({ a }_{ \rho }{ . }{ a }_{ x }) = ({ a }_{ \phi }{ . }{ a }_{ y }) = \cos { (\mathring { 40 } ) }= 0.766, ({ a }_{ \rho }{ . }{ a }_{ y }) = \sin { (\mathring { 40 } ) } = 0.643, and ({ a }_{ \phi }{ . }{ a }_{ x }) = – \sin { (\mathring { 40 } ) }\) = -0.643. Substituting these results in
dW = \(-(20 \times { 10 }^{ -6 } ) \left[ 28.4-35.8+47.7+85.3+222.9 \right] (6 \times { 10 }^{ -6 } ) = -41.8 nJ\)
