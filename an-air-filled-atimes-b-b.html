Guest Question.

An air-filled \(a\times b (b <a < 2b) \) rectangular waveguide is to be constructed to operate at \(3 (GHz)\) in the dominant mode. We desire the operating frequency to be at least \(20\)%, higher than the cutoff frequency of the dominant mode and also at least \(20\)% below the cutoff frequency of the next higher order mode.

\(a)\) Give a typical design for the dimensions \(a\) and \(b\).

\(b)\) Calculate for your design \(\beta ,{ u }_{ p },{ \lambda }_{ g }\), and the wave impedance at the operating frequency.

Admin
Solution
\(f=3\times { 10 }^{ 9 }\left( Hz \right) ,\quad \lambda =c/f=0.1\left( m \right)\)

 

Let \(a=kb,\quad 1<k<2,\quad { \left( { f }_{ c } \right) }_{ mn }=\frac { 3\times { 10 }^{ 8 } }{ 2a } \sqrt { { m }^{ 2 }+{ k }^{ 2 }{ n }^{ 2 } } \)

 

\(a)\) \({ \left( { f }_{ c } \right) }_{ 10 }=\frac { 1.5\times { 10 }^{ 8 } }{ a } \) for the dominant \({ TE }_{ 10 }\) mode

 

\(f>1.2{ \left( { f }_{ c } \right) }_{ 10 }:\quad a>0.06\left( m \right)\)

 

The next higher-order mode is \({ TE }_{ 10 }\) with \({ \left( { f }_{ c } \right) }_{ 10 }=\frac { 1.5\times { 10 }^{ 8 } }{ b } \)

 

For \(f<0.8{ \left( { f }_{ c } \right) }_{ 10 }:\quad b<0.04\left( m \right) \)

 

We choose \(a=6.5(cm) and b=3.5(cm)\)

 

\(b)\) \({ u }_{ p }=\frac { c }{ \sqrt { 1-{ \left( \lambda /2a \right) }^{ 2 } } } =4.70\times { 10 }^{ 8 }\left( m/s \right) \)

 

\({ \lambda }_{ g }=\frac { \lambda }{ \sqrt { 1-{ \left( \lambda /2a \right) }^{ 2 } } } =0.157\left( m \right) =15.7\left( cm \right)\)

 

\(\beta =\frac { 2\pi }{ { \lambda }_{ g } } =40.1\left( rad/m \right)\)

 

\({ \left( { Z }_{ TE } \right) }_{ 10 }=\frac { { \eta }_{ 0 } }{ \sqrt { 1-{ \left( \lambda /2a \right) }^{ 2 } } } =590\left( \Omega \right)\)
