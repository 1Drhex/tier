Guest Question.

In Fig. 5-3, let \(v_{s}=20 V , R_{s}=10 \Omega, R_{i}=990 \Omega, k=5, \text { and } R_{o}=3 \Omega\). Find (a) the Thevenin equivalent of the circuit seen by R1 and (b) v2 and the power dissipated in R1 for R1 = 0.5, 1, 3, 5, 10, 100, and 1000 Ω.

Admin
Solution

(a) The open-circuit voltage and short-circuit current at A–B terminal are \(v_{\text {o.c. }}=5 v_{1} \text { and } i_{\text {s.c. }}=5 v_{1} / 3\), respectively.

We find \(v_{1} \text { by dividing } v_{s} \text { between } R_{s} \text { and } R_{i} \text {. }\). Thus,

 

\(v_{1}=\frac{R_{i}}{R_{s}+R_{i}} v_{s}=\frac{990}{10+990}(20)=19.8 V\)

Therefore,

\(v_{\text {o.c. }}=5(19.8)=99 V \quad v_{ Th }=v_{\text {o.c. }}=99 V\)

 

\(i_{\text {s.c. }}=99 / 3=33 A \quad R_{ Th }=v_{\text {o.c. }} / i_{ s.c .}=3 \Omega\)

 

The Thevenin equivalent is shown in Fig. 5-32.

(b) With the load Rl connected, we have

\(v_{2}=\frac{R_{l}}{R_{l}+R_{ Th }} v_{ Th }=\frac{99 R_{l}}{R_{l}+3} \quad \text { and } \quad p=\frac{v_{2}^{2}}{R_{l}}\)

Table 5-3 shows the voltage across the load and the power dissipated in it for the given seven values of Rl. The load voltage is at its maximum when Rl = ∞. However, power delivered to Rl = ∞ is zero. Power delivered to Rl is maximum at Rl = 3 Ω, which is equal to the output resistance of the amplifier.
