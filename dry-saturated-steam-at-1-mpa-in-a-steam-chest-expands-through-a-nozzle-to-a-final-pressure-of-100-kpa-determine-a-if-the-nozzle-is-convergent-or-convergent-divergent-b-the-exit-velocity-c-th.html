Guest Question.

Dry, saturated steam at 1 MPa in a steam chest expands through a nozzle to a final pressure of 100 kPa. Determine (a) if the nozzle is convergent or convergent-divergent, (b) the exit velocity, (c) the dryness fraction at the exit and (c) the exit to throat area ratio. Assume the expansion to be isentropic throughout.

Admin
Solution

We have \(P_{0} = 1 MPa\) and we can get \(v_{0} = 0.19436m^{3} / kg\) from the steam table.

(a) Since the steam is initially dry and saturated, \(n = 1.135\) from Eqn. 9.4. From Eqn. 9.10, we have \(P^{\ast }=0.58\) \(P_{0} = 580 kPa.\)Since the flow is expanded to a final pressure of 100 kPa in the nozzle, it is clear that the nozzle is convergent-divergent.

\(n= \begin{cases} 1.3 & \quad if superheated \\ 1.035+ 0.1x_{1} & saturated mixture \end{cases}\) (9.4)

 

\(\frac{p^{\ast } }{P_{0} } =\left(\frac{2}{n+ 1} \right) ^{n/ \left(n-1\right) }\) (9.10)

(b) We have from Eqn. 9.7

\(h_{1} -h_{2} =\frac{n}{n-1} P_{1} v_{1} \left[1-\left(\frac{P_{2} }{P_{1} } \right) ^{\left(n-1/ n\right) } \right]\) (9.7)

 

\(h_{0} -h_{e} =\frac{n}{n-1} P_{0} v_{0} \left[1-\left(\frac{P_{e} }{P_{0} } \right) ^{\left(n-1/ n\right) } \right]\)

where the subscript e denotes the exit. Upon substituting numerical values, we get

\(h_{0} -h_{e} = 391.48 kJ/kg .\)

Therefore

\(V_{e} =\sqrt{2\left(h_{0}- h_{e} \right) } = 885m/s .\)

(c) From Eqn. 9.3, we can get

\(P_{v^{n} } = constant\) (9.3)

 

\(v_{e} =v_{0} \left(\frac{P_{e} }{P_{0} } \right) ^{-1/ n} =1.478m^{3}/ kg\)

Using steam tables, the dryness fraction at the exit can now be calculated as 0.87.

(d) From Eqn. 9.3, we can get

\(P_{v^{n} } = constant\) (9.3)

 

\(v^{\ast } =v_{0} \left(\frac{P^{\ast } }{P_{0} } \right) ^{-1/ n} =0.3141m^{3}/ kg\)

and

\(V^{\ast } =\sqrt{nP^{\ast }v^{ \ast } } = 454.7m/s .\)

Since

\(\dot{m} =\frac{A^{\ast }V^{\ast } }{v^{\ast } } =\frac{A_{e}V_{e} }{v_{e} }\)

we can obtain \(A_{e} / A^{\ast } = 2.42.\)

An alternative solution method is to use the Mollier diagram or the steam table. From the steam table, for dry saturated vapor at 1 MPa, we can get \(h_{0} = 2778.1 kJ/kg\) and \(s_{0}= 6.5865 kJ/kg.K.\) Since the expansion process is isentropic, we have at the exit, \(s_{e}= 6.5865 kJ/kg.K\) and \(P_{e}= 100 kPa.\) From steam tables, we can get \(x_{e}= 0.872\) and thus \(h_{e}= 2387.3 kJ/kg\) and \(v_{e}= 1.4773 m^{3} / kg.\) Therefore

\(V_{e} =\sqrt{2\left(h_{0}-h_{e} \right) } = 884m/s\)
