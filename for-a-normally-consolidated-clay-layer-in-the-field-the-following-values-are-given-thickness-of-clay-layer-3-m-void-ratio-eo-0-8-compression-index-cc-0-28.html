Guest Question.

For a normally consolidated clay layer in the field, the following values are given:
\(\bullet\) Thickness of clay layer 3 m
\(\bullet\) Void ratio (\(\textbf{e}_{o}\)) = 0.8
\(\bullet\) Compression index (\(\textbf{C}_{c}\)) = 0.28
\(\bullet\) Average effective pressure on the clay layer (\(\pmb{\sigma} ^{\prime}_{o}\)) = 130 kN/\(\textbf{m}^{\textbf{2}}\)
\(\bullet\) \(\pmb{\Delta}\pmb{\sigma} ^{\prime}\) = 50 kN/\(\textbf{m}^{\textbf{2}}\)
\(\bullet\) Secondary compression index (\(\textbf{C}_{\pmb{\alpha}}\)) = 0.02

What is the total consolidation settlement of the clay layer five years after the completion of primary consolidation settlement? (Note: Time for completion of primary settlement = 1.5 years.)

Admin
Solution

From Eq. (7.27), \(C^{\prime}_{\alpha} = \frac{ C_{\alpha}}{1 + e_{p}}\\ C^{j}_{\alpha} = \frac{ C_{\alpha}}{1 + e_{p}}\)

The value of \(e_{p}\) can be calculated as

\(e_{p} = e_{o} – \Delta e_{primary}\)

Combining Eqs. (7.12)

\(S_{p} = H\frac{\Delta e}{1 + e_{0}}\)

and (7.13),

\(\Delta e=C_{c}\left[\log \left(\sigma_{o}^{\prime}+\Delta \sigma^{\prime}\right)-\log \sigma_{o}^{\prime}\right]\)

we find that

\(\Delta e = C_{c} \log\left(\frac{\sigma^{\prime}_{o} + \Delta\sigma^{\prime}}{\sigma^{\prime}_{o} } \right) = 0.28 \log \left(\frac{130 + 50}{130}\right) \\ =0.04\)

Primary consolidation,\(S_{p} = \frac{\Delta eH}{1 + e_{o}} = \frac{(0.04)(3)}{1 + 0.8} = 0.067\,m\)

It is given that \(e_{o} = 0.8,\) and thus,

\(e_{p} = 0.8 – 0.04 = 0.76\)

Hence,

\(C^{\prime}_{\alpha} = \frac{0.02}{1 + 0.76} = 0.011\)

From Eq. (7.26),

\(S_{s} = C^{\prime}_{\alpha}H\log\left(\frac{t_{2}}{t_{1}}\right) = (0.011)(3)\log\left(\frac{5}{1.5}\right) \approx 0.017\,m\)

Total consolidation settlement = primary consolidation (\(S_{p}\)) + secondary settlement(\(S_{s}\)). So

Total consolidation settlement = \(0.067 + 0.017 = 0.084\,m \approx 84\,mm\)
