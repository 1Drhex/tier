Guest Question.

Select an ASTM A992 W-shape with a 14-in. nominal depth to carry nominal forces of 29 kips from dead load and 87 kips from live load in axial tension, as well as the following nominal moments:
\(\begin{array}{ll}M_{x D}=32.0 \text { kip-ft } & M_{x L}=96.0 \text { kip-ft } \\M_{y D}=11.3 \text { kip-ft } & M_{y L}=33.8 \text { kip-ft }\end{array}\)
The unbraced length is 30 ft and the ends are pinned. Assume the connections are made with no holes.

Admin
Solution

Material Properties:

ASTM A992 \(F_{y}=50 ksi \quad F_{u}=65 ksi\)

Manual
Table 2-3

Calculate the required strength

LRFD ASD
\(\begin{aligned}P_{u} &=1.2(29.0 kips )+1.6(87.0 kips ) \\&=174 kips \\M_{u x} &=1.2(32.0 kip – ft )+1.6(96.0 kip – ft ) \\&=192 kip – ft \\M_{u y} &=1.2(11.3 kip – ft )+1.6(33.8 kip – ft ) \\&=67.6 kip – ft\end{aligned}\) \(\begin{aligned}P_{a} &=29.0 kips +87.0 kips \\&=116 kips \\M_{a y} &=32.0 kip – ft +96 kip – ft \\&=128 kip – ft \\M_{a y} &=11.3 kip – ft +33.8 \text { kip-ft } \\&=45.1 kip – ft\end{aligned}\)

Verify that a W14×82 has sufficient available strength

Geometric Properties:

W14×82

\(A=24.0 \text { in. }^{2} \quad S_{x}=123 \text { in. }^{3} \quad Z_{x}=139 \text { in }^{3} \quad S_{y}=29.3 in ^{3}\)

 

\(Z_{y}=44.8 \text { in. }{ }^{3} \quad I_{y}=148 \text { in. }{ }^{4} \quad L_{p}=8.76 ft \quad L_{r}=33.1 ft\)

Manual
Table 1-1
Table 3-2

Calculate the nominal gross tensile strength

\(P_{n}=F_{y} A_{g}=(50 ksi )\left(24.0 \text { in. }^{2}\right)=1200 \text { kips }\)

Eqn. D2-1

Note that for a member with holes, the rupture strength of the member would also have to be computed using Specification Equation D2-2.

Calculate the nominal flexural strength for bending about the x-x axis

Yielding limit state

\(M_{n x}=M_{p}=F_{y} Z_{x}=50 ksi \left(139 in .^{3}\right)=6950 kip – in =579 \text { kip-ft }\)

Eqn. F2-1

Lateral-torsional buckling limit state

\(L_{b}=30.0 ft\)

 

Since \(L_{p}<L_{b} \leq L_{r}\), Equation F2-2 applies

 

Calculate lateral-torsion buckling modification factor

From Manual Table 3-1, \(C_{b}=1.14\), without considering the beneficial effects of the tension force. However, \(C_{b}\) may be increased because the column is in axial tension.

Section H1.2

\(P_{e y}=\frac{\pi^{2} E I_{y}}{L_{b}^{2}}=\frac{\pi^{2}(29,000 ksi )\left(148 in ^{4}\right)}{((30.0 ft )(12.0 in . / ft ))^{2}}=327 kips\)

 

LRFD ASD
\(\sqrt{1+\frac{P_{u}}{P_{e y}}}=\sqrt{1+\frac{174 kips }{327 kips }}=1.24\) \(\sqrt{1+\frac{1.5 P_{a}}{P_{e y}}}=\sqrt{1+\frac{1.5(116 kips )}{327 kips }}=1.24\)

 

\(C_{b}=1.24(1.14)=1.41\)

 

\(M_{n}=C_{b}\left[M_{p}-\left(M_{p}-0.7 F_{y} S_{x}\right)\left(\frac{L_{b}-L_{p}}{L_{r}-L_{p}}\right)\right] \leq M_{p}\)

Eqn. F2-2

\(M_{n}=1.41\left[6950 \text { kip-in. }-\left(6950 \text { kip-in. }-0.7(50 ksi )\left(123 in .^{3}\right)\right)\left(\frac{30.0 ft -8.76 ft }{33.1 ft -8.76 ft }\right)\right]\)

 

\(=6550 \text { kip }<M_{p} \text { therefore use: }\)

 

\(M_{n}=6550 \text { kip-in. or } 545 \text { kip-ft }\) controls

 

Local buckling limit state

Per Manual Table 1-1, the cross section is compact at \(F_{y}\) = 50 ksi; therefore, the local buckling limit state does not apply.

Manual
Table 1-1

Calculate the nominal flexural strength for bending about the y-y axis

Yielding limit state

Since W14×82 has compact flanges, only the limit state of yielding applies.

\(M_{n y}=M_{p}=F_{y} Z_{y} \leq 1.6 F_{y} S_{y}\)

Eqn. F6-1

\(=50 ksi \left(44.8 in .^{3}\right) \leq 1.6(50 ksi )\left(29.3 in .^{3}\right)\)

 

\(=2240 \text { kip-in }<2344 \text { kip-in., therefore use: }\)

 

\(M_{n y}=2240 \text { kip-in. or } 187 \text { kip-ft }\)

 

LRFD ASD
\(\begin{array}{l}\phi_{b}=\phi_{t}=0.90 \\P_{c}=\phi_{t} P_{n}=0.90(1200 \text { kips })=1080 \text { kips } \\M_{c x}=\phi_{b} M_{n x}=0.90(545 \text { kip }- ft )=491 \text { kip- } ft \\M_{c y}=\phi_{b} M_{n y}=0.90(187 kip – ft )=168 \text { kip- } ft\end{array}\) \(\begin{array}{l}\Omega_{b}=\Omega_{t}=1.67 \\P_{c}=P_{n} / \Omega_{t}=\frac{1200 \text { kips }}{1.67}=719 \text { kips } \\M_{c x}=M_{n x} / \Omega_{b}=\frac{545 \text { kip }- ft }{1.67}=327 \text { kip-ft } \\M_{c y}=M_{n y} / \Omega_{b}=\frac{187 \text { kip }- ft }{1.67}=112 \text { kip-ft }\end{array}\)

Sections D2
and F1

Check limit for Equation H1-1a

LRFD ASD
\(\frac{P_{r}}{\phi_{t} P_{n}}=\frac{P_{u}}{\phi_{t} P_{n}}=\frac{174 \text { kips }}{1080 kips }=0.161<0.2\) \(\frac{P_{r}}{P_{n} / \Omega_{t}}=\frac{P_{a}}{P_{n} / \Omega_{t}}=\frac{116 \text { kips }}{719 \text { kips }}=0.161<0.2\)

Therefore, Equation H1-1b applies

\(\frac{P_{r}}{2 P_{c}}+\left(\frac{M_{r x}}{M_{c x}}+\frac{M_{r y}}{M_{c y}}\right) \leq 1.0\)

Eqn. H1-1b

LRFD ASD
\(\begin{array}{l}\frac{174 kips }{2(1080 kips )}+\left(\frac{192 kip – ft }{491 kip – ft }+\frac{67.6 kip – ft }{168 kip – ft }\right) \leq 1.0 \\0.874<1.0 & \text { o.k. }\end{array}\) \(\begin{array}{l}\frac{116 \text { kips }}{2(719 \text { kips })}+\left(\frac{128 \text { kip-ft }}{327 \text { kip-ft }}+\frac{45.1 \text { kip-ft }}{112 \text { kip-ft }}\right) \leq 1.0 \\0.875<1.0 \quad \text { o.k. }\end{array}\)
