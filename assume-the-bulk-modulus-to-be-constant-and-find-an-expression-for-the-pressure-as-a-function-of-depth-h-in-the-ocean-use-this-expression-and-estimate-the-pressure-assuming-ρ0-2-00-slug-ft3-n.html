Guest Question.

Assume the bulk modulus to be constant and find an expression for the pressure as a function of depth h in the ocean. Use this expression and estimate the pressure assuming \(ρ_0 = 2.00 slug/ft^3\). Now, assume a constant density of 2.00 \(slug/ft^3\) and calculate the pressure and the percent error, assuming the estimate in the first calculation to be correct. Use depths of (a) 1500 ft, (b) 5000 ft, and (c) 15 000 ft.

Admin
Solution

Eq. 1.5.11 gives \(B= \rho \frac{dp}{d\rho }\mid _{T}\) But, dp = ρgdh. Therefore,

\(ρ gdh= \frac{B}{ρ}dρ or \frac{dρ }{ρ^2}=\frac{g}{B}dh \)

Integrate, using \(ρ_0 = 2.00 slug/ft^3, and B = 311,000 lb/in^2\):

\(\int\limits_{2}^{\rho }{\frac{dρ }{ρ^2}} =\frac{g}{B}\int\limits_{0}^{h}{dh}\) . ∴

–\(\left\lgroup\frac{1}{\rho }-\frac{1}{2} \right\rgroup =\left\lgroup\frac{32.2 ft/s^2}{311,000 (lb/in^2)\times 144 in^2/ ft^2} \right\rgroup h= 7.2\times 10^{-7}h\)

This gives \(\rho =\frac{1}{0.5- 7.2\times 10^{-7} h}\)

Now, \(p=\int\limits_{0}^{h}{ρ gdh}= \int\limits_{0}^{h}{\frac{g}{0.5- 7.2\times 10^{-7} h}dh }=\frac{g}{- 7.2\times 10^{-7}} ln(0.5 -7.2 \times 10^{-7}h )\)

If we assume ρ = const: p = ρ gh = 2.0× 32.2× h = 64.4h

b) For h = 5000 ft: \(p_{accurate} =\underline{ 323,200 psf} and p_{estimate} = \underline{322,000 psf} \).

% error =\(\frac{322,000 -323,200}{323,200} \times 100=\underline{-0.371 \%}\)
