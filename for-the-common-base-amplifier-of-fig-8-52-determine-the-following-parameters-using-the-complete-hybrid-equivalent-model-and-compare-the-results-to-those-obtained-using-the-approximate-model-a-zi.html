Guest Question.

For the common-base amplifier of Fig. 8.52, determine the following parameters using
the complete hybrid equivalent model and compare the results to those obtained using the approximate model.
(a) \(Z_{i}\) and \(Z^{′}_{i}\).

(b) \(A_{i}\) and \(A^{′}_{i}\).

(c) \(A_{v}\).

(d) \(Z_{o}\) and \(Z^{′}_{o}\).

Admin
Solution

The common-base hybrid parameters are derived from the common-emitter parameters using the approximate equations of Appendix A:

\(h_{ib} \cong \frac {h_{ie}}{1 + h_{fe}} = \frac{1.6 k \Omega}{1 + 110} = 14.41 \Omega\)

Note how closely the magnitude compares with the value determined from

\(h_{ib} = r_{e} = \frac{h_{ie}}{\beta} = \frac{1.6 k \Omega}{110} = 14.55 \Omega\)

 

\(h_{rb} \cong \frac{h_{ie}h_{oe}}{1 + h_{fe}} – h_{re} = \frac{(1.6 k \Omega)(20 \mu S)}{1 + 110} – 2 \times 10^{- 4} = 0.883 \times 10^{- 4}\)

 

\(h_{fb} \cong \frac{- h_{fe}}{1 + h_{fe}} = \frac{- 110}{1 + 110} = – 0.991\)

 

\(h_{ob} \cong \frac{h_{oe}}{1 + h_{fe}} = \frac{20 \mu S}{1 + 110} = 0.18 \mu S\)

Substituting the common-base hybrid equivalent circuit into the network of Fig.8.52 will then result in the small-signal equivalent network of Fig. 8.53. The Thévenin network for the input circuit will result in \(R_{Th} = 3 k \Omega \parallel 1 k \Omega = 0.75 k \Omega\) for \(R_{S}\) in the equation for \(Z_{o}\).

(a) Eq. (8.109): \(Z_{i} = \frac{V_{i}}{I_{i}} = h_{ib} – \frac{h_{fb} h_{rb} R_{L}}{1 + h_{ob} R_{L}} = 14.41\Omega – \frac{(-0.991)(0.883 \times 10^{- 4})(2.2 k \Omega)}{1 + (0.18 \mu S)(2.2 k \Omega)} = 14.41 \Omega + 0.19 \Omega = 14.60 \Omega\)

versus \(14.41 \Omega\) using \(Z_{i} \cong h_{ib}\).

\(Z^{′}_{i} = 3 k \Omega \parallel Z_{i} \cong Z_{i} = 14.60 \Omega\)

(b) Eq. (8.107): \(A_{i} = \frac{I_{o}}{I_{i}} = \frac{h_{fb}}{1 + h_{ob}R_{L}} = \frac{- 0.991}{1 + (0.18 \mu S)(2.2 k \Omega)} = – 0.991 = h_{fb}\)

Since \(3 k \Omega \gg Z_{i} , I^{′}_{i} \cong I_{i}\) and \(A^{′}_{i} = I_{o}/ I^{′}_{i} \cong – 1\) also.

(c) Eq. (8.108): \(A_{v} = \frac{V_{o}}{V_{i}} = \frac{- h_{fb}R_{L}}{h_{ib} + ( h_{ib} h_{ob} – h_{fb} h_{rb}) R_{L}} = \frac{-(- 0.991)(2.2 k \Omega)}{14.41 \Omega + \left[(14.41 \Omega)(0.18 \mu S) – (- 0.991)(0.883 \times 10^{- 4})\right] 2.2 k \Omega } = 149.25\)

versus 151.3 using \(A_{v} \cong – h_{fb} R_{L}/h_{ib}\).

(d) Eq. (8.110): \(Z_{o} = \frac{1}{h_{ob} – \left[h_{fb} h_{rb}/(h_{ib} + R_{S})\right] } = \frac{1}{0.18 \mu S – \left[(- 0.991)( 0.883 \times 10^{- 4}/14.41 \Omega +0.75 k \Omega)\right] } = \frac{1}{0.295 \mu S} =3.39 M \Omega\)

versus \(5.56 M \Omega\) using \(Z_{o} \cong 1/h_{ob}\). For \(Z^{′}_{o}\) as defined by Fig. 8.53:
\(Z^{′}_{o} = R_{C} \parallel Z_{o} = 2.2 k \Omega \parallel 3.39 M \Omega = 2.199 k \Omega\)

versus \( 2.2 k \Omega\) using \(Z^{′}_{o} \cong R_{C}\) .
