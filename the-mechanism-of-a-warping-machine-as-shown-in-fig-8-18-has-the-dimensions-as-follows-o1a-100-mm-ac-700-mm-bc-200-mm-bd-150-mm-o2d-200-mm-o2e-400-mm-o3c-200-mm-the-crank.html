Guest Question.

The mechanism of a warping machine, as shown in Fig. 8.18, has the dimensions as follows:
\(O_{1} A\)= 100 mm; AC = 700 mm ; BC = 200 mm ; BD = 150 mm ; \(O_{2} D\) = 200 mm ; \(O_{2} E\) mm ; \(O_{3} C=200 mm\).
The crank \(O_{1} A\) rotates at a uniform speed of 100 rad/s. For the given configuration, determine: 1. linear velocity of the point E on the bell crank lever, 2. acceleration of the points E and B, and 3. angular acceleration of the bell crank lever.

Admin
Solution

Given : \(\omega_{ AO 1}=100 rad / s ; O_{1} A=100 mm =0.1 m\)

We know that linear velocity of A with respect to \(O_{1}\), or velocity of A,

\(v_{ AOl }=v_{ A }=\omega_{ AOl } \times O_{1} A=100 \times 0.1=10 m / s\) …(Perpendicular to \(O_{1}\)A)

 

1. Linear velocity of the point E on bell crank lever

First of all draw the space diagram, as shown in Fig.8.19 (a), to some suitable scale. Now the velocity diagram, as shown in Fig.8.19 (b), is drawn as discussed below:

1. Since \(O_{1}, O_{2} \text { and } O_{3}\) are fixed points, therefore these points are marked as one point in the velocity diagram. From point \(o_{1}\), draw vector \(o_{1}\)a perpendicular to \(O_{1}\) A to some suitable scale, to represent the velocity of A with respect to O or velocity of A, such that

\(\text { vector } o_{1} a=v_{ AOl }=v_{ A }=10 m / s\)

2. From point a, draw vector ac perpendicular to AC to represent the velocity of C with respect to A (i.e. \(v_{ CA }\)) and from point \(o_{3}\) draw vector \(o_{3}\)c perpendicular to \(O_{3}\)C to represent the velocity of C with respect to \(O_{3}\) or simply velocity of C (i.e. \(v_{ C }\)). The vectors ac and \(o_{3}\)c intersect at point c.

3. Since B lies on AC, therefore divide vector ac at b in the same ratio as B divides AC in the space diagram. In other words, ab/ac = AB/AC

4. From point b, draw vector bd perpendicular to BD to represent the velocity of D with respect to B (i.e. \(v_{ DB }\)), and from point \(o_{2}\) draw vector \(o_{2}\)d perpendicular to \(O_{2}\)D to represent the velocity of D with respect to \(O_{2}\) or simply velocity of D (i.e. \(v_{ D }\)). The vectors bd and \(o_{2}\)d intersect at d.

5. From point \(o_{2}\), draw vector \(o_{2}\)e perpendicular to vector \(o_{2}\)d in such a way that

\(o_{2} e / o_{2} d=O_{2} E / O_{2} D\)

By measurement, we find that velocity of point C with respect to A,

\(v_{ CA }=\text { vector } a c=7 m / s\)

Velocity of point C with respect to \(O_{3}\),

\(v_{ CO 3}=v_{ C }=\text { vector } o_{3} c=10 m / s\)

Velocity of point D with respect to B,

\(v_{ DB }=\text { vector } b d=10.2 m / s\)

Velocity of point D with respect to \(O_{2}\),

\(v_{ DO 2}=v_{ D }=\text { vector } o_{2} d=2.8 m / s\)

and velocity of the point E on the bell crank lever,

\(v_{ E }=v_{ EO 2}=\text { vector } o_{2} e=5.8 m / s\)

 

2. Acceleration of the points E and B

Radial component of the acceleration of A with respect to \(O_{1}\) (or acceleration of A),

\(a_{ AO 2}^{r}=a_{ AO 1}=a_{ A }=\frac{v_{ AO 1}^{2}}{O_{1} A}=\frac{10^{2}}{0.1}=1000 m / s ^{2}\)

Radial component of the acceleration of C with respect to A,

\(a_{ CA }^{r}=\frac{v_{ CA }^{2}}{A C}=\frac{7^{2}}{0.7}=70 m / s ^{2}\)

Radial component of the acceleration of C with respect to \(O_{3}\)

\(a_{ CO 3}^{r}=\frac{v_{ CO 3}^{2}}{O_{3} C}=\frac{10^{2}}{0.2}=500 m / s ^{2}\)

Radial component of the acceleration of D with respect to B,

\(a_{ DB }^{r}=\frac{v_{ DB }^{2}}{B D}=\frac{(10.2)^{2}}{0.15}=693.6 m / s ^{2}\)

Radial component of the acceleration of D with respect to \(O_{2}\),

\(a_{ DO _{2}}^{r}=\frac{v_{ DO 2}^{2}}{O_{2} D}=\frac{(2.8)^{2}}{0.2}=39.2 m / s ^{2}\)

Radial component of the acceleration of E with respect to \(O_{2}\),

\(a_{ EO _{2}}^{r}=\frac{v_{ EO _{2}}^{2}}{O_{2} E}=\frac{(5.8)^{2}}{0.4}=84.1 m / s ^{2}\)

Now the acceleration diagram, as shown in Fig. 8.19 (c), is drawn as discussed below:

1. Since \(O_{1}, O_{2} \text { and } O_{3}\) are fixed points, therefore these points are marked as one point in the acceleration diagram. Draw vector \(o_{1}^{\prime} a^{\prime}\) parallel to \(O_{1}\)A, to some suitable scale, to represent the radial component of the acceleration of A with respect to \(O_{1}\) (or simply acceleration of A), such that

\(\text { vector } o_{1}^{\prime} a^{\prime}=a_{ AO _{1}}^{r}=a_{ A }=1000 m / s ^{2}\)

2. From point \(a^{\prime}\), draw \(a^{\prime}\)x parallel to AC to represent the radial component of the acceleration of C with respect to A (i.e. \(a_{ CA }^{r}\) ), such that

\(\text { vector } a^{\prime} x=a_{ CA }^{r}=70 m / s ^{2}\)

3. From point x, draw vector x\(c^{\prime}\) perpendicular to AC to represent the tangential component of the acceleration of C with respect to A (i.e. \(a_{ CA }^{t}\) ), the magnitude of which is yet unknown.

4. From point \(o_{3}^{\prime}\), draw vector \(o_{3}^{\prime}\) y parallel to \(O_{3}\)C to represent the radial component of the acceleration of C with respect to \(O_{3}\) (i.e. \(a_{ CO 3}^{r}\) ), such that

\(\text { vector } o_{3}^{\prime} y=a_{ CO 3}^{r}=500 m / s ^{2}\)

5. From point y, draw vector y\(c^{\prime}\) perpendicular to \(O_{3}\)C to represent the tangential component of the acceleration of C with respect to \(O_{3}\) (i.e. \(a_{ CO 3}^{t}\) ). The vectors x\(c^{\prime}\) and y\(c^{\prime}\) intersect at \(c^{\prime}\).

6. Join \(a^{\prime} c^{\prime}\). The vector \(a^{\prime} c^{\prime}\) represents the acceleration of C with respect to A (i.e. \(a_{ CA }\)).

7. Since B lies on AC, therefore divide vector \(a^{\prime} c^{\prime} \text { at } b^{\prime}\) in the same ratio as B divides AC in the space diagram. In other words, \(a^{\prime} b^{\prime} / a^{\prime} c^{\prime}=A B / A C\). Join \(b^{\prime} o_{2}^{\prime}\) which represents the acceleration of point B with respect to \(O _{2}\) or simply acceleration of B. By measurement, we find that

Acceleration of point B \(=\text { vector } o_{2}^{\prime} b^{\prime}=440 m / s ^{2}\)

8. Now from point \(b^{\prime}\), draw vector \(b^{\prime}\) z parallel to BD to represent the radial component of the acceleration of D with respect to B (i.e. \(a_{ DB }^{r}\) ), such that

\(\text { vector } b^{\prime} z=a_{ DB }^{r}=693.6 m / s ^{2}\)

9. From point z, draw vector z\(d^{\prime}\) perpendicular to BD to represent the tangential component of the acceleration of D with respect to B (i.e. \(a_{ DB }^{t}\) ), whose magnitude is yet unknown.

10. From point \(o_{2}^{\prime}\) , draw vector \(o_{2}^{\prime} z_{1}\) parallel to \(O_{2}\)D to represent the radial component of the acceleration of D with respect to \(O_{2}\) (i.e. \(a_{ DO _{2}}^{r}\) ), such that

\(\text { vector } o_{2}^{\prime} z_{1}=a_{ DO _{2}}^{r}=39.2 m / s ^{2}\)

11. From point \(z_{1}\), draw vector \(z_{1} d^{\prime}\) perpendicular to \(O _{2}\)D to represent the tangential component of the acceleration of D with respect to \(O _{2}\) (i.e. \(a_{ DO _{2}}^{t}\) ). The vectors \(z d^{\prime} \text { and } z_{1} d^{\prime} \text { intersect at } d^{\prime}\).

12. Join \(o_{2}^{\prime} d^{\prime}\). The vector \(o_{2}^{\prime} d^{\prime}\) represents the acceleration of D with respect to \(O_{2}\) or simply acceleration of D (i.e. \(a_{ DO 2} \text { or } a_{ D }\)).

13. From point \(o_{2}^{\prime}\), draw vector \(o_{2}^{\prime} e^{\prime}\) perpendicular to \(o_{2}^{\prime} d^{\prime}\) in such a way that

\(o_{2}^{\prime} e^{\prime} / o_{2}^{\prime} d^{\prime}=O_{2} E / O_{2} D\)

Note: The point \(e^{\prime}\) may also be obtained drawing \(a_{ EO _{2}}^{r} \text { and } a_{ EO _{2}}^{t}\) as shown in Fig. 8.19 (c).

By measurement, we find that acceleration of point E,

\(a_{ E }=a_{ EO 2}=\operatorname{vector}{o^{\prime}}_{2} e^{\prime}=1200 m / s ^{2}\)

 

3. Angular acceleration of the bell crank lever

By measurement, we find that the tangential component of the acceleration of D with respect to \(O_{2}\),

\(a_{ DO _{2}}^{t}=\text { vector } z_{1} d_{1}^{\prime}=610 m / s ^{2}\)

\(\therefore \) Angular acceleration of the bell crank lever

\(=\frac{a_{ DO 2}^{t}}{ O _{2} D}=\frac{610}{0.2}=3050 rad / s ^{2}(\text { Anticlockwise })\)
