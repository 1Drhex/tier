Guest Question.

A uniform plane wave with perpendicular polarization represented by Eqs. (7-135) and (7-136) is incident on a plane interface at \(z = 0\), a shown in Fig. 7-14. Assuming \({ \epsilon }_{ 2 }<{ \epsilon }_{ 1 }\), and \({ \theta }_{ i }>{ \theta }_{ c }\), \((a)\) obtain the phasor expressions for the transmitted field \(\left( { E }_{ t },{ H }_{ t } \right)\). and \((b)\) verify that the average power transmitted into medium \(2\) is zero

Admin
Solution

\(a)\) \(\sin { { \theta }_{ c } } =\sqrt { \frac { { \epsilon }_{ 2 } }{ { \epsilon }_{ 1 } } } \rightarrow \sin { { \theta }_{ t } } =\sqrt { \frac { { \epsilon }_{ 1 } }{ { \epsilon }_{ 2 } } } \sin { { \theta }_{ i } } >1\) for \({ \theta }_{ i }>{ \theta }_{ c }\)

 

\(\quad \quad \quad \quad \quad \quad \quad \quad \cos { { \theta }_{ t } } =-j\sqrt { \left( \frac { { \epsilon }_{ 1 } }{ { \epsilon }_{ 2 } } \right) \sin ^{ 2 }{ { \theta }_{ i } } -1 }\)

 

From Eq(7-141) and Eq(7-1142):

 

\(\overline { { E }_{ t } } \left( x,z \right) =\overline { { a }_{ y } } { E }_{ t0 }{ e }^{ -{ \alpha }_{ 1 }z }{ e }^{ -j{ \beta }_{ 2x }x },\)

 

\(\overline { { H }_{ t } } \left( x,z \right) =\frac { { E }_{ t0 } }{ { \eta }_{ 2 } } \left( \overline { { a }_{ x } } j{ \alpha }_{ 2 }+\overline { { a }_{ y } } \sqrt { \frac { { \epsilon }_{ 1 } }{ { \epsilon }_{ 2 } } } \sin { { \theta }_{ i } } \right) { e }^{ -{ \alpha }_{ 2 }z }{ e }^{ -j{ \beta }_{ 2x } },\)

 

Where \({ \beta }_{ 2x }={ \beta }_{ 1 }\sin { { \theta }_{ t } } -{ \beta }_{ 2 }\sqrt { \frac { { \epsilon }_{ 1 } }{ { \epsilon }_{ 2 } } } \sin { { \theta }_{ i } } \)

 

\({ \alpha }_{ 2 }={ \beta }_{ 2 }\sqrt { \left( \frac { { \epsilon }_{ 2 } }{ { \epsilon }_{ 1 } } \right) \sin ^{ 2 }{ { \theta }_{ i } } -1 }\)

 

\({ E }_{ t0 }=\frac { 2{ \eta }_{ 1 }\cos { { \theta }_{ i } } { E }_{ i0 } }{ { \eta }_{ 2 }\cos { { \theta }_{ i } } -j{ \eta }_{ 1 }\sqrt { \left( \frac { { \epsilon }_{ 2 } }{ { \epsilon }_{ 1 } } \right) \sin ^{ 2 }{ { \theta }_{ i } } -1 } }\) from Eq(7-148)

 

\(b)\) \({ \left( { \wp }_{ av } \right) }_{ 2z }={ \Re }_{ e }\left( { E }_{ ty }{ H }_{ tx }* \right) \)

 

\({ \tau }_{ \bot }=\frac { { E }_{ t0 } }{ { E }_{ i0 } } =\frac { { \eta }_{ 2 }\cos { { \theta }_{ i } } -{ \eta }_{ 1 }\cos { { \theta }_{ t } } }{ { \eta }_{ 2 }\cos { { \theta }_{ i } } +{ \eta }_{ 1 }\cos { { \theta }_{ t } } } \quad \quad \quad \left( 7-148 \right)\)
