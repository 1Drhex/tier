π-GROUPS FOR DRAG ON A SPHERE USING EXPONENT METHOD

The drag of a sphere, \(F_{D}\), in a flowing fluid is a function of the velocity V, the fluid density \(\rho\), the fluid viscosity \(\mu\), and the sphere diameter D. Find the \(\pi \text { -groups }\) using the exponent method.

Problem Definition

Situation: Given \(F_{D}=f(V, \rho, \mu, D)\).

Find: The \(\pi \text { -groups }\) using exponent method.

Plan

Follow the procedure for the exponent method in Table 8.2.

Admin
Solution

1. Dimensions of significant variables are

 

\([F]=\frac{M L}{T^{2}},[V]=\frac{L}{T},[\rho]=\frac{M}{L^{3}},[\mu]=\frac{M}{L T},[D]=L\)

 

2. Number of \(\pi \text { -groups is } 5-3=2\).

3. Form product with dimensions.

 

\(\frac{M L}{T^{2}}=\left[\frac{L}{T}\right]^{a} \times\left[\frac{M}{L^{3}}\right]^{b} \times\left[\frac{M}{L T}\right]^{c} \times[L]^{d}\)

 

\(=\frac{L^{a-3 b-c+d} M^{b+c}}{T^{a+c}}\)

 

4. Dimensional homogeneity. Equate powers of dimensions on each side.

 

\(L: \quad a-3 b-c+d=1\)

 

\(M: \quad b+c=1\)

 

\(T: \quad a+c=2\)

 

5. Solve for exponents a, b, and c in terms of d.

 

\(\left(\begin{array}{ccc}1 & -3 & -1 \\0 & 1 & 1 \\1 & 0 & 1\end{array}\right)\left(\begin{array}{l}a \\b \\c\end{array}\right)=\left(\begin{array}{c}1-d \\1 \\2\end{array}\right)\)

 

The value of the determinant is –1 so a unique solution is achievable. Solution is \(a=d, \quad b=d-1, \quad c=2-d\)

 

6. Write dimensional equation with exponents.

 

\(F=V^{d} \rho^{d-1} \mu^{2-d} D^{d}\)

 

\(F=\frac{\mu^{2}}{\rho}\left(\frac{\rho V D}{\mu}\right)^{d}\)

 

\(\frac{F \rho}{\mu^{2}}=\left(\frac{\rho V D}{\mu}\right)^{d}\)

 

There are two \(\pi \text { -groups }\):

 

\(\pi_{1}=\frac{F \rho}{\mu^{2}} \text { and } \pi_{2}=\frac{\rho V D}{\mu}\)

 

By dividing \(\pi_{1}\) by the square of \(\pi_{2}\) , the \(\pi_{1}\) group can be written as \(F_{D} /\left(\rho V^{2} D^{2}\right)\), so the functional form of the equation can be written as

 

\(\frac{F}{\rho V^{2} D^{2}}=f\left(\frac{\rho V D}{\mu}\right)\)

 

Review

1. The group of variables raised to the power forms a π-group.

2. The functional relationship between the two π-groups is obtained from experiment.
