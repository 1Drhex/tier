Guest Question.

Molecular Speeds in a Hydrogen Gas

A 0.500-mol sample of hydrogen gas is at 300 K.

(A) Find the average speed, the rms speed, and the most probable speed of the hydrogen molecules.

(B) Find the number of molecules with speeds between 400 m/s and 401 m/s.

Admin
Solution

Conceptualize Imagine a huge number of particles in a real gas, all moving in random directions with different speeds.

Categorize We cannot calculate the averages as was done in because the individual speeds of the particles are not known. We are dealing with a very large number of particles, however, so we can use the Maxwell- Boltzmann speed distribution function.

Analyze Use Equation 21.43 to find the average speed:

\(v_{\mathrm{avg}}=\sqrt{\frac{8 k_{\mathrm{B}} T}{\pi m_{0}}}=1.60 \sqrt{\frac{k_{\mathrm{B}} T}{m_{0}}}\) (21.43)

\(\begin{aligned}v_{\text {avg }} &=1.60 \sqrt{\frac{k_{\mathrm{B}} T}{m_{0}}}=1.60 \sqrt{\frac{\left(1.38 \times 10^{-23} \mathrm{~J} / \mathrm{K}\right)(300 \mathrm{~K})}{2\left(1.67 \times 10^{-27} \mathrm{~kg}\right)}} \\&=1.78 \times 10^{3} \mathrm{~m} / \mathrm{s}\end{aligned}\)

Use Equation 21.42 to find the rms speed:

\(v_{\mathrm{rms}}=\sqrt{\overline{v^{2}}}=\sqrt{\frac{3 k_{\mathrm{B}} T}{m_{0}}}=1.73 \sqrt{\frac{k_{\mathrm{B}} T}{m_{0}}}\) (21.42)

\(\begin{aligned}v_{\text {rms }} &=1.73 \sqrt{\frac{k_{\mathrm{B}} T}{m_{0}}}=1.73 \sqrt{\frac{\left(1.38 \times 10^{-23} \mathrm{~J} / \mathrm{K}\right)(300 \mathrm{~K})}{2\left(1.67 \times 10^{-27} \mathrm{~kg}\right)}} \\&=1.93 \times 10^{3} \mathrm{~m} / \mathrm{s}\end{aligned}\)

Use Equation 21.44 to find the most probable speed:

\(v_{\mathrm{mp}}=\sqrt{\frac{2 k_{\mathrm{B}} T}{m_{0}}}=1.41 \sqrt{\frac{k_{\mathrm{B}} T}{m_{0}}}\) (21.44)

\(\begin{aligned}v_{\mathrm{mp}} &=1.41 \sqrt{\frac{k_{\mathrm{B}} T}{m_{0}}}=1.41 \sqrt{\frac{\left(1.38 \times 10^{-23} \mathrm{~J} / \mathrm{K}\right)(300 \mathrm{~K})}{2\left(1.67 \times 10^{-27} \mathrm{~kg}\right)}} \\&=1.57 \times 10^{3} \mathrm{~m} / \mathrm{s}\end{aligned}\)

Use Equation 21.41 to evaluate the number of molecules in a narrow speed range between v and v + dv:

\(N_{v}=4 \pi N\left(\frac{m_{0}}{2 \pi k_{\mathrm{B}} T}\right)^{3 / 2} v^{2} e^{-m_{0} v^{2} / 2 k_{\mathrm{B}} T}\) (21.41)

\(\text { (1) } N_{v} d v=4 \pi N\left(\frac{m_{0}}{2 \pi k_{\mathrm{B}} T}\right)^{3 / 2} v^{2} e^{-m_{0} v^{2} / 2 k_{\mathrm{B}} T} d v\)

Evaluate the constant in front of \(v^2\):

\(\begin{aligned}4 \pi N\left(\frac{m_{0}}{2 \pi k_{\mathrm{B}} T}\right)^{3 / 2} &=4 \pi n N_{\mathrm{A}}\left(\frac{m_{0}}{2 \pi k_{\mathrm{B}} T}\right)^{3 / 2} \\&=4 \pi(0.500 \mathrm{~mol})\left(6.02 \times 10^{23} \mathrm{~mol}^{-1}\right)\left[\frac{2\left(1.67 \times 10^{-27} \mathrm{~kg}\right)}{2 \pi\left(1.38 \times 10^{-23} \mathrm{~J} / \mathrm{K}\right)(300 \mathrm{~K})}\right]^{3 / 2} \\&=1.74 \times 10^{14} \mathrm{~s}^{3} / \mathrm{m}^{3}\end{aligned}\)

Evaluate the exponent of e that appears in Equation (1):

\(-\frac{m_{0} v^{2}}{2 k_{\mathrm{B}} T}=-\frac{2\left(1.67 \times 10^{-27} \mathrm{~kg}\right)(400 \mathrm{~m} / \mathrm{s})^{2}}{2\left(1.38 \times 10^{-23} \mathrm{~J} / \mathrm{K}\right)(300 \mathrm{~K})}=-0.0645\)

Evaluate \(N_{v}\) dv using these values in Equation (1):

\(\begin{aligned}N_{v} d v &=\left(1.74 \times 10^{14} \mathrm{~s}^{3} / \mathrm{m}^{3}\right)(400 \mathrm{~m} / \mathrm{s})^{2} e^{-0.0645}(1 \mathrm{~m} / \mathrm{s}) \\&=2.61 \times 10^{19} \mathrm{molecules}\end{aligned}\)

Finalize In this evaluation, we could calculate the result without integration because dv = 1 m/s is much smaller than v = 400 m/s. Had we sought the number of particles between, say, 400 m/s and 500 m/s, we would need to integrate Equation (1) between these speed limits.
