Guest Question.

Select a W18 column of ASTM A36 steel, 26 ft. long, and subjected to a factored axial load of 500 kips. Assume that the column is pinned at both ends.

Admin
Solution

Since W18 shapes are not listed in the AISCM column load tables, we cannot use these tables to design this column. Procedure 1 in Section 5.6 will be followed.

\(P_{u} = 500 kips\),

KL = 1.0 × 26 ft. = 26 ft.

Cycle 1:

  1. Assume \(\phi_{c}F_{cr}=20ksi\lt F_{y}\) (36 ksi).
  2. \(A_{g .required}\geq \frac{P_{u}}{\phi_{c}F_{cr}} =\frac{500}{20} =25in^2\)
  3. Select W18 section from part 1 of the AISCM with \(A_g \gt 2.5in^2\).
    Try W18 × 86 with:
    \(A_g=25.3in^2\),
    \(r_x=7.77 in.^2\)and
    \(r_y=2.63in.^2\).
    \(\frac{KL}{r_{min}} =\frac{(1.0)(26 ft.)(12)}{2.63} =118.6\lt 200 \)OK
  4. Go to AISCM, Table 4-22, with the KL/r value from step 3 and obtain \(\phi_cF_{cr}=15.5\)ksi
  5. \(\phi_cP_{n}=(\phi_cF_{cr})(A_g)=(15.5ksi)(25.3in.^2)=392kips\lt P_u=500kips\)

The selected column is not adequate. Therefore, proceed to cycle 2.

 

Cycle 2:

  1. Assume \(\phi_cF_{cr}=15.5ksi\) (from step 4 of the previous cycle).
  2. \(A_{g .required}\geq \frac{P_{u}}{\phi_{c}F_{cr}} =\frac{500}{15.5} =32.2in^2\)
  3. Try W18 × 119 with
    \(A_g=35.1in^2>A_{g .required}\),
    \(r_x=7.90 in.\)and
    \(r_y=2.69in.\).
    \(\frac{KL}{r_{min}}= \frac{(1.0)(26 ft.)(12)}{2.69} =116<200\)OK
  4. Go to AISC, Table 4-22 with KL/r = 116 and obtain \(\phi_cF_{cr}=16.0 ksi\)
  5. The compression design strength is
    \(\phi_cP_{n}=(\phi_cF_{cr})(A_g)=(16.0 ksi)(35.1 in.^2)=561 kips>P_u=500kips\) OK
    Therefore, the W18 × 119 column is adequate.
  6. Check the slenderness criteria for compression elements:
    \(b_f=11.3 in.(b = 11.3/2 = 5.65 in.)\).
    \(t_f=1.06in\).
    \(t_w=0.655in\).
    \(h = d – 2k_{des} = 19.0 – (2)(1.46) = 16.08 in.\)

(Note: \(k_{des}\) is smaller than \(k_{det}\) and should be used for the design. The difference in these values is due to the variation in the fabrication processes.)

\(\frac{b}{t} \leq 0.56\sqrt{\frac{E}{F_y} } ;\frac{5.65}{1.06} =5.33\lt 0.56\sqrt{\frac{29,000}{36} } =15.9\) OK

\(\frac{h}{t_w} \leq 1.49\sqrt{\frac{E}{F_y} } ;\frac{16.08}{0.655} =24.5\lt 1.49\sqrt{\frac{29,000}{36} } =42.2\) OK

This implies that the column section is not slender; therefore, the design strength calculated above does not have to be reduced for slenderness effects.
