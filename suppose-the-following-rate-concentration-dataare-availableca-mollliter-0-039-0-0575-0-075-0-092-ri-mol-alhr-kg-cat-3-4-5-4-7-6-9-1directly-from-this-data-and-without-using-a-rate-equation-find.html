Guest Question.

suppose the following rate concentration data are available:

\(C_A mol/liter\) 0.039 0.0575 0.075 0.092
\(r_{\mathrm{A}}^{\prime}, \mathrm{mol} \mathrm{A} / \mathrm{hr} \cdot \mathrm{kg} \mathrm{cat}\) 3.4 5.4 7.6 9.1

 

Directly from this data, and without using a rate equation, find the size of packed bed needed to treat \(2000 \mathrm{~mol} / \mathrm{hr}\) of pure \(\mathrm{A}\) at \(117^{\circ} \mathrm{C}\) (or \(C_{\mathrm{A} 0}=0.1 \mathrm{~mol} /\) liter, \(\varepsilon_{\mathrm{A}}=3\) ) to \(35 \%\) conversion, all at \(3.2 \mathrm{~atm}\)

Note: Rate information such as this can be obtained from a differential reactor (see Table E18.2), or from other types of experimental reactors.

Admin
Solution

To find the amount of catalyst needed without using an analytic expression for the rate concentration relationship requires graphical integration of the plug flow performance equation, or

\(\frac{W}{F_{\mathrm{A} 0}}=\int_{0}^{0.35} \frac{d X_{\mathrm{A}}}{-r_{\mathrm{A}}^{\prime}}\)

 

Table 18.5

\(-{r}'_{A}\) (given) \(\frac{1}{-r_{\mathrm{A}}^{\prime}}\) \(C_A (given)\) \(X_{\mathrm{A}}=\frac{1-C_{\mathrm{A}} / 0.1}{1+3 C_{\mathrm{A}} / 0.1}\)
3.4 0.294 0.039 0.2812
5.4 0.186 0.0575 0.1563
7.6 0.1316 0.075 0.0778
9.1 0.11 0.092 0.02275

 

The needed \(1 /-r_{\mathrm{A}}^{\prime}\) versus \(X_{\mathrm{A}}\) data are determined in Table \(\mathrm{E} 18.5\) and are plotted in Fig. E18.5. Integrating graphically then gives

\(\int_{0}^{0.35} \frac{d X_{\mathrm{A}}}{-r_{\mathrm{A}}^{\prime}}=0.0735\)

Hence

\(\underline{\underline{W}}=\left(2000 \frac{\mathrm{mol} \mathrm{A}}{\mathrm{hr}}\right)\left(0.0735 \frac{\mathrm{hr} \cdot \mathrm{kg} \mathrm{cat}}{\mathrm{mol} \mathrm{A}}\right)=\underline{147 \mathrm{~kg} \mathrm{cat}}\)
