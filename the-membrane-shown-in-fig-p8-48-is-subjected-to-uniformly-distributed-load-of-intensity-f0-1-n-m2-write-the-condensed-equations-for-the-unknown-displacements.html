The membrane shown in Fig. P8.48 is subjected to uniformly distributed load of intensity \(f_{0}=1 N / m ^{2}\). Write the condensed equations for the unknown displacements.

Solution

The membrane shown in Fig. P8.48 is subjected to uniformly distributed load of intensity \(f_{0}=1 N / m ^{2}\). Write the condensed equations for the unknown displacements.

For 2 Ã— 2 mesh of linear rectangular elements, the only unknown nodal values are \(U_{5}\) and \(U_{6}\). All elements are identical, with the same element cofficient matrix. The equations governing the unknown nodal values are:

\(K_{55} U_{5}+K_{56} U_{6}=4 \frac{f_{0} a b}{4}-\left(K_{58} U_{8}+K_{59} U_{9}\right)\)

 

\(K_{56} U_{5}+K_{66} U_{6}=2 \frac{f_{0} a b}{4}-\left(K_{68} U_{8}+K_{69} U_{9}\right)\)

where \(U_{8}\) = 0.1875 m and \(U_{9}\) = 0.25 m. Using the element matrix in Eq. (8.2.54) (a = 0.025 m and b = 0.04 m), we can write the above equations as

\(\left[\begin{array}{rr}2.9667 & -0.8583 \\-0.8583 & 1.4833\end{array}\right]\left\{\begin{array}{c}U_{5} \\U_{6}\end{array}\right\}\) = \(10^{-2}\left\{\begin{array}{l}0.10 \\0.05\end{array}\right\}+\left\{\begin{array}{l}0.07083 \\0.05495\end{array}\right\}\) = \(\left\{\begin{array}{l}0.07183 \\0.05545\end{array}\right\}\)

The solution of these equations is \(U_{5}\) = 4.2072 cm and \(U_{6}\) = 6.1726 cm.

(8.2.54): \([K^{e}]=\frac{k_{e}}{6ab}\begin{bmatrix}2(a^{2}+b^{2}) &a^{2}-2b^{2} &-(a^{2}+b^{2}) &b^{2}-2a^{2}\\ a^{2}-2b^{2} &2(a^{2}+b^{2}) &b^{2}-2a^{2} &-(a^{2}+b^{2}) \\ -(a^{2}+b^{2}) &b^{2}-2a^{2} &2(a^{2}+b^{2}) &a^{2}-2b^{2} \\ b^{2}-2a^{2} &-(a^{2}+b^{2}) &a^{2}-2b^{2} &2(a^{2}+b^{2}) \end{bmatrix}\)
