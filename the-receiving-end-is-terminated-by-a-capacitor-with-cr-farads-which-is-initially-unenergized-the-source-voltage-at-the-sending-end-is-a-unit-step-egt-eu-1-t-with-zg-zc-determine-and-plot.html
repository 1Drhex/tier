Single-phase lossless-line transients: step-voltage source matched at sending end, capacitive load at receiving end

The receiving end is terminated by a capacitor with \(C_{R} \) farads, which is initially unenergized. The source voltage at the sending end is a unit step \(e_{G} \left(t\right) =Eu_{-1} \left(t\right) \), with \(Z_{G} =Z_{c}\). Determine and plot υ(x, t) versus time t at the sending end of the line.

Admin
Solution

From (13.2.12) with \(Z_{R}=\frac{1}{sC_{R} }\) and \(Z_{G} =Z_{c}\),

\(\Gamma _{S} \left(s\right)=\frac{\frac{Z_{G}\left(s\right) }{Z_{c} } -1}{\frac{Z_{G}\left(s\right) }{Z_{c}} +1}\) per unit (13.2.12)

\(\Gamma _{R}\left(s\right)=\frac{\frac{1}{sC_{R} Z_{c} }-1 }{\frac{1}{sC_{R}Z_{c} } +1}=\frac{-s+\frac{1}{Z_{c}C_{R} } }{s+\frac{1}{Z_{c}C_{R} } }\)

 

\(\Gamma _{S}\left(s\right) =\frac{1-1}{1+1} =0\)

Then, from (13.2.10), with \(E_{G}\left(s\right)=E/s \),

\(V\left(x,s\right) =E_{G}\left(s\right) \left[\frac{Z_{c} }{Z_{G}\left(s\right)+Z_{c} } \right]\left[\frac{e^{-sx/\nu}+\Gamma _{R}\left(s\right)e^{s\left[\left(x/\nu \right) -2\tau \right] } }{1-\Gamma _{R}\left(s\right) \Gamma _{S} \left(s\right)e^{-2s\tau } } \right] \) (13.2.10)

\(V\left(x,s\right) =\frac{E}{s} \left(\frac{1}{2} \right) \left[e^{-sx/\nu }+\left(\frac{-s+\frac{1}{Z_{c}C_{R} } }{s+\frac{1}{Z_{c}C_{R} } } \right) e^{s\left[\left(x/\nu \right)-2\tau \right] }\right] =\frac{E}{2} \left[\frac{e^{-sx/\nu } }{s}+\frac{1}{s} \left(\frac{-s+\frac{1}{Z_{c}C_{R} } }{s+\frac{1}{Z_{c} C_{R} } } \right) e^{s\left[\left(x/\nu \right) -2\tau \right] } \right]\)

Using partial fraction expansion of the second term above,

\(V\left(x,s\right)=\frac{E}{2}\left[\frac{e^{-sx/\nu } }{s}+\left(\frac{1}{s} -\frac{2}{s+\frac{1}{Z_{c} C_{R} } } \right)e^{s\left[\left(x/\nu \right)-2\tau \right] } \right]\)

The inverse Laplace transform is

\(\upsilon \left(x,t\right)= \frac{E}{2} u_{-1}\left(t-\frac{x}{\nu } \right) +\frac{E}{2}\left[1-2e^{\left(-1/Z_{c} C_{R} \right)\left(t+x/\nu -2\tau \right) } \right]u_{-1}\left(t+\frac{x}{\nu }-2\tau \right)\)

At the sending end, where x = 0,

\(\upsilon \left(0,t\right)= \frac{E}{2} u_{-1}\left(t\right) +\frac{E}{2} \left[1-2e^{\left(-1/Z_{c}C_{R} \right) \left(t-2\tau \right) } \right]u_{-1} \left(t-2\tau \right)\)

υ(0, t) is plotted in Figure 13.6. As in , a forward traveling step voltage wave of E/2 volts is initiated at the sending end at t = 0. At \(t=\tau \), when the forward traveling wave arrives at the receiving end, a backward traveling wave is initiated. The backward traveling voltage wave, an exponential with initial value -E/2, steady-state value +E/2, and time constant \(Z_{c} =C_{R} \), arrives at the sending end at \(t=2\tau \), where it is superimposed on the forward traveling wave. No additional waves are initiated, since the source impedance is matched to the line. In steady-state, the line and the capacitor at the receiving end are energized at E volts with zero current. The capacitor at the receiving end can also be viewed as a short circuit at the instant \(t=\tau \), when the forward traveling wave arrives at the receiving end. For a short circuit at the receiving end, \(\Gamma _{R} =-1\), and therefore the backward traveling voltage wavefront is -E/2, the negative of the forward traveling wave. However, in steady-state the capacitor is an open circuit, for which \(\Gamma _{R} =+1\), and the steadystate backward traveling voltage wave equals the forward traveling voltage wave.
