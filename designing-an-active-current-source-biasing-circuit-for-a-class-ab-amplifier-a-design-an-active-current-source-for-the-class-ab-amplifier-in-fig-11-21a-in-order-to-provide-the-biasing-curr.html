Designing an active current–source biasing circuit for a class AB amplifier

(a) Design an active current source for the class AB amplifier in Fig. 11.21(a) in order to provide the biasing current of \( I_{ R }=5.67 mA \) needed for . Assume \( V_{ CC }=V_{ EE }=12 V , V_{ BE }=0.7 V , \text { and } R_{ L }=50 \Omega \). (b) Use PSpice/SPICE to plot the transfer characteristic and the instantaneous \( i_{ N }, i_{ P }, \text { and } i_{ O } \text { for } v_{ I }=5 \sin (2000 \pi t) \). The PSpice model parameters for the transistors are

IS=1E-14 BF=50 VJE=0.7

and for the diodes are

IS=1E-13 BV=100

Admin
Solution

(a) The biasing current source \( I_{ R } \), shown in Fig. 11.22, can be produced by two pnp transistors \( Q _{1} \text { and } Q _{2} \) and a resistance \( R_{1} \). Thus,

\( I_{ R } \approx I_{ ref }=\frac{V_{ CC }-V_{ EBP }}{R_{ ref }} \) (11.75)

 

which, for \( I_{ R }=5.67 mA \), gives

\( R_{ ref }=\frac{V_{ CC }-V_{ EBP }}{I_{ R }}=\frac{12 V -0.7 V }{5.67 mA } \approx 2 k \Omega \)

 

(b) In practice, emitter resistances \( R_{ E 1} \text { and } R_{ E 2} \), shown in Fig. 11.22, are connected to ensure the stability of the biasing point.

The transfer characteristic for the circuit in Fig. 11.22 is shown in Fig. 11.23(a). It gives \( V_{ O (\max )} \equiv V (2)= 11.139 V \) and offset voltages of \( v_{ O }=560.4 mV \text { at } v_{ I }=0 \text { and } v_{ O }=0 \text { at } v_{ I }=-640.9 mV \). The plots of \( i_{ N }, i_{ P } , \text { and } i_{ O } \) are shown in Fig. 11.23(b), which give \( i_{ N (\text { peak })} \equiv I ( RE 1)=109.66 mA , i_{ P (\text { peak })} \equiv I ( RE 2)=-83.92 mA \), and \( i_{ O (\text { peak })} \equiv I ( RL )=109.59 mA \text { to }-83.83 mA \). Thus, because of the offset voltage, the load current is not symmetrical.
