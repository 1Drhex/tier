Guest Question.

(a) Show that the second-order system
\( \ddot{y} + 2\zeta \omega_{n} \dot{y} + \omega^{2}_{n} y = 0 , y(0) = y_{o} , \dot{y}(0) = 0\),
has the response
\(y(t) = y_{o}\frac{e^{- \sigma t}}{\sqrt{1 - \zeta^{2}}}\sin(\omega_{d}t + \cos^{- 1} \zeta)\).
(b) Prove that, for the underdamped case \((\zeta \lt 1)\), the response oscillations decay at a predictable rate (see Fig. 3.63) called the logarithmic decrement
\( \delta = \ln\frac{y_{o}}{y_{1}} = \sigma \tau_{d} = \ln \frac{\Delta y_{1}}{y_{1}} \cong \ln \frac{\Delta y_{i}}{y_{i}}\),
where
\(\tau_{d} = \frac{2 \pi}{\omega_{d}}\)
is the damped natural period of vibration.

Admin
Solution

(a) The system is second order \(\Longrightarrow Q(s) = s^{2} + 2 \zeta \omega_{n} s + \omega^{2}_{n}\). The initial condition response can be obtained by plugging a dirac delta at the input at the time 0 (this ”charges” the system immediately to its initial condition and after that the system evolves by itself).
\(Input_{effective} = y_{0} \delta(t)\)

 

\(L[Input_{effective}] = y_{0}\)
We do not know whether the transfer function has finite zeros or not, but further thought will reveal the presence of at least one finite zero in the H(s).
\(\underset{s \rightarrow \infty}{\lim} s H(s)y_{0} = y(t)|_{0 +}\)
where
\(H(s) = \frac{P(s)}{Q(s)} = \frac{P(s)}{s^{2} + 2 \zeta \omega_{n}s + \omega^{2}_{n}}\).
If P(s) were a constant (no zeros in the H(s)), then the limit in the initial value theorem would give always zero (which is wrong because we know that the initial value must be \(y_{0}\).) So we need a zero. We suggest using the following H(s):
\(H(s) = \frac{- s}{s^{2} + 2 \zeta \omega_{n}s + \omega^{2}_{n}}\),
\(Y(s) = H(s)y_{0} = \frac{- s y_{0}}{s^{2} + 2 \zeta \omega_{n}s + \omega^{2}_{n}} , = \frac{R_{+}}{s – P_{+}} + \frac{R_{-}}{s – P_{–}}\).

where

\(P_{+} = – \zeta \omega_{n} + j \omega_{n} \sqrt{1 – \zeta^{2}}\),

\(P_{–} = – \zeta \omega_{n} – j \omega_{n} \sqrt{1 – \zeta^{2}}\),

\(R_{+} = \frac{- \omega_{n} e^{j(\pi = \cos^{- 1} \zeta)}}{2 \omega_{n} \sqrt{1 – \zeta^{2}} e^{j \pi /s}}\),

\(R_{–} = R^{\ast}_{+}\).

Note: The residues can be calculated graphically.

\(R_{+} = \underset{s \rightarrow P_{+}}{\lim}[(s – P_{+}) Y(s)]\),

\(\Longrightarrow y(t) = R_{+}e^{P_{+} t} + R_{–}e^{P_{–}t}\)

 

\(y(t) = \frac{- e^{- \zeta \omega_{n}t}}{2 \sqrt{1 – \zeta^{2}}}\left[e^{+ j (\omega_{n} \sqrt{1 – \zeta^{2}} t + \pi /2 – \cos^{- 1} \zeta)} + e^{- j (\omega_{n} \sqrt{1 – \zeta^{2}} t + \pi /2 – \cos^{- 1} \zeta)}\right]\),

\(\Longrightarrow y(t) = y_{0} \frac{e^{- \sigma t}}{\sqrt{1 – \zeta^{2}}} \sin(\omega_{d}t – \cos^{- 1} \zeta)\).

(b)

\(\frac{dy(t)}{dt} = 0 \Longrightarrow t = \frac{n \pi}{\omega_{d}}\) (n is any integer)

\(t_{Max} = \frac{2 \pi}{\omega_{d}} n\)

 

\(y(t)|_{t_{Max}} \equiv y_{n} = y_{0} \frac{e^{- \sigma n \tau_{d}}}{\sqrt{1 – \zeta^{2}}} \sin(\cos^{- 1} \zeta)\).

Note:

\(\sin (- \cos^{- 1} \zeta) = \sqrt{1 – \zeta^{2}}\)

 

\(y_{n} = \frac{y_{0} \sqrt{1 – \zeta^{2}}}{\sqrt{1 – \zeta^{2}}} e^{- \sigma n \tau_{d}} (\ast)\)

(Proof of the first line)

\(\sigma = \ln \frac{y_{0}}{y_{n}} = \sigma \tau_{d}\),

From \((\ast)\)

\(y_{1} = y_{0}e^{- \sigma \tau_{d}} \Longrightarrow \ln \frac{y_{0}}{y_{n}} = \sigma \tau_{d}\).

(Proof of the second line)

\(\Delta y_{n} = y_{n – 1} – y_{n}\),

\(\Delta y_{n} = y_{0} e^{- \sigma n \tau_{d}} – y_{0} e^{- (n – 1) \sigma \tau_{d}} = y_{0} e^{- \sigma n \tau_{d}}(1 – e^{\sigma \tau_{d}})\),

\(\Longrightarrow \frac{\Delta y_{n}}{y_{n}} = \frac{y_{0} e^{- \sigma n \tau_{d}}}{y_{0}e^{ – \sigma n \tau_{d}}}(1 – e^{\sigma \tau_{d}})\),

\(\Longrightarrow \frac{\Delta y_{n}}{y_{n}} = \frac{\Delta y_{i}}{y_{i}}\) for all i, n .
