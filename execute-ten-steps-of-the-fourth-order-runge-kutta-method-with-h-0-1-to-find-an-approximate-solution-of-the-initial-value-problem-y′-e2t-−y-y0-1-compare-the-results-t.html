Guest Question.

Execute ten steps of the fourth-order Runge–Kutta method with h = 0.1 to find an approximate solution of the initial-value problem

\(y′= e^{2t} −y, y(0) = 1\)

Compare the results to those of the second-order Taylor method.

Admin
Solution

This is the same IVP as we considered in 1. Recall that the exact solution to the problem is \(y(t ) = 2/3e^{−t} +1/3e^{2t}\) .

To implement the Runge–Kutta method, we use \(f (t , y) = e^{2t} − y\) and compute \(a_{n}, b_{n}, c_{n}, and d_{n}\) as given by (7.3.18). Using the initial condition \((t_{0}, y_{0}) = (0, 1)\), we compute

\(a_{n} = f (t_{n}, y_{n})\) slope at left endpoint
\(b_{n} = f (t_{n} + \frac {1}{2}h, y_{n} + \frac {1}{2}ha_{n})\) slope at midpoint
\(c_{n} = f (t_{n} + \frac {1}{2}h, y_{n} + \frac {1}{2}hb_{n})\) updated slope at midpoint
\(d_{n} = f (t_{n} +h, y_{n} +hc_{n})\) slope at right endpoint

 

\(a_{0} = f (t_{0}, y_{0}) = f (0, 1) = e^{2·0} −1 = 0\)
\(b_{0} = f (t_{0} +\frac { h}{2}, y_{0} + \frac {ha_{0}}{2})= f (0.05,1+0.05 · 0) = f (0.05, 1)\)
\(= e^{2·0.05} −1 = 0.105170918\)
\(c_{0} = f (t_{0} + \frac {h}{2}, y_{0} + \frac {hb_{0}}{2})= f (0.05,1+0.05 · 0.105170918)\)
\(= f (0.05,1.005258546) = e^{2·0.05} −1.005258546 =0.099912372\)
\(d_{0} = f (t_{1}, y_{0} +hc_{0}) = f (0.1,1+0.1 · 0.099912372) = f (0.1,1.009991237)\)
\(= e^{2·0.1} −1.009991237 = 0.211411521\)
