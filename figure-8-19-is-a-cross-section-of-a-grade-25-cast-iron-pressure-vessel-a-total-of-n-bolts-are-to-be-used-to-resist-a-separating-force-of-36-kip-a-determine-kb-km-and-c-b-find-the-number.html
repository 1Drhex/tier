\(k_m=\frac{0.5774 \pi Ed}{2 \ln\left(5 \frac{0.5774l+0.5d}{0.5774l+2.5d}\right)}\)

is

\(k_m=\frac{0.5774 \pi Ed}{2 \ln\left(5 \frac{0.5774l+0.5d}{0.5774l+2.5d}\right)}=\frac{0.5774\pi \left(14\right)\left(0.625\right)}{2\ln\left[5\frac{0.5774\left(1.5\right)+0.5\left(0.625\right) }{0.5774\left(1.5\right)+2.5\left(0.625\right) } \right]}=8.95 \ Mlbf/in\)

Then, using Eqs. (8–31)

\(f_i=\begin{cases} 0.75F_P &for \ nonpermanent \ connections \ , \ reused \ fasteners \\0.90F_P &for \ permanent \ connections\end{cases} \)
and (8–32)\(F_P=A_tS_P\), we find the recommended preload to be

For N bolts, Eq. (8–29) \(n_L=\frac{S_PA_t-F_i}{CP}\)
can be written \(n_L=\frac{S_PA_t-F_i}{C\left({p_{total}}/{N}\right) } \)
\(N=\frac{Cn_LP_{total}}{S_PA_t-F_i} =\frac{0.368\left(2\right)\left(36\right) }{85\left(0.226\right)-14.4 } =5.52\)
Six bolts should be used to provide the specified load factor.
(c) With six bolts, the load factor actually realized is
\(n_L=\frac{85\left(0.226\right) -14.4}{0.368\left({36}/{6}\right) } =2.18\)
From Eq. (8–28),the yielding factor of safety is

\(n_P=\frac{S_PA_t}{CP+F_i} \) (8–28)
\(n_P=\frac{S_PA_t}{C\left({P_{total}}/{N}\right) +F_i} =\frac{52\left(0.226\right) }{0.368\left({36}/{6}\right)+14.4 } =1.16\)
From Eq. (8–30),the load factor guarding against joint separation is
\(n_0=\frac{F_i}{PP\left(1-C\right) }\) (8–30)

\(n_0=\frac{F_i}{\left({P_{total}}/{N}\right)\left(1-C\right) } =\frac{14.4}{\left({36}/{6}\right)-\left(1-0.368\right) } =3.80\)
