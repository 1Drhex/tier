Guest Question.

A 22-foot-tall glu-lam column is required to support a roof load (including snow) of 40 k. Assuming \( 8 \frac{3}{4 }^{ \prime \prime }\) in one dimension (to match the beam width above), determine the minimum column size if the top and bottom are pin supported.
Select from the following sizes:

\( 8 \frac{3}{4 }^{ \prime \prime }\) × 9″ (A = 78.75 in.² )

\(8 \frac{3}{4}^{\prime \prime} \times 1010 \frac{1}{2}^{\prime \prime}\left(A=91.88 in .^{2}\right)\)

\( 8 \frac{3}{4 }^{ \prime \prime }\)× 12″ (A = 105.0 in.² )

 

 

Admin
Solution

Glu-lam column: \(\left(F_{ c }=1,650 psi , E=1.8 \times 10^{6} psi \right)\)

Try \(8 \frac{3}{4}^{\prime \prime} \times 10 \frac{1}{2}^{\prime \prime}\)

 

\(8 \frac{3}{4}^{\prime \prime} \times 10 \frac{1}{2}^{\prime \prime}\left(A=91.9 in .^{2}\right)\)

 

\(\frac{\ell_{ e }}{d_{\min }}=\frac{22 \times 12}{8.75}=30.2<50 \)

 

(max. slenderness ratio)

\(F_{ cE }=\frac{0.418 E}{\left(\ell_{ e } / d\right)^{2}}=\frac{0.418\left(1.8 \times 10^{6}\right)}{(30.2)^{2}}=825 psi\)

 

\(F_{ c }^{*} \cong F_{ c } C_{ D }=1,650 psi (1.15)=1,898 psi\)

 

\(\frac{F_{ cE }}{F_{ c }^{*}}=\frac{825}{1,898}=0.43\)

 

 

From Table 9.3, \(C_{ p }=0.403\)

 

 

\(F_{c}^{\prime}=F_{c}^{*}C_{p}=1,898(.403)=765 psi\)

 

\(\begin{aligned}P_{ a } &=F_{ c }^{\prime} \times A=765 psi \left(91.9 in .^{2}\right)=70,300 \# \\&>40,000 \#\end{aligned}\)

 

Cycle again, trying a smaller, more economical section.

 

 

Try \(8 \frac{3}{4}^{\prime \prime} \times 9^{\prime \prime}\).

 

\(8 \frac{3}{4}^{\prime \prime} \times 9^{\prime \prime}\left(A=78.8 in .^{2}\right)\)

 

Because the critical dimension is still \( 8 \frac{3}{4}^{\prime \prime}\) , the values for \(F_{ cE }, F^{*}_{ c } \) , and \(F_{ c }^{\prime}\) all remain the same as in trial 1. The only change that affects the capability of the column is the available cross-sectional area.

 

\(\therefore P_{ a }=F_{ c }^{\prime} A=(765 psi )\left(78.8 in .^{2}\right)=60,300 \#\)

 

\(P_{ a }=60.3 k >40 k\)

 

\(\text { Use } 8 \frac{3}{4}^{\prime \prime} \times 9^{\prime \prime} \) glu-lam section.
