Guest Question.

Using the partial-fraction method, find the inverse z transforms of

(a) \(X(z)=\frac{z}{z^{2}+z+\frac{1}{2}}\)

(b) \(X(z)=\frac{z}{\left(z-\frac{1}{2}\right)\left(z-\frac{1}{4}\right)}\)

Admin
Solution

(a) On expanding X(z)/z into partial fractions as in Eq. (3.35), we get

 

\(X(z)=R_{0}+\sum_{i=1}^{P} \frac{R_{i} z}{z-p_{i}}\) (3.35)

 

\(\frac{X(z)}{z}=\frac{1}{z^{2}+z+\frac{1}{2}}=\frac{1}{\left(z-p_{1}\right)\left(z-p_{2}\right)}=\frac{R_{1}}{z-p_{1}}+\frac{R_{2}}{z-p_{2}}\) (3.37)

 

where

 

\(p_{1}=\frac{e^{j 3 \pi / 4}}{\sqrt{2}} \quad \text { and } \quad p_{2}=\frac{e^{-j 3 \pi / 4}}{\sqrt{2}}\)

 

Thus we obtain

 

\(R_{1}= R e s _{z=p_{1}}\left[\frac{X(z)}{z}\right]=-j \quad \text { and } \quad R_{2}= R e s _{z=p_{2}}\left[\frac{X(z)}{z}\right]=j\)

 

i.e., complex-conjugate poles give complex-conjugate residues, and so Eq. (3.37) gives

 

\(X(z)=\frac{-j z}{z-p_{1}}+\frac{j z}{z-p_{2}}\)

 

From Table 3.2, we now obtain

 

\(x(n T)=u(n T)\left(-j p_{1}^{n}+j p_{2}^{n}\right)\)

 

\(=\left(\frac{1}{2}\right)^{n / 2} u(n T) \frac{1}{j}\left(e^{j 3 \pi n / 4}-e^{-j 3 \pi n / 4}\right)\)

 

\(=2\left(\frac{1}{2}\right)^{n / 2} u(n T) \sin \frac{3 \pi n}{4}\)

 

Alternatively, we can expand X(z) into partial fractions using Eq. (3.36) as shown in part (b).

 

\(X(z)=R_{0}+\sum_{i=1}^{P} \frac{R_{i}}{z-p_{i}}\) (3.36)

 

(b) X(z) can be expressed as

 

\(X(z)=\frac{z}{\left(z-\frac{1}{2}\right)\left(z-\frac{1}{4}\right)}=R_{0}+\frac{R_{1}}{z-\frac{1}{2}}+\frac{R_{2}}{z-\frac{1}{4}}\) (3.38)

 

where

 

\(R_{0}=\lim _{z \rightarrow \infty} X(z)=\lim _{z \rightarrow \infty} \frac{z}{\left(z-\frac{1}{2}\right)\left(z-\frac{1}{4}\right)}\)

 

\(=\lim _{z \rightarrow \infty} \frac{1}{z}=0\)

 

\(R_{1}=\operatorname{Res}_{z=\frac{1}{2}} X(z)=\left.\frac{z}{\left(z-\frac{1}{4}\right)}\right|_{z=\frac{1}{2}}=2\)

 

\(R_{2}=\operatorname{Res}_{z=\frac{1}{4}} X(z)=\left.\frac{z}{\left(z-\frac{1}{2}\right)}\right|_{z=\frac{1}{4}}=-1\)

 

Hence Eq. (3.38) gives

 

\(X(z)=\frac{2}{z-\frac{1}{2}}+\frac{-1}{z-\frac{1}{4}}\)

 

and from Table 3.2

 

\(x(n T)=4 u(n T-T)\left[\left(\frac{1}{2}\right)^{n}-\left(\frac{1}{4}\right)^{n}\right]\)
