Guest Question.

A toroid having a cross section of rectangular shape is defined by the following surfaces: the cylinders \(\rho\) = 2 cm and \(\rho\) = 3 cm, and the planes z = 1 cm and z = 2.5 cm. The toroid carries a surface current density of -50\({ a }_{ z }\) A/m on the surface \(\rho\) = 3 cm. Find H at the point P\((\rho, \phi, z): (a) { P }_{ A }\) (1.5 cm, 0, 2 cm); (b) \({ P }_{ B }\) (2.1 cm, 0, 2 cm); (c) \({ P }_{ C } (2.7 cm, \pi /2, 2 cm)\); (d) \((3.5 cm, \pi /2, 2 cm)\).

Admin
Solution

Again, magnetic ﬁeld exists only inside the toroid cross section, and is given by \(H=\frac { { I }_{ encl } }{ 2\pi \rho } { a }_{ \phi }(2<\rho <3), (1< z <2.5)\) cm.
where \({ I }_{ encl }\) is found from the given current density: On the outer radius, the current is
\({ I }_{ outer }=-50(2\pi \times 3\times { 10 }^{ -2 }) = -3\pi A\)
This current is directed along negative z, which means that the current on the inner radius \((\rho = 2)\) is directed along positive z. Inner and outer currents have the same magnitude. It is the inner current that is enclosed by the circular integration path in \({ a }_{ \phi }\) within the toroid that is used in Ampere’s law. So \({ I }_{ encl } =+3\pi\) A. We can now proceed with what is requested:

a)The radius, \(\rho\) =1.5 cm, lies outside the cross section, and so \({ H }_{ A }\) =0
b)This point does lie inside the cross section, and the \(\phi \) and z values do not matter. We ﬁnd
\({ H }_{ B }=\frac { { I }_{ encl } }{ 2\pi \rho } { a }_{ \phi }=\frac { 3{ a }_{ \phi } }{ 2(2.1\times { 10 }^{ -2 }) } =71.4{ a }_{ \phi }A/m\)
c)again, \(\phi\) and z values make no difference, so
\({ H }_{ C }=\frac { 3{ a }_{ \phi } }{ 2(2.7\times { 10 }^{ -2 }) } =55.6{ a }_{ \phi }A/m\)
d)This point lies outside the cross section, and so \({ H }_{ D } =0\).
