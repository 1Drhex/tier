Guest Question.

Using the method of virtual work, determine the input crank torque, T12, required for equilibrium for the four-bar linkage subjected to external load P as shown in Fig. 11.5a.

ASSUMPTIONS

Friction effects and weights of the links can be neglected.

Admin
Solution

From , we recall that the position-difference vector from O4 to Q and the applied force at Q, respectively, are

\(R _{Q O_{4}}=1.84 \hat{ i }+4.65 \hat{ j } \text { in }\) (1)

 

\(P =120 lb \angle 220^{\circ}=-91.93 \hat{ i }-77.13 \hat{ j } lb\) (2)

 

Figure 11.21 presents a scale diagram of the system at the posture in question. If a small virtual displacement dθ2 is given to the input crank 2, this results in a small displacement dθ4 of the output crank 4, along with a small displacement \(d R _{Q O_{4}}\) of point Q, as shown.

Assuming that these virtual displacements are small and happen in a short time increment, dt, the velocity difference equation, Eq. (3.3), can be multiplied by dt to yield the relationship between \(d R _{Q O_{4}}\) and dθ4 , as follows:

 

\(V _{P Q}=\frac{d R _{P Q}}{d t}= \omega \times R _{P Q}\) (3.3)

 

\(\begin{aligned}d R _{Q O_{4}} &=d \theta_{4} \hat{ k } \times R _{Q O_{4}}=\left(d \theta_{4} \hat{ k } rad \right) \times(1.84 \hat{ i }+4.65 \hat{ j } \text { in }) \\&=(-4.65 \hat{ i }+1.84 \hat{ j } \text { in }) d \theta_{4}\end{aligned}\) (3)

Since the input torque, T12, and the applied force, P, are the only forces doing work during the chosen displacement, Eq. (11.22) can be written for this problem as

dU = dW. (11.22)

\(d U=T_{12} \hat{ k } \cdot d \theta_{2} \hat{ k }+ P \cdot d R _{Q O_{4}}=0\)

Substituting Eqs. (2) and (3) into this equation gives

\(T_{12} d \theta_{2}+(-91.93 \hat{ i }-77.13 \hat{ j } lb ) \cdot(-4.65 \hat{ i }+1.84 \hat{ j } \text { in }) d \theta_{4}=0\)

Then, dividing by dθ2, rearranging, and recognizing the first-order kinematic coefficient, we find

\(T_{12}=(-285.6 in \cdot lb ) \frac{d \theta_{4}}{d \theta_{2}}=-285.6 \theta_{4}^{\prime} in \cdot lb\) (4)

 

By dividing the numerator and denominator of this angular displacement ratio by dt, we recognize that this is equal to the angular-velocity ratio ω4/ω2, which can be determined from the angular velocity ratio theorem of Eq. (3.28).* Using the location of the instant center I24 and measurements from Fig. 11.21, we find

\(V _{P}=\left(x_{P}^{\prime} \hat{ i }+y_{P}^{\prime} \hat{ j }\right) \dot{\psi}\) (3.28)

 

\(\theta_{4}^{\prime}=\frac{d \theta_{4}}{d \theta_{2}}=\frac{\omega_{4}}{\omega_{2}}=\frac{R_{I_{24} I_{12}}}{R_{I_{24} I_{14}}}=\frac{14.65 in }{22.65 in }=0.6468 rad / rad\)

 

Finally, substituting this result into Eq. (4) gives the input torque,

\(T _{12}=(-285.6 \hat{ k } \text { in } \cdot lb )(0.6468 rad / rad )=-184.7 \hat{ k } \text { in } \cdot lb\)

* This technique of treating small displacement ratios as velocity ratios or first-order kinematic coefficients can often be extremely helpful, because, as we remember from Sec. 3.8, velocity relationships lead to linear equations rather than the nonlinear relations implied by position or displacement. Velocity polygons and instant-center methods are also helpful.
