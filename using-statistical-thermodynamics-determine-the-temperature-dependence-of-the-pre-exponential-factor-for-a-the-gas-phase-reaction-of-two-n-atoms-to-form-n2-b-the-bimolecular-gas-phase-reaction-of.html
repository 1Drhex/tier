Guest Question.

Using statistical thermodynamics, determine the temperature dependence of the pre-exponential factor for:
a) the gas-phase reaction of two N atoms to form \(N_{2}\).
b) the bimolecular gas-phase reaction of \(H_{2}O\)+ CO to form \(H_{2}\) and \( CO_{2}\).
c) the bimolecular gas-phase reaction of \(H_{2}\) + \(Cl_{2}\) to form HCl.

Admin
Solution

a) \(r=v \left[X^{+} \right] = k_{B}T/h \frac{Q_{N-N} }{Q^{2}_{N} } e^{-E/RT} \left[N \cdot \right]^{2} \)

pre-exponential factor = \(k_{\circ } = \left(k_{B} T/h\right) \frac{f^{3}_{tr} f^{2}_{rot} }{f^{6}_{tr} } \) because a N atom has no rotational or vibrational

modes. Therefore, \(k_{\circ }= \left(k_{B} T/h\right) \frac{\left(aT\right)^{3/2}\left(bT\right) }{\left(cT\right)^{3} } \Rightarrow k_{\circ } \ \alpha T^{1/2}\)

b) \(Q_{CO}= f^{3}_{tr} \ f^{2}_{rot} \ f_{vib} \) and \(Q_{H_{2}O }= f^{3}_{tr} \ f^{2}_{rot} \ f^{3}_{vib}\)

\(Q_{COOH_{2}}\) has \(3\times 5=15\)modes, with 15 – 7 = 8 degrees of vibrational freedom; therefore

\(k_{\circ }=\left(k_{B} T/h\right) \frac{f^{3}_{tr} \ f^{3}_{rot} \ f^{8}_{vib}}{f^{6}_{tr} \ f^{5}_{rot} \ f^{4}_{vib}} \cong \left(k_{B} T/h\right)\frac{1}{f^{3}_{tr} \ f^{2}_{rot}} \) because \(f_{vib}\cong 1\)

so \(k_{\circ } \ \alpha \left(k_{B} T/h\right)\frac{1}{\left(aT\right)^{3/2} \left(bT\right) } \) or \( k_{\circ } \ \alpha \ T^{-3/2} \)

c) \(Q_{H_{2} }\cong Q_{Cl_{2} } = f^{3}_{tr} \ f^{2}_{rot} \ f_{vib}\) ;assume non-linear activated complex, so degrees of

vibrational freedom = 3N – 7 = 5, and \(Q^{+}= Q_{H_{2} Cl_{2}}= f^{3}_{tr} \ f^{2}_{rot} \ f^{5}_{vib} \) ‘ Therefore,

\(k_{\circ }= \left(k_{B} T/h\right)\frac{f^{3}_{tr} \ f^{3}_{rot} \ f^{5}_{vib}}{f^{6}_{tr} f^{4}_{rot} f^{2}_{vib}} = \left(k_{B} T/h\right)\frac{1}{f^{3}_{tr} \ f_{rot}} \) because \(f_{vib} \cong 1 , so \ \ k_{\circ } \ \alpha \left(k_{B} T/h\right) \frac{1}{\left(aT\right)^{3/2} \left(bT\right)^{1/2} } \)

and \( k_{\circ } \ \alpha T^{-1} \)
