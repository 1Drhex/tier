Guest Question.

A point M moves along a half circle. The projection of its motion on the diameter \(AB\) is an uniform motion with the speed \(c\).

Determine

a) the speed \(v\left(\varphi \right)\) and the magnitude \(a\left(\varphi \right)\) of the acceleration,

b) the angle between the acceleration vector and the diameter \(AB\).

Admin
Solution

a) From the condition \(v=\sin \varphi =c\) follows

\(\underline{\underline{ v\left(\varphi \right)=\frac{c}{\sin \varphi } } }\) .

The acceleraton components \(a_{t} \) and \( a_{n}\) are determined with \(r\dot{\varphi } =v\) as

\(a_{t}=\frac{dv}{dt} =\frac{dv}{d\varphi }\dot{\varphi } =-\frac{c}{\sin ^{2}\varphi } \cos \varphi \frac{v}{r}=-\frac{c^{2} }{r}\frac{\cos \varphi }{\sin ^{3} \varphi } \) ,

\(a_{n}= \frac{v^{2} }{r} =\frac{c^{2} }{r\sin ^{2}\varphi } \) .

This finally leads to

\(\underline{\underline{a} }=\left|a\right| =\sqrt{a^{2}_{t} +a^{2}_{n} } =\frac{c^{2} }{r} \sqrt{\frac{\cos ^{2}\varphi }{\sin ^{6}\varphi }+\frac{1}{\sin ^{4}\varphi } } \)

 

\(=\frac{c^{2} }{r\sin ^{3}\varphi } \sqrt{\cos ^{2}\varphi +\sin ^{2}\varphi }=\underline{\underline{\frac{c^{2} }{r\sin ^{3}\varphi } } } \) .

b) From the figure can be seen:

\(\tan \psi =\tan \left(\varphi -\alpha \right)=\frac{a_{t} }{a_{n} } =\frac{-\frac{c^{2} }{r } \frac{\cos \varphi }{\sin ^{3}\varphi }}{\frac{c^{2} }{r\sin ^{2}\varphi } } \)

 

\(=-\cot \varphi =+\tan \left(\varphi -\pi / 2\right) \rightsquigarrow \underline{\underline{\alpha =\frac{\pi }{2} } } \) ,

i.e. the acceleration vector is perpendicular to AB.

Remark: the latter result can also be found without any calculation: if a component of \(v\) is constant then there is no acceleration in this direction, i.e. \(a\) is perpendicular to the direction of this component.
