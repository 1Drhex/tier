Guest Question.

Design the connection shown below. The required axial strength in the brace is 80 kips (LRFD) and 52 kips (ASD). The axial force may be either tension or compression. The length of the brace is 6 ft.

Admin
Solution

Material Properties:

 

Brace ASTM A500 Gr. B \(F_{y}=46 ksi \quad F_{u}=58 ksi\)

Plate ASTM A36 \(F_{y p}=36 ksi \quad F_{u}=58 ksi\)

Manual
Table 2-3

Geometric Properties:

 

Brace HSS3 1/2×3 1/2×4 \(A_{g}=2.91 \text { in. }^{2}\) r = 1.32 in. t = 0.233 in.
Plate t = 0.375 in.

Manual
Table 1-12

Obtain the available axial compression strength of the brace from Manual Table 4-4

 

\(\begin{array}{l}K=1.0 \\L_{b}=6.00 ft\end{array}\)

 

LRFD ASD
\(\phi_{c} P_{n}=98.4 \text { kips }\)

98.4 kips > 80.0 kips o.k.

 \(P_{n} / \Omega_{c}=65.4 \text { kips }\)

65.4 kips > 52.0 kips o.k.

Manual
Table 4-4

Obtain the available tension yielding strength of the brace from Manual Table 5-5

LRFD ASD
\(\phi_{t} P_{n}=120 \text { kips }\)

120 kips > 80.0 kips o.k.

 \(P_{n} / \Omega_{t}=80.2 \text { kips }\)

80.2 kips > 52.0 kips o.k.

Manual
Table 5-5

Calculate the tensile rupture strength of the brace

\(A_{n}=A_{g}-2\left(t_{w}\right)( slot )=2.91-2(0.233)(3 / 8+1 / 16)=2.71 in .{ }^{4}\)

 

\(\bar{x}=\frac{B^{2}+2 B H}{4(B+H)}=\frac{(31 / 2 in .)^{2}+2(31 / 2 in .)(31 / 2 in .)}{4(31 / 2 in +31 / 2 in .)}=1.31 in\)

Table D3.1

8 3/4 in. of overlap occurs. Try four 3/16-in. fillet welds, each 7-in. long

 

\(U=1-\frac{\bar{x}}{l}=1-\frac{1.31 in .}{7.00 in .}=0.813\)

 

\(A_{e}=U A_{n}=0.813\left(2.71 in .^{2}\right)=2.20 in .^{2}\)

 

\(P_{n}=F_{u} A_{e}=58 ksi \left(2.20 in .^{2}\right)=128 kips\)

Eqn. D2-2

Calculate the available tension rupture strength

LRFD ASD
\(\begin{array}{l}\phi_{t}=0.75 \\\phi_{t} P_{n}=0.75(128 kips )=95.8 \text { kips } \\95.8 \text { kips }>80.0 \text { kips } \quad \text { o.k. }\end{array}\) \(\begin{array}{l}\Omega_{t}=2.00 \\P_{n} / \Omega_{t}=(128 kips ) / 2.00=63.9 kips \\63.9 \text { kips }>52.0 kips \quad \text { o.k. }\end{array}\)

Calculate available strength of 3/16-in. weld of HSS to plate

LRFD ASD
\(\begin{aligned}\phi R_{n} &=4(7 \text { in. })(3 \text { sixteenths })\left(1.392^{\text {kips } / \text { in }} / \text { sixteenth }\right) \\&=117 \text { kips }>80.0 \text { kips } \quad \text { o.k. }\end{aligned}\) \(\begin{aligned}\frac{R_{n}}{\Omega} &=4(7 \text { in. })(3 \text { sixteenths })\left(0.928^{\text {kips } / \text { in. }} / \text { sixteenth }\right) \\&=78.0 \text { kips }>52.0 \text { kips } \quad \text { o.k. }\end{aligned}\)

Calculate the available strength of the weld of gusset plate to column

The slope of the member relative to the longitudinal weld along the column is 50 degrees.
Therefore the standard values for longitudinal fillet welds can be increased by:
\(\left(1.0+0.50 \sin ^{1.5} \theta\right)=1.34\)

or 34-percent.

Minimum weld is 3/16-in. fillet each side. Therefore:

LRFD ASD
\(\begin{aligned}\phi R_{n} &=2(16.0 \text { in. })(3)(1.392)(1.34) \\&=179 \text { kips }>80.0 \text { kips } \text { o.k. }\end{aligned}\) \(\begin{aligned}R_{n} / \Omega &=2(16.0 in .)(3)(0.928)(1.34) \\&=119 kips >52.0 \text { kips }\end{aligned}\)
