Guest Question.

For the design of a shallow foundation, the following data are given:
Foundation: \(L = 1.5 m, B = 1.0 m, D_{f} = 1.0 m \)
Soil: \( \varnothing = 25^{\circ} \)
\( c = 50 kPa \)
\( y = 17 kN/m^{3} \)
Modulus of deformation \( E_{s} = 1020 kPa \)
Poisson’s ratio \( \mu_{s} = 0.35 \)
Calculate the \( gross q_{ult} \) of the soil. Take the soil compressibility into consideration.

Admin
Solution

Assume incompressibility soil condition; hence,

\( gross q_{ult} = c_{d} N_{c} s_{c} d_{c} + \overline{q} N_{q} s_{q} d_{q} + 0.5 {y}'{B}'N_{y} s_{y} d_{y} \)

 

For \( \varnothing = 25^{\circ}, N_{c} = 20.72, N_{q} = 10.66, N_{y} = 10.88 \) (these values are computed from the Vesic equations of Table 4.2).

Table 4.6:

\( s_{c} = 1 + \frac{10.66}{20.72} \times \frac{1}{1.5} = 1.343 \)

 

\( s_{q} = 1 + \frac{1}{1.5}\tan 25 = 1.311 \)

 

\( s_{y} = 1 – 0.4 \times \frac{1}{1.5} = 0.733 > 0.6, use 0.733\)

 

\( \frac{D}{B} = \frac{1}{1} = 1 \rightarrow k = \frac{D}{B} = 1 \)

 

\( d_{c} = 1 + 0.4k = 1 + 0.4 \times 1 = 1.4 \)

 

\( d_{q} = 1 + 2 \tan \varnothing \left ( 1 – \sin \varnothing \right )^{2}k \)

 

\( = 1 + 2 \tan 25 \left ( 1 – \sin 25 \right )^{2}\left ( 1/1 \right ) = 1.311 \)

 

\( d_{y} = 1 \)

 

\( gross q_{ult} = 50 \times 20.72 \times 1.343 \times 1.4 \)

 

\( + 1 \times 17 \times 10.66 \times 1.311 \times 1.311 \)

 

\( + 0.5 \times 17 \times 1 \times 10.88 \times 0.733 \times 1 \)

 

\( = 1947.89 + 311.47 + 67.79 = 2327 kPa \)

 

The average overburden pressure in the influence zone, which is taken as pressure at the depth \( B/2 \) below the foundation level, is

\( \overline{q} = 1.5 \times 17 = 25.5 kPa \)

Equation :

\( I_{r} = \frac{G_{s}}{c + \overline{q}\tan \varnothing } = \frac{E_{s}}{2 \left ( 1 + \mu \right )} \times \frac{1}{c + \overline{q}\tan \varnothing} = \frac{1020}{2\left ( 1 + 0.35 \right )\left ( 50 + 25.5 \tan 25 \right )} = 6.1 \)

 

Equation :

\( = \frac{1}{2} \left \{ exp \left [ \left ( 3.30 – 0.45 \frac{B}{L} \right ) \cot \left ( 45 – \frac{\varnothing }{2} \right ) \right ] \right \} \)

 

\( I_{r\left ( cr \right )} = \frac{1}{2} \left \{ exp \left [ \left ( 3.30 – 0.45 \frac{1}{1.5} \right ) \cot \left ( 45 – \frac{25 }{2} \right ) \right ] \right \} = 55.53 \)

 

\( I_{r} < I_{r\left ( cr \right )}. \) Thus, the assumption of soil incompressibility is not justified. The compressibility factors of Equations (\( \zeta_{cc} = \zeta_{qc} – \frac{1 – \zeta_{qc} }{N_{c} \tan \varnothing } for \varnothing > 0 \)) and (\( \zeta_{qc} = exp \left \{ \left ( – 4.4 + 0.6 \frac{B}{L} \right ) \tan \varnothing + \left [ \frac{\left ( 3.07 \sin \varnothing \right )\left ( log 2 I_{r} \right )}{1 + \sin \varnothing } \right ]\right \} \)) need to be used in the bearing capacity equation.

 

Equation : \( \zeta_{qc} = exp \left \{ \left ( – 4.4 + 0.6 \frac{B}{L} \right ) \tan \varnothing + \left [ \frac{\left ( 3.07 \sin \varnothing \right )\left ( log 2 I_{r} \right )}{1 + \sin \varnothing } \right ]\right \} \)

 

\( = exp \left \{ \left ( – 4.4 + 0.6 \times \frac{1}{1.5} \right ) \tan 25 + \left [ \frac{\left ( 3.07 \sin 25 \right)\left ( log 12.2 \right )}{1 + \sin 25 } \right ]\right \} = 0.417 \)

 

Equation : \( \zeta_{cc} = \zeta_{qc} – \frac{1 – \zeta_{qc} }{N_{c} \tan \varnothing } for \varnothing > 0 \)

 

\( \zeta_{cc} = 0.417 – \frac{1 – 0.417 }{20.72 \times \tan 25} = 0.357 \)

 

\( \zeta_{yc} = \zeta_{qc} = 0.417 \)

 

\( gross q_{ult} = 1947.89 \left ( 0.357 \right ) + 311.47 \left ( 0.417 \right ) + 67.79 \left ( 0.417 \right ) = 854 kPa \)
