Guest Question.

Linearize the state-variable equations derived in for the system shown in Fig. 3.10 in a small vicinity of a normal operating point determined by the input force, \(F_{1} \left(t\right) =\bar{F} _{1}=const.\)

Admin
Solution

You solve this problem by following the five-step procedure introduced in Section 2.4.

Step 1. Derive the nonlinear model. The following state-variable equations were derived in :

\(\overset{\cdot }{z} _{1} =v_{1} ,\)

 

\(\overset{\cdot }{v} _{1}=- \frac{k_{1} }{m_{1} } z_{1} – \frac{b_{1} }{m_{1} } v_{1}+ \frac{c}{m_{1} }\left|z_{2}- z_{1} \right| \left(z_{2}- z_{1} \right) + \frac{1}{m_{1} } F_{1} \left(t\right) + g ,\)

 

\(\overset{\cdot }{z} _{2} ={v} _{2} ,\)

 

\(\overset{\cdot }{v} _{2} =- \frac{c}{m_{2} } \left|z_{2} – z_{1} \right| \left(z_{2} – z_{1} \right) + g.\)

Step 2. Determine the normal operating point. The normal operating point is determined by the constant input force, \(F_{1} \left(t\right) =\bar{F} _{1} =const\), and the corresponding unknown values of the four state variables, \(z_{1} =\bar{z} _{1}=const,v_{1} =\bar{v} _{1} =const,z_{2} =\bar{z} _{2} =const,\) and \(v_{2} =\bar{v} _{2} =const.\) The unknown normal operating–point values of the state variables can be found by their substitution into the state-variable equations, which yields

\(0=\bar{v} _{1},\)

 

\(0=- k_{1} \bar{z} _{1} + c\left|\bar{z} _{2}- \bar{z} _{1} \right| \left(\bar{z} _{2}- \bar{z} _{1} \right) + \bar{F} _{1} + m_{1} g,\)

 

\(0=\bar{v} _{2},\)

 

\(0=- c\left|\bar{z} _{2}- \bar{z} _{1} \right| \left(\bar{z} _{2}- \bar{z} _{1} \right)+ m_{2} g\)

The preceding equations describe the system at the normal operating point established in the system by the constant input force, \(\bar{F} _{1}\). It can be seen that at the normal operating point the system is at rest (the velocities of both masses are zero), whereas one can find the displacements of the two masses by solving the preceding equations and assuming that c > 0 to obtain

\(\bar{z} _{1} =\frac{1}{k_{1} } \left(\bar{F} _{1}+ m_{1}g+ m_{2} g \right),\)

 

\(\bar{z} _{2} =\frac{1}{k_{1} } \left(\bar{F} _{1}+ m_{1}g+ m_{2} g \right)+ \sqrt{\frac{m_{2} g}{c} } .\)

Step 3. Introduce incremental variables. Substitute \(z_{1} =\bar{z}_{1} + \hat{z} _{1}, z_{2} =\bar{z}_{2} + \hat{z} _{2},v_{1} =\bar{v}_{1} + \hat{v} _{1}, v_{2} =\bar{v}_{2} + \hat{v} _{2},\) and \(F_{1} \left(t\right) =\bar{F} _{1} + \hat{F} _{1}\) into the original state-variable equations:

\(\overset{\cdot }{\bar{z} } _{1} + \overset{\cdot }{\hat{z} } _{1}=\bar{v} _{1} + \hat{v} _{1} ,\)

 

\(\overset{\cdot }{\bar{v} } _{1} + \overset{\cdot }{\hat{v} } _{1}=- \frac{k_{1} }{m_{1} } \left(\bar{z} _{1}+ \hat{z}_{1} \right) – \frac{b_{1} }{m_{1} } – \left(\bar{v} _{1}+ \hat{v} _{1} \right) + \frac{1}{m_{1} } f_{NL} \left(\bar{z} _{21} + \hat{z} _{21} \right) + \frac{1}{m_{1} } \left(\bar{F} _{1}+ \hat{F} _{1} \right) + g ,\)

 

\(\overset{\cdot }{\bar{z} } _{2} + \overset{\cdot }{\hat{z} } _{2}=\bar{v} _{2} + \hat{v} _{2} ,\)

 

\(\overset{\cdot }{\bar{v} } _{2} + \overset{\cdot }{\hat{v} } _{2}=- \frac{1}{m_{2} }F_{NLS} \left(\bar{z} _{21}+ \hat{z} _{21} \right) + g,\)

where \(z_{21} =z_{2} – z_{1}\) and \(f_{NLS} \left(z_{21} \right)\) is the NLS characteristic. Substituting zeros for terms involving \(\bar{v} _{1} ,\bar{v} _{2}\) and derivatives of all constant components yields

\(\overset{\cdot }{\hat{z} } _{1}=\hat{v} _{2},\)

 

\( \overset{\cdot }{\hat{v} } _{1}=- \frac{k_{1} }{m_{1} } \left(\bar{z} _{1}+ \hat{z}_{1} \right) – \frac{b_{1} }{m_{1} }\hat{v} _{1} + \frac{1}{m_{1} } f_{NL} \left(\bar{z} _{21} + \hat{z} _{21} \right) + \frac{1}{m_{1} } \left(\bar{F} _{1}+ \hat{F} _{1} \right) + g,\)

 

\(\overset{\cdot }{\hat{z} } _{2}=\hat{v} _{2},\)

 

\( \overset{\cdot }{\hat{v} } _{2}=- \frac{1}{m_{2} }f_{NLS} \left(\bar{z} _{21}+ \hat{z} _{21} \right) + g,\)

Step 4. Linearize the nonlinear terms. Expand \(f_{NLS} \left(\bar{z} _{21}+ \hat{z} _{21} \right)\) into Taylor series, neglecting second- and higher-order terms:

\(f_{NLS} \left(\bar{z} _{21}+ \hat{z} _{21} \right) \approx f_{NLS} \left(\bar{z} _{21} \right) + \hat{z} _{21} \left(\frac{df_{NLS} }{dz_{21} } \right)_{\bar{z} _{21} } =c\left|\bar{z} _{2} – \bar{z} _{1} \right| \left(\bar{z} _{2}- \bar{z} _{1} \right) + \left(\hat{z} _{2} – \hat{z} _{1} \right) 2c\left|\bar{z} _{2}- \bar{z} _{1} \right|\) .

Step 5. Arrange the linearized equations into a final form. Substitute the linearized expression obtained in Step 4 into the equations derived in Step 3:

\(\overset{\cdot }{\hat{z} } _{1}=\hat{v} _{1} ,\)

 

\(\overset{\cdot }{\hat{v} } _{1}=- \frac{k_{1} }{m_{1} } \left(\bar{z} _{1}+ \hat{z}_{1} \right) – \frac{b_{1} }{m_{1} }\hat{v} _{1} + \frac{c}{m_{1} } \left[\left|\bar{z} _{2}- \bar{z} _{1} \right|\left(\bar{z} _{2} – \bar{z} _{1} \right) \right] + \frac{2c}{m_{1} } \left|\bar{z} _{2}- \bar{z} _{1}\right|\left(\hat{z} _{2}- \hat{z} _{1} \right) + \frac{1}{m_{1} } \left(\bar{F} _{1}+ \hat{F} _{1} \right) + g ,\)

 

\(\overset{\cdot }{\hat{z} } _{2}=\hat{v} _{2} ,\)

 

\( \overset{\cdot }{\hat{v} } _{2}=- \frac{c}{m_{2} } \left|\bar{z} _{2} – \bar{z} _{1} \right| \left(\bar{z} _{2} – \bar{z} _{1} \right) – \frac{2c}{m_{2} } \left|\bar{z} _{2} – \bar{z} _{1} \right|\left(\hat{z} _{2} – \hat{z} _{1} \right) + g .\)

Referring to the equations for the normal operating point derived in Step 2, it can be seen that all constant terms in the precedings equations cancel out to give the linearized state-variable equations:

\(\overset{\cdot }{\hat{z} } _{1}=\hat{v} _{1} ,\)

 

\(\overset{\cdot }{\hat{v} } _{1}=- \frac{k_{1} }{m_{1} } \hat{z} _{1}- \frac{b_{1} }{m_{1} } \hat{v} _{1} + \frac{2c}{m_{1} }\left|\bar{z} _{2}- \bar{z} _{1} \right|\left(\hat{z} _{2} – \hat{z} _{1} \right) + \frac{1}{m_{1} } \hat{F} _{1} ,\)

 

\(\overset{\cdot }{\hat{z} } _{2}=\hat{v} _{2} ,\)

 

\( \overset{\cdot }{\hat{v} } _{2}=- \frac{2c}{m_{2} } \left|\bar{z} _{2} – \bar{z} _{1} \right| \left(\bar{z} _{2} – \bar{z} _{1} \right).\)

An incremental spring stiffness that approximates the stiffness of the NLS in a small vicinity of the normal operating point is defined as

\(k_{inc} =2c\left|\bar{z} _{2}- \bar{z} _{1} \right| .\)

The linearized state-variable equations can now be written in a more compact form:

\(\overset{\cdot }{\hat{z} } _{1}=\hat{v} _{1} ,\)

 

\(\overset{\cdot }{\hat{v} } _{1}=- \frac{\left(k_{1}+ k_{inc} \right) }{m_{1} }\hat{z} _{1}- \frac{b_{1} }{m_{1} }\hat{v} _{1}+ \frac{k_{inc} }{m_{1} }\hat{z} _{2} + \frac{1}{m_{1} }\hat{F} _{1} ,\)

 

\(\overset{\cdot }{\hat{z} } _{2}=\hat{v} _{2} ,\)

 

\( \overset{\cdot }{\hat{v} } _{2}=\frac{k_{inc} }{m_{2} } \hat{z} _{1} – \frac{k_{inc} }{m_{2} } \hat{z} _{2}\)

The linearized equations can also be arranged into a matrix form:

\(\left [ \begin{matrix} \overset{\cdot }{\hat{z} } _{1} \\ \overset{\cdot }{\hat{v} } _{1} \\ \overset{\cdot }{\hat{z} } _{2} \\ \overset{\cdot }{\hat{v} } _{1} \end{matrix} \right ] =\left [ \begin{matrix} 0 & 1 & 0 & 0 \\ {- \left(k_{1}+ k_{inc} \right) }/{m_{1} } & {- b_{1} }/{m_{1} } & {k_{inc} }/{m_{1} } & 0 \\ 0 & 0 & 0 & 1 \\ {k_{inc} }/{m_{2} } & 0 & -{k_{inc} }/{m_{2} } & 0 \end{matrix} \right ] \left [ \begin{matrix} \hat{z} _{1} \\ \hat{v} _{1} \\ \hat{z} _{2} \\ \hat{v} _{2} \end{matrix} \right ] + \left [ \begin{matrix} 0 \\ {1}/{m_{1} } \\ 0 \\ 0 \end{matrix} \right ] \hat{F} _{1} .\)
