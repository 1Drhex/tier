Guest Question.

At point P(−3, 4, 5), express the vector that extends from P to Q(2, 0,−1) in; a) rectangular coordinates; b) cylindrical coordinates ; c) spherical coordinates ; d) Show that each of these vectors has the same magnitude

Admin
Solution

a) \({ R }_{ PQ }=Q-P=5{ a }_{ x }-4{ a }_{ y }-6{ a }_{ z } Then \left| { R }_{ PQ } \right| =\sqrt { 25+16+36 } =8.8\)

b)At P , \(\rho =5,\phi =\tan ^{ -1 }{ (4/-3) } =-{ 53 }^{ o } , z=5\) . Now

\({ R }_{ PQ }.{ a }_{ \rho }=(5{ a }_{ x }-4{ a }_{ y }-6{ a }_{ z }).{ a }_{ \rho }=5\cos { \phi } -4\sin { \phi } =6.20\) \({ R }_{ PQ }.{ a }_{ \phi }=(5{ a }_{ x }-4{ a }_{ y }-6{ a }_{ z }).{ a }_{ \phi }=-5\sin { \phi } -4\cos { \phi } =1.60\)

Thus

\({ R }_{ PQ }=6.20{ a }_{ \rho }+1.60{ a }_{ \phi }-6{ a }_{ z }\)

and \(\left| { R }_{ PQ } \right| =\sqrt { { 6.20 }^{ 2 }+{ 1.60 }^{ 2 }+{ 6 }^{ 2 } } =8.8\)

c) At P ,\(r=\sqrt { 9+16+25 } =\sqrt { 50 } =7.07 , \theta =\cos ^{ -1 }{ (5/7.07) } ={ 45 }^{ o }\)

and \(\phi \quad =\quad \tan ^{ -1 }{ (4/-3) } =-{ 53.1 }^{ o }\)

\({ R }_{ PQ }.{ a }_{ r }=(5{ a }_{ x }-4{ a }_{ y }-6{ a }_{ z }).{ a }_{ r }=5\sin { \theta } \cos { \phi } -4\sin { \theta } \sin { \phi } -6\cos { \theta } =0.14\) \({ R }_{ PQ }.{ a }_{\theta }=(5{ a }_{ x }-4{ a }_{ y }-6{ a }_{ z }).{ a }_{ \theta }=5\cos { \theta } \cos { \phi } -4\cos { \theta } \sin { \phi } -(-6)\sin { \theta } =8.62\) \({ R }_{ PQ }.{ a }_{ \phi }=(5{ a }_{ x }-4{ a }_{ y }-6{ a }_{ z }).{ a }_{ \phi }=-5\sin { \phi } -4\cos { \phi } =1.60\)

Thus \({ R }_{ PQ }=0.14{ a }_{ r }+8.62{ a }_{ \theta }+1.60{ a }_{ \phi }\)

and \(\left| { R }_{ PQ } \right| =\sqrt { { 0.14 }^{ 2 }+{ 8.62 }^{ 2 }+{ 1.60 }^{ 2 } } =8.8\)

d) Each does, as shown above
