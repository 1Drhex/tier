Guest Question.

An absorbing-emitting medium is contained between two large, parallel, isothemial, black plates at temperatures \(T_{1}\) and \(T_{2}\) = 0.5 \(T_{1}\) , respectively. Determine the steady-state temperature distribution within the medium and the total heat flux between the two plates, if heat is transferred by conduction and radiation. Discuss the influence of the conduction-to-radiation parameter, N, and of the optical thickness of the layer, \(τ_{L}\) .

Admin
Solution

The numerical solution to the governing equation may be found in a number of ways. We will employ here J + 1 equally spaced nodes τ = 0, Δτ , 2Δτ ,…, JΔτ = \(τ_{L}\) with nodal temperatures \(θ_{i}\) (i = 0,1,2,…, J) and simple finite-differencing for the conduction term,

\(\frac{d^{2}\theta }{d\tau ^{2}} \simeq \frac{\theta _{i+1}-2\theta _{i}+ \theta _{i-1}}{\Delta \tau ^{2}} + O(\Delta \tau ^{2})\),

with a truncation error of order \(\Delta \tau ^{2}\). The divergence of the radiative heat flux, equation (21.11 ) \(\frac{d\Psi _{R}}{d\tau } =2\left\{\int_{0}^{\tau }{}\frac{d\theta ^{4}}{d\tau ^{\prime }} (\tau ^{\prime }) E_{2}(\tau -\tau ^{\prime })d\tau ^{\prime } – \int_{\tau }^{\tau _{L}}{} \frac{d\theta ^{4}}{d\tau ^{\prime }} (\tau ^{\prime }) E_{2}(\tau ^{\prime }-\tau )d\tau ^{\prime }\right\}\) , will be calculated by approximating the emissive power, \(θ^{4}\) , by a. spline function, followed by analytical evaluation of the piecewise integrals. In order to obtain the same truncation error as for the conduction term, O(\(\Delta \tau ^{2}\)), the prediction of \(dθ^{4}/d\tau ^{\prime }\) must be accurate to O(\(\Delta \tau\)) [since the piecewise integration decreases the truncation error by O(\(\Delta \tau\))], Thus, for the emissive power a linear spline is sufficient, or

\(\theta ^{4}(\tau )=\theta ^{4}_{i}+B_{i}(\tau -\tau _{i})+ O(\Delta \tau ^{2})=\frac{\theta ^{4}_{i}(\tau _{i+1}-\tau )+\theta ^{4}_{i+1}(\tau -\tau _{i})}{\Delta \tau } +O(\Delta \tau ^{2}),\)

\(\frac{d\theta ^{4}}{d\tau } (\tau )=\frac{\theta ^{4}_{i+1}-\theta ^{4}_{i}}{\Delta \tau } +O(\Delta \tau ),\)

\(\tau _{i}\lt \tau \lt \tau _{i+1} , i= 0,1,2,…, J-1\)

Substituting this into equation (21.11) leads to

\(\left\lgroup\frac{d\Psi _{R}}{d\tau } \right\rgroup _{i} \simeq 2\sum\limits_{j=1}^{i}{} \frac{\theta ^{4}_{j}-\theta ^{4}_{j-1}}{\Delta \tau } \int_{\tau _{j-1}}^{\tau _{j}}{} E_{2}(\tau _{i}-\tau ^{\prime })d\tau ^{\prime }-2 \sum\limits_{j=i+1}^{J}{} \frac{\theta ^{4}_{j}-\theta ^{4}_{j-1}}{\Delta \tau } \int_{\tau _{j-1}}^{\tau _{j}}{} E_{2}(\tau ^{\prime }-\tau _{i})d\tau ^{\prime }\)

\(=\frac{2}{\Delta \tau } \sum\limits_{j=1}^{i}{}(\theta ^{4}_{j}-\theta ^{4}_{j-1}) \left[E_{3}(\tau _{i}-\tau _{j})-E_{3}(\tau _{i}-\tau _{j-1})\right] + \frac{2}{\Delta \tau } \sum\limits_{j=i+1}^{J}{}(\theta ^{4}_{j}-\theta ^{4}_{j-1}) \left[E_{3}(\tau _{j}-\tau _{i})-E_{3}(\tau _{j-1}-\tau _{i})\right]\)

\(=\frac{2}{\Delta \tau } \sum\limits_{j=1}^{J}{}(\theta ^{4}_{j}-\theta ^{4}_{j-1}) \left[E_{3}(\left|i-j\right|\Delta \tau ) -E_{3}(\left|i+1-j\right|\Delta \tau ) \right]\)

Equating both sides of equation (21.5) \(\frac{d^{2}\theta }{d\tau ^{2}}=\frac{1}{4N}\frac{d\Psi _{R}}{d\tau }\) , we find

\(\theta _{i-1}-2\theta _{i}+\theta _{i+1}=\frac{\Delta \tau }{2N} \sum\limits_{j=1}^{J}{}(\theta ^{4}_{j}-\theta ^{4}_{j-1}) \left[E_{3}(\left|i-j\right|\Delta \tau ) -E_{3}(\left|i+1-j\right|\Delta \tau ) \right] , i = 1,2,…,J-1,\)

\(\theta _{0}=1 , \theta _{j}=\theta _{L}\).

If N is relatively large (N > 0.1), heat transfer is dominated by conduction, and the solution proceeds as follows:

1. A temperature profile is guessed (e.g., the linear profile for pure conduction), and the (\(d\Psi _{R}/d\tau )_{i}\) are calculated based on these temperatures.
2. A new temperature profile is determined by inverting the simple tridiagonal matrix for θ.
3. The temperature profile is iterated on, using underrelaxation as necessary (as discussed in the previous example).
If N is small, radiation dominates, and the process should be reversed:
1. A temperature profile is guessed, the conduction contribution is calculated, and an emissive power field is determined by inverting the full matrix for the \(\theta ^{4}_{i}\) on the right-hand side.
2. A new temperature profile is deduced from the emissive powers, etc.

Once the temperature profile is known, the total heat flux follows from equations (21.9) \(\Psi _{R}(\tau )=2\left\{(J_[1]-1)E_{3}(\tau )-(J-{2}-\theta ^{4}_{L})E_{3}(\tau _{L}-\tau )-\int_{0}^{\tau }{}\frac{d\theta ^{4}}{d\tau ^{\prime }} (\tau ^{\prime })E_{3}(\tau -\tau ^{\prime })d\tau ^{\prime }- \int_{\tau }^{\tau_{L} }{}\frac{d\theta ^{4}}{d\tau ^{\prime }} (\tau ^{\prime })E_{3}(\tau ^{\prime }-\tau )d\tau ^{\prime } \right\}\) and (21.12) \(\Psi =\frac{q}{n^{2}\sigma T^{4}_{1}}=-4N\frac{d\theta }{d\tau } +\Psi _{R}= const\) as

\(\Psi _{i}\simeq -\frac{2N}{\Delta \tau } (\theta _{i+1}-\theta _{i-1})-\frac{2}{\Delta \tau } \left\{\sum\limits_{j=1}^{i}{}(\theta ^{4}_{j}-\theta ^{4}_{j-1})\int_{\tau _{j-1}}^{\tau _{j}}{} E_{3}(\tau _{i}-\tau ^{\prime })d\tau ^{\prime }+\sum\limits_{j=i+1}^{i}{}(\theta ^{4}_{j}-\theta ^{4}_{j-1})\int_{\tau _{j-1}}^{\tau _{j}}{} E_{3}(\tau ^{\prime }-\tau _{i})d\tau ^{\prime } \right\}\)

\(= -\frac{2N}{\Delta \tau } (\theta _{i+1}-\theta _{i-1})-\frac{2}{\Delta \tau } \left\{\sum\limits_{j=1}^{i}{}(\theta ^{4}_{j}-\theta ^{4}_{j-1})[ E_{4}(\tau _{i}-\tau _{j })-E_{4}(\tau _{i}-\tau _{j-1})]-\sum\limits_{j=i+1}^{i}{}(\theta ^{4}_{j}-\theta ^{4}_{j-1}) [E_{4}(\tau _{j}-\tau _{i })-E_{4}(\tau _{j-1}-\tau _{i}) ] \right\}\)

\(i=1,2,…,J-1\)

This value for the nondimensional heat flux should be the same for all nodes.
