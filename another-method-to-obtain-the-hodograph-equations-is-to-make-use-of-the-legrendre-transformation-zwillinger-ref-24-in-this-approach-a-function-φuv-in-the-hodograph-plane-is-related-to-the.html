Guest Question.

Another method to obtain the hodograph equations is to make use of the Legrendre transformation, Zwillinger, Ref. 24. In this approach a function, \(\Phi( u , v )\), in the hodograph plane is related to the velocity potential, \(\phi( x , y )\), by

\(\Phi= xl + yv -\phi\)

Use this relation to show that

(a) \(\frac{\partial \Phi}{\partial u }= x \text { and } \frac{\partial \Phi}{\partial v }= y\)

(b) \(\left\{\begin{array}{l}\frac{\partial^{2} \phi}{\partial x ^{2}}=\frac{1}{ J } \frac{\partial^{2} \Phi}{\partial v ^{2}} \\\frac{\partial^{2} \phi}{\partial x \partial y }=-\frac{1}{ J } \frac{\partial^{2} \Phi}{\partial u \partial v } \\\frac{\partial^{2} \phi}{\partial y ^{2}}=\frac{1}{ J } \frac{\partial^{2} \Phi}{\partial u ^{2}}\end{array}\right.\)

where \(J =\left|\begin{array}{ll}\Phi_{ uu } & \Phi_{ uv } \\\Phi_{ uv } & \Phi_{ vv }\end{array}\right|\).

(c) Finally, show that Eq.(12.27) is transformed into the following linear equation

\(\frac{\partial v}{\partial x}-\frac{\partial u}{\partial y}=0\) (12.27)

\(\left(1-\frac{u^{2}}{a^{2}}\right) \frac{\partial^{2} \Phi}{\partial v^{2}}+2 \frac{u v}{a^{2}} \frac{\partial^{2} \Phi}{\partial u \partial v}+\left(1-\frac{v^{2}}{a^{2}}\right) \frac{\partial^{2} \Phi}{\partial u^{2}}=0\)
Admin
Solution

(a) Differentiate \(\Phi=x u+y v-\phi(x, y)\) with respect to u and obtain

 

\(\frac{\partial \Phi}{\partial u }= u \frac{\partial x }{\partial u }+ x + v \frac{\partial y }{\partial u }-\frac{\partial \phi}{\partial x } \frac{\partial x }{\partial u }-\frac{\partial \phi}{\partial y } \frac{\partial y }{\partial u }\)

 

But from the definition of the velocity potential \(\frac{\partial \phi}{\partial x }= u \text { and } \frac{\partial \phi}{\partial y }= v\), so the above reduces to

 

\(\frac{\partial \Phi}{\partial u }= u \frac{\partial x }{\partial u }+ x + v \frac{\partial y }{\partial u }+-\frac{\partial \phi}{\partial x } \frac{\partial x }{\partial u }-\frac{\partial \phi}{\partial y } \frac{\partial y }{\partial u }= u \frac{\partial x }{\partial u }+ x + v \frac{\partial y }{\partial u }+- u \frac{\partial x }{\partial u }- v \frac{\partial y }{\partial u }= x\)

 

Similarly differentiate \(\Phi= xu + yV -\phi( x , y )\) with respect to v and obtain

 

\(\frac{\partial \Phi}{\partial v }= u \frac{\partial x }{\partial v }+ y + v \frac{\partial y }{\partial v }-\frac{\partial \phi}{\partial x } \frac{\partial x }{\partial v }-\frac{\partial \phi}{\partial y } \frac{\partial y }{\partial v }= u \frac{\partial x }{\partial v }+ y + v \frac{\partial y }{\partial v }- u \frac{\partial x }{\partial v }- v \frac{\partial y }{\partial v }= y\)

 

(b) Differentiate \(\partial \Phi / \partial u = x\) first with respect to u and also with respect to v to obtain

 

\(\begin{aligned}&\Phi_{\text {uu }}= x _{ u } \\&\Phi_{ uv }= x _{ v }\end{aligned}\)

 

Similarly using \(\partial \Phi / \partial v = y\) we obtain

 

\(\begin{aligned}&\Phi_{ vu }= y _{ u } \\&\Phi_{ vv }= y _{ v }\end{aligned}\)

 

Now use the chain rule and write

 

\(1=\frac{\partial u }{\partial u }=\frac{\partial\left(\phi_{ x }\right)}{\partial u }=\frac{\partial\left(\phi_{ x }\right)}{\partial x } \frac{\partial x }{\partial u }+\frac{\partial\left(\phi_{ x }\right)}{\partial y } \frac{\partial y }{\partial u }=\phi_{ xx } \Phi_{ uu }+\phi_{ xy } \Phi_{ vu }=1\)

 

Likewise

 

\(0=\frac{\partial u }{\partial v }=\frac{\partial\left(\phi_{ x }\right)}{\partial v }=\frac{\partial\left(\phi_{ x }\right)}{\partial x } \frac{\partial x }{\partial v }+\frac{\partial\left(\phi_{ x }\right)}{\partial y } \frac{\partial y }{\partial v }=\phi_{ xx } \Phi_{ uv }+\phi_{ xy } \Phi_{ vv }=0\)

 

The above pair may be written in matrix-vector form as

 

\(\left[\begin{array}{ll}\Phi_{ uu } & \Phi_{ vu } \\\Phi_{ uv } & \Phi_{ vv }\end{array}\right]\left[\begin{array}{l}\phi_{ xx } \\\phi_{ xy }\end{array}\right]=\left[\begin{array}{l}1 \\0\end{array}\right]\)

 

Solving this for \(\phi_{x x} \text { and } \phi_{x y}\) yields

 

\(\phi_{ xx }=\frac{\left|\begin{array}{cc}1 & \Phi_{ vu } \\0 & \Phi_{ vv }\end{array}\right|}{\left|\begin{array}{ll}\Phi_{ uu } & \Phi_{ vu } \\\Phi_{ uv } & \Phi_{ vv }\end{array}\right|}=\frac{\Phi_{ vv }}{\left|\begin{array}{ll}\Phi_{ uu } & \Phi_{ vu } \\\Phi_{ uv } & \Phi_{ vv }\end{array}\right|}\)

 

\(\phi_{ xy }=\frac{\left|\begin{array}{ll}\Phi_{ uu } & 1 \\\Phi_{ uv } & 0\end{array}\right|}{\left|\begin{array}{ll}\Phi_{ uu } & \Phi_{ vu } \\\Phi_{ uv } & \Phi_{ vv }\end{array}\right|}=\frac{-\Phi_{ uv }}{\left|\begin{array}{ll}\Phi_{ uu } & \Phi_{ vu } \\\Phi_{ uv } & \Phi_{ vv }\end{array}\right|}\)

 

Following the same procedure

 

\(1=\frac{\partial v }{\partial v }=\frac{\partial\left(\phi_{ y }\right)}{\partial v }=\frac{\partial\left(\phi_{ y }\right)}{\partial x } \frac{\partial x }{\partial v }+\frac{\partial\left(\phi_{ y }\right)}{\partial y } \frac{\partial y }{\partial v }=\phi_{ yx } \Phi_{ uv }+\phi_{ yy } \Phi_{ vv }=1\)

 

Likewise

 

\(0=\frac{\partial v }{\partial u }=\frac{\partial\left(\phi_{ x }\right)}{\partial u }=\frac{\partial\left(\phi_{ y }\right)}{\partial x } \frac{\partial x }{\partial u }+\frac{\partial\left(\phi_{ y }\right)}{\partial y } \frac{\partial y }{\partial u }=\phi_{ yx } \Phi_{ uu }+\phi_{ yy } \Phi_{ vu }=0\)

 

The above pair may be written in matrix-vector form as

 

\(\left[\begin{array}{cc}\Phi_{ uu } & \Phi_{ vu } \\\Phi_{ uv } & \Phi_{ vv }\end{array}\right]\left[\begin{array}{c}\phi_{ yx } \\\phi_{ yy }\end{array}\right]=\left[\begin{array}{l}0 \\1\end{array}\right]\)

 

Solving this for \(\phi_{ yx } \text { and } \phi_{ yy }\) yields

 

\(\phi_{ yy }==\frac{\Phi_{ uu }}{\left|\begin{array}{ll}\Phi_{ uu } & \Phi_{ vu } \\\Phi_{ uv } & \Phi_{ vv }\end{array}\right|}\)

 

\(\phi_{ yx }=\frac{-\Phi_{ vu }}{\left|\begin{array}{ll}\Phi_{ uu } & \Phi_{ vu } \\\Phi_{ uv } & \Phi_{ vv }\end{array}\right|}\)

 

Because of irrotationality

 

\(\frac{\partial v }{\partial x }-\frac{\partial u }{\partial y }=0=\phi_{ yx }-\phi_{ xy }\)

 

So \(\phi_{ xy }=\phi_{ yx }\). Accordingly, from the above we see that

 

\(\phi_{ xy }=\frac{-\Phi_{ uv }}{\left|\begin{array}{ll}\Phi_{ uu } & \Phi_{ vu } \\\Phi_{ uv } & \Phi_{ vv }\end{array}\right|}=\phi_{ yx }=\frac{-\Phi_{ vu }}{\left|\begin{array}{cc}\Phi_{ uu } & \Phi_{ vu } \\\Phi_{ uv } & \Phi_{ vv }\end{array}\right|}\)

 

or \(\Phi_{ uv }=\Phi_{ vu }\). Thus the denominators may be written as

 

\(J =\left|\begin{array}{ll}\Phi_{ uu } & \Phi_{ uv } \\\Phi_{ uv } & \Phi_{ vv }\end{array}\right|\)

 

Finally then we obtain the expressions requested i.e.,

 

\(\frac{\partial^{2} \phi}{\partial x ^{2}}=\frac{1}{ J } \frac{\partial^{2} \Phi}{\partial v ^{2}}\)

 

\(\frac{\partial^{2} \phi}{\partial x \partial y }=-\frac{1}{ J } \frac{\partial^{2} \Phi}{\partial u \partial v }\)

 

\(\frac{\partial^{2} \phi}{\partial y ^{2}}=\frac{1}{ J } \frac{\partial^{2} \Phi}{\partial u ^{2}}\)

 

(c) Equation (12.27)

 

\(\frac{\partial v}{\partial x}-\frac{\partial u}{\partial y}=0\) (12.27)

 

\(\left(1-\frac{u^{2}}{a^{2}}\right) \frac{\partial u}{\partial x}-\frac{u v}{a^{2}}\left(\frac{\partial v}{\partial x}+\frac{\partial u}{\partial y}\right)+\left(1-\frac{v^{2}}{a^{2}}\right) \frac{\partial v}{\partial y}=0\)

 

may be written as

 

\(\left(1-\frac{ u ^{2}}{ a ^{2}}\right) \frac{\partial^{2} \phi}{\partial x ^{2}}-\frac{ uv }{ a ^{2}}\left(\frac{\partial^{2} \phi}{\partial x \partial y }+\frac{\partial^{2} \phi}{\partial y \partial x }\right)+\left(1-\frac{ v ^{2}}{ a ^{2}}\right) \frac{\partial^{2} \phi}{\partial y ^{2}}=\)

 

\(\left(1-\frac{u^{2}}{a^{2}}\right) \frac{\partial^{2} \phi}{\partial x^{2}}-2 \frac{u v}{a^{2}} \frac{\partial^{2} \phi}{\partial x \partial y}+\left(1-\frac{v^{2}}{a^{2}}\right) \frac{\partial^{2} \phi}{\partial y^{2}}=0\)

 

The derivatives of the potential may be replaced by using the expressions from part (b)

 

\(\left(1-\frac{ u ^{2}}{ a ^{2}}\right) \frac{1}{ J } \frac{\partial^{2} \Phi}{\partial v ^{2}}-2 \frac{ uv }{ a ^{2}} \frac{1}{ J } \frac{\partial^{2} \Phi}{\partial u \partial v }+\left(1-\frac{ v ^{2}}{ a ^{2}}\right) \frac{1}{ J } \frac{\partial^{2} \Phi}{\partial u ^{2}}=0\)

 

Assuming that \(J \neq 0\) this becomes

 

\(\left(1-\frac{u^{2}}{a^{2}}\right) \frac{\partial^{2} \Phi}{\partial v^{2}}+2 \frac{u v}{a^{2}} \frac{\partial^{2} \Phi}{\partial u \partial v}+\left(1-\frac{v^{2}}{a^{2}}\right) \frac{\partial^{2} \Phi}{\partial u^{2}}=0\)
