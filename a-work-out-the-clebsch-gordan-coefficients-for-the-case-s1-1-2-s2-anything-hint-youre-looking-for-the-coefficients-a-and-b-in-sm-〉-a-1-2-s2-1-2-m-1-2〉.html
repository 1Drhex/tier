Guest Question.

(a) Work out the Clebsch–Gordan coefficients for the case \( s_{1}=1 / 2, s 2= \) anything. Hint: You’re looking for the coefficients A and B in

\( |s m\rangle=A\left|\frac{1}{2} s_{2} \frac{1}{2}\left(m-\frac{1}{2}\right)\right\rangle+B\left|\frac{1}{2} s_{2} \frac{-1}{2}\left(m+\frac{1}{2}\right)\right\rangle \).

such that |sm 〉 is an eigenstate of S² . Use the method of Equations 4.177 through 4.180. If you can’t figure out what \( S_{x}^{(2)} \) (for instance) does to \( \left|s_{2} m_{2}\right\rangle \) refer back to Equation 4.136 and the line before Equation 4.147.
Answer:

\( S^{2}=\left( S ^{(1)}+ S ^{(2)}\right) \cdot\left( s ^{(1)}+ S ^{(2)}\right)=\left(S^{(1)}\right)^{2}+\left(S^{(2)}\right)^{2}+2 S ^{(1)} \cdot S ^{(2)} \) (4.177).

\( S ^{(1)} \cdot S ^{(2)}|10\rangle=\frac{\hbar^{2}}{4} \frac{1}{\sqrt{2}}(2|\downarrow \uparrow\rangle-|\uparrow \downarrow\rangle+2|\uparrow \downarrow\rangle-|\downarrow \uparrow\rangle)=\frac{\hbar^{2}}{4}|10\rangle \) (4.178).

\( S ^{(1)} \cdot S ^{(2)}|00\rangle=\frac{\hbar^{2}}{4} \frac{1}{\sqrt{2}}(2|\downarrow \uparrow\rangle-|\uparrow \downarrow\rangle-2|\uparrow \downarrow\rangle+|\downarrow \uparrow\rangle)=-\frac{3 \hbar^{2}}{4}|00\rangle \) (4.179).

\( S^{2}|10\rangle=\left(\frac{3 \hbar^{2}}{4}+\frac{3 \hbar^{2}}{4}+2 \frac{\hbar^{2}}{4}\right)|10\rangle=2 \hbar^{2}|10\rangle \) (4.180).

\( S_{\pm}|s m\rangle=\hbar \sqrt{s(s+1)-m(m \pm 1)}|s(m \pm 1)\rangle \) (4.136).

\( S _{x}=\frac{\hbar}{2}\left(\begin{array}{ll} 0 & 1 \\ 1 & 0 \end{array}\right), \quad S _{y}=\frac{\hbar}{2}\left(\begin{array}{cc} 0 & -i \\ i & 0 \end{array}\right) \) (4.147).

\( A=\sqrt{\frac{s_{2} \pm m+1 / 2}{2 s_{2}+1}} ; \quad B=\pm \sqrt{\frac{s_{2} \mp m+1 / 2}{2 s_{2}+1}} \).

where the signs are determined by \( s=s_{2} \pm 1 / 2 \).

(b) Check this general result against three or four entries in Table 4.8.

Solution

(a) First note from Eq. 4.136 and line after Eq. 4.146 that

\( S_{\pm}|s m\rangle=\hbar \sqrt{s(s+1)-m(m \pm 1)}|s(m \pm 1)\rangle \) (4.136).

\( S _{+}=\hbar\left(\begin{array}{ll} 0 & 1 \\ 0 & 0 \end{array}\right), \quad S _{-}=\hbar\left(\begin{array}{ll} 0 & 0 \\ 1 & 0 \end{array}\right) \) (4.146).

\( S_{x}|s m\rangle=\frac{1}{2}\left[S_{+}|s m\rangle+S_{-}|s m\rangle\right] \).

\( =\frac{\hbar}{2}[\sqrt{s(s+1)-m(m+1)}|s m+1\rangle+\sqrt{s(s+1)-m(m-1)}|s m-1\rangle] \).

\( S_{y}|s m\rangle=\frac{1}{2 i}\left[S_{+}|s m\rangle-S_{-}|s m\rangle\right] \).

\( =\frac{\hbar}{2 i}[\sqrt{s(s+1)-m(m+1)}|s m+1\rangle-\sqrt{s(s+1)-m(m-1)}|s m-1\rangle] \).

Now, using Eqs. 4.177 and 4.147:

\( S^{2}=\left( S ^{(1)}+ S ^{(2)}\right) \cdot\left( s ^{(1)}+ S ^{(2)}\right)=\left(S^{(1)}\right)^{2}+\left(S^{(2)}\right)^{2}+2 S ^{(1)} \cdot S ^{(2)} \) (4.177).

\( S _{x}=\frac{\hbar}{2}\left(\begin{array}{ll} 0 & 1 \\ 1 & 0 \end{array}\right), \quad S _{y}=\frac{\hbar}{2}\left(\begin{array}{cc} 0 & -i \\ i & 0 \end{array}\right) \) (4.147).

\( S^{2}|s m\rangle=\left[\left(S^{(1)}\right)^{2}+\left(S^{(2)}\right)^{2}+2\left(S_{x}^{(1)} S_{x}^{(2)}+S_{y}^{(1)} S_{y}^{(2)}+S_{z}^{(1)} S_{z}^{(2)}\right)\right]\left[A\left|\frac{1}{2} \frac{1}{2}\right\rangle\left|s_{2} m-\frac{1}{2}\right\rangle+B\left|\frac{1}{2}-\frac{1}{2}\right\rangle\left|s_{2} m+\frac{1}{2}\right\rangle\right] \).

\( =A\left\{\left(S^{2}\left|\frac{1}{2} \frac{1}{2}\right\rangle\right)\left|s_{2} m-\frac{1}{2}\right\rangle+\left|\frac{1}{2} \frac{1}{2}\right\rangle\left(S^{2}\left|s_{2} m-\frac{1}{2}\right\rangle\right)\right. \) \( \left.+2\left[\left(S_{x}\left|\frac{1}{2} \frac{1}{2}\right\rangle\right)\left(S_{x}\left|s_{2} m-\frac{1}{2}\right\rangle\right)+\left(S_{y}\left|\frac{1}{2} \frac{1}{2}\right\rangle\right)\left(S_{y}\left|s_{2} m-\frac{1}{2}\right\rangle\right)+\left(S_{z}\left|\frac{1}{2} \frac{1}{2}\right\rangle\right)\left(S_{z}\left|s_{2} m-\frac{1}{2}\right\rangle\right)\right]\right\} \) \( +B\left\{\left(S^{2}\left|\frac{1}{2}-\frac{1}{2}\right\rangle\right)\left|s_{2} m+\frac{1}{2}\right\rangle+\left|\frac{1}{2}-\frac{1}{2}\right\rangle\left(S^{2}\left|s_{2} m+\frac{1}{2}\right\rangle\right)\right. \) \( \left.+2\left[\left(S_{x}\left|\frac{1}{2}-\frac{1}{2}\right\rangle\right)\left(S_{x}\left|s_{2} m+\frac{1}{2}\right\rangle\right)+\left(S_{y}\left|\frac{1}{2}-\frac{1}{2}\right\rangle\right)\left(S_{y}\left|s_{2} m+\frac{1}{2}\right\rangle\right)+\left(S_{z}\left|\frac{1}{2}-\frac{1}{2}\right\rangle\right)\left(S_{z}\left|s_{2} m+\frac{1}{2}\right\rangle\right)\right]\right\} \) \( =A\left\{\frac{3}{4} \hbar^{2}\left|\frac{1}{2} \frac{1}{2}\right\rangle\left|s_{2} m-\frac{1}{2}\right\rangle+\hbar^{2} s_{2}\left(s_{2}+1\right)\left|\frac{1}{2} \frac{1}{2}\right\rangle\left|s_{2} m-\frac{1}{2}\right\rangle\right. \) \( +2\left[\frac { \hbar } { 2 } | \frac { 1 } { 2 } – \frac { 1 } { 2 } \rangle \frac { \hbar } { 2 } \left(\sqrt{s_{2}\left(s_{2}+1\right)-\left(m-\frac{1}{2}\right)\left(m+\frac{1}{2}\right)}\left|s_{2} m+\frac{1}{2}\right\rangle\right.\right. \) \( \left.+\sqrt{s_{2}\left(s_{2}+1\right)-\left(m-\frac{1}{2}\right)\left(m-\frac{3}{2}\right)}\left|s_{2} m-\frac{3}{2}\right\rangle\right) \) \( +\left(\frac{i \hbar}{2}\right)\left|\frac{1}{2}-\frac{1}{2}\right\rangle \frac{\hbar}{2 i}\left(\sqrt{s_{2}\left(s_{2}+1\right)-\left(m-\frac{1}{2}\right)\left(m+\frac{1}{2}\right)}\left|s_{2} m+\frac{1}{2}\right\rangle\right. \) \( \left.\left.\left.-\sqrt{s_{2}\left(s_{2}+1\right)-\left(m-\frac{1}{2}\right)\left(m-\frac{3}{2}\right)}\left|s_{2} m-\frac{3}{2}\right\rangle\right)+\frac{\hbar}{2}\left|\frac{1}{2} \frac{1}{2}\right\rangle \hbar\left(m-\frac{1}{2}\right)\left|s_{2} m-\frac{1}{2}\right\rangle\right]\right\} \) \( +B\left\{\frac{3}{4} \hbar^{2}\left|\frac{1}{2}-\frac{1}{2}\right\rangle\left|s_{2} m+\frac{1}{2}\right\rangle+\hbar^{2} s_{2}\left(s_{2}+1\right)\left|\frac{1}{2}-\frac{1}{2}\right\rangle\left|s_{2} m+\frac{1}{2}\right\rangle\right. \) \( +2\left[\frac{\hbar}{2}\left|\frac{1}{2} \frac{1}{2}\right\rangle \frac{\hbar}{2}\left(\sqrt{s_{2}\left(s_{2}+1\right)-\left(m+\frac{1}{2}\right)\left(m+\frac{3}{2}\right)}\left|s_{2} m+\frac{3}{2}\right\rangle+\sqrt{s_{2}\left(s_{2}+1\right)-\left(m+\frac{1}{2}\right)\left(m-\frac{1}{2}\right)}\left|s_{2} m-\frac{1}{2}\right\rangle\right)\right. \) \( +\left(\frac{-i \hbar}{2}\right)\left|\frac{1}{2} \frac{1}{2}\right\rangle \frac{\hbar}{2 i}\left(\sqrt{s_{2}\left(s_{2}+1\right)-\left(m+\frac{1}{2}\right)\left(m+\frac{3}{2}\right)}\left|s_{2} m+\frac{3}{2}\right\rangle\right. \) \( \left.\left.\left.\left.-\sqrt{s_{2}\left(s_{2}+1\right)-\left(m+\frac{1}{2}\right)\left(m-\frac{1}{2}\right) \mid} s_{2} m-\frac{1}{2}\right\rangle\right)+\left(\frac{-\hbar}{2}\right)\left|\frac{1}{2}-\frac{1}{2}\right\rangle \hbar\left(m+\frac{1}{2}\right)\left|s_{2} m+\frac{1}{2}\right\rangle\right]\right\} \) \( =\hbar^{2}\left\{A\left[\frac{3}{4}+s_{2}\left(s_{2}+1\right)+m-\frac{1}{2}\right]+B \sqrt{s_{2}\left(s_{2}+1\right)-m^{2}+\frac{1}{4}}\right\}\left|\frac{1}{2} \frac{1}{2}\right\rangle\left|s_{2} m-\frac{1}{2}\right\rangle \) \( +\hbar^{2}\left\{B\left[\frac{3}{4}+s_{2}\left(s_{2}+1\right)-m-\frac{1}{2}\right]+A \sqrt{s_{2}\left(s_{2}+1\right)-m^{2}+\frac{1}{4}}\right\}\left|\frac{1}{2}-\frac{1}{2}\right\rangle\left|s_{2} m+\frac{1}{2}\right\rangle \)

\( =\hbar^{2} s(s+1)|s m\rangle=\hbar^{2} s(s+1)\left[A\left|\frac{1}{2} \frac{1}{2}\right\rangle\left|s_{2} m-\frac{1}{2}\right\rangle+B\left|\frac{1}{2}-\frac{1}{2}\right\rangle\left|s_{2} m+\frac{1}{2}\right\rangle\right] \).

\( \left\{\begin{array}{l} A\left[s_{2}\left(s_{2}+1\right)+\frac{1}{4}+m\right]+B \sqrt{s_{2}\left(s_{2}+1\right)-m^{2}+\frac{1}{4}}=s(s+1) A \\ B\left[s_{2}\left(s_{2}+1\right)+\frac{1}{4}-m\right]+A \sqrt{s_{2}\left(s_{2}+1\right)-m^{2}+\frac{1}{4}}=s(s+1) B \end{array}\right\} \) or

\( \left\{\begin{array}{l} A\left[s_{2}\left(s_{2}+1\right)-s(s+1)+\frac{1}{4}+m\right]+B \sqrt{s_{2}\left(s_{2}+1\right)-m^{2}+\frac{1}{4}}=0, \\ B\left[s_{2}\left(s_{2}+1\right)-s(s+1)+\frac{1}{4}-m\right]+A \sqrt{s_{2}\left(s_{2}+1\right)-m^{2}+\frac{1}{4}}=0, \end{array}\right\} \quad \text { or } \quad\left\{\begin{array}{l} A(a+m)+B b=0 \\ B(a-m)+A b=0 \end{array}\right\} \).

where \( a \equiv s_{2}\left(s_{2}+1\right)-s(s+1)+\frac{1}{4}, b \equiv \sqrt{s_{2}\left(s_{2}+1\right)-m^{2}+\frac{1}{4}} \). Multiply by (a-m) and b, then subtract:

\( A\left(a^{2}-m^{2}\right)+B b(a-m)=0 ; B b(a-m)+A b^{2}=0 \Rightarrow A\left(a^{2}-m^{2}-b^{2}\right)=0 \Rightarrow a^{2}-b^{2}=m^{2} \), or:

\( \left[s_{2}\left(s_{2}+1\right)-s(s+1)+\frac{1}{4}\right]^{2}-s_{2}\left(s_{2}+1\right)+m^{2}-\frac{1}{4}=m^{2} \),

\( \left[s_{2}\left(s_{2}+1\right)-s(s+1)+\frac{1}{4}\right]^{2}=s_{2}^{2}+s_{2}+\frac{1}{4}=\left(s_{2}+\frac{1}{2}\right)^{2} \), so

\( s_{2}\left(s_{2}+1\right)-s(s+1)+\frac{1}{4}=\pm\left(s_{2}+\frac{1}{2}\right) ; \quad s(s+1)=s_{2}\left(s_{2}+1\right) \mp\left(s_{2}+\frac{1}{2}\right)+\frac{1}{4} \).

Add \( \frac{1}{4} \) to both sides:

\( s^{2}+s+\frac{1}{4}=\left(s+\frac{1}{2}\right)^{2}=s_{2}\left(s_{2}+1\right) \mp\left(s_{2}+\frac{1}{2}\right)+\frac{1}{2}=\left\{\begin{array}{l} s_{2}^{2}+s_{2}-s_{2}-\frac{1}{2}+\frac{1}{2}=s_{2}^{2} \\ s_{2}^{2}+s_{2}+s_{2}+\frac{1}{2}+\frac{1}{2}=\left(s_{2}+1\right)^{2} \end{array}\right\} \),

So \( \left\{\begin{array}{l} s+\frac{1}{2}=\pm s_{2} \quad \Rightarrow s=\pm s_{2}-\frac{1}{2}=\left\{\begin{array}{l} s_{2}-\frac{1}{2} \\ -s_{2}-\frac{1}{2} \\ s+\frac{1}{2}=\pm\left(s_{2}+1\right) \end{array} \Rightarrow s=\pm\left(s_{2}+1\right)-\frac{1}{2}=\left\{\begin{array}{l} s_{2}+\frac{1}{2} \\ -s_{2}-\frac{3}{2} \end{array}\right.\right. \end{array}\right\} \).

\( \text { But } s \geq 0 \text {, so the possibilities are } s=s_{2} \pm 1 / 2 \text {. Then } \):

\( a=s_{2}^{2}+s_{2}-\left(s_{2} \pm \frac{1}{2}\right)\left(s_{2} \pm \frac{1}{2}+1\right)+\frac{1}{4} \).

\( =s_{2}^{2}+s_{2}-s_{2}^{2} \mp \frac{1}{2} s_{2}-s_{2} \mp \frac{1}{2} s_{2}-\frac{1}{4} \mp \frac{1}{2}+\frac{1}{4}=\mp s_{2} \mp \frac{1}{2}=\mp\left(s_{2}+\frac{1}{2}\right) \).

\( b=\sqrt{\left(s_{2}^{2}+s_{2}+\frac{1}{4}\right)-m^{2}}=\sqrt{\left(s_{2}+\frac{1}{2}\right)^{2}-m^{2}}=\sqrt{\left(s_{2}+\frac{1}{2}+m\right)\left(s_{2}+\frac{1}{2}-m\right)} \).

\( \therefore A\left[\mp\left(s_{2}+\frac{1}{2}\right)+m\right]=\mp A\left(s_{2}+\frac{1}{2} \mp m\right)=-B b=-B \sqrt{\left(s_{2}+\frac{1}{2}+m\right)\left(s_{2}+\frac{1}{2}-m\right)} \)

\( \Rightarrow A \sqrt{s_{2}+\frac{1}{2} \mp m}=\pm B \sqrt{s_{2}+\frac{1}{2} \pm m} . \quad \text { But }|A|^{2}+|B|^{2}=1 \), so

\( |A|^{2}+|A|^{2}\left(\frac{s_{2}+\frac{1}{2} \mp m}{s_{2}+\frac{1}{2} \pm m}\right)=\frac{|A|^{2}}{\left(s_{2}+\frac{1}{2} \pm m\right)}\left[s_{2}+\frac{1}{2} \pm m+s_{2}+\frac{1}{2} \mp m\right]=\frac{\left(2 s_{2}+1\right)}{\left(s_{2}+\frac{1}{2} \pm m\right)}|A|^{2}=1 \).

⇒ \( A=\sqrt{\frac{s_{2} \pm m+\frac{1}{2}}{2 s_{2}+1}} \).

\( B=\pm A \frac{\sqrt{s_{2}+\frac{1}{2} \mp m}}{\sqrt{s_{2}+\frac{1}{2} \pm m}}=\pm \sqrt{\frac{s_{2} \mp m+\frac{1}{2}}{2 s_{2}+1}} \).

(b) Here are four examples:

\( \text { (i) From the } 1 / 2 \times 1 / 2 \text { table }\left(s_{2}=1 / 2\right) \text {, pick } s=1 \text { (upper signs), } m=0 \text {. Then } \)

\( A=\sqrt{\frac{\frac{1}{2}+0+\frac{1}{2}}{1+1}}=\frac{1}{\sqrt{2}} ; \quad B=\sqrt{\frac{\frac{1}{2}-0+\frac{1}{2}}{1+1}}=\frac{1}{\sqrt{2}} \).

\( \text { (ii) From the } 1 \times 1 / 2 \text { table }\left(s_{2}=1\right) \text {, pick } s=3 / 2 \text { (upper signs), } m=1 / 2 \text {. Then } \) \( A=\sqrt{\frac{1+\frac{1}{2}+\frac{1}{2}}{2+1}}=\sqrt{\frac{2}{3}} ; \quad B=\sqrt{\frac{1-\frac{1}{2}+\frac{1}{2}}{2+1}}=\frac{1}{\sqrt{3}} . \) \( \text { (iii) From the } 3 / 2 \times 1 / 2 \text { table }\left(s_{2}=3 / 2\right) \text {, pick } s=1 \text { (lower signs), } m=-1 \text {. Then } \)

\( A=\sqrt{\frac{\frac{3}{2}+1+\frac{1}{2}}{3+1}}=\frac{\sqrt{3}}{2} ; \quad B=-\sqrt{\frac{\frac{3}{2}-1+\frac{1}{2}}{3+1}}=-\frac{1}{2} \).

\( \text { (iv) From the } 2 \times 1 / 2 \text { table }\left(s_{2}=2\right) \text {, pick } s=3 / 2 \text { (lower signs), } m=1 / 2 \text {. Then } \) \( A=\sqrt{\frac{2-\frac{1}{2}+\frac{1}{2}}{4+1}}=\sqrt{\frac{2}{5}} ; \quad B=-\sqrt{\frac{2+\frac{1}{2}+\frac{1}{2}}{4+1}}=-\sqrt{\frac{3}{5}} \text {. } \)

These all check with the values on Table 4.8, except that the signs (which are conventional) are reversed in (iii) and (iv). Normalization does not determine the sign of A (nor, therefore, of B).
