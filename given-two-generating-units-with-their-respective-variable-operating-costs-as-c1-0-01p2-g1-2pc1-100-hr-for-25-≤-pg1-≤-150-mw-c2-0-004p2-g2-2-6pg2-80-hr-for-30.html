Guest Question.

Given two generating units with their respective variable operating costs as

\(\begin{array}{lcc}C _{1}=0.01 P _{ G 1}^{2}+2 P _{ G 1}+100 \$ / hr & \text { for } 25 \leq P _{ G 1} \leq 150 MW \\C _{2}=0.004 P _{ G 2}^{2}+2.6 P _{ G 2}+80 \$ / hr & \text { for } 30 \leq P _{ G 2} \leq 200 MW\end{array}\)

determine the economically optimum division of generation for \(55 \leq P _{ L } \leq 350 MW\) . In particular, for \(P _{ L }=282 MW\) , compute \(P _{ G 1} \text { and } P _{ G 2} \text { . }\) Neglect transmission losses.

Admin
Solution

Choosing \(S_{\text {base }}\) as 100 MVA (3-Phase),

\(\begin{aligned}&\alpha_{1}=\left(S_{3 \phi \text { base }}\right)^{2} 0.01=100 ; \quad \alpha_{2}=40 \\&\beta_{1}=\left(S_{3 \phi \text { base }}\right) 2.00=200 ; \quad \beta_{2}=260 \\&\gamma_{1}=100 ; \quad \gamma_{2}=80\end{aligned}\)

In per unit, \(0.25 \leq P_{G 1} \leq 1.5 ; 0.3 \leq P_{G 2} \leq 2.0 ; 0.55 \leq P_{L} \leq 3.5\)

\(\lambda_{1}=\frac{\partial \zeta_{1}}{\partial P_{G 1}}=200 P_{G 1}+200 ; \quad \lambda_{2}=\frac{\partial \varsigma_{2}}{\partial P_{G 2}}=80 P_{G 2}+260\)

Calculate \(\lambda_{1} \text { and } \lambda_{2}\) for minimum generation conditions (Point 1, in Figure shown below).

Since \(\lambda_{2}>\lambda_{1}\) , in order to make \(\lambda \text { 's }\) equal, load unit 1 first until \(\lambda_{1}=284\) which occurs at

\(P_{G 1}=\frac{284-200}{200}=0.42 \text { (Point } 2 \text { in Figure) }\)

Now, calculate \(\lambda_{1} \text { and } \lambda_{2}\) at the maximum generation conditions:

Point 3 in Figure, now that \(\lambda_{1}>\lambda_{2}\) , unload unit 1 first until \(\lambda_{1}\) is brought down to \(\lambda_{1}=420\) which occurs at

\(P_{G 1}=\frac{420-200}{200}=1.10 \text { (Point } 4 \text { in Figure) }\)

Notice that, for \(0.72 \leq P_{L} \leq 3.1\) , it is possible to maintain equal \(\lambda \text { 's. }\) Equations are given by

\(\lambda_{1}=\lambda_{2} ; 200 P_{G 1}+200=80 P_{G 2}+260 ; \text { and } P_{G 1}+P_{G 2}=P_{L}\)

These linear relationships are depicted in the Figure below:

For \(P_{L}=282 MW =2.82 pu , P_{G 2}=2.82-P_{G 1}\) ;

\(\begin{aligned}&P_{G 1}=0.4 P_{G 2}+0.3=1.128-0.4 P_{G 1}+0.3 \\&1.4 P_{G 1}=1.428 \text { or } P_{G 1}=1.02=102 MW \\&P_{G 2}=2.82-1.02=1.8=180 MW\end{aligned}\)
