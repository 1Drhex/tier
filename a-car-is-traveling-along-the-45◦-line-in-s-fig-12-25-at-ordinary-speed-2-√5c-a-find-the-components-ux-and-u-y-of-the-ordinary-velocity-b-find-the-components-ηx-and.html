Guest Question.

A car is traveling along the 45◦ line in S (Fig. 12.25), at (ordinary) speed \((2 / \sqrt{5}) c\).

(a) Find the components \(u_{x} \text { and } u_{y}\) of the (ordinary) velocity.

(b) Find the components \(\eta_{x} \text { and } \eta_{y}\) of the proper velocity .

(c) Find the zeroth component of the 4-velocity, \(\eta^{0}\).

System \(\overline{ S }\) is moving in the x direction with (ordinary) speed \(\sqrt{2 / 5} c\), relative to S. By using the appropriate transformation laws:

(d) Find the (ordinary) velocity components \(\bar{u}_{x} \text { and } \bar{u}_{y} \text { in } \overline{ S }\).

(e) Find the proper velocity components \(\bar{\eta}_{x} \text { and } \bar{\eta}_{y} \text { in } \overline{ S }\).

(f) As a consistency check, verify that

\(\bar{\eta}=\frac{\overline{ u }}{\sqrt{1-\bar{u}^{2} / c^{2}}}\).

Admin
Solution

\(\text { (a) } u_{x}=u_{y}=u \cos 45^{\circ}=\frac{1}{\sqrt{2}} \frac{2}{\sqrt{5}} c=\sqrt{\frac{2}{5}} c\).

\(\text { (b) } \frac{1}{\sqrt{1-u^{2} / c^{2}}}=\frac{1}{\sqrt{1-4 / 5}}=\frac{\sqrt{5}}{\sqrt{5-4}}=\sqrt{5} . \therefore \eta=\frac{ u }{\sqrt{1-u^{2} / c^{2}}} \Rightarrow \eta_{x}=\eta_{y}=\sqrt{2}\)

(c) \(\eta_{0}=\gamma c=\sqrt{5} c\).

\(\text { (d) Eq. } 12.45 \Rightarrow\left\{\begin{array}{l}\bar{u}_{x}=\frac{u_{x}-v}{1-\frac{u_{xv}}{c^{2}}}=\frac{\sqrt{\frac{2}{5}} c-\sqrt{\frac{2}{5}} c}{1-\frac{2}{5}}=0 . \\\bar{u}_{y}=\frac{1}{\gamma}\left(\frac{u_{y}}{1-\frac{u x v}{c^{2}}}\right)=\sqrt{1-\frac{2}{5}} \frac{\sqrt{\frac{2}{5}} c}{1-\frac{2}{5}}=\frac{\sqrt{2 / 5}}{\sqrt{3 / 5}} c=\sqrt{\frac{2}{3}} c .\end{array}\right.\)

 

\(\left. \begin{matrix} \bar{u}_{x}=\frac{d \bar{x}}{d \bar{t}}=\frac{u_{x}-v}{\left(1-v u_{x} / c^{2}\right)}, \\ \bar{u}_{y}=\frac{d \bar{y}}{d \bar{t}}=\frac{u_{y}}{\gamma\left(1-v u_{x} / c^{2}\right)}, \\ \bar{u}_{z}=\frac{d \bar{z}}{d \bar{t}}=\frac{u_{z}}{\gamma\left(1-v u_{x} / c^{2}\right)}. \end{matrix} \right\}\) (12.45)

\(\text { (e) } \bar{\eta}_{x}=\gamma\left(\eta_{x}-\beta \eta^{0}\right)=\sqrt{1-\frac{2}{5}}\left(\sqrt{2} c-\sqrt{\frac{2}{5}} \sqrt{5} c\right)=0 . \quad \bar{\eta}_{y}=\eta_{y}=\sqrt{2} c \text {. }\)

(f)\(\frac{1}{\sqrt{1-\bar{u}^{2} / c^{2}}}=\frac{1}{\sqrt{1-2 / 3}}=\sqrt{3} ; \therefore \bar{\eta}=\sqrt{3} \overline{ u } \Rightarrow\left\{\begin{array}{l}\bar{\eta}_{x}=\sqrt{3} \bar{u}_{x}=0 \\\bar{\eta}_{y}=\sqrt{3} \bar{u}_{y}=\sqrt{2} c \end{array}\right\}\)
