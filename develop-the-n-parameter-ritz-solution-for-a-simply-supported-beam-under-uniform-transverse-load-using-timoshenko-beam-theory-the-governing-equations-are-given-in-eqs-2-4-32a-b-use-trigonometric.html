Guest Question.

Develop the N-parameter Ritz solution for a simply supported beam under uniform transverse load using Timoshenko beam theory. The governing equations are given in Eqs. (2.4.32a, b). Use Trigonometric functions to approximate w and \(\Psi \).

Eq (2.4.32a) : \(-\frac{\mathrm{d} }{\mathrm{d} x}\left [ S\left ( \frac{\mathrm{d} w}{\mathrm{d} x}+\phi _{x} \right ) \right ]+c_{f}w=q\)
Eq (2.4.32b) : \(-\frac{\mathrm{d} }{\mathrm{d} x}\left ( D\frac{\mathrm{d} \phi _{x}}{\mathrm{d} x} \right )+S\left ( \frac{\mathrm{d} w}{\mathrm{d} x}+\phi _{x} \right )=0\)

Admin
Solution

Assume solution of (w, \(\Psi \)) in the form,

\(wM = \sum_{j=1}^{M}b_{j}\phi _{j}\equiv \sum_{j=1}^{M}b_{j}\sin \frac{j\pi x}{L}\) , \(\Psi N = \sum_{j=1}^{N}c_{j}\psi _{j}\equiv \sum_{j=1}^{N}c_{j}\cos \frac{j\pi x}{L}\) (1)

Substitution of Eq. (1) into the weak forms (S = GAK and D = EI)

\(0=\int_{0}^{L}\left [ GAK \frac{\mathrm{d} v_{1}}{\mathrm{d} x} \left ( \frac{\mathrm{d} w}{\mathrm{d} x}+\Psi \right )+kv_{1}w-v_{1}q\right ]dx\) (2a)

\(0=\int_{0}^{L}\left [ EI\frac{\mathrm{d} v_{2}}{\mathrm{d} x}\frac{\mathrm{d} \Psi }{\mathrm{d} x}+GAK v_{2}\left ( \frac{\mathrm{d} w}{\mathrm{d} x} +\Psi \right ) \right ]dx\) (2b)

we obtain following system of algebraic equations,

\(\begin{bmatrix}[K^{11}] &[K^{12}] \\ [K^{21}] &[K^{22}] \end{bmatrix}\begin{Bmatrix}{b}\\ {c}\end{Bmatrix}=\begin{Bmatrix}{F^{1}}\\ {F^{2}}\end{Bmatrix}\) (3)

where

\(K_{ij}^{11}=\int_{0}^{L}\left ( GAK\frac{\mathrm{d} \phi _{i}}{\mathrm{d} x} \frac{\mathrm{d} \phi _{j}}{\mathrm{d} x}+k\phi _{i}\phi _{j}\right )dx\) , \(K_{ij}^{12}=\int_{0}^{L} GAK\frac{\mathrm{d} \phi _{i}}{\mathrm{d} x}\psi _{j}dx\) ,

\(K_{ij}^{21}=\int_{0}^{L} GAK\psi _{i}\frac{\mathrm{d} \phi _{j}}{\mathrm{d} x}dx\) , \(K_{ij}^{22}=\int_{0}^{L}\left ( EI\frac{\mathrm{d} \psi _{i}}{\mathrm{d} x}\frac{\mathrm{d} \psi _{j}}{\mathrm{d} x}+GAK \psi _{i} \psi _{j}\right )dx\) (4a)

\(F_{i}^{1}=\int_{0}^{L} \phi _{i}q dx\) , \(F_{i}^{2}=0\) (4b)

Substituting \(\phi _{i}=\sin (i\pi x/L)\) and \(\psi _{i}=\cos (i\pi x/L)\) into the above equations and evaluating the integrals, we obtain

\(K_{ij}^{11}=GAK \frac{L}{2}\left ( \frac{i\pi }{L} \right )\left ( \frac{j\pi }{L} \right )+\frac{kL}{2}\) , \(K_{ij}^{12}=GAK \frac{L}{2}\left ( \frac{i\pi }{L} \right )=K_{ji}^{21}\)

\(K_{ij}^{22}=\frac{L}{2}\left [ GAK+EI\left ( \frac{i\pi }{L} \right )\left ( \frac{j\pi }{L} \right ) \right ]\) (5a)

for i = j, and

\(K_{ij}^{\alpha \beta }=0\) , if \(i\neq j\) (5b)

\(F_{i}^{1}=-\frac{2q_{0}L}{i\pi }\) for i = odd and \(F_{i}^{1}=0\) for i = even (5c)
