Guest Question.

Use the variational method to estimate the ground-state energy of the hydrogen atom.

Admin
Solution

Of course, we already know the ground- state energy of hydrogen. Our goal here is to show how to implement the variational method in a step-by-step approach. The Hamiltonian for the hydrogen atom is

\( \hat{H}=\frac{\hat{P} ^{2} }{2\mu }-\frac{e^{2} }{|\hat{r}| }\)

 

As a first step, we need to make a judicious choice of the trial wave function. Since we are looking for the ground state, we need to choose a spherically symmetric wave function, since we know the ground state has I = 0. Moreover, for a sphe1ically symmetric potential energy, we know that for small r the radial wave function R(r) behaves as \(r^{l}\). Finally, since our goal is to can-y out the calculation analytically, we want to choose a wave function for which we can actually do the integrals needed to evaluate 〈E〉. There are two likely possibilities for the trial wave function, a simple exponential and a Gaussian. Here we will use the trial wave function

\( R(r)=Ne^{-br}\)

 

where N is the normalization constant.

First, we determine N:

\(1=\int_{0}^{\infty }{r^{2}dr|R(r)|^{2} }=|N|^{2}\int_{0}^{\infty }{r^{2}dre^{-2br} } =|N|^{2}\frac{1}{4b^{3} }\)

 

Choosing

\( N=2b^{3/2}\)

 

the full trial wave function is therefore

\(\psi =R(r)Y_{0.0}=\frac{1}{\sqrt{4\pi } }R(r)=\frac{b^{3/2} }{\sqrt{\pi } }e^{-br}\)

 

Consequently,

\(\left\langle E\right\rangle =\left\langle \psi |\hat{H}|\psi \right\rangle\)

 

\(=\int{d^{3}r\psi ^{\ast } }\left[-\frac{\hbar ^{2} }{2\mu }\left(\frac{d^{2} }{dr^{2} }+\frac{2}{r} \frac{d}{dr} \right) -\frac{e^{2} }{r} \right]\psi\)

 

\(= 4b^{3}\int_{0}^{\infty }{r^{2}dre^{-br}\left(-\frac{\hbar ^{2}b^{2} }{2\mu }+\frac{\hbar ^{2}b }{\mu r}-\frac{e^{2} }{r} \right) e^{-br} }\)

 

\(=4b^{2}\int_{0}^{\infty }{dr\left(-\frac{\hbar ^{2}b^{2} }{2\mu }r^{2}+\frac{\hbar ^{2}b }{\mu }r-e^{2}r \right)x^{-2br} }\)

 

\(=\frac{\hbar ^{2}b^{2} }{2\mu }-e^{2}b\)

 

Minimizing the expectation value of the energy:

\( \frac{d\left\langle E\right\rangle }{db}=\frac{\hbar ^{2}b }{\mu }-e^{2}=0\)

 

leading to

\( b=\frac{\mu e^{2} }{\hbar ^{2} }\)

 

Substituting this value forb into the expression for 〈E〉, we see that

\(E_{0}\leq \left\langle E\right\rangle=-\frac{\mu e^{4} }{2\hbar ^{2} }=\frac{1}{2} \mu c^{2}\alpha ^{2}\)

 

where \(\alpha =e^{2}/(\hslash c)\) is the fine-structure constant. Since the exact ground-state energy of the Hamiltonian is \(-\mu c^{2}\alpha ^{2}/2\) , the variational method in this case has yielded the exact energy. This is a consequence of our using a trial wave function that included the ground-state energy eigenfunction for a particular value of the parameter b. If we had used the same trial wave function for the three-dimensional harmonic oscillator, the upper limit on the ground-state energy would not be the exact eigenvalue. See Problem 12.8.
