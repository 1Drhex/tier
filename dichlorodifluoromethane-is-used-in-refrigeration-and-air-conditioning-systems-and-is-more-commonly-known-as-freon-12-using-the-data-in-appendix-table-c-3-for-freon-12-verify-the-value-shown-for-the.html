Guest Question.

Dichlorodifluoromethane is used in refrigeration and air-conditioning systems and is more commonly known as Freon-12. Using the data in Appendix Table C.3 for Freon-12, verify the value shown for the volumetric thermal-expansion coefficient. Repeat the calculations for helium at 400 K.

Admin
Solution

Equation 5.8 is the defining equation. Before it can be used, however, it must be rewritten in difference, rather than differential, form. Note that the data on Freon-12 in the appendix are for the saturated liquid state. Th e saturation pressure changes with temperature, so the information in Appendix Table C.3 is not constant-pressure data. It is all that is available in this text, however. The helium data are for constant-pressure conditions.

 

Assumptions

 

1. ∂ρ/∂T can be rewritten as Δρ/ΔT.
2. β is calculated on the basis of data available at −50 and −40°C for Freon-12, even though the corresponding saturation pressures are not equal as required by the definition of β (Equation 5.8).
3. β is calculated on the basis of data available at 366 and 477 K for helium, which are at constant pressure.

 

Equation 5.8 is rewritten as

\(\beta =-\frac{1}{\rho }\left ( \frac{\partial \rho }{\partial T} \right )_{p} \) (5.8)

\(\beta =-\frac{1}{\rho }\left ( \frac{\partial \rho }{\partial T} \right )_{p}\cong -\frac{1}{\rho }\frac{(\rho _{1}-\rho _{2})}{T_{1}-T_{2}}\)

 

From Appendix Table C.3 for Freon-12 we find

 

\(T_{1}=-0^{\circ}C,\;\;\;\rho _{1}=1.546(1000)\,kg/m^{3}\)

 

\(T_{2}=-40^{\circ}C,\;\;\;\rho _{2}=1.518(1000)\,kg/m^{3}\)

 

Substituting

 

\(\beta =-\frac{1}{1.546}\frac{1.546-1.518}{-50-(-40)}=1.811\times 10^{-3}/K\)

 

The accurate value shown in the table is \(2.63 × 10^{−3}/K\). The error introduced by using data at different pressures is

 

\(\frac{2.63-1.811}{2.63}\times 100\)%=31%

 

From Appendix Table D.3 for helium,

 

\(T_{1}=366\,K,\;\;\;\rho _{1}=0.13280\,kg/m^{3}\)

 

\(T_{2}=477\,K,\;\;\;\rho _{1}=0.10204\,kg/m^{3}\)

 

Substituting

 

\(\beta =-\frac{1}{0.13280}\frac{0.13280-0.10204}{366-477}=2.087\times 10^{-3}/K\)
