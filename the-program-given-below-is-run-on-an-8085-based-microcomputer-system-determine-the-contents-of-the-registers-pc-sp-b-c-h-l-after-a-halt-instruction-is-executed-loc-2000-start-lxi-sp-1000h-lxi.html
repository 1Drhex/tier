Guest Question.

The program given below is run on an 8085/based microcomputer system. Determine the contents of the registers: PC, SP, B, C, H, L after a halt instruction is executed.

LOC

2000 START LXI SP.1000H

LXI H, 2F37H

XRA A

MOV A, H

INX H

PUSH H

CZ 20FFH i

JMP 3000H

HLT

20FF ADD H

RZ

POP B

PUSHB

RM

3000 HLT

Admin
Solution

2000:LXI \(SP _{1}\), 1000H → SP ← 1000H

2003:LXI H, 2F37H → HL ← 2F37H

2006:XRA A→ A ←00H, Z21

2007:MOV A, H → A ← 2F

2008:IMX H → HL ← 2F38H

2009:Push H → SP ← OFFE

200A:CZ20FFH → As Z = 1, SP ← 0FFC

→ PC ← 20FF

due to call on zero

20FF:ADD, H → A + M → A → 2F + 2F

→ 5

2100:RZ → Z = 0

2101:POP B → SP ← 0FFE and BC ← 200D

2103:RM → If S = 1 in previous addition

3000:HLT → But S = 0, so condition false

After execution

PC = 3000

SP = 0FFE

B = 20

C = OD

H = 2F

L = 38
