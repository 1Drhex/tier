Guest Question.

The dynamic response of a stirred-tank bioreactor can be represented by the transfer function

\(\frac{C^{\prime}(s)}{C_{F}^{\prime}(s)}=\frac{4}{2 s+1}\)

where \(C^{\prime}\) is the exit substrate concentration, \(mol / L\), and \(C_{F}^{\prime}\) is the feed substrate concentration, \(mol / L\).

(a) Derive an expression for \(c^{\prime}(t)\) if \(c_{F}(t)\) is a rectangular pulse (Fig. 5.2) with the following characteristics:

\(c_{F}(t)= \begin{cases}2 & t<0 \\ 4 & 0 \leq t<2 \\ 2 & 2 \leq t<\infty\end{cases}\)

(b) What is the maximum value of \(c^{\prime}(t) ?\) When does it occur? What is the final value of \(c^{\prime}(t) ?\)

(c) If the initial value is \(c(0)=1\), how long does it take for \(c(t)\) to return to a value of \(1.05\) after it has reached its maximum value?

Admin
Solution

a) Using deviation variables, the rectangular pulse is

\(c_{F}^{\prime}= \begin{cases}0 & t<0 \\ 2 & 0 \leq t<2 \\ 0 & 2 \leq t \leq \infty\end{cases}\)

Laplace transforming this input yields

\(C_{F}^{\prime}(s)=\frac{2}{s}\left(1-e^{-2 s}\right)\)

The input is then given by

\(C^{\prime}(s)=\frac{8}{s(2 s+1)}-\frac{8 e^{-2 s}}{s(2 s+1)}\)

and from Table 3.1 the time domain function is

\(c^{\prime}(t)=8\left(1-e^{-t / 2}\right)-8\left(1-e^{-(t-2) / 2}\right) S (t-2)\)

b) By inspection of Eq. 1, the time at which this function will reach its maximum value is 2 , so maximum value of the output is given by

\(c^{\prime}(2)=8\left(1-e^{-1}\right)-8\left(1-e^{-0 / 2}\right) S (0)\)

and since the second term is zero, \(c^{\prime}(2)=5.057\)

c) By inspection, the steady state value of \(c^{\prime}(t)\) will be zero, since this is a first-order system with no integrating poles and the input returns to zero. To obtain \(c^{\prime}(\infty)\), simplify the function derived in a) for all time greater than 2 , yielding

\(c^{\prime}(t)=8\left(e^{-(t-2) / 2}-e^{-t / 2}\right)\)

which will obviously converge to zero.

Substituting \(c^{\prime}(t)=0.05\) in the previous equation and solving for \(t\) gives

t = 9.233
