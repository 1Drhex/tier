Guest Question.

The beam窶田olumn shown in Figure is pinned at both ends and is subjected to the loads shown. Bending is about the strong axis. Determine whether this member satisfies the appropriate AISC Specification interaction equation.

Admin
Solution

From Manual Table 6-2, the axial compressive design strength of a W10 \(\times\) 49 with \(F_{y}\) = 50 ksi and an effective length of \(L_{c} = K_{y}L = 1.0 \times 17 = 17 \)feet is
\(\phi _{c}P_{n} = 404 kips\)
Since bending is about the strong axis, the design moment, \(\phi _{b}M_{n},\) for\( C_{b}\) = 1.0 can be obtained from Manual Table 6-2.
For an unbraced length \(L_{b} = 17 ft,\)
\(\phi _{b}M_{n} = 197 ft-kips\)
For the end conditions and loading of this problem,\( C_{b} = 1.32\) (see Figurec).
For\( C_{b}\) = 1.32, the design strength is
\(\phi _{b}M_{n} = C_{b} \times 197 = 1.32(197) = 260\) ft-kips
This moment is larger than \(\phi _{b}M_{p} = 227 ft-kips\) (obtained from Table 6-2 as \(\phi _{b}M_{n}\) for\( L_{b}\) = 0), so the design moment must be limited to \(\phi _{b}M_{p}\). Therefore,

\(\phi _{b}M_{n} = 227 ft-kips\)
\(\textbf{Factored loads:}\)
\(P_{u} = 1.2P_{D} + 1.6P_{L} = 1.2(35) + 1.6(99) = 200.4 kips\)

\(Q_{u} = 1.2Q_{D} + 1.6Q_{L} = 1.2(5) + 1.6(12) = 25.2 kips\)
The maximum bending moment occurs at midheight, so
\(M_{u} = \frac{25.2(17)}{4}=107.1 \)ft-kips

Determine which interaction equation controls:
\(\frac{P_{u}}{\phi _{c}P_{n}} =\frac{200.4}{404}= 0.4960 > 0.2 \therefore\) Use Equation 6.3 (AISC Eq. H1-1a).

 

\(\frac{P_{u}}{\phi _{c}P_{n}} +\frac{8}{9}\left (\frac{M_{ux}}{\phi _{b}M_{nx}}+\frac{M_{uy}}{\phi _{b}M_{ny}} \right )=\frac{200.4}{404}+\frac{8}{9}\left ( \frac{107.1}{227}+0 \right )= 0.915 < 1.0 (OK)\)
This member satisfies the AISC Specification.
From Table 6-2, the allowable compressive strength of a W10 \(\times\) 49 with \(F_{y}\) = 50 ksi
and \(L_{c} = K_{y}L = 1.0 \times 17 = 17\) feet is
\(\frac{P_{n}}{\Omega _{c}}=269kips\)
Also from Table 6-2, for \(L_{b}\) = 17 ft and \(C_{b} = 1.0,\)

\(\frac{M_{n}}{\Omega _{b}}= 131\) ft-kips
From Figure c, \(C_{b}\) = 1.32. For \(C_{b} = 1.32,\)
\(\frac{M_{n}}{\Omega _{b}}= C_{b} \times 131 = 1.32(131) = 172.9\) ft-kips
This is larger than \(M_{p}/\Omega _{b}\) = 151 ft-kips, so the allowable moment must be limited
to \(M_{p}/\Omega _{b}\). Therefore,
\(\frac{M_{n}}{\Omega _{b}}= 151\) ft-kips
The total axial compressive load is
\(P_{a} = P_{D} + P_{L} = 35 + 99 = 134 kips\)
The total transverse load is
\(Q_{a} = Q_{D} + Q_{L} = 5 + 12 = 17 kips\)
The maximum bending moment is at midheight
\(M_{a} = \frac{17(17)}{4}= 72.25 ft-kips\)
Determine which interaction equation controls:
\(\frac{P_{a}}{P_{n}/\Omega _{c}} = \frac{134}{269} = 0.4981 >0.2 \therefore\) Use Equation 6.5 (AISC Equation H1-1a).

 

\(\frac{P_{a}}{P_{n}/\Omega _{c}}+\frac{8}{9}\left ( \frac{M_{ax}}{M_{nx}/\Omega _{b}}+\frac{M_{ay}}{M_{ny}/\Omega _{b}} \right )=\frac{134}{269}+\frac{8}{9}\left ( \frac{72.25}{151}+0\right )=0.923 < 1.0 (OK)\)
This member satisfies the AISC Specification.
