Guest Question.

Find the distribution function for the speed \(v_{⊥}\) for motion perpendicular to the z-axis.

 

Admin
Solution

First, we calculate the distribution function for velocity \(v_{⊥}\) perpendicular to the z-axis by integrating M(v) over \(v_{z},\) which we no longer care about. This gives

\(M \left(v_{\bot } \right) = \int_{-\infty }^{\infty }{M \left(v\right) dv_{z} } = \prod\limits_{i=x,y}{\left( \frac{m}{2\pi k_{B} T} \right)^{1/2} } exp \left[-mv^{2}_{i} / 2 k_{B} T\right] . \) (20.18)

Then we go to polar coordinates so that \(v_{x} = v_{\bot } \cos \phi and v_{y} = v_{\bot } \sin \phi\) to obtain

\(M \left(v_{\bot } \right) dv_{x} dv_{y} = \left(\frac{m}{2\pi k_{B} T} \right) exp \left[-mv^{2}_{\bot } /2 k_{B} T\right] d\phi v_{\bot } dv_{\bot }. \) (20.19)

Next we integrate on \(\phi \) from 0 to 2π to get the speed distribution function

\(\check{M} \left(v_{\bot } \right) = \left(\frac{m}{k_{B} T} \right) exp\left[-mv^{2}_{\bot } / 2k_{B} T\right] v_{\bot }.\) (20.20)

Thus the average speed for motion perpendicular to the z-axis is

\(\left\langle v_{\bot } \right\rangle = \int_{0}^{\infty }{\check{M} \left(v_{\bot } \right)} v_{\bot } dv_{\bot } = \int_{0}^{\infty }{\left(\frac{m}{k_{B} T} \right) } exp \left[-mv^{2}_{\bot } / 2k_{B} T\right] v^{2}_{\bot } dv_{\bot } = \left(\frac{\pi k_{B} T}{2m} \right)^{1/2} . \) (20.21)
