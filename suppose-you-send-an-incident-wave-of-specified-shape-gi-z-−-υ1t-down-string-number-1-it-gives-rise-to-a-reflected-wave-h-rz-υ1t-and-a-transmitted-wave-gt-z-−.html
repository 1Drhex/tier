Guest Question.

Suppose you send an incident wave of specified shape, \(g_{I}\left(z-v_{1} t\right)\), down string number 1. It gives rise to a reflected wave, \(h_{R}\left(z+v_{1} t\right)\), and a transmitted wave, \(g_{T}\left(z-v_{2} t\right)\). By imposing the boundary conditions 9.26 and 9.27, find \(h_{R}\) and \(g_{T}\) .

\(f\left(0^{-}, t\right)=f\left(0^{+}, t\right)\) (9.26)

\(\left.\frac{\partial f}{\partial z}\right|_{0^{-}}=\left.\frac{\partial f}{\partial z}\right|_{0^{+}}\) (9.27)

Admin
Solution

\(\text { Equation } 9.26 \Rightarrow g_{I}\left(-v_{1} t\right)+h_{R}\left(v_{1} t\right)=g_{T}\left(-v_{2} t\right) . \text { Now } \frac{\partial g_{I}}{\partial z}=-\frac{1}{v_{1}} \frac{\partial g_{I}}{\partial t} ; \frac{\partial h_{R}}{\partial z}=\frac{1}{v_{1}} \frac{\partial h_{R}}{\partial t} ; \frac{\partial g_{T}}{\partial z}=-\frac{1}{v_{2}} \frac{\partial g_{T}}{\partial t}\) .

\(\text { Equation } 9.27 \Rightarrow-\frac{1}{v_{1}} \frac{\partial g_{I}\left(-v_{1} t\right)}{\partial t}+\frac{1}{v_{1}} \frac{\partial h_{R}\left(v_{1} t\right)}{\partial t}=-\frac{1}{v_{2}} \frac{\partial g_{T}\left(-v_{2} t\right)}{\partial t} \Rightarrow g_{I}\left(-v_{1} t\right)-h_{R}\left(v_{1} t\right)=\frac{v_{1}}{v_{2}} g_{T}\left(-v_{2} t\right)+\kappa\) (where \(\kappa\) is a constant).

Adding these equations, we get \(2 g_{I}\left(-v_{1} t\right)=\left(1+\frac{v_{1}}{v_{2}}\right) g_{T}\left(-v_{2} t\right)+\kappa, \text { or } g_{T}\left(-v_{2} t\right)=\left(\frac{2 v_{2}}{v_{1}+v_{2}}\right) g_{I}\left(-v_{1} t\right)+\kappa^{\prime}\)

(where \(\kappa^{\prime} \equiv-\kappa \frac{v_{2}}{v_{1}+v_{2}}\) ). Now \(g_{I}(z, t), g_{T}(z, t), \text { and } h_{R}(z, t)\) are each functions of a single variable \(u \) (in the first case \(\left.u=z-v_{1} t, \text { in the second } u=z-v_{2} t, \text { and in the third } u=z+v_{1} t\right)\) .

Thus \(g_{T}(u)=\left(\frac{2 v_{2}}{v_{1}+v_{2}}\right) g_{I}\left(v_{1} u / v_{2}\right)+\kappa^{\prime}\).

Multiplying the first equation \(v_{1} / v_{2} \text { and subtracting, }\left(1-\frac{v_{1}}{v_{2}}\right) g_{I}\left(-v_{1} t\right)-\left(1+\frac{v_{1}}{v_{2}}\right) h_{R}\left(v_{1} t\right)=\kappa \Rightarrow h_{R}\left(v_{1} t\right)=\left(\frac{v_{2}-v_{1}}{v_{1}+v_{2}}\right) g_{I}\left(-v_{1} t\right)-\kappa\left(\frac{v_{2}}{v_{1}+v_{2}}\right), \text { or } h_{R}(u)=\left(\frac{v_{2}-v_{1}}{v_{1}+v_{2}}\right) g_{I}(-u)+\kappa^{\prime}\) .
