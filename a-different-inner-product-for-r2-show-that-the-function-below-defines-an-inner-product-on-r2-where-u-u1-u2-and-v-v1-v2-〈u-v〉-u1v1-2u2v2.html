Guest Question.

A Different Inner Product for \(R^2\)
Show that the function below defines an inner product on \(R^2\) , where \(u=\left(u_1,u_2\right) and v=\left(v_1,v_2\right). 〈u, v〉 = u_1v_1+ 2u_2v_2\).

Admin
Solution

1. The product of real numbers is commutative, so
\(〈u, v〉 = u_{1}v_{1}+ 2u_{2} v_{2}= v_{1} u_{1}+ 2v_{2} u_{2}= 〈v, u〉\).
2. Let \(w = \left(w_1, w_2\right) \ Then \ 〈u, v + w〉 = u_1 \left(v_1+ w_1\right) + 2u_2 \left(v_2+ w_2\right) = u_1v_1+ u_1w_1+ 2u_2v_2+ 2u_2w_2 = \left(u_1v_1+ 2u_2v_2\right) + \left(u_1w_1+ 2u_2w_2\right)= 〈u, v〉 + 〈u, w〉\).
3. If c is any scalar, then
\(c〈u, v〉 = c\left(u_1v_1+ 2u_2v_2\right) = \left(cu_1\right) v_1+ 2\left(cu_2\right) v_2= 〈cu, v〉\).
4. The square of a real number is nonnegative, so \(〈v, v〉 = v^{2}_{1} + v^{2}_{1}\geq 0\).
Moreover, this expression is equal to zero if and only if\( v = 0\) (that is, if and only if \(v_1= v_2= 0\)).
