Guest Question.

A hollow cylindrical tube with a rectangular cross section has external dimensions of 0.5 in by 1 in and a wall thickness of 0.05 in. Assume that the material is brass for which \(\sigma = 1.5 \times { 10 }^{ 7 }\) S/m. A current of 200 A dc is flowing down the tube. (a) What voltage drop is present across a 1 m length of the tube? (b) Find the voltage drop if the interior of the tube is filled with a conducting material for which \(\sigma = 1.5 \times { 10 }^{ 5 }\) S/m.

Admin
Solution

a) Converting all measurements to meters, the tube resistance over a 1 m length will be:
\({ R }_{ 1 } = \frac { 1 }{ \left( 1.5\times { 10 }^{ 7 } \right) \left[ \left( 2.54 \right) \left( 2.54/2 \right) \times { 10 }^{ -4 }-2.54\left( 1-.1 \right) \left( 2.54/2 \right) \left( 1-.2 \right) \times { 10 }^{ -4 } \right] } =7.38\times { 10 }^{ -4 }\Omega\)
The voltage drop is now V = \(I { R }_{ 1 } = 200(7.38\times { 10 }^{ -4 }) = 0.147\) V.
b) The resistance of the ﬁlling will be:
\({ R }_{ 2 } = \frac { 1 }{ \left( 1.5\times { 10 }^{ 5 } \right) \left( 1/2 \right) { \left( 2.54 \right) }^{ 2 }\times { 10 }^{ -4 }(.9)(.8) } =2.87\times { 10 }^{ -2 }\Omega\)
The total resistance is now the parallel combination of \({ R }_{ 1 }\) and \({ R }_{ 2 }\):
\({ R }_{ T } = { R }_{ 1 }{ R }_{ 2 }/({ R }_{ 1 } + { R }_{ 2 }) = 7.19\times { 10 }^{ -4 }\Omega\), and the voltage drop is now V = 200\({ R }_{ T }\) = .144 V.
