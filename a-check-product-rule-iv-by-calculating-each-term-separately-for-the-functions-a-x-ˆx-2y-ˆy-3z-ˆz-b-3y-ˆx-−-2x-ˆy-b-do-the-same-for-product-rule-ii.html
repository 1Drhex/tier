Guest Question.

(a) Check product rule (iv) (by calculating each term separately) for the functions

\(A=x\hat{x} +2y\hat{y}+3z\hat{z} ; B=3y\hat{x}-2x\hat{y}\)

(b) Do the same for product rule (ii).

(c) Do the same for rule (vi).

Admin
Solution

(a) A×B = \(\left | \begin{matrix}\hat{x} & \hat{y} & \hat{z} \\ x & 2y & 3z \\ 3y & -2x & 0 \end{matrix} \right |=\hat{x}(6xz)+\hat{y}(9zy) +\hat{z}\left(-2x^{2}-6y^{2} \right) \)

∇·(A×B) =\(\frac{\partial}{\partial x}(6xz)+\frac{\partial}{\partial y}(9zy)+\frac{\partial}{\partial z} \left(-2x^{2}-6y^{2} \right)=6z + 9z + 0 = 15z \)

∇×A =\(\hat{x}\left(\frac{\partial}{\partial y}(3z) −\frac{\partial}{\partial z}(2y)\right)+\hat{y}\left(\frac{\partial}{\partial z}(x) −\frac{\partial}{\partial x} (3z) \right)+\hat{z} \left(\frac{\partial}{\partial x} (2y) −\frac{\partial}{\partial y} (x)\right)=0 \); B·(∇×A) = 0

∇×B =\(\hat{x}\left(\frac{\partial}{\partial y}(0) −\frac{\partial}{\partial z}(-2x)\right)+\hat{y}\left(\frac{\partial}{\partial z}(3y) −\frac{\partial}{\partial x} (0) \right)+\hat{z} \left(\frac{\partial}{\partial x} (-2x) −\frac{\partial}{\partial y} (3y)\right)=-5\hat{z}; \) A·(∇×B) = −15z

∇·(A×B)\(\overset{?}{=} \)B·(∇×A) − A·(∇×B) = 0 − (−15z) = 15z.

(b) A·B = 3xy − 4xy = −xy ; ∇(A·B) = ∇(−xy) =\(\hat{x}\frac{\partial}{\partial x}(−xy) +\hat{y}\frac{\partial}{\partial y} (−xy) = −y\hat{x}− x\hat{y} \)

A×(∇×B) = \(\left | \begin{matrix} \hat{x} & \hat{y} & \hat{z} \\ x & 2y & 3z \\ 0 & 0 & -5 \end{matrix} \right | =\hat{x}(−10y) +\hat{y}(5x); \) B×(∇×A) = 0

(A·∇)B =\(\left(x\frac{\partial}{\partial x} +2y\frac{\partial}{\partial y} +3z\frac{\partial}{\partial z} \right)(3y \hat{x} − 2x \hat{y} ) =\hat{x}(6y) + \hat{y} (−2x)\)

(B·∇)A =\(\left(3y\frac{\partial}{\partial x} -2y\frac{\partial}{\partial y} \right)(x \hat{x} + 2y \hat{y} + 3z \hat{z} ) = \hat{x} (3y) + \hat{y} (−4x) \)

A×(∇×B) + B×(∇×A) + (A·∇)B + (B·∇)A

=\(−10y \hat{x} + 5x \hat{y} + 6y \hat{x} − 2x \hat{y} + 3y \hat{x} − 4x \hat{y} = −y \hat{x} − x \hat{y} =\) = ∇·(A·B).

(c) ∇×(A×B) =\(\hat{x} \left(\frac{\partial }{\partial y} \left(-2x^{2}-6y^{2} \right) -\frac{\partial}{\partial z}\left(9zy\right) \right) +\hat{y} \left(\frac{\partial}{\partial z}\left(6xz\right) -\frac{\partial}{\partial x} \left(-2x^{2}-6y^{2} \right) \right)+\hat{z} \left(\frac{\partial}{\partial x} (9zy)-\frac{\partial}{\partial y}(6xz) \right)\)

=\(\hat{x}(−12y − 9y) + \hat{y} (6x + 4x) +\hat{z} (0) = −21y \hat{x} + 10x \hat{y}\)

∇·A =\(\frac{\partial}{\partial x}(x) +\frac{\partial}{\partial y}(2y) +\frac{\partial}{\partial z}(3z) = 1 + 2 + 3 = 6;\) ∇·B =\(\frac{\partial}{\partial x}(3y) +\frac{\partial}{\partial y} (−2x) = 0 \)

(B·∇)A − (A·∇)B + A(∇·B) − B(∇·A) =\(3y \vec{x} − 4x \hat{y} − 6y \hat{x} + 2x \hat{y} − 18y \hat{x} + 12x \hat{y} = −21y \hat{x} + 10x \hat{y} \) = ∇×(A×B).
