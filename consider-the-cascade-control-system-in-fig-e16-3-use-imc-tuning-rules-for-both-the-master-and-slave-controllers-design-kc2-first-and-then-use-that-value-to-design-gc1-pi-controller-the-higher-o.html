Guest Question.

Consider the cascade control system in Fig. E16.3. Use IMC tuning rules for both the master and slave controllers. Design \(K_{c 2}\) first, and then use that value to design \(G_{c 1}\) (PI controller). The higher-order transfer function can be approximated first by a FOPTD model using a step test. Plot closed-loop responses for different values of the IMC closed-loop time constant for both outer loop and inner loop for a set point change.

Admin
Solution

For the inner controller (Slave controller), IMC tuning rules are used

\(G_{c 2} *=\frac{1}{G_{2}^{-}}=\frac{(2 s+1)(5 s+1)(s+1)}{\left(\tau_{c 2} s+1\right)^{3}}\)

Closed-loop responses for different values of \(\tau_{c 2}\) are shown below. A \(\tau_{c 2}\) value of 3 yields a good response.

For the Master controller,

\(G_{c 1} *=\frac{1}{G_{1}^{-}} \quad \text { where } \quad G_{1}^{-}=\frac{(2 s+1)(5 s+1)(s+1)}{\left(\tau_{c 1} s+1\right)^{3}} \frac{1}{(10 s+1)}\)

This higher-order transfer function is approximated by first order plus time delay using a step test:

Hence \(G_{1}^{-} \approx \frac{e^{-0.38 s}}{(15.32 s+1)}\)

From Table 12.1: (PI controller, Case G): \(K_{c}=\frac{15.32}{\tau_{c 1}+0.38}\) and \(\tau_{i}=15.32\) Closed-loop responses are shown for different values of \(\tau_{c 1 .}\) A \(\tau_{c 1}\) value of 7 yields a good response.

Hence for the master controller, \(K_{c}=2.07\) and \(\tau_{I}=15.32\)
