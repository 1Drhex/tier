Guest Question.

Network Function with Complex Poles. The network function of a second-order low-pass filter has the form
\(H(ω) = \frac{kω_0^2}{(jω)^2 + j2\zeta ω_0ω + ω_0^2}\)

This network function depends on three parameters: the dc gain k; the corner frequency \(ω_0\); and the damping ratio \(\zeta\). For convenience, we consider the case where k = 1. Then, using \(j^2 = -1\), we can write the network function as
\(H(ω) = \frac{ω_0^2}{ω_0^2 - ω^2 + j2\zeta ω_0ω}\)

Determine the asymptotic magnitude Bode plot of the second-order low-pass filter when the dc gain is 1.

Admin
Solution

The denominator of H(ω) contains a new factor, one that involves \(ω^2\). The asymptotic Bode plot is based on the approximation

\((ω_0^2 – ω^2) + j2\zeta ω_0ω\cong \begin{cases} ω_0^2 & ω \lt ω_0\\ – ω^2 & ω \gt ω_0 \end{cases}\)

Using this approximation, we can express H(ω) as

\(H(\omega)\cong \begin{cases} 1 & \omega \lt \omega_0 \\ -\frac{\omega_0^2}{\omega^2} & \omega \gt \omega_0 \end{cases}\)

The logarithmic gain is

\(20\log_{10} |H(\omega)| \cong \begin{cases} 0 & \omega \lt \omega_0 \\ 40\log_{10} \omega_0-40\log_{10} \omega & \omega \gt \omega_0 \end{cases}\)

The asymptotic magnitude Bode plot is shown in Figure 13.3-10. The actual magnitude Bode plot and the actual phase Bode plot are shown in Figure 13.3-11. The asymptotic Bode plot is a good approximation to the actual Bode plot when \(\omega \ll \omega_0 or \omega \gg \omega_0\). Near \(\omega = \omega_0\), the asymptotic Bode plot deviates from the actual Bode plot. At \(\omega = \omega_0\), the value of the asymptotic Bode plot is 0 dB, whereas the value of the actual Bode plot is

\(H(\omega_0) = \frac{1}{2\zeta}\)

As this equation and Figure 13.3-11 both show, the deviation between the actual and asymptotic Bode plot near \(\omega = \omega_0\) depends on \(\zeta\). The frequency \(\omega_0\) is called the corner frequency. The slope of the asymptotic Bode plot decreases by 40 dB/decade as the frequency increases past \(\omega = \omega_0\). In terms of the asymptotic Bode plot, the denominator of this network function acts like two poles at \(p = \omega_0\). If this factor were to appear in the numerator of a network function, it would act like two zeros at \(z = \omega_0\). The slope of the asymptotic Bode plot would increase by 40 dB/decade as the frequency increased past \(\omega = \omega_0\).
