Guest Question.

The four-bar linkage in the posture shown in Fig. 3.7a is driven by crank 2 at a constant angular velocity ω2 = 900 rev/min ccw. For this posture, determine the angular velocities of coupler link 3 and output link 4 and the velocities of point E in link 3 and point F in link 4.

Admin
Solution

GRAPHIC SOLUTION
First, we draw the linkage to a suitable scale. Then, we calculate the angular velocity of link 2 in radians per second; that is,

\( \omega_{2}=\left(900 \frac{ rev }{ min }\right)\left(2 \pi \frac{ rad }{ rev }\right)\left(\frac{1 min }{60 s }\right)=94.25 rad / s ccw \) (1)

Next we calculate the velocity of point B from Eqs. (3.3) and (3.4), noting that point A is fixed:

\(V _{P Q}=\frac{d R _{P Q}}{d t}= \omega \times R _{P Q}\) (3.3)

 

\(V _{P}= V _{Q}+ V _{P Q}\) (3.4)

 

\(V _{B}= \not V _{A}^{ 0 }+ V _{B A}= \omega _{2} \times R _{B A},\)

 

\(V_{B}=(94.25 rad / s )\left(\frac{4}{12} ft \right)=31.42 ft / s.\) (2)

 

It is important to note that the form “ω×R” was used for the velocity difference and not for the absolute velocity VB directly. Next, we choose a suitable velocity scale and an arbitrary location for an origin, denoted as OV, shown in Fig. 3.7b. Since VA = 0, the image point A is coincident with OV. We construct the line AB perpendicular to RBA and toward the lower left because of the counterclockwise sense of ω2; this line represents VBA.

Now we write two equations for the velocity of point C. Since the velocities of points C3 and C4 must be equal (links 3 and 4 are pinned together at C), we have

\(V_{C} = \overset{\sqrt{}\sqrt{}}{V_{B}} + \overset{?\sqrt{}}{V_{CB}} = \not V_{D}^{0} + \overset{?\sqrt{}}{V_{CD}}.\) (3)

 

We construct two lines in the velocity polygon; line BC is drawn from image point B perpendicular to RCB, and line DC is drawn from image point D (coincident with OV since VD = 0) perpendicular to RCD. We label the point of intersection of these two lines as image point C. When the lengths of these two lines are measured, we find that VCB = 38.4 ft/s and VC = VCD = 45.5 ft/s. The angular velocities of links 3 and 4 can now be obtained:

 

\(\omega_{3}=\frac{V_{C B}}{R_{C B}}=\frac{38.4 ft / s }{(18 / 12) ft }=25.6 rad / s ccw\) (4)

 

\(\omega_{4}=\frac{V_{C D}}{R_{C D}}=\frac{45.5 ft / s }{(11 / 12) ft }=49.64 rad / s ccw\) (5)

 

where the directions of ω3 and ω4 are obtained using the technique illustrated in Fig. 3.6c. There are several graphic methods for finding the velocity of point E. One method is to measure REB from the scale drawing of the linkage. Then, since points B and E are both attached to link 3, we calculate*

\(V_{E B}=\omega_{3} R_{E B}=(25.6 rad / s )\left(\frac{10.8}{12} ft \right)=23.04 ft / s\) (6)

We next construct the line BE in the velocity polygon, drawn to the chosen scale and perpendicular to REB . Then we solve† the velocity-difference equation

\(\overset{??}{V_{E}} = \overset{\sqrt{}\sqrt{}}{V_{B}} + \overset{\sqrt{}\sqrt{}}{V_{EB}}.\) (7)

From the velocity polygon, the velocity of point E is measured as

VE = 27.6 ft/s ∠165.3º.

Alternatively, the velocity of point E can be obtained from the velocity-difference equation

\(\overset{??}{V_{E}} = \overset{\sqrt{}\sqrt{}}{V_{C}} + \overset{\sqrt{}\sqrt{}}{V_{EC}}.\) (8)

by a procedure identical to that used for Eq. (7). This yields the triangle OVEC in the velocity polygon.

Suppose we wish to find VE without the intermediate step of calculating ω3. In this case we write Eqs. (7) and (8) simultaneously; that is,

\(V_{E} = \overset{\sqrt{}\sqrt{}}{V_{B}} + \overset{?\sqrt{}}{V_{EB}} = \overset{\sqrt{}\sqrt{}}{V_{C}} + \overset{?\sqrt{}}{V_{EC}}.\) (9)

 

Drawing lines EB (perpendicular to REB ) and EC (perpendicular to REC) in the velocity polygon, we find their point of intersection and so solve Eq. (9).

Perhaps the easiest method of solving for VE, however, is to take advantage of the concept of the velocity image of link 3. Recognizing that the velocity-image points B and C have already been found, we construct triangle BEC in the velocity polygon, similar in shape to triangle BEC in Fig. 3.7a. This locates point E in the velocity polygon and, therefore, gives a solution for the velocity of point E.

The velocity of point F can also be found by any of the previous methods using points C, D, and F of link 4. The result is

VF = 31.8 ft/s ∠130.9º.

For the purpose of comparison, an analytic solution is also presented here.

ANALYTIC SOLUTION

The first step is to perform a posture analysis of the linkage. Since this step was presented in Chap. 2, only the results are shown here. For the given link dimensions and the specified input variable θ2 = 120º, the postures of links 3 and 4 are θ3 = 20.92º and θ4 = 64.05º, respectively.

In vector form, the position-difference vectors corresponding to the links are:

\(R _{B A}=4 \text { in } \angle 120^{\circ}=-2 \hat{ i }+3.464 \hat{ j } \text { in },\)

 

\(R _{C B}=18 \text { in } \angle 20.92^{\circ}=16.813 \hat{ i }+6.427 \hat{ j } \text { in },\)

 

\(R _{C D}=11 \text { in } \angle 64.05^{\circ}=4.813 \hat{ i }+9.891 \hat{ j } \text { in },\)

 

\(R _{D A}=10 \text { in } \angle 0^{\circ}=10 \hat{ i } \text { in },\)

 

\(R _{E B}=10.77 \text { in } \angle-0.88^{\circ}=10.769 \hat{ i }-0.165 \hat{ j } \text { in },\)

 

\(R _{F D}=7.616 \text { in } \angle 40.85^{\circ}=5.761 \hat{ i }+4.981 \hat{ j } \text { in }.\)

 

We proceed as we did in the graphic solution—namely, the given input angular velocity is

\(\omega_{2}=\left(900 \frac{ rev }{ min }\right)\left(2 \pi \frac{ rad }{ rev }\right)\left(\frac{1 min }{60 s }\right)=94.25 rad / s ccw\) (10)

 

Then the velocity of point B is written as

\(V _{B}= \not V _{A}^{ 0 }+ V _{B A}= \omega _{2} \times R _{B A}\)

 

\(=-326.482 \hat{ i }-188.500 \hat{ j } in / s =377 in / s \angle-150^{\circ}.\) (11)

 

The velocity of point C can be written as

 

\(V_{C} = \overset{\sqrt{}\sqrt{}}{V_{B}} + \overset{?\sqrt{}}{V_{CB}} = \not V_{D}^{0} + \overset{?\sqrt{}}{V_{CD}}\)

 

\(= V _{B}+ \omega _{3} \times R _{C B}= \omega _{4} \times R _{C D}\) (12)

 

Substituting the position-difference vectors RCB and RCD and Eq. (11) into Eq. (12) and writing the result in matrix form gives

\( \left[\begin{array}{c} -326.482 in / s \\ -188.500 in / s \end{array}\right]+\left[\begin{array}{c} -6.427 in \\ 16.813 in \end{array}\right] \omega_{3}=\left[\begin{array}{c} -9.891 in \\ 4.813 in \end{array}\right] \omega_{4}. \) (13)

 

Solving Eq. (13), the angular velocities of links 3 and 4, respectively, are

ω3 = 25.382 rad/s (ccw) and ω4 = 49.501 rad/s (ccw). (14)

The velocities of points E and F can be obtained from the velocity-difference equation using the velocities of points B and D; that is

VE = VB +VEB = VB +ω3 × REB and VF = VD +VFD = VD + ω4 × RFD. (15)

Substituting the known data into Eq. (15) gives

\( V _{E}=-322.294 \hat{ i }+84.839 \hat{ j } \text { in } / s =27.8 ft / s \angle 165.25^{\circ}\)

and

\(V _{F}=-246.564 \hat{ i }+285.175 \hat{ j } \text { in } / s =31.4 ft / s \angle 130.85^{\circ}.\)

 

Note that these answers are in good agreement with the answers obtained by the graphic method.

*There is no restriction in our derivation that requires that REB lie along the material portion of link 3 in order to use Eq. (6), only that points E and B remain a constant distance apart.
†Note that numeric values should not be substituted into Eq. (7) directly. This is a vector equation and requires vector addition, not scalar; performing these vector operations is precisely the purpose of constructing the velocity polygon.
