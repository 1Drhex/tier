Guest Question.

Verify that a built-up, ASTM A572 grade 50, column with PL1in.×8in. flanges and a \(PL 1 / 4 in \cdot \times 15 in .\). web is sufficient to carry a dead load of 70 kips and live load of 210 kips in axial compression. The column length is 15 ft and the ends are pinned in both axes.

Admin
Solution

Material Properties:

\(\text { ASTM A572 Grade 50 } \quad F_{y}=50 ksi \quad F_{u}=65 ksi\) Manual Table 2-3

Geometric Properties:

Built-up Column \(d=17.0 \text { in. } \quad b_{f}=8.00 \text { in. } \quad t_{f}=1.00 \text { in. } \quad h=15.0 \text { in. } \quad t_{w}=0.250 \text { in. }\).

Calculate the required strength

LRFD ASD
\(P_{u}=1.2(70.0 kips) + 1.6(210 kips)\)

= 420 kips

 \(P_{a}=70.0 \text { kips }+210 \text { kips }\)

= 280 kips

Calculate built-up section properties (ignoring fillet welds)

\(A=2(8.00 \text { in. })(1.00 \text { in. })+(15.0 \text { in. })(0.250 \text { in. })=19.8 \text { in. }^{2}\)

 

\(I_{y}=\frac{2(1.00 \text { in. })(8.00 in .)^{3}}{12}+\frac{(15.0 in .)(0.250 in .)^{3}}{12}=85.4 in .{ }^{4}\)

 

\(r_{y}=\sqrt{\frac{I_{y}}{A}}=\sqrt{\frac{85.4 in ^{4}}{19.8 in ^{2}}}=2.08 in\)

 

\(I_{x}=\sum A d^{2}+\sum I_{x}\)

 

\(=2\left(8.00 in. ^{2}\right)(8.00 in .)^{2}+\frac{(0.250 in .)(15.00 in .)^{3}}{12}+\frac{2(8.0 in .)(1.0 in .)^{3}}{12}=1100 in. ^{4}\)

 

Calculate the elastic flexural buckling stress Section E7

For a pinned-pinned condition, K = 1.0 Commentary Table C-C2.2

Since the unbraced length is the same for both axes, the y-y axis will govern by inspection.

\(\frac{K L_{y}}{r_{y}}=\frac{1.0(15.0 ft )}{2.08 in. } \frac{12.0 in .}{ ft }=86.6\)

 

\(F_{e}=\frac{\pi^{2} E}{\left(\frac{K L}{r}\right)^{2}}=\frac{\pi^{2}(29,000 ksi )}{(86.6)^{2}}=38.2 ksi\) Eqn. E3-4

Calculate the elastic critical torsional buckling stress

Note: Torsional buckling will not govern if \(K L_{y}>K L_{z}\) , however, the check is included here to illustrate the calculation.

\(C_{w}=\frac{I_{y} h_{o}^{2}}{4}=\frac{85.4 in .{ }^{4}(16.0 in .)^{2}}{4}=5470 in .{ }^{6}\) Design Guide No. 9 Eqn. 3-5

 

\(J=\sum \frac{b t^{3}}{3}=\frac{2(8.00 in. )(1.00 in .)^{3}+(15.0 in. )(0.250 in. )^{3}}{3}=5.41 in ^{4}\) Design Guide No. 9 Eqn. 3-4

\(F_{e}=\left[\frac{\pi^{2} E C_{w}}{\left(K_{z} L\right)^{2}}+G J\right] \frac{1}{I_{x}+I_{y}}\) Eqn. E4-4

 

\(=\left[\frac{\pi^{2}(29,000 ksi )\left(5470 in ^{6}\right)}{(180 in )^{2}}+(11,200 ksi )\left(5.41 in .^{4}\right)\right] \frac{1}{1100 in ^{4}+85.4 in ^{4}}\)

 

= 92.2 ksi > 38.2 ksi

 

Therefore, the flexural buckling limit state controls.

\(\text { Use } F_{e}=38.2 ksi\)

Check for slender elements using Specification Section E7

Determine \(Q_{s}\), the unstiffened element (flange) reduction factor

\(\text { Calculate } k_{c}\)

 

\(k_{c}=\frac{4}{\sqrt{h / t_{w}}}=\frac{4}{\sqrt{15.0 / 0.250}}=0.516\) which is between 0.35 and 0.76 Table B4.1 Note [a]

 

\(\frac{b}{t}=\frac{4.00 in }{1.00 in .}=4.0<0.64 \sqrt{\frac{k_{c} E}{F_{y}}}=0.64 \sqrt{\frac{0.516(29,000 ksi )}{50 ksi }}=11.1\) Table B4.1 Case 4

Therefore, the flange is not slender.

\(Q_{s}=1.0\)

Determine \(Q_{a}\), the stiffened element (web) reduction factor

\(\frac{h}{t}=\frac{15.0 in .}{0.250 in .}=60.0>1.49 \sqrt{\frac{E}{F_{v}}}=1.49 \sqrt{\frac{29,000 ksi }{50 ksi }}=35.9\) , therefore the web is slender

 

\(Q_{a}=\frac{A_{e f f}}{A} \text { where } A_{\text {eff }}\) is effective area based on the reduced effective width of the web, \(b_{e}\). Section E7.2

For equation E7-17, take f as \(F_{c r} \text { with } Q=1.0\) Eqn. E7-16

KL /r = 86.6 from above

\(4.71 \sqrt{\frac{E}{Q F_{y}}}=4.71 \sqrt{\frac{29,000 ksi }{1.0(50 ksi )}}=113>86.6\)

 

When \(\frac{K L}{r} \leq 4.71 \sqrt{\frac{E}{Q F_{y}}}\)

 

\(F_{c r}=Q\left[0.658^{\frac{Q F_{y}}{F_{e}}}\right] F_{y}=1.0\left[0.658^{\frac{1.0(50 ksi )}{38.2 ksi }}\right](50 ksi )=28.9 ksi\) Eqn. E7-2

 

\(b_{e}=1.92 t \sqrt{\frac{E}{f}}\left[1-\frac{0.34}{(b / t)} \sqrt{\frac{E}{f}}\right] \leq b, \text { where } b=h\) Eqn. E7-17

 

\(=1.92(0.250 \text { in. }) \sqrt{\frac{29,000 ksi }{28.9 ksi }}\left[1-\frac{0.34}{(15.0 in . / 0.250 in .)} \sqrt{\frac{29,000 ksi }{28.9 ksi }}\right] \leq 15.0 \text { in. }\)

 

\(=12.5 \text { in. }<15.0 \text { in. }\) , therefore compute \(A_{e f f}\) with reduced effective web width.

\(A_{e f f}=b_{e} t_{w}+2 b_{f} t_{f}=(12.5 in .)(0.250 in .)+2(8.00 in .)(1.00 in .)=19.1 in .{ }^{2}\)

 

\(Q_{a}=\frac{A_{e f f}}{A}=\frac{19.1 in .^{2}}{19.8 in .{ }^{2}}=0.966\) Eqn. E7-16

 

\(Q=Q_{s} Q_{a}=(1.00)(0.966)=0.966\)

Determine whether Specification Equation E7-2 or E7-3 applies

KL/r = 86.6 from above

\(4.71 \sqrt{\frac{E}{Q F_{y}}}=4.71 \sqrt{\frac{29,000 ksi }{0.966(50 ksi )}}=115.4>86.6\)

 

Therefore, Specification Equation E7-2 applies.

\(F_{c r}=Q\left[0.658^{\frac{Q F_{y}}{F_{e}}}\right] F_{y}=0.966\left[0.658^{\frac{0.966(50 ksi )}{38.2 ksi }}\right](50 ksi )=28.5 ksi\) Eqn. E7-2

Calculate the nominal compressive strength

\(P_{n}=F_{c r} A_{g}=28.5 ksi \left(19.8 in ^{2}\right)=562 \text { kips }\) Eqn. E7-1

LRFD ASD
\(\phi_{c}=0.90\) \(\Omega_{ c }=1.67\)
\(\phi_{c} P_{n}=0.90(562 kips )\)

= 505 kips > 420 kips o.k.

 \(P_{n} / \Omega_{c}=562 kips / 1.67\)

= 336 kips > 280 kips o.k.
