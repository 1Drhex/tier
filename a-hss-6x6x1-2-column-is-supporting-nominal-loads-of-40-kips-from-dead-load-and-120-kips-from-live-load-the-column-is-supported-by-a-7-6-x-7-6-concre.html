Guest Question.

A HSS 6×6×1/2 column is supporting nominal loads of 40 kips from dead load and 120 kips from live load. The column is supported by a 7’-6” × 7’-6” concrete spread footing with a \(f_{ c }^{\prime}\) = 3000 psi. Size a base plate for this column.

Admin
Solution

Material Properties:

Column ASTM A500 Gr. B \(F_{y}=46 ksi \quad F_{u}=58 ksi\)

Base Plate ASTM A36 \(F_{y}=36 ksi \quad F_{u}=58 ksi\)

Manual
Table 2-3

Geometric Properties:

HSS 6×6×1/2 B = H = 6.00 in.

Calculate the required strength

LRFD ASD
\(\begin{aligned}P_{u} &=1.2(40.0 kips )+1.6(120 kips ) \\&=240 kips\end{aligned}\) \(\begin{aligned}P_{a} &=40.0 \text { kips }+120 \text { kips } \\&=160 \text { kips }\end{aligned}\)

Note: The procedure illustrated here is similar to that presented in AISC Design Guide 1- Column Base Plates

Try a base plate which extends 3 1/2 in. from each face of the HSS column, or 13 in. by 13 in.

Calculate the available strength for the limit state of concrete crushing

\(P_{p}=0.85 f_{c}^{\prime} A_{1} \sqrt{A_{2} / A_{1}} \text { where } A_{2} / A_{1} \leq 4 \)

Eqn. J8-2

\(\begin{array}{l}A_{1}=(13.0 . \text { in. })(13.0 in .)=169 in ^{2} \\A_{2}=(90.0 . in .)(90.0 in .)=8100 in ^{2}\end{array}\)

 

\(P_{p}=0.85(3 ksi )\left(169 in .^{2}\right) \sqrt{\frac{8100 in ^{2}}{169 in ^{2}}} \leq 1.7(3 ksi )\left(169 in .^{2}\right)\)

 

\(20,700 \text { kips } \leq 862 \text { kips }\)

 

\(\text { Use } P_{p}=862 \text { kips }\)

Note: The limit on the right side of Equation J8-2 will control when \(A_{2} / A_{1}\) exceeds 4.0

LRFD ASD
\(\begin{aligned}\phi_{c} &=0.60 \\\phi_{c} P_{p} &=0.60(862 \text { kips }) \\&=517 \text { kips }>240 \text { kips }\end{aligned}\) \(\begin{aligned}\Omega_{c} &=2.50 \\P_{p} / \Omega_{c} &=862 kips / 2.50 \\&=345 \text { kips }>160 kips\end{aligned}\)

Section J8

Calculate the pressure under the bearing plate and determine the required thickness

For a rectangular HSS, the distance m or n is determined using 0.95 times the depth and width of the HSS.

Note: When the HSS is large compared to the edge extension, a quick and conservative approach is to include in the bearing area only the surface within the distance m or n of the HSS wall within the shape.

\(m=n=\frac{N-0.95 \text { (outside dimension) }}{2}\)

 

\(=\frac{13.0 \text { in. }-0.95(6.00 in .)}{2}=3.65 \text { in. }>\frac{[6-2(1 / 2)]}{2}=2.50 \text { in. , therefore, the full area of }\)

the baseplate inside the HSS is effective in bearing.

The critical bending moment is the cantilever moment outside the HSS perimeter.

LRFD ASD
\(\begin{aligned}f_{p u} &=\frac{P_{u}}{A_{e f f}}=\frac{240 kips }{169 in .^{2}}=1.42 ksi \\M_{u} &=\frac{f_{p u} l^{2}}{2} \\Z &=\frac{t_{p}^{2}}{4}\end{aligned}\)

 

\(\phi_{b}=0.90\)

 

\(M_{n}=M_{p}=F_{y} Z\)

 

Equating:

\(M_{u}=\phi_{b} M_{n} \text { and solving for } t_{p} \text { gives: }\)

 

\(\begin{aligned}t_{p(\text { req })} &=\sqrt{\frac{2 f_{p u} l^{2}}{\phi_{b} F_{y}}} \\&=\sqrt{\frac{2(1.42 ksi )(3.65 in .)^{2}}{0.90(36 ksi )}}=1.08 in .\end{aligned}\)

 \(\begin{aligned}f_{p a} &=\frac{P_{a}}{A_{e f f}}=\frac{160 kips }{169 in .^{2}}=0.947 ksi \\M_{a} &=\frac{f_{p a} l^{2}}{2} \\Z &=\frac{t_{p}^{2}}{4}\end{aligned}\)

 

\(\Omega_{b}=1.67\)

 

\(M_{n}=M_{p}=F_{y} Z\)

Eqn. F11-1

Equating:

\(M_{a}=M_{n} / \Omega_{b} \text { and solving for } t_{p} \text { gives: }\)

 

\(\begin{aligned}t_{p(\text { req })} &=\sqrt{\frac{2 f_{p a} l^{2}}{F_{y} / \Omega_{b}}} \\&=\sqrt{\frac{2(0.947 ksi )(3.65 in .)^{2}}{(36 ksi ) / 1.67}}=1.08 in .\end{aligned}\)

Therefore, use a 1/4 in. thick plate.
