Guest Question.

For the steam-heated stirred-tank system model in Eqs. 2-51 and 2-52, assume that the steam temperature \(T_{s}\) is constant.

(a) Find a transfer function relating tank temperature \(T\) to inlet liquid temperature \(T_{i}\).

(b) What is the steady-state gain for this transfer function?

(c) Based on physical arguments only, should the gain be unity? Justify your answer.

\(m C \frac{d T}{d t}=w C\left(T_{i}-T\right)+h_{p} A_{p}\left(T_{w}-T\right)\) (2-51)

\(m_{w} C_{w} \frac{d T_{w}}{d t}=h_{s} A_{s}\left(T_{s}-T_{w}\right)-h_{p} A_{p}\left(T_{w}-T\right)\) (2-52)

Admin
Solution

a) The model for the system is given by

\(m C \frac{d T}{d t}=w C\left(T_{i}-T\right)+h_{p} A_{p}\left(T_{w}-T\right)\) (2-51)

\(m_{w} C_{w} \frac{d T_{w}}{d t}=h_{s} A_{s}\left(T_{s}-T_{w}\right)-h_{p} A_{p}\left(T_{w}-T\right)\) (2-52)

Assume that \(m, m_{ w }, C, C_{w}, h_{p}, h_{s}, A_{p}, A_{s}\), and \(w\) are constant. Rewriting the above equations in terms of deviation variables, and noting that

\(\frac{d T}{d t}=\frac{d T^{\prime}}{d t} \quad, \quad \frac{d T_{w}}{d t}=\frac{d T_{w}^{\prime}}{d t}\)

\(m C \frac{d T^{\prime}}{d t}=w C\left(T_{i}^{\prime}-T^{\prime}\right)+h_{p} A_{p}\left(T_{w}^{\prime}-T^{\prime}\right)\)

\(m_{w} C_{w} \frac{d T_{w}^{\prime}}{d t}=h_{s} A_{s}\left(0-T_{w}^{\prime}\right)-h_{p} A_{p}\left(T_{w}^{\prime}-T^{\prime}\right)\)

Taking Laplace transforms and rearranging,

\(\left(m C s+w C+h_{p} A_{p}\right) T^{\prime}(s)=w C T_{i}^{\prime}(s)+h_{p} A_{p} T_{w}^{\prime}(s)\) (1)

\(\left(m_{w} C_{w} s+h_{s} A_{s}+h_{p} A_{p}\right) T_{w}^{\prime}(s)=h_{p} A_{p} T^{\prime}(s)\) (2)

\(\text { Substituting in Eq. } 1 \text { for } T_{w}^{\prime}(s) \text { from Eq. } 2 \text {, }\)

\(\left(m C s+w C+h_{p} A_{p}\right) T^{\prime}(s)=w C T_{i}^{\prime}(s)+h_{p} A_{p} \frac{h_{p} A_{p}}{\left(m_{w} C_{w} s+h_{s} A_{s}+h_{p} A_{p}\right)} T^{\prime}(s)\)

Therefore,

\(\frac{T^{\prime}(s)}{T_{i}^{\prime}(s)}=\frac{w C\left(m_{w} C_{w} s+h_{s} A_{s}+h_{p} A_{p}\right)}{\left(m C s+w C+h_{p} A_{p}\right)\left(m_{w} C_{w} s+h_{s} A_{s}+h_{p} A_{p}\right)-\left(h_{p} A_{p}\right)^{2}}\)

b) The gain is \(\left[\frac{T^{\prime}(s)}{T_{i}^{\prime}(s)}\right]_{s=0}=\frac{w C\left(h_{s} A_{s}+h_{p} A_{p}\right)}{w C\left(h_{s} A_{s}+h_{p} A_{p}\right)+h_{s} A_{s} h_{p} A_{p}}\)

c) No, the gain would be expected to be one only if the tank were insulated so that \(h_{p}A_{p}= 0\). For the heated tank, the gain is not one because heat input changes as T changes.
