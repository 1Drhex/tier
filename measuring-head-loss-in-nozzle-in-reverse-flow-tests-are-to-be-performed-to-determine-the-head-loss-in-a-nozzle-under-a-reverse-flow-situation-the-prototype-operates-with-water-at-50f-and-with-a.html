MEASURING HEAD LOSS IN NOZZLE IN REVERSE FLOW

Tests are to be performed to determine the head loss in a nozzle under a reverse-flow situation. The prototype operates with water at 50°F and with a nominal reverse-flow velocity of 5 ft/s. The diameter of the prototype is 3 ft. The tests are done in a 1/12 scale model facility with water at 60°F. A head loss (pressure drop) of 1 psid is measured with a velocity of 20 ft/s. What will be the head loss in the actual nozzle?

Sketch:

Problem Definition

Situation: A 1/12 scale model tests for head loss in a reverse-flow nozzle. A pressure difference of 1 psid is measured with model at 20 ft/s.

Find: Pressure drop (psid) in actual nozzle.

Properties: Table F.5.: Water at \(50^{\circ} F , \rho=1.94 slugs / ft ^{3}\), \(v=1.41 \times 10^{-5} ft ^{2} / s ; \text { water at } 60^{\circ} F , \rho=1.94 \text { slugs } / ft ^{3}\) \(v=1.22 \times 10^{-5} ft ^{2} / s\).

Plan

The only significant \(\pi \text { -group }\) is the Reynolds number, so Eq. (8.4) reduces to \(C_{p}=f( Re )\). Dynamic similitude achieved if \(\operatorname{Re}_{m}=\operatorname{Re}_{p}\), then \(C_{p, m}=C_{p, p}\). From Fig. 8.6, if \(Re _{m}\), \(Re _{p}>10^{3}\), then \(C_{p, m}=C_{p, p}\).

\(C_{p}, c_{f}, C_{F}=f( Re , M , We , Fr )\) (8.4)

1. Calculate Reynolds number for model and prototype.

2. Check if both exceed \(10^{3}\). If not, model tests need to be reevaluated.

3. Calculate pressure coefficient.

4. Evaluate pressure drop in prototype.

Admin
Solution

1. Reynolds numbers

 

\(\operatorname{Re}_{m}=\frac{V D}{\nu}=\frac{20 ft / s \times(3 / 12 ft )}{1.22 \times 10^{-5} ft ^{2} / s }=4.10 \times 10^{5}\)

 

\(\operatorname{Re}_{p}=\frac{5 ft / s \times 3 ft }{1.41 \times 10^{-5} ft ^{2} / s }=1.06 \times 10^{6}\)

 

2. Both Reynolds numbers exceed \(10^{3}\). Therefore \(C_{p, m}=C_{p, p}\). The test is valid.

 

3. Pressure coefficient from model tests

 

\(C_{p, m}=\frac{\Delta p}{\frac{1}{2} \rho V^{2}}=\frac{1 lbf / in ^{2} \times 144 in ^{2} / ft ^{2}}{\frac{1}{2} \times 1.94 slug / ft ^{3} \times(20 ft / s )^{2}}=0.371\)

 

4. Pressure drop in prototype

 

\(\Delta p_{p}=0.371 \times \frac{1}{2} \rho V^{2}=0.371 \times 0.5 \times 1.94 \operatorname{slug} / ft ^{3} \times(5 ft / s )^{2}\)

 

\(=9.0 lbf / ft ^{2}=0.0625\) psid

 

Review

1. Because the Reynolds numbers are so much greater than \(10^{3}\), the equation for pressure drop is valid over a wide range of velocities.

2. This example justifies the independence of Reynolds
number referred to in Section 8.1.
