Guest Question.

The Euler-Bernoulli-von K´arm´an nonlinear beam theory [7]:

\(-\frac{\mathrm{d} }{\mathrm{d} x}\left \{ EA\left [ \frac{\mathrm{d} u}{\mathrm{d} x}+\frac{1}{2} \left ( \frac{\mathrm{d} w}{\mathrm{d} x} \right )^{2}\right ] \right \}=f\) for 0 <x<L

\(\frac{\mathrm{d^{2}} }{\mathrm{d} x^{2}}\left ( EI\frac{\mathrm{d^{2}}w }{\mathrm{d} x^{2}} \right )-\frac{\mathrm{d} }{\mathrm{d} x}\left \{ EA\frac{\mathrm{d} w}{\mathrm{d} x}\left [ \frac{\mathrm{d} u}{\mathrm{d} x}+\frac{1}{2} \left ( \frac{\mathrm{d} w}{\mathrm{d} x} \right )^{2}\right ] \right \}=q\)

u = w = 0 at x = 0, L; \(\left ( \frac{\mathrm{d} w}{\mathrm{d} x} \right )|_{x=0}=0\); \(\left ( EI\frac{\mathrm{d^{2}} w}{\mathrm{d} x^{2}} \right )|_{x=L}=M_{0}\)

where EA, EI, f, and q are functions of x, and \(M_{0}\) is a constant. Here u denotes the axial displacement and w the transverse deflection of the beam.

Admin
Solution

The first step of the formulation is to multiply each equation with a weight function, say \(v_{1}\) for the first equation and \(v_{2}\) for the second equation, and integrate over the interval (0, L). In the second step, carry out the integration-by-parts once in the first equation, twice in the first term of the second equation, and once in the second part of the second equation. Then use the fact that \(v_{1}(0)=v_{1}(L)\) = 0 (because u is specified there), \(v_{2}(0)=v_{2}(L)\) = 0 (because w is specified), and \((dv_{2}/dx)(0)\) = 0 (because dw/dx is specified at x = 0). In addition, we have \(EI(d^{2}w/dx^{2})=M_{0}\) at x = L. The final weak forms are given by

\(0=\int_{0}^{L}\left \{ EA\frac{\mathrm{d} v_{1}}{\mathrm{d} x}\left [ \frac{\mathrm{d} u}{\mathrm{d} x}+\frac{1}{2}\left ( \frac{\mathrm{d} w}{\mathrm{d} x}\right )^{2} \right ] -v_{1}f \right \}dx\) (1a)

\(0=\int_{0}^{L}\left \{ EI\frac{\mathrm{d^{2}} v_{2}}{\mathrm{d} x^{2}}\frac{\mathrm{d^{2}} w}{\mathrm{d} x^{2}}+EA\frac{\mathrm{d} v_{2}}{\mathrm{d} x}\frac{\mathrm{d} w}{\mathrm{d} x}\left [ \frac{\mathrm{d} u}{\mathrm{d} x}+\frac{1}{2}\left ( \frac{\mathrm{d} w}{\mathrm{d} x} \right )^{2} \right ]-v_{2}q \right \}dx\) \(-\left ( \frac{\mathrm{d} v_{2}}{\mathrm{d} x} \right )|_{L}M_{0}\)

Note that for this case the weak form is not linear in u or w. However, a functional can be constructed for this using the potential operator theory.The functional is given by

\(\Pi (u,w)=\int_{0}^{L}\left \{ \frac{EA}{2}\left [ \left ( \frac{\mathrm{d} u}{\mathrm{d} x}\right )^{2}+\frac{\mathrm{d} u}{\mathrm{d} x}\left ( \frac{\mathrm{d} w}{\mathrm{d} x} \right )^{2}+\frac{1}{2}\left ( \frac{\mathrm{d} w}{\mathrm{d} x} \right )^{4} \right ] +\frac{EI}{2}\left ( \frac{\mathrm{d^{2}} w}{\mathrm{d} x^{2}} \right )^{2}+uf + wq\right \}dx -\frac{\mathrm{d} w}{\mathrm{d} x}|_{L}M_{0}\)
