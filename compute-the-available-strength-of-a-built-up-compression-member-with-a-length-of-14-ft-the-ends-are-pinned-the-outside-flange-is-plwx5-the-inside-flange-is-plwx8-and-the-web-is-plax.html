Guest Question.

Compute the available strength of a built-up compression member with a length of 14 ft. The ends are pinned. The outside flange is \(PL 3 / 4 \times 5\), the inside flange is \(PL 3 / 4 \times 8\), and the web is\(PL 3 / 8 \times 10 1 / 2\). Material is ASTM A572 Grade 50.

Admin
Solution

Material Properties:

ASTM A572 \(F_{y}=50 ksi \quad F_{u}=65 ksi\)

Manual
Table 2-4

User note: There are no tables for special built-up shapes.

Determine if the shape has any slender elements

Check outside flange slenderness

\(\text { Calculate } k_{c}\)

\(k_{c}=\frac{4}{\sqrt{\frac{h}{t_{w}}}}=\frac{4}{\sqrt{\frac{10.5 in .}{0.375 in .}}}=0.756\) \(0.35 \leq k_{c} \leq 0.76 . \quad \text { o.k }\)

Table B4.1
Note [a]

\(\frac{b}{t}=\frac{2.50 \text { in. }}{0.75 \text { in. }}=3.33\)

 

\(\lambda_{r}=0.64 \sqrt{\frac{k_{c} E}{F_{y}}}=0.64 \sqrt{\frac{0.756(29,000 ksi )}{50 ksi }}=13.4\)

Table B4.1
Case 4

\(\frac{b}{t} \leq \lambda_{r}\) therefore, the outside flange is not slender.

Check inside flange slenderness

\(\frac{b}{t}=\frac{4.0 in .}{0.750 in .}=5.33\)

Table B4.1
Case 4

\(\frac{b}{t} \leq \lambda_{r}\) therefore, the inside flange is not slender.

Check web slenderness

\(\frac{h}{t}=\frac{10.5 in .}{0.375 in .}=28.0\)

Table B4.1
Case 14

\(\lambda_{r}=1.49 \sqrt{\frac{E}{F_{y}}}=1.49 \sqrt{\frac{29,000 ksi }{50 ksi }}=35.9\)

 

\(\frac{h}{t}<\lambda_{r}\) therefore, the web is not slender.

Calculate section properties (ignoring welds)

\(A_{g}=b_{f 1} t_{f 1}+h t_{w}+b_{f 2} t_{f 2}\) = \((8.00 \text { in. })(0.750 \text { in. })+(10.5 \text { in. })(0.375 \text { in. })+(5.00 \text { in. })(0.75 \text { in. })\)

 

\(=13.7 in .^{2}\)

 

\(\bar{y}=\frac{\Sigma A_{i} y_{i}}{\Sigma A_{i}}=\frac{\left(6.00 in ^{2}\right)(11.6 in .)+\left(3.94 in .^{2}\right)(6.00 in .)+\left(3.75 in .^{2}\right)(0.375 in .)}{\left(6.00 in ^{2}\right)+\left(3.94 in .^{2}\right)+\left(3.75 in .^{2}\right)}=6.92 in\)

 

Note that the neutral axis location is measured from the bottom of the outside flange.

\(I_{x}=\left[\frac{(8.0 \text { in. })(0.75 in .)^{3}}{12}+(8.0 \text { in. })(0.75 \text { in. })(4.71 in .)^{2}\right]+\)

 

\(\left[\frac{(0.375 in .)(10.5 in .)^{3}}{12}+(0.375 in. )(10.5 in .)(0.92 in .)^{2}\right]+\)

 

\(\left[\frac{(5.0 \text { in. })(0.750 in .)^{3}}{12}+(5.0 \text { in. })(0.750 in .)(6.55 in .)^{2}\right]=334 in. { }^{4}\)

 

\(r_{x}=\sqrt{\frac{I_{x}}{A}}=\sqrt{\frac{334 \text { in. }^{4}}{13.7 \text { in. }^{2}}}=4.94 \text { in. }\)

 

\(I_{y}=\left[\frac{(0.75 in .)(8.0 in .)^{3}}{12}\right]+\left[\frac{(10.5 in .)(0.375 in .)^{3}}{12}\right]+\left[\frac{(0.750 in. )(5.0 in .)^{3}}{12}\right]=39.9 in .^{4}\)

 

\(r_{y}=\sqrt{\frac{I_{y}}{A}}=\sqrt{\frac{39.9 in ^{4}}{13.7 in ^{2}}}=1.71 in\)

 

Calculate x-x axis flexural elastic critical buckling stress, \(F_{e}\)

 

\(\frac{K_{x} L}{r_{x}}=\frac{1.0(14.0 ft )(12 in . / ft )}{4.94 in .}=34.0\)

Section E3

\(F_{e}=\frac{\pi^{2} E}{\left(\frac{K L}{r}\right)^{2}}=\frac{\pi^{2}(29,000 ksi )}{(34.0)^{2}}=247 ksi\) does not control

Eqn. E3-4

Calculate the flexural- torsional critical elastic buckling stress

\(J=\Sigma\left(\frac{b t^{3}}{3}\right)=\frac{(8.00 \text { in. })(0.750 in. )^{3}}{3}+\frac{(10.5 in. )(0.375 in. )^{3}}{3}+\frac{(5.00 in. )(0.750 in. )^{3}}{3}=2.01 in. ^{4}\)

 

\(h_{o}=d-\frac{t_{f}}{2}-\frac{t_{f}}{2}=12.0 \text { in. }-\frac{0.750 \text { in. }}{2}-\frac{0.750 in .}{2}=11.3 \text { in. }\)

 

\(C_{w}=\frac{t_{f} h_{o}^{2}}{12}\left(\frac{b_{1}^{3} b_{2}^{3}}{b_{1}^{3}+b_{2}^{3}}\right)=\frac{(0.750 in .)(11.25 in .)^{2}}{12}\left(\frac{(8.00 in .)^{3}(5.00 in .)^{3}}{(8.00 in .)^{3}+(5.00 in .)^{3}}\right)=795 in .^{6}\)

 

Locate the shear center
Due to symmetry, both the centroid and the shear center lie on the y-axis. Therefore \(x_{o}=0\)

\(e=h_{o}\left(\frac{b_{1}^{3}}{b_{1}^{3}+b_{2}^{3}}\right)=11.3 in \cdot\left(\frac{(8.00 in .)^{3}}{(8.00 in .)^{3}+(5.00 in .)^{3}}\right)=9.04 in\)

Note that this shear center location is taken from the center of the outside flange. Therefore, add one-half the flange thickness to determine the shear center location measured from the bottom of the outside flange.

\(e+\frac{t_{f}}{2}=9.04 \text { in. }+\frac{0.75 \text { in. }}{2}=9.42 \text { in. }\)

 

Therefore \(y_{o}=\left(e+\frac{t_{f}}{2}\right)-\bar{y}=9.42 \text { in. }-6.92 \text { in. }=2.50 \text { in. }\)

 

\(\bar{r}_{0}^{2}=x_{0}^{2}+y_{0}^{2}+\frac{I_{x}+I_{y}}{A_{g}}=0+(2.50 in .)^{2}+\frac{334 in .^{4}+39.9 in \cdot^{4}}{13.7 in .^{2}}=33.5 in .^{2}\)

Eqn. E4-7

 

\(H=1-\frac{x_{0}^{2}+y_{0}^{2}}{\bar{r}_{0}^{2}}=1-\frac{0+(2.50 in .)^{2}}{33.5 in .^{2}}=0.813\)

Eqn. E4-8

Since the ends are pinned, K = 1.0
