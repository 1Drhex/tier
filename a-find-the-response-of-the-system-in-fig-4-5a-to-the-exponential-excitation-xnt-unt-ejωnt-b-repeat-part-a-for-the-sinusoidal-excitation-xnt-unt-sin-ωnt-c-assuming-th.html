Guest Question.

(a) Find the response of the system in Fig. 4.5a to the exponential excitation

\(x(n T)=u(n T) e^{j \omega n T}\)

(b) Repeat part (a) for the sinusoidal excitation

\(x(n T)=u(n T) \sin \omega n T\)

(c) Assuming that p < 1, find the response of the system to the sinusoidal excitation in part (b) as \(n \rightarrow \infty\).

Admin
Solution

(a) With the system initially relaxed, the use of Eq. (4.16) gives

 

\(y(n T)=x(n T)+p y(n T-T)\) (4.16)

 

\(\begin{aligned}y(0) &=e^{0}+p y(-T)=1 \\y(T) &=e^{j \omega T}+p y(0)=e^{j \omega T}+p \\y(2 T) &=e^{j 2 \omega T}+p y(T)=e^{j 2 \omega T}+p e^{j \omega T}+p^{2}\\ \cdots \cdots &\cdots \cdots \cdots \cdots \cdots \cdots \cdots \cdots \cdots \cdots \cdots \cdots\end{aligned}\)

 

\(\begin{aligned}y(n T) &=u(n T)\left(e^{j \omega n T}+p e^{j \omega(n-1) T}+\cdots+p^{(n-1)} e^{j \omega T}+p^{n}\right) \\&=u(n T) e^{j \omega n T}\left(1+p e^{-j \omega T}+\cdots+p^{n} e^{-j n \omega T)}\right) \\&=u(n T) e^{j \omega n T} \sum_{k=0}^{n} p^{k} e^{-j k \omega T}\end{aligned}\)

 

This is a geometric series with a common ratio \(p e^{-j \omega T}\) and, as in (b), the above sum can be obtained in closed form as

 

\(y(n T)=u(n T) \frac{e^{j \omega n T}-p^{(n+1)} e^{-j \omega T}}{1-p e^{-j \omega T}}\) (4.19)

 

Now consider the function

 

\(H\left(e^{j \omega T}\right)=\frac{1}{1-p e^{-j \omega T}}=\frac{e^{j \omega T}}{e^{j \omega T}-p}\) (4.20)

 

and let

 

\(H\left(e^{j \omega T}\right)=M(\omega) e^{j \theta(\omega)}\) (4.21)

 

where

 

\(M(\omega)=\left|H\left(e^{j \omega T}\right)\right|=\frac{1}{\sqrt{1+p^{2}-2 p \cos \omega T}}\) (4.22a)

 

and

 

\(\theta(\omega)=\arg H\left(e^{j \omega T}\right)=\omega T-\tan ^{-1} \frac{\sin \omega T}{\cos \omega T-p}\) (4.22b)

 

as can be easily shown. On using Eqs. (4.19)–(4.21), y(nT ) can be expressed as

 

\(y(n T)=u(n T) H\left(e^{j \omega T}\right)\left(e^{j \omega n T}-p^{(n+1)} e^{-j \omega T}\right)\)

 

\(=u(n T) M(\omega)\left(e^{j[\theta(\omega)+\omega n T]}-p^{(n+1)} e^{j[\theta(\omega)-\omega T]}\right)\) (4.23)

 

(b) The system is linear and so

 

\(y(n T)= R u(n T) \sin \omega n T= R u(n T) \frac{1}{2 j}\left(e^{j \omega n T}-e^{-j \omega n T}\right)\)

 

\(=\frac{1}{2 j}\left[ R u(n T) e^{j \omega n T}- R u(n T) e^{-j \omega n T}\right]\)

 

\(=\frac{1}{2 j}\left[y_{1}(n T)-y_{2}(n T)\right]\) (4.24)

 

where

 

\(y_{1}(n T)= R u(n T) e^{j \omega n T} \quad \text { and } \quad y_{2}(n T)= R u(n T) e^{-j \omega n T}\)

 

Partial response \(y_{1}(n T)\) can be immediately obtained from Eq. (4.23) in part (a) as

 

\(y_{1}(n T)=u(n T) M(\omega)\left(e^{j[\theta(\omega)+\omega n T]}-p^{(n+1)} e^{j[\theta(\omega)-\omega T]}\right)\) (4.25)

 

and since

 

\(y_{2}(n T)=\left. R u(n T) e^{j \omega n T}\right|_{\omega \rightarrow-\omega}\) (4.26)

 

partial response \(y_{2}(n T)\) can be obtained by replacing \(\omega \text { by }-\omega \text { in } y_{1}(n T)\), that is,

 

\(y_{2}(n T)=u(n T) M(-\omega)\left(e^{j[\theta(-\omega)-\omega n T]}-p^{(n+1)} e^{j[\theta(-\omega)+\omega T]}\right)\) (4.27)

 

From Eqs. (4.22a) and (4.22b), we note that M(ω) is an even function and θ(ω) is an odd function of ω, that is,

 

\(M(-\omega)=M(\omega) \quad \text { and } \quad \theta(-\omega)=-\theta(\omega)\)

 

Hence Eqs. (4.24), (4.25), and (4.27) yield

 

\(y(n T)=u(n T) \frac{M(\omega)}{2 j}\left(e^{j[\theta(\omega)+\omega n T]}-e^{-j[\theta(\omega)+\omega n T]}\right)\)

 

\(-u(n T) \frac{M(\omega)}{2 j} p^{(n+1)}\left(e^{j[\theta(\omega)-\omega T]}-e^{-j[\theta(\omega)-\omega T]}\right)\)

 

\(=u(n T) M(\omega) \sin [\omega n T+\theta(\omega)]\)

 

\(-u(n T) M(\omega) p^{(n+1)} \sin [\theta(\omega)-\omega T]\) (4.28)

 

We note that the response of the system consists of two components. For a given frequency ω, the first term is a steady sinusoid of fixed amplitude and the second term is a transient component whose amplitude is proportional to \(p^{(n+1)}\).

 

(c) If p < 1, then the transient term in Eq. (4.28) reduces to zero as \(n \rightarrow \infty\) since \(\lim _{n \rightarrow \infty} p^{(n+1)} \rightarrow 0\) and, therefore, we have

 

\(\tilde{y}(n T)=\lim _{n \rightarrow \infty} y(n T)=M(\omega) \sin [\omega n T+\theta(\omega)]\)

 

This is called the steady-state sinusoidal response and, as can be seen, it is a sinusoid of
amplitude M(ω) displaced by a phase angle θ(ω). Since the input is a sinusoid whose amplitude and phase angle are unity and zero, respectively, the system has introduced a gain M(ω) and a phase shift θ(ω), as illustrated in Fig. 4.12.
