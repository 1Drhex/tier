Guest Question.

The column footing shown in the scheme below measures 1.2 × 1.8 m and is subjected to a centric vertical load Q and two moments \( M_{B} \) and \( M_{L} \). If \( e_{B} = 0.12 m, e_{L} = 0.26 m \) and the depth of foundation \( D_{f} = 1 m \), determine the allowable load the foundation can carry using: (a) Terzaghi equations and effective area method, (b) the Meyerhof equations and effective area method, (c) thePrakash and Saran method. Use the safety factor against bearing capacity failure equals three.

Admin
Solution

(a) Terzaghi equations and effective area method

Table 4.2:

\( gross q_{ult} = c N_{c} s_{c} + {y}'D_{f} N_{q} + \frac{1}{2} {y}' B N_{y} s_{y} \)

 

The \(N_{c} \) term is 0, since the cohesion \(c = 0. B = {B}' \)

Table 4.3:

\( \varnothing = 35^{\circ}; N_{q} = 41.4; N_{y} = 42.4 \)

For rectangular footings, the Terzaghi \( s_{y} = 1 – 0.2 \frac{B}{L} \) ( see Section 4.6 )

\( {B}' = B – 2 e_{B} = 1.2 – 2 \times 0.12 = 0.96 m \)

 

\( {L}' = L – 2 e_{L} = 1.8 – 2 \times 0.26 = 1.28 m \)

 

\( gross q_{ult} = D_{f} {y}' N_{q} + \frac{1}{2} {y}' {B}' N_{y} s_{y} \)

 

\( = \left ( 1 \times 17 \right )\left ( 41.4 \right ) + \left ( 0.5 \times 17 \right )\left ( 0.96 \right )\left ( 42.4 \right )\left ( 1 – 0.2 \times \frac{0.96}{1.08} \right ) \)

 

\( = 703.8 + 284.4 = 988.2 kPa \)

Gross ultimate load \( = gross q_{ult} \times {B}'{L}' = 988.2 \times 0.96 \times 1.28 = 1214.3 kN \)

The allowable load (safe load) the footing can carry \( = \frac{1214.3}{SF = 3} = 405 kN \)

 

(b) Meyerhof equations and effective area method

Table 4.2:

\( gross q_{ult} = c N_{c} s_{c} d_{c} + {y}' D_{f} N_{q} s_{q} d_{q} + \frac{1}{2} {y}'{B}' N_{y} s_{y} d_{y} \)

 

The \(N_{c}\) term is 0, since the cohesion \(c = 0. {B}' = B\)

Table 4.5:

\( K_{p} = \tan^{2}\left ( 45 + \frac{\varnothing }{2} \right ) = 3.69\)

 

\( d_{q} = d_{y} = 1 + 0.1 \sqrt{K_{p}} \frac{D}{B} = 1 + 0.1 \times 1.92 \times \frac{1}{1.2} = 1.16 \)

 

\( s_{q} = s_{y} = 1 + 0.1 K_{p} \frac{D}{{B}'} = 1 + 0.1 \times 3.69 \times \frac{1}{0.96} = 1.38 \)

 

Table 4.4:

\( \varnothing = 35^{\circ}; N_{q} = 33.6; N_{y} = 37.8 \) ( by interpolation )

 

\( gross q_{ult} = \left ( 1 \times 17 \right) \left ( 33.6 \right ) \left ( 1.38 \right ) \left ( 1.16 \right ) \)

 

\( + \left ( 0.5 \times 17 \right) \left ( 0.96 \right ) \left ( 37.8 \right ) \left ( 1.38 \right ) \left ( 1.16 \right ) \)

 

\( = 914.4 + 493.8 = 1408.2 kPa \)

Gross ultimate load \( = gross q_{ult} \times {B}' {L}' = 1408.2 \times 0.96 \times 1.28 = 1730.4 kN \)

 

The allowable load (safe load) the footing can carry \( = \frac{1730.4}{SF = 3} = 577 kN \)

 

(c) Prakash and Saran method.

Equation :

\( Q_{ult} = BL \left [ {c}' N_{c\left ( e \right )} F_{cs\left ( e \right )} + q N_{q\left ( e \right )} F_{qs\left ( e \right )} + \frac{1}{2} y B N_{y\left ( e \right )} F_{ys\left ( e \right )} \right ] \)

 

The \( N_{c} \) term is 0, since the cohesion c=0

For more safety, use \( \left ( \frac{e_{B}}{B} = \frac{0.12}{1.2} = 0.1 \right ) \) or \( \left ( \frac{e_{L}}{L} = \frac{0.26}{1.8} = 0.14 \right ) \) whichever is greater; hence, use 0.14.

Figure : for \( \varnothing = 35^{\circ}\) and \( \frac{e}{B} = 0.14 \) estimate \( N_{y\left ( e \right )} = 20 \)

Figure : for \( \varnothing = 35^{\circ}\) and \( \frac{e}{B} = 0.14 \) estimate \( N_{q\left ( e \right )} = 29 \)

\( F_{qs\left ( e \right )} = 1\)

For the purpose of computing \( F_{ys\left ( e \right )}\) assume the soil is dense sand; hence,

\( F_{ys\left ( e \right )} = 1.0 + \left ( \frac{2 e}{B} – 0.68 \right ) \frac{B}{L} + \left [ 0.43 – \left ( \frac{3}{2} \right ) \left ( \frac{e}{B} \right ) \right ] \left ( \frac{B}{L} \right )^{2} \)

 

\( = 1.0 + \left ( 2 \times 0.2 – 0.68 \right ) \frac{1.2}{1.8} + \left [ 0.43 – \left ( \frac{3}{2} \right ) \left ( 0.2 \right ) \right ] \left ( \frac{1.2}{1.8} \right )^{2} = 0.873 \)

 

\( Q_{ult} = 1.2 \times 1.8 \left [ 1 \times 17 \times 29 \times 1 + \frac{1}{2} \times 17 \times 1.2 \times 20 \times 0.873 \right ] = 1450 kN \)

 

The allowable load (safe load) the footing can carry \( = \frac{1450}{SF = 3} = 483 kN \)
