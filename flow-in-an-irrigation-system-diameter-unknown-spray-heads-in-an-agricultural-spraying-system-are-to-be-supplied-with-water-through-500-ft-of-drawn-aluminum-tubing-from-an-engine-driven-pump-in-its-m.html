FLOW IN AN IRRIGATION SYSTEM: DIAMETER UNKNOWN

Spray heads in an agricultural spraying system are to be supplied with water through 500 ft of drawn aluminum tubing from an engine-driven pump. In its most efficient operating range, the pump output is 1500 gpm at a discharge pressure not exceeding 65 psig. For satisfactory operation, the sprinklers must operate at 30 psig or higher pressure. Minor losses and elevation changes may be neglected. Determine the smallest standard pipe size that can be used.

Admin
Solution

Given: Water supply system, as shown.

Find: Smallest standard \(D\).
Solution:

\(\Delta p, L\), and \(Q\) are known. \(D\) is unknown, so iteration is needed to determine the minimum standard diameter that satisfies the pressure drop constraint at the given flow rate. The maximum allowable pressure drop over the length, \(L\), is

\(\Delta p_{\max }=p_{1_{\max }}-p_{2_{\min }}=(65-30) \mathrm{psi}=35 \mathrm{psi}\)

Governing equations:

\(\begin{aligned}\left(\frac{p_{1}}{\rho}+\alpha_{1} \cancel{\frac{\bar{V}^2_1}{2}}+\cancel{gz_{1}}\right) &-\left(\frac{p_{2}}{\rho}+\alpha_{2} \cancel{\frac{\bar{V}^2_2}{2}}+\cancel{g z_{2}}\right)=h_{l_{T}} \\h_{l_{T}} &=h_{l}+\cancel{h_{l_m}}^{=0(3)}=f \frac{L}{D} \frac{\bar{V}_{2}^{2}}{2}\end{aligned}\)

Assumptions:

(1) Steady flow.
(2) Incompressible flow.
(3) \(h_{l_{T}}=h_{l}\), i.e., \(h_{l_{m}}=0\).
(4) \(z_{1}=z_{2}\).
(5) \(\bar{V}_{1}=\bar{V}_{2}=\bar{V} ; \alpha_{1} \simeq \alpha_{2}\).

Then

\(\Delta p=p_{1}-p_{2}=f \frac{L}{D} \frac{\rho \bar{V}^{2}}{2}\) (1)

 

Equation 1 is difficult to solve for \(D\) because both \(\bar{V}\) and \(f\) depend on \(D !\) The best approach is to use a computer application such as Excel to automatically solve for \(D\). For completeness here we show the manual iteration procedure. The first step is to express Eq. 1 and the Reynolds number in terms of \(Q\) instead of \(\bar{V}\) ( \(Q\) is constant but \(\bar{V}\) varies with \(D)\). We have \(\bar{V}=Q / A=4 Q / \pi D^{2}\) so that

\(\Delta p=f \frac{L}{D} \frac{\rho}{2}\left(\frac{4 Q}{\pi D^{2}}\right)^{2}=\frac{8 f L \rho Q^{2}}{\pi^{2} D^{5}}\) (2)

The Reynolds number in terms of \(Q\) is

\(\operatorname{Re}=\frac{\rho \bar{V} D}{\mu}=\frac{\bar{V} D}{\nu}=\frac{4 Q}{\pi D^{2}} \frac{D}{\nu}=\frac{4 Q}{\pi \nu D}\)

Finally, \(Q\) must be converted to cubic feet per second.

\(Q=1500 \frac{\mathrm{gal}}{\mathrm{min}} \times \frac{\mathrm{min}}{60 \mathrm{~s}} \times \frac{\mathrm{ft}^{3}}{7.48 \mathrm{gal}}=3.34 \mathrm{ft}^{3} / \mathrm{s}\)

For an initial guess, take nominal 4 in. (4.026 in. i.d.) pipe:

 

\(R e=\frac{4 Q}{\pi \nu D}=\frac{4}{\pi} \times 3.34 \frac{\mathrm{ft}^{3}}{\mathrm{~s}} \times \frac{\mathrm{s}}{1.21 \times 10^{-5} \mathrm{ft}^{2}} \times \frac{1}{4.026 \mathrm{in} .} \times 12 \frac{\mathrm{in} .}{\mathrm{ft}}=1.06 \times 10^{6}\)

For drawn tubing, \(e=5 \times 10^{-6} \mathrm{ft}\) (Table 8.1) and hence \(e / D=1.5 \times 10^{-5}\), so \(f \simeq 0.012\) (Eq. 8.37),

\(\frac{1}{\sqrt{f}}=-2.0 \log \left(\frac{e / D}{3.7}+\frac{2.51}{Re \sqrt{f}}\right) \quad \text { for turbulent flow }(Re \geq 2300)\) (8.37)

and

\(\begin{gathered}\Delta p=\frac{8 f L \rho Q^{2}}{\pi^{2} D^{5}}=\frac{8}{\pi^{2}} \times 0.012 \times 500 \mathrm{ft} \times 1.94 \frac{\text { slug }}{\mathrm{ft}^{3}} \times(3.34)^{2} \frac{\mathrm{ft}^{6}}{\mathrm{~s}^{2}} \\\times \frac{1}{(4.026)^{5} \text { in. }^{5}} \times 1728 \frac{\text { in. }^{3}}{\mathrm{ft}^{3}} \times \frac{\mathrm{lbf} \cdot \mathrm{s}^{2}}{\text { slug } \cdot \mathrm{ft}} \\\Delta p=172 \mathrm{lbf} / \mathrm{in} .^{2}>\Delta p_{\max }\end{gathered}\)

Since this pressure drop is too large, try \(D=6\) in. (actually 6.065 in. i.d.):

\(R e=\frac{4}{\pi} \times 3.34 \frac{\mathrm{ft}^{3}}{\mathrm{~s}} \times \frac{\mathrm{s}}{1.21 \times 10^{-5} \mathrm{ft}^{2}} \times \frac{1}{6.065 \mathrm{in} .} \times 12 \frac{\mathrm{in} .}{\mathrm{ft}}=6.95 \times 10^{5}\)

For drawn tubing with \(D=6\) in., \(e / D=1.0 \times 10^{-5}\), so \(f \simeq 0.013\) (Eq. 8.37), and

 

\(\begin{gathered}\Delta p=\frac{8}{\pi^{2}} \times 0.013 \times 500 \mathrm{ft} \times 1.94 \frac{\text { slug }}{\mathrm{ft}^{3}} \times(3.34)^{2} \frac{\mathrm{ft}^{6}}{\mathrm{~s}^{2}} \\\times \frac{1}{(6.065)^{5} \text { in. }^{5}} \times(12)^{3} \frac{\mathrm{in} \cdot{ }^{3}}{\mathrm{ft}^{3}} \times \frac{\mathrm{lbf} \cdot \mathrm{s}^{2}}{\text { slug } \cdot \mathrm{ft}} \\\Delta p=24.0 \mathrm{lbf} / \mathrm{in} .^{2}<\Delta p_{\max }\end{gathered}\)

Since this is less than the allowable pressure drop, we should check a 5 in (nominal) pipe. With an actual i.d. of 5.047 in.,

\(R e=\frac{4}{\pi} \times 3.34 \frac{\mathrm{ft}^{3}}{\mathrm{~s}} \times \frac{\mathrm{s}}{1.21 \times 10^{-5} \mathrm{ft}^{2}} \times \frac{1}{5.047 \mathrm{in} .} \times 12 \frac{\mathrm{in} .}{\mathrm{ft}}=8.36 \times 10^{5}\)

For drawn tubing with \(D=5\) in., \(e / D=1.2 \times 10^{-5}\), so \(f \simeq 0.0122\) (Eq. 8.37), and

\(\begin{gathered}\Delta p=\frac{8}{\pi^{2}} \times 0.0122 \times 500 \mathrm{ft} \times 1.94 \frac{\text { slug }}{\mathrm{ft}^{3}} \times(3.34)^{2} \frac{\mathrm{ft}^{6}}{\mathrm{~s}^{2}} \\\times \frac{1}{(5.047)^{5} \text { in. }^{5}} \times(12)^{3} \frac{\text { in. }^{3}}{\mathrm{ft}^{3}} \times \frac{\mathrm{lbf} \cdot \mathrm{s}^{2}}{\text { slug } \cdot \mathrm{ft}} \\\Delta p=56.4 \mathrm{lbf} / \mathrm{in} .^{2}>\Delta p_{\max }\end{gathered}\)

Thus the criterion for pressure drop is satisfied for a minimum nominal diameter of 6 in. pipe. \(\xleftarrow[]{\quad\quad\quad D}\)

 

This problem illustrates the method for manually iterating to calculate pipe diameter.

The Excel workbook for this problem automatically iterates to solve for the exact pipe diameter D that satisfies Eq. 1, without having to obtain the explicit equation (Eq. 2) for D first. Then all that needs to be done is to select the smallest standard pipe size that is equal to or greater than this value. For the given data, D = 5.58 in., so the appropriate pipe size is 6 in. The workbook can be used to perform numerous “what-ifs” that would be extremely time-consuming to do manually, e.g., to see how the required D is affected by changing the pipe length L. For example, it shows that reducing L to 250 ft would allow 5 in. (nominal) pipe to be used. The workbook can be modified for solving other case (d) type problems.
