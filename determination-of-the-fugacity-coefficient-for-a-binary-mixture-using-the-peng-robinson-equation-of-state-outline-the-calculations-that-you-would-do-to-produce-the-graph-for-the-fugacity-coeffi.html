Determination of the Fugacity Coefficient for a Binary Mixture Using the Peng–Robinson Equation of State

Outline the calculations that you would do to produce the graph for the fugacity coeffi cient of methane in a binary mixture of methane and ethane, as shown on the left side of Figure 7.3. Show the representative part of a MATLAB code that could be used to perform this calculation. Use mixing rules given by Equations (7.15), (7.17), and (7.18).

\(a_{\text {mix }}=\sum \sum y_{i} y_{j} a_{i j}\) (7.15)

\(a_{i j}=\sqrt{a_{i i} a_{j j}}\left(1-k_{i j}\right)\) (7.17)

\(b_{\text {mix }}=\sum y_{i} b_{i}\) (7.18)

Admin
Solution

We begin with the Peng–Robinson equation of state, Equation (4.25):

 

\(P=\frac{R T}{v-b}-\frac{a \alpha(T)}{v(v+b)+b(v-b)}\) (4.25)

 

\(P=\frac{R T}{v-b}-\frac{a \alpha}{v^{2}+2 b v-b^{2}}\) (E7.5A)

 

where the parameters of the pure species can be found from the critical temperature, the critical pressure, and Pitzer’s acentric factor, as follows:

 

\(a_{i}=\frac{0.45724 R^{2} T_{c, i}^{2}}{P_{c, i}}, \alpha_{i}=\left[1+\left(0.37464+1.54226 \omega_{i}-0.26992 \omega_{i}^{2}\right)\left(1-T_{r, i}^{0.5}\right)\right]^{2}\),

 

and,

 

\(b_{i}=\frac{0.07780 R T_{c, i}}{P_{c, i}}\) (E7.5B)

 

The van der Waals mixing rules are as follows:

 

\(a \alpha=y_{1}^{2}(a \alpha)_{1}+2 y_{1} y_{2}(a \alpha)_{12}+y_{2}^{2}(a \alpha)_{2} \text { and } b=y_{1} b_{1}+y_{2} b_{2}\) (E7.5C)

 

Applying the binary interaction parameter gives:

 

\(a \alpha=y_{1}^{2}(a \alpha)_{1}+2 y_{1} y_{2} \sqrt{(a \alpha)_{1}(a \alpha)_{2}}\left(1-k_{12}\right)+y_{2}^{2}(a \alpha)_{2}\) (E7.5D)

 

Finally, in a way similar to , the Peng–Robinson equation of state can be used to develop an expression for the fugacity coeffi cient of species 1. Such an expression is given in Table 7.1:

 

ln \(\hat{\varphi}_{1}=\frac{b_{1}}{b}(z-1)-\ln \left(\frac{(v-b) P}{R T}\right)\)

 

\(+\frac{a \alpha}{2 \sqrt{2} b R T}\left[\frac{b_{1}}{b}-\frac{2}{a \alpha}\left(y_{1}(a \alpha)_{1}+y_{2}(a \alpha)_{12}\right)\right] \ln \left[\frac{v+(1+\sqrt{2}) b}{v+(1-\sqrt{2}) b}\right]\)

 

where Equation (7.5A) can be used to find v and \(z(=P v / R T)\), given values of P and T. Using Equations (7.5B), (7.5C), and (7.5D), we can calculate numerical values for the fugacity coefficient with the parameter values from Table A.1 (Appendix A).

Parts of a MATLAB code that can be used in this way are shown here.
