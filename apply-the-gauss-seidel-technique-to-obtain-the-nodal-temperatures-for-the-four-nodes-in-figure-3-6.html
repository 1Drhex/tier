Guest Question.

Apply the Gauss-Seidel technique to obtain the nodal temperatures for the four nodes in Figure 3-6.

Admin
Solution

It is useful to think in terms of a resistance formulation for this problem because all the connecting resistances between the nodes in Figure 3-6 are equal; that is:

\(R=\frac{\Delta y}{k\Delta y} = \frac{\Delta x}{k\Delta y}=\frac{1}{k} \hspace{30 pt} \enspace [a]\)

Therefore, when we apply Equation (3-32) to each node, we obtain (qi =0):

\(T_{i} =\frac{\sum\limits_{j}{k_{j} T_{j} } }{\sum\limits_{j}{k_{j} } } \hspace{30 pt} \hspace{30 pt} [b]\)

Because each node has four resistances connected to it and k is assumed constant,

\(\sum\limits_{j}{k_{j} } =4k\)

and

\(T_{i} =\frac{1}{4} \sum\limits_{j}{T_{j} }\hspace{30 pt} \hspace{30 pt} [c]\)

We now set up an iteration table as shown and use initial temperature assumptions of 300 and \(200 ^{â—¦} C\). Equation (c) is then applied repeatedly until satisfactory convergence is achieved. In thetable, five iterations produce convergence with 0.13 degree. To illustrate the calculation, we can note the two specific cases below:

\(T(2)_{n=1}=\frac{1}{4}(500+100+T_{4}+T_{1} ) =\frac{1}{4} (500+100+200+275)=268.75\)

 

\(T(3)_{n=4}=\frac{1}{4}(100+T_{1}+T_{4}+100 ) =\frac{1}{4} (100+250.52+150.52+100)=150.26\)
