Guest Question.

A company that makes clutch disks for race cars has the cash flows shown for one department.

\((a)\) Determine the number of positive roots to the rate of return relation.

\((b)\) Calculate the internal rate of return.

\((c)\) Calculate the external rate of return using the return on invested capital (ROIC) approach with an investment rate of \(15\)% per year. (As assigned by your instructor, solve by hand and/or spreadsheet.)

Admin
Solution

\((a)\) Descartes’ rule of signs: \(2\) sign changes

Norstrom’s criterion; series starts negative; \(1\) sign change, therefore, one positive root

\((b) 0 = -65 + 30(P/F,i,1) + 84(P/F,i,2) – 10(P/F,i,3) – 12(P/F,i,4)\)

 

Solve for \(i\) by trial and error or spreadsheet.

\(i = 28.6\)% per year (spreadsheet)

A negative root of \(-56.0\)% is discarded.

 

\((c)\) Apply net-investment procedure steps because the investment rate \({ i }_{ i } = 15\)% is not equal to \(i*\) rate of \(28.6\)% per year.

Hand solution:

 

Step \(1: { F }_{ 0 } = -65 \quad \quad \quad\quad\quad { F }_{ 0 } < 0\) ; use \({ i }^{ \backprime \backprime }\)

 

\({ F }_{ 1 } = -65(1 + { i }_{ i }) + 30\quad \quad \quad\quad { F }_{ 1 } < 0;\) use \({ i }^{ \backprime \backprime }\)

 

\({ F }_{ 2 } = { F }_{ 1 }(1 + { i }_{ i }) + 84 \quad \quad\quad\quad { F }_{ 2 } > 0;\) use \({ i }_{ i } ({ F }_{ 2 }\) must be \(> 0\), because last two terms are negative)

 

\({ F }_{ 3 } = { F }_{ 2 }(1 + 0.15) -10 \quad \quad \quad { F }_{ 3 } > 0;\) use \({ i }_{ i } ({ F }_{ 3 }\) must be \(> 0,\) because last term is negative)

 

\({ F }_{ 4 } = { F }_{ 3 } (1 + 0.15) –12\)

 

Step \(2\): Set \({ F }_{ 4 } = 0\) and solve for \({ i }^{ \backprime \backprime }\) by trial and error.

\({ F }_{ 1 } = -65 – 65{ i }^{ \backprime \backprime } + 30\)

 

\({ F }_{ 2 } = (-65 – 65{ i }^{ \backprime \backprime } + 30)(1 + { i }_{ i }) + 84\)

 

\(\quad = -65 – 65{ i }_{ i } + 30 – 65{ i }^{ \backprime \backprime } – 65{ i }^{ \backprime \backprime }2 + 30{ i }^{ \backprime \backprime } +84\)

 

\(\quad = -65{ i }^{ \backprime \backprime }2 –100{ i }^{ \backprime \backprime } + 49\)

 

\({ F }_{ 3 } = (-65{ i }^{ \backprime \backprime }2 –100{ i }^{ \backprime \backprime }+ 49)(1.15) – 10\)

 

\( \quad = -74.8 { i }^{ \backprime \backprime }2 –115{ i }^{ \backprime \backprime } + 56.4 – 10\)

 

\(\quad = -74.8 { i }^{ \backprime \backprime }2 –115{ i }^{ \backprime \backprime } + 46.4\)

 

\({ F }_{ 4 } = (-74.8{ i }^{ \backprime \backprime }2 –115{ i }^{ \backprime \backprime } + 46.4)(1.15) –12\)

 

\(\quad = -86 { i }^{ \backprime \backprime }2 –132.3{ i }^{ \backprime \backprime } + 53.3 – 12\)

 

\(\quad = -86{ i }^{ \backprime \backprime }2 –132.3{ i }^{ \backprime \backprime } + 41.3\)

 

Solve by quadratic equation, trial and error, or spreadsheet.

\({ i }^{ \backprime \backprime } = 26.6\)% per year (spreadsheet)

 

Spreadsheet solution: Using the format and functions of Figure 7-12, \({ i }^{ \backprime \backprime } = 26.62\)%.
