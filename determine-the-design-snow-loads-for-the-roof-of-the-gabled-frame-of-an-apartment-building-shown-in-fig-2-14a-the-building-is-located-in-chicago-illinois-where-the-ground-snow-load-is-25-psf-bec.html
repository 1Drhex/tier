Guest Question.

Determine the design snow loads for the roof of the gabled frame of an apartment building shown in Fig. \(2.14(\mathrm{a})\). The building is located in Chicago, Illinois, where the ground snow load is 25 psf. Because of several trees near the structure, assume the exposure factor is \(C_{e}=1\)

Admin
Solution

Flat-Roof Snow Load.

\( \begin{aligned} p_{g} &=25 \mathrm{psf} \\ C_{e} &=1 \\ C_{t} &=1 \quad \text { (heated structure) } \\ I_{s} &=1 \quad \text { (from Table } 2.3 \text { for nonessential building, risk category II) } \end{aligned} \)

From Eq. (2.7), the flat-roof snow load is obtained as

\( \begin{aligned} p_{f} &=0.7 C_{e} C_{t} I_{s} p_{g}=0.7(1)(1)(1)(25) \\ &=17.5 \mathrm{psf} \end{aligned} \)

The slope is \(\theta=35^{\circ}\), which is greater than \(15^{\circ}\), so the minimum values of \(p_{f}\) need not be considered.

Sloped-Roof Snow Load. By applying Eq. (2.9), we compute the slope factor as

\( \begin{array}{c} \text { For warm roofs } \\ \left(C_{t} \leq 1.0\right) \end{array}\left\{\begin{array}{ll} C_{s}=1 & \text { for } 0 \leq \theta<30^{\circ} \\ C_{s}=1-\frac{\theta-30^{\circ}}{40^{\circ}} & \text { for } 30^{\circ} \leq \theta \leq 70^{\circ} \\ C_{s}=0 & \text { for } \theta>70^{\circ} \end{array}\right. \) (2.9)

\( C_{s}=1-\frac{\theta-30^{\circ}}{40^{\circ}}=1-\frac{35^{\circ}-30^{\circ}}{40^{\circ}}=0.88 \)

From Eq. (2.8), we determine the design sloped-roof snow load:

\( p_{s}=C_{s} p_{f}=0.88(17.5)=15.4 \mathrm{psf} \)

This load is called the balanced design snow load and is applied to the entire roof of the structure, as shown in Fig. \(2.14(\mathrm{~b})\).

As the slope is \(\theta=35^{\circ}\), which is greater than \(30.2^{\circ}\), the unbalanced snow load does not need to be considered.
