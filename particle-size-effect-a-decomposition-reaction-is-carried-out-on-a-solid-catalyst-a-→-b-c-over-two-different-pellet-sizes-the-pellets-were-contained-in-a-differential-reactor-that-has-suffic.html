Guest Question.

Particle size effect
A decomposition reaction is carried out on a solid catalyst

\(A \rightarrow B + C\)

over two different pellet sizes. The pellets were contained in a differential reactor that has sufficient turbulence such that the external mass transfer effects are negligible. We know that the adsorption coverage of A on the catalyst site is minimal and thus \(K_{A} \rightarrow \infty\). The reaction on the catalyst surface is thus virtually first order. The results of two experimental runs made under identical conditions are as shown in Table E17.3.1. Estimate the Thiele modulus and effectiveness factor for each pellet. How small should the pellets be made to virtually eliminate all internal diffusion effects on the overall reaction kinetics?

Admin
Solution

When \(K_{A} \rightarrow \infty\), the Thiele modulus is defined by Eq. (17.68) \(\phi=\lim _{K_{\beta} \rightarrow \infty} \frac{1}{a_{p}} \sqrt{\frac{r_{\max }^{\prime}}{2 D_{e A} C_{A S}}} \frac{\left[1-K_{\beta} \ln \left(1+K_{\beta}^{-1}\right]^{-\frac{1}{2}}\right.}{1+K_{\beta}}=\frac{1}{a_{p}} \sqrt{\frac{r_{\max }^{\prime}}{D_{e A} K_{A}}}\). , or

\(\phi=\frac{d_{p}}{6} \sqrt{\frac{r_{\max }^{\prime}}{D_{e A} K_{A}}}\) (E17.3.1)

and the effectiveness factor is given by Eq. (17.72),

\(\eta=\frac{\phi \operatorname{coth}(3 \phi)-1 / 3}{\phi^{2}}\). (17.72)

The effectiveness factor is defined by Eq. (17.47) \(\eta=\frac{r_{A, \text { obs }}}{r_{A S}}=\frac{\left.D_{e A} S \frac{d C_{A}}{d x}\right|_{x=\delta_{p}}}{-r_{A S}^{\prime} V_{p}}=\frac{a_{c}}{-r_{A S}^{\prime}}\left[2 \int_{C_{A e, 0}}^{C_{A S}}-r_{A}^{\prime} D_{e A} d C_{A}\right]^{\frac{1}{2}}\), or

\(\eta=\frac{r_{A, a b s}}{r_{A S}}\) (E17.3.2)

Assuming that the Thiele modulus for Run 1 is \(\phi_{1}\), from Eq. (E17.3.1) the Thiele modulus for Run 2 is given by

\(\phi_{2}=\frac{d_{p 2}}{d_{p 1}} \phi_{1}\), (E17.3.3)

 

TABLE E17.3.1
Pellet size, mm Measured rate, mol-A/(g-cat·s)
Run 1 2.54 2.00
Run 2 0.254 10.5

since only particle size has changed. From Eqs. (E17.3.2) and (17.72), we have

\(\frac{r_{A, \text { obs } 1}}{r_{A, \text { obs } 2}}=\frac{\eta_{1}}{\eta_{2}}=\frac{\phi_{1} \operatorname{coth}\left(3 \phi_{1}\right)-1 / 3}{\phi_{2} \operatorname{coth}\left(3 \phi_{2}\right)-1 / 3} \times \frac{\phi_{2}^{2}}{\phi_{1}^{2}}\), (E17.3.4)

which leads to

\(\frac{2.00}{10.5}=\frac{\phi_{1} \operatorname{coth}\left(3 \phi_{1}\right)-1 / 3}{0.1 \phi_{1} \operatorname{coth}\left(0.3 \phi_{1}\right)-1 / 3} \times 0.01\)

or

\(0.0525=\frac{0.1 \phi_{1} \operatorname{coth}\left(0.3 \phi_{1}\right)-1 / 3}{\phi_{1} \operatorname{coth}\left(3 \phi_{1}\right)-1 / 3}\). (E17.3.5)

This equation can be solved to give \(\phi_{1}\) = 5.9103. Therefore we can compute

\(\eta_{1}=\) 0.15965 by using Eq.(17.72);

\(\phi_{2}\) = 0.59103 by using Eq.(E17.3.3);

\(\eta_{2}\)= 0.83188 by using Eq.(17.72).

For negligible intraparticle diffusion effects, we set \(\eta_{3}\) > 0.95. From Eq. (17.72), we obtain \(\phi_{3}\) < 0.299529.

From Eq. (E17.3.1),

\(\frac{\phi_{3}}{\phi_{1}}=\frac{d_{p 3}}{d_{p 1}}\)

Thus

\(d_{p 3}=\frac{\phi_{3}}{\phi_{1}} d_{p 1}<\frac{0.299529}{5.9103} \times 2.54 mm =0.1287 mm\).

Therefore when the pellet size is smaller than 0.1287 mm, the internal diffusion effects are virtually eliminated.
