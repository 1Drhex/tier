Guest Question.

A pipeline in a plane splits into two line sections which then again merge. In the line ①, a globe valve \(G\) (pressure-loss coefficient \(\zeta_{1} \)) is built in, while in line ②, a clack valve \(C\) (pressureloss coefficient \(\zeta_{2}\) ) is present. The total volume flow throug the pipe system is given by \(Q\).

a) Calculate the volume flows in the pipes ① and ②.

b) Determine the pressure loss (related to \(\rho g\)) between inflow and offflow.

Given: \(D_{1}=1.4 m \), \(D_{2}=0.8 m\) , \(Q=5.0 m^{3}/ s \) , \(\zeta_{1}=1.3\) , \(\zeta_{2}=0.3\) .

Admin
Solution

a) The generalized BERNOULLI’s theorem between points ① and ③ (via pipe ① and via pipe ②) yields

\(\left. \begin{matrix}\frac{1}{2}\varrho v^{2}_{0}+p_{0}=\frac{1}{2}\varrho v^{2}_{1}+p_{1}=\frac{1}{2}\varrho v^{2}_{3}+p_{3} +\Delta p_{v_{1}} \\ \frac{1}{2}\varrho v^{2}_{0}+p_{0}=\frac{1}{2}\varrho v^{2}_{2}+p_{2}=\frac{1}{2}\varrho v^{2}_{3}+p_{3} +\Delta p_{v_{2}} \end{matrix} \right\} \rightsquigarrow \begin{matrix} \Delta p_{v_{1}}=\Delta p_{v_{2}} , \\ \zeta _{1}\frac{v^{2}_{1}\varrho }{2}=\zeta _{2}\frac{v^{2}_{2}\varrho }{2} \end{matrix} \).

With the volume flow \(Q=vA=vD^{2}\pi / 4 \) follows

\(\left(\frac{\zeta _{1} }{\zeta _{2} } \right)=\left(\frac{v_{2} }{v_{1} } \right)^{2}=\left(\frac{Q_{2} }{Q_{1} } \right)^{2} \left(\frac{D_{1} }{D_{2} } \right)^{4} \).

Using the continuity equation \(Q=Q_{1}+Q_{2} \) leads to the volume flow in pipe ①:

\(\underline{\underline{Q_{1} } } =\frac{Q}{1+\sqrt{\frac{\zeta _{1} }{\zeta _{2} } }\left(\frac{D_{2} }{D_{1} } \right)^{2} }=\frac{5}{1+\sqrt{\frac{1.3}{0.3} } \left(\frac{0.8}{1.4} \right)^{2} } =\underline{\underline{2.98m^{3}/ s } } \).

Thus, the volume flow in pipe ② is given by

\(\underline{\underline{Q_{2} } } =Q-Q_{1} =5-2.98=\underline{\underline{2.02 m^{3}/ s } } \).

b) From the volume flows the flow velocities are determined as

\(v_{1}=\frac{4Q_{1} }{D^{2}_{1} \pi }=\frac{4\cdot 2.98}{1.4^{2}\pi }=1.94\frac{m}{s} \) , \(v_{2}=\frac{4Q_{2} }{D^{2}_{1} \pi }=\frac{4\cdot 2.02}{0.8^{2}\pi }=4.02\frac{m}{s} \) .

Herewith, the (related) pressure loss can be calculated:

\(\underline{\underline{\frac{\Delta p_{v_{1} } }{\varrho g}=\frac{\zeta _{1}v^{2}_{1} }{2g}=0.25 m } } \) , \(\underline{\underline{\frac{\Delta p_{v_{2} } }{\varrho g}=\frac{\zeta _{2}v^{2}_{2} }{2g}=0.25 m } } \) .
