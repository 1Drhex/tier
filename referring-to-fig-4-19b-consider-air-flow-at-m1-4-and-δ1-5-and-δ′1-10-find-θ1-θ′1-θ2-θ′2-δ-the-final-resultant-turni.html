Guest Question.

Referring to Fig. 4.19b, consider air flow at \(M_{1} = 4\) and \(δ_{1} = 5°\) and \(\acute{\delta } _{1} = 10°\). Find \(θ_{1}, \acute{\theta } _{1}, θ_{2}, \acute{\theta } _{2}, δ\) (the final resultant turning angle) and all other properties in Regions 3 and \(\acute{3}\).

Admin
Solution

For \(M_{1} = 4\) and \(δ_{1} = 5°\), \(θ_{1} ≈ 18°\) from Fig. 4.12. Therefore, \(M_{1n} = M_{1} \sin18° ≈ 1.24\). Therefore, using shock table or Eqn. (4.20) \(M^{2}_{2} =1-\epsilon =2-M^{2}_{1}\) yields

\(\frac{p_{2} }{p_{1} }=1.6272\) and \(\beta =\frac{p_{2}- p_{1}}{p_{1} }=0.6272\)

\(\frac{p_{t2} }{p_{t1} }=0.9884 \frac{p_{t2} -p_{t1}}{p_{t1} } =1.2\)%

 

\(\frac{T_{2} }{T_{1} }=1.1531\)

 

\(M_{2n}=0.8183\)

 

\(M_{2t}= M_{1t}\sqrt{\frac{T_{1} }{T_{2} }} =M_{1} \cos \theta _{1} \sqrt{\frac{T_{1} }{T_{2} }}=4\cos 18° \sqrt{\frac{1}{1.1531} } =3.543\)

Therefore, \(M_{2} = 3.64\).

\(\acute{\delta } _{1} = 10°\) leads to \(\acute{\theta } _{1}=23°\), also from Fig. 4.12 (or Appendix E). Therefore, \(M_{1n} = M_{1} \sin23° ≈ 1.56\). Therefore, using the shock table or Eqn. (4.20) \(M^{2}_{2} =1-\epsilon =2-M^{2}_{1}\) yields

\(\frac{\acute{p}_{2} }{p_{1}}=2.673 \) and \(\beta =\frac{p_{\acute{p}_{2}}- p_{1}}{p_{1} }=1.673\)

\(\frac{\acute {p}_{t2} }{p_{t1} }=0.910\frac{\acute {p}_{t2}- p_{t1}}{p_{t1} }=9\)%

 

\(\frac{\acute{T} _{2} }{T_{1} }=1.361\)

 

\(\acute {M}_{2n}=0.681\)

 

\(\acute {M}_{2t}= M_{1t}\sqrt{\frac{T_{1} }{T_{2} }} =M_{1} \cos \acute {\theta} _{1} \sqrt{\frac{T_{1} }{T_{2} }}=4\cos 23° \sqrt{\frac{1}{1.1531} } =3.156\)

Therefore, \(\acute {M}_{2} = 3.229\).

Because the flow in Regions \(\acute {2}\) is turning through a larger angle upward than that in Regions 2 downward, it is reasonable to expect that the final flow is turned upward from the incident flow. By guessing the value of δ, \(p_{3}\), and \(\acute {p}_{3}\) can be found. If their values are equal, then the guessed value of δ is the final flow deflection angle and other properties in Regions 3 and \(\acute {3}\) can be determined.

First guess δ = 6°, then \(δ_{2} = δ_{1} + δ = 5° + 6° = 11°\) for \(M_{2} = 3.64\), \(θ_{2} ≈ 23.0°\).

Therefore, \(M_{2n} = M_{2} \sin23.0° ≈ 1.42\) and \(\frac{p_{3} }{p_{2} }≈ 2.186\) and \(\frac{p_{3} }{p_{1} }=\frac{p_{3} }{p_{2} } \times \frac{p_{2} }{p_{1} }≈ 2.186 \times 1.627 ≈ 3.557\).

Then \(\acute {δ}_{2} = \acute {δ}_{1} + δ = 10° – 6° = 4°\) for \(\acute {M}_{2} = 3.229, \acute {θ}_{1} ≈ 20.0°\). Therefore, \(\acute {M}_{2n} = \acute {M}_{2} \sin20° ≈ 1.10\) and \( \frac{\acute {p}_{3} }{p_{2} }≈ 1.245\) and \(\frac{\acute {p}_{3} }{p_{1} }=\frac{\acute {p}_{3} }{\acute {p}_{2} } \times \frac{\acute {p}_{2} }{p_{1} }≈ 1.245 \times 2.673 ≈ 3.352\).

Therefore for \(δ = 6°\), \(\frac{p_{3}}{p_{1}} \gt \frac{\acute{p}_{3} }{p_{1}} \).

Guess \(δ = 5°\), then \(δ_{2} = δ_{1} + δ = 5° + 5° = 10°\) for \(M_{2} = 3.64\), \(θ_{2} ≈ 22.0°\).

Therefore, \(M_{2n} = M_{2} \sin22.0° ≈ 1.36\) and \(\frac{p_{3} }{p_{2} } ≈ 1.991\) and \(\frac{p_{3} }{p_{1} }=\frac{p_{3} }{p_{2} } \times \frac{p_{2} }{p_{1} }≈ 1.991 \times 1.627 ≈ 3.239\).

For \(δ = 5°\), then \(\acute {δ}_{2} = \acute {δ}_{1} + δ = 10° – 5° = 5°\) for \(\acute {M}_{2} = 3.229, \acute {θ}_{2} ≈ 20.0°\).

Therefore, \(\acute {M}_{2n} = \acute {M}_{2} \sin21° ≈ 1.16\) and \(\frac{\acute {p}_{3} }{p_{2} }≈ 1.403\) and \(\frac{\acute {p}_{3} }{p_{1} }=\frac{\acute {p}_{3} }{\acute {p}_{2} } \times \frac{\acute {p}_{2} }{p_{1} }≈ 1.403\times 2.673 ≈3.750\)

Therefore for \(δ = 5°\), \(\frac{p_{3} }{p_{1} }\lt \frac{\acute p_{3} }{p_{1} }\). The solution must be 5° < δ < 6°.

One may assume the final solution is \(δ = 5.5°\) . The best final value can be found by further iterations. Once the value of δ is found, the answers for \(M_{3}, \acute {M}_{ 3}\) and all other properties can then be found following similar procedures to those in and .
