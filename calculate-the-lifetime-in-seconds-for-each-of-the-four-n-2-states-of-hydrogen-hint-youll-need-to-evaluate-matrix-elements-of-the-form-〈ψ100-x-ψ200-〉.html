Guest Question.

Calculate the lifetime (in seconds) for each of the four n = 2 states of hydrogen. Hint: You’ll need to evaluate matrix elements of the form \( \left\langle\psi_{100}|x| \psi_{200}\right\rangle,\left\langle\psi_{100}|y| \psi_{211}\right\rangle \) , and so on. Remember that \( x=r \sin \theta \cos \phi \), \( y=r \sin \theta \sin \phi, \text { and } z=r \cos \theta \). Most of these integrals are zero, so inspect them closely before you start calculating. Answer: \( 1.60 \times 10^{-9} \) seconds for all except \( \psi_{200} \), which is infinite.

Admin
Solution

In we calculated the matrix elements of z; all of them are zero except \( \langle 100|z| 210\rangle=\frac{2^{8}}{3^{5} \sqrt{2}} a \). As for x and y, we noted that \( |100\rangle,|200\rangle \text {, and }|210\rangle \text { are even (in } x, y) \text {, whereas }|21 \pm 1\rangle \) is odd. So the only non-zero matrix elements are \( \langle 100|x| 21 \pm 1\rangle \text { and }\langle 100|y| 21 \pm 1\rangle \). Using the wave functions in :

\( \langle 100|x| 21 \pm 1\rangle=\frac{1}{\sqrt{\pi a^{3}}}\left(\frac{\mp 1}{8 \sqrt{\pi a^{3}}}\right) \frac{1}{a} \int e^{-r / a} r e^{-r / 2 a} \sin \theta e^{\pm i \phi}(r \sin \theta \cos \phi) r^{2} \sin \theta d r d \theta d \phi \).

\( =\mp \frac{1}{8 \pi a^{4}} \int_{0}^{\infty} r^{4} e^{-3 r / 2 a} d r \int_{0}^{\pi} \sin ^{3} \theta d \theta \int_{0}^{2 \pi}(\cos \phi \pm i \sin \phi) \cos \phi d \phi \).

\( =\frac{\mp 1}{8 \pi a^{4}}\left[4 !\left(\frac{2 a}{3}\right)^{5}\right]\left(\frac{4}{3}\right)(\pi)=\mp \frac{2^{7}}{3^{5}} a \).

\( \langle 100|y| 21 \pm 1\rangle=\frac{\mp 1}{8 \pi a^{4}}\left[4 !\left(\frac{2 a}{3}\right)^{5}\right]\left(\frac{4}{3}\right) \int_{0}^{2 \pi}(\cos \phi \pm i \sin \phi) \sin \phi d \phi \).

\( =\frac{\mp 1}{8 \pi a^{4}}\left[4 !\left(\frac{2 a}{3}\right)^{5}\right]\left(\frac{4}{3}\right)(\pm i \pi)=-i \frac{2^{7}}{3^{5}} a \).

\( \langle 100| r | 200\rangle=0 ; \quad\langle 100| r | 210\rangle=\frac{2^{7} \sqrt{2}}{3^{5}} a \hat{k} ; \quad\langle 100| r | 21 \pm 1\rangle=\frac{2^{7}}{3^{5}} a(\mp \hat{i}-i \hat{j}), \text { and hence } \)

\( \wp^{2}=0(\text { for }|200\rangle \rightarrow|100\rangle), \quad \text { and }|\wp|^{2}=(q a)^{2} \frac{2^{15}}{3^{10}}(\text { for }|210\rangle \rightarrow|100\rangle \text { and }|21 \pm 1\rangle \rightarrow|100\rangle) \).

Meanwhile, \( \omega=\frac{E_{2}-E_{1}}{\hbar}=\frac{1}{\hbar}\left(\frac{E_{1}}{4}-E_{1}\right)=-\frac{3 E_{1}}{4 \hbar} \) so for the three l = 1 states:

\( A=-\frac{3^{3} E_{1}^{3}}{2^{6} \hbar^{3}} \frac{(e a)^{2} 2^{15}}{3^{10}} \frac{1}{3 \pi \epsilon_{0} \hbar c^{3}}=-\frac{2^{9}}{3^{8} \pi} \frac{E_{1}^{3} e^{2} a^{2}}{\epsilon_{0} \hbar^{4} c^{3}}=\frac{2^{10}}{3^{8}}\left(\frac{E_{1}}{m c^{2}}\right)^{2} \frac{c}{a} \).

\( =\frac{2^{10}}{3^{8}}\left(\frac{13.6}{0.511 \times 10^{6}}\right)^{2} \frac{\left(3.00 \times 10^{8} m / s \right)}{\left(0.529 \times 10^{-10} m \right)}=6.27 \times 10^{8} / s ; \quad \tau=\frac{1}{A}=1.60 \times 10^{-9} s \).

for the three l = 1 states (all have the same lifetime); \( \tau=\infty \) for the l = 0 state.
