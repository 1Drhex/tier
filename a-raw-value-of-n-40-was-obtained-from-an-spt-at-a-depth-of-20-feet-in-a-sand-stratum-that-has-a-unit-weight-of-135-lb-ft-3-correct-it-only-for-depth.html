Guest Question.

A raw value of N = 40 was obtained from an SPT at a depth of 20 feet in a sand stratum that has a unit weight of 135 \({ lb }/{ { ft }^{ 3 } }\) . Correct it only for depth.

Admin
Solution

Any of these three methods will provide acceptable answers. Notice how similar their results are from
each other:
1. Using the Bazaraa Method (1967):
\({ N }_{ corrected }\)=\(\frac { 4{ N }^{ ' } }{ 1+2{ P }_{ o } } \quad if\quad { P }_{ o }\le 1.5\quad { kpis }/{ { ft }^{ 2 } } and \quad { N }_{ corrected }\)=\(\frac { 4{ N }^{ ' } }{ 3.25+0.5{ P }_{ o } } \quad if\quad { P }_{ o }\ge 1.5\quad { kpis }/{ { ft }^{ 2 } }\) but

\({ p }_{ o }\)=\(\frac { (20ft)(135\quad { lb }/{ { ft }^{ 3 } }) }{ 1000\quad { lb }/{ kip } }\) =\(2.70\quad { kips }/{ { ft }^{ 2 } }>1.5\quad { kips }/{ { ft }^{ 2 } }\)

therefore \({ N }_{ corrected }\)=\(\frac { 4{ N }^{ ' } }{ 3.25+0.5{ P }_{ o } }\) =\(\frac { 4(40) }{ 3.25+0.5(2.70{ kpis }/{ { ft }^{ 2 } }) }\) =35

 

2. Using the Peck Method (1974):

\({ N }_{ corrected }\)=\({ N }^{ ' }{ C }_{ N }\quad where\quad { C }_{ N }\)=\(0.77\log _{ 10 }{ \frac { 20 }{ { P }_{ 0 } } } if\quad { p }_{ 0 }\quad is\quad in\quad tons/{ ft }^{ 2 }\) or \({ C }_{ N }\)=\(0.77\log _{ 10 }{ \frac {1915 }{ { P }_{ 0 } } } if\quad { p }_{ 0 }\quad is\quad in\quad kN/{ m }^{ 2 }\) but \({ p }_{ 0 }\)=\(\frac { (20ft)(135\quad lb/{ ft }^{ 3 }) }{ 2000\quad lb/ton }\) =\(1.35\quad tons/{ ft }^{ 2 }\)=\(2.70\quad kips/{ ft }^{ 2 }\)

\(\therefore \) \({ C }_{ N }\)=0.77\(\log _{ 10 } \)\(\frac { 20 }{ 1.35tons/{ ft }^{ 2 } } \)=0.90

\(\therefore \) \({ N }_{ corrected }\)=(40)(0.90)=36

3. The Liao-Whitman Method (1986), as used in Exploration-03

\({ N }_{ corrected }\)=\({ N }^{ ' }\sqrt { \frac { 100 }{ { p }_{ 0 } } } with\quad { p }_{ 0 }\quad in\quad kN/{ m }^{ 2 }\) or =\({ N }^{ ' }\sqrt { \frac { 2.000psf }{ { p }_{ 0 } } } with\quad { p }_{ 0 }\quad in\quad psf\)

but \({ p }_{ 0 }\)=\((1.35\quad tons/{ ft }^{ 2 })\left[ \frac { 96.1\quad kN/{ m }^{ 2 } }{ 1\quad ton/{ ft }^{ 2 } } \right]\) =129.7\(kN/{ m }^{ 2 }\)

\(\therefore \)\({ N }_{ corrected }\)=\(40\sqrt { \frac { 100\quad kN/{ m }^{ 2 } }{ 129.7\quad kN/{ m }^{ 2 }\quad } }\) =35
