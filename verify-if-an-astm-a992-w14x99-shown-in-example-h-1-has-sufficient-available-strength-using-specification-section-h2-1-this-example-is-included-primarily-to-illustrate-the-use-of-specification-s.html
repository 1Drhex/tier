Guest Question.

Verify if an ASTM A992 W14×99 shown in has sufficient available strength, using Specification Section H2.1. This example is included primarily to illustrate the use of Specification Section \(K L_{x}=K L_{y}=L_{b}=14.0 ft\)

LRFD ASD
\(\begin{array}{l}P_{u}=400 \text { kips } \\M_{u x}=250 \text { kip-ft } \\M_{u y}=80.0 \text { kip-ft }\end{array}\) \(\begin{array}{l}P_{a}=267 \text { kips } \\M_{a x}=167 \text { kip-ft } \\M_{a y}=53.3 \text { kip-ft }\end{array}\)
Admin
Solution

Material Properties:

ASTM A992 \(F_{y}=50 ksi \quad F_{u}=65 ksi\)

Manual
Table 2-3

Geometric Properties:

W14×99 \(A=29.1 \text { in. }^{2} \quad S_{x}=157 \text { in. }^{3} \quad S_{y}=55.2 in ^{3}\)

Manual
Table 1-1

Calculate the required flexural and axial stresses

LRFD ASD
\(\begin{array}{l}f_{a}=\frac{P_{u}}{A}=\frac{400 kips }{29.1 in .^{2}}=13.7 ksi \\f_{b w}=\frac{M_{u x}}{S_{x}}=\frac{250 kip – ft }{157 in .^{3}}\left(\frac{12 in .}{ ft }\right)=19.1 ksi \\f_{b z}=\frac{M_{u y}}{S_{y}}=\frac{80.0 kip – ft }{55.2 in .^{3}}\left(\frac{12 in .}{ ft }\right)=17.4 ksi\end{array}\) \(\begin{array}{l}f_{a}=\frac{P_{a}}{A}=\frac{267 kips }{29.1 in .^{2}}=9.18 ksi \\f_{b w}=\frac{M_{a x}}{S_{x}}=\frac{167 kip – ft }{157 in .^{3}}\left(\frac{12 in .}{ ft }\right)=12.8 ksi \\f_{b z}=\frac{M_{a y}}{S_{y}}=\frac{53.3 kip – ft }{55.2 in .^{3}}\left(\frac{12 in }{ ft }\right)=11.6 ksi\end{array}\)

Calculate the available flexural and axial stresses from the available strengths in

LRFD ASD
\(\begin{array}{l}F_{a}=\phi_{c} F_{c r}=\frac{\phi_{c} P_{n}}{A}=\frac{1,130 kips }{29.1 in .^{2}}=38.8 ksi \\F_{b w}=\frac{\phi_{b} M_{n x}}{S_{x}}=\frac{645 kip – ft }{157 in .{ }^{3}}\left(\frac{12 in .}{ ft }\right)=49.3 ksi \\F_{b z}=\frac{\phi_{b} M_{n y}}{S_{y}}=\frac{311 kip – ft }{55.2 in .{ }^{3}}\left(\frac{12 in .}{ ft }\right)=67.6 ksi\end{array}\) \(\begin{array}{l}F_{a}=\frac{F_{c r}}{\Omega_{c}}=\frac{P_{n}}{\Omega_{c} A}=\frac{751 kips }{29.1 in ^{2}}=25.8 ksi \\F_{b w}=\frac{M_{n x}}{\Omega_{b} S_{x}}=\frac{431 kip – ft }{157 in .^{3}}\left(\frac{12 in .}{ ft }\right)=32.9 ksi \\F_{b z}=\frac{M_{n y}}{\Omega_{b} S_{y}}=\frac{207 kip – ft }{55.2 in .^{3}}\left(\frac{12 in .}{ ft }\right)=45.0 ksi\end{array}\)

As shown in the LRFD calculation of \(F_{b z}\) above, the available flexural stresses can exceed the yield stress in cases where the available strength is governed by yielding and the yielding strength is calculated using the plastic section modulus.

Calculate the combined stress ratio

LRFD ASD
\(\left|\frac{f_{a}}{F_{a}}+\frac{f_{b w}}{F_{b w}}+\frac{f_{b z}}{F_{b z}}\right| \leq 1.0\) \(\left|\frac{f_{a}}{F_{a}}+\frac{f_{b w}}{F_{b w}}+\frac{f_{b z}}{F_{b z}}\right| \leq 1.0\)
\(\left|\frac{13.7 ksi }{38.8 ksi }+\frac{19.1 ksi }{49.3 ksi }+\frac{17.4 ksi }{67.6 ksi }\right|=0.998<1.0\) o.k. \(\left|\frac{9.18 ksi }{25.8 ksi }+\frac{12.8 ksi }{32.9 ksi }+\frac{11.6 ksi }{45.0 ksi }\right|=1.00\) o.k.

Eqn. H2-1

Note: this check is made at a point. The designer must therefore select which point along the length is critical, or check multiple points if the critical point can not be readily determined.
