Guest Question.

A drilled shaft is shown in Figure 14.39. The uncorrected average standard penetration number (\(\textbf{N}_{60}\)) within a distance of 2\(\textbf{D}_{b}\) below the base of the shaft is about 30. Determine
a. The ultimate load-carrying capacity
b. The load-carrying capacity for a settlement of 12 mm. Use Eq. (14.95).

\(\beta = 2.0 - 0.15 z^{0.75}_{i}\qquad (0.25 \leq \beta \leq 1.8)\)
Admin
Solution
Part a

From Eqs. (14.93) and (14.95),

\(f_{i} = \beta \sigma^{\prime}_{ozi}\\and\\\beta = 2.0 – 0.15 z^{0.75}\\\text{For this problem, }z_{i} = 6/2 = 3 m, so\\\beta = 2 – (0.15)(3)^{0.75} = 1.658\\and\\\sigma^{\prime}_{ozi} = \gamma z_{i} = (16)(3) = 48 kN/m^{2}\\Thus,\\f_{i} = (48)(1.658) = 79.58 kN/m^{2}\\and\\\Sigma f_{i}p\Delta L_{i} = (79.58)(\pi \times 1)(6) = 1500 kN\)

From Eq. (14.97), \(q_{p}(kN/m^{2}) = 57.5 N_{60} \leq 4.3 MN/m^{2}\\q_{p} = 57.5 N_{60} = (57.5)(30) = 1725 kN/m^{2}\)

Note that \(D_{b}\) is greater than 1.27 m. So we will use Eq. (14.98). \(q_{pr} = \frac{1.27}{D_{b}(m)}q_{p}\qquad (for D_{b} \geq 1.27 m)\\q_{pr} = \left(\frac{1.27}{D_{b}}\right) q_{p} = \left(\frac{1.27}{1.5}\right) (1725) \approx 1461 kN/m^{2}\\Now\\q_{pr}A_{p} = (1461)\left(\frac{\pi}{4} \times 1.5^{2}\right) \approx 2582 kN\\Hence,\\Q_{u(net)} = q_{pr}A_{p} + \Sigma f_{i}p\Delta L_{i} = 2582 + 1500 = \textbf{4082 kN}\)

Part b

We have

\(\frac{\text{Allowable settlement}}{D_{s}} = \frac{12}{(1.0)(1000)} = 0.12 = 1.2 \%\)

Table 14.10 shows, that for a normalized settlement of 1.2%, the normalized load is about 0.974. Thus, the side load transfer is \((0.974) (1500) \approx 1461 kN.\\Similarly,\\\frac{\text{Allowable settlement}}{D_{b}} = \frac{12}{(1.5)(1000)} = 0.008 = 0.8\%\)

Table 14.11 indicates, that for a normalized settlement of 0.8%, the normalized base load is about 0.25. So the base load is \((0.25)(2582) = 645.5 kN\). Hence, the total load is

\(Q = 1461 + 645.5 \approx \textbf{2102 kN}\)
