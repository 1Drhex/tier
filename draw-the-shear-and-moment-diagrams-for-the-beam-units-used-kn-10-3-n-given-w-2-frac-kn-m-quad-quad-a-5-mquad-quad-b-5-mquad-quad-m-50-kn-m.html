Guest Question.

Draw the shear and moment diagrams for the beam.

Units Used:

\(kN\) = \({ 10 }^{ 3 }\) \(N\)

Given:

\(w\) = \(2\) \(\frac { kN }{ m } \quad \quad a\) = \(5\) \(m\quad \quad b\) = \(5\) \(m\quad \quad M\) = \(50\) \(kN.m\)

Admin
Solution

Guesses \(\quad A=1N\quad \quad C=1N\)

Given \(\quad wa\left( b+\frac { a }{ 2 } \right) -A\left( a+b \right) -M=0\quad \quad A+C-wa=0\quad \quad \left( \begin{matrix} A \\ C \end{matrix} \right) =Find\left( A,C \right) \)

 

\({ x }_{ 1 }=0,0.001a..a\quad \quad { V }_{ 1 }\left( { x }_{ 1 } \right) =\left( A-w{ x }_{ 1 } \right) \frac { 1 }{ kN } \quad \quad { M }_{ 1 }\left( { x }_{ 1 } \right) =\left( A{ x }_{ 1 }-w{ x }_{ 1 }\frac { { x }_{ 1 } }{ 2 } \right) \frac { 1 }{ kN.m }\)

 

\({ x }_{ 2 }=a,1.001a..a+b\quad \quad { V }_{ 2 }\left( { x }_{ 2 } \right) =-C\frac { 1 }{ kN } \quad \quad { M }_{ 2 }\left( { x }_{ 2 } \right) =\left[ -M+C\left( a+b-{ x }_{ 2 } \right) \right] \frac { 1 }{ kN.m }\)
