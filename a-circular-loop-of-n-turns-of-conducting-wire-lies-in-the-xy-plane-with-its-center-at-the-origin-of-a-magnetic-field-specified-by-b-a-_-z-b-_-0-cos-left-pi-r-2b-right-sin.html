Guest Question.

A circular loop of \(N\) turns of conducting wire lies in the \(xy\)-plane with its center at the origin of a magnetic field specified by \(B={ a }_{ z }{ B }_{ 0 }\cos { \left( \pi r/2b \right) } \sin { \omega t }\) , where \(b\) is the radius of the loop and \(\omega\) is the angular frequency. Find the emf induced in the loop.

Admin
Solution

The problem specified a stationary loop in a time-varying magnetic field; hence Eq. (6-12) can be used directly to find the induced emf, \(\gamma \) . The magnetic flux linking each turn of the circular loop is

\(\Phi =\int _{ S }^{ }{ B } ds\)

 

\(\quad =\int _{ 0 }^{ b }{ \left[ { a }_{ z }{ B }_{ 0 }\cos { \left( \frac { \pi r }{ 2b } \right) \sin { \omega t } } \right] .\left( { a }_{ z }2\pi rdr \right) }\)

 

\(\quad =\frac { 8{ b }^{ 2 } }{ \pi } \left( \frac { \pi }{ 2 } -1 \right) { B }_{ 0 }\sin { \omega t }\)

 

Since there are \(N\) turns. the total flux linkage is \(N\Phi\), and we obtain

 

\(\gamma =-N\frac { d\Phi }{ dt }\)

 

\(\quad =-\frac { 8N }{ \pi } { b }^{ 2 }\left( \frac { \pi }{ 2 } -1 \right) { B }_{ 0 }\omega \cos { \omega t } \quad \quad \left( V \right) \)

 

As the phase of \(\cos { \omega t }\) leads that of \(\sin { \omega t }\) by \({ 90 }^{ o }\), we see that the phase of the induced emf lags behind that of the flux linkage by \({ 90 }^{ o }\).

 

\(\gamma =-\frac { d\Phi }{ dt } \quad \quad \left( V \right) \quad \quad \left( 6-12 \right) \)
