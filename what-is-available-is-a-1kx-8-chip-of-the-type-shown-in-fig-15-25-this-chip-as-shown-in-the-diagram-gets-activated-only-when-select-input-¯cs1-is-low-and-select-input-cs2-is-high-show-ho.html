Guest Question.

What is available is a \(1K× 8\) chip of the type shown in Fig. 15.25. This chip, as shown in the diagram, gets activated only when select input \(\overline{CS1}\) is LOW and select input CS2 is HIGH. Show how two such ROMs can be connected to get \(2K×8\) ROM without using any additional logic.

Admin
Solution

• Figure 15.26 shows the arrangement.

• The address bit \(AB_{10}\) is low for the first 1024 address inputs (from \(00000000000 to 01111111111\)) and ROM-1 is selected.

• For the remaining \(1024\) address inputs (from \(10000000000 to 11111111111\)), the \(AB_{10}\) bit is HIGH, thus enabling ROM-2.
