Guest Question.

The arched surface \(AB\) is shaped in the form of a quarter circle. If it has a length \(L\), determine the horizontal and vertical components of the resultant force caused by the water acting on the surface.The density of water is \({ \rho }{ w }\)

Units Used:

\(Mg = { 10 }^{ 3 } kg \)

 

\(kN = { 10 }^{ 3 }N \)

Given:

\(L = 8 m \)

 

\({ \rho }_{ w } = 1.0 \frac { Mg }{ { m }^{ 3 } } \)

 

\(a = 3 m \)

 

\(b = 2 m \)

 

\(g = 9.81 \frac { m }{ { s }^{ 2 } }\)
Admin
Solution
\({ F }_{ 3 }={ \rho }_{ w }gabL\quad\quad\quad \quad \quad \quad\quad \quad \quad { F }_{ 3 }=470.88kN\)

 

\({ F }_{ 2 }={ \rho }_{ w }gabL\quad \quad\quad \quad\quad\quad \quad \quad \quad { F }_{ 2 }=470.88kN\)

 

\({ F }_{ 1 }={ \rho }_{ w }g\frac { b }{ 2 } bL\quad\quad \quad\quad \quad\quad \quad \quad \quad { F }_{ 1 }=156.96kN\)

 

\(W=\left( { b }^{ 2 }-\frac { \pi { b }^{ 2 } }{ 4 } \right) L{ \rho }_{ w }g\quad \quad \quad\quad \quad W=67.368kN\)

 

\({ F }_{ x }={ F }_{ 1 }+{ F }_{ 2 }\quad \quad\quad \quad\quad \quad \quad \quad { F }_{ x }=628kN\)

 

\({ F }_{ y }={ F }_{ 2 }+W\quad \quad \quad\quad\quad \quad \quad \quad { F }_{ y }=538kN\)
