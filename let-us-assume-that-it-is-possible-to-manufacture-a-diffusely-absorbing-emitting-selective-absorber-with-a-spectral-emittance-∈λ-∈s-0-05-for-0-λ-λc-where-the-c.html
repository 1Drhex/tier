Guest Question.

Let us assume that it is possible to manufacture a diffusely absorbing/emitting selective absorber with a spectral emittance \(\epsilon _{\lambda } =\epsilon _{s} =0.05\) for \(0\lt \lambda \lt \lambda _{c}\) and \(\epsilon _{\lambda } =\epsilon _{c} =0.95\) for \(\lambda \gt \lambda _{c}\) , where the cutoff wavelength can be varied through manufacturing methods. Determine the optimum cutoff wavelength for a solar collector with an absorber plate at 350 K that is exposed to solar irradiation of \(q_{sun} =1000W/m^{2}\) at an angle of \(\theta _{s} =30^{\circ }\) ° off-normal. What is the net radiative energy gain for such a collector?

Admin
Solution

A simple energy balance on the surface, using equations

\(\epsilon (T)=\frac{E(T)}{E_{b}(T) }\) , and \(H_{\lambda } (r,\lambda )=\int_{2\pi }^{}{I_{\lambda }(r,\lambda ,\widehat{s_{i} } )\cos \theta _{i}d\Omega _{i} }\) . leads to

\(q_{net} =E-H^{\prime }_{abs} (\theta _{s})=\epsilon E_{b} -\alpha ^{\prime } (\theta _{s})H^{\prime }(\theta _{s})\)

where \(q_{net} \gt 0\) if a net amount of energy leaves the surface, \(q_{net} \lt 0\) if energy is collected. Total, hemispherical emittance follows from equation

\(\epsilon (T)=\frac{\int_{0}^{\infty }{E_{\lambda }(T,\lambda ) d\lambda } }{E_{b }(T)} =\frac{1}{n^{2} \sigma T^{4} } \int_{0}^{\infty } \epsilon _{\lambda } (T,\lambda ) E_{b \lambda } (T,\lambda ) d\lambda\) .

while total, directional absorptance is determined from equation

\(\alpha ^{\prime } (r,\widehat{s_{i} } )\equiv \frac{\int_{0}^{\infty }\alpha ^{\prime }_{\lambda }(T,\lambda ,\widehat{s_{i} })I_{\lambda } (r,\widehat{s_{i} } ) d\lambda }{\int_{0}^{\infty }I_{\lambda } (r,\widehat{s_{i} } ) d\lambda }\) .

For our diffuse absorber we have \(\alpha ^{\prime }_{\lambda } (\lambda ,\theta )=\epsilon _{\lambda } (\lambda )\) and

\(\epsilon =\frac{1}{\sigma T^{4}_{coll} } \left[\epsilon _{s}\int_{0}^{\lambda _{c} }{E_{b\lambda } (T_{coll},\lambda )d\lambda }+ \epsilon _{c}\int_{\lambda _{c}}^{\infty }{E_{b\lambda } (T_{coll},\lambda )d\lambda } \right] =\epsilon _{s}+\frac{(\epsilon _{c}-\epsilon _{s})}{\sigma T^{4}_{coll}} \int_{\lambda _{c}}^{\infty } E_{b\lambda } (T_{coll},\lambda )d\lambda\) ,

\(\alpha = \frac{1}{\sigma T^{4}_{sun} } \left[\epsilon _{s}\int_{0}^{\lambda _{c} }{E_{b\lambda } (T_{sun},\lambda )d\lambda }+ \epsilon _{c}\int_{\lambda _{c}}^{\infty }{E_{b\lambda } (T_{sun},\lambda )d\lambda } \right] =\epsilon _{s}+\frac{(\epsilon _{c}-\epsilon _{s})}{\sigma T^{4}_{sun}} \int_{\lambda _{c}}^{\infty } E_{b\lambda } (T_{sun},\lambda )d\lambda\) .

Substituting these expressions into our energy balance leads to

\(q_{net}= \epsilon _{s}(\sigma T^{4}_{coll} -q_{sun}\cos \theta _{s} )+(\epsilon _{c} -\epsilon _{s})\int_{\lambda _{c} }^{\infty }{\left[E_{b\lambda } (T_{coll},\lambda ) -\frac{q_{sun}\cos \theta _{s}}{\sigma T^{4}_{sun}}E_{b\lambda } (T_{sun},\lambda ) \right] } d\lambda\) .

Optimizing the value of \(\lambda _{c}\) implies finding a maximum for \(q_{net}\) . Therefore, from Leibnitz’ rule (see, e.g., [76]), which states that

\(\frac{d}{dx} \int_{a(x)}^{b(x)}{f(x,y)dy}=\frac{db}{dx}f(x,b)-\frac{da}{dx}f(x,a)+\int_{a}^{b}{\frac{df}{dx} (x,y)dy}\) ,

we find \(\frac{d q_{net}}{d \lambda _{c} } =-(\epsilon _{c} -\epsilon _{s})\left[E_{b\lambda } (T_{coll},\lambda _{c} )-\frac{q_{sun}\cos \theta _{s}}{\sigma T^{4}_{sun}}E_{b\lambda } (T_{sun},\lambda _{c} ) \right]=0\) ,

or \(E_{b\lambda } (T_{coll},\lambda _{c})=\frac{q_{sun}\cos \theta _{s}}{\sigma T^{4}_{sun}} E_{b\lambda } (T_{sun},\lambda _{c} )\) .

Note that the cutoff wavelength does not depend on the values for \(\epsilon _{c}\) and \(\epsilon _{s}\) . Using Planck’s law, equation

\(E_{b\lambda }(T,\lambda )=\frac{2\pi hc^{2}_{0} }{n^{2} \lambda ^{5} [e^{h c_{0}/n\lambda kT}-1 ]} , (n=const)\) ,

with n = 1 (surroundings are air), the last expression reduces to

\(exp(C_{2} /\lambda _{c} T_{coll} )-1=\frac{\sigma T^{4}_{sun} }{q_{sun}\cos \theta _{s}} [exp(C_{2} /\lambda _{c} T_{sun} )-1]\) .

This transcendental equation needs to be solved by iteration. As a first guess one may employ Wien’s distribution, equation

\(E_{b\lambda } \simeq \frac{2\pi hc^{2}_{0} }{n^{2} \lambda ^{5} } e^{-h c_{0}/n\lambda kT}=\frac{C_{1} }{n^{2} \lambda ^{5}} e^{-C_{2}/n\lambda T} , \frac{h c_{0}}{n\lambda kT} \gg 1\) ,

(dropping two “-1” terms). \(exp(C_{2}/\lambda _{c} T_{coll})\simeq \frac{\sigma T^{4}_{sun}}{q_{sun}\cos \theta _{s}} exp (C_{2}/\lambda _{c} T_{sun})\)

or \(exp\left[\frac{C_{2}}{\lambda _{c}}\left(\frac{1}{T_{coll}}-\frac{1}{T_{sun}} \right) \right]\simeq \frac{\sigma T^{4}_{sun}}{q_{sun}\cos \theta _{s}}\) ,

\(\lambda _{c}\simeq C_{2} \left(\frac{1}{T_{coll}}-\frac{1}{T_{sun}} \right)/\ln \frac{\sigma T^{4}_{sun}}{q_{sun}\cos \theta _{s}} =14.388\left(\frac{1}{350}-\frac{1}{5777} \right) \mu m/\ln \frac{5.670\times 10^{-8} \times 5777^{4} }{1000\times \cos 30^{\circ } }=3.45\mu m\) .

Iterating the full Planck’s law leads to a cutoff wavelength of \(\lambda _{c}=3.69 \mu m\) . Substituting these values into the expressions for emittance and absorptance,

\(\epsilon =\epsilon _{s} +(\epsilon _{c}-\epsilon _{s})[1-f(\lambda _{c}T_{coll})]=0.95-0.90+0.90f(3.69\times 350)=0.05+0.90\times 0.00413=0.054\) ,

\(\alpha =\epsilon _{s} +(\epsilon _{c}-\epsilon _{s})[1-f(\lambda _{c}T_{sun})]= 0.05+0.90\times f(3.69\times 5777)=0.05+0.90\times 0.98785=0.939\) .

The net heat flux follows then as

\(q_{net} =0.054\times 5.760\times 10^{-8} \times 350^{4} -0.939\times 1000\times \cos 30^{\circ } =-767W/m^{2}\) .

Actually, neither \(f(\lambda _{c} T_{coll})\simeq 0\) nor \(f(\lambda _{c} T_{sun})\simeq 1\) ~ 1 is particularly sensitive to the exact value of \(\lambda _{c}\) , because there is very little spectral overlap between solar radiation (95% of which is in the wavelength \(range^{8}\) of \(\lambda \lt 2.2\mu m\)) and blackbody emission at 350 K (95% of which is at \(\lambda \gt 5.4\mu m\)) .

Surfaces can be made directionally selective by mechanically altering the surface finish on a microscale (microgrooves) or macroscale. For example, large V-grooves (large compared with the wavelengths of radiation) tend to reflect incoming radiation several times for near-normal incidence, as indicated in Fig. 3-35 (from Trombe and coworkers [96]) for an opening angle of \(\gamma = 30^{\circ }\), each time absorbing a fraction of the beam. The number of reflections decreases with increasing incidence angle, down to a single reflection for incidence angles \(\theta \gt 90^{\circ } -\gamma\) (or \(60^{\circ }\) in the case of Fig. 3-35). Hollands [97] has shown that this type of surface has a significantly higher normal emittance, which is important for collection of solar irradiation, than hemispherical emittance, which governs emission losses. A similarly shaped material, with flat black bottoms, was theoretically analyzed by Perlmutter and Howell [98]. Their analytical values for directional emittance were experimentally confirmed by Brandenberg and Clausen [27], as illustrated in Fig. 3-36.

Based on a blackbody at 5777 K. This number remains essentially unchanged for true, extraterrestrial solar irradiation [95], while the 95% fraction moves to even shorter wavelengths if atmospheric absorption is taken into account (cf. Fig, 1-3).
