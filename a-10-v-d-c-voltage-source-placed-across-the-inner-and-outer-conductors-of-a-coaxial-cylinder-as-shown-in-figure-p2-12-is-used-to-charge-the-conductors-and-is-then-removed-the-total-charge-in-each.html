Guest Question.

A 10-V d.c. voltage source, placed across the inner and outer conductors of a coaxial cylinder as shown in Figure P2-12, is used to charge the conductors and is then removed. The total charge in each conductor is ± Q. The inner conductor has a radius of a = 2 cm, the radius of the outer conductor is 4 cm, and the length of the cylinder is \(\ell = 6 cm\). Assuming no field fringing and free space between the conductors, find the:

(a) Electric field intensity between the conductors in terms of Q.

(b) Total charge Q on the inner and outer conductors.

(c) Surface charge density on the inner and outer conductors.

(d) Electric flux density between the conductors.

(e) Capacitance between the conductors.

(f) Energy stored between the conductors.

Solution

Using Colomb’s Law

(a)

\(\underset { S }{ \oiint } =\underline { D } \cdot d\underline { s } =\int _{ 0 }^{ \ell }{ \int _{ 0 }^{ 2\pi }{ ({ \hat { a } }_{ p }{ D }_{ p })\cdot } { \hat { a } }_{ p }pd\phi dz= } =\int _{ 0 }^{ \ell }{ \int _{ 0 }^{ 2\pi }{ { \varepsilon }_{ 0 }{ E }_{ p } } pd\phi dz= } 2\pi { \varepsilon }_{ 0 }\ell _{ p }{ E }_{ p }=Q\)

or

\({ E }_{ p }=\frac { Q }{ 2\pi { \varepsilon }_{ 0 }\ell } \frac { 1 }{ p } =\frac { Q }{ p } (3\times { 10 }^{ 11 })=3\times { 10 }^{ 11 }\frac { Q }{ p }\)

Since the voltage is 10 Volts, then

\(V=_{ a }\int _{ }^{ b }{ \underline { E } \cdot d\underline { \ell } } =_{ a }\int _{ }^{ b }{ ({ \hat { a } }_{ p }{ E }_{ p })\cdot } { \hat { a } }_{ p }dp=3\times { 10 }^{ 11 }Q_{ a }\int _{ }^{ b }{ \frac { dp }{ p } } =3\times { 10 }^{ 11 }Q\ell n(p)\overset { b }{ \underset { a }{ | } } \\ V=10=3\times { 10 }^{ 11 }Q\ell n(\frac { b }{ a } )=3\times { 10 }^{ 11 }Q\ell n(2)=3\times { 10 }^{ 11 }Q(0.69315)\\ Q=\frac { 10 }{ 3\times { 10 }^{ 11 }(0.69315) } =48.0898\times { 10 }^{ -12 }\quad C\)

(b)

\(Q=48.0898\times { 10 }^{ -12 }\quad C\)

(C)

\(q_{ sa }=-\frac { Q }{ { A }_{ a } } =-\frac { Q }{ 2\pi a\ell } =-\frac { 48.0898\times { 10 }^{ -12 } }{ 2\pi (2\times { 10 }^{ -2 })(6\times { 10 }^{ -2 }) } =-6.378\times { 10 }^{ -9 }C/{ m }^{ 2 }\\ q_{ sb }=\frac { Q }{ { A }_{ b } } =-\frac { Q }{ 2\pi b\ell } =\frac { 48.0898\times { 10 }^{ -12 } }{ 2\pi (4\times { 10 }^{ -2 })(6\times { 10 }^{ -2 }) } =3.189\times { 10 }^{ -9 }C/{ m }^{ 2 }\)

(d)

\(D={ \varepsilon }_{ 0 }{ E }_{ p }=8.854\times { 10 }^{ -12 }(\frac { 14.42695 }{ p } )=\frac { 1.27736 }{ p } { 10 }^{ -10 }C/{ m }^{ 2 }\)

(e)

\(C=\frac { Q }{ V } =\frac { 48.0898\times { 10 }^{ -12 } }{ 10 } =4.80898\times { 10 }^{ -12 }\) farads
