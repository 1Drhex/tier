Guest Question.

Elemental balance

Consider the fermentation of carbon monoxide in aqueous solution using the bacterium Methylotrophicum to produce acetic acid, butyric acid, and additional cells. Based on the reported yields, stoichiometry is given by

\(\begin{aligned}& CO +\left(-v_{ o _{2}}\right) O _{2}+\left(-v_{ N _{2}}\right) N _{2}+\left(-v_{ H _{2} O }\right) H _{2} O \rightarrow \\&0.52 CO _{2}+0.21 CH _{3} COOH +0.006 CH _{3} CH _{2} CH _{2} COOH +v_{ x } X\end{aligned}\)

where X represents the Methylotrophicum cells. Assuming that cells produced in this reaction can be represented by the Roels average biomass (that is, 1 C-mole cells = \(CH _{1.8} O _{0.5} N _{0.2}\)), determine all the remaining stoichiometric coefficients.

Admin
Solution

Let X =\(CH _{1.8} O _{0.5} N _{0.2}\). To find the remaining stoichiometric coefficients, we need to perform atom balances individually. In this reaction, we have four atoms involved: C, O, N, and H. Four balance equations can be written independently.

C-mole balance:

\(1-0 v_{ O _{2}}-0 v_{ N _{2}}-0 v_{ H _{2} O }=0.52+0.21 \times 2+0.006 \times 4+v_{ x }\) (E3.3.1)

O-mole balance:

\(1-2 v_{ O _{2}}-0 v_{ N _{2}}-v_{ H _{2} O }=0.52 \times 2+0.21 \times 2+0.006 \times 2+0.5 v_{ x } \) (E3.3.2)

N-mole balance:

\(0-0 v_{ O _{2}}-2 v_{N_{2}}-0 v_{ H _{2} O }=0.52 \times 0+0.21 \times 0+0.006 \times 0+0.2 v_{ x }\) (E3.3.3)

H-mole balance:

\(0-0 v_{ O _{ z }}-0 v_{ N _{ 2 }}-2 v_{ H _{2} O }=0.52 \times 0+0.21 \times 4+0.006 \times 8+1.8 v_{x}\) (E3.3.4)

 

Eq. (E3.3.1) \(1-0 v_{ O _{2}}-0 v_{ N _{2}}-0 v_{ H _{2} O }=0.52+0.21 \times 2+0.006 \times 4+v_{ x }\) through (E3.3.4) \(0-0 v_{ O _{ z }}-0 v_{ N _{ 2 }}-2 v_{ H _{2} O }=0.52 \times 0+0.21 \times 4+0.006 \times 8+1.8 v_{x}\) need to be solved simultaneously to find all the stoichiometric coefficients. From Eq. (E3.3.1) \(1-0 v_{ O _{2}}-0 v_{ N _{2}}-0 v_{ H _{2} O }=0.52+0.21 \times 2+0.006 \times 4+v_{ x }\) , we obtain

\(v_{x}=1-0.52-0.21 \times 2-0.006 \times 4=0.036\) (E3.3.5)

 

Using Eq. (E3.3.4) \(0-0 v_{ O _{ z }}-0 v_{ N _{ 2 }}-2 v_{ H _{2} O }=0.52 \times 0+0.21 \times 4+0.006 \times 8+1.8 v_{x}\) , we have

\(v_{ H _{2} O }=\left(0.21 \times 4+0.006 \times 8+1.8 v_{ x }\right) \div(-2)=-0.4764\) (E3.3.6)

 

Eq. (E3.3.3) \(0-0 v_{ O _{2}}-2 v_{N_{2}}-0 v_{ H _{2} O }=0.52 \times 0+0.21 \times 0+0.006 \times 0+0.2 v_{ x }\) leads to

\(v_{ N _{2}}=0.2 v_{ x } \div(-2)=-0.0036\) (E3.3.7)

 

Eq. (E3.3.2) \(1-2 v_{ O _{2}}-0 v_{ N _{2}}-v_{ H _{2} O }=0.52 \times 2+0.21 \times 2+0.006 \times 2+0.5 v_{ x } \) can be re-arranged to yield

\(v_{ O _{2}}=\left(1-v_{ H 2 O }-0.52 \times 2-0.21 \times 2-0.006 \times 2-0.5 v_{ x }\right) / 2=-0.0068\) (E3.3.8)

 

This concludes the determination. The final stoichiometry is given by

\(\begin{aligned}& CO +0.0068 O _{2}+0.0036 N _{2}+0.4764 H _{2} O \rightarrow \\&\quad 0.52 CO _{2}+0.21 CH _{3} COOH +0.006 CH _{3} CH _{2} CH _{2} COOH +0.036 CH _{1.8} O _{0.5} N _{0.2}\end{aligned}\)
