Solution

To plot the compaction curve, the dry density corresponding to the different water content and bulk density pairs can be determined with the relationship \(\rho _{b} =\rho _{d} \left(1+w\right)\), as shown in Table 2.4.

Table 2.4 Compaction data, determination of dry density.

Dry density \(\left[g/ cm^{3} \right] \) Bulk density \(\left[g/ cm^{3} \right] \) Water content \(\left[\%\right] \)
1.94 2.07 6.8
1.97 2.14 8.5
1.99 2.18 9.4
2.01 2.21 10.2
2.00 2.23 11.3
1.96 2.21 12.5
1.93 2.19 13.6

For determining zero air void line and 5% air voids, we can use the following relationships:

\(\rho _{d} =\frac{G_{s} \rho _{w}}{ 1+e} \Leftrightarrow \rho _{d} =\frac{G_{s} \rho _{w}}{1+\frac{wG_{s} }{S_{r} } }\)

 

\(\therefore \rho _{d\left(ZAV , S_{r}=1 \right) } \frac{G_{s} \rho _{w} }{1+wG_{s} }\)

\(A_{v} =\frac{V_{a} }{V}\) and \(\left(1-A_{v} \right) =1-\frac{V_{a} }{V} =\frac{V }{V}-\frac{V_{a} }{V}=\frac{V-V_{a} }{V}=\frac{V_{w}+ V_{s}}{V}\) or \( V=\frac{V_{w}+V_{s} }{\left(1-A_{{v} } \right) } \)

\(\rho _{d} =\frac{\rho _{b}}{\left(1+w\right) } = \frac{M}{V\left(1+w\right) }= \frac{M_{s} +M_{w} } {V\left(1+w\right) }\)

By replacing V for the previous expression, the following expression is obtained:

\(\rho _{d} =\frac{\left(M_{s}+M_{w} \right)\left(1-A_{\nu } \right) }{\left(V_{w}+V_{s} \right)\left(1+w\right) }\)

and we know that

\(\rho _{s} =\frac{M_{s} }{V_{s} }\) and \(G_{s} =\frac{\rho _{s}}{\rho _{w}} \) then \(V_{s}=\frac{M_{s}}{G_{s}\rho _{w}} \) and \(\rho _{w}=\frac{W_{w} }{V_{w} }\) then \(V_{w}=\frac{M_{w}}{\rho _{w}} \) or \(V_{w}=\frac{wM_{w}}{\rho _{w}}\)

then,

\(\rho _{d} =\frac{\left(M_{s}+M_{w} \right)\left(1-A_{\nu } \right) }{\left\lgroup\frac{wM_{s}}{\rho _{w} } +\frac{M_{s}}{G_{s}\rho _{w} } \right\rgroup\left(1+w\right) }=\frac{M_{s}\left\lgroup1+\frac{M_{w}}{M_{s}} \right\rgroup\left(1-A_{\nu } \right) }{\frac{M_{s} }{\rho _{w} }\left\lgroup w+\frac{1}{G_{s} } \right\rgroup \left(1+w\right) } =\frac{\rho _{w} \left(1+w\right)\left(1-A_{\nu } \right) }{\left\lgroup w+\frac{1}{G_{s} } \right\rgroup \left(1+w\right) } \)

 

\(=\frac{\rho _{w}\left(1-A_{\nu } \right) }{\frac{1}{G_{s} } \left(G_{s}w+1\right) } = \frac{G_{s}\rho _{w} \left(1-A_{\nu } \right) }{\left(G_{s} w+1\right) } \)

 

\(\rho _{d} = \frac{G_{s}\rho _{w} \left(1-A_{\nu } \right) }{\left(G_{s} w+1\right) } \)
