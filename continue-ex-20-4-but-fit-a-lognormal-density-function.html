Guest Question.

Continue Ex. 20–4, but fit a lognormal density function.

Admin
Solution

From Eqs. (20–18) and (20–19),

\(μ_{y} = ln μ_{x} − ln \sqrt{1 + C^{2}_{x}} ≈ ln μ_{x} −\frac{1}{2}C^{2}_{x}\) (20–18)
\( \hat{σ}_{y} =\sqrt{ln (1 + C^{2}_{x})} ≈ C_{x}\) (20–19)

\(μ_{y} = ln μ_{x} − ln \sqrt{1 + C^{2}_{x}} = ln 63.625 − \frac{1}{2} ln (1 + 0.040 773^{2}) = 4.1522\)
\( \hat{σ}_{y} =\sqrt{ln (1 + C^{2}_{x})}=\sqrt{ln (1 + 0.040 773^{2})} = 0.0408\)

The probability density of a lognormal distribution is given in Eq. (20–17) as

\(f (x) =\begin{cases} \frac{1}{x \hat{σ}_{y}\sqrt{2π}}exp\left[−\frac{1}{2}\left(\frac{ln x − μ_{y}}{\hat{σ}_{y}}\right)^{2}\right]&for x > 0\\0& for x ≤ 0\end{cases}\) (20–17)

\(g(x) =\frac{1}{x (0.0408)\sqrt{2π}} exp\left[−\frac{1}{2} \left(\frac{ln x − 4.1522}{0.0408}\right)^{2}\right] for x > 0\)

For example, g(63.625) = 0.1537. This lognormal density has been added to Table 20–5.
Plot the lognormal PDF superposed on the histogram of Ex. 20–4 along with the normal density. As seen in Fig. 20–8, both normal and lognormal densities fit well.
