Guest Question.

Consider the following amplitude modulated signal: S(t) = cos(2000 π t) + 4 cos(2400 π t) + cos(2800 π t). The ratio (accurate to three decimal places) of the power of the massage signal to the power of the carrier signal is _______.

Admin
Solution

amplitude modulated signal is given as

\(\begin{aligned}S(t)=& \cos (2000 \pi t)+4 \cos (2400 \pi t)+\cos (2800 \pi t) \\S_{ AM }(t)=& \frac{\mu A_{ C }}{2} \cos \left[2 \pi\left(f_{ c }-f_{ m }\right) t\right]+A_{ C } \cos \left(2 \pi f_{ C } t\right) \\&+\frac{\mu A_{ C }}{2} \cos \left[2 \pi\left(f_{ C }-f_{ m }\right) t\right] \\S(t)=& A_{ c } \cdot \cos 2 \pi f c t+\frac{A c \mu}{2} \cdot \cos \pi(f c+f m) t \\&+\frac{A c \mu}{2} \cdot \cos 2 \pi\left(f_{c}-f m\right) t \\P_{ t }=& P_{ c }+P_{ SB }\end{aligned}\)

From the given data

\(\begin{aligned}&A_{ c }=4 \text { and } \frac{A c \mu}{2}=1 \\&P_{ C }=\frac{A C}{2}=8 \\&P_{ SB }=2\left\{\frac{A c \mu}{2 \sqrt{2}}\right\}^{2}=\left(\frac{A c \mu}{2}\right)^{2}\end{aligned}\)

= (1) = 1

\(\frac{P_{ SB }}{P_{ C }}=\frac{1}{8}=0.125\)

Hence, the correct answer is 0.12 to 0.13.
