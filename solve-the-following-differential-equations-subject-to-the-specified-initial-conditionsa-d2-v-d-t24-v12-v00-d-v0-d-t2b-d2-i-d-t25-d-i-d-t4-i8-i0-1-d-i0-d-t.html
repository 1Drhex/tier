Guest Question.

Solve the following differential equations subject to the specified initial conditions
(a) \(d^{2} v / d t^{2}+4 v=12, v(0)=0, d v(0) / d t=2\)
(b) \(d^{2} i / d t^{2}+5 d i / d t+4 i=8, i(0)=-1, d i(0) / d t=0\)
(c) \(d^{2} v / d t^{2}+2 d v / d t+v=3, v(0)=5, d v(0) / d t=1\)
(d) \(d^{2} i / d t^{2}=2 d i / d t=5 i=10, i(0)=4, d i(0) / d t=-2\)

Admin
Solution

(a) \(\mathrm{s}^{2}+4=0\) which leads to \(\mathrm{s}_{1,2}=\pm \mathrm{j} 2\) (an undamped circuit)

\(\begin{array}{l}\mathrm{v}(\mathrm{t})=\mathrm{V}_{\mathrm{s}}+\mathrm{Acos} 2 \mathrm{t}+\mathrm{B} \sin 2 \mathrm{t} \\\\4 \mathrm{V}_{\mathrm{s}}=12 \text { or } \mathrm{V}_{\mathrm{s}}=3 \\\\\mathrm{v}(0)=0=3+\mathrm{A} \text { or } \mathrm{A}=-3 \\\\\mathrm{dv} / \mathrm{dt}=-2 \mathrm{A} \sin 2 \mathrm{t}+2 \mathrm{B} \cos 2 \mathrm{t}\end{array}\)

\(\mathrm{dv}(0) / \mathrm{dt}=2=2 \mathrm{B}\) or \(\mathrm{B}=1,\) therefore \(\mathrm{v}(\mathrm{t})={(3-3 \cos 2 \mathrm{t}+\sin 2 \mathrm{t}) \mathrm{V}}\)
(b) \(\mathrm{s}^{2}+5 \mathrm{s}+4=0\) which leads to \(\mathrm{s}_{1,2}=-1,-4\)

\(\begin{array}{l}\mathrm{i}(\mathrm{t})=\left(\mathrm{I}_{\mathrm{s}}+\mathrm{Ae}^{-\mathrm{t}}+\mathrm{Be}^{-4 t}\right) \\\\4 \mathrm{I}_{\mathrm{s}}=8 \text { or } \mathrm{I}_{\mathrm{s}}=2 \\\\\mathrm{i}(0)=-1=2+\mathrm{A}+\mathrm{B}, \text { or } \mathrm{A}+\mathrm{B}=-3(1) \\\\\mathrm{di} / \mathrm{dt}=-\mathrm{Ae}^{-\mathrm{t}}-4 \mathrm{Be}^{-4 \mathrm{t}} \\\\\mathrm{di}(0) / \mathrm{dt}=0=-\mathrm{A}-4 \mathrm{B}, \text { or } \mathrm{B}=-\mathrm{A} / 4(2)\end{array}\)

From (1) and (2) we get A=-4 and B=1

\(\mathrm{i}(\mathrm{t})={\left(2-4 \mathrm{e}^{-\mathrm{t}}+\mathrm{e}^{-4 \mathrm{t}}\right) \mathrm{A}}\)

(c) \(\quad \mathrm{s}^{2}+2 \mathrm{s}+1=0, \mathrm{s}_{1,2}=-1,-1\)

\(\begin{array}{l}\mathrm{v}(\mathrm{t})=\left[\mathrm{V}_{\mathrm{s}}+(\mathrm{A}+\mathrm{Bt}) \mathrm{e}^{-\mathrm{t}}\right], \mathrm{V}_{\mathrm{s}}=3 \\\\\mathrm{v}(0)=5=3+\mathrm{A} \text { or } \mathrm{A}=2 \\\\\mathrm{dv} / \mathrm{dt}=\left[-(\mathrm{A}+\mathrm{Bt}) \mathrm{e}^{-t}\right]+\left[\mathrm{Be}^{-t}\right] \\\\\mathrm{dv}(0) / \mathrm{dt}=-\mathrm{A}+\mathrm{B}=1 \text { or } \mathrm{B}=2+1=3 \\\\\mathrm{v}(\mathrm{t})={\left[3+(2+3 \mathrm{t}) \mathrm{e}^{-\mathrm{t}}\right] \mathrm{V}}\end{array}\)

(d) \(\quad \mathrm{s}^{2}+2 \mathrm{s}+5=0, \mathrm{s}_{1,2}=-1+\mathrm{j} 2,-1-\mathrm{j} 2\)
\(\mathrm{i}(\mathrm{t})=\left[\mathrm{I}_{\mathrm{s}}+(\mathrm{Acos} 2 \mathrm{t}+\mathrm{B} \sin 2 \mathrm{t}) \mathrm{e}^{-t}\right],\) where \(5 \mathrm{I}_{\mathrm{s}}=10\) or \(\mathrm{I}_{\mathrm{s}}=2\)

\(\begin{array}{l}\mathrm{i}(0)=4=2+\mathrm{A} \text { or } \mathrm{A}=2 \\\\\mathrm{di} / \mathrm{dt}=\left.[-(\mathrm{Acos} 2 \mathrm{t}+\mathrm{B} \sin 2 \mathrm{t}) \mathrm{e}^{-t}\right]+\left[(-2 \mathrm{A} \sin 2 \mathrm{t}+2 \mathrm{B} \cos 2 \mathrm{t}) \mathrm{e}^{-\mathrm{t}}\right] \\\\\operatorname{di}(0) / \mathrm{dt}=-2=-\mathrm{A}+2 \mathrm{B} \text { or } \mathrm{B}=0 \\\\\mathrm{i}(\mathrm{t})=\left[2+(2 \cos 2 \mathrm{t}) \mathrm{e}^{-\mathrm{t}}\right] \mathrm{A}\end{array}\)
