Design a welded bracket plate, using 70 ksi electrodes, to support the following loads:
\(\begin{array}{l}P_{D}=9 \text { kips } \\P_{L}=27 \text { kips }\end{array}\)

Material Properties:
Plate Material ASTM A36 \(F_{y}=36 ksi \quad F_{u}=58 ksi\)

Solution
LRFD ASD
\(R_{u}=1.2(9 \text { kips })+1.6(27 kips )=54 \text { kips }\) \(R_{a}=9 \text { kips }+27 \text { kips }=36 \text { kips }\)

\(\begin{aligned}M_{u} &=P_{u}(a l)=(54 kips )(0.603)(18 \text { in. }) \\&=586 \text { in-kips }\end{aligned}\)

 \(\begin{aligned}D_{r e q}=& \frac{P_{a} \Omega}{C C_{1} l}=& \frac{(36 kips )(2.00)}{(1.47)(1.0)(18 \text { in. })} \\=& 2.72 \rightarrow 3 \text { sixteenths } \\&<1 / 2 \text { in. }-1 / 16 \text { in. }=7 / 16 o.k.\end{aligned}\)
