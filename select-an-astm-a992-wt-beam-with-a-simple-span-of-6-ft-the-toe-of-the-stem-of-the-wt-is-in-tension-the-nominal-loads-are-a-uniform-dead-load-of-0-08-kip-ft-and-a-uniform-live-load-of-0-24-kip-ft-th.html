Guest Question.

Select an ASTM A992 WT beam with a simple span of 6 ft. The toe of the stem of the WT is in tension. The nominal loads are a uniform dead load of 0.08 kip/ft and a uniform live load of 0.24 kip/ft. There is no deflection limit for this member. Assume full lateral support.

Admin
Solution

Material properties:

ASTM A992 \(F_{y}=50 ksi \quad F_{u}=65 ksi\)

Manual
Table 2-3

Calculate the required flexural strength

LRFD ASD
\(w_{u}=1.2(0.0800 kip / ft )+1.6(0.240 kip / ft )\)

= 0.480 kip/ft

 \(w_{a}=0.0800 kip / ft +0.240 kip / ft\)

= 0.320 kip/ft
\(M_{u}=\frac{0.480 kip / ft (6.00 ft )^{2}}{8}=2.16 kip – ft\) \(M_{a}=\frac{0.320 kip / ft (6.00 ft )^{2}}{8}=1.44 kip-ft\)

Try WT 5×6

Geometric Properties:

WT 5×6 \(I_{x}=4.35 \text { in. }^{4} \quad Z_{x}=2.20 \text { in. }{ }^{3} \quad S_{x}=1.22 \text { in. }{ }^{3} \quad b_{f}=3.96 \text { in }\)

 

\(t_{f}=0.210 \text { in. } \quad \bar{y}=1.36 \text { in. } \quad S_{x c}=\frac{I_{x}}{y_{c}}=\frac{4.35 in ^{4}}{1.36 in .}=3.20 in .^{3}\)

Manual
Table 1-8

Calculate the nominal flexural strength, \(M_{n}\)

Flexural yielding limit state

\(M_{p}=F_{y} Z_{x} \leq 1.6 M_{y}\) for stems in tension

Eqn. F9-2

\(1.6 M_{y}=1.6 F_{y} S_{x}=1.6(50 ksi )\left(1.22 in .^{3}\right)=97.6 \text { kip-in. }\)

 

\(M_{p}=F_{y} Z_{x}=(50 ksi )\left(2.20 in .^{3}\right)=110 kip – in .>97.6 kip – in .\) therefore use

 

\(M_{p}=97.6 \text { kip-in. or } 8.13 \text { kip-ft }\)

 

\(M_{n}=M_{p}=8.13 \text { kip-ft }\) controls

Eqn. F9-1

Lateral-torsional buckling limit state

Section F9.2

Because the WT is fully braced and the stem is in tension, no check of the lateral-torsional buckling limit state is required. Note that if the stem is in compression, Equation F9-4 must be checked even for fully braced members, since the equation converges to the web local buckling limit state check at an unbraced length of zero. See Commentary Section F9.

Flange local buckling limit state

Check flange compactness

\(\lambda=\frac{b_{f}}{2 t}=\frac{3.96 in .}{2(0.210 in .)}=9.43\)

 

\(\lambda_{p}=0.38 \sqrt{\frac{E}{F_{ y }}}=0.38 \sqrt{\frac{29,000 ksi }{50 ksi }}=9.15<9.43\) ; therefore the flange is not compact.

Table B4.1,
Case 7

Check flange slenderness

\(\lambda_{r}=1.0 \sqrt{\frac{E}{F_{ y }}}=1.0 \sqrt{\frac{29,000 ksi }{50 ksi }}=24.1>9.43\) , therefore the flange in not slender.

Table B4.1,
Case 7

Calculate critical flange local buckling stress

For a Tee with a noncompact flange, the critical stress is:

\(F_{c r}=F_{y}\left(1.19-0.50\left(\frac{b_{f}}{2 t_{f}}\right) \sqrt{\frac{F_{y}}{E}}\right)=(50 ksi )\left(1.19-0.50(9.43) \sqrt{\frac{50 ksi }{29,000 ksi }}\right)=49.7 ksi\)

Eqn. F9-7

Calculate the nominal flexural strength

\(M_{n}=F_{c r} S_{x c}=49.7 ksi \left(3.20 in ^{3}\right)=159 kip \text { -in. or } 13.3 \text { kip-ft }\) does not control

Eqn. F9-6

Calculate the available flexural strength

LRFD ASD
\(\phi_{b}=0.90\) \(\Omega_{b}=1.67\)
\(\phi_{b} M_{n}=0.90(8.13 kip – ft )\) = 7.32 kip-ft> 2.16 kip-ft o.k. \(M_{n} / \Omega_{b}=\frac{8.13 kip – ft }{1.67}\) = 4.87 kip-ft >1.44 kip-ft o.k.
