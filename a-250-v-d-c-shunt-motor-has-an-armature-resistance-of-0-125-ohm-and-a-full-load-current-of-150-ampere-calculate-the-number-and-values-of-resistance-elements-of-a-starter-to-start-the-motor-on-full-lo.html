Guest Question.

A 250 V D C shunt motor has an armature resistance of 0.125 ohm and a full load current of 150 ampere. Calculate the number and values of resistance elements of a starter to start the motor on full load, the maximum current not to exceed 200 A.

Admin
Solution
\(\begin{array}{l}\text { As the motor is to start against its full load, hence the minimum current is its full load current i.e., }\\150 A .\end{array}\)

 

\(I_{\max }=200 A ; I_{\min }=150 A\)

 

\(\therefore \quad K=\frac{I_{\max }}{I_{\min } .}=\frac{200}{150}=\frac{4}{3}\)

 

\(\text { Now } \quad R_{1}=\frac{V}{I_{\max }}=\frac{250}{200}=1.25 ohm \text { and }R_{a}=0.125 ohm\) (given)

 

\( \text { Now } \quad\left(\frac{I_{\max }}{I_{\min }}\right)^{n}=\frac{R_{1}}{R_{a}} \quad \text{ or } \quad\left(\frac{4}{3}\right)^{n}=\frac{1.25}{0.125}=10\)

 

Taking logs on both sides, we have

 

\( n . \log \frac{4}{3}=\log 10 \text { or } n \times 0.1248=1 \text { or } n=8\)

 

\(R_{2}=\frac{R_{1}}{K}=1.25 \times \frac{3}{4}=0.938 \Omega \quad ; \quad R_{3}=\frac{R_{2}}{K}=0.938 \times \frac{3}{4}=0.703 \Omega\)

 

\(R_{4}=\frac{R_{2}}{K}=0.703 \times \frac{3}{4}=0.527 \Omega ; \quad R_{5}=\frac{R_{4}}{K}=0.527 \times \frac{3}{4}=0.395 \Omega\)

 

\(R_{6}=\frac{R_{5}}{K}=0.395 \times \frac{3}{4}=0.296 \Omega ; \quad R_{7}=\frac{R_{6}}{K}=0.222 \times \frac{3}{4}=0.222 \Omega\)

 

\(R_{8}=\frac{R_{7}}{K}=0.222 \times \frac{3}{4}=0.167 \Omega ; \quad R_{9}=\frac{R_{8}}{K}=0.167 \times \frac{3}{4}=0.125 \Omega=R_{a}\)

 

\(\text { Resistance of various steps in the order of } 1 \text { st to } 8 \text { th, }\)

 

\(r_{1}=R_{1}-R_{2}=1.25-0.931=0.312 \Omega \quad ; \quad r_{2}=R_{2}-R_{3}=0.938-0.703=0.235 \Omega\)

 

\(r_{3}=R_{3}-R_{4}=0.703-0.527=0.176 \Omega ; \quad r_{4}=R_{4}-R_{5}=0.527-0.395=0.132 \Omega\)

 

\(r_{5}=R_{5}-R_{6}=0.395-0.296=0.099 \Omega ; \quad r_{6}=R_{6}-R_{7}=0.296-0.222=0.074 \Omega\)

 

\(r_{7}=R_{7}-R_{8}=0.222-0.167=0.055 \Omega ; \quad r_{8}=R_{8}-R_{9}=0.167-0.125=9.042 \Omega\)
