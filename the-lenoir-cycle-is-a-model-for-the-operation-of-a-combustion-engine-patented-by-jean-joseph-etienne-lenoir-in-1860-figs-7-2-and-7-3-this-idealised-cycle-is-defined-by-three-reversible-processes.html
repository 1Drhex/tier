Guest Question.

The Lenoir cycle is a model for the operation of a combustion engine patented by Jean Joseph Etienne Lenoir in 1860 (Figs. 7.2 and 7.3). This idealised cycle is defined by three reversible processes :

  • \(1 \longrightarrow 2\) isochoric compression
  • \(2 \longrightarrow 3\) adiabatic expansion
  • \(3 \longrightarrow 1\) isobaric contraction

Assume that the cycle is performed on an ideal gas characterised by the coefficient c found in relation (5.62). The following values of some state variables of the gas are assumed to be known : the pressure \(p_1\), volumes \(V_1\) and \(V_3\), temperature \(T_1\) and the number of moles of gas N. Analyse this cycle by using the following instructions :

U = c N R T (5.62)

a) Draw the (p, V) and (T, S) diagrams of the cycle.
b) Determine the entropy variation \(ΔS_{12}\) of the gas during the isochoric process \(1 \longrightarrow 2\).
c) Express the temperature \(T_2\) in terms of the heat exchanged \(Q_{12}\) during the isochoric process \(1 \longrightarrow 2\).
d) Determine the pressure \(p_2\) in terms of the pressure \(p_1\), the volume \(V_1\) and the heat exchanged \(Q_{12}\).
e) Determine the pressure \(p_3\) in terms of the pressure \(p_2\) and volumes \(V_2\) and \(V_3\).
f ) Determine the work \(W_{23}\) performed during the adiabatic process \(2 \longrightarrow 3\) and the heat \(Q_{23}\) exchanged during this process.
g) Find the work \(W_{31}\) performed during the isobaric process \(3 \longrightarrow 1\) and the heat \(Q_{31}\) exchanged during this process.
h) Find the efficiency of the cycle ηL defined in conformity with relation (7.38) as,

\( η = − \frac {W}{Q^+}\). (7.38)

\( η_L = − \frac {W_{23} + W_{31}}{Q_{12}}\).

Express the efficiency \(η_L\) in terms of the temperatures \(T_1, T_2\) and \(T_3\).

Admin
Solution

a) For the isochoric process, \(V = V_1 = V_2 = const\) (Fig. 7.2). For the adiabatic process according to relation (5.90), p (V) = const/Vγ where γ > 1 and const =\( p_1 V^γ_1 = p_2 V^γ_2\) , which is a monotonously decreasing convex function of V. For the isobaric process, \(p (V) = p_3 = p_1 = const\).

\( pV^γ = const \) (5.90)

For the isochoric process, according to relation (7.20), \(T (S) = T_1 \exp ((S − S_1) /cNR)\), which is a monotonously increasing function of S (Fig. 7.3). For the adiabatic process, \( S = S_1 = S_2 = const\). For the isobaric process, according to relation (7.23), \(T (S) = T_1 \exp ((S − S_1) / (c + 1) NR)\), which is a monotonously increasing function of S.

\( \Delta S_{if} = \int_{S_i}^{S_f}{dS} = c N R \int_{T_i}^{T_f}{\frac{dT}{T} } = c N R \ln \biggl(\frac{T_f}{T_i}\biggr) \) (7.20)

\( \Delta S_{if} = \int_{S_i}^{S_f}{dS} = \int_{i}^{f}{\frac{dH}{T} } = (c+1) N R \int_{T_i}^{T_f}{\frac{dT}{T} } = (c+1) N R \ln \Bigl(\frac{T_f}{T_i}\Bigr) \) (7.23)

b) According to relation (7.20), the work performed during the isochoric process vanishes,

\( \Delta S_{if} = \int_{S_i}^{S_f}{dS} = c N R \int_{T_i}^{T_f}{\frac{dT}{T} } = c N R \ln \biggl(\frac{T_f}{T_i}\biggr) \) (7.20)

\( \Delta S_{12} = c N R \ln \Bigl(\frac{T_2}{T_1}\Bigr) = (c + 1) N R \ln \Bigl(\frac{V_1}{V_3}\Bigr) \).

c) According to relation (7.19), the heat exchanged during the isochoric process is written as,

\( Q_{if} = \Delta U_{if} = \int_{U_i}^{U_f}{dU} = c N R \int_{T_i}^{T_f}{dT} = c N R (T_f – T_i) \). (7.19)

\( Q_{12} = \Delta U_{12} = \int_{U_1}^{U_2}{dU} = c N R \int_{T_1}^{T_2}{dT} = c N R (T_2 – T_1) \).

Thus,

\( T_2 = T_1 + \frac{Q_{12}}{c N R} \).

d) Using the ideal gas equation of state (5.47), the pressure \(p_2\) is given

by,

p V = N R T (5.47)

\( p_2 = \frac{N R T_2}{V_2} = \frac{N R }{V_1} \Bigl(T_1 + \frac{Q_{12}}{c N R}\Bigr) = p_1 + \frac{Q_{12}}{c V_1} \).

e) According to relation (5.90) for an adiabatic process,

\( pV^γ = const \) (5.90)

\( p_3 = p_2 \Bigl( \frac{V_2}{V_3}\Bigr)^γ \).

f ) According to relations (7.14) and (7.13), the work performed during the adiabatic process is given by,

\(W_{if} = \Delta U_{if} = c N R \int_{T_i}^{T_f}{dT} = c N R(T_f -T_i)\) (7.14)

\( Q_{if} = \int_{i}^{f}{T dS} =0 \) (7.13)

\(W_{23} = \Delta U_{23} = c N R \int_{T_2}^{T_3}{dT} = c N R(T_3 -T_2)\)

and there is no heat exchanged,

\( Q_{23} = \int_{2}^{3}{T dS} =0 \)

g) According to relation (7.21) and (7.22) characterising the isobaric process, the work performed can be written as,

\( W_{if}= – \int_{i}^{f}{p dV } = -p \int_{V_i}^{V_f}{dV} = – p (V_f – V_i) = – N R ( T_f -T_i) \) (7.21)

\( Q_{if} = \Delta H_{if} = \int_{H_i}^{H_f}{dH} = (c+1)N R \int_{T_i}^{T_f}{dT} = (c+1) N R (T_f -T_i) \). (7.22)

\( W_{31}= – \int_{3}^{1}{p dV } = -p \int_{V_3}^{V_1}{dV} = – p (V_1 – V_3) = – N R ( T_3 -T_1) \)

and the heat exchanged is given by,

\( Q_{31} = \Delta H_{31} = \int_{H_3}^{H_1}{dH} = (c+1)N R \int_{T_3}^{T_1}{dT} = (c+1) N R (T_1 -T_3) \).

This heat transfer is not taking place during an isothermal process. Thus, it cannot be described by a heat transfer to a thermal reservoir.

h) Applying the results above to the definition of \(η_L\) and using the coefficient γ = (c + 1) /c, we can write,

\( η_L = \frac {W_{23} + W_{31}}{Q_{12}} = \frac {c(T_2 – T_3)+(T_1 -T_3)}{c(T_2 – T_1)}\)

\( = \frac {T_1+cT_2-(c+1)T_3}{c(T_2 – T_1)} = \frac {(γ − 1) T_1 + T_2 − γ T_3}{T_2 – T_1} = 1- γ \frac {T_3 − T_1}{T_2 – T_1} \).
