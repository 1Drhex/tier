Guest Question.

Use the prototype low-pass op amp filter from , along with magnitude and frequency scaling, to compute the resistor values for a lowpass filter with a gain of \( 5\) , a cutoff frequency of \( 1000 Hz\), and a feedback capacitor of \(0.01\mu F\). Construct a Bode plot of the resulting transfer function’s magnitude.

Admin
Solution

To begin, use frequency scaling to place the cutoff frequency at \( 1000 Hz\):

\(K_{f} =\omega ^{\prime }_{c} / \omega _{c} =2\pi \left(1000\right) /1=6283.185\),

where the primed variable has the new value and the unprimed variable has the old value of the cutoff frequency. Then compute the magnitude scale factor that, together with \(K_{f} =6283.185\), will scale the capacitor to \(0.01\mu F\):

\(K_{m} =\frac{1}{K_{f}} \frac{C}{C^{\prime } } =\frac{1}{\left(6283.185\right)\left(10^{-8} \right) } =15,915.5\).

Since resistors are scaled only by using magnitude scaling,

\(R^{\prime }_{1}=R^{\prime }_{2}=K_{m}R=\left(15,915.5\right) \left(1\right) =15,915.5 \Omega\).

Finally, we need to meet the passband gain specification. We can adjust the scaled values of either \(R_{1}\) or \(R_{2}\), because \(K=R_{2}/R_{1}\). If we adjust \(R_{2}\), we will change the cutoff frequency, because \(\omega _{c}=1/R_{2}C\). Therefore, we can adjust the value of \(R_{1}\) to alter only the passband gain:

\(R_{1}=R_{2}/K=\left(15,915.5\right)/\left(5\right)=3183.1 \Omega\).

The final component values are

\(R_{1}=3183.1 \Omega\), \(R_{2}=15,915.5 \Omega\), \(C=0.01\mu F\).

The transfer function of the filter is given by

\(H\left(s\right)=\frac{-31,415.93}{s+6283.185}\).
