Guest Question.

Lightning strikes a lossy dielectric sphere \(\epsilon =1.2{ \epsilon }_{ 0 }\), \(\sigma =10\left( S/m \right)\) of radius \(0.1 (m)\) at time \(t = 0\), depositing uniformly in the sphere a total charge \(l (mC).\) Determine, for all \(t\),

\(a)\) the electric field intensity both inside and outside the sphere,

\(b)\) the current density in the sphere.

\(a)\) \(R<b:\quad \overline { { E }_{ i } } =\overline { { a }_{ R } } \frac { \left( 4\pi /a \right) { R }^{ 3 }\rho }{ 4\pi \epsilon{ R }^{ 2 } } =\overline { { a }_{ R } } \frac { { \rho }_{ 0 }R }{ 3\epsilon } { e }^{ -\left( \sigma /\epsilon \right) t }=\overline { { a }_{ R } } 7.5\times { 10 }^{ 9 }R{ e }^{ -9.42\times { 10 }^{ 11 }t }\left( V/m \right)\)
