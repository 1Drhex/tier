Guest Question.

Design a single-angle connection between a W16×50 beam and a W14×90 column flange to support the following beam end reactions:
\(\begin{array}{l}R_{D}=9 \text { kips } \\R_{L}=27 \text { kips }\end{array}\)
Use ¾ in. diameter ASTM A325-N bolts to connect the supported beam to the single angle.
Use 70 ksi electrode welds to connect the single angle to the column flange.

Material Properties:
\(\begin{array}{lccc}\text { W16}\times \text {50 } & \text { ASTM A992 } & F_{y}=50 ksi & F_{u}=65 ksi \\\text { W14}\times \text {90 } & \text { ASTM A992 } & F_{y}=50 ksi & F_{u}=65 ksi \\\text { Angle } & \text { ASTM A36 } & F_{y}=36 ksi & F_{u}=58 ksi\end{array}\)

\(R_{u}=1.2(9 kips )+1.6(27 kips )=54 \text { kips }\)

 

Design single angle, bolts, and welds

 

Since the half-web dimension of a W16×50 is less than 1/4 in., the tabulated values may conservatively be used.

 

Try a four bolt single angle (L4×3×3/8).

 

\(\phi R_{n}=63.6 \text { kips }>54 \text { kips }\) o.k.

 

 

Also with a 3/16-in fillet weld size

 

\(\phi R_{n}=56.6 \text { kips }>54 \text { kips }\) o.k.

 

 

Use four-bolt single angle L4×3×3/8. The 3-in. leg will be shop welded to the column flange and the 4 in. leg will be field bolted to the beam web.

 

Check supported beam web

 

The bearing strength of the beam web per bolt is s = 3 in., 3/4 in. diameter bolts, standard holes

 

\(\begin{aligned}\phi R_{n} &=\phi r_{n} t_{w} n \\&=(87.8 kips / in .)(0.380 in .)(4 \text { bolts }) \\&=133 kips >54 kips o.k.\end{aligned}\)

 

Check support

 

Minimum support thickness for the 3/16-in. welds is 0.286 in.

 

 

\( t_{w}=0.710 \text { in }>0.286 \text { in }\) o.k.
