Analysing the effect of an exponentially switched constant perturbation
A ‘constant’ perturbation was switched on exponentially starting at t = 0. Evaluate the probability of finding a system in state 2 given that initially it was in state 1, and illustrate the role of transients.

Method The perturbation is given by eqn 6.72 and the solution is expressed by eqn 6.73. To find the probability that the system is in state 2, we need to form \(P_{2} =\left|c_{2}(t) \right| ^{2}\) for a general value of k and then to plot \(P_{2}\) against t. For example plots, set \(\lambda =k/ω_{21} \) and plot \(P_{2} /(\left|V\right|/ω_{21} )^{2} \), with \(\left|V\right| =H^{(1)}_{21} /\hbar\), for \(\lambda =0.01,0.1,\) and \(1\) , which correspond to switching rates increasing in 10-fold steps.

\(H^{(1)} (t)=\begin{cases}0 & for & t\lt 0 \\H^{(1)}(1-e^{-kt} ) & for& t\geq 0\end{cases}\) (6.72)

\(c_{f} (t)=\frac{1}{i\hbar } H^{(1)}_{fi} \int_{0}^{t}{(1-e^{-kt} )e^{iω_{fi}t }dt } =\frac{1}{i\hbar } H^{(1)}_{fi} \left\{\frac{e^{iω_{fi}t }-1}{iω_{fi} }+\frac{e^{-(k-iω_{fi} )t}-1 }{k-iω_{fi} } \right\}\) (6.73)

Admin
Solution

From eqn 6.73 with \(\lambda =k/ω_{21}\) and \(x=w_{21} t\),

\(P_{2} (t)=\frac{\left|V\right|^{2}p_{2} (t)}{ω^{2}_{21} } \)

with

\(p_{2}(t) =\frac{1}{1+\lambda ^{2} } \left\{1+2\lambda ^{2}-\left[2\lambda ^{2}\cos x+(2-e^{-\lambda x} )e^{-\lambda x}+2\lambda (1-e^{-\lambda x} ) \sin x \right] \right\} \)

This function is plotted for \(\lambda =0.01,0.1,1\) in Fig. 6.17.

Comment Notice how slow switching (λ= 0.01) generates hardly any transients, whereas rapid switching (λ = 1) is like an impulsive shock to the system, and causes the population to oscillate violently between the two states. For very rapid switching (λ >> 1), \(p_{2}\) varies as 2(1 − cos x), and so it oscillates between 0 and 4 with an average value of 2: such rapid switching is like a hammer blow.
