Guest Question.

Given a part with \(S_{u t}=151 kpsi\) and at the critical location of the part,\(S_{e}=67.5 kpsi .\)
For the loading of Fig. 6–33, estimate the number of repetitions of the stress-time block in Fig. 6–33 that can be made before failure.

Admin
Solution

From Fig. 6–18, for \(S_{u t}=151 \text { kpsi, } f=0.795\). From Eq. (6–14),

 

\(a=\frac{\left(f S_{u t}\right)^{2}}{S_{e}}\) (6–14)

 

\(a=\frac{\left(f S_{u t}\right)^{2}}{S_{e}}=\frac{[0.795(151)]^{2}}{67.5}=213.5 kpsi\)

 

From Eq. (6–15),

 

\(b=-\frac{1}{3} \log \left(\frac{f S_{u t}}{S_{e}}\right)\) (6–15)

 

\(b=-\frac{1}{3} \log \left(\frac{f S_{u t}}{S_{e}}\right)=-\frac{1}{3} \log \left[\frac{0.795(151)}{67.5}\right]=-0.0833\)

 

So,

 

\(S_{f}=213.5 N^{-0.0833} \quad N=\left(\frac{S_{f}}{213.5}\right)^{-1 / 0.0833}\) (1), (2)

 

We prepare to add two columns to the previous table. Using the Gerber fatigue criterion, Eq. (6–47), with \(S_{e}=S_{f}\), and n = 1, we can write

 

\(\text { Gerber } \frac{n \sigma_{a}}{S_{e}}+\left(\frac{n \sigma_{m}}{S_{u t}}\right)^{2}=1\) (6–47)

 

\(S_{f}=\left\{\begin{array}{ll}\frac{\sigma_{a}}{1-\left(\sigma_{m} / S_{u t}\right)^{2}} & \sigma_{m}>0 \\S_{e} & \sigma_{m} \leq 0\end{array}\right.\) (3)

 

where \(S_{f}\) is the fatigue strength associated with a completely reversed stress, \(\sigma_{\text {rev }}\), equivalent to the fluctuating stresses.

 

Cycle 1: \(r=\sigma_{a} / \sigma_{m}=70 / 10=7\), and the strength amplitude from Table 6–7, is

 

\(S_{a}=\frac{7^{2} 151^{2}}{2(67.5)}\left\{-1+\sqrt{1+\left[\frac{2(67.5)}{7(151)}\right]^{2}}\right\}=67.2 kpsi\)

 

Since \(\sigma_{a}>S_{a}\), that is, \(70>67.2\), life is reduced. From Eq. (3),

 

\(S_{f}=\frac{70}{1-(10 / 151)^{2}}=70.3 kpsi\)

 

and from Eq. (2)

 

\(N=\left(\frac{70.3}{213.5}\right)^{-1 / 0.0833}=619\left(10^{3}\right) \text { cycles }\)

 

Cycle 2: r = 10/50 = 0.2, and the strength amplitude is

 

\(S_{a}=\frac{0.2^{2} 151^{2}}{2(67.5)}\left\{-1+\sqrt{1+\left[\frac{2(67.5)}{0.2(151)}\right]^{2}}\right\}=24.2 kpsi\)

 

\(\text { Since } \sigma_{a}<S_{a} \text { , that is } 10<24.2 \text { , then } S_{f}=S_{e}\) and indefinite life follows. Thus, \(N \rightarrow \infty\)

 

\(\text { Cycle } 3: r=10 /-30=-0.333 \text { , and since } \sigma_{m}<0, S_{f}=S_{e} \text { , }\) indefinite life follows and \(N \rightarrow \infty\)

Cycle Number \(S_{f}, \text { kpsi }\) N, cycles
1 70.3 \(619\left(10^{3}\right)\)
2 67.5 \(\infty\)
3 67.5 \(\infty\)

 

From Eq. (6–58) the damage per block is

 

\(D=\sum \frac{n_{i}}{N_{i}}\) (6–58)

 

\(D=\sum \frac{n_{i}}{N_{i}}=N\left[\frac{1}{619\left(10^{3}\right)}+\frac{1}{\infty}+\frac{1}{\infty}\right]=\frac{N}{619\left(10^{3}\right)}\)

 

Setting D = 1 yields \(N=619\left(10^{3}\right)\) cycles.
