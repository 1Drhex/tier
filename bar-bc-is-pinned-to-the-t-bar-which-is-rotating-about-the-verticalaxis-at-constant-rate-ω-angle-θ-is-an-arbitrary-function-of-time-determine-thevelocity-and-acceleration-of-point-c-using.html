Guest Question.

Bar BC is pinned to the T-bar, which is rotating about the verticalaxis at constant rate Ω. Angle θ is an arbitrary function of time. Determine thevelocity and acceleration of point C using the following alternative approaches: (a)attach the xyz reference frame to the T-bar; (b) attach the xyz reference frame to bar BC.

Admin
Solution

By employing two different approaches, this example provides insight into the decisions one must make. In the first approach, attaching \(x y z\) to the \(\mathrm{T}\) bar means that the sole rotation of \(x y z\) is \(\Omega\) about the fixed vertical axis. Thus we define the \(Z\) axis such that \(\bar{e}_{1}=\bar{K}\). Because \(\Omega\) is specified to be constant, the general descriptions of angular motion are

\(\bar{\omega}=\Omega \bar{e}_{1}=\Omega \bar{K}, \quad \bar{\Omega}_{1}=\overline{0}, \quad \bar{\alpha}=\bar{\Omega}_{1} \times \bar{K}=\overline{0}\)

Any point on the vertical axis of rotation is stationary. Selecting the origin \(O^{\prime}\) to be point \(A\) simplifies the description of \(\bar{r}_{P / O^{\prime}}\) and gives\(\bar{v}_{O^{\prime}}=\bar{a}_{O^{\prime}}=\overline{0}\). We orient \(x y z\) consistently with the way in which the configuration of the system is specified and show our choice in a line sketch.

To analyze the relative motion we visualize bringing \(x y z\) to rest by setting \(\Omega=0\). Bar \(B C\) would still move in the vertical plane as it pivots about pin \(B .\) Thus end \(C\) moves in a circular path at angular speed \(\dot{\theta}\) relative to \(x y z .\) Polar coordinates having origin at point \(B\) are suitable for describing this relative motion, with the radial distance \(R=L_{2}\) and \(\theta\) being the polar angle. The associated unit vectors are shown in the sketch. The corresponding descriptions of the relative velocity and acceleration are

\(\left(\bar{v}_{C}\right)_{x y z}=L_{2} \dot{\theta} \bar{e}_{\theta}, \quad\left(\bar{a}_{C}\right)_{x y z}=-L_{2} \dot{\theta}^{2} \bar{e}_{R}+L_{2} \ddot{\theta} \bar{e}_{\theta}\)

The axes of \(x y z\) are convenient directions for representing vectors, so we use them as the global directions. (An alternative definition would align the \(\hat{x}\) axis with bar \(B C\), and place the \(\hat{z}\) axis in the vertical plane. This would simplify the description of \(\bar{e}_{R}\) and \(\bar{e}_{\theta}\), but slightly complicate \(\bar{K} .\) ) The global descriptions of the unit vectors are

\(\bar{K}=-\bar{k}, \quad \bar{e}_{R}=\cos \theta \bar{i}-\sin \theta \bar{k}, \quad \bar{e}_{\theta}=-\sin \theta \bar{i}-\cos \theta \bar{k}\)

The corresponding relative position is

\(\bar{r}_{C / O^{\prime}}=\left(L_{1}+L_{2} \cos \theta\right) \bar{i}-L_{2} \sin \theta \bar{k}\)

We may now assemble the individual terms. For velocity we have

\(\bar{v}_{C}=\left(\bar{v}_{C}\right)_{x y z}+\bar{\omega} \times \bar{r}_{C / O^{\prime}}\)

\(\begin{aligned} \\=L_{2} \dot{\theta}(-\sin \theta \bar{i}-\cos \theta \bar{k})+(-\Omega \bar{k}) \times\left[\left(L_{1}+L_{2} \cos \theta\right) \bar{i}-L_{2} \sin \theta \bar{k}\right] ,\\\\\bar{v}_{C}=-L_{2} \dot{\theta} \sin \theta \bar{i}-\left(L_{1}+L_{2} \cos \theta\right) \Omega \bar{j}-L_{2} \dot{\theta} \cos \theta \bar{k}\end{aligned}\)

A shortcut for the evaluation of \(\bar{\omega} \times\left(\bar{\omega} \times \bar{r}_{C / O^{\prime}}\right)\) is to retain \(\bar{\omega} \times \bar{r}_{C / O^{\prime}}\) from the velocity analysis. Thus we have

\(\begin{aligned}\bar{a}_{C}=&\left(\bar{a}_{C}\right)_{x y z}+\bar{\omega} \times\left(\bar{\omega} \times \bar{r}_{C / O^{\prime}}\right)+2 \bar{\omega} \times\left(\bar{v}_{C}\right)_{x y z} \\\\=&-L_{2} \dot{\theta}^{2}(\cos \theta \bar{i}-\sin \theta \bar{k})+L \ddot{\theta}(-\sin \theta \bar{i}-\cos \theta \bar{k})+(-\Omega \bar{k}) \\\\& \times\left[-\left(L_{1}+L_{2} \cos \theta\right) \Omega \bar{j}\right]+2(-\Omega \bar{k}) \times\left[L_{2} \dot{\theta}(-\sin \theta \bar{i}-\cos \theta \bar{k})\right] \\\\=&\left[-L_{2} \dot{\theta}^{2} \cos \theta-L \ddot{\theta} \sin \theta-\left(L_{1}+L_{2} \cos \theta\right) \Omega^{2}\right] \bar{i}+2 L \Omega \dot{\theta} \sin \theta \bar{j} \\\\&+\left(L_{2} \dot{\theta}^{2} \sin \theta-L \ddot{\theta} \cos \theta\right) \bar{k} .\end{aligned}\)

Obviously, changing the selection of the moving reference frame should not alter the result. To avoid confusion with the preceding analysis, let \(x^{\prime} y^{\prime} z^{\prime}\) denote the moving reference frame that is attached to bar \(B C\). The angular velocity of this bar is the sum of \(\Omega\) about the vertical axis and \(\theta\) about an axis that is perpendicular to the plane of the T-bar. Both simple rotations are shown in a new sketch.

As was done in the previous analysis, fixed \(X Y Z\) can be used to describe \(\bar{e}_{1} .\) To describe the \(\dot{\theta}\) rotation we attach \(x_{2} y_{2} z_{2}\) to the T-bar, with the \(y_{2}\) axis outward from the plane of the sketch, so that \(\bar{e}_{2}=\bar{j}_{2}\). The angular velocity \(\bar{\Omega}_{2}\) then consists solely of the rotation about the vertical axis. Thus

\(\bar{\omega}=\Omega \bar{e}_{1}+\dot{\theta} \bar{e}_{2}=\Omega \bar{K}+\dot{\theta} \bar{j}_{2}, \quad \bar{\Omega}_{1}=\overline{0}, \quad \bar{\Omega}_{2}=\Omega \bar{K}\)

The angular acceleration corresponding to constant \(\Omega\) and variable \(\dot{\theta}\) is

\(\bar{\alpha}=\ddot{\theta} \bar{j}_{2}+\dot{\theta}\left(\bar{\Omega}_{2} \times \bar{j}_{2}\right)=\ddot{\theta} \bar{j}_{2}+\dot{\theta} \Omega\left(\bar{K} \times \bar{j}_{2}\right)\)

The guideline for the selection of the origin \(O^{\prime}\) requires that it be a point in bar \(B C\), which is the body to which \(x^{\prime} y^{\prime} z^{\prime}\) is attached. The only such point executing a simple motion is end \(B\), which follows a circular path in the horizontal plane at constant angular speed \(\Omega .\) We may describe this motion in cylindrical coordinates whose axis is \(z_{2}\), with the \(x_{2}\) axis parallel to arm \(A B\), which is the \(\bar{e}_{R}\) direction. Then the \(y_{2}\) axis is opposite to \(\bar{e}_{\theta}\). Correspondingly, the motion of the origin is given by

\(\bar{v}_{O^{\prime}}=-L_{1} \Omega \bar{j}_{2}, \quad \bar{a}_{O^{\prime}}=-L_{1} \Omega^{2} \bar{i}_{2}\)

Because \(x y z\) has been defined to be attached to bar \(B C\), point \(C\) remains fixed from the viewpoint of this reference frame. Thus,

\(\left(\bar{v}_{C}\right)_{x y z}=\left(\bar{a}_{C}\right)_{x y z}=\overline{0}\)

Orienting \(x^{\prime} y^{\prime} z^{\prime}\) as depicted in the sketch facilitates the description of relative position. We correspondingly find that

\(\bar{K}=\sin \theta \bar{i}^{\prime}-\cos \theta \bar{k}^{\prime}, \quad \bar{j}_{2}=\bar{j}^{\prime}, \quad \bar{i}_{2}=\cos \theta \bar{i}^{\prime}+\sin \theta \bar{k}^{\prime}, \quad \bar{r}_{C / O^{\prime}}=L_{2} \bar{i}^{\prime}\)

The global descriptions of the angular motion variables are

\(\begin{array}{c}\bar{\omega}=\Omega\left(\sin \theta \bar{i}^{\prime}-\cos \theta \bar{k}^{\prime}\right)+\dot{\theta} \bar{j}^{\prime} \\\\\bar{\alpha}=\ddot{\theta} \bar{j}^{\prime}+\dot{\theta} \Omega\left[\left(\sin \theta \bar{i}^{\prime}-\cos \theta \bar{k}^{\prime}\right) \times \bar{j}^{\prime}\right]=\ddot{\theta} \bar{j}^{\prime}+\dot{\theta} \Omega\left(\cos \theta \bar{i}+\sin \theta \bar{k}^{\prime}\right)\end{array}\)

The nonzero terms in the general velocity equation are

\(\begin{aligned}\bar{v}_{C} &=\bar{v}_{O^{\prime}}+\bar{\omega} \times \bar{r}_{C / O^{\prime}} \\\\&=-L_{1} \Omega \bar{j}^{\prime}+\left[\Omega\left(\sin \theta \bar{i}^{\prime}-\cos \theta \bar{k}^{\prime}\right)+\dot{\theta} \bar{j}^{\prime}\right] \times L_{2} \bar{i}^{\prime} \\\\&=-\left(L_{1}+L_{2} \cos \theta\right) \Omega \bar{j}^{\prime}-L_{2} \dot{\theta} \bar{k}^{\prime}\end{aligned}\)

The nonzero terms in the general acceleration equation are

\(\begin{aligned}\bar{a}_{C}=& \bar{a}_{O^{\prime}}+\bar{\alpha} \times \bar{r}_{C / O^{\prime}}+\bar{\omega} \times\left(\bar{\omega} \times \bar{r}_{C / O^{\prime}}\right) \\\\=&-L_{1} \Omega^{2}\left(\cos \theta \bar{i}^{\prime}+\sin \theta \bar{k}^{\prime}\right)+\left[\ddot{\theta} \bar{j}^{\prime}+\dot{\theta} \Omega\left(\cos \theta \bar{i}+\sin \theta \bar{k}^{\prime}\right)\right] \times L_{2} \bar{i}^{\prime} \\\\&+\left[\Omega\left(\sin \theta \bar{i}^{\prime}-\cos \theta \bar{k}^{\prime}\right)+\dot{\theta} \bar{j}^{\prime}\right] \times\left(-L_{2} \Omega \cos \theta \bar{j}^{\prime}-L_{2} \dot{\theta} \bar{k}^{\prime}\right) \\\\=&\left[-\left(L_{1}+L_{2} \cos \theta\right) \Omega^{2} \cos \theta-L_{2} \dot{\theta}^{2}\right] \bar{i}^{\prime}+2 L_{2} \Omega \dot{\theta} \sin \theta \bar{j}^{\prime} \\\\&+\left[-\left(L_{1}+L_{2} \cos \theta\right) \Omega^{2} \sin \theta-L_{2} \ddot{\theta}\right] \bar{k}^{\prime}\end{aligned}\)

Different global coordinate systems were used to represent each set of results, so the components of \(\bar{v}_{C}\) and \(\bar{a}_{C}\) are not identical. One check that the vectors are consistent lies in the fact that the \(y^{\prime}\) axis for the second analysis coincides with the \(\hat{y}\) axis for the first. Correspondingly, we see that those velocity and acceleration components match. Another way to verify that both sets of results represent the same vectors is to apply the rotation transformation between \(x^{\prime} y^{\prime} z^{\prime}\) and \(\hat{x} \hat{y} \hat{z}\) to transform one set of components to the other. A third approach is to sketch the vector resultant of similar terms. For example, the \(\left(L_{1}+L_{2} \cos \theta\right) \Omega^{2}\) terms obtained in either analysis represent a centripetal acceleration that is perpendicular to the vertical axis directed from point \(B\) to point \(A\).
