Guest Question.

A shunt generator supplies 195 A at 220 V. Armature resistance is 0.02 ohm, shunt field resistance is 44 ohm. If the iron and friction losses amount to 1600 watt, find (i) emf generated; (ii) copper losses; (iii) b.h.p. of the engine driving the generator. (iv) commercial, mechanical and electrical efficiency.

Admin
Solution
\(\text { The conventional circuit is shown in Fig. } 5.54 \text { . }\)

 

\(\text { Shunt field current, } \quad I_{s h}=\frac{V}{R_{s h}}=\frac{220}{44}=5 A\)

 

\(\text { Armature current, } \quad I_{a}=I_{L}+I_{s h}=195+5=200 A\)

 

\(\text { (i) Generated or induced emf, } \quad E=V+I_{a} R_{a}\)

 

\(\begin{array}{l}=220+200 \times 0.02 \\=224 V \end{array}\)

 

\(\begin{array}{l}\\\begin{array}{l}\text { Armature copper loss }=i_{a}^{2} R_{a}=(200)^{2} \times 0 \cdot 02=800 W \\\text { Shunt field copper loss }=I_{s h}^{2} R_{S h}=(5)^{2} \times 44=1100 W \\\quad \text { (ii) } \text { Total copper losses }=800+1100=1900 W \end{array}\\\text { Output power }=V I_{L}=220 \times 195=42900 W\\\text { Input power }=42900+1600+1900=46400 W\end{array}\)

 

\(\text { (iii) B.H.P. of the engine driving the generator }=\frac{46400}{735 \cdot 5}=63 \cdot 08 H . P \text { . }\)

 

\(\text { (iv) Commercial efficiency, } \eta=\frac{\text { final output of machine }}{\text { input to machine }}\)

 

\(=\frac{42900}{46400} \times 100=92.45 \%\)

 

\(\text { Mechanical efficiency, } \quad \eta_{m}=\frac{\text { Power developed in armature }}{\text { Power input }}\)

 

\(=\frac{E_{g} I_{a}}{46400} \times 100=\frac{224 \times 200}{46400} \times 100=96.55 \% \)

 

\(\text { Electrical efficiency, } \eta_{e}=\frac{\text { Electrical Power output }}{\text { Power generated in armature }}\)

 

\( =\frac{V I_{L}}{E_{g} I_{a}} \times 100=\frac{220 \times 195}{224 \times 200} \times 100=95.76 \% \)

 

Alternatively, commercial efficiency,

 

\( \eta =\eta_{m} \times \eta_{e} \)

 

\( =\frac{96.55}{100} \times \frac{95.76}{100} \times 100=92.45 \%(\text { True as above })\)
