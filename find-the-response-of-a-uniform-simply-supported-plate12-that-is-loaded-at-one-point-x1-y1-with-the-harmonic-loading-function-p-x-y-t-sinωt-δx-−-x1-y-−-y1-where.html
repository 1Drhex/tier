Guest Question.

Find the response of a uniform, simply supported plate12 that is loaded at one point (\(x_{1}\), \(y_{1}\)) with the harmonic loading function

 

\(P(x, y, t)=\sin \Omega t \delta\left(x-x_{1}, y-y_{1}\right)\)

 

where the δ is the Dirac delta function, defined as

 

\(\begin{aligned}&\int_{-\infty}^{\infty} \int_{-\infty}^{\infty} \delta\left(x-x_{1}, y-y_{1}\right) d x d y \\&= \begin{cases}1 & \text { if } x=x_{1} \text { and } y=y_{1} \\0 & \text { if } x \neq x_{1} \text { and } / \text { or } y \neq y_{1} .\end{cases}\end{aligned}\)
Admin
Solution

From the previous formulation, the normal modes \(\hat{W}_{j k}(x, y)\) and frequencies \(\omega_{j k}\) are known. Assuming zero initial conditions, the response is given by Equation 8.41, where

 

\(w(x, y, t)=\sum_{j=1}^{\infty} \sum_{k=1}^{\infty} \hat{W}_{j k}(x, y) F_{j k}(t)\)

 

\(\begin{aligned}&F_{j k}(t) \\&=\int_{0}^{t}\left[\int_{0}^{a} \int_{0}^{b} P(\zeta, \eta, \tau) \hat{W}_{j k}(\zeta, \eta) d \zeta d \eta\right] g_{j k}(t-\tau) d \tau\end{aligned}\)

 

When substituting the two-dimensional Kronecker delta function for the force \(P\) (\(x\), \(y\), \(t\)), the double integral is reduced to a single value at \(x\) = \(x_{1}\) and \(y\) = \(y_{1}\), with all other values equal to zero,

 

\(\begin{aligned}F_{j k}(t) &=\int_{0}^{t} \hat{W}_{j k}\left(x_{1}, y_{1}\right) g_{j k}(t-\tau) d \tau \\&=\int_{0}^{t} \frac{2}{\sqrt{\rho h a b}} \sin \Omega \tau \sin \frac{j \pi x_{1}}{a} \sin \frac{k \pi y_{1}}{b} g_{j k}(t-\tau) d \tau .\end{aligned}\)

 

The function\(g_{j k}(t)\) is the impulse response function, which for an undamped oscillator is

 

\(g_{j k}(t)=\frac{1}{\omega_{j k}} \sin \omega_{j k} t, \quad t \geq 0\)

 

The complete solution is then

 

\(\begin{aligned}w(x, y, t) &=\frac{4}{\rho h a b} \sum_{j=1}^{\infty} \sum_{k=1}^{\infty} \frac{1}{\omega_{j k}} \sin \frac{j \pi x_{1}}{a} \sin \frac{k \pi y_{1}}{b} \\& \times \sin \frac{j \pi x}{a} \sin \frac{k \pi y}{b} \int_{0}^{t} \sin \Omega \tau \sin \omega_{j k}(t-\tau) d \tau .\end{aligned}\)

 

Note that for certain values of \(x_{1}\) and \(y_{1}\) there are modes that cannot be excited by the given force. These are the modes where \(j x_{1} / a\) and \(k y_{1} / b\) are integers and, therefore, sine equals zero. If the loading function is more complicated, a more intricate algebraic procedure is needed to evaluate \(F_{j k}(t)\).
