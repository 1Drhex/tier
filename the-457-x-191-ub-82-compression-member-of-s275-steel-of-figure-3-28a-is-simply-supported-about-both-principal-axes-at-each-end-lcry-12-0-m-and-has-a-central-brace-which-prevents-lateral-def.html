Guest Question.

The 457 × 191 UB 82 compression member of S275 steel of Figure 3.28a is simply supported about both principal axes at each end \((L_{cr,y} = 12.0 m)\), and has a central brace which prevents lateral deflections in the minor principal plane \((L_{cr,z} = 6.0 m)\). Check the adequacy of the member for a factored axial compressive load corresponding to a nominal dead load of 160 kN and a nominal imposed load of 230 kN.

Admin
Solution

Factored axial load. \(N_{Ed} = (1.35 × 160) + (1.5 × 230) = 561 kN\)
Classifying the section.
For S275 steel with \(t_{f} = 16 mm, f_{y} = 275 N/mm^2\)

\(\varepsilon = (235/275)^{0.5} = 0.924\)

 

\(c_{f} /(t_{f}\varepsilon ) = [(191.3 − 9.9 − 2 × 10.2)/2]/(16.0 × 0.924) = 5.44 < 14\)

 

\(c_{w} = (460.0 − 2 × 16.0 − 2 × 10.2) = 407.6 mm\)

 

\(c_{w} /(t_{w}\varepsilon ) = 407.6/(9.9 × 0.924) = 44.5 > 42\)
and so the web is Class 4 (slender).
Effective area.

\(\bar{\lambda }_{p} =\sqrt{\frac{f_{y} }{\sigma _{cr} } } =\frac{\bar{b}/t}{28.4\varepsilon \sqrt{k_{\sigma } } } =\frac{407.6/9.9}{28.4 × 0.924 ×\sqrt{4.0} } =0.784\)

 

\(p=\frac{\bar{\lambda } p − 0.055(3 + ψ)}{\bar{\lambda }^2_{p} }=\frac{0.784 − 0.055(3 + 1)}{0.784^2}=0.918\)

 

\(d − d_{eff} = (1 − 0.918) × 407.6 = 33.6 mm\)

 

\(A_{eff} = 104 × 10^2 − 33.6 × 9.9 = 10 067 mm^2\)
Cross-section compression resistance.

\(N_{c,Rd}=\frac{A_{eff} f_{y} }{\gamma _{M0} } =\frac{10 067 × 275}{1.0}=2768 kN > 561 kN = N_{Ed}\)
Member buckling resistance.

\(\bar{\lambda } _{y} =\sqrt{\frac{A_{eff} f_{y} }{N_{cr,y} } } =\frac{L_{cr,y} }{i_{y} } \frac{\sqrt{A_{eff}/A } }{\lambda _{1} } =\frac{12 000}{(18.8 × 10)}\frac{\sqrt{10 067/10 400} }{93.9 × 0.924} =0.724\)

 

\(\bar{\lambda } _{z} =\sqrt{\frac{A_{eff} f_{y} }{N_{cr,z} } } =\frac{L_{cr,z} }{i_{z} } \frac{\sqrt{A_{eff}/A } }{\lambda _{1} } =\frac{6 000}{(4.23 × 10)}\frac{\sqrt{10 067/10 400} }{93.9 × 0.924} = 1.608 > 0.724\)

Buckling will occur about the minor (z) axis. For a rolled UB section (with h/b > 1.2 and \(t_{f} \)≤ 40 mm), buckling about the z-axis, use buckling curve (b) with α=0.34

\(Φ_{z} = 0.5[1 + 0.34(1.608 − 0.2) + 1.608^2] = 2.032\)

 

\(\chi _{z}=\frac{1}{2.032 +\sqrt{2.032^2 − 1.608^2} } =0.305\)

 

\(N_{b,z,Rd}=\frac{\chi A_{eff}f_{y} }{\gamma_{M1} } =\frac{0.305 × 10 067 × 275}{1.0}=844 kN > 561 kN = N_{Ed}\)
and so the member is satisfactory.
