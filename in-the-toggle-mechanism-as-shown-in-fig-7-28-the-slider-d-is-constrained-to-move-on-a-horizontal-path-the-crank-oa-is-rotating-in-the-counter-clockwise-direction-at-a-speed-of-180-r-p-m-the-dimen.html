Guest Question.

In the toggle mechanism, as shown in Fig. 7.28, the slider D is constrained to move on a horizontal path. The crank OA is rotating in the counter-clockwise direction at a speed of 180 r.p.m
The dimensions of various links are as follows :
OA = 180 mm ; CB = 240 mm ; AB = 360 mm ; and BD = 540 mm.
For the given configuration, find : 1. Velocity of slider D, 2. Angular velocity of links AB, CB and BD; 3. Velocities of rubbing on the pins of diameter 30 mm at A and D, and 4. Torque applied to the crank OA, for a force of 2 kN at D.

Admin
Solution

Given : \(N_{ AO }=180 r.p.m . \text { or } \omega_{ AO }=2 \pi \times 180 / 60=18.85 rad / s\)

Since the crank length OA = 180 mm = 0.18 m, therefore velocity of A with respect to O or velocity of A (because O is a fixed point),

\(v_{ AO }=v_{ A }=\omega_{ AO } \times O A=18.85 \times 0.18=3.4 m / s\) …(Perpendicular to OA)

 

1. Velocity of slider D

First of all draw the space diagram, to some suitable scale, as shown in Fig. (a). Now the velocity diagram, as shown in Fig.(b), is drawn as discussed below :

1. Draw vector oa perpendicular to OA, to some suitable scale, to represent the velocity of A with respect to O or velocity of A (i.e.\(v_{ AO } \text { or } v_{ A }\)),such that

\(\text { vector } o a=v_{ AO }=v_{ A }=3.4 m / s\)

2. Since point B moves with respect to A and also with respect to C, therefore draw vector ab perpendicular to A B to represent the velocity of B with respect to A i.e. \(v_{ BA }\), and draw vector cb perpendicular to CB to represent the velocity of B with respect to C, i.e. \(v_{ BC }\). The vectors ab and cb intersect at b.

3. From point b, draw vector bd perpendicular to BD to represent the velocity of D with respect to B i.e. \(v_{ DB }\), and from point c draw vector cd parallel to the path of motion of the slider D (which is along CD) to represent the velocity of D, i.e. \(v_{ D }\). The vectors bd and cd intersect at d.

By measurement, we find that velocity of the slider D,

\(v_{ D }=\text { vector } c d=2.05 m / s\)

 

2. Angular velocities of links AB, CB and BD

By measurement from velocity diagram, we find that

Velocity of B with respect to A,

\(v_{ BA }=\text { vector } a b=0.9 m / s\)
Velocity of B with respect to C,

\(v_{ BC }=v_{ B }=\text { vector } c b=2.8 m / s\)

and velocity of D with respect to B,

\(v_{ DB }=\text { vector } b d=2.4 m / s\)

We know that A B = 360 mm = 0.36 m ; CB = 240 mm = 0.24 m and BD = 540 mm = 0.54 m.

\(\therefore \) Angular velocity of the link A B,

\(\omega_{ AB }=\frac{v_{ BA }}{A B}=\frac{0.9}{0.36}=2.5 rad / s (\text { Anticlockwise about } A)\)

Similarly angular velocity of the link CB,

\(\omega_{ CB }=\frac{v_{ BC }}{C B}=\frac{2.8}{0.24}=11.67 rad / s \text { (Anticlockwise about } \left.C\right)\)

and angular velocity of the link BD,

\(\omega_{ BD }=\frac{v_{ DB }}{B D}=\frac{2.4}{0.54}=4.44 rad / s (\text { Clockwise about } B)\)

 

3. Velocities of rubbing on the pins A and D

Given : Diameter of pins at A and D,

\(D_{ A }=D_{ D }=30 mm =0.03 m\)

\(\therefore \) \(\text { Radius, } r_{ A }=r_{ D }=0.015 m\)

We know that relative angular velocity at A

\(=\omega_{ BC }-\omega_{ BA }+\omega_{ DB }=11.67-2.5+4.44=13.61 rad / s\)

and relative angular velocity at D

\(=\omega_{ DB }=4.44 rad / s\)

\(\therefore \) Velocity of rubbing on the pin A

= 13.61 × 0.015 = 0.204 m/s = 204 mm/s

and velocity of rubbing on the pin D

= 4.44 × 0.015 = 0.067 m/s = 67 mm/s

 

4. Torque applied to the crank OA

Let \(T_{ A }\) = Torque applied to the crank OA, in N-m

\(\therefore \) Power input or work supplied at A

\(=T_{ A } \times \omega_{ AO }=T_{ A } \times 18.85=18.85 T_{ A } N – m\)

We know that force at D,

\(F_{D}=2 kN =2000 N\) …(Given)

\(\therefore \) Power output or work done by D,

\(=F_{ D } \times v_{ D }=2000 \times 2.05=4100 N – m\)

Assuming 100 per cent efficiency, power input is equal to power output.

\(\therefore \) \(18.85 T_{ A }=4100 \text { or } T_{ A }=217.5 N – m\)
