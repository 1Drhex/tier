Guest Question.

Suppose

\( \Psi(x, 0)=\frac{A}{x^{2}+a^{2}}, \quad(-\infty<x<\infty) \)

for constants A and a.

(a) Determine A, by normalizing Ψ(x,0).

(b) Find \( \langle x\rangle,\left\langle x^{2}\right\rangle, \text { and } \sigma_{x} \) (at time t = 0) .

(c) Find the momentum space wave function Φ (p,0) ,and check that it is normalized.

(d) Use Φ (p,0) to calculate \( \langle p\rangle,\left\langle p^{2}\right\rangle, \text { and } \sigma_{p} \) (at time t = 0).

(e) Check the Heisenberg uncertainty principle for this state.

Admin
Solution

(a)

\( 1=|A|^{2} \int_{-\infty}^{\infty} \frac{1}{\left(x^{2}+a^{2}\right)^{2}} d x=2|A|^{2} \int_{0}^{\infty} \frac{1}{\left(x^{2}+a^{2}\right)^{2}} d x=\left.2|A|^{2} \frac{1}{2 a^{2}}\left[\frac{x}{x^{2}+a^{2}}+\frac{1}{a} \tan ^{-1}\left(\frac{x}{a}\right)\right]\right|_{0} ^{\infty} \)

\( =\frac{1}{a^{2}}|A|^{2} \frac{1}{a} \tan ^{-1}(\infty)=\frac{\pi}{2 a^{3}}|A|^{2} \quad \Rightarrow A=a \sqrt{\frac{2 a}{\pi}} \).

(b)

\( \langle x\rangle=A^{2} \int_{-\infty}^{\infty} \frac{x}{\left(a^{2}+x^{2}\right)^{2}} d x=0 \).

\( \left\langle x^{2}\right\rangle=2 A^{2} \int_{0}^{\infty} \frac{x^{2}}{\left(a^{2}+x^{2}\right)^{2}} d x \). [ \( \text { Let } y \equiv \frac{x^{2}}{a^{2}}, x=a \sqrt{y}, d x=\frac{a}{2 \sqrt{y}} d y \)].

\( =\frac{2 a^{2}}{\pi} \int_{0}^{\infty} \frac{y^{1 / 2}}{(1+y)^{2}} d y=\frac{2 a^{2}}{\pi} \frac{\Gamma(3 / 2) \Gamma(1 / 2)}{\Gamma(2)}=\frac{2 a^{2}}{\pi} \frac{(\sqrt{\pi} / 2)(\sqrt{\pi})}{1} = a^2 \).

\( \sigma_{x}=\sqrt{\left\langle x^{2}\right\rangle-\langle x\rangle^{2}}= a \).

(c)

\( \Phi(p, 0)=\frac{A}{\sqrt{2 \pi \hbar}} \int_{-\infty}^{\infty} e^{-i p x / \hbar} \frac{1}{x^{2}+a^{2}} d x \) . [\( \operatorname{But} e^{-i p x / \hbar}=\cos \left(\frac{p x}{\hbar}\right)-i \sin \left(\frac{p x}{\hbar}\right) \), and sine is odd.]

\( =\frac{2 A}{\sqrt{2 \pi \hbar}} \int_{0}^{\infty} \frac{\cos (p x / \hbar)}{x^{2}+a^{2}} d x=\frac{2 A}{\sqrt{2 \pi \hbar}}\left(\frac{\pi}{2 a} e^{-|p| a / \hbar}\right) = \sqrt{\frac{a}{\hbar}} e^{-|p| a / \hbar} \).

\( \int_{-\infty}^{\infty}|\Phi(p, 0)|^{2} d p=\frac{a}{\hbar} \int_{-\infty}^{\infty} e^{-2|p| a / \hbar} d p=\left.\frac{2 a}{\hbar}\left(\frac{e^{-2 p a / \hbar}}{-2 a / \hbar}\right)\right|_{0} ^{\infty}=1 \).

(d)

\( \langle p\rangle=\frac{a}{\hbar} \int_{-\infty}^{\infty} p e^{-2|p| a / \hbar} d p = 0 \).

\( \left\langle p^{2}\right\rangle=2 \frac{a}{\hbar} \int_{0}^{\infty} p^{2} e^{-2 p a / \hbar} d p=\frac{2 a}{\hbar} 2\left(\frac{\hbar}{2 a}\right)^{3} = frac{\hbar^{2}}{2 a^{2}} \). \( \sigma_{p}=\sqrt{\left\langle p^{2}\right\rangle-\langle p\rangle^{2}}=\frac{\hbar}{\sqrt{2} a} \).

(e) \( \sigma_{x} \sigma_{p}=a \frac{\hbar}{\sqrt{2} a}=\sqrt{2} \frac{\hbar}{2}>\frac{\hbar}{2} \).
