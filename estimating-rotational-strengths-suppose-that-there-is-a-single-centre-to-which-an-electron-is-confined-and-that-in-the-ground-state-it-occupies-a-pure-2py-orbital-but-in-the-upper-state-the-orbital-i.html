Estimating rotational strengths
Suppose that there is a single centre to which an electron is confined, and that in the ground state it occupies a pure \(2p_{y}\)-orbital but in the upper state the orbital is a mixture of the form \(\left|1\right\rangle =\left|2p_{x} \right\rangle \cos \zeta +\left|3d_{yz} \right\rangle \sin \zeta \), where \(\zeta \) is a parameter (we encountered this parametrization of normalized two-component superpositions in Section 6.1) and the orbitals are Slater orbitals (Section 7.14). Evaluate the rotational strength of the transition as a function of \(\zeta \).

Method The expression for the rotational strength is given in eqn 12.101. In this model, only the z-components contribute. For the matrix elements of \(m_{z} =\gamma _{e} l_{z} \) we use \(l_{z} \propto \partial /\partial \varphi \) and recognize that \(p_{x} \propto \cos \varphi \) and \(p_{y} \propto \sin \varphi \). For the matrix elements of \(\mu _{z} =-ez\), write \(z=r \cos \theta \) and use the form of the STOs specified in Section 7.14.

\(R_{n0} =im\mu _{0n} \cdot m_{n0} \) (12.101)

Admin
Solution

For the matrix elements of \(l_{z}\) we use

\(l_{z} \psi _{2p_{y} } =\frac{\hslash }{i} \frac{\partial}{\partial \varphi } ƒ(r)\sin\theta \sin\varphi =-i\hbar ƒ(r)\sin \theta \cos\varphi =-i\hbar \psi _{2p_{x} } \)

or \(l_{z} \left|2p_{y} \right\rangle =-i\hbar \left|2p_{x} \right\rangle\). From this relation and denoting \( \left|2p_{y} \right\rangle\) as \( \left|0 \right\rangle\), it follows that

\(\left\langle1\left|m_{z} \right|0 \right\rangle =\gamma _{e} \left\{\left\langle2p_{x}\left|l_{z} \right|2p_{y} \right\rangle \cos \zeta + \left\langle3d_{yz}\left|l_{z} \right|2p_{y} \right\rangle \sin \zeta\right\} \)

 

\( =\gamma _{e}(-i\hbar )\left\langle2p_{x} \mid 2p_{x}\right\rangle \cos\zeta =i\mu _{B} \cos\zeta \)

where we have introduced the Bohr magneton through \(\mu _{B} =-\gamma _{e} \hbar \). For the electric transition dipole we need the explicit form of the orbitals, and use

\(\psi _{2p_{y} } =\left(\frac{3}{4\pi } \right) ^{1/2} \sin\theta \sin\varphi \left(\frac{2^{5}\zeta ^{5}_{p} }{4!} \right) ^{1/2} re^{-\zeta _{p}r }\) \(\zeta _{p} =\frac{Z^{\ast }_{p} }{n_{p}a_{0} } \)

 

\(\psi _{3d_{yz} } =\frac{1}{2} \left(\frac{15}{4\pi } \right) ^{1/2} \sin2\theta \sin\varphi \left(\frac{2^{7}\zeta ^{7}_{p} }{6!} \right) ^{1/2} r^{2} e^{-\zeta _{d}r }\) \(\zeta _{d} =\frac{Z^{\ast }_{d} }{n_{d}a_{0} } \)

The matrix element evaluates to

\(\left\langle0\left|\mu _{z} \right| 1\right\rangle =-e\left\langle2p_{y}\left|r\cos\theta \right| 3d_{yz} \right\rangle \sin\zeta =-ea_{0} \left\{\frac{2^{6}(6\zeta ^{5}_{p} \zeta ^{7}_{d})^{1/2} }{(\zeta _{p}+\zeta _{d} )^{7} } \right\} \sin\zeta \)

Therefore, from eqn 12.101 with \(2\sin\zeta \cos\zeta =\sin2\zeta \), we find

\(R_{10} =-ea_{0} \mu _{B} \left\{\frac{2^{5}(6\zeta ^{5}_{p} \zeta ^{7}_{d})^{1/2} }{(\zeta _{p}+\zeta _{d} )^{7} } \right\} \sin2\zeta \)

Comment The rotational constant is greatest when \(\zeta =\pi /4\) or \(5\pi /4\). For an O atom, \(\zeta_{p} =2.25/a_{0} \) and \(\zeta_{d} =0.33/a_{0} \), and then

\(R_{10} =(1.30\times 10^{-54}C^{2}m^{3} s^{-1} )\times \sin2\zeta \)
