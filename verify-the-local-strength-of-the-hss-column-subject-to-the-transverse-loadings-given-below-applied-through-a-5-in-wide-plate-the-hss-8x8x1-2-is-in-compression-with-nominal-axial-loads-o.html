Guest Question.

Verify the local strength of the HSS column subject to the transverse loadings given below, applied through a 5 in. wide plate. The HSS 8×8×1/2 is in compression with nominal axial loads of \(P_{D \text { column }}\) = 54 kips and \(P_{L \text { column }}=\) = 162 kips. The HSS has negligible flexural required strength and the connecting member is 4 in. wide.

Admin
Solution

Material Properties:

Column ASTM A500 Gr. B \(F_{y}=46 ksi \quad F_{u}=58 ksi\)

Plate ASTM A36 \(F_{y p}=36 ksi \quad F_{u}=58 ksi\)

Manual
Table 2-3
Table 2-4

Geometric Properties:

Column HSS8×8×1/2 B = 8.00 in. t = 0.465 in.

Plate \(B_{p}=5.50 \text { in. } \quad t_{p}=0.500 \text { in }\).

Manual
Table 1-12

Check the limits of applicability of Specification Section K1

1) Strength: \(F_{y}\) less than or equal to 52 ksi for HSS \(F_{y}=46 ksi\) o.k.

Section K1.2

2) Ductility: \(F_{y} / F_{u} \leq 0.8 \text { for HSS }\) for HSS \(F_{y} / F_{u}=0.79\) o.k.

3) \(0.25<B_{p} / B<1.0\) \(B_{p} / B=0.69\) o.k.

Section
K1.3b

4) \(B / t \leq 35\) \(B / t=17\) o.k.

Calculate the required strength

LRFD ASD
Transverse force from the plate

\(\begin{aligned}P_{u} &=1.2(10.0 kips )+1.6(30.0 kips ) \\&=60.0 kips\end{aligned}\)

Column axial force

\(\begin{aligned}P_{r} &=P_{u \text { column }} \\&=1.2(54.0 \text { kips })+1.6(162 \text { kips }) \\&=324 \text { kips }\end{aligned}\)

 Transverse force from the plate

\(\begin{aligned}P_{a} &=10.0 kips +30.0 \text { kips } \\&=40.0 kips\end{aligned}\)

Column axial force

\(\begin{aligned}P_{r} &=P_{a \text { column }} \\&=54.0 kips +162 kips \\&=216 kips\end{aligned}\)

Calculate available local yielding strength for uneven load distribution in the loaded plate

\(R_{n}=\left[\frac{10 F_{y} t}{\left(\frac{B}{t}\right)}\right] B_{p} \leq F_{y p} t_{p} B_{p}\)

Eqn. K1-2

\(R_{n}=\left[\frac{10(46 ksi )(0.465 in .)}{\left(\frac{8.00 in .}{0.465 in .}\right)}\right] 5.50 in. \leq 36 ksi (0.500 in. )(5.50 in .)\)

 

\(=68.4 \text { kips } \leq 99.0 \text { kips o.k. }\)
LRFD ASD
\(\begin{array}{c}\phi=0.95 \\\phi R_{n}=0.95(68.4 kips )=65.0 kips \\65.0 \text { kips }>60.0 kips \quad \text { o.k. }\end{array}\) \(\begin{array}{c}\Omega=1.58 \\R_{n} / \Omega=\frac{68.4 kips }{1.58}=43.3 kips \\43.3 \text { kips }>40.0 \text { kips } \quad \text { o.k. }\end{array}\)

Section
K1.3b

Check shear yielding (punching)

This limit state does not control when \(B_{p}>B-2 t\), nor when \(B_{p}<0.85 B\).

 

\(B-2 t=8.00 \text { in. }-2(0.500 \text { in. })=7.00 \text { in. }\)

 

0.85B= 0.85(8.00 in.) = 6.80 in.

Therefore, since \(B_{p}<6.80\) in. this limit state does not control.

Check sidewall strength

This limit state does not control unless the chord member and branch member (connecting element) have the same width.
