Guest Question.

(a) A current filament I is formed into a circle, \(\rho\) = a, in the z = \(\acute { z}\) plane. Find \({ H }_{ z }\) at P(0, 0, z) if I flows in the \({ a }_{ \phi }\) direction. (b) Find \({ H }_{ z }\) at P caused by a uniform surface current density K = \({ K }_{ 0 }{ a }_{ \phi }\), flowing on the cylindrical surface, \(\rho\) = a, 0 < z < h. The results of part (a) should help.

Admin
Solution

a) Use the Biot-Savart law
\(H=\int { \frac { IdL\times { a }_{ R } }{ 4\pi { R }^{ 2 } } }\)
where in this case I dL = \(I d\phi { a }_{ \phi }, { a }_{ R }=\left[ -a{ a }_{ \rho }+\left( z-\acute { z } \right) { a }_{ z } \right] /R\), and R=\(\sqrt { { a }^{ 2 }+{ \left( z-\acute { z } \right) }^{ 2 } }\) .The setup becomes
\(H=\int _{ 0 }^{ 2\pi }{ \frac { Iad\phi { a }_{ \phi }\times \left[ -a{ a }_{ \rho }+\left( z-\acute { z } \right) { a }_{ z } \right] }{ 4\pi { \left[ { a }^{ 2 }+{ \left( z-\acute { z } \right) }^{ 2 } \right] }^{ 3/2 } } } = \int _{ 0 }^{ 2\pi }{ \frac { Ia\left[ a{ a }_{ z }+\left( z-\acute { z } \right) { a }_{ \rho } \right] d\phi }{ 4\pi { \left[ { a }^{ 2 }+{ \left( z-\acute { z } \right) }^{ 2 } \right] }^{ 3/2 } } }\)

At this point we need to be especially careful. Note that we are integrating a vector with an \({ a }_{ \rho }\) component around a complete circle, where the vector has no \(\phi \) dependence. This sum of all \({ a }_{ \rho }\) components will be zero – even though this doesn’t happen when we go ahead with the integration without this knowledge. The problem is that the integral “interprets” \({ a }_{ \rho }\) as a constant direction, when in fact – as we know –\({ a }_{ \rho }\) continually changes direction as \(\phi \) varies. We drop the \({ a }_{ \rho }\) component in the integral to give

\(H=\int _{ 0 }^{ 2\pi }{ \frac { I{ a }^{ 2 }{ a }_{ z }d\phi }{ 4\pi { \left[ { a }^{ 2 }+{ \left( z-\acute { z } \right) }^{ 2 } \right] }^{ 3/2 } } } = \frac { \pi { a }^{ 2 }I{ a }_{ z } }{ 2\pi { \left[ { a }^{ 2 }+{ \left( z-\acute { z } \right) }^{ 2 } \right] }^{ 3/2 } } = \frac { m }{ 2\pi { \left[ { a }^{ 2 }+{ \left( z-\acute { z } \right) }^{ 2 } \right] }^{ 3/2 } } A/m\)
where m = \(\pi { a }^{ 2 }I{ a }_{ z }\) is the magnetic moment of the loop.
b) Using part a, we can write down the differential ﬁeld at P arising from a circular current ribbon of differential height, d\(\acute { z }\) , at location \(\acute { z }\) . The ribbon is of radius a and carries current \({ K }_{ 0 }d\acute { z } { a }_{ \phi }\)A:
\(dH=\frac { \pi { a }^{ 2 }{ K }_{ 0 }d\acute { z } { a }_{ z } }{ 2\pi { \left[ { a }^{ 2 }+{ \left( z-\acute { z } \right) }^{ 2 } \right] }^{ 3/2 } } A/m\)

The total magnetic ﬁeld at P is now the sum of the contributions of all differential rings that comprise the cylinder:

\({ H }_{ z }=\int _{ 0 }^{ h }{ } \frac { \pi { a }^{ 2 }{ K }_{ 0 }d\acute { z } }{ 2\pi { \left[ { a }^{ 2 }+{ \left( z-\acute { z } \right) }^{ 2 } \right] }^{ 3/2 } } =\frac { { a }^{ 2 }{ K }_{ 0 } }{ 2 } \int _{ 0 }^{ h }{ } \frac { d\acute { z } }{ { \left[ { a }^{ 2 }+{ z }^{ 2 }-2z\acute { z } +{ \left( \acute { z } \right) }^{ 2 } \right] }^{ 3/2 } } \\ =\frac { { a }^{ 2 }{ K }_{ 0 } }{ 2 } \frac { 2(2\acute { z } -2z) }{ 4{ a }^{ 2 }\sqrt { { a }^{ 2 }+z^{ 2 }-2z\acute { z } +{ \left( \acute { z } \right) }^{ 2 } } } |_{ 0 }^{ h }=\frac { { K }_{ 0 }(\acute { z } -z) }{ 2\sqrt { { a }^{ 2 }+{ \left( \acute { z } -z \right) }^{ 2 } } } |_{ 0 }^{ h }\\ =\frac { { K }_{ 0 } }{ 2 } \left[ \frac { (h-z) }{ \sqrt { { a }^{ 2 }+{ \left( h-z \right) }^{ 2 } } } +\frac { z }{ \sqrt { { a }^{ 2 }+{ z }^{ 2 } } } \right] A/m\)
