Guest Question.

Re-estimate the Two- and Three-Suffix Margules Parameters by the Method of Linear Regression

Determine the model parameters for the two-suffix and the three-suffix Margules equations using the liquid–vapor equilibrium data for a binary system of benzene (1)–cyclohexane (2) at 10°C. Mole fractions of liquid vs. total pressure are reported in Table E8.9A.

Admin
Solution

The excess Gibbs energy can be written:

 

\(g^{E}=R T\left[x_{1} \ln \gamma_{1}+x_{2} \ln \gamma_{2}\right]=R T\left[x_{1} \ln \left(\frac{y_{1} P}{x_{1} P_{1}^{\text {sat }}}\right)+x_{2} \ln \left(\frac{y_{2} P}{x_{2} P_{2}^{\text {sat }}}\right)\right]\) (E8.11A)

 

where we have rearranged Equation (8.10) to give:

 

\(y_{i} P=x_{i} \gamma_{i} P_{i}^{\text {sat }}\) (8.10)

 

\(\gamma_{i}=\frac{y_{i} P}{x_{i} P_{i}^{\text {sat }}}\)

 

and then substituted for \(\gamma_{i}\) Thus, we can use the experimental data from Table E8.9A to obtain values for \(g^{E}\) at every experimental point.

We can rewrite the two-suffix and three-suffix Margules equations by dividing \(g^{E} \text { by }\left(x_{1} x_{2}\right)\). The two-suffix Margules equation becomes:

 

\(\frac{g^{E}}{x_{1} x_{2}}=A\) (E8.11B)

 

Thus, the average of \(g^{E} / x_{1} x_{2}\) over all the data will give the best prediction for the value of A. The three-suffix Margules equation becomes:

 

\(\frac{g^{E}}{x_{1} x_{2}}=A+B\left(x_{1}-x_{2}\right)\) (E8.11C)

 

A plot of \(g^{E} / x_{1} x_{2} \text { vs. } x_{1}-x_{2}\) should give a straight line with slope B and intercept A. The data from Table E8.9A are plotted in this form in Figure E8.11.

The average of the data gives the value for the two-suffix Margules parameter:

 

\(A=1409[ J / mol ]\)

 

This value is close to those obtained by the objective functions in . Linear regression gives the best-fit line to be:

 

\(\frac{g^{E}}{x_{1} x_{2}}=1402+75.1\left(x_{1}-x_{2}\right)[J / mol ]\) (E8.11D)

 

Hence, we get:

 

\(A=1402[ J / mol ] \quad \text { and } \quad B=75.1[ J / mol ]\)

 

These values are close to those obtained in .
