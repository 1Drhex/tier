Guest Question.

Designing a Dc Inductor

Design a dc inductor of L = 450 µH. The dc current is \(I_L = 7.2 A\) with a ripple of ΔI = 1 A. Assume window factor \(K_u = 0.4\). Use power core with a graded air gap.

Admin
Solution

The peak inductor current \(I_m = I_L + ΔI = 7.2 + 1 = 8.2 A\). The inductor energy, \(W_t=\frac{1}{2}\times LI_m^2=\frac{1}{2}\times 450\times 10^{-6}\times 8.2^2=15 mJ \). From Table 13.1

Table 13.1 Core Configuration Constants

Pot core 433 632 -0.17 \(P_{cu} = P_{fe}\)
Power core 403 590 -0.12 \(P_{cu} \gg P_{fe}\)
E-laminated core 366 534 -0.14 \(P_{cu} = P_{fe}\)
C-core 323 468 -0.14 \(P_{cu} = P_{fe}\)
Single-coil 395 569 -0.14 \(P_{cu} \gg P_{fe}\)
Tape-wound core 250 365 -0.13 \(P_{cu} = P_{fe}\)

for power core, \(K_j = 403\) and x = -0.12. Let us choose \(B_m = 0.3\). From Eq. (13.64),

\(A_p=\left[\frac{LI^2\times 10^4}{B_cK_uK_j} \right]^{\frac{1}{1-x} } (cm^4)\) (13.64)

\(A_p=\left[\frac{2\times 0.015\times 10^4}{0.4\times 403\times 0.3} \right]^{\frac{1}{1+0.12} }=8.03 cm^4\)

Choose the power-core type, 55090-A2 (Magnetics, Inc.) with \(A_p = 8.06 cm^4\), core weight \(W_t = 0.131 kg\), core area \(A_c = 1.32 cm^2\), magnetic path length \(l_c = 11.62 cm\), and mean length of a turn \(l_{mt} = 6.66 cm\).

From Eq. (13.56), \(J = K_j A_p^ x = 403 × 8.03^{-0.12} = 313.8 A/cm^2\).

Substituting NI from Eq. (13.54) into \(B_m = μ_oμ_r H = μ_oμ_r NI/l_c\) and simplifying, we get

\(NI=K_uW_aJ\) (13.54)

\(\mu _r=\frac{B_mI_c\times 10^5}{\mu _oW_aJK_u}\) (13.65)

\(=\frac{0.3\times 11.62\times 10^5}{4\pi \times 6.11\times 313.8\times 0.4}=36.2\)

Find the material with \(\mu _r\geq 36.2\). Let us select type MPP-330T (Magnetics, Inc.) that gives \(L_c = 86 mH\) with \(N_c = 1000\) turns. Thus, the number of required turns is

\(N=N_c\sqrt{\frac{L}{L_c} }\) (13.66)

 

\(=1000\sqrt{\frac{450\times 10^{-6}}{86\times 10^{-3}} }=72\)

The bare-wire cross-sectional area is \(A_w = I_m/J = 8.2/313.8 = 0.026 cm^2\).

we find the primary AWG number 14 with \(Aw = 0.02082 cm^2\) and \(\sigma =82.8 \mu \Omega /cm\). The winding resistance is \(R=I_{mt}N_\sigma \times 10^{-6}=6.66\times 72\times 82.8 =0.04 \Omega\). The copper loss is \(P_{cu} = I^2_L R = 7.2^2 × 0.04 = 2.1 W\).

Note: Using Eq. (13.61),

\(L=\frac{N^2}{\Re _c+\Re _g} =\frac{\mu _oA_c}{l_g+\frac{l_c}{\mu _r} }N^2\) (13.61)

the air-gap length \(l_g\) with a discrete air gap is given by

\(l_g=\frac{\mu _oA_cN^2}{L }-\frac{l_c}{\mu _r}\)

\(=\left[\frac{4\times \pi \times 10^{-7}\times 1.32\times 72^2}{450\times 10^{-6}}-\frac{11.62}{36.2} \right]\times 10^{-2}=0.19 cm \) (13.67)
