Guest Question.

In Yukawa’s original theory (1934), which remains a useful approximation in nuclear physics, the “strong” force between protons and neutrons is mediated by the exchange of π-mesons. The potential energy is

\( V(r)=-r_{0} V_{0} \frac{e^{-r / r_{0}}}{r} \) (8.83)

where r is the distance between the nucleons, and the range \( r_0 \) is related to the mass of the meson: \( r_{0}=\hbar / m_{\pi} c \). Question: Does this theory account for the existence of the deuteron (a bound state of the proton and the neutron)?
The Schrödinger equation for the proton/neutron system is (see ):

\( -\frac{\hbar^{2}}{2 \mu} \nabla^{2} \psi( r )+V(r) \psi( r )=E \psi( r ) \) (8.84).

where μ is the reduced mass (the proton and neutron have almost identical masses, so call them both m), and r is the position of the neutron (say) relative to the proton: \( r = r _{n}- r _{p} \). Your task is to show that there exists a solution with negative energy (a bound state), using a variational trial wave function of the form

\( \psi_{\beta}( r )=A e^{-\beta r / r_{0}} \) (8.85).

(a) Determine A, by normalizing \( \psi_{\beta}( r ) \).

(b) Find the expectation value of the Hamiltonian \( \left(H=-\frac{\hbar^{2}}{2 \mu} \nabla^{2}+V\right) \) in the state \( \psi_{\beta} \). Answer:

\( E(\beta)=\frac{\hbar^{2}}{2 \mu r_{0}^{2}} \beta^{2}\left[1-\frac{4 \gamma \beta}{(1+2 \beta)^{2}}\right], \quad \text { where } \gamma \equiv \frac{2 \mu r_{0}^{2}}{\hbar^{2}} V_{0} \) (8.86).

(c) Optimize your trial wave function, by setting \( d E(\beta) / d \beta=0 \) . This tells
you β as a function of γ (and hence—everything else being constant—of \( V_0\) ), but let’s use it instead to eliminate γ in favor of β:

\( E_{\min }=\frac{\hbar^{2}}{2 \mu r_{0}^{2}} \frac{\beta^{2}(1-2 \beta)}{(3+2 \beta)} \) (8.87).

(d) Setting \( \hbar^{2} / 2 \mu r_{0}^{2}=1, \text { plot } E_{\min } \text { as a function of } \beta, \text { for } 0 \leq \beta \leq 1 \).

What does this tell you about the binding of the deuteron? What is the minimum value of \( V_0\) for which you can be confident there is a bound state (look up the necessary masses)? The experimental value is 52 MeV.

Admin
Solution

(a) To normalize the variational state

\( 1=\int\left|\psi_{\beta}( r )\right|^{2} d^{3} r \).

\( =A^{2} 4 \pi \int_{0}^{\infty} e^{-2 \beta r / r_{0}} r^{2} d r \).

\( =A^{2} 4 \pi\left(\frac{r_{0}}{2 \beta}\right)^{3} \int_{0}^{\infty} e^{-z} z^{2} d z \).

\( =A^{2} \frac{\pi r_{0}^{3}}{\beta^{3}} \).

and

\( A=\sqrt{\frac{\beta^{3}}{\pi r_{0}^{3}}} \).

(b) Now we calculate the expectation value of the Hamiltonian:

\( \langle H\rangle=\iint \psi_{\beta}^{*}(r)\left[-\frac{\hbar^{2}}{2 \mu} \frac{1}{r^{2}}\left(r^{2} \frac{\partial \psi_{\beta}}{\partial r}\right)-r_{0} V_{0} \frac{e^{-r / r_{0}}}{r} \psi_{\beta}(r)\right] r^{2} d r d \Omega \).

\( =4 \pi A^{2}\left(\int_{0}^{\infty} e^{-\beta r / r_{0}}\left[\frac{\hbar^{2}}{2 \mu} \frac{\beta}{r_{0}^{2}}\left(\frac{2 r_{0}}{r}-\beta\right) e^{-\beta r / r_{0}}\right] r^{2} d r\right. \)

\( \left.-r_{0} V_{0} \int e^{-(2 \beta+1) r / r_{0}} r d r\right) \).

In the first integral we let \( z=2 \beta r / r_{0} \text { and in the second integral we let } z=(2 \beta+1) r / r_{0} \text { to get } \)

\( \langle H\rangle=\frac{4 \beta^{3}}{r_{0}^{3}}\left[\frac{\hbar^{2} r_{0}}{4 \mu \beta} \int_{0}^{\infty}\left(z-\frac{1}{4} z^{2}\right) e^{-z} d z-\frac{r_{0}^{3} V_{0}}{(2 \beta+1)^{2}} \int_{0}^{\infty} z e^{-z} d z\right] \).

\( =\frac{4 \beta^{3}}{r_{0}^{3}}\left[\frac{\hbar^{2} r_{0}}{4 \mu \beta}\left(1-\frac{1}{2}\right)-\frac{r_{0}^{3} V_{0}}{(2 \beta+1)^{2}}\right] \).

\( =\frac{\hbar^{2} \beta^{2}}{2 \mu r_{0}^{2}}-\frac{4 \beta^{3}}{(2 \beta+1)^{2}} \frac{\hbar^{2} \gamma}{2 \mu r_{0}^{2}} \).

which can be put in the desired form:

\( E(\beta)=\frac{\hbar^{2}}{2 \mu r_{0}^{2}} \beta^{2}\left[1-\frac{4 \beta \gamma}{(2 \beta+1)^{2}}\right] \).

(c) ] Finding the value of β that minimizes the energy:

\( \frac{d E(\beta)}{d \beta}=0 \).

\(\frac{\hbar^{2}}{2 \mu r_{0}^{2}}\left[2 \beta-\frac{12 \beta^{2} \gamma}{(2 \beta+1)^{2}}+2 \frac{4 \beta^{3} \gamma}{(2 \beta+1)^{3}} 2\right]=0 \).

\( (2 \beta+1)^{3}-6 \beta(2 \beta+1) \gamma+8 \beta^{2} \gamma=0 \).

\( (2 \beta+1)^{3}-2 \beta(3+2 \beta) \gamma=0 \).

Solving this for γ,

\( \gamma=\frac{(2 \beta+1)^{3}}{2 \beta(3+2 \beta)} \),

and plugging back into our expression for the energy we have

\( E_{\min }=\frac{\hbar^{2}}{2 \mu r_{0}^{2}} \beta^{2}\left[1-\frac{4 \beta}{(2 \beta+1)^{2}} \frac{(2 \beta+1)^{3}}{2 \beta}\right] \).

\( =\frac{\hbar^{2}}{2 \mu r_{0}^{2}} \beta^{2}\left[1-\frac{4 \beta}{(2 \beta+1)^{2}} \frac{(2 \beta+1)^{3}}{2 \beta(3+2 \beta)}\right] \).

which simplifies to

\( E_{\min }=\frac{\hbar^{2} \beta^{2}}{2 \mu r_{0}^{2}} \frac{1-2 \beta}{3+2 \beta} \).

(d) From our plot we see that \( E_{\min }<0 \text { for } \beta>1 / 2 \). We can turn this into a value for γ; there will be a negative-energy (or bound) state when

\( \gamma>\frac{(2 \cdot 1 / 2+1)^{3}}{2 \cdot 1 / 2(3+2 \cdot 1 / 2)}=2 \).

which in turn means

\( \frac{2 \mu_{0} r_{0}^{2}}{\hbar^{2}} V_{0}>2 \).

or

\( V_{0}>\frac{\hbar^{2}}{\mu r_{0}^{2}}=2 \frac{\left(m_{\pi} c^{2}\right)^{2}}{m_{p} c^{2}}=42 MeV \).

So we know there is a bound state for \( V_{0}>42 MeV \) but a better variational wave function could show a bound state for a smaller value of \( V_{0} \).
