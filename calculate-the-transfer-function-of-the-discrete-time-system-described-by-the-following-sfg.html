Guest Question.

Calculate the transfer function of the discrete-time system described by the following SFG:

 

Admin
Solution

First suggested solution: Derivation of the mathematical model describing the system.
From the SFG we get:

\(w_{1}[n] = w_{4}[n] – x[n]\) (1)

\(w_{2}[n] =αw_{1}[n] \) (2)
\(w_{3}[n] = w_{2}[n] + x[n]\) (3)
\(w_{4}[n] = w_{3}[n – 1] \) (4)
\(y[n] = w_{2} [n] + w_{4} [n]\) (5)

Assuming zero initial conditions we z-transform the mathematical model:

\(W_{1}(z) =W_{4}(z) – X(z)\) (6)

\(W_{2}(z) =αW_{1}(z)\) (7)

\(W_{3}(z) =W_{2}(z) + X(z)\) (8)
\(W_{4}(z)=W_{3}(z)z^{-1}\) (9)
\(Y(z)=W_{2}(z)+W_{4}(z)\) (10)
\(W_{2}(z)\overset{(6)}{=}\alpha W_{4}(z)-\alpha X(z) \) (11)
\(W_{4}(z)\overset{(8)}{=}z^{-1}W_{2}(z)+z^{-1}X(z)\) (12)
\(W_{2}(z)\overset{(12)}{=}\alpha z^{-1}W_{2}(z)+\alpha z^{-1}X(z)-\alpha X(z)\Rightarrow\)
\(W_{2}(z)=\frac{\alpha z^{-1}X(z)-\alpha X(z)}{1-\alpha z^{-1} }\) (13)
\(W_{4}(z)=\alpha z^{-1}W_{4}(z)-\alpha z^{-1}X(z)+z^{-1} X(z)\Rightarrow \)
\(W_{4}(z)=\frac{z^{-1} X(z)-\alpha z^{-1}X(z)}{1-\alpha z^{-1} }\) (14)

Therefore, from relation (14) the transfer function of the discrete system is derived:

\(H(z)=\frac{Y(z)}{X(z)}=\frac{z^{-1}-\alpha }{1-\alpha z^{-1}}\) (15)

Then, the difference equation that describes the system is:

\(h[n]=\alpha ^{n-1}u[n-1]-\alpha ^{n+1}u[n]\) (16)

Second suggested solution: Using Mason’s gain formula

It is true that:\(H(z)=\frac{\sum\limits_{n=1}^{k}{T_{n}D_{n}} }{D} =\frac{T_{1}D_{1}+T_{2}D_{2}+T_{3}D_{3}+T_{4}D_{4}}{D} \) (17)

where: \(T_{n}\) is the gain of the n-th forward path that connects the input with the output.
In the SFG of this problem, there are four paths.

\(T_{1}=-1.\alpha ,T_{2}=z^{-1},T_{3}=-1.\alpha .z^{-1},T_{4}=\alpha .z^{-1}\) (18)

D is the determinant of the diagram and is given by

\(D=1-\sum{L_{1}}+\sum{L_{2}}-\sum{L_{3}}=1-\alpha z^{-1}\) (19)

\(D_{n}\) is the subdeterminant of \(T_{n}\) path and is calculated from (19), not taking into account the loops touching the n-th forward path. Obviously \(D_{n} = 1.\)
Thus, the transfer function of the discrete system using Mason’s formula would be:

\((17)\overset{(18)}{\underset{(19)}{\Rightarrow }} H(z)=\frac{-1.\alpha +z^{-1}+(-1.\alpha .z^{-1})+\alpha .z^{-1}}{1-\alpha z^{-1}}=\frac{-\alpha +z^{-1}}{1-\alpha z^{-1}}\) (20)

The result is the same with the one derived at the first suggested solution.
