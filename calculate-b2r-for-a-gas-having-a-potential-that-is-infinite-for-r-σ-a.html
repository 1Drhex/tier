Guest Question.

Calculate \(B_{2}(r)\) for a gas having a potential that is infinite for r < σ , has a square well of depth ε for r between σ and σ + a, and is zero for r > σ + a.

Admin
Solution

The relevant functions are shown in the following table:

Region r <σ σ< r < σ + a σ + a < r
u(r) \(\infty \) −ε 0
g(r) 0 \(exp(\beta \varepsilon )\) 1
f (r) -1 \(exp(\beta \varepsilon ) − 1\) 0

 

The integrations are straightforward and result in

\(B_{2}\left(T\right) = \frac{2\pi }{3} \sigma ^{3} \left[1 – \left(e^{\beta \varepsilon } – 1 \right) \frac{\left(\sigma + a\right)^{3} – \sigma ^{3}}{\sigma ^{3}} \right],\) (20.81)

which agrees with the hard sphere gas for a = 0. For βε ≪ 1, we can expand the exponential to get the high-temperature result

\(B_{2}\left(T\right) = \frac{2\pi }{3} \sigma ^{3} \left[1 – \frac{\varepsilon }{k_{B}T} \frac{\left(\sigma + a\right)^{3} – \sigma ^{3}}{\sigma ^{3}} \right].\) (20.82)

Then the equation for p can be written approximately in the van der Waals form

\(p + a_{0}/v^{2} = k_{B}T/\left(v – b_{0}\right) ,\) (20.83)

where \(v = 1/n, a_{0} = \varepsilon \left(2\pi /3\right) \left[\left(\sigma + a\right)^{3} – \sigma ^{3}\right] , and b_{0} = \left(2\pi /3\right) \sigma ^{3}.\)
