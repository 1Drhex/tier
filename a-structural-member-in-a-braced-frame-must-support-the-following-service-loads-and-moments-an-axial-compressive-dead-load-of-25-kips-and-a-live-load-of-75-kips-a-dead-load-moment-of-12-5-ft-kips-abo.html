Guest Question.

A structural member in a braced frame must support the following service loads and moments: an axial compressive dead load of 25 kips and a live load of 75 kips; a dead load moment of 12.5 ft-kips about the strong axis and a live load moment of 37.5 ft-kips about the strong axis; a dead load moment of 5 ft-kips about the weak axis and a live load moment of 15 ft-kips about the weak axis. The moments occur at one end; the other end is pinned. The member length is 15 feet. The frame analysis was a first order analysis with reduced member stiffnesses, so the moment amplification method with \(K_x\) = 1.0 can be used. In the direction perpendicular to the frame \(K_y = \)1.0. There are no transverse loads on the member. Use A992 steel and select a W10 shape.

 

Admin
Solution

The factored axial load is
\(P_nt = P_u = 1.2P_D + 1.6P_L = 1.2(25) + 1.6(75) = 150 kips\)
(There is no amplification of axial loads in members braced against sidesway.) The factored moments are
\(M_{ntx} = 1.2(12.5) + 1.6(37.5) = 75.0 ft-kips\)
\(M_{nty} = 1.2(5) + 1.6(15) = 30.0 ft-kips\)
The amplification factor \(B_1 \)can be estimated as 1.0 for purposes of making a trial selection. For the two axes,
\(M_{ux} = B_{1x}M_{ntx} = 1.0(75) = 75 ft-kips\)
\(M_{uy} = B_{1y}M_{nty} = 1.0(30) = 30 ft-kips\)
Try a W10 \(\times \)49 : From Table 6-2,
\(\phi _{c} P_n = 449 kips, \phi _{b}M_{nx} = 204 ft-kips, \phi _{b}M_{ny} = 106 ft-kips\)
Determine which interaction equation to use:
\(\frac {P_u}{\phi _{c}P_n}= \frac{150}{449} = 0.3341 > 0.2 \therefore\) AISC Equation H1-1a controls.
As a rough check (remember that B1 has not yet been computed and \(C_b\) has not been accounted for),
\(\frac {P_u}{\phi _{c}P_n} + \frac{8}{9} \left ( \frac{M_{ux}}{\phi _{b}M_{nx} } +\frac{M_{uy}}{\phi _{b}M_{ny}} \right )= 0.3341 +\frac{8}{9}\left ( \frac{75}{204 } +\frac{30}{106} \right )= 0.912 < 1.0 (OK)\)
It is likely that\( B_{1x}\) and \( B_{1y}\) will be equal to 1.0. In addition, the inclusion of\( C_b \) will increase the value of \(\phi _{b}M_{nx}\) and reduce the result of the interaction equation, so this shape is probably satisfactory.
Calculate \(B_{1}\) for each axis:
\(C_m = 0.6 – 0.4\left ( \frac{M_1}{M_2} \right )= 0.6 – 0.4\left ( \frac{0}{M_2} \right )= 0.6\) (for both axes)
From Section 6.5, for each axis,
\(P_{e1} = \frac{\pi^{2} EI^{*}}{(L_{c1})^{2}}\)

and the modified stiffness is

\(EI^{*} = 0.8\tau _{b}EI = 0.8(1.0)EI = 0.8EI\)
Therefore,

 

\(P_{e1x} = \frac{\pi^{2}(0.8EI_x) }{(L_{cx})^{2}}=\frac{\pi^{2}(0.8)(29,000)(272)}{(15 \times 12)^{2}}= 1922kips\)

 

\(B_{1x} = \frac{C_{mx}}{1 -\frac{P_u}{P_{e1x}}}=\frac{0.6}{1 – \frac{150}{1922}}= 0.651 < 1.0 \therefore B_{1x} = 1.0\) as assumed.

 

\(P_{e1y} = \frac{\pi^{2}(0.8EI_y) }{(L_{cy})^{2}}=\frac{\pi^{2}(0.8)(29,000)(93.4)}{(15 \times 12)^{2}}= 660 kips\)

 

\(B_{1y} = \frac{C_{my}}{1 -\frac{P_u}{P_{e1y}}}=\frac{0.6}{1 – \frac{150}{660}}= 0.776 < 1.0 \therefore B_{1y} = 1.0\) as assumed.

From Figure g, \( C_b = 1.67.\) Modify \( \phi _{b}M_{nx} \) to account for\( C_b.\)
\(\phi _{b}M_{nx} \times C_b = 204(1.67) = 341 ft-kips\)
For\( L_b = 0,\)
\(\phi _{b}M_{nx} = \phi _{b}M_{px} = 227 ft-kips\)
Since 341 ft-kips > 227 ft-kips, the upper limit of 227 ft-kips controls, and
\(\phi _{b}M_{nx} = 227 ft-kips.\)
Check AISC Equation H1-1a:
\(\frac {P_u}{\phi _{c}P_n} + \frac{8}{9} \left ( \frac{M_{ux}}{\phi _{b}M_{nx} } +\frac{M_{uy}}{\phi _{b}M_{ny}} \right )= 0.3341 +\frac{8}{9}\left ( \frac{75}{227 } +\frac{30}{106} \right )\)
= 0.909 < 1.0 (OK)
Try the next smaller shape. Try a W10 \(\times \) 45, with \(\phi _{c}P_{n} = 333 k, \phi _{b}M_{nx}= 175 ft-kips,\)
and \( \phi _{b}M_{ny} 76.1 ft-kips.\)

\(P_{e1x} =\frac{\pi^{2}(EI^{*}_{x}) }{(L_{cx})^{2}}= \frac{\pi^{2}(0.8EI_x) }{(L_{cx})^{2}}=\frac{\pi^{2}(0.8)(29,000)(248)}{(15 \times 12)^{2}}= 1753kips\)

 

\(B_{1x} = \frac{C_{mx}}{1 -\frac{P_u}{P_{e1x}}}=\frac{0.6}{1 – \frac{150}{1753}}= 0.656 < 1.0 \therefore B_{1x} = 1.0\) as assumed.

 

\(P_{e1y} = \frac{\pi^{2}(0.8EI_y) }{(L_{cy})^{2}}=\frac{\pi^{2}(0.8)(29,000)(53.4)}{(15 \times 12)^{2}}= 377 kips\)

 

\(B_{1y} = \frac{C_{my}}{1 -\frac{P_u}{P_{e1y}}}=\frac{0.6}{1 – \frac{150}{377}}= 0.996 < 1.0 \therefore B_{1y}\) = 1.0 as assumed.
Modify\( \phi _{b}M_{nx}\) to account for \(C_b.\)
\(\phi _{b}M_{nx} \times C_b = 175(1.67) = 292 ft-kips\)
For \(L_b = 0,\)
\(\phi _{b}M_{nx} = \phi _{b}M_{px} = 206 ft-kips\)
Since 292 ft-kips . 206 ft-kips, the upper limit of 206 ft-kips controls, and
\(\phi _{b}M_{nx} = 206 ft-kips.\)
Check AISC Equation H1-1a:
\(\frac {P_u}{\phi _{c}P_n} + \frac{8}{9} \left ( \frac{M_{ux}}{\phi _{b}M_{nx} } +\frac{M_{uy}}{\phi _{b}M_{ny}} \right )= 0.3341 +\frac{8}{9}\left ( \frac{75}{206 } +\frac{30}{76.1} \right ) \)
=1.12 > 1.0 (N.G.)
Use a W10 \(\times\) 49.
The required axial load strength is
\(P_a = P_D + P_L = 25 + 75 = 100 kips\)
The moments at each end are
\(M_{ntx} = M_D + M_L = 12.5 + 37.5 = 50.0 ft-kips\)
\(M_{nty} = M_D + M_L = 5 + 15 = 20.0 ft-kips\)
If we assume that \(B_1 \)= 1.0, The required bending moment strengths are
\(M_{ax} = B_{1}M_{ntx} = 1.0(50) = 50 ft-kips\)
\(M_{ay} = B_{1}M_{nty} = 1.0(20) = 20 ft-kips\)
The effective length for compression and the unbraced length for bending are the same:
\(L_c = L_b = 15 ft\)
From Table 6-2, try a W10 \(\times\) 49, with \(P_{n}/\Omega _{c} \)= 299 kips, \(M_{nx}/\Omega _{b} \)= 136 ft-kips, and \(M_{ny}/\Omega _{b} \)= 70.6 ft-kips.

Determine which interaction equation to use:

\(\frac{P_a}{P_n/\Omega _{c}} =\frac{100}{299} = 0.3344 > 0.2 \therefore\) AISC Equation H1-1a controls.
As a rough check (remember that B1 has not yet been computed and Cb has not been accounted for),
\(\frac{P_a}{P_n/\Omega _{c}} +\frac{8}{9} \left ( \frac{ M_{ax}}{M_{nx}/\Omega _{b}} + \frac{ M_{ay}}{M_{ny}/\Omega _{b}}\right )= 0.3344 +\frac{8}{9} \left ( \frac{50}{136} +\frac{20}{70.6} \right )= 0.913 > 1.0 (OK)\)
It is likely that \(B_{1x}\) and \(B_{1y}\) will be equal to 1.0. In addition, the inclusion of \(C_b \)will
increase the value of \(\phi _{b}M_{nx}\) and reduce the result of the interaction equation, so this
shape is probably satisfactory.
Calculate B1 for each axis:
\(C_m = 0.6 – 0.4\left ( \frac{M_1}{M_2} \right )= 0.6 – 0.4\left ( \frac{0}{M_2} \right )= 0.6\) (for both axes)
From Section 6.5, for each axis,
\(P_{e1x} =\frac{\pi^{2}EI* }{(L_{c1})^{2}}\)
and the modified stiffness is
\(EI^{*} = 0.8\tau _{b}EI = 0.8(1.0)EI = 0.8 EI\)

Therefore,
\(P_{e1x} = \frac{\pi^{2}(0.8EI_x) }{(L_{cx})^{2}}=\frac{\pi^{2}(0.8)(29,000)(272)}{(15 \times 12)^{2}}= 1922kips\)

 

\(B_{1x} = \frac{C_{mx}}{1 -\frac{1.60P_a}{P_{e1x}}}=\frac{0.6}{1 – \frac{1.60(100)}{1922}}= 0.654 < 1.0 \therefore B_{1x} = 1.0 \)as assumed.

 

\(P_{e1y} = \frac{\pi^{2}(0.8EI_y) }{(L_{cy})^{2}}=\frac{\pi^{2}(0.8)(29,000)(93.4)}{(15 \times 12)^{2}}= 660 kips\)

 

\(B_{1y} = \frac{C_{my}}{1 -\frac{1.60P_a}{P_{e1y}}}=\frac{0.6}{1 – \frac{1.60(100)}{660}}= 0.792 < 1.0 \therefore B_{1y} = 1.0\) as assumed.
From Figure g, \( C_b = 1.67.\) Modify \( M_{nx}/\Omega _{b}\) to account for \( C_b.\)
\(\frac{M_{nx}}{\Omega _{b}} \times C_b = 136(1.67) = 227 ft-kips\)

For \(L_b = 0\)
\(\frac{M_{nx}}{\Omega _{b}} =\frac{M_{px}}{\Omega _{b}} = 151 ft-kips\)
Since 227 ft-kips > 151 ft-kips, the upper limit of 151 ft-kips controls, and
\(\frac{M_{nx}}{\Omega _{b}} = 151 ft-kips\)
Check AISC Equation H1-1a:
\(\frac{P_a}{P_n/\Omega _{c}} +\frac{8}{9} \left ( \frac{ M_{ax}}{M_{nx}/\Omega _{b}} + \frac{ M_{ay}}{M_{ny}/\Omega _{b}}\right )= 0.3344 +\frac{8}{9} \left ( \frac{50}{151} +\frac{20}{70.6} \right )\)

= 0.883 < 1.0 (OK)
Try the next smaller shape. Try W10 \(\times \)45, with \(P_{n}/\Omega _{c} \)= 222 kips, \(M_{nx}/\Omega _{b} =117\) ft-kips, and \(M_{ny}/\Omega _{b} = 50.6 ft-kips.\)
\(P_{e1x} =\frac{\pi^{2}(EI*_{x}) }{(L_{cx})^{2}}= \frac{\pi^{2}(0.8EI_x) }{(L_{cx})^{2}}=\frac{\pi^{2}(0.8)(29,000)(248)}{(15 \times 12)^{2}}= 1753kips\)

 

\(B_{1x} = \frac{C_{mx}}{1 -\frac{1.60P_a}{P_{e1x}}}=\frac{0.6}{1 – \frac{1.60(100)}{1753}}= 0.660 < 1.0 \therefore B_{1x} = 1.0\) as assumed.

 

\(P_{e1y} = \frac{\pi^{2}(0.8EI_y) }{(L_{cy})^{2}}=\frac{\pi^{2}(0.8)(29,000)(53.4)}{(15 \times 12)^{2}}= 377.4 kips\)

 

\(B_{1y} = \frac{C_{my}}{1 -\frac{1.60P_a}{P_{e1y}}}=\frac{0.6}{1 – \frac{1.60(100)}{377.4}}= 1.042\)
\(M_{ay} = B_{1y}M_{nty} = 1.042(20) = 20.8 ft-kips\)
Modify \(M_{nx}/\Omega _{b}\) to account for \(C_b:\)
\(\frac{M_{nx}}{\Omega _{b}} \times C_b = 117(1.67) = 195 ft-kips\)
For\( L_b = 0,\)
\(\frac{M_{nx}}{\Omega _{b}} =\frac{M_{px}}{\Omega _{b}} = 137 ft-kips\)
Since 195 ft-kips > 137 ft-kips, the upper limit of 137 ft-kips controls, and
\(\frac{M_{nx}}{\Omega _{b}} =137 ft-kips.\)
Check AISC Equation H1-1a:
\(\frac{P_a}{P_n/\Omega _{c}} +\frac{8}{9} \left ( \frac{ M_{ax}}{M_{nx}/\Omega _{b}} + \frac{ M_{ay}}{M_{ny}/\Omega _{b}}\right )=\frac{100}{222} +\frac{8}{9} \left ( \frac{50}{137} +\frac{20.8}{50.6} \right )\)
= 1.14 > 1.0 (N.G.)
Use a W10 \(\times \)49.
