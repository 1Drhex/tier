Guest Question.

If disk A is stationary, \( w_{A} = 0 \)and disk B rotates at\( w_{B} =20 rad/s \), determine the torque T required to sustain the motion. Plot your results of torque (vertical axis) versus the gap thickness for \(0\leq t\leq 0.1 \). The gap contains SAE10 oil for which\( \mu = 0.02 N\cdot s/m^{2} \). Assume the velocity profile is linear.

Admin
Solution
0.1 0.08 0.06 0.04 0.02 0 \(\tau \left ( 10^{-3} \right ) m\)
0.628 0.785 1.05 1.57 3.14 \(\lim\) \(T\left ( N\cdot m \right )\)

Oil is a Newtonian fluid. The velocities of the oil on the surfaces of disks A and B are \(U _{A} = w_{A} r = 0 and U _{B} = w_{B} R=\left ( 20 r \right ) m/s \) Since the velocity profile is assumed to be linear as shown in Fig. a,

\(\frac{du}{dy} = \frac{U_{A} – U_{B}}{t} = \frac{0 – 20 r}{t} = \left ( -\frac{20r}{t} \right ) s^{-1}\)

Applying Newton’s law of viscosity,
\(\tau = \mu \left | \frac{du}{dy} \right | = \left ( 0.02 N\cdot s/m^{2} \right )\left ( \frac{20r}{t} \right )=\left ( \frac{0.4r}{t} \right ) N/m^{2}\)

 

The shaded differential element shown in Fig. b has an area of \(dA = 2\pi r \). Thus
\(dF = \tau dA =\left ( \frac{0.4r}{t} \right )\left ( 2 \pi r dr \right ) = \left ( \frac{0.8}{t} \right )r^{2} \)dr.Moment equilibrium about point O in Fig. b requires

\(+\sum M_{O} =0; T – \int r dF = 0\)
\(T – \int_{0}^{0.1 m} r\left [ \left ( \frac{0.8 \pi }{t} \right )r^{2} dr\right ] = 0\)
\(T = \int_{0}^{0.1 m}\left ( \frac{0.8 \pi }{t} \right )r^{3} dr\)
\(T = \left ( \frac{0.8 \pi }{t} \right )\left ( \frac{r^{4}}{4} \right )\int |_{0}^{0.1 m}\)
\(T = \left [ \frac{20\left ( 10^{-6} \right )\pi }{t}N\cdot m \right ] \)where t is in m
The plot of T vs t is shown Fig. c.
