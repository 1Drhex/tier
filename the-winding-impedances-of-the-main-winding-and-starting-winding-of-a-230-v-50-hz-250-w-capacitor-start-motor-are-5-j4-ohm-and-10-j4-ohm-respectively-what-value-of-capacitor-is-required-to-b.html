Guest Question.

The winding impedances of the main winding and starting winding of a 230 V, 50 Hz, 250 W capacitor start motor are (5 + j4) ohm and (10 + j4) ohm respectively. What value of capacitor is required to be connected in series with the starting winding to obtain maximum torque at the start.

Admin
Solution

Here, \(V = 230 V; f = 50 Hz; P_{o} = 250 W; \overline{Z}_{m} = \left(5 + j4\right) ohm; \overline{Z}_{s} = \left(10 + j4\right) ohm \)

\(\phi _{m} = \tan ^{-1} \frac{4}{5} = 38.66^{\circ } lagging\)

 

\(\phi _{s} = \tan ^{-1} \frac{4}{10} = 28.8^{\circ } lagging\)

 

The phasor diagram of the motor is shown in Fig. 11.21. Here, the main winding current \(I_{m}\) lags behind the voltage vector by \(\phi _{m} = 38.66^{\circ } \), whereas, the starting winding current lags behind the voltage vector by \(\phi _{s} = 21.8^{\circ }\). The starting torque will be maximum when phase angle between \(I_{s}\) and \(I_{m}\) i.e., \(\theta = 90^{\circ } \).

Let C farad be the value of capacitor to be connected in series with the starting winding to obtain maximum torque at start.

\(\phi^{\prime }_{s} = 38.66^{\circ } – 90^{\circ } = – 51.34^{\circ }\)

 

From phasor diagram; \(\tan \left(- 51.34^{\circ }\right) = \frac{4 – X_{c}}{10} \)

\(-1.25 = \frac{4 – X_{c}}{10} \)

 

\( X_{c} = 16.5 ohm\)

 

Capacitance of capacitor, \( C = \frac{1}{2 \pi f X_{c}} = \frac{1}{2 \pi \times 50 \times 16.5} = \mathbf{193 \mu F} \)
