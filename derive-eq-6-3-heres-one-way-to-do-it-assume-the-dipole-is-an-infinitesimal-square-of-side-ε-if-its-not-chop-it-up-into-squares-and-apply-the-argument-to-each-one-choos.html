Guest Question.

Derive Eq. 6.3. [Here’s one way to do it: Assume the dipole is an infinitesimal square, of side ε (if it’s not, chop it up into squares, and apply the argument to each one). Choose axes as shown in Fig. 6.8, and calculate F = I∫(dl × B) along each of the four sides. Expand B in a Taylor series—on the right side, for instance,

F = ∇(m · B) (6.3)

\(B = B (0, \epsilon, z) \cong B (0,0, z)+\left.\epsilon \frac{\partial B }{\partial y}\right|_{(0,0, z)}\) .

For a more sophisticated method, see .]

Admin
Solution
\(d F =I\{(d y \hat{ y }) \times B (0, y, 0)+(d z \hat{ z }) \times B (0, \epsilon, z)-(d y \hat{ y }) \times B (0, y, \epsilon)-(d z \hat{ z }) \times B (0,0, z)\}\)

 

\(=I\{-(d y \hat{ y }) \times \underbrace{[ B (0, y, \epsilon)- B (0, y, 0)]}_{\approx \epsilon \frac{\partial B }{\partial z}}+(d z \hat{ z }) \times \underbrace{[ B (0, \epsilon, z)- B (0,0, z)]}_{\approx \epsilon \frac{\partial B }{\partial y}}\}\)

 

\(\Rightarrow I \epsilon^{2}\left\{\hat{ z } \times \frac{\partial B }{\partial y}-\hat{ y } \times \frac{\partial B }{\partial z}\right\} .\left[\text { Note that }\left.\left.\int d y \frac{\partial B }{\partial z}\right|_{0, y, 0} \approx \epsilon \frac{\partial B }{\partial z}\right|_{0,0,0} \text { and }\left.\left.\int d z \frac{\partial B }{\partial y}\right|_{0,0, z} \approx \epsilon \frac{\partial B }{\partial y}\right|_{0,0,0} .\right]\)

 

\(F=m\left\{\left|\begin{array}{ccc}\hat{ x } & \hat{ y } & \hat{ z } \\0 & 0 & 1 \\\frac{\partial B_{x}}{\partial y} & \frac{\partial B_{y}}{\partial y} & \frac{\partial B_{z}}{\partial y}\end{array}\right|-\left|\begin{array}{ccc}\hat{ x } & \hat{ y } & \hat{ z } \\0 & 1 & 0 \\\frac{\partial B_{x}}{\partial z} & \frac{\partial B_{y}}{\partial z} & \frac{\partial B_{z}}{\partial z}\end{array}\right|\right\} =m\left\{\hat{ y } \frac{\partial B_{x}}{\partial y}-\hat{ x } \frac{\partial B_{y}}{\partial y}-\hat{ x } \frac{\partial B_{z}}{\partial z}+\hat{ z } \frac{\partial B_{x}}{\partial z}\right\}\)

 

\(=m\left[\hat{ x } \frac{\partial B_{x}}{\partial x}+\hat{ y } \frac{\partial B_{x}}{\partial y}+\hat{ z } \frac{\partial B_{x}}{\partial z}\right] \quad\left(\text { using } \nabla \cdot B =0 \text { to write } \frac{\partial B_{y}}{\partial y}+\frac{\partial B_{z}}{\partial z}=-\frac{\partial B_{x}}{\partial x}\right)\) .

 

But \(m \cdot B =m B_{x}(\text { since } m =m \hat{ x }, \text { here }), \text { so } \nabla ( m \cdot B )=m \nabla\left(B_{x}\right)=m\left(\frac{\partial B_{x}}{\partial x} \hat{ x }+\frac{\partial B_{x}}{\partial y} \hat{ y }+\frac{\partial B_{x}}{\partial z} \hat{ z }\right)\) .

Therefore \(F = \nabla ( m \cdot B )\) . qed
