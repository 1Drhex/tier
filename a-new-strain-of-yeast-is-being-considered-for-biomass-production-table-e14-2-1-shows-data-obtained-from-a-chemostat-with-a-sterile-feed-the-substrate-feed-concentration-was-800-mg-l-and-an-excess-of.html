Guest Question.

A new strain of yeast is being considered for biomass production. Table E14.2.1 shows data obtained from a chemostat with a sterile feed. The substrate feed concentration was 800 mg/L and an excess of oxygen was used at a pH of 5.5 and T = 35°C. Determine \(\mu_{\max }, K_{ S }, Y F_{ X / S }, k_{ d }, \text { and } m_{ S }\), assuming \(\mu_{\text {net }}=\frac{\mu_{\max } S}{K_{S}+S}-k_{d}\).

Admin
Solution

This is a case of parametric estimation as \(\mu_{\max }, K_{ S }, Y F_{ X / S }, k_{ d }\), and \(m_{ S }\), are all kinetic parameters needing to be determined from the experimental data set. Mass balance of the biomass inside the chemostat leads to:

\(\frac{d X}{d t}=D\left(X_{0}-X\right)+\left(\mu_{G}-k_{d}\right) X\) (E14.2.1)

where D is dilution rate and D = Q/V. Since the feed medium is sterile, \(X_{0}\) = 0, and the system is at steady state (dX/dt = 0), then Eq. (E14.2.1) gives rise to

X = 0 (E14.2.2)

or

\(D =\mu_{ net }=\mu_{ G }- k _{ d }\) (E14.2.3)

Combine with the Monod equation

\(\mu_{G}=\frac{\mu_{\max } S}{K_{S}+S}\) (E14.2.4)

We obtain

\(S=\frac{K_{S}\left(D+k_{d}\right)}{\mu_{max }-D-k_{d}}\) (E14.2.5)

Thus, from the mass balance of the cells in the chemostat, we are able to relate the substrate concentration with the dilution rate with the given kinetic model (Monod equation).

We next perform a mass balance on the substrate in the chemostat:

\(Q\left(S_{0}-S\right)+r_{S} V=\frac{d(S V)}{d t}\) (E14-2.6)

At steady state, nothing would change with time. Together with the biomass-yield factor (for total biomass growth), we obtain

\(X=\frac{D}{\mu_{G}} Y F_{X / S}\left(S_{0}-S\right)\) (E14.2.7)

Substitute Eqs. (E14.2.3) and (E14.2.5) into Eq. (E14.2.7), we obtain

\(X=Y F_{X/ S} D\left\lgroup \frac{S_{0}}{D+k_{d}}-\frac{K_{S}}{\mu_{\max }-D-k_{d}} \right\rgroup \) (E14.2.8)

Eqs. (E14.2.5) and (E14.2.8) define the chemostat operation. Now it is a matter of fitting the experimental data to these two equations to obtain the kinetic parameters.

The strategy here is to make the error between Eqs. (E14.2.5) and (E14.2.8) and the experimental data minimum. Before we look at the error between the model prediction and the data, we should examine the potential errors in the data set. Examining the data set, we can see that the errors of S and X are not uniform. While the errors for all the data on biomass concentration X may be regarded uniform (e.g., \(\pm\)1), however, the errors of S may be \(\pm\)1 for S > 100 and \(\pm\) 0.1 for S < 100. Therefore, when we calculate the error we need to scale the errors to the same level so that the final parameters are the best estimates from the data. A scaling factor w is thus needed to include the errors of the substrate concentrations. Table E14.2.2 shows the results. When minimizing \(\sqrt{\sum\left(S-S_{C}\right)^{2} \omega^{2}+\left(X-X_{C}\right)^{2}}\) by altering \(\mu_{max }, K_{ S }, Y F_{ X / S }, k_{ d }\), we obtain

\(\mu_{\max }=0.8223 h ^{-1}, K_{ S }=88.30 mg / L , Y F X / S =0.6129 g – cells / g – S , k_{ d }=0.0304 h ^{-1}\)

and

\(\sqrt{\Sigma\left(S-S_{C}\right)^{2} \omega^{2}+\left(X-X_{C}\right)^{2}}=16.434 \).

Now we are still missing the maintenance coefficient mS. To compute the maintenance coefficient, we need to go back to definition, Eq. (13.27) \(\mu_{ d }=k_{ d }\) or

\(-r_{ S }=\left(Y F_{ S / X} \mu_{ net }+m_{ s }\right) X\) (E14.2.9)

When carrying out mass balance, we have used

\(-r_{ S }=Y F_{ S / X } \mu_{ G } X=\left(Y F_{ S / X } \mu_{ net }+m_{ s }\right) X\) (E14.2.10)

Therefore,

\(-r_{ S }=Y F_{ S / X } \mu_{ G } X=\left[Y F_{ S/X }\left(\mu_{ G }-k_{ d }\right)+m_{ s }\right] X\) (E14.2.11)

That is

\(m_{ s }=Y F_{ S/X } k_{ d }=k_{ d } / Y F_{ X/S }\) (E14.2.12)

We can then calculate the maintenance coefficient, which is given by \(m_{ s }=k_{ d } / Y F_{ X / S }=0.04967 h ^{-1}\)
