Guest Question.

When a unit ramp input is applied to the unity feedback system having closed loop transfer function

\(\frac { C\left( s \right) }{ R\left( s \right) } =\frac { { K }_{ s+b } }{ { s }^{ 2 }+as+b } ,\left( a>0,b>0,K>0 \right)\) ,the steady state error will be

\((A)0\) \((B)\frac { a }{ b }\)

\((C)\frac { a+K }{ b}\) \((D)\frac { a-K }{ b}\)

Admin
Solution

Closed loop transfer function is

\(\quad \quad \quad \frac { C\left( s \right) }{ R\left( s \right) } =\frac { { K }_{ s+b } }{ { s }^{ 2 }+as+b }\)

And is closed loop unity feedback system.

Error, cal be calculated as

\(\quad \quad \quad E\left( s \right) =R\left( s \right) -C\left( s \right)\)

 

\(\quad \quad \quad\quad =R\left( s \right) \left[ 1-\frac { { K }_{ s+b } }{ { s }^{ 2 }+as+b } \right] \)

Steady state error, can be calculated as

\(\quad \quad \quad { e }_{ ss }=\underset { x\rightarrow \infty }{ \lim _{ }{ } } \left[ \frac { { s }^{ 2 }+as+b-ks-b }{ { s }^{ 3 }+a{ s }^{ 2 }+b } \right] \)

 

\(\quad \quad \quad\quad =\underset { x\rightarrow \infty }{ \lim _{ }{ } } \left[ \frac { { s }^{ 2 }+as-ks }{ { s }^{ 3 }+a{ s }^{ 2 }+bs } \right] ……\frac { 0 }{ 0 }\)

Using L-hospital method, we get

\(\quad \quad \quad\underset { x\rightarrow \infty }{ \lim _{ }{ } } \frac { 2s+a-k }{ 3{ s }^{ 2 }+2sa+b } =\frac { a-k }{ b }\)

Hence, the correct option is \((D).\)
