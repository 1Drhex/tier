Guest Question.

The voltage \(v=V_{0} e^{-a|t|}\) for a > 0 is connected across a parallel combination of a resistor and a capacitor as shown in Fig. 6-22(a). (a) Find currents \( i_{C}, i_{R}, \text { and } i=i_{C}+i_{R} \). (b) Compute and graph \( v, i_{C}, i_{R} \text {, and } i \text { for } V_{0}=10 V , C=1 \mu F , R=1 M \Omega \text {, and } a=1 \text {. }\)

Admin
Solution

(a) See (a) in Table 6-3 for the required currents.

(b) See (b) in Table 6-3. Figures 6-22(b)–(e) show the plots of \(v, i_{C}, i_{R}\), and i, respectively, for the given data. During t > 0, i = 0, and the voltage source does not supply any current to the RC combination. The resistor current needed to sustain the exponential voltage across it is supplied by the capacitor.

Table 6-3

Time v \(i_{C}=C d v / d t\) \(i_{R}=v / R\) \(i=i_{C}+i_{R}\)
(a) t < 0

t > 0

 \(\begin{aligned}&v=V_{0} e^{a t} \\&v=V_{0} e^{-a t}\end{aligned}\) \(\begin{aligned}i_{C} &=C V_{0} a e^{a t} \\i_{C} &=-C V_{0} a e^{-a t}\end{aligned}\) \(\begin{aligned}&i_{R}=\left(V_{0} / R\right) e^{a t} \\&i_{R}=\left(V_{0} / R\right) e^{-a t}\end{aligned}\) \(\begin{aligned}&i=v_{0}(C a+1 / R) e^{a t} \\&i=V_{0}(-C a+1 / R) e^{-a t}\end{aligned}\)
(b) t < 0

t > 0

 \(\begin{aligned}&v=10 e^{t} \\&v=10 e^{-t}\end{aligned}\) \(\begin{aligned}&i_{C}=10^{-5} e^{t} \\&i_{C}=-10^{-5} e^{-t}\end{aligned}\) \(\begin{aligned}&i_{R}=10^{-5} e^{t} \\&i_{R}=10^{-5} e^{-t}\end{aligned}\) \(\begin{aligned}&i=2\left(10^{-5} e^{t}\right) \\&i=0\end{aligned}\)
