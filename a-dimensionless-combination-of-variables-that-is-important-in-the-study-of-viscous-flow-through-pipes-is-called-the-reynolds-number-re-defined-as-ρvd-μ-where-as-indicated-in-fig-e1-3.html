Guest Question.

A dimensionless combination of variables that is important in the study of viscous flow through pipes is called the Reynolds number, Re, defined as \({\rho VD}/{\mu } \) where, as indicated in Fig. E1.3, \(\rho \) is the fluid density, V the mean fluid velocity, D the pipe diameter, and \( \mu \) the fluid viscosity. A Newtonian fluid having a viscosity of 0.38 \( {N\cdot s}/{m^{2} } \) and a specific gravity of 0.91 flows through a 25-mm-diameter pipe with a velocity of 2.6 m/s.
FIND Determine the value of the Reynolds number using
(a) SI units and
(b) BG units.

 

Admin
Solution

(a) The fluid density is calculated from the specific gravity as

\( \rho =SG_{\rho_{ H_{2}°C} } =0.91(1000{kg}/{m^{3} })=910{kg}/{m^{3} } \)

and from the definition of the Reynolds number

\( Re=\frac{\rho VD}{\mu } =\frac{(910{kg}/{m^{3} })(2.6{m}/{s})(25mm)(10^{-3}{m}/{mm} )}{0.38{N\cdot s}/{m^{2} }} =156{{(kg\cdot m}/{s^{2})}/{N} } \)

However, since 1 N = 1 \( kg\cdot {m}/{s^{2} } \) it follows that the Reynolds number is unitless (dimensionless)—that is,

Re = 156

COMMENT The value of any dimensionless quantity does not depend on the system of units used if all variables that make up the quantity are expressed in a consistent set of units. To check this we will calculate the Reynolds number using BG units.

(b) We first convert all the SI values of the variables appearing in the Reynolds number to BG values by using the conversion factors from Table 1.3. Thus,

\( Re=\frac{(1.77{slugs}/{ft^{3} })(8.53{ft}/{s})(8.20\times 10^{-2} ft)}{7.94\times 10^{-3}{Ib\cdot s}/{ft^{2} } } =156{({slug\cdot ft}/{s^{2} })}/{Ib }=156 \)

since 1 lb = 1 \( {slug\cdot ft}/{s^{2} } \).

COMMENT The values from part (a) and part (b) are the same, as expected. Dimensionless quantities play an important role in fluid mechanics, and the significance of the Reynolds number, as well as other important dimensionless combinations, will be discussed in detail in Chapter 7. It should be noted that in the Reynolds number it is actually the ratio \( {\mu }/{\rho } \) that is important, and this is the property that we have defined as the kinematic viscosity.
