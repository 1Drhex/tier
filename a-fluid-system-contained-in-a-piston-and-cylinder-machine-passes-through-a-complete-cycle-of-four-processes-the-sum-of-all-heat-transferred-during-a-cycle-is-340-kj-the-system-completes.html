Guest Question.

A fluid system, contained in a piston and cylinder machine, passes through a complete cycle of four processes. The sum of all heat transferred during a cycle is – 340 kJ. The system completes 200 cycles per min.Complete the following table showing the method for each item, and compute the net rate of work output in kW.

ΔE(kJ/min) W(kJ/min) Q (kJ/min) Process
_ 4340 0 1–2
_ 0 42000 2–3
– 73200 _ 42000 3–4
_ _ _ 4–1
Admin
Solution

Sum of all heat transferred during the cycle = – 340 kJ.

Number of cycles completed by the system = 200 cycles/min.

Process 1–2 :

Q = Δ E + W

0 = Δ E + 4340

\(\therefore\) Δ E = – 4340 kJ/min.

 

Process 2–3 :

Q = Δ E + W

42000 = Δ E + 0

ΔE = 42000 kJ/min.

 

Process 3–4 :

Q = ΔE + W

– 4200 = – 73200 + W

\(\therefore\) W = 69000 kJ/min.

 

Process 4–1 :

ΣQ = – 340 kJ
Cycle

The system completes 200 cycles/min

 

\(\therefore\) \( Q_{1-2}+Q_{2-1}+Q_{3-4}+Q_{4-1}=-340\times 200=-68000\frac{kJ}{min}\)

 

or \( 0 + 42000 + (– 4200) + Q_{4-1} = – 68000\)

 

\(Q_{4-1} = – 105800\) kJ/min.

 

Now, ∫ dE = 0, since cyclic integral of any property is zero.

 

\(\Delta E_{1-2}+\Delta E_{2-3}+\Delta E_{3-4}+\Delta E_{4-1}=0\)

 

\(– 4340 + 42000 + (– 73200) + Δ E_{4-1} = 0\)

 

\(\therefore\) \( \Delta E_{4-1}=35540 \frac{kJ}{min}\).

 

\( \therefore\) \( W_{4-1}=Q_{4-1}-\Delta E_{4-1}\)

 

\(= – 105800 – 35540 = – 141340 \frac{kJ}{min}\)

 

The completed table is given below :

ΔE(kJ/min) W(kJ/min) Q(kJ/min) Process
– 4340 4340 0 1–2
42000 0 42000 2–3
– 73200 69000 – 4200 3–4
– 73200 – 141340 – 105800 4–1

Since ΣQ = ΣW
cycle cycle

 

Rate of work output = – 68000 kJ/min = \(-\frac{68000}{60}\) kJ/s or kW

 

= 1133.33 kW.
