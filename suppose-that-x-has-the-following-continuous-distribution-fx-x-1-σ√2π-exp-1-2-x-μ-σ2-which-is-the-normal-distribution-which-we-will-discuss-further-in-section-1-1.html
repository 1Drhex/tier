Guest Question.

Suppose that X has the following continuous distribution:

\(f_{X}(x)=\frac{1}{\sigma \sqrt{2 \pi}} \exp \left\{-\frac{1}{2}\left(\frac{x-\mu}{\sigma}\right)^{2}\right\}\)

which is the normal distribution, which we will discuss further in Section 1.10.4. If Z = (X − μ)/σ , then what is \(f_{Z}(z)\)? (Note, we use Z intentionally here, rather than Y , because as we shall see in Section 1.10.8, Z is the so-called standard normal.)

Admin
Solution

In order to determine \(f_{Z}(z)\), we need to know both \(f_{X}(x)\) and dx/dz . We know \(f_{X}(x)\) is the normal distribution, as shown above. To compute dx/dz we need an expression for x, which we can get by inverting the given relationship for Z (note, for the computation of the slope, we assume that both X and Z are known, and are replaced by their lowercase equivalents):

 

\(x=g^{-1}(z)=\mu+\sigma z\)

 

which gives us

 

\(\left|\frac{d x}{d z}\right|=\left|\frac{d g^{-1}(z)}{d z}\right|=\sigma\)

 

Putting these results together gives us

 

\(f_{ Z }(z)=f_{X}(x)\left|\frac{d x}{d z}\right|=f_{X}(\mu+\sigma z) \sigma\)

 

\(=\frac{1}{\sqrt{2 \pi}} \exp \left\{-\frac{1}{2} z^{2}\right\}\)

 

Notice that the parameters μ and σ have now disappeared from the distribution of Z . As we shall see, Z is also normally distributed with \(\mu_{Z}=0 \text { and } \sigma_{ Z }=1\).

The question now arises as to what happens if the function \(Y=g(X)\) is not one to one. The answer is that the probabilities of all the X = x values which lead to each y are added into the probability that Y = y. That is, if \(g\left(x_{1}\right), g\left(x_{2}\right), \ldots\) all lead to the same value of y, then

 

\(f_{Y}(y)=f_{X}\left(x_{1}\right)\left|\frac{d x_{1}}{d y}\right|+f_{X}\left(x_{2}\right)\left|\frac{d x_{2}}{d y}\right|+\cdots\)

 

The number of terms on the right-hand-side generally depends on y, so this computation over all y can be quite difficult. For example, the function \(Y=a+b X+c X^{2}+d X^{3}\) might have three values of x leading to the same value of y over some ranges in y but only one value of x leading to the same value of y on other ranges.
