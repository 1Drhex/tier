Guest Question.

Find all solutions to the system of linear equations

x1+2×2− x3 +3×5=7 x2−4×3 + x5=−2 x4−2×5=1 (16)

Admin
Solution

In this system \(x_{3}\) and \(x_{5}\) are free variables, so we set each equal to a free parameter

x3=s1andx5=s2

It remains to determine the values of the leading variables.
Substituting \(x_{5}\) into the bottom equation, we have

x4−2s2=1

so that \(x_{4}=1+2s_{2}\). Substituting our values for \(x_{3}\) and \(x_{5}\) into th next equation up gives

x2−4s1+s2=−2

so that \(x_{2}=-2+4s_{1}- s_{2}\). Finally, substituting in for \(x_{2}\), \(x_{3}\), and \(x_{5}\) in the top equation, we have

x1+2(−2+4s1−s2)−s1+3s2=7

Hence \(x_{1}=11-7s_{1}- s_{2}\). Therefore the general solution is

x1=11−7s1−s2x2=−2+4s1−s2x3=s1x4=1+2s2x5=s2

where \(s_{1}\) and \(s_{2}\) can be any real numbers.
