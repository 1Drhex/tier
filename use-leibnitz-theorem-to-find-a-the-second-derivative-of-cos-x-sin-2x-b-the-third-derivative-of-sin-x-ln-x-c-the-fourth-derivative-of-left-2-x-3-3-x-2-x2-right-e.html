Guest Question.

Use Leibnitz’ theorem to find
(a) the second derivative of cos x sin 2x,
(b) the third derivative of sin x ln x,
(c) the fourth derivative of \(\left( 2{ x }^{ 3 }+3{ x }^{ 2 }+x+2 \right) { e }^{ 2x }\).

Admin
Solution

Leibnitz’ theorem states that if \(y(x) = u(x)v(x)\) and the rth derivative of a function f(x) is denoted by \(f^{ (r) }\) then

\({ y }^{ (n) }=\sum _{ k=0 }^{ n }{ ^{ n }{ C }_{ k } } { u }^{ (k) }{ v }^{ (n-k) }\).

So,

(a) \(\frac { { d }^{ 2 }\left( \cos { x } \sin { 2x } \right) }{ { dx }^{ 2 } } =\left( -\cos { x } \right) \left( \sin { 2x } \right) +2\left( -\sin { x } \right) \left( 2\cos { 2x } \right)\)

\(+\left( \cos { x } \right) \left( -4\sin { 2x } \right)\)

\(=-5\cos { x } \sin { 2x } -4\sin { x } \cos { 2x }\)

\(=2\sin { x } \left[ -5\cos ^{ 2 }{ x } -2\left( 2\cos ^{ 2 }{ x } -1 \right) \right]\)

\(=2\sin { x } \left( 2-9\cos ^{ 2 }{ x } \right)\).

(b) \( \frac { { d }^{ 3 }\left( \cos { x } \ln { x } \right) }{ { dx }^{ 3 } } =\left( -\cos { x } \right) \left( \ln { x } \right) +3\left( -\sin { x } \right) \left( { x }^{ -1 } \right)\)

\(+3\left( \cos { x } \right) \left( -{ x }^{ -2 } \right) +\left( \sin { x } \right) \left( 2{ x }^{ -3 } \right)\)

\(=\left( 2{ x }^{ -3 }-3{ x }^{ -1 } \right) \sin { x } -\left( 3{ x }^{ -2 }+\ln { x } \right) \cos { x }\).

(c) We note that the nth derivative of \({e }^{ 2x }\) is \({ 2 }^{ n }{e }^{ 2x }\) and that the 4th derivative of a cubic polynomial is zero. And so,

\(\frac { { d }^{ 4 }\left[ \left( 2{ x }^{ 3 }+3{ x }^{ 2 }+x+2 \right) { e }^{ 2x } \right] }{ { dx }^{ 4 } }\)

\(=\left( 0 \right) \left( { e }^{ 2x } \right) +4\left( 12 \right) \left( { 2e }^{ 2x } \right) +6\left( 12x+6 \right) \left( { 4e }^{ 2x } \right)\)

\(+4\left( 6{ x }^{ 2 }+6x+1 \right) \left( 8{ e }^{ 2x } \right) +\left( { 2x }^{ 3 }+3{ x }^{ 2 }+x+2 \right) \left( 16{ e }^{ 2x } \right)\)

\(=16\left( { 2x }^{ 3 }+15{ x }^{ 2 }+31x+19 \right) { e }^{ 2x }\).
