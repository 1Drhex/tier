Guest Question.

If \(A=\rho\cos\phi a_{\rho}+\sin \phi a_{\phi}\), evaluate \(\oint{A}\cdot dl\) around the path shown in Figure 3.23. Confirm this by using Stokes’s theorem.

Admin
Solution

Let

\(\oint_{L}A\cdot dl=\left[\int_{a}^{b}+\int_{b}^{c}+\int_{c}^{d}+\int_{d}^{a}\right] A\cdot dl\)

where path L has been divided into segments ab, bc, cd, and da as in Figure 3.23. Along ab, \(\rho=2\) and \(dl=\rho d\phi a_{\phi}\). Hence

\(\int_{a}^{b}A\cdot dl=\int_{\phi=60^{\circ}}^{30^{\circ}}\rho\sin\phi d\phi=2\left(-\cos\phi\right)|_{60^{\circ}}^{30^{\circ}}=-\left(\sqrt{3}-1\right)\)

Along bc, \(\phi=30^{\circ}\) and \(dl=d\rho a_{\rho}\). Hence

\(\int_{b}^{c}A\cdot dl=\int_{\rho =2}^{5}\rho\cos\phi d\rho =\cos30^{\circ}\frac{\rho^{2}}{2}|_{2}^{5}=\frac{21\sqrt{3}}{4}\)

Along cd, \(\rho =5\) and \(dl=\rho d\phi a_{\phi }\). Hence

\(\int_{c}^{d}A\cdot dl=\int_{\phi=30^{\circ}}^{60^{\circ}}\rho\sin\phi d\phi=5\left(-\cos\phi\right)|_{30^{\circ}}^{60^{\circ}}=\frac{5}{2} \left(\sqrt{3}-1\right)\)

Along da, \(\phi=60^{\circ}\) and \(dl=d\rho a_{\rho}\). Hence

\(\int_{d}^{a}A\cdot dl=\int_{\rho =5}^{2}\rho\cos\phi d\rho =\cos60^{\circ}\frac{\rho^{2}}{2}|_{5}^{2}=-\frac{21}{4}\)

Putting all these together results in

\(\oint_{L}A\cdot dl=-\sqrt{3}+1+\frac{21\sqrt{3} }{4}+\frac{5\sqrt{3} }{2}-\frac{5}{2}-\frac{21}{4}\)

\(=\frac{27}{4}\left(\sqrt{3}-1 \right)=4.941\)

From Stokes’s theorem (because L is a closed path)

\(\oint_{L}A\cdot dl=\int_{S}\left(\nabla \times A\right)\cdot dS\)

But

\(dS=\rho d\phi d\rho a_{z}\)

and

\(\nabla \times A=a_{\rho }\left[\frac{1}{\rho }\frac{\partial A_{z}}{\partial \phi } -\frac{\partial A_{\phi} }{\partial z} \right] + a_{\phi }\left[\frac{\partial A_{\rho }}{\partial z } -\frac{\partial A_{z} }{\partial \rho }\right] + a_{z }\frac{1}{\rho }\left[\frac{\partial}{\partial \rho }\left(\rho A_{\phi }\right) -\frac{\partial A_{\rho } }{\partial\phi } \right]\)

\(=\left(0-0\right) a_{\rho} +\left(0-0\right) a_{\phi}+\frac{1}{\rho } \left(1+\rho \right)\sin\phi a_{z}\)

Hence:

\(\int_{S}\left(\nabla \times A\right)\cdot dS=\int_{\phi=30^{\circ}}^{60^{\circ}}\int_{\rho =2}^{5}\frac{1}{\rho}\left(1+\rho \right)\sin\phi\rho d\rho d\phi\)

\(=\int_{30^{\circ}}^{60^{\circ}}\sin\phi d\phi\int_{2}^{5}\left(1+\rho \right) d\rho=-\cos\phi |_{30^{\circ}}^{60^{\circ}}\left(\rho +\frac{\rho ^{2}}{2} \right)|_{2}^{5} \)

\(=\frac{27}{4}\left(\sqrt{3} -1\right) =4.941 \)
