Guest Question.

A layered soil is shown in Figure 7.24. Given:

• H1 =1.5 m \(K_{1} \) =\( 10^{-4}\) cm/sec

• H2 = 3 m \(K_{2} \) =3.2 × \( 10^{-2}\) cm/sec

• H3 = 2 m \(K_{3} \) =4.1×\( 10^{-5}\) cm/sec

Estimate the ratio of equivalent hydraulic conductivity,\(\frac{K_{H\left(eq\right) } }{K_{V\left(eq\right) }} \)

Figure 7.24 A layered soil profile

Admin
Solution

From Eq. (7.40),

\(K_{H\left(eq\right)} =\frac{1}{H} \left(K_{H1} H_{1} + K_{H2} H_{2} + K_{H3} H_{3}+…K_{Hn} H_{n} \right) \)

 

\(K_{H\left(eq\right)} =\frac{1}{H} \left(K_{H1} H_{1} + K_{H2} H_{2} + K_{H3} H_{3} \right) \)

 

\(=\frac{1}{\left(1.5+3+2\right) } \left[\left(10^{-4} \right)\left(1.5\right) +\left(3.2 \times 10^{-2} \right) \left(3\right) +\left(4.1\times 10^{-5} \right) \left(2\right) \right] =148.05 \times 10^{-4} cm/sec \)

Again, from Eq. (7.45),

\(K_{V\left(eq\right) } =\frac{H}{\left(\frac{H_{1} }{K_{V_{1} } } \right)+ \left(\frac{H_{2} }{K_{V_{2} } } \right) + \left(\frac{H_{3} }{K_{V_{3} } } \right) +… \left(\frac{H_{n} }{K_{V_{n} } } \right)}\)

 

\(K_{V\left(eq\right) } =\frac{H}{\left(\frac{H_{1} }{K_{V_{1} } } \right)+ \left(\frac{H_{2} }{K_{V_{2} } } \right) + \left(\frac{H_{3} }{K_{V_{3} } } \right) }\)

 

\(= \frac{1.5 + 3 + 2}{\left(\frac{1.5}{10^{-4} } \right) +\left(\frac{3}{3.2\times 10^{-2} } \right) +\left(\frac{2}{4.1\times 10^{-5} } \right) }\) \(= 1.018 \times 10^{-4} cm/sec\)

Hence,

\(\frac{K_{H\left(eq\right) }}{K_{V\left(eq\right) }} =\frac{148.05\times 10^{-4}}{1.018 \times 10^{-4} } = 145.4\)
