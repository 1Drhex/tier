Guest Question.

Determine the reactions at the supports of the beam shown in Fig. 4.2 using stiffness method. E = 29000 ksi, I = 446\(in^{4} \).

Admin
Solution

Steps 1-3

Fig. 4.3 shows the beam after executing steps 1 through 3. Member 1 is defined as node number 1-2 and member 2 is defined as node number 2-3. This order needs to be used when writing the [K] matrix using the action and displacement vector using Eq. 4.2 and Eq. 4.3

\(\left\{A\right\}=\left\{\begin{matrix} A_{1} \\ A_{2} \\ A_{3} \\ A_{4} \end{matrix} \right\}\)and \(\left\{D\right\}=\left\{\begin{matrix} D_{1} \\ D_{2} \\ D_{3} \\ D_{4} \end{matrix} \right\}\) respectively. Note that it could have been defined differently; like member 1 could have been defined as 2-1.The Kinematic Indeterminacy of this structure is 3 and the unknown displacements are\(D_{1}\).\(D_{2}\)and\(D_{3}\)(translation at joint 1 and rotations at joint 2 and 3).Figure 4.3 shows the displacement numbering is done starting with the free displacements followed by the restrained displacements.In this problem\(D_{1}\),\(D_{2}\) and\(D_{3}\) are free displacements (unknowns) while \(D_{4}\) through \(D_{6}\) are the restrained displacements,which are zero in this problem. As explained earlier,the actions\(\left\{A_{F} \right\} \)corresponding to\(\left\{D_{F} \right\} \)are known (which can easily be obtained from the fixed end moments).The\(\left\{A_{R} \right\} \)corresponding to the known\(\left\{D_{R} \right\}\) are the
unknowns.

Step 4

The stiffness matrix \(\left\{K_{1} \right\}\)for member 1 is obtained from Eq. 4.2

as using the given properties of the beam element in the given problem.

Table 4.1: Stiffness matrix for member 1

The above matrix, \(\left[K_{1} \right]\),relates the action vector\(\left\{A_{5},A_{4},A_{6},A_{3},\right\} ^{t} \) to the displacement vector\(\left\{D_{5},D_{4},D_{6},D_{3},\right\} ^{t} \).pointed out earlier in the basic procedure, displacements numbering follows the order of translation followed by rotations. The same order is followed by the corresponding actions as well. This is very important as noted earlier. A derivation of some terms used in calculating the stiffness matrix [K] is shown below.\(K_{55}=k_{66}=-k_{65}=-k_{56}=\frac{12EI}{L^{3} }=\frac{1\left(29000\right)\left(446\right) }{\left(10*12\right) ^{3} }=89.82 \)Similarly, other terms in the stiffness matrix\(K_{1}\)can be calculated. In exactly same way,\(K_{2}\)the stiffness matrix for member 2 can be
assembled using Eq. 4.2 and Eq. 4.3 as,

\(\left\{A\right\}=\left\{\begin{matrix} A_{1} \\ A_{2} \\ A_{3} \\ A_{4} \end{matrix} \right\}\)and \(\left\{D\right\}=\left\{\begin{matrix} D_{1} \\ D_{2} \\ D_{3} \\ D_{4} \end{matrix} \right\}\)

Table 4.2: Stiffness matrix for member 2

The above [K] matrix relates the action vector\(\left\{A_{6},A_{3},A_{2},A_{1},\right\} ^{t} \) to the displacement vector\(\left\{D_{6},D_{3},D_{2},D_{1},\right\} ^{t} \). Again, translations are numbered first followed by rotations

Step 5

The structure stiffness matrix for this problem \(\left[K_{J} \right]\)can be assembled using Eq. 4.4\(K_{Jij}=\sum{} ^{n}_{1}K_{A_{i-D_{j} } }\)Where, n = Number of terms with the same i-j location.\(K_{A_{i-D_{j} } }\)=Value in the element stiffness matrix [K]corresponding to\(A_{i}\) and\(D_{j}\)and the assembled matrices\(\left[K_{1} \right]\)and \(\left[K_{2} \right]\).
