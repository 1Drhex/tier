Guest Question.

Determine the \(P/A\) factor value for \(i = 7.75\)% and \(n = 10\) years, using the three methods described previously.

Admin
Solution

Factor formula: Apply the formula from inside the front cover of the book for the \(P/A\) factor. Showing 5-decimal accuracy,

\(( P/A ,7.75\)%\(,10) = \frac { { \left( 1+i \right) }^{ n }-1 }{ i{ \left( 1+i \right) }^{ n } } =\frac { { \left( 1.0775 \right) }^{ 10 }-1 }{ 1.0775{ \left( 1.0775 \right) }^{ 10 } } =\frac { 1.10947 }{ 0.16348 } \)

 

\(\quad \quad\quad\quad\quad \quad \quad\quad=6.78641\)

 

Spreadsheet: Utilize the spreadsheet function in Figure 2–9, that is, \(= – PV(7.75\)%\(,10,1),\) to
display \(6.78641\).
Linear interpolation: Use Figure 2–10 as a reference for this solution. Apply the Equation
[2.16] and [2.17] sequence, where \(x\) is the interest rate \(i\) , the bounding interest rates are
\({ i }_{ 1 } = 7\)% and \({ i }_{ 1 } = 8\)%, and the corresponding \(P/A \)factor values are \({ f }_{ 1 } = ( P/A ,7\)%\(,10) = 7.0236\) and \({ f }_{ 2 } = ( P/A ,8\)%\(,10) = 6.7101\). With 4-place accuracy,

 

\(\quad \quad\quad f={ f }_{ 1 }+\frac { \left( x-{ x }_{ 1 } \right) }{ \left( { x }_{ 2 }-{ x }_{ 1 } \right) } \left( { f }_{ 2 }-{ f }_{ 1 } \right) \quad \quad \quad \left[ 2.16 \right]\)

 

\(\quad \quad\quad f={ f }_{ 1 }+\frac { a }{ b } c={ f }_{ 1 }+d\quad \quad\quad \quad \quad \left[ 2.17 \right]\)

 

\(\quad \quad\quad f={ f }_{ 1 }+\frac { \left( i-{ i }_{ 1 } \right) }{ \left( { i }_{ 2 }-{ i }_{ 1 } \right) } \left( { f }_{ 2 }-{ f }_{ 1 } \right) =7.0236+\frac { \left( 7.75-7 \right) }{ \left( 8-7 \right) } \left( 6.7101-7.0236 \right)\)

 

\(\quad \quad\quad =7.0236+\left( 0.75 \right) \left( -0.3135 \right) =7.0236-0.2351\)

 

\(\quad \quad\quad =6.7885\)

 

Comment
Note that since the \(P/A\) factor value decreases as i increases, the linear adjustment is negative
at \(-0.2351\). As is apparent, linear interpolation provides an approximation to the correct factor
value for \(7.75\)% and 10 years, plus it takes more calculations than using the formula or spreadsheet
function. It is possible to perform two-way linear interpolation for untabulated \(i\) and \(n\)
values; however, the use of a spreadsheet or factor formula is recommended.
