Guest Question.

For the Wheatstone bridge circuit in Figure 9-12, find the loop currents. Use the loop currents to solve for the current in each resistor (branch current).

Admin
Solution

Assign three counterclockwise loop currents(\(I_{A}\),\(I_{B}\), and \(I_{C}\)) as shown in Figure 9- 12. Then write the loop equations. The equations for the loops are

Loop A: -12 + 330(\(I_{A}\)– \(I_{B}\)) + 300(\(I_{A}\) – \(I_{C}\)) = 0

Loop B: 330(\(I_{B}\) – \(I_{A}\)) + 360\(I_{B}\) + 1000(\(I_{B}\) – \(I_{C}\)) = 0

Loop C: 300(\(I_{C}\) – \(I_{A}\)) + 1000(\(I_{C}\)– \(I_{B}\)) + 390 \(I_{C}\) = 0

Rearrange the equations into standard form:

Loop A : 630\(I_{A}\) – 330\(I_{B}\) – 300\(I_{C}\) = 12 V

Loop B : -330\(I_{A}\) + 1690\(I_{B}\) – 1000\(I_{C}\) = 0

Loop C : -300\(I_{A}\) – 1000\(I_{B}\) + 1690\(I_{C}\) = 0

You can solve these equations with substitution, but this is tedious with three unknowns. The determinant method or directly solving with your calculator are simpler ways. Units are not shown until the end of the problem. Evaluating the characteristic determinant using the expansion method.

\(\left|\begin{matrix} 630 \\ -330 \\ -300 \end{matrix} \ \ \begin{matrix} -330 \\ 1690 \\ -1000 \end{matrix} \ \ \begin{matrix} -300 \\ -1000 \\ 1690 \end{matrix} \right| \begin{matrix} 630 \\ -330 \\ -300 \end{matrix} \ \ \ \ \begin{matrix} -330 \\ 1690 \\ -1000 \end{matrix}\)

 

\(= [(630)( 1690)(1690) + (-330)(-1000)(-300) + (-300)(-330)(-1000)]— [(—300)(1690)(-300) + (-1000)(-1000)(630) + (1690)(-330)(-330)]\)

 

\(= 635202000 \)

Solving for \(I_{A}\):

\(\frac{\left|\begin{matrix} 12 \\ 0 \\ 0 \end{matrix} \ \ \begin{matrix} -330 \\ 1690 \\ -1000 \end{matrix} \ \ \begin{matrix} -300 \\ -1000 \\ 1690 \end{matrix} \right|}{635202000} = \frac{(12)(1690|(1690) – (12)(-1000)(-1000)}{635202000} = 0.0351 \ A = 35.1 \ mA\)

Solving for \(I_{B}\):

\(\frac{\left|\begin{matrix} 630 \\ -330\\ -300 \end{matrix} \ \ \begin{matrix} 12 \\ 0\\ 0 \end{matrix} \ \ \begin{matrix} -300 \\ -1000 \\ 1690 \end{matrix} \right|}{635202000} = \frac{(12)(-1000)(-300) – (-330)(12)(1690)}{635202000} = 0.0162 \ A = 16.2 \ mA\)

Solving for \(I_{C}\):

\(\frac{\left|\begin{matrix} 630 \\ -330\\ -300 \end{matrix} \ \ \begin{matrix} -330 \\ 1690 \\ -1000 \end{matrix} \ \ \begin{matrix} 12 \\ 0 \\ 0 \end{matrix} \right|}{635202000} = \frac{(12)(-330)(-1000) – (-300)(1690)( 12)}{635202000} = 0.0158 \ A = 15.8 \ mA \)

The current in \(R_{1}\) is the difference between\(I_{A}\)and \(I_{B}\) :

\(I_{1}\) = (\(I_{A}\) – \(I_{B}\)) = 35.1 mA – 16.2 mA = 18.9 mA

The current in \(R_{2}\) is the difference between \(I_{A}\) and \(I_{C}\) :

\(I_{2}\) = (\(I_{A}\) – \(I_{C}\)) = 35.1 mA – 15.8 mA = 19.3 mA

The current in \(R_{3}\) is \(I_{B}\) :
\(I_{3}\) = \(I_{B}\) = 16.2 mA

The current in \(R_{4}\) is \(I_{C}\) :
\(I_{4}\) = \(I_{C}\) = 15.8 mA

The current in \(R_{L}\) is the difference between \(I_{B}\) and \(I_{C}\):

\(I_{L}\)= (\(I_{B}\) – \(I_{C}\) ) = 16.2 mA – 15.8 mA = 0.4 mA
