Guest Question.

Determine if a 14-ft long HSS10×6×3/8 ASTM A500 grade B column filled with \(f_{c}^{\prime}\) = 5 ksi normal weight concrete can support a dead load of 56 kips and a live load of 168 kips in axial compression. The column is pinned at both ends and the concrete at the base bears directly on the base plate. At the top, the load is transferred to the concrete in direct bearing.

Admin
Solution

Calculate the required compressive strength

LRFD ASD
\(\begin{aligned}P_{u} &=1.2(56 kips )+1.6(168 kips ) \\&=336 kips\end{aligned}\) \(\begin{aligned}P_{a} &=56 \text { kips }+168 \text { kips } \\&=224 \text { kips }\end{aligned}\)

The available strength in axial compression can be determined directly from the Manual at KL = 14 ft as:

LRFD ASD
\(\begin{array}{l}\phi_{c} P_{n}=353 \text { kips } \\353 \text { kips }>336 \text { kips }\end{array}\)

o.k.

 \(\begin{array}{l}P_{n} / \Omega_{c}=236 \text { kips } \\236 \text { kips }>224 \text { kips }\end{array}\)

o.k.

\(=\left(10.4 in .^{2}\right)(46 ksi )+0.85\left(48.4 in ^{2}\right)(5 ksi )=684 \text { kips }\)

 

\(C_{3}=0.6+2\left(\frac{A_{s}}{A_{c}+A_{s}}\right)=0.6+2\left(\frac{10.4 in .^{2}}{48.4 in .^{2}+10.4 in ^{2}}\right)=0.954 \geq 0.90\)
