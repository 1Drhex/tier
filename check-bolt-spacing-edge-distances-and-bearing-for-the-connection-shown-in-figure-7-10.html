Guest Question.

Check bolt spacing, edge distances, and bearing for the connection shown in Figure 7.10.

Admin
Solution

From AISC J3.3, the minimum spacing in any direction is

2 2⁄3d=2.667(\(\frac{3}{4}\))=2.00 in.

Actual spacing =2.50 in. > 2.00 in. (OK)

From AISC Table J3.4, the minimum edge distance in any direction is 1 inch.

Actual edge distance = \(1\frac{1}{4}\) in. > 1 in. (OK)

For computation of the bearing strength, use a hole diameter of

\(h=d+\frac{1}{16}=\frac{3}{4}+\frac{1}{16}=\frac{13}{16}\) in.

Check bearing on both the tension member and the gusset plate. For the tension member and the holes nearest the edge of the member,

\(\ell_{c}=\ell_{e}-\frac{h}{2}=1.25-\frac{13/16}{2}=0.8438\) in.

\(R_{n}=1.2\ell_{c}t F_{u}≤2.4dtF_{u}\)

\(1.2\ell_{c}t F_{u}=1.2(0.8438)(\frac{1}{2})(58)=29.36\) kips

Check upper limit:

\(2.4dtF_{u}=2.4(\frac{3}{4})(\frac{1}{2})(58)=52.20\) kips

29.36 kips < 52.20 kips \(∴\) Use \(R_{n }= 29.36\) kips/bolt

(This result means that \(\ell_{c}\) is small enough so that it must be accounted for.)
For the other holes,

\(\ell_{c}=s-h=2.5-\frac{13}{16}=1.688\) in.

\(R_{n}=1.2\ell_{c}t F_{u}≤2.4dtF_{u}\)

\(1.2\ell_{c}t F_{u}=1.2(1.688)(\frac{1}{2})(58)=58.74\) kips

Upper limit (the upper limit is independent of \(\ell_{c}\) and is the same for all bolts):

\(2.4dtF_{u}=52.20\) kips < 58.74 kips \(∴\) Use \(R_{n }= 52.20\) kips/bolt

(This result means that \(\ell_{c}\) is large enough so that it does not need to b accounted for. Hole deformation controls.)

The bearing strength for the tension member is

\(R_{n }= 2(29.36) + 2(52.20) = 163.1\) kips

For the gusset plate and the holes nearest the edge of the plate,

\(\ell_{c}=\ell_{e}-\frac{h}{2}=1.25-\frac{13/16}{2}=0.8438\) in.

\(R_{n}=1.2\ell_{c}t F_{u}≤2.4dtF_{u}\)

\(1.2\ell_{c}t F_{u}=1.2(0.8438)(\frac{3}{8})(58)=22.02\) kips

Upper limit \(=2.4dtF_{u}=2.4(\frac{3}{4})(\frac{3}{8})(58)\)

=39.15 kips > 22.02 kips \(∴\) Use \(R_{n }= 22.02\) kips/bolt

For the other holes,

\(\ell_{c}=s-h=2.5-\frac{13}{16}=1.688\) in.

\(R_{n}=1.2\ell_{c}t F_{u}≤2.4dtF_{u}\)

\(1.2\ell_{c}t F_{u}=1.2(1.688)(\frac{3}{8})(58)=44.06\) kips

Upper limit \(= 2.4dtF_{u} = 39.15\) kips < 44.06 kips \(∴\) Use \(R_{n }= 39.12\) kips/bolt

The bearing strength for the gusset plate is

\(R_{n}=2(22.02)+2(39.15)=122.3\) kips

The gusset plate controls. The nominal bearing strength for the connection is therefore

\(R_{n} = 122.3\) kips

The design strength is \(\phi R_{n}=0.75(122.3)=91.7\) kips.

The required strength is

\(R_{u} = 1.2 D+1.6 L=1.2(15)+1.6(45)= 90.0\) kips < 91.7 kips (OK)

The allowable strength is \(\frac{R_{n}}{Ω}=\frac{122.3}{2.00}=61.2\) kips.

The required strength is

\(R_{a} = D + L = 15 + 45 = 60\) kips < 61.2 kips (OK)

Bearing strength, spacing, and edge-distance requirements are satisfied.
