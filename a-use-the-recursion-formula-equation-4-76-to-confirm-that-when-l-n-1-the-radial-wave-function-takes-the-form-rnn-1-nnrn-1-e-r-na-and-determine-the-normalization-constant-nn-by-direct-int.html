Guest Question.

(a) Use the recursion formula (Equation 4.76) to confirm that when l = n-1 the radial wave function takes the form

\( R_{n(n-1)}=N_{n} r^{n-1} e^{-r / n a} \) ,

\( c_{j+1}=\frac{2(j+\ell+1-n)}{(j+1)(j+2 \ell+2)} c_{j} \) (4.76)

and determine the normalization constant \( N_n \) by direct integration.

(b) Calculate 〈r〉 and 〈r²〉 for states of the form \( \psi_{n}(n-1) m \).

(c) Show that the “uncertainty” in \( r\left(\sigma_{r}\right) \text { is }\langle r\rangle / \sqrt{2 n+1} \) for such states.
Note that the fractional spread in r decreases, with increasing n (in this sense the system “begins to look classical,” with identifiable circular “orbits,” for large n). Sketch the radial wave functions for several values of n, to illustrate this point.

Admin
Solution

(a) Equation 4.75 \( \Rightarrow R_{n(n-1)}=\frac{1}{r} \rho^{n} e^{-\rho} v(\rho) \), \( \rho \equiv \frac{r}{n a} ; \quad \text { Eq. } 4.76 \Rightarrow c_{1}=\frac{2(n-n)}{(1)(2 n)} c_{0}=0 \).

\( R_{n \ell}(r)=\frac{1}{r} \rho^{\ell+1} e^{-\rho} v(\rho) \) (4.75).

\( c_{j+1}=\frac{2(j+\ell+1-n)}{(j+1)(j+2 \ell+2)} c_{j} \) (4.76).

So \( v(\rho)=c_{0}, \quad \text { and hence } \quad R_{n(n-1)}=N_{n} r^{n-1} e^{-r / n a}, \quad \text { where } \quad N_{n} \equiv \frac{c_{0}}{(n a)^{n}} \).

\( 1=\int_{0}^{\infty}|R|^{2} r^{2} d r=\left(N_{n}\right)^{2} \int_{0}^{\infty} r^{2 n} e^{-2 r / n a} d r=\left(N_{n}\right)^{2}(2 n) !\left(\frac{n a}{2}\right)^{2 n+1} \); \( N_{n}=\left(\frac{2}{n a}\right)^{n} \sqrt{\frac{2}{n a(2 n) !}} \).

(b)

\( \left\langle r^{l}\right\rangle=\int_{0}^{\infty}|R|^{2} r^{l+2} d r=N_{n}^{2} \int_{0}^{\infty} r^{2 n+l} e^{-2 r / n a} d r \).

\( \langle r\rangle=\left(\frac{2}{n a}\right)^{2 n+1} \frac{1}{(2 n) !}(2 n+1) !\left(\frac{n a}{2}\right)^{2 n+2}=\left(n+\frac{1}{2}\right) n a .\)

\( \left\langle r^{2}\right\rangle=\left(\frac{2}{n a}\right)^{2 n+1} \frac{1}{(2 n) !}(2 n+2) !\left(\frac{n a}{2}\right)^{2 n+3}=(2 n+2)(2 n+1)\left(\frac{n a}{2}\right)^{2}=\left(n+\frac{1}{2}\right)(n+1)(n a)^{2} \).

(c)

\( \sigma_{r}^{2}=\left\langle r^{2}\right\rangle-\langle r\rangle^{2}=\left[\left(n+\frac{1}{2}\right)(n+1)(n a)^{2}-\left(n+\frac{1}{2}\right)^{2}(n a)^{2}\right] \).

\( =\frac{1}{2}\left(n+\frac{1}{2}\right)(n a)^{2}=\frac{1}{2(n+1 / 2)}\langle r\rangle^{2} ; \quad \sigma_{r}=\frac{\langle r\rangle}{\sqrt{2 n+1}} \).

Maxima occur at: \( \frac{d R_{n, n-1}}{d r}=0 \Rightarrow(n-1) r^{n-2} e^{-r / n a}-\frac{1}{n a} r^{n-1} e^{-r / n a}=0 \Rightarrow r=n a(n-1) \).
