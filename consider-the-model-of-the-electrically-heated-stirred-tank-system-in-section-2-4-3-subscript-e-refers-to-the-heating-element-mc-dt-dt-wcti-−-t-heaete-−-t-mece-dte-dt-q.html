Guest Question.

Consider the model of the electrically heated stirred-tank system in Section 2.4.3. Subscript \(e\) refers to the heating element:

\(\begin{aligned}m C \frac{d T}{d t} &=w C\left(T_{i}-T\right)+h_{e} A_{e}\left(T_{e}-T\right) \\m_{e} C_{e} \frac{d T_{e}}{d t} &=Q-h_{e} A_{e}\left(T_{e}-T\right)\end{aligned}\)

(a) Derive transfer functions relating changes in outlet temperature \(T\) to changes in the two input variables: heater input \(Q\) (assuming no change in inlet temperature), and inlet temperature \(T_{i}\) (for no change in heater input).

(b) Show how these transfer functions are simplified when negligible thermal capacitance of the heating element \(\left(m_{e} C_{e} \rightarrow 0\right)\) is assumed.

Admin
Solution

(a) First write the steady-state equations:

\(\begin{aligned}&0=w C\left(\bar{T}_{i}-\bar{T}\right)+h_{e} A_{e}\left(\bar{T}_{e}-\bar{T}\right) \\&0=\bar{Q}-h_{e} A_{e}\left(\bar{T}_{e}-\bar{T}\right)\end{aligned}\)

Now subtract the steady-state equations from the dynamic equations

\(m C \frac{d T}{d t}=w C\left[\left(T_{i}-\bar{T}_{i}\right)-(T-\bar{T})\right]+h_{e} A_{e}\left[\left(T_{e}-\bar{T}_{e}\right)-(T-\bar{T})\right]\) (1)

\(m_{e} C_{e} \frac{d T_{e}}{d t}=(Q-\bar{Q})-h_{e} A_{e}\left[\left(T_{e}-\bar{T}_{e}\right)-(T-\bar{T})\right]\) (2)

Note that \(d T / d t=d T^{\prime} / d t\) and \(d T_{e} / d t=d T_{e}^{\prime} / d t\). Substitute deviation variables; then multiply (1) by \(1 / wC\) and (2) by \(1 /\left( h _{e} A_{e}\right)\).

\(\frac{m}{w} \frac{d T^{\prime}}{d t}=-\left(T^{\prime}-T_{i}^{\prime}\right)+\frac{h_{e} A_{e}}{w C}\left(T_{e}^{\prime}-T^{\prime}\right)\) (3)

\(\frac{m_{e} C_{e}}{h_{e} A_{e}} \frac{d T_{e}^{\prime}}{d t}=\frac{Q^{\prime}}{h_{e} A_{e}}-\left(T_{e}^{\prime}-T^{\prime}\right)\) (4)

Eliminate one of the output variables, \(T ^{\prime}( s )\) or \(T _{e}^{\prime}(s)\), by solving (4) for it, and substituting into (3). Because \(T_{e}^{\prime}(s)\) is the intermediate variable, remove it. Then rearranging gives:

\(\begin{aligned}&{\left[\frac{m m_{e} C_{e}}{w h_{e} A_{e}} s^{2}+\left(\frac{m_{e} C_{e}}{h_{e} A_{e}}+\frac{m_{e} C_{e}}{w C}+\frac{m}{w}\right) s+1\right] T^{\prime}(s)} \\&=\left(\frac{m_{e} C_{e}}{h_{e} A_{e}} s+1\right) T_{i}^{\prime}(s)+\frac{1}{w C} Q^{\prime}(s)\end{aligned}\)

Because both inputs influence the dynamic behavior of \(T ^{\prime}\) ‘, it is necessary to develop two transfer functions for the model. The effect of \(Q ^{\prime}\) on \(T ^{\prime}\) can be derived by assuming that \(T _{i}\) is constant at its nominal steady-state value, \(\overline{ T }_{i}\). Thus, \(T_{i}^{\prime}=0\) and the previous equation can be rearranged as:

\(\frac{T^{\prime}(s)}{Q^{\prime}(s)}=\frac{1 / w C}{b_{2} s^{2}+b_{1} s+1}=G_{1}(s) \quad\left(T_{i}^{\prime}(s)=0\right)\)

Similarly, the effect of \(T _{i}^{\prime}\) on \(T ^{\prime}\) is obtained by assuming that \(Q =\overline{ Q }\) (that is, \(Q ^{\prime}=0\) ):

\(\frac{T^{\prime}(s)}{T_{i}^{\prime}(s)}=\frac{\frac{m_{e} C_{e}}{h_{e} A_{e}} s+1}{b_{2} s^{2}+b_{1} s+1}=G_{2}(s) \quad\left(Q^{\prime}(s)=0\right)\)

where

\(b_{1} \text { is defined to be } \frac{m_{e} C_{e}}{h_{e} A_{e}}+\frac{m_{e} C_{e}}{w C}+\frac{m}{w}\)

 

\(b_{2} \text { is defined to be } \frac{m m_{e} C_{e}}{w h_{e} A_{e}}\)

By the superposition principle, the effect of simultaneous changes in both inputs is given by

\(T^{\prime}(s)=G_{1}(s) Q^{\prime}(s)+G_{2}(s) T_{i}^{\prime}(s)\)

(b)

The limiting behavior of \(m _{e} C_{e}\) going to zero has \(b _{2}=0\) and \(b_{1}=m / w\) and simplifies the last equation to

\(T^{\prime}(s)=\frac{1 / w C}{\frac{m}{w} s+1} Q^{\prime}(s)+\frac{1}{\frac{m}{w} s+1} T_{i}^{\prime}(s)\)
