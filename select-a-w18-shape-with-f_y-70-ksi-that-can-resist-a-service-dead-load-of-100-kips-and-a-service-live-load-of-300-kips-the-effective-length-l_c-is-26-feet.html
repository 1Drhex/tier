Guest Question.

Select a W18 shape with \(F_y = 70\) ksi that can resist a service dead load of 100 kips and a service live load of 300 kips. The effective length \(L_c\) is 26 feet.

Admin
Solution

\(P_u = 1.2D + 1.6L = 1.2(100) + 1.6(300) = 600 kips\)
Try \(F_{cr} = 47 ksi (an arbitrary choice of two-thirds F_y):\)
Required \(A_g = \frac{P_u}{\phi _{c}F_{cr}}=\frac{600}{0.90(47)}= 14.2 in.^2\)

Try a W18 \(\times 50:\)
\(A_g = 14.7 in.^2 > 14.2 in.^2 (OK)\)
\(\frac{L_c}{r_{min}}= \frac{26 \times 12}{1.65} = 189.1\)
\(F_e = \frac{\pi ^2 E}{(L_c/r)^2} =\frac{\pi ^2(29,000)}{(189.1)^2} = 8.004 ksi\)
\(4.71\sqrt{ \frac{E}{F_y}}= 4.71\sqrt{ \frac{29,000}{70}} = 95.87\)

Since\(\frac{L_c}{r} > 4.71\sqrt{ \frac{E}{F_y}} ,\) AISC Equation E3-3 applies.
\(F_{cr} = 0.877F_e = 0.877(8.004) = 7.020 ksi\)
\(\phi _{c}P_n = \phi _{c}F_{cr}A_g = 0.90(7.020)(14.7) = 92.9 kips < 600 kips\) (N.G.)
(Note: This shape is slender for compression, but since it did not work, there is no need to compute a reduced strength.)
Because the initial estimate of \(F_{cr}\) was so far off, assume a much lower value.
Increase the derived value of 7.020 ksi by one-third of the difference between the initial estimate and 7.020 ksi.

\(Try F_{cr} = 7.020 + \frac{1}{3}(47 – 7.020) = 20.3 ksi:\)
Required \(A_g = \frac{P_u}{\phi _{c}F_{cr}}=\frac{600}{0.90(20.3)} = 32.8 in.^2\)

Try a W18 \(\times 119:\)
\(A_g = 35.1 in.^2 > 32.8 in.^2 (OK)\)
\(\frac{L_c}{r_{min}}= \frac{26 \times 12}{2.69} = 116.0\)
\(F_e = \frac{\pi ^2 E}{(L_c/r)^2} =\frac{\pi ^2(29,000)}{(116.0)^2} = 21.27 ksi\)
Since
\(\frac{L_c}{r} > 4.71\sqrt{ \frac{E}{F_y}}= 95.87,\) AISC Equation E3-3 applies.
\(F_{cr} = 0.877F_e = 0.877(21.27) = 18.65 ksi\)
\(\phi _{c}P_n = \phi _{c}F_{cr}A_g = 0.90(18.65)(35.1) = 589 kips < 600 kip\) (N.G.)
This is very close, so try the next larger size.

Try a W18 \(\times 130:\)
\(A_g = 38.3 in.^2\)

\(\frac{L_c}{r_{min}}= \frac{26 \times 12}{2.70} = 115.6 < 200 (OK)\)
\(F_e = \frac{\pi ^2 E}{(L_c/r)^2} =\frac{\pi ^2(29,000)}{(115.6)^2} = 21.42 ksi\)
\(\frac{L_c}{r} > 4.71\sqrt{ \frac{E}{F_y}}= 95.87,\) AISC Equation E3-3 applies.
\(F_{cr} = 0.877F_e = 0.877(21.42) = 18.79 ksi\)
\(\phi _{c}P_n = \phi _{c}F_{cr}A_g = 0.90(18.79)(38.3) = 648 kips > 600 kips (OK)\)
This shape is not slender (there is no footnote in the dimensions and properties table to indicate that it is), so local buckling does not have to be investigated.

Use a W18 \(\times 130.\)
The ASD solution procedure is essentially the same as for LRFD, and the same trial values of Fcr will be used here.
\(P_a = D + L = 100 + 300 = 400 kips\)

Required \(A_g = \frac{P_a}{0.6F_{cr}}=\frac{400}{0.6(47)}=14.2 in.^2\)

\(A_g = 14.7 in.^2 > 14.2 in.^2 (OK)\)
\(\frac{L_c}{r_{min}}= \frac{26 \times 12}{1.65} = 189.1\)
\(F_e = \frac{\pi ^2 E}{(L_c/r)^2} =\frac{\pi ^2(29,000)}{(189.1)^2} = 8.004 ksi\)
\(4.71\sqrt{ \frac{E}{F_y}} =4.71\sqrt{ \frac{29,000}{70}} =95.87\)

\(\frac{P_n}{\Omega _{c}}= 0.6F_{cr}A_g = 0.6(7.020)(14.7) = 61.9 kips < 400 kips (N.G.)\)
(Note: This shape is slender for compression, but since it did not work, there is no need to compute a reduced strength.)
