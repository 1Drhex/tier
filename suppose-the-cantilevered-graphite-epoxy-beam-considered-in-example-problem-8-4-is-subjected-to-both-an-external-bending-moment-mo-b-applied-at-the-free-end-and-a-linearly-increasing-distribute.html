Guest Question.

Suppose the cantilevered graphite–epoxy beam considered in is subjected to both an external bending moment \(M_o^b\) applied at the free end and a linearly increasing distributed load \(q(x) = (q_o/L)x\), as shown in Figure 8.26. Perform the following:

a. Use the method of superposition to obtain an analytical expression giving the predicted deflection w(z).

b. Plot numerical values of the predicted deflections, assuming the following beam length and loading:

L = 1 m

\(q_0 =200 N/m\) \(M_0^b =-40 N-m\)

 

Admin
Solution

a. Deflections induced by a concentrated moment \(M_o^b\) and distributed load q(x) when acting separately are included in Table B.1.

TABLE B.1
Deflection and Slopes of Cantilever Beams

w(x) = beam deflection (positive downwards)
w’(x) = dw/dx = slope of the deflection curve
\(\overline{IE}_{xx}\)= effective flexural rigidity
\(w(x)=\frac{Px^2}{6\overline{IE}_{xx} }(3L-x)\)
\(w^\prime (x)=\frac{Px}{2\overline{IE}_{xx} }(2L-x)\)
for 0 ≤ x ≤ a:
\(w (x)=\frac{Px^2}{6\overline{IE}_{xx} }(3a-x) w^\prime (x)=\frac{Px}{2\overline{IE}_{xx} }(2a-x)\)
for a ≤ x ≤ L:
\(w (x)=\frac{Pa^2}{6\overline{IE}_{xx} }(3x-a) w^\prime (x)=\frac{Pa^2}{2\overline{IE}_{xx} }\)
\(w (x)=\frac{Px^2}{24\overline{IE}_{xx} }(6L^2-4Lx+x^2)\)
\(w^\prime (x)=\frac{Px}{6\overline{IE}_{xx} }(3L^2-3Lx+x^2)\)
for 0 ≤ x ≤ a:
\(w (x)=\frac{qx^2}{24\overline{IE}_{xx} }(6a^2-4ax+x^2)\)
\(w^\prime (x)=\frac{qx}{6\overline{IE}_{xx} }(3a^2-3ax+x^2)\)
for a ≤ x ≤ L:
\(w (x)=\frac{qa^3}{24\overline{IE}_{xx} }(4x-a) w^\prime (x)=\frac{qa^3}{6\overline{IE}_{xx} }\)
for 0 ≤ x ≤ a:
\(w (x)=\frac{qbx^2}{12\overline{IE}_{xx} }(3L+3a-2x) w^\prime (x)=\frac{qbx}{2\overline{IE}_{xx} }(L+a-x) \)
for a ≤ x ≤ L:
\(w (x)=\frac{q}{24\overline{IE}_{xx} }(x^4-4Lx^3+6L^2x^2-4a^3x+a^4)\)
\(w^\prime (x)=\frac{q}{6\overline{IE}_{xx} }(x^3-3Lx^2+3L^2x-a^3)\)
\(w (x)=\frac{q_0x^2}{120L\overline{IE}_{xx} }(20L^3-10L^2x+x^3)\)
\(w^\prime (x)=\frac{q_0}{24L\overline{IE}_{xx} }(8L^3-6L^2x+x^3)\)
\(w (x)=\frac{q_0x^2}{120L\overline{IE}_{xx} }(10L^3-10L^2x+5Lx^2-x^3)\)
\(w^\prime (x)=\frac{q_0x}{24L\overline{IE}_{xx} }(4L^3-6L^2x+4Lx^2-x^3)\)
\(w(x)=\frac{-M_0^bx^2}{2\overline{IE}_{xx} } w^\prime (x)=\frac{-M_0^bx}{\overline{IE_{xx}} } \)
for 0 ≤ x ≤ a: for a ≤ x ≤ L:
\(w(x)=\frac{-M_0^bx^2}{\overline{2IE}_{xx} } \) \(w(x)=\frac{-M_0^ba}{2\overline{IE}_{xx} }(2x-a)\)
\(w^\prime (x)=\frac{-M_0^bx}{2\overline{IE}_{xx} }\) \(w^\prime (x)=\frac{-M_0^ba}{\overline{IE}_{xx} }\)

The deflections due to \(M_o^b\) acting alone are given by

\(w(x)|_{M_0^b}=\frac{M_0^bx^2}{2(\overline{IE}_{xx} )}\)

The deflections due to the distributed load are given by

\(w(x)|_{q(x)}=\frac{q_0x^2}{120L(\overline{IE}_{xx} )}(20L^3-10L^2x+x^3)\)

Applying the principle of superposition, the beam deflections caused by both load components acting simultaneously is simply the sum of these two solutions:

\(w(x)= w(x)|_{M_0^b}+w(x)|_{q(x)}=\left\lgroup\frac{M_0^bx^2}{2(\overline{IE}_{xx} )} \right\rgroup+\left\lgroup\frac{q_0x^2}{120L(\overline{IE}_{xx} )}(20L^3-10L^2x+x^3) \right\rgroup\)

b. Note that the specified numerical value of the concentrated bending moment is algebraically negative:

\(M_0^b=-40 N − m\)

Hence, the concentrated moment is acting in a sense opposite to that shown in Figure 8.26. The distributed load q(x) will tend to deflect the beam downwards (i.e., in the positive z-direction), whereas \(M_o^b\) will tend to deflect the beam upwards (in the negative z-direction).

Using the values specified for beam dimensions, material properties, and loads, deflections are given (in millimeters) by

\(w(x) = (0.1977 x^2 )(20 − 10x + x^3 ) − 2.372 x^2 \)

which can be rearranged as

\(w(x) = 1.582 x^2 − 1.977 x^3 + 0.1977 x^5\)

A plot of predicted deflections is shown in Figure 8.27. As before both the deflection and slope are zero at the left end, as required by the clamped boundary condition. A locally maximum positive (downwards) deflection of 0.160 mm occurs at an axial position of x = 0.563 m. The globally maximum deflection occurs at the free end, where a negative (upward) deflection of
0.198 mm occurs.
