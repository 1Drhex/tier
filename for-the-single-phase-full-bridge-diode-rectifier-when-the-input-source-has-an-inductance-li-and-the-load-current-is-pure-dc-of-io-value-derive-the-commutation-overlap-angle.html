Guest Question.

For the single-phase full-bridge diode rectifier, when the input source has an inductance \(L _{ i }\) and the load current is pure dc of \(\overline{ I }_{ o }\) value, derive the commutation overlap angle.

Admin
Solution

Solution

The power circuit for derivation of the commutation angle is shown in Fig. 4.29. Also, Fig. 4.30 shows the key circuit waveforms during commutation. During commutation the following equation holds:

 

\(v _{ i }- L _{ i } \frac{ di _{ i }}{ dt }=0 \quad \text { for } \quad \pi<\omega t \leq \pi+\mu\)

 

Multiplying the above equation by dÏ‰t:

 

\(\begin{aligned}& v _{ i } d \omega t = L _{ i } \frac{ di _{ i }}{ dt } d \omega t \text { for } \pi<\omega t \leq \pi+\mu \text { or } \quad v _{ i } d \omega t =\omega L _{ i } d i _{ i } \\&\text { for } \pi<\omega t \leq \pi+\mu\end{aligned}\)

 

By intergrading both parts:

 

\(\int_{\pi}^{\pi+\mu} v _{ i } d \omega t =\int_{\overline{ I }_{ o }}^{-\overline{ I }_{ o }} \omega L _{ i } di _{ i } \text { or } \int_{\pi}^{\pi+\mu} \sqrt{2} \widetilde{ V }_{ i } \sin \omega \operatorname{td}(\omega t )=\int_{\overline{ I }_{ o }}^{-\overline{ I _{ o }}} \omega L _{ i } di _{ i }\)

 

\(\sqrt{2} \widetilde{ V }_{ i }(\cos \pi-\cos (\pi+\mu))=-2 \omega L _{ i } \overline{ I }_{ o } \quad \text { or } \quad \sqrt{2} \widetilde{ V }_{ i }(-1+\cos \mu)=-2 \omega L _{ i } \overline{ I }_{ o }\)

or

\(\mu=\cos ^{-1}\left(1-\frac{2 \omega L _{ i } \overline{ I }_{ o }}{\sqrt{2} \widetilde{ V }_{ i }}\right)\)

 

The time of one commutation or overlap interval is given by:

\(\Delta t =\text { commutation time }=\frac{\mu}{\omega}=\frac{1}{\omega} \cos ^{-1}\left(1-\frac{2 \omega L _{ i } \overline{ I }_{ o }}{\sqrt{2} \widetilde{ V }_{ i }}\right)\)

 

As can be seen from Fig. 4.30, the output voltage for each overlap interval is reduced by an amount equal to the area A. Therefore, each overlap interval is reducing the average output voltage of the rectifier by:

 

\(\begin{aligned}\overline{ V }_{\mu} &=\frac{ A }{ T }=\frac{\int_{0}^{\mu} v _{ i } d (\omega t )}{2 \pi} \\&=\frac{\int_{0}^{\mu} \sqrt{2} \widetilde{ V }_{ i } \sin \omega td (\omega t )}{2 \pi} \\&=\frac{\sqrt{2} \widetilde{ V }_{ i }}{2 \pi}(1-\cos \mu)\end{aligned}\)

 

Since in the single-phase full-bridge diode rectifier there are two overlap intervals per cycle, the average output voltage is reduced by:

 

\(\overline{ V }_{ o (\text { loss })}=2 \overline{ V }_{\mu}=\frac{\sqrt{2} \widetilde{ V }_{ i }}{\pi}(1-\cos \mu)\)
