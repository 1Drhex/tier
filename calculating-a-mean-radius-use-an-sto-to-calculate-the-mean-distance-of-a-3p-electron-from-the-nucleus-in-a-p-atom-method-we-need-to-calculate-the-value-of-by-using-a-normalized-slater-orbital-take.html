Calculating a mean radius
Use an STO to calculate the mean distance of a \(3p\) electron from the nucleus in a \(P\) atom.
Method We need to calculate the value of \(\left\langle r\right\rangle \) by using a normalized Slater orbital. Take \(Z_{eff}\) from Table 7.3. The spherical harmonics are normalized to 1; for other integrations, use

\(\int_{0}^{\infty }{x^{n}e^{-ax} dx} =\frac{n!}{a^{n+1} } \)

Solution

For a \(P3p\) orbital we take \(n_{eff}=3\) and \(Z_{eff}=4.8864\), then eqn 7.41 gives

\(\psi _{nlm_{l} } (r,\theta ,\varphi )=Nr^{n_{eff} -1} e^{-Z_{eff}\rho /n_{eff} } Y_{lm_{l} } (\theta ,\varphi )\) (7.41)

\(\psi _{3p} =Nr^{2} e^{-4.8864\rho /3 } Y_{1m_{l} } (\theta ,\varphi )=Nr^{2} e^{-1.6288r /a_{0} } Y_{1m_{l} } (\theta ,\varphi )\)

To determine N we use

\(\int{\psi ^{2}_{3p} d\tau } =N^{2} \int_{0}^{\infty }{r^{6}e^{-3.2576r/a_{0} } dr} \int_{0}^{\pi }{\int_{0}^{2\pi }{\left|Y_{1m_{l} }(\theta ,\varphi ) \right|^{2}\sin \theta d\theta }d\varphi } =N^{2} \int_{0}^{\infty }{r^{6}e^{-3.2576r/a_{0} }dr } \)

 

\(=N^{2} \frac{6!a^{7}_{0} }{3.2576^{7} } \)

It follows that

\(N=\left(\frac{3.2576^{7} }{6!a^{7}_{0} } \right) ^{1/2} =\frac{2.3253}{a^{7/2}_{0} }\)

To calculate the expectation value of r, we need to evaluate only the radial integral:

\(\left\langle r\right\rangle =N^{2} \int_{0}^{\infty }{r^{7}e^{-3.2576r/a_{0} } dr} =\frac{3.2576^{7} }{6!a^{7}_{0} } \times \frac{7!a^{8}_{0} }{3.2576^{8} } =\frac{7a_{0} }{3.2576} =2.1488a_{0} \)
