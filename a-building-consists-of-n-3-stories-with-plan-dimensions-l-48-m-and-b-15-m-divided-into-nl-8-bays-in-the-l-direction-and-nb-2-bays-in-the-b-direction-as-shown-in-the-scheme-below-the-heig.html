Guest Question.

A building consists of n = 3 stories with plan dimensions L = 48 m and B = 15 m; divided into \( N_{L} = 8 \) bays in the L direction and \( N_{B} = 2 \) bays in the B direction, as shown in the scheme below. The height of each story is \( h = 3.2 m \). The floors of the building are \( d_{floor} = 250 mm\) thick. Shear walls, intended to resist overturning, are located at both ends of the building and are \( t = 300 mm\) thick by \( b_{w} = 4 m \) wide on plan. A water tank, \( d_{tank} = 2 m \) deep by \( l_{tank} = 5 m \) long by \( b_{tank} = 5 m \) wide, sits over the shear wall at one end of the building. The shear walls are supported by strip foundations of length \( l_{fdn} = 6.5 m \), width \( b_{fdn} = 2 m \) and \( d_{fdn} = 1.5 m \). The following characteristic imposed/wind actions act on the building:

• roof loading \( q_{rf ,k} = 0.6 kPa \)
• office floor loading \( q_{off ,k} = 2.5 kPa \)
• partition loading \( q_{par ,k} = 0.8 kPa \)
• wind (horizontal) \( q_{w,k} = 1.15 kPa \)

The characteristic unit weight of reinforced concrete is \( y_{c,k} = 25 kN/m^{3} \) and of water \( y_{w,k} = 10 kN/m^{3} \).
Required: the way in which actions should be combined according to Eurocode (EN 1990), in a way that is suitable for geotechnical design of foundations.

Admin
Solution

Geometry

The total plan area of building is

\( A_{total} = L \times B = 48 \times 15 = 720 m^{2} \)

 

The tributary area above the stability wall is

\( A = \left ( \frac{B + b_{w}}{2} \right ) \times \frac{1}{2} \left ( \frac{L}{N_{L}} \right ) = \left ( \frac{15 + 4}{2} \right ) \times \frac{1}{2} \left ( \frac{48}{8} \right ) = 28.5 m^{2} \)

 

Characteristic permanent actions
Self-weight of slabs:

• Floor \( g_{fl,Gk} = y_{c, k} \times d_{floor} = 25 \times 0.25 = 6.25 kPa \)

• Screed on roof \( g_{scr,Gk} = 1.5 kPa \)

• Raised floor \( g_{r, −fl,Gk} = 0.5 kPa \) ( removable )

Self-weight of water tank on roof-only half total weight is carried by the core wall:

\( W_{tank,Gk} = \frac{1}{2} y_{w, k} \times d_{tank} \times l_{tank} \times b_{tank} = \frac{1}{2} \times 10 \times 2 \times 5 \times 5 \)

 

\( = 250 kN \) ( removable )

 

Self-weight of core wall:

\( W_{wall,Gk} = y_{c, k} t_{w} b_{w} \left ( nh \right ) = 25 \times 0.3 \times 4 \times 3 \times 3.2 = 288 kN \)

 

Self-weight of strip foundation:

\( W_{fdn,Gk} = y_{c, k} d_{fdn} b_{fdn} l_{fdn} = 25 \times 1.5 \times 2 \times 6.5 = 488 kN\)

 

Total self-weight of non-removable members (normal to ground):

\( N_{Gk1} = A \left ( n g_{fl,Gk} + g_{scr,Gk} \right ) + W_{wall,Gk} + W_{fdn,Gk} \)

 

\( = 28.5 \left ( 3 \times 6.25 + 1.5 \right ) + 288 + 488 = 1353 kN \)

 

Total self-weight of removable members (normal to ground):

\( N_{Gk2} = A \left ( n – 1 \right ) g_{r, −fl,Gk} + W_{tank, Gk} \)

 

\( = 28.5 \left ( 3 – 1 \right ) \times 0.5 + 250 = 279 kN\)

 

Characteristic variable actions
Imposed actions (normal to ground):

• On roof

\( N_{rf ,Qk} = q_{rf, k} A = 0.6 \times 28.5 = 171 kN \)

• On floor

\( N_{fl ,Qk} = \left ( n – 1 \right )\left ( q_{off ,k} + q_{par,k}\right ) A \)

 

\( = \left ( 3 – 1 \right )\left ( 2.5 + 0.8 \right ) \times 28.5 = 188.1 kN \)

 

Wind actions (horizontal direction):

• On roof

\( Q_{w, rf , Qk} = q_{w, k} \left ( \frac{h}{2} \right )\left ( \frac{L}{2} \right ) = 1.15 \left ( \frac{3.2}{2} \right )\left ( \frac{48}{2} \right ) = 44.2 kN \)

 

• On each floor

\( Q_{w, fl , Qk} = q_{w, k} \left ( h \right ) \left ( \frac{L}{2} \right ) \)

 

\( = 1.15 \left ( 3.2 \right ) \left ( \frac{48}{2} \right ) = 88.3 kN \)

 

Total wind action (normal to ground):

\( N_{w, Qk} = 0 kN \)

 

Moment effect of wind action (on ground):

• First floor

\( M_{w, Qk1} = Q_{w, fl, Qk} \left [ \left ( n – 2 \right ) h + d_{fdn} \right ] \)

 

\( = 88.3 \left [ \left ( 3 – 2 \right ) \left ( 3.2 \right ) + 1.5 \right ] = 415 kN.m \)

• Second floor

\( M_{w, Qk2} = Q_{w, fl, Qk} \left [ \left ( n – 1 \right ) h + d_{fdn} \right ] \)

 

\( = 88.3 \left [ \left ( 3 – 1 \right ) \left ( 3.2 \right ) + 1.5 \right ] = 698 kN.m \)

• Roof

\( M_{w, Qk3} = Q_{w, rf, Qk} \left [ \left ( n – 0 \right ) h + d_{fdn} \right ] \)

 

\( = 44.2 \left [ \left ( 3 – 0 \right )\left ( 3.2 \right ) + 1.5 \right ] = 490 kN.m \)

• total

\( M_{w, Qk} = \sum \) Moment effect of wind action

 

\( = 415 + 698 + 490 = 1603 kN.m \)

 

Combinations of actions for persistent and transient design situations – ULS verification

Combination 1 – wind as leading variable action, vertical actions unfavourable, partial factors from Set B in Table 6.6 or Section 6.3.4(G).

• Partial factors:

• On permanent actions \( γ_{G} = 1.35\)

• On variable actions (wind) \( γ_{Q,w} = 1.5\)

• On variable actions (imposed loads) \( γ_{Q,i} = 1.5\)

 

• Combination factors (Table 6.3):

• For wind (leading variable action) \( \Psi _{w} = 1.0 \)

• For imposed load in office areas (Category B) \( \Psi _{fl} = \Psi _{0, i,B} = 0.7\)

• For imposed load on roof (Category H) \( \Psi _{rf} = \Psi _{0, i,H} = 0.0\) (also see EN 1991-1-1: Clause 3.3.2)

 

• Design value of normal action effect:

\( N_{Ed} = y_{G} \left ( N_{Gk1} + N_{Gk2} \right ) + y_{Q,w} \Psi_{w} N_{w, Qk} + y_{Q, I} \left ( \Psi_{fl} N_{fl, Qk} + \Psi_{rf} N_{rf, Qk} \right ) \)

 

\( = 1.35 \left ( 1353 + 279 \right ) + 0 + 1.5 \left ( 0.7 \times 188.1 + 0^{*} \right ) = 2401 kN \)

 

∗ See EN 1991-1-1 §3.3.2(1)

• Design value of moment effect:

\( M_{Ed} = y_{Q,w} \Psi_{w}M_{w, Qk} = 1.5 \times 1 \times 1603 = 2405 kN.m \)

 

• Maximum bearing pressure \( \left ( P_{max, Ed} \right ) \) on underside of strip foundation:

• Check eccentricity \( e_{l} \):

\( e_{l} = \left ( M_{Ed} / N_{Ed} \right ) = \left ( 2405 / 2401 \right ) = 1.00 m < \frac{l_{fdn}}{6} \) ( OK.)

 

\( P_{max, Ed} = \frac{N_{Ed}}{A}\left ( 1 + \frac{6 e_{l}}{l_{fdn}} \right ) \) Equation or

 

\( P_{max, Ed} = \left ( \frac{N_{Ed}}{b_{fdn} l_{fdn}} \right ) + \left ( \frac{6 M_{Ed}}{b_{fdn} l_{fdn}^{2}} \right ) \)

 

\( = \left ( \frac{2401}{2 \times 6.5} \right ) + \left ( \frac{6 \times 2405}{2 \times 6.5^{2}} \right ) = 184.69 + 170.77 = 355.5 kPa \)

 

Combination 2 – wind as leading variable action, vertical actions favourable \( \left ( y_{G} = 1 and y_{Q, i} = 0 \right ) \) , partial factors from Set B in Table 6.6 or Subsection 6.3.4(G).

• Design value of normal action effect:

\( N_{Ed} = y_{G, fav} \left ( N_{Gk1} + N_{Gk2} \right ) = \left ( 1 \right ) \left ( 1353 + 279 \right ) = 1632 kN \)

 

• Design value of moment effect:

\( M_{Ed} = y_{Q,w} \Psi_{w} M_{w, Qk} = 1.5 \times 1 \times 1603 = 2405 kN.m \)

 

• Maximum bearing pressure \( \left ( P_{max,Ed} \right ) \) on underside of strip foundation:

• Check eccentricity \( e_{l} \):

\( e_{l} = \left ( M_{Ed} / N_{Ed} \right ) = \left ( 2405 / 1632 \right ) = 1.47 m > \frac{l_{fdn}}{6} \) ( Not OK.)

Therefore, Equation ( \( q = \frac{R}{BL} \left ( 1 \pm \frac{6 e_{x}}{B} \pm \frac{6 e_{y}}{L} \right ) \) ) is not applicable.
Since tension cannot occur between ground and the underside of the footing, the triangular bearing pressure distribution shown below may be considered (see Section 2.6.2 of Chapter 2).

\( a = \frac{l_{fdn}}{2} – e_{l} = \left ( \frac{1}{2} \right ) \left ( l_{fdn} – 2 e_{l} \right ) \)

 

\( N_{Ed} = \left ( \frac{P_{max, Ed} \times 3 a}{2} \right ) \left ( b_{fdn} \right ) = \left ( P_{max, Ed} \right ) \left ( \frac{3}{4} \right )\left ( b_{fdn} \right ) \left ( l_{fdn} – 2 e_{l} \right ) \)

 

\( P_{max, Ed} = \frac{4 N_{Ed}}{3 b_{fdn} \left ( l_{fdn} – 2 e_{l} \right )} = \frac{4 \times 1632}{3 \times 2 \left ( 6.5 – 2 \times 1.47 \right )} = 306 kPa \)

 

However, TCC (2006) recommends a rectangular ULS pressure distribution diagram to be considered and not triangular, as shown below.

Accordingly, the maximum bearing pressure \( \left (P_{max, Ed} \right ) \) on the underside of the strip foundation is

\( P_{max, Ed} = \frac{N_{Ed}}{b_{fdn} \times 2 a} = \frac{1632}{2 \times 2 \left ( \frac{1}{2} \right ) \left ( l_{fdn} – 2 e_{l} \right )} = \frac{1632}{2 \left ( 6.5 – 2 \times 1.47 \right )} = 229 kPa \)

 

Combination 3 – imposed loads as leading variable action, vertical actions unfavourable, partial factors from Set B in Table 6.6 or Section 6.3.4(G).

• Partial factors:

• On permanent actions \( y_{G} = 1.35 \)

• On variable actions (imposed loads) \( y_{Q, i} = 1.5 \)

• On variable actions (imposed loads) (wind) \( y_{Q, w} = 1.5 \)

 

• Combination factors (Table 6.3):

• For imposed load in office areas (Category B) \( \Psi_{fl} = 1.0 \)

• For imposed load on roof (Category H) \( \Psi_{rf} = 1.0 \)

• For wind \( \Psi_{w} = \Psi_{0, w} = 0.6 \)

 

• Design value of normal action effect:

\( N_{Ed} = y_{G} \left ( N_{Gk1} + N_{Gk2} \right ) + y_{Q, i} \left ( \Psi_{fl} N_{fl, Qk} + \Psi_{rf} N_{rf, Qk} \right ) + y_{Q,w} \Psi_{0,w} N_{w, Qk}\)

 

\( = 1.35 \left ( 1353 + 279 \right ) + 1.5 \left ( 1 \times 188.1 + 1 \times 17.1 \right ) + 0 = 2511 kN \)

 

• Design value of moment effect:

\( M_{Ed} = y_{Q,w} \Psi_{0,w} M_{w, Qk} = 1.5 \times 0.6 \times 1603 = 1443 kN.m \)

 

• Maximum bearing pressure \( \left (P_{max, Ed} \right ) \) on underside of strip foundation:

• Check eccentricity \( e_{l} \):

\( e_{l} = \left ( M_{Ed} / N_{Ed} \right ) = \left ( 1443 / 2511 \right ) = 0.57 m < \frac{l_{fdn}}{6} \) ( OK.)

 

\( P_{max, Ed} = \left ( \frac{N_{Ed}}{b_{fdn} l_{fdn}} \right ) + \left ( \frac{6 M_{Ed}}{b_{fdn} l_{fdn}^{2}} \right ) \)

 

\( = \left ( \frac{2511}{2 \times 6.5} \right ) + \left ( \frac{6 \times 1443}{2 \times 6.5^{2}} \right ) = 193.15 + 102.46 = 296 kPa \)

 

Combination 4 – wind as leading variable action, vertical actions unfavourable, partial factors from Set C in Table 6.6 or Section 6.3.4(G).

• Partial factors:

• On permanent actions \( y_{G} = 1.0 \)

• On variable actions (wind) \( y_{Q, w} = 1.3 \)

• On variable actions (imposed loads) \( y_{Q, i} = 1.3 \)

 

• Combination factors (Table 6.3):

• For wind (leading variable action) \( \Psi_{w} = 1.0 \)

• For imposed load in office areas (Category B) \( \Psi_{fl} = \Psi_{0, i,B} = 0.7 \)

• For imposed load on roof (Category H) \( \Psi_{rf} = \Psi_{0, i,H} = 0.0 \) (also see EN 1991-1-1: Clause 3.3.2)

 

• Design value of normal action effect:

\( N_{Ed} = y_{G} \left ( N_{Gk1} + N_{Gk2} \right ) + y_{Q, w} \Psi_{w} N_{w, Qk} + y_{Q, I} \left ( \Psi_{fl} N_{fl, Qk} + \Psi_{rf} N_{rf, Qk} \right )\)

 

\( = 1.0 \left ( 1353 + 279 \right ) + 0 + 1.3 \left ( 0.7 \times 188.1 + 0^{*} \right ) = 1803 kN \)

 

∗ See EN 1991-1-1 §3.3.2(1)

 

• Design value of moment effect:

\( M_{Ed} = y_{Q,w} \Psi_{w} M_{w, Qk} = 1.3 \times 1 \times 1603 = 2084 kN.m \)

 

• Maximum bearing pressure \( \left (P_{max, Ed} \right ) \) on underside of strip foundation.

• Check eccentricity \( e_{l} \):

\( e_{l} = \left ( M_{Ed} / N_{Ed} \right ) = \left ( 2084 / 1803 \right ) = 1.16 m < \frac{l_{fdn}}{6} \) ( OK.)

 

\( P_{max, Ed} = \left ( \frac{N_{Ed}}{b_{fdn} l_{fdn}} \right ) + \left ( \frac{6 M_{Ed}}{b_{fdn} l_{fdn}^{2}} \right ) \)

 

\( = \left ( \frac{1803}{2 \times 6.5} \right ) + \left ( \frac{6 \times 2084}{2 \times 6.5^{2}} \right ) = 138.69 + 147.98 = 286.67 kPa \)

 

Combination 5– wind as leading variable action, vertical actions favourable \( \left ( y_{G} = 1 and y_{Q, i} = 0 \right ) \), partial factors from Set C in Table 6.6 or Section 6.3.4(G).

• Design value of normal action effect:

\( N_{Ed} = y_{G, fav} \left ( N_{Gk1} + N_{Gk2} \right ) = \left ( 1 \right ) \left ( 1353 + 279 \right ) = 1632 kN \)

 

• Design value of moment effect:

\( M_{Ed} = y_{Q,w} \Psi_{w} M_{w, Qk} = 1.3 \times 1 \times 1603 = 2084 kN.m \)

 

• Maximum bearing pressure \( \left (P_{max, Ed} \right ) \) on underside of strip foundation:

• Check eccentricity \( e_{l} \):

\( e_{l} = \left ( M_{Ed} / N_{Ed} \right ) = \left ( 2084 / 1632 \right ) = 1.27 m < \frac{l_{fdn}}{6} \) ( OK.)

 

\( P_{max, Ed} = \left ( \frac{N_{Ed}}{b_{fdn} l_{fdn}} \right ) + \left ( \frac{6 M_{Ed}}{b_{fdn} l_{fdn}^{2}} \right ) \)

 

\( = \left ( \frac{1632}{2 \times 6.5} \right ) + \left ( \frac{6 \times 2084}{2 \times 6.5^{2}} \right ) = 125.54 + 147.98 = 273.52 kPa \)

 

Combination 6 – imposed loads as leading variable action, vertical actions unfavourable, partial factors from Set C in Table 6.6 or Section 6.3.4(G).

• Partial factors:

• On permanent actions \( y_{G} = 1.0 \)

• On variable actions (imposed loads) \( y_{Q, i} = 1.3 \)

• On variable actions (imposed loads) (wind) \( y_{Q, w} = 1.3 \)

 

• Combination factors (Table 6.3):

• For imposed load in office areas (Category B) \( \Psi_{fl} = 1.0\)

• For imposed load on roof (Category H) \( \Psi_{rf} = 1.0\)

• For wind \( \Psi_{w} = \Psi_{0, w} = 0.6\)

 

• Design value of normal action effect:

\( N_{Ed} = y_{G} \left ( N_{Gk1} + N_{Gk2} \right ) + y_{Q, i} \left ( \Psi_{fl} N_{fl, Qk} + \Psi_{rf} N_{rf, Qk} \right ) + y_{Q, w} \Psi_{0,w} N_{w, Qk} \)

 

\( = 1.0 \left ( 1353 + 279 \right ) + 1.3 \left ( 1 \times 188.1 + 1 \times 17.1 \right ) + 0 = 1899 kN \)

 

• Design value of moment effect:

\( M_{Ed} = y_{Q, w} \Psi_{0,w} N_{w, Qk} = 1.3 \times 0.6 \times 1603 = 1250 kN.m \)

• Check eccentricity \( e_{l} \):

\( e_{l} = \left ( M_{Ed} / N_{Ed} \right ) = \left ( 1250 / 1899 \right ) = 0.66 m < \frac{l_{fdn}}{6} \) ( OK.)

\( = \left ( \frac{1899}{2 \times 6.5} \right ) + \left ( \frac{6 \times 1250}{2 \times 6.5^{2}} \right ) = 146.08 + 88.76 = 234.84 kPa \)
