Guest Question.

A 4″\(\phi \) standard weight steel pipe (\(F_{y} \) = 50 ksi) supports a roof framing system as shown. The timber beam-to-column connection is considered to be a pin, while the base of the column is rigidly embedded into the concrete. If the load from the roof is 35 k, is the column adequate?

Admin
Solution

Four-inch”-diameter standard weight pipe:

A = 3.17 in.²

r = 1.51 in.

Although the theoretical K is 0.7 for the support condition shown, the AISC-recommended value for use in design is

K = 0.80

\(K \ell / r=\frac{0.80 \times\left(18^{\prime} \times 12 in . / ft .\right)}{1.51^{\prime \prime}}=114.43\)

Using Table 9.2,

\(F_{ a }\)=11.4 ksi

\(P_{ a }=11.4 k / in .^{2} \times 3.17 in .^{2}=36.14 k >35 k\)

The column is adequate.
