Guest Question.

Consider beam 6 of Table A–9, which is a simply supported beam having a concentrated load F not in the center. Develop the deflection equations using singularity functions.

Admin
Solution

First, write the load intensity equation from the free-body diagram,

 

\(q=R_{1}\langle x\rangle^{-1}-F\langle x-a\rangle^{-1}+R_{2}\langle x-l\rangle^{-1}\) (1)

 

Integrating Eq. (1) twice results in

 

\(V=R_{1}\langle x\rangle^{0}-F\langle x-a\rangle^{0}+R_{2}\langle x-l\rangle^{0}\) (2)

 

\(M=R_{1}\langle x\rangle^{1}-F\langle x-a\rangle^{1}+R_{2}\langle x-l\rangle^{1}\) (3)

 

Recall that as long as the q equation is complete, integration constants are unnecessary for V and M; therefore, they are not included up to this point. From statics, setting V = M = 0 for x slightly greater than l yields \(R_{1}=F b / l \text { and } R_{2}=F a / l\). Thus Eq. (3) becomes

 

\(M=\frac{F b}{l}\langle x\rangle^{1}-F\langle x-a\rangle^{1}+\frac{F a}{l}\langle x-l\rangle^{1}\)

 

Integrating Eqs. (4–12) and (4–13) as indefinite integrals gives

 

\(\frac{M}{E I}=\frac{d^{2} y}{d x^{2}}\) (4–12)

 

\(\theta=\frac{d y}{d x}\) (4–13)

 

\(E I \frac{d y}{d x}=\frac{F b}{2 l}\langle x\rangle^{2}-\frac{F}{2}\langle x-a\rangle^{2}+\frac{F a}{2 l}\langle x-l\rangle^{2}+C_{1}\)

 

\(\text { EI } y=\frac{F b}{6 l}\langle x\rangle^{3}-\frac{F}{6}\langle x-a\rangle^{3}+\frac{F a}{6 l}\langle x-l\rangle^{3}+C_{1} x+C_{2}\)

 

Note that the first singularity term in both equations always exists, so \(\langle x\rangle^{2}=x^{2}\) and \(\langle x\rangle^{3}=x^{3}\). Also, the last singularity term in both equations does not exist until x = l, where it is zero, and since there is no beam for \(x>1\) we can drop the last term. Thus

 

\(E I \frac{d y}{d x}=\frac{F b}{2 l} x^{2}-\frac{F}{2}\langle x-a\rangle^{2}+C_{1}\) (4)

 

\(\text { EI } y=\frac{F b}{6 l} x^{3}-\frac{F}{6}\langle x-a\rangle^{3}+C_{1} x+C_{2}\) (5)

 

The constants of integration \(C_{1} \text { and } C_{2}\) are evaluated by using the two boundary conditions y = 0 at x = 0 and y = 0 at x = l. The first condition, substituted into Eq. (5), gives \(C_{2}=0\) (recall that \(\langle 0-a\rangle^{3}=0\)). The second condition, substituted into Eq. (5), yields

 

\(0=\frac{F b}{6 l} l^{3}-\frac{F}{6}(l-a)^{3}+C_{1} l=\frac{F b l^{2}}{6}-\frac{F b^{3}}{6}+C_{1} l\)

 

Solving for \(C_{1}\) gives

 

\(C_{1}=-\frac{F b}{6 l}\left(l^{2}-b^{2}\right)\)

 

Finally, substituting \(C_{1} \text { and } C_{2}\) in Eq. (5) and simplifying produces

 

\(y=\frac{F}{6 E I l}\left[b x\left(x^{2}+b^{2}-l^{2}\right)-l\langle x-a\rangle^{3}\right]\) (6)

 

Comparing Eq. (6) with the two deflection equations for beam 6 in Table A–9, we note that the use of singularity functions enables us to express the deflection equation with a single discontinuous equation.
