Guest Question.

Designing a depletion-load NMOS inverter Design a depletion-load NMOS inverter, as shown in Fig. 15.16(a), to obtain a noise margin of \( NM _{ H } \geq 2.7 V \). The threshold voltages are \( V_{ tL }=-2.5 V , V_{ tD }=1 V \), and \( K_{ L }=20 \mu A / V ^{2} \). The supply voltage is \( V_{ DD }=5 V \text {. Assume } V_{ OH }=V_{ DD }=5 V \), load capacitance \( C_{ L }=0.5 pF \), and frequency \( f_{ clk }=5 MHz \).

(a) Find the design parameter \( K_{ R }=K_{ D } / K_{ L } \), neglecting the body effect.
(b) Calculate \( NM _{ L } \) if the body effect is neglected.
(c) Calculate \( NM _{ H } \) if the body effect is included. Assume \( \gamma=0.5 \sqrt{ V } \text { and } 2 \phi_{ f }=0.6 V \).
(d) Calculate the low-to-high propagation time\( t_{ pLH } \).
(e) Calculate the high-to-low propagation time \(t_{ pHL }\).
(f) Calculate the delay-power product (DP).
(g) Use PSpice/SPICE to verify your results.

Admin
Solution

(a) \( NM _{ H }=V_{ OH }-V_{ IH } \geq 2.7 V \text {. Thus, for } V_{ OH }=V_{ DD }=5 V.\)

 

\( V_{ IH }=V_{ OH }- NM _{ H } \leq 2.3 V \)

 

For \( V_{ IH } \leq 2.3 V , \) Eq. (15.56) yields

\( V_{ IH }=\frac{2\left|V_{ tL }\right|}{\sqrt{3 K_{R}}}+V_{ tD } \) (15.56)

 

\( K_{ R }=\frac{\left(2 V_{ tL }\right)^{2}}{3\left(V_{ IH }-V_{ tD }\right)^{2}} \) (15.59)

 

which gives \( K_{ R }=4.93 \). Since a higher value gives a lower \( V_{ IH } \text {, we choose } K_{ R }=5 \text {. For this value of } K_{ R } \), Eq. (15.56) gives \( V_{ IH }=2.29 V \text {, and Eq. (15.57) gives } v_{ O }=0.65 V \).

(b) For \( K_{ D }=K_{ L } \times K_{ R }=100 \mu A / V ^{2} \) , Eq. (15.49) gives

\(K_{ L }\left[-2 V_{ tL }\left(V_{ DD }-v_{ O }\right)-\left(V_{ DD }-v_{ O }\right)^{2}\right]=K_{ D }\left(v_{ I }-V_{ tD }\right)^{2}\) (15.49)

 

\(20 \mu \times\left[(-2) \times(-2.5) \times\left(5-v_{ O }\right)-\left(5-v_{ O }\right)^{2}\right]=20 \mu \times 5\left(v_{ I }-1\right)^{2}\) (15.60)

Setting \( d v_{ O } / d v_{ I }=-1 \), we get

\( v_{ I }=0.2 v_{ O }+0.5 \)

Solving these two equations, we get \( V_{ IL }=1.46 V \text { and } v_{ O }=4.78 V \text {. For } v_{ I }=V_{ OH } \text { and } v_{ O }=V_{ OL } \), Eq. (15.55) yields

\( K_{ L }\left(-V_{ tL }\right)^{2}=K_{ D }\left[2\left(v_{ I }-V_{ tD }\right) v_{ O }-v_{ O }^{2}\right] \) (15.55)

 

\( K_{ D }\left[2\left(V_{ OH }-V_{ tD }\right) V_{ OL }-V_{ O L }^{2}\right]=K_{ L } V_{ tL }^{2} \) (15.61)

 

which, for \( V_{ OH }=5 V \text { and } K_{ R }=5 \text {, gives } V_{ OL }=0.16 V \). Thus,

\( NM _{ L }=V_{ IL }-V_{ OL }=1.46-0.16=1.3 V \)

(c) With the body effect, at \( v_{ O }=0.65 V \), Eq. (15.58) gives

\( V_{ tL }=V_{ tL 0}+\gamma\left|\sqrt{v_{ O }+2 \phi_{ f }}-\sqrt{2 \phi_{ f }}\right| \) (15.58)

 

\(\begin{aligned}V_{ tL } &=V_{ tL 0}+\gamma\left|\sqrt{v_{ O }+2 \phi_{ f }}-\sqrt{2 \phi_{ f }}\right| \\&=-2.5+0.5|\sqrt{0.65+0.6}-\sqrt{0.6}|=-2.33 V\end{aligned}\) (15.62)

 

With this value, Eq. (15.56) gives \( V_{ IH }=2.20 V \). Thus,

\( V_{ IH }=\frac{2\left|V_{ tL }\right|}{\sqrt{3 K_{R}}}+V_{ tD } \) (15.56)

 

\( NM _{ H }=V_{ OH }-V_{ IH }=5-2.20=2.80 V \)

 

which is better than the specified 2.7 V.

(d) The inverter with a load capacitor \( C _{ L } \) is shown in Fig. 15.19(a). As \( v_{ I } \text { goes low to } V_{ OL }, M _{ D } \) turns off immediately and the capacitor \( C _{ L } \) is charged up by the drain current \( i_{ DL } \). The operating point, shown in Fig. 15.19(b), moves along the load line of \( M _{ L } \) from point Q of \( V_{ OL } \) to midpoint P of \( V_{ O (\text { mid })}=\left(V_{ OH }+V_{ OL }\right) / 2 \). The charging of the capacitor is shown in Fig. 15.19(c). The output voltage is given by

\(C_{ L } \frac{d v_{ O }}{d t}=i_{ DL }=K_{ L } V_{ tL }^{2}\) (15.63)

which, after integration from \( V_{ OL } \text { to } V_{ O ( mid )} \text {, gives } t_{ pLH } \) as

 

\(\begin{aligned}t_{ pLH } &=\frac{C_{ L }\left(V_{ OH }-V_{ OL }\right)}{2 K_{ L } V_{ tL }^{2}} \\&=\frac{0.5 pF \times(5-0.16)}{2 \times 20 \mu \times 2.5^{2}}=9.7 ns\end{aligned}\) (15.64)

 

(e) As \( v_{ I } \text { goes high to } V_{ OH }, M _{ D } \) turns on and the capacitor \( C _{ L } \text { discharges through } M _{ D } \). The operating point moves along the load line of \( M _{ D } \text { from point } B \text { of } V_{ OH } \text { to midpoint } D \text { of } V_{ O ( mid )} \) . The output voltage is given by

\( C_{ L } \frac{d v_{ O }}{d t}=i_{ DD }-i_{ DL }=i_{ DD }-K_{ L } V_{ tL }^{2} \) (15.65)

 

\( V_{ OL }=0.16 V , V_{ OH }=V_{ DD }=5 V \), and

 

\( V_{ O ( mid )}=\frac{V_{ OH }+V_{ OL }}{2}=2.6 V \)

 

Thus,

\(\begin{aligned}i_{ DD }\left(V_{ OH }\right) &=K_{ D }\left(V_{ OH }-V_{ tD }\right)^{2}=100 \mu \times(5-1)^{2}=1600 \mu A \\i_{ DD }\left(V_{ O ( mid )}\right) &=K_{ D }\left[2\left(V_{ OH }-V_{ tD }\right) v_{ O }-v_{ O }^{2}\right]=100 \mu \times\left[2 \times(5-1) \times 2.6-2.6^{2}\right]=1404 \mu A \\i_{ DL }\left(V_{ O ( mid )}\right) &=K_{ L } V_{ tL }^{2}=20 \mu \times 2.5^{2}=125 \mu A \\i_{ CHL } &=\frac{1600 \mu+1404 \mu-125 \mu}{2}=1440 \mu A\end{aligned}\)

 

Using Eq. (15.46), we have

\( t_{ pHL }=\frac{C_{ L }\left(V_{ OH }-V_{ O ( mid )}\right)}{I_{ CHL }} \) (15.46)

 

\( t_{ pHL }=0.5 pF \times \frac{(5-2.6) V }{1440 \mu A }=0.83 ns \)

 

which is much shorter than \( t_{ pLH }=9.7 ns \). Assuming that \( i_{ DD } \) is much greater than \( i_{ DL }\), Eq. (15.65) can be simplified to

\( C_{ L } \frac{d v_{ O }}{d t}=i_{ DD }=K_{ D }\left(V_{ DD }-V_{ tD }\right)^{2} \)

 

which, after integration between \( V_{ OH } \text { and }\left(V_{ OH }+V_{ OL }\right) / 2 \text {, gives } t_{ pHL } \) as

\(\begin{aligned}t_{ pHL } &=\frac{C_{ L }\left(V_{ OH }-V_{ OL }\right)}{2 K_{ D }\left(V_{ DD }-V_{ tD }\right)^{2}} \\&=\frac{0.5 pF \times(5-0.16)}{2 \times 100 \mu \times(5-1)^{2}}=0.76 ns\end{aligned}\) (15.66)

(close to 0.83 ns).

(f) The propagation time \( t_{ pd } \text {, which is the average of } t_{ pLH } \text { and } t_{ pHL } \), is

\( t_{ pd }=\frac{t_{ pLH }+t_{ pHL }}{2}=\frac{9.7 ns +0.76 ns }{2}=5.2 ns \)

 

The current at \( v_{ O }=V_{ OL } \) becomes

\(\begin{aligned}&i_{ DL }=i_{ DD }=K_{ L } V_{ tL }^{2}=125 \mu A \\&P_{\text {static }}=\frac{5 \times 125 \mu A }{2}=312.5 \mu W\end{aligned}\)

 

Since \( P_{\text {dynamic }}=\int_{\text {clk }} C_{ L } V_{ DD }^{2}-5 MHz \times 0.5 pF \times 5^{2}-62.5 \mu W ,\)

 

\( P_{ D }=P_{\text {static }}+P_{\text {dynamic }}=312.5 \mu+62.5 \mu=375 \mu W \)

Therefore, the delay-power product is

\( DP =P_{ D } \times t_{ pd }=375 \mu W \times 5.2 ns =1.95 pJ \)

(g) The PSpice schematic is shown in Fig. 15.20. The PSpice plot of the VTC is shown in Fig. 15.21, which gives \( V_{ IL }=1.4522 V \text { (expected value is } 1.46 V \text { ), } V_{ IH }=2.2702 V \text { (expected value is } 2.29 V \text { ), } V_{ OL }=0.16 V \) (expected value is 0.16 V) at \( V_{ OH }=5 V , V_{ I (\operatorname{tran} 1)}=2.1172(\text { expected value is } 1+2.5 \sqrt{1 / 5}=2.118 V )\), and \( V_{ O (\operatorname{tran} 1)}=2.815 V \) (expected value is 5 – 2.5 = 2.5 V). The VTC values are very close to the expected values.

NOTE: We can find \( d v_{ O } / d v_{ I }(=-1) \) points by plotting dv(2) in Probe.
