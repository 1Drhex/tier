Guest Question.

The steady, incompressible flow field for two-dimensional flow is given by the following velocity components: \( \upsilon_{x} = 16 y - x \) and \( \upsilon_{y} = 16 x + y \). Determine the equation for the stream function and the velocity potential.

Admin
Solution

First, let’s check to make sure continuity is satisfied:

 

\( \frac{\partial \upsilon_{x}}{\partial x} + \frac{\partial \upsilon_{y}}{\partial y} = \frac{\partial }{\partial x} \left ( 16 y – x \right ) + \frac{\partial }{\partial y} \left ( 16 x + y \right ) = – 1 + 1 = 0 \)

 

So continuity is satisfied, which is a necessary condition for us to proceed.

We defined the stream function as

 

\( \frac{\partial \Psi }{\partial y} = \upsilon_{x} \)

 

and
\( – \frac{\partial \Psi }{\partial x} = \upsilon_{y} \)

 

Thus,

\( \upsilon_{x} = \frac{\partial \Psi }{\partial y} = 16 y – x \)

 

\( \upsilon_{y} = – \frac{\partial \Psi }{\partial x} = 16 x + y \)

 

We can begin by integrating equation (\(\upsilon_{x} = \frac{\partial \Psi }{\partial y} = 16 y – x \)) or equation (\(\upsilon_{y} = – \frac{\partial \Psi }{\partial x} = 16 x + y \)). Either will result in the same answer. We will choose to begin by integrating equation (\(\upsilon_{x} = \frac{\partial \Psi }{\partial y} = 16 y – x \)) partially with respect to y:

 

\( \Psi = 8 y^{2} – x y + f_{1}\left ( x \right ) \)

 

where \(f_{1} \left ( x \right ) \) is an arbitrary function of x.
Next, we take the other part of the definition of the stream function, equation (\( – \frac{\partial \Psi }{\partial x} = \upsilon_{y} \)), and differentiate equation (\( \Psi = 8 y^{2} – xy + f_{1}\left ( x \right ) \)) with respect to x:

 

\( \upsilon_{y} = – \frac{\partial \Psi }{\partial x} = y – f_{2} \left ( x \right ) \)

 

Here, \( f_{2} \left ( x \right ) \) is \( \frac{df}{dx}, \) since f is a function of the variable x.
The result is that we now have two equations for \( \upsilon_{y}, \) equations (\( \upsilon_{y} = – \frac{\partial \Psi }{\partial x} = 16 x + y \)) and (\( \upsilon_{y} = – \frac{\partial \Psi }{\partial x} = y – f_{2} \left ( x \right ) \)). We can now equate these and solve for \( f_{2} \left ( x \right ) : \)

 

\( \upsilon_{y} = y – f_{2} \left ( x \right ) = 16 x + y \)

 

Solving for \( f_{2} \left ( x \right ) \),

 

\( f_{2} \left ( x \right ) = – 16 x \)

 

So

\( f_{1} \left ( x \right ) = – 16 \frac{x^{2}}{2} = – 8 x^{2} + C \)

 

The integration constant C is added to the above equation since f is a function of x only. The final equation for the stream function is

 

\( \Psi = 8 y^{2} – x y – 8 x^{2} + C \)

 

The constant C is generally dropped from the equation because the value of a constant in this equation is of no significance. The final equation for the stream function is

 

\( \Psi = 8 y^{2} – x y – 8 x^{2} \)

 

One interesting point is that the difference in the value of one stream line in the flow to another is the volume flow rate per unit width between the two streamlines.
Next we want to find the equation for the velocity potential. Since a condition for the velocity potential to exist is irrotational flow, we must first determine whether the flow in this example is irrotational.

To do this, we must satisfy equation.

 

\( \omega_{z} = \frac{1}{2} \left ( \frac{\partial \upsilon_{y}}{\partial x} – \frac{\partial \upsilon_{x}}{\partial y}\right ) = \frac{1}{2} \left ( 16 – 16 \right ) = 0 \)

 

So the flow is irrotational, as required.
We now want to determine the equation for the velocity potential. The velocity potential is defined by equation :

 

\( \upsilon = \bigtriangledown \varnothing \)

or

\( \frac{\partial \varnothing }{\partial x} = \upsilon_{x} = 16 y – x \)

 

\( \varnothing = 16 x y – \frac{x^{2}}{2} + f\left ( y \right ) \)

 

Differentiating with respect to y and equating to \( \upsilon_{x} \),

 

\( \frac{\partial \varnothing }{\partial y} = 16 x + \frac{d}{dy} f \left ( y \right ) = 16 x + y \)

 

Thus,

\( \frac{d}{d y} f \left ( y \right ) = y \)

 

and

\( f \left ( y \right ) = \frac{y^{2}}{2} \)

 

so that the final equation for the velocity potential is

\( \varnothing = 16 x y – \frac{x^{2}}{2} + \frac{y^{2}}{2} \)
