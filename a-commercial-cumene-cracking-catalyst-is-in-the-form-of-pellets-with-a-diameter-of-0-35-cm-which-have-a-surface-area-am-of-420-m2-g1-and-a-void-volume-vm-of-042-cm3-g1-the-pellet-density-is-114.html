Guest Question.

A commercial cumene cracking catalyst is in the form of pellets with a diameter of 0.35 cm which have a surface area, \(A_{m}\), of 420 \(m^{2}g^{-1}\) and a void volume, \(V_{m}\) , of 0.42 \(cm^{3}g^{-1}\). The pellet density is 1.14 g \(cm^{-3}\). The measured \(1^{st}\)-order rate constant for this reaction at 685K was 1.49 \(cm^{3} s^{-1} g^{-1}\). Assume that Knudsen diffusion dominates and the path length is determined by the pore diameter, \(d_{p}\) . An average pore radius can be estimated from the relationship \(\overline{r} _{p} =2 \overline{V}_{m} /A_{m}\) if the pores are modeled as noninterconnected cylinders (see equation 4.94 :\(\overline{r} _{p} = \frac{2 \overline{V}_{m}}{A_{m}}= \frac{2\varepsilon }{A_{m}\rho _{p}} \)). Assuming isothermal operation, calculate the Thiele modulus and determine the effectiveness factor, \( \eta \),under these conditions.

Admin
Solution

Using the Thiele modulus:

\(\eta =\frac{\Re }{\Re _{o} } = k_{meas} /k_{o}= 3/\phi \left\lgroup\frac{1}{\tanh \phi }- \frac{1}{\phi } \right\rgroup \) and \(\phi =R_{p} k_{o} ^{1/2} / D_{eff}\ ^{1/2} \Rightarrow k_{o}=\frac{\phi }{3} \frac{ k_{meas}}{\left\lgroup 1/ \tanh \phi – \frac{1}{\phi } \right\rgroup } = \phi ^{2} D_{eff}/R^{2}_{p} \Rightarrow \phi \left\lgroup 1/ \tanh \phi – \frac{1}{\phi } \right\rgroup = \frac{k_{meas} \Re ^{2} }{3D_{eff}} \)

 

\(R_{p}= \overline{r}_{p} = \frac{2 V_{g} }{S_{g}}= 2\left\lgroup\frac{0.42 \ cm^{3}/g }{\left(420 \ m^{2}/g \right) \left(100 \ cm/m\right) ^{2} } \right\rgroup = 2.0 \times 10^{-7} cm \)

 

\(D_{eff} \cong D_{Kn}= 1/3 \ d_{p} \overline{v}= 1/3\left(4.0 \times 10^{-7} cm \right) \left[\frac{8\left(1.38 \times 10^{-16} erg /K\right) \left(685 K\right) }{\pi \left(120 amu\right) \left(1.66\times 10^{-24} g/amu\right) } \right]^{1/2} = 4.63 \times 10^{-3} \frac{cm^{2} }{s} \)

 

\(\overline{v}= \left[\frac{8k_{B}T}{\pi m} \right]^{1/2} = \left[\frac{8 R T}{\pi \left(M W\right) } \right]^{1/2}= 3.48 \times 10^{4}cm/s\) (Eq.4.76)

 

\(\phi \left\lgroup\frac{1}{\tanh \phi }- \frac{1}{\phi } \right\rgroup = \frac{\left( 1.49 \ cm^{3} / s\cdot g\right) \left(1.14 \ g/ cm^{3} \right) \left(0.35 \ cm/2 \right) ^{2} }{3\left(4.6 \times 10^{-3} cm^{2}/s \right) } = 3.75 \cong 3.8 \) so

\(\frac{\phi }{\tanh \phi} = 4.8 \Rightarrow \phi =4.8 \) , and \(\eta =\frac{3}{4.8}\left\lgroup1- \frac{1}{4.8} \right\rgroup = 0.495 \cong 0.5\)
