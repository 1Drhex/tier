Guest Question.

Consider the second-order system
\(G(s) = \frac{1}{s^{2} + 2 \zeta s + 1}\).
We would like to add a transfer function of the form \(D(s) = K(s+a)/(s+b)\) in series with G(s) in a unity-feedback structure.

(a) Ignoring stability for the moment, what are the constraints on K, a, and b so that the system is Type 1?

(b) What are the constraints placed on K, a, and b so that the system is both stable and Type 1?
(c) What are the constraints on a and b so that the system is both Type 1 and remains stable for every positive value for K?

Admin
Solution

(a) In a unity feedback structure, the error is \(1/(1+GD)\) and, as we saw, to be Type 1, there needs to be a pole at s = 0 in the product GD. Since there is no such pole in G, it must be supplied by D, thus, the answer is

b = 0

(b) To assure stability, all poles of the closed loop must be in the left half plane, for which the criterion is by Routh. Thus the characteristic equation is

\(s(s^{2} + 2 \zeta s + 1) + K(s + a) = 0\)

and the Routh array is

\(\begin{array}{l l} 1 & 1 + K \\2 \zeta & a K \\ \frac{2 \zeta (1 + K) – a K}{2 \zeta} \\ a K \end{array} \)

Thus the requirements are

\(2 \zeta(1 + K) – a K \gt 0\)

\(a K \gt 0\)

(c) If we assume that \(\zeta \gt 0\) and, for this part, that \(a \gt 0\) also, the requirements can be reduced to

\(K \gt 0\)

\(2 \zeta + K (2 \zeta – a) \gt 0\)

If \(a \lt 2 \zeta\), inspection of these conditions shows that the system will be stable for all positive values of K. On the other hand, if \(a \gt 2 \zeta\), then the requirement is

\(0 \lt K \lt \frac{2 \zeta}{a – 2 \zeta}\)

Extra credit: work out the case for \(\zeta \lt 0\). Note to the Instructor: you might come back to this problem in chapter 5 and verify this point using the rule of asymptotes.
