Guest Question.

Step test data have been obtained for the off-gas \(CO_{2}\) concentration response obtained from changing the feed rate to a bioreactor. Use Smith’s method as well as nonlinear regression based on Excel and MATLAB to estimate parameters in a second-order model from experimental step response data shown in Figure below. For all three methods, assume θ = 0 because the response curve becomes nonzero immediately after t = 0. Compare the results with an FOPTD model that is fit using the 63.2% response method to estimate the time constant.

Admin
Solution

Smith’s Method
The two points of interest are the 20% response time, \(t_{20} = 1.85 min\), and the 60% response time, \( t_{60} = 5.0 min\). Hence, \( t_{20}/ t_{60} = 0.37\).
From first Figure on the right, ζ = 1.3 and \(t_{60}/\tau = 2.8\); thus, \(\tau\)= 5.0/2.8 = 1.79 min. Because the model is overdamped, the two time constants can be calculated from the following expressions modified from Equations
\(\tau _{1}=\frac{\tau }{\zeta -\sqrt{\zeta ^{2}-1} } \quad (\zeta \geq 1)\)
and
\(\tau _{2}=\frac{\tau }{\zeta +\sqrt{\zeta ^{2}-1} } \quad (\zeta \geq 1)\) :
\(\tau _{1}=\tau \zeta +\tau \sqrt{\zeta ^{2}-1} , \quad \tau _{2}=\tau \zeta -\tau \sqrt{\zeta ^{2}-1}\)
Solving gives \(\tau_{1}\) = 3.81 min and \(\tau_{2}\) = 0.84 min.
For Figure above the 63.2% response is estimated to occur at t = 5.3 min. Using the slope at the point of inflection, we can estimate the time delay to be θ = 0.7 min.
Note that \(\tau\) = 4.6 min, which is approximately equal to the sum of \(\tau_{1}\) and \(\tau_{2}\) for the second-order model.

Nonlinear Regression
Using Excel and MATLAB, we calculate the time constants in Equation
\(y(t)=KM(1-\frac{\tau _{1}e^{-t/\tau _{1}}-\tau _{2}e^{-t/\tau _{2}}}{\tau _{1}-\tau _{2}} )\)
that minimize the sum of the squares of the errors between experimental data and model predictions (see Equation
\(\min _{\beta j}S=\sum\limits_{i=1}^{N}{(Y_{i}-\hat{y}_{i} )^{2}} \) ).
The data consisted of 25 points between t = 0 and t = 12 min with a sampling period of 0.5 min. A comparison of the model parameters and the sum of squared errors for each method is shown below; the time delay is set to zero for the three second-order methods.

Clearly, the nonlinear regression method is superior in terms of the goodness of fit, as measured by the sum of squares of the prediction error, but the required calculations are more complicated. Note that the nonlinear regression methods employed by Excel and MATLAB produce identical results.
The step responses are plotted in second Figure on the right; all three calculated models give an acceptable fit to the original step response curve. In fact, the nonlinear regression model is indistinguishable from the experimental response. Nonlinear regression does not depend on graphical correlations and always provides a better fit to the data. It also permits the experimental step test to be terminated before the final steady state is reached; however, sufficient response data must be obtained for the regression method to be effective.
