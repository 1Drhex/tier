Guest Question.

A white noise signal x(n) of zero expected value and variance \(ﾏダ{x}^{2}\) is applied to an LTI system with impulse response \(h(n) = (0.6)^{n}u(n)\) producing the output v(n).
The signal v(n) is then applied to a second LTI system of impulse response \(g(n) = (0.8)^{n}u(n),\) producing the output y(n). Calculate the expected values and variances of the signals v(n) and y(n).

Admin
Solution

For the first system with \(h(n) = (0.6)^{n}u(n)\), it is:

\(E[v]=E[x]\sum\limits_{n=-\infty }^{+\infty }{h(n)}=E[x]H(e^{j0})=0,\) since E[x]=0.
\(\sigma _{v}^{2}=\sigma _{x}^{2}\sum\limits_{n=-\infty }^{+\infty }{|h(n)|^{2}}=\underbrace{ \sigma _{x}^{2}\frac{1}{2\pi }\int\limits_{-\pi }^{\pi }{|H(e^{j\omega })|^{2}d\omega } }_{by\ using\ Parseval's\ theorem} \)

\(=\sigma _{x}^{2}\frac{1}{2\pi }\int\limits_{-\pi }^{\pi }{H(e^{j\omega })H(e^{-j\omega })d\omega }=\sigma _{x}^{2}\frac{1}{2\pi }\int\limits_{-\pi }^{\pi }{\frac{1}{1-0.6e^{-j\omega }}\frac{1}{1-0.6e^{j\omega }}d\omega }\)
\(=\sigma _{x}^{2}\frac{1}{2\pi }\int\limits_{-\pi }^{\pi }{\frac{1}{1.36-1.2\cos \omega }d\omega }=\sigma _{x}^{2}\frac{1}{2\pi }\frac{2\pi }{\sqrt{(1.36)^{2}-(-1.2)^{2}} }\)
\(=\sigma _{x}^{2}\frac{1}{0.64 }=\frac{\sigma _{x}^{2}}{(0.8)^{2}}=1.5625\sigma _{x}^{2},\)

where the fourth equality is due to the fact that:

\(H(e^{j\omega })=DTFT\left\{x(n)\right\}=DTFT\left\{0.6^{n}u(n)\right\}=1/(1-0.6e^{-j\omega }),\)

while the last integral is solved by the use of the relation:

\(\int\limits_{0}^{2\pi }{\frac{d\omega }{a+b\cos \omega } }=\frac{2\pi }{\sqrt{a^{2}-b^{2}} }.\)

For the second system with \(g(n) = (0.8)^{n}u(n),\) it is:

\(E[y]=E[v]\sum\limits_{n=-\infty }^{+\infty }{g(n)}=E[v]G(e^{j0})=0,\) since E[v]=0.
\(\sigma _{y}^{2}=\sigma _{v}^{2}\sum\limits_{n=-\infty }^{+\infty }{|g(n)|^{2}}= \sigma _{v}^{2}\frac{1}{2\pi }\int\limits_{-\pi }^{\pi }{|G(e^{j\omega })|^{2}d\omega }\)
\(=\sigma _{v}^{2}\frac{1}{2\pi }\int\limits_{-\pi }^{\pi }{G(e^{j\omega })G(e^{-j\omega })d\omega }=\sigma _{v}^{2}\frac{1}{2\pi }\int\limits_{-\pi }^{\pi }{\frac{1}{1-0.8e^{-j\omega }}\frac{1}{1-0.8e^{j\omega }}d\omega }\)
\(=\sigma _{v}^{2}\frac{1}{2\pi }\int\limits_{-\pi }^{\pi }{\frac{1}{1.64-1.6\cos \omega }d\omega }=\sigma _{v}^{2}\frac{1}{2\pi }\frac{2\pi }{\sqrt{(1.64)^{2}-(-1.6)^{2}} } \)
\(=\frac{\sigma _{v}^{2}}{0.36} =\frac{\sigma _{v}^{2}}{(0.6)^{2}}=\frac{\frac{\sigma _{x}^{2}}{(0.8)^{2}} }{(0.6)^{2}}=\frac{\sigma _{x}^{2}}{(0.48)^{2}}=4.34\sigma _{x}^{2} . \)
