Guest Question.

A fill is to be placed over the soil profile shown in Figure 12.7. Using program SETTRATE, develop a plot of consolidation settlement vs. time. Both the SP and CL/ML strata are normally consolidated.

Admin
Solution

Silty clay/clayey silt stratum:

Compute using program SETTRATE. See Figure 12.8 for typical screen. The results are plotted in the table on the next page.

Sand stratum:

The consolidation settlement in the sand stratum will occur as quickly as the load is applied. Therefore, we only need to compute the ultimate consolidation settlement.

At midheight in the sand stratum:

\(\sigma ^{\prime }_{z0}=\sum{\gamma H – u} =\left(17.8 kN/m^{3}\right) \left(1.2 m\right) +\left(18.1 kN/m^{3}\right) \left(0.95 m\right) – \left(9.8 kN/m^{3}\right) \left(0.95 m\right) =29.2 kPa\)

 

\(\sigma ^{\prime }_{zf}=\sigma ^{\prime }_{z0}+\gamma _{fill}H_{fill}=29.2 kPa+\left(18.0 kN/m^{3}\right) \left(3.2 m\right) =86.8 kPa\)

Per Table 11.4,

Material \(C_{α}/C_{c}\)
Granular soils, including rockfill 0.02 ± 0.01
Shale and mudstone 0.03 ± 0.01
Inorganic days and silts 0.04 ± 0.01
Organic days and silts 0.05 ± 0.01
Peat and muskeg 0.06 ±0.01
\(C_{c}/(1+e_{0})=0.007\)

 

\(\left(\delta _{c}\right) _{ult}=\sum{\frac{C_{c}}{1+e_{0}} H\log \left\lgroup\frac{\sigma ^{\prime }_{zf}}{\sigma ^{\prime }_{z0}} \right\rgroup } =\left(0.007\right) \left(4.3 m\right) \log \left\lgroup\frac{86.8 kPa}{29.2 kPa} \right\rgroup =14 mm\)

Gravelly sand stratum:

This soil is classified as very dense, so we can assume its compressibility is negligible compared to that of the other strata.
