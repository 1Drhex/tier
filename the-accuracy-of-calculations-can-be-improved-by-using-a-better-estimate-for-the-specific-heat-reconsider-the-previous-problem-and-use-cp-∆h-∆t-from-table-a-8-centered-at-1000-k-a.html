Guest Question.

The accuracy of calculations can be improved by using a better estimate for the specific heat. Reconsider and use \(C_{ P }=\Delta h / \Delta T\), from Table A.8 centered at 1000 K.
A mixture of 50% carbon dioxide and 50% water by mass is brought from 1500 K, 1 MPa to 500 K, 200 kPa in a polytropic process through a steady state device. Find the necessary heat transfer and work involved using values from Table A.5.

Admin
Solution

Using values from Table A.8 we estimate the heat capacities

\(\begin{aligned}& C _{ P CO _{2}}=\frac{1096.36-849.72}{1100-900}=1.2332 kJ / kg K \\& C _{ P H _{2} O }=\frac{2226.73-1768.6}{1100-900}=2.2906 kJ / kg K\end{aligned}\)

Eq.11.23: \(C _{ P \text { mix }}=\Sigma c _{ i } C _{ Pi }=0.5 \times 1.2332+0.5 \times 2.2906=1.7619 kJ / kg K\)

Eq.11.15: \(R _{\operatorname{mix}}=\Sigma c _{ i } R _{ i }=0.5 \times 0.1889+0.5 \times 0.4615=0.3252 kJ / kg K\)

Process \(Pv ^{ n }= C \quad \Rightarrow \quad n =\ln \left( P _{1} / P _{2}\right) / \ln \left( v _{2} / v _{1}\right) \quad \text { and use } Pv = RT\)

\(n =\ln \left(\frac{1000}{200}\right) / \ln \left(\frac{500 \times 1000}{200 \times 1500}\right)=3.1507\)

Work is from Eq.7.18

\(w =-\int vdP =-\frac{ n }{ n -1}\left( P _{ e } v _{ e }- P _{ i } v _{ i }\right)=-\frac{ nR }{ n -1}\left( T _{ e }- T _{ i }\right)= 4 7 6 . 4 k J / kg\)

Heat transfer from energy equation

\(q = h _{ e }- h _{ i }+ w =1.7619(500-1500)+476.4=- 1 2 8 5 . 5 kJ / kg\)

……………………………………………

Eq.11.23 : \(C_{p 0 mix }=c_{A} C_{p 0 A}+c_{B} C_{p 0 B}\)

Eq.11.15:
\(\begin{aligned}R_{\operatorname{mix}} &=\frac{1}{m}\left(n_{A} \bar{R}+n_{B} \bar{R}\right) \\&=\frac{1}{m}\left(m_{A} R_{A}+m_{B} R_{B}\right) \\&=c_{A} R_{A}+c_{B} R_{B}\end{aligned}\)

Eq.7.18 :
\(\begin{aligned}w &=-\int_{i}^{e} v d P \quad \text { and } \quad P v^{n}=\text { constant }=C^{n} \\w &=-\int_{i}^{e} v d P=-C \int_{i}^{e} \frac{d P}{P^{1 / n}} \\&=-\frac{n}{n-1}\left(P_{e} v_{e}-P_{i} v_{i}\right)=-\frac{n R}{n-1}\left(T_{e}-T_{i}\right)\end{aligned}\)
