Guest Question.

The stream function for a flow field is defined by \( \psi = 2r^3 sin 2\Theta \) . Determine the magnitude of the velocity of fluid particles at point \(r = 1 m, u = (p^3) rad,\) and plot the streamlines for \(\psi _1 = 1 m^2/s and \psi _2 = 2 m^2/s. \)

Admin
Solution

We consider ideal fluid flow.

The velocity components are

\(v_r = \frac{1}{r} \frac{\partial \psi }{\partial θ } ; v_r = \frac{1}{r}[2r^3(2 cos 2θ ) ] = (4r^2 cos 2θ ) m/s \)

 

\(v_θ=- \frac{\partial \psi }{\partial r}: v_θ = – ( -6r^2 sin 2θ ) m/s \)

The continuity equation\( \frac{v_r}{r} + \frac{\partial vr}{\partial r} + \frac{1}{r}\frac{\partial v_θ }{\partial θ } = 4r cos 2θ + 8r cos 2θ + (-12r cos 2θ ) = 0 \)is indeed satisfied.

At point \( r = 1 m, θ = π/3 rad,\)

 

\(v_r = 4(1^2) cos [2(\frac{\pi }{3})] = – 2m/s \)

 

\(v_θ = -6(1^2) sin [2\frac{\pi }{3}]= -5.196 m/s\)

Thus, the magnitude of the velocity is

\(V = \sqrt{v_r^2 + v_θ ^2} = \sqrt{( -2 m/s)^2 + ( -5.196 m/s)^2 } \)

= 5.57 m/s

For\( \psi = 1 m^2/s,\)
