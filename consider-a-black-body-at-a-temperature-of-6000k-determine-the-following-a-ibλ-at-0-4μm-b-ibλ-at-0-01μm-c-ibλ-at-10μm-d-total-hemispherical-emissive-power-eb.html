Guest Question.

Consider a black body at a temperature of 6000K. Determine the following. (a) \(I_{b_{1} \lambda} \text { at } 0.4 \mu m\) (b) \(I_{b, \lambda} \text { at } 0.01 \mu m\) (c) \(I_{b, \lambda}\) at\(10 \mu m\) (d) total hemispherical emissive power \(E_{b}(T)\) (e) \(I_{b, \lambda}\) corresponding to \(\lambda_{\max }\) (f)ratio of \(I_{b, \lambda} \text { at } \lambda_{\max } \text { to } I_{b, \lambda} \text { at } 10 \mu m\) (g) fraction of radiation in the visible part of the spectrum.

Admin
Solution

Temperature of the black body T = 6000K

\(\text { (a) } I _{b, \lambda} \text { at } 0.4 \mu m\)

 

\(I_{b, \lambda}=\frac{c_{1} \lambda^{-5}}{\left[e^{\frac{c_{2}}{\lambda T}-1}\right]}\) (2.190)

 

\(I_{b, \lambda}=\frac{1.198 \times 10^{8}(0.4)^{-5}}{\left[e^{\frac{1.433 \times 10^{4}}{0.4 \times 6000}}-1\right]}\) (2.191)

 

\(I_{b, \lambda}=2.91 \times 10^{7} W / m ^{2} \mu m sr\) (2.192)

 

\(\text { (b) } I_{b, \lambda} \text { at } 0.01 \mu m =0\) (hardly any radiation in this ultraviolet part of the spectrum from the sun)

 

\(\text { (c) } I_{b, \lambda} \text { at } 10 \mu m =4420 W / m ^{2} \mu m sr\)

 

\(\text { (d) } E _{b}( T )=\sigma T ^{4}=7.35 \times 10^{7} W / m ^{2}\)

 

\(\text { (e) } \lambda_{\max x} T =2898 \mu mK ; \lambda_{\max }=0.483 \mu m\) (this corresponds to blue light as the wavelength of blue radiation is between \( 0.45 \mu m \text { to } 0.49 \mu m !\) Of course, if we want to do it for solar radiation, instead of 6000K, we will use 5800 K.) \(I _{b, \lambda} \text { corresponding to } \lambda_{\max }=3.181 \times 10^{7} W / m^{2} \mu m s r\)

 

\(\text { (f) } I _{b, \lambda} \text { at } \lambda_{\max } / I _{b, \lambda} \text { at } 10 \mu m =7196\)
This is a very important result. The \(I _{b, \lambda}\) corresponding to visible blue light divided by the \(I_{b, \lambda}\) at infrared of \(10 \mu m\) for the temperature of 6000K, which corresponds to the outer temperature of the sun, we can see that the ratio of intensity of visible radiation to infrared radiation is about 7196.

 

(g) Fraction in the visible part of the spectrum: \(\lambda_{1}=0.4 \mu m , \lambda_{2}=0.7 \mu m\) , T = 6000 K, \(\lambda_{1} T =2400 \mu mK , \lambda_{2} T =4200 \mu mK\).

The fraction corresponding to \(\lambda_{1} T =2400 \mu mK , F _{0-\lambda_{1} T}=0.14\) .

The fraction corresponding to \(\lambda_{2} T =4200 \mu mK , F _{0-\lambda_{2} T}=0.516\)

\(F_{\lambda_{1}-\lambda_{2}}=0.376 \text { or } 37.6 \%\)

 

Please do not underestimate the importance of this result because 0 to \(\infty\) is so huge and the wavelength band of \(0.4 \mu m -0.7 \mu m\) is so small, yet the fraction of radiation in this band is almost 40% of the total radiation in the \(0-\infty\) band. So, nearly 40% of the solar radiation is concentrated in a very small portion of the spectrum, namely the visible range. If the temperature of the sun were not 6000K, the peak would be in some other part of the spectrum and we would not be getting enough “visible” radiation.

So now the question arises: if we have the temperature of the sun to be 6000K, which makes the earth so habitable, when all of it it started with the big bang theory (assuming this to be the correct theory to explain the origins of the universe), it was basically a cooling problem. It started with some initial conditions. If the initial conditions were to be different, then the temperature of the outer sun would have been different, which means that the fraction of the radiation falling would have been different and the earth’s temperature may have been different because of which we all may not have been here today! Why were the initial conditions chosen that way, or who chose them to be so?
