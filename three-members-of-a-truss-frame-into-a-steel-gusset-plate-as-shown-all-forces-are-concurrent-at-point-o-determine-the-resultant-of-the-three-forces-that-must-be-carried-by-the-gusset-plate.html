Guest Question.

Three members of a truss frame into a steel gusset plate as shown. All forces are concurrent at point O. Determine the resultant of the three forces that must be carried by the gusset plate.

Admin
Solution
\(F_{1y} = + F_{1} \cos 30^{\circ} = 10k (0.866) = 8.66k\)

 

\(F_{1x} = + F_{1} \sin 30^{\circ} = 10k (0.50) = 5k\)

 

\(F_{2} = – F_{2x} = -12k\)

 

\(F_{3x} = + \frac{1}{\sqrt{2} } (F_{3}) = + \frac{18k}{\sqrt{2} }\)

 

\(F_{3y} = – \frac{1}{\sqrt{2} } (F_{3}) = – \frac{18k}{\sqrt{2} }\)

 

\(R_{x} = \sum{F_{x}} = +5k – 12k + \frac{18k}{\sqrt{2} } = + 5.73k\)

 

\(R_{y} = \sum{F_{y}} = +8.66k – \frac{18k}{\sqrt{2} } = – 4.07k\)

 

\(\tan \theta = \frac{R_{y}}{R_{x}} = \frac{4.07}{5.73} = 0.710\)

 

\(\theta = \tan ^{-1} (0.710) = 35.4^{\circ} from horizontal \)

 

 

\(\sin \theta = \frac{R_{y}}{R}\)

 

\(R = \frac{R_{y}}{\sin \theta } = \frac{R_{y}}{\sin 35.4^{\circ} }\)

 

\(\therefore R = \frac{4.07k}{(0.579)} = 7.03k\)
