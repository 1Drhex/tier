Designing a MOS differential pair with an active current source

(a) Design a MOS differential pair as shown in Fig. 9.13, in which one input terminal is grounded. The output is taken from the drain of transistor \( M _{1} \). The DC biasing current is \( I_{ Q }=10 mA , \text { and } V_{ DD }=V_{ SS }=15 V \). The MOSFETs are identical and have \( K_{ n }=1.25 mA / V ^{2} \text { and } V_{ t }=1 V . A \). A small-signal voltage gain of \( A_{1}-10 V / V \) is required. Assume \( V_{M}=\infty \).

(b) Calculate the design values of \( A_{ d }, A_{ c } \), and CMRR.
(c) Find the maximum permissible value of \( R_{ D }\) and the corresponding voltage gain \( A_{ d } \).

Admin
Solution

(a) \( I_{ Q }=10 mA , \text { and } I_{ D 1}=I_{ D 2}=I_{ Q } / 2=10 mA / 2=5 mA \). From Eq. (9.54),

\(\begin{aligned}g_{ m } &=2 K_{ n }\left(V_{ GS }-V_{ t }\right) \\&=2 \overline{2 K_{ n } I_{ Q }}\end{aligned}\) (9.54)

 

\(g_{ m }=\sqrt{2 \times K_{ n } I_{ Q }}=\sqrt{2 \times 1.25 m \times 10 m }=5 mA / V\)

 

From Eq. (9.37), the DC gateâ€“source voltage is

\(i_{ D }=K_{ n }\left(v_{ GS }-V_{ t }\right)^{2}\) (9.37)

 

\(V_{ GS 1}=-\left(\sqrt{\frac{I_{ D 1}}{K_{ n }}}+V_{ t }\right)=-\left(\sqrt{\frac{5 mA }{1.25 m }}+1\right)=-2.236 V\)

 

Therefore, the voltage at the source terminal with respect to the ground is \( V_{ SR }=-V_{ GS }=-2.236 V \), and

\(R_{ SS }=\frac{V_{ SR }-V_{ SS }}{I_{ SS }}=\frac{(-2.236+15) V }{10 mA }=1.3 k \Omega\)

 

(b) \( v_{ G 1}=V_{ G 1}+v_{ g 1}, V_{ G 1}=0, \text { and } v_{ G 2}=v_{ g 2}=0 \). Then

\(\begin{aligned}&v_{ id }=v_{ g 1}-v_{ g 2}=v_{ g 1} \\&v_{ ic }=\frac{v_{ g 1}+v_{ g 2}}{2}=\frac{v_{ g 1}}{2}\end{aligned}\)

 

From Eq. (9.7),

\(v_{ ol }=v_{ oc }+\frac{v_{ od }}{2}\) (9.7)

 

\(v_{ ol }=v_{ oc }+\frac{v_{ od }}{2}=A_{ c } v_{ ic }+A_{ d } v_{ id }=A_{ c } \frac{v_{ g 1}}{2}+A_{ d } v_{ g 1}=\frac{v_{ g 1}}{2}\left(A_{ c }+2 A_{ d }\right)\)

 

Substituting for \( A_{ d } \text { and } A_{ C } \) from Eqs. (9.55) and (9.57) gives the voltage gain \( A_{1} \) as

\(A_{ d }=\frac{v_{ od } / 2}{v_{ id } / 2}=-g_{ m }\left(r_{ o 1} \| R_{ D }\right)\) (9.55)

 

\( A_{ c }=\frac{v_{ oc }}{v_{ ic }}=\frac{-g_{ m } R_{ D }}{1+g_{ m } 2 R_{ SS }}\) (9.57)

 

\(A_{1}=\frac{v_{ ol }}{v_{ g 1}}=\frac{1}{2}\left(A_{ c }+2 A_{ d }\right)=-\frac{1}{2}\left[\frac{g_{ m } R_{ D }}{1+2 g_{ m } R_{ SS }}+2 g_{ m } R_{ D }\right]\)

 

Substituting \( A_{1}=-10, g_{ m }=5 mA / V , \text { and } R_{ SS }=1.3 k \Omega \) into the above equation gives \( R_{ D }=1.93 k \Omega \).

 

\( r_{ ol }=\frac{V_{ M }}{I_{ D 1}}=\infty \)

 

From Eq. (9.55),

\(A_{ d }=-g_{ m }\left(R_{ D } \| r_{ o 1}\right)=-5 m \times(1.93 k \| 20 k )=-8.8 V / V\)

 

From Eq. (9.57),

\(A_{ c }=\frac{-g_{ m } R_{ D }}{1+g_{ m } \times 2 R_{ SS }}=\frac{-5 m \times 1.93 k }{1+5 m \times 2 \times 1.7 k }=-0.64 V / V\)

 

Thus, \( CMRR =\left|A_{ d } / A_{ c }\right|=8.8 / 0.64=13.76, \text { or } 22.78 dB \text {. } \)

(c) For \( I_{ D }=I_{ Q } / 2=5 mA \text { and } V_{ DD }=15 V \), the maximum value of \( R_{ D } \) is

\(R_{ D (\max )}=\frac{V_{ DD }}{I_{ D }}=\frac{15 V }{5 mA }=3 k \Omega\)

which gives the maximum value of

\( A_{ d (\max )}=-g_{ m } R_{ D (\max )}=-5 mA \times 3 k \Omega–15 V / V\)
