Guest Question.

The stream function for a flow field is defined by \(\psi = (4/r^2) sin 2\Theta \) . Show that continuity of the flow is satisfied, and determine the r and \(\Theta \)velocity components of fluid particles at point \(r = 2 m, \Theta = (\pi/4) rad \). Plot the streamline that passes through this point.

Admin
Solution

We consider ideal fluid flow.

Using the r and\( θ \) velocity components with respect to stream function

\(v_r = \frac{1}{r}\frac{\partial \psi }{\partial θ } = \frac{1}{r}(\frac{4}{r^2})(2 cos 2 θ ) = \frac{8}{r^3} \cos 2θ\)

 

\(v_θ =- \frac{\partial \psi }{\partial r} = – (- \frac{8}{r^3} sin 2 θ ) = \frac{8}{r^3} \sin 2θ \)

The continuity equation\(\frac{v_r}{r} + \frac{\partial v_r}{\partial r} + \frac{1}{r} \frac{\partial v_θ }{\partial θ } = \frac{8}{r^4}\cos 2θ + (- \frac{24}{r^4} cos 2θ ) + \frac{16}{r^4} cos 2θ = 0 \) is indeed satisfied.

 

At point r = 2 m and \( θ =\frac{\pi }{4} \)

 

\(v_r = \frac{8}{2^3} cos[2(\frac{\pi }{4})] = 0 \)

 

\(v_θ = \frac{8}{2^3} sin[2(\frac{\pi }{4})] = 1 m/s \)

 

\(\psi = \frac{4}{2^2} sin [2(\frac{\pi }{4})] =1 \)

 

Therefore, the stream function that passes through this point is

\(1 = \frac{4}{r^2} \sin 2θ \)

 

\(r^2 = 4 \sin 2 θ \)
