Guest Question.

The physical half-life is 1.7 hours \(For^{ \ \ 113m }_{}\)In.

a. A sample \(of^{ \ \ 113m }_{}\)In has a mass of 2 µg. How \(many^{ \ \ 113m }_{}\)In atoms are present in the sample?

b. How many \(^{113m}In \) atoms remain after 4 hours have elapsed?

c. What is the activity of the sample when t = 4.0 hours?

d. Specific activity is defined as the activity per unit mass of a radioactive sample. What is the specific activity of \(the^{ \ \ 113m }_{}\)In sample after 4 hours?

e. \(Enough^{ \ \ 113m }_{}\)In must be obtained at 4 p.m. Thursday to provide 300 kBq at 1 p.m. Friday. How \(much^{ \ \ 113m }_{}\)In should be obtained?

Admin
Solution

a.

\(Number \ of \ atoms \ N =\frac{(No. \ of \ grams)(No \ of \ atoms/gram − atomic \ mass)}{No. \ of \ grams/gram − atomic \ mass}\)

 

\(\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ =\frac{(2 × 10^{-6} g)(6.02 × 10^{23} atoms/gram − atomic\ mass)}{113 grams/gram − atomic \ mass} \)

 

\(\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ = 1.07 × 10^{16} \ atoms\)

b.

\(Number \ of \ atoms \ remaining = N = N_{0}e^{-\lambda t}\)

 

\(Since \ \lambda = 0.693/T_{1/2}\)

 

\(N=N_{0}e^{-\left(0.693/T_{1/2}\right)^{t} }\)

 

\(\ \ \ =\left(1.07 \times 10^{16}atoms \right) e^{-\left(0.693/1.7hr\right) 4.0 \ hr}\)

 

\(\ \ \ =\left( 1.07 \times 10^{16}atoms \right) \left(0.196\right)\)

 

\(\ \ \ =2.10 \times 10^{15} atoms \ remaining\)

c.

\(A=\lambda N=\frac{0.693}{T_{1/2}}N\)

 

\(\ \ \ =\frac{0.693\left(2.1\times 10^{15}atoms\right) }{1.7 \ hr\left(3600 \ sec/hr\right) }\)

 

\(\ \ \ =2.4 \times 10^{11} dps\)

In units of activity

\(A=2.4 \times 10^{11} Bq\)

Because of the short physical half-life (large decay constant) of \(^{113m}In\), a very small mass of this nuclide possesses high activity.

d.

\(Specific \ activity =\frac{2.4 \times 10^{11} Bq}{2 \times 10^{-6}g}=1.2 \times 10^{17} \frac{Bq}{g}\)

e.

\(A=A_{0}e^{-\lambda t}\)

 

\(A_{0}= A e^{\lambda t}= A e^{0.693 t/T_{1/2}}\)

 

\(\ \ \ \ \ =\left(300 kBq\right)e^{\left(0.693\right)\left(21 \ hr\right)/1.7 \ hr }\)

 

\(A=1.57 \times 10^{9}Bq = 1.57 \ GBq = activity \ that \ should \ be \ obtained \ at \ 4 p.m. Thursday\)
