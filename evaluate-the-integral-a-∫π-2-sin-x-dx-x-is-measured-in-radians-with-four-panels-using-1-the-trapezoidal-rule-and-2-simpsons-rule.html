Guest Question.

Evaluate the integral \(A =\int_{0}^{\pi /2}{\sin x}dx\) (x is measured in radians) with four panels, using (1) the trapezoidal rule; and (2) Simpson’s rule.

Admin
Solution

Because the range of integration is 0 ≤ x ≤ π/2 rad and the number of panels is four, we get Δx = (π/2)/4 = π/8 rad. The following table is convenient for carrying out the quadrature.

f(x) = sin x x(rad) i
0 0 1
0.3827 π/8 2
0.7071 π/4 3
0.9239 3π/8 4
1 π/2 5

Part 1: Trapezoidal Rule

Using Eq. (A.3) \(A\approx \sum\limits_{i=1}^{n}{A_{i}= (f_{1}+2f_{2}+f_{3}+…+2f_{n}+f_{n+1})}\frac{\Delta x}{2}\) wih n = 4 and Δx = π/8 rad, we get

\(A ≈ ( f_{1} + 2 f_{2} + 2 f_{3} + 2 f_{4} + f_{5})\frac{\Delta x}{2}\)

= [0 + 2(0.3827 + 0.7071 + 0.9239) + 1.0]\(\frac{π}{16}\)

= 0.9871

Part 2: Simpson’s Rule

With n = 4 and Δx = π/8 rad, Eq. (A.6) \(A\approx \sum\limits_{i=1,3,5…}^{n-1}{(A_{i}+A_{i+1})}=(f_{1}+4f_{2}+2f_{3}+4f_{4}+…+2f_{n-1}+4f_{n}+f_{n+1})\frac{\Delta x}{3}\) becomes

\(A ≈ ( f_{1} + 4 f_{2} + 2 f_{3} + 4 f_{4} + f_{5})\frac{\Delta x}{3}\) \(= [0 + 4(0.3827) + 2(0.7071) + 4(0.9239) + 1.0]\frac{π}{24}\)

= 1.0002

Because the exact value of the integral is 1.0000, you can see that Simpson’s rule is considerably more accurate than the trapezoidal rule for this problem. A major source of error in the trapezoidal rule is that the curvature of the function f (x) = sin x has the same sign throughout the interval 0 ≤ x ≤ π/2 rad. As mentioned previously, the trapezoidal rule does not perform well in problems of this type.
