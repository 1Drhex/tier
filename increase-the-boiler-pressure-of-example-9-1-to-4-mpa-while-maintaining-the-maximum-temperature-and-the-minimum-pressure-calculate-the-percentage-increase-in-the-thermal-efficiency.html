Guest Question.

Increase the boiler pressure of to 4 MPa while maintaining the maximum temperature and the minimum pressure. Calculate the percentage increase in the thermal efficiency.

Admin
Solution

\( \text { Neglecting the work of the pump, the enthalpy } h_{2} \text { remains unchanged: } h_{2}=192 kJ / kg . \text { At } 400^{\circ} C \text { and } 4 MPa \text {, the } \).

\( \text { enthalpy and entropy are } s_{3}=6.7698 kJ / kg \cdot K \text { and } h_{3}=3214 kJ / kg \text {. State } 4 \text { is in the quality region. Using } s_{4}=s_{3} \text {, the quality is } \)

found to be

\( x_{4}=\frac{s_{4}-s_{f}}{s_{f g}}=\frac{6.7698-0.6491}{7.5019}=0.8159 \).

Observe that the moisture content has increased to 18.4 percent, an undesirable result. The enthalpy of state 4 is then

\( h_{4}=h_{f}+x_{4} h_{f g}=192+(0.8159)(2393)=2144 kJ / kg \).

\( \text { The heat addition is, } q_{B}=h_{3}-h_{2}=3214-192=3022 kJ / kg \text { and the turbine work output is } \)

\( w_{T}=h_{3}-h_{4}=3214-2144=1070 kJ / kg \).

Finally, the thermal efficiency is

\( \eta=\frac{w_{T}}{q_{B}}=\frac{1070}{3022}=0.3541 \).

The percentage increase in efficiency from that of is

\( \% \text { increase }=\left(\frac{0.3541-0.3232}{0.3232}\right)(100)=9.55 \% \).
