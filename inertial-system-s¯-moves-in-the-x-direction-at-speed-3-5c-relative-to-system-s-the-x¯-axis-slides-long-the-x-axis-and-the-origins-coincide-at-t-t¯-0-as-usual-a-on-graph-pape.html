Guest Question.

Inertial system \(\overline{ S } \text { moves in the } x \text { direction at speed } \frac{3}{5} c\) relative to system S. (The \(\bar{x}\) axis slides long the x axis, and the origins coincide at \(t=\bar{t}=0\), as usual.)

(a) On graph paper set up a Cartesian coordinate system with axes ct and x. Carefully draw in lines representing \(\overline{ x }\) = –3, –2, –1, 0, 1, 2, and 3. Also draw in the lines corresponding to \(\overline{c t}\) = –3, –2, –1, 0, 1, 2, and 3. Label your lines clearly

(b) In \(\overline{ S }\), a free particle is observed to travel from the point \(\bar{x}=-2 \text { at time } c \bar{t}=-2\) to the point \(\bar{x}=2 \text { at } c \bar{t}=+3\). Indicate this displacement on your graph. From the slope of this line, determine the particle’s speed in S.

(c) Use the velocity addition rule to determine the velocity in S algebraically, and check that your answer is consistent with the graphical solution in (b).

Admin
Solution
\(\text { (b) } \frac{c}{v}=\text { slope }=\frac{9.25}{8.75}\)

 

\(\Rightarrow v=\frac{8.75}{9.25} c=\frac{35}{37} c\)

 

\(\text { (c) } v^{\prime}=\frac{4}{5} c, \text { so } v=\frac{\frac{4}{5} c+\frac{3}{5} c}{1+\frac{4}{5} \cdot \frac{3}{5}}\)

 

\(=\frac{(7 / 5) c}{(37 / 25)}=\frac{35}{37} c\)
