Guest Question.

A mobile robot is equipped with a sensor that can detect if the floor is clean or not (Z ∈ {clean, dirty}). The mobile system is also equipped with a cleaning system (a set of brushes, a vacuum pump, and a dust bin) that can clean the floor, but the cleaning system is engaged only if the robot believes the floor needs cleaning (U ∈ {clean, null}). Again we would like to determine if the floor is clean or not (X ∈ {clean, dirty}). The initial probability (belief) that the floor is clean is

\(\text { bel }\left(X_{0}=\text { clean }\right)=0.5\)

The correctness of the sensor measurements is given with a statistical model of the sensor:

\(\begin{array}{ll}P\left(Z_{k}=\text { clean } \mid X_{k}=\text { clean }\right)=0.8, & P\left(Z_{k}=\text { dirty } \mid X_{k}=\text { clean }\right)=0.2 \\P\left(Z_{k}=\text { dirty } \mid X_{k}=\text { dirty }\right)=0.9, & P\left(Z_{k}=\text { clean } \mid X_{k}=\text { dirty }\right)=0.1\end{array}\)

The outcome probabilities if the robot decides to perform floor cleaning are the following:

\(\begin{aligned}&P\left(X_{k}=\operatorname{clean} \mid X_{k-1}=\text { clean }, U_{k-1}=\text { clean }\right)=1 \\&P\left(X_{k}=\operatorname{dirt} y \mid X_{k-1}=\text { clean, } U_{k-1}=\text { clean }\right)=0 \\&P\left(X_{k}=\text { clean } \mid X_{k-1}=\operatorname{dirty}, U_{k-1}=\text { clean }\right)=0.8 \\&P\left(X_{k}=\operatorname{dirty} \mid X_{k-1}=\operatorname{dirty}, U_{k-1}=\text { clean }\right)=0.2\end{aligned}\)

If the cleaning system is not activated, the following outcome probabilities can be assumed:

\(\begin{aligned}&P\left(X_{k}=\text { clean } \mid X_{k-1}=\text { clean }, U_{k-1}=\text { null }\right)=1 \\&P\left(X_{k}=\text { dirty } \mid X_{k-1}=\text { clean }, U_{k-1}=\text { null }\right)=0 \\&P\left(X_{k}=\text { clean } \mid X_{k-1}=\text { dirty }, U_{k-1}=\text { null }\right)=0 \\&P\left(X_{k}=\text { dirty } \mid X_{k-1}=\text { dirty }, U_{k-1}=\text { null }\right)=1\end{aligned}\)

Assume that the mobile system first makes an action and then receives the measurement data. Determine the belief \(b e l_{p}\left(x_{k}\right)\) based on the action taken (prediction) and the belief \(\operatorname{bel}\left(x_{k}\right)\) based on measurement \(\operatorname{bel}\left(x_{k}\right)\) (correction) for the following sequence of actions and measurements:

\(\begin{array}{ccc}k & U_{k-1} & Z_{k} \\\hline 1 & \text { null } & \text { dirty } \\2 & \text { clean } & \text { clean } \\3 & \text { clean } & \text { clean }\end{array}\)
Admin
Solution

Without loss of generality let us introduce the following notation: \(X_{k}\) ∈ {clean, dirty} as \(X_{k} \in\left\{x_{k}, \bar{x}_{k}\right\}\), \(Z_{k}\) ∈ {clean, dirty} as \(Z_{k} \in\left\{z_{k}, \bar{z}_{k}\right\}\), and \(U_{k}\) ∈ (clean, null) as \(U_{k} \in\left\{u_{k}, \bar{u}_{k}\right\}\). Let us use Algorithm 3. In time step k = 1, when the action ¯u0 = null is active we can determine the predicted belief that the floor is clean is the following:

\(\begin{aligned}b e l_{p}\left(x_{1}\right) &=\sum_{x_{0} \in X} P\left(x_{1} \mid x_{0}, \bar{u}_{0}\right) \operatorname{bel}\left(x_{0}\right) \\&=P\left(x_{1} \mid \bar{x}_{0}, \bar{u}_{0}\right) \operatorname{bel}\left(\bar{x}_{0}\right)+P\left(x_{1} \mid x_{0}, \bar{u}_{0}\right) \operatorname{bel}\left(x_{0}\right) \\&=0 \cdot 0.5+1 \cdot 0.5 \\&=0.5\end{aligned}\)

 

and the predicted belief that the floor is dirty:

 

\(\begin{aligned}\operatorname{bel}_{p}\left(\bar{x}_{1}\right) &=\sum_{x_{0} \in X} P\left(\bar{x}_{1} \mid x_{0}, \bar{u}_{0}\right) \operatorname{bel}\left(x_{0}\right) \\&=P\left(\bar{x}_{1} \mid \bar{x}_{0}, \bar{u}_{0}\right) \operatorname{bel}\left(\bar{x}_{0}\right)+P\left(\bar{x}_{1} \mid x_{0}, \bar{u}_{0}\right) \text { bel }\left(x_{0}\right) \\&=1 \cdot 0.5+0 \cdot 0.5 \\&=0.5\end{aligned}\)

Since no action has been taken, the state probabilities are unchanged. Based on the measurement \(\bar{z}_{1}\) = dirty the belief can be corrected:

\(\operatorname{bel}\left(x_{1}\right)=\eta p\left(\bar{z}_{1} \mid x_{1}\right) \operatorname{bel}_{p}\left(x_{1}\right)=\eta 0.2 \cdot 0.5=\eta 0.1\)

and

\(\operatorname{bel}\left(\bar{x}_{1}\right)=\eta p\left(\bar{z}_{1} \mid \bar{x}_{1}\right) \text { bel }_{p}\left(\bar{x}_{1}\right)=\eta 0.9 \cdot 0.5=\eta 0.45\)

After the normalization factor η is evaluated:

\(\eta=\frac{1}{0.1+0.45}=1.82\)

The final values of the belief can be determined:

\(\operatorname{bel}\left(x_{1}\right)=0.182, \quad \text { bel }\left(\overline{x_{1}}\right)=0.818\)

The procedure can be repeated for time step k = 2, where\(u_{1}\) = clean and \(z_{2}\) = clean:

\(\begin{aligned}b e l_{p}\left(x_{2}\right) &=0.8364, & & \text { bel }_{p}\left(\bar{x}_{2}\right)=0.1636 \\\operatorname{bel}\left(x_{2}\right) &=0.9761, & & \operatorname{bel}\left(\bar{x}_{2}\right) &=0.0239\end{aligned}\)

And it can be continued for time step k = 3, where \(u_{2}\) = clean and \(z_{3}\) = clean:

\(\begin{aligned}& b e l_{p}\left(x_{3}\right) &=0.9952, & & \text { bel }_{p}\left(\bar{x}_{3}\right) &=0.0048 \\\operatorname{bel}\left(x_{3}\right) &=0.9994, & & \operatorname{bel}\left(\bar{x}_{3}\right) &=0.0006\end{aligned}\)
