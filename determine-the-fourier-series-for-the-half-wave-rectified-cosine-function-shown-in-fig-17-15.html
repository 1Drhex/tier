Guest Question.

Determine the Fourier series for the half-wave rectified cosine function shown in Fig. 17.15.

Admin
Solution

This is an even function so that \( b_n = 0\).Also, \( T = 4, ω_0= 2p/T= π/2 \) Over a period,

\( f(t)=\begin{cases} 0, & -2<t<-1 \\ \cos \frac{\pi }{2}t, & -1<t<1 \\ 0, & 1<t<2 \end{cases} \)

 

\( a _0 = \frac{2}{T} \int_{0}^{T/2}{f(t)} dt = \frac{2}{4} \biggl[\int_{0}^{1}{\cos \frac{\pi }{2}t } dt + \int_{1}^{2}{0} dt\biggr] = \frac{1}{2} \frac{2}{\pi } \sin \frac{\pi }{2} t\mid ^1_0 = \frac{1}{\pi } \)

 

\(a _n = \frac{4}{T} \int_{0}^{T/2}{f(t) \cos n\omega _0 t} dt = \frac{4}{4} \biggl[\int_{0}^{1}{\cos \frac{\pi }{2} t \cos \frac{n\pi t}{2} dt+0} \biggr] \)

 

But \( \cos A \cos B = \frac{1}{2} [\cos(A + B) + \cos(A B)] \). Then

\(a _n = \frac{1}{2} \int_{0}^{1}{\biggl[\cos \frac{\pi }{2}(n+1) t+\cos \frac{\pi }{2} (n-1) t\biggr] }dt \)

 

For n = 1,

\( a _1 = \frac{1}{2} \int_{0}^{1}{[\cos \pi t+1]} dt = \frac{1}{2} \biggl[\frac{\sin \pi t}{\pi } +t\biggr]\mid ^1_0 = \frac{1}{2} \)

 

For n>1,

\( a _n = \frac{1}{\pi (n+1)} \sin \frac{\pi }{2} (n+1) + \frac{1}{\pi (n-1)} \sin \frac{\pi }{2} (n-1) \)

 

For n = odd (n = 1, 3, 5, ……), (n +1) and (n – 1) are both even, so

\( \sin \frac{\pi }{2} (n+1) = 0 =\sin \frac{\pi }{2} (n-1) \), n = odd

For n = even (n = 2, 4, 6, ……), (n + 1) and (n – 1) are both odd. Also,

\( \sin \frac{\pi }{2} (n+1) = -\sin \frac{\pi }{2} (n-1) = \cos \frac{n\pi }{2} = (-1)^{n/2} \) , n = even

Hence,

\( a _n =\frac{(-1)^{n/2}}{\pi (n+1)} + \frac{-(-1)^{n/2}}{\pi (n-1)} = \frac{-2(-1)^{n/2}}{\pi (n^2-1)} \) n = even

Thus,

\( f(t) = \frac{1}{\pi } + \frac{1}{2} \cos \frac{\pi }{2} t-\frac{2}{\pi } \sum\limits_{n=even}^{\infty }{\frac{(-1)^{n/2}}{(n^2-1)} \cos \frac{n\pi }{2} t} \)

 

To avoid using n = 2, 4, 6, ……and also to ease computation, we can replace n by 2k, where k = 1,2,3,….and obtain

\( f(t) = \frac{1}{\pi } + \frac{1}{2} \cos \frac{\pi }{2} t-\frac{2}{\pi } \sum\limits_{k=1}^{\infty }{\frac{(-1)^{k}}{(4k^2-1)} \cos k π t} \)

 

which is a Fourier cosine series.
