Guest Question.

Find the Laplace transforms of the following functions, using the information in Table 3.1. (However, some of the individual terms in these functions may not have Laplace transforms.)

\(\text { (a) } f(t)=5+e^{-3 t}+t e^{-4 t}\)

 

\(\text { (b) } f(t)=\sin (4 t)+t-3+e^{-(t-3)}+5 / t\)

 

\(\text { (c) } f(t)=t \cos (4 t)+t / 5\)

 

\(\text { (d) } f(t)=(t-1) \cos (4(t-1))+t^{2}\)

Solution

(a)

\(f(t)=5+e^{-3 t}+t e^{-4 t}\)

 

Transform each term using rules 2, 5, and 7 from Table 3.1, respectively.

 

\(F(s)=\frac{5}{s}+\frac{1}{s+3}+\frac{1}{(s+4)^{2}}\)

 

(b)

\(f(t)=\sin (4 t)+(t-3) S(t-3)+e^{-(t-3)} S(t-3)+\frac{5}{t}\)

 

  • To transform sin(4t), use rule 14 from Table 3.1
  • To transform (t-3)S(t-3) use rules 3 and 26 together. To use rule 26, set f(t) = t and \(t_{0}=3\).
  • To transform \(e ^{-(t-3)} S (t-3)\) use rules 5 and 26 together. To use rule 26, set \(f(t)= e ^{-t}\) and \(t_{0}=3\).
  • Note that there is no Laplace transform for 1/t.

 

\(F(s)=\frac{4}{s^{2}+16}+\frac{e^{-3 s}}{s^{2}}+\frac{e^{-3 s}}{s+1}+5 L \left(\frac{1}{t}\right)\)

 

(c)

\(f(t)=e^{-t} \cos (4 t)+\frac{t}{5}\)

 

  • To transform the first term, use rule 18 from Table 3.1
  • To transform the second term, use rule 3 from Table 3.1

 

\(F(s)=\frac{s+1}{(s+1)^{2}+16}+\frac{1}{5 s^{2}}\)

 

(d)

\(f(t)=S(t-1) \cos (4(t-1))+t^{2}\)

 

  • To transform the first term, use rules 15 and 26 together. To use rule 26, set \(f(t)=\cos (4 t) \text { and } t_{0}=1\).
  • To transform the second term, use rule 4 of Table 3.1.

 

\(F(s)=e^{-s} \frac{s}{s^{2}+16}+\frac{2}{s^{3}}\)
