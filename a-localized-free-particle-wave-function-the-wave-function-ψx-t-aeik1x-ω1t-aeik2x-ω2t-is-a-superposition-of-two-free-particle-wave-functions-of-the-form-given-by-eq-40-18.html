A LOCALIZED FREE-PARTICLE WAVE FUNCTION

The wave function \(\Psi(x, t)=A e^{i\left(k_{1} x-\omega_{1} t\right)}+A e^{i\left(k_{2} x-\omega_{2} t\right)}\) is a superposition of two free-particle wave functions of the form given by Eq. (40.18). Both \(k_1\) and \(k_2\) are positive. (a) Show that this wave function satisfies the Schrödinger equation for a free particle of mass m. (b) Find the probability distribution function for Ψ(x, t).

\(\Psi(x, t)=A e^{i(k x-\omega t)}=A e^{i k x} e^{-i \omega t}\) (40.18)

Admin
Solution

IDENTIFY and SET UP:

Both wave functions \(A e^{i\left(k_{1} x-\omega_{1} t\right)}\) and \(A e^{i\left(k_{2} x-\omega_{2} t\right)}\) represent a particle moving in the positive x-direction, but with different momenta and kinetic energies: \(p_{1}=\hslash k_{1}\) and \(E_{1}=\hslash \omega_{1}=\hslash^{2} k_{1}^{2} / 2 m\) for the first function, \(p_{2}=\hslash k_{2} \text { and } E_{2}=\) \(\hbar \omega_{2}=\hslash^{2} k_{2}^{2} / 2 m\) for the second function. To test whether a superposition of these is also a valid wave function for a free particle, we’ll see whether our function Ψ(x, t) satisfies the free-particle Schrödinger equation, Eq. (40.15). We’ll use the derivatives of the exponential function: \((d / d u) e^{a u}=a e^{a u} \text { and }\left(d^{2} / d u^{2}\right) e^{a u}=a^{2} e^{a u}\).

The probability distribution function 0 \(|\Psi(x, t)|^{2}\) is the product of Ψ(x, t) and its complex conjugate.

\(-\frac{\hslash^{2}}{2 m} \frac{\partial^{2} \Psi(x, t)}{\partial x^{2}}=i \hslash \frac{\partial \Psi(x, t)}{\partial t}\) (40.15)

EXECUTE:

(a) If we substitute Ψ(x, t) into Eq. (40.15), the left-hand side of the equation is

\(\begin{aligned}-\frac{\hslash^{2}}{2 m} \frac{\partial^{2} \Psi(x, t)}{\partial x^{2}} &=-\frac{\hslash^{2}}{2 m} \frac{\partial^{2}\left(A e^{i\left(k_{1} x-\omega_{1} t\right)}+Ae^{i\left(k_{2} x-\omega_{2} t\right)}\right)}{\partial x^{2}} \\&=-\frac{\hslash^{2}}{2 m}\left[\left(i k_{1}\right)^{2} A e^{i\left(k_{1} x-\omega_{1} t\right)}+\left(i k_{2}\right)^{2} Ae^{i\left(k_{2} x-\omega_{2} t\right)}\right] \\&=\frac{\hslash^{2} k_{1}^{2}}{2 m} Ae^{i\left(k_{1}x-\omega_{1} t\right)}+\frac{\hslash^{2} k_{2}^{2}}{2 m} A e^{i\left(k_{2} x-\omega_{2} t\right)}\end{aligned}\)

The right-hand side is

\(\begin{aligned}i \hslash \frac{\partial \Psi(x, t)}{\partial t} &=i \hslash\frac{\partial\left(A e^{i\left(k_{1} x-\omega_{1} t\right)}+A e^{i\left(k_{2} x-\omega_{2} t\right)}\right)}{\partial t} \\&=i \hslash\left[\left(-i \omega_{1}\right) Ae^{i\left(k_{1} x-\omega_{1} t\right)}+\left(-i \omega_{2}\right) A e^{i\left(k_{2} x-\omega_{2} t\right)}\right] \\&=\hslash \omega_{1} A e^{i\left(k_{1} x-\omega_{1} t\right)}+\hslash \omega_{2} A e^{i\left(k_{2} x-\omega_{2} t\right)}\end{aligned}\)

The two sides are equal, provided that \(\hslash \omega_{1}=\hslash^{2} k_{1}^{2} / 2 m\) and \(\hbar \omega_{2}=\hbar^{2} k_{2}^{2} / 2 m\). These are just the relationships that we noted above. So we conclude that \(\Psi(x, t)=A e^{i\left(k_{1} x-\omega_{1} t\right)}+A e^{i\left(k_{2} x-\omega_{2} t\right)}\) is a valid free-particle wave function. In general, if we take any two wave functions that are solutions of the Schrödinger equation and then make a superposition of these to create a third wave function Ψ(x, t), then Ψ(x, t) is also a solution of the Schrödinger equation.

(b) The complex conjugate of Ψ(x, t) is

\(\Psi^{*}(x, t)=A^{*} e^{-i\left(k_{1} x-\omega_{1} t\right)}+A^{*} e^{-i\left(k_{2} x-\omega_{2} t\right)}\)

Hence

\(\begin{aligned}&|\Psi(x, t)|^{2} \\&=\Psi^{*}(x, t) \Psi(x, t) \\&=\left(A^{*} e^{-i\left(k_{1}x-\omega_{1} t\right)}+A^{*} e^{-i\left(k_{2} x-\omega_{2} t\right)}\right)\left(Ae^{i\left(k_{1} x-\omega_{1} t\right)}+A e^{i\left(k_{2} x-\omega_{2} t\right)}\right) \\&=A^{*} A\left[\begin{array}{c}e^{-i\left(k_{1} x-\omega_{1} t\right)} e^{i\left(k_{1} x-\omega_{1} t\right)}+e^{-i\left(k_{2} x-\omega_{2} t\right)} e^{i\left(k_{2} x-\omega_{2}t\right)} \\+e^{-i\left(k_{1} x-\omega_{1} t\right)} e^{i\left(k_{2} x-\omega_{2} t\right)}+e^{-i\left(k_{2} x-\omega_{2} t\right)} e^{i\left(k_{1} x-\omega_{1} t\right)}\end{array}\right] \\&=|A|^{2}\left[e^{0}+e^{0}+e^{i\left[\left(k_{2}-k_{1}\right) x-\left(\omega_{2}-\omega_{1}\right) t\right]}+e^{-i\left[\left(k_{2}-k_{1}\right) x-\left(\omega_{2}-\omega_{1}\right)t\right]}\right]\end{aligned}\)

To simplify this expression, recall that \(e^0\) = 1. From Euler’s formula, \(e^{i \theta}=\cos \theta+i \sin \theta\) and \(e^{-i \theta}=\cos \theta-i \sin \theta, \text { so } e^{i \theta}+e^{-i \theta}=\) \(2 \cos \theta\).

Hence

\(\begin{aligned}|\Psi(x, t)|^{2} &=|A|^{2}\left\{2+2 \cos \left[\left(k_{2}-k_{1}\right) x-\left(\omega_{2}-\omega_{1}\right) t\right]\right\} \\&=2|A|^{2}\left\{1+\cos\left[\left(k_{2}-k_{1}\right) x-\left(\omega_{2}-\omega_{1}\right) t\right]\right\}\end{aligned}\)

 

EVALUATE: Figure 40.5 is a graph of the probability distribution function \(|\Psi(x, t)|^{2}\) at \(t = 0\). The value of \(|\Psi(x, t)|^{2}\) varies between 0 and \(4|A|^{2}\); probabilities can never be negative! The particle has become somewhat localized: The particle is most likely to be found near a point where \(|\Psi(x, t)|^{2}\) is maximum (where the functions \(A e^{i\left(k_{1} x-\omega_{1} t\right)} \text { and } A e^{i\left(k_{2} x-\omega_{2} t\right)}\) interfere constructively) and is very unlikely to be found near a point where \(|\Psi(x, t)|^{2} = 0\) (where \(A e^{i\left(k_{1} x-\omega_{1} t\right)}\) and \(A e^{i\left(k_{2} x-\omega_{2} t\right)}\) interfere destructively).

Note also that the probability distribution function is not stationary: It moves in the positive x-direction like the particle that it represents. To see this, recall from Section 15.3 that a sinusoidal wave given by \(y(x, t)=A \cos (k x-\omega t)\) moves in the positive x-direction with speed \({v}=\omega / k \text {; since }|\Psi(x, t)|^{2}\) includes a term \(\cos \left[\left(k_{2}-k_{1}\right) x-\left(\omega_{2}-\omega_{1}\right) t\right]\). The subscript “av” reminds us that \(v_{av}\) represents the average value of the particle’s speed.
The price we pay for localizing the particle somewhat is that, unlike a particle represented by Eq. (40.18), it no longer has either a definite momentum or a definite energy. That’s consistent with the Heisenberg uncertainty principles: If we decrease the uncertainties about where a particle is and when it passes a certain point, the uncertainties in its momentum and energy must increase.

The average momentum of the particle is \(p_{\mathrm{av}}=\left(\hslash k_{2}+\hslash k_{1}\right) / 2\), which is the average of the momenta associated with the free-particle wave functions we added to create Ψ(x, t). This corresponds to the particle having an average speed \(\bar{v}_{\mathrm{av}}=p_{\mathrm{av}} / m= \left(\hslash k_{2}+\hslash k_{1}\right) / 2 m\). Can you show that this is equal to the expression \(v_{\mathrm{av}}=\left(\omega_{2}-\omega_{1}\right) /\left(k_{2}-k_{1}\right)\) that we found above?
