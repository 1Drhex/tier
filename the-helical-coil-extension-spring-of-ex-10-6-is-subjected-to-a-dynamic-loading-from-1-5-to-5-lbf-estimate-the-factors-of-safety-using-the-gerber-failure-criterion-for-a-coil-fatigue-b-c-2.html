Guest Question.

The helical coil extension spring of is subjected to a dynamic loading from 1.5 to 5 lbf. Estimate the factors of safety using the Gerber failure criterion for (a) coil fatigue, (b) coil yielding, (c) end-hook bending fatigue at point A of Fig. 10–6a, and (d) end-hook torsional fatigue at point B of Fig. 10–6b.

Admin
Solution

A number of quantities are the same as in Ex. 10–6: \( d = 0.035 \ in \ , \ S_{ut} = 264.7 \ kpsi \ ,\ D = 0.213 \ in \ , \ r_1 = 0.106 \ in \ C = 6.086 \ , \ K_B = 1.234 \ , \ \left(K\right)_A = 1.14 \ , \ \left(K\right)_B = 1.18 \ , \ N_b = 12.17 \ turns \ , \ L_0 = 0.817 \ in \ , \ k = 17.91 \ lbf/in \ , \ F_i = 1.19 \ lbf\ , \ \ and \left(\tau_i\right)_{uncorr} = 15.1 \ kpsi\). Then
\(F_a = {\left(F_{max} − F_{min}\right)}/{2} = {\left(5 − 1.5\right)}/{2} = 1.75 \ lbf \\ F_m = {\left(F_{max} + F_{min}\right)}/{2} = {\left(5 + 1.5\right)}/{2} = 3.25 \ lbf\)
The strengths from Ex. 10–6 include \(S_{ut} = 264.7 \ kpsi \ , \ S_y = 198.5 \ kpsi \ , \ and \ S_{sy} =119.1 \ kpsi\).
The ultimate shear strength is estimated from Eq. (10–30) as

\(S_{su}=0.67S_{ut} = 0.67\left(264.7\right) = 177.3 \ kpsi\)
(a) Body-coil fatigue:
\(\tau_a = \frac{8K_B F_aD}{\pi d^3}= \frac{8\left(1.234\right)1.75\left(0.213\right)}{\pi\left(0.035^3\right)}\left(10^{−3}\right) = 27.3 \ kpsi \\ \tau_m = \frac{F_m}{F_a} \tau_a = \frac{3.25}{1.75}27.3 = 50.7 \ kpsi\)

Using the Zimmerli data of Eq. (10–28)

\(S_{su}=35 \ kpsi \ S_{sm}=55 \ kpsi \)

gives

\(S_{se}=\frac{S_{su}}{1-\left(\frac{S_{sm}}{S_{su}} \right)^2 } =\frac{35}{1-\left(\frac{55}{177.3}^2 \right) } =38.7 \ kpsi\)

 

From Table 6–7, p. 307,the Gerber fatigue criterion for shear is

Fatigue factor of safety
\(n_f=\frac{1}{2} \left(\frac{S_{ut}}{\sigma _m} \right)^2\frac{\sigma _a}{S_e}\left[-1+\sqrt{1+\left(\frac{2\sigma _mS_e}{S_{ut}\sigma _a} \right)^2 } \right] \ \sigma _m\gt 0\)

 

\(\left(n_f\right )_{body} =\left(\frac{S_{su}}{\tau_m}\right) ^2\frac{\tau_a}{S_{se}} \left[-1+\sqrt{1+\left(2+\frac{\tau_m}{S_{su}}\frac{S_{se}}{\tau_a} \right)^2 } \right] \\\\ =\frac{1}{2} \left(\frac{177.3}{38.7} \right)^2\frac{27.3}{38.7} \left[-1+\sqrt{1+\left(2\frac{50.7}{177.3}\frac{38.7}{27.3} \right)^2 } \right] =1.24\)

(b) The load-line for the coil body begins at \(S_{sm} = \tau_i \ and \ has \ a \ slope \ r = {\tau_a}/{\left(\tau_m − \tau_i\right )}\). It can be shown that the intersection with the yield line is given by \(\left(S_{sa}\right)_y =\left[{r}/{\left(r + 1\right)}\right]\left(S_{sy} − \tau_i\right )\). Consequently, \(\tau_i = \left({F_i}/{F_a}\right)\tau_a = \left({1.19}/{1.75}\right)27.3 = 18.6 \ kpsi \ , \ r = 27.3 \left(50.7 − 18.6\right) = 0.850\) , and

\(\left(S_{sa}\right) _y=\frac{0.850}{0.850+1} \left(119.1-18.6\right) -46.2 \ kpsi\)
Thus,
\(\left(n_y\right)_{body}=\frac{\left(S_{sa}\right)_y }{\tau _a} =\frac{46.2}{27.3} =1.69\)
(c) End-hook bending fatigue: using Eqs. (10–34)

\(\sigma _A=F\left[\left(K\right)_A\frac{16D}{\pi d^3} +\frac{4}{\pi d^2} \right] \)

 

and (10–35)

 

\(\left(K\right) _A=\frac{4C^2_1-C_1-1}{4C_1\left(C_1-1\right) } \ \ C_1=\frac{2r_1}{d} \) gives

 

\(\sigma _a=F_a\left[\left(K\right)_A\frac{16D}{\pi d^3}+\frac{4}{\pi d^2} \right] \\\\ =1.75\left[1.14\frac{16\left(0.213\right) }{\pi \left(0.035^3\right) } +\frac{4}{\pi \left(0.035^2\right) } \right] \left(10^{-3}\right) =52.3 \ kpsi \\\\ \sigma _m=\frac{F_m}{F_a} \sigma _a=\frac{3.25}{1.75}52.3 =97.1 \ kpsi\)

To estimate the tensile endurance limit using the distortion-energy theory,

\(S_e={S_{se}}/0.577={38.7}/{0.577}=67.1 \ kpsi\)

Using the Gerber criterion for tension gives

\(\left(n_f\right) _A=\frac{1}{2} \left(\frac{S_{ut}}{\sigma _m} \right) ^2\frac{\sigma _a}{S_e} \left[-1+\sqrt{1+\left(2\frac{\sigma _m}{S_{ut}} \frac{S_e}{\sigma _a} \right)^2 } \right] \\ =\frac{1}{2} \left(\frac{264.7}{97.1} \right)^2\frac{52.3}{67.1} \left[-1+\sqrt{1+\left(2\frac{97.1}{264.7}\frac{67.1}{52.3} \right)^2 } \right] =1.08\)

(d) End-hook torsional fatigue: from Eq. (10–36)

\(\tau _B=\left(K\right) _B\frac{8FD}{\pi d^3} \) \(\left(\tau _B\right) =\left(K\right) _B\frac{8FD}{\pi d^3} =1.18\frac{8\left(1.75\right)0.213 }{\pi \left(0.035^3\right) } \left(10^{-3}\right) =26.1 \ kpsi \\ \left(\tau _m\right) _B=\frac{F_m}{F_a} \left(\tau _a\right)_B =\frac{3.25}{1.75} 26.1=48.5 \ kpsi\)

Then, again using the Gerber criterion, we obtain

\(\left(n_f\right) _B=\frac{1}{2} \left(\frac{S_{su}}{\tau _m} \right) ^2\frac{\tau _a}{S_{se}} \left[-1+\sqrt{1+\left(2\frac{\tau _m}{S_{su}}\frac{S_{se}}{\tau _a} \right)^2 } \right] \\ =\frac{1}{2}\left(\frac{177.3}{48.5} \right) ^2\frac{26.1}{38.7} \left[-1+\sqrt{1+\left(2\frac{48.5}{177.3}\frac{38.7}{26.1} \right) ^2} \right] =1.30\)
