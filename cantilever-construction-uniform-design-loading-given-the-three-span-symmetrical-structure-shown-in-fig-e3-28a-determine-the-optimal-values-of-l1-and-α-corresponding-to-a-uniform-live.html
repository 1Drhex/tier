Guest Question.

Cantilever Construction—Uniform Design Loading

Given: The three-span symmetrical structure shown in Fig. E3.28a.

Determine: The optimal values of \(L_{1}\) and \(\alpha\) corresponding to a uniform live loading w.

Admin
Solution

Solution: Using the results of the previous example, first, we establish the influence lines for the moment at mid-span of ab \(\left(M_{1-1}\right)\), at point b \(\left(M_{ b }\right)\), and at mid-span of member cd \(\left(M_{2-2}\right)\). They are plotted in Fig. E3.28b.

 

We suppose that the uniformly distributed loading can be applied on an arbitrary segment of a span. We start with the side span, ab. Based on the influence line, we load span ab (Fig. E3.28c).

 

Next, we load the center span. Loading the segment bcd produces the maximum values for \(M_{ b }\) and \(M_{ cd }\) (Fig. E3.28d). The third option is to load the center span (Fig. E3.28e).

 

The peak values for these loading schemes are

 

\(M_{ ab }=\frac{w L_{1}^{2}}{8}\)

 

\(M_{ cd }=\frac{w L_{2}^{2}}{8}(1-2 \alpha)^{2}\)

 

\(M_{ bc }=\frac{w L_{2}^{2}}{2}\left[\alpha^{2}+\alpha(1-2 \alpha)\right]\)

 

The remaining steps are the same as for the previous example. We want to use a constant crosssection for the total span and therefore equate the design moments. This step results in values for \(\alpha\) and \(L_{1}\).

 

Setting \(M_{ bc }=M_{ cd }\) results in

 

\(\frac{1}{8}(1-2 \alpha)^{2}=\left[\alpha^{2}+\alpha(1-2 \alpha)\right] 8 \alpha^{2}-8 \alpha+1=0 \downarrow \alpha=\frac{1}{2}\left(1-\frac{\sqrt{2}}{2}\right)=0.147\)

 

Setting \(M_{ ab }=M_{ cd }\) leads to

 

\(\frac{w L_{1}^{2}}{8}=\frac{w L_{2}^{2}}{8}(1-2 a)^{2}=\frac{w L_{2}^{2}}{8}\left(\frac{\sqrt{2}}{2}\right)^{2}\) \(L_{2}=0.707 L_{2}\)

 

\(\therefore L_{1}=\frac{\sqrt{2}}{2}\)

 

Lastly, \(L_{2}\) is related to the total span by

 

\(2 L_{1}+L_{2}=L_{ T }\)

 

\(\Downarrow\)

 

\((1+\sqrt{2}) L_{2}=L_{ T }\)

 

\(\Downarrow\)

 

\(L_{2}=0.414 L_{ T }\)

 

These results are close to the values based on using a single concentrated load.
