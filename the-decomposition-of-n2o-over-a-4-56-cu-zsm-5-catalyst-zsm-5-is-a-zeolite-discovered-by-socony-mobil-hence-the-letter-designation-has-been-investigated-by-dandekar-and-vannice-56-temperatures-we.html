Solution

Plotting in rate vs. in \(\ell n P_{i}\) using a power rate law gives the following reaction orders:

T(K) Reaction Order
\(N_{2}O\) \(O_{2}\) \(N_{2}\)
623 0.08 -0.031 0
653 0.24 -0.12 0
673 0.31 -0.07 0

The simplest L-H model would be for unimolecular decomposition:

(1) \(2\left[N_{2}O + * \xrightleftharpoons[]{ K_{N_{2} O} } N_{2} O * \right]\)

(2) \(2 \left[ N_{2} O * \xrightarrow[]{K} N_2 +O*\right] \)

(3) \(2 O* \xrightleftharpoons[]{1/K_{O_{2} } } O_{2} + 2 *\)

——————————————–

\(2 N_{2}O \Longrightarrow 2 N_{2} + O_{2}\)

 

\(r_{m} = \frac{1}{m} \frac{d N_{N_{2} } }{dt} = \frac{1}{m} \left\lgroup – \frac{d N_{N_{2}O } }{ dt} \right\rgroup = k \left[N_{2} O* \right]\)

From (1) : \(K_{N_{2}O }= \frac{\left[N_{2}O * \right] }{ P_{N_{2}O } \left[*\right], } , so \left[N_{2}O * \right]= K_{N_{2}O } P_{N_{2}O }\left[*\right]\)

From (3) : \(K_{O_{2} } = \frac{\left[O*\right]^{2} }{P_{O_{2} } \left[*\right]^{2} } , so \left[O*\right] = K^{1/2}_{O_{2} } P^{1/2}_{O_{2} } \left[*\right]\)

Site balance gives: \(L= \left[*\right] + \left[N_{2}O *\right]+ \left[O*\right]\)

\(L= \left[*\right]+ K_{N_{2}O } P_{N_{2}O } \left[*\right] + K^{1/2}_{O_{2} } P^{1/2}_{O_{2} } \left[*\right], and , \left[*\right]= L/\left(1+ K_{N_{2}O } P_{N_{2}O } + K^{1/2}_{O_{2} } P^{1/2}_{O_{2} } \right) ,\)

consequently,

\(r= K K_{N_{2}O } P_{N_{2}O } \left[*\right] = LK K_{N_{2}O } P_{N_{2}O }/ \left(1+ K_{N_{2}O } P_{N_{2}O }+K^{1/2}_{O_{2} } P^{1/2}_{O_{2} } \right)\)

Arrhenius plots ofthe fitting parameters listed in Table 2 provide the following values:

For \(K_{N_{2}O } : \Delta H^{o}_{ad} = – 17 \ kcal \ mole ^{-1} \ and \Delta S^{o}_{ad} – 21 \ cal \ mole ^{-1} K^{-1}\) (e.u)

For \(K_{O_{2} } : \Delta H^{o}_{ad} = – 25 \ kcal \ mole ^{-1} \ and \Delta S^{o}_{ad} =- 35 e.u\)

For \(K : E_{RDS} = 57 \ kcal \ mole ^{-1}\)

The enthalpy and entropy values for adsorption fulfill all the guidelines in Table 6.9, thus they are consistent.
From either a linear extrapolation of the high- P portions of the two isotherms in Figure 1 or using the difference between the two at 100 Torr CO pressure, the irreversible uptake is 580 μ \(mole \) CO \(g^{-1}\). The dispersion of Cu is: \(D_{Cu} = Cu_{s}/ Cu_{tot}\) ,and with \(CO_{ad}/ Cu_{s}=1\) ,

\(D_{Cu}= \frac{580 \mu mole \ Cu_{s} g^{-1}_{cat} }{\left(0.0456 \ Cu \ g^{-1}_{cat} \right) \left(mole \ Cu /63.55 \ g\ Cu \right) \left(10^{6} \ \mu mole/mole \right) } = 0.81 \)

Under differential reaction conditions, \(P_{O_{2} } \approx O\) and can be ignored; therefore, an easy way is to choose a known differential rate and correct for temperature, for example:

\(\frac{r_{823K} }{r_{673 K} } = \frac{K_{823K} }{K_{673 K} } = \frac{e^{-36200 cal/mole/\left(1.987 \ cal/mole/mole.K\right) \left(823K\right)} }{e^{-36200 cal/mole/\left(1.987 \ cal/mole/mole.K\right) \left(673K\right)} } = 139 \) thus

 

\(TOF_{823K} = \frac{r_{823K}}{ Cu_{s}} = \frac{\left(12.6 \ \mu mole/s\cdot g\right) \left(13 \ atm^{-1} \right) \left(0.0666 \ atm\right) \left(139\right) }{ \left[1+ \left(13 \ atm^{-1} \right) \left(0.0666 \ atm\right) \right] \left(580 \ \mu mole \ Cu_{s}/g\right) } = 1.4 s^{-1} \)
