Guest Question.

Finding the parameters of a practical noninverting op-amp circuit The noninverting amplifier in Fig. 3.9(a) has \( R_{1}=10 k \Omega \text { and } R_{ F }=10 k \Omega \). The op-amp parameters are \( A_{ o }=2 \times 10^{5}, f_{ b }=10 Hz , R_{ o }=75 \Omega \), and \( R_{ i }=2 M \Omega \). The frequency of the input signal is \( f_{ s }=10 kHz \). Determine (a) the unity-gain bandwidth \( f_{ bw } \) of the op-amp, (b) the closed-loop voltage gain \( A_{ f } \), and (c) the closed-loop break frequency \( f_{ c } \) of the op-amp circuit.

Admin
Solution

Using Eq. (3.28), we find the frequency-dependent voltage gain of the noninverting amplifier to be

\( A_{ f }=\frac{v_{ O }}{v_{ S }}=\frac{A_{ o }\left(R_{1}+R_{ F }\right)}{A_{ o } R_{1}+R_{1}+R_{ F }}=\frac{1+R_{ F } / R_{1}}{1+\left(1+R_{ F } / R_{1}\right) \not A_{ o }}=\frac{1+R_{ F } / R_{1}}{1+x} \) (3.28)

 

\( A_{ f }(j \omega)=\frac{1+R_{ F } / R_{1}}{1+\left(1+R_{ F } / R_{1}\right) / A_{ o }(j \omega)} \)

Substituting the frequency-dependent gain A(jω) from Eq. (3.2), we get

\( A_{0}(j \omega)=\frac{A_{ o }}{1+j \omega / \omega_{ b }}=\frac{A_{ o }}{1+j f / f_{ b }} \) (3.2)

 

\( A_{ f }(j \omega)=\frac{1+R_{ F } / R_{1}}{1+\left(1+R_{ F } / R_{1}\right) / A_{ o }+j f\left(1+R_{ F } / R_{1}\right) /\left(A_{ o } f_{ b }\right)} \) (3.33)

 

since \( \omega=2 \pi f \). If we assume that \( \left(1+R_{ F } / R_{1}\right)<<A_{0} \), which is generally the case, and substitute \( f_{ bw }=A_{ o } f_{ b } \), Eq. (3.33) becomes

\( A_{ f }(j \omega)=\frac{1+R_{ F } / R_{1}}{1+j f\left(1+R_{ F } / R_{1}\right) / f_{ bw }} \) (3.34)

 

which gives the closed-loop break (or 3-dB) frequency as

\( f_{ c }=\frac{f_{ bw }}{1+R_{ F } / R_{1}}=\frac{f_{ bw } R_{1}}{R_{1}+R_{ F }}=\beta f_{ bw }=\beta A_{ o } f_{ b } \) (3.35)

 

where \( \beta=R_{1} /\left(R_{1}+R_{ F }\right) \) is called the feedback ratio, or the feedback factor. This should not be confused with the current gain βF of bipolar transistors in Chapters 1 and 8.

The closed-loop DC gain is \( \left(1+R_{ F } / R_{1}\right) \) (as expected), and this gain also falls off at a rate of -20 dB decade after a break frequency of \( f_{ c }=\beta f_{ bw } \).

For \( R_{1}=10 k \Omega, R_{ F }=10 k \Omega, A_{ o }=2 \times 10^{5}, f_{ b }=10 Hz , \text { and } f=f_{ s }=10 kHz, \)

 

\(\begin{aligned}&\frac{R_{ F }}{R_{1}}=\frac{10 k \Omega}{10 k \Omega}=1 \\&\beta=\frac{R_{1}}{\left(R_{1}+R_{ F }\right)}=\frac{10 k \Omega}{(10 k \Omega+10 k \Omega)}=0.5\end{aligned}\)

 

which is small compared to \( A_{ o }=2 \times 10^{5} \).

(a) From Eq. (3.4),

\( f_{ bw }=A_{ o } f_{ b } \) (3.4)

 

\( f_{ bw }=A_{ o } f_{ b }=2 \times 10^{5} \times 10=2 MHz \)

 

 

(b) From Eq. (3.34), we get

\(\begin{aligned}A_{ f }(j \omega) &=\frac{1+1}{1+j f(1+1) /\left(f_{ bw }\right)}=\frac{2}{1+j\left(10 \times 10^{3}\right) \times 2 /\left(2 \times 10^{6}\right)} \\&=\frac{2}{1+j 0.01}=1.9999 \angle-0.57^{\circ}\end{aligned}\)

 

(c) From Eq. (3.35), we get

\( f_{ c }=\beta A_{ o } f_{ b }=0.5 \times 2 \times 10^{5} \times 10=1 MHz \)
