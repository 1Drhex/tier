CALCULATING DISCHARGE AND MANNING’S n USING CHEZY EQUATION

If a channel with boulders has a slope of 0.0030, is 100 ft wide, has an average depth of 4.3 ft, and is known to have a friction factor of 0.130, what is the discharge in the channel and what is the numerical value of Manning’s n for this channel?

Problem Definition

Situation: Uniform flow, channel with known f.

Find:

1. Discharge, Q, in cfs.

2. Numerical value of Manning’s n for this channel with boulders.

Plan

1. Find velocity using Eq. (15.9) with known f . Estimate \(R_{h}\) to be y, which is 4.3 ft.

\(V=\sqrt{\frac{8 g}{f} R_{h} S_{0}}\) (15.9)

2. Calculate discharge from Q = VA.

3. Solve for Manning’s n using Chezy equation for traditional units (Eq. 15.16).

\(Q=\frac{1.49}{n} A R_{h}^{2 / 3} S_{0}^{1 / 2}\) (15.16)

Admin
Solution

1. Velocity

 

\(V=\left[\sqrt{\frac{(8)\left(32.2 ft / s ^{2}\right)}{0.130}}\right][\sqrt{(4.3 ft )(0.0030)}]=5.06\) ft/s

 

2. Discharge

 

\(Q=V A=(5.06)(100 \times 4.3)=2176\) cfs

 

3. Manning’s n, using the traditional unit equation (Eq. 15.16)

 

\(n=\frac{1.49}{Q} A R_{h}^{2 / 3} S_{0}^{1 / 2}\)

 

\(n=\left(\frac{1.49}{2176 ft ^{3} / s }\right)\left(100 \times 4.3 ft ^{2}\right)(4.3 ft )^{2 / 3}(0.003)^{1 / 2}\)

 

\(n=0.0426\)

 

Review

Note: This calculated value of n is within the range of typicalvalues in Table 15.1 under the category of “Unlined Canals,Cut in rock.”

Note: This example and show that f in the
Darcy-Weisbach equation can be related to Manning’s n for uniform-flow conditions.
