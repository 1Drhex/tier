Guest Question.

Standard air-filled rectangular waveguides have been designed for the radar bands listed in Table 6-4. The inner dimensions of a waveguide suitable for X -band applications arc: \(a = 2.29 \)cm (\(0.90\) in.) and \(h = 1.02\) cm (\(0.40\) in.). If it is desired that the waveguide operates only in the dominant \({ TE }_{ 10 }\) mode and that the opera ting frequency be at lea t \(25\)% above the cutoff frequency of the \({ TE }_{ 10 }\) mode but no higher than \(95\)% of the next higher cutoff frequency, what is the allowable operating-frequency range?

Band Frequency Range (GHz) Wavelength Range (cm)
U 40-60 0.75-0.50
Ka 26.5-40 1.13-0.75
K 18-26.5 l.67-1.13
Ku 12.4-18 2.42-1.67
X 8-12.4 3.75-2.42
C 4-8 7.5-3.75
S 2-4 15-7.5
L 1-2 30-15

Solution

For \(a=2.29\times { 10 }^{ -2 } (m)\) and \(h = 1.02 \times { 10 }^{ -2 } (m)\). the two mode having the lowest cutoff frequencies are \({ TE }_{ 10 }\) and \({ TE }_{ 20 }\). Using Eq. (9-68), we find

 

\(\left( { f }_{ c } \right) _{ 10 }=\frac { c }{ 2a } =\frac { 3\times { 10 }^{ 8 } }{ 2\times 2.29\times { 10 }^{ -2 } } =6.55\times { 10 }^{ 9 }\quad \left( Hz \right) \)

 

\(\left( { f }_{ c } \right) _{ 20 }=\frac { c }{ a } =13.10\times { 10 }^{ 9 }\quad \left( Hz \right)\)

 

Thus the allowable operating-frequency range under the specified conditions is

 

\(1.25\left( { f }_{ c } \right) _{ { TE }_{ 10 } }\le f\le 0.95\left( { f }_{ c } \right) _{ { TE }_{ 10 } }\)

 

or

 

\(8.19\quad \left( GHz \right) \le f\le 12.45\quad \left( GHz \right)\)

 

\(\left( { f }_{ c } \right) _{ mn }=\frac { h }{ 2\pi \sqrt { \mu \epsilon } } =\frac { 1 }{ 2\sqrt { \mu \epsilon } } \sqrt { { \left( \frac { m }{ a } \right) }^{ 2 }+{ \left( \frac { n }{ b } \right) }^{ 2 } } \quad \quad \left( Hz \right) \quad \quad \quad \left( 9-68 \right) \)
