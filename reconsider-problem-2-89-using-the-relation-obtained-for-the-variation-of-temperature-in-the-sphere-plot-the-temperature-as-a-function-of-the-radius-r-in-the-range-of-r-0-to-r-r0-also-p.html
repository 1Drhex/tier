Guest Question.

Reconsider . Using the relation obtained for the variation of temperature in the sphere, plot the temperature as a function of the radius r in the range of r = 0 to \( r=r_{0} \). Also, plot the center temperature of the sphere as a function of the thermal conductivity in the range of \( 10 W / m \cdot{ }^{\circ} C \) to \( 400 W / m \cdot{ }^{\circ} C \). Discuss the results. Use the EES (or other) software.

Admin
Solution

“GIVEN”

r_0=0.04 “[m]”

g_dot=4E7 “[W/m^3]”

T_s=80 “[C]”

k=15 “[W/m-C], Parameter to be varied”

“ANALYSIS”

T=T_s+g_dot/(6*k)*(r_0^2-r^2)

“Temperature distribution as a function of r”

“r is the parameter to be varied”

T_0=T_s+g_dot/(6*k)*r_0^2
