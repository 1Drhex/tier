Guest Question.

oscillator The ground-state energy eigenfunction of the harmonic oscillator

\(\left\langle x|0\right\rangle=\left(\frac{m\omega }{\pi \hbar }\right) ^{1/4} e^{-m\omega x^{2}/2\hbar }\)

is a Gaussian. Show that we can generate the coherent state |α) by simply displacing the ground state from its equilibrium position by a distance d by applying the translation operator

\(\hat{T} (d)=e^{-i\hat{p} _{x} }d/\hbar =e^{\sqrt{mw/2\hbar }d(\hat{a} ^{\dagger }-\hat{a} ) }\)

to the ground state, that is,

\( \hat{T}(d)|0〉=|\alpha 〉\)

provided we set \(\alpha =\sqrt{mw/2\hbar }d \)

Admin
Solution

This is a subtle problem. Notice that if we set \(\alpha =\sqrt{mw/2\hbar }d\) at t = 0, it becomes complex as time evolves. With this in mind, it is helpful to introduce a generalized translation, or displacement, operator

\( \hat{D}(\alpha ) = e^{(\alpha \hat{a} ^{\dagger } -\alpha ^{\ast }\hat{a})}\)

which reduces to

\(\hat{T}(d)=e^{-i\hat{p} _{x}d/h } =e^{\sqrt{mw/2\hbar }d(\hat{a} ^{\dagger }-\hat{a} ) }\)

when \(\alpha =\sqrt{mw/2\hbar }d\).To establish that

\( \hat{D}(\alpha )|0 〉=|\alpha 〉\)

we need to make use of the identity

\(e^{\check{A}+\check{B} }=e^{\check{A} }e^{\check{B} }e^{-\left[\check{A},\check{B} \right]/2 }\)

which holds when the operators \(\hat{A}\) and \(\hat{A}\) each commute with the commutator \(\left[\hat{A},\hat{B}\right] \) See Problem 7.19. In our case, we take \(\hat{A}=\alpha \hat{a} ^{\dagger }\) and \(\hat{B}=-\alpha ^{\ast }\hat{a} \) Since the commutator \(\left[\hat{a} ,\hat{a} ^{\dagger } \right]=1\) 1, which clearly commutes with \(\hat{a}\) and \(\hat{a} ^{\dagger }\) , we can safely apply this operator identity in this case. Therefore

\(\hat{D}(\alpha )=e^{\alpha \hat{a} ^{\dagger } }e^{-\alpha ^{\ast }\hat{a} } e^{\left|\alpha \hat{a} ^{\dagger } -\alpha ^{\ast }\hat{a} \right| /2}\)

and consequently

\(\hat{D}(\alpha )|0〉=e^{-\left|\alpha \right| ^{2}/2 }\sum\limits_{n=0}^{\infty }{\frac{a^{n} }{\sqrt{n!} }|n〉 }=|\alpha 〉\)

In deriving this result we have made use of the Taylor series expansion for the operators \(e^{\alpha \hat{a} ^{\dagger } }\) and \(e^{-\alpha ^{\ast }\hat{a} }\) . Thus displacing the ground state from its equilibrium position generates a coherent state, a Gaussian wave function that oscillates back and forth, as indicated in Fig. 7 .11.
