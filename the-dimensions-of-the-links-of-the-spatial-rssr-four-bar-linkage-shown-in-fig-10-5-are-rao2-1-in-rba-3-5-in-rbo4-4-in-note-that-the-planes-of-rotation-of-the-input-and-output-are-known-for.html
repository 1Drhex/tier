Guest Question.

The dimensions of the links of the spatial RSSR four-bar linkage shown in Fig. 10.5 are \( R_{A O_{2}}=1 \text { in, } R_{B A}=3.5 \text { in, } R_{B O_{4}}=4 \text { in. } \) Note that the planes of rotation of the input and output are known. For the input angle θ2 = −45°, determine the postures of the coupler 3 and output link 4.

Admin
Solution

GRAPHIC SOLUTION

If we replace link 4 by vector \(R _{B O_{4}}.\) then the only unknown is the angle θ4, since the magnitude of the vector and the plane of rotation are given. Similarly, if link 3 is replaced by vector \(R _{B A},\) then the magnitude is known, and there are only two unknown spheric coordinate angular direction variables for this vector. The loop-closure equation is

\(\left( R _{B A}\right)+\left(- R _{B O_{4}}\right)+\left( R _{A O_{2}}- R _{O_{4} O_{2}}\right)= 0\) (1)

 

We identify this as case 2d in Table 10.1, requiring the solution of a second-degree (quadratic) polynomial, and hence it yields two solutions.

This problem is solved graphically using two orthographic views, the frontal and profile views. If we imagine, in Fig. 10.5, that the coupler is disconnected from the output crank at B and allowed to occupy any possible position with respect to A, then B of link 3 must lie on the surface of a sphere of known radius with its center at A. With joint B still disconnected, the locus of B of link 4 is a circle of known radius about O4 in a plane parallel to the yz plane. Therefore, to solve this problem, we need to find the two points of intersection of a circle and a sphere.

The solution is shown in Fig. 10.6, where the subscripts F and P denote projections on the frontal and profile planes, respectively. First, we locate points O2, A, and O4 in both
views. In the profile view, we draw a circle of radius \(R_{B O_{4}}=4\) in about center O4P; this is the locus of point B4. This circle appears in the frontal view as the vertical line MFO4FNF.

Next, in the frontal view, we construct the outline of a sphere with AF as center and coupler length RBA = 3.5 in as its radius. If MFO4FNF is regarded as the trace of a plane normal to the frontal view plane, the intersection of this plane with the sphere appears as the full circle in the profile view, having diameter MPNP = MFNF. The circular arc of radius \(R_{B O_{4}}\) intersects the circle at two points, yielding two solutions. One of the points is chosen as BP and is projected back to the frontal view to locate BF. Links 3 and 4 are drawn next as the dashed lines shown in the profile and frontal views of Fig. 10.6.

Measuring the x, y, and z projections from the graphic solution, we can write the vectors for each link:

 

\(\begin{aligned}R _{O_{4} O_{2}} &=3.00 \hat{ i }-2.00 \hat{ k } \text { in } \\R _{A O_{2}} &=0.71 \hat{ i }-0.71 \hat{ j } \text { in } \\R _{B A} &=2.30 \hat{ i }+1.95 \hat{ j }+1.77 \hat{ k } i n \\R _{B O_{4}} &=1.22 \hat{ j }+3.81 \hat{ k } \text { in }\end{aligned}\)

These components were obtained from a true-size graphic solution; better accuracy is possible, of course, by making the drawings two or four times actual size.

ANALYTIC SOLUTION

The given information establishes that

 

\(\begin{aligned}R _{O_{4} O_{2}} &=3.000 \hat{ i }-2.000 \hat{ k } \text { in } \\R _{A O_{2}} &=\cos \left(-45^{\circ}\right)+\sin\left(-45^{\circ}\right)=0.707 \hat{ i }-0.707 \hat{ j } \text { in } \\R _{B A} &=R_{B A}^{x} \hat{ i }+R_{B A}^{y} \hat{ j }+R_{B A}^{z} \hat{ k } \\R _{B O_{4}} &=-4.000 \sin \theta_{4} \hat{ j }+4.000 \cos \theta_{4} \hat{ k } \text { in. }\end{aligned}\)

 

The same loop-closure equation as in Eq. (1) above must be solved. Substituting the given information gives

 

\(\begin{aligned}&\left(R_{B A}^{x} \hat{ i }+R_{B A}^{y} \hat{ j }+R_{B A}^{z} \hat{ k}\right)+\left(4.000 \sin \theta_{4} \hat{ j }-4.000 \cos \theta_{4} \hat{ k }\right) \\&\quad+(0.707 \hat{ i }-0.707 \hat{ j }-3.000 \hat{ i }+2.000 \hat{ k })= 0\end{aligned}\)

 

Then, separating this equation into its \(\hat{ i }, \hat{ j }, \text { and } \hat{ k }\) components and rearranging gives the following three equations:

 

\(\begin{aligned}&R_{B A}^{x}=2.293 in , \\&R_{B A}^{y}=-4.000 \sin \theta_{4}+0.707 in \\&R_{B A}^{z}=4.000 \cos \theta_{4}-2.000 in\end{aligned}\)

 

Next, we square and add these equations, and, remembering that RBA = 3.500 in, we obtain

 

\((2.293 in )^{2}+\left(-4.000 \sin \theta_{4}+0.707 in \right)^{2}+\left(4.000 \cos \theta_{4}-2.000 in \right)^{2}=(3.500 in )^{2} .\)

 

Expanding this equation and rearranging gives

\(-5.656 \sin \theta_{4}-16.000 \cos \theta_{4}+13.508 in ^{2}=0\)

We now have a single equation with only one unknown, namely, θ4. However, it contains both sine and cosine terms (a transcendental equation). A standard technique for dealing with this situation is to convert the equation into a second-order (quadratic) polynomial by using the tangent of half-angle identities:

 

\(\sin \theta_{4}=\frac{2 \tan \left(\theta_{4} / 2\right)}{1+\tan ^{2}\left(\theta_{4} / 2\right)} \quad \text { and } \quad \cos \theta_{4}=\frac{1-\tan ^{2}\left(\theta_{4} / 2\right)}{1+\tan ^{2}\left(\theta_{4} / 2\right)}\)

 

Substituting these identities into the above equation and multiplying by the common denominator gives a single quadratic equation in one unknown, tan( θ4/2), that is,

 

\(-11.312 \tan \left(\theta_{4} / 2\right)-16.000\left[1-\tan ^{2}\left(\theta_{4} / 2\right)\right]+13.508\left[1+\tan ^{2}\left(\theta_{4} / 2\right)\right]=0\)

or

\(29.508 \tan ^{2}\left(\theta_{4} / 2\right)-11.312 \tan \left(\theta_{4} / 2\right)-2.492=0\)

This quadratic equation can now be solved for two roots, which are

 

\(\tan \frac{\theta_{4}}{2}=\left\{\begin{aligned} 0.5398 ; & \theta_{4}=-123.3^{\circ} \\-0.1565 ; & \theta_{4}=162.2^{\circ}\end{aligned}\right.\)

 

Of these two roots, we choose the second as the solution of interest. Back-substituting into the previous conditions, we find numeric values for the four vectors, namely:

 

\(\begin{aligned}R _{O_{4} O_{2}} &=3.000 \hat{ i }-2.000 \hat{ k } \text { in } \\R _{A O_{2}} &=0.707 \hat{ i }-0.707 \hat{ j } \text { in } \\R _{B A} &=2.293 \hat{ i }+1.930 \hat{ j }+1.809 \hat{ k } \text { in } \\R _{B O_{4}} &=1.223 \hat{ j }+3.809 \hat{ k } \text { in }\end{aligned}\)

 

These results match those of the graphic solution within graphic error.
