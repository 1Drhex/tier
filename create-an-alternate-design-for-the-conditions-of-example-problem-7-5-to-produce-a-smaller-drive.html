Guest Question.

Create an alternate design for the conditions of to produce a smaller drive.

Admin
Solution

Objective Design a smaller chain drive for the application in .
Given Power transmitted = 15 hp to a conveyor
Speed of motor = 900 rpm; output speed range = 230 to 240 rpm
Analysis Use a multistrand design to permit a smaller-pitch chain to be used to transmit the same design power (21.0 hp) at the same speed (900 rpm). Use the design data presented in this section. The solution procedure is developed within the Results section of the problem solution.

Results Let’s try a four-strand chain for which the power capacity factor is 3.3. Then the required power per strand is

\(P = 21.0/3.3 = 6.36\) hp
From Table 7–14, we find that a no. 40 chain (1/2-in pitch) with a 17-tooth sprocket will be satisfactory. Type B lubrication, oil bath, can be used.
For the required large sprocket,

\(N_2 = N_1 × ratio = 17(3.83) = 65.11\)
Let’s use \(N_2 = 65\) teeth.
The sprocket diameters are

\(\begin{aligned}&P D_{1}=\frac{p}{\sin \left(180^{\circ} / N_{1}\right)}=\frac{0.500\mathrm{in}}{\sin \left(180^{\circ} / 17\right)}=2.721 \mathrm{in} \\&P D_{2}=\frac{p}{\sin \left(180 \% / N_{2}\right)}=\frac{0.500 \mathrm{in}}{\sin (180 \% / 65)}=10.349 \mathrm{in}\end{aligned}\)
For the center distance, let’s try the minimum recommended: \(C=30\) pitches.
\(30(0.50 \mathrm{in})=15.0 \mathrm{in}\)
The chain length is
\(L_{c}=2(30)+\frac{65+17}{2}+\frac{(65-17)^{2}}{4 \pi^{2}(30)}=102.9 \text { pitches }\)
Specify the integer length, \(L_{c}=104\) pitches \(=104(0.50)=52.0\) in. The actual maximum center distance is
\(\begin{aligned}&C D=\frac{1}{4}\left[104-\frac{65+17}{2}+\sqrt{\left.\left[104-\frac{65+17}{2}\right]^{2}-\frac{8(65-17)^{2}}{4 \pi^{2}}\right]}\right. \\&C D=30.54\text { pitches }=30.54(0.50)=15.272 \text { in }\end{aligned}\)
Compute the angle of wrap of the chain for each sprocket using Equations (7-21) and (7-22). Note that the minimum angle of wrap should be \(120^{\circ}\).
For the small sprocket,
\(\begin{aligned}&\theta_{1}=180^{\circ}-2 \sin ^{-1}\left[\left(P D_{2}-P D_{1}\right) /2 C\right] \\&\theta_{1}=180^{\circ}-2 \sin ^{-1}[(10.349-2.721) /(2(15.272))]=151.1^{\circ}\end{aligned}\)

Because this is greater than 120°, it is acceptable.
For the larger sprocket,

\(\begin{aligned}&\theta_{1}=180^{\circ}+2 \sin ^{-1}\left[\left(P D_{2}-P D_{1}\right) /2 C\right] \\&\theta_{2}=180^{\circ}+2 \sin ^{-1}[(10.349-2.721) /(2(15.272))]=208.9^{\circ}\end{aligned}\)

 

Comments: Summary:
Figure 7–38(b) shows the new design to the same scale as the first design. The space reduction is significant.

Chain: No. 40, 1/2-in pitch, four-strand, 104 pitches, 52.0 in length
Sprockets: No. 40–4 (four strands), 1/2-in pitch
Small: 17 teeth, \(PD_1 = 2.721\) in
Large: 65 teeth, \(PD_2 = 10.349\) in
Maximum center distance: 15.272 in
Type B lubrication (oil bath)
