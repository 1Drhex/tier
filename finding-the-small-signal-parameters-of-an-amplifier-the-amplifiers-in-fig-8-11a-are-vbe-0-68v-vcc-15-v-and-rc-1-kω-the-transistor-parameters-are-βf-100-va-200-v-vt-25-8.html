Guest Question.

Finding the small-signal parameters of an amplifier The amplifiers in Fig. 8.11(a) are \( V_{ BE }=0.68 V \), \( V_{ CC }=15 V , \text { and } R_{ C }=1 k \Omega \). The transistor parameters are \( \beta_{ F }=100, V_{ A }=200 V , V_{ T }=25.8 mV \), and \( I_{ S }=3.3 \times 10^{-14} A \).

(a) Find the DC-biasing point \( I_{ B }, I_{ C }, \text { and } V_{ CE } \).

(b) Find the small-signal transistor model parameters \( r_{\pi}, r_{ o } \text {, and } g_{ m } \).

(c) Find the small-signal amplifier parameters \( R_{ i }, R_{0}, \text { and } A_{\text {vo }}\).

Admin
Solution

(a) From Eq. (8.1), \( I_{ C }=I_{ S } \exp \left(v_{ BE } / V_{ T }\right)=3.3 \times 10^{-14} \times \exp \left[0.682 /\left(25.8 \times 10^{-3}\right)\right]=9.97 mA \), \( I_{ B }=I_{ C } / \beta_{ F }=9.97 mA / 100=99.7 \mu A , \text { and } V_{ CE }=V_{ CC }-R_{ C } I_{ C }=15 V -9.97 mA \times 1 k \Omega=5.03 V \).

 

\( i_{ C }=I_{ C , n }=I_{ E , n }-I_{ B , n }=I_{ S } e^{v_{ BE } / V_{ T }} \) (8.1)

 

(b) From Eq. (8.40), \( r_{\pi}=V_{ T } / I_{ B }=25.8 mV / 99.7 \mu A =258.8 \Omega \).

 

\( \frac{1}{r_{\pi}}=\frac{i_{ b }}{v_{ be }}=\left.\frac{d i_{ B }}{d v_{ BE }}\right|_{\text {at } Q \text {-point }}=\frac{I_{ B }}{V_{ T }}=\frac{I_{ B }}{25.8 mV } \) (8.40)

 

From Eq. (8.42), \( g_{ m }=I_{ C } / V_{ T }=9.97 mA / 25.8 mV =386 mA / V \).

 

\( g_{ m }=\frac{i_{ c }}{v_{ be }}=\left.\frac{\beta_{ F } d i_{ B }}{d v_{ BE }}\right|_{\text {at } Q_{\text {-point }}}=\frac{\beta_{ f } I_{ B }}{V_{ T }}=\frac{I_{ C }}{V_{ T }}=\frac{\beta_{ f }}{r_{\pi}} \) (8.42)

 

 

From Eq. (8.43), \( r_{ o }=V_{ A } / I_{ C }=200 V / 9.97 mA =20.06 k \Omega \).

 

\( \frac{1}{r_{ o }}=\frac{i_{ c }}{v_{ ce }}=\left.\frac{d i_{ C }}{d v_{ CE }}\right|_{\text {at } Q_{\text {-point }}}=\frac{I_{ C }}{ V _{ A }+V_{ CE }}=\frac{I_{ C }}{V_{ A }} \quad \text { for } V_{ A }>>V_{ CE } \) (8.43)

 

From Eq. (8.7), \( r_{ o }=\left(V_{ CE }+V_{ A }\right) / I_{ C }=(5.03+200) V / 9.97 mA =20.56 k \Omega \).

 

\( g_{ o }=\frac{1}{r_{ o }}=\frac{d i_{ C }}{d v_{ CE }}=\frac{I_{ C }}{V_{ CE }+V_{ A }} \) (8.7)

 

(c) \( R_{ i }=r_{\pi}=258.8 \Omega, \quad R_{ o }=r_{ o } \| R_{ C }=952.5 \Omega, \quad G_{ mo }=g_{ m }=386 mA / V , \text { and } A_{ vo }=-g_{ m }\left(r_{ o } \| R_{ C }\right)= -386 mA / V \times(20.56 k \Omega \| 1 k )=-368.5 V / V\)
