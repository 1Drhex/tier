Guest Question.

Suppose that a collector system supplies heat to an industrial process. The collector inlet temperature (process return temperature) varies as shown in Table 11.9 but, for a certain hour, is constant during the month. The calculation is done for the month of April, where \(\overline{K}_{T}\) = 0.63. The system is located at 35°N latitude and the collector characteristics are FRUL = 5.92 W/m2-°C, FR(\(\tau \alpha \))n = 0.82, tilted at 40°, and the incidence angle modifier constant bo = 0.1. The weather conditions are also given in the table. Calculate the energy output of the collector.

Solution

First, the incidence angle is calculated, from which the incidence angle modifier is estimated. The estimations are done on the half hour; for the hour 8–9, the hour angle is 52.5°. From Eq. (2.20),

\(\cos (\theta )=\sin (L-\beta )\sin (\delta )+\cos (L-\beta)\cos (\delta )\cos (h)\)

\(=\sin (35-40 )\sin (9.41 )+\cos (30-45)\cos (9.41 )\cos (52.5)=0.584\) or θ = 54.3°

From Eq. (4.25),

\(K_{\theta }=\frac{(\tau \alpha) }{(\tau \alpha)_{n} } =1-b_{o}\left[\frac{1}{\cos (\theta )}-1 \right] =1-0.1\left[\frac{1}{\cos (54.3)}-1 \right] =0.929\)

The dimensionless critical radiation level, Xc, is given by Eq. (11.40):

\(X_{c}=\frac{I_{tc}}{\overline{I_{t}} } =\frac{F_{R}U_{L}(T_{i}-T_{a})}{F_{R}(\tau \alpha )_{n}\frac{(\tau \alpha )}{(\tau \alpha )_{n}}\overline{I}_{t} } =\frac{5.92(25-9)\times 3600}{0.82\times 0.929\times 1.52\times 10^6} =0.294\)

\(\overline{H}_{o}\)= 35.8 MJ/m2. From the input data and Eq. (2.82),

\(\overline{K}_{T}=\frac{\overline{H} }{\overline{H}_{o} } \)

\(\overline{H} =\overline{K}_{T}\overline{H}_{o} =35.8\times 0.63 = 22.56\) MJ/m2

To avoid repeating the same calculations as in previous examples, some values are given directly. Therefore, hss = 96.7°, α = 0.709, and β = 0.376. From Eq. (2.84a),

\(r=\frac{\pi }{24} (\alpha +\beta \cos (h))\frac{\cos (h)-\cos (h_{ss})}{\sin (h_{ss} )-\left\lgroup\frac{2\pi h_{ss}}{360} \right\rgroup \cos (h_{ss})} \)

\(=\frac{\pi }{24} (0.709 +0.376 \cos (54.3))\frac{\cos (52.5)-\cos (96.7)}{\sin (96.7 )-\left\lgroup\frac{2\pi (96.7)}{360} \right\rgroup \cos (96.7)} \)

=0.075

From Eq. (2.83), we have

\(r_{d}=\left\lgroup\frac{\pi }{24}\right\rgroup \frac{\cos (h)-\cos (h_{ss})}{\sin (h_{ss} )-\left\lgroup\frac{2\pi h_{ss}}{360} \right\rgroup \cos (h_{ss})}\)

\(=\left\lgroup\frac{\pi }{24}\right\rgroup \frac{\cos (52.5)-\cos (96.7)}{\sin (96.7 )-\left\lgroup\frac{2\pi (96.7)}{360} \right\rgroup \cos (96.7)}\)

= 0.080

From Eq. (11.29),

\(\overline{R}_{h} =\frac{\overline{I}_{t} }{\overline{I} } =\frac{\overline{I}_{t}}{r\overline{H}} =\frac{1.52}{0.075\times 22.56} =0.898\)

The monthly average hourly clearness index, \(\overline{K}_{T}\), is given by Eq. (11.28):

\(\overline{K}_{T} =\frac{\overline{I}}{\overline{I}_{o}} =\frac{r}{r_{d}}\overline{K}_{T}=\frac{r}{r_{d}} \frac{\overline{H} }{\overline{H}_{o} } =[\alpha +\beta \cos (h)]\overline{K}_{T}\)

\(=[0.709+0.376\cos (52.5)]\times 0.63\)

=0.591

From Eq. (11.26c),

\(X_{m}=1.85+0.169\frac{\overline{R}_{h} }{\overline{k}^2_{T} } +0.0696\frac{\cos (\beta )}{\overline{k}^2_{T}} -0.981\frac{\overline{k}_{T}}{\cos ^2(\delta )} \)

\(=1.85+0.169\frac{0.898 }{(0.591)^2 } +0.0696\frac{\cos (40 )}{(0.591)^2} -0.981\frac{(0.591)}{\cos ^2(9.41 )}\)

=1.841

From Eq. (11.26b),

\(g=\frac{X_{m}-1}{2-X_{m}} =\frac{1.841-1}{2-1.841} =5.289\)

From Eq. (11.26a),

\(\Phi =\left|\left|g\right|-\left[g^2+(1+2g)\left\lgroup1-\frac{X_{c}}{X_{m}} \right\rgroup^2 \right]^{1/2} \right| \)

\(=\left|\left|5.289\right|-\left[5.289^2+(1+2\times 5.289)\left\lgroup1-\frac{0.294}{1.841} \right\rgroup^2 \right]^{1/2} \right|\)

=0.723

Finally the useful gain (UG) of the collector for that hour is (April has 30 days):

\(F_{R}(\tau \alpha )_{n}=\frac{(\tau \alpha )}{(\tau \alpha )_{n}} N\overline{I}_{t} \Phi =0.82\times 0.929\times 30\times 1.52\times 0.723\)

=25.11 MJ/m2
