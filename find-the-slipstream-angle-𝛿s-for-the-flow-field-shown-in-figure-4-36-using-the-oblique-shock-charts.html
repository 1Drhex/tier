Guest Question.

Find the slipstream angle \(\delta_{s}\) for the flow field shown in Figure 4.36, using the oblique shock charts.

Admin
Solution

From oblique shock chart 1 (Figure A.1), for \(M_{\infty}=2.0\) and \(\theta_{1}=10^{\circ}\), we get

\(\beta_{1}=39^{\circ}\)

 

For \(M_{\infty}=2.0\) and \(\theta_{1}=10^{\circ}\), from oblique shock chart 2, we get

\(M_{1}=1.665, \frac{p_{1}}{p_{\infty}}=1.686\)

 

For \(M_{\infty}=2.0\) and \(\theta_{2}=5^{\circ}\), from oblique shock chart 1, we get

\(\beta_{2}=34.5^{\circ}\)

 

For \(M_{\infty}=2.0\) and \(\theta_{2}=5^{\circ}\), from oblique shock chart 2, we get

\(M_{2}=1.805 \text { and } \frac{p_{2}}{p_{\infty}}=1.323\)

 

Because of the slipstream, the properties in regions 3 and 4 have to be calculated by trial and error only.

Trial 1

Let the down stream flow be parallel to upstream flow. For \(\theta_{3}=10^{\circ}\) and \(M_{1}=1.665\), from oblique shock chart 1,

\(\beta_{3}=48.5^{\circ}\)

 

Therefore,

\(M_{1 n 3}=M_{1} \sin \beta_{3}\\[0.5cm] =1.665 \times \sin 48.5^{\circ} \\[0.5cm] = 1.247\)

 

For \(M_{1 n 3}=1.247\), from the normal shock table,

\(p_{3} / p_{1}=1.656\)

 

Therefore,

\(\frac{p_{3}}{p_{\infty}}=\frac{p_{3}}{p_{1}} \times \frac{p_{1}}{p_{\infty}} \\[0.5cm] = 1.656 √ó 1.686 \\[0.5cm] = 2.79\)

 

For \(\theta_{4}=5^{\circ} \text { and } M_{2}=1.805\),

\(\beta_{4}=38^{\circ}\)

 

Therefore,

\(M_{2 n 4}=M_{2} \sin \beta_{4} \\[0.5cm] =1.805 \times \sin 38^{\circ} \\[0.5cm] = 1.111\)

 

From the normal shock table, for \(M_{2 n 4}=1.111\),

\(\frac{p_{4}}{p_{2}}=1.271\)

 

Therefore,

\(\frac{p_{4}}{p_{\infty}}=\frac{p_{4}}{p_{2}} \times \frac{p_{2}}{p_{\infty}} \\[0.5cm] = 1.271 √ó 1.323 \\[0.5cm] = 1.68\)

 

Because \(p_{3} / p_{\infty}>p_{4} / p_{\infty}\), the slip stream has to be deflected downwards so that the two pressures become equal.

Trial 2

Let

\(\frac{p_{3}}{p_{\infty}}=\frac{p_{4}}{p_{\infty}}=2.25\)

 

Therefore,

\(\frac{p_{3}}{p_{1}}=\frac{p_{3}}{p_{\infty}} \times \frac{p_{\infty}}{p_{1}} \\[0.5cm] =2.25 \times \frac{1}{1.686} \\[0.5cm] = 1.334\)

 

For \(p_{3} / p_{1}=1.334\), from the normal shock table,

\(M_{1 n 3}=1.135\)

 

Therefore,

\(\beta_{3}=\sin ^{-1}\left(\frac{M_{1 n 3}}{M_{1}}\right) \\[0.5cm] =43^{\circ}\)

 

For \(M_{1}=1.665\) and \(\beta_{3}=43^{\circ}\), from oblique shock chart 1,

\(\theta_{3}=6.5^{\circ}\)

 

i.e. \(3.5^{\circ}\) below freestream direction. Similarly,

\(\frac{p_{4}}{p_{2}}=\frac{p_{4}}{p_{\infty}} \times \frac{p_{\infty}}{p_{2}} \\[0.5cm] =2.25 \times \frac{1}{1.323} \\[0.5cm] = 1.7\)

 

For \(p_{4} / p_{2}=1.7\), from the normal shock table,

\(M_{2 n 4}=1.27\)

 

Therefore,

\(\beta_{4}=\sin ^{-1}\left(\frac{M_{2 n 4}}{M_{2}}\right)=\sin ^{-1}\left(\frac{1.27}{1.805}\right) \\[0.5cm] =44.7^{\circ}\)

 

For \(M_{2}=1.805\) and \(\beta_{4} =44.7^{\circ},\) from oblique shock chart 2 (Figure A.2),

\(\theta_{4}=11^{\circ}\)

 

i.e. \(6^{\circ}\) below the freestream direction.

Trial 3

With downstream flow \(4.5^{\circ}\) below upstream flow, For \(\theta_{3}=5.5^{\circ}\) and \(M_{1}=1.665\), from oblique shock chart 1,

\(\beta_{3}=42.5^{\circ}\)

 

Therefore,

\(M_{1 n 3}=M_{1} \sin 42.5=1.665 \times \sin 42.5 \\[0.5cm] = 1.125\)

 

From the normal shock table, for \(M_{1 n 3}=1.125\),

\(\frac{p_{3}}{p_{1}}=1.31\)

 

Therefore,

\(\frac{p_{3}}{p_{\infty}}=\frac{p_{3}}{p_{1}} \times \frac{p_{1}}{p_{\infty}} \\[0.5cm] = 1.31 √ó 1.686 \\[0.5cm] = 2.21\)

 

For \(\theta_{4}=9.5^{\circ}\) and \(M_{2}=1.805 \text { , }\) from oblique shock chart 1.

\(\beta_{4}=43.5^{\circ}\)

 

Therefore,

\(M_{2 n 4}=M_{2} \sin \beta_{4} \\[0.5cm] =1.805 \times \sin 43.5^{\circ} \\[0.5cm] = 1.242\)

 

For \(M_{2 n 4}=1.242\), from the normal shock table,

\(\frac{p_{4}}{p_{2}}=1.63\)

 

Thus,

\(\frac{p_{4}}{p_{\infty}}=\frac{p_{4}}{p_{2}} \times \frac{p_{2}}{p_{\infty}} \\[0.5cm] = 1.63 √ó 1.323 \\[0.5cm] = 2.16\)

 

In trial 3, it can be seen that pressures \(P_{3}\) and \(P_{4}\) are nearly equal, i.e. the assumed slipstream deflection \(\left(\delta_{s}\right)\) of \(4.5^{\circ}\) is the correct value.

Note: This problem could have been solved using oblique shock tables too. But to practice the use oblique shock charts, this example has been solved with charts.
