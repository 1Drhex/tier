Guest Question.

(a) Charge \(q_{A}\) is at rest at the origin in system S; charge \( q_{B} \text { flies by at speed } v\) on a trajectory parallel to the x axis, but at y = d. What is the electromagnetic force on \(q_{B}\) as it crosses the y axis?

(b) Now study the same problem from system \(\overline{ S },\) which moves to the right with speed \(v\). What is the force on \(q_{B} \text { when } q_{A} \text { passes the } \bar{y}\) axis? [Do it two ways: (i) by using your answer to (a) and transforming the force; (ii) by computing the fields in \(\overline{ S }\) and using the Lorentz force law.]

Admin
Solution

(a) Fields of A at B: \(E =\frac{1}{4 \pi \epsilon_{0}} \frac{q_{A}}{d^{2}} \hat{ y } ; B = 0\) . So force on \(q_{B} \text { is } \mathbf { F } = \frac { 1 } { 4 \pi \epsilon _ { 0 } } \frac { q _ { A } q _ { B } } { d ^ { 2 } } \hat { \mathbf { y } } .\)

(b) (i) From Eq. 12.67: \(\overline{ F }=\frac{\gamma}{4 \pi \epsilon_{0}} \frac{q_{A} q_{B}}{d^{2}} \hat{ y }\) (Note: here the particle is at rest in \(\overline{ S }\).)

\(\overline{ F }_{\perp}=\frac{1}{\gamma} F _{\perp}, \quad \bar{F}_{\|}=F_{\|}\) (12.67)

(ii) From Eq. 12.93, with \(\theta=90^{\circ}: \bar{E}=\frac{1}{4 \pi \epsilon_{0}} \frac{q_{A}\left(1-v^{2} / c^{2}\right)}{\left(1-v^{2} / c^{2}\right)^{3 / 2}} \frac{1}{d^{2}} \hat{ y }=\frac{\gamma}{4 \pi \epsilon_{0}} \frac{q_{A}}{d^{2}} \hat{ y }\)

 

\(E =\frac{1}{4 \pi \epsilon_{0}} \frac{\gamma_{0} q R }{\left(\gamma_{0}^{2} R^{2} \cos ^{2} \theta+R^{2} \sin ^{2} \theta\right)^{3 / 2}}=\frac{1}{4 \pi \epsilon_{0}} \frac{q\left(1-v_{0}^{2} / c^{2}\right)}{\left[1-\left(v_{0}^{2} / c^{2}\right) \sin ^{2} \theta\right]^{3 / 2}} \frac{\hat{ R }}{R^{2}}\) (12.93)

(this also follows from Eq. 12.109).

\(\begin{array}{lll}\bar{E}_{x}=E_{x}, & \bar{E}_{y}=\gamma\left(E_{y}-v B_{z}\right), & \bar{E}_{z}=\gamma\left(E_{z}+v B_{y}\right) \\\bar{B}_{x}=B_{x}, & \bar{B}_{y}=\gamma\left(B_{y}+\frac{v}{c^{2}} E_{z}\right), & \bar{B}_{z}=\gamma\left(B_{z}-\frac{v}{c^{2}} E_{y}\right)\end{array}\) (12.109)

\(\overline{ B } \neq 0, \text { but since } v_{B}=0 \text { in } \overline{ S }\) , there is no magnetic force anyway, and \( \overline{ F }=\frac{\gamma}{4 \pi \epsilon_{0}} \frac{q_{A} q_{B}}{d^{2}} \hat{ y }\) (as before).
