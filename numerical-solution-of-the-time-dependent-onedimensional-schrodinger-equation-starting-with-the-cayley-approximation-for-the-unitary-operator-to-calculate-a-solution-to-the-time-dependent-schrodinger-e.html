Guest Question.

Numerical solution of the time-dependent onedimensional Schrodinger equation . Starting with the Cayley approximation for the unitary operator to calculate a solution to the time-dependent Schrodinger equation , derive a finite difference scheme to compute a numerical solution to the one-dimensional Schrodinger equation for the case of a spatially varying effective mass.

Admin
Solution

Finite difference approximation for the one-dimensional Schrodinger equation in the case of a constant effective mass Our starting point is the one-dimensional time-dependent Schrodinger equation for a particle with a constant effective mass m∗ in an arbitrary time-independent potential V (x) :

\(H\psi \left ( x,t \right )=- \frac{h^{2}}{2m^{\ast }}\frac{\partial ^{2}\psi \left ( x,t \right )}{\partial x^{2}}+ V\left ( x \right )\psi \left ( x,t \right )=ih\frac{\partial \psi \left ( x,t \right )}{\partial t}\) (13.84)

For the time being, let us ignore the time dependence and consider the total derivative in space as opposed to the partial derivative. Using equally spaced intervals along the x-axis at integer multiples of a small increment Δx, we first write a finite difference version of the kinetic operator using the approximation [7]

\(\frac{\mathrm{d}^{2}\psi _{i} }{\mathrm{d} x^{2}} = \frac{\psi _{i+1}- 2\psi _{i} + \psi _{i-1}}{\left ( \Delta x \right )^{2}}\) (13.85)

At location i (i.e. in the ith mesh), the expression for the left-hand side of Equation (13.84) can be written as

\(H\psi_{i}=-\left ( \frac{h^{2}}{2m} \right )\left ( \frac{\psi _{i+1}- 2\psi _{i} + \psi _{i-1}}{\left ( \Delta x \right )^{2}} \right ) +V_{i}\psi _{i}\) (13.86)

For a conservative system whose Hamiltonian is time independent (because the potential is time independent), a formal solution to the Schrodinger equation can be written as

\(\psi (x,t) = U\left ( t,t_{0} \right )\psi \left ( x,t_{0} \right )=e^{-j\left ( t-t_{0} \right )H/h}\psi \left ( x,t_{0} \right )\) (13.87)

where j is the imaginary square-root of −1.
Calling \(\delta t = t-t_{0}\), and using the Cayley approximation for the unitary operator \( e^{-j\delta t H/h}\) , we get

\(U\left ( t,t_{0} \right ) = e^{-j\delta t H/h}=\frac{1- j\delta tH/2h}{1+ j\delta tH/2h}= \frac{2h- j\delta tH}{2h+ j\delta tH}\) (13.88)

To characterize the wave function at different time intervals separated by a small increment δt, we use a superscript n which takes integer multiples of \(\delta t\). With the use of Equation (13.84), the wave functions at two different times separated by a time interval \(\delta t\) are related as follows:

\(\psi _{i}^{n+1} = \frac{2h- j\delta tH}{2h+ j\delta tH}\psi _{i}^{n}\) (13.89)

This last equation can be rearranged as

\(\left ( 2h+ j\delta tH \right )\psi _{i}^{n+1} =\left ( 2h-j\delta tH \right )\psi _{i}^{n}\) (13.90)

Using the expression for \(H\psi _{i}\)in Equation (13.86), we get

\(2h\psi _{i}^{n+1} -\frac{j\delta th^{2}}{2m^{\ast }}\left ( \frac{\psi _{i+1}^{n+1}- 2 \psi _{i}^{n+1}+ \psi _{i-1}^{n+1}}{(\Delta x)^{2}}-\frac{2m^{\ast }}{h^{2}} V_{i}^{n+1}\psi _{i}^{n+1}\right )\)

 

\(=2h\psi _{i}^{n}+\frac{j\delta th^{2}}{2m^{\ast }}\left ( \frac{\psi _{i+1}^{n}- 2 \psi _{i}^{n}+ \psi _{i-1}^{n}}{(\Delta x)^{2}}-\frac{2m^{\ast }}{h^{2}} V_{i}^{n}\psi _{i}^{n}\right )\) (13.91)

The last equation leads to an implicit algorithm for obtaining the wave function at the next time step, i.e., at time t = (n + 1)δt, at three discrete points in space (points (i − 1)Δx, iΔx, and (i + 1)Δx):

\(\psi _{i+1}^{n+1} +\psi _{i}^{n+1} \left ( \frac{2j\left ( \Delta x \right )^{2}h}{\delta t}-2-\left ( \Delta x \right )^{2} V_{n}^{i} \right )+\psi _{i-1}^{n+1}\)

 

\(=-\psi _{i+1}^{n}+ \psi _{i}^{n}\left ( \frac{2j\left ( \Delta x \right )^{2}h}{\delta t}+2+\left ( \Delta x \right )^{2} V_{n}^{i} \right )-\psi _{i-1}^{n}\) (13.92)

This implicit algorithm is called the Crank–Nicholson scheme. The latter ensures the conservation of the probability density over time, i.e., the integral over the simulation domain of the probability density of the wave packet is conserved. This probability is equal to unity if the original wave packet is normalized.

For a given initial wave packet, repeated application of the implicit algorithm (13.92) allows one to calculate the wave packet at any later time across the entire solution domain. The implementation of the boundary conditions at both ends of the solution domain is described later.
