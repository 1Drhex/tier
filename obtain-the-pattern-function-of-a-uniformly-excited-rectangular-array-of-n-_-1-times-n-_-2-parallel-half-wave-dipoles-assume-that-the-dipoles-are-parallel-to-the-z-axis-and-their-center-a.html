Guest Question.

Obtain the pattern function of a uniformly excited rectangular array of \({ N }_{ 1 }\times { N }_{ 2 }\) parallel half-wave dipoles. Assume that the dipoles are parallel to the z-axis and their center arc spaced \({ d }_{ 1 }\) and \({ d }_{ 2 }\) apart in the \(x\)– and \(y\)– directions, respectively.

Admin
Solution

From Eqs(10-39) and (10-60):

 

\(\left| { E }_{ \theta } \right| =\frac { j60{ I }_{ m }{ N }_{ 1 }{ N }_{ 2 } }{ R } { e }^{ -j\beta R }\left| \frac { \cos { \left( \frac { \pi }{ 2 } \cos { \phi } \right) } }{ \sin { \theta } } { A }_{ x }\left( { \psi }_{ x } \right) { A }_{ y }\left( { \psi }_{ y } \right) \right|\)

 

Where \({ A }_{ x }\left( { \psi }_{ x } \right) =\frac { 1 }{ { N }_{ 1 } } \frac { \sin { \left( { N }_{ 1 }{ \psi }_{ x }/2 \right) } }{ \sin { \left( { \psi }_{ x }/2 \right) } } ,\quad { \psi }_{ x }=\frac { \beta d }{ 2 } \sin { \theta } \cos { \phi } \)

 

\({ A }_{ y }\left( { \psi }_{ y } \right) =\frac { 1 }{ { N }_{ 2 } } \frac { \sin { \left( { N }_{ 2 }{ \psi }_{ y }/2 \right) } }{ \sin { \left( { \psi }_{ y }/2 \right) } } ,\quad { \psi }_{ y }=\frac { \beta d }{ 2 } \sin { \theta } \cos { \phi } \)

 

\(\left| F\left( \theta ,\phi \right) \right| =\frac { 1 }{ { N }_{ 1 }{ N }_{ 2 } } \left| \left[ \frac { \cos { \left( \frac { \pi }{ 2 } \cos { \phi } \right) } }{ \sin { \theta } } \right] \frac { \sin { \left( \frac { { N }_{ 1 }{ \psi }_{ x } }{ 2 } \right) } \sin { \left( \frac { { N }_{ 2 }{ \psi }_{ y } }{ 2 } \right) } }{ \sin { \left( \frac { { \psi }_{ x } }{ 2 } \right) } \sin { \left( \frac { { \psi }_{ y } }{ 2 } \right) } } \right|\)

 

\({ E }_{ \theta }={ \eta }_{ 0 }{ H }_{ \phi }=\frac { j60{ I }_{ m } }{ R } { e }^{ -j\beta R }\left[ \frac { \cos { \left( \frac { \pi }{ 2 } \cos { \phi } \right) } }{ \sin { \theta } } \right] \quad \quad \left( 10-39 \right) \)
