Guest Question.

Figure 15.10 represents a rotating system that has been idealized for illustrative purposes. A weightless shaft that is supported in bearings at A and B rotates with an angular velocity \(\omega =100 \hat{ i } rad / s\). Three mass particles, with weights \( w_{1}=2 oz , w_{2}=1 oz , \text { and } w_{3}=1.5 oz \), are connected to the shaft and rotate with it, causing unbalance. Determine the magnitudes and orientations of the bearing reactions at A and B.

Admin
Solution

The inertia force caused by each rotating mass particle is

 

\(\begin{aligned}&m_{1} r_{1} \omega^{2}=\frac{(2 oz )(3 in )(100 rad / s )^{2}}{\left(386 in / s ^{2}\right)(16 oz / b )}=9.72 lb \\&m_{2} r_{2} \omega^{2}=\frac{(1 oz )(2 in )(100 rad / s )^{2}}{\left(386 in / s ^{2}\right)(16 oz / b )}=3.24 lb\end{aligned}\)

 

and

 

\(m_{3} r_{3} \omega^{2}=\frac{(1.5 oz )(2.5 in )(100 rad / s )^{2}}{\left(386 in / s ^{2}\right)(16 oz / lb )}=6.07 lb\)

 

These three forces are each parallel to the yz plane and can be written in vector form as

 

\(\begin{aligned}& F _{1}=m_{1} r_{1} \omega^{2} \angle \theta_{1}=9.72 lb \angle 0^{\circ}=9.72 \hat{ j } lb \\& F _{2}=m_{2} r_{2} \omega^{2} \angle \theta_{2}=3.24 lb \angle 120^{\circ}=-1.62 \hat{ j }+2.81 \hat{ k } lb\end{aligned}\)

 

and

\(F _{3}=m_{3} r_{3} \omega^{2} \angle \theta_{3}=6.07 lb \angle 195^{\circ}=-5.86 \hat{ j }-1.57 \hat{ k } lb\)

where the angles θ1, θ2, and θ3 are measured counterclockwise from the y axis when viewed from the positive x axis. The moments of these forces taken about the bearing at A must be balanced by the moment of the bearing reaction at B. Therefore,

\(\begin{aligned}\sum M _{A}=&(1 \hat{ i } \text { in }) \times(9.72 \hat{ j } lb )+(3 \hat{ i } \text { in }) \times(-1.62 \hat{ j }+2.81 \hat{ k } lb ) \\&+(4 \hat{ i } \text { in }) \times(-5.86 \hat{ j }-1.57 \hat{ k } lb )+(6 \hat{ i } \text { in }) \times\left( F _{B}\right)= 0\end{aligned}\)

Solving this equation, the bearing reaction at B is

\(F _{B}=3.10 \hat{ j }-0.36 \hat{ k } lb =3.12 lb \angle-6.6^{\circ}\)

To find the bearing reaction at A, we repeat the analysis. Taking moments about B gives

\(\begin{aligned}\sum M _{B}=&(-2 \hat{ i } \text { in }) \times(-5.86 \hat{ j }-1.57 \hat{ k } 1 b )+(-3 \hat{ i } \text { in }) \times(-1.62 \hat{ j }+2.81 \hat{ k } l b ) \\&+(-5 \hat{ i } \text { in }) \times(9.72 \hat{ j } 1 b )+(-6 \hat{ i } in ) \times\left( F _{A}\right)= 0\end{aligned}\)

Then solving this equation, the bearing reaction at A is

\(F _{A}=-5.34 \hat{ j }-0.88 \hat{ k } lb =5.41 lb \angle 189.4^{\circ}\)

Note that these are rotating reactions and that static (stationary) components caused by gravity are not included.
