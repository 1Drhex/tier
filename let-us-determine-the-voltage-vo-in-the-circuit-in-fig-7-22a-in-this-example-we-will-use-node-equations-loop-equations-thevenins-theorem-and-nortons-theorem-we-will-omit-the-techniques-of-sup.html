Guest Question.

Let us determine the voltage \(V_{o}\) in the circuit in Fig. 7.22a. In this example we will use node equations, loop equations, Thevenin’s theorem, and Norton’s theorem. We will omit the techniques of superposition and source transformation. Why?

Admin
Solution

(1) Nodal Analysis To perform a nodal analysis, we label the
node voltages and identify the supernode as shown in Fig. 7.22b.The constraint equation for the supernode is

\(V_{3}+12\angle 0° = V_{1}\)

and the KCL equations for the nodes of the network are

\(\frac{V_{1}- V_{2}}{-j1}+ \frac{V_{3}- V_{2}}{1}-4 \angle 0° + \frac{V_{3}- V_{o}}{1} + \frac{V_{3}}{j1}=0\)

 

\(\frac{V_{2}- V_{1}}{-j1}+\frac{V_{2}- V_{3}}{1}- 2 \left( \frac{V_{3}- V_{o}}{1}\right) =0\)

 

\(4 \angle 0°+\frac{V_{o}- V_{3}}{1}+\frac{ V_{o}}{1}=0\)

At this point we can solve the foregoing equations using a matrix analysis or, for example, substitute the first and last equations into the remaining two equations, which yields

\(3V_{o}- (1+j)V_{2}= -4(4+j12)\)

 

\(-(4+j2)V_{o}+(1+j)V_{2}=12+j16\)

Solving these equations for \(V_{o}\) yields

\(V_{o}=\left(\frac{-(8+j4)}{1+j2} \right)\)

 

\(\ \ \ \ =+4\angle 143.13°V\)

(2) Loop Analysis The mesh currents for the network are defined in Fig. 7 .22c. The constraint equations for the circuit are

\(I_{2}= -4\angle 0°\)

 

\(I_{x}=I_{4}-I_{2}=I_{4}+4\angle 0°\)

 

\(I_{3}=2I_{x}=2I_{4}+8\angle 0°\)

The KVL equations for mesh 1 and mesh 4 are

\(-j1I_{1}+1(I_{1}-I_{3})=-12\angle 0°\)

 

\(j1(I_{4}-I_{3})+1(I_{4}-I_{2})+1I_{4}=0\)

Note that if the constraint equations are substituted into the second KVL equation, the only unknown in the equation is \(I_{4}\) . This substitution yields

\(I_{4}=+4\angle 143.13°V\)

and hence

\(V_{o}=+4\angle 143.13°V\)

(3) Thevenin’s Theorem In applying Thevenin’s theorem, we will find the open-circuit voltage and then determine the Thevenin equivalent impedance using a test source at the open circuit terminals. We could determine the Thevenin equivalent impedance by calculating the short-circuit current; however, we will determine this current when we apply Norton’s theorem. The open-circuit voltage is determined from the network in Fig. 7.23a. Note that \(I^{'}_{x}= 4\angle 0°\) and since \(2I^{'}_{x}\) flows through the inductor, the open-circuit voltage \( V_{oc}\) is

\(V_{oc}=-1(4\angle 0°)+j1(2I^{'}_{x})\)

 

\(\ \ \ \ =-4+j8 \ V\)

To determine the Thevenin equivalent impedance, we tum off the independent sources, apply a test voltage source to the output terminals, and compute the current leaving the test source. As shown in Fig. 7.23b, since 1; flows in the test source, KCL requires that the current in the inductor be 1; also. KVL around the mesh containing the test source indicates that

\(j1I^{''}_{x}-1I^{''}_{x}- V_{tcx}=0\)

Therefore,

\(I^{''}_{x}=\frac{-V_{test}}{-I^{''}_{x}}\)

Then

\(Z_{Th}= \frac{V_{test}}{-I^{''}_{x}}\)

 

\(\ \ \ \ = 1-j \ \Omega \)

If the Thevenin equivalent network is now connected to the load, as shown in Fig. 7 .23c, the output voltage \( V_{o}\) is found to be

\(V_{o}=\frac{-4+8j}{2-j1}(1)\)

 

\(\ \ \ \ = +4\angle 143.13°V\)

(4) Norton’s Theorem In using Norton’s theorem, we will find the short-circuit current from the network in Fig. 7 .24a. Once again, using the supernode, the constraint and KCL equations are

\(V_{3}+12\angle 0°=V_{1}\)

 

\(\frac{V_{2}-V_{1}}{-j1}+ \frac{V_{2}-V_{3}}{1}-2 I^{''}_{x} =0\)

 

\(\frac{V_{1}-V_{2}}{-j1}+ \frac{V_{3}-V_{2}}{1}-4\angle 0° + \frac{V_{3}}{j1}+ I^{''}_{x}=0\)

 

\(I^{''}_{x}=\frac{V_{3}}{1} \)

Substituting the first and last equations into the remaining equations yields

\((1 + j)V_{2} – (3 + j)I^{''}_{x} = jl2 \)

 

\(-(1 + j)V_{2} + (2)I^{''}_{x} = 4 – jl2 \)

Solving these equations for\(I^{''}_{x}\) yields

\(I^{''}_{x}=\frac{-4}{1+j}A\)

The KCL equation at the right-most node in the network in
Fig. 7.24a is

\(I^{''}_{x}=4\angle 0°+I_{sc}\)

Solving for \(I_{sc}\), we obtain

\(I_{sc}=\frac{-(8+j4)}{1+j}A\)

The Thevenin equivalent impedance was found earlier to be

\(Z_{Th}=1-j\Omega\)

Using the Norton equivalent network, the original network is reduced to that shown in Fig. 7 .24b. The voltage \(V_{o}\) is then

\(V_{o}=\frac{-(8+j4)}{1+j}\left[\frac{(1)(1-j)}{1+1-j} \right] \)

 

\( \ \ \ \ =4\left[\frac{3-j}{3+j} \right] \)

 

\(\ \ \ \ = +4\angle -143.13°V\)
