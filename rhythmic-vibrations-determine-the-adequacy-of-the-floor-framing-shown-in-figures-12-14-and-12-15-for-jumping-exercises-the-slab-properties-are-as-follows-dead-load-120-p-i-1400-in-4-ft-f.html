Guest Question.

Rhythmic Vibrations

Determine the adequacy of the floor framing shown in Figures 12-14 and 12-15 for jumping exercises. The slab properties are as follows:

Dead load = 120 p

I = 1400\( in.^4\)/ft.

\(f^′_ c\) = 5000 psi

\(γ_c = 145\) pcf

Admin
Solution

Slab Deflection:

From Table 12-3,

\(w_p\) = 4.2 psf

\(E_{c}=33w^{1.5} _{c} \sqrt{f^′_{c} }\)

 

\(=33(145)^{1.5} \sqrt{5000 } = 4074 ksi\)

w = (120 psf + 4.2 psf) = 124.2 lb./ft.

= 10.35 lb./in.

\(\Delta _{s} = \frac{5wL^4}{384EI}\)

 

\( = \frac{(5)(10.35)(22\times 12)^4}{(384)(4,074,000)(1400)} = 0.111 in.\)

 

Girder Deflection:

w = (120 psf + 4.2 psf)(22 ft.) = 2732 lb./ft.

= 228 lb./in.

\(\Delta _{g}=\frac{5wL^4}{384EI}\)

 

\(=\frac{(5)(228)(16 \times 12)^4}{(384)(29,000,000)(712)}= 0.195 in.\)

 

Natural Frequency of the System:

\(f_{n}= 0.18\sqrt{\frac{g}{\Delta _{s}+\Delta _{g} } }\)

 

\( = 0.18\sqrt{\frac{386}{0.111+0.195 } } =6.39 Hz\)

 

Minimum Required Natural Frequency for Each Harmonic (eq. (12-35)):

\(f_{n required}\geq f_{f}\sqrt{1 +\left(\frac{k}{a_{o}/g } \right)\left(\frac{\alpha _{i}w_{p} }{w_{t} } \right) }\) (12-35)

First harmonic:

\(2.5\sqrt{1 +\left[\frac{2.0}{0.055} \right]\left[\frac{(1.5)(4.2)}{124.2} \right] }= 4.22 Hz\lt f_{n}= 6.39 Hz\) OK

Second harmonic:

\(5.0\sqrt{1 +\left[\frac{2.0}{0.055} \right]\left[\frac{(0.6)(4.2)}{124.2} \right] }= 6.59 Hz\gt f_{n}= 6.39 Hz\) Not Good

Third harmonic:

\(7.5\sqrt{1 +\left[\frac{2.0}{0.055} \right]\left[\frac{(0.1)(4.2)}{124.2} \right] }= 7.95 Hz\gt f_{n}= 6.39 Hz\) Not Good

The floor system does not meet the criteria given in equation (12-35), so the more accurate analysis needs to be performed:

\(\frac{a_{pi} }{g}= \frac{1.3\alpha _{i}w_{p} }{w_{t}\sqrt{\left[\left(\frac{f_{n} }{f_{f} } \right) ^{2}-1 \right] ^{2}+\left(\frac{2\beta f_{n} }{f_{f} } \right) ^{2} } }\)

First harmonic:

\(\frac{a_{p1} }{g}= \frac{ (1.3)(1.5)(4.2) }{124.2\sqrt{\left[\left(\frac{6.39 }{2.5 } \right) ^{2}-1 \right] ^{2}+\left[\frac{(2)(0.06)(6.39)}{2.5 }\right] ^{2} } } = 0.0119\)

Second harmonic:

\(\frac{a_{p2} }{g}= \frac{ (1.3)(0.6)(4.2) }{124.2\sqrt{\left[\left(\frac{6.39 }{5.0 } \right) ^{2}-1 \right] ^{2}+\left[\frac{(2)(0.06)(6.39)}{5.0 }\right] ^{2} } } = 0.0405\)

Third harmonic:

\(\frac{a_{p3} }{g}= \frac{ (1.3)(0.1)(4.2) }{124.2\sqrt{\left[\left(\frac{6.39 }{7.5 } \right) ^{2}-1 \right] ^{2}+\left[\frac{(2)(0.06)(6.39)}{7.5 }\right] ^{2} } } = 0.0150\)

The effective maximum acceleration, considering all of the harmonics, is

\(\frac{a_{pm} }{g}=\left(\frac{a^{1.5} _{pi} }{g} \right) ^{2/3 }\leq \frac{a_{o} }{g}\)

 

\(= (0.0119^{1.5} + 0.0405^{1.5}+0.0150^{1.5} )^{2/3}=0.503\lt \frac{a_{o} }{g}= 0.055.\)

From Table 12-4, the recommended acceleration limit is a range between 4% and 7% in this case, the average between these two values (5.5%) is taken as the acceleration limit.
The floor acceleration is less than the specified limit, so this floor is adequate for jumping exercises. If this floor were to be used for jumping exercises adjacent to a dining area, the average acceleration limit, from Table 12-4, would be 2%. Therefore, this floor would not be adequate in that case.
