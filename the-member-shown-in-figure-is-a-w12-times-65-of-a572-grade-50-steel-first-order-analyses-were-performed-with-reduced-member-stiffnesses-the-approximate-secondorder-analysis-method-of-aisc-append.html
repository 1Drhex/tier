Guest Question.

The member shown in Figure is a W12 \(\times\) 65 of A572 Grade 50 steel. First-order analyses were performed with reduced member stiffnesses. The approximate secondorder analysis method of AISC Appendix 8 can be used, making this a direct analysis method. For LRFD, the analysis results for the controlling factored load combination are \(P_{nt} = 300\) kips, \(M_{ntx} = \)135 ft-kips, and \(M_{nty} = 30\) ft-kips. For ASD, the analysis results for the controlling load combination are \(P_{nt} = 200\) kips, \(M_{ntx} = 90\) ft-kips, and \(M_{nty} = 20\) ft-kips. Use\( K_{y} = 1.0\), and investigate this member for compliance with the AISC Specification.

Admin
Solution

Compute the strong axis bending moments:
\(C_{mx} = 0.6 – 0.4 \frac{M_1}{M2}= 0.6 – 0.4(0) = 0.6\)
Since a modified flexural rigidity, EI*, was used in the frame analysis, it must also be used in the computation of \(P_{e1}\). From Equation 6.8,
\(EI* = 0.8\tau_{b} EI = 0.8(1.0)EI = 0.8EI\)
(In this book, we assume that the requirements for using \(\tau_{b}\) = 1.0 have been met.)
\(L_{c1} = L_{cx} = K_{x}L = 1.0(15 \times 12) = 180 in\).
\(P_{e1x} =\frac{\pi ^{2}EI*}{(L_{c1})^{2}}=\frac{\pi ^{2}(0.8EI_x)}{(L_{c1})^{2}}= \frac{\pi ^{2}(0.8)(29,000)(533)}{(180)^{2}} = 3767kips\)

 

\(B_{1x} =\frac {C_{mx}}{1 – (\alpha P_{r}/P_{e1x})}= \frac {C_{mx}}{1 – (1.00P_{u}/P_{e1x})}= \frac {0.6}{1- (300/3767)} \)
\(=0.652 < 1.0 \therefore Use B_{1x} = 1.0.\)
The required moment strength is

 

\(M_r = M_{ux} = B_{1x}M_{ntx} + B_{2x}M_{ℓtx} = 1.0(135) + 0 = 135.0 ft-kips\)
From Table 6-2 with \(C_b = 1.0 and L_b = 15 feet\), the moment strength is
\(\phi _{b}M_{nx} = 340 ft-kips and \phi _{b}M_{px} = 356 ft-kips\)
From Figure 5.15g, \(C_b = 1.67\) and

\(C_{b} \times (\phi _{b}M_{nx} for C_{b} = 1.0) = 1.67(340) = 567.8 ft-kips\)
This result is larger than \(\phi _{b}M_{px}\); therefore, use \(\phi _{b}M_{nx} = \phi _{b}M_{px} =356\) ft-kips.
\textbf{Compute the weak axis bending moments:}

 

\(C_{my} = 0.6 – 0.4\frac{M_1}{M2}= 0.6 – 0.4(0) = 0.6\)

 

\(P_{e1y} =\frac{\pi ^{2}EI*}{(L_{c1})^{2}}=\frac{\pi ^{2}(0.8EI_y)}{(K_{y}L)^{2}}= \frac{\pi ^{2}(0.8)(29,000)(174)}{(1.0 \times 15 \times 12)^{2}} = 1230 kips\)

where\( L_{c1} = K_{1}L = K_{y}L.\)
\(B_{1y} =\frac {C_{my}}{1 – (\alpha P_{r}/P_{e1y})}= \frac {C_{my}}{1 – (1.00P_{u}/P_{e1y})}= \frac {0.6}{1- (300/1230)}\)
\( =0.794 < 1.0 \therefore Use B_{1y} = 1.0.\)
The required moment strength is
\(M_r = M_{uy} = B_{1y}M_{nty} + B_{2y}M_{ℓty}= 1.0(30) + 0 = 30 ft-kips\)
From Table 6-2, \(\phi _{b}M_{ny}\) = 161 ft-kips.
Because the flange of this shape is noncompact (see footnote in the dimensions and properties table), the weak axis bending strength is limited by FLB (see Section 5.15 of this book and Chapter F of the AISC Specification). This is accounted for in the Table 6-2 value.
The required axial compressive strength is
\(P_r = P_u = P_{nt} + B_{2}P_{ℓt} = 300 + 0 = 300 kips\)
and the axial compressive design strength from Table 6-2 is
\(\phi _{c}P_{n} = 663 kips\)
Determine which interaction formula to use:
\(\frac{P_u}{\phi _{c}P_{n}}=\frac{300}{665} =0.4525 > 0.2 \therefore \)Use Equation 6.3 (AISC Equation H1-1a).

 

\(\frac{P_u}{\phi _{c}P_{n}}+\frac{8}{9}\left (\frac{M_{ux}}{\phi _{b} M_{nx}} + \frac{M_{uy}}{\phi _{b} M_{ny}} \right )=0.4525 +\frac{8}{9}\left (\frac{135}{356} +\frac{30}{161} \right )= 0.955 < 1.0 (OK)\)
The W12 \(\times\) 65 is satisfactory.

\(\textbf{Compute the strong axis bending moments:}\)
\(C_{mx} = 0.6 – 0.4 \frac{M_1}{M2}= 0.6 – 0.4(0) = 0.6\)
Since a modified flexural rigidity, EI*, was used in the frame analysis, it must also be used in the computation of \(P_{e1}\). From Equation 6.8,
\(EI* =0.8\tau_{b} EI = 0.8(1.0)EI = 0.8EI\)

 

\(P_{e1x} =\frac{\pi ^{2}EI*}{(K_{1}L)^{2}}=\frac{\pi ^{2}(0.8EI_{x})}{(K_{x}L)^{2}}= \frac{\pi ^{2}(0.8)(29,000)(533)}{(1.0 \times 15 \times 12)^{2}} = 3767kips\)

 

\(B_{1x} =\frac {C_{mx}}{1 – (\alpha P_{r}/P_{e1x})}= \frac {C_{mx}}{1 – (1.60P_{a}/P_{e1x})}= \frac {0.6}{1- (1.60 \times 200/3767)}= 1.015\)
\(=0.656 < 1.0 \therefore Use B_{1x} = 1.0.\)
The required moment strength is
\(M_r = M_{ax} = B_{1x}M_{ntx} + B_{2x}M_{ℓtx} = 1.0(90) + 0 =90 ft-kips\)
From Table 6-2 with \(C_{b} = 1.0 \)and\( L_{b} = 15\) feet, the moment strength is
\(\frac{M_{nx}}{\Omega _{b}}=226 ft-kips \)and
\(\frac{M_{px}}{\Omega _{b}}=237 ft-kips\)
From Figure 5.15g,\( C_b = 1.67 \)and
\(C_{b} \times \left ( \frac{M_{nx}}{\Omega _{b}} for C_{b} = 1.0 \right ) = 1.67(226) = 377.4 ft-kips\)
This result is larger than
\(\frac{M_{px}}{\Omega _{b}} ; therefore, use \frac{M_{nx}}{\Omega _{b}}= \frac{M_{px}}{\Omega _{b}}= 237 ft-kips\).

\(\textbf{Compute the weak axis bending moments:}\)
\(C_{my} = 0.6 – 0.4\frac{M_1}{M2}= 0.6 – 0.4(0) = 0.6\)

 

\(P_{e1y} =\frac{\pi ^{2}EI*}{(K_{1}L)^{2}}=\frac{\pi ^{2}(0.8EI_y)}{(K_{y}L)^{2}}= \frac{\pi ^{2}(0.8)(29,000)(174)}{(1.0 \times 15 \times 12)^{2}} = 1230 kips\)

 

\(B_{1y} =\frac {C_{my}}{1 – (\alpha P_{r}/P_{e1y})}= \frac {C_{my}}{1 – (1.60P_{a}/P_{e1y})}= \frac {0.6}{1- (1.60 \times 200/1230)}\)
=0.811 < 1.0 \(\therefore \)Use \(B_{1y} = 1.0.\)
The required moment strength is
\(M_r = M_{ay} = B_{1y}M_{nty} + B_{2y}M_{ℓty}= 1.0(20) + 0 = 20 ft-kips\)
From Table 6-2,
\(M_{ny}/\Omega _{b} = 107 ft-kips\)
Because the flange of this shape is noncompact, the weak axis bending strength is limited by FLB (see Section 5.15 of this book and Chapter F of the AISC Specification). This is accounted for in the Table 6-2 value.
The required axial compressive strength is
\(P_{a} = P_{nt} + B_{2}P_{ℓt} = 200 + 0 = 200 kips\)
For \(L_C = 1.0(15) = 15 feet\), the allowable compressive strength from the column load tables is
\(\frac{P_{n}}{\Omega _{c}}=441 kips\)
\(\textbf{Check the interaction formula:}\)
\(\frac{P_{a}}{P_{n}/\Omega _{c}}=\frac{ 200}{441} = 0.4535 > 0.2 \therefore\) Use Equation 6.5 (AISC Equation H1-1a).
\(\frac{P_{a}}{P_{n}/\Omega _{c}}=\frac{8}{9}\left (\frac{M_{ax}}{ M_{nx}/\Omega _{b}} + \frac{M_{ay}}{ M_{ny}/\Omega _{b}} \right )=0.4535 +\frac{8}{9}\left ( \frac{90}{237}+\frac{20}{107} \right ) \)
= 0.957 < 1.0 (OK)
The W12 \(\times\) 65 is satisfactory.
