Guest Question.

For the three-phase bridge fully controlled thyristor rectifier, the following specifications are given: rms input line-to-line voltage: 415 V 50 Hz, load: 100-Ω resistor in series with a large value inductor so that ωL〉〉R. Calculate the average, rms, and maximum values of the current flowing through a thyristor.

Admin
Solution

Solution

The average and rms values of the current flowing through a thyristor are respectively
given by:

 

\(\begin{aligned} \overline{ I }_{\text {Thyristor }} &=\frac{1}{ T } \int_{ o }^{2 \pi} i _{\text {Thyristor }}(\omega t ) d \omega t \\ &=\frac{1}{2 \pi} \int_{\alpha+30^{\circ}}^{150^{\circ}+\alpha} \overline{ I }_{0} d \omega t =\left.\frac{\overline{ I }_{0}}{2 \pi}(\omega t )\right|_{\alpha+30^{\circ}}^{150^{\circ}+\alpha} \\ &=\frac{\overline{ I }_{ o }}{2 \pi}\left(\alpha+150^{\circ}-\alpha-30^{\circ}\right)=\frac{\overline{ I }_{ o }}{3} \mid A \end{aligned}\)

 

\(\begin{aligned} \widetilde{ I }_{\text {Thyristor }} &=\left[\frac{1}{ T } \int_{ o }^{2 \pi} i _{\text {Thyristor }}^{2}(\omega t ) d \omega t \right]^{1 / 2}=\left[\frac{1}{2 \pi} \int_{\alpha+30^{\circ}}^{150^{\circ}+\alpha} \overline{ I }_{ o }^{2} d \omega t \right]^{1 / 2} \\ &=\left[\frac{\overline{ I }_{ o }^{2}}{2 \pi}\left(\frac{2 \pi}{3}\right)\right]^{1 / 2}=\frac{\overline{ I }_{ o }}{\sqrt{3}} A \quad \text { where } \overline{ I }_{ o }=\frac{\overline{ V }_{ o }}{ R } A \end{aligned}\)

 

The maximum values of the above currents result when the maximum output current
average value is \(\overline{ I }_{0}\). This happens whenα = 0° and is given by the following equation:

 

\(\overline{ I }_{ o (\max )}=\frac{\overline{ V }_{ o (\max )}}{ R }=\frac{\overline{ V }_{ o \left(\alpha=0^{\circ}\right)}}{ R }=\frac{3 \sqrt{6} \widetilde{ V }_{ i }}{\pi R } \cos \alpha=\frac{3 \sqrt{6}(415 / \sqrt{3})}{\pi \cdot 100} \cos 0^{\circ}=5.6 A\)

 

Consequently,

\(\widehat{ I }_{\text {Thyristor }(\max )}=5.6 A , \quad \overline{ I }_{\text {Thyristor }(\max )}=1.866 A , \quad \widetilde{ I }_{\text {Thyristor }(\max )}=3.23 A .\)
