Guest Question.

Determine the \(E\) field caused by a spherical cloud of electron with a volume charge density \({ \rho }_{ v }=-{ \rho }_{ O }\) for \(0\le R\le b\) (both \({ \rho }_{ O }\) and \(b\) are positive) and \({ \rho }_{ v }= 0\) for \(R>b\).

Admin
Solution

First we recognize that the given source condition has spherical symmetry. The appropriate Gaussian surfaces must therefore be concentric spherical surface . We must find the E field in two regions, as shown in Fig. 3-7.

a) \(0\le R\le b\)

 

A hypothetical spherical Gaussian surface \({ S }_{ i }\) with \(R < b\) is constructed within the electron cloud. On this surface, \(E\) is radial and has a constant magnitude:

\(E={ a }_{ R }{ E }_{ R },\quad \quad ds={ a }_{ R }ds\)

 

The total outward \(E\) flux is

\(\oint _{ { S }_{ 1 } }^{ }{ E } ds={ E }_{ R }\int _{ { S }_{ 1 } }^{ }{ ds } ={ E }_{ R }4\pi { R }^{ 2 }\)

 

The total charge enclosed within the Gaussian surface is

\(Q=\int _{ V }^{ }{ { \rho }_{ v }dv }\)

 

\(\quad =-{ \rho }_{ o }\int _{ V }^{ }{ dv } =-{ \rho }_{ o }\frac { 4\pi }{ 3 } { R }^{ 2 }\)

 

Substitution into Eq. (3-6) yield

\(E=-{ a }_{ R }\frac { { \rho }_{ o } }{ 3{ \epsilon }_{ o } } R,\quad \quad 0\le R\le b\)

 

\(\oint _{ S }^{ }{ E } .ds=\frac { Q }{ { \epsilon }_{ O } } \quad \quad \quad \left( 3-5 \right)\)

 

We see that within the uniform electron cloud the \(E\) field is directed toward the center and has a magnitude proportional to the distance from the center.

\(b)\) \(R\ge b\)

 

For this case we construct a spherical Gaussian surface \({ S }_{ o }\) with \(R > h\) outside the electron cloud. We obtain the same expression for \(\oint _{ S }^{ }{ E } .ds\) as in case \((a).\) The total charge enclosed is

\(Q={ \rho }_{ o }\frac { 4\pi }{ 3 } { b }^{ 3 }\)

 

Consequently,

\(E=-{ a }_{ R }\frac { { \rho }_{ o }{ b }^{ 3 } }{ 3{ \epsilon }_{ 0 }{ R }^{ 2 } } .\quad \quad \quad R\ge b\)

 

Notice that this relationship follows the inverse square law and could have been obtained directly from Eq. (3-8). We see that outside the charged cloud. the \(E\) field is exactly the same as if the total charge has been concentrated on a ·point charge at the center. This result hold . in general, for any spherically symmetrical charged region even when \({ \rho }_{ v }\) is a function of \(R\).
