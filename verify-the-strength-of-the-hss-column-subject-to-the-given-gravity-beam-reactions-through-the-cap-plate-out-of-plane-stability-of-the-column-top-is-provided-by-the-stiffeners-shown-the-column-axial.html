Guest Question.

Verify the strength of the HSS column subject to the given gravity beam reactions through the cap plate. Out of plane stability of the column top is provided by the stiffeners shown. The column axial forces are RDL = 24 kips and RLL = 30 kips.

Admin
Solution

Material Properties:

Beam ASTM A992 \(F_{y}=50 ksi \quad F_{u}=65 ksi\)

Column ASTM A500 Gr. B \(F_{y}=46 ksi \quad F_{u}=58 ksi\)

Cap Plate ASTM A36 \(F_{y p}=36 ksi \quad F_{u}=58 ksi\)

Manual
Table 2-3

Geometric Properties:

Beam W18×35 d = 17.7 in. \(b_{f}=6.00 \text { in. } \quad t_{w}=0.300 \text { in. }\)

\(t_{f}=0.425 \text { in. } \quad k_{l}=0.750 \text { in. }\).

Manual
Table 1-1

Column HSS8×8× 1/4 t = 0.233 in. Manual Table 1-12

Cap Plate t = 0.500 in.

Calculate the required strength

LRFD ASD
\(\begin{aligned}R_{\underline{u}} &=1.2(24.0 kips )+1.6(30.0 kips ) \\&=76.8 kips\end{aligned}\) \(\begin{aligned}R_{\underline{a}} &=24.0 kips +30.0 kips \\&=54.0 kips\end{aligned}\)

Assume the vertical beam reaction is transmitted to the HSS through bearing of the cap plate at the two column faces perpendicular to the beam.

Calculate bearing length, N, at bottom of W18×35

Section K1.6

\(N=2 k_{l}+5 t_{f}=2(0.750 in .)+5(0.425 in .)=3.63 in\)

Check limit for number of HSS faces contributing

\(5 t_{p}+N=5(0.500 in .)+3.63 in. =6.13 in .<8.00 \text { in. therefore, only } 2 \text { walls contribute }\)

Calculate the nominal local wall yielding strength of the HSS

For each of the two walls:

\(R_{n}=F_{y} t\left(5 t_{p}+N\right) \leq B F_{y} t\)

Eqn. K1-11

\(=46 ksi (0.233 in. )(5(0.500 in .)+3.63 in. ) \leq 8.00 in .(46 ksi )(0.233 in .)\)

 

\(65.7 \text { kips } \leq 85.7 \text { kips }\)

 

Use \(R_{n}\) = 65.7 kips per wall

 

Calculate the available local wall yielding strength of the HSS

LRFD ASD
\(\begin{array}{l}\phi=1.00 \\\phi R_{n}=1.00(65.7 kips )(2 \text { walls }) \\\quad=131 \text { kips }>76.8 \text { kips } \quad \text { o.k. }\end{array}\) \(\begin{array}{l}\Omega=1.50 \\R_{n} / \Omega=(65.7 kips / 1.50)(2 \text { walls }) \\\quad=87.6 \text { kips }>54.0 \text { kips } \quad \text { o.k. }\end{array}\)

Section K1.6

Calculate the nominal wall local crippling strength of the HSS

Section K1.6

For each of the two walls:

\(R_{n}=0.8 t^{2}\left[1+(6 N / B)\left(t / t_{p}\right)^{1.5}\right]\left[E F_{y} t_{p} / t\right]^{0.5}\)

Eqn. K1-12

\(R_{n}=0.8(0.233 \text { in. })^{2}\left[1+\left(\frac{6(3.63 in .)}{8.00 in .}\right)\left(\frac{0.233 in .}{0.500 in .}\right)^{1.5}\right]\left[\frac{29,000 ksi (46 ksi )(0.500 in .)}{0.233 in. }\right]^{0.5}\)

 

= 137 kips per wall

Calculate the available local wall local crippling strength of the HSS

LRFD ASD
\(\begin{aligned}\phi &=0.75 \\\phi R_{n} &=0.75(137 kips )(2 \text { walls }) \\&=207 \text { kips }>76.8 \text { kips }\end{aligned}\) \(\begin{aligned}\Omega &=2.00 \\R_{n} / \Omega &=(137 kips / 2.00)(2 \text { walls }) \\&=137 kips >54.0 kips \quad o . k\end{aligned}\)

Section K1.6

The presence of the stiffeners eliminates the need to check the beam web for local web yielding and local web crippling.
