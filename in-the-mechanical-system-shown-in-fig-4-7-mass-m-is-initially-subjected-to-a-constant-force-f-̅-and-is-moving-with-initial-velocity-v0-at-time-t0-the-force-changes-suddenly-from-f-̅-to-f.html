Guest Question.

In the mechanical system shown in Fig. 4.7, mass m is initially subjected to a constant force \(\overline{F}\) and is moving with initial velocity \(v_{0}\). At time \(t_{0}\) the force changes suddenly from \(\overline{F}\) to \(\overline{F} + \Delta F\). Find the velocity and acceleration of the mass for time \(t\gt t_{0}\).

Admin
Solution

Using force F(t) applied to the mass as the input and using the velocity of the mass v(t) as the output variable, we can write the input–output equation as

\(m\dot{v} + bv=F\left(t\right) .\)

Because the velocity of the mass cannot change suddenly, the initial condition is
\(v\left(t_{0} \right) =v_{0} .\)

A mathematical form describing the input force occurring at \(t=t_{0}\) is

\(F\left(t\right) =\overline{F} + \Delta FU_{s} \left(t- t_{0} \right) .\)

The input–output equation can be rewritten in the form

\(m\dot{v} + bv =\overline{F} + \Delta FU_{s} \left(t- t_{0} \right) .\)

The general expression for the step response of a first-order model given by Eq. (4.30)

\(y\left(t\right) =y_{ss} -\left(y_{ss}- y_{0} \right) e^{- \frac{t}{\tau } } ,\)

can be used here but it has to be modified to reflect the initial time as \(t_{0}\), rather than as 0.
The modified form of Eq. (4.30) is

\(y\left(t\right) =y_{ss} -\left(y_{ss}- y_{0} \right) e^{- \frac{\left(t- t_{0} \right) }{\tau } } ,\)

or, when velocity \(v\left(t\right)\) is substituted as the output variable,

\(v\left(t\right) =v_{ss} -\left(v_{ss}- v_{0} \right) e^{- \frac{\left(t- t_{0} \right) }{\tau } } ,\)

with the general form of the solution known, the problem is reduced to finding three unknowns in the preceding equation: initial velocity \(v_{0}\), steady-state velocity \(v_{ss}\), and time constant \(\tau\) . The initial velocity resulting from a constant initial force \(\overline{F}\) can be found from the input–output equation when both velocity and force are made constant and equal to their respective initial values:

\(m\dot{v_{0} } + bv_{0} =\overline{F} .\)

The first term on the left-hand side is zero and thus the initial velocity is

\(v_{0} =\frac{\overline{F} }{b}.\)

One can find the steady-state velocity of the mass by taking limits of both sides of the input–output equation for time approaching infinity:

\(\underset{t\rightarrow \infty }{\lim } \left(m\dot{v} + bv\right) =\underset{t\rightarrow \infty }{\lim }\left[\overline{F}+ \Delta FU_{s}\left(t- t_{0} \right) \right] ,\)

and hence

\(m\underset{t\rightarrow \infty }{\lim } \dot{v} + b\underset{t\rightarrow \infty }{\lim }v=\overline{F} + \Delta F.\)

Because the velocity reaches a constant value of \(v_{ss}\) for time approaching infinity, the first term on the left-hand side is zero and thus the steady-state velocity is

\(v_{ss} =\frac{\left(\overline{F} + \Delta F\right) }{b} .\)

The time constant, defined by Eq. (4.21) \(\tau ={a_{1} }/{a_{0} }\), is

\(\tau =\frac{m}{b} .\)

Substituting the expressions found for the three unknowns into the general form of the solution gives

\(v\left(t\right) =\frac{\left(\overline{F}+ \Delta F \right) }{b} – \left[\frac{\left(\overline{F}+ \Delta F \right) }{b}- \frac{\overline{F} }{b} \right] e^{- \left({b}/{m}\right)\left(t- t_{0} \right) } ,\)

and hence

\(v\left(t\right) =\frac{\overline{F} }{b} + \left(\frac{\Delta F}{b} \right) \left[1- e^{-\left({b}/{m}\right)\left(t- t_{0} \right) } \right] .\)

It should also be noted that the steady-state gain for this system, defined by Eq. (4.33) \(K=\frac{1}{a_{0} }\), is

\(K=\frac{1}{b} .\)

With the steady-state gain, the velocity of the mass for \(t\gt t_{0}\) can be expressed in yet another form:

\(v\left(t\right) =K\left[\overline{F}+ \Delta F\left(1- e^{\frac{t- t_{0} }{\tau } } \right) \right] .\)

The response of acceleration for \(t\gt t_{0}\) can be obtained by differentiation of the velocity step response :

\(a\left(t\right) =\left(\frac{\Delta F}{m} \right) e^{- \left({b}/{m}\right)\left(t- t_{0} \right) } =\left(\frac{K}{\tau } \right) \Delta Fe^{- \frac{\left(t- t_{0} \right) }{\tau } } .\)

Both the velocity and acceleration step response curves are plotted in Fig. 4.8.
A conclusion from these considerations is that a step response of a first-order model is always an exponential function of time involving three parameters: initial value, steadystate value, and time constant. If the initial value is specified in the problem statement, the other two parameters can be found simply by inspection of the model input–output equation, by use of the relations defined in Eqs. (4.21) \(\tau ={a_{1} }/{a_{0} }\) and (4.29), \(y_{ss} =\underset{t\rightarrow \infty }{\lim } y\left(t\right) =\frac{A}{a_{0} } .\) A step response of a firstorder model can thus be found in a very simple manner, without solving the differential equation.
