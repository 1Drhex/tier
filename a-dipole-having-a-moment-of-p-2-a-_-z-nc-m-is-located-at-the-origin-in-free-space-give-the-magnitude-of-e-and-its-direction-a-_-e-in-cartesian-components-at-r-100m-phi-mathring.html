Guest Question.

A dipole, having a moment of P = 2\({ a }_{ z }\) nC. m is located at the origin in free space. Give the magnitude of E and its direction \({ a }_{ E }\) in Cartesian components at r = 100m, \(\phi =\mathring { 90 }\), and \(\theta \)= : (a) \(\mathring { 0 }\); (b) \(\mathring { 30 }\); (c) \(\mathring { 90 }\).

Admin
Solution

Begin with
E = \(\frac { P }{ 4\pi { \epsilon }_{ 0 }{ \left( r \right) }^{ 3 } } \left[ 2\cos { \theta } { { a }_{ r } }+\sin { \theta } { a }_{ \theta } \right]\)
from which
\(\left| { E } \right| =\frac { P }{ 4\pi { \epsilon }_{ 0 }{ \left( r \right) }^{ 3 } } { \left[ 4\cos ^{ 2 }{ \theta } +\sin ^{ 2 }{ \theta } \right] }^{ 1/2 } = \frac { P }{ 4\pi { \epsilon }_{ 0 }{ \left( r \right) }^{ 3 } } { \left[ 1 +3\cos ^{ 2 }{ \theta } \right] }^{ 1/2 }\)
Now
\({ E }_{ x }=E{ .a }_{ x }\frac { P }{ 4\pi { \epsilon }_{ 0 }{ \left( r \right) }^{ 3 } } \left[ 2\cos { \theta } { { a }_{ r } }.{ a }_{ x }+\sin { \theta } { a }_{ \theta }{ a }_{ x } \right] =\frac { P }{ 4\pi { \epsilon }_{ 0 }{ \left( r \right) }^{ 3 } } { \left[ 3\cos ^{ }{ \theta } \sin ^{ }{ \theta } \cos { \phi } \right] }\)
then
\({ E }_{ y }=E{ .a }_{ y }\frac { P }{ 4\pi { \epsilon }_{ 0 }{ \left( r \right) }^{ 3 } } \left[ 2\cos { \theta } { { a }_{ r } }.{ a }_{ y }+\sin { \theta } { a }_{ \theta }{ a }_{ y } \right] =\frac { P }{ 4\pi { \epsilon }_{ 0 }{ \left( r \right) }^{ 3 } } { \left[ 3\cos ^{ }{ \theta } \sin { \theta } \sin { \phi } \right] }\)
and
\({ E }_{ z }=E{ .a }_{ z }\frac { P }{ 4\pi { \epsilon }_{ 0 }{ \left( r \right) }^{ 3 } } \left[ 2\cos { \theta } { { a }_{ r } }.{ a }_{ z }+\sin { \theta } { a }_{ \theta }{ a }_{ z } \right] =\frac { P }{ 4\pi { \epsilon }_{ 0 }{ \left( r \right) }^{ 3 } } { \left[ 2\cos ^{ 2 }{ \theta } – \sin ^{ 2 }{ \theta } \right] }\)
Since \(\phi\) is given as \(\mathring { 90 } , { E }_{ x}\) = 0, and the field magnitude becomes
\(\left| { E }_{ \left( \phi =\mathring { 90 } \right) } \right| = \sqrt { { E }_{ y }^{ 2 }+{ E }_{ z }^{ 2 } } =\frac { P }{ 4\pi { \epsilon }_{ 0 }{ \left( r \right) }^{ 3 } } { { \left[ 9\cos ^{ 2 }{ \theta } \sin ^{ 2 }{ \theta } +{ \left( 2\cos ^{ 2 }{ \theta } -\sin ^{ 2 }{ \theta } \right) }^{ 2 } \right] } }^{ 1/2 }\)
Then the unit vector becomes (again at \(\phi = \mathring { 90 }\) ):
\({ a }_{ E }=\frac { 3\cos { \theta } \sin { \theta } { { a }_{ y }+ }\left( 2\cos ^{ 2 }{ \theta } -\sin ^{ 2 }{ \theta } \right) { a }_{ z } }{ { \left[ 9\cos ^{ 2 }{ \theta } \sin ^{ 2 }{ \theta } +{ \left( 2\cos ^{ 2 }{ \theta } -\sin ^{ 2 }{ \theta } \right) }^{ 2 } \right] }^{ 1/2 } }\)
Now with r = 100 m and P =\(2\times { 10 }^{ -9 }\).
\(\frac { P }{ 4\pi { \epsilon }_{ 0 }{ \left( r \right) }^{ 3 } } =\frac { 2\times { 10 }^{ -9 } }{ 4\pi \left( 8.854\times { 10 }^{ -12 } \right) { 10 }^{ 6 } } = 1.80 \times { 10 }^{ -5 }\)

Using the above formulas, we ﬁnd at \(\theta =\mathring { 0 } , \left| { E } \right| =(1.80 \times { 10 }^{ -5 })(2)=36.0\mu V/m\) and \({ a }_{ E } = { a }_{ z }. At \theta =\mathring { 0 }\), we find \(\left| { E } \right| = (1.80 \times { 10 }^{ -5 }){ { \left[ 1.69+1.56 \right] } }^{ 1/2 } = 32.5 \mu V/m\) and \({ a }_{ E } = (1.30{ a }_{ y } + 1.25{ a }_{ z })/1.80 = 0.72 { a }_{ x} + 0.69 { a }_{ z }. At \theta = \mathring { 90 } , \left| { E } \right| =(1.80 \times { 10 }^{ -5 })(1)=18.0\mu V/m\) and \({ a }_{ E } = -{ a }_{ z }\).
