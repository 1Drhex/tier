Guest Question.

Figure 2.25 shows a thin-film resistor in an integrated circuit. How hot will it get with 2 W dissipated over its 0.1 × 0.2-in surface (100 W/in²)? Assume the ground plane is at 80°C.

Admin
Solution

Since we have three different media through which this heat will be transferred, we must take into consideration thermal conduction within each. With the help of Eq. 2.14 and Table 2.4,

\(P_{heat} = \frac{dQ_{heat} }{dt} = k\left\lgroup\frac{A}{L} \right\rgroup\Delta T = \frac{1}{\lambda }\left\lgroup\frac{A}{L} \right\rgroup\Delta T = \frac{\Delta T}{\Re _{therm} }\)

(k = thermal conductance) (λ = thermal resistivity) (\(\Re _{therm}\) = thermal resistance)

TABLE 2.4 Typical Thermal Resistivities (λ) in Units of °C.in/W

we get the individual heat transfers through each region:

\(\Delta T_{1-2\left(in ceramic\right) } = \lambda _{ceramic}\left\lgroup\frac{L}{A} \right\rgroup P_{dis} = \frac{\left(2.13\right)\left(0.025\right)\left(2\right) }{\left(0.1\right)\left(0.2\right) } = 5.3^{\omicron }C\)

 

\(\Delta T_{2-3\left(in grease\right) } = \lambda _{grease}\left\lgroup\frac{L}{A} \right\rgroup P_{dis} = \frac{\left(46\right)\left(0.002\right)\left(2\right) }{\left(0.1\right)\left(0.2\right) } = 9.2^{\omicron }C\)

 

\(\Delta T_{3-4\left(in aluminum\right) } = \lambda _{aluminum}\left\lgroup\frac{L}{A} \right\rgroup P_{dis} = \frac{\left(0.23\right)\left(0.125\right)\left(2\right) }{\left(0.1\right)\left(0.2\right) } = 2.9^{\omicron }C\)

Adding this together:

\(\Delta T_{1-4} = 5.3^{\omicron }C + 9.2^{\omicron }C + 2.9^{\omicron }C = 17.4^{\omicron }C\)

Adding this to the 80°C for the aluminum ground plane leads to an estimated 100°C for maximum resistor temperature. This is a conservative estimate, since it neglects transverse heat spreading.
