Guest Question.

The “half-sinusoidal” waveform shown in Fig. 18.2 represents the voltage response obtained at the output of a half-wave rectifier circuit, a nonlinear circuit whose purpose is to convert a sinusoidal input voltage to a (pulsating) approximation to dc. Find the Fourier series representation of this waveform.

Admin
Solution

Identify the goal of the problem.

We are presented with a periodic function and are asked to find the Fourier series representation. If not for the removal of all negative voltages, the problem would be trivial, as only one sinusoid would be required.

Collect the known information.

In order to represent this voltage as a Fourier series, we must first determine the period and then express the graphical voltage as an analytical function of time. From the graph, the period is seen to be

T = 0.4 s

and thus

\(f_{0}=2.5 Hz\)

and

\(\omega_{0}=5 \pi rad / s\)

Devise a plan.

The most straightforward approach is to apply Eqs. [9] to [11] to calculate the set of coefficients \(a_{0}, a_{n}, \text { and } b_{n}\). To do this, we need a functional expression for v(t), the most straightforward being defined over the interval t = 0 to t = 0.4 as

 

\(a_{0}=\frac{1}{T} \int_{0}^{T} f(t) d t\) [9]

 

\(a_{n}=\frac{2}{T} \int_{0}^{T} f(t) \cos n \omega_{0} t d t\) [10]

 

\(b_{n}=\frac{2}{T} \int_{0}^{T} f(t) \sin n \omega_{0} t d t\) [11]

 

\(v(t)= \begin{cases}V_{m} \cos 5 \pi t & 0 \leq t \leq 0.1 \\ 0 & 0.1 \leq t \leq 0.3 \\ V_{m} \cos 5 \pi t & 0.3 \leq t \leq 0.4\end{cases}\)

 

However, choosing the period to extend from t = −0.1 to t = 0.3 will result in fewer equations and, hence, fewer integrals:

 

\(v(t)=\left\{\begin{array}{lr}<br /> V_{m} \cos 5 \pi t & -0.1 \leq t \leq 0.1 \\<br /> 0 & 0.1 \leq t \leq 0.3<br /> \end{array}\right.\) [12]

 

This form is preferable, although either description will yield the correct results.

 

Construct an appropriate set of equations.

The zero-frequency component is easily obtained:

 

\(a_{0}=\frac{1}{0.4} \int_{-0.1}^{0.3} v(t) d t=\frac{1}{0.4}\left[\int_{-0.1}^{0.1} V_{m} \cos 5 \pi t d t+\int_{0.1}^{0.3} 0 d t\right]\)

 

The amplitude of a general cosine term is

 

\(a_{n}=\frac{2}{0.4} \int_{-0.1}^{0.1} V_{m} \cos 5 \pi t \cos 5 \pi n t d t\)

 

and the amplitude of a general sine term is

 

\(b_{n}=\frac{2}{0.4} \int_{-0.1}^{0.1} V_{m} \cos 5 \pi t \sin 5 \pi n t d t\)

 

which, in fact, is always zero, and hence will not be considered further.

 

Determine if additional information is required.

The form of the function we obtain upon integrating is different when n is unity than it is for any other choice of n. If n = 1, we have

 

\(a_{1}=5 V_{m} \int_{-0.1}^{0.1} \cos ^{2} 5 \pi t d t=\frac{V_{m}}{2}\) [13]

 

whereas if n is not equal to unity, we find

 

\(a_{n}=5 V_{m} \int_{-0.1}^{0.1} \cos 5 \pi t \cos 5 \pi n t d t\)

 

Attempt a solution.

Solving, we find that

 

\(a_{0}=\frac{V_{m}}{\pi}\) [14]

 

\(a_{n}=5 V_{m} \int_{-0.1}^{0.1} \frac{1}{2}[\cos 5 \pi(1+n) t+\cos 5 \pi(1-n) t] d t\)

 

or

 

\(a_{n}=\frac{2 V_{m}}{\pi} \frac{\cos (\pi n / 2)}{1-n^{2}} \quad(n \neq 1)\) [15]

 

(A similar integration shows that \(b_{n}=0\) for any value of n, and the Fourier series thus contains no sine terms.) The Fourier series is therefore obtained from Eqs. [1], [13], [14], and [15]:

 

\(f(t)=a_{0}+\sum_{n=1}^{\infty}\left(a_{n} \cos n \omega_{0} t+b_{n} \sin n \omega_{0} t\right)\) [1]

 

\(v(t)=\frac{V_{m}}{\pi}+\frac{V_{m}}{2} \cos 5 \pi t+\frac{2 V_{m}}{3 \pi} \cos 10 \pi t-\frac{2 V_{m}}{15 \pi} \cos 20 \pi t\)

 

\(+\frac{2 V_{m}}{35 \pi} \cos 30 \pi t-\cdots\) [16]

 

Verify the solution. Is it reasonable or expected?

Our solution can be checked by plugging values into Eq. [16] and truncating after a specific number of terms. Another approach, however, is to plot the function as shown in Fig. 18.3 for n = 1, 2, and 6.

As can be seen, as more terms are included, the more the plot resembles that of Fig. 18.2.
