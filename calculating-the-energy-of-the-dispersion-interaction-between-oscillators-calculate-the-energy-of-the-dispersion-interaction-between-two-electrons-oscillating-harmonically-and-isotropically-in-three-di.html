Calculating the energy of the dispersion interaction between oscillators
Calculate the energy of the dispersion interaction between two electrons oscillating harmonically and isotropically in three dimensions about centres separated by a distance R, and express the answer in terms of their polarizabilities.

Method We base the answer on eqn 12.37. For the matrix elements, we use the values in , but we need to distinguish the frequencies, force constants, and masses by subscripts A and B for the two ‘atoms’ (an ‘atom’ is an electron and its centre). Let the states of atoms A and B be labelled by \(\nu _{A}\) and \(\nu _{B}\), respectively. The selection rules result in the restriction of the sum in eqn 12.37 to only four terms, so it may be evaluated explicitly. For the relation to the polarizabilities, use the results obtained in .

\(E^{(2)} =-\frac{2}{3} \left(\frac{1}{4\pi \varepsilon _{0}R^{3} } \right) ^{2} \sum\limits_{n_{A},n_{B}\neq 0 }{\frac{(\mu _{A,0_{A}n_{A} }\cdot \mu _{A,n_{A}0_{A} } )(\mu _{B,0_{B}n_{B} }\cdot \mu _{B,n_{B}0_{B} } )}{\Delta E_{n_{A}0_{A} } +\Delta E_{n_{B} 0_{B} } } } \) (12.37)

Admin
Solution

The sum we require has the following four non-zero terms:

\(E^{(2)} =-\frac{2}{3} \left(\frac{1}{4\pi \varepsilon _{0}R^{3} } \right) ^{2} \left\{\frac{\left|\left\langle \nu _{A}\left|\mu _{A}\right| \nu _{A}+1 \right\rangle \right|^{2} \left|\left\langle \nu _{B}\left|\mu _{B}\right| \nu _{B}+1 \right\rangle \right|^{2} }{\hbar (\omega _{A}+\omega _{B}) } +\frac{\left|\left\langle \nu _{A}\left|\mu _{A}\right| \nu _{A}+1 \right\rangle \right|^{2} \left|\left\langle \nu _{B}\left|\mu _{B}\right| \nu _{B}-1 \right\rangle \right|^{2}}{\hbar (\omega _{A}-\omega _{B}) } +\frac{\left|\left\langle \nu _{A}\left|\mu _{A}\right| \nu _{A}-1 \right\rangle \right|^{2} \left|\left\langle \nu _{B}\left|\mu _{B}\right| \nu _{B}+1 \right\rangle \right|^{2}}{\hbar (-\omega _{A}+\omega _{B}) }+\frac{\left|\left\langle \nu _{A}\left|\mu _{A}\right| \nu _{A}-1 \right\rangle \right|^{2} \left|\left\langle \nu _{B}\left|\mu _{B}\right| \nu _{B}-1 \right\rangle \right|^{2}}{\hbar (-\omega _{A}-\omega _{B}) } \right\} \)

Then, with the matrix elements from ,

\(E^{(2)} =-\frac{2}{3} \left(\frac{1}{4\pi \varepsilon _{0}R^{3} } \right) ^{2} \left(\frac{3\hbar e^{2} }{2m_{A}\omega _{A} } \right) \left(\frac{3\hbar e^{2} }{2m_{B}\omega _{B} } \right)\)

 

\(\times \left\{\frac{(\nu _{A}+1 )(\nu _{B}+1 )}{\hbar( \omega_{A}+\omega _{B}) } +\frac{(\nu _{A}+1 )\nu _{B}}{\hbar( \omega_{A}-\omega _{B}) }-\frac{\nu _{A}(\nu _{B}+1 )}{\hbar( \omega_{A}-\omega _{B})}-\frac{\nu _{A}\nu _{B}}{\hbar( \omega_{A}+\omega _{B})} \right\}\)

We have used the relation

\(\left|\left\langle \nu \left|\mu \right| \nu +1\right\rangle \right| ^{2} =\left|\left\langle \nu \left|\mu_{x} \right| \nu +1\right\rangle \right| ^{2}+\left|\left\langle \nu \left|\mu_{y} \right| \nu +1\right\rangle \right| ^{2}+\left|\left\langle \nu \left|\mu_{z} \right| \nu +1\right\rangle \right| ^{2}=3\left(\frac{\hbar e^{2} }{2m\omega _{0} } \right) (\nu +1)\)

and its analogues. It then follows that

\(E^{(2)} =-\frac{2}{3} \left(\frac{1}{4\pi \varepsilon _{0}R^{3} } \right) ^{2}\left(\frac{\hbar e^{4} }{m_{A}m_{B}\omega _{A}\omega _{B} } \right) \left\{\frac{(1+2\nu _{B} )\omega _{A}-(1+2\nu _{A} )\omega _{B} }{\omega^{2}_{A} -\omega ^{2}_{B} } \right\}\)

At this stage we can replace m by \(k_{f} /\omega ^{2} \) for each oscillator, and use the results from and eqn 12.19 that \(\alpha ^{\prime } =e^{2} /4\pi \varepsilon _{0} k_{f} \), to obtain

\(\alpha ^{\prime } =\frac{\alpha }{4\pi \varepsilon _{0} } \) (12.19)

\(E^{(2)} =-\frac{3}{2} \left(\frac{\alpha ^{\prime }_{A} \alpha ^{\prime }_{B}}{R^{6} } \right) \hbar \omega _{A} \omega _{B} \left\{\frac{(1+2\nu _{B} )\omega _{A} -(1+2\nu _{A} )\omega _{B}} {\omega ^{2}_{A}-\omega ^{2}_{B} } \right\} \)

When the two oscillators are in their ground states \((\nu _{A} = \nu _{B} = 0)\), this expression simplifies to

\(E^{(2)} =-\frac{3}{2} \left(\frac{\alpha ^{\prime }_{A} \alpha ^{\prime }_{B}}{R^{6} } \right) \left(\frac{\hbar \omega_{A} \omega_{B} }{\omega_{A} + \omega_{B}} \right) \)

Comment Keep this exact result in mind and compare it with the approximate London formula that we derive below: the two expressions have identical structures. In this case only a very limited number of transitions are allowed, and the closure approximation on which the London formula is based is exact.
