Guest Question.

The following are the parameters of a 230V, 50 Hz, 4-pole, single phase induction motor \(R_{1} = 2.2 \Omega; X_{1}= 3.0 \Omega; R’_{2} = 3.8 \Omega; X’_{2} = 2.1 \Omega; X_{m} = 86 \Omega .\)

Calculate the input current and power when the motor is operating at full-load speed of 1410 rpm.

If the mechanical losses (iron and friction loss) are 60 W, determine gross power developed, useful power at the shaft, shaft torque and efficiency.

Admin
Solution

Here, \(V = 230 V; f = 50 Hz; P = 4; N = 1410 rpm\)

\(R_{1} = 2.2 \Omega; X_{1}= 3.0 \Omega; R^{\prime }_{2} = 3.8 \Omega; R^{\prime }_{2} = 2.1 \Omega; X_{m} = 86 \Omega \)

 

For equivalent circuit, refer to Fig. 11.6 (b).

Synchronous speed, \(N_{S} = \frac{120 f}{P} = \frac{120 \times 50}{4} = 1500 rpm\)

Slip, \(S = \frac{N_{S} – N}{N_{S}} = \frac{1500 – 1410}{1500} = 0.06\)

Now, \(\frac{R^{\prime }_{2} }{2S} = \frac{3.8}{2 \times 0.06} = 31.67 \Omega ; \frac{X^{\prime }_{2} }{2} = \frac{2.1}{2} = 1.05 \Omega\)

\(\frac{R^{\prime }_{2} }{2\left(2 – S\right) } = \frac{3.8}{2 \left(2 – 0.06\right) } = 0.98 \Omega ; \frac{X_{m} }{2} = \frac{86}{2} = 43 \Omega\)

 

\(\overline{Z}_{f} = \frac{j\frac{X_{m}}{2}\left\lgroup\frac{R^{\prime }_{2} }{2S} + j \frac{X^{\prime }_{2} }{2}\right\rgroup }{\frac{R^{\prime }_{2} }{2S} + j \left\lgroup\frac{X_{m}}{2} + \frac{X^{\prime }_{2} }{2}\right\rgroup } = \frac{j43\left(31.67 + j1.05\right) }{31.67 + j\left(43 + 1.05\right) } \\[0.5cm] \quad \; \, = \left(25.22\angle 37.61^{\circ }\right) ohm = \left(19.98 + j 15.39\right) \Omega \)

 

\(\overline{Z}_{b} = \frac{j\frac{X_{m}}{2}\left\lgroup\frac{R^{\prime }_{2} }{2\left(2 – S\right) } + j \frac{X^{\prime }_{2} }{2}\right\rgroup }{\frac{R^{\prime }_{2} }{2\left(2 – S\right)} + j \left\lgroup\frac{X_{m}}{2} + \frac{X^{\prime }_{2} }{2}\right\rgroup } \\[0.5cm] \quad \; \, = \frac{j43\left(0.98 + j1.05\right) }{0.98 + j\left(43 + 1.05\right) } \\[0.5cm] \quad \; \,= 1.40\angle 48.3^{\circ } = \left(0.933 + j 1.05\right) \Omega \)

 

Total impedance, \(\overline{Z}_{T} = \overline{Z}_{1} + \overline{Z}_{f} + \overline{Z}_{b} \\[0.5cm] \hspace{30 pt} \hspace{30 pt} \qquad \quad \; = \left(2.2 + j3\right) + \left(19.98 + j15.39\right) + \left(0.933 + j1.05\right) \\[0.5cm] \hspace{30 pt} \hspace{30 pt} \qquad \quad \; = \left(23.11 + j 19.44\right) \Omega = 30.2 \angle 40.1^{\circ} \)

Input current, \(\overline{I}_{I} = \frac{\overline{V}}{\overline{Z}_{T}} = \frac{230 \angle 0^{\circ}}{30.2 \angle 40.1^{\circ}} = 7.616\angle -40.1^{\circ} A \)

\(I_{I} = \mathbf{7.616 A} \)

 

Input power, \( P = I^{2}_{1} R_{T} = \left(7.616\right)^{2} \times 23.11 = \mathbf{1340 W} \)

Alternately, \( P = V I_{1} \cos \phi = 230 \times 7.616 \times \cos 40.1^{\circ } = \mathbf{1340 W} \)

The voltage across AB, i.e.,

\( \overline{V}_{f} = \overline{I} _{1} \overline{Z}_{f} = 7.616 \angle -40.1^{\circ } \times 25.22 \angle 37.61^{\circ } \\[0.5cm] \quad \; \, = 200.5\angle -2.5^{\circ } V\)

 

Current in the forward parallel branch, i.e.,

\(\overline{I}_{f} = \frac{\overline{V}_{f}}{\overline{Z}_{2}’} = \frac{200.5 \angle -2.5^{\circ}}{31.67 + j 1.05} = \frac{200.5 \angle -2.5^{\circ}}{31.69\angle 1.9^{\circ}} \\[0.5cm] \quad \; \, = 6.327\angle -4.4^{\circ} A \)

 

The voltage across BC, i.e.,

\( \overline{V}_{b} = \overline{I} _{1} \overline{Z}_{b} = 7.616 \angle -40.1^{\circ } \times 1.40 \angle 48.3^{\circ } \\[0.5cm] \quad \; \,= 10.66\angle 8.2^{\circ } \)

 

Current in the backward parallel branch, i.e.,

\(\overline{I}_{b} = \frac{\overline{V}_{b}}{\overline{Z}_{2}’} = \frac{10.66 \angle 8.2^{\circ}}{0.98 + j 1.05} = \frac{10.66 \angle 8.2^{\circ}}{1.436\angle 47^{\circ}} = 7.42\angle -38.8^{\circ} A \)

 

\(Effective resistance of the forward branch = \frac{R^{\prime }_{2} }{2S} = \frac{3.8}{2\times 0.06} = 31.67 \Omega \)

 

\(The actual resistance existing in the branch = \frac{3.8}{2} = 1.9 \Omega \)

 

It depicts that (31.67 – 1.9 = 29.77 Ω) is the electrical equivalent resistance of mechanical load.

∴ Power developed in the forward branch

\(P_{mf} = I^{2}_{f} \times 29.77 = \left(6.327\right)^{2} \times 29.77 = 1192 W\)

 

\( Effective resistance of the backward branch = \frac{3.8}{2 \left(2 – S\right) } =\frac{3.8}{2 \left(2 – 0.06\right) } = 0.98 \Omega \)

 

\(Actual resistance existing in the branch = \frac{3.8}{2} = 1.9 \Omega \)

 

Electrical equivalent resistance representing mechanical load

\( = \left(0.98 – 1.9\right) = –0.92 \Omega \)

 

The negative sign indicates that the power developed produces a backward torque.

Power developed in the backward branch,

\(P_{mb} = I^{2}_{b} \times \left(-0.92\right) = \left(7.42\right)^{2} \times \left(-0.92\right) = -51 W\)

 

Hence, total mechanical power developed,

\(P_{m} = P_{mf} – P_{mb} = 1192 – 51 = 1141 W\)

 

Power at the shaft, \(P_{Sh} = P_{m} – mechanical loss = 1141 – 60 = \mathbf{1081 W} \)

\(Shaft torque = \frac{P_{Sh}}{2\pi N / 60} = \frac{60 \times 1081}{2\pi \times 1410} = \mathbf{7.32 Nm} \)

 

\(Efficiency = \frac{P_{out}}{P_{in}} = \frac{1081}{1340} = 0.8067 = \mathbf{80.67\%} \)
