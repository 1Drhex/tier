Guest Question.

Designing an enhancement-load NMOS inverter Design an enhancement-load NMOS inverter, as shown in Fig. 15.12(a), to obtain a noise margin of \( NM _{ L } \geq 0.8 V \). The threshold voltages are \( V_{ tL }=V_{ tD }=1 V \), and \( K_{ L }=20 \mu A / V ^{2} \). The supply voltage is \( V_{ DD }=5 V \). Assume load capacitance \( C_{ L }=0.5 pF \) and clock frequency \( f_{\text {clk }}=5 MHz \).

(a) Find the design parameter \( K_{ R }=K_{ D } / K_{ L } \), neglecting the body effect.
(b) Calculate \( NM _{ L } \) if the body effect is included. Assume \( \gamma=0.5 \sqrt{ V } \text { and } 2 \phi_{ f }=0.6 V \).
(c) Calculate the low-to-high propagation time \( t_{ pLH } \).
(d) Calculate the high-to-low propagation time \( t_{ pHL } \).
(e) Calculate the delay-power product (DP).

Admin
Solution
\( NM _{ L }=V_{ IL }-V_{ OL } \geq 0.8 V \text {. Thus, for } V_{ IL }=V_{ tD }=1 V,\)

 

\( V_{ OL }=V_{ IL }- NM _{ L } \leq 0.2 V \)

Technically, \( V_{ OH }=V_{ DD }-V_{ tD }=5-1=4 V \). However, to simplify the analysis for finding an expression for \( K_{ R } \), we will assume \( V_{ OH }=V_{ IH }=V_{ DD }=5 V \text {. For } V_{ IH }=5 V , M _{ L } \text { and } M _{ D } \) operate in the saturation and nonsaturation (ohmic) regions, respectively. Thus, for \( v_{ I }=V_{ IH }=V_{ OH } \text { and } v_{ O }=V_{ OL } \), Eq. (15.33) yields

\( K_{ L }\left(V_{ DD }-v_{ O }-V_{ tL }\right)^{2}=K_{ D }\left[2\left(v_{ I }-V_{ tD }\right) v_{ O }-v_{ O }^{2}\right] \) (15.33)

 

\( K_{ D }\left[2\left(V_{ OH }-V_{ tD }\right) V_{ OL }-V_{ OL }^{2}\right]=K_{ L }\left(V_{ DD }-V_{ OL }-V_{ tL }\right)^{2} \) (15.39)

which gives the desired value of the geometry ratio \( K_{ R } \) as

\( K_{ R }=\frac{K_{ D }}{K_{ L }}=\frac{\left(V_{ DD }-V_{ OL }-V_{ tL }\right)^{2}}{2\left(V_{ OH }-V_{ tD }\right) V_{ OL }-V_{ OL }^{2}} \) (15.40)

(a) Without the body effect, \( V_{ tD }=1 V . \text { For } V_{ OL }=0.2 V \text { and } V_{ OH }=5 V , \text { Eq. (15.40) gives } K_{ R }=9.26 \). Since a higher value gives a lower value of \( V_{ OL } \), we choose \( K_{ R } = 10 \). For this value of \( K_{ R } \), we can find \( V_{ OL } \) by solving the quadratic equation in Eq. (15.39):

\( 10\left[2(5-1) V_{ OL }-V_{ OL }^{2}\right]=\left(5-V_{ OL }-1\right)^{2} \)

which gives \( V_{ OL }=0.19 V \text { or } 7.81 V \) (not physically meaningful). Thus, \( V_{ OL }=0.19 V \), which is less than 0.2 V.

(b) With the body effect of \( V_{ O }=0.2 V \), Eq. (15.38) gives

\( V_{ tL }=V_{ tD 0}+\gamma\left|\sqrt{v_{ O }+2 \phi_{ f }}-\sqrt{2 \phi_{ f }}\right| \) (15.38)

 

\(\begin{aligned}V_{ tL } &=V_{ tD }+\gamma\left|\sqrt{v_{ O }+2 \phi_{ f }}-\sqrt{2 \phi_{ f }}\right| \\&=1+0.5|\sqrt{0.2+0.6}-\sqrt{0.6}|=1.06 V\end{aligned}\) (15.41)

 

With this value of \( V_{ tL }=1.06 V \), Eq. (15.39) gives

\( 10\left[2(5-1) V_{ OL }-V_{ OL }^{2}\right]=\left(5-1.06-V_{ OL }\right)^{2} \)

which gives \( V_{ OL }=0.18 V \). Thus,

\( NM _{ L }=V_{ tL }-V_{ OL }=1-0.18=0.82 V \)

which is better than the specified 0.8 V.

(c) The NMOS inverter with a load capacitor \( C _{ L } \) is shown in Fig. 15.14(a). As \( v_{ I } \text { goes low to } V_{ OL }, M _{ D } \) turns off immediately, and the capacitor \( C _{ L } \) is charged up by the drain current \( i_{ DL } \). The operating point, shown in Fig. 15.14(b), moves along the load line of \( M _{ L } \) from point Q of \( V_{ OL } \text { to midpoint } P \text { of } V_{ O ( mid )}=\left(V_{ OH }+V_{ OL }\right) / 2 \).
The charging of the capacitor is shown in Fig. 15.14(c). The output voltage is given by

\( C_{ L } \frac{d v_{ O }}{d t}=i_{ DL }=K_{ L }\left(V_{ DD }-v_{ O }-V_{ tL }\right)^{2} \) (15.42)

which can be integrated from \( V_{ OL } \text { to } V_{ O ( mid )} \) to give the low-to-high propagation time \( t_{ pLH } \), as shown in Fig. 15.14(d). To simplify the analysis, we will find the average value of the charging current \( I_{ CLH } \) and then find an approximate value of \( t_{ pLH } \); that is,

\( t_{ pLH }=\frac{C_{ L }\left(V_{ O ( mid )}-V_{ OL }\right)}{I_{ CLH }} \) (15.43)

where \( I_{\text {CLH }} \) is given by

\( I_{ CLH }=\frac{i_{ C }\left(V_{ OL }\right)+i_{ C }\left(V_{ O ( mid )}\right)}{2} \) (15.44)

For \( V_{ OL }=0.2 V \),

 

\(\begin{aligned}&V_{ OH }=V_{ DD }-V_{ tL }=5-1=4 V \\&V_{ O ( mid )}=\frac{V_{ OH }+V_{ OL }}{2}=2.1 V\end{aligned}\)

 

Then

\(\begin{aligned}&i_{ C }\left(V_{ OL }\right)=i_{ DL }\left(V_{ OL }\right)=K_{ L }\left(V_{ DD }-v_{ O }-V_{ tL }\right)^{2}=20 \mu \times(5-0.2-1)^{2}=288.8 \mu A \\&i_{ C }\left(V_{ O ( mid )}\right)=K_{ L }\left(V_{ DD }-V_{ O ( mid )}-V_{ tL }\right)^{2}=20 \mu \times(5-2.1-1)^{2}=72.2 \mu A \\&I_{ CLH }=\frac{288.8 \mu+72.2 \mu}{2}=180.5 \mu A\end{aligned}\)

Thus, \( t_{ pLH }=0.5 pF \times \frac{(2.1-0.2) V }{180.5 \mu A }=5.26 ns \)

 

(d) As \( v_{ I } \text { goes high to } V_{ OH }, M _{ D } \) turns on and the capacitor \( C _{ L } \text { discharges through } M _{ D } \). The operating point moves along the load line of \( M _{ D } \text { from point } B \text { of } V_{ OH } \text { to midpoint } D \text { of } V_{ O ( mid )}=\left(V_{ OH }+V_{ OL }\right) / 2 \). The output voltage is given by

\( C_{ L } \frac{d v_{ O }}{d t}=i_{ DD }-i_{ DL } \) (15.45)

 

where \( i_{ DD }= \begin{cases}K_{ D }\left(V_{ OH }-V_{ tD }\right)^{2} & \text { for }\left(V_{ OH }-V_{ tD }\right) \geq v_{ O }>V_{ O ( mid )} \\ K_{ D }\left[2\left(V_{ OH }-V_{ tD }\right) v_{ O }-v_{ O }^{2}\right] & \text { for } v_{ O } \leq V_{ O ( mid )}\end{cases} \)

Using the average value of the capacitor discharging current \( i_{ CHL } \), we can find the approximate value of \( t_{ pHL } \) from

\( t_{ pHL }=\frac{C_{ L }\left(V_{ OH }-V_{ O ( mid )}\right)}{I_{ CHL }} \) (15.46)

where \( i_{ CHL } \) is given by

\( i_{ CHL }=\frac{i_{ DD }\left(V_{ OH }\right)+i_{ DD }\left(V_{ O ( mid )}\right)-i_{ DL }\left(V_{ O ( mid )}\right)}{2} \) (15.47)

 

For \( K_{ D }=K_{ R } K_{ L }=200 \mu A / V ^{2} \),

 

\(\begin{aligned}V_{ OL } &=0.2 V \\V_{ OH } &=V_{ DD }-V_{ tL }=5-1=4 V \\V_{ O ( mid )} &=\frac{V_{ OH }+V_{ OL }}{2}=2.1 V\end{aligned}\)

 

Thus,

\(\begin{aligned}i_{ DD }\left(V_{ OH }\right) &=K_{ D }\left(V_{ OH }-V_{ tD }\right)^{2}=200 \mu \times(4-1)^{2}=1800 \mu A \\i_{ DD }\left(V_{ O ( mid )}\right) &=K_{ D }\left[2\left(V_{ OH }-V_{ tD }\right) V_{ O }-V_{ O }^{2}\right]=200 \mu \times\left[2 \times(4-1) \times 2.1-2.1^{2}\right]=1638 \mu A \\i_{ DL }\left(V_{ O ( mid )}\right) &=K_{ L }\left(V_{ DD }-V_{ O }-V_{ tL }\right)^{2}=20 \mu \times(5-2.1-1)^{2}=72.2 \mu A \\i_{ CHL } &=\frac{1800+1638-72.2}{2}=1683 \mu A \\t_{ pHL } &=0.5 pF \times \frac{(4-2.1) V }{1683 \mu A }=0.56 ns\end{aligned}\)

which is much shorter than \( t_{ pLH }=5 ns \). The value of \( i_{ DL }\left(V_{ O ( mid )}\right) \), which is much smaller than that of \( i_{ DD }\left(V_{ OH }\right) \), can often be neglected.

(e) The propagation time \( t_{ pd }\), which is the average of \( t_{ pLH } \text { and } t_{ pHL } \), is

\( t_{ pd }=\frac{t_{ pLH }+t_{ pHL }}{2}=\frac{5.26 ns +0.56 ns }{2}=2.91 ns \)

 

Then \( P_{\text {static }}=\frac{V_{ DD } i_{ D }\left(V_{ DL }\right)} {2}=\frac{V_{ DD } K_{ L }\left(V_{ DD }-V_{ OL }-V_{ tL }\right)^{2}}{2} \) (15.48)

 

\( =\frac{5 \times 20 \mu \times(5-0.2-1)^{2}}{2}=722 \mu W \)

 

\(\begin{aligned}&\text { and } P_{\text {dynamic }}=f_{\text {clk }} C_{ L } V_{ DD }^{2}=5 MHz \times 0.5 pF \times 5^{2}=62.5 \mu W \\&\text { so } \quad P_{ D }=P_{\text {static }}+P_{\text {dynamic }}=722 \mu+62.5 \mu=784.5 \mu W\end{aligned}\)

 

Therefore, the delay-power product is

\( DP =P_{ D } \times t_{ pd }=784.5 \mu W \times 2.91 ns =2.28 pJ \)
