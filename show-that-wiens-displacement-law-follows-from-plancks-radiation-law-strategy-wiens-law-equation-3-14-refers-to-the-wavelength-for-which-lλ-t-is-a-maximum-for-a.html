Guest Question.

Show that Wien’s displacement law follows from Planck’s radiation law. Strategy Wien’s law, Equation (3.14), refers to the wavelength for which \(\ell(\lambda, T)\) is a maximum for a given temperature. From calculus we know we can find the maximum value of a function for a certain parameter by taking the derivative of the function with respect to the parameter, set the derivative to zero, and solve for the parameter.

\(\lambda_{\max } T=2.898 \times 10^{-3} m \cdot K\) (3.14)

Admin
Solution

Therefore, to find the value of the Planck radiation law for a given wavelength we set \(d \ell / d \lambda=0\) and solve for λ.

 

\(\frac{d\ell(\lambda, T)}{d \lambda}=0 \quad \text { for } \lambda=\lambda_{\max }\)

 

\(\left.2 \pi c^{2} h \frac{d}{d \lambda}\left[\lambda^{-5}\left(e^{h c / \lambda k T}-1\right)^{-1}\right]\right|_{\lambda_{\max }}=0\)

 

\(-5 \lambda_{\max }^{-6}\left(e^{h c / \lambda_{\max } k T}-1\right)^{-1}-\lambda_{\max }^{-5}\left(e^{h c / \lambda_{\max } k T}-1\right)^{-2}\)

 

\(\times\left(\frac{-h c}{k T \lambda_{\max }^{2}}\right) e^{h c / \lambda_{\max } k T}=0\)

 

Multiplying by \(\lambda_{\max }^{6}\left(e^{h c / \lambda_{\max } k T}-1\right)\) results in

 

\(-5+\frac{h c}{\lambda_{\max } k T}\left(\frac{e^{h c / \lambda_{\max } k T}}{e^{h c / \lambda_{\max } k T}-1}\right)=0\)

 

Let

 

\(x=\frac{h c}{\lambda_{\max } k T}\)

 

Then

 

\(-5+\frac{x e^{x}}{e^{x}-1}=0\)

 

and

 

\(x e^{x}=5\left(e^{x}-1\right)\)

 

This is a transcendental equation and can be solved numerically (try it!) with the result \(x \approx 4.966\), and therefore

 

\(\frac{h c}{\lambda_{\max } k T}=4.966\)

 

\(\lambda_{\max } T=\frac{h c}{4.966 k}=\frac{1240 eV \cdot nm }{4.966\left(8.617 \times 10^{-5} \frac{ eV }{ K }\right)} \frac{10^{-9} m }{ nm }\)

 

and finally,

 

\(\lambda_{\max } T=2.898 \times 10^{-3} m \cdot K\)

 

which is the empirically determined Wien’s displacement law.
