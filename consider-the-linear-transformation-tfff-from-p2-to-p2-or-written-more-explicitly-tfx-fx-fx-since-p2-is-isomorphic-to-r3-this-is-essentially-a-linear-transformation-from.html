Guest Question.

Consider the linear transformation:

\(T(f)=f^{\prime}+f^{\prime \prime}\) from \(P_{2}\) to \(P_{2}\) or, written more explicitly, \(T(f(x))=f^{\prime}(x)+f^{\prime \prime}(x)\)

Since \(P_{2}\) is isomorphic to \(\mathbb{R}^3\) , this is essentially a linear transformation from \( \mathbb{R}^3\) to \(\mathbb{R}^3\) , given by a 3 × 3 matrix \(B\) . Let’s see how we can find this matrix.

Admin
Solution

If we let \(f(x) = a + bx + cx^2\), then we can write transformation \(T\) as:

\(T\left(a+b x+c x^{2}\right)=\left(a+b x+c x^{2}\right)^{\prime}+\left(a+b x+c x^{2}\right)^{\prime \prime}=(b+2 c x)+2 c=(b+2 c)+2 c x \) .

Next let’s write the input \(f(x) = a + bx + cx^2\) and the output \( T(f(x))= (b+2c)+2cx \) in coordinates with respect to the standard basis \( \mathfrak{B}=(1,x,x^2)\) of \(P_{2}\):

\(f(x)=a+b x+c x^{2} \xrightarrow[]{T} T(f(x))=(b+2 c)+2 c x \)

 

\( \downarrow L_{\mathfrak{B}} \hspace{30 pt} \downarrow L_{\mathfrak{B}}\)

 

\([f(x)]_{\mathfrak{B}}=\left[\begin{array}{l}a \\b \\c\end{array}\right] \quad \longrightarrow[T(f(x))]_{\mathcal{\mathfrak{B}}}=\left[\begin{array}{c} b+2 c \\2 c \\0\end{array}\right]\)

Written in \(\mathfrak{B}\)-coordinates, transformation \(T\) takes \([f(x)]_{\mathfrak{B}}\):

\([T(f(x))]_{\mathfrak{B}}=\left[\begin{array}{c}b+2 c \\2 c \\0 \end{array}\right]=\left[ \begin{array}{lll}0 & 1 & 2 \\0 & 0 & 2 \\0 & 0 &0 \end{array}\right]\left[\begin{array}{l}a \\b \\c\end{array}\right]=\left[\begin{array}{lll}0 & 1 & 2 \\0 & 0 & 2 \\0 & 0 & 0\end{array}\right][f(x)]_{\mathfrak{B}}\)

The matrix: \( B=\begin{bmatrix} 0&1&2\\0&0&2\\0&0&0 \end{bmatrix} \)

is called the \(\mathfrak{B}\)-matrix of transformation \(T\). It describes the transformation \(T\) if input and output are written in \(\mathfrak{B}\)-coordinates. Let us summarize our work in two diagrams:
