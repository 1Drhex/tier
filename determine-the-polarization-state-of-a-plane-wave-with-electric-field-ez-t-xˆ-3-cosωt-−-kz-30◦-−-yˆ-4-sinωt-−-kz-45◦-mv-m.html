Guest Question.

Determine the polarization state of a plane wave with electric field \(E(z, t) = \hat{x } 3 cos(ωt − kz + 30^◦)−\hat{y } 4 sin(ωt − kz + 45^◦) (mV/m).\)

Admin
Solution

We begin by converting the second term to a cosine reference,

\(E=\hat{x } 3 cos(ωt − kz + 30^◦)−\hat{y } 4 cos(ωt − kz + 45^◦- 90^◦)\)

 

\(=\hat{x } 3 cos(ωt − kz + 30^◦)−\hat{y } 4 cos(ωt − kz – 45^◦)\)

The corresponding field phasor \(\tilde{E} (z)\) is

\(\tilde{E} (z)= \hat{x } 3e^{−jkz}e^{j30^◦} − \hat{y } 4e^{−jkz}e^{−j45^◦}\)

 

\(=\hat{x } 3e^{−jkz}e^{j30^◦}+ \hat{y } 4e^{−jkz}e^{−j45^◦}e^{j180^◦}\)

 

\(=\hat{x } 3e^{−jkz}e^{j30^◦}+ \hat{y } 4e^{−jkz}e^{j135^◦}\)

where we have replaced the negative sign of the second term with \(e^{j180^◦}\) in order to have positive amplitudes for both terms, thereby allowing us to use the definitions given in Section 7-3.3. According to the expression for \(\tilde{E} (z)\), the phase angles of the x and y components are \(δ_x = 30^◦ and δ_y = 135^◦\), giving a phase difference \(δ = δ_y − δ_x = 135^◦ − 30^◦ = 105^◦.\) The auxiliary angle \(ψ_0\) is obtained from

\(ψ_0=\tan ^{-1}(\frac{a_y}{a} )=\tan ^{-1}(\frac{4}{3} )= 53.1^◦.\)

 

From Eq. (7.59a),

\(\tan 2γ = (\tan 2ψ_0) \cos δ \quad (-\pi /2\leq \gamma \leq \pi /2)\) (7.59a)

\(tan 2γ = (tan 2ψ_0) cos δ = tan 106.2^◦ cos 105^◦ = 0.89\)

 

which gives two solutions for γ, namely \( γ = 20.8^◦ and γ = −69.2^◦.\) Since cos δ < 0, the correct value of γ is \(−69.2^◦.\)

From Eq. (7.59b),

\(\sin 2χ = (\sin 2ψ_0) \sin δ \quad (-\pi /4\leq χ \leq \pi /4)\) (7.59b)

\(sin 2χ = (sin 2ψ_0) sin δ\)

 

\(= sin 106.2^◦ sin 105^◦\)

 

\(= 0.93 or χ = 34.0^◦.\)

 

The magnitude of \(\) indicates that the wave is elliptically polarized and its positive polarity specifies its rotation as left handed.
