Guest Question.

Equipment ratings for the four-bus power system shown in Figure 7.14 are as follows:

Generator G1: \(500 MVA , 13.8 kV , X ^{\prime \prime}=0.20 \text { per unit }\)
Generator G2: \(750 MVA , 18 kV , X ^{\prime \prime}=0.18 \text { per unit }\)
Generator G3: \(1000 MVA , 20 kV , X ^{\prime \prime}=0.17 \text { per unit }\)
Transformer T1: \(500 MVA , 13.8 \Delta / 500 Y kV , X =0.12 \text { per unit }\)
Transformer T2: \(750 MVA , 18 \Delta / 500 Y kV , X =0.10 \text { per unit }\)
Transformer T3: \(1000 MVA , 20 \Delta / 500 Y kV , X =0.10 \text { per unit }\)
Each 500 kV line: \(X _{1}=50 \Omega\).

A three-phase short circuit occurs at bus 1, where the prefault voltage is 525 kV. Prefault load current is neglected. Draw the positive-sequence reactance diagram in per unit on a 1000-MVA, 20-kV base in the zone of generator G3. Determine (a) the Thévenin reactance in per unit at the fault, (b) the subtransient fault current in per unit and in kA rms, an (c) contributions to the fault current from generator G1 and from line 1–2.

 

Admin
Solution

\(X_{g 1}^{\prime \prime}=(0.20)\left(\frac{1000}{500}\right)=0.40 \text { per unit }\) \(X _{T 1}=(0.12)(1000 / 500)=0.24 \text { per unit }\)

 

\(\begin{aligned}& X _{g 2}^{\prime \prime}=(0.18)(1000 / 750)=0.24 Per \text { unit } \\&X_{j 3}^{\prime \prime}=0.17 \text { per unit }\end{aligned}\) \(\begin{aligned}&X_{T 2}=(0.10)(1000 / 750)=0.1333 \text { per unit } \\&X_{T 3}=0.10 \text { per unit }\end{aligned}\)

 

\(Z_{\text {base } 4}=\frac{V_{\text {base 4 }}}{S_{\text {base }}}=\frac{(500)^{2}}{1000}=250 \Omega\) \(X_{\text {Line } 1.2}=X_{\text {Line } 2.3}=X_{\text {Line2.4 }}=\frac{50}{250}=0.20 \text { per unit }\)

 

(a)

\(\begin{aligned}X_{T h}=&(0.40+0.24) / /[0.20+(0.20+0.10+0.17) / /(0.20+0.1333+0.24)] \\&=0.64 / /[0.20+0.47 / / 0.5733]=0.64 / / 0.4583 \\&=\underline{\underline{0.2670}} \text { per unit }\end{aligned}\)

 

(b) \(\bar{V}_{F}=\frac{525}{500}=1.05 \angle 0^{\circ} \text { per unit }\) \(I_{b a s e 4}=\frac{S_{\text {base }}}{\sqrt{3} \nabla_{\text {base } 4}}=\frac{1000}{(\sqrt{3})(500)}=1.155 kA\)

\(\bar{I}_{F}^{\prime \prime}=\frac{\bar{V}_{F}}{\bar{Z}_{T h}}=\frac{1.05 \angle 0^{\circ}}{j 0.2670}= \underline{\underline{ -j 3.933}} \text { per unit }=(-j 3.933)(1.155)=\underline{\underline{-j 4.541 kA}}\)

 

(c) Using current division:

\(\bar{I}_{g 1}^{\prime \prime}=\bar{I}_{F}^{\prime \prime} \frac{0.4583}{(0.4583+0.64)}=(-j 3.933)(0.4173)=\underline{\underline{-j 1.641}} \text { per unit }=\underline{\underline{-j 1.896}} kA\)

 

\(\bar{I}_{\text {Line} 1.2}^{\prime \prime}=\bar{I}_{F}^{\prime \prime}\left(\frac{0.64}{0.4583+0.64}\right)=(-j 3.933)(0.5827)=\underline{\underline{-j 2.292}} \text { per unit }=\underline{\underline{-j 2.647}} kA\)
