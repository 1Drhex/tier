Guest Question.

A binary communication system makes use of the symbols ‘zero’ and ‘one’. There are channel errors. Consider the following events:

\(\begin{aligned}&x_{0}: \text { a 'zero' is transmitted } \\&x_{1}: \text { a 'one' is transmitted } \\&y_{0}: \text { a 'zero' is received } \\&y_{1}: \text { a 'one' is received }\end{aligned}\)

The following probabilities are given: \(P\left(x_{0}\right)=\frac{1}{2} P\left(y_{0} / x_{0}\right)=\frac{3}{4} \text { and } P\left(y_{0} / x_{1}\right)=\frac{1}{2} \text {. }\)The information in bits that you obtain when you learn which symbol has been received (while you know that a ‘zero’ has been transmitted) is _____.

Admin
Solution

It is given that,

P\(\left(x_{0}\right)\) = 1/2

\(P\left(y_{0} / x_{0}\right)=3 / 4 \quad P\left(y_{0}\right) \cdot P\left(x_{0}\right)=P\left(x_{0}, y_{0}\right)\)

 

\(\begin{aligned}P\left(y_{1} / x_{0}\right) &=1 / 4 \quad \frac{3}{4} \times 1 / 2=P\left(x_{0}, y_{0}\right) \\\frac{3}{8} &=P\left(x_{0}, y_{0}\right) \\P\left(x_{0}, y_{1}\right) &=P\left(x_{0}\right) P\left(y_{1}\right)\end{aligned}\)

= 1/2 × 1/4 = 1/8.

(Using the expression of entropy for conditional probability)

\(\begin{aligned}H\left(y / x_{0}\right) &=P\left(y_{0} / x_{0}\right) \log _{2} \frac{1}{ P \left( y _{0} / x _{0}\right)}+P\left(y_{1} / x_{0}\right) \\\log _{2} & \frac{1}{ P \left( y _{1} / x _{0}\right)} \\&=\frac{-3}{4} \log _{2} \frac{3}{4}-\frac{1}{4} \log _{2} \frac{1}{4}\end{aligned}\)

H \(\left(y / x_{0}\right)\) = 0.81 bits per symbol.
Hence, the correct Answer is (0.81)
