Guest Question.

Let us again solve the velocity analysis of , this time using the method of superposition. There are two cases that must be considered: case (a), where link 3 is temporarily considered frozen whereas link 2 has an angular velocity of ω2 = 10 rad/s ccw, and case (b), where link 2 is temporarily considered frozen whereas link 3 has an angular velocity of ω3 = 5 rad/s cw.

Admin
Solution

The angular velocity of link 4 can be written as a linear combination of case (a) and case (b). Using the angular velocity ratio theorem, this gives

\(\omega_{4}=\left(\frac{R_{I_{24}^{2} I_{12}^{2}}}{R_{I_{24}^{2} I_{14}^{2}}}\right) \omega_{2}+\left(\frac{R_{I_{34}^{3}I_{13}^{3}} }{R_{I_{34}^{3} I_{14}^{3}}}\right) \omega_{3},\) (1)

 

where the superscripts in the instant center labels denote the input variable that is moving whereas all others are temporarily considered frozen. Similarly, the angular velocity of link 5 can be written as

\(\omega_{5}=\left(\frac{R_{I_{25}^{2} I_{12}^{2}}}{R_{I_{25}^{2} I_{15}^{2}}}\right) \omega_{2}+\left(\frac{R_{I_{35}^{3} I_{13}^{3}}}{R_{I_{35}^{3} I_{15}^{3}}}\right) \omega_{3}.\) (2)

Comparing Eqs. (1) and (2) with Eqs. (5.4), the first-order kinematic coefficients of links 4 and 5 can be written as

\(\omega_{i}=d \theta_{i} / d t=\theta_{i j}^{\prime} \omega_{j}+\theta_{i k}^{\prime} \omega_{k} \text {. }\) (5.4)

 

\(\theta_{42}^{\prime}=\frac{R_{I_{24}^{2} I_{12}^{2}}}{R_{I_{24}^{2} I_{14}^{2}}}, \quad \theta_{43}^{\prime}=\frac{R_{I_{34}^{3} I_{13}^{3}}}{R_{I_{34}^{3} I_{14}^{3}}}, \quad \theta_{52}^{\prime}=\frac{R_{I_{25}^{2} I_{12}^{2}}}{R_{I_{25}^{2} I_{15}^{2}}}, \quad \text { and } \quad \theta_{53}^{\prime}=\frac{R_{I _{35}^{3} I_{13}^{3}}}{R_{I_{35}^{3} I_{15}^{3}}} .\) (3)

 

Note that a sign convention is not necessary if directed lines are used when measuring the relative locations of instant centers. However, if a sign convention is preferred, then each first-order kinematic coefficient is negative if the relative instant center \(I_{i j}^{k}\) is between the absolute instant centers \(I_{1 i}^{k} \text { and } I_{1 j}^{k}\) and positive if the relative instant center \(I_{i j}^{k}\) is outside the absolute instant centers\(I_{1 i}^{k} \text { and } I_{1 j}^{k}\).

Case (a): Link 3 is temporarily considered frozen. The linkage is temporarily regarded as a four-bar linkage, as shown in Fig. 5.12, with input from the rotation of link 2 alone.

From the Aronhold-Kennedy theorem, the secondary instant center \(I_{14}^{2}\) is the point of intersection of the line containing instant centers \(I_{15}^{2} \text { and } I_{45}^{2},\) and the line containing instant centers \(I_{12}^{2} \text { and } I_{24}^{2}.\) Similarly, the secondary instant center \(I_{25}^{2}\) is the point of intersection of the line containing instant centers \(I_{24}^{2} \text { and } I_{45}^{2}\), and the line containing instant centers \(I_{12}^{2} \text { and } I_{15}^{2}\).

Using the convention that lines proceed positive to the right, the distances between the instant centers, measured from the scaled drawing of Fig. 5.12, are

\(R_{I_{24}^{2} I_{12}^{2}}=-2.50 \text { in, } \quad R_{I_{24}^{2} I_{14}^{2}}=10.56 \text { in, } \quad R_{I_{25}^{2} I_{12}^{2}}=-7.76 \text { in, and } R_{I_{25}^{2} I_{15}^{2}}=-17.03 \text { in. }\) (4)

Case (b): Link 2 is temporarily considered frozen. The linkage is temporarily regarded as a four-bar linkage, as shown in Fig. 5.13, with input from the rotation of link 3 alone.

From the Aronhold-Kennedy theorem, the secondary instant center \(I_{34}^{3}\) is the point of intersection of the line containing instant centers \(I_{13}^{3} \text { and } I_{14}^{3} \text {, }\) and the line containing instant centers \(I_{35}^{3} \text { and } I_{45}^{3} \text {. }\) Similarly, the secondary instant center \(I_{15}^{3}\) is the point of intersection of the line containing instant centers \(I_{14}^{3} \text { and } I_{45}^{3},\) and the line containing instant centers \( I_{13}^{3} \text { and } I_{35}^{3} .\)

Continuing with the convention that lines proceed positive to the right, the distances between the instant centers, measured from the scaled drawing of Fig. 5.13, are

\(R_{I_{34}^{3} I_{13}^{3}}=18.06 \text { in, } \quad R_{I_{34}^{3} I_{14}^{3}}=25.62 \text { in, } \quad R_{I_{35}^{3} I_{13}^{3}}=4.00 \text { in, and } R_{I_{35}^{3} I_{15}^{3}}=-36.67 \text { in. }\) (5)

 

Substituting Eqs. (4) and (5) into Eqs. (3), the first-order kinematic coefficients of links 4 and 5 are

\(\theta_{42}^{\prime}=\frac{R_{I_{24}^{2} I_{12}^{2}}}{R_{I_{24}^{2} I_{14}^{2}}}=\frac{-2.50 \text { in }}{10.56 \text { in }}=-0.237, \quad \theta_{43}^{\prime}=\frac{R_{I_{34}^{3} I_{13}^{3}}}{R_{I_{34}^{3} I_{14}^{3}}}=\frac{18.06 \text { in }}{25.62 \text { in }}=0.705,\) (6a)

 

\(\theta_{52}^{\prime}=\frac{R_{I_{25}^{2} I_{12}^{2}}}{R_{I_{25}^{2} I_{15}^{2}}}=\frac{-7.76 \text { in }}{-17.03 in }=0.455, \quad \theta_{53}^{\prime}=\frac{R_{I_{35}^{3} I_{13}^{3}}}{R_{I_{35}^{3} I_{15}^{3}}}=\frac{4.00 in }{-36.67 in }=-0.109\) (6b)

Note that the values of the first-order kinematic coefficients of links 4 and 5, given by Eqs. (6), are all nondimensional (rad/rad) and are all in good agreement with the values found analytically in .

Substituting Eqs. (6) into Eq. (5.4), the angular velocities of links 4 and 5, respectively, are

\(\omega_{4}=-0.237(10 rad / s )+0.705(-5 rad / s )=-5.90 rad / s ( cw ),\)

 

\(\omega_{5}=0.455(10 rad / s )-0.109(-5 rad / s )=5.10 rad / s ccw.\)

Note that the answers for the angular velocities of coupler links 4 and 5 from the graphic methods ( and ) are in good agreement with the answers obtained from the analytic methods ( and ).
