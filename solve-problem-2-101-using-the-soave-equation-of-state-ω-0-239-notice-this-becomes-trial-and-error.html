Guest Question.

Solve using the Soave equation of state, ω = 0.239. Notice this becomes trial and error.

Admin
Solution

Carbon dioxide from table A.2: \(T _{ c }=304.1 K , P _{ c }=7380 kPa\)

\(T _{ r }= T / T _{ c }=333.15 / 304.1=1.09553\)

For Soave EOS see Appendix D (very close to Redlich-Kwong)

\(P=\frac{R T}{v-b}-\frac{a}{v^{2}+b v}\)

where the parameters are from Table D.1 and D.4

ω = 0.239

f = 0.48 + 1.574ω – 0.176\( \omega^{2}\) = 0.84613

\(a_{0}=0.42748\left[1+f\left(1-T_{r}^{1 / 2}\right)\right]^{2}=0.394381\)

 

\(b =0.08664 \frac{ RT _{ c }}{ P _{ c }}=0.08664 \times \frac{0.1889 \times 304.1}{7380}=0.0006744 m ^{3} / kg\)

 

\(a =0.394381 \frac{ R ^{2} T _{ c }^{2}}{ P _{ c }}=0.394381 \times \frac{0.1889^{2} \times 304.1^{2}}{7380}=0.176342 kPa \left( m ^{3} / kg \right)^{2}\)

 

The equation is:

\(P =\frac{ RT }{ v – b }-\frac{ a }{ v ^{2}+ bv }\)

 

\(10000=\frac{0.1889 \times 333.15}{ v -0.0006744}-\frac{0.176342}{ v ^{2}+0.0006744 \times v }\)

 

Trial and error on v (start guided by Table B.3.2):

 

\(\begin{array}{llll}v =0.0035 m ^{3} / kg & \Rightarrow & P =10202 kPa & \text { so } v \text { larger } \\v =0.0036 m ^{3} / kg & \Rightarrow & P =10051 kPa & \text { so } v \text { close } \\v = 0 . 0 0 3 6 3 m ^{3} / kg & \Rightarrow & P =10006 kPa & OK\end{array}\)

Equations of State
The cubic equation of state and contains as a special case the ideal-gas law:
\(P=\frac{R T}{v-b}-\frac{a}{v^{2}+c b v+d b^{2}}\)
where (a, b) are parameters and (c, d ) define the model as shown in the following table with the acentric factor (ω) and
\(b=b_{0} R T_{c} / P_{c} \quad \text { and } \quad a=a_{0} R^{2} T_{c}^{2} / P_{c}\)
The acentric factor is defined by the saturation pressure at a reduced temperature \(T_{r}=0.7\)
\(\omega=-\frac{\ln P_{r}^{ sat } \text { at } T_{r}=0.7}{\ln 10}-1\)
