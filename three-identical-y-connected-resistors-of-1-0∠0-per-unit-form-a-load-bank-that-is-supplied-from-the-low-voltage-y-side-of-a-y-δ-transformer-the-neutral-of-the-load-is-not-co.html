Guest Question.

Three identical Y-connected resistors of 1.0∠0° per unit form a load bank that is supplied from the low-voltage Y-side of a Y–Δ transformer. The neutral of the load is not connected to the neutral of the system. The positive- and negative-sequence currents flowing toward the resistive load are given by

\(I_{a, 1}=1 \angle 4.5^{\circ} \text { per unit; } \quad I_{a, 2}=0.5 \angle 250^{\circ} \text { per unit }\)

and the corresponding voltages on the low-voltage Y-side of the transformer are

\(V_{a n, 1}=1 \angle 45^{\circ} per\) per unit (Line-to-neutral voltage base)

\(V_{a n, 2}=0.5 \angle 250^{\circ}\) per unit (Line-to-neutral voltage base)

Determine the line-to-line voltages and the line currents in per unit on the high-voltage side of the transformer. Account for the phase shift.

Admin
Solution
\(\begin{gathered}\bar{V}_{A 1}=1 \angle 45^{\circ}+30^{\circ}=1 \angle 75^{\circ}=0.2588+j 0.9659 \\\bar{V}_{A 2}=0.5 \angle 250-30^{\circ}=0.5 \angle 220^{\circ}=-0.383-j 0.321 \\\bar{V}_{A}=\bar{V}_{A 1}+\bar{V}_{A 2}=0.12422+j 0.645=0.656 \angle 100.91^{\circ}\end{gathered}\)

 

\(\begin{aligned}&\bar{V}_{B 1}=a^{2} \bar{V}_{A 1}=1 \angle 315^{\circ}=1 \angle-45^{\circ}=0.7071-j 0.7071 \\&\bar{V}_{B 2}=a \bar{V}_{A 2}=0.5 \angle 340^{\circ}=0.5 \angle-20^{\circ}=0.469-j 0.171 \\&\bar{V}_{B}=\bar{V}_{B 1}+\bar{V}_{B 2}=1.177-j 0.878=1.47 \angle-36.7^{\circ}\end{aligned}\)

 

\(\begin{aligned}&\bar{V}_{C 1}=a \bar{V}_{A 1}=1 \angle 195^{\circ}=-0.9659-j 0.2588 \\&\bar{V}_{C 2}=a^{2} \bar{V}_{A 2}=0.5 \angle 100^{\circ}=-0.0868+j 0.492 \\&\bar{V}_{C}=\bar{V}_{C 1}+\bar{V}_{C 2}=-1.053-j 0.234=1.078 \angle 167.49^{\circ}\end{aligned}\)

 

Line-to-line voltages are given by: [in pu on line-neutral voltage base]

\(\begin{aligned}\bar{V}_{A B}=\bar{V}_{A}-\bar{V}_{B} &=-1.301+j 1.523 \\&=2 \angle 130.5^{\circ} \leftarrow \\\bar{V}_{B C}=\bar{V}_{B}-\bar{V}_{C} &=2.32-j 1.112 \\&=2.57 \angle-25.6^{\circ} \leftarrow \\\bar{V}_{C A}=\bar{V}_{C}-\bar{V}_{A} &=-1.177-j 0.411 \\&=1.25 \angle-160.75^{\circ} \leftarrow\end{aligned}\) \(\left\{\begin{array}{l}\underline{\text { Note: Divide by }} \sqrt{3} \\\text { if the base is } \\\text { line-to-line voltage }\end{array}\right.\)

 

Load impedance in each phase is \(1 \angle 0^{\circ} pu\) .

\(\therefore \bar{I}_{a 1}=\bar{V}_{a 1} \text { in pu; } \bar{I}_{a 2}=\bar{V}_{a 2} \text { in } pu\)

Thus \(\bar{I}_{A}=\bar{V}_{A} \text { in } pu\)

\(\left.\begin{array}{l}\bar{I}_{A}=0.656 \angle 100.91^{\circ} pu \\\bar{I}_{B}=1.47 \angle-36.7^{\circ} pu \\\bar{I}_{C}=1.078 \angle 167.49^{\circ} pu\end{array}\right\}\)
