Guest Question.

Obtain the Fourier transform of the triangular pulse \( q_{\tau }(t) \) shown in Fig. 2.10a.

Admin
Solution

From Fig. 2.10a, we have

\( q_{\tau }(t)= \begin{cases} 1-\frac{2\left|t\right| }{\tau } & for \left|t\right|\leq \tau /2 \\ 0 & for \left|t\right|\gt \tau /2 \end{cases} \)

We note that the given triangular pulse can be generated by performing the integration

\( q_{\tau }(t)=\int_{-\infty }^{t}{\frac{2x(t)}{\tau } }dt \) (2.51)

where

\( x(t)=p_{\tau /2}(t+\frac{1}{4}\tau )-p_{\tau /2}(t-\frac{1}{4}\tau ) \)

(see ). If we differentiate both sides in Eq. (2.51), we get

\( \frac{dq_{\tau }(t) }{dt}=\frac{2x(t)}{\tau } \) (2.52 a)

If we apply the time-differentiation theorem (Theorem 2.11) with k =1 to the left-hand side of Eq. (2.52a), we have

\( \digamma [\frac{dq_{\tau }(t) }{dt}]=j\omega Q_{\tau }(j\omega ) \) (2.52 b)

Ontheotherhand,ifweapplytheFouriertransformtotheright-handsideofEq.(2.52a), we get

\( \digamma [\frac{{2x }(t) }{dt}]=\frac{2X(j\omega )}{\tau } \) (2.52 c)

Therefore, from Eqs. (2.52a)–(2.52c)

\( \digamma [\frac{dq_{\tau }(t) }{dt}]=\digamma [\frac{2x(t)}{\tau }] \)

or

\( Q_{\tau }(j\omega ) =\frac{2X(j\omega )}{\tau } \)

and from Eq. (2.50), we get

\( X(j\omega )= (e^{j\omega \tau /4}-e^{-j\omega \tau /4} ) \digamma p_{\tau /2} (t)=\frac{4j\sin ^{2}\omega \tau /4 }{\omega } \) (2.50)

\( Q_{\tau }(j\omega ) =\frac{8\sin ^{2} \omega \tau /4}{\omega ^{2} \tau } \)

or

\( q_{\tau }(t) \leftrightarrow \frac{8\sin ^{2} \omega \tau /4}{\omega ^{2} \tau } \)

The amplitude spectrum is illustrated in Fig.2.10b. The phase spectrum is zero for all frequencies since the signal is a real,even function of time (seeEqs.(2.36a) and (2.36b)).

\( \Re e X(j\omega )=2\int_{0}^{\infty }{x(t)\cos \omega t}dt \) (2.36 a)

 

\( \Im m X(j\omega )=0 \) (2.36 b)
