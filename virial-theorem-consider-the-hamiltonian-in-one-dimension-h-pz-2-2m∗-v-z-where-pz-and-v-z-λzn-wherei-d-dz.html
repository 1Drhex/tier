Guest Question.

Virial theorem. Consider the Hamiltonian in one dimension

\(H = \frac{p_{z}^{2}}{2m^{*}} + V\left ( z \right )\) , (2.49)

where

\(p_{z} = \frac{h}{i}\frac{d}{dz}\) (2.50)

and
\(V\left ( z \right )= \lambda z^{n}\), (2.51)

where λ is real and n is an integer.
First prove the following result:
\(\left [ H, zp_{z} \right ]= ih - \left [ 2 \left ( \frac{p_{z}^{2}}{2m^{*}} \right )+ \lambda nz^{n} \right ]\), (2.52)

where H is the Hamiltonian above.
Starting with the results of the previous step, prove that if |φ is an eigenstate
of H, the following is true:
\(2\left \langle T \right \rangle = n\left \langle V \right \rangle\), (2.53)

where \(\left \langle ... \right \rangle\) stands for the expectation value in the state |φ and T is the kinetic energy operator expressed as
\(T=\frac{-h^{2}}{2m^{*}}\frac{d^{2}}{dz^{2}}\) . (2.54)

Equation (2.53) is known as the virial theorem.

Admin
Solution

Preliminary lemma: If \( |\phi >\) is an eigenstate of H, then \(< \phi |\left [ H,A \right ]\phi > = 0\), for any quantum mechanical operator A.

The expectation value of any quantum mechanical operator represents the average value of the corresponding physical variable which we expect to measure in an experiment. Therefore, the expectation value of a quantum mechanical operator must be real (we cannot measure an imaginary quantity). Since the expectation value of any Hermitian operator is always real (see Appendix C), all legitimate quantum mechanical operators are Hermitian.

Since \(|\phi >\) is an eigenstate of H, the following relation holds: \(H|\phi > = E|\phi >\), where E is the eigenenergy. Recalling that both H and A are Hermitian and that E is real, we obtain

\(< \phi |\left [ H,A \right ]\phi > =\int d^{3}\vec{r}\phi ^{\ast }\left ( \vec{r} \right )\left ( HA-AH \right )\phi \left ( \vec{r} \right )\)

 

\(= \int d^{3}\vec{r}\left \{ \left [ AH\phi \left ( \vec{r} \right ) \right ] ^{\ast }\phi \left ( \vec{r} \right )-\phi ^{\ast }\left ( \vec{r} \right )AH\phi \left ( \vec{r} \right )\right \}\)

 

\(= E\int d^{3}\vec{r}\left \{ \phi ^{\ast } \left ( \vec{r} \right )A\phi \left ( \vec{r} \right ) – \phi ^{\ast } \left ( \vec{r} \right )A\phi \left ( \vec{r} \right ) \right \} \) (2.55)

 

This property will be used below.

Starting with the operator identity

\(\left [ A,BC \right ]= \left [ A,B \right ]C +B\left [ A,C \right ]\) (2.56)

we get

\(\left [ H,zp_{z} \right ]= \left [ H,z \right ]p_{z}+z\left [ H,p_{z} \right ]\) (2.57)

 

Substituting the Hamiltonian given in Equation (2.49) in the preceding equation leads to

\(\left [ H,zp_{z} \right ]= \left [ \left ( \frac{p_{z}^{2}}{2m^{\ast }}+\lambda _{z}^{n} \right ), z \right ] p_{z}+z\left [ \left ( \frac{p_{z}^{2}}{2m^{\ast }}+\lambda _{z}^{n} \right ) ,p_{z} \right ]\)

 

\(= \left [ \frac{p_{z}^{2}}{2m^{\ast }},z \right ]p_{z}+\lambda \left [ z^{n} ,z \right ]p_{z}+\frac{z}{2m^{\ast }}\left [ p_{z}^{2},p_{z} \right ]+ z\left [ \lambda z^{n} , p_{z}\right ]\) (2.58)

 

The second and third commutators are equal to zero. Furthermore,

\(\left [ p_{z}, z^{n} \right ]= – ihnz^{n-1}\) (2.59)

and

\(\left [ z,p_{z}^{2} \right ] = 2ihp_{z}\) (2.60)

Using these results, we get

\(\left [ H,zp_{z} \right ] = \frac{1}{2m^{\ast }}\left ( -2ihp_{z} \right )p_{z} +\lambda z \left ( ihnz^{n-1} \right )\)

 

\( = \frac{1}{2m^{\ast }}\left ( -2ihp_{z} \right )p_{z} +i \lambda hnz^{n}\)

 

\( = ih \left [ -2\left ( \frac{p_{z}^{2}}{2m^{\ast }} \right )+\lambda nz^{n} \right ]\) (2.61)

Since \(|\phi >\) is an eigenstate of H, we can apply the preliminary lemma derived earlier
and get

\(< \left ( H,zp_{z} \right )> = 0= ih\left ( -2 < T> + n> \right ) .\) (2.62)

In other words,

\(2< T > =n < V > .\) (2.63)
