Guest Question.

Consider the \(causal^{14}\) system governed by

\(m \ddot{x}+c \dot{x}+k x=F(t)\)

Determine the particular solution by solving the system with \(x(0)=0 \text { and } \dot{x}(0)=0\) where \(F(t)=\sin \omega t \cdot u(t)\) and assuming c = 0.

 

Admin
Solution

The particular solution is the general solution with\(x()=\dot{x}(0)=0\) only if the excitation F(t) is nonzero for t ≥ 0. The steady-state solution is

\(x_{s}(t)=\left[\frac{1}{k} \cdot \frac{1}{1-\left(\omega / \omega_{n}\right)^{2}}\right] \operatorname{sin} \omega t\)

This solution cannot be obtained by the Laplace transform method since the range of integration of the Laplace transform covers positive time only. The function \(F(t)=\sin \omega t\) is harmonic over all time and the convolution cannot account for its values for t < 0. The general solution of Equation 4.30 is the sum of the transient and steady-state solutions, given by

\(\begin{aligned}x(t) &=x(0) \cos \omega_{n} t \\&+\left[\frac{\dot{x}(0)}{\omega_{n}}-\frac{1}{k} \cdot \frac{\omega / \omega_{n}}{1-\left(\omega / \omega_{n}\right)^{2}}\right] \sin \omega_{n} t \\&+\left[\frac{1}{k} \cdot \frac{1}{1-\left(\omega / \omega_{n}\right)^{2}}\right] \operatorname{sin} \omega t\end{aligned}\)

With \(x(0)=\dot{x}(0)=0\), let \(x(t)=y(t)\). It is obvious that

\(\begin{aligned}y(t) &=\left[\frac{1}{k} \cdot \frac{1}{1-\left(\omega / \omega_{n}\right)^{2}}\right]\left(\operatorname{sin} \omega t-\frac{\omega}{\omega_{n}} \sin \omega_{n} t\right) \\& \neq x_{o}(t)\end{aligned}\)

The first two terms of the general solution, defined as \(x(t)\),

\(z(t) \equiv x(0) \cos \omega_{n} t+\frac{\dot{x}(0)}{\omega_{n}} \sin \omega_{n} t\)

coincide with the response in free vibration to which the initial conditions are matched. Thus the term

\(x(t)-x_{0}(t)-z(t)=-\left[\frac{1}{k} \cdot \frac{1}{1-\left(\omega / \omega_{n}\right)^{2}}\right] \sin \omega_{n} t\)

arises because the general solution is used to satisfy the initial conditions in order to establish the two arbitrary constants. Suppose that the system is subjected to a positive-time harmonic driving force,

\(F(t)= \begin{cases}\sin \omega t & \text { if } t \geq 0 \\ 0 & \text { if } t<0\end{cases}\)

with c = 0. Since F (t) is non-zero only for t ≥ 0, the
assumption of a solution of the form \(x(t)=A \cos \omega t+\) \(B \sin \omega t\) cannot satisfy the governing equation. Instead, the steady-state solution can be found using the convolution

\(x_{o}(t)=\int_{0}^{t} F(\tau) g(t-\tau) d \tau\)

giving

\(x_{o}(t)=\left[\frac{1}{k} \cdot \frac{1}{1-\left(\omega / \omega_{n}\right)^{2}}\right]\left(\sin \omega t-\frac{\omega}{\omega_{n}} \operatorname{sin} \omega_{n} t\right)\)

for t ≥ 0, Thus, equation \(x(t)=y(t) \neq x_{o}(t)\) in this case.
