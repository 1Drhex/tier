Guest Question.

Analysis of Existing Diagonal Brace Connection

Determine the force distribution at the gusset plate connection interfaces for the existing diagonal brace connection shown in Figure 11-42, assuming that

a. The gusset-to-beam connection is more rigid than the gusset-to-column connection, and

b. The gusset-to-column connection is more rigid than the gusset-to-beam connection.

Assume a 1⁄2-in. setback between the gusset plate and the face of the column.

Admin
Solution

1. \( d_{c}\)(W14 × 90) = 14.0 in.; \(d_{b}\)(W24 × 94) = 24.3 in.

0.5\( d_{c}\)(W14 × 90) = 7.0 in.; 0.5\( d_{b}\)(W24 × 94) = 12.15 in.

θ = 55° from the geometry of the connection

X = 30 in. and Y = 18 in. (see Figure 11-42)

2. The actual centroidal distances are calculated as follows:

\(\bar{\alpha }=\frac{X}{2} + 0.5 in. = \frac{30 in.}{2} + 0,5 in. =15.5\) in.

\(\bar{\beta }=\frac{Y}{2} = \frac{18 in.}{2} = 9\) in.

3. If the gusset-to-beam connection is more rigid than the gusset-to-column connection (e.g., if a welded connection is used for the gusset-to-beam connection and a bolted connection is used for the gusset-to-column connection),

\(\beta =\overline{\beta }\) (from step 2) = 9 in. Substituting this in equation (11-49) yields

\(\alpha – \beta \tan \theta =0.5d_{b}\tan\theta -0.5d_{c}.\) (11-49)

\(\alpha – \beta \tan\theta =e_{b}\tan\theta -e_{c}.\)

∝ – (9 in.)(tan 55°) = (12.15 in.)(tan 55°) – 7.0 in.

Therefore, set \(\alpha = 23.2 in. =\overline{\alpha }\)

from step 2. Thus, we find that a moment exists on the gusset-to-beam connection interface; the moment and forces on this interface are calculated as follows:

\(r =\sqrt{(\alpha _{step3} +0.5d_{c} )^{2}+(\overline{\beta _{step2} } +0.5d_{c})^{2} }\)

 

\(=\sqrt{(23.2 in. + 7 in.)^{2}+(9 in. + 12.15 in.)^{2} }= 36.9\) in.

The forces on the gusset-to-beam interface are

\(H_{B}=\frac{\alpha _{step3} }{r_{step3} }P_{brace} =\frac{23.2 in.}{36.9 in.}(150 k) = 94.3\) kips

\(V_{B}=\frac{0.5 d_{b} }{r_{step3} }P_{brace} =\frac{12.15 in.}{36.9 in.}(150 k) = 49.4\) kips

\(M_{u,g–b} = V_{B}\left|\alpha -\overline{\alpha } \right|\) = (49.4 k)(23.2 in. – 15.5 in.)

= 380.4 in.-kips = 31.7 ft.-kips

The forces on the gusset-to-column interface are

\(H_{C}=\frac{0.5 d_{c} }{r }P_{brace} =\frac{7.0 in.}{36.9 in.}(150 k) = 28.5\) kips

\(V_{C}=\frac{\beta}{r }P_{brace} =\frac{9 in.}{36.9 in.}(150 k) = 36.6\) kips

\(M_{u,g–c}\)= 0 ft.-kips

4. If the gusset-to-column connection is more rigid than the gusset-to-beam connection (e.g., if a welded connection is used for the gusset-to-column connection and a bolted connection is used for the gusset-to- beam connection), set \(\alpha =\overline{\alpha }\) (from step 2) = 15.5 in. Substituting this in equation (11-49) yields

\(\alpha – \beta \tan\theta =0.5d_{b}\tan\theta -0.5d_{c}.\) (11-49)

15.5 in. -β (tan 55°) = (12.15 in.)(tan 55°) – 7.0 in.

Therefore, β =3.6 in. ≠\(\overline{\beta }\) from step 2. Thus, we find that a moment exists on the gusset-to-column connection interface; the moment and forces on this interface are calculated as follows:

\(r =\sqrt{(\overline{\alpha }_{step2} +0.5d_{c} )^{2}+(\beta _{step4} +0.5d_{c})^{2} }\)

 

\(=\sqrt{(15.5 in. + 7.0 in.)^{2}+(3.6 in. + 12.15 in.)^{2} }= 27.5\) in.

The forces on the gusset-to-column interface are

\(H_{C}=\frac{0.5d_{c} }{r_{step4} } P_{brace} = \frac{7.0 in.}{27.5 in.}(150 k) = 38.2\) kips

\(V_{C}=\frac{\beta _{step4} }{r_{step4} }P_{brace} =\frac{3.6 in.}{27.5 in.}(150 k) = 19.6\) kips

\(M_{u,g–c} = H_{C}\left|\beta -\overline{\beta} \right| = (38.2 k)(3.6 in. – 9 in.) \)

 

= 206.3 in.-kips = 17.2 ft.-kips

The forces on the gusset-to-beam interface are

\(H_{B}=\frac{\alpha}{r}P_{brace} =\frac{15.5 in.}{27.5 in.}(150 k) = 84.5\) kips

\(V_{B}=\frac{0.5d_{b} }{r}P_{brace} =\frac{12.15 in.}{27.5 in.}(150 k) = 66.3\) kips

\(M_{u,g–c}\)= 0 ft.-kips

The available strength of the gusset plate connection is determined using the methods presented in Chapters 9 and 10 for calculating bolt and weld capacities.

For more information and detailed design examples of gusset-plated connections, the reader should refer to Chapter 2 of reference 18.
