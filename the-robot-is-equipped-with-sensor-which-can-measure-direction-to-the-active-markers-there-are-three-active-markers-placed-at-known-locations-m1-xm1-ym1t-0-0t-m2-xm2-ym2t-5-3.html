Guest Question.

The robot is equipped with sensor which can measure direction to the active markers. There are three active markers placed at known locations \(m _{1}=\left[x_{m 1}, \gamma_{m 1}\right]^{T}\) = \([0,0]^{T}\) , \(m _{2}=\left[x_{m 2}, \gamma_{m 2}\right]^{T}\) =\([5,3]^{T}\), and \(m _{3}=\left[x_{m 3}, \gamma_{m 3}\right]^{T}\) = \([1,5]^{T}\) . At a current robot pose q = \([x,y,φ]^{T}\) measured directions are given as a relative angles\(\alpha_{1}\) = −2.7691, \(\alpha_{2}\) = −0.3585, and \(\alpha_{3}\) = 1.4277.
What is the current robot pose q?

There are many possible solutions to solve this triangulation problem,
and here two possibilities are shown. In the first part of the solution, particle
swarm optimization (PSO) is applied. Some more details about PSO can
be found in Section 3.3.9. In the second part of the solution a popular
geometric algorithm based on the intersection of circular arcs is used.

Admin
Solution

The current robot pose q =\([x,y,φ]^{T}\) and measured angles \(\alpha_{j}\)(j = 1, 2, 3)
are related by the relations in Eq. (5.69). The task of the PSO algorithm
is to find unknowns (x, y, and φ) so that the relations in Eq. (5.69) are
valid. Each particle position presents one candidate solution (\(q_{i}\)) and during
optimizations a swarm of particles updates toward more optimal solutions.
The criteria for optimality of the ith particle solution is given as follows:

 

\(J_{i}=f\left(q_{i}\right)=\sum_{j=1}^{3}\left(\alpha_{j}-\hat{\alpha}_{j}\right)^{2}\)

 

\(\begin{aligned}\tan \left(\alpha_{1}+\varphi\right) &=\frac{\gamma_{m 1}-\gamma}{x_{m 1}-x} \\\tan \left(\alpha_{2}+\varphi\right) &=\frac{\gamma_{m 2}-y}{x_{m 2}-x} \\\tan \left(\alpha_{3}+\varphi\right) &=\frac{y_{m 3}-y}{x_{m 3}-x}\end{aligned}\)

 

where ˆαj is the simulated measurement of the ith particle that is obtained from Eq. (5.69):

 

\(\hat{\alpha}_{j}=\arctan \frac{\gamma_{m j}-\gamma}{x_{m j}-x}-\varphi\)

Note that the arctan function must return a correct angle in range (−π, π] (in Matlab function atan2 should be used).
The correct solution is q = \([2,2.5,π/6]^{T}\) , and the solution script is given in Listing 5.6 and the resulting situation is shown in Fig. 5.11.

Among many analytical solutions for the triangulation problem, the most often used one computes the intersection of circular arcs. Here the basic idea is described and only the final analytical solution is used to compute robot pose. The complete derivation of the algorithm can be found in [3]. The algorithm is based on three circles where each circle is defined by three points: any pair of markers \(m_{i}\), \(m_{j}\), \((i, j=1,2,3, i \neq j)\), and robot position x, y (see Fig. 5.12).

The robot position and the two markers are connected by two lines
with the angle between the lines \(\alpha_{i j}=\alpha_{j}-\alpha_{i}\). The centers and radius of
those three circles are

 

\(\begin{aligned}c_{i j} &=\left[\begin{array}{l}x_{a j} \\\gamma_{c i j}\end{array}\right]=\frac{1}{2}\left(m_{i}+m_{j}+\left[\begin{array}{l}\left(\gamma_{m i}-\gamma_{m j} \cot \alpha_{i j}\right) \\\left(x_{m j}-x_{m i} \cot \alpha_{i j}\right)\end{array}\right]\right) \\r_{i j} &=\frac{\left\|m_{i}-m_{j}\right\|}{2 \sin \alpha_{i j}}\end{aligned}\)

Because \(\alpha_{1 3}=\alpha_{1 2}-\alpha_{3 3}\) only two of those angles are independent and the two belonging circles (for \(\alpha_{1 2}\) and \(\alpha_{2 3}\)) are

 

\(\begin{aligned}&\left(x-x_{12}\right)^{2}+\left(y-y_{12}\right)^{2}=r_{12}^{2} \\&\left(x-x_{23}\right)^{2}+\left(y-y_{23}\right)^{2}=r_{23}^{2}\end{aligned}\)

 

The intersection of both circles (Eq. 5.73) is the solution for the robot
position. For more details on how to obtain the analytical solution see [3]. Here, only the final solution is stated. A new temporary coordinate frame is defined so that \(m_{2}\) is in its origin and \(m_{3}\) lies on its x-axis. With these temporary coordinates the solution is easier to find. The rotation matrix from reference to temporary coordinates reads

\(R=\left[\begin{array}{cc}\cos \beta & -\sin \beta \\\sin \beta & \cos \beta\end{array}\right]\)

where \(\beta=\tan \frac{\gamma_{m 3}-\gamma_{m} 2}{x_{m 3}-x_{m 2}}\). Markers expressed in temporary coordinates \(\bar{m}_{i}=\left[\bar{x}_{m i}, \bar{y}_{m i}\right]^{T}\) are obtained by transformation \(\bar{m}_{i}=R^{-1}\left(m_{i}-m_{2}\right)\) The intersection of the circles in temporal coordinates (¯x, ¯y) reads

 

\(\left[\begin{array}{c}\bar{x} \\\bar{y}\end{array}\right]=\bar{x}_{m 3} \frac{1-\eta \cot \alpha_{12}}{1-\eta^{2}}\left[\begin{array}{c}1 \\-\eta\end{array}\right]\)

 

where \(\eta=\frac{\bar{x}_{m, 3}-\bar{x}_{m 1}-\bar{y}_{m 1} \cot \alpha_{12}}{\bar{x}_{m 3} \cot \alpha_{23}-\bar{y}_{m 1}+\bar{x}_{m 1} \cot \alpha_{12}}\). The solution in the reference coordinates is

 

\(\left[\begin{array}{l}x \\y\end{array}\right]=m_{2}+R\left[\begin{array}{l}\bar{x} \\\bar{y}\end{array}\right]\)

and orientation of the robot is

 

\(\varphi=\arctan \frac{Y_{m} 1-\gamma}{x_{m} 1-x}-\alpha_{1}\)

 

A complete algorithm to obtain the solution is given in Matlab Listing 5.7.The solution is shown graphically in Fig. 5.13.
