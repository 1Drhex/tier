Guest Question.

Designing a BJT inverter Design a BJT inverter, as shown in Fig. 15.35, to drive five identical inverters (N = 5) and to give \( V_{ OH }=3.5 V \text { and } NM _{ L }=0.4 V \). The transistor has \( I_{ C (\max )}=5 mA , \beta_{ F }=100 \text { to } 150 \), \( V_{ BE (\text { cut-in })}=0.6 V , V_{ BE ( sat )}=0.8 V , t_{ d }=1 ns , \text { and } t_{ s }=2 ns . \text { The supply voltage } V_{ CC } \text { is } 5 V \).

(a) Find the values of \(R_{ C } \text { and } R_{ B } \).
(b) Calculate \( k_{\text {ODF }} \text { and } NM _{ H } \).
(c) Find the maximum fan-out N for \( V_{ IH }=2.0 V \text { and } k_{ ODF }=1 \).
(d) Calculate the propagation delay \( t_{ pd } \). Assume that each load is an NMOS and can be represented by a capacitance \( C_{ B }=2 pF \) in series with resistance \( R_{ B }=1.8 k \Omega \).
(e) Calculate the delay-power product (DP) at a frequency \( f_{ clk }=5 MHz \).

Admin
Solution

(a) \( N=5, V_{ OH }=3.5 V , V_{ IL }=V_{ BE ( cut – in )}=0.6 V \), and

\( V_{ OL }=V_{ IL }- NM _{ L }=0.6-0.4=0.2 V \)

For \( V_{ CE (\text { sat })}=V_{ OL }=0.2 V \text { (for } V_{ T }=25.8 mV \text { and } \beta_{ F }=100 \text { ), Eq. (15.78) gives } \beta_{ F (\text { forced })}=71.8 \). Using Eq. (15.77), we have

\( I_{ C ( sat )}=\frac{V_{ CC }-V_{ CE ( sat )}}{R_{ C }} \) (15.77)

 

\( V_{ CE \text { (sat) }}=V_{ T } \ln \left(\frac{10+9 I_{ C (\text { sat })} / I_{ B \text { (sat) }}}{1-I_{ C (\text { sat })} / \beta_{ F } I_{ B (\text { sat })}}\right)=V_{ T } \ln \left(\frac{10+9 \beta_{ F (\text { forced })}}{1-\beta_{ F \text { (forced) }} / \beta_{ F }}\right) \) (15.78)

 

\( R_{ C } \geq \frac{\left(V_{ CC }-V_{ CE ( max )}\right)}{I_{ C ( sat )}}=\frac{(5-0.2) V }{5 mA }=960 \Omega \)

We choose \( R_{ C }=1 k \Omega \). When the input is low, \( Q _{1} \text { is off and } v_{ O }=V_{ OH } \). The base current \( I_{ B } \) of each load stage is

\( I_{ B }=\frac{V_{ OH }-V_{ BE ( sat )}}{R_{ B }} \)

 

and \( V_{ OH }=V_{ CC }-N I_{ B } R_{ C } \text {. Eliminating } I_{ B } \text { and solving for } V_{ OH } \), we get

\( V_{ OH }=\frac{V_{ CC }+N R_{ C } V_{ BE ( sat )} / R_{ B }}{1+N R_{ C } / R_{ B }}=\frac{R_{ B } V_{ CC }+N R_{ C } V_{ BE ( sat )}}{R_{ B }+N R_{ C }} \) (15.79)

 

which, for \( V_{ OH }=3.5 V , N=5 \text {, and } V_{ BE \text { (sat) }}=0.8 V \text {, gives } R_{ B }=9 R_{ C }=9 k \Omega \).

(b) We have

\(\begin{aligned}&I_{ B }=\frac{I_{ C (\text { sat })}}{\beta_{ F (\text { forced })}}=\frac{5 mA }{71.8}=70 \mu A \\&I_{ B (\text { sat })}=\frac{I_{ C (\text { sat })}}{\beta_{ F }}=\frac{5 mA }{100}=50 \mu A \\&k_{ ODF }=\frac{I_{ B }}{I_{ B (\text { sat })}}=\frac{70}{50}=1.40\end{aligned}\)

 

\(V_{ IH }=R_{ B } I_{ B }+V_{ BE ( sat )}\) (15.80)

 

\( =9 k \Omega \times 70 \mu A +0.8 V =1.43 V \)

 

Thus, \( NM _{ H }=V_{ OH }-V_{ IH }=3.5-1.43=2.07 V \). This noise margin is a measure of the safety factor that allows the load transistors to remain saturated despite changes in supply voltage, temperature, and manufacturing tolerances.

(c) When \( Q _{1} \) is off, the output voltage \( v_{ O } \) becomes

\( v_{ O }=V_{ CC }-N I_{ B } R_{ C }=V_{ CC }-N k_{ ODF } I_{ B ( sat )} R_{ C }=V_{ CC }-N \frac{k_{ ODF }\left(V_{ CC }-V_{ CE ( sat )}\right)}{\beta_{ F }} \)

For regeneration of logic levels at the load gates, \( v_{ O } \geq V_{ IH } \), and we find the fan-out N as

\( N \leq \frac{\beta_{ F }\left(V_{ CC }-V_{ IH }\right)}{k_{ ODF }\left(V_{ CC }-V_{ CE ( sat )}\right)} \) (15.81)

\( \leq 100 \times \frac{5-1.4}{1.39 \times(5-0.2)}=53.4 \)

Hence, the maximum fan-out is N = 53.

(d) When \( Q _{1} \) turns off, each load is represented by a capacitance \( C_{ B } \text { and a resistance } R_{ B } \). The equivalent circuit is shown in Fig. 15.36(a). For identical loads, all branches are effectively in parallel as far as the output node is concerned, and Fig. 15.36(a) can be simplified to Fig. 15.36(b). That is, the time constant for low to high can be found from

\(\begin{aligned}\tau_{r 1} & \approx N C_{ B }\left(R_{ C }+\frac{R_{ B }}{N}\right) \\&=5 \times 2 pF \times\left(1 k \Omega+\frac{9 k \Omega}{5}\right)=28 ns\end{aligned}\) (15.82)

 

The output voltage vO during charging can be found from

\( v_{ O }=V_{ CC }-\frac{R_{ C } V_{ CC }}{R_{ C }+R_{ B } / N} e^{-t / \tau_{ r 1}} \)

from which we can find the time required for \( v_{ O } \text { to rise from } V_{ OL }=0.2 V \text { to } V_{ O ( mid )} \), which is equal to \( \left(V_{ OH }+V_{ OL }\right) / 2=1.85 V \); that is,

\(\begin{aligned}t_{ r 1} &=t_{2}\left(\text { for } v_{ O }=V_{ O ( mid )}\right)-t_{1}\left(\text { for } v_{ O }=V_{ OL }\right)=\tau_{ r 1} \ln \frac{V_{ CC }-V_{ OL }}{V_{ CC }-V_{ O ( mid )}} \\&=28 ns \times \ln \left[\frac{(5-0.2)}{(5-1.85)}\right]=11.79 ns\end{aligned}\) (15.83)

 

Thus, \( t_{ pLH }=t_{ s }+t_{ r 1}=2+11.79=13.79 ns \).
When \( Q_{1} \) is turned on to saturation, each capacitor discharges through the transistor. The equivalent circuit is shown in Fig. 15.36(c). The time constant for high to low can be found from

\(\begin{aligned}\tau_{ fl } & \approx C_{ B } R_{ B } \\&=2 pF \times 9 k \Omega=18 ns\end{aligned}\) (15.84)

The output voltage \( v_{ O } \) during discharging can be found from

\( v_{ O }=V_{ OH } e^{-t / \tau_{ n }} \)

from which we can find the time for \( v_{ O } \text { to fall from } V_{ OH }=3.5 V \text { to }\left( V _{ OH }+V_{ OL }\right) / 2=1.85 V \); that is,

\(\begin{aligned}t_{ f 1} &=\tau_{ fl } \ln \left(\frac{2 V_{ OH }}{V_{ OH }+V_{ OL }}\right) \\&=18 ns \times \ln \left(2 \times \frac{3.5}{3.5+0.2}\right)=11.48 ns\end{aligned}\) (15.85)

Thus, \( t_{ pHL }=t_{ d }+t_{ fl }=1+11.48=12.48 ns \). The propagation time is

\( t_{ pd }=\frac{t_{ pHL }+t_{ pLH }}{2}=13.135 ns \)

(e) The total power loss in the two junctions is found as follows:

\(\begin{aligned}&P_{\text {static }}=V_{ CE ( sat )} I_{ C ( sat )}+V_{ BE ( sat )} I_{ B } \\&\quad=0.2 \times 5 mA +0.8 \times 70 \mu A =1.056 mW \\&P_{\text {dynamic }}=N f_{ clk } C_{ B } V_{ OH }^{2}=5 \times 5 MHz \times 2 pF \times 3.5^{2}=0.61 mW \\&P_{ D }=P_{\text {static }}+P_{\text {dynamic }}=1.67 mW\end{aligned}\)

Therefore, the delay-power product is

\( DP =P_{ D } \times t_{ pd }=1.67 mW \times 13.135 ns =21.9 pJ \)
