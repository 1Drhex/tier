Guest Question.

A velocity field is defined as \( u = 2(x^2 + y^2) ft/s, v = (-4xy) ft/s.\) Determine the stream function and the circulation around the rectangle shown. Plot the streamlines for \( \psi _0 = 0, \psi _1 = 1 ft^2/s, and \psi _2 = 2 ft^2/s. \)

Admin
Solution

We consider ideal fluid flow.
Since the continuity equation \(\frac{\partial u}{\partial x} = \frac{\partial v}{\partial y} = 4x + (-4x) = 0 \)is satisfied, the stream function can be established. Using the definition of the velocity components, with respect to stream function,

\(\frac{\partial \psi }{\partial y} = u; \frac{\partial \psi }{\partial y} = 2(x^2 + y^2)\)

Integrating this equation with respect to y,

\(\psi = 2(x^2 y + \frac{1}{3}y^3 ) + f(x) \)(1)
Also,

\(– \frac{\partial \psi }{\partial x} = v;\)

 

\(– \frac{\partial \psi }{\partial x} = -4xy\)

 

\(– \frac{\partial}{\partial x} [2(x^2y + \frac{1}{3}y^3 ) + f(x)] = -4xy\)

 

\(4xy + \frac{\partial}{\partial x} [f(x)] = 4xy\)

 

\(\frac{\partial}{\partial x} [f(x)] = 0\)

 

Integrating this equation with respect to x,

f(x) = C

Substituting this result into Eq. (1),

\(\psi = 2(x^2y + \frac{1}{3}y^3 ) + C\)

C is an arbitary constant. If we set it equal to zero then the stream function can be expressed as

\(\psi = 2(x^2y + \frac{1}{3}y^3 ) \)

For\( \psi =0, 0 = 2y(x^2 + \frac{y^2 }{3})\) since \( x^2 + \frac{y^2}{3} \neq 0 , \) then

y=0

For \(\psi = 1 ft^2/s,\)

\(1= 2y(x^2 + \frac{y^2 }{3})\)

 

\(x^2 = \frac{3 – 2y^3}{6y} 0 \lt y \lt 1.145\)

 

For\( \psi = 2 ft^/s,\)

\(2= 2y(x^2 + \frac{y^2 }{3})\)

 

\(x^2 = \frac{6- 2y^3}{6y} 0 \lt y \lt 1.442 \)
