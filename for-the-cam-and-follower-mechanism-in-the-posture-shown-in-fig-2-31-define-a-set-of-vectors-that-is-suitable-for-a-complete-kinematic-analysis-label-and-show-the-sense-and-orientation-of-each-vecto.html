Guest Question.

For the cam-and-follower mechanism in the posture shown in Fig. 2.31, define a set of vectors that is suitable for a complete kinematic analysis. Label and show the sense and orientation of each vector on the mechanism. Write the vector loop equation(s) and identify suitable input(s) for the mechanism. Identify the known quantities, the unknown variables, and any constraints. If you identify constraints, then write the constraint equation(s).

Admin
Solution

Since the circular cam, link 2, and the roller follower, link 3, are in rolling contact, let us consider them linked together by a (fictitious) arm, link 23. This leads us to consider the set of vectors shown in Fig. 2.32.

The vector loop equation for the mechanism can be written as

\(\overset{\sqrt{}I}{R_{2}} + \overset{\sqrt{}? }{R_{23}} – \overset{\sqrt{}? }{R_{4}} – \overset{\sqrt{}\sqrt{} }{R_{12}} – \overset{\sqrt{}\sqrt{}}{R_{11}} = 0.\) (1)

This is a valid loop-closure equation and, given the angle θ2 as input, it can be solved for the postures of link 4 and of the fictitious link 23. However, this does not yet allow a solution for the rotation of the roller, link 3.

Since there is rolling contact between cam 2 and roller 3 at point C, let us consider the apparent angular displacements of links 2 and 3 as they would appear to an observer riding on the fictitious arm, link 23. Such an observer would observe

ρ3Δθ3/23 = −ρ2Δθ2/23, (2)

which gives the following relationship between the absolute angular displacements:

ρ3(Δθ3 − Δθ23) = −ρ2(Δθ2 − Δθ23). (3)

Rearranging this equation gives

ρ3Δθ3 = (ρ2 +ρ3)Δθ23 −ρ2Δθ2. (4)

Since the angular displacement of link 2 is known (the input) and Δθ23 is determined from Eq. (1), the angular displacement, Δθ3, can be determined from Eq. (4). It is worth noting that we can obtain a solution for the posture of link 3 even though no vector is attached to or rotates with link 3.
