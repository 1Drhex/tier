Solve with variable specific heats using Table F.5.
In a Brayton cycle the inlet is at 540 R, 14 psia and the combustion adds 350 Btu/lbm. The maximum temperature is 2500 R due to material considerations. Find the maximum permissible compression ratio and for that the cycle efficiency using cold air properties.

Admin
Solution

Combustion: \(h _{3}= h _{2}+ q _{ H } ; \quad{ }_{2} w _{3}=0 \text { and } T _{\max }= T _{3}=2500 R\)

\(h _{2}= h _{3}- q _{ H }=645.721-350=295.72 Btu / lbm\)

From Table F.5 find \(T _{2} \text { from } h _{2}\)

\(T _{2}=1216.8 R ; s _{ T 2}^{ o }=1.83883 ; T _{1}=540 R ; s _{ T 1}^{ o }=1.63979 Btu / lbm – R\)

Reversible adiabatic compression leads to constant s, from Eq.6.19:

\(P _{2} / P _{1}=\exp \left[\left( s _{ T 2}^{ o }- s _{ T 1}^{ o }\right) / R \right]=\exp (2.90313)= 1 8 . 2 3 1\)

Reversible adiabatic expansion leads to constant s, from Eq.6.19

\(s _{ T 4}^{ o }= s _{ T 3}^{ o }+ R \ln \left( P _{4} / P _{3}\right)=2.03391+\frac{53.34}{778} \ln \frac{1}{18.231}=1.83487 Btu / lbm – R\)

From Table F.5.1 by linear interpolation \(T _{4}=1197.9 R , h _{4}=290.905 Btu / lbm\)

\(\begin{aligned}& w _{ T }= h _{3}- h _{4}=645.721-290.905=354.82 Btu / lbm \\& w _{ C }= h _{2}- h _{1}=295.72-129.18=166.54 Btu / lbm \\& w _{\text {net }}= w _{ T }- w _{ C }=354.82-166.54= 1 8 8 . 2 8 Btu / lbm \\&\eta= w _{\text {net }} / q _{ H }=188.28 / 350= 0 . 5 3 8\end{aligned}\)
