Guest Question.

A series \(RLC\) circuit has \(R = 2 Ω\), \(L = 1 H\) and \(C= 1 F\). The capacitor is initially charged to \(2 V\) and the initial current in the inductor is \(2 A\) at \(t = 0^{-}\). Find the zero-input response of capacitor voltage and circuit current.

Admin
Solution

The differential equation governing the capacitor voltage \(v_{C}(t)\) is

\(\frac{d^{2} v_{C}}{d t^{2}}+2 \frac{d v_{C}}{d t}+v_{C}=0 \text { for } t \geq 0^{+}\)

 

The characteristic equation is \(\gamma^{2}+2 \gamma+1=0\) and its roots \(\alpha_{1,2}=-1 \mathrm{~s}^{-1} \text {and }-1 \mathrm{~s}^{-1}\) The trial solution to be attempted is of the form \(v_{C}(t)=A_{1} e^{-t}+A_{2} t e^{-t} \mathrm{~V}\) Applying initial conditions at \(t = 0^{+}\), we get two equations in \(A_{1}\) and \(A_{2}.\)

\(v_{C}\left(0^{+}\right)=A_{1}=2 \mathrm{~V} ;\left.\frac{d v_{C}}{d t}\right|_{\left(0^{+}\right)}=\frac{I_{o}}{C}=2 \mathrm{~V} / \mathrm{s}=\left.\left[-A_{1} e^{-t}-A_{2} t e^{-t}+A_{2} e^{-t}\right]\right|_{\left(0^{+}\right)}=-A_{1}+A_{2}\)

 

\(\begin{aligned} &\therefore A_{1}=2 \mathrm{~V} \text { and } A_{2}=4 \mathrm{~V} \mathrm{~s}^{-1}\\ &\therefore v_{C}(t)=2 e^{-t}+4 t e^{-t} \mathrm{~V} \text { for } t \geq 0^{+} \text {and } i(t)=C \frac{d v_{C}}{d t}=-2 e^{-t}-4 t e^{-t}+4 e^{-t}=2 e^{-t}-4 t e^{-t} \mathrm{~A} \text { for } t \geq 0\\ &\begin{array}{l} v_{L}(t)=L \frac{d i}{d t}=1 \frac{d i}{d t}=-2 e^{-t}+4 t e^{-t}-4 e^{-t}=-6 e^{-t}+4 t e^{-t} \mathrm{~V} \text { for } t \geq 0^{+} \\ v_{R}(t)=R i=2 i=4 e^{-t}-8 t e^{-t} \mathrm{~V} \text { for } t \geq 0^{+} \end{array} \end{aligned}\)

 

These waveforms are shown in Fig. 12.1-3.

case-3 \(α_{1}\), and \(α_{2}\) complex and conjugates with negative real parts

Now we come up against the most interesting case of all. This occurs when \(R<2 \sqrt{L / C}.\)

The quantity \(\left(\frac{R^{2}}{4 L^{2}}-\frac{1}{L C}\right)\) is negative under this condition and the roots of characteristic equation become

 

\(\alpha_{1,2}=-\frac{R}{2 L} \pm j \sqrt{\frac{1}{L C}-\frac{R^{2}}{4 L^{2}}}\)

 

We define three new symbols – mainly for convenience at present. But they will turn out to be important parameters in circuit studies soon. They are defined as in Eqn. 12.1-6.

\(\text { Let } \xi=\frac{R}{2 \sqrt{L / C}}, \omega_{n}=\frac{1}{\sqrt{L C}} \text { and } \omega_{d}=\sqrt{\frac{1}{L C}-\frac{R^{2}}{4 L^{2}}}=\sqrt{1-\xi^{2}} \omega_{n} (12-6-1)\)

 

The roots of characteristic equation can be expressed in terms of these new symbols as \(\alpha_{1}=-\xi \omega_{n}+j \sqrt{1-\xi^{2}} \omega_{n} \text { and } \alpha_{2}=-\xi \omega_{n}-j \sqrt{1-\xi^{2}} \omega_{n}\)

\(\begin{array}{l} v_{C}(t)=V_{o} \frac{\alpha_{2} e^{\alpha_{1} t}-\alpha_{1} e^{\alpha_{2} t}}{\alpha_{2}-\alpha_{1}}-\frac{I_{o}}{C} \frac{e^{\alpha_{1} t}-e^{\alpha_{2} t}}{\alpha_{2}-\alpha_{1}} \mathrm{~V} \text { for } t \geq 0^{+} \\ \alpha_{1,2}=-\xi \omega_{n} \pm j \sqrt{1-\xi^{2}} \omega_{n}=-\xi \omega_{n} \pm j \omega_{d} \Rightarrow \alpha_{2}-\alpha_{1}=-2 j \omega_{d} \end{array}\)

 

\(\frac{\text { Simplifying } \frac{\alpha_{2} e^{\alpha_{1} t}}{\alpha_{2}-\alpha_{1}}}{=\frac{-1}{2 j \sqrt{1-\xi^{2} \omega_{n}}}\left[\left(-\xi \omega_{n}-j \omega_{d}\right) e^{\left(-\xi \omega_{n}+j \omega_{d}\right) t}-\left(-\xi \omega_{n}+j \omega_{d}\right) e^{\left(-\xi \omega_{n}-j \omega_{d}\right) t}\right]}\)

 

Now we use Eqn. 12.1-4 for the zero-input response for \(v_{C}(t)\) and substitute these expressions for \(\alpha_{1}\) and \(\alpha_{2}\) and employ Euler’s formula for algebraic simplification of the resulting expression.

\(\begin{aligned} &=\frac{-e^{-\xi \omega_{n} t}}{2 j \sqrt{1-\xi^{2}} \omega_{n}}\left[\left(-\xi \omega_{n}-j \omega_{d}\right) e^{\left(j \omega_{d}\right) t}-\left(-\xi \omega_{n}+j \omega_{d}\right) e^{\left(-j \omega_{d}\right) t}\right]\\ &=\frac{-e^{-\xi \omega_{n} t}}{2 j \sqrt{1-\xi^{2}} \omega_{n}}\left[-\xi \omega_{n}\left(e^{\left(j \omega_{d}\right) t}-e^{\left(-j \omega_{d}\right) t}\right)-j \omega_{d}\left(e^{\left(j \omega_{d}\right) t}+e^{\left(-j \omega_{d}\right) t}\right)\right]\\ &=\frac{-e^{-\xi \omega_{n} t}}{2 j \sqrt{1-\xi^{2}} \omega_{n}}\left[-\xi \omega_{n}\left(2 j \sin \omega_{d} t\right)-j \omega_{d}\left(2 \cos \omega_{d} t\right)\right] \text { (by Euler's formula) }\\ &=e^{-\xi \omega_{n} t}\left[\frac{\xi}{\sqrt{1-\xi^{2}}} \sin \omega_{d} t+\cos \omega_{d} t\right]\left(\because \omega_{d}=\sqrt{1-\xi^{2}} \omega_{n}\right) \end{aligned}\)

 

\(\frac{\text { Similarly simplifying } \frac{e^{\alpha_{1} t}-e^{\alpha_{2} t}}{\alpha_{2}-\alpha_{1}}}{\frac{e^{\alpha_{1} t}-e^{\alpha_{2} t}}{\alpha_{2}-\alpha_{1}}=-\frac{e^{-\xi \omega_{n} t}}{\sqrt{1-\xi^{2}} \omega_{n}} \sin \omega_{d} t}\)

 

\(\therefore v_{C}(t)=V_{o} e^{-\xi \omega_{n} t}\left[\frac{\xi}{\sqrt{1-\xi^{2}}} \sin \omega_{d} t+\cos \omega_{d} t\right]+\frac{I_{o}}{C} \frac{e^{-\xi \omega_{n} t}}{\sqrt{1-\xi^{2}} \omega_{n}} \sin \omega_{d}\)

 

Substituting \(\omega_{n}=\frac{1}{\sqrt{L C}}\) in the second term ,

\(v_{C}(t)=V_{o} e^{-\xi \omega_{n} t}\left[\frac{\xi}{\sqrt{1-\xi^{2}}} \sin \omega_{d} t+\cos \omega_{d} t\right]+I_{o} \sqrt{\frac{L}{C}} e^{-\xi \omega_{n} t} \frac{1}{\sqrt{1-\xi^{2}}} \sin \omega_{d} t \mathrm{~V}\)

 

Therefore, the zero-input response for capacitor voltage in this case with \(0 \leq \xi<1\) is given by the following expression in terms of \(\xi\) and \(\omega_{n}\)

\(\begin{aligned} \quad v_{C}(t) &=e^{-\xi \omega_{n} t}\left[\left(\frac{V_{o} \xi+I_{o} \sqrt{\frac{L}{C}}}{\sqrt{1-\xi^{2}}}\right) \sin \sqrt{1-\xi^{2}} \omega_{n} t+V_{o} \cos \sqrt{1-\xi^{2}} \omega_{n} t\right] \mathrm{V} \text { for } t \geq 0^{+} \\ \text {where } \omega_{n} &=1 / \sqrt{L C}, \xi=\frac{R}{2 \sqrt{L / C}} \end{aligned}(12-1-7)\)

 

That, indeed, looks slightly complicated. We will take it up through special cases soon. But before that, here is how it looks for a series \(RLC\) circuit with \(L = 1 H\), \(C = 1 F\), \(R = 0.5 W\), \(V_{o} = 2 V\) and \(I_{o} = 2 A.\)

The value of \(\xi \text { is } 0.25 \text { and } \omega_{n} \text { is } 1 \text { . }\)

 

The zero-input response has an exponentially damped sinusoidal shape. All the circuit variables oscillate with the amplitude of oscillation decreasing with time. An exponential function governs the decrease in amplitude with time. The numbers \(\xi\) and \(\omega_{n}\) decide the decay rate of this exponential function. They also decide the time-interval between successive zero-crossings of circuit variables.
