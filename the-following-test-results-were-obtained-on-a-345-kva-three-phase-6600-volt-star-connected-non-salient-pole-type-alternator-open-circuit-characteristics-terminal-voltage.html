Guest Question.

The following test results were obtained on a 345 kVA , three-phase, 6600 volt, star connected non salient pole type alternator. Open circuit characteristics.

\(\begin{array}{lllllll}\text { Terminal voltage in volt } & 1400 & 2100 & 5600 & 6600 & 7240 & 8100 \\\text { Exciting current in ampere } & 20 & 30 & 46.5 & 58 & 76.5 & 96\end{array}\)

Short circuit characteristics : Stator current 35 A with an exciting current of 50 A . Leakage reactance on full load is \( 8 \%\) Neglect armature resistance. Calculate the exciting current (for f u l load ) at p.f. 0-8 lagging and at unity.

 

 

Admin
Solution

Reducing all line voltages to phase voltage, for a star connected machine.

 

\( \frac{1400}{\sqrt{3}}, \frac{2100}{\sqrt{3}}, \frac{5600}{\sqrt{3}}, \frac{6600}{\sqrt{3}}, \frac{7240}{\sqrt{3}}, \frac{8100}{\sqrt{3}}=808,1212,3233,3810,4180,4676 \)

 

Plot the O C C as shown in Fig. 6.58

 

Plot the S C C from the given data. It is a straight line as shown in Fig. 6.58..

 

Leakage reactance drop \( =8 \%=\frac{8}{100} \times \frac{6600}{\sqrt{3}}=304.8\) volt

 

Terminal voltage (phase value), \( V=\frac{6600}{\sqrt{3}}=3810 V\) is taken as reference vector. Since armature

 

resistance is neglected, there is only leakage reactance drop.

 

\( \therefore\)

\( I X_{S}=304.8\) volt (leads the current vector by \( 90^{\circ}\)

 

Full-load current, \( \quad I=\frac{k V A \times 1000}{\sqrt{3} V_{L}}=\frac{345 \times 1000}{\sqrt{3} \times 6600}=30 A\)

 

 

For pf 0.8 lagging

 

For pf 0.8 lagging, current logs behind terminal voltage V by an angle \( \phi=\cos ^{-1} 0.8=36.87^{\circ} \) , as shown in vector diagram

 

\(\begin{aligned}E &=\sqrt{(V \cos \phi+I R)^{2}+\left(V \sin \phi+IX_{S}\right)^{2}} \\&=\sqrt{(3810 \times 0.8+0)^{2}+(3890 \times0.6+304.8)^{2}}=4000 V\end{aligned}\)

 

 

\( \text { Phase difference between } E \text { and } V \text { is } \delta=\sin ^{-1} \frac{I X_{S} \cos \phi}{E}=\sin ^{-1} \frac{304.8 \times 0.8}{4000}=3.5 "\)

 

 

From O C C graph, corresponding to 4000 volt, the field current is 62 ampere. This excitation \(\left(I_{f 1}\right)\) is called normal excitation which is drawn at right angles to E . This is represented by vector oa. From short-circuit characteristic, corresponding to full load current of 30 A, the exciting current is 42.85 A. This excitation \(\left(I_{f 2}\right)\) is in phase opposition with the current I and represents the field excitation of armature reaction and leakage reactance drop. The vector (a b) is drawn in parallel to current vector in phase opposition to balance the effect of armature reaction. Now, the resultant of these two vectors is given by o b whose magnitude \(\left(I_{f}\right)\) is calculated as follows.

 

\(\begin{aligned}o b^{2} &=o a^{2}+a b^{2}-2 o a \times a b \times \cos (90+\phi+\delta) \\&=(62)^{2}+(42.85)^{2}-2 \times 62 \times 42.85 \times \cos \left(90+36.87^{\circ}+3.5^{\circ}\right) \\&=3844+1800+2 \times 62 \times 42.85 \times \cos \left(49.63^{\circ}\right)\end{aligned}\)

 

 

\( o b=\sqrt{3844+1800+2 \times 62 \times 42.85 \times 0.64}=95 A\)

 

 

If, it is required to find the regulation, then corresponding to this field excitation the open circuit voltage \(E_{o}\) can be read off from the O C C curve. This voltage \(E_{o}\) always lags the excitation o b by \(90^{\circ}\) as shown in Fig. 6.59.

 

 

For Unity p.f.

 

For unity p f, the current I is in phase with the terminal voltage V and the leakage reactance drop \( I X_{S}\) equal to 304.8 volt is drawn at right angle to the current I or at right angles to voltage vector V. as shown in Fig. 6.60.

 

\(\text { Hence from the vector diagram opt, we have }\)

 

\(E=\sqrt{V^{2}+\left(I X_{S}\right)^{2}}=\sqrt{(3810)^{2}+(304.8)^{2}}=3822 V\)

 

\(\text { Phase difference between } E \text { and } V \text { is say } \delta^{\prime}\)

 

\(\delta^{\prime}=\sin ^{-1} \frac{I X_{S}}{E}=\sin ^{-1} \frac{304.8}{3822}=4.58^{\circ}\)

 

\(\text { Corresponding to } 3822 V \text { , the field excitation from } O C C \text { curve is } 59.8 A\)

 

\(\text { Full load current at unity p.f. }\)

 

\(I=\frac{345 \times 1000}{\sqrt{3} \times 6600}=30 A\)

 

\(\text { From short circuit characteristic } 30 \text { A corresponds to exciting current of } 42.85 A\)

 

Now \(oa^{\prime}\). is drawn at right angles to E and is equal to 59.8 A. Now \( a^{\prime} b^{\prime}\) equal to 42.85 A is drawn parallel to current vector. The vector \( o b^{\prime}\) is the vector sum of \( o a^{\prime}\) and \( a^{\prime} b^{\prime}\).

 

\(\begin{aligned}o b^{\prime} &=\sqrt{o a^{2}+a^{\prime} b^{\prime 2}-2 o a^{\prime} \times a^{\prime} b^{\prime} \times \cos (90+\theta)} \\&=\sqrt{(59.8)^{2}+(42.85)^{2}+2 \times 59.8 \times 42.85 \times \cos 85.44^{\circ}} \\&=76.25 A \end{aligned}\)

 

Again if it is required to find the regulation then corresponding to this field current of 76.25 A the open-circuit voltage \( E_{o}\) can be determined which always lags the excitation \( o b^{\prime}\) by \( 90^{\circ} . \)

 

\(\therefore \quad \% \text { age regulation }=\frac{E_{o}-V}{V} \times 100\)
