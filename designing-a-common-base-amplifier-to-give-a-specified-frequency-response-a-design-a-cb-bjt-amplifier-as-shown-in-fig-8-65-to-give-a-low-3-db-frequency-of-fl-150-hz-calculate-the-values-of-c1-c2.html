Guest Question.

Designing a common-base amplifier to give a specified frequency response

(a) Design a CB BJT amplifier as shown in Fig. 8.65 to give a low 3-dB frequency of \(f_{ L }=150 Hz\). Calculate the values of \(C_{1}, C_{2} \text {, and } C_{ B }\). The circuit parameters are \(C_{\pi}=15 pF , C_{\mu}=1 pF , g_{ m }=57.14 m \mho, \beta_{ f }=80\), \(R_{ s }=200 \Omega, r_{\pi}=1.4 k \Omega, R_{ E }=330 \Omega, R_{ C }=5 k \Omega, R_{1}=7 k \Omega, R_{2}=4.3 k \Omega, \text { and } R_{ L }=5 k \Omega\).

(b) Calculate the high 3-dB frequency \(f_{ H }\).

Admin
Solution
\(R_{ B }=R_{1}\left\|R_{2}=7 k \Omega\right\| 4.3 k \Omega=2.66 k \Omega\)

(a) The design steps are as follows:

Step 1. Calculate the equivalent resistances \(R_{ Cl }, R_{ C 2}, \text { and } R_{ CB }\).
From Eq. (8.189),

\(R_{ i }=R_{ E }\left\|r_{\pi}\right\| R_{ t }=R_{ E }\left\|r_{\pi}\right\| \frac{r_{\pi}}{\beta_{ f }}=R_{ E } \| \frac{r_{\pi}}{1+\beta_{ f }}\) (8.189)

\(R_{ i }=330 \Omega \|\left[\frac{1.4 k \Omega}{1+80}\right]=16.4 \Omega\)

 

and \(R_{ C 1}=R_{ s }+R_{ i }=200+16.4=216.4 \Omega\)

From Eq. (8.191),

\(R_{ C 2}=R_{ C }+R_{ L }\) (8.191)

 

\(R_{ C 2}=R_{ C }+R_{ L }=5 k \Omega+5 k \Omega=10 k \Omega\)

 

From Eq. (8.193),

\(R_{ CB }=R_{ B } \|\left[r_{\pi}+\left(1+\beta_{ f }\right)\left(R_{ s } \| R_{ E }\right)\right]\) (8.193)

 

\(R_{ CB }=2.66 k \Omega \|[1.4 k \Omega+(1+80)(200 \Omega \| 330 \Omega)]=2.16 k \Omega\)

 

Step 2. At the lowest resistance, the dominant cutoff frequency is \(f_{ Cl }=f_{ L }=150 Hz\).

Step 3. Calculate the required value of \(C_{1}\) from Eq. (8.190):

\(f_{ C 1}=\frac{1}{2 \pi R_{ C 1} C_{1}}\) (8.190)

 

\(f_{ C 1}=\frac{1}{2 \pi R_{ C 1} C_{1}}=\frac{1}{2 \pi \times 216.4 \times C_{1}}=150 Hz \text { or } C_{1}=4.9 \mu F\)

 

Step 4. At the next higher resistance \(R_{ CB }=2.16 k \Omega, f_{ CB }=f_{ L } / 10=150 / 10=15 Hz.\)

 

Step 5. Calculate the required value of \( C_{ B } \) from Eq. (8.194):

\(f_{ CB }=\frac{1}{2 \pi R_{ CB } C_{ B }}\) (8.194)

 

\(f_{ CB }=\frac{1}{2 \pi R_{ CB } C_{ B }}=\frac{1}{2 \pi \times 2.16 k \Omega \times C_{ B }}=15 Hz \text { or } C_{ B }=4.91 \mu F\)

 

Step 6. At the highest resistance \( R_{ C 2}=10 k \Omega, f_{ C 3}=f_{ L } / 20=150 / 20=7.5 Hz .\)

Step 7. Calculate the required value of \( C_{2} \) from Eq. (8.192):

\(f_{ C 2}=\frac{1}{2 \pi R_{ C 2} C_{2}}\) (8.192)

 

\(f_{ C 2}=\frac{1}{2 \pi R_{ C 2} C_{2}}=\frac{1}{2 \pi \times 10 k \Omega \times C_{2}}=7.5 Hz \text { or } C_{2}=2.12 \mu F\)

 

(b) From Eq. (8.195),

\(R_{ C \mu}=R_{ C } \| R_{ L }\) (8.195)

 

\(R_{ C \mu}=R_{ C }\left\|R_{ L }=5 k \Omega\right\| 5 k \Omega=2.5 k \Omega\)

 

From Eq. (8.196),

\(R_{ C \pi}=\left(R_{ S } \| R_{ E }\right)\left\|r_{\pi}\right\| \frac{1}{g_{ m }}\) (8.196)

 

\(R_{ C \pi}=(200 \Omega \| 330 \Omega)\|1.4 k \Omega\| \frac{1}{57.14 m \mho}=15.2 \Omega\)

 

From Eq. (8.197), the high 3-dB frequency is

\(f_{ H }=\frac{1}{2 \pi\left(R_{ C \pi} C_{\pi}+R_{ C \mu} C_{\mu}\right)}\) (8.197)

 

\( f_{ H }=\frac{1}{2 \pi\left(R_{ C \pi} C_{\pi}+R_{ C \mu} C_{\mu}\right)}=\frac{1}{2 \pi(15.2 \Omega \times 15 pF +2.5 k \Omega \times 1 pF )}=58.34 MHz\)
