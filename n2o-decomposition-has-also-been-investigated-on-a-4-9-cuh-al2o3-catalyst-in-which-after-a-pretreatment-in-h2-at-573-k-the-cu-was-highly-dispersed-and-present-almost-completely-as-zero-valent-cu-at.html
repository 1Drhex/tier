Guest Question.

\(N_{2}O\) decomposition has also been investigated on a 4.9% \(Cu/η - Al_{2}O_{3}\) catalyst in which, after a pretreatment in \(H_{2}\) at 573 K, the Cu was highly dispersed and present almost completely as zero-valent Cu at the surface [56]. These small Cu crystallites exhibited significant deactivation, which was attributed to strong oxygen adsorption at the Cu surface, except at high temperatures above 823 K. Additional studies indeed showed that rapid \(O_{2}\) desorption required much higher temperatures than with the Cu/ZSM-5 catalysts; therefore, \(O_{2}\) adsorption/desorption was assumed to be reversible, rather than quasi-equilibrated, and no RDS existed, as shown in the sequence of steps proposed below, where * represents an active site:

(1) \(N_{2}O+ * \xrightleftharpoons[]{K_{N_{2}O}} N_{2}O *\)

 

(2) \(N_{2}O* \xrightarrow[]{K} N_{2} + O*\)

 

(3) \(2 O* \xrightleftharpoons[K_{-1}]{K_{1}} O_{2}+ 2*\)

Assume that no surface species can be neglected and that \(K_{1}\)and \(K_{-1}\) already contain a factor of 2 (due to \(d\xi /dt= dN_{i}/ v_{i} dt, thus \ d\left[O*\right]/dt= 2K^{\prime }_{1} \left[O*\right]^{2}\) for step 3 in the forward direction), and derive the rate expression. This is algebraically complex, and demonstrates how rate expressions can become complicated once one moves away from some of the common simplified models. Suggestion: solve for [O*] first to substitute into the site balance, then solve that for [*]. The optimized rate parameters are listed in the table below and the apparent activation energy was 34.4 kcal \( mole^{-1}\). Are they reasonable? Evaluate them to as great an extent as possible based on the rules in Table 6.9 and 6.10. (You need to use only the maximum rate). The adsorption of \(N_{2}O\) was used to count \(Cu^{o}_{s}\) sites, and 354 mmole ‘O’ atoms was adsorbed per g catalyst (see Chapter 3.3.4.3). What was the dispersion of the Cu?

Solution

Step 2 defines the rate:

\(r_{m}= \frac{1}{m} \frac{dN_{N_{2} } }{dt} = \frac{1}{m} \left\lgroup-\frac{dN_{N_{2}O } }{dt} \right\rgroup = k \left[N_{2}O*\right]\)

 

Step 1 gives : \(K_{N_{2}O } = \frac{\left[N_{2}O*\right]}{ P_{N_{2_{O} } } \left[*\right] } , so \left[N_{2}O*\right] = K_{N_{2}O } P_{N_{2}O } \left[*\right]\)

 

Assuming all surface species are included, a site balance gives

\(L= \left[*\right] + \left\lfloor N_{2}O* \right\rfloor + \left\lfloor O* \right\rfloor\)

To remove the unknown [0 *], the SSA must be used:

\(\frac{d\left[O*\right] }{dt}= k \left[N_{2}O*\right]+ k_{-1} P_{O_{2} }[*]^2- k_{1} \left[O*\right]^{2} =0\)

and

\(\left[O*\right]= \left\lgroup\frac{K\left[N_{2}O*\right]+ k_{-1} P_{O_{2}} \left[*\right]^{2} }{k_{1}} \right\rgroup ^{1/2} \)

 

Then \(L= \left[*\right]+ K_{N_{2}O } P_{N_{2}O } \left[*\right]+ \left\lgroup\frac{K\left[N_{2}O*\right]+ k_{-1} P_{O_{2}} \left[*\right]^{2} }{k_{1}} \right\rgroup ^{1/2}= z\left[*\right]+ \left(x\left[*\right]+y \left[*\right]\right)^{1/2} \)

 

where \(z= 1 + K_{N_{2}O } P_{N_{2}O }, x = k K_{N_{2}O } P_{N_{2}O }/ k_{1}, and \ y = k_{-1} P_{O_{2}}/k_{1}\)

Rearranging and squaring each side gives:

\(x\left[*\right]+ y \left[*\right]^{2}= \left(L-z \left[*\right]\right)^{2}= L^{2}- 2 Lz \left[*\right] + z^{2}\left[*\right]^{2} \)

and

\(\left(y-z^{2}\right) \left[*\right]^{2}+ \left(x+ 2 Lz\right)\left[*\right] – L^{2} =0\)

The solution for this quadratic expression is:

\(\left[*\right] = \frac{\left(x+ 2 Lz \right)\pm \left(x^{2}+ 4 L xz + 4 y L^{2}\right)^{1/2} }{2 \left(y-z^{2}\right) } \)

Substituting back and using the positive root gives:

\(\left[*\right] = \frac{k K_{N_{2}O } P_{N_{2}O }/k_{1}+ 2 L \left(1+ K_{N_{2}O } P_{N_{2}O } \right) }{ -2 \left[k_{-1} P_{O_{2}}/k_{1}- \left(1+ K_{N_{2}O } P_{N_{2}O }\right)^{2} \right] } +\)

 

\(\frac{\left[ \left\lgroup\frac{k K_{N_{2}O } P_{N_{2}O }}{k_{1}} \right\rgroup ^{2} + \frac{4 L K K_{N_{2}O } P_{N_{2}O } \left(1+ K_{N_{2}O } P_{N_{2}O }\right) }{k_{1}} + \frac{4 L^{2} k_{-1} P_{O_{2}} }{k_{1}} \right]^{1/2} }{ -2 \left[k_{-1} P_{O_{2}} / k_{1} – \left(1+ K_{N_{2}O } P_{N_{2}O }\right)^{2} \right] }\)

Because the rate is: \(r_{m}= K \left[N_{2}O*\right]= K K_{N_{2}O } P_{N_{2}O }\left[*\right]\) , the final rate expression is:

\(r_{m}= \frac{d P_{N_{2}O }\left\{1+ \left(a+c\right)P_{N_{2}O }+ \left[a P_{N_{2}O } + \left(a^{2}+ ac\right) P_{N_{2}O }+ b P_{O_{2} } \right]^{1/2} \right\} }{ \left(1+ c P_{N_{2}O }\right)^{2}- b P_{O_{2} }}\)

 

where \(a= k K_{N_{2}O }/k_{1}, b = k_{1}/k_{-1}, c = K_{N_{2}O } \ and \ d= LK K_{N_{2}O }\) Note that these four

parameters can be combined in various ways to give the values in Table 1.
An Arrhenius plot of the adsorption equilibrium constant,\(K_{N_{2}O }\) , gives \(\Delta H^{o}_{ad}= -25.2 Kcal \ mole^{-1}\) and \(\Delta S^{o}_{ad}= -33 e.u\)

which satisfy the guidelines in Table 6.9.
The rate constant Lk represents a unimolecular decomposition reaction on the surface, soguideline 2 in Table 6.10 can be applied:

\(r_{a}= r_{m}/A_{cu.g}= \left\lgroup\frac{1.1 \ \mu mole}{s\cdot g \ cat} \right\rgroup \left\lgroup\frac{1 \ g \ cat}{2\left(354 \ \mu mole \ Cu_{s} \right) } \right\rgroup \left\lgroup\frac{\sim 10^{15} Cu_{s} }{ cm^{2} Cu} \right\rgroup \cong 2\times 10^{12} \frac{molecule}{s \ cm^{2}}\)

or, with correction for activation energy:

\(LA_{d}e^{-34400/1.987843} = 2\times 10^{12}= 1.2\times 10^{-9} \ L A_{d}\) and \(L A_{d}= 2\times 10^{21} molecule/s cm_{s} \)

Both values are below \(10^{28}\) molecule/s \(cm^{2}\) .
