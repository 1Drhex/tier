Guest Question.

For the network of Fig. 8.49, determine the following parameters using the complete hybrid equivalent model and compare to the results obtained using the approximate model.
(a) \(Z_{i}\) and \(Z^{′}_{i}\).
(b) \(A_{v}\).
(c) \(A_{i} = I_{o} /I_{i} \) and \(A^{′}_{i} = I_{o}/I^{′}_{i}\)
(d) \(Z_{o}\) (within \(R_{C}\)) and
\(Z^{′}_{o}\) (including \(R_{C}\)).

Admin
Solution

Now that the basic equations for each quantity have been derived, the order in which they are calculated is arbitrary. However, the input impedance is often a useful quantity to know and therefore will be calculated first. The complete common-emitter hybrid equivalent circuit has been substituted and the network redrawn as shown in Fig. 8.50. A Thévenin equivalent circuit for the input section of Fig. 8.50 will result in the input equivalent of Fig. 8.51 since \(E_{Th} \cong V_{S}\) and \(R_{Th} \cong R_{S} = 1 k \Omega\) (a result of \(R_{B} = 470 k \Omega\)being much greater than \(R_{S} = 1 k \Omega\)). In this example, \(R_{L} = R_{C}\) and \(I_{o}\) is defined as the current through \(R_{C}\) as in previous examples of this chapter. The output impedance \(Z_{o}\) as defined by Eq. (8.110) \(Z_{o} = \frac{V_{o}}{I_{o}} = \frac{1}{h_{o} – \left[h_{f} h_{r}/(h_{i} + R_{S})\right] }\) is for the output transistor terminals only. It does not include the effects of \(R_{C} . Z^{′}_{o}\) is simply the parallel combination of \(Z_{o}\) and \(R_{L}\). The resulting configuration of Fig. 8.51 is then an exact duplicate of the defining network of Fig. 8.48, and the equations derived above can be applied.

(a) Eq. (8.109): \(Z_{i} = \frac{V_{i}}{I_{i}} = h_{ie} – \frac{h_{fe} h_{re} R_{L}}{1 + h_{oe} R_{L}} = 1.6 k \Omega – \frac{(110)(2 \times 10^{- 4})(4.7 k \Omega)}{1 + (20 \mu S)(4.7 k \Omega)} = 1.6 k \Omega – 94.52 \Omega = 1.51 k \Omega\)

versus \(1.6 k \Omega\) using simply \(h_{ie}\).

\(Z^{′}_{i} = 470 k \Omega \parallel Z_{i} \cong Z_{i} = 1.51 k \Omega\)

(b) Eq. (8.108): \(A_{v} = \frac{V_{o}}{V_{i}} = \frac{- h_{fe}R_{L}}{h_{ie} + ( h_{ie} h_{oe} – h_{fe} h_{re}) R_{L}} = \frac{-(110)(4.7 k \Omega)}{1.6 k \Omega + \left[(1.6 k \Omega)(20 \mu S) – (110)(2 \times 10^{- 4})\right]4.7 k \Omega } = \frac{- 517 \times 10^{3} \Omega}{1.6 k \Omega + (0.032 – 0.022)4.7 k \Omega} = \frac{- 517 \times 10^{3} \Omega}{1.6 k \Omega + 47 \Omega} = – 313.9 \)

versus – 323.125 using \(A_{v} \cong – h_{fe} R_{L}/h_{ie}\)

(c) Eq. (8.107): \(A_{i} = \frac{I_{o}}{I_{i}} = \frac{h_{fe}}{1 + h_{oe}R_{L}} = \frac{110}{1 + (20 \mu S)(4.7 k \Omega)} = \frac{110}{1 + 0.094} = 100.55\)

versus 110 using simply \( h_{fe} \). Since \(470 k \Omega \gg Z_{i} , I^{′}_{i} \cong I_{i}\) and \(A^{′}_{i} \cong 100.55\) also

(d) Eq. (8.110): \(Z_{o} = \frac{V_{o}}{I_{o}} = \frac{1}{h_{oe} – \left[h_{fe} h_{re}/(h_{ie} + R_{S})\right] } = \frac{1}{20 \mu S – \left[(110)( 2 \times 10^{- 4})/(1.6 k \Omega + 1 k \Omega)\right] } = \frac{1}{20 \mu S – 8.46 \mu S} = \frac{1}{11.54 \mu S} = 86.66 k \Omega\)

which is greater than the value determined from \(1/h_{oe} = 50 k \Omega\).

 

\(Z^{′}_{o} = R_{C} \parallel Z_{o} = 4.7 k \Omega \parallel 86.66 k \Omega = 4.46 k \Omega\)

versus \(4.7 k \Omega\) using only \(R_{C}\).
