Guest Question.

A transistor with β = 100 is to be used in the collector-emitter feedback bias configuration as shown in Fig. 8.7. Assume Rc = 1.5 K, Re = 2 K Rb = 370 K, VCC = 12 V, VBE = 0.6 V and ICO = 0. (a) Calculate the collector to emitter voltage VCE, and (b) Find the stability factor S.

Admin
Solution

(a) Using Rc = 1.5 K, Re = 2 K, Rb = 370 K, VCC = 12 V, VBE = 0.6 V and ICO = 0 in Eq. (8.29), \(I_{C}\approx \frac{(V_{CC}-V_{BE})+(R_{e}+R_{c}+R_{b})I_{CO}}{R_{e}+R_{c}+R_{b}/\beta } \) , the collector current can be obtained as

\(I_{C}= \frac{(V_{CC}-V_{BE})+(R_{e}+R_{c}+R_{b})I_{CO}}{R_{e}+R_{c}+R_{b}/\beta } \)

\(=\frac{(12-0.6)V+0V}{(2+1.5+370/100)K} \)

= 1.58 mA

Since ICO= 0, the base current is

\(I_{B}=\frac{I_{C}}{\beta }=15.8 \mu A \)

Applying KVL around the collector loop in Fig. 8.7, we can get

VCC – (IC + IB )(Rc + Re ) – VCE = 0

Thus VCE can be calculated as

VCE = VCC – (IC + IB )(Rc + Re )

= 12 – (1.58 + 0.0158 ) mA × (1.5 + 2 ) K

= 6.41 V

(b) The stability factor S can be obtained from Eq. (8.28) , \(S=\frac{1+\beta }{1+\beta (R_{e}+R_{c})/(R_{e}+R_{c}+R_{b})} \) , as

\(S=\frac{1+100}{1+100(2+1.5)/(2+1.5+370)} \)

= 52.14

which is less than the value of S (= 65) obtained in the example of Sec. 8.4 for emitter-feedback bias circuit. Similarly, using the same values of Rc, Rb and β in Eq. (8.11), \(S=\frac{1+\beta }{1+\beta R_{c}/(R_{c}+R_{b})} \) , we can obtain S = 71.95 in collector-feedback bias. This example thus clearly illustrates that the collector-emitter feedback bias circuit has the highest stability among the fixed-bias, collector-feedback bias and emitter-feedback bias circuits.
