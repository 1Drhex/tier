Guest Question.

Determine if a 14 ft tall W10×45 steel section encased in a 24 in.×24in. concrete column with f′c = 5 ksi, is adequate to support a dead load of 350 kips and a live load of 1050 kips in axial compression. The concrete section has 8-#8 longitudinal reinforcing bars and #4 transverse ties @ 12in. o/c, The column is pinned at both ends and the load is
applied directly to the concrete encasement.

Admin
Solution

Material Properties:

Column W10×45 \(F_{y}=50 ksi \quad F_{u}=65 ksi\)

Manual
Table 2-3

Concrete \(f_{c}^{\prime}=5 ksi \quad E_{c}=3,900 ksi (145 pcf \text { concrete })\)

Reinforcement \(F_{y s t}=60 ksi\)

Geometric Properties:

\(\begin{array}{l} W 10 \times 45: \\A_{s}=13.3 in ^{2} \\I_{y}=53.4 in .\end{array}\)

Manual
Table 1-1

Reinforcing steel:

\(\left.A_{s r}=6.32 in .^{2} \quad \text { (the area of } 1-\# 8 \text { bar is } 0.79 in .^{2}, \text { per } ACI \right)\)

 

\(I_{s r}=\frac{\pi r^{4}}{4}+A d^{2}=8 \frac{\pi(0.50)^{4}}{4}+6(0.79)(9.5)^{2}=428 in .{ }^{4}\)

 

Concrete

\(\begin{array}{l}A_{c}=A_{c g}-A_{s}-A_{s r}=576 in .^{2}-13.3 in .^{2}-6.32 in .^{2}=556 in .^{2} \\I_{c}=I_{c g}-I_{s}-I_{s r}=27,200 in .^{4}\end{array}\)

Note: The weak axis moment of inertia is used as part of a slenderness check.

Limitations:

1) Normal weight concrete \(10 ksi \geq f_{c}^{\prime} \geq 3 ksi \quad f_{c}^{\prime}=5 ksi \quad \text { o.k. }\)

Section I1.2

2) \(F_{y s t} \leq 75 ksi \quad F_{y s t}=60 ksi \quad \text { o.k. }\)

3) The cross-sectional area of the steel core shall comprise at least one percent of the total composite cross section.

Section I2.1a

\(13.3 in .^{2}>(0.01)\left(576 in .^{2}\right)=5.76 in .^{2} \text { o.k. }\)

4) Concrete encasement of the steel core shall be reinforced with continuous longitudinal bars and lateral ties or spirals. The minimum transverse reinforcement shall be at least 0.009 \(\text { in. }^{2}\) of tie spacing.

\(0.20 in .^{2} / 12 in .=0.0167 in. ^{2} / in .>0.009 in .^{2} / in . \quad \text { o.k. }\)

5) The minimum reinforcement ratio for continuous longitudinal reinforcing, \(\rho_{ sr }\), shall be 0.004.

\(\rho_{s r}=\frac{A_{s r}}{A_{g}}=\frac{6.32 in .^{2}}{576 in .{ }^{2}}=0.011>0.004 \quad \text { o.k. }\)

Eqn. I2-1

Calculate the total required compressive strength

LRFD ASD
\(\begin{aligned}P_{u} &=1.2(350 kips )+1.6(1050 kips ) \\&=2100 kins\end{aligned}\) \(\begin{aligned}P_{a} &=350 \text { kips }+1050 \text { kips } \\&=1400 \text { kips }\end{aligned}\)

Calculate the available compressive strength

\(P_{o}=A_{s} F_{y}+A_{s r} F_{y r}+0.85 A_{c} f_{c}^{\prime}\)

 

\(=\left(13.3 in .^{2}\right)(50 ksi )+\left(6.32 in .^{2}\right)(60 ksi )+0.85\left(556 in .^{2}\right)(5 ksi )=3410 kips\)

Eqn. I2-2

\(C_{1}=0.1+2\left(\frac{A_{s}}{A_{c}+A_{s}}\right)=0.1+2\left(\frac{13.3 in .^{2}}{556 in .^{2}+13.3 in .^{2}}\right)=0.15\)

Eqn. I2-5

\(\begin{aligned}E I_{e f f} &=E_{s} I_{s}+0.5 E_{s} I_{s r }+C_{1} E_{c} I_{c} \\&=(29,000 ksi )\left(53.4 in .^{4}\right)+0.5(29,000 ksi )\left(428 in .^{4}\right)+(0.15)\left(3,900 in .^{4}\right)(27,200) \\&=23,700,000 kip – in .^{2}\end{aligned}\)

Eqn. I2-4

User note: K value is for this case K = 1.0.

\(P_{e}=\pi^{2}\left(E I_{e f f}\right) /(K L)^{2}=\pi^{2}\left(23,700,000 \text { kip-in. }^{2}\right) /((1.0)(14 \text { in. })(12 in . / ft ))^{2}=8,290 \text { kips }\)

Eqn. I2-3

\(\frac{P_{o}}{P_{e}}=\frac{3,410 \text { kips }}{8,290 kips }=0.411\)

 

\(0.411 \leq 2.25 \text { Therefore use Eqn. I2-2 to solve } P_{n}\)

Section I2.1b

\(P_{n}=P_{o}\left[0.658^{\left(\frac{P_{o}}{P_{e}}\right)}\right]=(3,410 \text { kips })\left[0.658^{(0.411)}\right]=2870 \text { kips }\)

Eqn. I2-6

LRFD ASD
\(\begin{array}{c}\phi_{c}=0.75 \\\phi_{c} P_{n}=0.75(2870 kips )=2150 kips \\2150 \text { kips }>2100 \text { kips } \quad \text { o.k. }\end{array}\) \(\begin{array}{c}\Omega_{c}=2.00 \\P_{n} / \Omega_{c}=2870 \text { kips } / 2.00=1440 \text { kips } \\1440 \text { kips }>1400 \text { kips } \quad \text { o.k. }\end{array}\)

Section I2.1b

Because the entire load in the column was applied directly to the concrete, accommodations must be made to transfer an appropriate portion of the axial force to the steel column. This force is transferred as a shear force at the interface between the two materials.

 

Determine the number and spacing of 1/2-in. diameter headed shear studs to transfer the axial force.

Material Properties:

Conc. \(f_{c}^{\prime}=5 ksi \quad E_{c}=4070 ksi\)

Shear Studs \(F_{u}=65 ksi\)

Manual Table
3-21

Geometric Properties:

\(\begin{array}{ll} W 10 \times 45 & A_{s t}=13.3 in .^{2} \quad A_{\text {flange }}=4.97 \text { in. }^{2} \quad A_{w e b}=3.36 \text { in. }^{2} & d_{s t}=10.1 \text { in. } \\\text { Shear Studs } & A_{s c}=0.196 \text { in. }^{2} \\\text { Conc. } & d_{c}=24 \text { in. }\end{array}\)

Manual
Table 1-1

Calculate the shear force to be transferred

LRFD ASD
\(V=\frac{P_{u}}{\phi_{c}}=\frac{2,100}{0.75}=2,800 kips\) \(V=P_{a} \Omega_{c}=1,400(2)=2,800 \text { kips }\)
\(V^{\prime}=V\left(A_{s} F_{y} / P_{o}\right)=2800 kips \left(\left(13.3 in .^{2}\right)(50 ksi ) /(3410 kips )\right)=546 kips\)

Eqn. I2-10

Calculate the nominal strength of one 1/2 in. diameter shear stud connector

\(Q_{n}=0.5 A_{s c} \sqrt{f_{c}^{\prime} E_{c}} \leq A_{s c} F_{u}\)

 

\(0.5 A_{s c} \sqrt{f_{c}^{\prime} E_{c}}=0.5\left(0.196 in .^{2}\right) \sqrt{(5 ksi )(3900 ksi )}=13.7 kips\)

Eqn. I2-12

\(A_{s c} F_{u}=\left(0.196 in ^{2}\right)(65 ksi )=12.7 kips\)

Therefore use 12.7 kips.

Calculate the number of shear studs required to transfer the total force, V’

\(V^{\prime} / Q_{n}=546 \text { kips } / 12.7 \text { kips }=43\)

An even number of studs are required to be placed symmetrically on two faces. Therefore use 22 studs minimum per flange

Determine the spacing for the shear studs

Section
I2.1f

The maximum stud spacing is 16 in.

The available column length is 14ft (12 in./ft) = 168 in. and the maximum spacing is

= 168 in./(22+1) = 7.3 in.

Therefore, on the flanges, use single studs @ 7 in.

Stud placement is to start 10.5 in. from one end.

Determine the length of the studs for the flanges;

\(\left(\frac{d_{c}-d_{s t}}{2}\right)-3 \text { in. }=\left(\frac{24 \text { in. }-10.1 \text { in. }}{2}\right)-3 \text { in. }=3.95 \text { in. }\)

Therefore use 3 ½ in. in length for the flanges.

Note: The subtraction of 3 in. is to ensure sufficient cover.

Summary: use 2-in. diameter shear stud connectors as shown on each flange, spaced @ 7 in.
