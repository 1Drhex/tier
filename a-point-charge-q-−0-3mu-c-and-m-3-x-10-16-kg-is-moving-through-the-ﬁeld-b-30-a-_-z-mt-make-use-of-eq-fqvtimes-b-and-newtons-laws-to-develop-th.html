Guest Question.

A point charge, \(Q= −0.3\mu C\) and \(m = 3 × { 10 }^{ -16 }\) kg, is moving through the ﬁeld \(B = 30{ a }_{ z }\) mT. Make use of Eq. \(F=Qv\times B\) and Newton’s laws to develop the appropriate differential equations, and solve them, subject to the initial condition at t =0,v = 3 ×\({ 10 }^{ 5 }{ a }_{ x}\)m/s at the origin. Solve these equations to evaluate at t = \(3\mu s\): a) the position P(x,y,z) of the charge; b) its velocity; c) and its kinetic energy.

Admin
Solution

We begin by visualizing the problem. Using F = qv × B, we ﬁnd that a positive charge moving along positive \({ a }_{ x}\), would encounter the z-directed B ﬁeld and be deﬂected into the negative y direction.

Motion along negative y through the ﬁeld would cause further deﬂection into the negative x direction. We can construct the differential equations for the forces in x and in y as follows:

\({ F }_{ x }a_{ x }=m\frac { d{ \upsilon }_{ x } }{ dt } a_{ x }=q{ \upsilon }_{ y }a_{ y }\times Ba_{ z }=qB{ \upsilon }_{ y }a_{ x }\)
\({ F }_{ y }a_{ y }=m\frac { d{ \upsilon }_{ y } }{ dt } a_{ y }=q{ \upsilon }_{ x }a_{ x }\times Ba_{ z }=-qB{ \upsilon }_{ x }a_{ y }\)
or
\(\frac { d{ \upsilon }_{ x } }{ dt } =\frac { qB }{ m } { \upsilon }_{ y }\)
and
\(\frac { d{ \upsilon }_{ y } }{ dt } =-\frac { qB }{ m } { \upsilon }_{ x }\)
To solve these equations, we ﬁrst differentiate (2) with time and substitute (1), obtaining:
\(\frac { { d }^{ 2 }{ \upsilon }_{ y } }{ d{ t }^{ 2 } } =-\frac { qB }{ m } \frac { d{ \upsilon }_{ x } }{ dt } =-{ \left( \frac { qB }{ m } \right) }^{ 2 }{ \upsilon }_{ y }\)

Therefore, \({ \upsilon }_{ y } =A sin(qBt/m)+\acute { A } cos(qBt/m)\). However, at t =0,\({ \upsilon }_{ y }\) =0, and so \(\acute { A }\) =0, leaving \({ \upsilon }_{ y } = A sin(qBt/m)\). Then, using (2),

\({ \upsilon }_{ x }=-\frac { m }{ qB } \frac { d{ \upsilon }_{ y } }{ dt } =-A\cos { \left( \frac { qBt }{ m } \right) }\)

Now at t = 0, \({ \upsilon }_{ x } = { \upsilon }_{ x0 } = 3\times { 10 }^{ 5 }\). Therefore \(A = −{ \upsilon }_{ x0 }\), and so \({ \upsilon }_{ x } = { \upsilon }_{ x0 } cos(qBt/m)\), and \({ \upsilon }_{ y } =−{ \upsilon }_{ x0 } sin(qBt/m)\). The positions are then found by integrating \({ \upsilon }_{ x }\) and \({ \upsilon }_{ y }\) over time:

\(x(t)=\int { } { \upsilon }_{ x0 }\cos { \left( \frac { qBt }{ m } \right) } dt+C=\frac { m{ \upsilon }_{ x0 } }{ qB } \sin { \left( \frac { qBt }{ m } \right) } +C\)
where C = 0, since x(0) = 0. Then
\(y(t)=\int { } { -\upsilon }_{ x0 }\sin { \left( \frac { qBt }{ m } \right) } dt+D =\frac { m{ \upsilon }_{ x0 } }{ qB } \cos { \left( \frac { qBt }{ m } \right) } +D\)

We require that y(0)=0, so D =−\(({ m\upsilon }_{ x0 })/(qB)\), and ﬁnally \(y(t) = −{ m\upsilon }_{ x0 }/qB [1−\cos( qBt/m)]\). Summarizing, we have, using \(q =−3\times { 10 }^{ – 7} C, m = 3\times { 10 }^{ -16 }kg, B = 30\times { 10 }^{ -3 }T\), and \({ \upsilon }_{ x0 } = 3 \times { 10 }^{ 5 }m/s\):
\(x (t)=\frac { m{ \upsilon }_{ x0 } }{ qB } \sin { \left( \frac { qBt }{ m } \right) } =-{ 10 }^{ -2 }\sin { (-3\times { 10 }^{ -7 }t) } m\)
\(y(t)=-\frac { m{ \upsilon }_{ x0 } }{ qB } \left[ 1-\cos { \left( \frac { qBt }{ m } \right) } \right] ={ 10 }^{ -2 }\left[ 1-\cos { (-3\times { 10 }^{ 7 }t) } \right] m\)
\({ \upsilon }_{ x }(t) = { \upsilon }_{ x0 }\cos { \left( \frac { qBt }{ m } \right) } =3\times { 10 }^{ 5 }\cos { (-3\times { 10 }^{ 7 }t) } m/s\)
\({ \upsilon }_{ y }(t)={ -\upsilon }_{ x0 }\sin { \left( \frac { qBt }{ m } \right) } = -3 \times { 10 }^{ 5 }\sin{ (-3\times { 10 }^{ 7 }t) } m/s\)
The answers are now:
a) At t = \(3 \times { 10 }^{ -6 }\) s, x = 8.9 mm, y = 14.5 mm, and z = 0.
b) At t = \(3 \times { 10 }^{ -6 } s, { \upsilon }_{ x }= – 1.3 \times { 10 }^{ 5 }m/s, { \upsilon }_{ y }= 2.7 \times { 10 }^{ 5 } m/s\), and so
\(v(t = 3 \mu s) = – 1.3 \times { 10 }^{ 5 } { a }_{ x }+ 2.7 \times { 10 }^{ 5 } { a }_{ y } m/s\)
whose magnitude is \(\upsilon = 3 \times { 10 }^{ 5 }\) m/s as would be expected.
c) Kinetic energy is K.E.= (1/2) m \({ \upsilon }^{ 2 } = 1.35 \mu J\) at all times.
