Guest Question.

Shear Force Distribution

Given: The one-story frame defined in Fig. E11.4a.

Determine: The column shears and the diagonal brace forces.

Admin
Solution

Using (11.19), the frame stiffness is

 

\(k_{\text {frame }}=\frac{3 E}{h^{3}}\left\{\frac{2 I_{ CE }}{1+(1 / 2)\left(\left(I_{ CE } / h\right) /\left(I_{ b } / L\right)\right)}+\sum_{\text {intercol }} \frac{I_{ CI }}{1+(1 / 4)\left(\left(I_{ CI } / h\right) /\left(I_{ b } / L\right)\right)}\right\}\) (11.19)

 

\(k_{frame}=\frac{3EI_{c}}{h^{3}}\left ( \frac{2}{1+(1/2)((I_{c}/h)/(I_{b}/L))}+\frac{3}{1+(1/4)((I_{c}/h)/(I_{b}/L))} \right )\)

 

\(=\frac{3EI_{c}}{h^{3}}\left ( \frac{8}{5}+\frac{8}{3} \right )=12.8\frac{EI_{c}}{h^{3}}\)

 

The brace stiffness follows from (11.20). Note that there are two braces.

 

\(k_{\text {brace }_{(\text {chevron })}}=\frac{2 A E}{h}\left(\sin \theta_{1} \cos ^{2} \theta_{1}\right)\) (11.20)

 

\(k_{brace}=2\frac{A_{brace}E}{h}(\sin \theta \cos ^{2}\theta )=\frac{A_{brace}E}{h}2(0.447)(0.894)^{2}=0.714\frac{A_{brace}E}{h}\)

 

Noting (11.16), the individual forces are related by

 

\(\begin{array}{l}P_{\text {frame }}=\frac{k_{\text {frame }}}{k_{ T }} P \\P_{\text {brace }}=\frac{k_{\text {brace }}}{k_{ T }} P\end{array}\) (11.16)

 

\(P_{frame}=\left ( \frac{k_{frame}}{k_{brace}} \right )P_{brace}=17.9\left ( \frac{I_{c}}{A_{brace}h^{2}} \right )P_{brace}\)

 

Summing the forces leads to

 

\(P_{frame}+P_{brace}=P\Rightarrow \left ( 17.9\frac{I_{c}}{A_{brace}h_{2}}+1 \right )P_{brace}=40\)

 

Then

 

\(P_{brace}=\frac{40}{1+(17.9)((40(10)^{6})/((650)(3000)^{2}))}=35.6\) KN

 

\(P_{frame}=4.36\) KN

 

The force in each diagonal brace is given by

 

\(F_{b}=\frac{P_{brace}}{2\cos \theta }=19.9\) KN

 

We evaluate the column shear forces using the corresponding stiffness factors defined by (11.11) and (11.12).

 

\(k_{ BE }=\frac{3 E I_{ CE }}{h^{3}}\left\{\frac{1}{1+(1 / 2)\left(\left(I_{ CE } / h\right) /\left(I_{ b } / L\right)\right)}\right\}=\frac{3 E I_{ CE }}{h^{3}} f_{ BE }\) (11.11)

 

\(k_{ BI }=\frac{3 E I_{ CI }}{h^{3}}\left\{\frac{1}{1+(1 / 4)\left(\left(I_{ CI } / h\right) /\left(I_{ b } / L\right)\right)}\right\}=\frac{3 E I_{ CI }}{h^{3}} f_{ BI }\) (11.12)

 

\(k_{E}=\frac{3EI_{c}}{h^{3}}\left ( \frac{4}{5} \right )\)

 

\(k_{I}=\frac{3EI_{c}}{h^{3}}\left ( \frac{8}{9} \right )\)

 

\(\frac{V_{E}}{V_{I}}=\frac{k_{E}}{k_{I}}=\frac{4/5}{8/9}=\frac{9}{10}\)

 

\(\therefore V_{E}=0.9V_{I}\)

 

Summing the shears,

 

\(2V_{E}+3V_{I}=4.36\Rightarrow 1.8V_{I}+3V_{I}=4.36\)

 

Then

 

\(V_{I}=0.91\) KN

 

\(V_{E}=0.82\) KN

 

Note that the brace carries the major portion of the story shear.
