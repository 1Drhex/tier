Guest Question.

For the spatial RSSR four-bar linkage in the posture shown in Fig. 10.7, the constant angular velocity of link 2 is \(\omega_{2}=40 \widehat{ k } rad / s\). Find the angular velocities and angular accelerations of links 3 and 4 and the velocity and acceleration of point B for the specified posture.

Admin
Solution

GRAPHIC POSTURE SOLUTION

The graphic posture analysis follows exactly the procedure given in Sec. 10.3 and . The solution is shown in Fig. 10.8

ANALYTIC POSTURE SOLUTION

The analytic posture solution also follows the procedure given in Sec. 10.3 and . For the given input angle, the results are:

 

\(\begin{aligned}R _{O_{4} O_{2}} &=12.000 \hat{ i }+3.000 \hat{ k } \text { in } \\R _{A O_{2}} &=2.000 \hat{ i }+3.464 \hat{ j } i n \\R _{B A} &=10.000 \hat{ i }+2.746 \hat{ j }+10.838 \hat{ k } \text { in } \\R _{B O_{4}} &=6.210 \hat{ j }+7.838 \hat{ k } \text { in }\end{aligned}\)

 

GRAPHIC VELOCITY AND ACCELERATION SOLUTIONS

The determination of the velocities and accelerations of a spatial mechanism by graphic means can be conducted in the same manner as for a planar mechanism. However, the posture information as well as the velocity and acceleration vectors often do not appear in their true lengths in the front, top, and profile views, but are foreshortened. This means that spatial motion problems usually require the use of auxiliary views where the vectors appear in true lengths.

The velocity solution for this example is shown in Fig. 10.9 with notation corresponding to that used in many texts on descriptive geometry. The subscripts F, T, and P designate the front, top, and profile views, and the subscripts 1 and 2 indicate the first and second auxiliary views, respectively.

The steps to obtain the solution to the velocity problem are as follows:

1. We first construct to scale the front, top, and profile views of the linkage and designate each point.

2. Next, we calculate VA as above and place this vector in position with its origin at A in the three views. The velocity VA shows in true length in the frontal view. We designate its terminus as point aF and project this point into the top and profile views to find aT and aP.

3. The magnitude of velocity VB is unknown, but its direction is perpendicular to \(R _{B O_{4}},\) , and, once the problem is solved, it will show in true size in the profile view. Therefore, we construct a line in the profile view that originates at point AP (which becomes the origin of our velocity polygon) and corresponds in direction to that of VB (perpendicular to \( R_{B_{P}O_{4 P} \) ). We then choose any point dP on this line and project it into the front view (dF) and top view (dT ) to establish the line of action of VB in those views.

4. The equation to be solved is

VB = VA +VBA, (1)

where VA and the directions of VB and VBA are known. We note that VBA must lie in a plane perpendicular (in space) to RBA, but its magnitude and that of VB are unknown.

Vector VBA must originate at the terminus of VA and must lie in a plane perpendicular to RBA; it terminates by intersecting line Ad or its extension. To find the plane perpendicular to RBA, we start by constructing the first auxiliary view, which shows vector RBA in true length, so we construct the edge view of plane 1 parallel to ATBT and project RBA to this plane. In this projection, we note that distances k and l are the same in this first auxiliary view as in the frontal view. The first auxiliary view of AB is A1B1, which is true length. We also project points a and d into this view, but the remaining links need not be projected.

5. In this step, we construct a second auxiliary view, plane 2, such that the projection of AB upon it is a point. Then, all lines drawn parallel to this plane are perpendicular to RBA. The edge view of such a plane must be perpendicular to A1B1 extended. In this example, it is convenient to choose this plane so that it contains point a; therefore, we construct the edge view of plane 2 through point a1 perpendicular to A1B1 extended. Now we project points A, B, a, and d into this plane. Note that the distances—m, for example—of points from plane 1 must be the same in the top view and in the second auxiliary view.

6. We now extend line A1d1 until it intersects the edge view of plane 2 at b1, and we find projection b2 of this point into plane 2 where the projector meets line A2d2 extended. Now both points a and b lie in plane 2, and any line drawn in plane 2 is perpendicular to RBA. Therefore, line ab is VBA, and the second auxiliary view of that line shows its true length. Line A2b2 is the projection of VB on the second auxiliary plane, but not in its true length, since point A is not in plane 2.

7. To simplify the reading of the drawing, step 7 is not shown; those who follow the first six steps carefully should have no difficulty with the seventh. We can project the three vectors into the top, front, and profile views. Velocity VB can be measured from the profile view where it appears in true length. The result is

\(V _{B}=200 \hat{ j }-158 \hat{ k } \text { in } / s.\)

When all vectors have been projected into these three views, we can measure their x, y, and z projections directly.

8. If we assume that the idle freedom of spin is not active and, therefore, that the angular velocity vector ω3 is perpendicular to RBA and appears in true length in the second auxiliary view,* then the magnitudes of the angular velocities can be determined from the equations

 

\(\omega_{3}=\frac{V_{B A}}{R_{B A}}=\frac{242 in / s }{15 in }=16.13 rad / s\)

 

\(\omega_{4}=\frac{V_{B O_{4}}}{R_{B O_{4}}}=\frac{255 in / s }{10 in }=25.50 rad / s\)

 

Therefore, we can draw the angular velocity vectors in the views where they appear in their true lengths and project them into the views where their vector components can be measured. The results are

 

\(\begin{aligned}& \omega _{3}=-7.69 \hat{ i }+13.71 \hat{ j }+3.63 \hat{ k } rad / s \\& \omega _{4}=25.50 \hat{ i } rad / s\end{aligned}\)

 

The steps to obtain the solution to the acceleration problem, also not shown, are followed in an identical manner, using the same two auxiliary planes. The equation to be solved is

 

\(A _{B O_{4}}^{n}+ A _{B O_{4}}^{t}= A _{A O_{2}}^{n}+ A _{A O_{2}}^{t}+ A _{B A}^{n}+ A _{B A}^{t}\) (2)

 

where vectors \(A _{B O_{4}}^{n}, A _{A O_{2}}^{n}, \quad A_{A O_{2}}^{t}, \text { and } A _{B A}^{n}\) can be determined now that the velocity analysis has been completed. Also, we see that \(A _{B O_{4}}^{t} \text { and } A _{B A}^{t}\) have known directions. Therefore, the solution can proceed exactly as for the velocity polygon; the only difference in approach is that there are more known vectors. The final results are

 

\(\begin{aligned}& \alpha _{3}=-527 \hat{ i }+619 \hat{ j }+329 \hat{ k } rad / s ^{2} \\& \alpha _{4}=-865 \hat{ i } rad / s ^{2} \\& A _{B}=2750 \hat{ j }-10450 \hat{ k } \text { in } / s ^{2}\end{aligned}\)

 

ANALYTIC VELOCITY AND ACCELERATION SOLUTIONS

From the given information and the constraints, the angular velocities and accelerations can be written as

 

\(\begin{array}{ll}\omega _{2}=40 \hat{ k } rad / s , & \alpha _{2}= 0 \\\omega _{3}=\omega_{3}^{x} \hat{ i }+\omega_{3}^{y} \hat{ j }+\omega_{3}^{z} \hat{ k }, & \alpha _{3}=\alpha_{3}^{x} \hat{ i }+\alpha_{3}^{y} \hat{ j }+\alpha_{3}^{z} \hat{ k } \\\omega _{4}=\omega_{4} \hat{ i }, & \alpha _{4}=\alpha_{4} \hat{ i } .\end{array}\)

 

First, we find the velocity of point A as the velocity difference from point O2. Thus,

\(\begin{aligned}V _{A} &= V _{A O_{2}}=\omega_{2} \times R _{A O_{2}}=(40 \hat{ k } rad / s ) \times(2.000 \hat{ i }+3.464 \hat{ j } \text { in }) \\&=-138.560 \hat{ i }+80.000 \hat{ j } in / s\end{aligned}\) (3)

 

Similarly, for link 3, the velocity difference is

 

\(\begin{aligned}V _{B A}=&\left(\omega_{3}^{x} \hat{ i }+\omega_{3}^{y} \hat{ j }+\omega_{3}^{z} \hat{ k }\right) \times(10.000 \hat{ i }+2.746 \hat{ j }+10.838 \hat{ k } \text { in }) \\=&\left(10.838 \omega_{3}^{y}-2.746 \omega_{3}^{z}\right) \hat{ i }+\left(10.000 \omega_{3}^{z}-10.838 \omega_{3}^{x}\right) \hat{ j }, \\&+\left(2.746 \omega_{3}^{x}-10.000 \omega_{3}^{y}\right) \hat{ k } \text { in }\end{aligned}\) (4)

 

and for link 4 the velocity of point B is

 

\(\begin{aligned}V _{B} &= V _{B O_{4}}= \omega _{4} \times R _{B O_{4}}=\left(\omega_{4} \hat{ i }\right) \times(6.210 \hat{ j }+7.838 \hat{ k } \text { in }) \\&=-(7.838 \text { in }) \omega_{4} \hat{ j }+(6.210 \text { in }) \omega_{4} \hat{ k }\end{aligned}\) (5)

 

The next step is to substitute these into the velocity difference equation,

VB = VA +VBA. (6)

After this, we separate the \(\hat{ i }, \hat{ j }, \text { and } \hat{ k }\) components to obtain the three algebraic equations:

\(10.838 \text { in } \omega_{3}^{y}-2.746 \text { in } \omega_{3}^{z}=138.560 in / s\) (7)

 

\(-10.838 \text { in } \omega_{3}^{x}+10.000 \text { in } \omega_{3}^{z}+7.838 \text { in } \omega_{4}=-80.000 in / s\) (8)

 

\(2.746 \text { in } \omega_{3}^{x}-10.000 \text { in } \omega_{3}^{y}-6.210 \text { in } \omega_{4}=0.000 \text {. }\) (9)

 

We now have three equations; however, we note that there are four unknowns, \(\omega_{3}^{x}, \omega_{3}^{y}, \omega_{3}^{z}, \text { and } \omega_{4}.\) This would not occur in most problems but does here because of the idle freedom of the coupler to spin about its own axis. Since this spin does not affect the input-output relationship, we will find the same result for ω4 regardless of the rate of spin. One way to proceed, therefore, would be to choose one component of ω3 and give it a value, thus setting the rate of spin, and then solve for the other unknowns. Another approach is to set the rate of spin about the coupler link axis to zero by requiring that

 

\(\begin{aligned}\omega _{3} \times R _{B A} &=0, \\R_{B A}^{x} \omega_{3}^{x}+R_{B A}^{y}\omega_{3}^{y}+R_{B A}^{z} \omega_{3}^{z} &=0, \\\left(10.000 \text { in) } \omega_{3}^{x}+(2.746 \text { in }) \omega_{3}^{y}+(10.838 \text { in })\omega_{3}^{z}\right.&=0 .\end{aligned}\) (10)

 

Equations (7) through (10) can now be solved simultaneously for the four unknowns. The result is

 

\(\begin{aligned}& \omega _{3}=-7.692 \hat{ i }+13.704 \hat{ j }+3.625 \hat{ k } rad / s \\& \omega _{4}=-25.468 \hat{ i } rad / s\end{aligned}\)

 

Substituting these angular velocities into Eq. (5), the velocity of point B is

\(V _{B}=199.618 \hat{ j }-158.156 \hat{ k } \text { in } / s\)

Turning next to acceleration analysis, we compute the following components:

\(A _{A O_{2}}^{n}=\omega_{2} \times\left(\omega_{2} \times R _{A O_{2}}\right)=-3200.00 \hat{ i }-5542.56 \hat{ j } in / s ^{2}\) (11)

 

 

\(A _{A O_{2}}^{t}= \alpha _{2} \times R _{A O_{2}}= 0\) (12)

 

 

\(A _{B A}^{n}= \omega _{3} \times\left( \omega _{3} \times R _{B A}\right)=-2601.01 \hat{ i }-714.26 \hat{ j }-2818.99 \hat{ k } \text { in } / s ^{2}\) (13)

 

\(\begin{aligned}A _{B A}^{t}=& \alpha _{3} \times R _{B A}=\left(\alpha_{3}^{x} \hat{ i }+\alpha_{3}^{y} \hat{ j }+\alpha_{3}^{z} \hat{ k }\right) \times(10.000 \hat{ i }+2.746 \hat{ j }+10.838 \hat{ k } \text { in }) \\=&\left(10.838 \alpha_{3}^{y}-2.746 \alpha_{3}^{z}\right) \hat{ i }+\left(10.000 \alpha_{3}^{z}-10.838 \alpha_{3}^{x}\right) \hat{ j } \\&+\left(2.746 \alpha_{3}^{x}-10.000 \alpha_{3}^{y}\right) \hat{ k } \text { in }\end{aligned}\) (14)

 

\(A _{B O_{4}}^{n}= \omega _{4} \times\left( \omega _{4} \times R _{B O_{4}}\right)=-4028.05 \hat{ j }-5084.03 \hat{ k } in / s ^{2}\) (15)

 

\(A _{B O_{4}}^{t}=\alpha_{4} \times R _{B O_{4}}=-7.838 \alpha_{4} \hat{ j }+6.210 \alpha_{4} \hat{ k } \text { in. }\) (16)

 

These quantities are now substituted into the acceleration difference equation,

 

\(A _{B O_{4}}^{n}+ A _{B O_{4}}^{t}= A _{A O_{2}}^{n}+ A _{A O_{2}}^{t}+ A _{B A}^{n}+ A _{B A}^{t}\) (17)

 

and separated into \(\hat{ i }, \hat{ j }, \text { and } \hat{ k }\) components. Along with the condition \(\alpha_{3} \times R _{B A} = 0\) for the spin of the idle freedom, this results in four equations in four unknowns as follows:

 

\(\begin{aligned}&\begin{aligned}10.838 \text { in } \alpha_{3}^{y}-2.746 \text { in } &\alpha_{3}^{z}=5801.01 in / s ^{2}, \\-10.838 \text { in } \alpha_{3}^{x}+10.000 \text { in }\alpha_{3}^{z}+7.838 \text { in } & \alpha_{4}=2228.61 in / s ^{2}, \\2.746 \text { in } \alpha_{3}^{x}-10.000 \text { in } \alpha_{3}^{y}-6.210 \text { in } &\alpha_{4}=-2265.04 in / s ^{2},\end{aligned}\\&10.000 \text { in } \alpha_{3}^{x}+2.746 \text { in } \alpha_{3}^{y}+10.838 \text { in } \alpha_{3}^{z}=0.00 \text {. }\end{aligned}\)

 

The coefficients (on the left-hand side) of these acceleration equations are identical to those of the velocity equations [Eqs. (7)–(10)]. This is always the case. These equations are now solved to give the desired accelerations.

 

\(\alpha_{3}=-526.94 \hat{ i }+618.71 \hat{ j }+329.43 \hat{ k } rad / s ^{2}\)

 

\(\alpha_{4}=-864.59 \hat{ i } rad / s ^{2}\)

 

\(A _{B}= A _{B O_{4}}^{n}+ A _{B O_{4}}^{t}=2748.61 \hat{ j }-10453.10 \hat{ k } \text { in } / s ^{2}\)

* This is the same assumption we used in the analytic solution when we wrote the equation ω3 × RBA = 0.
