Guest Question.

A point charge for which Q = 2 × \({ 10 }^{ -16 }\)C and m= 5 × \({ 10 }^{ -26 }\)kg is moving in the combined ﬁelds \(E = 100{ a }_{ x } −200{ a }_{ y }+300{ a }_{ z }\) V/m and \(B = −3{ a }_{ x } + 2{ a }_{ y } − { a }_{ z }\) mT. If the charge velocity at t = 0 is v(0)\(=(2{ a }_{ x } −3{ a }_{ y } −4{ a }_{ z }) × { 10 }^{ 5 } m/s\). (a) give the unit vector showing the direction in which the charge is accelerating at t = 0. (b)ﬁnd the kinetic energy of the charge at t =0.

Admin
Solution

a) Use F(t = 0) = \(q\left[ E+\left( v(0)\times B \right) \right]\) , where
v(0) \(\times B = \left( 2{ a }_{ x }-3{ a }_{ y }-4{ a }_{ z } \right) { 10 }^{ 5 }\times \left( -3{ a }_{ x }+2{ a }_{ y }-{ a }_{ z } \right) { 10 }^{ -3 }=1100{ a }_{ x }+1400{ a }_{ y }-500{ a }_{ z }\)
So the force in newtons becomes
\(F(0) = \left( 2\times { 10 }^{ -16 } \right) \left[ \left( 100+1100 \right) { a }_{ x }+\left( 1400-200 \right) { a }_{ y }+\left( 300-500 \right) { a }_{ z } \right] =4\times { 10 }^{ -14 }\left[ { 6a }_{ x }+6{ a }_{ y }-{ a }_{ z } \right] \)
The unit vector that gives the acceleration direction is found from the force to be
\({ a }_{ F }=\frac { { 6a }_{ x }+6{ a }_{ y }-{ a }_{ z } }{ \sqrt { 73 } } =.70{ a }_{ x }+.70{ a }_{ y }-.12{ a }_{ z }\)
b)
\(K.E.=\frac { 1 }{ 2 } m{ \left| v(0) \right| }^{ 2 }=\frac { 1 }{ 2 } \left( 5\times { 10 }^{ -26 }kg \right) { \left( 5.39\times { 10 }^{ 5 }m/s \right) }^{ 2 }=7.25\times { 10 }^{ -15 }J=7.25fJ\)
