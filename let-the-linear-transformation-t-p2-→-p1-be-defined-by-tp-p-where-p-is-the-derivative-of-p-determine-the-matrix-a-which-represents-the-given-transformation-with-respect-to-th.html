Guest Question.

Let the linear transformation \(T: P_{2} \rightarrow P_{1}\) be defined by

\(T(\mathbf{p})=\mathbf{p}^{\prime}\)

where \(\mathbf{p}^{\prime}\) is the derivative of \(\mathbf{p}\). Determine the matrix A which represents the given transformation with respect to the ordered bases

\(B=\left[1, x, x^{2}\right] \text { and } C=[1, x]\)

for \(P_{2}\) and \(P_{1}\) respectively. Find \(T\left(2 x^{2}+3 x+1\right)\) by using this matrix \(\mathrm{A}\).

 

Admin
Solution

First we look at how \(T\) behaves towards the basis (axes) vectors. Applying the given linear transformation to the vectors in basis (axes) \(B=\left[1, x, x^{2}\right]\) we have

\(\begin{array}{cl}T(1)=1^{\prime}=0 & \text { [differentiating }] \\T(x)=x^{\prime}=1 & \quad \text { [differentiating }] \\T\left(x^{2}\right)=\left(x^{2}\right)^{\prime}=2 x & \quad \text { [differentiating }]\end{array}\)

We need to write each of these as the coordinates of the basis (axes) \(C=[1, x]\) :

\(\begin{aligned}T(1) &=0=a(1)+b(x) \text { gives } a=0 \text { and } b=0 \\T(x) &=1=c(1)+d(x) \text { gives } c=1 \text { and } d=0 \\&T\left(x^{2}\right)=2 x=e(1)+f(x) \text { gives } e=0 \text { and } f=2\end{aligned}\)

What is our matrix \(A\) equal to?

\(\mathbf{A}=\left(\left[\begin{array}{ll}T & (1)\end{array}\right]_{C} \quad[T(x)]_{C} \quad\left[\begin{array}{lll}T & \left(x^{2}\right)\end{array}\right]_{C}\right)=\left(\begin{array}{lll}a & c & e \\b & d & f\end{array}\right)=\left(\begin{array}{lll}0 & 1 & 0 \\0 & 0 & 2\end{array}\right)\)

For \(T\left(2 x^{2}+3 x+1\right)\) we have \(\mathbf{p}=2 x^{2}+3 x+1\) so \([\mathbf{p}]_{B}=\left(\begin{array}{l}1 \\ 3 \\ 2\end{array}\right)\) (coefficients of \(1, x\) and \(x^{2}\), and it must be in this order, because the given basis is ordered, \(B=\left[1, x, x^{2}\right]\). Remember, a basis is a set of vectors which represent axes, so the order of the basis vectors matters.):

\([T(\mathbf{p})]_{C}=\mathbf{A}[\mathbf{p}]_{B}=\left(\begin{array}{lll}0 & 1 & 0 \\0 & 0 & 2\end{array}\right)\left(\begin{array}{l}1 \\3 \\2\end{array}\right)=\left(\begin{array}{l}3 \\4\end{array}\right)\)

The entries 3 and 4 in the right hand column vector are the coefficients of the basis \(C=[1, x]\), which means that we have

\(T\left(2 x^{2}+3 x+1\right)=3(1)+4(x)=3+4 x\)

Checking this result by differentiating the quadratic:

\(T\left(2 x^{2}+3 x+1\right)=\left(2 x^{2}+3 x+1\right)^{\prime}=4 x+3\)

Thus the matrix \(\mathbf{A}\) does give the derivative of the quadratic polynomial \(2 x^{2}+3 x+1\)
