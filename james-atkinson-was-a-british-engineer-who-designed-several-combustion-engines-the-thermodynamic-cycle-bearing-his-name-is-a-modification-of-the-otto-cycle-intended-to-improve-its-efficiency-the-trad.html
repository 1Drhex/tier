Guest Question.

James Atkinson was a British engineer who designed several combustion engines. The thermodynamic cycle bearing his name is a modification of the Otto cycle intended to improve its efficiency. The trade-off in achieving higher efficiency is a decrease in the work performed per cycle. The idealised Atkinson cycle consists of the following reversible processes:

• \( 1 \longrightarrow 2 \): adiabatic compression
• \( 2 \longrightarrow 3 \): isochoric heating
• \( 3 \longrightarrow 4 \): isobaric heating
• \( 4 \longrightarrow 5 \): adiabatic expansion
• \( 5 \longrightarrow 6 \): isochoric cooling
• \( 6 \longrightarrow 1 \): isobaric cooling

Assume that the adiabatic processes are reversible and that the cycle is operated on an ideal gas characterised by,

p V = N R T

U = c N R T

\( γ = \frac{c+1}{c} \).

The following physical quantities that characterise the cycle are assumed to be known :
volumes \(V_1, V_2\) and \(V_6\), pressures \(p_1 and p_3\), temperature \(T_5\), and the number of moles N of gas. Analyse this cycle by using the following instructions :
a) Draw the (p, V) diagram of the Atkinson cycle.
b) Determine the pressures \(p_2, p_4, p_5, p_6\), the volumes \(V_3, V_4, V_5\) and temperatures \(T_1, T_2, T_3, T_4, T_6\), in terms of the known physical quantities.
c) Find the works \(W_{12}, W_{23}, W_{34}, W_{45}, W_{56}, W_{61}\) and the work W performed per cycle.
d) Find the heat transfers \(Q_{12}, Q_{23}, Q_{34}, Q_{45}, Q_{56}, Q_{61}\) and the heat \(Q^+ = Q_{23} + Q_{34}\) provided to the gas.
e) Determine the efficiency of the Atkinson cycle,

\( η_A = − \frac {W}{Q^+} \).

Admin
Solution

a) For the adiabatic process, according to relation (5.90), \(p (V) = const/V^γ\) where γ > 1 and \(const = p_1 V^γ_1 = p_2 V^γ_2\) or \(const = p_4 V^γ_4 = p_5 V^γ_5\), which are monotonously decreasing convex functions of V (Fig. 7.6). For the isochoric processes, \(V = V_2 = V_3 = const\) or \(V = V_5 = V_6 = const\). For the isobaric processes, \(p (V) = p_3 = p_4 = const\) or \(p (V) = p_6 = p_1 = const\).

\( pV^γ = const\) (5.90)

b) Using relation (5.90) and the ideal gas equation of state (5.47), the pressures are given by,

pV = NR T (5.47)

\( p_2 = p_1 \Bigl(\frac{V_1}{V_2}\Bigr)^\gamma \).

\( p_4 = p_3 \).

\( p_5 = \frac{N R T_5}{V_6} \).

\( p_6 = p_1 \).

the volumes by,

\( V_3 = V_2 \).

\( V_4 = \Bigl(\frac{N R T_5}{p_3}\Bigr)^{\frac{1}{\gamma}} V^{\frac{\gamma -1}{\gamma}}_6 \).

\( V_5 = V_6 \).

The temperatures are written as,

\( T_1 = \frac{p_1 V_1}{N R}\).

\( T_2 = \frac{p_1 V_1}{N R} \Bigl(\frac{V_1}{V_2}\Bigr)^{\gamma -1}\).

\( T_3 = \frac{p_3 V_3}{N R}\).

\( T_4 = \Bigl(\frac{p_3 V_6}{N R}\Bigr)^{\frac{\gamma -1}{\gamma}} T^{\frac{1}{\gamma}}_5 \).

\( T_6 = \frac{p_1 V_6}{N R} \).

c) According to relation (7.14), the work performed during the adiabatic compression and expansion is written as,

\( W_{if} =\Delta U_{if} = c N R \int_{T_i}^{T_f}{dT} = c N R (T_f -T_i) \). (7.14)

\( W_{12} =\Delta U_{12} = c N R \int_{T_1}^{T_2}{dT} = c N R (T_2 -T_1) \).

\( W_{45} =\Delta U_{45} = c N R \int_{T_4}^{T_5}{dT} = c N R (T_5 -T_4) \).

According to relation (7.18), there is no work performed during the isochoric heating and cooling,

\( W_{if} = – \int_{i}^{f}{p dV} =0\). (7.18)

\( W_{23} = W_{56} = 0 \)

According to relation (7.21), the work performed during the isobaric processes can be written as,

\( W_{if} = – \int_{i}^{f}{p dV} = – p \int_{V_i}^{V_f}{dV} = −p (V_f − V_i) = −N R (T_f − T_i) \) (7.21)

\( W_{34} = – \int_{3}^{4}{p dV} = – p_3 \int_{V_4}^{V_3}{dV} = −p_3(V_4 − V_3) = N R (T_4 − T_3) \)

\( W_{61} = – \int_{6}^{1}{p dV} = – p_1 \int_{V_6}^{V_1}{dV} = −p_1(V_1 − V_6) = N R (T_1 − T_6) \).

The work performed per cycle reads,

\( W = W_{12} + W_{34} + W_{45} + W_{61} \).

\( = c N R (T_2 − T_ + T_5 − T_4) + N R (T_4 − T_3 + T_1 − T_6) \).

d) According to relation (7.13), there is no heat exchanged during the adiabatic compression and expansion,

\( Q_{if} = \int_{i}^{f}{T dS} = 0 \). (7.13)

\( Q_{12} = Q_{45} = 0 \).

According to relation (7.19), the heat exchanged during the isochoric heating and cooling is given by,

\( Q_{if} = \Delta U_{if} = \int_{U_i}^{U_f}{dU} = c N R \int_{T_i}^{T_f}{dT} = c N R (T_f -T_i) \). (7.19)

\( Q_{23} = \Delta U_{23} = \int_{U_2}^{U_3}{dU} = c N R \int_{T_2}^{T_3}{dT} = c N R (T_3 -T_2) \).

\( Q_{56} = \Delta U_{56} = \int_{U_5}^{U_6}{dU} = c N R \int_{T_5}^{T_6}{dT} = c N R (T_6 -T_5) \).

According to relation (7.22), the heat exchanged during the isobaric processes can be written as,

\( Q_{if} =\Delta H_{if} = \int_{H_i}^{H_f}{dH} = (c+1) N R \int_{T_i}^{T_f}{dT} = (c+1) N R (T_f -T_i) \). (7.22)

\( Q_{34} =\Delta H_{34} = \int_{H_3}^{H_4}{dH} = (c+1) N R \int_{T_3}^{T_4}{dT} = (c+1) N R (T_4- T_3) \).

\( Q_{61} =\Delta H_{61} = \int_{H_6}^{H_1}{dH} = (c+1) N R \int_{T_6}^{T_1}{dT} = (c+1) N R (T_1 -T_6) \).

The heat provided from the hot reservoir reads,

\( Q^+ = Q_{23} + Q_{34} = c N R (T_3 − T_2) + (c + 1) NR (T_4 − T_3) \).

e) Using the definition of the efficiency (7.38), we obtain,

\( η = − \frac{W}{Q^+} \). (7.38)

\( η_A = − \frac{W}{Q^+} = – \frac{c (T_2 − T_1 + T_5 − T_4) + (T_4 − T_3 + T_1 − T_6)}{c (T_3 − T_2) + (c + 1) (T_4 − T_3)} \).

\( = \frac{(T_1 − T_2 + T_4 − T_5) + (γ − 1) (T_3 − T_4 + T_6 − T_1)}{(T_3 − T_2) + γ (T_4 − T_3)} \).
