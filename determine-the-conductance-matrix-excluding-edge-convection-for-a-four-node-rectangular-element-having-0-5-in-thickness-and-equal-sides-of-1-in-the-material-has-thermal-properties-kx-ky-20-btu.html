Guest Question.

Determine the conductance matrix (excluding edge convection) for a four-node, rectangular element having 0.5 in. thickness and equal sides of 1 in. The material has thermal properties \(\ K_{x} = K_{y} = 20 Btu/(hr-ft-°F) \) and \(\ h = 50 Btu/(hr-ft^{2} -°F) \) .

Admin
Solution

The element with node numbers is as shown in Figure 7.9 and the interpolation functions, Equation 6.56, are

Equation (6.56a):

\(\ N_{1}(r,s)=\frac{1}{4}(1-r)(1-s) \)

 

\(\ N_{2}(r,s)=\frac{1}{4}(1+ r)(1-s) \)

 

\(\ N_{3}(r,s)=\frac{1}{4}(1+ r)(1+ s) \)

 

\(\ N_{4}(r,s)=\frac{1}{4}(1-r)(1+ s) \)

Equation (6.56b):

\(\ \phi (x,y)=\phi (r,s)=N_{1}(r,s)\phi _{1} + N_{2}(r,s)\phi _{2}+ N_{3}(r,s)\phi _{3}+ N_{4}(r,s)\phi _{4} \)

in terms of the normalized coordinates r and s. For the 1-in. square element, we have 2a = 2b = 1 and dA = dx dy = ab dr ds. The partial derivatives in terms of the normalized coordinates, via the chain rule, are

\(\ \frac{\partial N_{i} }{\partial x}=\frac{\partial N_{i} }{\partial r} \frac{\partial r}{\partial x}=\frac{1}{a}\frac{\partial N_{i} }{\partial r} \) i = 1, 4

 

\(\ \frac{\partial N_{i} }{\partial y}=\frac{\partial N_{i} }{\partial s} \frac{\partial s}{\partial y}=\frac{1}{b}\frac{\partial N_{i} }{\partial s} \) i = 1, 4

Therefore, Equation 7.37 becomes

\(\ \left[k^{(e)} \right]=\int\limits_{A}^{}{\int\limits_{}^{}{(K_{x}\left[\frac{\partial N}{\partial x}\right] ^{T}\left[\frac{\partial N}{\partial x}\right] + K_{y}\left[\frac{\partial N}{\partial y}\right] ^{T}\left[\frac{\partial N}{\partial y}+\right] )t dA} }+ 2h\int\limits_{A}^{}{\int\limits_{}^{}{\left[N\right] ^{T} \left[N\right] dA } } \)

 

\(\ \left[k^{(e)} \right]=\int\limits_{-1}^{1}{\int\limits_{-1}^{1}{(K_{x}\left[\frac{\partial N}{\partial r}\right] ^{T}\left[\frac{\partial N}{\partial r}\right]\frac{1}{a^{2} } + K_{y}\left[\frac{\partial N}{\partial s}\right] ^{T}\left[\frac{\partial N}{\partial s}\right]\frac{1}{b^{2} } )tab dr ds} }+ 2h\int\limits_{-1}^{1}{{\left[N\right] ^{T} \left[N\right]ab dr ds} } \)

or, on a term by term basis,

\(\ K_{ij}=\int\limits_{-1}^{1}{\int\limits_{-1}^{1}{(K_{x}\frac{\partial N_{i} }{\partial r}\frac{\partial N_{j} }{\partial r}\frac{1}{a^{2} } +K_{y}\frac{\partial N_{i} }{\partial s}\frac{\partial N_{j} }{\partial s}\frac{1}{b^{2} } )tab dr ds} } + 2h\int\limits_{-1}^{1}{\int\limits_{-1}^{1}{ N_{i}N_{j}ab dr ds} } \) i,j= 1,4

or

\(\ K_{ij}=\int\limits_{-1}^{1}{\int\limits_{-1}^{1}{(K_{x}\frac{\partial N_{i} }{\partial r}\frac{\partial N_{j} }{\partial r}\frac{b}{a} +K_{y}\frac{\partial N_{i} }{\partial s}\frac{\partial N_{j} }{\partial s}\frac{a}{b} )t dr ds} } + 2h\int\limits_{-1}^{1}{\int\limits_{-1}^{1}{ N_{i}N_{j}ab dr ds} } \) i,j= 1,4

Assuming that \(\ K_{x} \) and \(\ K_{y} \) are constants, we have
\(\ K_{ij}=K_{x}t\frac{b}{a}\int\limits_{-1}^{1}{\int\limits_{-1}^{1}{\frac{\partial N_{i} }{\partial r}\frac{\partial N_{j} }{\partial r}}dr ds }+ K_{y}t\frac{a}{b}\int\limits_{-1}^{1}{\int\limits_{-1}^{1}{\frac{\partial N_{i} }{\partial s}\frac{\partial N_{j} }{\partial s}}dr ds }+ 2hab\int\limits_{-1}^{1}{\int\limits_{-1}^{1}{N_{i}N_{j} }dr ds } \) i,j= 1,4
The required partial derivatives are
\(\ \frac{\partial N_{1} }{\partial r}=\frac{1}{4}(s-1) \) \(\ \frac{\partial N_{1} }{\partial s}=\frac{1}{4}(r-1) \)

 

\(\ \frac{\partial N_{2} }{\partial r}=\frac{1}{4}(1-s) \) \(\ \frac{\partial N_{2} }{\partial s}=-\frac{1}{4}(1+ r) \)

 

\(\ \frac{\partial N_{3} }{\partial r}=\frac{1}{4}(1+ s) \) \(\ \frac{\partial N_{3} }{\partial s}=\frac{1}{4}(1+ r) \)

 

\(\ \frac{\partial N_{4} }{\partial r}=-\frac{1}{4}(1+ s) \) \(\ \frac{\partial N_{4} }{\partial s}=\frac{1}{4}(1-r) \)
Substituting numerical values (noting that a = b), we obtain, for example,

\(\ K_{11}=20\int\limits_{-1}^{1}{\int\limits_{-1}^{1}{\left[\frac{1}{16}(s-1)^{2}+\frac{1}{16}(r-1)^{2} \right](\frac{0.5}{12} ) }dr ds }+ 2(50)\int\limits_{-1}^{1}{\int\limits_{-1}^{1}{\frac{1}{16}(1-r)^{2}(1-rs)^{2}(\frac{0.5}{12} )^{2} }dr ds } \)

Integrating ﬁrst on r,

\(\ K_{11}=\frac{20(0.5)}{16(12)}\int\limits_{-1}^{1}{(s-1)^{2}r\mid ^{1}_{-1}+ \frac{(r-1)^{3} }{3}\mid ^{1}_{-1}ds }-\frac{100}{16}(\frac{0.5}{12} )^{2}\int\limits_{-1}^{1}{(1-s)^{2}\frac{(1-r)^{3} }{3} \mid ^{1}_{-1}ds} \)

or

\(\ K_{11}=\frac{20(0.5)}{16(12)}\int\limits_{-1}^{1}{\left[(s-1)^{2}(2)+ \frac{8}{3} \right] ds}+ \frac{100}{16}(\frac{0.5}{12} )^{2}\int\limits_{-1}^{1}{(1-s)^{2}\frac{8}{3} ds } \)

Then, integrating on s, we obtain

\(\ K_{11}=\frac{20(0.5)}{16(12)}\left(\frac{2(s-1)^{3} }{3}+ \frac{8}{3}s \right)\mid ^{1}_{-1} -\frac{100}{16}\left(\frac{0.5}{12} \right) ^{2}\left(\frac{8}{3} \right)\left(\frac{(1-s)^{3} }{3} \right) ^{1}_{-1} \)

or

\(\ K_{11}=\frac{20(0.5)}{16(12)}\left(\frac{16}{3}+ \frac{16}{3} \right) + \frac{100}{16}\left(\frac{0.5}{12} \right) ^{2}\left(\frac{8}{3} \right)\left(\frac{8}{3} \right) =0.6327 Btu/ (hr-°F) \)

The analytical integration procedure just used to determine \(\ K_{11} \) is not the method used by ﬁnite element software packages; instead, numerical methods are used, primarily the Gauss quadrature procedure discussed in Chapter 6. If we examine the terms in the integrands of the equation deﬁning \(\ K_{ij} \) , we ﬁnd that the integrands are quadratic functions
of r and s. Therefore, the integrals can be evaluated exactly by using two Gauss points in r and s. Per Table 6.1, the required Gauss points and weighting factors are \(\ r_{i},s_{j}=\pm 0.57735 \) and \(\ W_{i},W_{j}=1.0 \) , i,j= 1,2 . .Using the numerical procedure for \(\ K_{11} \) ,we write

\(\ K_{11}=K_{x}t\frac{b}{a}\int\limits_{-1}^{1}{\int\limits_{-1}^{1}{\frac{1}{16}(s-1)^{2}dr ds } } +K_{y}t\frac{b}{a}\int\limits_{-1}^{1}{\int\limits_{-1}^{1}{\frac{1}{16}(r-1)^{2}dr ds } } + 2hab\int\limits_{-1}^{1}{\frac{1}{16}(r-1)^{2}(s-1)^{2} dr ds} \)

 

\(\ =K_{x}t\frac{b}{a}\sum\limits_{i=1}^{2}{\sum\limits_{j=1}^{2}{\frac{1}{16}W_{i}W_{j}(s_{j}-1 )^{2} } } + K_{y}t\frac{a}{b}\sum\limits_{i=1}^{2}{\sum\limits_{j=1}^{2}{\frac{1}{16}W_{i}W_{j}(r_{i}-1 )^{2} } } + 2hab\sum\limits_{i=1}^{2}{\sum\limits_{j=1}^{2}{\frac{1}{16}W_{i}W_{j} (1-r_{i})^{2}(1-s_{j})^{2}} } \)

and, using the speciﬁed integration points and weighting factors, this evaluates to

\(\ K_{11}=K_{x}t\frac{b}{a}(\frac{1}{3} )+K_{y}t\frac{a}{b}(\frac{1}{3} )+ 2hab(\frac{4}{9} ) \)

It is extremely important to note that the result expressed in the preceding equation is the correct value of \(\ K_{11} \) for any rectangular element used for the two-dimensional heat conduction analysis discussed in this section. The integrations need not be repeated for each element; only the geometric quantities and the conductance values need be substituted to obtain the value. Indeed, if we substitute the values for this example, we obtain

\(\ K_{11} \) = 0.6327 Btu/(hr-°F)

as per the analytical integration procedure.

Proceeding with the Gaussian integration procedure (calculation of some of these terms are to be evaluated as end-of-chapter problems), we ﬁnd

\(\ K_{11}=K_{22}=K_{33}=K_{44}= 0.6327 Btu/(hr-°F) \)

Why are these values equal?

The off-diagonal terms (again using the numerical integration procedure) are calculated as

\(\ K_{12}= − 0.1003 \)

 

\(\ K_{13}= − 0.2585 \)

 

\(\ K_{14}= − 0.1003 \)

 

\(\ K_{23}= − 0.1003 \)

 

\(\ K_{24}= − 0.2585 \)

 

\(\ K_{34}= − 0.1003 \)

Btu/(hr-°F), and the complete element conductance matrix is

\(\ \left[K^{(e)} \right]=\left [ \begin{matrix} 0.6327 & −0.1003 & −0.2585 & −0.1003 \\ −0.1003 & 0.6327 & −0.1003 & −0.2585 \\ −0.2585 & −0.1003 & 0.6327 & −0.1003 \\ −0.1003 & −0.2585 & −0.1003 & 0.6327 \end{matrix} \right ] Btu/(hr-°F) \)
