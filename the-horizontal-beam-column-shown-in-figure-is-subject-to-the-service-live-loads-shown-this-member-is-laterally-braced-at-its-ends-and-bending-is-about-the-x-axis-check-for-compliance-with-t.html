Guest Question.

The horizontal beam–column shown in Figure is subject to the service live loads shown. This member is laterally braced at its ends, and bending is about the x-axis. Check for compliance with the AISC Specification. \(K_x = K_y = 1.0.\)

Admin
Solution

The factored axial load is
\(P_u = 1.6(28) = 44.8 kips\)
The factored transverse loads and bending moment are
\(Q_u = 1.6(28) = 44.8 kips\)

\(w_u = 1.2(0.035) = 0.042 kips/ft\)

\(M_u =\frac {44.8(10)}{4}{0.042(10)^{2}}{8} = 112.5 ft-kips\)
This member is braced against sidesway, so \(M_{ℓt} = 0.\)
Compute the moment amplification factor: For a member braced against sidesway and transversely loaded, \(C_m \)can be taken as 1.0. A more accurate value can be found in the Commentary to AISC Appendix 8:
\(C_{m} = 1 + \Psi\left ( \frac{\alpha P_{r}}{P_{e1}} \right ) \) (AISC Equation C-A-8-2)
From Commentary Table C-A-8.1, \(\Psi = -0.2 \)for the support and loading conditions
of this beam–column. For the axis of bending,
\(P_{e1} =\frac{\pi ^{2}EI}{(L_{c1})^{2}}= \frac{\pi ^{2}EI_{x}}{(K_{x}L)^{2}}= \frac{\pi ^{2}(29,000)(127)}{(10 \times 12)^{2}} = 2524 kips\)

 

\(C_{m} =1 + \Psi\left ( \frac{\alpha P_{r}}{P_{e1}} \right )= 1 – 0.2\left ( \frac{1.00\alpha P_{u}}{P_{e1}} \right ) = 1 – 0.2\left ( \frac{44.8}{2524} \right ) = 0.9965\)
The amplification factor is

 

\(B_1 =\frac {C_m}{1 – (\alpha P_{r}/P_{e1})}= \frac {C_m}{1 – (1.00P_{u}/P_{e1})}= 0.9965\)
The amplified bending moment is

\(M_{u} = B_{1}M_{nt} + B_{2}M_{ℓt} = 1.015(112.5) + 0 = 114.2 ft-kips\)
From Manual Table 6-2, for\( L_b\) = 10 ft and \(C_b = 1,\)

\(\phi _{b}M_{n} = 123 ft-kips\)
Because the beam weight is very small in relation to the concentrated live load,\( C_{b}\) may be taken from Figure c as 1.32. This value results in a design moment of
\(\phi _{b}M_{n} = 1.32(123) = 162.4 ft-kips\)
This moment is greater than \(\phi _{b}M_{p} = 130 \)ft-kips, so the design strength must be limited to this value. Therefore,
\(\phi _{b}M_{n} = 130 ft-kips\)
Check the interaction formula: From Manual Table 6-2, for \(L_{c} = 10 ft,\)

\(\phi _{c}P_{n} = 359 kips\)

\(\frac{P_{u}}{\phi _{c}P_{n}}= \frac{44.8}{359} = 0.1248 < 0.2 \therefore \) Use Equation 6.4 (AISC Equation Hl-lb).

\(\frac{P_{u}}{2\phi _{c}P_{n}}+\left ( \frac{M_{ux}}{\phi _{b} M_{nx}} + \frac{M_{uy}}{\phi _{b} M_{ny}} \right )= \frac{0.1248}{2}+\left ( \frac{114.2}{130}+0 \right )= 0.941 < 1.0 (OK)\)
A W8 \(\times\) 35 is adequate.
The applied axial load is
\(P_a = 28 kips\)
The applied transverse loads are
\(Q_a = 28 kips and w_a = 0.035 kips/ft\)
and the maximum bending moment is
\(M_{nt} = \frac{28(10)}{4}+\frac{0.035(10)^{2}}{8} = 70.44 ft-kips\)
The member is braced against end translation, \(so M_{ℓt} = 0.\)
Compute the moment amplification factor: For a member braced against sidesway and transversely loaded, \(C_m \)can be taken as 1.0. A more accurate value can be found in the Commentary to AISC Appendix 8:
\(C_{m} = 1 + \Psi\left ( \frac{\alpha P_{r}}{P_{e1}} \right )\) (AISC Equation C-A-8-2)
From Commentary Table C-A-8.1, \(\Psi\)= -0.2 for the support and loading conditions of this beam–column. For the axis of bending,
\(P_{e1} =\frac{\pi ^{2}EI}{(L_{c1})^{2}}= \frac{\pi ^{2}EI_{x}}{(K_{x}L)^{2}}= \frac{\pi ^{2}(29,000)(127)}{(10 \times 12)^{2}} = 2524 kips\)

 

\(C_{m} =1 + \Psi\left ( \frac{\alpha P_{r}}{P_{e1}} \right )= 1 – 0.2\left ( \frac{1.60\alpha P_{a}}{P_{e1}} \right ) = 1 – 0.2\left ( \frac{1.60 \times 28}{2524} \right ) = 0.9965\)

 

\(B_1 =\frac {C_m}{1 – (\alpha P_{r}/P_{e1})}= \frac {C_m}{1 – (1.60P_{u}/P_{e1})}= \frac {0.9965}{1- (1.60 \times 28/2524)}= 1.015\)

 

\(M_a = B_{1}M_{nt} = 1.015(70.44) = 71.50 ft-kips\)
From Table 6-2 with \(C_{b} \)= 1.0 and \(L_{b} \)= 10 feet, the moment strength is
\(\frac {M_{n}}{\Omega _{b}}= 82.0 ft-kips\)
Because the beam weight is very small in relation to the concentrated live load, \(C_{b}\) may be taken from Figure c as 1.32. This results in an allowable moment of
\(\frac {M_{n}}{\Omega _{b}}= 1.32(82.0) = 108.2 ft-kips\)
This result is larger than\(\frac {M_{p}}{\Omega _{b}}= 86.6; therefore, use\frac {M_{n}}{\Omega _{b}}=\frac {M_{p}}{\Omega _{b}}= 86.6 ft-kips.\)
Compute the axial compressive strength: From Table 6-2, for\( L_{c} = 10 ft,\)
\(\frac {P_{n}}{\Omega _{c}} =238 kips\)
Determine which interaction formula to use:
\(\frac {P_{a}}{P_{n}/\Omega _{c}}= \frac {28}{238} = 0.1176 < 0.2 \therefore \)Use Equation 6.6 (AISC Equation H1-1b).

 

\(\frac {P_{a}}{2P_{n}/\Omega _{c}}+\left ( \frac{M_{ax}}{ M_{nx}/\Omega _{b}} + \frac{M_{ay}}{ M_{ny}\Omega _{b}} \right )=\frac {0.1176}{2}+\left ( \frac {71.50}{86.6 }+0\right )\)
= 0.884 <1.0 (OK)
The W8 \(\times\) 35 is adequate.
