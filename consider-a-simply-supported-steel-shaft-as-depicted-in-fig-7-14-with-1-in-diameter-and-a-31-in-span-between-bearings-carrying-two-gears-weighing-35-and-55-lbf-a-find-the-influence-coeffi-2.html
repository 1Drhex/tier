Guest Question.

Consider a simply supported steel shaft as depicted in Fig. 7–14, with 1 in diameter and a 31-in span between bearings, carrying two gears weighing 35 and 55 lbf.
(a) Find the influence coefficients.
(b) Find \(∑wy\) and \(∑wy^{2}\) and the first critical speed using Rayleigh’s equation, Eq. (7–23).

\(ω_{1} = \sqrt { \frac { g \sum{w_{i} y_{i}}} { \sum {w_{i} y^{2}_{i}}}}\) (7–23)

(c) From the influence coefficients, find \(ω_{11}\) and \(ω_{22}\).
(d) Using Dunkerley’s equation, Eq. (7–32), estimate the first critical speed.

\(\frac {1}{ω^{2}_{1}} \dot{=} \sum\limits_{1=1}^{n}{\frac {1}{ω^{2}_{i i}}}\) (7-32)

(e) Use superposition to estimate the first critical speed.
( f ) Estimate the shaft’s intrinsic critical speed. Suggest a modification to Dunkerley’s equation to include the effect of the shaft’s mass on the first critical speed of the attachments.

Admin
Solution

(a)

\(I =\frac {πd^{4}}{64} =\frac {π(1)^{4}}{64} = 0.049 09 in^{4}\)

\(6E I l = 6(30)10^{6}(0.049 09)31 = 0.2739(10^{9}) lbf · in^{3}\)

From Eq. set (7–24),

\(δ_{i j}=\begin{cases}\frac {b_{j} x_{i}}{6E I l} (l^{2} − b^{2}_{j} − x^{2}_{i}) & x_{i} ≤ a_{i} \\\frac {a_{j} (l − x_{i} )}{6E I l}(2lx_{i} − a^{2}_{j} − x^{2}_{i}) & x_{i} > a_{i}\end{cases}\) (7-24)

\(δ_{11} =\frac {24(7)(31^{2} − 24^{2} − 7^{2})}{0.2739(10^{9})} = 2.061(10^{−4}) in/lbf\)
\(δ_{22} =\frac {11(20)(31^{2} − 11^{2} − 20^{2})}{0.2739(10^{9})} = 3.534(10^{−4}) in/lbf\)
\(δ_{12} = δ_{21} =\frac {11(7)(31^{2} − 11^{2} − 7^{2})}{0.2739(10^{9})} = 2.224(10^{−4}) in/lbf \)

2.224(\(10^{-4}\)) 2.061(\(10^{-4}\)) 1

\(y_{1} = w_{1}δ_{11} + w_{2}δ_{12} = 35(2.061)10^{−4} + 55(2.224)10^{−4} = 0.019 45 in\)
\(y_{2} = w_{1}δ_{21} + w_{2}δ_{22} = 35(2.224)10^{−4} + 55(3.534)10^{−4} = 0.027 22 in\)

(b)

\(\sum{w_{i} y_{i} }= 35(0.019 45) + 55(0.027 22) = 2.178 lbf · in\)

\(\sum{w_{i} y^{2}_{i} }=35(0.019 45)^{2} + 55(0.027 22)^{2} = 0.053 99 lbf · in^{2}\)

\(ω = \sqrt{\frac {386.1(2.178)}{0.053 99}} = 124.8 rad/s , or 1192 rev/min\)

(c)

\(\frac{1}{\omega_{11}^{2}}=\frac{w_{1}}{g} \delta_{11}\)

\(ω_{11} = \sqrt {\frac {g}{w_{1}δ_{11}}} =\sqrt {\frac {386.1}{35(2.061)10^{−4}} }= 231.4 rad/s, or 2210 rev/min\)

\(ω_{22} = ]\sqrt {\frac {g}{w_{2}δ_{22}}} = \sqrt { \frac {386.1}{55(3.534)10^{−4}}} = 140.9 rad/s, or 1346 rev/min\)

(d)

\(\frac {1}{ω^{2}_{1}} \dot {=} \sum { \frac {1}{ω^{2}_{i i}} }=\frac {1}{231.4^{2}} + \frac {1}{140.9^{2}} = 6.905(10^{−5})\) (1)

\(ω_{1} \dot {=} \sqrt{ \frac {1}{6.905(10^{−5})} } = 120.3 rad/s, or 1149 rev/min\)

which is less than part b, as expected.

(e) From Eq. (7–24),

\(δ_{cc} = \frac {b_{cc} x_{cc} (l^{2} − b^{2}_{cc} − x^{2}_{cc})}{6E I l} = \frac {15.5(15.5)(31^{2} − 15.5^{2} − 15.5^{2})}{0.2739(10^{9})}\)

\(= 4.215(10^{−4}) in/lbf\)

From Eq. (7–33),

\(w_{1c} = w_{1} \frac {δ_{11}}{δ_{cc}}\) (7–33)

\(w_{1c} = w_{1} \frac {δ_{11}}{δ_{cc}} = 35 \frac {2.061(10^{−4})}{4.215(10^{−4})} = 17.11 lbf\)
\(w_{2c} = w_{2} \frac {δ_{22}}{δ_{cc}} = 55 \frac {3.534(10^{−4})}{4.215(10^{−4})} = 46.11 lbf\)

\(ω = \sqrt { \frac {g}{δ_{cc} \sum {w_{ic}} } }= \sqrt { \frac {386.1}{4.215(10^{−4})(17.11 + 46.11)}} = 120.4 rad/s, or 1150 rev/min\)

which, except for rounding, agrees with part d, as expected.

( f ) For the shaft, \(E = 30(10^{6})\) psi, γ = 0.282 lbf/\(in^{3}\), and \(A = π(1^{2})/4 = 0.7854 in^{2}\).Considering the shaft alone, the critical speed, from Eq.(7–22), is

\(ω_{1} =\left( \frac {π}{l}\right)^{2} \sqrt {\frac{EI}{m}} = \left( \frac {π}{l}\right)^{2} \sqrt {\frac {gE I}{Aγ}} \) (7.22)

\(ω_{s}= \left( \frac {π}{l}\right)^{2} \sqrt {\frac{gE I}{Aγ}}= \left( \frac {π}{31}\right)^{2} \sqrt {\frac {386.1(30)10^{6}(0.049 09)}{0.7854(0.282)}}\)

= 520.4 rad/s, or 4970 rev/min

We can simply add \(1/ω^{2}_{s}\) to the right side of Dunkerley’s equation, Eq.(1), to include the shaft’s contribution,

\(\frac {1}{ω^{2}_{1}} \dot{=} \frac {1}{520.4^{2}} + 6.905(10^{−5}) = 7.274(10^{−5})\)

\(ω_{1} \dot{=} 117.3 rad/s, or 1120 rev/min\)

which is slightly less than part d, as expected.

The shaft’s first critical speed \(ω_{s}\) is just one more single effect to add to Dunkerley’s equation. Since it does not fit into the summation, it is usually written up front.

\(\frac {1}{ω^{2}_{1}} \dot{=} \frac {1}{ω^{2}_{s}} +\sum\limits_{i=1}^{n}{\frac {1}{ω^{2}_{ii}}}\) (7-34)

Common shafts are complicated by the stepped-cylinder geometry, which makes the influence-coefficient determination part of a numerical solution.
