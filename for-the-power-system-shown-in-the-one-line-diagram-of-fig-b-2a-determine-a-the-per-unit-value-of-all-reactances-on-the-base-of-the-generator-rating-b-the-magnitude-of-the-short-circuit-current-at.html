Guest Question.

For the power system shown in the one-line diagram of Fig. B-2(a) determine:

a.The per-unit value of all reactances on the base of the generator rating.

b.The magnitude of the short-circuit current at points A and B. (Neglect the motor’s short-circuit current contributions.)

Admin
Solution

a.The base values are given by the generator rating.

\(S_{b}= \)50 MVA \(V_{b_{L-L}}= \)13.8 kV

\( I_{b} = \frac{50\times 10^{3}}{\sqrt{3}(13.8) } = 2091.85 A \)

The base impedance at 13.8 kV is

\( Z_{b}=\left(\frac{13.8}{50} \right)^{2} = 3.81 \Omega / phase \)

Generator. The per-unit value of the generator’s reactance is

\( X_{d} = \frac{0.95}{3.81} = \underline{\underline{0.25 pu} } \)

Transformer. Referring the transformer’s reactance to the low-voltage winding, we have

\( X_{eL}= \left(\frac{N_{1}}{N_{2}} \right)^{2}\)

=\( 17.85\left(\frac{13.8}{69} \right)^{2} \)

=\(\underline{\underline{0.714 \Omega / phase} }\)

In per unit,

\(X_{e} = \frac{0.714}{3.81}= \underline{\underline{0.187 pu} }\)

Cable. Referring the cable’s reactance to the low-voltage winding of the transformer, we have

\( X_{c_{L}} = 41.4\left(\frac{13.8}{69} \right)^{2} = 1.656 \Omega / phase \)

In per unit,

\( X_{c}= 1.656/3.81 =\underline{\underline{0.435 pu} } \)

The per-phase equivalent circuit, using per-unit values, is shown in Fig. B-2(b).

b.The magnitude of the short-circuit current at A is:

Using ohmic values,

\( I = \left|\frac{V_{L-N}}{Z} \right|= \frac{13,800/\sqrt{3} }{0.95} = \underline{\underline{8.39 k A} } \)

Using per-unit values,

\( I = \left|\frac{V}{Z} \right|= \frac{1.0}{0.25} = 4.0 pu \)

=\( 4(2091.85)= \underline{\underline{8.39 kA} } \)

The magnitude of the short-circuit current at B is:

Using ohmic values,

\( I = \frac{V_{L-N}}{Z} = \frac{13,800\sqrt{3} }{(0.95)+ 0.714} = \underline{\underline{4.79 kA} }\)

Using per-unit values,

\( I = \left|\frac{V}{Z} \right|= \frac{1.0}{0.25+0.187} \)

= 2.29 pu

= 2.29(2091.85)

=\( \underline{\underline{4.79 k A} } \)

Please note that when we speak of a short circuit, we usually mean a solid, three-phase short circuit
