Guest Question.

The Hamiltonian for a charge q in a one-dimensional harmonic oscillator in a classical (not quantized) electric field, which we will take here to have the time-dependence \(|E|=|E_{0}|e^{-t/\pmb{\tau} }\) such as would arise if the oscillator were situated between the plates of a discharging parallel-plate capacitor, is given by

\(\hat{H}=\frac{\hat{P} _{x} ^{2} }{2m}+\frac{1}{2}mω^{2}\hat{x} ^{2}-q\hat{x}|E_{0}|x^{-t/\tau }\)

 

Choose

\( \hat{H} _{0}=\frac{\hat{P} ^{2} _{x} }{2m}+\frac{1}{2}mω^{2}\hat{x} ^{2}\)

 

and

\( \hat{H} _{1}(t)=-q\hat{x}|E_{0}|e^{-t/\pmb{\tau} }\)

 

Suppose that at t = 0 the oscillator is in then= 0 ground state. What is the probability that the oscillator is in an excited state at \(t = \infty\) ?

Admin
Solution

From (14.96):

\(c_{f}(t) = \delta _{fi} – \frac{i}{\hbar} \int_{0}^{t}{dt^{\prime } e^{i(E^{(0)}_{f}-E^{(0)}_{i})t^{\prime}/\hbar} \left\langle E^{(0)}_{f}\left|\hat{H}_{1}(t^{\prime }) \right| E^{(0)}_{i} \right\rangle + \cdot \cdot \cdot } \)

 

\(c_{n}(\infty )=\frac{iq|E_{0}| }{\hbar }\int_{0}^{\infty }{dt^{\prime }e^{inωt^{\prime } }e^{-{\prime }/\tau }\left\langle n|\hat{x}|0 \right\rangle }\) \( n\neq 0\)

 

Expressing the position operator in terms of raising and lowering operators, we find

\(c_{n}(\infty )=\frac{iq|E_{0}| }{\hbar }\sqrt{\frac{\hbar }{2mω} }\int_{0}^{\infty }{dt^{′}e^{inwt^{′} } e^{-t^{′}/τ }\left\langle n|(\hat{a} +\hat{a} ^{\dagger } )|0\right\rangle }\)

 

\(=\frac{iq|E_{0}| }{\hbar }\sqrt{\frac{\hbar }{2mω}}\int_{0}^{\infty }{dt^{′} }e^{inωt^{′} }x^{-t^{′}/\tau }\left\langle n|1\right\rangle\)

 

Thus only \(c_{1}(\infty ) \) is nonzero:

\(c_{1}(\infty ) =\frac{iq|E_{0}| }{\hbar }\sqrt{\frac{\hbar }{2mω} }\int_{0}^{\infty }{dt^{′}e^{iωt^{′} }e^{-t/\tau } }\)

 

\(=\frac{iq|E_{0}| }{\hbar }\sqrt{\frac{\hbar }{2mw} }\frac{\tau }{1-iω\tau }\)

 

The probability of the oscillator making the transition from the ground state to the first excited state is given by

\(|c_{1}(\infty )|^{2}=\frac{(q|E_{0}|\tau )^{2} }{2m\hbar ω}\frac{1}{1+ω^{2}\tau ^{2} }\)

 

This result is our first hint of how a selection rule might arise: Since the position operator in the peiturbing electric dipole Hamiltonian \( \hat{H} _{1} \) involves a single raising and a single lowering operator, only transitions in which the quantum number n of the oscillator changes by 1 are permitted through first order in perturbation theory.
