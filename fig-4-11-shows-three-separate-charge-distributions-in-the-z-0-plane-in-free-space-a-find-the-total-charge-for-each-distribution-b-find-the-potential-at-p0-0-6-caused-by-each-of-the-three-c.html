Guest Question.

Fig 4.11 shows three separate charge distributions in the z = 0 plane in free space. (a) Find the total charge for each distribution. (b) Find the potential at P(0, 0, 6) caused by each of the three charge distributions acting alone. (c) Find \({ V }_{ P }\).

Admin
Solution

a)Line charge along the y axis:
\({ Q }_{ 1 }=\int _{ 3 }^{ 5 }{ } \pi \times { 10 }^{ -9 }dy=2\pi \times { 10 }^{ -9 }C=6.28nC\)
Line charge in an arc at radius \(\rho\) = 3:
\({ Q }_{ 2 } =\int _{ \mathring {10 } }^{ \mathring { 70} }{ \left( { { 10 }^{ -9 } } \right) 3d\phi =4.5 } { \times 10 }^{ -9 }\left( 70-10 \right) \frac { 2\pi }{ 360 } = 4.71{ \times 10 }^{ -9 }C = 4.71 nC\)
sheet charge :
\({ Q }_{ 3 }=\int _{ \mathring { 10 } }^{ \mathring { 70 } }{ \int _{ 1.6 }^{ 3.5 }{ } \left( { { 10 }^{ -9 } } \right) \rho d\rho d\phi =5.07 } { \times 10 }^{ -9 }C = 5.07 nC\)
b) Line charge along y axis:
\({V }_{P1 }=\int _{ 3 }^{ 5 }{ } \frac { \rho LdL }{ 4\pi { \epsilon }_{ 0 }R } =\int _{ 3 }^{ 5 }{ } \frac { \pi \times { 10 }^{ -9 }dy }{ 4\pi { \epsilon }_{ 0 }\sqrt { { \left( y \right) }^{ 2 }+{ \left( 6 \right) }^{ 2 } } } =\frac { { 10 }^{ 3 } }{ 4\times 8.854 } \ln { \left( y+\sqrt { { \left( y \right) }^{ 2 }+{ \left( 6 \right) }^{ 2 } } \right) { | }_{ 3 }^{ 5 } } = 7.83 V\)
Line charge in an arc a radius \(\rho\) = 3:
\({V }_{P2 }=\int _{ \mathring { 10 } }^{ \mathring { 70 } }{ } \frac { \left( 1.5\times { 10 }^{ -9 } \right) 3d\phi }{ 4\pi { \epsilon }_{ 0 }\sqrt { { \left( 3 \right) }^{ 2 }+{ \left( 6 \right) }^{ 2 } } } =\frac { { Q }_{ 2 } }{ 4\pi { \epsilon }_{ 0 }\sqrt { 45 } } =6.31V\)
Sheet charge:
\({V }_{P3 } = \int _{ \mathring { 10 } }^{ \mathring { 70 } } \int _{ 1.6 }^{ 3.5 } \frac { \left( { 10 }^{ -9 } \right) \rho d\rho \phi }{ 4\pi { \epsilon }_{ 0 }\sqrt { { \left( \rho \right) }^{ 2 }+{ \left( 6 \right) }^{ 2 } } } =\frac { 60\times { 10 }^{ -9 } }{ 4\pi \left( 8.854\times { 10 }^{ -12 } \right) } \left( \frac { 2\pi }{ 360 } \right) \int _{ 1.6 }^{ 3.5 } \frac { \rho d\rho }{ \sqrt { { \left( \rho \right) }^{ 2 }+36 } } =9.42\sqrt { { \left( \rho \right) }^{ 2 }+36 } { | }_{ 1.6 }^{ 3.5 }=6.93V\)
c) This will be the sum of the three results of part b, or
\({V }_{P } = {V }_{P1 } + {V }_{P2 } + {V }_{P3 } = 7.83 + 6.31 + 6.93 = 21.1 V\)
