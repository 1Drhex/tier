Guest Question.

The source current in the circuit in Fig. 7.6-8 is \(i_{S}(t) =\) \(I_{m}\) \(cosω t A\). Find \(w\) and \(k\) such that the current \(i_{y}\) is in phase with \(i_{S}(t)\) and has the same amplitude as that of \(i_{S}(t)\).

Admin
Solution

The phasor equivalent circuit is shown in Fig. 7.6-9.

The dependent source current value is \(= \frac{k R_{u} I_{s}}{R_{a}+R_{b}}\) This source is transformed into a dependent voltage source of value \(=\frac{k R_{a} R_{c} \boldsymbol{I}_{s}}{R_{a}+R_{b}}=\alpha \boldsymbol{I}_{s}\) in series with \(R_{c}\). Source transformation theorem is applicable under sinusoidal steady-state condition. Three meshes and the mesh current phasors are as shown in Fig. 7.6-9. The mesh impedance matrix can be written by inspection. Hence, the mesh equations in matrix form will be as below.

\(\left[\begin{array}{ccc} R_{c}+R+\frac{1}{j \omega C} & -R & 0 \\ -R & 2 R+\frac{1}{j \omega C} & -R \\ 0 & -R & 2 R+\frac{1}{j \omega C} \end{array}\right]\left[\begin{array}{c} I_{1} \\ I_{2} \\ I_{3} \end{array}\right]=\left[\begin{array}{c} -\alpha I_{s} \\ 0 \\ 0 \end{array}\right]\)

 

The determinant of the mesh impedance matrix after some simplification is

\(\Delta_{Z}=\left[R^{3}+3 R_{\mathrm{c}} R^{2}-\frac{R_{\mathrm{c}}+5 R}{(\omega C)^{2}}\right]-j\left[\frac{R\left(6 R+4 R_{\mathrm{c}}\right)}{(\omega C)}-\frac{1}{(\omega C)^{3}}\right]\)

 

We need to solve for \( I_{3}\).

\(I_{3}=\frac{-R^{2} \alpha I_{s}}{\Delta_{Z}}\)

 

We want the angle of \(I_{3}\) to be the same as that of \(I_{s}\). This is possible only if \(ΔZ\) is a negative real number because the numerator is a real negative number. Therefore imaginary part of \(ΔZ\) should go to zero at the \(ω\) we are looking for.

Therefore, \(\left[\frac{R\left(6 R+4 R_{c}\right)}{(\omega C)}-\frac{1}{(\omega C)^{3}}\right]=0 \Rightarrow \omega=\frac{1}{R C} \frac{1}{\sqrt{6+4 \beta}}, \text { where } \beta=\frac{R_{c}}{R}\)

 

We also want the magnitude of the two currents to be the same. Therefore,

\(\left[R^{3}+3 R_{c} R^{2}-\frac{R_{c}+5 R}{(\omega C)^{2}}\right]=-R^{2} \alpha\)

 

Substituting \(\omega=\frac{1}{R C} \frac{1}{\sqrt{6+4 \beta}}, \text { where } \beta=\frac{R_{c}}{R} \text { and } \alpha=\frac{k R_{a} R_{c}}{R_{a}+R_{b}}\) and simplifying, we get the required value of \(k\) as, \(k =\frac{\left(R_{a}+R_{b}\right)}{R_{b}}\left[4 \frac{R_{c}}{R}+23+29 \frac{R}{R_{c}}\right]\)

[This is the small-signal equivalent circuit used to analyse the conditions for sinusoidal oscillation to take place in a Transistor Phase Shift Oscillator circuit that is widely used to generate low power sinusoidal signals up to about \(20 kHz\) frequency.]
