Guest Question.

An elliptic filter is required satisfying the following specifications:

• Passband edge \(\omega_{p}: \sqrt{0.9}\) rad/s

• Stopband edge \(\omega_{a}: 1 / \sqrt{0.9}\) rad/s

• Maximum passband loss \(A_{p}: 0.1\) dB

• Minimum stopband loss \(A_{a}: 50.0\) dB

Form \(H_{N}(s)\).

Admin
Solution

From Eqs. (10.86)–(10.90)

 

\(k^{\prime}=\sqrt{1-k^{2}}\) (10.86)

 

\(q_{0}=\frac{1}{2}\left(\frac{1-\sqrt{k^{\prime}}}{1+\sqrt{k^{\prime}}}\right)\) (10.87)

 

\(q=q_{0}+2 q_{0}^{5}+15 q_{0}^{9}+150 q_{0}^{13}\) (10.88)

 

\(D=\frac{10^{0.1 A_{a}}-1}{10^{0.1 A_{p}}-1}\) (10.89)

 

\(n \geq \frac{\log 16 D}{\log (1 / q)}\) (10.90)

 

\(k=0.9 \quad k^{\prime}=0.435890 \quad q_{0}=0.102330\)

 

\(q=0.102352 \quad D=4,293,090 \quad n \geq 7.92 \quad \text { or } \quad n=8\)

 

From Eqs. (10.91)–(10.99) the transfer-function coefficients in Table 10.1 can be obtained. The corresponding loss characteristic is plotted in Fig. 10.17. The actual value of \(A_{a}\) is 50.82 dB according to Eq. (10.84).

 

\(A_{a}=10 \log \left(\frac{10^{0.1 A_{p}}-1}{16 q^{n}}+1\right)\) (10.84)

 

\(\Lambda=\frac{1}{2 n} \ln \frac{10^{0.05 A_{p}}+1}{10^{0.05 A_{p}}-1}\) (10.91)

 

\(\sigma_{0}=\left|\frac{2 q^{1 / 4} \sum_{m=0}^{\infty}(-1)^{m} q^{m(m+1)} \sinh [(2 m+1) \Lambda]}{1+2 \sum_{m=1}^{\infty}(-1)^{m} q^{m^{2}} \cosh 2 m \Lambda}\right|\) (10.92)

 

\(W=\sqrt{\left(1+k \sigma_{0}^{2}\right)\left(1+\frac{\sigma_{0}^{2}}{k}\right)}\) (10.93)

 

\(\Omega_{i}=\frac{2 q^{1 / 4} \sum_{m=0}^{\infty}(-1)^{m} q^{m(m+1)} \sin \frac{(2 m+1) \pi \mu}{n}}{1+2 \sum_{m=1}^{\infty}(-1)^{m} q^{m^{2}} \cos \frac{2 m \pi \mu}{n}}\) (10.94)

 

\(V_{i}=\sqrt{\left(1-k \Omega_{i}^{2}\right)\left(1-\frac{\Omega_{i}^{2}}{k}\right)}\) (10.95)

 

\(a_{0 i}=\frac{1}{\Omega_{i}^{2}}\) (10.96)

 

\(b_{0 i}=\frac{\left(\sigma_{0} V_{i}\right)^{2}+\left(\Omega_{i} W\right)^{2}}{\left(1+\sigma_{0}^{2} \Omega_{i}^{2}\right)^{2}}\) (10.97)

 

\(b_{1 i}=\frac{2 \sigma_{0} V_{i}}{1+\sigma_{0}^{2} \Omega_{i}^{2}}\) (10.98)

 

\(H_{0}= \begin{cases}\sigma_{0} \prod_{i=1}^{r} \frac{b_{0 i}}{a_{0 i}} & \text { for odd } n \\ 10^{-0.05 A_{p}} \prod_{i=1}^{r} \frac{b_{0 i}}{a_{0 i}} & \text { for even } n\end{cases}\) (10.99)
