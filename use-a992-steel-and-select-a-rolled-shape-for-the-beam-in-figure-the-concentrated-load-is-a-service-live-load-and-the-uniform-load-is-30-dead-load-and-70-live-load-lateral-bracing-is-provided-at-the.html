Guest Question.

Use A992 steel and select a rolled shape for the beam in Figure. The concentrated load is a service live load, and the uniform load is 30% dead load and 70% live load. Lateral bracing is provided at the ends and at midspan. There is no restriction on deflection.

Admin
Solution

Neglect the beam weight and check it later.
\(w_{D} = 0.30(3) = 0.9 kips/ft\)

\(w_{L} = 0.70(3) = 2.1 kips/ft\) \(w_{u} = 1.2(0.9) + 1.6(2.1) = 4.44 kips/ft\)

\(P_{u} = 1.6(9) = 14.4 kips\)
The factored loads and reactions are shown in Figure 5.30. Next, determine themoments required for the computation of C_{b}. The bending moment at a distance x from the left end is
\(M = 60.48x – 4.44\frac{x1}{2}=60.48x – 2.22x^{2} (for x \leq 12 ft)\)

For x = 3 ft, \(M_{A} = 60.48(3) – 2.22(3)^{2} = 161.5 ft-kips\)

For x = 6 ft,\( M_{B} = 60.48(6) – 2.22(6)^{2} = 283.0 ft-kips\)

For x = 9 ft, \(M_{C} = 60.48(9) – 2.22(9)^{2} = 364.5 ft-kips\)

For x = 12 ft,\( M_{max} = M_{u} = 60.48(12) – 2.22(12)^{2} = 406.1 ft-kips\)

 

\(C_{b}=\frac{12.5M_{max}}{2.5M_{max} + 3M_{A} + 4M_{B} + 3M_{C}}\)

 

\(=\frac{12.5(406.1)}{2.5(406.1) + 3(161.5) + 4(283.0) + 3(364.5)}= 1.36\)
Enter the charts with an unbraced length \(L_{b}\) = 12 ft and a bending moment of
\(\frac{M_{u}}{C_{b}}= \frac{406.1}{1.36}=299 ft-kips\)
Try W21 \(\times\) 48:
\(\phi _{b}M_{n}\) = 311 ft-kips (for \(C_{b} \)= 1)
Since \(C_{b}\) = 1.36, the actual design strength is 1.36(311) = 423 ft-kips. But the design strength cannot exceed \(\phi _{b}M_{p}\), which is only 398 ft-kips (obtained from the chart),so the actual design strength must be taken as
\(\phi _{b}M_{n} =\phi _{b}M_{p} = 398 ft-kips < M_{u} = 406.1 ft-kips (N.G.)\)
For the next trial shape, move up in the charts to the next solid curve and try
W18 \(\times\) 55. For \(L_{b}\) = 12 ft, the design strength from the chart is 335 ft-kips for
\(C_{b} \)= 1. The strength for \(C_{b} \)= 1.36 is
\(\phi _{b}M_{n} = 1.36(335) = 456 ft-kips > \phi _{b}M_{p} = 420 ft-kips\)
\(\therefore \phi _{b}M_{n} = \phi _{b}M_{p} = 420 ft-kips > M_{u} = 406.1 ft-kips (OK)\)
Check the beam weight.
\(M_{u} = 406.1 +\frac{1}{8} (1.2 \times 0.055)(24)^{2} = 411 ft-kips < 420 ft-kips (OK)\)

The maximum shear is
\(V_{u} = 60.48 +\frac{1.2(0.055)}{2}(24) = 61.3 kips\)
From the \(Z_{x}\) tables,
\(\phi _{v}V_{n} = 212 kips > 61.3 kips (OK)\)
Use a W18 \(\times\) 55.
The applied loads are
\(w_{a} = 3 kips/ft and P_{a} = 9 kips\)
The left-end reaction is
\(\frac{w_{a}L + P_{a}}{2}=\frac{3(24) + 9}{2}= 40.5 kips\)
and the bending moment at a distance x from the left end is
\(M = 40.5x – 3x\frac{x1}{2}= 40.5x – 1.5x^{2} (for x \leq 12 ft)\)
Compute the moments required for the computation of Cb:
For x = 3 ft, \(M_{A} = 40.5(3) – 1.5(3)^{2} = 108.0 ft-kips\)
For x = 6 ft, \(M_{B} = 40.5(6) – 1.5(6)^{2} = 189.0 ft-kips\)
For x = 9 ft, \(M_{C} = 40.5(9) – 1.5(9)^{2} = 243.0 ft-kips\)
For x = 12 ft,\( M_{max} = 40.5(12) – 1.5(12)^{2} = 270.0 ft-kips\)

 

\(C_{b}=\frac{12.5M_{max}}{2.5M_{max} + 3M_{A} + 4M_{B} + 3M_{C}}\)

 

\(=\frac{12.5(270)}{2.5(270) + 3(108) + 4(189) + 3(243)}= 1.36 \)
Enter the charts with an unbraced length \(L_{b}\) = 12 ft and a bending moment of
\(\frac{M_{a}}{C_{b}}= \frac{270}{1.36}=199 ft-kips\)
Try W21 \(\times\) 48: For \(C_{b}\) = 1,
\(M_{n}/\Omega _{b} = 207 ft-kips\)
For \(C_{b}\) = 1.36, the actual allowable strength is 1.36(207) = 282 ft-kips, but the strength cannot exceed \(M_{p}/\Omega _{n}\), which is only 265 ft-kips (this can be obtained from the chart), so the actual allowable strength must be taken as

\(\frac{M_{n}}{\Omega _{b}}=\frac{M_{p}}{\Omega _{b}}= 265 ft-kips < M_{a} = 270 ft-kips (N.G.)\)
Move up in the charts to the next solid curve and try W18 \(\times\) 55. For \(L_{b}\) = 12 ft, the allowable strength for \(C_{b}\) = 1 is 223 ft-kips. The strength for \(C_{b}\) = 1.36 is
\(\frac{M_{n}}{\Omega _{b}}=1.36(223) = 303 ft-kips >\frac{M_{p}}{\Omega _{b}}= 280 ft-kips\)
\(\therefore \frac{M_{n}}{\Omega _{b}}=\frac{M_{p}}{\Omega _{b}} =280 ft-kips > M_{a} = 270 ft-kips (OK)\)
Account for the beam weight.
\(M_{a} = 270 +\frac{1}{8}(0.055)(24)^{2} = 274 ft-kips < 280 ft-kips (OK)\)
The maximum shear is
\(V_{a} = \frac{9 + 3.055(24)}{2}= 41.2 kips\)
From the \(Z_{x}\) table (or the uniform load table)
\(\frac{V_{n}}{\Omega _{v}}= 141 kips > 41.2 kips (OK)\)
Answer Use a W18 \(\times\) 55.
