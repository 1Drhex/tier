Guest Question.

Obtain a parallel realization of the transfer function

\(H(z)=\frac{10 z^{4}-3.7 z^{3}-1.28 z^{2}+0.99 z}{\left(z^{2}+z+0.34\right)\left(z^{2}+0.9 z+0.2\right)}\)

using canonic sections.

Admin
Solution

We first need to find the poles of the transfer function. We have

 

\(H(z)=\frac{10 z^{4}-3.7 z^{3}-1.28 z^{2}+0.99 z}{\left(z-p_{1}\right)\left(z-p_{2}\right)\left(z-p_{3}\right)\left(z-p_{4}\right)}\)

 

where

 

\(p_{1}, p_{2}=0.5 \mp j 0.3\)

 

\(p_{3}=-0.4\)

 

\(p_{4}=-0.5\)

 

If we expand H(z)/z into partial fractions, we get

 

\(\frac{H(z)}{z}=\frac{R_{1}}{z-0.5+j 0.3}+\frac{R_{2}}{z-0.5-j 0.3}+\frac{R_{3}}{z+0.4}+\frac{R_{4}}{z+0.5}\)

 

The ith residue of H(z)/z is given by

 

\(R_{i}=\left.\frac{\left(z-p_{i}\right) H(z)}{z}\right|_{z=p_{i}}\)

 

\(R_{i}=\left.\frac{\left(z-p_{i}\right) H(z)}{z}\right|_{z=p_{i}}\)

 

and through routine arithmetic or through the use of MATLAB, we get

 

\(R_{1}=\left.\frac{\left(10 z^{4}-3.7 z^{3}-1.28 z^{2}+0.99 z\right)}{z\left(z-p_{2}\right)\left(z-p_{3}\right)\left(z-p_{4}\right)}\right|_{z=p_{1}}\)

 

\(=\left.\frac{\left(10 z^{3}-3.7 z^{2}-1.28 z+0.99\right)}{\left(z-p_{2}\right)\left(z-p_{3}\right)\left(z-p_{4}\right)}\right|_{z=p_{1}}\)

 

= 1.0

 

Similarly,

 

\(R_{2}=1 \quad R_{3}=3 \quad R_{4}=5\)

 

and thus

 

\(H(z)=\frac{z}{z-0.5+j 0.3}+\frac{z}{z-0.5-j 0.3}+\frac{3 z}{z+0.4}+\frac{5 z}{z+0.5}\)

 

Now, if we combine the first two and the last two partial fractions into second-order transfer functions, we get

 

\(H(z)=\frac{2 z^{2}-z}{z^{2}-z+0.34}+\frac{8 z^{2}+3.5 z}{z^{2}+0.9 z+0.2}\)

 

or

 

\(H(z)=H_{1}(z)+H_{2}(z)\)

 

where

 

\(H_{1}(z)=\frac{2-z^{-1}}{1-z^{-1}+0.34 z^{-2}} \quad \text { and } \quad H_{2}(z)=\frac{8+3.5 z^{-1}}{1+0.9 z^{-1}+0.2 z^{-2}}\)

 

Using canonic sections, the parallel realization shown in Fig. 8.14 can be obtained.
