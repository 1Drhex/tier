Guest Question.

For the single-phase full-bridge fully controlled thyristor rectifier of Fig. 5.1(a), the following specifications are given \(v _{ i }=230 \sqrt{2} \sin (100 \pi t ), R =4 \Omega, L =40 mH , E =80 V\), and firing angle \(\alpha=30^{\circ}\). Calculate the following:

a) The average output current.
b) The rms output current.
c) The power delivered to source E.
d) The input power factor.

 

 

Admin
Solution

Solution

\(\begin{aligned}\varphi &=\text { load impedance angle }=\tan ^{-1}\left(\frac{\omega L}{R}\right)=\tan ^{-1}\left(\frac{2 \pi fL }{ R }\right) \\&=\tan ^{-1}\left(\frac{2 \pi \times 50 \times 0.04}{4}\right)=72.34^{\circ}\end{aligned}\)

 

Since \(\varphi>\alpha\), the output current is continuous.

a) Using Eq. (5.16) the average output voltage is:

 

\(\overline{ V }_{ o }=\frac{1}{\pi} \int_{\alpha}^{\pi+\alpha} \sqrt{2} \widetilde{ V }_{ i } \sin (\omega t ) d (\omega t )=\frac{2 \sqrt{2} \widetilde{ V }_{ i }}{\pi} \cos \alpha=0.9 \widetilde{ V }_{ i } \cos \alpha\) (5.16)

 

\(\overline{ V }_{ o }=\frac{2 \sqrt{2} \widetilde{ V }_{ i }}{\pi} \cos \alpha=\frac{2 \sqrt{2} \times 230}{\pi} \cos 30^{\circ}=179 V\)

 

Since \(v _{ o }= V _{ R }+ v _{ L }+ E\) then the average voltage across the resistor is given by:

 

\(\overline{ V }_{ R }=\overline{ V }_{ o }-\overline{ V }_{ L }- E =\overline{ V }_{ o }- E =\overline{ I }_{ o } R \quad\left(\text { since } \overline{ V }_{ L }=0\right)\)

or

\(\overline{ I }_{ o }=\frac{\overline{ V }_{ o }- E }{ R }=\frac{179-80}{4}=24.8 A\)

 

b) The rms output current is:

 

\(\widetilde{I}_{o}=\sqrt{\bar{I}_{ o }^{2}+\left(\frac{\widehat{ I }_{ o , 2}}{\sqrt{2}}\right)^{2}+\left(\frac{\widehat{ I }_{ o , 4}}{\sqrt{2}}\right)^{2}+\cdots} \approx \sqrt{\overline{ I _{ o }^{-2}+\left(\frac{\widehat{ I }_{ o , 2}}{\sqrt{2}}\right)^{2}}}\)

 

where

\(\overline{ I }_{ o }=\) average value of the output current;

\(\widehat{ V }_{ o , 2}=\) amplitude of the output voltage 2nd harmonic component \(=\sqrt{\hat{ V }_{ o 1,2}^{2}+\hat{ V }_{ o 2,2}^{2}}\)

\(\widehat{ I }_{o,2}=\) amplitude of the output current 2nd harmonic component \(=\frac{\hat{ V }_{ o , 2}}{\sqrt{ R ^{2}+(2 \omega L )^{2}}}\)

From Eqs. (5.17) and (5.18), the following results are obtained:

 

\(\begin{aligned}=& \frac{-2 \sqrt{2} \widetilde{V}_{i}}{\pi} \int_{0}^{\alpha} \frac{[\sin (\omega t-n \omega t)+\sin (\omega t+n \omega t)]}{2} d (\omega t) \\&+\frac{2 \sqrt{2} \widetilde{V}_{i}}{\pi} \int_{\alpha}^{\pi} \frac{[\sin (\omega t-n \omega t)+\sin (\omega t+n \omega t)]}{2} d(\omega t) \\=&\left.\frac{\sqrt{2} \widetilde{V}_{i}}{\pi}\left[\frac{\cos (1-n) \omega t}{1-n}+\frac{\cos (1+n) \omega t}{1+n}\right]\right|_{0} ^{\alpha} \\&-\left.\frac{\sqrt{2} \widetilde{V}_{i}}{\pi}\left[\frac{\cos (1-n) \omega t}{1- n }+\frac{\cos (1+n) \omega t}{1+n}\right]\right|_{\alpha} ^{\pi} \\=& \frac{\sqrt{2} \widetilde{V}_{i}}{\pi}\left[\frac{\cos ( n +1) \alpha}{ n +1}-\frac{\cos ( n -1) \alpha}{ n -1}\right] \quad n =2,4,6, \cdots\end{aligned}\) (5.17)

 

\(\begin{aligned}\widehat{ V }_{ o 2, n }=& \frac{2}{2 \pi} \int_{0}^{2 \pi} v _{ o } \sin ( n \omega t ) d (\omega t )=\frac{2}{\pi} \int_{0}^{\pi} v _{ o } \sin ( n \omega t ) d (\omega t ) \\=& \frac{2}{\pi} \int_{0}^{\alpha}\left(-\sqrt{2} \widetilde{ V }_{ i }\right) \sin (\omega t ) \sin ( n \omega t ) d (\omega t ) \\&+\frac{2}{\pi} \int_{\alpha}^{\pi}\left(\sqrt{2} \widetilde{ V }_{ i }\right) \sin (\omega t ) \sin ( n \omega t ) d (\omega t ) \\=& \frac{2 \sqrt{2} \widetilde{ V }_{ i }}{\pi}\left[\frac{\sin ( n +1) \alpha}{ n +1}-\frac{\sin ( n -1) \alpha}{ n -1}\right] \quad n =2,4,6, \cdots\end{aligned}\) (5.18)

 

\(\begin{aligned}\widehat{ V }_{ o1 , 2} &=\frac{2 \sqrt{2} \widetilde{ V }_{ i }}{\pi}\left[\frac{\cos 3 \alpha}{3}-\cos \alpha\right]=207\left(\frac{\cos 90^{\circ}}{3}-\cos 30^{\circ}\right) \\&=207(0-0.866)=-179.27 V\end{aligned}\)

 

\(\begin{aligned}\widehat{ V }_{o2,2} &=\frac{2 \sqrt{2} \widetilde{ V }_{ i }}{\pi}\left[\frac{\sin 3 \alpha}{3}-\sin \alpha\right]=\frac{2 \sqrt{2} \times 230}{\pi}\left[\frac{\sin 90^{\circ}}{3}-\sin 30^{\circ}\right] \\&=207\left(\frac{1}{3}-0.5\right)=-34.5 V\end{aligned}\)

 

Therefore,

 

\(\widehat{ V }_{ o , 2}=\sqrt{(-34.5)^{2}+(-179.27)^{2}}=182.5 V\)

 

The amplitude of the output current second harmonic component is:

 

\(\widehat{ I }_{ o , 2}=\frac{182.5}{\sqrt{4^{2}+(2 \omega L )^{2}}}=\frac{182.5}{\sqrt{16+\left(2 \times 120 \pi \times 40 \times 10^{-3}\right)^{2}}}=\frac{182.5}{30.4}=6 A\)

 

And its rms value is \(\tilde{ I }_{ o , 2}=\frac{6}{\sqrt{2}}=4.2 A\)

Therefore,

 

\(\widetilde{ I }_{ o }=\sqrt{(24.8)^{2}+(4.2)^{2}}=25.15 A\)

c) The power absorbed by the load dc source is:

\(P _{ E }=\overline{ I }_{ o } E =(24.83)(80)=1986.6 W\)

 

d) Assuming that the rectifier is ideal, the power factor is:

\(\lambda=\frac{P_{i}}{S_{i}}=\frac{P_{o}}{S_{i}}=\frac{\widetilde{I}_{o}^{2} R+P_{E}}{\widetilde{V}_{i} \widetilde{I}_{i}}=\frac{(25.15)^{2}(4)+1986.6}{(230)(25.15)}=0.78\)
