Guest Question.

Calculate the expectation value of E and \(E^{2}\) in the state \(|n_{k,s} 〉\).

Admin
Solution

The electric field operator is given by

\(\hat{E}=-\frac{1}{c}\frac{\partial \hat{A} }{\partial t}\)

 

\( =i\sum\limits_{k,s}{\sqrt{2\pi \hbar w} }\left(\hat{a} _{k,s}\varepsilon (k,s)\frac{e^{i(k.r-wt)} }{\sqrt{V} } – \hat{a}^{\dagger }_{k,s}\varepsilon (k,s)\frac{e^{-i(k.r-wt)} }{\sqrt{V} } \right)\)

 

where we have used ( 14.48)

\(\hat{A}= \sum\limits_{k,s}{c\sqrt{\frac{2\pi \hbar }{w} }\left(\hat{a} _{k,s}\varepsilon (K,s) \frac{e^{i(k.r-wt)} }{\sqrt{V} }+\hat{a} _{k,s} ^{\dagger }\varepsilon (K,s)\frac{e^{-i(k.r-wt)} }{\sqrt{V} } \right) }\)

 

for the vector potential operator\(\hat{A}\).Since

\(\left\langle n_{k,s}|\hat{a} _{k,s} ^{\dagger }|n_{k,s} \right\rangle =\sqrt{n_{k,s}+1 }\left\langle n_{k,s}|(n+1)_{k,s} \right\rangle=0\)

 

and

\( \left\langle n_{k,s}|\hat{a} _{k,s}|n_{k,s} \right\rangle=\sqrt{n_{k,s} }\left\langle x_{k,s}|(n-1)_{k,s} \right\rangle =0\)

 

we see that

\( \left\langle n_{k,s}|\hat{E}|n_{k,s} \right\rangle =0\)

 

Since \(\hat{E} ^{2}\) contains terms of the form \(\hat{a} _{k,s}\hat{a} ^{\dagger } _{k,s}\) and \(\hat{a} ^{\dagger } _{k,s}\hat{a} _{k,s}\), namely a creation operator followed by an annihilation operator and an annihilation operator followed by a creation operator, respectively, \(\left\langle n_{k,s} |\hat{E} ^{2}|n_{k,s} \right\rangle\) does not vanish. As our derivation of the electric field energy (14.37)

\(\frac{1}{8\pi }\int{d^{3}r } \frac{1}{c}\frac{\partial A }{\partial t}.\frac{1}{c} \frac{\partial A}{\partial t} \)

 

\(=\frac{1}{8\pi }\int{d^{3}r\sum\limits_{k,s}{\left(\frac{-iω}{c}c_{k,s}\varepsilon (k,s)\frac{e^{i(k.r-ωt)} }{\sqrt{V} }+\frac{iω}{c}c_{k,s} ^{\ast }\varepsilon (k,s)\frac{e^{-i(k.r-ωt)} }{\sqrt{V} } \right) } }\).

 

\(\sum\limits_{k^′ , s^′ }{\left(\frac{-i ω^′}{c}c_{k^′ ,s^′ }\varepsilon (k^′ ,s^′ )\frac{e^{i(k^′ .r- ω^′ t)} }{\sqrt{V} }+\frac{i ω^′ }{c}c_{k^′ ,s^′} ^{\ast }\varepsilon (k^′ , s^′ )\frac{e^{-i(k^′ . r- ω^′t)} }{\sqrt{V} } \right) }\)

 

illustrates, we need to take into account that the operator \(\hat{E}\) involves a sum over all possible modes, namely over all possible values of k and s. If the only terms in this sum that mattered were the annihilation and creation operators corresponding to the particular k and s that label the state \(|n_{k.s}〉\), we would obtain

\(\left\langle n_{k.s}|\hat{E} ^{2}|n_{k,n} \right\rangle=\frac{2\pi \hslash ω}{V}〈n_{k,s}| \left( \hat{a} _{k,s}\hat{a} ^{\dagger } _{k,s}+\hat{a} ^{\dagger } _{k,s }\hat{a} _{k,s}-\hat{a} _{k,s}\hat{a} _{k,s}e^{2i(k.r-ωt)}-\hat{a} ^{\dagger } _{k,s}\hat{a} ^{\dagger } _{k,s}e^{-2i(k.r-ωt)} \right)|n_{k.s}〉\)

 

\(=\frac{2\pi \hbar ω}{V}\left\langle n_{k,s}|\left(2\hat{a} ^{\dagger } _{k,s }\hat{a} _{k,s}+1\right) |n_{k,s} \right\rangle\)

 

\(=\frac{4\pi }{V}\left(n_{k,s}+\frac{1}{2} \right)\hbar ω\)

 

Thus the uncertainty per mode in the electric field strength in the vacuum is
given by

\(\sqrt{\left\langle 0_{k,s}|\hat{E} ^{2}|0_{k,s} \right\rangle-\left\langle 0_{k,s}|\hat{E}|0_{k,s} \right\rangle ^{2} }=\sqrt{\frac{2\pi }{V}\hbar w }\)

 

Therefore, although \(\left\langle 0_{k,s}|\hat{E}|0_{k,s} \right\rangle=0\), there are nonzero vacuum fluctuations in the electric field strength. It can be argued that these fluctuations are the cause of spontaneous emission of a photon when an atom makes a transition from an excited state. We will examine spontaneous emission in hydrogen in Section 14.7.

\(\int{(\triangledown \times A).dS}=\int{B.dS}\)

 

According to (14.53), the full vacuum state|0〉 is the vacuum for each mode, namely

\(|0〉=|0_{k_{1},s_{1} }〉\otimes |0_{k_{2},s_{2} }〉 \otimes 0_{k_{3},s_{3} }〉\otimes …\) (14.53)

Thus

\(\left\langle 0\left|\hat{E}^{2} \right|0 \right\rangle = \sum\limits_{k^{\prime } , s^{\prime }}{\frac{2\pi }{V} \left\langle 0 \left|\left(2\hat{a}^{\dagger }_{k^{\prime } , s^{\prime }} +1 \right) \hbar \omega ^{\prime } \right| 0 \right\rangle = \frac{4\pi }{V} E_0} \)

 

where \(E_{0}\) is the sum of the zero-point energies for each mode, which is consistent with

\(\left\langle 0|\hat{H}|0 \right\rangle=\frac{1}{8\pi }\int{d^{3}r\left\langle 0|(\hat{E} ^{2}+\hat{B}^{2} )|0\right\rangle }=E_{0}\)

 

Clearly, the electric field pottion or the Hamiltonian is contributing one-half the total zero-point energy.
