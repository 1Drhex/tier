Guest Question.

The infinite-duration signal

\(x(t)=\frac{\sin \omega_{c} t}{\pi t}\)

where \(\omega_{c}=1\) rad/s, is to be converted into a corresponding finite-duration signal \(x_{w}(t)\) by using the rectangular window defined in Eq. (7.18) with τ = 15.0 s. (a) Obtain X( jω), the frequency spectrum of x(t). (b) Repeat part (a) for the modified signal \(x_{w}(t)\). (c) Plot the difference between \(X(j \omega) \text { and } X_{w}(j \omega)\).

\(w_{R}(t)= \begin{cases}1 & \text { for }-\tau \leq t \leq \tau \\ 0 & \text { otherwise }\end{cases}\) (7.18)

Admin
Solution

(a) From Table 6.2, we have

 

\(X(j \omega)=p_{2 \omega_{c}}(\omega)\) (7.21)

 

where

 

\(p_{2 \omega_{c}}(\omega)= \begin{cases}1 & \text { for }|\omega|<\omega_{c} \\ 0 & \text { otherwise }\end{cases}\)

 

(b) From Eqs. (7.19), (7.20b), and (7.21), the frequency spectrum of the modified signal is obtained as

 

\(W_{R}(j \omega)=\frac{2 \sin \omega \tau}{\omega}\) (7.19)

 

\(=\frac{1}{2 \pi} \int_{-\infty}^{\infty} X(j \varpi) W(j \omega-j \varpi) d \varpi\) (7.20b)

 

\(X_{w}(j \omega)=\frac{1}{2 \pi} \int_{-\infty}^{\infty} X(j \varpi) W_{R}(j \omega-j \varpi) d \varpi\)

 

\(=\frac{1}{\pi} \int_{-\infty}^{\infty} p_{2 \omega_{c}}(\varpi) \frac{\sin (\omega-\varpi) \tau}{\omega-\varpi} d \varpi\)

 

\(=\frac{1}{\pi} \int_{-\omega_{c}}^{\omega_{c}} \frac{\sin (\omega-\varpi) \tau}{\omega-\varpi} d w\) (7.22)

 

The real integral in Eq. (7.22) can be evaluated by means of some numerical formula, for example, Simpson’s one-third rule given by

 

\(\int_{x_{1}}^{x_{m+1}} f(x) d x=\frac{\Delta x}{3}\left(y_{1}+4 \sum_{i \text { even }}^{m} y_{i}+2 \sum_{i \text { odd }}^{m-1} y_{i}+y_{m+1}\right)\) (7.23)

 

where m is assumed to be an even integer.

The graphical construction for the convolution integral of Eq. (7.22) for τ = 15.0 s and \(\omega_{c}=1\) rad/s is illustrated in Fig. 7.15. The spectrum of \(x_{w}(t), \text { that is, } X_{m}(j \omega)\) is illustrated in Fig. 7.15e.

(c) The difference between \(X_{w}(\omega) \text { and } X(\omega)\), which is plotted in Fig. 7.15 f , is seen to oscillate about zero and becomes significantly larger at frequencies close to the band edges \(\pm \omega_{c}\). The oscillations in \(X_{w}(\omega)\) are caused by the side lobes in the spectrum of the window, as was anticipated earlier. The larger difference near the band edges is due to the finite main-lobe width and it is caused by the fact that convolution tends to smooth the value of the function in the region of a discontinuity. If the ripple ratio and main-lobe width of the window function were decreased, the spectrum of the finite-duration signal \(x_{w}(t)\) would tend to approach that of the infinite-duration signal x(t).
