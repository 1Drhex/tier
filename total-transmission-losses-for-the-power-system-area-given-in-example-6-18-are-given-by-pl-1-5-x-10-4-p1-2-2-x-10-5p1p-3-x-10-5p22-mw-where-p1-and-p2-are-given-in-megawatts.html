Guest Question.

Economic dispatch solution including generator limits and line losses

Total transmission losses for the power system area given in are given by

\(P_{L} =1.5\times 10^{-4} P^{2}_{1} +2\times 10^{-5}P_{1}P_{2} +3\times 10^{-5} P^{2}_{2} MW\)

where \(P_{1}\) and \(P_{}\) are given in megawatts. Determine the output of each unit, total transmission losses, total load demand, and total operating cost \(C_{T}\) when the area λ = 16.00 $/MWh.

Admin
Solution

Using the incremental operating costs from in (6.12.13),

\(\lambda =\frac{dC_{i} }{dP_{i} } \left(L_{i} \right) =\frac{dC_{i} }{dP_{i} }\left(\frac{1}{1-\frac{\partial P_{L} }{\partial P_{i} } } \right)\) i = 1, 2, …, N (6.12.13)

\(\frac{dC_{1} }{dP_{1} }\left(\frac{1}{1-\frac{\partial P_{L} }{\partial P_{1} } } \right)=\frac{10+16\times 10^{-3}P_{1} }{1-\left(3\times 10^{-4}P_{1}+2\times 10^{-5}P_{2} \right) }\)=16.00

\(\frac{dC_{2} }{dP_{2} }\left(\frac{1}{1-\frac{\partial P_{L} }{\partial P_{2} } } \right)=\frac{8+18\times 10^{-3}P_{2} }{1-\left(6\times 10^{-4}P_{2}+2\times 10^{-5}P_{1} \right) } \)=16.00

Rearranging the two equations,

\(20.8\times 10^{-3} P_{1} +32\times 10^{-5} P_{2} \)=6.00

\(32\times 10^{-5} P_{1} +18.96\times 10^{-3} P_{2} \)=8.00

Solving,

\(P_{1}\) = 282 MW \(P_{2}\) = 417 MW

Using the equation for total transmission losses,

\(P_{L}=1.5\times10^{-4}\left(282\right) ^{2} +2\times 10^{-5} \left(282\right) \left(417\right)+3\times 10^{-5} \left(417\right)^{2}=19.5 MW \)

From (6.12.10), the total load demand is

\(P_{1} +P_{2}+…+P_{N}-P_{L}=P_{T}\) (6.12.10)

\(P_{T}=P_{1}+P_{2}P_{L}=282+417-19.5=679.5 MW\)

Also, using the cost formulas given in , the total operating cost is

\(C_{T}=C_{1}+C_{2}=10\left(282\right) +8\times 10^{-3}\left(282\right) ^{2}+8\left(417\right) +9\times 10x^{-3} \left(417\right)^{2} =8357\) $/h

Note that when transmission losses are included, λ given by (6.12.13) is no longer the incremental operating cost of the area. Instead, λ is the unit incremental operating cost \(dC_{i}/ dP_{i}\) multiplied by the unit penalty factor \(L_{i}\).
