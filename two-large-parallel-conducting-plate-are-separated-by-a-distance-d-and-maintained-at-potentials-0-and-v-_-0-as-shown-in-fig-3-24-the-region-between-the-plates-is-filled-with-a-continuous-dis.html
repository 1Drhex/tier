Guest Question.

Two large parallel conducting. plate are separated by a distance d and maintained at potentials \(0\) and \({ V }_{ 0 }\) , as shown in Fig. 3-24. The region between the plates is filled with a continuous distribution of electron having a volume density of charge \({ \rho }_{ v }=-{ \rho }_{ o }y/d\). Assuming negligible fringing effect at the edges. determine a) the potential at any point between the plates, and \(b\)) the surface charge densities on the plates.

Admin
Solution

\(a)\) Here the governing equation is Poisson’s equation (3-127), which simplifies to

\(\frac { { d }^{ 2 }V\left( y \right) }{ d{ y }^{ 2 } } =\frac { { \rho }_{ o } }{ { \epsilon }_{ o }d } y\quad \quad \quad \left( 3-132 \right) \)

 

Integrating Eq. (3-132) twice, we have

\(V\left( y \right) =\frac { { \rho }_{ o } }{ { 6\epsilon }_{ o }d } { y }^{ 3 }+{ C }_{ 1 }y+{ C }_{ 2 }\)

 

The boundary conditions on the two conducting plates are:

At \(y=0,\quad V=0={ C }_{ 2 }\)

 

At \(y=d,\quad V={ V }_{ 0 }=\frac { { \rho }_{ o }{ d }^{ 2 } }{ { 6\epsilon }_{ o } } +{ C }_{ 1 }d,\quad { C }_{ 1 }=\frac { { V }_{ 0 } }{ d } -\frac { { \rho }_{ o }d }{ { 6\epsilon }_{ o } } \)

 

Substituting the above values of \({ C }_{ 1 }\) and \({ C }_{ 2 }\) in Eq. (3- 133), we obtain the solution for Poisson’s equation (3-132).

\(V\left( y \right) =\frac { { \rho }_{ o } }{ { 6\epsilon }_{ o }d } { y }^{ 3 }+\left( \frac { { V }_{ 0 } }{ d } -\frac { { \rho }_{ o }d }{ { 6\epsilon }_{ o } } \right) y\)

 

The electric field intensity is – \(\nabla\) V.

\(E\left( y \right) =-{ a }_{ y }\frac { \partial V }{ \partial y } =-{ a }_{ y }+\left[ \frac { { \rho }_{ o } }{ { 2\epsilon }_{ o }d } { y }^{ 2 }\left( \frac { { V }_{ 0 } }{ d } -\frac { { \rho }_{ o }d }{ { 6\epsilon }_{ o } } \right) \right]\)

 

\(b)\) The surface charge densities on the conducting plates can be found from the boundary condition given in Eq. (3-46).

At the lower plate, \(y = 0.\)

\({ a }_{ n }={ a }_{ y },\quad { \rho }_{ sl }={ \epsilon }_{ o }{ E }_{ yl }=-\frac { { { \epsilon }_{ o }V }_{ 0 } }{ d } +\frac { { \rho }_{ o }d }{ { 6 } }\)

 

At the upper plate, \(y = d.\)

\({ a }_{ n }={ -a }_{ y },\quad { \rho }_{ su }=-{ \epsilon }_{ o }{ E }_{ yu }=-\frac { { { \epsilon }_{ o }V }_{ 0 } }{ d } +\frac { { \rho }_{ o }d }{ { 3 } }\)

 

(n this case,\({ \rho }_{ su }\neq -{ \rho }_{ sl }\), and it no longer is meaningful to compute the capacitance.

\({ E }_{ n }=\frac { { \rho }_{ s } }{ { \epsilon }_{ o } } \quad \quad \quad \left( 3-46 \right) \)

 

\(\frac { { \partial }^{ 2 }V }{ \partial { x }^{ 2 } } +\frac { { \partial }^{ 2 }V }{ \partial { y }^{ 2 } } +\frac { { \partial }^{ 2 }V }{ \partial { z }^{ 2 } } =-\frac { { \rho }_{ v } }{ \epsilon } \quad \quad \quad \left( 3-46 \right)\)
