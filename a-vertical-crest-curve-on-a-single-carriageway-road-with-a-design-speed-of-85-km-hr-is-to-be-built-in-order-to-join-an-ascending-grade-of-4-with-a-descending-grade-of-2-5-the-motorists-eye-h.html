Guest Question.

A vertical crest curve on a single carriageway road with a design speed of 85 km/hr is to be built in order to join an ascending grade of 4% with a descending grade of 2.5%. The motorist’s eye height is assumed to be 1.05m while the object height is assumed to be 0.26m.

(1) Calculate the minimum curve length required in order to satisfy the requirements of minimum sight stopping distance.
(2) Recalculate the minimum curve length with the object height assumed to be zero.

Admin
Solution

Solution (1):

p = +0.04
q = -0.025
From Table 6.7 the desirable minimum stopping distance is 160m.

Table 6.7 Stopping sight distances for different design speeds (source: TD 9/93 (DoT, 1993))

Design speed (km/hr) Stopping sight distance (m)
50 60 70 85 100 120
70 90 120 160 215 295 Desirable minimum
50 70 90 120 160 215 Absolute minimum

\(e=-\left(q-p\right)\frac{L}{8}=-(-0.025-0.04)×160÷8\)

=1.3 m

Since e >\(H_{1}\) , S ≤L as the sight distance is contained within the curve length.
Therefore, using Equation 6.49:

\(L_{m}=\frac{AS^{2}}{\left[\sqrt{2H_{1}}+ \sqrt{2H_{2}} \right] ^{2} }=\frac{0.065×160^{2}}{\left[\sqrt{2×1.05}+ \sqrt{2×0.26} \right] ^{2} }=353 \)

Solution (2):

If the object height is assumed to be zero, then Equation 6.49 reduces to
Equation 6.58:

\(L=\frac{AS^{2}}{2H_{1}}=\frac{0.065×160^{2}}{2×1.05}\)

=792 m

Thus the required crest curve length more than doubles in value if the object
height is reduced to zero.
