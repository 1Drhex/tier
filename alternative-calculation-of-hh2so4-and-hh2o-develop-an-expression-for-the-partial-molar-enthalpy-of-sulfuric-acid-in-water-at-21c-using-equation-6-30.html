Guest Question.

Alternative Calculation of \(\bar{H}_{ H _{2} SO _{4}}\) and \(\bar{H}_{ H _{2} O }\)

Develop an expression for the partial molar enthalpy of sulfuric acid in water at 21째C using Equation (6.30).

\(\bar{K}_{1}=k-x_{2} \frac{ d k}{ d x_{2}}\) (6.30)

Admin
Solution

If we apply Equation (6.30) to this system, we get:

 

\(\bar{H}_{ H _{2} SO _{4}}=h-x_{ H _{2} O } \frac{ d h}{ d x_{ H _{2} O }}\) (E6.15A)

 

Thus, we begin with the expression for the molar enthalpy given by Equation (E6.13A).

 

\(h=1.596 x_{ H _{2} SO _{4}}+1.591 x_{ H _{2} O }-74.40 x_{ H _{2} SO _{4}} x_{ H _{2} O }\left(1-0.561 x_{ H _{2} SO _{4}}\right)[ kJ / mol ]\) (E6.13A)

 

\(h=1.596 x_{ H _{2} SO _{4}}+1.591 x_{ H _{2} O }-74.40 x_{ H _{2} SO _{4}} x_{ H _{2} O }\left(1-0.561 x_{ H _{2} SO _{4}}\right)[ kJ / mol ]\) (E6.15B)

 

Differentiating Equation (E6.15B) gives:

 

\(\frac{ d h}{ d x_{ H _{2} O }}=-1.596+1.591-74.40\left[\left(x_{ H _{2} SO _{4}}-x_{ H _{2} O }\right)\left(1-0.561 x_{ H _{2} SO _{4}}\right)+0.561 x_{ H _{2} SO _{4}} x_{ H _{2} O }\right]\) (E6.15C)

 

Substituting Equations (E6.15B) and (E6.15C) into (E6.15A) gives:

 

\(\bar{H}_{ H _{2} SO _{4}}=1.596\left(x_{ H _{2} SO _{4}}+x_{ H _{2} O }\right)-74.40 x_{ H _{2} O }^{2}\left(1-0.561 x_{ H _{2} SO _{4}}\right)+41.74 x_{ H _{2} SO _{4}} x_{ H _{2} O }^{2}\)

 

\(\bar{H}_{ H _{2} SO _{4}}=1.596-74.40 x_{ H _{2} O }^{2}+83.48 x_{ H _{2} SO _{4}} x_{ H _{2} O }^{2}[ kJ / mol ]\) (E6.15D)

 

Expressions (E6.15D) and (E6.13C) are identical.
