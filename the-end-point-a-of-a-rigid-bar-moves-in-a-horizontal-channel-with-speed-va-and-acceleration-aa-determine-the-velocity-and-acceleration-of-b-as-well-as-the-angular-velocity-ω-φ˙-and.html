Guest Question.

The end point \(A\) of a rigid bar moves in a horizontal channel with speed \(v_{A}\) and acceleration \(a_{A} \).

Determine the velocity and acceleration of \(B\) as well as the angular velocity \(\omega =\dot{\varphi } \) and the angular acceleration \(\dot{\omega } \) of the bar.

Admin
Solution

The velocity and acceleration of \(B\) are vertically directed. With that information, the \(velocity diagram\) can be plotted and from the graph we find by inspection:

\(l\omega =\frac{v_{A} }{\cos \varphi } \rightsquigarrow \underline{\underline{\omega } } =\dot{\varphi } =\underline{\underline{\frac{v_{A}}{l\cos \varphi } } } \) ,

\(\underline{\underline{v_{B}=v_{A}\tan \varphi } } \) .

From the \(acceleration diagram\) follow in the same way

\(l\dot{\omega } =\frac{a_{A} }{\cos \varphi }+\omega ^{2}l\tan \varphi \)

 

\(\rightsquigarrow \underline{\underline{\dot{\omega } } }=\underline{\underline{\frac{a_{A} }{l\cos \varphi } +\frac{v^{2}_{A}\sin \varphi}{l^{2}\cos ^{3} \varphi } } } \) ,

\(\underline{\underline{a_{B} } }=a_{A}\tan \varphi +\frac{l\omega ^{2} }{\cos \varphi }\)

 

\(=\underline{\underline{a_{A}\tan \varphi +\frac{v^{2}_{A} }{l\cos ^{3}\varphi } } } \) .

The problem can also be solved purely analytically. We introduce an appropriate coordinate system and from the coordinates

\(x_{A}=l\sin \varphi \) ,

\(y_{B}=l\cos \varphi \)

one obtains with \(\dot{x}_{A}=v_{A}\) and \(\ddot{x}_{A}=\dot v_{A}=a_{A} \)

 

\(\dot{x}_{A }=l\dot{\varphi } \cos \varphi \rightsquigarrow \underline{\underline{\omega =\dot{\varphi }=\frac{v_{A} }{l\cos \varphi } } }\) ,

\(\underline{\underline{\dot{\omega } } }=\frac{\dot{v}_{A}l\cos \varphi +v_{A}l\dot{\varphi } \sin \varphi }{l^{2}\cos ^{2}\varphi }=\underline{\underline{ \frac{a_{A} }{l\cos \varphi }+\frac{v^{2}_{A}\sin \varphi }{l^{2}\cos ^{3} \varphi } } } \) ,

\(\underline{\underline{\dot{y}_{B} } } =v_{B}=-l\dot{\varphi } \sin \varphi =\underline{\underline{-v_{A}\tan \varphi } } \) ,

\(\underline{\underline{\ddot{y}_{B} } } =a_{B}=-\dot{v }_{A} \tan \varphi -v_{A}\frac{\dot{\varphi } }{\cos ^{2}\varphi }= \underline{\underline{-a_{A}\tan \varphi -\frac{v^{2}_{A} }{l\cos^{3}\varphi } } } \) .

Note that since \(y\) is positive upwards directed, the quantities \(\dot{y}_{B} \) and
\(\ddot{y}_{B} \) have a negative sign.

The velocity \(v_{B} \) and the angular velocity \(\omega\) can also be determined by using the center of instantaneous rotation Π. This point Π is given by the intersection of the perpendiculars to \(v_{A}\) and \(v_{B}\) , the directions of which are known. In this way we obtain

\(v_{A}=\omega l\cos \varphi \rightsquigarrow \underline{\underline{\omega } } =\underline{\underline{\frac{v_{A} }{l\cos \varphi } } } \) ,

\(\underline{\underline{v_{B} } }=\omega l\sin \varphi =\underline{\underline{v_{A}\tan \varphi } } \) .

 

Remark: From the derived relations it can be seen that \(\dot{\omega } \) and \(a_{B}\) are nonzero even when point \(A\) moves at constant speed (\(a_{A}=0\) ).
