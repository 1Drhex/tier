Guest Question.

Select an appropriate steel column section 24 feet long, braced at midheight about the weak axis, that supports a load of 350 k. Use a W14 section. (See Example )

Admin
Solution

P = 350k
Try a W 14×74 (A = 21.8 in.2; rx = 6.04″; ry = 2.48″)
Weak Axis:
L = 12′; KL = 12′

 

\(\frac{KL}{r_{y}} = \frac{(1)(12^{'}\times 12in./ft.)}{2.48^{"}} = 58\)

Strong Axis:

L = 24′; KL = 24′

 

\(\frac{KL}{r_{x}} = \frac{(1)(24^{'}\times 12in./ft.)}{6.04^{"}} = 47.7\)

 

The weak axis governs the design.

Enter the slenderness ratio table with \(\frac{KL}{r_{y}} = 58.\)

Fa = 17.62ksi
Pa = Fa × A = 17.62 k/in.2 × 21.8in.2 = 384k
Pa = 384k > Pact. = 350k ∴ OK

Efficiency: \(\frac{P_{act.}}{P_{allow}} \times 100\% = \frac{350k}{384k} \times 100\% = 91\%\)

Try for a more efficient column section.
W 14× 68: (A = 20.0in.2 ; rx = 6.01″; ry = 2.46″)
Weak Axis:

 

\(KL = 12^{'} ; \frac{KL}{r_{y}} = \frac{(1)(12^{'}\times 12in./ft.)}{2.46^{"}} = 58.5\)

 

Strong Axis:

 

\(KL = 24^{'} ; \frac{KL}{r_{x}} = \frac{(1)(24^{'}\times 12in./ft.)}{6.01^{"}} = 47.9\)

 

The weak axis governs.

\(\frac{KL}{r_{y}} = 58.5;\) Fa = 17.57ksi

 

Pa = Fa × A = 17.57 k/in.2 × 20.0in.2 = 351.4k
Pa = 351.4k > Pact. = 350k ∴ OK

Efficiency: \(\frac{P_{act.}}{P_{a}} \times 100\% = \frac{350k}{351.4k} \times 100\% = 99.6\%\)

 

Use: W14× 68
