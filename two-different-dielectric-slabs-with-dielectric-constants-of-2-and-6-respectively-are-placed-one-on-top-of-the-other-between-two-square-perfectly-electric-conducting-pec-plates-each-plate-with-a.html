Guest Question.

Two different dielectric slabs, with dielectric constants of 2 and 6, respectively, are placed one on top of the other, between two square perfectly electric conducting (PEC) plates, each plate with an area of \(1 { cm }^{ 2 }\), as shown in the figure. The thickness of each dielectric slab is 1 cm. A 10-Volt d.c. power supply is placed between the two plates forming a parallel-plate capacitor, and is maintained connected at all times. Find (in terms of \({ Îµ }_{ 0 }\) when applicable) the following:

(a) Electric field in each slab (in V/meter).

(b) Electric flux density in each slab (in \(C/{ m }^{ 2 }\)).

(c) Total charge in each of the two PEC plates (in C).

(d) Total capacitance of the parallel-plate capacitor (in farads) using its definition based on the charge and voltage.

(e) Capacitance of each slab (in farads) based on the definition of capacitance (using plate area, separation and permittivity of the medium). (f) Total capacitance (in farads) of the parallel-plate capacitor, using the capacitances of part (e). How does this capacitance compares with that of part (d)? Are they the same or different? Explain.

Admin
Solution
\(A=1C{ m }^{ 2 },\quad V=10Volt,\quad { \varepsilon }_{ r1 }=2,\quad { \varepsilon }_{ r2 }=6,\quad { d }_{ 1 }={ d }_{ 2 }=1Cm\)

(a)

\({ E }_{ 1 }d_{ 1 }+{ E }_{ 2 }d_{ 2 }=10\Rightarrow { E }_{ 1 }+{ E }_{ 2 }=\frac { 10 }{ 10^{ -2 } } =10^{ 3 }V/m\\ D_{ 1 }=D_{ 2 }\Rightarrow { \varepsilon }_{ 1 }{ E }_{ 1 }={ \varepsilon }_{ 2 }{ E }_{ 2 }\Rightarrow { E }_{ 1 }=\frac { { E }_{ 2 } }{ { \varepsilon }_{ 1 } } ({ \varepsilon }_{ 2 })=\frac { { \varepsilon }_{ 2 } }{ { \varepsilon }_{ 1 } } { E }_{ 2 }=\frac { 6 }{ 2 } { E }_{ 2 }=3{ E }_{ 2 }\\ { E }_{ 1 }+{ E }_{ 2 }=3{ E }_{ 2 }+{ E }_{ 2 }=4{ E }_{ 2 }={ 10 }^{ 3 }\Rightarrow { E }_{ 2 }=0.25\times { 10 }^{ 3 }V/m,\quad { E }_{ 1 }=3{ E }_{ 2 }=0.75\times { 10 }^{ 3 }V/m\\ { E }_{ 1 }=0.75\times { 10 }^{ 3 }V/m,\quad { E }_{ 2 }=0.25\times { 10 }^{ 3 }V/m\)

(b)

\({ D }_{ 1 }={ \varepsilon }_{ 1 }{ E }_{ 1 }=2{ \varepsilon }_{ 0 }(0.75\times { 10 }^{ 3 })=1.5{ \varepsilon }_{ 0 }\times { 10 }^{ 3 }C/{ m }^{ 2 }\\ { D }_{ 2 }={ \varepsilon }_{ 2 }{ E }_{ 2 }=6{ \varepsilon }_{ 0 }(0.25\times { 10 }^{ 3 })=1.5{ \varepsilon }_{ 0 }\times { 10 }^{ 3 }C/{ m }^{ 2 }\)

(C)

\({ Q }_{ upper }={ A }_{ 1 }{ q }_{ s1 }={ 10 }^{ -4 }{ \varepsilon }_{ 0 }(1.5\times { 10 }^{ 3 })=0.15{ \varepsilon }_{ 0 }C\\ { Q }_{ lower }={ A }_{ 2 }{ q }_{ s2 }={ 10 }^{ -4 }{ \varepsilon }_{ 0 }(1.5\times { 10 }^{ 3 })=0.15{ \varepsilon }_{ 0 }C\)

(d)

\({ C }_{ 1 }=\frac { Q }{ { V }_{ 1 } } =\frac { Q_{ 1 } }{ { E }_{ 1 }{ d }_{ 1 } } =\frac { 0.15{ \varepsilon }_{ 0 } }{ (0.75\times { 10 }^{ 3 })({ 10 }^{ -2 }) } =0.02{ \varepsilon }_{ 0 }F\\ C_{ 2 }=\frac { Q_{ 2 } }{ { V }_{ 1 } } =\frac { 0.15{ \varepsilon }_{ 0 } }{ { E }_{ 2 }{ d }_{ 2 } } =\frac { 0.15{ \varepsilon }_{ 0 } }{ (0.25\times { 10 }^{ 3 })({ 10 }^{ -2 }) } =0.06{ \varepsilon }_{ 0 }F\\ C_{ T }=\frac { Q }{ { V } } =\frac { 0.15{ \varepsilon }_{ 0 } }{ 10 } =0.015{ \varepsilon }_{ 0 }F\)

(e)

\({ C }_{ 1 }={ \varepsilon }_{ 1 }\frac { { A }_{ 1 } }{ d_{ 1 } } =2{ \varepsilon }_{ 0 }\frac { { 10 }^{ -4 } }{ { 10 }^{ -2 } } =0.02{ \varepsilon }_{ 0 }\\ { C }_{ 2 }={ \varepsilon }_{ 2 }\frac { { A }_{ 2 } }{ d_{ 2 } } =6{ \varepsilon }_{ 0 }\frac { { 10 }^{ -4 } }{ { 10 }^{ -2 } } =0.06{ \varepsilon }_{ 0 }\\ { C }_{ T }=\frac { { C }_{ 1 }{ C }_{ 2 } }{ { C }_{ 1 }{ +C }_{ 2 } } =\frac { (0.02{ \varepsilon }_{ 0 })(0.06{ \varepsilon }_{ 0 }) }{ (0.02{ \varepsilon }_{ 0 }+0.06{ \varepsilon }_{ 0 }) } =\frac { 0.02(0.06) }{ 0.08 } { \varepsilon }_{ 0 }\\ { C }_{ T }=\frac { 2(6) }{ 8 } \times { 10 }^{ -2 }{ \varepsilon }_{ 0 }=0.015{ \varepsilon }_{ 0 }\)

(f) They are the same, as they should be.
