Guest Question.

Starting from the Rodrigues formula, derive the orthonormality condition for Legendre polynomials:

\( \int_{-1}^{1} P_{\ell}(x) P_{\ell^{\prime}}(x) d x=\left(\frac{2}{2 \ell+1}\right) \delta_{\ell \ell^{\prime}} \). (4.34).

Hint: Use integration by parts.

Admin
Solution

\( \int_{-1}^{1} P_{\ell}(x) P_{\ell^{\prime}}(x) d x=\frac{1}{2^{\ell} \ell !} \frac{1}{2^{\ell^{\prime}} \ell^{\prime} !} \int_{-1}^{1}\left[\left(\frac{d}{d x}\right)^{\ell}\left(x^{2}-1\right)^{\ell}\right]\left[\left(\frac{d}{d x}\right)^{\ell^{\prime}}\left(x^{2}-1\right)^{\ell^{\prime}}\right] d x \).

If \( \ell \neq \ell^{\prime} \), we may as well let \( \ell \) be the larger of the two \( \left(\ell>\ell^{\prime}\right) \). Integrate by parts, pulling successively each derivative off the first term onto the second:

\( 2^{\ell} \ell ! 2^{\ell^{\prime}} \ell^{\prime} ! \int_{-1}^{1} P_{\ell}(x) P_{\ell^{\prime}}(x) d x=\left.\left[\left(\frac{d}{d x}\right)^{\ell-1}\left(x^{2}-1\right)^{\ell}\right]\left[\left(\frac{d}{d x}\right)^{\ell^{\prime}}\left(x^{2}-1\right)^{\ell^{\prime}}\right]\right|_{-1} ^{1} \)

\( -\int_{-1}^{1}\left[\left(\frac{d}{d x}\right)^{\ell-1}\left(x^{2}-1\right)^{\ell}\right]\left[\left(\frac{d}{d x}\right)^{\ell^{\prime}+1}\left(x^{2}-1\right)^{\ell^{\prime}}\right] d x \).

\( =\ldots \text { (boundary terms) } \ldots+(-1)^{\ell} \int_{-1}^{1}\left(x^{2}-1\right)^{\ell}\left(\frac{d}{d x}\right)^{\ell^{\prime}+\ell}\left(x^{2}-1\right)^{\ell^{\prime}} d x \).

\( \text { But }(d / d x)^{\ell^{\prime}+\ell}\left(x^{2}-1\right)^{\ell^{\prime}}=0, \text { because }\left(x^{2}-1\right)^{\ell^{\prime}} \) is a polynomial whose highest power is \( 2 \ell^{\prime} \) , so more than \( 2 \ell^{\prime} \) derivatives will kill it, and \( \ell^{\prime}+\ell>2 \ell^{\prime} \). Now, the boundary terms are of the form:

\( \left.\left[\left(\frac{d}{d x}\right)^{\ell-n}\left(x^{2}-1\right)^{\ell}\right]\left[\left(\frac{d}{d x}\right)^{\ell^{\prime}+n-1}\left(x^{2}-1\right)^{\ell^{\prime}}\right]\right|_{-1} ^{+1}, n=1,2,3, \ldots, \ell \).

Look at the first term: \( \left(x^{2}-1\right)^{\ell}=\left(x^{2}-1\right)\left(x^{2}-1\right) \ldots\left(x^{2}-1\right) ; \ell \) So 0, 1, 2, ….. \(\ell \) -1 derivatives will still leave at least one overall factor of \( \left(x^{2}-1\right) \). [Zero derivatives leaves \(\ell \) factors; one derivative leaves \( \ell-1: d / d x\left(x^{2}-1\right)^{\ell}=2 \ell x\left(x^{2}-1\right)^{\ell-1} \); two derivatives leaves \( \ell -2:d^2/dx^2(x^2-1)^\ell =2\ell(x^2-1)^{\ell -1} +2\ell (\ell -1)2x^2(x^2-1^{\ell -2}) \), and so on.] So the boundary terms are all zero, and hence \( \int_{-1}^{1} P_{\ell}(x) P_{\ell^{\prime}}(x) d x=0 \).

This leaves only the case \( \ell=\ell^{\prime} \). Again the boundary terms vanish, but this time the remaining integral does not:

\( \left(2^{\ell} \ell !\right)^{2} \int_{-1}^{1}\left[P_{\ell}(x)\right]^{2} d x=(-1)^{\ell} \int_{-1}^{1}\left(x^{2}-1\right)^{\ell} \underbrace{\left(\frac{d}{d x}\right)^{2 \ell}\left(x^{2}-1\right)^{\ell}}_{(d / d x)^{2 \ell}\left(x^{2 \ell}\right)=(2 \ell) !} d x \).

\( =(-1)^{\ell}(2 \ell) ! \int_{-1}^{1}\left(x^{2}-1\right)^{\ell} d x=2(2 \ell) ! \int_{0}^{1}\left(1-x^{2}\right)^{\ell} d x \).

\( \text { Let } x \equiv \cos \theta, \text { so } d x=-\sin \theta d \theta, \quad\left(1-x^{2}\right)=\sin ^{2} \theta, \quad \theta: \pi / 2 \rightarrow 0 \) Then

\( \int_{0}^{1}\left(1-x^{2}\right)^{\ell} d x=\int_{\pi / 2}^{0}(\sin \theta)^{2 \ell}(-\sin \theta) d \theta=\int_{0}^{\pi / 2}(\sin \theta)^{2 \ell+1} d \theta \).

\( =\frac{(2)(4) \cdots(2 \ell)}{(1)(3)(5) \cdots(2 \ell+1)}=\frac{\left(2^{\ell} \ell !\right)^{2}}{1 \cdot 2 \cdot 3 \cdots(2 \ell+1)}=\frac{\left(2^{\ell} \ell !\right)^{2}}{(2 \ell+1) !} \).

\( \therefore \int_{-1}^{1}\left[P_{\ell}(x)\right]^{2} d x=\frac{1}{\left(2^{\ell} \ell !\right)^{2}} 2(2 \ell) ! \frac{\left(2^{\ell} \ell !\right)^{2}}{(2 \ell+1) !}=\frac{2}{2 \ell+1} . \quad \text { So } \int_{-1}^{1} P_{\ell}(x) P_{\ell^{\prime}}(x) d x=\frac{2}{2 \ell+1} \delta_{\ell \ell^{\prime}} \). QED
