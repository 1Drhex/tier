Guest Question.

Figure 6.27 illustrates an early model F-16A and a later F-16C. The differences between the stabilators of the two aircraft are apparent. The increase in stabilator area was made to all but the earliest F-16As to increase pitch control authority. Table 6.3 lists descriptive data for each aircraft. Determine the effect of this change on the F-16’s neutral point location and static margin, for both subsonic and supersonic flight.

Table 6.3 Descriptive parameters for an early F-16A and an F-16C

\(\begin{array}{ccc}\hline \hline \text { Item } & \text { Early F-16A } & \text { F-16C } \\\hline \text { Wing: } & & \\S, ft ^{2} & 300 & 300 \\c_{\text {root }}, ft & 16.5 & 16.5 \\c_{\text {tip }}, ft & 3.5 & 3.5 \\b, ft & 30 \text { (no missiles or rails) } & 30 \text { (no missiles or rails) } \\x \text { of root chord leading } & 0 & 0 \\\quad \text { edge, ft } & \text { (20 ft aft of fuselage nose) } & \text { (20 ft aft of fuselage nose) } \\\Lambda_{L E}, \text { degrees } & 40 & 40\\\text { Stabilator: } & & \\S_{ t }, ft ^{2} & 108 & 135 \\c_{\text {root }}, ft & 10 & 11 \\c_{\text {tip }}, ft & 2 & 3 \\b, ft & 18 & 18 \\x \text { of root chord leading } & & \\\text { edge, } ft & 17.5 & 17 \\\Lambda_{L E}, \text { degrees } & 40 & 40\\\text { Strake (exposed): } & & \\S_{\text {strake }}, ft ^{2} & 20 & 20 \\c_{\text {root }}, ft & 9.6 & 9.6 \\c_{\text {tip }}, ft & 0 & 0 \\b, ft & 2 & 2 \\x \text { of root chord leading } & & \\\text { edge, } ft & -8 & -8 \\\Lambda_{L E}, \text { degrees (avg.) } & 80 & 80 \\\text { Fuselage } & & \\l_{f} & 48.5 & 48.5 \\w_{f} & 5 & 5 \\\text { Whole airplane } & & \\\bar{x} \text { (relative to M.A.C.) } & .35 & .35\end{array}\)

 

Admin
Solution

We will run the complete analysis for the subsonic case, then repeat
it for supersonic flight. The stability analysis begins by estimating the location of the aerodynamic center of the wing/strake/fuselage combination, which will be the same for both aircraft. For the F-16 wing alone,

\(\begin{gathered}\lambda=c_{\text {tip }} / c_{\text {root }}=3.5 ft / 16.5 ft =0.212 \\MAC =\frac{2}{3} c_{\text {root }} \frac{1+\lambda+\lambda^{2}}{1+\lambda}=11.4 ft \\y_{\text {MAC }}=\frac{b}{6} \frac{1+2 \lambda}{1+\lambda}=5.875 ft\end{gathered}\)

For subsonic flight,

\(\begin{aligned}x_{ ac } &=y_{ MAC } \tan \Lambda_{ LE }+0.25 MAC \\&=(5.875 ft ) \tan 40 deg +0.25(11.4 ft )=7.8 ft\end{aligned}\)

Adding the effect of the strake,

\(\begin{aligned}&\lambda_{\text {strake }}=c_{\text {tip }} / c_{\text {root }}=0 ft / 9.6 ft =0 \\& MAC =\frac{2}{3} c_{\text {root }} \frac{1+\lambda+\lambda^{2}}{1+\lambda}=6.4 ft\end{aligned}\)

 

\(y_{ MAC }=\frac{b}{6} \frac{1+2 \lambda}{1+\lambda}=0.33 ft\)

 

\(\begin{aligned}x_{ ac _{\text {strake }}} &=y_{ MAC _{\text {strake }}} \tan \Lambda_{ LE _{\text {strake }}}+0.25 MAC _{\text {strake }} \\&=(0.33 ft ) \tan 80 deg +0.25(6.4 ft )=3.5 ft\end{aligned}\)

 

but these are defined relative to the leading edge of the strake root chord, not the wing root chord. From Table 6.3, the strake root is 8 ft forward of the wing root, so that relative to the wing

\(x_{ ac _{\text {strake }}}=-4.5 ft \text { (subsonic) }\)

and

\(x_{ ac _{\text {wing }+\text { srrake }}}=\frac{x_{ ac _{\text {wing }}} S+\left(x_{\text {ac }_{\text {srrake }}}-x_{ ac _{\text {wing }}}\right) S_{\text {strake }}}{S+S_{\text {strake }}}=6.5 ft\)

Now, adding the effect of the fuselage, using \(C_{L_{\alpha_{\text {wing } / \text { strake }}}}=0.068 / deg\) (predicted in Sec. 4.7) and the fact that the wing-root leading edge is 20 ft aft of the fuselage nose, so that \(l_{ ac _{\text {wing } / \text { strake }}}=20 ft +x_{ ac _{\text {wing/strake }}}\)

\(\begin{aligned}x_{ ac _{\text {wing }+\text { strake +fuselage }}} &=x_{ ac _{\text {wing }+\text { strake }}}-\frac{l_{f} w_{f}^{2}\left[0.005+0.111\left(\frac{l_{ ac _{\text {wing }}+\text { strake }}}{l_{f}}\right)^{2}\right]}{S C_{L_{\alpha_{\text {wing }+\text { strake }}}}} \\&=6.5 ft -\frac{48.5 ft (5 ft )^{2}\left[0.005+0.111\left(\frac{26.5 ft }{48.5 ft }\right)^{2}\right]}{300 ft ^{2}(0.068 / deg )(57.3 deg / rad )} \\&=6.4 ft\end{aligned}\)

 

Next, the aerodynamic center of the F- 1 6 A stabilator is located for subsonic flight.

\(\begin{gathered}\lambda_{\text {stabilator }}=c_{\text {tip }} / c_{\text {root }}=2 ft / 10 ft =0.2 \\MAC =\frac{2}{3} c_{\text {root }} \frac{1+\lambda+\lambda^{2}}{1+\lambda}=6.9 ft \\y_{ MAC }=\frac{b}{6} \frac{1+2 \lambda}{1+\lambda}=3.5 ft\end{gathered}\)

 

\(\begin{aligned}x_{ ac _{ stab }} &=y_{ MAC _{ stab }} \tan \Lambda_{ LE _{ sab }}+0.25 MAC _{ stab } \\&=(3.5 ft ) \tan 40 deg +0.25(6.9 ft )=4.7 ft\end{aligned}\)

 

This is defined relative to the leading edge of the stabilator root chord. From Table 6.3, the stabilator root is 17.5 ft aft of the wing root, so that relative to the wing

\(x_{ ac _{\text {stab }}}=22.2 ft\)

But the distance of interest for the stabilator is \(l_{t}\), the distance from the stabilator’s aerodynamic center to the aircraft center of gravity. Table 6.3 lists the center of gravity as 0.35 MAC, so that relative to the wing root

\(\begin{aligned}x_{ cg } &=y_{ MAC } \tan \Lambda_{ LE }+0.35 MAC \\&=(5.875 ft ) \tan 40 deg +0.35(11.4 ft )=8.9 ft\end{aligned}\)

and

\(l_{t}=x_{ ac _{\text {stab }}}-x_{ cg }=22.2 ft -8.9 ft =13.3 ft\)

It is now possible to calculate tail volume ratio:

\(V_{H}=\frac{S_{t} l_{t}}{S \bar{c}}=\frac{108 ft ^{2}(13.3 ft )}{300 ft ^{2}(11.4 ft )}=0.42\)

Because the F-16’s \(\bar{x}_{ cg }\) is specified relative to the leading edge of the MAC, it is convenient (and common) to express \(\overline{ X }_{\text {ac }_{\text {wing }+\text { strake }+\text { fuselage }}}\) and \(\bar{x}_{n}\) relative to the same reference. This requires subtracting the distance between the root leading edge and the MAC leading edge from the value of \(x_{ ac }\) . The expression for \(\bar{x}_{ ac _{\text {wing }+\text { strake }+\text { fuselage }}}\) then becomes

\(\bar{x}_{ a c_{w i n g}+\text { srake }+\text { fuselage }}=\frac{x_{ ac _{\text {wing }+\text { srrake }+\text { fuselage }}}}{\bar{c}}=\frac{6.4 ft -(5.875 ft ) \tan 40 deg }{11.4 ft }=0.13\)

Recall from Sec. 4.7 that \(\partial \varepsilon / \partial \alpha=0.64\).

\(\begin{aligned}\bar{x}_{n} &=\bar{x}_{ ac _{\text {wing }+\text { srakac }+\text { fiuselage }}}=V_{H} \frac{C_{L_{\alpha t}}}{C_{L_{\alpha}}}\left(1-\frac{\partial \varepsilon}{\partial \alpha}\right) \\&=0.13+0.42\left(\frac{0.0536}{0.0572}\right)(1-0.64)=0.27\end{aligned}\)

So that the F-16As static margin is

\(\begin{aligned}SM &=\bar{x}_{n}-\bar{x} \\&=0.27-0.35=-0.08\end{aligned}\)

Similar calculations for the F-16C yield

\(SM =0.36-0.35=+0.01\)

Now we repeat the calculations for supersonic flight, when the aerodynamic centers of the wing and stabilator shift from the quarter-chord of their MAC to the halfchord.

\(\begin{aligned}x_{ ac } &=y_{ MAC } \tan \Lambda_{ LE }+0.50 MAC \\&=(5.875 ft ) \tan 40 deg +0.50(11.4 ft )=10.6 ft\end{aligned}\)

Adding the effect of the strake,

\(\begin{aligned}x_{ ac _{\text {strake }}} &=y_{ MAC _{\text {strake }}} \tan \Lambda_{ LE _{\text {strake }}}+0.50 MAC _{\text {strake }} \\&=(0.33 ft ) \tan 80 deg +0.50(6.4 ft )=5.1 ft\end{aligned}\)

But these are defined relative to the leading edge of the strake-root chord, not the wing-root chord. From Table 6.3, the strake root is 8 ft forward of the wing root, so that relative to the wing

\(x_{\text {ac }_{\text {strake }}}=-2.9 ft\)

and

\(x_{ ac _{\text {wing }+\text { strake }}}=\frac{x_{ ac _{\text {wing }}} S+\left(x_{\text {ac } \left._{\text {strake }}-x_{ ac _{\text {wing }}}\right)}{S_{\text {strake }}}\right.}{S+S_{\text {strake }}}=9.1 ft\)

To perform the supersonic calculation for fuselage effect, supersonic lift-curve slope must be predicted. A specific Mach number must be chosen. For M = 1.5

\(C_{L_{\alpha}}=\frac{4}{\sqrt{M_{\infty}^{2}-1}}=0.051 / deg\)

 

\(x_{ a c_{\text {wing }+\text { strake + fuselage }}}=x_{ ac _{\text {wing }+\text { strake }}}-\frac{l_{f} w_{f}^{2}\left[0.005+0.111\left(\frac{l_{ ac _{\text {wing }+\text { strake }}}}{l_{f}}\right)^{2}\right]}{S C_{L_{\alpha \text { wing }+\text { strake }}}}\)

 

\(\begin{aligned}&=9.1 ft -\frac{48.5 ft (5 ft )^{2}\left[0.005+0.111\left(\frac{29.1 ft }{48.5 ft }\right)^{2}\right]}{300 ft ^{2}(0.051 / deg )(57.3 deg / rad )} \\&=9.0 ft\end{aligned}\)

Next, the aerodynamic center of the F-16A stabilator is located:

\(\begin{gathered}\lambda_{\text {stabilator }}=c_{\text {tip }} / c_{\text {root }}=2 ft / 10 ft =0.2 \\MAC =\frac{2}{3} c_{\text {root }} \frac{1+\lambda+\lambda^{2}}{1+\lambda}=6.9 ft \\y_{\text {MAC }}=\frac{b}{6} \frac{1+2 \lambda}{1+\lambda}=3.5 ft\end{gathered}\)

 

\(\begin{aligned}x_{ ac _{\text {stab }}} &=y_{ MAC _{\text {stab }}} \tan \Lambda_{ LE _{\text {stab }}}+0.4 MAC _{\text {stab }} \\&=(3.5 ft ) \tan 40 deg +0.50(6.9 ft )=6.4 ft\end{aligned}\)

 

This is defined relative to the leading edge of the stabilator root chord. From Table 6.3, the stabilator root is 17.5 ft aft of the wing root, so that relative to the wing

\(x_{ ac _{\text {stab }}}=23.9 ft\)

But the distance of interest for the stabilator is \(l_{t}\), the distance from the stabilator’s aerodynamic center to the aircraft center of gravity. Table 6.3 lists the center of gravity as 0.35 MAC, so that relative to the wing root

\(\begin{aligned}x_{ cg } &=y_{ MAC } \tan \Lambda_{ LE }+0.35 MAC \\&=(5.875 ft ) \tan 40 deg +0.35(11.4 ft )=8.9 ft\end{aligned}\)

and

\(l_{t}=x_{ ac _{ stab }}-x_{ cg }=23.9 ft -8.9 ft =15 ft\)

It is now possible to calculate tail volume ratio

\(V_{H}=\frac{S_{t} l_{t}}{S \bar{c}}=\frac{108 ft ^{2}(15 ft )}{300 ft ^{2}(11.4 ft )}=0.47\)

Recall from Sec. 4.7 that \(\partial \varepsilon / \partial \alpha=0.64\) . Because the F-16’s \(\bar{x}_{ cg }\) is specified relative to the leading edge of the MAC, it is convenient (and common) to express \(\bar{x}_{ ac _{\text {wing }+\text { strake +fuselage }}} \text { and } \bar{x}_{n}\) relative to the same reference. This requires subtracting the distance between the root leading edge and the MAC leading edge from the value of \(x_{ ac }\) . The expression for supersonic \(\bar{X}_{\text {ac }_{\text {wing }+\text { strake }+\text { fuselage }}}\) then becomes

\(\bar{x}_{ a c_{\text {wing }+\text { strake + fuselage }}}=\frac{x_{\text {ac }_{\text {wing }+\text { srake }+\text { fuselage }}}}{\bar{c}}=\frac{9 ft -(5.875 ft ) \tan 40 deg }{11.4 ft }=0.36\)

 

And for supersonic neutral point

\(\begin{aligned}\bar{x}_{n} &=\bar{x}_{ ac _{\text {wing }+\text { strake +fuselage }}}=V_{H} \frac{C_{L_{\alpha t}}}{C_{L_{\alpha}}}\left(1-\frac{\partial \varepsilon}{\partial \alpha}\right) \\&=0.36+0.47\left(\frac{0.051}{0.0544}\right)(1-0.64)=0.52\end{aligned}\)

So that the F-16A’s supersonic static margin is

\(\begin{aligned}SM &=\bar{x}_{n}-\bar{x} \\&=0.52-0.35=+0.17\end{aligned}\)

Similar calculations for the F-16C yield

\(SM =0.61-0.35=+0.26\)

Figure 6.28 plots neutral point locations calculated for the F-16C vs Mach number and compares them with actual values. Note that, despite the F-16’s relatively complex aerodynamics, the method produced reasonably good estimates.
