Guest Question.

Let D = \(5 { r }^{ 2 }{a }_{ r } mC/{ m }^{ 2 }\) for r < 0.08 m, and D = \(0.1 {a }_{ r }/ { r }^{ 2 }C/{ m }^{ 2 }\) for r > 0.08 m. (a) Find \({ \rho }_{ \upsilon }\) for r = 0.06m. (b) Find \({ \rho }_{ \upsilon }\) for r = 0.1 m. (c) What surface charge density could be located at r = 0.08 m to cause D = 0 for r > 0.08 m?

Admin
Solution

a) This radius lies within the first region, and so
\({ \rho }_{ \upsilon }=\triangledown\) .D=\(\frac { 1 }{ { r }^{ 2 } } \frac { d }{ dr } \left( { r }^{ 2 }{ D }_{ r } \right) =\frac { 1 }{ { r }^{ 2 } } \frac { d }{ dr } \left( 5.00{ r }^{ 4 } \right) =20r mC/{ m }^{ 3 }\)
which when evaluated at r = 0.06 yields \({ \rho }_{ \upsilon }(r = .06) = 1.20 mC/{ m }^{ 3 }\)
b)This is in the region where the second ﬁeld expression is valid. The \(1/{ r }^{ 2 }\) dependence of this ﬁeld yields a zero divergence (shown in Problem 3.23), and so the volume charge density is zero at 0.1 m
c) The total surface charge should be equal and opposite to the total volume charge. The latter is
Q = \(\int _{ 0 }^{ 2\pi }{ } \int _{ 0 }^{ \pi }{ } \int _{ 0 }^{ .08 }{ 20r(mC/{ m }^{ 3 }) } { r }^{ 2 }\sin { \theta } dr d\theta d\phi = 2.57\times { 10 }^{ -3 } mC = 2.57\mu C\)
So now
\({ \rho }_{ S }=-\left[ \frac { 2.57 }{ 4\pi { \left( .08 \right) }^{ 2 } } \right] =-32\mu C/{ m }^{ 2 }\)
