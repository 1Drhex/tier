Guest Question.

The cable will break when the maximum tension reaches \({ T }_{ max }\). Determine the minimum sag h if it supports the uniform distributed load \(w\).

Given:

\(kN\) = \({ 10 }^{ 3 }\) \(N\)

\({ T }_{ max }\) = \(10\) \(kN\)

\(w\) = \(600\) \(\frac { N }{ m }\)

\(a\) = \(25\) \(m\)

Admin
Solution

The equation of the cable:

\(y=\frac { 1 }{ { F }_{ H } } \int { \int { w } } dxdx\quad \quad \quad y=\frac { 1 }{ { F }_{ H } } \left( \frac { w{ x }^{ 2 } }{ 2 } +{ C }_{ 1 }x+{ C }_{ 2 } \right) \quad \quad \quad \frac { dy }{ dx } =\frac { 1 }{ { F }_{ H } } \left( wx+{ C }_{ 1 } \right)\)

 

Boundary Conditions:

\(y=0\) at \(x=0\), then from Eq.[1] \(\quad \quad \quad 0=\frac { 1 }{ { F }_{ H } } \left( { C }_{ 2 } \right) \quad \quad \quad { C }_{ 2 }=0\)

 

\(\frac { d }{ dx } y=0\) at \(x=0\) then from Eq.[2] \(\quad \quad \quad 0=\frac { 1 }{ { F }_{ H } } \left( { C }_{ 1 } \right) \quad \quad \quad { C }_{ 1 }=0\)

 

Thus,\(\quad y=\left( \frac { w }{ 2{ F }_{ H } } \right) { x }^{ 2 }\quad \quad \frac { dy }{ dx } =\frac { w }{ { F }_{ H } } x\quad \quad h\frac { w }{ 2{ F }_{ H } } { \left( \frac { a }{ 2 } \right) }^{ 2 }\quad \quad { F }_{ H }=\frac { w }{ 2h } { \left( \frac { a }{ 2 } \right) }^{ 2 }\)

 

\(\tan { \left( { \theta }_{ max } \right) } =\frac { wa }{ 2{ F }_{ H } } \quad \quad \quad \quad \quad \quad \cos { \left( { \theta }_{ max } \right) } =\frac { 2{ F }_{ H } }{ \sqrt { 4{ F }_{ H }^{ 2 }+{ \left( wa \right) }^{ 2 } } }\)

 

\({ T }_{ max }=\frac { { F }_{ H } }{ \cos { \left( { \theta }_{ max } \right) } } =\sqrt { { F }_{ H }^{ 2 }+\frac { { \left( wa \right) }^{ 2 } }{ 4 } } =\frac { wa }{ 2 } \sqrt { \frac { { a }^{ 2 } }{ 16{ h }^{ 2 } } +1 }\)

 

Guess \(\quad h=1m\)

Given \(\quad { T }_{ max }=\frac { wa }{ 2 } \sqrt { \frac { { a }^{ 2 } }{ 16{ h }^{ 2 } } +1 } \quad \quad \quad h=Find\left( h \right) \quad \quad \quad h=7.09m\)
