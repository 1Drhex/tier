Guest Question.

A point mass M moves according to Fig. 1.8a along a straight line. The square of the velocity decreases linearly with x. The mass passes the location \(\ x=0 \) at\(\ t=0 \) with a velocity \(\ v_{0} \gt 0 \) and at the location \(\ x=x_{1}\gt 0 \) it has the velocity \(\ v_{1}=0 \).
At what time does the point mass reach the position \(\ x_{1} \) and what is its acceleration?

Admin
Solution

First, we need to describe the velocity. A linear relation between \(\ v^{2} \) and \(\ x \) can be generally written as \(\ v^{2}=bx+c \). The constants b and c follow from consideration of the given values:

\(\ v\left(x=0\right)=v_{0}\rightarrow c=v^{2}_{0},v\left(x=x_{1}\right)=0\rightarrow b=-\frac{c}{x_{1}} =-\frac{v^{2}_{0}}{x_{1}}. \)

Thus,

\(\ v=v\left(x\right) =v_{0} \sqrt{1-\frac{x}{x_{1} } } . \)

Figure 1.8b shows \(\ v\left(x\right) \) in the phase plane. From \(\ v = dx/dt \) it follows by separation of variables and indeterminate integration that

\(\ t=\int{\frac{dx}{v_{0}\sqrt{1-\frac{x}{x_{1}} } } }=-2\frac{x_{1} }{v_{0}} \sqrt{1-\frac{x}{x_{1} }}+C. \)

The integration constant C is determined from the initial condition \(\ x\left(0\right)=0: \)

\(\ 0=-2\frac{x_{1}}{v_{0}}\sqrt{1}+C \rightarrow C=2\frac{x_{1}}{v_{0}}. \)

With this, the time \(\ t_{1} \) when the mass reaches\(\ x = x_{1} \) is found as

\(\ \underline{\underline{t_{1}} } =t\left(x_{1}\right) =C=\underline{\underline{2\frac{x_{1}}{v_{0}} } } . \)

The acceleration is determined by application of the chain rule:

\(\ \underline{\underline{a} } =\frac{dv}{dt} =\frac{dv}{dx}\frac{dx}{dt}=\frac{dv}{dx}v=-\frac{v_{0}}{2x_{1}} \frac{1}{\sqrt{1-\frac{x}{x_{1}} } }v_{0} \sqrt{1-\frac{x}{x_{1}} } =-\underline{\underline{\frac{v^{2}_{0}}{2x_{1}} } } . \)

Since a = const, we see that the motion is a uniform acceleration.
As a check, we can compute the velocity and position via integration from the known acceleration:

\(\ v=at+v_{0}=-\frac{v^{2}_{0}}{2x_{1}} t+v_{0} \),

 

\(\ x=-\frac{v^{2}_{0}}{4x_{1}} t^{2}+v_{0}t, \)

Elimination of t leads to the given velocity-position relation.
