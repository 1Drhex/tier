Guest Question.

A steel plate of dimensions \( 2.5\times 1.5\times 0.08 \) m
and weighing 23.1kN is hoisted by steel cables with lengths
\( { L }_{1 }=3.2 \) m and \( { L }_{2 }=3.9 \) m that are each attached to the plate by
a clevis and pin (see figure). The pins through the clevises are
18 mm in diameter and are located 2.0 m apart. The orientation
angles are measured to be \( \theta =94.4° \) and \( \alpha = 54.9° \) .
For these conditions, first determine the cable forces \( { T }_{ 1 } \) and
\( { T }_{ 2 } \) , then find the average shear stress \( { \tau }_{ aver } \) in both pin 1 and pin
2, and then the average bearing stress \( { \sigma }_{ b } \) between the steel plate
and each pin. Ignore the mass of the cables.

Admin
Solution

NUMERICAL DATA

\( {L }_{1 }=3.2m \)
\( {L }_{2 }=3.9m \)
\( \alpha =54.9\left( \frac { \pi }{ 180 } \right) ㎭. \)
\( a=0.6m\quad \quad \quad b=1m \)
\( W=77.0\left( 2.5\times 1.5\times 0.08 \right) \quad \quad \quad W=23.1kN \)
(\( 77=wt \) density of steel, \( kN/{ m }^{ 3 } \))

SOLUTION APPROACH

STEP (1) \( d=\sqrt { { a }^{ 2 }+{ b }^{ 2 } } \quad \quad \quad d=1.166m \)

STEP (2) \( { \theta }_{ 1 }=atan\left( \frac { a }{ b } \right) \quad \quad \quad { \theta }_{ 1 }\frac { 180 }{ \pi } =30.964 degrees \)

STEP (3)-Law of cosines

\( H=\sqrt { { d }^{ 2 }+{ L }_{ 1 }^{ 2 }-2d{ L }_{ 1 }\cos { \left( \theta +{ \theta }_{ 1 } \right) } } \quad \quad \quad H=3.99m \)

STEP (4) \( { \beta }_{ 1 }=a\cos { \left( \frac { { L }_{ 2 }^{ 2 }+{ H }^{ 2 }-{ d }^{ 2 } }{ 2{ L }_{ 1 }H } \right) } \quad \quad \quad { \beta }_{ 1 }\frac { 180 }{ \pi } =13.789 degrees \)

STEP (5) \( { \beta }_{ 2 }=a\cos { \left( \frac { { L }_{ 2 }^{ 2 }+{ H }^{ 2 }-{ d }^{ 2 } }{ 2{ L }_{ 1 }H } \right) } \quad \quad \quad { \beta }_{ 2 }\frac { 180 }{ \pi } =16.95 degrees \)

STEP (6)
Check \( \left( { \beta }_{ 1 }+{ \beta }_{ 2 }+\theta +\alpha \right) \frac { 180 }{ \pi } =180.039 degrees \)

Statics \( { T }_{ 1 }\sin { \left( { \beta }_{ 1 } \right) } ={ T }_{ 2 }\sin { \left( { \beta }_{ 2 } \right) } \)

\( { T }_{ 1 }={ T }_{ 2 }\left( \frac { \sin { \left( { \beta }_{ 2 } \right) } }{ \sin { \left( { \beta }_{ 1 } \right) } } \right) \) \( { T }_{ 1 }\cos { \left( { \beta }_{ 1 } \right) } +{ T }_{ 2 }\cos { \left( { \beta }_{ 2 } \right) }=W \) \( { T }_{ 2 }=\frac { W }{ \cos { \left( { \beta }_{ 1 } \right) } \frac { \sin { \left( { \beta }_{ 2 } \right) } }{ \sin { \left( { \beta }_{ 1 } \right) } } +\cos { \left( { \beta }_{ 2 } \right) } } \quad \quad \quad { T }_{ 2 }=10.77 kN\quad \quad \quad \leftarrow \) \( { T }_{ 1 }={ T }_{ 2 }\frac { \sin { \left( { \beta }_{ 2 } \right) } }{ \sin { \left( { \beta }_{ 1 } \right) } } \quad \quad \quad { T }_{ 1 }=13.18kN\quad \quad \quad \leftarrow \)

\( { T }_{ 1 }\cos { \left( { \beta }_{ 1 } \right) } +{ T }_{ 2 }\cos { \left( { \beta }_{ 2 } \right) } =23.1\)
SHEAR & BEARING STRESSES

\( { d }=18㎜\quad \quad \quad t=100㎜ \)
\( { A }_{ S }=\frac { \pi }{ 4 } { d }_{ p }^{ 2 }\quad \quad \quad { A }_{ b }=t{ d }_{ p } \)

\( { \tau }_{ 1ave }=\frac { \frac { { T }_{ 1 } }{ 2 } }{ { A }_{ S } } \quad \quad \quad { \tau }_{ 1ave }=25.9㎫\quad \quad \quad \leftarrow \) \( { \tau }_{ 2ave }=\frac { \frac { { T }_{ 2 } }{ 2 } }{ { A }_{ S } } \quad \quad \quad { \tau }_{ 2ave }=21.2㎫\quad \quad \quad \leftarrow \) \( { \sigma }_{ b1 }=\frac { { T }_{ 1 } }{ { A }_{ b } } \quad \quad \quad { \sigma }_{ b1 }=7.32㎫\quad \quad \quad \leftarrow \) \( { \sigma }_{ b2 }=\frac { { T }_{ 2 } }{ { A }_{ b } } \quad \quad \quad { \sigma }_{ b2 }=5.99㎫\quad \quad \quad \leftarrow \)
