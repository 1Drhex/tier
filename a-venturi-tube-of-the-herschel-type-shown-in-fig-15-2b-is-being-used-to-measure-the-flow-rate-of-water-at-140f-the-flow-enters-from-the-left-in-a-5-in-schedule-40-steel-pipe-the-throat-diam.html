Guest Question.

A venturi tube of the Herschel type shown in Fig. 15.2(b) is being used to measure the flow rate of water at 140°F. The flow enters from the left in a 5-in Schedule 40 steel pipe. The throat diameter d is 2.200 in. The venturi is rough cast. The manometer fluid is mercury (sg = 13.54) and the deflection h is 7.40 in. Compute the velocity of flow in the pipe and the volume flow rate in gal/min.

Admin
Solution

We will use Eq. (15–6) to compute the velocity of flow in the pipe, \(v_1\). Then, we will find the volume flow rate from \( Q = A_1v_1\).

First let’s document pertinent data and compute some of the basic parameters in Eq. (15–6).

Fluid flowing in the pipe: water at 140°F; \(\gamma_w = 61.4 lb/ft^3\), \( \nu = 5.03 \times 10^{-6} ft^2/s\) (from Appendix A)

Manometer fluid: mercury (sg = 13.54); \(\gamma_m = \left(13.54\right) \left(62.4lb/ft^3\right) =844.9lb/ft^3.\)

Pipe dimensions: D = 0.4206 ft, \( A_1 = 0.1390 ft^2\) (from Appendix F)

Throat dimensions: d = (2.20 in) (1.0 ft/12 in) = 0.1833 ft, \( A_2 = \pi d^2/4 = 0.02640 ft^2\)

Then,

\( A_1/A_2 = \left(0.1390ft^2\right) /\left(0.0264ft^2\right) =5.265\)

\( \beta = d/D= \left(0.1833ft\right) /\left(0.4206ft\right) =0.436.\)

Note that 0.30 <\(\beta \) < 0.75, within the recommended range

Figure 15.3 applies giving the value of the discharge coefficient C for the rough-cast venturi. Let’s assume that the Reynolds number for the flow of water in the pipe is greater than \( 2.0 \times 10^5\) and use the value of C = 0.984 as the first estimate. This must be checked later when the Reynolds number is known and adjusted according to Fig. 15.3 if \( N_R \lt 2.0 \times 10^5. \)

Let’s evaluate the term \(\left[\left(\gamma_m/\gamma_f\right)-1\right] \) first:

\( \left[\left(\gamma_m/\gamma_f\right)-1\right] =\left[\left(844.9lb/ft^3/61.4lb/ft^3\right)-1 \right] =12.76\)

Also, let’s convert the h value to feet:

\( h = \left(7.40in\right) \left(1ft/12in\right) =0.6167ft\)

Now we can compute v, from Eq. (15–6):

\( v_1 = C\sqrt{\frac{2gh\left[\left(\gamma_m/\gamma_f\right)-1 \right] }{\left(A_1/A_2\right)^2 -1 } } =0.984\sqrt{\frac{2\left(32.2ft/s^2\right)\left(0.6167ft\right) \left(12.76\right) }{\left(5.265\right)^2 -1 } } \)

\( v_1 = 4.285ft/s\)

Now we must check the Reynolds number for the flow in the pipe using this value:

\( N_R = \frac{v_1D}{\nu} =\frac{\left(4.285ft/s\right)\left(0.4206ft\right) }{5.03 \times 10^{-6}} =3.58 \times 10^5\)

We note that this value is greater than \( 2 \times 10^5\) as we initially assumed. Then, the value for the discharge coefficient, C = 0.984, is correct and the calculation for \( v_1 \) is also correct. If the Reynolds number was less than \( 2 \times 10^5\), we would read a new value of C from Fig. 15.3 and recompute the velocity.

Result Now we complete the problem by computing the volume flow rate Q:

\( Q = A_1 v_1 = \left(0.1390ft^2\right) \left(4.285ft/s\right) =0.596 ft^3/s\)

Converting this to gal/min, we obtain

\( Q = \left(0.596ft^3/s\right) \left[\left(449gal/min\right)/1.0ft^3/s \right] =267 gal/min\)
