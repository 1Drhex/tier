Design a stiffened seated connection between a W21×68 beam and a W14×90 column flange, to support the following end reactions:
\(\begin{array}{l}R_{D}=21 \text { kips } \\R_{L}=62.5 \text { kips }\end{array}\)
Use 3/4 in. diameter ASTM A325-N bolts in standard holes to connect the supported beam to the seat plate and top angle. Use 70 ksi electrode welds to connect the stiffener and top angle to the column flange.

Determine weld requirements for seat plate

Use 5/16-in. fillet welds on each side of the stiffener. Minimum length of seat-plate-to-column flange weld is 0.2(L) = 3 in.
