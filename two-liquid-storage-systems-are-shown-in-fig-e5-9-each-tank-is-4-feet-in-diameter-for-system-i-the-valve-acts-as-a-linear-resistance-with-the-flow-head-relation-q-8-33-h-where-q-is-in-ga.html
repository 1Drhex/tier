Guest Question.

Two liquid storage systems are shown in Fig. E5.9. Each tank is 4 feet in diameter. For System I, the valve acts as a linear resistance with the flow-head relation \(q=8.33 h\), where \(q\) is in gal/min and \(h\) is in feet. For System II, variations in liquid level \(h\) do not affect exit flow rate \(q\). Suppose that each system is initially at steady state with \(\bar{h}=6 ft\) and \(\bar{q}_{i}=50 gal / min\) and that at time \(t=0\) the inlet flow rate suddenly changes from 50 to \(70 gal / min\). For each system, determine the following information:

(a) The transfer function \(H^{\prime}(s) / Q_{i}^{\prime}(s)\) where the primes denote deviation variables.

(b) The transient response \(h(t)\).

(c) The new steady-state levels.

(d) If each tank is \(8 ft\) tall, which tank overflows first? When?

(e) Verify the results in (d) using computer simulation.

Admin
Solution

a) Material balance over tank 1.

\(A \frac{d h}{d t}=C\left(q_{i}-8.33 h\right)\)

 

\(\begin{aligned}\text{where }\quad &A=\pi \times(4)^{2} / 4=12.6 ft ^{2} \\&C=0.1337 \quad \frac{ ft ^{3} / min }{ USGPM }\end{aligned}\)

 

\(\begin{aligned}&A s H^{\prime}(s)=C Q_{i}^{\prime}(s)-(C \times 8.33) H^{\prime}(s) \\&\frac{H^{\prime}(s)}{Q_{i}^{\prime}(s)}=\frac{0.12}{11.28 s+1}\end{aligned}\)

For tank 2,

\(A \frac{d h}{d t}=C\left(q_{i}-q\right)\)

 

\(A s H^{\prime}(s)=C Q_{i}^{\prime}(s) \quad, \quad \frac{H^{\prime}(s)}{Q_{i}^{\prime}(s)}=\frac{0.011}{s}\)

b)

\(Q_{i}^{\prime}(s)=20 / s\)

 

\(\begin{aligned}\text{For tank 1, }\quad &H^{\prime}(s)=\frac{2.4}{s(11.28 s+1)}=\frac{2.4}{s}-\frac{27.1}{11.28 s+1}\\&h(t)=6+2.4\left(1- e ^{-t / 11.28}\right)\end{aligned}\)

 

\(\begin{aligned}\text{For tank 2, }\quad &H^{\prime}(s)=0.22 / s^{2}\\&h(t)=6+0.22 t\end{aligned}\)

c)

\(\text { For tank } 1, \quad h(\infty)=6+2.4-0=8.4 ft\)

 

\(\text { For tank } 2, \quad h(\infty)=6+(0.22 \times \infty)=\infty ft\)

d)

\(\text { For tank } 1, \quad 8=6+2.4\left(1- e ^{-t / 11.28}\right)\)

 

\(\text{For tank 2,}\quad\begin{aligned}&h=8 ft \text { at } t=20.1 min \\&8=6+0.22 t \\&h=8 ft \text { at } t=9.4 min\end{aligned}\)

Tank 2 overflows first, at 9.4 min.

e)

The red line \(\left( h ^{\prime}( t )=2 ft\right.\), or \(h ( t )=8 ft\) ) shows that tank 2 overflows first at \(9.4 min\).
