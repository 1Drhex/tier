Guest Question.

A symmetrical \(p^+\)–n–\(p^+\) bipolar transistor has the following properties:

Emitter Base
\(A=10^{-4}cm^2\) \(N_{a}=10^{17}\) \(N_d=10^{15}cm^{-3}\)
\(W_{b}=1\mu m\) \(\tau _{n}=0.1\mu s\) \(\tau _{p}=10\mu s\)
\(\mu _{p}=200\) \(\mu_{n}=1300cm^2/V-s\)
\(\mu _{n}=700\) \(\mu_{p}=450cm^2/V-s\)
Admin
Solution

In the base,

\(p_{n}={n_{i}}^2/n_{n}=(1.5\times 10^{10})^2/10^{15}=2.25\times 10^5\)

\(D_{p}=450(0.0259)=11.66,L_{p}=(11.66\times 10^{-5})^{1/2}=1.08\times 10^{-2}\)

\(W_{b}/L_{p}=10^{-4}/1.08\times 10^{-2}=9.26\times 10^{-3}\)

\(I_{ES}=I_{CS}=qA\left(D_{p}/L_{p}\right)p_{n}ctnh \left(W_{b}/L_{p}\right)=(1.6\times 10^{-19})(10^{-4}) (11.66/1.08\times 10^{-2}) (2.25\times 10^5)ctnh 9.26\times 10^{-3}=4.2\times 10^{-13}A\)

\(\Delta p_{E}=p_{n}e^{qV_{EB}/KT},\Delta p_{c}\approx0\)

\(\Delta p_{E}=2.25\times 10^5\times e^{0.3/0.0259}=2.4\times 10^{10}\)

\(I_{B}=qA(D_{p}/L_{p})\Delta p_{E}\tanh \left(W_{b}/2L_{p}\right) \)

or

\(I_{B}=\frac{Q_{b}}{\tau _{p}}=qAW_{b}\Delta p_{E}/2\tau _{p}=\pmb{1.9\times 10^{-12}A}\)

In the emitter,
\(D_{n}=700(0.0259)=18.13\)

\(L_{n}=(18.13\times 10^{-7})^{1/2}=1.35\times 10^{-3}\)

 

\(I_{En}=\frac{qAD_{n}^E}{L_{n}^E}n_{p}^Ee^{qV_{EB}/KT}\)

 

\(I_{Ep}=\frac{qAD_{p}^B}{L_{p}^B}p_{n}^Bctnh\frac{W_{b}}{L_{p}^B}e^{qV_{EB}/KT}\)

 

\(\gamma =\frac{I_{Ep}}{I_{En}+I_{Ep}}=\left[1+\frac{I_{En}}{I_{Ep}}\right]^{-1}\)

 

\(\begin{matrix} \gamma =\left[1+\frac{D^E_n/L^E_nn^E_p}{D^B_p/L^B_pn^B_n}\tanh \frac{W_b}{L_p}\right]^{-1} & \left(Use \frac{n^E_p}{p^B_n}=\frac{n^B_n}{p^E_p}\right) \end{matrix} \)

 

\(=\left[1+\frac{18.13\times 1.08\times 10^{-2}\times 10^{15}}{11.66\times 1.35\times 10^{-3}\times10^{17}}\tanh 9.26\times 10^{-3}\right]^{-1}=\pmb{0.99885}\)

 

\(B=sech\frac{W_{b}}{L_{p}}=sech 9.26\times 10^{-3}=\pmb{0.99996}\)

 

\(\alpha =B\gamma =(0.99885)(0.99996)=\pmb{0.9988}\)

 

\(\beta =\frac{\alpha }{1-\alpha }=\frac{0.9988}{0.0012}=\pmb{832}\)
