Guest Question.

The 4-cm-diameter smooth, horizontal pipe of Fig. E7.10 transports \(0.004 m^3/s\) of water at 20°C. Using the power-law profile, approximate (a) the friction factor, (b) the maximum velocity, (c) the radial position where u = V, (d) the wall shear, (e) the pressure drop over a 10-m length, and (f) the maximum velocity using Eq. 7.6.16.

\(\frac{u_{\max }}{u_{\tau}}=2.44 \ln \frac{u_{\tau} r_{0}}{\nu}+5.7\) (smooth pipes) (7.6.16)

Admin
Solution

(a) The average velocity is calculated to be

\(V=\frac{Q}{A}=\frac{0.004}{\pi \times 0.02^{2}}=3.18\)

The Reynolds number is

\(\mathrm{Re}=\frac{V D}{\nu}=\frac{3.18 \times 0.04}{10^{-6}}=1.27 \times 10^{5}\)

From Table \(7.1\) we see that \(n \cong 7.5\) and from Eq. 7.6.21,

\( n = \frac{1}{\sqrt{f}} (7.6.21)\)

\(\begin{aligned}f &=\frac{1}{n^{2}} \\&=\frac{1}{7.5^{2}}=0.018\end{aligned}\)

(b) The maximum velocity is found using Eq. \(7.6 .20\) to be

\(V=\frac{1}{\pi r_{0}^{2}} \int_{0}^{r_{0}} \bar{u}(r) 2 \pi r d r=\frac{2 n^{2}}{(n+1)(2 n+1)} u_{\max } (7.6.20)\)

\(\begin{aligned}u_{\max } &=\frac{(n+1)(2 n+1)}{2 n^{2}} V \\&=\frac{8.5 \times 16}{2 \times 7.5^{2}} \times 3.18=3.84 \mathrm{~m} / \mathrm{s}\end{aligned}\)

(c) The distance from the wall where \(u=V=3.18 \mathrm{~m} / \mathrm{s}\) is found using Eq. \(7.6 .19\) as follows:

\(\frac{\bar{u}}{u_{\max }}=\left(\frac{y}{r_{0}}\right)^{1 / n} (7.6.19)\)

\(\begin{aligned}\frac{\bar{u}}{u_{\max }} &=\left(\frac{y}{r_{0}}\right)^{1 / 7.5} \\\therefore y &=r_{0}\left(\frac{u}{u_{\max }}\right)^{7.5} \\&=2\left(\frac{3.18}{3.84}\right)^{7.5}=0.49 \mathrm{~cm}\end{aligned}\)

The radial position is thus

\(\begin{aligned}r &=r_{0}-y \\&=2-0.49=1.51 \mathrm{~cm}\end{aligned}\)

(d) The wall shear is found using Eq. 7.3.19 and is

\( f = \frac{\tau_{0}}{\frac{1}{8} \rho V^{2}} (7.3.19)\)

\(\begin{aligned}\tau_{0} &=\frac{1}{8} \rho V^{2} f \\&=\frac{1}{8} \times 1000 \times 3.18^{2} \times 0.018=23 \mathrm{~Pa}\end{aligned}\)

(e) The pressure drop is calculated using Eq. \(7.6 .18\) with \(\Delta p / L=-d p / d x\) to be

\( \tau_{0} = – \frac{r_{0}}{2} \frac{d p}{d x} (7.6.18)\)

\(\begin{aligned}\Delta p &=\frac{2 \tau_{0} L}{r_{0}} \\&=\frac{2 \times 23 \times 10}{0.02}=23000 \mathrm{~Pa} \quad \text { or } \quad 23 \mathrm{kPa}\end{aligned}\)

(f) To use Eq. \(7.6 .16\) we must know the shear velocity. It is

\(\begin{aligned}u_{\tau} &=\sqrt{\frac{\tau_{0}}{\rho}} \\&=\sqrt{\frac{23}{1000}}=0.152 \mathrm{~m} / \mathrm{s}\end{aligned}\)

Then we find \(u_{\max }\) to be

\(u_{\max }=0.152\left(2.44 \ln \frac{0.152 \times 0.02}{10^{-6}}+5.7\right)=3.84 \mathrm{~m} / \mathrm{s}\)

This is the same as that given by the power-law formula in part (b). This answer is considered to be more accurate if it differs from that of Eq. 7.6.20. Note that the experimental data do not allow for accuracy in excess of three significant digits, and often to only two significant digits.
