Guest Question.

Given the potential field in free space, V = \(80\phi { a }_{ \phi }\)(cyl. coord.), find : (a) the energy stored in the region \(2< \rho < 4 cm, 0 < \phi <0.2\pi . 0 < z < 1m\); (b) the potential difference \({ V }_{ AB }\) for A\((3 cm, \phi = 0, z = 0)\) and B\((3cm, 0.2\pi, 1m)\); (C) the maximum value of the energy density in the specified region.

Admin
Solution

a) First we find
\(E=-\triangledown V=-\frac { 1 }{ \rho } \frac { dV }{ d\phi } { a }_{ \phi }=-\frac { 80 }{ \rho } { a }_{ \phi }V/m\)
Then
\({ W }_{ E }=\int _{ \upsilon }^{ }{ } { \omega }_{ E }d\upsilon =\int _{ 0 }^{ 1 }{ } \int _{ 0 }^{ 0.2\pi }{ } \int _{ .02 }^{ .04 }{ } \frac { 1 }{ 2 } { \epsilon }_{ 0 }\frac { { \left( 80 \right) }^{ 2 } }{ { \rho }^{ 2 } } \rho d\rho d\phi dz=640\pi { \epsilon }_{ 0 }\ln { \left( \frac { .04 }{ .02 } \right) } =12.3nJ\)
b) Use
\({ V }_{ AB }=-\int _{ B }^{ A }{ } E.dL=-\int _{ .2\pi }^{ 0 } -\frac { 80 }{ \rho } { a }_{ \phi }.{ a }_{ \phi }\rho d\phi =-80(0.2\pi )=-16\pi V\)
c)The energy density is
\({ \omega }_{ E }=\frac { 1 }{ 2 } { \epsilon }_{ 0 }{ E }^{ 2 } = \frac { 1 }{ 2 } { \epsilon }_{ 0 }\frac { 6400 }{ { \rho }^{ 2 } }\)
This will maximize at the lowest value of \(\rho\) in the speciﬁed range, which is \(\rho\) = 2 cm. So
\({ \omega }_{ E , max }=\frac { 1 }{ 2 } { \epsilon }_{ 0 }\frac { 6400}{ {.02 }^{ 2} }=7.1 \times { 10 }^{ -5 } J/{ m}^{ 3} = 71 \mu J/{ m}^{ 3}\)
