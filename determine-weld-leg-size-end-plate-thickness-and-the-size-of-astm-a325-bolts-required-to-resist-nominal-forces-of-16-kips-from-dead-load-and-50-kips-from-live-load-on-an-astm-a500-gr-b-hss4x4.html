Guest Question.

Determine weld leg size, end plate thickness, and the size of ASTM A325 bolts required to resist nominal forces of 16 kips from dead load and 50 kips from live load on an ASTM A500 Gr. B HSS4×4×1/4 section. The end plate is ASTM A36.

Admin
Solution

Material Properties:

Strut ASTM A500 Gr. B \(F_{y}=46 ksi \quad F_{u}=58 ksi\)

End Plate ASTM A36 \(F_{y}=36 ksi \quad F_{u}=58 ksi\)

Manual
Table 2-3

Geometric Properties:

Strut HSS 4×4×1/4 t = 0.233 in. A = 3.37 \(\text { in. }^{2}\)

End Plate b = a = 1.50 in.

Manual
Table 1-12

Calculate the required tensile strength

LRFD ASD
\(\begin{aligned}P_{u} &=1.2(16.0 kips )+1.6(50.0 \text { kips }) \\&=99.2 kips\end{aligned}\) \(\begin{aligned}P_{a} &=16.0 \text { kips }+50.0 \text { kips } \\&=66.0 \text { kips }\end{aligned}\)

Preliminary size of the (4) ASTM A325 bolts

LRFD ASD
\(r_{u t}=\frac{P_{u}}{n}=\frac{99.2 kips }{4}=24.8 kips\) Try 3/4 -in. diameter ASTM A325 bolts \(\phi r_{n}=29.8 \text { kips }\) \(r_{a t}=\frac{P_{a}}{n}=\frac{66.0 kips }{4}=16.5 kips\) Try 3/4 -in. diameter ASTM A325 bolts \(r_{n} / \Omega=19.9 \text { kips }\)

Manual
Table 7-2

Calculate required end-plate thickness with consideration of prying action

\(a^{\prime}=a+\frac{d_{b}}{2} \leq 1.25 b+\frac{d_{b}}{2}\)

Manual
Section 9

\(=1.50 \text { in. }+\frac{3 / 4}{2} \leq 1.25(1.50 \text { in. })+\frac{3 / 4}{2}\)

 

\(=1.88 \text { in. } \leq 2.25 \text { in. } \quad \text { o.k. }\)

 

\(b^{\prime}=b-\frac{d_{b}}{2}=1.5 \text { in. }-\frac{3 / 4}{2}=1.12 \text { in. }\)

 

\(\rho=\frac{b^{\prime}}{a^{\prime}}=\frac{1.12}{1.88}=0.596\)

 

\(\delta=1-\frac{d^{\prime}}{p}=1-\frac{(3 / 4+1 / 8)}{4.0}=0.781\)

 

LRFD ASD
\(\beta=\frac{1}{\rho}\left(\frac{\phi r_{n}}{r_{u t}}-1\right)=\frac{1}{0.596}\left(\frac{29.8}{24.8}-1\right)=0.338\)

 

\(\begin{aligned}\alpha^{\prime} &=\frac{1}{\delta}\left(\frac{\beta}{1-\beta}\right) \\&=\frac{1}{0.781}\left(\frac{0.338}{1-0.338}\right)=0.654 \leq 1.0\end{aligned}\)

 \(\beta=\frac{1}{\rho}\left(\frac{r_{n} / \Omega}{r_{a t}}-1\right)=\frac{1}{0.596}\left(\frac{19.9}{16.5}-1\right)=0.346\)

 

\(\begin{aligned}\alpha^{\prime} &=\frac{1}{\delta}\left(\frac{\beta}{1-\beta}\right) \\&=\frac{1}{0.781}\left(\frac{0.346}{1-0.346}\right)=0.677 \leq 1.0\end{aligned}\)

Manual
Section 9

LRFD ASD
\(\begin{aligned}t_{r e q} &=\sqrt{\frac{4.44 r_{u t} b^{\prime}}{p F_{u}\left(1+\delta \alpha^{\prime}\right)}} \\&=\sqrt{\frac{4.44(24.8 kips )(1.12 in. )}{4.00 in. (58 ksi )(1+0.781(0.654))}} \\&=0.593 in .\end{aligned}\)

 

 

Use 5/8 in. end plate, \(t_{1}\) > 0.593, further bolt check for prying not required.

 

Use (4) 3/4 in. diameter A325 bolts

 \(\begin{aligned}t_{r e q} &=\sqrt{\frac{6.66 r_{a t} b^{\prime}}{p F_{u}\left(1+\delta \alpha^{\prime}\right)}} \\&=\sqrt{\frac{6.66(16.5 kips )(1.12 in .)}{4.00 in \cdot(58 ksi )(10.781(0.677))}} \\&=0.589 in .\end{aligned}\)

Manual
Section 9

Use 5/8 in. end plate, \(t_{1}\) > 0.589, further bolt check for prying not required.

 

Use (4) 3/4 in. diameter A325 bolts

Calculate the weld size required

\(F_{w}=0.60 F_{e x x}\left(1.0+0.5 \sin ^{1.5}(\theta)\right)=0.60(70 ksi )\left(1.0+0.5 \sin ^{1.5}(90)=63.0 ksi \right.\)

Eqn. J2-5

\(l=4(4.00 in .)=16.0 in\)

 

LRFD ASD
For shear and fillet welds \(\phi\)= 0.75

 

\(\begin{aligned}w & \geq \frac{P_{u}}{\phi F_{w}(0.707) l} \\& \geq \frac{99.2 \text { kips }}{0.75(63.0 ksi )(0.707)(16.0 in .)} \\&=0.186 in .\end{aligned}\)

 For shear and fillet welds \(\Omega\) = 2.00

 

\(\begin{aligned}w & \geq \frac{\Omega P_{a}}{F_{w}(0.707) l} \\& \geq \frac{2.00(66.0 kips )}{(63.0 ksi )(0.707)(16.0 in .)} \\&=0.185 in.\end{aligned}\)

Manual
Section 8

Check minimum weld size requirements

For t = 5/8 in. minimum weld = 1/4 in.

Table J2.4

LRFD ASD
0.250 in. > 0.186 in. Use 1/4 in. weld leg size 0.250 in. > 0.185 in. Use 1/4 in. weld leg size

Results

Use 1/4 in. weld with 5/8 in. end plate and (4) ¾-in. diameter ASTM A325 bolts.
