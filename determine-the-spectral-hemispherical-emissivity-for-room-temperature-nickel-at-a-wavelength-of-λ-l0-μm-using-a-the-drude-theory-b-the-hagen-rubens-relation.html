Guest Question.

Determine the spectral, hemispherical emissivity for room-temperature nickel at a wavelength of λ= l0μm, using (a) the Drude theory, (b) the Hagen-Rubens relation.

Admin
Solution

We first need to determine the optical constants n and k from either theory, then calculate the hemispherical emissivity from equation

\(\epsilon _{\shortparallel } =\frac{8n}{n^{2} +k^{2} } \left(1-\frac{n}{n^{2} +k^{2}}\ln [(n+1)^{2}+ k^{2} ]+\frac{(n^{2} -k^{2})}{k(n^{2} +k^{2})}\tan ^{-1} \frac{k}{n+1} \right)\),

\(\epsilon _{\bot }=8n \left(1-n\ln \frac{(n+1)^{2} +k^{2}}{n^{2} +k^{2}}+\frac{(n^{2} -k^{2})}{k} \tan ^{-1}\frac{k}{n(n+1)+ k^{2}} \right)\) ,

\(\epsilon _{\lambda } =\frac{1}{2} (\epsilon _{\shortparallel }+\epsilon _{\bot })\).

or read it from Fig. 3-10.

(a) Using values for nickel from Table 3.3 in equation

\(\varepsilon ^{\prime } (\nu )=\varepsilon _{0} -\frac{\nu ^{2}_{p} }{\nu ^{2}+\gamma ^{2} } , \varepsilon ^{\prime \prime }(\nu )=\frac{\nu ^{2}_{p} \gamma }{\nu (\nu ^{2}+\gamma ^{2})}\) .

we find with \(\nu =c_{0} /\lambda =2.998\times 10^{8} m/s/10^{-5}m=2.998\times 10^{13} Hz\),

\(\varepsilon ^{\prime } =1.0-3.72\times 10^{30}/[(2.998\times 10^{13})^{2} +(1.62\times 10^{13})^{2} ]=1-3204=-3203\),

\(\varepsilon ^{\prime \prime }=3204\times 1.62 \times 10^{13}/2.998\times 10^{13}=1731\),

\(n^{2} =0.5\times \left(-3208+\sqrt{3208^{2}+1731^{2} } \right) =219\),

\(k^{2}=0.5\times \left(3208+\sqrt{3208^{2}+1731^{2} } \right)=3422\),

and

\(n=14.8 , k=58.5 , k/n=58.5/14.8=3.95\).

To use Fig. 3-10 we first determine \(\rho _{n\lambda }\) as

\(\rho _{n\lambda } =\frac{13.8^{2}+58.5^{2} }{15.8^{2} +58.5^{2}} =0.984\) ,

and

\(\epsilon _{n\lambda }=1-\rho _{n\lambda }=0.016\).

From Fig. 3-10 \(\epsilon _{\lambda }/\epsilon _{n\lambda }\simeq 1.29\). and, therefore, \(\epsilon _{\lambda }\simeq 0.021\) .

(b) Using the Hagen-Rubens relation we find, from equation

\(\epsilon _{n\lambda }=\frac{2}{n} -\frac{2}{n^{2} } +…..,\)

\(\epsilon _{n\lambda }\simeq \frac{2}{\sqrt{30\lambda \sigma _{dc} } }-\frac{1}{15\lambda \sigma _{dc}} ,\lambda\) in \(cm , \sigma _{dc}\) in \(\Omega ^{-1} cm^{-1}\) .

\(\epsilon _{n\lambda }=\frac{2}{\sqrt{30\times 10^{-3} \times 1.28\times 10^{5} } }-\frac{1}{15\times 10^{-3} \times 1.28\times 10^{5}} =0.032\) .

Further, with \(n\simeq k\simeq \sqrt{30\times 10^{-3} \times 1.28\times 10^{5} }=62.0\), we obtain from Fig. 3-10 \(\epsilon _{\lambda }/\epsilon _{n\lambda }\simeq1.275\) and \(\epsilon _{\lambda } \simeq 0.041\) .
