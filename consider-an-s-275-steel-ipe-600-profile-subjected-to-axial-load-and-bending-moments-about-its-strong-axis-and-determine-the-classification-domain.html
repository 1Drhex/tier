Material data:
S 275 steel grade \(f_y\)= 275 MPa (\(F_y \)= 39.9 ksi)
EC3 procedure
For S 275 steel grade \(ε = \sqrt{\frac{235}{f_{y}}}=\sqrt{\frac{235}{275}}=0.924\)
Flange
\(\frac{c}{t}=\frac{b-t_{w}-(2.r)}{2.t_{f}}=\frac{220−12− (2×24)}{2×19}=\frac{160}{38}=4 21 ≤ 9 ε=9×0.924 = 8.32\)
Flange is class 1
Web in compression
\(\frac{c}{t}=\frac{h− 2.t_{f}− 2.r}{t_{w}}=\frac{600−(2×19) −(2×24)}{12}=\frac{514}{12}=42.83 > 42.ε=42×0.924 = 38.83\)
Web in compression is class 4
Web in flexure
\(\frac{c}{t}=\frac{h− 2.t_{f}− 2.r}{t_{w}}=\frac{600−(2×19) −(2×24)}{12}=\frac{514}{12}=42.83 < 72.ε=72×0.924 = 66.56\)
Web in flexure is class 1
If member is subjected to axial load and bending, class for the section changes depends on the entity of axial load
The parameter α for transition from class 1 to class 2 is therefore determined
\(\alpha =\frac{396.ε + c/t_{w}}{13.(c/t_{w})}=\frac{396 × 0.924 + 42.83}{13 × 42.83}=0.7341\)
In case of pure flexure, limit moment \(M_{Lim}\) coincides with plastic moment, and is determined as follows
\(M_{Lim} =M_{pl} =W_{pl}.f_{y} = 3512 × 275.10^{-3}= 965.8 kNm(712.3 kip-ft)\)
Compute with Eqs. (4.12a) and (4.12b) limit values for axial load and bending moment at transition from class 1 to class 2

\(N^{1-2}_{Ed}=\begin{bmatrix}2\cdot \frac{396\cdot \varepsilon +(c/t_{w})}{13\cdot (c/t_{w})}-1\end{bmatrix}\cdot (c\cdot t_{w})\cdot f_{y}\) (4.12a)

\(M^{1-2}_{Ed}=M_{pl}-\frac{\begin{Bmatrix}\begin{bmatrix}2\cdot \frac{396\cdot \varepsilon +(c/t_{w})}{13\cdot (c/t_{w})}-1\end{bmatrix}\cdot c\end{Bmatrix}^{2}\cdot t_{w}}{4}\cdot f_{y}\) (4.12b)

\(N^{1-2}_{Ed}=[2.a-1].(c.t_{w}).f_{y}=[2×0.7341−1] × (514 × 12) × 275.10^{-3}= 794.2 kN(174.5)kips = 794.2 kN (174.5) kips\)
\(M_{Ed}^{1-2}=M_{pl}-\frac{{[2.\frac{369.ε + (c/t_{w}}{13.(c/t_{w})}-1].c}^{2}t_{w}}{4}f_{y}=965.8-\frac{{[2\times \frac{396 × 0.924 + 42.83}{13 × 42.83}-1]\times 514}^{2}\times 12}{4}× 275.10^{−6}= 965.8 – 47.8 = 918.0 kNm(178.5 kip-ft)\)
In the same way, limit values for axial load and bending moment at transition from class 2 to class 3 shall be
\(N^{2-3}_{Ed}=\left [ 2.\frac{456.\varepsilon+(c/t)}{13.(c/t)}-1 \right ].(d.t_{w}).f_{y}=\left [ 2.\frac{456\times 0.924+42.83}{13.42.83} -1\right ]\times (514\times 12)\times 275.10^{-3}=1131.9 kN(254.5 kips)\)
\(M_{Ed}^{2-3}=M_{pl}-\frac{\left \{ [2.\frac{456.\varepsilon +(c/t_{w})}{13.(c/t_{w})}-1].c \right \}^{2}.t_{w}}{4}.f_{y}=965.8\frac{\left \{ \left [ 2\times \frac{456\times 0.924 + 42.83}{13\times 42.83}-1 \right ]\times 514 \right \}^{2}\times 12}{4}\times 275.10^{-6} = 965.8-97.1 = 868.7 kNm\)(640.7 kips-ft)
Using the same method also for elastic behaviour, taking into account the relationship between axial load and parameter ψ, related to elastic stress distribution in web, limit values for axial load and bending moment at transition from class 3 to class 4 are computed according to Eqs. (4.14a) and (4.14b)
\(N^{3-4}_{Ed}=\frac{1}{2}.\left[\frac{42.\varepsilon -0.67.(c/t)}{0.33.(c/t)}+1\right ].A.f_{y}=\frac{1}{2}\left [ \frac{42\times 0.924-0.67\times (42.83)}{0.33\times 42.83}+1 \right ]\times 156\times 275.10^{-1}=\frac{1}{2}.(0.7154+1)\times 156\times 275.10^{-1}=3682 kN(827.7)kips\)
\(M_{Ed}^{3-4}=\left \{ f_{y}-\frac{1}{2}.\left [ \frac{42.\varepsilon-0.67.(c/t_{w})}{0.33.(c/t_{w})}+1 \right ].f_{y} \right \}.W_{el}=\left \{ 275-\frac{1}{2}\times \left [ \frac{42\times 0.924-0.67\times 42.83}{0.33\times 42.83}+1 \right ]\times 275 \right \}\times 3069.10^{-6}= (275−235.87)\times 3069.10^{-6} = 120.1 kNm (88.6 kip-ft)\)
The compressive strength for axial load only \(N_{pl}\) computed using gross area is
\(N_{pl} = A.f_{y} = 156 × 275 10^{−1} = 4290 kN(964.4) kips\)
When \(N_{pl}\) is larger than limit axial load at transition from class 3 to class 4, there are no limitations for compressive strength
The section is class 4 due to high web slenderness, therefore, compressive strength shall be computed using the effective area (for a definition, see Section 4.3.3, for the calculation,)
\(A_{eff}=149.5 cm^{2}2(23.17 in.^{2})\)
Then member compressive strength for axial load only \(N_{Lim}\) shall be:
\(N_{Lim} = A_{eff}.f_{y} = 149.5 × 275.10^{−1} = 4112 kN(924.4 kips)\)
AISC procedure
For members subjected to flexure and axial load AISC prescribes to compute classification in case of axial load only and in case of bending moment only
Flange
(1) Axial load
\(\frac{b}{t}=\frac{b_{w}}{2.t_{f}}=\frac{8.66}{2\times 0.748}=5.79<=0.56\sqrt{\frac{E}{F_{y}}}=0.56\times \sqrt{\frac{29000}{39.9}}=15.1\)
Flange is non-slender
Web \(\frac{h}{t_{w}}=\frac{d-(2.t_{f})-(2.r)}{t_{w}}=\frac{23.62-(2\times0.748)-(2\times 0.945)}{0.472}=\frac{20.23}{0.472}=42.9> 1.49\sqrt{\frac{E}{F_{y}}}=1.49\times \sqrt{\frac{29000}{39.9}}=40.2\)
Web is slender
S 275 steel IPE 600 section, subjected to axial load, has nonslender flanges and slender web, therefore it is classified as slender (2) Flexure
Flange
Limiting width-to-thickness ratio compact/non-compact \(\lambda _{p}=0.38\sqrt{\frac{E}{F_{y}}}=0.38\times \sqrt{\frac{29000}{39.9}}=10.24\)
Limiting width-to-thickness ratio non-compact/slender
\(\lambda _{r}=1.0\sqrt{\frac{E}{F_{y}}}=1.0\times \sqrt{\frac{29000}{39.9}}=26.96\)
\(\frac{b}{t}=\frac{b_{w}}{2.t_{f}}=\frac{8.66}{2\times 0.748}=5.79\)
Flange is compact
Web
Limiting width-to-thickness ratio compact/non-compact
\(\lambda _{p}=3.76\sqrt{\frac{E}{F_{y}}}=3.76\times \sqrt{\frac{29000}{39.9}}=101.4\)
Limiting width-to-thickness ratio non-compact/slender
\(\lambda _{r}=5.70\sqrt{\frac{E}{F_{y}}}=5.70\times \sqrt{\frac{29000}{39.9}}=153.7\)
\(\frac{h}{t_{w}}=\frac{d-(2.t_{f})-(2.r)}{t_{w}}=\frac{23.62-(2\times 0.748) -(2\times 0.945)}{0.472}=\frac{20.23}{0.472}=42.9\)
Web is compact
S 275 steel IPE 600 section, subjected to flexure, has class compact
flanges and compact web, therefore, it is classified as compact Being the section non-slender for axial load only, the nominal compressive strength \(P_{n}\) shall be\(P_{n}=F_{cr}A_{g}=\left [0.658\frac{QFy}{Fe}\right ]QF_{y}A_{g}\)
For low slenderness \(KL/r value, F_{e}\gg QF_{y} \), therefore, this expression can be approximated as
\(P_{n} = F_{cr}A_{g} = QF_{y}A_{g}\)
\(Q=Q_{s}.Q_{a}\)
\(Q_{s} = 1\)(flanges are non-slender)
Compute \(Q_{a}\) because the web is slender
\(\frac{h}{t_{w}}=42.9> 1.49\sqrt{\frac{E}{f}}=1.49\times \sqrt{\frac{29000}{39.9}=40.2}\)
\((f=F_{cr}=39.9 Ksi)\)
Then \(b_{e}=1.92t\sqrt{\frac{E}{f}}\left [ 1-\frac{0.34}{(b/t)}\sqrt{\frac{E}{f}} \right ]=1.92 × 0.472 ×\sqrt{\frac{29000}{39.9}}\times \left [ 1-\frac{0.34}{42.8}\times \sqrt{\frac{29000}{39.9}} \right ]\)= 19.2 in.< h=20.23 in.(488mm< 514mm)
Effective area
\(A_{e}=A_{g}-(h-b_{e})t_{w}=24.18−(20.23−19.2)×0.472=23.69 in.^{2}(152.8 cm_{2})\)
\(Q_{a}=A_{e}/A_{g}=23.69/24.18=0.98\)
\(Q=Q_{s}.Q_{a}=1\times 0.98=0.98\)
\(P_{n}=QF_{y}A_{g}\)=0.98 × 39.9×24.18 =945.5 kips(4206 kN)
