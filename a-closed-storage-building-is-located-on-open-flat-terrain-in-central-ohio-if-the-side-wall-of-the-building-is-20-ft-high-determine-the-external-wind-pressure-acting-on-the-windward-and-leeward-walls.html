Guest Question.

A closed storage building is located on open flat terrain in central Ohio. If the side wall of the building is 20 ft high, determine the external wind pressure acting on the windward and leeward walls. Each wall is 60 ft long. Assume the roof is essentially flat.

Admin
Solution
\(V = 105 mi/h\)

 

\({ k }_{ zt }=1.0\)

 

\({ k }_{d }=1.0\)

 

\(q=0.00256{ k }_{ z }{ k }_{ zt }{ k }_{ d }{ V }^{ 2 }\)

 

\(\quad =0.00256{ k }_{ z }\left( 1.0 \right) \left( 1.0 \right) { \left( 105 \right) }^{ 2 }\)

 

\(\quad =28.22{ k }_{ z }\)

 

From Table 1–5

 

\(z\) \({ k }_{ z }\) \({ q }_{ z }\)
\(0–15\) \(0.85\) \(23.99\)
\(20\) \(0.9\) \(25.4\)

 

Thus, for windward wall

\(p=qG{ C }_{ p }-{ q }_{ h }\left( G{ C }_{ pi } \right)\)

 

\(\quad =q\left( 0.85 \right) \left( 0.8 \right) -25.40\left( \pm 0.18 \right)\)

 

\(\quad =0.68 q\pm 4.572\)

 

\({ p }_{ 0-15 }=0.68\left( 23.99 \right) \mp 4.572=11.7 psf\) or \(20.9 psf \)

 

\({ p }_{ 20 }=0.68\left( 25.40 \right) \mp 4.572=12.7 psf\) or \(21.8 psf\)

 

Leeward wall

\(\frac { L }{ B } =\frac { 60 }{ 60 } =1\) so that \({ C }_{ p }=-0.5\)

 

\(p=qG{ C }_{ p }-{ q }_{ h }\left( G{ C }_{ pi } \right)\)

 

\(p =25.40\left( 0.85 \right) \left( -0.5 \right) -25.40\left( \pm 0.18 \right)\)

 

\(p=-15.4 psf\) or \(-6.22 psf\)
