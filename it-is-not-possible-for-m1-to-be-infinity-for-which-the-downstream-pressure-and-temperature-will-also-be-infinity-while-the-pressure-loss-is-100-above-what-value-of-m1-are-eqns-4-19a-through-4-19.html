Guest Question.

It is not possible for \(M_{1}\) to be infinity, for which the downstream pressure and temperature will also be infinity while the pressure loss is 100%. Above what value of \(M_{1}\) are Eqns. (4.19a) \(M ^{2}_{2}= \underset{M_{1}\rightarrow \infty }{\lim } \left[\frac{M ^{2}_{1}+\frac{2}{\gamma -1} }{\frac{2\gamma }{\gamma -1}M ^{2}_{1} -1} \right] \) through (4.19f) \(\frac{p_{t2}}{p_{t1}} =e^{-\left(s_{2} -s_{1} \right) /R} \approx \left[\frac{\gamma +1}{\gamma -1} \right] ^{\gamma /\left(\gamma -1\right) } \left[\frac{\gamma +1}{2\gamma M ^{2}_{1}} \right] ^{1/\left(\gamma -1\right) } =\left[\frac{1}{\beta } \right]^{1/\left(\gamma -1\right) } \) good approximations?

Admin
Solution

One estimation can be obtained by assuming \(M_{2}\) to be 110% of \(\sqrt{\left(\gamma -1\right) /2\gamma } \).

In this case for \(Î³ = 1.4\), \(M_{2}\approx 0.416 \) and the normal shock table (Appendix D) gives \(M_{1}\approx 4.94 \). The results with and without the strong shock assumption for fluid properties are

\(\frac{T_{2} }{T_{1} }\) = 5.684 4.745 (strong shock approximation)

\(\frac{p_{2} }{p_{1} }\) = 28.304 28.47 (strong shock approximation)

\(\frac{\rho_{2} }{\rho_{1} }\) = 4.980 6.0 (strong shock approximation)

\(\frac{p_{t2} }{p_{t1} }\) = 0.065 0.122 (strong shock assumption)

If \(M_{2}\) is 105% of \(\sqrt{\left(\gamma -1\right) /2\gamma }\), then \(M_{2}\approx 0.397\) and \(M_{1}\approx 7.0\). Therefore,

\(\frac{T_{2} }{T_{1} }\) = 10.469 9.528 (strong shock approximation)

\(\frac{p_{2} }{p_{1} }\) = 57.0 57.167 (strong shock approximation)

\(\frac{\rho_{2} }{\rho_{1} }\) = 5.444 6.0 (strong shock approximation)

\(\frac{p_{t2} }{p_{t1} }\) = 0.015 0.021 (strong shock assumption)
