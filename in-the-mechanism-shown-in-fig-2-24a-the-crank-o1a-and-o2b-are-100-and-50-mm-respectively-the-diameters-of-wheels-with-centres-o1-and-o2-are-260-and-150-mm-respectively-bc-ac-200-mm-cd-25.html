Guest Question.

In the mechanism shown in Fig.2.24(a), the crank \( O_{1} A \text { and } O_{2} B \) are 100 and 50 mm, respectively. The diameters of wheels with centres \( O_{1} \text { and } O_{2} \) are 260 and 150 mm, respectively. BC = AC = 200 mm, CD = 250 mm. The wheels roll on each other. The crank \( O_{1} A \) rotates at 120 rpm. Determine (a) the velocity of the slider D, (b) the angular velocities of links BC and CD and (c) the torque at \( O_{1} B \) when the force required at D is 4 kN.

Admin
Solution

\( \omega=2 \pi \times \frac{120}{60}=12.57 rad / s \).

\( v_{a}=12.57 \times 0.1=1.257 m / s \).

\( v_{e}=v_{a} \times \frac{O_{1} E}{O_{1} A}=1.257 \times \frac{130}{100}=1.634 m / s \),

\( v_{f}=v_{e} \).

\( v_{b}=v_{f} \times \frac{O_{2} B}{O_{2} F}=1.634 \times \frac{50}{60}=1.362 m / s \).

\( \text { Draw } o_{1} a \perp O_{1} A \text { to represent the velocity } v_{a} \text { of point } A \text { to a scale of } 1 cm =0.25 m / s \), as shown in Fig.2.24(b). Extend \( o_1a\) to e such that

\( o_{1} e=o_{1} a \times \frac{O_{1} E}{O_{1} A} \),

\( =5 \times \frac{130}{100}=6.5 cm\).

\( \text { From } o_{2} \text { draw a line perpendicular to } O_{2} B \text { to intersect } o_{1} e \text { rotated about } o_{1} \text { at } f \text {. Locate point } b \text { on } o_{2} f \text { such that } \).

\( O_{2} b=o_{2} f \times \frac{O_{2} B}{O_{2} F} \).

\( =6.5 \times \frac{50}{75}=4.33 cm \). \( \left(\because o_{2} f=o_{2} e\right) \).

Draw a line perpendicular to BC at b and draw another line perpendicular to AC at ‘a’ to intersect at c. Now draw a line perpendicular to CD at c and draw another line from g parallel to the path of motion of slider D to intersect at d. Then,

\( \text { (a) Velocity of slider, } v_{d g}=g d=2.4 cm =0.6 m / s \).

\( \text { (b) } v_{b c}=c b=1.4 cm =0.35 m / s \).

\( \omega_{b c}=\frac{v_{b c}}{B C}=\frac{0.35}{0.2}=1.75 rad / s ( cw ) \).

\( v_{d c}=c d=4.6 cm =1.15 m / s \).

\( \omega_{c d}=\frac{v_{d c}}{D C}=\frac{1.15}{0.25}=4.6 rad / s ( ccw ) \).

\( \text { (c) } T \omega=F_{d} v_{d g} \).

\( T=4000 \times \frac{0.6}{12.57}=190.93 Nm \).
