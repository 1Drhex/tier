Guest Question.

Redesign the lowpass filter of using the von Hann, Hamming, and Blackman windows.

Admin
Solution

The impulse response is the same as in , that is,

 

\(h(n T)=\frac{1}{n \pi} \sin \omega_{c} n T\)

 

On multiplying h(nT ) by the appropriate window function and then using Eqs. (9.15) and (9.14), in this order, we obtain

 

\(H^{\prime}(z)=z^{-(N-1) / 2} H(z)\) (9.14)

 

\(H_{w}(z)= Z [w(n T) h(n T)]=\frac{1}{2 \pi j} \oint_{\Gamma} H(v) W\left(\frac{z}{v}\right) v^{-1} d v\) (9.15)

 

\(H_{w}^{\prime}(z)=z^{-(N-1) / 2} \sum_{n=0}^{(N-1) / 2} \frac{a_{n}^{\prime}}{2}\left(z^{n}+z^{-n}\right)\)

 

where \(a_{0}^{\prime}=w(0) h(0) \quad \text { and } \quad a_{n}^{\prime}=2 w(n T) h(n T)\)

 

The amplitude responses of the three filters are given by

 

\(M(\omega)=\left|\sum_{n=0}^{(N-1) / 2} a_{n}^{\prime} \cos \omega n T\right|\)

 

These are plotted in Fig. 9.7 for \(N=21 \text { and } \omega_{s}=10\) rad/s. As expected, the amplitude of the passband ripple is reduced, and the minimum stopband attenuation as well as the transition width are increased progressively from the von Hann to the Hamming to the Blackman window.
