Guest Question.

For step , ramp, and parabolic inputs, find the steady-state error for the feedback control system shown in Figure 13.17(a) if

\( G_{1}\left ( s \right ) = \frac{10}{s\left ( s+1 \right )}\) (13:76)

Admin
Solution

First find G(s), the product of the z.o.h. and the plant.

\(G\left ( s \right ) = \frac{10\left ( 1-e^{-Ts} \right )}{s^{2}\left ( s+1 \right )} = 10\left ( 1-e^{-Ts} \right )\left [ \frac{1}{s^{2}} -\frac{1}{s} + \frac{1}{s+1}\right ] \)(13:77)

The z-transform is then

\(G\left ( z \right ) = 10\left ( 1-z^{-1} \right )\left [ \frac{Tz}{\left ( z-1 \right )^{2}} -\frac{z}{z-1} + \frac{z}{z-e^{-T}}\right ]= 10\left [ \frac{T}{z-1} – 1 + \frac{z-1}{z-e^{-T}}\right ]\) (13:78)

For a step input,

\(K_{p}=\lim_{z\rightarrow 1}G\left ( z \right ) = \infty ; e^{\ast }\left ( \infty \right ) =\frac{1}{1+ K_{p}} = 0\) (13:79)

For a ramp input,

\(K_{\nu }=\frac{1}{T}\lim_{z\rightarrow 1}\left ( z-1 \right )G\left ( z \right ) = 10 ; e^{\ast }\left ( \infty \right ) =\frac{1}{K_{\nu }} =0.1\) (13:80)

For a parabolic input,

\(K_{a}=\frac{1}{T^{2}}\lim_{z\rightarrow 1}\left ( z-1 \right )^{2}G\left ( z \right ) = 0 ; e^{\ast }\left ( \infty \right ) =\frac{1}{K_{a}} =\infty\) (13:81)

You will notice that the answers obtained are the same as the results obtained for the analog system. However, since stability depends upon the sampling interval, be sure to check the stability of the system after a sampling interval is established before making steady-state error calculations .

We can use MATLABâ€™s command dcgain(Gz) to find steady-state errors. The command evaluates the dc gain of Gz, a digital LTI transfer function object, by evaluating Gz at z = 1. We use the dc gain to evaluate, Kp, Kv, and Ka. You will input T, the samplinginterval, through the keyboard to test stability.
