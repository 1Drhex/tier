Guest Question.

Compute the classical partition function \(Z^{*}_{C}\) for a single diatomic molecule consisting of point particles having masses \(m_{1} and m_{2}\) separated by a fixed distance ℓ (no vibrational mode) and having a magnetic moment \( \mu_{B}\) that makes an angle θ with an applied magnetic field B. The molecule is confined to a box of volume V and its Hamiltonian has the form (which we need to write in terms of canonical momenta)

\(H = \frac{m_{1}}{2} \dot{r}^{2}_{1} + \frac{m_{2}}{2} \dot{r}^{2}_{2} - \mu _{B} B \cos \theta . \) (20.102)

Admin
Solution

We make a canonical transformation to a center of mass coordinate \(R = (m_{1}r_{1} + m_{2}r_{2})/(m_{1} + m_{2})\) and a relative coordinate \(r = r_{1} − r_{2}\) but then an additional transformation of the relative coordinate into spherical polar coordinates with fixed radius \( r = \ell,\) azimuthal angle φ and polar angle θ.

The kinetic energy in H takes the well-known form

\(\frac{m_{1}}{2} \dot{r}^{2}_{1} + \frac{m_{2}}{2} \dot{r}^{2}_{2} = \frac{M}{2} \dot{R}^{2} + \frac{\mu_{red}}{2} \left[\ell^{2} \dot{\theta }^{2} + \ell^{2} \sin^{2} \theta \dot{\varphi }^{2} \right] ,\) (20.103)

where \(M = m_{1} + m_{2}\) is the total mass and \(\mu_{red} = m_{1}m_{2}/M\) is the reduced mass.

The canonical momenta are \(P_{i} = \partial H / \partial \dot{R}_{i} = M\dot{R}_{i}, p_{\theta } = \partial H / \partial \dot{\theta } = \mu \ell^{2} \dot{\theta}, and p_{\varphi } = \partial H / \partial \dot{\varphi } = \mu \ell^{2} \sin^{2} \theta \dot{\varphi },\) so the
Hamiltonian becomes

\(H = \sum\limits_{i=1}^{3}{\frac{p^{2}_{i} }{2M} + \frac{p^{2}_{\theta } }{2I} + \frac{p^{2}_{\varphi } }{2I \sin^{2} \theta} } – \mu _{B} B \cos \theta ,\) (20.104)

where \(I = \mu_{red} \ell^{2}\) is the moment of inertia about the center of mass of the molecule.

See Section F.1.1 in Appendix F for an explicit evaluation of I. Since the transformation is canonical, the partition function is

\(Z^{*}_{C} = \frac{1}{h^{5}} \int{exp\left[- \beta H\right] dp_{1} dp_{2} dp_{3} dR_{1} dR_{2} dR_{3} dp_{\theta } dp_{\varphi } d\theta d\varphi } .\) (20.105)

The factor of \(h^{−5}, \) rather than \(h^{−6}, \) arises because the magnitude of \(\ell\) has been assumed to be constant (no vibrational mode). Since exp[−βH] factors, we can perform the R and P integrals immediately to obtain

\(\frac{1}{h^{3}} \int{exp \left[- \beta \sum\limits_{i=1}^{3}{\frac{p^{2}_{i} }{2M}} \right] dp_{1} dp_{2} dp_{3} dR_{1} dR_{2} dR_{3}} = Vn_{Q},\) (20.106)

where \(n_{Q} = \left(Mk_{B}T / 2\pi \hbar^{2} \right)^{3/2} .\) The integrals over \(dp_{θ} and dp_{φ}\) are well-known Gaussian integrals, resulting in

\(\frac{1}{h} \int{exp \left[- \beta \frac{p^{2}_{\theta } }{2I} \right] dp_{\theta } } = \left(\frac{2\pi I}{h^{2}\beta } \right)^{1/2} ; \frac{1}{h} \int{exp \left[- \beta \frac{p^{2}_{\varphi } }{2I \sin^{2} \theta}\right] dp_{\varphi } } = \sin \theta \left(\frac{2\pi I}{h^{2}\beta } \right)^{1/2}.\) (20.107)

The crucial point to note is that the scale factor sin θ gets trapped inside the remaining θ integral, resulting in

\(Z^{*}_{C} = Vn_{Q} \left(\frac{2\pi I}{h^{2}\beta } \right) \int{exp \left[\beta \mu _{B} B \cos \theta \right] \sin \theta d\theta d\varphi } = Vn_{Q} \left(\frac{2I}{\hbar ^{2}\beta } \right) \frac{\sin h\left(\beta \mu _{B} B\right) }{\beta \mu _{B}B} .\) (20.108)

We see that the partition function is the product of three factors, \(Vn_{Q}\) which is the partition function for translation of a structureless ideal gas of molecules having molecular mass M, a factor \(\left(2I / \hbar ^{2} \beta \right),\) which is the partition function for a rigid diatomic molecule rotating about its center of mass, and a factor \(sinh(βμ_{B}B)/ μ_{B}B\) which is the partition function for a magnetic dipole.
