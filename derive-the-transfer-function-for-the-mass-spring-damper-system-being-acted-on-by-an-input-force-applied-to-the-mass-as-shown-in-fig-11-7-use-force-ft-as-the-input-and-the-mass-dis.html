Guest Question.

Derive the transfer function for the mass–spring–damper system being acted on by an input force applied to the mass, as shown in Fig. 11.7. Use force F(t) as the input and the mass displacement x(t) as the output variable.

Admin
Solution

The input–output equation for a linear mass–spring–dashpot system was derived in
Chap. 2, Eq. (2.34):

\(m\frac{d^{2}x_{1} }{dt^{2} } +b\frac{dx_{1} }{dt} +kx_{1} =F_{i} .\)

 

\(m\frac{d^{2}x }{dt^{2} } +b\frac{dx }{dt} +kx =F\left(t\right) .\)

Assuming zero initial conditions, \(x\left(0^{-} \right) =0\) and \(\dot{x} \left(0^{-} \right) =0\), and taking the Laplace transform of both sides of the input–output equation yields

\(ms^{2} X\left(s\right) +bsX\left(s\right)+kX\left(s\right)=F\left(s\right) ;\)

hence, the system transfer function is

\(T\left(s\right) =\frac{X\left(s\right)}{F\left(s\right)} =\frac{1}{ms^{2}+bs+k } .\)

It can be seen that this second-order system has no zeros and two poles, \(s_{1}\) and \(s_{1}\), which can be found when the system characteristic equation is solved:

\(ms^{2}+bs+k=0.\)

The poles are easily found by the solution of this quadratic equation:

\(s_{1},s_{2} ={\left(-b\pm \sqrt{b^{2} -4mk} \right) }/{2m}.\)

By use of the damping ratio (see Section 4.4), \(\zeta ={b}/{\left(2\sqrt{mk} \right) },\) when \(\zeta\geq 1\), the poles are equal to negative inverse time constants,

\(s_{1},s_{2} =-\frac{1}{\tau _{1} } ,-\frac{1}{\tau _{2} },\)

and, when \(0\lt \zeta \lt 1.0\), the poles are complex-conjugate numbers related to the system damping ratio and natural frequency:

\(s_{1},s_{2} =-\zeta w_{n} \pm jw_{n}\sqrt{1-\zeta ^{2} } ,\)

where

\(w_{n}=\sqrt{{k}/{m} } .\)

The availability of the system transfer function offers an alternative way for solving for the response of linear systems (an alternative to analytical solutions of input–output equations or computer solutions of state-variable equations). From the definition of the transfer function, the Laplace transform of the output signal can be calculated as

\( Y\left(s\right) =U\left(s\right)T\left(s\right).\) (11.17)

One can then find the system response in the time domain by taking the inverse transform of \(Y\left(s\right)\) by using partial fraction expansion described in Appendix 2 or by using MATLAB, which is the topic of Section 11.7:

\(y\left(t\right) =L^{-1} \left\{Y\left(s\right) \right\} .\) (11.18)

In many situations, it is of practical value to engineers to be able to quickly evaluate a general character of the system step response rather than to calculate a detailed and complete response. In such situations, the system transfer function can be used to evaluate the step response immediately after the input signal is applied, at time \(t=0^{+}\) , and at steady state for time approaching infinity. Based on the initial-value theorem (see Appendix 2), the system response at time \(t=0^{+}\) is

\(y \left(0^{+}\right) =\underset{s\rightarrow \infty }{\lim } sY\left(s\right) =\underset{s\rightarrow \infty }{\lim } sU\left(s\right)T.\left(s\right)\) (11.19)

For a step input of magnitude A, we have

\(y \left(0^{+}\right) =\underset{s\rightarrow \infty }{\lim } s\frac{A}{s}T \left(s\right) =A\underset{s\rightarrow \infty }{\lim } T\left(s\right).\) (11.20)

Thus the value of the step response at \(t=0^{+}\) is equal to the limiting value of the transfer function as s → ∞ multiplied by the magnitude of the step input.

The value of the final value of the steady-state step response can be found with the final-value theorem (Appendix 2):

\(y{ss} =\underset{t\rightarrow \infty }{\lim } y=\underset{s\rightarrow \infty }{\lim } sY\left(s\right)=\underset{s\rightarrow \infty }{\lim } sU\left(s\right)T\left(s\right).\) (11.21)

For a step input of magnitude \(A,U\left(s\right)={A}/{s},\)

\(y_{ss} =A\underset{s\rightarrow 0}{\lim } T\left(s\right) .\) (11.22)

According to this equation, the final value of the steady-state step response is the same as the limit of the value of the system transfer function when s → 0 multiplied by the magnitude of the step input.
