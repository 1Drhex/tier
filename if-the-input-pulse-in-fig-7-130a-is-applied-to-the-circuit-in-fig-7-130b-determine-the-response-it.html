Guest Question.

If the input pulse in Fig. 7.130(a) is applied to the circuit in Fig. 7.130(b), determine the response i(t).

Admin
Solution

Since \(v_{S }\)=10[ u(t)- u(t- 1)] , this is the same as saying that a 10 V source is turned on at t = 0 and a -10 V source is turned on later at t = 1. This is shown in the figure below.

\(\begin{array}{l}\text { For } 0<\mathrm{t}<1, \quad \mathrm{i}(0)=0, \quad \mathrm{i}(\infty)=\frac{10}{5}=2 \\\mathrm{R}_{\mathrm{th}}=5 \| 20=4, \quad \tau=\frac{\mathrm{L}}{\mathrm{R}_{\mathrm{th}}}=\frac{2}{4}=\frac{1}{2} \\\mathrm{i}(\mathrm{t})=\mathrm{i}(\infty)+[\mathrm{i}(0)-\mathrm{i}(\infty)] \mathrm{e}^{-\mathrm{t} / \tau} \\\mathrm{i}(\mathrm{t})=2\left(1-\mathrm{e}^{-2 \mathrm{t}}\right) \mathrm{A} \\\mathrm{i}(1)=2\left(1-\mathrm{e}^{-2}\right)=1.729 \\\text { For } \mathrm{t}>1, \quad \mathrm{i}(\infty)=0 \quad \text { since } \mathrm{v}_{\mathrm{s}}=0 \\\mathrm{i}(\mathrm{t})=\mathrm{i}(1) \mathrm{e}^{-(\mathrm{t}-1) / \mathrm{t}} \\\mathrm{i}(\mathrm{t})=1.729 \mathrm{e}^{-2(\mathrm{t}-1)} \mathrm{A}\end{array}\)

Thus,

\(\mathrm{i}(\mathrm{t})=\left\{\begin{array}{cc}2\left(1-\mathrm{e}^{-2 \mathrm{t}}\right) \mathrm{A} & 0<\mathrm{t}<1 \\1.729 \mathrm{e}^{-2(\mathrm{t}-1)} \mathrm{A} & \mathrm{t}>1\end{array}\right.\)
