Guest Question.

Use Euler’s method to obtain a numerical solution of the differential equation

\(4\frac{dx}{dt} + x=4\)

over a period of time from 0 to 12 s. The initial condition is \(x\left(0\right) =10\), and the system time constant is 4 s.

Admin
Solution

First rewrite the differential equation in the same form as that of Eq. (5.1):

\(\frac{dx}{dt}=f\left(x,t\right) ,\)

 

\(\frac{dx}{dt}=-0.25x+1.\)

By use of Euler’s formula (5.6)

\(x\left(t_{0}+ \Delta t \right) \thickapprox x\left(t_{0} \right) + f\left(x_{0},t_{0} \right) \Delta t\)

the successive values of x are calculated as follows:

\(x\left(1\right) =x\left(0\right) + \left[- 0.25x\left(0\right) + 1\right] \Delta t,\)

 

\(x\left(2\right) =x\left(1\right) + \left[- 0.25x\left(1\right) + 1\right] \Delta t\)

 

The first step in implementing the numerical solution is to choose a value of the integration
interval Δt. Although rules of thumb and implications associated with this choice are
discussed in greater detail in Section 5.4, the reader can gain some appreciation for the
problem by referring to Fig. 5.1. The Euler method approximates the function as a series of
straight lines. Because it is well known that the solution to this equation is an exponential
function (see Section 4.3), it would seem reasonable to choose a time step that is somehow
related to the exponential time constant \(\tau\) . For this system, the time constant is 4 s, and to illustrate this relationship, the Euler method is implemented for Δt’s of 0.5, 1.0, 2.0, and 4.0 s.
A simple script was written for MATLAB to solve this equation and is shown in Table 5.1. Note that MATLAB script is fairly easy to understand and it would be rather straightforward to go from this simple script to another programming environment such as basic or c .
As already stated, this program was run for various values of integration time step
(dt in the script file) and the results were compared with the exact solution, which can
be found to be

\(x\left(t\right) =6e^{- {t}/{4}} + 4.\)
