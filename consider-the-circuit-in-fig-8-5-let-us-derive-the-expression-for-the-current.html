Guest Question.

Consider the circuit in Fig. 8.5. Let us derive the expression for the current.

Admin
Solution

The KVL equation for this circuit is

\(L\frac{di(t)}{dt} + Ri(t) = V_{M}\) cos ωt

Since the input forcing function is \(V_{M}\) cos ωt, we assume that the forced response component of the current i(t) is of the form

i(t) = A cos (ωt + Φ)

which can be written using Eq. (8.11) as

Eq. (8.11): sin (α + β) = sin α cos β + cos α sin β
cos (α + β) = cos α cos β – sin α sin β
sin (α – β) = sin α cos β – cos α sin β
cos (α – β) = cos α cos β + sin α sin β

i(t) = A cos Φ cos ωt – A sin Φ sin ωt
= \(A_{1}\) cos ωt + \(A_{2}\) sin ωt

Note that this is, as we observed in Chapter 7, of the form of the forcing function cos ωt and its derivative sin ωt. Substituting this form for i(t) into the preceding differential equation yields

\(L\frac{d}{dt} + (A_{1} cos\,ωt + A_{2} sin\,ωt) + R(A_{1} cos\,ωt + A_{2} sin\,ωt)= V_{M}\) cos ωt

Evaluating the indicated derivative produces

\(-A_{1} ωL sin\,ωt + A_{2} ωL cos\,ωt + RA_{1} cos\,ωt + RA_{2} sin\,ωt = V_{M}cos\,ωt\)

By equating coefficients of the sine and cosine functions, we obtain

\(-A_{1} ωL + A_{2}\) R = 0
\(A_{1} R + A_{2} ωL = V_{M}\)

that is, two simultaneous equations in the unknowns \(A_{1}\) and \(A_{2}\). Solving these two equations for \(A_{1}\) and \(A_{2}\) yields

\(A_{1} = \frac{RV_{M}}{R^{2} + ω^{2}L^{2}}\)
\(A_{2} = \frac{ωLV_{M}}{R^{2} + ω^{2}L^{2}}\)

Therefore,

i(t) = \(\frac{RV_{M}}{R^{2} + ω^{2}L^{2}} cos\,ωt + \frac{ωLV_{M}}{R^{2} + ω^{2}L^{2}}\) sin ωt

which, using the last identity in Eq. (8.11), can be written as

i(t) = A cos (ωt + Φ)

where A and Φ are determined as follows:

A cos Φ = \(\frac{RV_{M}}{R^{2} + ω^{2}L^{2}}\)
A sin Φ = \(\frac{-ωLV_{M}}{R^{2} + ω^{2}L^{2}}\)

Hence,

tan Φ = \(\frac{A sin\,Φ}{A cos\,Φ} = – \frac{ωL}{R}\)

and therefore,

Φ = – \(tan^{-1}\frac{ωL}{R}\)

and since

\((A\,cos\,Φ)^{2} + (A\,sin\, Φ)^{2} = A^{2}(cos^{2}\,Φ + sin^{2}\,Φ) = A^{2}\)
\(A^{2} = \frac{R^{2}V^{2}_{M}}{(R^{2} + ω^{2}L^{2})^{2}} +\frac {(ωL)^{2} V^{2}_{M}}{(R^{2} + ω^{2}L^{2})^{2}}\)
= \(\frac {V^{2}_{M}}{R^{2} + ω^{2}L^{2}}\)
A = \(\frac{V_{M}}{\sqrt{R^{2} + ω^{2}L^{2}}}\)

Hence, the final expression for i(t) is

i(t) =\(\frac{V_{M}}{\sqrt{R^{2} + ω^{2}L^{2}}} cos \Big(ωt – tan^{-1}\frac{ωL}{R}\Big)\)

The preceding analysis indicates that Φ is zero if L = 0 and hence i(t) is in phase with v(t). If R = 0, Φ = –90°, and the current lags the voltage by 90°. If L and R are both present, the current lags the voltage by some angle between 0° and 90°.
