Guest Question.

A 10-minute count of sample plus background yields 10,000 counts, and a 6-minute count of background alone yields 1296 counts. What are the net sample count rate and its standard deviation?

Admin
Solution
\(Sample \ plus \ background \ count \ rate=\frac{10.000 \ counts}{10 \ min }= 1000 \ cpm\)

 

\(Background \ count \ rate =\frac{1296\ counts}{6 \ min} = 216 \ cpm\)

 

\(\sigma _{C} sample \ plus \ background \ count \ rate =\sqrt{\frac{c}{t} }= \sqrt{\frac{1000 \ cpm}{10 \ min} }= \sqrt{100}\)

 

\(\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ = 10 \ cpm\)

 

\(\sigma _{C} for \ background \ count \ rate =\sqrt{\frac{c}{t} }= \sqrt{\frac{216 \ cpm}{6 \ min} }= \sqrt{36}= 6 \ cpm\)

 

\(Net \ sample \ count \ rate = \left(A-B\right)\pm \sqrt{\sigma ^{2}_{A} + \sigma ^{2}_{B} }\)

 

\(\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ = \left(1000- 216\right)\pm \sqrt{\left(10\right)^{2} + \left(6\right)^{2} }\)

 

\(\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ = 784 \pm \sqrt{136}\)

 

\(\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ = 784 \pm 12\)
