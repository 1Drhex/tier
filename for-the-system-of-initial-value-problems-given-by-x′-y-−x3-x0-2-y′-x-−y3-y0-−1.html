Guest Question.

For the system of initial-value problems given by

\(x′= y −x^{3}, x(0) = 2\)
\(y′= x −y^{3}, y(0) = −1\) (6.4.11)

estimate the solution to the IVP up to t = 5 using h = 0.1 and comment on the behavior of the trajectory.

Admin
Solution

In the given problem, if we take the perspective that x′= f (t ,x, y) and y′=g (t ,x, y), then it follows that \(f (t ,x, y)=y−x^{3}\) and \(g (t ,x, y)=x−y^{3}\). Applying (6.4.9) with h = 0.1, we have

\(x_{n+1} = x_{n} +0.1 · (y_{n} −x^{3}_{n})\)
\(y_{n+1} = y_{n} +0.1 · (x_{n} −y^{3}_{n})\)

Beginning this iteration with \(x_{0}\) = 2 and y\(y_{0}\)= −1, we generate the following table.

\(y_{n}\) \(x_{n}\) \(t_{n}\)
-1 2 0
-0.7 1.1 0.1
-0.5557 0.8968 0.2
−0.448849846 0.769180708 0.3
¦ ¦ ¦
0.994533281 0.994536765 4.7
0.995618024 0.995620126 4.8
0.996488877 0.996490144 4.9
0.997187534 0.997188297 5

In the table, we see behavior consistent with the fact that the equilibrium point (1, 1) of the system is a stable attracting node. In addition, the numerical data is in agreement with the graphical behavior we expect based on the direction field in figure 6.4 where we first considered the given nonlinear system.
