Guest Question.

The field windings of the two similar tramway motors are connected in series and the motors are coupled to perform a test. The following test results were obtained while one machine acts as a motor and the other acts as a generator. Motor: Armature Current: 56 A, Armature Voltage: 590 V, Voltage drop a cross field winding:40 V ,Generator: Armature Current: 44A, Armature Voltage: 400V, Field Voltage drop: 40V, Resistance of each armature: 0.3 ohm. Calculate the efficiency of motor and generator.

Admin
Solution

Here,

\( I_{a m} =56 A ; V_{a m}=590 V ; V_{s e}=40 V \)

 

\( I_{a g} =44 A ; V_{a g}=400 V ; V_{s e}=40 V \)

 

\( R_{a m} =R_{a g}=0.3 \Omega\)

 

Input to the whole set \( =V_{1} I_{a m}=(40+590+40) \times 56=37520 W\)

 

Output of the set \( =V_{a g} I_{a g}=400 \times 44=17600 W\)

 

Total losses of the set = Input -Output =37520-17600=19920 W

 

Series field and armature Cu loss of motor

 

\( =V_{\text {sem }} \times I_{a m}+I_{a m}^{2} R_{a m} \)

 

\( =40 \times 56+(56)^{2} \times 0.3=3180.8 W\)

 

Series field and armature Cu loss of generator

 

\( =V_{\operatorname{seg}} \times I_{a m}+I_{a g}^{2} R_{a g}\)

 

\( =40 \times 56+(44)^{2} \times 0.3=2820.8 W \)

 

\( \text { Total Cu losses of the set } =3180.8+2820.8=6001.6 W\)

 

Stray losses of the set =19920-6001.6=13918.4 W

Stray losses of each machine =\( \frac{13918.4}{2}=6959.2 W\)

 

For motor efficiency

\( \text { Motor input } =\left(V_{a m}+V_{\text {sem }}\right) \times I_{a m}=(590+40) \times 56=35280 W \)

 

\( \text { Motor losses } =3180.8+6959.2=10140 W\)

 

\( \text { Motor efficiency, } \eta_{m} =\frac{\text { motor input }-\text { losses }}{\text { motor input }} \times 100\)

 

\( =\frac{35280-10140}{35280} \times 100=71.26 \%\)

 

For generator efficiency:

\( \text { Generator output } =V_{a g} I_{a g}=400 \times 44=17600 W \)

 

\( \text { Generator losses } =2820.8+6959.2=9780 W\)

 

\( \text { Generator efficiency, } \eta_{ g } =\frac{\text { Generator output }}{\text { Generator output + losses }} \times 100 \)

 

\( =\frac{17600}{17600+9780} \times 100=64.28 \% \)
