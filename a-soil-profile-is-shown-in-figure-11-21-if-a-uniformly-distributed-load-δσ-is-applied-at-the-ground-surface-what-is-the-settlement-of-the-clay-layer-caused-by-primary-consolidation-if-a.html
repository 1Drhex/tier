Guest Question.

A soil profile is shown in Figure 11.21. If a uniformly distributed load, Δσ, is applied at the ground surface, what is the settlement of the clay layer caused by primary consolidation if

a. The clay is normally consolidated

b. The preconsolidation pressure \( (\sigma ^{\prime }_{c} )=200KN/m^{2}\)

c. \(\sigma ^{\prime }_{c} =150KN/m^{2}\)

Use \(C_{s}\approx \frac{1}{5} C_{c^{.} } \)

Admin
Solution

Part a

The average effective stress at the middle of the clay layer is

\(\sigma ^{\prime }_{\omicron } =2\gamma _{dry} + 4[\gamma _{sat(sand)}-\gamma _{w} ]+\frac{3.5}{2}[\gamma _{sat(clay)}-\gamma _{w} ]\)

 

\(\sigma ^{\prime }_{\omicron }=(2)(14)+4(18-9.81)+1.75(19-9.81)=76.08 kN/m^{2}\)

From Eq. (11.29),

\(S_{c} =\frac{C_{c} H}{1+e_{o} } \log \left(\frac{\sigma ^{\prime }_{o} +\Delta \sigma ^{\prime } }{\sigma ^{\prime }_{o}} \right) \)

From Eq. (11.33),

\(C_{c} =0.009(LL-10)=0.009(40-10)=0.27\)

so

\(S_{c}=\frac{(0.27)(3.5)}{1+0.8} \log \left(\frac{76.08 + 100}{76.08} \right) =0.191 m = 191 mm\)

part b

\(\sigma ^{\prime }_{o}+\Delta \sigma ^{\prime }=76.08 + 100 = 176.08 kN/m^{2}\)

 

\(\sigma ^{\prime }_{c}=200 kN/m^{2}\)

 

Because \(\sigma ^{\prime }_{o}+\Delta \sigma ^{\prime }\lt \sigma ^{\prime }_{c}\) use Eq. (11.31):

\(S_{c} =\frac{C_{s}H }{1+e_{o} } \log \left(\frac{\sigma ^{\prime }_{o}+\Delta \sigma ^{\prime }}{\sigma ^{\prime }_{o}} \right) \)

 

\(C_{s}=\frac{C_{c}}{5} =\frac{0.27}{5} =0.054\)

 

\(S_{c}=\frac{(0.054)(3.5)}{1+0.8} \log \left(\frac{76.08 + 100}{76.08} \right) =0.038 m = 38 mm\)

 

Part c

\(\sigma ^{\prime }_{o}=76.08 kN/m^{2}\)

 

\(\sigma ^{\prime }_{o}+\Delta \sigma ^{\prime }=176.08 kN/m^{2}\)

 

\(\sigma ^{\prime }_{c}=150 kN/m^{2}\)

Because \(\sigma ^{\prime }_{o}\lt \sigma ^{\prime }_{c}\lt \sigma ^{\prime }_{o}+\Delta \sigma ^{\prime }\) use Eq. (11.32):

\(S_{c} = S_{c} =\frac{C_{s}H }{1+e_{o} } \log \frac{\sigma ^{\prime }_{c} }{\sigma ^{\prime}_{o} } +\frac{C_{s}H }{1+e_{o} } \log \left(\frac{\sigma ^{\prime }_{o}+\Delta \sigma ^{\prime }}{\sigma ^{\prime }_{c}} \right) \)

 

\(=\frac{(0.054)(3.5)}{1+0.8} \log \left(\frac{150}{76.08} \right) +\frac{(0.27)(3.5)}{1.8} \log \left(\frac{176.08}{150} \right) \approx 0.0675 m = 67.5 mm\)
