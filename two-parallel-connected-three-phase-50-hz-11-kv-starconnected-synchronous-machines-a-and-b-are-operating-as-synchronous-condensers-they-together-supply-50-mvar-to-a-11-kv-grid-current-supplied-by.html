Guest Question.

Two parallel connected, three-phase, \(50 Hz, 11 kV\), starconnected synchronous machines \(A\) and \(B\) are operating as synchronous condensers. They together supply \(50 MVAR\) to a \(11 kV \) grid. Current supplied by both the machines are equal. Synchronous reactances of machine \(A\) and machine \(B \) are \( 1 Ω\) and \(3 Ω\), respectively. Assuming the magnetic circuit to be linear, the ratio of excitation current of machine \(A\) to that of machine \(B\) is _____. (Give the answer up to two decimal places).

Admin
Solution

As the machines works at same current and same voltage, so they supply same reactive power.

As machines are operating as synchronous condenser so they will work as overexcited synchronous motor.

Total current,

\(\quad \quad \quad { I }_{ T }=\frac { 50 MVA }{ \sqrt { 3 } 11 KV } = 2.624 KA\)

 

\(\quad \quad \quad { I }_{ A}={ I }_{ B }=\frac { { I }_{ T } }{ 2 } =\frac { 2.624 KA }{ 2 } =1.312 KA\)

Current taken by motor will be leading because the motor is working as synchronous condenser,

Also, \( \vec { { E }_{ A } } =\vec { { V }_{ T } } -j\vec { { I }_{ a } } { X }_{ s }\)

\(\quad \quad \quad =\frac { 11 KV }{ \sqrt { 3 } } -j(1.312\angle { 90 }^{ o } KA)(1)\)

 

\(\quad \quad \quad =\frac { 11 }{ \sqrt { 3 } } +1.312\times 1=7.662 KV\)

Similarly \(\vec { { E }_{ B } } =\frac { 11 KV }{ \sqrt { 3 } } -j(1.312\angle { 90 }^{ o } KA)(3)\)

\(\quad \quad \quad =\frac { 11 }{ \sqrt { 3 } } +1.312\times 3=10.286 KV\)

Hence \(\frac { { I }_{ A } }{ { I }_{ B } } \propto \frac { { E }_{ A } }{ { E }_{ B } } =\frac { 7.662 }{ 10.286 } =0.74\)

Hence, the correct answer is \((0.74).\)
