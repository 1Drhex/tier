Guest Question.

Determine the proportions of four bar mechanism, by using three precision points, to generate \(y=x^{1.5}\) , where x varies between 1 and 4. Assume \(\theta_{ S }=30^{\circ} ; \Delta \theta=90^{\circ}\) ; \(\phi_{ S }=90^{\circ} ; \text { and } \Delta \phi=90^{\circ}\) . Take length of the fixed link AD as 25 mm.

Admin
Solution

Given : \(x_{ S }=1 ; x_{ F }=4 ; \quad \theta_{ S }=30^{\circ} ; \quad \Delta \theta=\theta_{ F }-\theta_{ S }=90^{\circ} ; \quad \phi_{ S }=90^{\circ}\) ; \(\Delta \phi=\phi_{ F }-\phi_{ S }=90^{\circ} ; d=25 mm\)

We have already calculated the three values of x and y for the above given data in . These values are :

\(x_{1}=1.2 ; \quad x_{2}=2.5 ; \quad \text { and } \quad x_{3}=3.8\)

 

\(y_{1}=1.316 ; \quad y_{2}=3.952 ; \quad \text { and } \quad y_{3}=7.41\)

 

The corresponding values of \(\theta \text { and } \phi \text { are }\)

 

\(\theta_{1}=36^{\circ} ; \theta_{2}=75^{\circ} ; \text { and } \theta_{3}=114^{\circ}\)

 

\(\phi_{1}=94.06^{\circ} ; \phi_{2}=127.95^{\circ} ; \text { and } \phi_{3}=172.41^{\circ}\)

 

We know that the Freudenstein’s equation is

\(k_{1} \cos \phi-k_{2} \cos \theta+k_{3}=\cos (\theta-\phi)\) …(i)

 

where \(k_{1}=\frac{d}{a} ; k_{2}=\frac{d}{c} ; \quad \text { and } \quad k_{3}=\frac{a^{2}-b^{2}+c^{2}+d^{2}}{2 a c}\) …(ii)

 

Now for the three different positions of the mechanism, the equation (i) may be written three times as follows :

\(k_{1} \cos 94.06^{\circ}-k_{2} \cos 36^{\circ}+k_{3}=\cos \left(36^{\circ}-94.06^{\circ}\right)\)

 

or \(-0.0708 k_{1}-0.809 k_{2}+k_{3}=0.529\) …(iii)

 

Similarly \(k_{1} \cos 127.95^{\circ}-k_{2} \cos 75^{\circ}+k_{3}=\cos \left(75^{\circ}-127.95^{\circ}\right)\)

 

or \(-0.615 k_{1}-0.259 k_{2}+k_{3}=0.6025\) …(iv)

 

and \(k_{1} \cos 172.41^{\circ}-k_{2} \cos 114^{\circ}+k_{3}=\cos \left(114^{\circ}-172.41^{\circ}\right)\)

 

\(-0.9912 k_{1}+0.4067 k_{2}+k_{3}=0.5238\) …(v)

Solving three simultaneous equations (iii), (iv) and (v), we get

\(k_{1}=0.6 ; k_{2}=0.453 ; \text { and } k_{3}=0.12\)

Now from equation (ii),

\(a=\frac{d}{k_{1}}=\frac{25}{0.6}=41.7 mm\) \(c=\frac{d}{k_{2}}=\frac{25}{0.453}=55.2 mm\)

 

and \(b=\left(a^{2}+c^{2}+d^{2}-k_{3} \times 2 a c\right)^{1 / 2}\)

 

\(=\left[(41.7)^{2}+(55.2)^{2}+(25)^{2}-0.12 \times 2 \times 41.7 \times 55.2\right]^{1 / 2}=69.7 mm\)

The designed four bar mechanism \(A B_{2} C_{2} D\) in one position \(\text { (i.e. for } \left.\theta_{2}, x_{2} \text { and } \phi_{2}, y_{2}\right)\) is shown by thick lines in Fig. 25.13.

The other two positions of the four bar mechanism may be drawn by joining \(B_{1} C_{2}\) (i.e.\(\left.\theta_{1}, x_{1} \text { and } \phi_{1}, y_{1}\right) \text { and } B_{3} C_{3}\left(\text { i.e. } \theta_{3}, x_{3} \text { and } \phi_{3}, y_{3}\right)\)

Note : In the above example, the motion of input link and output link is taken clockwise.
