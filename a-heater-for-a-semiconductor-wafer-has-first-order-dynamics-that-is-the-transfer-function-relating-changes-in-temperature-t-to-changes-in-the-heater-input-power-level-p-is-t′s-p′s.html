Guest Question.

A heater for a semiconductor wafer has first order dynamics, that is, the transfer function relating changes in temperature T to changes in the heater input power level P is

\(\frac{T^{\prime}(s)}{P^{\prime}(s)}=\frac{K}{\tau s+1}\)

where \(K\) has units \(\left[{ }^{\circ} C / Kw \right]\) and \(\tau\) has units \([ min ]\).

The process is at steady state when an engineer changes the power input stepwise from 1 to \(2 kW\). She notes the following:

(i) The process temperature initially is \(100^{\circ} C\).

(ii) Four minutes after changing the power input, the temperature is \(400^{\circ} C\).

(iii) Thirty minutes later the temperature is \(500^{\circ} C\).

(a) What are \(K\) and \(\tau\) in the process transfer function?

(b) If at another time the engineer changes the power input linearly at a rate of \(0.5 kW / min\), what can you say about the maximum rate of change of process temperature: When will it occur? How large will it be?

Admin
Solution

(a) For a step change in input of magnitude M:

\(y(t)=K M\left(1-e^{-t / \tau}\right)+y(0)\)

We note that \(K M=y(\infty)-y(0)=500-100=400^{\circ} C\)

Then \(K=\frac{400^{\circ} C}{(2-1) K w}=400^{\circ} C / K w\)

At time \( t=4, y(4)=400^{\circ} C ; \text { thus, } \frac{400-100}{500-100}=1-e^{-4 / \tau} \text {, or } \tau=2.89 min\)

\(\therefore \frac{T^{\prime}(s)}{P^{\prime}(s)}=\frac{400}{2.89 s+1}\left[{ }^{\circ} C / K w\right]\)

(b) For an input ramp change with slope a = 0.5Kw/ min :

\(K a=400 \times 0.5=200^{\circ} C / min\)

This maximum rate of change will occur as soon as the transient has died out, i.e., after \(5 \times 2.89 \min \approx 15\) min have elapsed.
