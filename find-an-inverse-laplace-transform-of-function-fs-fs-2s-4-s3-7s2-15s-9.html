Guest Question.

Find an inverse Laplace transform of function F(s):

\( F(s)\frac{2s + 4}{s^{3}+ 7s^{2} + 15s + 9} \).

Admin
Solution

The first step is to find the poles of F(s). There is a single pole at s =–1 and a pole of multiplicity 2 at s =–3. F(s) can thus be rewritten as

\( F(s)\frac{2s + 4}{(s + 1)(s + 3)^{2} } \).

Hence the expanded form of F(s) is

\( F(s) = \frac{C_{11} }{s+1}+ \frac{C_{21} }{s+3}+\frac{C_{22} }{\left(s+3\right) ^{2} } \).

By use of Eqs. (A2.13), the constants \( C_{11},C_{21}\), and \( C_{22} \)are found to be

\( C_{ij} =\frac{1}{\left(p_{i}-j \right)! } \underset{s\rightarrow s_{i} }{\lim } \frac{d^{p_{i}-j }\left[F\left(s\right)\left(s-s_{i} \right) ^{p_{i}-j } \right] }{ ds^{p_{i}-j}} \). (A2.13)

\( C_{11} = 0.5, C_{21} = −0.5 ,C_{22} = 1\).

The partial fraction expansion of F(s) becomes

\( F(s) = \frac{0.5 }{s+1}- \frac{0.5 }{s+3}+\frac{1}{\left(s+3\right) ^{2} } \).

The inverse Laplace transforms of the three simple terms on the right-hand side of the preceding equation can be found in Table A2.1, yielding the solution

\( f (t) = 0.5e^{−t} − 0.5e^{−3t}+ te^{−3t}\).
