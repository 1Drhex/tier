Guest Question.

A pipeline company is responsible for pumping hexane from a manufacturer to a distributor. The company wishes to install a meter in the pipeline to monitor the flow rate. A fluid meters reference book was consulted, and it has been decided to install a 6 × 4 venturi meter in the line.
The discharge coefficient for the meter is a constant at 0.984 for a Reynolds number greater than \(2 × 10^5\), provided that pressure taps are located as indicated in Figure 5.8. The meter is to be installed in an upward-sloping configuration that makes an angle of 20° with the horizontal, as shown in Figure 5.10. It is proposed to use a hexane-over-mercury manometer. Generate a calibration curve for the meter for a flow rate up to \(0.15 m^3/s\).

Admin
Solution

From the Appendix we read:
hexane \(ρ = 0.657(1 000) kg/m^3\) \(µ = 0.297 \times 10^{-3} N·s/m^2\)
mercury \(ρ_m = 13.6(1 000) kg/m^3\) [Appendix Table B.1]
6-nom sch 40 \(D_1 = 15.41 cm\) \(A_1 = 186.50 cm^2\)
4-nom sch 40 \(D_2 = 10.23 cm\) \(A_2 = 82.19 cm^2\) [Appendix Table C.1]

For the manometer, we have
\(\frac{(p_{1} – p_2)g_c}{ρg}+ z_1 – z_2 = ∆h \left\lgroup \frac{ρ_m}{ρ}– 1 \right\rgroup= ∆h \left\lgroup\frac{13.6(1 000)}{0.657(1 000)} – 1 \right\rgroup = 19.7 ∆h\)
Substituting into Equation 5.18 gives
\(Q_{th} = A_2 \sqrt{\frac{2g∆h(ρ_m/ρ – 1)}{1 – D^4_2/D^4_1}} \left\lgroup two-liquid\ manometer \right\rgroup\) (5.18)
\(Q_{th} = 82.19 \times 10^{-4}\left\lgroup \frac{2 (9.81)(19.7)∆h}{1 –\left\lgroup\frac{ 10.23}{15.41}\right\rgroup^4}\right\rgroup^{1/2}\)

or \(Q_{th} = 0.18\sqrt{∆h}\)
A tabulation of \(Q_{th}\) versus ∆h is provided in Table 5.4.
The Reynolds number for the flowing hexane is
\(Re = \frac{V_2D_2}{ν} = \frac{4ρQ_{ac}}{πD^2µg_c}= \frac{4(0.657)(1 000)C_vQ_{th}}{π(0.102 3)(0.297 \times 10^{-3})}\)
or
\(\frac{Re}{C_v} = 2.75 \times 10^7 Q_{th}\)
Results of calculations made with this equation are also provided in Table 5.4. We see that for all ∆h values greater than 0.2, the ratio \(Re/C_v\) is greater than \(2 \times 10^5\). Now because \(C_v\) is always less than one, the Reynolds number will be greater than \(2 \times 10^5\). So for the calculations of this example,\(C_v = 0.984\), which is listed in the table. The actual flow rate \(Q_{ac} (= C_vQ_{th})\) is also listed. A graph of flow rate versus ∆h is provided in Figure 5.11.
