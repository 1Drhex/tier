Guest Question.

Consider the transfer function

\(G(s)=\frac{Y(s)}{U(s)}=\frac{10}{(5 s+1)(3 s+1)}\)

What is \(y(t \rightarrow \infty)\) for the following inputs:

(a) step input of height \(M\)

(b) unit impulse input \(\delta(t)\)

(c) \(\sin t\)

(d) unit rectangular pulse (Eq. 3-13, \(h=1\) )

\(f(t)= \begin{cases}0 & t<0 \\ h & 0 \leq t<t_{w} \\ 0 & t \geq t_{w}\end{cases}\) (3-13)

Admin
Solution

(a)

\(Y(s)=G(s) U(s)=\frac{10}{(5 s+1)(3 s+1)} \frac{M}{s}\)

 

\(Y(s)=\left(\frac{a_{1}}{5 s+1}+\frac{a_{2}}{3 s+1}+\frac{a_{3}}{s}\right) M\)

Partial fraction expansion:

\(a_{1}=125, a_{2}=-45, a_{3}=10\).

\(Y(s)=\left(\frac{125}{5 s+1}-\frac{45}{3 s+1}+\frac{10}{s}\right) M\)

Inverse Laplace:

\(y(t)=\left(25 e^{-t / 5}-15 e^{-t / 3}+10\right) M\)

Then,

\(\underset{t\rightarrow \infty }{\lim } y(t)=\underset{t\rightarrow \infty }{\lim } \left(25 e^{-t / 5}-15 e^{-t / 3}+10\right) M=10 M\)

Or, final value theorem from Chapter 3 applies:

\(\underset{t\rightarrow \infty }{\lim } y(t)=\underset{s\rightarrow 0 }{\lim } s Y(s)=\underset{s\rightarrow 0 }{\lim } s G(s) U(s)=\lim _{s \rightarrow 0} s \frac{10}{(5 s+1)(3 s+1)} \frac{M}{s}=10 M\)

(b)

\(Y(s)=G(s) U(s)=\frac{10}{(5 s+1)(3 s+1)}\)

 

\(Y(s)=\frac{a_{1}}{5 s+1}+\frac{a_{2}}{3 s+1}\)

Partial fraction expansion:

\(a_{1}=25, a_{2}=-15\).

\(Y(s)=\frac{25}{5 s+1}-\frac{15}{3 s+1}\)

Inverse Laplace:

\(y(t)=5 e^{-t / 5}-5 e^{-t / 3}\)

Then,

\(\underset{t\rightarrow \infty }{\lim } y(t)=\underset{t\rightarrow \infty }{\lim } 5 e^{-t / 5}-5 e^{-t / 3}=0\)

Or, final value theorem from Chapter 3 applies:

\(\underset{t\rightarrow \infty }{\lim } y(t)=\underset{s\rightarrow 0 }{\lim } s Y(s)=\lim _{s \rightarrow 0} s G(s) U(s)=\underset{s\rightarrow 0 }{\lim } s \frac{10}{(5 s+1)(3 s+1)}=0\)

(c)

\(Y(s)=G(s) U(s)=\frac{10}{(5 s+1)(3 s+1)} \frac{1}{s^{2}+1}\)

 

\(Y(s)=\frac{a_{1}}{5 s+1}+\frac{a_{2}}{3 s+1}+\frac{a_{3}+b_{3} j}{s+j}+\frac{a_{3}-b_{3} j}{s-j}\)

Partial fraction expansion:

\(a_{1}=625 / 26, a_{2}=-27 / 2, a_{3}=-2 / 13, b_{3}=7 / 26 \text {. }\)

 

\(Y(s)=\frac{625 / 26}{5 s+1}-\frac{27 / 2}{3 s+1}+\frac{-2 / 13+7 / 26 j}{s+j}+\frac{-2 / 13-7 / 26 j}{s-j}\)

Inverse Laplace:

\(y(t)=125 / 26 e^{-t / 5}-9 / 2 e^{-t / 3}-7 / 13 \sin t-4 / 13 \cos t\)

Then, \(\lim _{t \rightarrow \infty} y(t)\) does not converge.

(d)

\(Y(s)=G(s) U(s)=\frac{10}{(5 s+1)(3 s+1)} \frac{1}{s}\left(1-e^{-t w^{s}}\right)\)

According to part (a), we have:

\(y(t)=\left(25 e^{-t / 5}-15 e^{-t / 3}+10\right)-S\left(t-t_{w}\right)\left(25 e^{-(t-10) / 5}-15 e^{-(t-10) / 3}+10\right)\)

Then,

\(\underset{t\rightarrow \infty }{\lim } y(t)=\underset{t\rightarrow \infty }{\lim } \left(25 e^{-t / 5}-15 e^{-t / 3}+10\right)-\left(25 e^{-(t-10) / 5}-15 e^{-(t-10) / 3}+10\right)=0\)
