Guest Question.

Use Newton-Raphson to find a solution to

\(\left[\begin{array}{c}e^{x_{1} x_{2}} \\\cos \left(x_{1}+x_{2}\right)\end{array}\right]=\left[\begin{array}{l}1.2 \\0.5\end{array}\right]\)

where \(x_{1} \text { and } x_{2}\) are in radians. (a) Start with \(x_{1}(0)=1.0 \text { and } x_{2}(0)= 0.5\) and continue until (6.2.2) is satisfied with ε = 0.005. (b) Show that Newton-Raphson diverges for this example if \(x_{1}(0)=1.0 \text { and } x_{2}(0)=2.0\)

____________________

Eq. (6.2.2): \(\left|\frac{x_{k}(i+1)-x_{k}(i)}{x_{k}(i)}\right|<\varepsilon \quad \text { for all } k=1,2, \ldots, N\)

Admin
Solution

(a) \(e^{x_{1} x_{2}}-1.2=0\)

\(\begin{aligned}&\cos \left(x_{1}+x_{2}\right)-0.5=0 \\&J(i)=\left.\frac{d f}{d x}\right|_{x=x(i)}=\left[\begin{array}{ll}x_{2} e^{x_{1} x_{2}} & x_{1} e^{x_{1} x_{2}} \\-\sin \left(x_{1}+x_{2}\right) & -\sin \left(x_{1}+x_{2}\right)\end{array}\right]_{x=x(i)}\end{aligned}\)

In general form

\(x(i+1)=x(i)-J^{-1}(i) f(x)\)

Solving using MATLAB with \(x_{1}(0)=1, x_{2}(0)=1, \varepsilon=0.001:\)

i 0 1 2 3 4 5
\(x_{1}\) 1 0.6836 0.8956 0.8315 0.8267 0.8266
\(x_{2}\) 0.5 0.386 0.1518 0.2157 0.2205 0.2206
ε _ 0.3164 0.6068 0.4210 0.0226 0.0001

After 5 iterations ε < 0.005. Therefore,

\(\begin{aligned}&x_{1}=0.8266 \\&x_{2}=0.2206\end{aligned}\)

 

(b) When the initial conditions are changed, the solution diverges as each iteration results in very large solutions (on the order of \(1 e^{83}\) ). The error is (until the program fails):
