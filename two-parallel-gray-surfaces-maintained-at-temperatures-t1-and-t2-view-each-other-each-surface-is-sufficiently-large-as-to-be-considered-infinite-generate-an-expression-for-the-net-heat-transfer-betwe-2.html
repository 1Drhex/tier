Guest Question.

Two parallel gray surfaces maintained at temperatures \( T_{1} \) and \( T_{2} \) view each other. Each surface is sufficiently large as to be considered infinite. Generate an expression for the net heat transfer between these surfaces.using the methods developed in this section.

Admin
Solution

For this case n = 3 and the problem formulation will involve 7 equations—one for each surface. Part (a). Each of the surfaces is at a known temperature in this case, thus equation ( \( \left ( 1 – F_{ii} + \frac{ \varepsilon_{i}}{\rho_{i}} \right )J_{i} – \sum_{j = 1, j \neq i}^{n} F_{ij} J_{j} = \frac{\varepsilon_{i}}{\rho_{i}} E_{bi} \) ) applies. The following conditions are known:

 

\( T_{1} = 1100 K T_{2} = 550 K T_{3} = 0 K \)

 

\( F_{11} = 0 F_{21} = 0.2 F_{31} = 0.2 \)

 

\( F_{12} = 0.2 F_{23} = 0 F_{32} = 0.2 \)

 

\( F_{13} = 0.2 F_{23} = 0.8 F_{33} = 0.6 \)

 

\( \varepsilon_{1} = 1 \varepsilon_{2} = 1 \varepsilon_{3} = 1 \)

 

We can write the following:

 

\( \left ( 1 + \frac{\varepsilon_{1}}{\rho_{1}} \right ) J_{1} – \left [ F_{12} J_{2} + F_{13} J_{3} \right ] = \frac{\varepsilon_{1}}{\rho_{1}} E_{b1} \)

 

\( \left ( 1 + \frac{\varepsilon_{2}}{\rho_{2}} \right ) J_{2} – \left [ F_{21} J_{1} + F_{23} J_{3} \right ] = \frac{\varepsilon_{2}}{\rho_{2}} E_{b2} \)

 

\( \left ( 1 – F_{33} + \frac{\varepsilon_{3}}{\rho_{3}} \right ) J_{3} – \left [ F_{31} J_{1} + F_{32} J_{2} \right ] = \frac{\varepsilon_{3}}{\rho_{3}} E_{b3} \)

 

which, for the given conditions, reduce to

 

\( J_{1} = E_{b1} = \sigma T_{1}^{4} \)

 

\( J_{2} = E_{b2} = \sigma T_{2}^{4} \)

 

\( J_{3} = 0 \)

 

The net heat leaving plate 1 is thus, according to equation ( \( J_{i} – \sum_{j=1}^{n} F_{ij} J_{j} = \frac{q_{i}}{A_{i}} \) ), equal to

 

\( q_{1} = A_{1} \left [ J_{1} – F_{12} J_{2} \right ] \)

 

\( = A_{1} \left [ \sigma T_{1}^{4} – 0.2 \sigma T_{2}^{4} \right ] \)

 

\( = 4 m^{2} \left ( 5.676 \times 10^{-8} W/m^{2} \cdot K^{4} \right )\left [ 1100^{4} – 0.2 \left ( 550 \right )^{4} \right ] K^{4} = 328.3 kW \)

 

Part (b). Values of \( T_{i}\) and \( F_{ij}\) remain the same. The only change from part (a) is that \( \varepsilon_{3} = 0 \). The set of equations applying to the three surfaces are again

 

\( \left ( 1 + \frac{\varepsilon_{1}}{\rho_{1}} \right ) J_{1} – \left [ F_{12} J_{2} + F_{13} J_{3} \right ] = \frac{\varepsilon_{1}}{\rho_{1}} E_{b1} \)

 

\( \left ( 1 + \frac{\varepsilon_{2}}{\rho_{2}} \right ) J_{2} – \left [ F_{21} J_{1} + F_{23} J_{3} \right ] = \frac{\varepsilon_{2}}{\rho_{2}} E_{b2} \)

 

\( \left ( 1 + F_{33} + \frac{\varepsilon_{3}}{\rho_{3}} \right ) J_{3} – \left [ F_{31} J_{1} + F_{32} J_{2} \right ] = \frac{\varepsilon_{3}}{\rho_{3}} E_{b3} \)

 

as before. Substituting values for \( T_{i}, F_{ij} \), and \( \varepsilon_{i} \), we have

\( J_{1} = E_{b1} = \sigma T_{1}^{4} \)

 

\( J_{2} = E_{b2} = \sigma T_{2}^{4} \)

 

\( \left ( 1 – F_{33} \right )J_{3} – F_{31}J_{1} – F_{32}J_{2} = 0 \)

 

The expression for \( q_{i}\) is

\( q_{1} = A_{1} \left [ J_{1} – F_{12}J_{2} – F_{13}J_{3} \right ] = A_{1} \left [ J_{1} – F_{12}J_{2} – \frac{F_{13}}{ 1 – F_{33}} \left ( F_{31}J_{1} + F_{32}J_{2} \right ) \right ] \)

 

\( = A_{1} \left [ J_{1} \left ( 1 – \frac{F_{13} F_{31}}{1 – F_{33}} \right ) – J_{2} \left ( F_{12} + \frac{F_{13} F_{32}}{1 – F_{33}} \right ) \right ] \)

 

and, with numerical values inserted, we obtain

\( q_{1} = 4 \left ( 5.676 \times 10^{-8}\right )\left \{ \left (1100\right )^{4}\left [1 – \frac{\left (0.8\right )\left ( 0.2\right )}{1 – 0.6}\right ] – \left ( 550 \right )^{4}\left [0.2 + \frac{\left (0.8\right )\left (0.2 \right )}{1 – 0.6}\right ] \right \}= 187.0 kW \)

 

Part (c). Values of \( T_{i}\) and \( F_{ij} \) remain the same. Emissivities are

\( \varepsilon_{1} = 0.4 \varepsilon_{2} = 0.8 \varepsilon_{3} = 1 \)

 

Equations for the three surfaces are, again,

\( \left ( 1 + \frac{\varepsilon_{1}}{\rho_{1}} \right ) J_{1} – \left [ F_{12} J_{2} + F_{13} J_{3} \right ] = \frac{\varepsilon_{1}}{\rho_{1}} E_{b1} \)

 

\( \left ( 1 + \frac{\varepsilon_{2}}{\rho_{2}} \right ) J_{2} – \left [ F_{21} J_{1} + F_{23} J_{3} \right ] = \frac{\varepsilon_{2}}{\rho_{2}} E_{b2} \)

 

\( \left ( 1 + F_{33} + \frac{\varepsilon_{3}}{\rho_{3}} \right ) J_{3} – \left [ F_{31} J_{1} + F_{32} J_{2} \right ] = \frac{\varepsilon_{3}}{\rho_{3}} E_{b3} \)

 

which become

\( \left ( 1 + \frac{0.4}{0.6} \right ) J_{1} – \left [ F_{12} J_{2} + F_{13} J_{3} \right ] = \frac{0.4}{0.6} E_{b1} \)

 

\( \left ( 1 + \frac{0.8}{0.2} \right ) J_{2} – \left [ F_{21} J_{1} + F_{23} J_{3} \right ] = \frac{0.8}{0.2} E_{b2} \)

 

\( J_{3} = 0 \)

 

We now have

\( 1.67 J_{1} – 0.2 J_{2} = 0.67 E_{b1} \)

 

\( 5 J_{2} – 0.2 J_{1} = 4 E_{b2} \)

 

Solving these two equations simultaneously for \( J_{1} \) and \( J_{2} \), we get

\( J_{1} = 33900 W/m^{2} \)

 

\( J_{2} = 5510 W/m^{2} \)

 

and the value for \( q_{i} \) is evaluated as

 

\( q_{1} = \left [ 33900 – \frac{5510}{5} \right ] 4 = 131.2 kW \)
