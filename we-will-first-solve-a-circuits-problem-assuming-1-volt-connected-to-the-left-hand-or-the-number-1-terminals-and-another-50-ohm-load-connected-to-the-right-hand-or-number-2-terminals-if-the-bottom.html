Guest Question.

We will first solve a circuits problem assuming 1 volt connected to the left-hand (or the number 1) terminals and another 50-ohm load connected to the right-hand (or number 2) terminals. If the bottom rail is the reference node, we have just two unknown node voltages, say \(V_{b^{\prime} e}\) and \(y_{2}\). We simply sum the currents flowing out of each of these nodes, thus:

 

\(\frac{v_{b^{\prime} e}-1}{25}+\frac{v_{b^{\prime} e}}{200}+j 1.6 \pi \times 10^{-3} v_{b^{\prime} e}+j 10^{-5} \pi\left(v_{b^{\prime} e}-v_{2}\right)=0\)

 

\(\frac{v_{2}}{4000}+\frac{v_{2}}{50}+0.25 v_{b^{\prime} e}+j 10^{-5} \pi\left(v_{2}-v_{b^{\prime} e}\right)=0\)
Admin
Solution

We will multiply the first equation by 200, move the one volt to the right-hand side, and multiply the second equation by 4000, getting

 

\(v_{b^{\prime} e}(8+1+j(0.32 \pi+0.002 \pi))-j 0.002 \pi v_{2}=8\)

 

\(v_{b^{\prime} e}(1000-j 0.04 \pi)+v_{2}(81+j 0.04 \pi)=0\).

 

We might find it preferable to have the polar form of complex numbers. Then the equations would be

 

\(v_{b^{\prime} e}\left(9.0567 \angle 6.413^{\circ}\right)+V_{2}\left(.0628 \angle 90^{\circ}\right)=8\)

 

\(v_{b^{\prime} e}\left(1000 \angle-0.0072^{\circ}\right)+v_{2}\left(81 \angle 0.0889^{\circ}\right)=0\).

 

Solving the simultaneous equations by any convenient method, the student may compare his/her answers to the authorâ€™s results of

 

\(v_{b^{\prime} e}=0.889 \angle-1.501^{\circ} \text { and } v_{2}=10.97 \angle 178.338^{\circ}\).

 

Of course, we did all this to determine the scattering coefficients at this frequency.

\(S_{11}\) is the input reflection coefficient,

 

\(\Gamma_{i n}=\frac{Z_{i n}-Z_{0}}{Z_{i n}+Z_{0}}\)

 

Since the input voltage was one volt, we can get \(Z_{i n} \text { as } 1 / i_{i n}, \text { where }\)

 

\(i_{i n}=\frac{1-v_{b^{\prime} e}}{25}=0.00455 \angle 11.80^{\circ}\)

 

\(\text { Then } Z_{\text {in }}=219.64 \angle-11.80^{\circ} \text { and }\)

 

\(S_{11}=\frac{219.64 \angle-11.80^{\circ}-50 \angle 0^{\circ}}{219.64 \angle-11.80^{\circ}+50 \angle 0^{\circ}}=0.636 \angle-5.60^{\circ}\)

 

Now, \(S_{21}=b_{2} / a_{1}\), because we made \(a_{2}=0\) by connecting \(Z_{0}\), although the one volt we assumed contains both \(a_{1} \text { and } b_{1}\). However, we can say \(1=a_{1}+b_{1}=\) \(a_{1}\left(1+S_{11}\right) \text { and thus }\)

 

\(a_{1}=\frac{1}{1+S_{11}} \text { and } S_{21}=v_{2}\left(1+S_{11}\right)=17.94 \angle 176.16^{\circ}\)

 

These magnitudes are very close to the DC values quoted above, but the capacitors have begun to cause a bit of a phase shift.
Now, we recall that

 

\(S_{12}=\frac{b_{1}}{a_{2}}\),

 

so what we need to do is work our way through some voltage dividers. By inspection, we can say that

 

\(b_{1}=\frac{50}{75} v_{b^{\prime} e}\)

 

in which \(v_{b^{\prime} e}\) is itself a division of \(\nu_{2}\) appearing across a complex parallel impedance. The 75 â„¦ in parallel with 200 â„¦ gives an equivalent R in parallel with the 80 pF, of 54.54. The impedance of R in parallel with C can be massaged to a less hairy form of R/(1 + jÏ‰RC), and then

 

\(v_{b^{\prime} e}=\frac{\frac{R}{1+j \omega R C}}{\frac{R}{1+j \omega R C}+\frac{1}{j \omega C_{b^{\prime} c}}} v_{2}=\frac{R}{R\left(1+\frac{C}{C_{b^{\prime} c}}\right)+\frac{1}{j \omega C_{b^{\prime} c}}} v_{2} ; v_{1}=\frac{2}{3} v_{b^{\prime} e}\).

 

Hence

 

\(S_{12}=\frac{0.66667 \times 54.54}{161 \times 54.54-\frac{j 10^{5}}{\pi}}=0.001101 \angle 74.576^{\circ}\)

 

Next we set out to write the output admittance by writing \(i_{2}\) in terms of \(v_{2}\);

\(i_{2}=\frac{v_{2}}{4000}+v_{b^{\prime} e}(0.25)+\left(v_{2}-v_{b^{\prime}e}\right) j \omega C_{b^{\prime} c}\)

 

\(=v_{2}\left(\frac{1}{4000}+j \omega C_{b^{\prime} c}\right)+v_{b^{\prime} e}\left(0.25-j \omega C_{b^{\prime} c}\right)\)

 

However, remembering that \(v_{1}=(2 / 3) v_{b^{\prime} e}\) and also \(v_{1}=S_{12} v_{2}\), we may write

 

\(v_{b^{\prime} e}=1.5 S_{12} v_{2}\),

 

so we may write the output admittance by writing \(i_{2} \text { in terms of } v_{2} \text { : }\)

 

\(\frac{1}{4000}+j \omega C_{b^{\prime} c}+1.5 S_{12}\left(0.25-j \omega C_{b^{\prime} c}\right)=7.98 \times 10^{-4} \angle 63.23^{\circ}\)

 

The resulting reflection coefficient will be

 

\(S_{22}=\frac{1 / Y-50}{1 / Y+50}=0.965 \angle-4.08^{\circ}\).
