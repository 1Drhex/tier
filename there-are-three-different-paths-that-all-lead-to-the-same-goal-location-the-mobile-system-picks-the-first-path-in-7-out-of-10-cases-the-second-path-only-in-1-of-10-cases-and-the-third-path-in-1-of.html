Guest Question.

There are three different paths that all lead to the same goal location. The mobile system picks the first path in 7 out of 10 cases, the second path only in 1 of 10 cases, and the third path in 1 of 5 cases. There is a 5%, 10%, and 8% chance of an obstacle on the first, second, and third paths, respectively.
1. Determine the probability that the mobile system encounters an obstacle when going toward the goal.
2. The mobile system encountered on an obstacle. What is the probability that this happened on the first path?

Admin
Solution

Let us denote the first, second, and third path with symbols A1, A2, and A3, respectively, and the obstacle on any path with symbol B. The probabilities of selecting a specific path are the following: \(P\left(A_{1}\right)\) = 0.7, \(P\left(A_{2}\right)\) = 0.1, and \(P\left(A_{3}\right)\) = 0.2. The probability of encountering an obstacle on the first path is \(P\left(B \mid A_{1}\right)\) = 0.05, on the second path \(P\left(B \mid A_{2}\right)\) = 0.1, and on the third path \(P\left(B \mid A_{3}\right)\) = 0.08.

1. The probability that the mobile system encounters an obstacle is

\(P(B)=P\left(B \mid A_{1}\right) P\left(A_{1}\right)+P\left(B \mid A_{2}\right) P\left(A_{2}\right)+P\left(B \mid A_{3}\right) P\left(A_{3}\right)\)

= 0.05 · 0.7 + 0.1 · 0.1 + 0.08 · 0.2
= 0.061

2. The probability that the mobile system gets stuck on the first path can be calculated using Bayes’ rule (6.12):

 

\(P(x \mid y)=\frac{P(y \mid x) P(x)}{P(y)}\)

 

 

\(\begin{aligned}P\left(A_{1} \mid B\right) &=\frac{P\left(B \mid A_{1}\right) P\left(A_{1}\right)}{P(B)} \\&=\frac{P\left(B \mid A_{1}\right) P\left(A_{1}\right)}{P\left(B \mid A_{1}\right) P\left(A_{1}\right)+P\left(B \mid A_{2}\right) P\left(A_{2}\right)+P\left(B \mid A_{3}\right) P\left(A_{3}\right)}\end{aligned}\)

 

 

\(\begin{aligned}&=\frac{0.05 \cdot 0.7}{0.05 \cdot 0.7+0.1 \cdot 0.1+0.08 \cdot 0.2} \\&=0.5738\end{aligned}\)
