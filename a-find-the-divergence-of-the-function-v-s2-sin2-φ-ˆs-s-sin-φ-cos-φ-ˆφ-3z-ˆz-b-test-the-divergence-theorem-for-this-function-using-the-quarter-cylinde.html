Guest Question.

(a) Find the divergence of the function

\(v=s\left(2+\sin ^2\phi \right)\hat{s}+s\sin \phi \cos \phi \hat{\pmb{\phi} }+3z\hat{z}\) .

(b) Test the divergence theorem for this function, using the quarter-cylinder (radius 2, height 5) shown in Fig. 1.43.
(c) Find the curl of v.

Admin
Solution

(a) \(∇\cdot v=\frac{1}{s}\frac{\partial}{\partial s}\left(s s\left(2+\sin ^2\phi \right) \right)+\frac{1}{s}\frac{\partial}{\partial \phi }\left(s\sin \phi \cos \phi \right)+\frac{\partial}{\partial z}(3z) \)

=\(\frac{1}{s}2s \left(2+\sin ^2\phi \right)+\frac{1}{s}s\left(\cos ^2\phi -\sin ^2\phi \right)+3 \)

=\(4 + 2\sin ^2\phi +\cos ^2\phi -\sin ^2\phi +3\)

=\(4 +\sin ^2\phi +\cos ^2\phi +3=8\) .

(b) \(\int{}\left(∇\cdot v\right)d\tau =\int{}(8)s ds d\phi dz = 8 \int_{0}^{2}{} s ds\int_{0}^{\frac{\pi }{2} }{}d\phi \int_{0}^{5}{}dz = 8(2)\left(\frac{\pi }{2} \right) (5) = 40\pi \) .

Meanwhile, the surface integral has five parts:

top: \(z = 5, da = s ds d\phi \hat{z}; v·da = 3z s ds d\phi= 15s ds d\phi . \int{}v·da = 15\int_{0}^{2}{}s ds\int_{0}^{\frac{\pi }{2} }{}d\phi = 15\pi \) .

bottom: \(z = 0, da = −s ds d\phi \hat{z}; v·da = −3z s ds d\phi = 0. \int{}v·da = 0 \) .

back: \(\phi =\frac{\pi }{2} , da = ds dz \hat{\pmb{\phi}} ; v·da = s\sin \phi \cos \phi ds dz = 0.\int{}v·da = 0 \) .

left: \(\phi = 0, da = −ds dz \hat{\pmb{\phi} }; v·da = −s \sin \phi \cos \phi ds dz = 0.\int{}v·da = 0. \)

front: \(s = 2, da = s d\phi dz \hat{s} ; v·da = s\left(2 +\sin ^2\phi \right)s d\phi dz = 4 \left(2 +\sin ^2\phi \right)d\phi dz \)

\(\int{} v·da = 4\int_{0}^{\frac{\pi }{2} }{} \left(2+\sin ^2\phi \right) d\phi \int_{0}^{5}{}dz = (4)\left(\pi +\frac{\pi }{4} \right)(5) = 25\pi \) .

So \(\oint{}v·da = 15\pi + 25\pi = 40\pi \) .

(c) \(∇\times v=\left(\frac{1}{s}\frac{\partial }{\partial \phi }(3z) −\frac{\partial}{\partial z}\left(s\sin \phi \cos \phi \right) \right)\hat{s}+\left(\frac{\partial}{\partial z}\left(s\left(2+\sin ^2\phi \right) \right) -\frac{\partial}{\partial s}(3z) \right)\hat{\pmb{\phi} } \)

+ \(\frac{1}{s}\left(\frac{\partial}{\partial s}\left(s^2\sin \phi \cos \phi \right)-\frac{\partial}{\partial \phi } \left(s\left(2+\sin ^2\phi \right) \right) \right) \hat{z} \)

= \(\frac{1}{s} \left(2s\sin \phi \cos \phi -s2\sin \phi \cos \phi \right)\hat{z}=0 \) .
