Guest Question.

The surfaces \(\rho \)= 3 and 5, \(\phi\) = \(\mathring { 100 }\) and \(\mathring { 130 }\) , and z = 3 and 4.5 identify a closed surface. (a) Find the volume enclosed. (b) Find the total area of the enclosing surface. (c) Find the total length of the twelve edges of the surface. (d) Find the length of the longest straight line that lies entirely within the volume.

Admin
Solution

Vol = \(\int _{ 3 }^{ 4.5 } \int _{ \mathring { 100 } }^{ \mathring { 130 } } \int _{ 3 }^{ 5 } \rho d\rho d\phi dz \)= 6.28
Note : The limits in the \(\phi\) integration must be converted to radians ( as was done here, but not shown).
b)
Area =\(2 \int _{ \mathring { 100 } }^{ \mathring { 130 } } \int _{ 3 }^{ 5 } \rho d\rho d\phi \) =\( \int _{ 3 }^{ 4.5 } \int _{ \mathring { 100 } }^{ \mathring { 130 } } 3 d\phi dz + \int _{ 3 }^{ 4.5 } \int _{ \mathring { 100 } }^{ \mathring { 130 } } 5 d\phi dz + 2\int _{ 3 }^{ 4.5 } \int _{ 3 }^{ 5 } d\rho dz \)= 20.7
c)
Length =\( 4 \times 1.5 + 4 \times 2 + 2 \times \left[ \overset {\mathring { 30 } }{ \mathring { 360 } } \times 2\pi \times 3 + \overset {\mathring { 30 } }{ \mathring { 360 } } \times 2\pi \times 5 \right] \) = 22.4
d) this will be between the points A\((\rho = 3,\phi = \mathring { 100 }, z = 3)\)and B\((\rho = 5,\phi = \mathring { 130 }, z = 4.5)\). Performing point transformation to cartesian coordinates, these become A\((x = -0.52,y = 2.95, z = 3)\) and B\((x = -3.12,y = 3.83, z = 4.5)\). Taking A and B as vectors directed from the origin, the requested length is
Length = \(\left| B-A\right|\) = \(\left| (-2.69,0.88,1.5) \right|\) = 3.21
