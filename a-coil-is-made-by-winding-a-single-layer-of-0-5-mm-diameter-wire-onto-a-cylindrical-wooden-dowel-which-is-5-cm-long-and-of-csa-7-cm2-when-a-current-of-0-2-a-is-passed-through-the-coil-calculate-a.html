Guest Question.

A coil is made by winding a single layer of \(0.5 \mathrm{~mm}\) diameter wire onto a cylindrical wooden dowel, which is \(5 \mathrm{~cm}\) long and of csa \(7 \mathrm{~cm}^{2}\). When a current of \(0.2 \mathrm{~A}\) is passed through the coil, calculate (a) the mmf produced, (b) the flux density, and (c) the flux produced.

Admin
Solution

(a) \(I=0.2 \mathrm{~A} ; \ell=5 \times 10^{-2} \mathrm{~m} ; A=7 \times 10^{-4} \mathrm{~m}^{2} ; d=0.5 \times 10^{-3} ; \mu_{\mathrm{r}}=1\) (wood)
Since \(F=N I\) ampere turn, we first need to calculate the number of turns of wire on the coil. Consider Fig. 4.10 which represents the coil wound onto the dowel.
From Fig. 4.10 it may be seen that the number of turns may be obtained by dividing the length of the dowel by the diameter (thickness) of the wire.
\(\begin{aligned} \text { Thus, } N &=\frac{\ell}{d}=\frac{50}{0.5} \\ \text { so, } N &=100 \\ \text { Thus, } F &=100 \times 0.2=20 \text { At} \end{aligned}\)

(b) \(B=\frac{\Phi}{A}\) tesla, or \(B=\mu_{0} \mu_{\mathrm{r}} H\) tesla
but since we do not yet know the value for the flux, but can calculate the value for \(H\), then the second equation needs to be used.

\(\begin{aligned}H &=\frac{F}{\ell} \text { ampere turn } / \text { metre }=\frac{20}{5 \times 10^{-2}} \\\text { so, } H &=400 \mathrm{At} / \mathrm{m} \\\text { and, } B &=4 \pi \times 10^{-7} \times 1 \times 400=5.026 \times 10^{-4} \\\text { so, } B &=503 \mu \mathrm{T} \end{aligned}\) \(\begin{aligned}\text {(c) }\Phi &=B A \text { weber }=503 \times 10^{-6} \times 7 \times 10^{-4} \\\text { thus, } \Phi &=0.352 \mu \mathrm{Wb} \end{aligned}\)
