Guest Question.

An aircraft with \(C_{D_{0}}=0.020, k_{1}=0.12, \text { and } k_{2}=0\) is flying at h = 30,000 ft and \(M_{\infty}=0.8\). If the aircraft has a wing area of 375 \(ft ^{2}\) and it weighs 25,000 Ib, what is its drag coefficient, and how much drag is it generating? If the aircraft is in SLUF, how much thrust is its engine producing? If its \(C_{L_{\max }}=1.8\), what is its stall speed at that altitude?

Admin
Solution

The atmospheric conditions for this situation are obtained from the standard atmosphere table for \(h=30,000 ft \text { as } \rho=0.00089 slug / ft ^{3} \text { and } a= 994.8 ft/s,\) so

\(V_{\infty}=M_{\infty} a=(0.8)(994.8 ft / s )=795.8 ft / s\)

and

\(q=\frac{1}{2} \rho V^{2}=\frac{1}{2}\left(0.00089 \text { slug/ft }^{3}\right)(795.8)^{2}=281.8 lb / ft ^{2}\)

 

so, using Eq. (5.18)

\(C_{L}=\frac{W}{q S}=\frac{25,000 lb }{\left(281.8 lb / ft ^{2}\right)\left(375 ft ^{2}\right)}=0.2366\)

and, using Eq. (5.19)

\(\begin{aligned}&C_{D}=C_{D_{0}}+k C_{L}^{2}=0.020+0.12(0.2366)^{2}=0.027 \\&D=C_{D} q S=0.027\left(281.8 lb / ft ^{2}\right)\left(375 ft ^{2}\right)=2823 lb\end{aligned}\)

If the aircraft is in SLUE, then

\(T=T_{R}=D=2823 lb\)

Finally, its stall speed in SLUF is given by Eq. (5.20) as

\(V_{\text {stall }}=\sqrt{\frac{2 W}{\rho S C_{L_{\max }}}}=\sqrt{\frac{2(25,000 lb )}{\left(0.00089 slug / ft ^{3}\right)\left(375 ft ^{2}\right)(1.8)}}=288.5 ft / s\)

____________________________________________________________________________________________________________________

Eq. (5.18) : \(C_{L}=\frac{W}{q S}\)

Eq. (5.19) : \(C_{D}=C_{D_{0}}+k C_{L}^{2}, \quad D=C_{D} q S\)

Eq. (5.20) : \(V_{\text {stall }}=\sqrt{\frac{2 W}{\rho S C_{L_{\max }}}}\)
