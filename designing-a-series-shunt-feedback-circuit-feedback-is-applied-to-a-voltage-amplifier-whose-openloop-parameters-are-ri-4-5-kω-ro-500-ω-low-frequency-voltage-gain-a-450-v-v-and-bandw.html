Guest Question.

Designing a series-shunt feedback circuit Feedback is applied to a voltage amplifier whose openloop parameters are \( R_{ i }=4.5 k \Omega, R_{ o }=500 \Omega \), low-frequency voltage gain A = -450 V/V, and bandwidth \( BW =f_{ H }=10 kHz \). The load resistance is \( R_{ L }=10 k \Omega \). Determine the values of the feedback network so that the following specifications are satisfied:

(a) Bandwidth with feedback \( f_{ Hf }=1 MHz , R_{ if }>R_{ i }, \text { and } R_{ of }<R_{ o } \)

(b) \( R_{ if }=50 R_{ i } \text { and } R_{ of }<R_{ o } \)

(c) \( R_{\text {if }}>R_{ i } \text { and } R_{\text {of }}=R_{ o } / 250 \)

Admin
Solution
\( R_{ i }=4.5 k \Omega, R_{ o }=500 \Omega, A=-450 V / V , \text { and } f_{ H }=10 kHz.\)

(a) Since the input resistance \( R_{\text {if }} \) should increase and the output resistance \( R_{\text {of }} \) should decrease, we can see from Table 10.1 that the feedback must be of the series-shunt type. Thus, we can use the equations derived in Sec. 10.6. The feedback network consists of \( R_{1} \text { and } R_{ F }\), as shown in Fig. 10.47. Since A is negative, the value of \( v_{ f } \) will be negative. Thus, \( v_{ f } \text { is added to } v_{ s } \) rather than subtracted from it \( \text { (i.e., } v_{ s }+v_{ f }=v_{ e } \text { ) } \), and this is accomplished by connecting the feedback signal to the negative terminal of \( v_{ S } \). To minimize the loading effects, \( \left(R_{1}+R_{ F }\right) \) must be much larger than \( R_{ L } \). The condition

\( \left(R_{1}+R_{ F }\right)>>R_{ L } \) (10.98)

is generally satisfied by choosing \( R_{1}+R_{ F }=10 R_{ L } \). Since the gain–bandwidth product remains constant, Eq. (10.24) gives \( A f_{ H }=A_{ f } f_{ Hf } \). That is,

\(\text { Gain-bandwidth product GBW }=A_{ o } f_{ H }\) (10.24)

 

\(450 \times 10 kHz =A_{ f } \times 1 MHz\)

which gives \( A_{ f }=450 / 100=4.5 V / V \). From Eq. (10.37),

\(A_{ f }=\frac{A}{1+\beta A}\) (10.37)

 

\( 4.5=\frac{450}{1+450 \beta} \)

which gives β = 0.22. The feedback factor β is related to \( R_{1} \text { and } R_{ F } \) by

\( \beta=\frac{R_{ F }}{R_{1}+R_{ F }}=\frac{R_{ F }}{10 R_{ L }} \) (10.99)

 

which, for \( \beta=0.22 \text { and } R_{ L }=10 k \Omega \), gives

\(\begin{array}{ll}& R_{ F }=10 \beta R_{ L }=10 \times 0.22 \times 10 k =22 k \Omega \\\text { and } & R_{1}=10 R_{ L }-R_{ F }=100 k -22 k =78 k \Omega\end{array}\)

(b) Since \( R_{ if }=50 R_{ i } \text { and } R_{ if }=R_{ i }(1+\beta A) \) from Eq. (10.40), we get

\(R_{ if }=\frac{v_{ s }}{i_{ i }}=\frac{v_{ e }(1+\beta A)}{v_{ e } / R_{ i }}=(1+\beta A) R_{ i }\) (10.40)

 

\(50=1+\beta A \quad(\text { for negative feedback })\)

which, for \( |A|=450 \text {, gives } \beta=0.109 \text {. For } \beta=0.109 \text { and } R_{ L }=10 k \Omega \), Eq. (10.99) gives

\(\begin{aligned}&\quad R_{ F }=10 \beta R_{ L }=10 \times 0.109 \times 10 k \approx 11 k \Omega \\&\text { and } \quad R_{1}=10 R_{ L }-R_{ F }=100 k -11 k =89 k \Omega\end{aligned}\)

(c) Since \( R_{ of }=R_{ o } / 250 \text { and } R_{ of }=R_{ o } /(1+\beta A) \) from Eq. (10.44), we get

\( R_{ of }=\frac{R_{ o }}{1+\beta A} \) (10.44)

\( 250=1+\beta A \)

which, for \( |A|=450 \text {, gives } \beta=0.5533 \text {. For } \beta=0.5533 \text { and } R_{ L }=10 k \Omega \text {, } \) Eq. (10.99) gives

\(\begin{array}{ll}& R_{ F }=10 \beta R_{ L }=10 \times 0.5533 \times 10 k \approx 55 k \Omega \\\text { and } & R_{1}=10 R_{ L }-R_{ F }=100 k -55 k =45 k \Omega\end{array}\)

NOTE: The above solution gives approximate values for the designer to start with. More accurate values of \( A _{ f }, R_{ if }, \text { and } R_{ of } \) could be found by considering the loading effects of the feedback network and the load resistance (as in ). The design steps should be repeated until the desired specifications are met.
