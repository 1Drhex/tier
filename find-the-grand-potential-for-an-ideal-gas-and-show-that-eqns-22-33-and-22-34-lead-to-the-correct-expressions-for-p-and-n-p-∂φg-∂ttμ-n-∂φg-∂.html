Guest Question.

Find the grand potential for an ideal gas, and show that eqns 22.33 and 22.34 lead to the correct expressions for p and \( p = -\left(\frac{\partial \Phi _{G} }{V} \right)_{T,\mu },\) (22.33) \(N = -\left(\frac{\partial \Phi _{G} }{\partial \mu } \right)_{T,V}.\) (22.34)

Admin
Solution

Using eqns 21.36 and 22.15 we have that

\( F = -Nk_{B}T\ln \left(\frac{Ve}{N\lambda ^{3}_{th}} \right)\)
\( = Nk_{B}T\left[\ln (n\lambda ^{3}_{th})-1\right].\) (21.36)

 

\( \mu =k_{B}T\ln (n\lambda ^{3}_{th}).\) (22.15)

 

\( \Phi _{G} = Nk_{B}T\left[\ln (n\lambda ^{3}_{th})-1\right]-Nk_{B}T\ln (n\lambda ^{3}_{th})\)

 

\( = -Nk_{B}T,\) (22.35)

and using the ideal gas equation \((pV = Nk_{B}T))\) this becomes

\( \Phi _{G}=-pV.\) (22.36)

We can check that eqn 22.34 leads to the correct value of p by evaluating

\( \left(\frac{\partial \Phi _{G}}{\partial \mu } \right)_{T,V}=\left(\frac{\partial \Phi _{G}}{\partial N } \right)_{T,V}\left(\frac{\partial N}{\partial \mu } \right)_{T,V},\) (22.37)

and since \(\left(\frac{\partial \Phi _{G}}{\partial N } \right)_{T,V}=-k_{B}T\) (from eqn 22.35) and \(\left(\frac{\partial \mu}{\partial N} \right)_{T,V} =-k_{B}T/N\) we have that

\( \left(\frac{\partial \Phi _{G}}{\partial \mu } \right)_{T,V}=-k_{B}T\times \frac{N}{k_{B}T}=-N ,\) (22.38)

justifying eqn 22.34. Similarly,\(^{3}\)

\(^{3}\)Using the reciprocity theorem, with T held constant for all terms.

\(\left(\frac{\partial \Phi _{G}}{\partial V} \right)_{T,\mu } =-\left(\frac{\partial \Phi _{G}}{\partial \mu } \right)_{T,V}\left(\frac{\partial \mu }{\partial V} \right)_{T,\Phi_{G} }=N\left(\frac{\partial \mu }{\partial V} \right)_{T,\Phi_{G} }\) (22.39)

and since the constraint of constant T and constant \(\Phi_{G} = −Nk_{B}T\) means constant T and N, and using \(N = nV ,\) we can use eqn 22.15 to obtain

\( \mu =k_{B}T\ln (n\lambda ^{3}_{th}).\) (22.15)

 

 

\(\left(\frac{\partial \mu }{\partial V} \right)_{T,V}=k_{B}T\left(\frac{\partial \ln (N\lambda ^{3}_{th}/V) }{\partial V } \right)_{T,N}=-\frac{k_{B}T}{V},\) (22.40)

and eqn 22.39 becomes

\( \left(\frac{\partial \Phi _{G}}{\partial V} \right)_{T,\mu }=-\frac{Nk_{B}T}{V}=-p,\) (22.41)

thus justifying eqn 22.33.
