Guest Question.

Let G = 15r\({ a }_{ \phi }\). (a) Determine \(\oint { } G .dL\) for the circular path r = 5, \(\theta = \mathring { 25 }\), \(0\le \phi \le 2\pi\). (b) Evaluate \(\int _{ S }^{ }{ } \left( \triangledown \times G \right)\) .dS over the spherical cap r = 5, \(0\le \theta \le \mathring { 25 }, 0\le \phi \le 2\pi\).

Admin
Solution

a) \(\oint { } G.dL=\int _{ 0 }^{ 2\pi }{ 15(5) } { a }_{ \phi }.{ a }_{ \phi }(5)\sin { (\mathring { 25 } ) } d\phi =2\pi (375)\sin { (\mathring { 25 } ) } =995.8\)
b) When evaluating the curl of G using the formula in spherical coordinates, only one of the six terms survives:
\(\triangledown \times G=\frac { 1 }{ r\sin { \theta } } \frac { \partial \left( { G }_{ \phi }\sin { \theta } \right) }{ \partial \theta } { a }_{ r }=\frac { 1 }{ r\sin { \theta } } 15r\cos { \theta } { a }_{ r }=15\cot { \theta } { a }_{ r }\)
Then
\(\int _{ S }^{ }{ } (\triangledown \times G).dS=\int _{ 0 }^{ 2\pi }{ } \int _{ 0 }^{ \mathring { 25 } }{ } 15\cot { \theta } { a }_{ r }.{ a }_{ r }{ \left( 5 \right) }^{ 2 }\sin { \theta } d\theta d\phi \\ =2\pi \int _{ 0 }^{ \mathring { 25 } }{ } 15\cos { \theta } \left( 25 \right) d\theta =2\pi (15)(25)\sin { \left( \mathring { 25 } \right) } =995.8\)
