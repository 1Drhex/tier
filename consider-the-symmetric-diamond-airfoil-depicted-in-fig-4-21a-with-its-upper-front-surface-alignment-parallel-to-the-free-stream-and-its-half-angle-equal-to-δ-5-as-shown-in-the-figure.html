Guest Question.

Consider the symmetric diamond airfoil depicted in Fig. 4.21a, with its upper front surface alignment parallel to the free stream and its half angle equal to \(δ = 5°\), as shown in the figure. Assume the free stream Mach number \(M_{1} = 4\) and \(P_{1} = 20 kPa\) and a chord length of 0.6 m. Find \(C_{L}\) and \(C_{D}\).

Admin
Solution

One way of finding \(C_{L}\) and \(C_{D}\) is to first find L and D as described by Eqn. (4.57) \(C_{p}\equiv \frac{p-p_{1}}{\frac{1}{2}\rho _{1} V^{2}_{1} } = \frac{p-p_{1}}{\frac{1}{2} \frac{\gamma p_{1}}{\gamma RT_{1}} V^{2}_{1} } =\frac{2}{\gamma M^{2}_{1} } \frac{p-p_{1}}{p_{1}} \). There are now, however, four surfaces to consider. Region 2 in the figure is not disturbed and \(p_{2} = p_{1}\).

Region 3:

\(M_{1} = 4\) and \(δ_{13} = 10°\) leads to \(θ_{13} ≈ 23°\) and \(M_{1n} = M_{1} \sin23° ≈ 1.56\). Therefore, using the shock table or Eqn. (4.20) \(M^{2}_{2} =1-\epsilon =2-M^{2}_{1}\) yields

\(\frac{p_{3} }{p_{1} }=2.673\)

 

\(\frac{p_{t3} }{p_{t1} }=0.910\)

 

\(\frac{T_{3} }{T_{1} }=1.361\)

 

\(M_{3n}=0.681\)

 

\(M_{3t} =M_{1t} \sqrt{\frac{T_{1} }{T_{3} }} =M_{1} \cos \theta _{13} \sqrt{\frac{T_{1} }{T_{3} }}=4 \cos 23° \sqrt{\frac{1}{1.361} }=3.156\)

Therefore \(M_{3} = 3.229\) and the corresponding Prandtl-Meyer angle is \(\nu _{3}\approx 54°\)

Region 4:

\(\nu _{2}\approx 65.78° \nu _{4}=\nu _{2}+\Delta \nu _{24}=65.78°+10°=75.78°\) and \(M_{4} ≈ 4.88\),

\(\frac{p_{4} }{p_{t4} } =0.0022\)

 

\(\frac{p_{4} }{p_{1} }=\frac{p_{4} }{p_{t4} } \times \frac{p_{t4} }{p_{t2} }\times \frac{p_{t2} }{p_{t1} }\times \frac{p_{t1} }{p_{1} }=\left(0.0022\right) \left(1\right) \left(1\right) \left(\frac{1}{0.0066} \right)=0.333 \)

Region 5:

\(\nu _{5}=\nu _{3}+\Delta \nu _{35}=54°+10°=64°\). Thus \(M_{5} ≈ 3.87\) and \(\frac{p_{5} }{p_{t5} }=0.0079\).

\(\frac{p_{5} }{p_{1} }=\frac{p_{5} }{p_{t5} } \times \frac{p_{t5} }{p_{t3} }\times \frac{p_{t3} }{p_{t1} }\times \frac{p_{t1} }{p_{1} }=\left(0.0079\right) \left(1\right) \left(0.910\right) \left(\frac{1}{0.0066} \right)=1.089\)

Eqn. (4.57) states

\(C_{p}\equiv \frac{p-p_{1}}{\frac{1}{2}\rho _{1} V^{2}_{1} } = \frac{p-p_{1}}{\frac{1}{2} \frac{\gamma p_{1}}{\gamma RT_{1}} V^{2}_{1} } =\frac{2}{\gamma M^{2}_{1} } \frac{p-p_{1}}{p_{1}} \)

Using Eqn. (4.58) \(C_{L}\equiv \frac{\left(p_{L} -p_{U}\right) c \cos \alpha }{\frac{1}{2} \rho _{1} V^{2}_{1} c} =\left(C_{pL}-C_{pU}\right) \cos \alpha \) by incorporating the angle of the four surfaces, each of which has a length of \(c/2 \cos δ\), leads to

\(C_{L}=\frac{2}{\gamma M^{2}_{1}} \left[\left(\frac{p_{3}-p_{1}}{p_{1}} \right) \frac{1}{2 \cos \delta } \cos 2\delta +\left(\frac{p_{5}-p_{1}}{p_{1}} \right) \frac{1}{2 \cos \delta } \cos 0-\left(\frac{p_{2}-p_{1}}{p_{1}} \right) \frac{1}{2 \cos \delta } \cos 0-\left(\frac{p_{4}-p_{1}}{p_{1}} \right) \frac{1}{2 \cos \delta } \cos 2\delta \right] \)

 

\(C_{D}=\frac{2}{\gamma M^{2}_{1}} \left[\left(\frac{p_{3}-p_{1}}{p_{1}} \right) \frac{1}{2 \cos \delta } \sin 2\delta +\left(\frac{p_{5}-p_{1}}{p_{1}} \right) \frac{1}{2 \cos \delta } \sin 0-\left(\frac{p_{2}-p_{1}}{p_{1}} \right) \frac{1}{2 \cos \delta } \sin 0-\left(\frac{p_{4}-p_{1}}{p_{1}} \right) \frac{1}{2 \cos \delta } \sin 2\delta \right] \)

Because \(2δ = 10°, = 1.4\); \(p_{2} = p_{1}\), \(\frac{p_{3}-p_{1}}{p_{1}} =1.673\), \(\frac{p_{5}-p_{1}}{p_{1}} =0.089\), \(\frac{p_{4}-p_{1}}{p_{1}} =-0.667\) and \(M_{1} = 4\), \(C_{L} = 0.1072\) and \(C_{D} = 0.018\).

Now if the thin-airfoil theory is used, α = δ = 5° = 0.0873 (radians), \(\frac{dh}{dx} =\tan \alpha \approx \alpha \), \(\alpha _{c}=0\) because of symmetry, and also \(\alpha _{c}=0\) 0 along the whole chord length.

\(C_{L}=\frac{4\alpha }{\sqrt{M^{2}_{1} -1} } =\frac{4\times 0.0873}{\sqrt{4^{2}-1} } =0.090\)

 

\(C_{D}=\frac{4\alpha }{\sqrt{M^{2}_{1} -1} }\left[\alpha ^{2}+ \overline{\alpha ^{2}_{c}} +\overline{\left(\frac{dh}{dx}\right)^{2}} \right] =\frac{4}{\sqrt{4^{2}-1} } \left[0.0873^{2}+0+0.0873^{2}\right] =0.0158\)
