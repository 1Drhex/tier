Guest Question.

A 240 V , D C series motor has a resistance of 0.2 ohms, when the line current is 40 A, speed is 1800 rpm. Find the resistance to be added in series with the motor (i) to make the motor run at 900 rpm when line current is 60 A (i i) to limit the speed to 3600 rpm when line current is 10 A (iii) speed of motor when it is connected directly to mains and line current is 60 A. Assume flux at 60 A is 1.8 times of flux at 40 A.

Admin
Solution
\(\begin{array}{l}\text { Here, } V=240 V ; R_{m}=0.2 \Omega ; I_{a 1}=40 A ; N_{1}=1800 rpm \\\text { Back emf } E_{b 1}=V-I_{a 1} R_{m}=240-40 \times 0.2=232 V \\\text { For } I_{a 1}=40 A ; flux =\phi_{1} \\\text { For } I_{a 2}=60 A ; flux =\phi_{2} \text { where } \phi_{2}=1.8 \phi_{1}\end{array}\)

 

\(\text { (i) } N_{2}=900 rpm ; I_{a 2}=60 A ; \text { then } \phi_{2}=1.8 \phi_{1}\)

 

Let R be the resistance added in motor circuit

\( E_{b 2} =V-I_{a 2}\left(R_{m}+R\right)=240-60(0.2+R) \)

 

\( =240-12-60 R=228-60 R\)

 

Now

\( N \propto \frac{E_{b}}{\phi} \quad \text { or } \quad \frac{N_{2}}{N_{1}}=\frac{E_{b 2}}{\phi_{2}} \times \frac{\phi_{1}}{E_{b 1}}\)

 

\(\text { or } \quad \frac{900}{1800}=\frac{(228-60 R)}{1.8 \phi_{1}} \times \frac{\phi_{1}}{232}\)

 

\(\text { or } \quad 228-60 R =\frac{1.8 \times 232 \times 900}{1800}\)

 

\(\text { or } \quad 228-60 R =208.8 \quad \text { or } \quad R=0.32 \Omega\)

 

\(\text { (ii) } N_{3}=3600 rpm ; I_{a 3}=10 A ; \phi_{3} \propto I_{a 3} \text { and } \phi_{1} \propto I_{a 1}\)

 

\(\text { Let } R_{1} \text { be the resistance added in series with the motor }\)

 

\(\begin{aligned}E_{b 3} &=V-I_{a 3}\left(R_{m}+R_{1}\right)=240-10\left(0.2+ R _{1}\right) \\&=240-2-10 R _{1}=238-10 R _{1} \\N & \propto \frac{E_{b}}{\phi} ; \frac{N_{3}}{N_{1}}=\frac{E_{b 3}}{\phi_{3}} \times \frac{\phi_{1}}{E_{b 1}}\end{aligned}\)

 

\(\begin{array}{l}\frac{N_{3}}{N_{1}}=\frac{E_{b_{3}}}{E_{b_{3}}} \times \frac{I_{a 1}}{I_{b_{1}}}\\\text { (Considering linear magnetisation) }\end{array}\)

 

\(\frac{3600}{1800}=\frac{\left(238-10 R_{1}\right)}{10} \times \frac{40}{232}\)

 

\(\begin{array}{l}\text { or }\\238-10 R _{1}=\frac{3600 \times 10 \times 232}{1800 \times 40}\\\text { or } \quad 238-10 R _{1}=116 \text { or } R_{1}=12.2 \Omega \end{array}\)

 

\(\text { (iii) } N_{4}=? ; I_{a 4}=60 A \text { and } \phi_{4}=1.8 \phi_{1}\)

 

\(E_{b 4}=V-I_{a 4}\left(R_{m}\right)=240-60 \times 0.2=228 V\)

 

\(\frac{N_{4}}{N_{1}}=\frac{E_{b 4}}{\phi_{2}} \times \frac{\phi_{1}}{E_{b 1}}\)

 

\(\begin{array}{l}\text { or } \quad N_{4}=\frac{228}{1.8 \phi_{1}} \times \frac{\phi_{1}}{232} \times 1800\\\text { or }\\N_{4}=982.76 rpm \end{array}\)
