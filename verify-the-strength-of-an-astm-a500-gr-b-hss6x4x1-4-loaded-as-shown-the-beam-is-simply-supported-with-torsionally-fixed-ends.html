Guest Question.

Verify the strength of an ASTM A500 Gr. B HSS6×4×1/4 loaded as shown. The beam is simply supported with torsionally fixed ends.

Admin
Solution

Material Properties:

ASTM A500 Gr. B \(F_{y}=46 ksi \quad F_{u}=58 ksi\)

Manual
Table 2-3

Geometric properties:

HSS 6×4×1/4 \(\frac{h}{t}=22.8 \quad \frac{b}{t}=14.2 \quad t=0.233 \text { in. } \quad Z_{x}=8.53 \text { in. }{ }^{3}\)

Manual
Table 1-11

Calculate the required strengths

LRFD ASD
\(\begin{aligned}w_{u} &=1.2(0.460 kip / ft )+1.6(1.38 kip / ft ) \\&=2.76 kip / ft\end{aligned}\)

\(\begin{aligned}V_{r} &=V_{u}=\frac{w_{u} l}{2} \\&=\frac{2.76 kip / ft (8.00 ft )}{2}=11.0 kips\end{aligned}\) \(\begin{aligned}M_{r} &=M_{u}=\frac{w_{u} l^{2}}{8} \\&=\frac{2.76 kip / ft (8.00 ft )^{2}(12 in . / ft )}{8} \\&=265 kip – in .\end{aligned}\)

\(\begin{aligned}T_{r} &=T_{u}=\frac{w_{u} l e}{2} \\&=\frac{2.76 kip / ft (8.00 ft )(6.00 in .)}{2} \\&=66.2 kip – in .\end{aligned}\)

 \(\begin{aligned}w_{a} &=0.460 kip / ft +1.38 kip / ft \\&=1.84 kip / ft\end{aligned}\)

\(\begin{aligned}V_{r} &=V_{a}=\frac{w_{a} l}{2} \\&=\frac{1.84 kip / ft (8.00 ft )}{2}=7.36 kips\end{aligned}\) \(\begin{aligned}M_{r} &=M_{a}=\frac{w_{a} l^{2}}{8} \\&=\frac{1.84 kip / ft (8.00 ft )^{2}(12 in / ft )}{8} \\&=177 kip – in .\end{aligned}\)

\(\begin{aligned}T_{r} &=T_{a}=\frac{w_{a} l e}{2} \\&=\frac{1.84 kip / ft (8.00 ft )(6.00 in .)}{2} \\&=44.2 kip – in .\end{aligned}\)

Calculate the nominal shear strength

\(h=6.00 in .^{2}-3(0.233 in .)=5.30 in\)

Section G5

\(A_{w}=2 h t=2(5.30 in .)(0.233 in .)=2.47 in .^{2}\)

 

\(k_{v}=5\)

 

Calculate the web shear coefficient

\(\frac{h}{t_{w}}=22.8 \leq 1.10 \sqrt{\frac{k_{ v } E}{F_{y}}}=1.10 \sqrt{\frac{5(29,000 ksi )}{46 ksi }}=61.7 \text { therefore, } C_{v}=1.0\)

Eqn. G2-3

 

\(V_{n}=0.6 F_{y} A_{w} C_{v}=0.6(46 ksi )\left(2.47 in .^{2}\right)(1.0)=68.2 kips\)

Eqn. G2-1

 

Calculate the available shear strength

LRFD ASD
\(\begin{array}{l}\phi_{ v }=0.90 \\V _{c}=\phi_{v} V_{n}=0.90(68.2 kips )=61.4 kips\end{array}\) \(\begin{array}{l}\Omega_{ v }=1.67 \\V_{c}=V_{n} / \Omega_{v}=68.2 kips / 1.67=40.8 kips\end{array}\)

Section G1

Calculate the nominal flexural strength

Section F7

Flexural yielding limit state

\(M_{n}=M_{p}=F_{y} Z_{x}=46 ksi \left(8.53 in .^{3}\right)=392 \text { kip-in. }\)

Eqn. F7-1

Flange local buckling limit state

\(\frac{b}{t}=14.2<1.12 \sqrt{\frac{E}{F_{y}}}=28.1\) therefore the flange is compact and the flange local buckling limit state does not apply.

Case 12

Web local buckling limit state

\(\frac{h}{t}=22.8<2.42 \sqrt{\frac{E}{F_{y}}}=60.8\) therefore the the web is compact and the web local buckling limit state does not apply.

Table B4.1
Case 13

Therefore \(M_{n}\) = 392 kip-in., controlled by the flexural yielding limit state.

Calculate the available flexural strength

LRFD ASD
\(\begin{aligned}\phi_{b} &=0.90 \\M_{c} &=\phi_{b} M_{n}=0.90(392 \text { kip-in. })=353 \text { kip-in. }\end{aligned}\) \(\begin{array}{l}\Omega_{b}=1.67 \\M_{c}=M_{n} / \Omega_{b}=392 \text { kip-in. } / 1.67=235 \text {kip-in. }\end{array}\)

Section F1

Take the available torsional strength from

LRFD ASD
\(T_{c}=\phi_{T} T_{n}=0.90(278 \text { kip-in. })=250 \text { kip-in. }\) \(T_{c}=T_{n} / \Omega_{T}=278 \text { kip-in. } / 1.67=166 \text { kip-in. }\)

Check combined strength

LRFD ASD
\(\left(\frac{P_{r}}{P_{c}}+\frac{M_{r}}{M_{c}}\right)+\left(\frac{V_{r}}{V_{c}}+\frac{T_{r}}{T_{c}}\right)^{2} \leq 1.0\)

 

\(\left(0+\frac{265 kip – in .}{353 kip – in .}\right) +\left(\frac{11.0 kips }{61.4 kips }+\frac{66.2 kip – in .}{250 kip – in .}\right)^{2}\)

=0.948 < 1.0 o.k.

 \(\left(\frac{P_{r}}{P_{c}}+\frac{M_{r}}{M_{c}}\right)+\left(\frac{V_{r}}{V_{c}}+\frac{T_{r}}{T_{c}}\right)^{2} \leq 1.0\)

Eqn. H3-6

\(\left(0+\frac{177 kip – in .}{235 kip – in .}\right) +\left(\frac{7.36 \text { kips }}{40.8 \text { kips }}+\frac{44.2 \text { kip-in. }}{166 \text { kip-in. }}\right)^{2}\)
