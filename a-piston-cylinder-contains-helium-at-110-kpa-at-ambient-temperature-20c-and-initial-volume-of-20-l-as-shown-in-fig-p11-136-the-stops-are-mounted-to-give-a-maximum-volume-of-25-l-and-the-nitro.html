Guest Question.

A piston/cylinder contains helium at 110 kPa at ambient temperature 20°C, and initial volume of 20 L as shown in Fig. P11.136. The stops are mounted to give a maximum volume of 25 L and the nitrogen line conditions are 300 kPa, 30°C. The valve is now opened which allows nitrogen to flow in and mix with the helium. The valve is closed when the pressure inside reaches 200 kPa, at which point the temperature inside is 40°C. Is this process consistent with the second law of thermodynamics?

Admin
Solution
\(\begin{aligned}& P _{1}=110 kPa , T _{1}=20{ }^{\circ} C , V _{1}=20 L , V _{\max }=25 L = V _{2} \\& P _{2}=200 kPa , T _{2}=40{ }^{\circ} C , P _{ i }=300 kPa , T _{ i }=30{ }^{\circ} C\end{aligned}\)

Constant P to stops, then constant \(V = V _{\max } \Rightarrow W _{ cv }= P _{1}\left( V _{2}- V _{1}\right)\)

\(\begin{aligned}Q _{ cv } &= U _{2}- U _{1}+ W _{ cv }- n _{ i } \overline{ h }_{ i }; \\&= n _{2} \overline{ h }_{2}- n _{1} \overline{ h }_{1}- n _{ i } \overline{ h }_{ i }-\left( P _{2}- P _{1}\right) V _{2} \\&= n _{ A }\left(\overline{ h }_{ A 2}-\overline{ h }_{ Ai }\right)+ n _{ B }\left(\overline{ h }_{ B 2}-\overline{ h }_{ B 1}\right)-\left( P _{2}- P _{1}\right) V _{2}\end{aligned}\)

 

\(\begin{aligned}& n _{ B }= n _{1}= P _{1} V _{1} / \overline{ R } T _{1}=110 \times 0.02 / 8.3145 \times 293.2=0.0009 kmol \\& n _{2}= n _{ A }+ n _{ B }= P _{2} V _{2} / \overline{ R } T _{2}=200 \times 0.025 / 8.3145 \times 313.2=0.00192 kmol \\& n _{ A }= n _{2}- n _{ B }=0.00102 kmol\end{aligned}\)

 

Mole fractions: \(y _{ A 2}=0.00102 / 0.00192=0.5313, \quad y _{ B 2}=0.4687\)

\(\begin{aligned}Q _{ cv }=& 0.00102 \times 28.013 \times 1.042(40-30)+0.0009 \times 4.003 \times 5.193(40-20) \\&-(200-110) 0.025 \\=& 0.298+0.374-2.25=-1.578 kJ\end{aligned}\)

 

\(\begin{aligned}S _{\text {gen }} &= n _{2} \overline{ s }_{2}- n _{1} \overline{ s }_{1}- n _{ i } \overline{ s }_{ i }- Q _{ cv } / T _{0} \\&= n _{ A }\left(\overline{ s }_{ A 2}-\overline{ s }_{ Ai }\right)+ n _{ B }\left(\overline{ s }_{ B 2}-\overline{ s }_{ B 1}\right)- Q _{ cv } / T _{0}\end{aligned}\)

 

\(\begin{aligned}&\overline{ s }_{ A 2}-\overline{ s }_{ Ai }=29.189 \ln \frac{313.2}{303.2}-\overline{ R } \ln \frac{0.5313^{*} 200}{300}=9.5763 \\&\overline{ s }_{ B 2}-\overline{ s }_{ B 1}=20.7876 \ln \frac{313.2}{293.2}-\overline{ R } \ln \frac{0.4687^{*} 200}{110}=2.7015\end{aligned}\)

 

\(\begin{aligned}S _{ gen } &=0.00102 \times 9.5763+0.0009 \times 2.7015+1.578 / 293.2 \\&= 0 . 0 1 7 6 ~ k J / K > 0 \quad \text { Satisfies } 2 \text { nd law. }\end{aligned}\)
