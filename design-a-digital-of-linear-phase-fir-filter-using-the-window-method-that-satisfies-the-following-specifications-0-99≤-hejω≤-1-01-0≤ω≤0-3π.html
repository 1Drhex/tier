Guest Question.

Design a digital, of linear phase, FIR filter using the window method that satisfies the following specifications:

\(0.99\leq |H(e^{j\omega })|\leq 1.01,\) 0≤|ω|≤0.3π

\(|H(e^{j\omega })|\leq 0.01,\) 0.35≤|ω|≤π

Admin
Solution

Since the pass band ripple and the stop band ripple are the same, we can focus only on the requirement of the stop band ripple. A stop band ripple \(δ_{2} = 0.01\) corresponds to a stop band attenuation of -40dB. Therefore, based on Table 6.2, we can use the Hanning window for which an attenuation of approximately -44dB is obtained. The transition band, according to the specifications, is Δω = 0.05π ⇒ Δf = 0.025.

The required filter order is \(N=\frac{3.1}{f}= 12.4,\) and it is \(w[n]=0.5-0.5\cos (\frac{2\pi n}{124} )\) for \(0\leq n\leq 124.\)

For an ideal low-pass filter with a cut-off frequency defined by the point corresponding to the middle of the transition band, i.e., \(ω_{c} =0.325π,\) and such a delay that \(h_{i}[n]\) is symmetrically placed in the interval 0 ≤ n ≤ 124, that is N/2 = 62, it is:

\(h_{i}[n]=\frac{\sin (0.325\pi (n-62))}{\pi (n-62)}\)

Thus, the filter corresponds to

\(h[n]=[0.5-0.5\cos \frac{2\pi n}{124}].\frac{\sin (0.325\pi (n-62))}{\pi (n-62)}, 0\leq n\leq 124 \)

If, instead of the Hanning window, we used the Hamming or Blackman window, we would exceed the threshold of the ripple requirements in the pass band and the stop band, as a larger filter order would be required. For example, using the Blackman window, the order of the filter, which satisfies the transition band specification would be \(N=\frac{5.5}{0.025}=220.\)
