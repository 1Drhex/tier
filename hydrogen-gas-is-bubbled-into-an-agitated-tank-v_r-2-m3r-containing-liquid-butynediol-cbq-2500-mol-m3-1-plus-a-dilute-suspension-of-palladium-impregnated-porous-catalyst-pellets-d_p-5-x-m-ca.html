Guest Question.

Hydrogen gas is bubbled into an agitated \(\operatorname{tank}\left(V_{r}=2 \mathrm{~m}^{3} r\right)\) containing liquid butynediol \(\left(C_{\mathrm{B} 0}=2500 \mathrm{~mol} / \mathrm{m}^{3} l\right)\) plus a dilute suspension of palladium-impregnated porous catalyst pellets \(\left(d_{p}=5 \times 10^{-5} \mathrm{~m}\right.\) cat, \(\rho_{s}=1450 \mathrm{~kg} / \mathrm{m}^{3}\) cat, \(\mathscr{D}_{e}=\) \(5 \times 10^{-10} \mathrm{~m}^{3} \mathrm{l} / \mathrm{m}\) cat \(\cdot \mathrm{s}, f_{s}=0.0055\) ). Hydrogen dissolves in the liquid \(\left(H_{\mathrm{A}}=\right.\) \(148000 \mathrm{~Pa} \cdot \mathrm{m}^{3} l / \mathrm{mol}\) ) and reacts with the butynediol on the catalyst surface as follows (see Fig. E22.2a):

\(\ \mathrm{H}_{2}(g \rightarrow l)+\text { butynediol }(l) \frac{\text { on }}{\text { catalyst }} \text { butynediol }\)

and at \(35^{\circ} \mathrm{C}\)

\(-r_{\mathrm{A}}^{\prime}=k^{\prime} C_{\mathrm{A}} C_{\mathrm{B}} \quad \text { and } \quad k^{\prime}=5 \times 10^{-5} \mathrm{~m}^{6} l / \mathrm{kg} \cdot \mathrm{mol} \text { cat } \cdot \mathrm{s}\)

Unused hydrogen is recompressed and recirculated, and the whole operation takes place at 1.46 atm and \(35^{\circ} \mathrm{C}\). Find out how long it will take for \(90 \%\) conversion of reactant.

Additional Data

The mass transfer rates are given as

\(\left(k_{\mathrm{A} i} \mathrm{a}_{i) g+l}=0.277 \frac{\mathrm{m}^{3} l}{\mathrm{~m}^{3} r \cdot \mathrm{s}} \quad k_{\mathrm{Ac}}=4.4 \times 10^{-4} \frac{\mathrm{m}^{3} l}{\mathrm{~m}^{3} r \cdot \mathrm{s}}\right.\)
Admin
Solution

First compare \(C_{\mathrm{A}}\) and \(C_{\mathrm{B}}\) :

\(\left.\begin{array}{l}C_{\mathrm{A}}==\frac{p_{\mathrm{A}}}{H_{\mathrm{A}}}=\frac{1.46(101325)}{148000}=10 \mathrm{~mol} / \mathrm{m}^{3} l \\C_{\mathrm{B} 0}=2500 \text { and } C_{\mathrm{Bf}}=250 \mathrm{~mol} / \mathrm{m}^{3} l\end{array}\right\} \begin{array}{l}\text { Both at the beginning and } \\\text { at the end of the batch run } \\C_{\mathrm{B}}>C_{\mathrm{A}} ; \text { therefore, the } \\\text { system is in extreme } 1\end{array}\)

Although \(C_{\mathrm{A}}\) stays constant throughout the batch run, \(C_{\mathrm{B}}\) does not, so we will have a changing rate with time and with \(C_{\mathrm{B}}\). Therefore, let us evaluate the rate at any particular value of \(C_{\mathrm{B}}\).

\(a_{c}=\frac{6 f_{s}}{d_{p}}=\frac{6(0.0055)}{5 \times 10^{-5}}=660 \mathrm{~m}^{2} \mathrm{cat} / \mathrm{m}^{3} l\)

therefore,

\(k_{\mathrm{A} c} a_{c}=4.4 \times 10^{-4}(660)=0.29 \frac{\mathrm{m}^{3} l}{\mathrm{~m}^{3} r \cdot \mathrm{s}}\)

and

\(M_{T}=L \sqrt{\frac{k^{\prime} C_{\mathrm{B}} \rho_{s}}{\mathscr{D}_{e}}}=\frac{5 \times 10^{-5}}{6} \sqrt{\frac{\left(5 \times 10^{-5}\right) C_{\mathrm{B}}(1450)}{5 \times 10^{-10}}}=0.1 C_{\mathrm{B}}^{1 / 2}\) (i)

Replacing in the rate expression of Eq. 18 gives

\({{-r}''}''_{A}=\frac{1}{\frac{1}{H_{A}k_{Ag}a_{i}}+\frac{1}{k_{Al}a_{i}}+\frac{1}{k_{Ac}a_{c}}+\frac{1}{({{k}''}''_{A}\overline{C}^{m}_{B}\overline{C}^{n-1}_{A})\varepsilon _{A}.f_{s}}}\) (Eq 18)

\(\begin{aligned}-r_{\mathrm{A}}^{\prime \prime \prime \prime} &=\frac{1}{\frac{1}{0.277}+\frac{1}{0.29}+\frac{1}{\left(5 \times 10^{-5}\right) C_{\mathrm{B}}(1450)\left(\mathscr{E}_{\mathrm{A}}\right)(0.0055)}}.\frac{14.6(101325)}{148000} \\&\) (ii)

\(=\frac{1}{0.70584+\frac{250.8}{C_{\mathrm{B}}\left(\mathscr{E}_{\mathrm{A}} \text { at } M_{T}=0.1 C_{\mathrm{B}}^{1 / 2}\right)}}\end{aligned}\)

Now from Eq. 12 the reaction time is given by

\(t=\frac{V_{l}}{b V_{r}} \int_{C_{\mathrm{Bf}}}^{c_{\mathrm{BO}}} \frac{d C_{\mathrm{B}}}{\left(-r_{\mathrm{A}}^{\prime \prime \prime \prime}\right)}\)

\(\underline{t=3460 \mathrm{~s}}, \text { or } \underline{\underline{58 \mathrm{~min}}}\)
