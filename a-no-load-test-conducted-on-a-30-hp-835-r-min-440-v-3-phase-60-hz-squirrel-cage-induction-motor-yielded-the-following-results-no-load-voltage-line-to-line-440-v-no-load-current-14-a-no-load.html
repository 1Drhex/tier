Guest Question.

A no-load test conducted on a 30 hp, 835 r/min, 440 V, 3-phase, 60 Hz, squirrel-cage induction motor yielded the following results:
No-load voltage (line-to-line): 440 V
No-load current: 14 A
No-load power: 1470 W
Resistance measured between two terminals: 0.5 立
The locked-rotor test, conducted at reduced voltage, gave the following results:
Locked-rotor voltage (line-to-line): 163 V
Locked-rotor power: 7200 W
Locked-rotor current: 60 A
Determine the equivalent circuit of the motor.

Admin
Solution

Assuming the stator windings are connected in wye, the resistance per phase is

 

\(r_{1}=0.5 \Omega / 2=0.25 \Omega\)

 

From the no-load test we find

 

\(\begin{aligned}S_{ NL } &=E_{ NL } I_{ NL } \sqrt{3}=440 \times 14 \sqrt{3} \\&=10669 VA \\P_{ NL } &=1470 W \\Q_{ NL } &=\sqrt{S_{N L}^{2}-P_{ NL }^{2}}=\sqrt{10669^{2}-1470^{2}} \\&=10568 var \\X_{ m } &=E_{ NL }^{2} / Q_{ NL }=440^{2} / 10568 \\&=18.3 \Omega \\R_{ m } &=E_{ NL }^{2} / P_{ NL }=440^{2} /\left(1470-3 \times 14^{2} \times 0.25\right) \\&=146 \Omega\end{aligned}\)

 

From the locked-rotor test we find

 

\(\begin{aligned}S_{ LR } &=E_{ LR } I_{ LR } \sqrt{3}=163 \times 60 \sqrt{3} \\&=16939 VA \\P_{ LR } &=7200 W \\Q_{ LR } &=\sqrt{S_{ LR }^{2}-P_{ LR }^{2}}=\sqrt{16939^{2}-7200^{2}} \\&=15333 \text { var } \\I_{ LR } &=60 A\end{aligned}\)

 

Total leakage reactance referred to stator is

 

\(x=\frac{Q_{ LR }}{3 I_{ LR }^{2}}=\frac{15333}{3 \times 60^{2}}=1.42 \Omega\)

 

Total resistance referred to stator is

 

\(\begin{aligned}r_{1}+r_{2} &=P_{ LR } / 3 I_{ LR }^{2}=7200 /\left(3 \times 60^{2}\right) \\&=0.67 \Omega \\r_{1} &=0.25 \Omega \\r_{2} &=0.67-0.25=0.42 \Omega\end{aligned}\)

 

(In a squirrel-cage motor, \(R_{2}=r_{2} \text { because } R_{ x }=0\); see Eq. 1.)

 

\(R_{2}=r_{2}+R_{ x }\) (Equation 1)
