Guest Question.

The streamwise velocity component for a laminar boundary layer is sometimes assumed to be roughly approximated by the linear relation \(u=\frac{y}{\delta }u_{e}\) where\( \delta =1.25*10^{-2}\sqrt{x} \) Assume that we are trying to approximate the flow of air at standard sea-level conditions past a flat plate where \( u_{e} = 2.337 m/s\). Calculate the streamwise distribution of the displacement thickness\(\left ( \delta ^{*} \right )\) , the velocity at the edge of the boundary layer\(\left ( \nu _{e} \right )\) , and the skin-friction coefficient\( \left ( C_{f} \right )\). Compare the values obtained assuming a linear velocity profile with the more exact solutions presented in this chapter

Admin
Solution

– As given in Table 1.2 , the standard day atmospheric conditions at sea level are:

\(\rho _{\infty }=1.2250kg/m^{3} and \mu _{\infty }=1.7894*10^{-5}kg/s\cdot m\)

So, for constant-property flow past a flat plate,

\(Re_{x}=\frac{\rho _{\infty }u_{e}x}{\mu _{\infty }}=1.60*10^{5}x\)

Using the definition for the displacement thickness of an incompressible boundary layer, equation (4.26),

equation 4.26:

\(\delta ^{*}=\int_{0}^{\delta }\left ( 1-\frac{u}{u_{e}} \right )dy \) \(\delta ^{*}=\int_{0}^{\delta }\left ( 1-\frac{u}{u_{e}} \right )dy=\delta \int_{0}^{1}\left ( 1-\frac{u}{u_{e}} \right )d\left ( \frac{y}{\delta } \right )\)

Notice that, since we have \(u/u_{e}\) in terms of \(y/\delta \) we have changed our independent variable from y to \(y/\delta\) We must also change the upper limit on our integral from \(\delta\) to 1. Therefore, since:

\(\frac{u}{u_{e}}=\frac{y}{\delta }and \delta =1.25*10^{-2}\sqrt{x}\)

Then

\(\delta ^{*}=1.25*10^{-2}\sqrt{x}\int_{0}^{1}\left ( 1-\frac{y}{\delta } \right )d\left ( \frac{y}{\delta } \right )=0.625*10^{-2}\sqrt{x}\)

for the linear profile.

Using the equation for the more exact formulation [equation (4.27)], and noting that.

equation 4.27:

\(\frac{\delta ^{*}}{x}=\frac{\sqrt{2}\left ( \eta _{e}-f_{e} \right )}{\sqrt{Re_{x}}}=\frac{1.72}{\sqrt{Re_{x}}}\)

\(Re_{x}=1.60*10^{5}x\), we find that

\(\delta ^{*}=0.430*10^{-2}\sqrt{x}\)

Using the continuity equation, we would find that the linear approximation gives a value for \(\nu _{e}\)

\(\nu _{e}=\frac{3.125*10^{-3}}{\sqrt{x}}u_{e}\)

Using the more exact formulation of equation (4.25) yields:

equation 4.25:

\(\frac{\nu _{e}}{u_{e}}=\frac{0.84}{\sqrt{Re_{x}}}\)

 

\(\nu _{e}-\frac{0.84}{\sqrt{Re_{x}}}u_{e}-\frac{2.10*10^{-3}}{\sqrt{x}}u_{e}\)

Finally, we find that the skin friction for the linear velocity approximation is given by:

\(\tau =\mu \left ( \frac{\partial u}{\partial y} \right )_{y=0}=\frac{\mu u_{e}}{\delta }\)

Therefore, the skin-friction coefficient is:

\(C_{f}=\frac{\pi }{\frac{1}{2}\rho _{\infty }u_{e}^{2}}=\frac{2\mu _{\infty }}{\rho _{\infty }u_{e}\delta }=\frac{2}{1.60*10^{5}\left ( 1.25*10^{-2}\sqrt{x} \right )}=\frac{1.00*10^{-3}}{\sqrt{x}}\)

For the more exact formulation,

\(C_{f}=\frac{0.664}{\sqrt{Re_{x}}}=\frac{1.66*10^{-3}}{\sqrt{x}}\)

Summarizing these calculations provides the following comparison:

More exact solution linear approximation
\(0.430*10^{-2}\sqrt{x}\) \(0.625*10^{-2}\sqrt{x}\) \(\delta ^{*}\)
\(\left ( 2.10*10^{-3}u_{e} \right )/\sqrt{x}\) \(\left ( 3.125*10^{-3}u_{e} \right )/\sqrt{x}\) \(\nu _{e}\)
\(\left ( 1.66*10^{-3} \right )/\sqrt{x}\) \(\left ( 1.00*10^{-3} \right )/\sqrt{x}\) \(C_{f}\)

Comparing the velocity profiles, which are presented in Fig. 4.7 , the reader should be able to use physical reasoning to determine that these relationships are intuitively correct. That is, if you use a linear profile the shear would be less than that for the exact solution, where \( \delta ^{*} and\nu _{e}\) would be greater for the linear profile.

In this example, we assumed that the boundary-layer thickness d was\( 1.25*10^{-2}\sqrt{x}\) which is the value obtained using the more exact formulation [i.e., equation (4.23)]. However, if we had used the integral approach to determine the value of d for a linear profile, we would have obtained:

equation 4.23:

\( \frac{\delta }{x}=\frac{5.0}{\sqrt{Re_{x}}}\) \(\delta =\frac{3.464}{\sqrt{Re_{x}}}=0.866*10^{-2}\sqrt{x}\)

Although this is considerably less than the assumed (more correct) value, the values of the other parameters (e.g., \(\delta ^{*} andc_{f}\) ) would be in closer agreement with those given by the more exact solution.
