Guest Question.

Forced Response to an Exponential Input. Find the forced response for the inductor current \(i_f\) for the parallel RLC circuit shown in Figure 9.7-1 when \(i_s = 8e^{-2t} A\). Let R = 6 立, L = 7 H, and C = 1/42 F.

Admin
Solution

The source current is applied at t = 0 as indicated by the unit step function u(t). After t = 0, the KCL equation at the upper node is

\(i + \frac{v}{R} + C \frac{dv}{dt} = i_s\) (9.7-3)

We note that
\(v = L \frac{di}{dt}\) (9.7-4)

so
\(\frac{dv}{dt} = L \frac{d^2i}{dt^2}\) (9.7-5)

Substituting Eqs. 9.7-4 and 9.7-5 into Eq. 9.7-3, we have

\(i + \frac{L}{R} \frac{di}{dt} + CL \frac{di^2}{dt^2} = i_s\)

Then we divide by LC and rearrange to obtain the familiar second-order differential equation

\(\frac{d^2i}{dt^2} + \frac{1}{RC} \frac{di}{dt} + \frac{1}{LC} i = \frac{i_s}{LC}\) (9.7-6)

Substituting the component values and the source \(i_s\), we obtain

\(\frac{d^2i}{dt^2} + 7 \frac{di}{dt} + 6 i = 48e^{-2t}\) (9.7-7)

We wish to obtain the forced response, so we assume that the response will be

\(i_f = Be^{-2t}\) (9.7-8)

where B is to be determined. Substituting the assumed solution, Eq. 9.7-8, into the differential equation, we have

\(4Be^{-2t} + 7 (-2Be^{-2t}) + 6 Be^{-2t} = 48e^{-2t}\)

or \((4 – 14 + 6)Be^{-2t} = 48e^{-2t}\)

Therefore, B = – 12 and

\(i_f = -12e^{-2t} A\)
