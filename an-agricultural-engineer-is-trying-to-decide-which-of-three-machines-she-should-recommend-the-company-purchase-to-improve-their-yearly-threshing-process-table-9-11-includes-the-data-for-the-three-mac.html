Guest Question.

An agricultural engineer is trying to decide which of three machines she should recommend the company purchase to improve their yearly threshing process. Table 9.11 includes the data for the three machines under consideration with the threshing machines listed in order of the least initial cost to the highest initial cost. For this example, the incremental rate of return for each incremental increase in initial cost using net present worth analysis is calculated including the do nothing alternative. The company uses a MARR of 7% for large equipment. Figures 9.8 through 9.10 are the net cash flow diagrams for the three threshing machines.

TABLE 9.11 Threshing Machine Data

Cost or Disbursements Threshing Machine 1 Threshing Machine 2 Threshing Machine 3
Initial cost $140,000.00 $160,000.00 $200,000.00
Yearly revenue $70,000.00 $87,500.00 $130,000.00
Yearly expenses $20,000.00 $30,000.00 $50,000.00
Life in years 4 4 4
Admin
Solution

First, calculate the rate of return of the net cash flow between machine 1 and the do nothing alternative using net present worth analysis:

\(0 =-P_0 + (A_1 +A_2)(P/F,i,n)\)

=- $ 140,000.00 + ($70,000.00 – $ 20,000.00)(P/A,i,4)

Try 20%

= -$140,000.00 + $50,000.00(2.5887)

= – $ 140,000.00 + $ 129,435.00

= -$10,565.00

Try 15%

= -$140,000.00 + $50,000.00(2.8549)

= -$140,000.00 +$142,745.00

=$ 2,745.00

Use interpolation and Table 9.12 to calculate the rate of return:

\(IROR = c +\left\lgroup\frac{a}{b} \right\rgroup d\)

 

\(IROR = 15 +\left\lgroup\frac{2,745.00 – 0}{2,745.00- (-10,565.00)} \right\rgroup \times (20 – 15)\)

 

\(= 15 +\left\lgroup\frac{2,745.00}{13,310.00} \right\rgroup \times 5\)

= 15 +(0.206236×5)

= 15 +1.03118

= 16.03 %

Therefore, since the ROR > MARR, 16.03% > 7%, select machine 1 and compare it to machine 2.

Second, calculate the incremental rate of return of machine 2 minus machine 1 using net present worth analysis:

\(0 =P_0 +(A_1 +A_2)(P/A,i.n)\)

= – $20,000.00 +($17,500.00 – $ 10,000.00)(P/A,i,n)

Try 15%

= – $20,000.00 + $7,500.00(2.8549)

= – $20,000.00 + $21,411.75

= $ 1,411.75

Try 20%

= – $20,000.00 + $7,500.00(2 5887)

= – $20,000.00 + $19,414.25

– $ 585.75

Use interpolation and Table 9.13 to solve for the incremental rate of return:

\(IROR = c +\left\lgroup\frac{a}{b} \right\rgroup d\)

 

\(IROR = 15+\left\lgroup\frac{1,411.75 – 0}{1,411.75 – (-585.75)} \right\rgroup \times (20 – 15)\)

 

\(= 15 +\left\lgroup\frac{1,411.75}{1,997.50} \right\rgroup \times 5\)

=15 +(0.706758 ×5)

= 15 + 3.533792

= 18.53 %

Therefore, since the IROR > MARR, 18.52% > 7%, select machine 2 and compare it to machine 3.

Third, calculate the incremental rate of return for machine 3 minus machine 2 using net present worth analysis:

\(0=P_0 +(A_1 + A_2)(P/A,i,n)\)

= -$40,000.00 +($42,500.00 – $20,000.00)(P/A,i,4)

Try 40%

= -$40,000.00 +$22,500.00(1.8492)

= -$40,000.00 +$ 41,607.00

= $ 1,607.00

Try 50%

= -$ 40,000.00 + $ 22,500.00(1.6049)

= -$ 40,000.00 +$ 36,110.25

= – $3,89.75

Use interpolation and Table 9.14 to calculate the incremental rate of return:

\(IROR = c +\left\lgroup\frac{a}{b} \right\rgroup d\)

 

\(IROR = 40+\left\lgroup\frac{1,607.00 – 0}{1,607.00 – (-3,889.75)} \right\rgroup \times (50 – 40)\)

 

\(= 40 +\left\lgroup\frac{1607.00}{5496.75} \right\rgroup \times 10\)

= 40 +(0.292355)

= 40 + 2.923546

= 42.92 %

Therefore, since the IROR > MARR, 42.92% > 7%, select machine 3.
