Guest Question.

Saturated vapor R-410A at 80 F is throttled to 30 \(lbf / in .^{2}\) in a steady flow process. Find the exit temperature, neglecting kinetic energy, using the generalized charts, Fig. D.2 and repeat using Table F.9.

Admin
Solution

R-410A throttling process, R = 1.98589/72.585 = 0.02736 Btu/lbm-R

\(T _{1}=80 F , x _{1}=1.00, P _{2}=30 lbf / in .^{2}\),

Energy Eq.: \(h _{2}- h _{1}=\left( h _{2}- h _{2}^{*}\right)+\left( h _{2}^{*}- h _{1}^{*}\right)+\left( h _{1}^{*}- h _{1}\right)=0\)

Generalized charts, \(T _{ r1 }=\frac{539.7}{620.1}=0.87\)

From D.2: \(\left( h _{1}^{*}- h _{1}\right)=0.02736 \times 620.1 \times 0.75=12.72 Btu / lbm\)

To get \(C _{ P 0}\), use h values from Table F.9 at low pressure (5 psia).

\(C _{ P 0} \approx \frac{137.6-133.75}{80-60}=0.1925 Btu / lbm R\)

Substituting into energy Eq.: \(\left( h _{2}- h _{2}^{*}\right)+0.1925\left( T _{2}-30\right)+12.72=0\)

at \(P_{ r 2}=\frac{30}{711}=0.042\)

Assume \(T _{2}=10 F =469.7 R => T _{ r 2}=\frac{469.7}{620.1}=0.757\)

\(\left( h _{2}^{*}- h _{2}\right)=0.02736 \times 620.1 \times 0.07=1.19\)

Substituting,

\(-1.19+0.1925(10-80)+12.72=-1.945\)

Guess \(T _{2}=20 F =479.7 R \Rightarrow T _{ r 2}=\frac{479.7}{620.1}=0.774\)

\(\left( h _{2}^{*}- h _{2}\right)=0.02736 \times 620.1 \times 0.07=1.19\)

Substituting,

\(\begin{aligned}-1.19+0.1925(20-80)+12.72=-0.02 \text { nearly } OK \\\left.\Rightarrow T _{2}= 2 0 F \text { (maybe re-estimate } C _{ P 0} \text { at avg } T \right)\end{aligned}\)

R-410A tables, F.9: \(T _{1}=80 F , x _{1}=1.0 \Rightarrow h _{1}=122.14 Btu / lbm \)

\(h _{2}= h _{1}=122.14 Btu / lbm , \quad P _{2}=30 lbf / in .^{2} \Rightarrow T _{2}= 8 . 1 5 F\)
