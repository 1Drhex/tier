Guest Question.

Calculate the primary consolidation settlement of the 3-m-thick clay layer (Figure 7.25) that will result from the load carried by a 1.5-m square footing. The clay is normally consolidated. Use 2 : 1 method for calculation of Δσ′

Admin
Solution

For normally consolidated clay, from Eq. (7.14) we have

 

\(S_{p}=\frac{C_{c} H}{1 + e_{0}}\log\left(\frac{\sigma_{o}^{\prime} + \Delta \sigma^{\prime}}{\sigma_{o}^{\prime}}\right) \)

 

where

 

\(C_{c} = 0.009(LL – 10) = 0.009(40 – 10) = 0.27\\H = 3000\,mm\\e_{0} = 1.0\\ \sigma^{\prime}_{0} = 4.5 \times \gamma_{dry(sand)} + 1.5[\gamma_{sat(sand)} – 9.81] + \frac{3}{2}[\gamma_{sat(clay)} – 9.81]\\ =4.5 \times 15.7 + 1.5(18.9 – 9.81) + 1.5(17.3 – 9.81) = 95.52\,kN/m^{2}\)

 

In order to calculate \(\Delta\sigma^{\prime}\), we can prepare the following table:

 

\(\textbf{z}\\\textbf{(cm)}\) \(\textbf{B + z}\\\textbf{(m)}\) \(\textbf{q}^{\textbf{a}}\\\textbf{(kN/}\textbf{m}^{\textbf{2}}\textbf{)}\) \(\Delta\sigma^{\prime}\\\textbf{[Eq. (7.47)]}\)
4.5 6.0 395.6 24.72 = \(\Delta\sigma^{\prime}_{t}\)
6.0 7.5 395.6 15.82 = \(\Delta\sigma^{\prime}_{m}\)
7.5 9.0 395.6 10.99 = \(\Delta\sigma^{\prime}_{b}\)

 

\(^{a}q = \frac{890}{1.5 \times 1.5} = 395.6\,kN/m^{2}\)

 

From Eq. (7.46)

 

\(\Delta\sigma^{\prime}_{av} = \frac{\Delta\sigma_{t} + 4\Delta\sigma_{m} + \Delta\sigma_{b}}{6} = \frac{24.76 + (4)(15.82) + 10.99}{6} \approx 16.5\,kN/m^{2}\\S_{p} = \frac{(0.27)(3000)}{1 + 1}\log \left(\frac{95.52 + 16.5}{95.52}\right) = \textbf{28.0 mm} \)

 

Note: If we use Table 6.8 and Eq. (6.45) \( \Delta\sigma_{c} = qI_{c}\) to estimate \(\Delta\sigma^{\prime}_{av}\), the value of \(S_{p}\) will be 21.3 mm.
