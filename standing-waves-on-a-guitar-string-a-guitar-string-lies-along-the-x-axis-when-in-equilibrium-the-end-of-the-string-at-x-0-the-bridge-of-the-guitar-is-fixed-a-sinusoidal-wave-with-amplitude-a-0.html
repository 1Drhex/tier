STANDING WAVES ON A GUITAR STRING

A guitar string lies along the x-axis when in equilibrium. The end of the string at x = 0 (the bridge of the guitar) is fixed. A sinusoidal wave with amplitude A = 0.750 mm = 7.50 × \(10^{-4}\) m and frequency f = 440 Hz, corresponding to the red curves in Fig. 15.24, travels along the string in the -x-direction at 143 m/s. It is reflected from the fixed end, and the superposition of the incident and reflected waves forms a standing wave. (a) Find the equation giving the displacement of a point on the string as a function of position and time. (b) Locate the nodes. (c) Find the amplitude of the standing wave and the maximum transverse velocity and acceleration.

Admin
Solution

IDENTIFY and SET UP:

This is a kinematics problem (see Problem-Solving Strategy 15.1 in Section 15.3). The target variables are: in part (a), the wave function of the standing wave; in part (b), the locations of the nodes; and in part (c), the maximum displacement y, transverse velocity \(v_y\), and transverse acceleration \(a_y\). Since there is a fixed end at x = 0, we can use Eqs. (15.28) and (15.29) to describe this standing wave. We will need the relationships \(\omega=2 \pi f, v=\omega / k \text {, and } v=\lambda f\).

\(y(x, t)=\left(A_{\mathrm{SW}} \sin k x\right) \sin \omega t\) (15.28)

\(\begin{aligned}x &=0, \frac{\pi}{k}, \frac{2 \pi}{k}, \frac{3 \pi}{k}, \ldots \\&=0,\frac{\lambda}{2}, \frac{2 \lambda}{2}, \frac{3 \lambda}{2}, \ldots\end{aligned}\) (15.29)

EXECUTE:

(a) The standing-wave amplitude is \(A_{SW}\) = 2A =1.50 × \(10^{-3}\) m (twice the amplitude of either the incident or reflected wave). The angular frequency and wave number are

\(\begin{aligned}&\omega=2 \pi f=(2 \pi \mathrm{rad})\left(440\mathrm{~s}^{-1}\right)=2760 \mathrm{rad} / \mathrm{s} \\&k=\frac{\omega}{v}=\frac{2760 \mathrm{rad} / \mathrm{s}}{143 \mathrm{~m} / \mathrm{s}}=19.3 \mathrm{rad} / \mathrm{m}\end{aligned}\)

Equation (15.28) then gives

\(\begin{aligned}y(x, t) &=\left(A_{\mathrm{SW}} \sin k x\right) \sin \omega t \\&=\left[\left(1.50 \times 10^{-3} \mathrm{~m}\right) \sin (19.3 \mathrm{rad} / \mathrm{m})x\right] \sin (2760 \mathrm{rad} / \mathrm{s}) t\end{aligned}\)

(b) From Eq. (15.29), the positions of the nodes are x = 0, λ/2, λ, 3λ/2, c. The wavelength is λ = v/f = (143 m/s)/(440 Hz) = 0.325 m, so the nodes are at x = 0, 0.163 m, 0.325 m, 0.488 m, c.

(c) From the expression for y(x, t) in part (a), the maximum displacement from equilibrium is \(A_{SW}\) = 1.50 × \(10^{-3}\) m = 1.50 mm. This occurs at the antinodes, which are midway between adjacent nodes (that is, at x = 0.081 m, 0.244 m, 0.406 m, c).

For a particle on the string at any point x, the transverse (y-) velocity is

\(\begin{aligned}v_{y}(x, t)=& \frac{\partial y(x, t)}{\partial t} \\=&\left[\left(1.50\times 10^{-3} \mathrm{~m}\right) \sin (19.3 \mathrm{rad} / \mathrm{m})x\right] \\& \times[(2760 \mathrm{rad} / \mathrm{s}) \cos (2760 \mathrm{rad} /\mathrm{s}) t] \\=&[(4.15 \mathrm{~m} / \mathrm{s}) \sin (19.3 \mathrm{rad} / \mathrm{m})x] \cos (2760 \mathrm{rad} / \mathrm{s}) t\end{aligned}\)

At an antinode, sin (19.3 rad/m)x = ±1 and the transverse velocity varies between +4.15 m/s and -4.15 m/s. As is always the case in SHM, the maximum velocity occurs when the particle is passing through the equilibrium position (y = 0).

The transverse acceleration \(a_y\)(x, t) is the second partial derivative of y(x, t) with respect to time. You can show that

\(\begin{aligned}a_{y}(x, t)=& \frac{\partial v_{y}(x, t)}{\partial t}=\frac{\partial^{2}y(x, t)}{\partial t^{2}} \\=&\left[\left(-1.15 \times 10^{4} \mathrm{~m} / \mathrm{s}^{2}\right) \sin (19.3 \mathrm{rad} / \mathrm{m}) x\right] \\& \times \sin (2760 \mathrm{rad}/\mathrm{s}) t\end{aligned}\)

At the antinodes, the transverse acceleration varies between \(+1.15 \times 10^{4} \mathrm{~m} / \mathrm{s}^{2} \text { and }-1.15 \times 10^{4} \mathrm{~m} / \mathrm{s}^{2}\).

 

EVALUATE: The maximum transverse velocity at an antinode is quite respectable (about 15 km/h, or 9.3 mi/h). But the maximum transverse acceleration is tremendous, 1170 times the acceleration due to gravity! Guitar strings are actually fixed at both ends; we’ll see the consequences of this in the next section.
