Bolted Beam Splice
Repeat for a bolted moment connection assuming \(M_{u}= 195\) ft .k and\(V_{u}= 50\) k. Use 7⁄8-in.-diameter A490 SC bolts in STD holes and a Class B faying surface.

Admin
Solution

From AISCM, Table 1-1,

Table 1-1 Structural steel material properties

Steel Type ASTM Designation or Grade of Structural Steel \(F_{y}\) (ksi) \(F_{y}\) (ksi)

A53 Grade B 35 60

d = 15.9 in. T = 13-5/8 in.

\(t_{f} =0.43\) in.

\(b_{f}=6.99\) in.

\(t_{w} = 0295\) in. \(S_{x} = 56.5 in.^{3}\)

From AISCM, Table 3-6,\(\phi _{b}M_{p}= 240\) ft-kips.

Table 3-6 Tabulated vertical wind pressures at overhangs*

Zone \(\frac{Tabulated Vertical Pressures, psf}{Load Case 1}\)
End-Zone\( E_{oH}\) -21.6
End-Zone\( G_{oH}\) -16.9

Flange force:

\(P_{uf}=\frac{M}{d}=\frac{195}{15.9/12} =148\) kips

Bolts in Top and Bottom Flanges:

In this case, slip is a strength limit state since slip at the joint would cause additional beam deflection (see discussion in Chapter 9).

From AISCM, Table 7-4, \(\phi _{v}R_{n}= (1.43)(16.5) = 23.6\) kips/bolt.

\(N =\frac{P_{uf} }{\phi _{v}R_{n} }=\frac{148}{23.6}= 6.24 \therefore\) Use 8 bolts.

Check the reduced flexural strength of the W16 × 36:

\(F_{y}/F_{u}= 50/65 = 0.77\lt 0.80 \rightarrow Y_{t}=1.0\)

 

\(A_{fg}= (6.99)(0.43) = 3.00 in.^{3}\)

 

\(A_{fn}= A_{fg}- \sum{A_{holes} }\)

 

\(=3.00 – \left[(2)\left(\frac{7}{8}+\frac{1}{8} \right)(0.43) \right]= 2.14 in.^{2}\)

 

\(F_{u}A_{fn}\)= (65)(2.14) = 139 kips

\(Y_{t}F_{y}A_{fg}\) = (1.0)(50)(3.00) = 150 kips

\(\phi _{b}M_{n} =\phi _{b}F_{u}S_{x}\left(\frac{A_{fn} }{A_{fg} } \right) = (0.9)(65)(56.5)\left(\frac{2.14}{3.0} \right)=2357 \)in.K

=196 ft.k>195 ft.k, OK

Plates

Assume that the top and bottom plates are still 5⁄8 in. by 9 in. Tension on the gross area was checked in ; now we will check tension on the net area (see Figure 11-23).

Tension on the net plate area:

\(A_{g}= (0.625 in.)(9 in.) = 5.62 in.^{2}\)

 

\(A_{n} = A_{g} – \sum{A_{holes} }= 5.62 -\left[(2)\left(\frac{7}{8}+\frac{1}{8} \right)(0.625) \right]= 4.38 in.^{2}\)

 

\(A_{n}\leq 0.85A_{g}= (0.85)(5.62) = 4.78 in.^{2}. Use A_{n} = 4.38 in.^{2}.\)

 

\(A_{e} = A_{n}U = (4.38 in.^{2})(1.0) = 4.38 in.^{2}\)

 

Strength based on the effective area is

\(\phi P_{n} =\phi F_{u}A_{e}= (0.75)(58)(4.38 in.^{2}) = 190 kips \gt P_{uf} = 148\) kips OK

Check bolt bearing (see Figure 11-23):

Since \(t_fF_{ub} = (0.43)(65) = 27.9\) kips/in. \(< t+pF_{up} = (0.625)(58) = 36.2\) kips/in., bearing on the beam flange will control.

\(L_{c1}= 2 in. – \left[0.5\left(\frac{7}{8}+\frac{1}{8} \right) \right]=1.50\) in.

\(L_{c2}=L_{c3}=L_{c4} = 3 in. – \left[\left(\frac{7}{8}+\frac{1}{8} \right)\right]= 2.0\) in.

For bolt 1, \( \phi R_{n}= \phi1.2 L_{c}tF_{u}\leq \phi2.4dtF_{u} \)

=(0.75)(1.2)(1.50 in.)(0.43 in.)(65 ksi)

<(0.75)(2.4)(0.875)(0.43 in.)(65 ksi)

=37.7 kips < 44.0 kips

\(\phi R_{n}= 37.7\) kips

For bolts 2, 3, and 4, \(\phi R_{n}\)= (0.75)(1.2)(2.0 in.)(0.43 in.)(65 ksi)

< (0.75)(2.4)(0.875)(0.43 in.)(65 ksi)

= 50.3 kips > 44.0 kips

\(\phi R_{n}\) =44.0 kips.

\(\phi R_{n}\)= (2)(37.1 k) +(6)(44.0 k)=338 kips> \(P_{uf}\) = 148 kips OK for bearing

Block shear:

\(A_{gv} =(2)(3 in. + 3 in. + 3 in. + 2 in.)(0.43 in.) = 9.46 in.^{2}\)

 

\(A_{gt}=(1.49 in. + 1.49 in.)(0.43 in.) = 1.28 in.^{2}\)

 

\(A_{nv}= A_{gv}- \sum{A_{holes} }= 9.46 -\left[(3.5)(2)\left(\frac{7}{8}+\frac{1}{8} \right)(0.43)\right]= 6.45in.^{2}\)

 

\(A_{nt}= A_{gt}- \sum{A_{holes} }= 1.28 -\left[\left(\frac{7}{8}+\frac{1}{8} \right)(0.43)\right]= 0.85in.^{2}\)

 

The available block shear strength is found from Chapter 4 (\( U_{bs} = 1.0\)):

\(\phi P_{n}= \phi (0.60F_{u}A_{nv}+U_{bs}F_{u}A_{nt})\leq \phi (0.60F_{y}A_{gv}+U_{bs}F_{u}A_{nt})\)

 

=0.75[(0.60)(65)(6.45) + (1.0)(65)(0.85)]

≤ 0.75[(0.60)(50)(9.46) + (1.0)(65)(0.85)]

230 kips < 254 kips

\(\phi P_{n} = 230 kips > P_{uf} = 148\) kips OK

Compression:

K = 0.65 (Figure 5-3)

L = 2 in. + 0.5 in. + 2 in. = 4.5 in.

\(r_{y}=\frac{t}{\sqrt{12} }= \frac{0.625}{\sqrt{12} }= 0.180\) in.

\(\frac{KL}{r}=\frac{(0.65)(4.5)}{0.180}= 16.2\)

 

Since KL/r < 25,\( F_{cr} = F_{y} \)(AISC, Section J4.4).

\(\phi P_{n} = \phi F_{cr}A_{g}= (0.9)(36)(0.625)(9) = 182 kips \gt P_{uf} = 148\) kips OK

Shear strength:

A 3⁄8-in. by 13-in. plate will be used (same size used for previous example). Assuming that a shear strength of 50 kips is needed, AISCM, Table 7-7 can be used to check the capacity of the bolt group (see Figure 11-24):

e = 1.5 in. + 0.5 in. + 1.5 in. = 3.5 in.

Using N = 4 and s = 3 in., C = 2.58.

Assuming (4)- 7⁄8-in.-diameter A490 bolts, \(\phi _{v}R_{n}= 27.1\) kips/bolt.

The strength of the bolt group is (2.58)(27.1) = 69.9 kips > 50 kips. OK

Check bearing on beam web:

\(L_{c}(all bolts) = 3 in. -\left(\frac{7}{8}+\frac{1}{8} \right)= 2.0\) in.

\(\phi R_{n}\) = (0.75)(1.2)(2.0 in.)(0.295 in.)(65 ksi)

<(0.75)(2.4)(0.875)(0.295 in.)(65 ksi)

=34.5 kips > 30.2 kips

= 30.2 kips

\(\phi R_{n}\) = (4)(30.2) = 120 kips > 50 kips OK for bearing
