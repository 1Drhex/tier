Guest Question.

The transfer function

\(H_{A}(s)=\frac{H_{0} s^{4}}{\prod_{j=1}^{2}\left(s-p_{j}\right)\left(s-p_{j}^{*}\right)}\)

where

\(H_{0}=0.9885531 \quad p_{1}, p_{1}^{*}=-2.047535 \pm j 1.492958 \quad\) \(p_{2}, p_{2}^{*}=-0.3972182 \pm j 1.688095\)

represents a highpass Chebyshev filter with a passband edge of 2 rad/s and a passband ripple of 0.1 dB. Obtain a corresponding discrete-time transfer function employing the matched-z transformation method. Use a sampling frequency of 10 rad/s.

Admin
Solution

The value of L in Eq. (11.15) is generally zero for highpass filters, according to Table 11.4. Hence \(H_{D}(z)\) can be readily formed as

 

\(H_{D}(z)=(z+1)^{L} \frac{H_{0} \prod_{i=1}^{M}\left(z-e^{z_{i} T}\right)}{\prod_{i=1}^{N}\left(z-e^{p_{i} T}\right)}\) (11.15)

 

Table 11.4 Typical values of L in Eq. (11.15)
Type of filter Lowpass Highpass Bandpass Bandstop
Butterworth N 0 N/2 0
Chebyshev N 0 N/2 0
Inverse-Chebyshev,
N odd 1 0 N/A N/A
N even 0 0 1 for N/2 odd 0
0 for N/2 even
Elliptic,
N odd 1 0 N/A N/A
N even 0 0 1 for N/2 odd 0
0 for N/2 even

 

\(H_{D}(z)=H_{0} \frac{\left(1-2 z+z^{2}\right)^{2}}{\prod_{j=1}^{2}\left(b_{0 j}+b_{1 j} z+z^{2}\right)}\)

 

where

 

\(\begin{array}{ll}b_{01}=7.630567 \times 10^{-2} & b_{11}=-3.267079 \times 10^{-1} \\b_{02}=6.070409 \times 10^{-1} & b_{12}=-7.608887 \times 10^{-1} \\ &H_{0} =2.076398 \times 10^{-1} &\end{array}\)

 

The above value of \(H _{0}\) was chosen to give a maximum passband gain of unity, which corresponds to a minimum passband loss of zero dB. The loss characteristic of the derived filter is compared with that of the analog filter in Fig. 11.4.
