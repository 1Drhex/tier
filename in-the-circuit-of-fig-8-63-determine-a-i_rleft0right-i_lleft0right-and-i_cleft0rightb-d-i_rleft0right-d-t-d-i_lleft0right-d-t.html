Guest Question.

In the circuit of Fig. 8.63, determine:

(a) \(i_{R}\left(0^{+}\right), i_{L}\left(0^{+}\right)\), and \(i_{C}\left(0^{+}\right)\)

(b) \(d i_{R}\left(0^{+}\right) / d t, d i_{L}\left(0^{+}\right) / d t,\) and \(d i_{C}\left(0^{+}\right) / d t\)

(c) \(i_{R}(\infty), i_{L}(\infty)\), and \(i_{C}(\infty)\)

Admin
Solution

(a) At t = 0-, the equivalent circuit is shown in Figure (a).

60||20 = 15 kohms, \(i_{R}(0-)\) = 80/(25 + 15) = 2mA.

By the current division principle,

\(\mathrm{i}_{\mathrm{L}}(0-)=60(2 \mathrm{mA}) /(60+20)=1.5 \mathrm{mA} \\\)
\(\mathrm{v}_{\mathrm{C}}(0-)=0\)

At t=0+

\(\mathrm{v}_{\mathrm{C}}(0+)=\mathrm{v}_{\mathrm{C}}(0-)=0\)

\(\mathrm{i}_{\mathrm{L}}(0+)=\mathrm{i}_{\mathrm{L}}(0-)={1.5 \mathrm{mA}} \\\)
\(80=\mathrm{i}_{\mathrm{R}}(0+)(25+20)+\mathrm{v}_{\mathrm{C}}(0-) \\\)
\(\mathrm{i}_{\mathrm{R}}(0+)=80 / 45 \mathrm{k}={1.778 \mathrm{mA}} \\\)

But \(\mathrm{i}_{\mathrm{R}}=\mathrm{i}_{\mathrm{C}}+\mathrm{i}_{\mathrm{L}}\)

\(1.778=\mathrm{i}_{\mathrm{C}}(0+)+1.5 \text { or } \mathrm{i}_{\mathrm{C}}(0+)={{0 . 2 7} 8 \mathrm{mA}}\) \(\begin{array}{ll}\text { (b) } & \text { v }_{\mathrm{L}}(0+)=\mathrm{v}_{\mathrm{C}}(0+)=0\end{array}\)

But

\(\begin{array}{l}\begin{aligned}\mathrm{v}_{\mathrm{L}}=\mathrm{Ldi}_{\mathrm{L}} / \mathrm{dt} & \text { and } \mathrm{di}_{\mathrm{L}}(0+) / \mathrm{dt}=\mathrm{v}_{\mathrm{L}}(0+) / \mathrm{L}=0\\\\\operatorname{di}_{\mathrm{L}}(0+) / \mathrm{dt}={0}\end{aligned} \\\\\text{Again, } 80=45 \mathrm{i}_{\mathrm{R}}+\mathrm{v}_{\mathrm{C}} \\\\0 \quad=45 \mathrm{di}_{\mathrm{R}} / \mathrm{dt}+\mathrm{dv}_{\mathrm{C}} / \mathrm{dt}\end{array}\)

But, \(\quad \mathrm{d} \mathrm{v}_{\mathrm{C}}(0+) / \mathrm{dt}=\mathrm{i}_{\mathrm{C}}(0+) / \mathrm{C}=0.278 \mathrm{mohms} / 1 \mu \mathrm{F}=278 \mathrm{V} / \mathrm{s}\)

Hence, \(\quad \operatorname{di}_{\mathrm{R}}(0+) / \mathrm{dt}=(-1 / 45) \mathrm{d} \mathrm{v}_{\mathrm{C}}(0+) / \mathrm{dt}=-278 / 45\)

\(\mathrm{di}_{\mathrm{R}}(0+) / \mathrm{dt}={-6.1778 \mathrm{A} / \mathrm{s}} \\\)
Also, \(\mathrm{i}_{\mathrm{R}}=\mathrm{i}_{\mathrm{C}}+\mathrm{i}_{\mathrm{L}} \\\)
\(\mathrm{di}_{\mathrm{R}}(0+) / \mathrm{dt}=\mathrm{di}_{\mathrm{C}}(0+) / \mathrm{dt}+\operatorname{di}_{\mathrm{L}}(0+) / \mathrm{dt} \\\)
\(-6.1788=\mathrm{di}_{\mathrm{C}}(0+) / \mathrm{dt}+0, \text { or } \mathrm{di}_{\mathrm{C}}(0+) / \mathrm{dt}={-6.1788 \mathrm{A} / \mathrm{s}}\)

(c) As t approaches infinity, we have the equivalent circuit in Figure (b).

\(\begin{array}{l}\mathrm{i}_{\mathrm{R}}(\infty)=\mathrm{i}_{\mathrm{L}}(\infty)=80 / 45 \mathrm{k}={1.778 \mathrm{mA}} \\\\\mathrm{i}_{\mathrm{C}}(\infty)=\mathrm{Cdv}(\infty) / \mathrm{dt}={0}\end{array}\)
