Guest Question.

A solar energy collector uses a single glass cover with a thickness of 4 mm. In the visible solar range, the refraction index of glass, n, is 1.526 and its extinction coefficient K is 32 \(m^{-1}\). Calculate the reflectivity, transmissivity, and absorptivity of the glass sheet for the angle of incidence of 60° and at normal incidence (0°).

Admin
Solution

Angle of incidence = 60°
From Eq. (2.44), \(n = \frac{n_{2}}{n_{1}} = \frac{\sin \theta _{1}}{\sin \theta _{2}}\)

the refraction angle \(θ_{2}\) is calculated as

\(\theta _{2} = \sin ^{-1}\left[\frac{\sin \theta _{1}}{n} \right] =\sin ^{-1}\left[\frac{\sin (60)}{1.526} \right]\) = 34.6°

From Eq. (2.51), the transmittance can be obtained as

\(\tau _{a} = e^{\left[-\frac{KL}{\cos (\theta _{2})} \right] } = e^{\left[-\frac{32(0.004)}{\cos (34.6)} \right] } \) = 0.856

From Eqs. (2.45) and (2.46),

\(t_{\bot } = \frac{\sin ^2 (\theta _{2}- \theta _{1})}{\sin ^2 (\theta _{2}+ \theta _{1})} = \frac{\sin ^2 (34.6- 60)}{\sin ^2 (34.6+ 60)}\) = 0.185

\(t_{\parallel } = \frac{\tan ^2 (\theta _{2}- \theta _{1})}{\tan ^2 (\theta _{2}+ \theta _{1})}=\frac{\tan ^2 (34.6- 60)}{\tan ^2 (34.6+ 60)}\) = 0.001

From Eqs. (2.52a)–(2.52c),

\(\tau _{\bot } = \frac{\tau _{\alpha }(1-r_{\bot })^2}{1-(r_{\bot }\tau _{\alpha })^2} = \tau _{\alpha }\frac{1-r_{\bot }}{1+r_{\bot }}\left\lgroup\frac{1-r^{2}_{\bot } }{1-(r_{\bot}\tau _{\alpha })^2} \right\rgroup \) , \(\alpha _{\bot } = (1- \tau _{\alpha })\left\lgroup\frac{1-r_{\bot } }{1-r_{\bot}\tau _{\alpha }} \right\rgroup\)

we have

\(\tau =\frac{\tau _{\alpha }}{2} \left\{\frac{1-r_{\bot }}{1+r_{\bot }}\left[\frac{1-r^{2}_{\bot }}{1-(r_{\bot }\tau _{\alpha })^2} \right]+\frac{1-r_{\parallel }}{1+r_{\parallel }} \left[\frac{1-r^{2}_{\parallel }}{1-(r_{\parallel }\tau _{\alpha })^2} \right] \right\} =\frac{\tau _{\alpha }}{2} \left[\tau _{\bot + \tau _{\parallel }}\right] =\)

\(\frac{0.856}{2} \left\{\frac{1-0.185}{1+0.185}\left[\frac{1-(0.185)^{2}}{1-(0.185\times 0.856)^2} \right]+\frac{1-0.001}{1+0.001} \left[\frac{1-0.001^{2}}{1-(0.001\times 0.856)^2} \right] \right\} \)= 0.428 (0.681 + 0.998) = 0.719

\(\rho =\frac{1}{2} \left[r_{\bot}(1+\tau_{\alpha }\tau_{\bot } )+r_{\parallel }(1+\tau_{\alpha }\tau_{\parallel } )\right] \) = 0.5[0.185(1+0.856 ×0.681) + 0.001(1+0.856 ×0.998)] = 0.147

\(\alpha =\frac{(1-\tau _{\alpha })}{2} \left[\left\lgroup\frac{1-r_{\bot }}{1-r_{\bot }\tau _{\alpha }} \right\rgroup +\left\lgroup\frac{1-r_{\parallel }}{1-r_{\parallel }\tau _{\alpha }} \right\rgroup \right]=\frac{(1-0.856)}{2} \left[\left\lgroup\frac{1-0.185}{1-0.185\times 0.856} + \frac{1-0.001}{1-0.001\times 0.856} \right\rgroup \right] \) = 0.142

Normal incidence
At normal incidence, \(\theta _{1}= 0^\circ \) and \(\theta _{2}= 0^\circ\) . In this case,\(\tau _{\alpha }\) is equal to 0.880.
There is no polarization at normal incidence; therefore, from Eq. (2.49),

\(r_{(0)}= r_{\bot } = r_{\parallel } = \left\lgroup\frac{n-1}{n+1} \right\rgroup ^2 = \left\lgroup\frac{1.526 -1}{1.526 +1} \right\rgroup^2\) = 0.043

From Eqs. (2.52a)–(2.52c), we have

\(\tau =\tau _{\alpha }\frac{1 – r_{(0)}}{1 + r_{(0)}} \left[\frac{1-r^{2}_{(0)} }{1- (r_{(0)} \tau _{\alpha })^2} \right] =0.880 \left\{\frac{1 – 0.043}{1 + 0.043} \left[\frac{1-0.043^2 }{1- (0.043 \times 0.880)^2} \right]\right\} \) = 0.807

\(\rho = r_{(0)} (1 + \tau _{\alpha }\tau _{(0)}) = 0.043 (1 + 0.880 \times 0.807)\) = 0.074

\(\alpha = ( 1-\tau _{\alpha }) \left\lgroup\frac{1-r_{(0)}}{1- r_{(0)} \tau _{\alpha }} \right\rgroup = (1- 0.880) \left\lgroup\frac{1- 0.043}{1- 0.043 \times 0.880} \right\rgroup\) = 0.119
