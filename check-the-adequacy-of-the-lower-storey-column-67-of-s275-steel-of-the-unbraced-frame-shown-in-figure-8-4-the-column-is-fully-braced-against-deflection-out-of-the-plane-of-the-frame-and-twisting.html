Guest Question.

Check the adequacy of the lower-storey column 67 (of S275 steel) of the unbraced frame shown in Figure 8.4. The column is fully braced against deflection out of the plane of the frame and twisting.

Admin
Solution

Design actions.
The first-order actions are \(N_{67} = 136.7 kN\) and \(M_{76} = 119.3\) kNm (Figure 8.4)
and the amplified moment is 141.8 kNm ().

\(H_{Ed} = 10 + 20 = 30 kN\) and

 

\(V_{Ed} = (20 + 40 + 20) + (40 + 80 + 40) = 240 kN.\)

 

\(H_{Ed} /V_{Ed} = 30/240 = 0.125 < 0.15,\)

and so the effects of sway imperfections should be included.

\(φ_{0} = 1/200\)

 

\(α_{h} = 2/√10 = 0.632 < 2/3 and so α_{h} = 2/3\)

 

m = 1 (one of the columns will have less than the average force),

\(α_{m} = √{0.5(1 + 1/1)} = 1.0,\) and

 

\(φ = (1/200) × (2/3) × 1.0 = 0.00333\)

The increased horizontal forces are

\(10 + 0.00333 × 80 = 10.27\) kN for the upper storey, and

\(20 + 0.00333 × 240 = 20.80 kN\) for the lower storey.

If the frame is reanalysed for these increased forces, then the axial force will increase from 136.7 to \(N_{Ed} = 137.2 kN\), and the moment from \(119.3 × 1.188 = 141.8 kNm\) to \(M_{y,Ed} = 121.9 × 1.188 = 144.8 kNm.\)

Section resistance.

\(t_{f} = 12.5 mm, f_{y} = 275 N/mm^2\)

 

\(ε =\sqrt{}(235/275) = 0.924\)

 

\(c_{f} /(t_{f} ε) = (204.3/2 − 7.9/2 − 10.2)/(12.5 × 0.924) = 7.62 < 9\) and the flange is Class 1.

The worst case for the web classification is when the web is fully plastic in compression, for which

\(c_{w}/(t_{w}ε) = (206.2 − 2 × 12.5 − 2 × 10.2)/(7.9 × 0.924) = 22.0 < 33\) and the web is Class 1.

Thus the section is Class 1.

\(γ_{M0} = 1.0\)

 

\(N_{c,Rd} = 66.3 × 10^2 × 275/1.0 N = 1823 kN = N_{pl,Rd}\)

 

\(0.25 N_{pl,Rd} = 0.25 × 1826 = 455.8 kN > 137.2 kN = N_{Ed}\) ,

 

\(0.5 h_{w}t_{w}f_{y} = 0.5(206.2 − 2 × 12.5) × 7.9 × 275 N = 196.8 kN > 137.2 kN = N_{Ed}\) ,

 

and so no allowance need be made for the effect of axial force on the plastic resistance moment.

\(M_{N,Rd} = M_{pl,Rd} = 567 × 10^3 × 275/1.0 Nmm = 155.9 kNm > 144.8 kNm = M_{Ed}\)

and the section resistance is adequate.

Member resistance.
Because the member is continuously braced out-of-plane, beam lateral buckling and column minor axis and torsional buckling need not be considered.

Thus \(M_{z,Ed} = 0, χ_{LT} = 1.0\), and

 

\(\bar{\lambda } _{0} = 0, C_{mLT} = 1.0, a_{LT} = 0, and b_{LT} = 0\)

 

\(L_{cr} = 5000\)

 

\(N_{cr} = π^2 × 210 000 × 5259 × 10^4/5000^2 N = 4360 kN\)

 

\(\frac{h}{b} = 206.2/204.3 = 1.009 < 1.2, t_{f} = 12.5 < 100\), and so for a S275 steel UC buckling about the y axis, use buckling curve b.

α = 0.34

\(\bar{\lambda }_{y}=\sqrt{(66.3 × 10^2 × 275)/(4360 × 10^3)} = 0.647\)

 

\(Φ = 0.5 × [1 + 0.34 × (0.647 − 0.2) + 0.647^2] = 0.785\)

 

\(χ_{y} =\frac{1}{0.785 +\sqrt{0.785^2 − 0.647^2}}= 0.813\)

Using Annex A,

\(µ_{y} =\frac{1 − 137.2/4360}{1 − 0.813 × 137.2/4360}= 0.994\)

 

\(ψ_{y} = 0/144.9 = 0\)

 

\(C_{my} = C_{my,0} = 0.79 + 0.21 × 0 + 0.36 × (0 − 0.33) × 137.2/4360 = 0.786\)

 

\(w_{y} = 567 × 10^3/510 × 10^3 = 1.112 < 1.5\)

 

\(\bar{\lambda }_{max} = \bar{\lambda }_{y} = 0.647\)

 

\(N_{Rk} = N_{pl,Rd} × γ_{M0} = 1823 × 1.0 = 1823 kN\)

 

\(γ_{M1} = 1.0\)

 

\(n_{pl} = 137.2/(1823/1.0) = 0.0753\)

 

\(C_{yy} = 1 + (1.112 − 1)[(2 − \frac{1.6}{ 1.112} × 0.785^2 × 0.647 − \frac{1.6}{ 1.112} ×0.785^2 × 0.647^2)× 0.0753 − 0]= 1.009\)

 

\(k_{yy} = 0.785 × \frac{1.0 ×0.994}{ 1 − 137.2/4360} × \frac{1}{ 1.009} = 0.798\)

Substituting into equation 6.61 of EC3,

\(\frac{qL^{3}}{2\sqrt{(EI_{z}GI_{t})} } = 15\left\{1 +\frac{1.8(ε − 0.3)}{\sqrt{(1 + 1.8^2(ε − 0.3)^2)} } \right\} + 40(K − 2)\left\{1 +\frac{2.8(ε − 0.4)}{\sqrt{(1 + 2.8^2(ε − 0.4)^2)} } \right\}\)

 

\(\frac{137.2}{ 0.813 × 1823} + 0.798 × \frac{144.8}{1.0 × 155.9} = 0.834 < 1.0\)

and so the member resistance is adequate.

 

Alternatively, using Annex B,

\(C_{my} = 0.9\)

 

\(k_{yy} = 0.9\left\{1 + (0.647 − 0.2)\frac{137.2}{0.813 × 1823/1.0} \right\} = 0.937\)

Substituting into equation 6.61 of EC3,

\(\frac{qL^{3}}{2\sqrt{(EI_{z}GI_{t})} } = 15\left\{1 +\frac{1.8(ε − 0.3)}{\sqrt{(1 + 1.8^2(ε − 0.3)^2)} } \right\} + 40(K − 2)\left\{1 +\frac{2.8(ε − 0.4)}{\sqrt{(1 + 2.8^2(ε − 0.4)^2)} } \right\}\)

 

\(\frac{137.2}{ 0.813 × 1823} + 0.937 × \frac{144.8}{1.0 × 155.9} = 0.963 < 1.0\)

 

and so the member resistance is adequate.
