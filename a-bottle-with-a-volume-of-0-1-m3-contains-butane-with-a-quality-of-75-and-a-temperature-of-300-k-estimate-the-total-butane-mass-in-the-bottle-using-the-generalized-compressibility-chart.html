Guest Question.

A bottle with a volume of 0.1 \(m ^{3}\) contains butane with a quality of 75% and a temperature of 300 K. Estimate the total butane mass in the bottle using the generalized compressibility chart.

Admin
Solution

We need to find the property v the mass is: m = V/v

so find v given \(T _{1}\) and x as : \(v = v _{ f }+ x v _{ fg }\)

Table A.2: Butane \(T _{ c }=425.2 K \quad P _{ c }=3.8 MPa =3800 kPa \)

\(T _{ r }=300 / 425.2=0.705 \quad \Rightarrow\)

 

From Fig. D.1 or table D.4: \(Z _{ f } \approx 0.02 ; \quad Z _{ g } \approx 0.9 ; \quad P _{ r \text { sat }}=0.1\)

 

\(P = P _{ sat }= P _{ r \text { sat }} \times P _{ c }\) = 0.1× 3.80 MPa ×1000 kPa/MPa = 380 kPa

 

\(v _{ f }= Z _{ f } RT / P\) = 0.02 × 0.14304 kJ/kgK × 300 K/380 kPa = 0.00226 \(m ^{3} / kg\)

 

\(v _{ g }= Z _{ g } RT / P\) = 0.9 × 0.14304 kJ/kgK × 300 K/380 kPa = 0.1016 \(m ^{3} / kg\)

 

v = 0.00226 + 0.75 × (0.1016 – 0.00226) = 0.076765 \(m ^{3} / kg\)

 

\(m =\frac{ V }{ v }=\frac{0.1}{0.076765}=1.303 kg\)
