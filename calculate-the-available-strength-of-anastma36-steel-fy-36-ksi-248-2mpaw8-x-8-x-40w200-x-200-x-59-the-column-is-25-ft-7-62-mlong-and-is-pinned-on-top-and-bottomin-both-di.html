Design procedure
According to AISC 360-10, there is only one column design curve to be used for any kind of sections. So the maximum compressive strength is associated with the minimum slenderness. Therefore, the column strength calculation should be performed by taking into consideration member slenderness about both principal axes, performing the following steps:
• cross-section classification;
• check slenderness ratio about strong axis \((KL_{x}/r_{x})\);
• check slenderness ratio about weak axis \((KL_{y}/r_{y})\);
• choose the higher slenderness ratio;
• calculate the elastic critical buckling stress \((F_{e})\);
• calculate the flexural buckling stress \((F_{cr})\);
• compute the nominal compressive strength \((P_{n})\);
• compute the available strength (LRFD:\(\phi_{c}P_{n}\); ASD:\(P_{n}/\Omega_{c}\)).

Solution

Check the slenderness ratio about both axes (by assuming K = 1.0)
\(\frac{KL_{x}}{r_{x}}=\frac{1\times (25.0.12)}{3.53}=84.9 \)governs
\(\frac{KL_{y}}{r_{y}}=\frac{1\times (12.5.12)}{2.04}=73.5\)
Calculate via Eq. (6.34) \(F_{e}=\frac{\pi^{2}E}{\left ( \frac{KL}{r} \right )^{2}}\)the elastic critical buckling stress \((F_{e})\).
\(F_{e}=\frac{\pi ^{2}E}{\left ( \frac{KL_{x}}{r_{x}} \right )}=\frac{3.14^{2}\times 29000 ksi}{(84.9)^{2}}\)=39.7 ksi (273.2 MPa)
Calculate the flexural buckling stress \((F_{cr})\).
Check limit:\(4.71\sqrt{\frac{E}{QF_{y}}}=4.71\times \sqrt{\frac{29000}{1\times 36}}=133.7> 84.9\)
Because \(\frac{KL_{x}}{r_{x}}\leq 4.71\sqrt{\frac{E}{(QF_{y})}}\)then, based on Eq.(6.32):\(\frac{Kl}{r}\leq 4.71\sqrt{\frac{E}{QF_{y}}}\left ( or\frac{(QF_{y})}{F_{e}\leq 2.25} \right ):\)
\(F_{cr}=\left [ 0.658^{\frac{(QF_{y})}{F_{e}}} \right ](QF_{y})=\left [ 0.658^{\frac{(1\times 36)}{39.7}} \right ]\times (1\times 36)=24.61 ksi(169.7 Mpa)\)
Compute via Eq. (7.31)\(\chi_{LT}=\frac{1}{\phi _{LT}+\sqrt{\phi^{2}_{LT} -\bar{\lambda}_{LT^{2}}}}\) the nominal compressive strength \((P_{n})\).
\(P_{n} = F_{cr}A_{g}\) =24.61 × 11.7 = 286.9 kips (1281 kN)
Compute the available strength.
LFRD: \(\phi _{c}P_{n}\)=0.90 × 287.9 = 259.1 kips (1153 kN)
ASD: \(P_{n}/\Omega _{c}\) = 287.9/1.67 = 172.4 kips (767 1kN)
