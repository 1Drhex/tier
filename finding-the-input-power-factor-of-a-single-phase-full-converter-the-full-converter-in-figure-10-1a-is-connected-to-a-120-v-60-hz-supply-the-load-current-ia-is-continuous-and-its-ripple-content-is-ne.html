Finding the Input Power Factor of a Single-Phase Full Converter

The full converter in Figure 10.1a is connected to a 120-V, 60-Hz supply. The load current \(I_a\) is continuous and its ripple content is negligible. The turns ratio of the transformer is unity. (a) Express the input current in a Fourier series; determine the HF of the input current, DF, and input PF. (b) If the delay angle is \(α = π/3\), calculate \(V_{dc}, V_n, V_{rms}, HF, DF\), and PF.

Admin
Solution

a. The waveform for input current is shown in Figure 10.1c and the instantaneous input current can be expressed in a Fourier series as

\(i_s(t)=a_0+\sum\limits_{n=1,2,. . .}^{\infty }{(a_n \cos nωt + b_n \sin nωt)}\)

where

\(a_0=\frac{1}{2\pi }\int_{\alpha }^{2\pi +\alpha }{i_s(t)d(\omega t)}=\frac{1}{2\pi }\left[\int_{\alpha }^{\pi +\alpha }{I_ad(\omega t)}-\int_{\pi +\alpha }^{2\pi +\alpha }{I_ad(\omega t)} \right]=0\)

 

\(a_n=\frac{1}{\pi }\int_{\alpha }^{2\pi +\alpha }{i_s(t) \cos nωt d(ωt)}\)

 

\(=\frac{1}{\pi} \left[\int_{\alpha }^{\pi +\alpha }{I_a \cos nωt d(ωt)}-\int_{\pi +\alpha }^{2\pi +\alpha }{I_a \cos nωt d(ωt)} \right]\)

 

\(=\frac{4I_a}{n\pi } \sin nα\) for \(n = 1, 3, 5,. . .\)

\(=0\) for \(n =2,4,. . .\)

 

\(b_n=\frac{1}{\pi} \int_{\alpha }^{2\pi +\alpha }{i(t) \sin nωt d(ωt)}\)

 

\(=\frac{1}{\pi} \left[\int_{\alpha }^{\pi +\alpha }{I_a \sin nωt d(ωt)}-\int_{\pi +\alpha }^{2\pi +\alpha }{I_a \sin nωt d(ωt)} \right]\)

 

\(=\frac{4I_a}{n\pi } \cos nα\) for \(n=1,3,5,. . . \)

\(=0 for n=2,4,. . .\)

Because \(a_0 = 0\), the input current can be written as

\(i_s(t)=\sum\limits_{n=1,3,5,. . .}^{\infty }{\sqrt{2}I_n \sin (n\omega t+\phi _n) }\)

where

\(\phi _n=tan^{-1}\frac{a_n}{b_n}=-n\alpha\) (10.4)

and \(ϕ_n\) is the displacement angle of the nth harmonic current. The rms value of the nth harmonic input current is

\(I_{sn}=\frac{1}{\sqrt{2} }(a^2_n+b^2_n)^{1/2}=\frac{4I_a}{\sqrt{2}n\pi }=\frac{2\sqrt{2}I_a }{n\pi }\) (10.5)

and the rms value of the fundamental current is

\(I_{s1}=\frac{2\sqrt{2}I_a }{\pi }\)

The rms value of the input current can be calculated from Eq. (10.5) as

\(I_s=\left(\sum\limits_{n=1,3,5,. . .}^{\infty }{I^2_{sn}} \right)^{1/2}\)

\(I_s\) can also be determined directly from

\(I_s=\left[\frac{2}{2\pi }\int_{\alpha }^{\pi +\alpha }{I^2_ad(\omega t)} \right]^{1/2}=I_a\)

From Eq. (3.22)

\(HF=\left(\frac{I^2_s-I^2_{s1}}{I^2_{s1}} \right)=\left[\left(\frac{I_s}{I_{s1}} \right)^2-1 \right]^{1/2}\) (3.22)

the HF is found as

\(HF=\left[\left(\frac{I_s}{I_{s1}} \right)^2-1 \right]^{1/2}=0.483\) or \(48\% \)

From Eqs. (3.21) and (10.4), the DF is

\(DF = \cos ϕ\) (3.21)

\(DF = \cos ϕ_1 = \cos(-α)\) (10.6)

From Eq. (3.23) the PF is found as

\(PF= \frac{V_sI_{s1}}{V_sI_{s}}\cos \phi =\frac{I_{s1}}{I_s}\cos \phi \) (3.23)

\(PF=\frac{I_{s1}}{I_s}\cos(-\alpha )=\frac{2\sqrt{2} }{\pi }\cos \alpha\) (10.7)

b. \(α = π/3\)

\(V_{dc} = \frac{2V_m}{\pi } \cos \alpha =54.02 V\) and \( V_n = 0.5 pu\)

\(V{rms}=\frac{V_m}{\sqrt{2} } =V_s =120 V\)

 

 

\(I_{s1}\left(2\sqrt{2}\frac{I_a}{\pi } \right)= 0.90032 I_a\) and \(I_s=I_a\)

 

\(HF=\left[\left(\frac{I_s}{I_{s1}} \right)^2-1 \right]^{1/2} = 0.4834\) or \(48.34\%\)

 

\(ϕ_1 = -α\) and \(DF= \cos(-\alpha )= \cos \frac{-\pi }{3}=0.5\)

 

\(PF= \frac{I_{s1}}{I_s}\cos(-\alpha )= 0.45\) (lagging)

Note: The fundamental component of input current is always \(90.03\%\) of \(I_a\) andthe HF remains constant at \(48.34\%\).
