Guest Question.

Designing an emitter follower

(a) Design an emitter follower with the topology shown in Fig. 8.34(a). Use transistor Q2N2222, for which minimum \( \beta_{ F }=100, \text { nominal } \beta_{ F }=173, I_{ S }=3.295 \times 10^{-14} A , \text { and } V_{ A }=200 V \). The operating collector current is set at \( I_{ C }=10 mA \). The DC power supply is \( V_{ CC }=15 V \). Assume \( V_{ BE }=0.7 V , R_{ L }=5 k \Omega, \text { and } R_{ s }=250 \Omega \).

(b) Use PSpice/SPICE to verify your results in part (a).

Admin
Solution

(a) Step 1. Design the biasing circuit. \( I_{ C }=10 mA , \text { and } V_{ CC }=15 V \). We will design for the worst-case value of \( \beta_{ F }=100 \).

 

\(\begin{aligned}&I_{ E }=\frac{\left(1+\beta_{ F }\right) I_{ C }}{\beta_{ F }}=\frac{101 \times 10 mA }{100}=10.1 mA \\&I_{ B }=\frac{I_{ C }}{\beta_{ F }}=\frac{10 mA }{100}=0.1 mA\end{aligned}\)

 

From Table 8.1 for Fig. 8.19, \( V_{ E }=V_{ CC } / 2=15 / 2=7.5 V \), which gives the value of

\( R_{ E }=\frac{V_{ E }}{I_{ E }}=\frac{7.5}{10.1 m A }=743 \Omega \)

 

The power rating of \( R_{ E } \) is

 

\( P_{ RE }=I_{ E }^{2} R_{ E }=(10.1 mA )^{2} \times 743=758 mW \)

 

The base voltage \( V_{ B } \) becomes

\( V_{ B }=V_{ E }+V_{ BE }=7.5+0.7=8.2 V \)

 

The value of \( R_{ B } \) can be found from

\(R_{ B }=\frac{V_{ CC }-V_{ B }}{I_{ B }}=\frac{\left( V _{ CC }-V_{ B }\right) \beta_{ F }}{I_{ C }}\) (8.104)

 

\( =\frac{(15-8.2) V \times 100}{10 mA }=68 k \Omega \)

 

The power rating of \( R_{ B } \) is

 

\( P_{ RB }=I_{ B }^{2} R_{ B }=(0.1 mA )^{2} \times 68 k \Omega=0.68 mW \)

 

Step 2. Find the small-signal parameters of the transistor. The results of give \( r_{\pi}=258 \Omega , g_{ m }=387.6 mA / V , \beta_{ F }=100 \text {, and } r_{ o 1}=20 k \Omega \).

Step 3. Find the values of \(C_{1} \text { and } C_{2} \text {. Let us choose } C_{1}=C_{2}=10 \mu F \text {. }\)

Step 4. Evaluate the values of the input resistance \( R_{ i } \), the open-circuit voltage gain \( A_{ vo } \), and the output resistance \( R_{ o }\). From Eq. (8.96)

 

\( R_{ x }=\frac{v_{ b }}{i_{ b }}=r_{\pi}+\left(1+\beta_{ f }\right)\left(r_{ o 1}\left\|r_{ o 2}\right\| R_{ L }\right) \) (8.96)

 

\( R_{ x }=r_{\pi}+\left(1+\beta_{ f }\right)\left(r_{ o 1}\left\|R_{ E }\right\| R_{ L }\right)=258+(1+100) \times(20 k \|743\| 5 k )=65.59 k \Omega \)

 

From Eq. (8.97),

 

\( R_{ i }=\frac{v_{ b }}{i_{ s }}=R_{ B } \| R_{ x } \) (8.97)

 

\(\begin{aligned}&R_{ i }=R_{ B }\left\|R_{ x }=68 k \right\| 65.59 k =33.4 k \Omega \\&R_{ in }=\frac{v_{ S }}{i_{ s }}=R_{ i }+R_{ s }=33.4 k +250=33.6 k \Omega\end{aligned}\)

 

From Eq. (8.102),

 

\( A_{ vo }=\frac{\left(1+g_{ m } r_{\pi}\right)\left(r_{ o 1} \| R_{ E }\right)}{1+\left(1+g_{ m } r_{\pi}\right)\left(r_{ o 1} \| R_{ E }\right)}=\frac{\left(1+\beta_{ f }\right)\left(r_{ o 1} \| R_{ E }\right)}{1+\left(1+\beta_{ f }\right)\left(r_{ o 1} \| R_{ E }\right)} \) (8.102)

 

\(\begin{aligned}A_{ vo } &=\frac{\left(1+\beta_{ f }\right)\left(R_{ E } \| r_{ o 1}\right)}{\left[r_{\pi}+\left(1+\beta_{ f }\right)\left(R_{ E } \| r_{ o1 }\right)\right]} \\&=\frac{(1+100) \times(743 \| 20 k )}{[258+(1+100) \times(743 \| 20 k )]}=0.9966\end{aligned}\)

 

From Eq. (8.103),

 

\(R_{ o }=\left(r_{ o 1} \| R_{ E }\right) \| \frac{r_{\pi}+\left(R_{ s } \| R_{ B }\right)}{1+\beta_{ f }}\) (8.103)

 

\( R_{ o }=R_{ E }\left\|r_{ o 1}\right\| \frac{r_{\pi}+\left(R_{ s } \| R_{ B }\right)}{1+\beta_{ f }}=743\|20 k \| \frac{258+250 \| 68 k }{1+100}=5 \Omega \)

 

The output resistance including \( R_{ L } \) is

 

\( R_{\text {out }}=R_{ L }\left\|R_{ o }=5 k \right\| 5 \approx 4.99 \Omega \)

 

(b) The emitter-follower circuit for PSpice simulation is shown in Fig. 8.35.

The PSpice plots, which are shown in Fig. 8.36 for \( v_{ s }=1 V , \text { give } v_{ o }=987 mV \text { and } R_{ in }=V_{ s ( rms )} / I_{ s ( rms )}= 41.98 k \Omega \)(expected value is 33.6 kΩ), and the voltage gain is \( A_{ v }=v_{ o } / v_{ s }=0.987 \) (expected value is 0.9966). If we run the simulation with a very large value of \( R_{ L } \), tending to infinity \( \text { (say, } R_{ L }=10 G \Omega \text { ) } \), the output voltage will be the maximum \( v_{ o (\max )} \). Then if we connect the normal load \( \left(\text { say, } R_{ L }=5 k \Omega\right) \) and run the simulation, the output voltage should drop because of the current flow through the output resistance \( R_{ o } \) of the amplifier. PSpice simulation gives \( v_{0}=987 mV \text { and } v_{0(\max )}=990 mV \text { for } R_{ L }=10 G \Omega \text {. Thus, } R_{ o } \) can be found from

 

\( \Delta v_{ o }=v_{ o (\max )}-v_{ o }=R_{ o } i_{ L }=\frac{R_{ o } v_{ o }}{R_{ L }} \)

 

which gives

\( R_{ o }=\frac{R_{ L }\left(v_{ o ( max )}-v_{ o }\right)}{v_{ o }}=\frac{5 k \Omega \times(990-987) mV }{987 mV }=15 \Omega \)

 

NOTE: This example simulates the steps that would normally be used to measure the output resistance of an amplifier in the laboratory. As expected, the simulated results differ from the design values, and the design calculations should be modified. Even if we modify the design to satisfy the design specifications, the results can be expected to differ from those that would be obtained in the laboratory, although not significantly.

If we ran a PSpice simulation of the linear circuit shown in Fig. 8.34(c), the results would be closer to the expected values, but they would not take into account the nonlinear behavior of the transistor.
