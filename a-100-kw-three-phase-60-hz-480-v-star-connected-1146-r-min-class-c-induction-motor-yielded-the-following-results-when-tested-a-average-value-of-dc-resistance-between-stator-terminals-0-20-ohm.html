Guest Question.

A 100 kW, three-phase, 60 Hz, 480 V, star-connected, 1146 r/min, Class C induction motor yielded the following results when tested:

a. Average value of dc resistance between stator terminals: 0.20 ohms

b. No-load test: \(V_{exc}\) = 480 V, L-L

\(P_{exc}\) = 1920 W, three-phase

\(I_{exc}\) = 40 A
c. Locked-rotor test:
\(V_{z}\) = 32 V, L-L
\(P_{z}\) = 2560 W, three-phase
\(I_{z}\) = 65.6 A
f = 15 Hz
Determine the per-phase equivalent circuit of the motor.

Admin
Solution

No-Load Test (Fig. 3-28)

Using Eq. (3.63) \(R^{′}_{m}=\frac{P_{exc}}{I_{exc}^{2}} \),

\( R^{′}_{m} =\frac{P_{exc}}{I_{exc}^{2}} = \frac{1920}{3\left(40\right)^{2} } =0.40\Omega /phase \)

From Eq. (3.66) \( R_{m}=\frac{2}{3}R^{′}_{m} ohms/phase \),

\( R_{m}=\frac{2}{3}\left(0.40\right)=0.27 \Omega / phase \)

The per-phase stator resistance is

\( R_{1}=\frac{0.2}{2}=0.10 \Omega / phase \)

The rotational loss equivalent resistance is found by using Eq. (3.65) \( R^{′}_{m} = R_{1}+R_{m}+R_{rotational losses} \).

\( R_{r.l.}=0.40-0.10-0.27=0.033\Omega / phase \)

This resistance is not explicitly shown in the equivalent circuit, but its power consumption will be part of the power developed by the motor. From Eq. (3.62) \( Z^{′}_{m} = \frac{V_{exc}}{I_{exc}}\), the gross magnetizing impedance is

\( Z^{′}_{m} = \frac{V_{exc}}{I_{exc}}=\frac{480/\sqrt{3} }{40}=6.92\Omega / phase \)

The gross magnetizing reactance is

\( X^{′}_{m} = \sqrt{6.92^{2}-0.40^{2}} = 6.92\Omega / phase \)

Locked-Rotor Test (Fig 3-29)

From the data of the locked-rotor test and Eqs. (3.58) \( Z_{e} = \frac{V_{z}}{I_{z}} \) and (3.59)\( R_{e} = \frac{P_{z}}{I_{z}^{2}} = R_{1}+R_{2} \), we obtain

\( Z_{e} =\frac{V_{z}}{I_{z}} = \frac{32/\sqrt{3} }{65.6} = 0.28 \Omega / phase \)

 

\( R_{e} = \frac{P_{z}}{I_{z}^{2}} = \frac{2560}{3\left(65.6\right)^{2} }=0.2 \Omega / phase \)

The rotor resistance, as seen from the stator, is found by using Eq. (3.59) \( R_{e} = \frac{P_{z}}{I_{z}^{2}} = R_{1} + R_{2} \).

\( R_{2}=R_{e}-R_{1} \)

=\( 0.20 – 0.10\)

=\(0.10\Omega /phase\)

The equivalent motor reactance is found by using Eq. (3.60)\( X_{e}=\sqrt{Z_{e}^{2} – R_{e}^{2}}=X_{1}+X_{2} \).

\( X_{e}=\sqrt{Z_{e}^{2}-R_{e}^{2}}=\sqrt{0.28^{2}-0.20^{2}}=0.20 \Omega /phase \)

The stator and rotor reactances can be found by using Tables 3-5

and Eq. (3.60)\( X_{e}=\sqrt{Z_{e}^{2}-R_{e}^{2}}=X_{1}+X_{2}\), as follows:

\( X_{e} = X_{1}+X_{2} \)

Substituting, we get

\( 0.2 = X_{1}+\frac{X_{1}}{0.43} \)

From the last two equations,

\(X_{1} = 0.06\Omega /phase\)

and

\( X_{2} = 0.14\Omega /phase\)

The above reactances were measured at a frequency of 15 Hz. Thus, at the rated frequency of 60 Hz,

\( X_{1} = 4(0.06) = 0.24\Omega /phase \)

and

\( X_{2}=4(0.14) = 0.56\Omega /phase \)

The magnetizing reactance is found by using Eq. (3.67)\(X^{′}_{m} = X_{1}+R_{m} \):

\( X_{m}=X^{′}_{m}-X_{1} \)

=\( 6.93-0.24 = 6.69\Omega /phase \)

The per-phase equivalent circuit, as obtained from the data of the given tests, is shown in Fig. 3-30.
