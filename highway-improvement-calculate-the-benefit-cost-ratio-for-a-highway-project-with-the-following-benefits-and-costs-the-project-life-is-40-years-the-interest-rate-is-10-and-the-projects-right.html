Guest Question.

Highway Improvement
Calculate the benefit/cost ratio for a highway project with the following benefits and costs. The project life is 40 years, the interest rate is 10%, and the project’s right-of-way is worth $5M in 40 years.

$20M Construction cost (includes acquiring right-of-way)
350K Annual maintenance
3M Repaving every 8 years
1M Annual value of lives saved (1 per year) \(\left[\frac{\$ 20}{\text { hour }} \times \frac{.25 \text { hours }}{\text { trip }} \times \frac{250,000 \text { trips }}{\text { year }}\right]\)
1.25M Time savings for commercial traffic
1M Time savings for commuter and recreational traffic \(\left[\frac{\$ 5}{\text { hour }} \times \frac{.25 \text { hours }}{\text { trip }} \times \frac{800,000 \text { trips }}{\text { year }}\right]\)
Admin
Solution

Most of the costs are stated in annual terms; thus, it will be slightly easier to compute the ratio in annual terms. The benefits include lives saved and time savings for two groups. The cost equation shown below includes the construction cost, the annual maintenance, the periodic repaving, and in the final term, the “correction” for the repaving that does not occur in year 40 and the rightof-way value.
B = 1M + 1.25M + 1M = $3.25M
C = 20M(A/P, 10%, 40) + .35M + 3M(A/F, 10%, 8) – (3M + 5M)(A/F, 10%, 40)
= 2.046M + .35M + .262M + .018M = $2.64M
B/C = 3.25M/2.64M = 1.23

Since 1.23 is greater than 1, build it!
To convert the benefits and costs to present values, it is easiest to simply multiply B and C by (P/A,10%,40). Since both the numerator and the denominator are multiplied by the same number, the ratio is unchanged.
