Guest Question.

The axisymmetric element of Figure 9.11 is part of a body rotating with angular velocity 10 rad/s about the z axis and subjected to gravity in the negative z direction. Compute the equivalent nodal forces. Density is 7.3 \(\ (10)^{4} lb-s²/in.^{4} \)

Admin
Solution

For the stated conditions, we have

\(\ R_{B}=r\omega ^{2}=100r in./s^{2} \)

\(\ Z_{B}=-g=-386.4 in./s^{2} \)

Using the interpolation functions as given in ,

\(\ f_{r1} = 2\pi \rho \int\limits_{A}^{}{N_{1} R_{B}r dr dz }= 2\pi \rho (100)\int\limits_{3}^{4}{\int\limits_{0}^{4-r}{(4-r-z)r^{2} dz dr } }=0.84 Ib \)

\(\ f_{r2} = 2\pi \rho \int\limits_{A}^{}{N_{2} R_{B}r dr dz }= 2\pi \rho (100)\int\limits_{3}^{4}{\int\limits_{0}^{4-r}{(r-3)r^{2} dz dr } }=0.98 Ib \)

\(\ f_{r3} = 2\pi \rho \int\limits_{A}^{}{N_{3} R_{B}r dr dz }= 2\pi \rho (100)\int\limits_{3}^{4}{\int\limits_{0}^{4-r}{zr^{2} dz dr } }=0.84 Ib \)

\(\ f_{z1} = 2\pi \rho \int\limits_{A}^{}{N_{1} Z_{B}r dr dz }= -2\pi \rho (386.4)\int\limits_{3}^{4}{\int\limits_{0}^{4-r}{(4-r-z)r dz dr } }=-1.00 Ib \)

\(\ f_{z2} = 2\pi \rho \int\limits_{A}^{}{N_{2} Z_{B}r dr dz }= -2\pi \rho (386.4)\int\limits_{3}^{4}{\int\limits_{0}^{4-r}{(r-3)r dz dr } }=-1.08 Ib \)

\(\ f_{z3} = 2\pi \rho \int\limits_{A}^{}{N_{3} Z_{B}r dr dz }= -2\pi \rho (386.4)\int\limits_{3}^{4}{\int\limits_{0}^{4-r}{zr dz dr } }=-1.00 Ib \)

The integrations required to obtain the given results are straightforward but algebraically tedious. Another approach that can be used and is increasingly accurate for decreasing element size is to evaluate the body forces and the integrand at the centroid of the cross section of the element area as an approximation. Using this approximation, it can be shown that

\(\ \int\limits_{A}^{}{N_{i} (\bar{r},\bar{z} )\bar{r} dz dr } =\frac{\bar{r} A}{3} \) i = 1, 3

so the body forces are allocated equally to each node. For the present example, the result is

\(\ f_{r1}= f_{r2}= f_{r3}= 0.88 Ib \)

\(\ f_{z1}= f_{z2}= f_{z3}= -1.03 Ib \)

Note that, within the numerical accuracy used here, the total radial force and the total axial force are the same for the two methods.
