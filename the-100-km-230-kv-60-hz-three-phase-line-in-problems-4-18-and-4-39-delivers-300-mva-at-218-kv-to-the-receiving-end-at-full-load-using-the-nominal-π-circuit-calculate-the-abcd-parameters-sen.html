Guest Question.

The 100-km, 230-kV, 60-Hz, three-phase line in Problems and delivers 300 MVA at 218 kV to the receiving end at full load. Using the nominal π circuit, calculate the ABCD parameters, sending-end voltage, and percent voltage regulation when the receiving-end power factor is (a) 0.9 lagging, (b) unity, and (c) 0.9 leading. Assume a 50°C conductor temperature to determine the resistance of this line.

Admin
Solution

(a) From , \(R=0.1128 \frac{\Omega}{ mi }\left(\frac{1 mi }{1.609 km }\right)=0.0701 \Omega / km\)

\(\bar{z}=0.0701+j 0.506=0.511 \angle 82.11^{\circ} \Omega / km ; \bar{y}=3.229 \times 10^{-6} \angle 90^{\circ} s / km\)

From Problems and

\(\begin{aligned}\bar{A}=\bar{D}=1+\frac{\bar{Y} \bar{Z}}{2}=& 1+\frac{1}{2}\left(3.229 \times 10^{-6} \times 100 \angle 90^{\circ}\right)\left(0.511 \times 100 \angle 82.11^{\circ}\right) \\&=0.9918 \angle 0.0999^{\circ} \text { per unit }\end{aligned}\)

 

\(\bar{B}=\bar{Z}=\bar{z} l=0.511 \times 100 \angle 82.11^{\circ}=51.1 \angle 82.11^{\circ} \Omega\)

 

\(\begin{aligned}\bar{C}=\bar{Y}\left(1+\frac{\bar{Y} \bar{Z}}{4}\right) &=\left(3.229 \times 10^{-4} \angle 90^{\circ}\right)\left[1+0.004125 \angle 172.11^{\circ}\right] \\=& 3.216 \times 10^{-4} \angle 90.033^{\circ} S\end{aligned}\)

 

\(\begin{aligned}&\bar{V}_{R}=\frac{218}{\sqrt{3}} \angle 0^{\circ}=125.9 \angle 0^{\circ} kV _{ LN } \\&\bar{I}_{R}=\frac{300}{218 \sqrt{3}} \angle-\cos ^{-1} 0.9=0.7945 \angle-25.84^{\circ} kA\end{aligned}\)

 

\(\begin{aligned}\bar{V}_{s}=\bar{A} \bar{V}_{R}+\overline{B I}_{R} &=0.9918 \angle 0.0999(125.9)+51.1 \angle 82.11^{\circ}\left(0.7945 \angle-25.84^{\circ}\right) \\&=151.3 \angle 12.98^{\circ} kV _{ LN }\end{aligned}\)

 

\(\begin{aligned}&V_{s}=151.3 \sqrt{3}=262 kV _{ LL } \\&V_{R N L}=V_{S} / A=262 / 0.9918=264.2 kV _{ LL } \\&\% V R=\frac{V_{R N L}-V_{R F L}}{V_{R F L}} \times 100=\frac{264.2-218}{218} \times 100=21.2 \%\end{aligned}\)

 

(b) \(\bar{I}_{R}=0.7945 \angle 0^{\circ} kA\)

\(\begin{aligned}\bar{V}_{S} &=0.9918 \angle 0.0999^{\circ}(125.9)+51.1 \angle 82.11^{\circ}\left(0.7945 \angle 0^{\circ}\right) \\&=136.6 \angle 17.2^{\circ} kV _{ LN } ; V_{S}=136.6 \sqrt{3}=236.6 kV _{ LL }\end{aligned}\)

 

\(\begin{aligned}&V_{R N L}=V_{S} / A=236.6 / 0.9918=238.6 kV _{ LL } \\&\% V R=\frac{238.6-218}{218} \times 100=9.43 \%\end{aligned}\)

 

(c)

\(\begin{aligned}&\bar{I}_{R}=0.7945 \angle 25.84^{\circ} kA \\&\bar{V}_{S}=124.9 \angle 0.0999^{\circ}+40.6 \angle 107.95^{\circ}=118.9 \angle 19.1^{\circ} kV _{ LN } \\&V_{S}=118.9 \sqrt{3}=205.9 kV _{ LL } \\&\% V R=\frac{205.9-218}{218} \times 100=-5.6 \%\end{aligned}\)
