Guest Question.

The platform has mass \(M\) and center of mass located at \(G\). If it is lifted using the three cables,determine the force in each of these cables.

Units Used:

\(Mg\) = \({ 10 }^{ 3 }\) \(kg\quad \quad kN\) = \({ 10 }^{ 3 }\) \(N \quad \quad g\) = \(9.81\) \(\frac { m }{ { s }^{ 2 } }\)

Given:

\(M\) = \(3\) \(Mg\)

\(a\) = \(4\) \(m \)

\(b\) = \(3\) \(m \)

\(c\) = \(3\) \(m\)

\(d\) = \(4\) \(m \)

\(e\) = \(2\) \(m\)

Admin
Solution

The initial guesses are:

\({ F }_{ AC }=10N\quad \quad { F }_{ BC }=10N\quad \quad { F }_{ DE }=10N\)

Given

\(\frac { b({ F }_{ AC }) }{ \sqrt { { a }^{ 2 }+{ b }^{ 2 } } } -\frac { c({ F }_{ BC }) }{ \sqrt { { a }^{ 2 }+{ c }^{ 2 } } } =0\)

 

\(Mge-\left( { F }_{ AC } \right) a\frac { d+e }{ \sqrt { { a }^{ 2 }+{ b }^{ 2 } } } -{ F }_{ BC }\frac { a(d+e) }{ \sqrt { { a }^{ 2 }+{ c }^{ 2 } } } =0\)

 

\(\frac { a }{ \sqrt { { a }^{ 2 }+{ c }^{ 2 } } } { F }_{ BC }(b+c)-Mgb+{ F }_{ DE }b=0\)

 

\(\left( \begin{matrix} { F }_{ AC } \\ { F }_{ BC } \\ { F }_{ DE } \end{matrix} \right) =Find({ F }_{ AC },{ F }_{ BC },{ F }_{ DE })\quad \quad \quad \left( \begin{matrix} { F }_{ AC } \\ { F }_{ BC } \\ { F }^{ DE } \end{matrix} \right) =kN\)
