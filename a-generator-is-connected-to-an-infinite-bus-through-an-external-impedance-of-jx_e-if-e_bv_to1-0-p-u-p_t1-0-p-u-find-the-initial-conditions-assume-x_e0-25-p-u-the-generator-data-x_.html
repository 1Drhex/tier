Guest Question.

A generator is connected to an infinite bus through an external impedance of \(jx_{e} \). If \(E_{b}=V_{to}=1.0 p.u \). \(P_{t}=1.0 \) p.u. Find the initial conditions. Assume \(x_{e}=0.25 \) p.u.

The generator data: \(x_{d}=1.8,x_{q}=1.7,x_{d}^{'}=0.17,x_{q}^{'}=0.23,R_{a}=0.0,T_{d}^{'}=0.4sec,T_{q}^{'}=0.1sec,H=4sec,f_{B}=60 \)Hz

Admin
Solution

The generator terminal bus angle \(\theta _{o}\) is found from

\(P_{t}=\frac{V_{t}E_{b}\sin \theta _{o}}{x_{e}}\)

Substituting values,

\(\sin \theta _{o}=0.25,\theta _{o}=14.48^{\circ}\)

 

\(\hat{I_{a}}=\frac{\hat{V_{t}-E_{b}\angle 0}}{jx_{e}}=\frac{1.0\angle 14.48^{\circ}-1.0\angle 9^{\circ}}{j0.25}=1.008\angle 7.24^{\circ}\)

 

\(\hat{E_{qo}}=V_{t}\angle \theta _{o}+jx_{q}\hat{I_{a}}\)

 

\(=1.0\angle 14.48^{\circ}+j1.7\cdot 1.008\angle 7.24^{\circ}\)

 

\(=2.090\angle 68.91^{\circ}=E_{qo}\angle \delta _{o}\)

 

\(E_{qo}=2.090,\delta _{o}=68.91^{\circ}\)

The initial armature current components are

\(i_{do}=-I_{a}\sin \left ( \delta _{o}-\phi _{o} \right )=-1.008\sin \left ( 68.91-7.24 \right )\)

=-0.8873

\(i_{qo}=-I_{a}\cos \left ( \delta _{o}-\phi _{o} \right )=-1.008\cos \left ( 68.91-7.24 \right )\)

=0.4783

\(E_{fdo}=E_{qo}-\left ( x_{d}-x_{q} \right )i_{do}=2.090+0.1\cdot 0.8873\)

=2.179

\(E_{qo}^{'}=E_{fdo}+\left ( x_{d}-x_{d}^{'} \right )i_{do}=0.7327\)

 

\(E_{do}^{'}=-\left ( x_{q}-x_{q}^{'} \right )i_{qo}=-0.7031\)

 

\(T_{eo}=E_{qo^{i_{qo}}}^{'}+E_{do^{i_{do}}}^{'}+\left ( x_{d}^{'}-x_{q}^{'} \right )i_{do}i_{qo}\)

=1.0

(Check:\(T_{eo}=P_{t}+I_{a}^{2}R_{a}\))

The initial slip, \(S_{mo}\) can be assumed to be equal to zero.
