Guest Question.

The complex electric field inside an infinitely long rectangular pipe, with all four vertical walls perfectly electric conducting, as shown in Figure P1-29, is given by

\(E={ \hat { a } }_{ z }(1+j)\sin { (\frac { \pi }{ a } x) } \sin { (\frac { \pi }{ b } y) }\)

Assuming that there are no sources withinthe box and \(a ={ λ }_{ 0 }, b = 0.5{ λ }_{ 0 }\), and \(μ ={ μ }_{ 0 }\),where \({ λ }_{ 0 }=\) free space, infinite medium wavelength, find the:

(a) Conductivity.

(b) Dielectric constant.of the medium within the box.

Admin
Solution
\(\underline { E } ={ \hat { a } }_{ z }(1+j)\sin { (\frac { \pi }{ a } x) } \sin { (\frac { \pi }{ b } y) }\)

Using Maxuwell’s equation of \(\nabla \times \underline { E } =j\omega { \mu }_{ 0 }\underline { H }\) leads to

\(\underline { H } -\frac { 1 }{ j\omega \mu _{ 0 } } \left[ { \hat { a } }_{ x }\frac { ə{ E }_{ z } }{ əy } -{ \hat { a } }_{ y }\frac { ə{ E }_{ z } }{ əx } \right] =-\frac { (1+j) }{ j\omega \mu _{ 0 } } \left[ { \hat { a } }_{ x }(\frac { \pi }{ b } )\cos { (\frac { \pi }{ b } y)\sin { (\frac { \pi }{ a } x)-{ \hat { a } }_{ y } } } (\frac { \pi }{ a } )\sin { (\frac { \pi }{ b } y)\cos { (\frac { \pi }{ a } x) } } \right]\)

Now using Maxuwell’s equation of \(\nabla \times \underline { H } =\underline { J } _{ c }+j\omega \varepsilon \underline { E } =\sigma \underline { E } +j\omega \varepsilon \underline { E }\) leads to

\(\nabla \times \underline { H } ={ -\hat { a } }_{ z }\frac { 1 }{ j\omega { \mu }_{ 0 } } \left[ (\frac { \pi }{ a } )^{ 2 }+(\frac { \pi }{ b } )^{ 2 } \right] \sin { (\frac { \pi }{ b } y) } \sin { (\frac { \pi }{ a } x) } { =\hat { a } }_{ z }(\sigma +j\omega \varepsilon )(1+j)\sin { (\frac { \pi }{ b } y) } \sin { (\frac { \pi }{ a } x) }\)

Equating both sides, we can write that

\((\sigma +j\omega \varepsilon )=\frac { 1 }{ j\omega { \mu }_{ 0 } } \left[ (\frac { \pi }{ a } )^{ 2 }+(\frac { \pi }{ a } )^{ 2 } \right] \Rightarrow \sigma =0\\ { \varepsilon }_{ r }=\frac { \varepsilon }{ { \varepsilon }_{ 0 } } =\frac { 1 }{ \omega ^{ 2 }{ \mu }_{ 0 }{ \varepsilon }_{ 0 } } \left[ (\frac { \pi }{ a } )^{ 2 }+(\frac { \pi }{ a } )^{ 2 } \right]\)
