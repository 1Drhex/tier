Guest Question.

Figure 8–19 is a cross section of a grade 25 cast-iron pressure vessel. A total of N bolts are to be used to resist a separating force of 36 kip.
(a) Determine \(k_{b}, k_{m}\), and C.
(b) Find the number of bolts required for a load factor of 2 where the bolts may be reused when the joint is taken apart.
(c) With the number of bolts obtained in part (b), determine the realized load factor for overload, the yielding factor of safety, and the load factor for joint separation.

Admin
Solution

(a) The grip is l = 1.50 in. From Table A–31, the nut thickness is \(\frac{35}{64} \text { in. }\) Adding two threads beyond the nut of \(\frac{2}{11}\)in gives a bolt length of

 

\(L=\frac{35}{64}+1.50+\frac{2}{11}=2.229 \text { in }\)

 

From Table A–17 the next fraction size bolt is \(L=2 \frac{1}{4}\) From Eq. (8–13), the thread length is \(L_{T}=2(0.625)+0.25=1.50 \text { in. }\) Thus, the length of the unthreaded portion

 

\(L_{T}=\left\{\begin{array}{ll}2 d+\frac{1}{4} \text { in } & L \leq 6 \text { in } \\2 d+\frac{1}{2} \text { in } & L>6 \text { in }\end{array}\right.\) (8–13)

 

in the grip is \(l_{d}=2.25-1.50=0.75 \text { in. }\) The threaded length in the grip is \(l_{t}=l-\)\(l_{d}=0.75 \text { in. }\) From Table 8–2, \(A_{t}=0.226 in ^{2}\) The major-diameter area is \(A_{d}=\)\(\pi(0.625)^{2} / 4=0.3068 in ^{2}\).The bolt stiffness is then

 

\(k_{b}=\frac{A_{d} A_{t} E}{A_{d} l_{t}+A_{t} l_{d}}=\frac{0.3068(0.226)(30)}{0.3068(0.75)+0.226(0.75)}\)

 

= 5.21 Mlbf/in

 

From Table A–24, for no. 25 cast iron we will use E = 14 Mpsi. The stiffness of the members, from Eq. (8–22), is

 

\(k_{m}=\frac{0.5774 \pi E d}{2 \ln \left(5 \frac{0.5774 l+0.5 d}{0.5774 l+2.5 d}\right)}\) (8–22)

 

\(\begin{aligned}k_{m} &=\frac{0.5774 \pi E d}{2 \ln \left(5 \frac{0.5774 l+0.5 d}{0.5774l+2.5d}\right)}=\frac{0.5774 \pi(14)(0.625)}{2 \ln \left[5 \frac{0.5774(1.5)+0.5(0.625)}{0.5774(1.5)+2.5(0.625)}\right]} \\&=8.95 Mlbf / \text { in }\end{aligned}\)

 

If you are using Eq. (8–23), from Table 8–8, A = 0.778 71 and B = 0.616 16, and

 

\(\frac{k_{m}}{E d}=A \exp (B d / l)\) (8–23)

 

\(\begin{aligned}k_{m} &=E d A \exp (B d / l) \\&=14(0.625)(0.77871) \exp [0.61616(0.625) / 1.5]\end{aligned}\)

 

\(=8.81 Mlbf / in\)

 

which is only 1.6 percent lower than the previous result.
From the first calculation for \(k_{m}\), the stiffness constant C is

 

\(C=\frac{k_{b}}{k_{b}+k_{m}}=\frac{5.21}{5.21+8.95}=0.368\)

 

(b) From Table 8–9, \(S_{p}=85 kpsi\). Then, using Eqs. (8–31) and (8–32), we find the recommended preload to be

 

\(F_{i}=\left\{\begin{array}{ll}0.75 F_{p} & \text { for nonpermanent connections, reused fasteners } \\0.90 F_{p} & \text { for permanent connections }\end{array}\right.\) (8–31)

 

\(F_{p}=A_{t} S_{p}\) (8–32)

 

\(F_{i}=0.75 A_{t} S_{p}=0.75(0.226)(85)=14.4 kip\)

 

For N bolts, Eq. (8–29) can be written

 

\(n_{L}=\frac{S_{p} A_{t}-F_{i}}{C P}\) (8–29)

 

\(n_{L}=\frac{S_{p} A_{t}-F_{i}}{C\left(P_{\text {total }} / N\right)}\) (1)

 

or \(N=\frac{C n_{L} P_{\text {total }}}{S_{p} A_{t}-F_{i}}=\frac{0.368(2)(36)}{85(0.226)-14.4}=5.52\)

 

Six bolts should be used to provide the specified load factor.
(c) With six bolts, the load factor actually realized is

 

\(n_{L}=\frac{85(0.226)-14.4}{0.368(36 / 6)}=2.18\)

 

From Eq. (8–28), the yielding factor of safety is

 

\(n_{p}=\frac{S_{p} A_{t}}{C P+F_{i}}\) (8–28)

 

\(n_{p}=\frac{S_{p} A_{t}}{C\left(P_{\text {total }} / N\right)+F_{i}}=\frac{85(0.226)}{0.368(36 / 6)+14.4}=1.16\)

 

From Eq. (8–30), the load factor guarding against joint separation is

 

\(n_{0}=\frac{F_{i}}{P(1-C)}\) (8–30)

 

\(n_{0}=\frac{F_{i}}{\left(P_{\text {total }} / N\right)(1-C)}=\frac{14.4}{(36 / 6)(1-0.368)}=3.80\)
