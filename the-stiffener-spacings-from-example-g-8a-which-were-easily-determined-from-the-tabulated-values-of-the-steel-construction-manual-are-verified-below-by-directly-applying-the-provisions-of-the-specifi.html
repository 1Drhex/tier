Guest Question.

The stiffener spacings from , which were easily determined from the tabulated values of the Steel Construction Manual, are verified below by directly applying the provisions of the Specification.

Admin
Solution

Verify the shear strength of end panel

Section G2.1b

\(k_{v}=5+\frac{5}{(a / h)^{2}}=5+\frac{5}{(1.80)^{2}}=6.54\)

 

\(h / t_{w}=33.0 \text { in. } / 0.313 \text { in. }=105\)

 

Check a/h limits

\(a / h=\frac{59 \text { in. }}{33 \text { in. }}=1.80 \leq 3.0\)

 

\(a / h=1.80<\left[\frac{260}{\left(h / t_{w}\right)}\right]^{2}=\left[\frac{260}{105}\right]^{2}=6.13\)

 

Therefore, use \(k_{v}\) = 6.54.

Tension field action is not allowed since the panel is an end panel.

Since \(h / t_{w}>1.37 \sqrt{k_{v} E / F_{y}}=1.37 \sqrt{(6.54)(29,000 ksi ) /(36 ksi )}=99.5\)

 

\(C_{v}=\frac{1.51 E k_{v}}{\left(h / t_{w}\right)^{2} F_{y}}=\frac{1.51(29,000 ksi )(6.54)}{(105)^{2} 36 ksi }=0.721\)

Eqn. G2-5

\(V_{n}=0.6 F_{y} A_{w} C_{v}=0.6(36 ksi )\left(11.3 in .^{2}\right)(0.721)=176 kips\)

Eqn. G2-1

LRFD ASD
\(\phi_{v}=0.90\)

\(\phi_{v} V_{n}=0.90(176 kips )=159 \text { kips }\)

159 kips > 154 kips o.k.

 \(\Omega_{v}=1.67\)

\(V_{n} / \Omega_{v}=(176 kips ) / 1.67=106 kips\)

106 kips > 102 kips o.k.

Section G1

Verify the shear strength of the second panel

a/h for the second panel was 3.0

\(k_{v}=5+\frac{5}{(a / h)^{2}}=5+\frac{5}{(3.0)^{2}}=5.56\)

Check a/h limits

\(a / h=\frac{99 \text { in. }}{33 \text { in. }}=3.00 \leq 3.0\)

Section G2.1

\(a / h=3.00<\left[\frac{260}{\left(h / t_{w}\right)}\right]^{2}=\left[\frac{260}{105}\right]^{2}=6.13\)

 

Therefore use \(k_{v}=\) = 5.56

 

Since \(h / t_{w}>1.37 \sqrt{k_{v} E / F_{y}}=1.37 \sqrt{(5.56)(29,000 ksi ) /(36 ksi )}=91.7\)

 

\(C_{v}=\frac{1.51 E k_{v}}{\left(h / t_{w}\right)^{2} F_{y}}=\frac{1.51(29,000 ksi )(5.56)}{(105)^{2}(36 ksi )}=0.613\)

Eqn. G2-5

Since \(h / t_{w}>1.10 \sqrt{k_{v} E / F_{y}}=1.10 \sqrt{5.56(29,000 ksi ) /(36 ksi )}=73.6\) , use Eqn. G3-2

 

\(V_{n}=0.6 F_{y} A_{w}\left[C_{v}+\frac{1-C_{v}}{1.15 \sqrt{1+(a / h)^{2}}}\right]\)

Eqn. G3-2

\(=0.6(36 ksi )\left(11.3 \text { in. }^{2}\right)\left[0.613+\frac{1-0.613}{1.15 \sqrt{1+(3.0)^{2}}}\right]\)

 

\(V_{n}=176 kips\)

Calculate the available shear strength

LRFD ASD
\(\phi_{v}=0.90\)

\(\phi_{v} V_{n}=0.90(176 kips )=158 \text { kips }\)

156 kips > 127 kips o.k.

 \(\Omega_{v}=1.67\)

\(V_{n} / \Omega_{v}=(176 kips ) / 1.67=105 kips\)

105 kips > 84.0 kips o.k.
