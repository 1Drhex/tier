Guest Question.

A pump is driven from an engine crank-shaft by the mechanism as shown in Fig. 8.20. The pump piston shown at F is 250 mm in diameter and the crank speed is 100 r.p.m. The dimensions of various links are as follows:
OA = 150 mm ; AB = 600 mm ; BC = 350 mm ; CD = 150 mm; and DE = 500 mm.
Determine for the position shown : 1. The velocity of the cross-head E, 2. The rubbing velocity of the pins A and B which are 50 mm diameter. 3. The torque required at the crank shaft to overcome a presure of 0.35 \(N / mm ^{2}\),and 4. The acceleration of the cross-head E.

Admin
Solution

Given : \(N_{ AO }=100 \text { r.p.m. or } \omega_{ AO }=2 \pi \times 100 / 60=10.47 rad / s \) ; OA = 150 mm = 0.15 m ; A B = 600 mm = 0.6 m ; BC = 350 mm = 0.35 m ; CD = 150 mm = 0.15 m ; DE = 500 mm = 0.5 m

We know that velocity of A with respect to O or velocity of A,

\(v_{ AO }=v_{ A }=\omega_{ AO } \times O A=10.47 \times 0.15=1.57 m / s\) …(Perpendicular to OA)

 

1. Velocity of the cross-head E

First of all, draw the space diagram, to some suitable scale, as shown in Fig. (a). Now the velocity diagram, as shown in Fig. (b), is drawn as discussed below:

1. Since O and C are fixed points, therefore these points are marked as one point in the velocity diagram. Now draw vector oa perpendicular to OA, to some suitable scale, to represent the velocity of A with respect ot O or the velocity of A, such that

\(\text { vector } o a=v_{ AO }=v_{ A }=1.57 m / s\)

2. From point a, draw vector ab perpendicular to A B to represent the velocity of B with respect to A (i.e. \(v_{ BA }\)), and from point c draw vector cb perpendicular to CB to represent the velocity of B with respect to C (i.e. \(v_{ BC }\)). The vectors ab and cb intersect at b.

By measurement, we find that

\(v_{ BA }=\text { vector } a b=1.65 m / s\)

and \(v_{ BC }=v_{ B }=\text { vector } cb =0.93 m / s\)

3. From point c, draw vector cd perpendicular to CD or vector cb to represent the velocity of D with respect to C or velocity of D, such that

vector cd : vector cb = CD: CB \(\text { or } \quad v_{ DC }: v_{ BC }=C D: C B\)

\(\therefore \) \(\frac{v_{ DC }}{v_{ BC }}=\frac{C D}{C B} \text { or } v_{ DC }=v_{ BC } \times \frac{C D}{C B}=0.93 \times \frac{0.15}{0.35}=0.4 m / s\)

4. From point d, draw vector de perpendicular to DE to represent the velocity of E with respect to D (i.e. \(v_{ ED }\)), and from point o draw vector oe parallel to the path of motion of E (which is vertical) to represent the velocity of E or F. The vectors oe and de intersect at e.

By measurement, we find that velocity of E with respect to D,

\(v_{ ED }=\text { vector } d e=0.18 m / s\)

and velocity of the cross-head E,

\(v_{ EO }=v_{ E }=\text { vector } oe =0.36 m / s\)

 

2. Rubbing velocity of the pins at A and B

We know that angular velocity of A with respect to O,

\(\omega_{ AO }=10.47 rad / s\) …(Anticlockwise)

Angular velocity of B with respect to A,

\(\omega_{ BA }=\frac{v_{ BA }}{A B}=\frac{1.65}{0.6}=2.75 rad / s\) …(Anticlockwise)

and angular velocity of B with respect to C,

\(\omega_{ BC }=\frac{v_{ BC }}{C B}=\frac{0.93}{0.35}=2.66 rad / s\) …(clockwise)

We know that diameter of pins at A and B,

\(d_{ A }=d_{ B }=50 mm =0.05 m\) …(Given)

or Radius \(r_{ A }=r_{ B }=0.025 m\)

\(\therefore \) Rubbing velocity of pin at A

\(=\left(\omega_{ AO }-\omega_{ BA }\right) r_{ A }=(10.47-2.75) 0.025=0.193 m / s\)

and rubbing velocity of pin at B

\(=\left(\omega_{ BA }+\omega_{ BC }\right) r_{ B }=(2.75+2.66) 0.025=0.135 m / s\)

 

3. Torque required at the crankshaft

Given: Pressure to overcome by the crankshaft,

\(p_{ F }=0.35 N / mm ^{2}\)

Diameter of the pump piston

\(D_{ F }=250 mm\)

\(\therefore \) Force at the pump piston at F,

\(F_{ F }=\text { Pressure } \times \text { Area }=p_{ F } \times \frac{\pi}{4}\left(D_{ F }\right)^{2}=0.35 \times \frac{\pi}{4}(250)^{2}=17183 N\)

Let \(F_{ A }\) = Force required at the crankshaft at A.

Assuming transmission efficiency as 100 per cent,

Work done at A = Work done at F

\(F_{ A } \times v_{ A }=F_{ F } \times v_{ F } \text { or } F_{ A }=\frac{F_{ F } \times v_{ F }}{v_{ A }}=\frac{17183 \times 0.36}{1.57}=3940 N\) \(\ldots\left(\because v_{ F }=v_{ E }\right)\)

\(\therefore \) Torque required at the crankshaft,

\(T_{ A }=F_{ A } \times O A=3940 \times 0.15=591 N – m\)

Acceleration of the crosshead E

We know that the radial component of the acceleration of A with respect to O or the acceleration of A,

\(a_{ AO }^{r}=a_{ A }=\frac{v_{ AO }^{2}}{O A}=\frac{(1.57)^{2}}{0.15}=16.43 m / s ^{2}\)

Radial component of the acceleration of B with respect to A,

\(a_{ BA }^{r}=\frac{v_{ BA }^{2}}{A B}=\frac{(1.65)^{2}}{0.6}=4.54 m / s ^{2}\)

Radial component of the acceleration of B with respect to C.

\(a_{ BC }^{r}=\frac{v_{ BC }^{2}}{C B}=\frac{(0.93)^{2}}{0.35}=2.47 m / s ^{2}\)

and radial component of the acceleration of E with respect to D,

\(a_{ ED }^{r}=\frac{v_{ ED }^{2}}{D E}=\frac{(0.18)^{2}}{0.5}=0.065 m / s ^{2}\)

Now the acceleration diagram, as shown in Fig.(c), is drawn as discussed below:

1. Since O and C are fixed points, therefore these points are marked as one point in the acceleration diagram. Draw vector \(o^{\prime} a^{\prime}\) parallel to OA, to some suitable scale, to represent the radial component of the acceleration of A with respect to O or the acceleration of A, such that

\(\text { vector } o^{\prime} a^{\prime}=a_{ AO }^{r}=a_{ A }=16.43 m / s ^{2}\)

2. From point \(a^{\prime}\), draw vector \(a^{\prime}\)x parallel to A B to represent the radial component of the acceleration of B with respect to A (i.e. \(a_{ BA }^{r}\) ), such that

\(\text { vector } a^{\prime} x=a_{ BA }^{r}=4.54 m / s ^{2}\)

3. From point x, draw vector x\(b^{\prime}\) perpendicular to A B to represent the tangential component of the acceleration of B with respect to A (i.e. \(a_{ BA }^{t}\) ) whose magnitude is yet unknown.

4. Now from point \(c^{\prime}\), draw vector \(c^{\prime}\) y parallel to CB to represent the radial component of the acceleration of B with respect to C (i.e. \(a_{ BC }^{r}\) ), such that

\(\text { vector } c^{\prime} y=a_{ BC }^{r}=2.47 m / s ^{2}\)

5. From point y, draw vector \(y b^{\prime}\) perpendicular to CB to represent the tangential component of the acceleration of B with respect to C (i.e. \(a_{ BC }^{t}\) ). The vectors \(y b^{\prime} \text { and } x b^{\prime}\) intersect at \(b^{\prime} . \text { Join } c^{\prime} b^{\prime}\) and \(a^{\prime} b^{\prime}\). The vector \(c^{\prime} b^{\prime}\) represents the acceleration of B with respect to C (i.e. \(a_{ BC }\)) or the acceleration of B (i.e. \(a_{ B }\)) and vector \(a^{\prime} b^{\prime}\) represents the acceleration of B with respect to A (i.e. \(a_{ BA }\)).

By measurement, we find that

\(a_{ BC }=a_{ B }=\text { vector } c^{\prime} b^{\prime}=9.2 m / s ^{2}\)

and \(a_{ BA }=\text { vector } a^{\prime} b^{\prime}=9 m / s ^{2}\)

6. From point \(c^{\prime}\), draw vector \(c^{\prime} d^{\prime}\) perpendicular to CD or vector \(c^{\prime} b^{\prime}\) to represent the acceleration of D with respect to C or the acceleration of D (i.e. \(a_{ DC } \text { or } a_{ D }\)), such that

\(\text { vector } c^{\prime} d^{\prime}: \text { vector } c^{\prime} b^{\prime}=C D: C B \text { or } a_{ D }: a_{ BC }=C D: C B\)

\(\therefore \) \(\frac{a_{ D }}{a_{ BC }}=\frac{C D}{C B} \text { or } a_{ D }=a_{ BC } \times \frac{C D}{C B}=9.2 \times \frac{0.15}{0.35}=3.94 m / s ^{2}\)

7. Now from point \(d^{\prime}\), draw vector \(d^{\prime}\)z parallel to DE to represent the radial component of E with respect to D (i.e. \(a_{ ED }^{r}\) ), such that

\(\text { vector } d^{\prime} z=a_{ ED }^{r}=0.065 m / s ^{2}\)

8. From point z, draw vector \(z e^{\prime}\) perpendicular to DE to represent the tangential component of the acceleration of E with respect to D (i.e. \(a_{ ED }^{t}\) ) whose magnitude is yet unknown.

9. From point \(o^{\prime}\), draw vector \(o^{\prime} e^{\prime}\) parallel to the path of motion of E (which is vertical) to represent the acceleration of E. The vectors z\(e^{\prime}\) and \(o^{\prime} e^{\prime}\) intersect at \(e^{\prime}\).

By measurement, we find that acceleration of the crosshead E,

\(a_{ E }=\text { vector } o^{\prime} e^{\prime}=3.8 m / s ^{2}\)
