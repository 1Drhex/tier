Guest Question.

Two different dielectric slabs, with dielec tric constants of 2 and 6, respectively, are placed side-by-side between four rectangular PEC plates. Each dielectric slab is square (1 cm by 1 cm; area = \(1{ cm }^{ 2 })\) and each with a thickness of 1 cm. Each of the top andbottom PEC plates has dimensions of 1 cm by 1 cm, as shown in the figure. A 10-Volt d.c. power supply is placed between the twoplates forming two separate parallel-platecapacitors.For each capacitor, find the following:

(a) Electric field in each slab \(({ E }_{ 1 }\) and \({ E }_{ 2 }\))(in V/meter).

(b) Electric flux density in each slab (\({ D }_{ 1 }\) and \({ D }_{ 2 }\)) (in \(C/{ m }^{ 2 }\)).

(c) Total charge density in each of the twoPEC plates (\({ q }_{ 1 }\) and \({ q }_{ 2 }\)) (in \(C/{ m }^{ 2 }\)).

(d) Total charge in each of the two PECplates (\({ Q }_{ 1 }\) and \({ Q }_{ 2 }\)) (in C).

(e) Capacitance in each of the parallel-platecapacitors (\({ C }_{ 1 }\) and \({ C }_{ 2 }\)) (in farads) usingthe definition based on the charge andvoltage.

(f) Total capacitance \({ C }_{ T }\) (in farads) of theparallel-plate capacitor using the capac-itances from part (e).

(g) Capacitance of each capacitor (infarads) based on the definition of capac-itance for each (using plate area, sepa-ration and permittivity of the medium).

(h) Total capacitance \({ C }_{ T }\) (in farads) ofthe parallel plate capacitors, using thecapacitances of part (g).

(i) How do the capacitances of parts (f)and (h) compare? Are they the same ordifferent? Should they be the same ordifferent? Explain.

Solution

(a)

\({ E }_{ 1 }=\frac { 10 }{ 1cm } (\frac { 100cm }{ 1m } )={ 10 }^{ +3 }\frac { 10 }{ 1cm } V/m\\ { E }_{ 2 }=\frac { 10 }{ 1cm } (\frac { 100cm }{ 1m } )={ 10 }^{ +3 }\frac { 10 }{ 1cm } V/m\)

(b)

\({ D }_{ 1 }={ E }_{ 1 }\varepsilon _{ 1 }={ 10 }^{ +3 }(2\times 8.854\times { 10 }^{ -12 })17.708{ \times 10 }^{ -9 }C/{ m }^{ 2 }\\ { D }_{ 2 }={ E }_{ 2 }\varepsilon _{ 2 }={ 10 }^{ +3 }(6\times 8.854\times { 10 }^{ -12 })53.124\times { 10 }^{ -9 }C/{ m }^{ 2 }\)

(C)

\({ q }_{ 1 }=17.708{ \times 10 }^{ -9 }C/{ m }^{ 2 }\\ { q }_{ 2 }=53.124\times { 10 }^{ -9 }C/{ m }^{ 2 }\)

(d)

\(Q_{ 1 }={ q }_{ 1 }A=17.708{ \times 10 }^{ -9 }\frac { C }{ { m }^{ 2 } } (1\times 10^{ -4 }{ m }^{ 2 })=17.708{ \times 10 }^{ -13 }=1.7708{ \times 10 }^{ -12 }C\\ Q_{ 2 }={ q }_{ 2 }A=53.124{ \times 10 }^{ -9 }\frac { C }{ { \mu }^{ 2 } } (1\times 10^{ -4 }{ m }^{ 2 })=53.124{ \times 10 }^{ -13 }=5.3124{ \times 10 }^{ -12 }C\)

(e)

\({ C }_{ 1 }=\frac { { Q }_{ 1 } }{ { V }_{ 1 } } =\frac { 1.7708{ \times 10 }^{ -12 } }{ 10 } =1.7708{ \times 10 }^{ -13 }\) farads

\({ C }_{ 2 }=\frac { { Q }_{ 2 } }{ { V }_{ 2 } } =\frac { 5.3124{ \times 10 }^{ -12 } }{ 10 } =5.3124{ \times 10 }^{ -13 }\) farads

(f)

\(C_{ T }={ C }_{ 1 }+{ C }_{ 2 }=(1.7708+5.3124){ \times 10 }^{ -13 }=7.0832{ \times 10 }^{ -13 }=0.70832{ \times 10 }^{ -12 }\) farads

(g)

\({ C }_{ 1 }={ \varepsilon }_{ 1 }\frac { { A }_{ 1 } }{ { S }_{ 1 } } =2(8.854{ \times 10 }^{ -12 })\frac { { 10 }^{ -4 } }{ { 10 }^{ -2 } } =17.708{ \times 10 }^{ -14 }=1.7708{ \times 10 }^{ -13 }\) farads

\({ C }_{ 2 }={ \varepsilon }_{ 2 }\frac { { A }_{ 2 } }{ { S }_{ 2 } } =6(8.854{ \times 10 }^{ -12 })\frac { { 10 }^{ -4 } }{ { 10 }^{ -2 } } =53.124{ \times 10 }^{ -14 }=5.3124{ \times 10 }^{ -13 }\) farads

(h)

\({ C }_{ T }={ C }_{ 1 }+{ C }_{ 2 }=(1.7708+5.3124){ \times 10 }^{ -13 }=7.0824{ \times 10 }^{ -13 }=0.70824{ \times 10 }^{ -12 }\) farads

(i) They are the same as they should. Both are derived based on basic and related principles.
