Guest Question.

Synthesize graphically, a four-bar mechanism for three positions of input and output cranks with \( \psi_{12}=35^{\circ}, \phi_{23}=30^{\circ}, \psi_{12}=40^{\circ} \text {, and } \psi_{23}=60^{\circ} \text {, where } \phi_{i j} \text { and } \psi_{i j} \) are angular distances between position i and j of input and output links respectively.

Admin
Solution

Given: \( \phi_{12}=35^{\circ}, \phi_{23}=30^{\circ} ; \psi_{12}=40^{\circ}, \psi_{23}=60^{\circ} \).

∴ \( \phi_{13}=35+30=65^{\circ} \text { and } \psi_{13}=100^{\circ} \).

\( \alpha_{12}=\frac{1}{2}\left(\phi_{12}-\psi_{12}\right)=\frac{1}{2}(35-40)=-2.5^{\circ} \).

\( \alpha_{13}=\frac{1}{2}\left(\phi_{13}-\psi_{13}\right)=\frac{1}{2}(65-100)=-17.5^{\circ} \).

Angle subtended by the coupler at the relative pole=angle subtended by the fixed link at the relative pole.
The four-bar mechanism with three positions may be synthesized as explained below (Fig.16.31):

\( \text { 1. Select relative poles } R_{12} \text { and } R_{13} \text { at positions arbitrarily chosen. Draw } \alpha_{12}=2.5^{\circ} \text { at } R_{12} \text { and } \angle \alpha_{13}= 17.5^{\circ} \text { at } R_{13}\).

\( \text { 2. Let } x_{2} \text { and } y_{2} \text { lines intersect at } A \text { and } x_{1} \text { and } y_{1} \text { lines at } B \text {. Join } A B \text { to get the coupler. } \).

\( \text { 3. Select the fixed point } O_{2} \text { of the fixed link } O_{2} O_{4} \text { at an arbitrary position. Join } O_{2} R_{12} \text { and } O_{2} R_{13} \).

\( \text { 4. Draw } \angle O_{2} R_{12} Z_{1}=\angle \alpha_{12}=2.5^{\circ} \text { and } \angle O_{2} R_{13} Z_{2}=\angle \alpha_{13}=17.5^{\circ} \text {. Let } R_{12} Z_{1} \text { and } R_{13} Z_{2} \text { lines intersect }\text { at } O_{4}\).

\( \text { 5. Join } O_{2} A, O_{2} O_{4} \text { and } O_{4} B \text {. Then } O_{2} A B O_{4} \text { is the required four-bar mechanism shown in Fig. 16.31 } \).

\( b=\frac{1}{2}\left(x_{f}-x_{i}\right)=\frac{1}{2}(90-0)=45^{\circ} \).

\( x_{m}=45+45 \cos \left[\frac{(2 m-1) \pi}{6}\right] \).

\( x_{1}=45+45 \cos \frac{\pi}{6}=83.97^{\circ}, y_{1}=0.994 \).

\( x_{2}=45+45 \cos \frac{\pi}{2}=45^{\circ}, y_{2}=0.707 \).

\( x_{3}=45+45 \cos \frac{5 \pi}{6}=6.03^{\circ}, y_{3}=0.105 \).

\( \text { Given } \Delta \theta=120^{\circ}, \Delta \phi=60^{\circ} \).

Scale factors are:

\( r_{x}=\frac{\Delta \theta}{x_{f}-x_{i}}=\frac{120}{90-0}=\frac{120}{90}=\frac{4}{3} \).

\( y_{i}=\sin 0^{\circ}=0, y_{f}=\sin 90^{\circ}=1 \).

\( r_{y}=\frac{\Delta \phi}{y_{f}-y_{i}}=\frac{60}{1-0}=60 \).

Now \( r_{x}=\frac{\theta_{m}-\theta_{i}}{x_{m}-x_{i}} \).

∴ \( \frac{\theta_{1}-105}{83.97-0}=\frac{4}{3} \).

\( \theta_{1}=216.96^{\circ} \).

\( \frac{\theta_{2}-105}{45-0}=\frac{4}{3}, \theta_{2}=165^{\circ} \).

\( \frac{\theta_{3}-105}{6.03-0}=\frac{4}{3}, \theta_{3}=113.04^{\circ} \).

\( r_{y}=\frac{\phi_{m}-\phi_{i}}{y_{m}-y_{i}}\).

\( \frac{\phi_{1}-60}{0.994-0}=60, \phi_{1}=119.64^{\circ} \).

\( \frac{\phi_{2}-60}{0.707-0}=60, \phi_{2}=102.42^{\circ} \).
