Guest Question.

Magnetostatics treats the “source current” (the one that sets up the field) and the “recipient current” (the one that experiences the force) so asymmetrically that it is by no means obvious that the magnetic force between two current loops is consistent with Newton’s third law. Show, starting with the Biot-Savart law (Eq. 5.34) and the Lorentz force law (Eq. 5.16), that the force on loop 2 due to loop 1 (Fig. 5.61) can be written as

\(F _{2}=-\frac{\mu_{0}}{4 \pi} I_{1} I_{2} \oint \oint \frac{\hat{ ᴫ }}{ᴫ^{2}} d l _{1} \cdot d l _{2}\) (5.91)

\(B ( r )=\frac{\mu_{0}}{4 \pi} \int \frac{ I \times \hat{ᴫ}}{ᴫ^{2}} d l^{\prime}=\frac{\mu_{0}}{4 \pi} I \int \frac{d l ^{\prime} \times \hat{ ᴫ }}{ᴫ^{2}}\) (5.34)

\(F _{ mag }=\int I(d l \times B )\) (5.16)

In this form, it is clear that \(F _{2}=- F _{1}, \text { since } \hat{ ᴫ } \) changes direction when the roles of 1 and 2 are interchanged. (If you seem to be getting an “extra” term, it will help to note tha \(d l _{2} \cdot \hat{ᴫ}=d ᴫ\) .)

Admin
Solution

From the Biot-Savart law, the field of loop \(\# 1 \text { is } B =\frac{\mu_{0} I_{1}}{4 \pi} \oint_{1} \frac{d l _{1} \times \hat{ ᴫ }}{ᴫ^{2}}\) ; the force on loop #2 is

\(F =I_{2} \oint_{2} d l _{2} \times B =\frac{\mu_{0}}{4 \pi} I_{1} I_{2} \oint_{1} \oint_{2} \frac{d l _{2} \times\left(d l _{1} \times \hat{ ᴫ }\right)}{ᴫ^{2}}\) . Now \(d l _{2} \times\left(d l _{1} \times \hat{ ᴫ }\right)=d l _{1}\left(d l _{2} \cdot \hat{ ᴫ }\right)-\hat{ ᴫ }\left(d l _{1} \cdot d l _{2}\right)\) , so

\(F =-\frac{\mu_{0}}{4 \pi} I_{1} I_{2}\left\{\oint \oint \frac{\hat{ ᴫ }}{ᴫ^{2}}\left(d l _{1} \cdot d l _{2}\right)-\oint d l _{1} \oint \frac{\left(d l _{2} \cdot \hat{ ᴫ }\right)}{ ᴫ ^{2}}\right\}\)

The first term is what we want. It remains to show that the second term is zero:

\(ᴫ =\left(x_{2}-x_{1}\right) \hat{ x }+\left(y_{2}-y_{1}\right) \hat{ y }+\left(z_{2}-z_{1}\right) \hat{ z }, \text { so } \nabla_{2}(1 / ᴫ)=\frac{\partial}{\partial x_{2}}\left[\left(x_{2}-x_{1}\right)^{2}+\left(y_{2}-y_{1}\right)^{2}+\left(z_{2}-z_{1}\right)^{2}\right]^{-1 / 2} \hat{ x }\)

 

\(+\frac{\partial}{\partial y_{2}}\left[\left(x_{2}-x_{1}\right)^{2}+\left(y_{2}-y_{1}\right)^{2}+\left(z_{2}-z_{1}\right)^{2}\right]^{-1 / 2} \hat{ y }+\frac{\partial}{\partial z_{2}}\left[\left(x_{2}-x_{1}\right)^{2}+\left(y_{2}-y_{1}\right)^{2}+\left(z_{2}-z_{1}\right)^{2}\right]^{-1 / 2} \hat{ z }\)

 

\(=-\frac{\left(x_{2}-x_{1}\right)}{ᴫ^{3}} \hat{ x }-\frac{\left(y_{2}-y_{1}\right)}{ᴫ^{3}} \hat{ y }-\frac{\left(z_{2}-z_{1}\right)}{ᴫ^{3}} \hat{ z }=-\frac{ ᴫ }{ᴫ^{3}}=-\frac{\hat{ ᴫ }}{ᴫ^{2}} . \text { So } \oint \frac{\hat{ ᴫ }}{ᴫ^{2}} \cdot d l _{2}=-\oint \nabla_{2}\left(\frac{1}{ᴫ}\right) \cdot d l _{2}=0 \text { (by }\) Corollary 2 in Sect. 1.3.3). qed
