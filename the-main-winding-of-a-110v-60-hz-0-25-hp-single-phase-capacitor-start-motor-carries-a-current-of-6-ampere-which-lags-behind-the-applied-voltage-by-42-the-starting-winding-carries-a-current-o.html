Guest Question.

The main winding of a 110V, 60 Hz, 0.25 hp, single phase capacitor start motor carries a current of 6-ampere which lags behind the applied voltage by 42°. The starting winding carries a current of 4A which leads the voltage vector by 40°. Calculate (i) the total starting current and the power factor. (ii) the phase angle between the main winding current and starting winding current.

Admin
Solution

Here, \(I_{m} = 6 A; \phi _{m} = 42^{\circ } lagging; I_{s} = 4 A; \phi _{s} = 40^{\circ } leading \\[0.5cm] V = 110 V; f = 60 Hz; 0.25 HP\)

Considering supply voltage as a reference phasor

\(\overline{V} = V \angle 0^{\circ } = 110 \angle 0^{\circ } = \left(110 – j0\right) V \)

 

Main winding current, \(\overline{I}_{m} = 6 \angle -42^{\circ } A = \left(4.6 – j3.86\right) A \)

Starting winding current, \(\overline{I}_{s} = 4 \angle 40^{\circ } A = \left(2.97 + j2.68\right) A \)

Total starting current, \(\overline{I} = \overline{I}_{m} + \overline{I}_{s} = \left(4.6 – j3.86\right) + \left(2.97 + j2.68\right) \\[0.5cm] \hspace{30 pt} \hspace{30 pt} \qquad \quad \quad = \left(7.57 – j 1.18\right) A = 7.66\angle -8.9^{\circ} A\)

Starting current, \(I = \mathbf{7.66 A} \)

Power factor, \(\cos \phi = \cos 8.9^{\circ } = 0.988 \mathbf{lagging} \)

Phase difference between \(I_{m}\) and \(I_{s}\), \(\theta = \phi_{s} – \left(- \phi _{m}\right) = 40^{\circ } – \left(-42^{\circ }\right) = \mathbf{82^{\circ}} \)
