Guest Question.

The bar shown in Fig. 6–16 is subjected to a repeated moment \(0 \leq M \leq 1200 lbf \cdot \text { in. }\).The bar is AISI 4430 steel with \(S_{u t}=185 kpsi , S_{y}=170 kpsi \text { , and } K_{ Ic }=73 kpsi \sqrt{\text { in. }}\).Material tests on various specimens of this material with identical heat treatment indicate worst-case constants of \(C=3.8\left(10^{-11}\right)(\text { in } / \text { cycle }) /( kpsi \sqrt{\text { in }})^{m} \text { and } m=3.0 .\).As shown, a nick of size 0.004 in has been discovered on the bottom of the bar.Estimate the number of cycles of life remaining.

Admin
Solution

The stress range \(\Delta \sigma\) is always computed by using the nominal (uncracked) area. Thus

 

\(\frac{I}{c}=\frac{b h^{2}}{6}=\frac{0.25(0.5)^{2}}{6}=0.01042 in ^{3}\)

 

Therefore, before the crack initiates, the stress range is

 

\(\Delta \sigma=\frac{\Delta M}{I / c}=\frac{1200}{0.01042}=115.2\left(10^{3}\right) psi =115.2 kpsi\)

 

which is below the yield strength. As the crack grows, it will eventually become long enough such that the bar will completely yield or undergo a brittle fracture. For the ratio of \(S_{y} / S_{u t}\) it is highly unlikely that the bar will reach complete yield. For brittle fracture, designate the crack length as \(a_{f}\). If \(\beta=1\), then from Eq. (5–37) with \(K_{ I }=K_{ I c}\), we approximate \(a_{f}\) as

 

\(K_{I}=\beta \sigma \sqrt{\pi a}\) (5–37)

 

\(a_{f}=\frac{1}{\pi}\left(\frac{K_{ Ic }}{\beta \sigma_{\max }}\right)^{2} \approx \frac{1}{\pi}\left(\frac{73}{115.2}\right)^{2}=0.1278 \text { in }\)

 

From Fig. 5–27, we compute the ratio \(a_{f} / h\) as

 

\(\frac{a_{f}}{h}=\frac{0.1278}{0.5}=0.256\)

 

Thus a \(a_{f} / h\) varies from near zero to approximately 0.256. From Fig. 5–27, for this range \(\beta\) is nearly constant at approximately 1.07. We will assume it to be so, and re-evaluate \(a_{f}\) as

 

\(a_{f}=\frac{1}{\pi}\left(\frac{73}{1.07(115.2)}\right)^{2}=0.112 \text { in }\)

 

Thus, from Eq. (6–6), the estimated remaining life is

 

\(\int_{0}^{N_{f}} d N=N_{f}=\frac{1}{C} \int_{a_{i}}^{a_{f}} \frac{d a}{(\beta \Delta \sigma \sqrt{\pi a})^{m}}\) (6–6)

 

\(N_{f}=\frac{1}{C} \int_{a_{i}}^{a_{f}} \frac{d a}{(\beta \Delta \sigma \sqrt{\pi a})^{m}}=\frac{1}{3.8\left(10^{-11}\right)} \int_{0.004}^{0.112} \frac{d a}{[1.07(115.2) \sqrt{\pi a}]^{3}}\)

 

\(=-\left.\frac{5.047\left(10^{3}\right)}{\sqrt{a}}\right|_{0.004} ^{0.112}=64.7\left(10^{3}\right) \text { cycles }\)
