Guest Question.

In an application, the piecewise-continuous function

\(D(\omega)=\left\{\begin{array}{cl}2 \omega & \text { for } \quad 0 \leq \omega<6 \\12 & \text { for } 6 \leq \omega<12 \\-\omega+24 & \text { for } 12 \leq \omega<16 \\8 & \text { for } 16 \leq \omega<22\end{array}\right.\)

(see Fig. 16.3) has to be approximated by a polynomial of the form

\(P(\omega)=\sum_{k=0}^{5} a_{k} \omega^{k}\)

Using Algorithm 5, obtain a set of coefficients \(a_{k} \text { for } k=0,1, \ldots, 5\) that minimizes the difference between D(ω) and P(ω) in the range \(0 \leq \omega \leq 22\) in a least-squares sense.

Admin
Solution

A suitable objective function can be constructed as

 

\(\Psi( x )=\frac{1}{2} L_{2}^{2}=\frac{1}{2} \sum_{i=1}^{12}\left[D\left(\omega_{i}\right)-P\left(\omega_{i}\right)\right]^{2}\)

 

where \(\omega_{i}=2 i-2\) and

 

\(x =\left[\begin{array}{llll}x_{1} & x_{2} & \cdots & x_{6}\end{array}\right]^{T}=\left[\begin{array}{llll}a_{0} & a_{1} & \cdots & a_{5}\end{array}\right]^{T}\)

 

by sampling the error D(ω)− P(ω) at 12 points. The first partial derivatives of \(\Psi( x )\) can be readily determined as

 

\(\frac{\partial \Psi( x )}{\partial x_{k}}=-\sum_{i=1}^{12}\left[D\left(\omega_{i}\right)-P\left(\omega_{i}\right)\right] \omega_{i}^{k}\)

 

for k = 1, 2, . . . , 6. Using Algorithm 5 with an initial point \(x _{0}=\left[\begin{array}{llll}0 & 0 & \ldots & 0\end{array}\right]^{T}\) and a termination tolerance \(\varepsilon_{1}=10^{-6}\), the coefficients in Table 16.1 were obtained. The progress of the algorithm is illustrated in Table 16.2. The number of function evaluations is equal to the number of evaluations of the objective function \(\Psi( x )\) plus the number of evaluations of the partial derivative function \(\partial \Psi( x ) / \partial x_{k}\). The polynomial P(ω) is compared with D(ω) in Fig. 16.3 and the error between the two is plotted versus frequency
