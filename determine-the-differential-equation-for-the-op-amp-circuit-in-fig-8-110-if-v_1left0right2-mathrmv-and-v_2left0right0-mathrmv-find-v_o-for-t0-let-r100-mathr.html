Guest Question.

Determine the differential equation for the op amp circuit in Fig. 8.110 . If \(v_{1}\left(0^{+}\right)=2 \mathrm{V}\) and \(v_{2}\left(0^{+}\right)=0 \mathrm{V}\) find \(v_{o}\) for t>0 . Let R=100 \(\mathrm{k} \Omega\) and C=1 \(\mu \mathrm{F}\)

Admin
Solution

At the input of the first op amp,

(\(v_{o}\) – 0)/R = Cd(\(v_{1}\) – 0)/dt (1)

At the input of the second op amp,

(-\(v_{1}\) – 0)/R = Cd\(v_{2}\)/dt (2)

Let us now examine our constraints. Since the input terminals are essentially at ground, then we have the following,

\(v_{o}\) = –\(v_{2}\) or \(v_{2}\) = –\(v_{o}\) (3)

Combining (1), (2), and (3), eliminating v1 and v2 we get,

\(\frac{\mathrm{d}^{2} \mathrm{v}_{\mathrm{o}}}{\mathrm{dt}^{2}}-\left(\frac{1}{\mathrm{R}^{2} \mathrm{C}^{2}}\right) \mathrm{v}_{\mathrm{o}}=\frac{\mathrm{d}^{2} \mathrm{v}_{\mathrm{o}}}{\mathrm{dt}^{2}}-100 \mathrm{v}_{\mathrm{o}}=0\)

Which leads to \(\mathrm{s}^{2}-100=0\)
Clearly this produces roots of -10 and +10 .
And, we obtain,

\(\begin{array}{l}\mathrm{v}_{\mathrm{o}}(\mathrm{t})=\left(\mathrm{Ae}^{+10 \mathrm{t}}+\mathrm{Be}^{-10 \mathrm{t}}\right) \mathrm{V} \\\\\text { At } \mathrm{t}=0, \mathrm{v}_{\mathrm{o}}(0+)=-\mathrm{v}_{2}(0+)=0=\mathrm{A}+\mathrm{B}, \text { thus } \mathrm{B}=-\mathrm{A}\end{array}\)

This leads to \(\mathrm{v}_{\mathrm{o}}(\mathrm{t})=\left(\mathrm{Ae}^{+10 \mathrm{t}}-\mathrm{Ae}^{-10 \mathrm{t}}\right) \mathrm{V} .\) Now we can use \(\mathrm{v}_{1}(0+)=2 \mathrm{V}\)

\(\text { From }(2), \mathrm{v}_{1}=-\mathrm{R} \mathrm{Cdv}_{2} / \mathrm{dt}=0.1 \mathrm{dv}_{\mathrm{o}} / \mathrm{dt}=0.1\left(10 \mathrm{Ae}^{+10 \mathrm{t}}+10 \mathrm{Ae}^{-10 t}\right)\) \(\begin{array}{l}\mathrm{v}_{1}(0+)=2=0.1(20 \mathrm{A})=2 \mathrm{A} \text { or } \mathrm{A}=1 \\\\\text { Thus, } \mathrm{v}_{\mathrm{o}}(\mathrm{t})={\left(\mathrm{e}^{+10 \mathrm{t}}-\mathrm{e}^{-10 t}\right) \mathrm{V}}\end{array}\)

It should be noted that this circuit is unstable (clearly one of the poles lies in the right_half-plane).
