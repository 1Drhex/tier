Guest Question.

The elementary liquid-phase reaction

\(\mathrm{A}+2 \mathrm{~B} \stackrel{k_{1}}{\stackrel{\rightleftarrows}{k_{2}}} \mathrm{R}\)

with rate equation

\(-r_{\mathrm{A}}=-\frac{1}{2} r_{\mathrm{B}}=\left(12.5 \text { liter }^{2} / \mathrm{mol}^{2} \cdot \min \right) C_{\mathrm{A}} C_{\mathrm{B}}^{2}-\left(1.5 \mathrm{~min}^{-1}\right) C_{\mathrm{R}}, \quad\left[\frac{\mathrm{mol}}{\text { liter } \cdot \mathrm{min}}\right]\)

is to take place in a 6-liter steady-state mixed flow reactor. Two feed streams, one containing \(2.8 \mathrm{~mol} \mathrm{~A} /\) liter and the other containing \(1.6 \mathrm{~mol} \mathrm{~B} /\) liter, are to be introduced at equal volumetric flow rates into the reactor, and \(75 \%\) conversion of limiting component is desired (see Fig. E5.3). What should be the flow rate of each stream? Assume a constant density throughout.

Admin
Solution

The concentration of components in the mixed feed stream is

\(\begin{array}{l}C_{\mathrm{A} 0}=1.4 \mathrm{~mol} / \text { liter } \\C_{\mathrm{B} 0}=0.8 \mathrm{~mol} / \text { liter } \\C_{\mathrm{R} 0}=0\end{array}\)

These numbers show that \(\mathrm{B}\) is the limiting component, so for \(75 \%\) conversion of \(\mathrm{B}\) and \(\varepsilon=0,\) the composition in the reactor and in the exit stream is

\(\begin{aligned}C_{\mathrm{A}} &=1.4-0.6 / 2=1.1 \mathrm{~mol} / \mathrm{liter} \\C_{\mathrm{B}} &=0.8-0.6=0.2 \mathrm{~mol} / \text { liter or } 75 \% \text { conversion } \\C_{\mathrm{R}} &=0.3 \mathrm{~mol} / \mathrm{liter}\end{aligned}\)

Writing the rate and solving the problem in terms of B we have at the conditions within the reactor

\(\begin{aligned}-r_{\mathrm{B}} &=2\left(-r_{\mathrm{A}}\right)=(2 \times 12.5) C_{\mathrm{A}} C_{\mathrm{B}}^{2}-(2 \times 1.5) C_{\mathrm{R}} \\&=\left(25 \frac{\text { liter }^{2}}{\mathrm{~mol}^{2} \cdot \mathrm{min}}\right)\left(1.1 \frac{\mathrm{mol}}{\text { liter }}\right)\left(0.2 \frac{\mathrm{mol}}{\text { liter }}\right)^{2}-\left(3 \mathrm{~min}^{-1}\right)\left(0.3 \frac{\mathrm{mol}}{\text { liter }}\right) \\&=(1.1-0.9) \frac{\mathrm{mol}}{\text { liter } \cdot \min }=0.2 \frac{\mathrm{mol}}{\text { liter } \cdot \mathrm{min}}\end{aligned}\)

For no density change, the performance equation of Eq. 13 gives

\(\tau=\frac{V}{v}=\frac{C_{\mathrm{B} 0}-C_{\mathrm{B}}}{-r_{\mathrm{B}}}\)

Hence the volumentric flow rate into and out of the reactor is

\(\begin{array}{l}v=\frac{V\left(-r_{\mathrm{B}}\right)}{C_{\mathrm{B} 0-} C_{\mathrm{B}}} \\\underline{v}=\frac{(6 \text { liter })(0.2 \mathrm{~mol} / \mathrm{liter} \cdot \mathrm{min})}{(0.8-0.6) \mathrm{mol} / \text { liter }}=\underline{\underline{6} \text { liter } / \mathrm{min}}\end{array}\)

or 3 liter/min of each of the two feed streams
