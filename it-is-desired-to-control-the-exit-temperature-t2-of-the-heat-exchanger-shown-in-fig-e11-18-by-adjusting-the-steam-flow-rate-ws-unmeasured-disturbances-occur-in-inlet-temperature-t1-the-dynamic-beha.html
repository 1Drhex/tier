Guest Question.

It is desired to control the exit temperature \(T_{2}\) of the heat exchanger shown in Fig. E11.18 by adjusting the steam flow rate \(w_{s} .\) Unmeasured disturbances occur in inlet temperature \(T_{1} .\) The dynamic behavior of the heat exchanger can be approximated by the transfer functions

\(\begin{aligned}&\frac{T_{2}^{\prime}(s)}{W_{s}^{\prime}(s)}=\frac{2.5 e^{-s}}{10 s+1}[=] \frac{{ }^{\circ} F }{ lb / s } \\&\frac{T_{2}^{\prime}(s)}{T_{1}^{\prime}(s)}=\frac{0.9 e^{-2 s}}{5 s+1}[=] \text { dimensionless }\end{aligned}\)

where the time constants and time delays have units of seconds. The control valve has the following steady-state characteristic:

\(w_{s}=0.6 \sqrt{p-4}\)

where \(p\) is the controller output expressed in \(mA\). At the nominal operating condition, \(p=12 mA\). After a sudden change in the controller output, \(w_{s}\) reaches a new steady-state value in \(20 s\) (assumed to take five time constants). The temperature transmitter has negligible dynamics and is designed so that its output signal varies linearly from 4 to \(20 mA\) as \(T_{2}\) varies from 120 to \(160{ }^{\circ} F\).

(a) If a proportional-only feedback controller is used, what is \(K_{c m}\) ? What is the frequency of the resulting oscillation when \(K_{c}=K_{c m} ?\) (Hint: Use the direct-substitution method and Euler’s identity.)

(b) Estimate \(K_{c m}\) using direct substitution and a \(1 / 1\) Padé approximation for the time-delay term. Does this analysis provide a satisfactory approximation?

Admin
Solution

\(\begin{aligned}&G_{c}(s)=K_{c} \\&G_{V}(s)=\frac{K_{V}}{\tau_{V} s+1} \\&K_{v}=\left.\frac{d w_{s}}{d p}\right|_{p=12}=\frac{0.6}{2 \sqrt{12-4}}=0.106 \frac{1 bm / s }{ ma } \\&5 \tau_{v}=20 s \quad \tau_{v}=4 s \\&G_{p}(s)=\frac{2.5 e^{-s}}{10 s+1} \\&G_{m}(s)=K_{m}=\frac{(20-4) ma }{(160-120)^{\circ} F}=0.4 \frac{ ma }{{ }^{\circ} F}\end{aligned}\)

Characteristic equation is

\(1+\left(K_{c}\right)\left(\frac{0.106}{4 s+1}\right)\left(\frac{2.5 e^{-s}}{10 s+1}\right)(0.4)=0\) (1)

a) Substituting \(s= j \omega\) in (1) and using Euler’s identity

\(e ^{- j \omega}=\cos \omega- j \sin \omega\)

gives

\(-40 \omega^{2}+14 j \omega+1+0.106 K_{c}(\cos \omega- j \sin \omega)=0\)

Thus

\(-40 \omega^{2}+1+0.106 K_{c} \cos \omega=0\) (2)

and

\(14 \omega-0.106 K_{c} \sin \omega=0\) (3)

From (2) and (3),

\(\tan \omega=\frac{14 \omega}{40 \omega^{2}-1}\) (4)

Solving (4), \(\omega=0.579\) by trial and error.

Substituting for \(\omega\) in (3) gives

\(K_{c}=139.7=K_{c m}\)

Frequency of oscillation is \(0.579 rad / sec\)

b) Substituting the Pade approximation into (1) gives:

\(\begin{gathered}e^{-s} \approx \frac{1-0.5 s}{1+0.5 s} \\20 s^{3}+47 s^{2}+\left(14.5-0.053 K_{c}\right) s+\left(1+0.106 K_{c}\right)=0\end{gathered}\)

Substituting \(s= j \omega\) in above equation, we have:

\(-47 \omega^{2}+1+0.106 K_{c}+\left[-20 \omega^{3}+14.5 \omega-0.053 K_{c} \omega\right] j=0\)

Thus, we have:

\(\left\{\begin{array} { c }{ – 4 7 \omega ^ { 2 } + 1 + 0 . 1 0 6 \mathrm { K } _ { c } = 0 } \\{ – 2 0 \omega ^ { 3 } + 1 4 . 5 \omega – 0 . 0 5 3 K _ { c } \omega = 0 }\end{array} \Rightarrow \left\{\begin{array}{c}\omega=0.587 \\K_{c}=143.46\end{array}\right.\right.\)

Therefore, the maximum gain, \(K_{c}=143.46\), is a satisfactory approximation of the true value of \(139.7\) in (a) above
