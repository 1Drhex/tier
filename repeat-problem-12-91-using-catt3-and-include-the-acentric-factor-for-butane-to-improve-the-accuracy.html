Guest Question.

Repeat using CATT3 and include the acentric factor for butane to improve the accuracy.

Admin
Solution

\(C _{4} H _{10}\) cycle

\(\begin{aligned}& T _{1}=80{ }^{\circ} C , x _{1}=1.0 ; T _{3}=30^{\circ} C , x _{3}=0.0 \\& T _{ r1 }=\frac{353.2}{425.2}=0.831\end{aligned}\)

From CATT3 with ω = 0.199:

\(\begin{aligned}P _{1} &=0.2646 \times 3800=1005 kPa \\\left( h _{1}^{*}- h _{1}\right) &=0.1430 \times 425.2 \times 0.5685=34.6 \\\left( s _{1}^{*}- s _{1}\right) &=0.1430 \times 0.4996=0.0714\end{aligned}\)

 

\(T _{ r 3}=\frac{303.2}{425.2}=0.713\)

 

From CATT3 with ω = 0.199: \(P _{3}=0.07443 \times 3800=282.8 kPa\)

sat. liq.: \(\left( h ^{*}- h _{ f }\right)= R T _{ c } \times 6.048=367.74 ; \quad\left( s ^{*}- s _{ f }\right)= R \times 8.399=1.201\)

sat. vap.: \(\left( h ^{*}- h _{ g }\right)= RT _{ c } \times 0.202=12.28 ; \quad\left( s ^{*}- s _{ g }\right)= R \times 0.201=0.0287\)

Because of the combination of properties of \(C _{4} H _{10}\) (particularly the large \(C _{ P 0} / R\)), \(s _{1}\) is larger than \(s _{ g }\) at \(T _{3}\). To demonstrate,

\(\begin{aligned}&\left( s _{1}^{*}- s _{ g 3}^{*}\right)=1.716 \ln \frac{353.2}{303.2}-0.1430 \ln \frac{1005}{282.8}=0.0806 \\&\left( s _{1}- s _{ g 3}\right)=-0.0714+0.0806+0.0287=+0.0379 kJ / kg K\end{aligned}\)

so that \(T _{2 S }\) will be \(> T _{3}\), as shown in the T-s diagram. A number of other heavy hydrocarbons also exhibit this behavior.

Assume \(T _{2 S }=315 K , \quad \Rightarrow T _{ r 2 S }=315 / 425.2=0.741\)

From CATT3:

\(\begin{aligned}&\left( h _{2 S }^{*}- h _{2 S }\right)= R T _{ c } \times 0.183=11.13 \text { and } \quad\left( s _{2 S }^{*}- s _{2 S }\right)= R \times 0.1746=0.025 \\&\left( s _{1}^{*}- s _{2 S }^{*}\right)=1.716 \ln \frac{353.2}{315}-0.1430 \ln \frac{1005}{282.8}=+0.01509 \\&\left( s _{1}- s _{2 S }\right)=-0.0714+0.01509+0.025=-0.031\end{aligned}\)

Repeat at \(T _{2 S }=310 K\) to get \(T _{ r 2 S }=0.729\) ,

\(\begin{aligned}&\left( h _{2 S }^{*}- h _{2 S }\right)= RT _{ c } \times 0.1907=11.595 \text { and }\left( s _{2 S }^{*}- s _{2 S }\right)= R \times 0.1853=0.0265 \\&\left( s _{1}^{*}- s _{2 S }^{*}\right)=1.716 \ln \frac{353.2}{310}-0.1430 \ln \frac{1005}{282.8}=+0.04255 \\&\left( s _{1}- s _{2 S }\right)=-0.0714+0.04255+0.0265=-0.0023 \quad \text { very close to } 0, OK\end{aligned}\)

⇒ \(T_{2 S}=310 K\)

\(\begin{aligned}&\left(h_{1}^{*}-h_{2 S}^{*}\right)=1.716(353.2-310)=74.13 \\&w_{S T}=h_{1}-h_{2 S}=-34.6+74.13+11.6=51.1 kJ / kg\end{aligned}\)
