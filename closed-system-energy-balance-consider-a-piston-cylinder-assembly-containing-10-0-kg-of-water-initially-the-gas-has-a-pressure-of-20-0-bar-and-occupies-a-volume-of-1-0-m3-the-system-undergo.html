Closed System Energy Balance

Consider a piston–cylinder assembly containing 10.0 kg of water. Initially, the gas has a pressure of 20.0 bar and occupies a volume of 1.0 \(m ^{3}\). The system undergoes a reversible process in which it is compressed to 100 bar. The pressure volume relationship during this process is given by:

\(P v^{1.5}=\text { const }\)

(a) What is the initial temperature?

(b) Calculate the work done during this process.

(c) Calculate the heat transferred during this process.

(d) What is the fi nal temperature?

Admin
Solution

(a) We need two independent intensive properties to constrain the state of the system. Once values for these properties are determined, we can use the steam tables to find other properties. The specifi c volume of the initial state can be determined as follows:

 

\(\hat{v}=\frac{V}{m}=\frac{1.0}{10.0}=0.10\left[\frac{ m ^{3}}{ kg }\right]\)

 

The values in the steam tables are in units of Pa, so we convert 20 bar to 2 MPa. If we look at Table B.4 (Appendix B), we see that at P = 2 MPa to three signifi cant fi gures the volume at the saturation temperature is 0.100 \(\left[\frac{m^{3}}{k g}\right]\). Therefore, looking at Table B.2, we see the temperature is 212.4°C (or slightly above).

 

(b) To calculate the work it is useful to draw a schematic of the process. We defi ne the initial state as state 1 and the fi nal state as state 2, as shown in Figure E2.4.

 

Because this is a reversible process, we can write:

 

\(\hat{w}=-\int P_{E} d \hat{v}=-\int P d \hat{v}\)

 

To fi nd the upper limit on the integrand, we need to know the specifi c volume of the fi nal state, \(\hat{v}_{2}\). We can calculate \(\hat{v}_{2}\) from the equation in the problem statement:

 

\(P \hat{v}^{1.5}=\text { const }=P_{1} \hat{v}_{1}^{1.5}=P_{2} \hat{v}_{2}^{1.5}\)

 

Solving for \(\hat{v}_{2}\) gives:

 

\(\hat{v}_{2}=\left(\frac{P_{1} \hat{v}_{1}^{1.5}}{P_{2}}\right)^{1 / 1.5}=\left(0.2 \times 0.1^{1.5}\right)^{1 / .5}=0.0342\left[\frac{ m ^{3}}{ kg }\right]\)

 

Now we can solve for work:

 

\(\hat{w}=-\int_{0.1}^{0.0342} P d \hat{v}=-\int_{0.1}^{0.0342} \frac{P_{1} \hat{v}_{1}^{1.5}}{\hat{v}^{1.5}} d \hat{v}=2 P_{1} \hat{v}_{1}^{1.5}\left[\frac{1}{\hat{v}_{2}^{0.5}}-\frac{1}{\hat{v}_{1}^{0.5}}\right]_{0.1}^{0.0342}=284\left[\frac{ kJ }{ kg }\right]\)

 

The sign of work is positive because we are adding energy to the system during the compression. (c) To fi nd heat, we can apply the fi rst law:

 

\(q=\Delta u-w\)

 

Because we solved for work in part (b), we need only to determine the internal energy in states 2 and 1 from the steam tables. In part (a), we found that state 1 is approximately saturated vapor at 20 bar. Looking at Table B.2, we find:

 

\(u_{1}=2600.3\left[\frac{ kJ }{ kg }\right]\)

 

For \(u_{2}\), we look at Table B.4 at a pressure of 10 MPa (100 bar). In this case, we need to interpolate:

 

\(\frac{u_{2}-u\left(T=500^{\circ} C \right)}{u\left(T=550^{\circ} C \right)-u\left(T=500^{\circ} C \right)}=\frac{v_{2}-v\left(T=500^{\circ} C \right)}{v\left(T=550^{\circ} C \right)-v\left(T=500^{\circ} C \right)}\)

 

Solving for \(u_{2}\),

 

\(u_{2}=u\left(T=500^{\circ} C \right)+\left[u\left(T=550^{\circ} C \right)-u\left(T=500^{\circ} C \right)\right]\)

 

\(\left[\frac{v_{2}-v\left(T=500^{\circ} C \right)}{v\left(T=550^{\circ} C \right)-v\left(T=500^{\circ} C \right)}\right]\)

 

\(=3045.8+[3144.5-3045.8]\left[\frac{0.0342-0.03279}{0.03564-0.03279}\right]\)

 

\(=3094.6\left[\frac{ kJ }{ kg }\right]\)

 

Thus,

 

\(q=\Delta u-w=u_{2}-u_{1}-w=210\left[\frac{ kJ }{ kg }\right]\)

 

Since the value of q is positive, heat transfers from the system to the surroundings. (d) To find \(T_{2}\), we must also interpolate. From Table B.4,we get:

 

\(T_{2}=500^{\circ} C +\left[550^{\circ} C -500^{\circ} C \right]\left[\frac{v_{2}-v\left(T=500^{\circ} C \right)}{v\left(T=550^{\circ} C \right)-v\left(T=500^{\circ} C \right)}\right]\)

 

\(=500^{\circ} C +\left[50^{\circ} C \right]\left[\frac{0.0342-0.03279}{0.03564-0.03279}\right]\)

 

\(=525^{\circ} C\)
