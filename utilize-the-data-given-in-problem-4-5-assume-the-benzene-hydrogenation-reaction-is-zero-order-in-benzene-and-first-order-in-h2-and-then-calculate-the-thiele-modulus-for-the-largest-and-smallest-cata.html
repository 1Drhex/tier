Guest Question.

Utilize the data given in , assume the benzene hydrogenation reaction is zero order in benzene and first order in \(H_{2}\), and then calculate the Thiele modulus for the largest and smallest catalyst particles assuming all the Pd is in the mesopores. What is the Thiele modulus for the largest \(\left(500 \ \mu \right)\) particles if all the Pd were dispersed in only the micropores? Should there be any concern about pore diffusion limitations for any of the three possibilities? Why?

Admin
Solution

Thiele modulus \(\phi = \frac{R_{p} k^{1/2} C^{\frac{n-1}{2} }_{s} }{D^{1/2}_{eff} } \)

 

Rate \(= k^{'}{P}_{H_{2}}=kC_{H_2}= \left\lgroup\frac{1.99 \ \mu molw Bz}{s\cdot g} \right\rgroup \left(0.60 \ g/cm^{3} \right)= \frac{1.19 \times 10^{-6} mole \ Bz}{s \cdot cm^{3}} \)

 

\(C_{H_2}= \frac{n}{V} =\frac{P}{RT}=\frac{\left(710/760 \ atm\right) }{\left\lgroup\frac{82.06 \ cm^{3}\cdot atm}{g \ mole\cdot K} \right\rgroup \left(413K\right)} = 2.76 \times 10^{-5}\frac{mole}{cm^{3}} \)

 

\(k=\frac{Rate}{C_{H_2} }= \frac{1.19\times 10^{-6} \ mole/s\cdot cm^{3} }{2.76\times 10^{-6} \ mole/cm^{3}} = 0.0432 \ s^{-1} \)

 

\(D_{H_2}=1/3 \bar{v}_{H_2} \lambda \)

 

\(\bar{v}_{H_2} =\left[\frac{8\left(1.38\times 10^{-16} erg/K\right) \left(413K\right) }{\pi \left(2 \ amu\right) \left(1.66\times 10^{-24}g/amu\right) } \right] ^{1/2} = 2.1\times 10^{5} cm/s\)

 

\(\lambda =\frac{\left\lgroup\frac{82.06 \ cm^{3}\cdot atm } {gmole \cdot K} \right\rgroup \left(413K\right) \left\lgroup\frac{1 \ mole}{6.02 \times 10^{23} molecules} \right\rgroup } {\sqrt{2} \pi \left(2.4\times 10^{-8}cm \right) ^{2} \left(1 \ atm \ total \ P\right) } = 2.2\times 10^{-5}cm\)

 

\(D_{H_2}=1/3\left(2.1 \times 10^{5} cm/s \right) \left(2.2 \times 10^{-5} cm \right)=1.5 \ cm^{2}/s\)

 

\(D_{eff}\cong D_{K_n}=1/3 \bar{v} d_{p}\) where \(d_{p} \) =pore diameter

Case 1: Smallest particle, all Pd in mesopores:

\(D_{K_n}=1/3\left(2.1 \times 10^{5}cm/s\right)\left(2.5 \times 10^{-9}m\right)\left(10^{2}cm/m\right)=0.175 \ cm^{2}/s\)

 

\(\phi =\frac{\left(10\times 10^{-6}m\right)\left(10^{2}cm/m\right)\left(0.0432/s\right)^{1/2} }{\left(0.175 \ cm/s\right)^{1/2}} = 5.0\times 10^{-4}\)

 

Case 2: Largest particle, all Pd in mesopores:

\(\phi =\frac{\left(500\times 10^{-6}m\right)\left(10^{2}cm/m\right)\left(0.0432/s\right)^{1/2} }{\left(0.175 \ cm/s\right)^{1/2}} = 0.025\)

 

Case 3: Largest particle, all Pd in micropores:

\(D_{K_n}=1/3\left(2.1 \times 10^{5}cm/s\right)\left(0.9 \times 10^{-9}m\right)\left(10^{2}cm/m\right)=6.3\times 10^{-3}\ cm^{2}/s\)

 

\(\phi =\frac{\left(500\times 10^{-6}m\right)\left(10^{2}cm/m\right)\left(0.0432/s\right)^{1/2} }{\left(6.3\times 10^{-3}\ cm^{2}/s\right)^{1/2}} = 0.13\)

 

No concern – in all cases \(\eta \) is essentially unity.
