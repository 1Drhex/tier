Guest Question.

Determine the surface temperature of a cylindrical tank measuring 0.75 m in diameter and 1.2 m high. The tank contains a transformer immersed in an oil bath that produces a uniform surface temperature condition. All heat loss from the surface may be assumed to be due to natural convection to surrounding air at 295 K. The heat dissipation rate from the transformer is constant at 1.5 KW.

Admin
Solution

Surface areas that apply are

 

\( A_{top} = A_{bottom} = \frac{\pi }{4} \left ( 0.75 m \right )^{2} = 0.442 m^{2} \)

 

\( A_{side} = \pi \left ( 1.2 m \right ) \left ( 0.75 m \right ) = 2.83 m^{2} \)

 

The total heat transfer is the sum of the contributions from the three surfaces. This may be written as

\( q_{total} = \left [ h_{t} \left ( 0.442 \right ) + h_{b} \left ( 0.442 \right ) + h_{s} \left ( 2.83 \right ) \right ]\left ( T – 295 \right ) \)

 

where subscripts t (top), b (bottom), and s (sides) apply to the surfaces in question.
Rewriting this expression in terms of Nu, we have

 

\( q_{total} = \left [Nu_{t} \frac{k}{0.75} \left ( 0.442 \right ) + N u_{b} \frac{k}{0.75} \left ( 0.442 \right ) + N u_{s} \frac{k}{1.2} \left ( 2.83 \right ) \right ] \left ( T – 295 \right ) \)

 

or

\( q_{total} = \left [ 0.589 Nu_{t} + 0.589 N u_{b} + 2.36 N u_{s} \right ] k \left ( T – 295 \right ) \)

 

A complication exists in solving this equation, because the unknown quantity is surface temperature. The procedure to be used is trial and error, where, initially, a surface temperature value is assumed for property evaluation, and then solved for T. This new value for surface temperature will then be used and the procedure continued until the resulting temperature agrees with the value used in finding fluid properties.
To begin the problem, we assume that \( T_{surface} = 38.5 K, \) Properties will thus be evaluated at \( T_{f} = 340 K. \) For air at \( 340 K, \nu = 1.955 \times 10^{-5} m^{2}/s, k = 0.0293 W/m \cdot K, \alpha = 2.80 \times 10^{-5} m^{2} /s, Pr = 0.699, \) and \( \beta g/ \nu^{2} = 0.750 \times 10^{8} 1/K \cdot m^{3}. \)

For the vertical surface,

\( Gr = \frac{\beta g}{\nu^{2}} L^{3} \Delta T \)

 

\( = \left ( 0.750 \times 10^{8} 1/K \cdot m^{3} \right )\left ( 1.2 m \right )^{3} \left ( 90 k \right ) = 11.7 \times 10^{9} \)

 

According to equation (\( \frac{D}{L} \geq \frac{35}{Gr_{L}^{1/4}} \)), the effect of curvature may be neglected if

 

\( \frac{D}{L} \geq \frac{35}{\left ( 11.7 \times 10^{9} \right )^{1/4}} = 0.106 \)

 

In the present case D/L = 0.75/1.2 = 0.625; thus, the vertical surface will be treated using equations for a plane wall. Values must now be determined for \( Nu_{t}, Nu_{b} \), and \( Nu_{s} \). Equations ( \( 2 \times 10^{7} < Ra_{L} < 3 \times 10^{10} Nu_{L} = 0.14 Ra_{L}^{1/3}\) ), ( \( 3 \times 10^{5} < Ra_{L} < 10^{10} Nu_{L} = 0.27 Ra_{L}^{1/4}\) ), and ( \( Nu_{L} = \left \{ 0.825 + \frac{0.387 Ra_{L}^{1/6}}{\left [ 1 + \left ( 0.492 / Pr \right )^{9/16} \right ]^{8/27}} \right \}^{2} \) ) will be employed. The three values for Nu are determined as follows:

 

\( Nu_{t} : \)

 

\( L = A/p = \frac{\pi D^{2} /4}{\pi D} = \frac{D}{4} = 0.1875 m \)

 

\( N u_{t} = 0.14 \left [ \left ( 0.750 \times 10^{8} \right )\left ( 0.1875 \right )^{3} \left ( 90 \right )\left ( 0.699 \right ) \right ]^{1/3} = 44.0 \)

 

\( N u_{b} : \)

 

\( N u_{b} = 0.27 \left [ \left ( 0.750 \times 10^{8} \right )\left ( 0.1875 \right )^{3} \left ( 90 \right )\left ( 0.699 \right ) \right ]^{1/4} = 20.2 \)

 

\( N u_{s} : \)

 

\( N u_{s} = \left \{ 0.825 + \frac{0.387 \left [ \left ( 0.750 \times 10^{8} \right )\left ( 1.2 \right )^{3} \left ( 90 \right )\left ( 0.699 \right ) \right ]^{1/6}}{\left [ 1 + \left ( \frac{0.492}{0.699} \right )^{9/16} \right ]^{8/27}} \right \}^{2} = 236 \)

 

The solution for T is now

 

\( T = 295 + \frac{1500 W}{\left [ 0.589 \left ( 44 \right ) + 0.589 \left ( 20.2 \right ) + 2.36 \left ( 236 \right ) \right ] \left ( 0.0293 \right )} = 381.1 K \)

 

Using this as the new estimate for \( T_{surface}, \) we have a film temperature, \( T_{f} \cong 338 K. \) Air properties at this temperature are

\( \nu = 1.936 \times 10^{-5} m^{2}/s k = 0.0291 W/m \cdot K \)

 

\( \alpha = 2.77 \times 10^{-5} m^{2}/s Pr = 0.699 \beta g / \nu^{2} = 0.775 \times 10^{8} 1/K \cdot m^{3} \)

 

The new values for Nu become

 

\( N u_{t} = 0.14 \left [ \left ( 0.775 \times 10^{8} \right )\left ( 0.1875 \right )^{3} \left ( 86 \right )\left ( 0.699 \right ) \right ]^{1/3} = 43.8 \)

 

\( N u_{t} = 0.27 \left [ \left ( 0.775 \times 10^{8} \right )\left ( 0.1875 \right )^{3} \left ( 86 \right )\left ( 0.699 \right ) \right ]^{1/4} = 20.10 \)

 

\( N u_{s} = \left \{ 0.825 + \frac{0.387 \left [ 0.775 \times 10^{8} \left ( 1.2 \right )^{3} \left ( 86 \right )\left ( 0.699 \right ) \right ]^{1/6}}{\left [ 1 + \left ( 0.492 /0.699 \right )^{9/16} \right ]^{8/27}} \right \}^{2} = 235 \)

 

The revised value for \( T_{s} \) is now

 

\( T_{s} = 295 + \frac{1500 / 0.0293 }{\left [ 0.589 \left ( 43.8 \right ) + 0.589 \left ( 20.1 \right ) + 2.36 \left ( 235 \right ) \right ]} = 381.4 K \)

 

This result is obviously close enough, and the desired result for surface temperature is

 

\( T_{surface} \cong 381 K \)
