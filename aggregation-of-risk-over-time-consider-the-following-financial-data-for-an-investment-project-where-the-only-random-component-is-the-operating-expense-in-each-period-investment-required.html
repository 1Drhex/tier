Aggregation of Risk over Time

Consider the following financial data for an investment project, where the only random component is the operating expense in each period:

  • Investment required = $10,000
  • Project life = 3 years
  • Expected salvage value = $0
  • Annual operating revenue = $20,000
  • Annual operating expenses are random variables with the following means and variances:
\(X_{3}\) \(X_{2}\) \(X_{1}\) n
$15,000 $13,000 $9,000 Mean
1,000,000 490,000 250,000 Variance
Admin
Solution

Given: Cash flow data and input probability distributions.
Find: NPW distribution.

Step 1. Calculate the net proceeds from disposal of the equipment at the end of the project life:

Salvage value = 0

Book value = \(10,000 – \sum\limits_{n = 1}^{3}{D_{n} } = \$3,333\)

Taxable gain or loss = 0 – $3,333 = ($3,333)
Loss credit = 0.4($3,333) = $1,333
Net proceeds from sale = Salvage value + Loss credit = $1,333

Step 2. Construct a generalized cash flow table (Table 12.14) as a function of the random variable \(X_{n}\).

The means and variances of random variables (\(X_{1}\), \(X_{2}\), & \(X_{3}\)) are given as

\(E\left[X_{1}\right]\) = 9,000 \(E\left[X_{2}\right]\) = 13,000 \(E\left[X_{3}\right]\) = 15,000

\(Var\left[X_{1}\right]\) = 250,000 \(Var\left[X_{2}\right]\) = 490,000 \(Var\left[X_{3}\right]\) = 1,000,000

Table 12.14 Project’s After-Tax Cash Flows

Cash Flow
3 2 1 0 Description
-10,000 Investment
12,000 12,000 12,000 Revenue × (0.6)
\(-0.6X_{3}\) \(-0.6X_{2}\) \(-0.6X_{1}\) -O&M × (0.6)
667 1,334 667 \(D_{n}\) × (0.4)
1,333 Net proceeds from sale
14,000 13,334 12,667 -10,000 Net cash flow
\(-0.6X_{3}\) \(-0.6X_{2}\) \(-0.6X_{1}\)

The net present worth is then

\(PW\left(12\%\right) = \left[-10,000+\frac{12,667}{1.12}+\frac{13,334}{1.12^{2} } +\frac{14,000}{1.12^{3} } \right]\)

\(– 0.6 \left[\frac{X_{1} }{1.12} +\frac{X_{2} }{1.12^{2} } +\frac{X_{3} }{1.12^{3} }\right]\)

\(= 21,905 – 0.6 \left[\frac{X_{1} }{1.12} +\frac{X_{2} }{1.12^{2} } +\frac{X_{3} }{1.12^{3} }\right]\)

Case 1. Independent Cash Flows: Using Eqs.

\(E\left[PW\left(i\right) \right] =\sum\limits_{n = 0}^{N}{\frac{E\left(A_{n} \right) }{\left(1+i\right) ^{n} } }\) (12.11)

and

\(Var\left[PW\left(i\right) \right] =\sum\limits_{n = 0}^{N}{\frac{Var\left(A_{n} \right) }{\left(1+i\right) ^{2n} } }\) (12.12)

, we obtain

\(E\left[PW\left(12\%\right) \right] = 21,905 – 0.6 \left[\frac{E\left[X_{1}\right] }{1.12} +\frac{E\left[X_{2}\right] }{1.12^{2} } +\frac{E\left[X_{3}\right] }{1.12^{3} }\right]\)

= 21,905 – 17,446
= $4,459

\(Var\left[PW\left(12\%\right) \right] =\left(\frac{-0.6}{1.12} \right) ^{2} Var\left[X_{1} \right] + \left(\frac{-0.6}{1.12^{2} } \right) ^{2} Var\left[X_{2} \right] + \left(\frac{-0.6}{1.12^{3} } \right) ^{2} Var\left[X_{3} \right]\)

= 71,747.45 + 112,103.39 + 182,387.20
= 366,240

\(\sigma \left[PW\left(12\%\right) \right] = \$605.18\)

Case 2. Dependent Cash Flows: If the random variables \(X_{1}\) , \(X_{2}\) and \(X_{3}\) are partially correlated with the correlation coefficients \(\rho _{12} = 0.3\) , \(\rho _{13} = 0.5\) , and \(\rho _{23} = 0.4\) , respectively (see Figure 12.10), then the expected value and the variance of the NPW can be calculated with Eq:

\(Var \left[PW\left(i\right) \right] = \sum\limits_{n = 0}^{N}{\frac{Var \left(A_{n} \right) }{\left(1+i\right) ^{2n} } } +2 \sum\limits_{n = 0}^{N}{ } \sum\limits_{s = n+1}^{N}{\frac{\rho _{ns}\sigma _{n}\sigma _{s} }{\left(1+i\right) ^{n+s}} }\) (12.13)

\(E\left[PW\left(12\%\right) \right] = 21,905 – 0.6 \left[\frac{E\left[X_{1}\right] }{1.12} +\frac{E\left[X_{2}\right] }{1.12^{2} } +\frac{E\left[X_{3}\right] }{1.12^{3} }\right]\)

= 21,905 – 17,446
= $4,459

Var[PW(12%)] = (Original variance) + (Covariance terms)
= 366,240

\(+2 \left(\frac{-0.6}{1.12} \right) \left(\frac{-0.6}{1.12^{2} } \right) \rho _{12} \sigma _{1} \sigma _{2}\)

\(+2 \left(\frac{-0.6}{1.12} \right) \left(\frac{-0.6}{1.12^{3} } \right) \rho _{13} \sigma _{1} \sigma _{3}\)

\(+2 \left(\frac{-0.6}{1.12^{2}} \right) \left(\frac{-0.6}{1.12^{3} } \right) \rho _{23} \sigma _{2} \sigma _{3}\)

= 366,240

\(+2 \left(\frac{0.6^{2} }{1.12^{3}} \right)\left(0.3\right) \left(500\right) \left(700\right)\)

\(+2 \left(\frac{0.6^{2} }{1.12^{4}} \right)\left(0.5\right) \left(500\right) \left(1,000\right)\)

\(+2 \left(\frac{0.6^{2} }{1.12^{5}} \right)\left(0.4\right) \left(700\right) \left(1,000\right)\)

= 366,240 + 53,810.59 + 114,393.25 + 114,393.25
= 648,837

\(\sigma \left[PW\left(12\%\right) \right] =\$805.50\)

Note that the variance increases significantly when at least some of the random variables are positively correlated.

Comments: How is information such as the preceding used in decision making? Most probability distributions are completely described by six standard deviations— three above, and three below, the mean. As shown in Figure 12.11, the NPW of this project would almost certainly fall between $2,643 and $6,275 for the independent case and between $2,043 and $6,876 for the dependent case. In either situation, the NPW below \(3\sigma\) from the mean is still positive, so we may say that the project is quite safe. If that figure were negative, it would then be up to the decision maker to determine whether it is worth investing in the project, given its mean and standard deviation.

If we assume a normal probability distribution, we could find the probability that the project’s NPW will be greater than $5,000. With case 1,

\(Pr\left(X\geq x\right) = Pr \left(PW\left(12\%\right) \geq \$5,000 \right)\)

\(= 1 – Pr \left(PW\left(12\%\right)\leq \$5,000 \right)\)

\(= 1 – \Phi \left(\frac{X-\mu }{\sigma } \right) =1-\Phi \left(\frac{5,000 – 4,459}{605.18} \right)\)

\(= 1-\Phi \left(0.89\right)\)

= 1 – 0.81317
= 18.68%

Similarly, we can find the probability that the project will lose money (or the project’s NPW will be negative), which turns out to be 0.

\(Pr\left(X\leq x\right) = Pr \left(PW\left(12\%\right) \leq 0 \right)\)

\(=\Phi \left(\frac{X-\mu }{\sigma } \right) =\Phi \left(\frac{0 – 4,459 }{605.18 } \right) = \Phi \left(-7.3681\right)\)

\(= 1 – \Phi \left(7.3681\right)\)

= 0

See Appendix C for procedures on how to use the normal probability distribution.
