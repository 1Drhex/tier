Guest Question.

In this problem you will develop an alternative approach to degenerate perturbation theory. Consider an unperturbed Hamiltonian \( H^{0} \) with two degenerate states \( \psi_{a}^{0} \text { and } \psi_{b}^{0} \text { (energy } E^{0} \)), and a perturbation H′. Define the operator that projects onto the degenerate subspace:

\( P_{D}=\left|\psi_{a}^{0}\right\rangle\left\langle\psi_{a}^{0}|+| \psi_{b}^{0}\right\rangle\left\langle\psi_{b}^{0}\right| \). (7.100).

The Hamiltonian can be written

\( H=H^{0}+H^{\prime}=\tilde{H}^{0}+\tilde{H}^{\prime} \) (7.101).

where

\( \tilde{H}^{0}=H^{0}+P_{D} H^{\prime} P_{D}, \quad \tilde{H}^{\prime}=H^{\prime}-P_{D} H^{\prime} P_{D}\) (7.102).

The idea is to treat \( \tilde{H}^{0} \) as the “unperturbed” Hamiltonian and \( \tilde{H}^{\prime} \) as the perturbation; as you’ll soon discover, \( \tilde{H}^{0} \) is nondegenerate, so we can use ordinary nondegenerate perturbation theory.

(a) First we need to find the eigenstates of \( \tilde{H}^{0} \) .

i. Show that any eigenstate \( \psi_{n}^{0} \text { (other than } \psi_{a}^{0} \text { or } \psi_{b}^{0} \text { ) of } H^{0} \) is also an eigenstate of \( \tilde{H}^{0} \) with the same eigenvalue.

ii. Show that the “good” states \( \psi^{0}=\alpha \psi_{a}^{0}+\beta \psi_{b}^{0} \) (with α and β determined by solving Equation 7.30) are eigenstates of \( \tilde{H}^{0} \) with energies \( E^{0}+E_{\pm}^{1} \).

\( \underbrace{\left(\begin{array}{ll} W_{a a} & W_{a b} \\ W_{b a} & W_{b b} \end{array}\right)}_{W}\left(\begin{array}{l} \alpha \\ \beta \end{array}\right)=E^{1}\left(\begin{array}{l} \alpha \\ \beta \end{array}\right) \) (7.30).

(b) Assuming that \( E_{+}^{1} \text { and } E_{-}^{1} \) are distinct, you now have a nondegenerate unperturbed Hamiltonian \( \tilde{H}^{0} \) and you can do nondegenerate perturbation theory using the perturbation \( \tilde{H}^{\prime} \). Find an expression for the energy to second order for the states \( \psi_{\pm}^{0} \text { in (ii) } \).

Comment: One advantage of this approach is that it also handles the case where the unperturbed energies are not exactly equal, but very close: \( E_{a}^{0} \approx E_{b}^{0} \). In this case one must still use degenerate perturbation theory; an important example of this occurs in the nearly-free electron approximation for calculating band structure.

Solution

(a)

\( \tilde{H}_{0}\left|\psi_{n}^{0}\right\rangle=H_{0}\left|\psi_{n}^{0}\right\rangle+P_{D} H^{\prime} P_{D}\left|\psi_{n}^{0}\right\rangle . \text { But } P_{D}\left|\psi_{n}^{0}\right\rangle=\left|\psi_{a}^{0}\right\rangle\left\langle\psi_{a}^{0} \mid \psi_{n}^{0}\right\rangle+\left|\psi_{b}^{0}\right\rangle\left\langle\psi_{b}^{0} \mid \psi_{n}^{0}\right\rangle=0 \), because eigenstates of \( H_{0} \) belonging to different eigenvalues are orthogonal. And \( H_{0}\left|\psi_{n}^{0}\right\rangle=E_{n}^{0}\left|\psi_{n}^{0}\right\rangle, \text { so } \tilde{H}_{0}\left|\psi_{n}^{0}\right\rangle=E_{n}^{0}\left|\psi_{n}^{0}\right\rangle \).

\( \tilde{H}_{0}\left|\psi^{0}\right\rangle=H_{0}\left|\psi^{0}\right\rangle+P_{D} H^{\prime} P_{D}\left|\psi^{0}\right\rangle . \quad \text { But } H_{0}\left|\psi^{0}\right\rangle=E^{0}\left|\psi^{0}\right\rangle, \text { and } P_{D}\left|\psi^{0}\right\rangle=\left|\psi^{0}\right\rangle \), so

\( =E^{0}\left(\alpha\left|\psi_{a}^{0}\right\rangle+\beta\left|\psi_{b}^{0}\right\rangle\right)+\left(\left|\psi_{a}^{0}\right\rangle\left\langle\psi_{a}^{0}|+| \psi_{b}^{0}\right\rangle\left\langle\psi_{b}^{0}\right|\right) H^{\prime}\left(\alpha\left|\psi_{a}^{0}\right\rangle+\beta\left|\psi_{b}^{0}\right\rangle\right) \).

\( =E^{0}\left(\alpha\left|\psi_{a}^{0}\right\rangle+\beta\left|\psi_{b}^{0}\right\rangle\right)+\alpha\left(W_{a a}\left|\psi_{a}^{0}\right\rangle+W_{b a}\left|\psi_{b}^{0}\right\rangle\right)+\beta\left(W_{a b}\left|\psi_{a}^{0}\right\rangle+W_{b b}\left|\psi_{b}^{0}\right\rangle\right) \).

\( =\left(\alpha E^{0}+\alpha W_{a a}+\beta W_{a b}\right)\left|\psi_{a}^{0}\right\rangle+\left(\beta E^{0}+\alpha W_{b a}+\beta W_{b b}\right)\left|\psi_{b}^{0}\right\rangle \).

\( =\left(\alpha E^{0}+\alpha E^{1}\right)\left|\psi_{a}^{0}\right\rangle+\left(\beta E^{0}+\beta E^{1}\right)\left|\psi_{b}^{0}\right\rangle=\left(E^{0}+E^{1}\right)\left|\psi^{0}\right\rangle \).

In the last line I used Equations 7.27 and 7.29; \( E^{1} \text { can be either } E_{+}^{1} \text { or } E_{-}^{1} \) from Equation 7.33.

\( \alpha W_{a a}+\beta W_{a b}=\alpha E^{1} \) (7.27).

\( \alpha W_{b a}+\beta W_{b b}=\beta E^{1} \) (7.29).

\( E_{\pm}^{1}=\frac{1}{2}\left[W_{a a}+W_{b b} \pm \sqrt{\left(W_{a a}-W_{b b}\right)^{2}+4\left|W_{a b}\right|^{2}}\right] \) (7.33).

(b) Using nondegenerate perturbation theory, to second order (Equations 7.9 and 7.15), for \( \psi_{+} \):

\( E_{n}^{\prime}=\left\langle\psi_{n}^{0}\left|H^{\prime}\right| \psi_{n}^{0}\right\rangle \) (7.9).

\( E_{n}^{2}=\sum_{m \neq n} \frac{\left|\left\langle\psi_{m}^{0}\left|H^{\prime}\right| \psi_{n}^{0}\right\rangle\right|^{2}}{E_{n}^{0}-E_{m}^{0}} \) (7.15).

\( E_{+}=\left(E^{0}+E_{+}^{1}\right)+\left\langle\psi_{+}^{0}\left|\tilde{H}^{\prime}\right| \psi_{+}^{0}\right\rangle+\sum_{m \neq+} \frac{\left|\left\langle\psi_{m}^{0}\left|\tilde{H}^{\prime}\right| \psi_{+}^{0}\right\rangle\right|^{2}}{E^{0}+E_{+}^{1}-E_{m}^{0}} \).

But

\( \left\langle\psi_{+}^{0}\left|\tilde{H}^{\prime}\right| \psi_{+}^{0}\right\rangle=\left\langle\psi_{+}^{0}\left|H^{\prime}\right| \psi_{+}^{0}\right\rangle-\left\langle\psi_{+}^{0}\left|P_{D} H^{\prime} P_{D}\right| \psi_{+}^{0}\right\rangle=0 \).

because \( P_{D}\left|\psi_{+}^{0}\right\rangle=\left|\psi_{+}^{0}\right\rangle \text { and }\left\langle\psi_{+}^{0}\right| P_{D}=\left\langle\psi_{+}^{0}\right| \), while

\( \left\langle\psi_{m}^{0}\left|\tilde{H}^{\prime}\right| \psi_{+}^{0}\right\rangle=\left\langle\psi_{m}^{0}\left|H^{\prime}\right| \psi_{+}^{0}\right\rangle-\left\langle\psi_{m}^{0}\left|P_{D} H^{\prime} P_{D}\right| \psi_{+}^{0}\right\rangle= \begin{cases}\left\langle\psi_{m}^{0}\left|H^{\prime}\right| \psi_{+}^{0}\right\rangle, & (m \neq \pm) \\ 0, & (m=\pm)\end{cases} \).

\( \text { (note that } \left.P_{D}\left|\psi_{m}^{0}\right\rangle=0 \text { if } m \neq \pm\right) \text {. So } \).

\( E_{+}=E^{0}+E_{+}^{1}+\sum_{m \neq \pm} \frac{\left|\left\langle\psi_{m}^{0}\left|H^{\prime}\right| \psi_{+}^{0}\right\rangle\right|^{2}}{E^{0}+E_{+}^{1}-E_{m}^{0}} \)

\( \text { (Change the subscripts }+\text { to }-\text { for } E_{-} \text {.) Typically one drops the } E^{1} \) term in the denominator, since it is really a third-order correction.
