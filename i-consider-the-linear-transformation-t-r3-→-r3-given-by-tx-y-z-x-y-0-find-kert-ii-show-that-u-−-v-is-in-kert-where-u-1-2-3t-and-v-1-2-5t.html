Guest Question.

(i) Consider the linear transformation \(T: \mathbb{R}^{3} \rightarrow \mathbb{R}^{3}\) given by \(T\left(\left(\begin{array}{l}x \\ y \\ z\end{array}\right)\right)=\left(\begin{array}{l}x \\ y \\ 0\end{array}\right)\). Find \(\operatorname{ker}(T)\).

(ii) Show that \(\mathbf{u}-\mathbf{v}\) is in \(\operatorname{ker}(T)\) where \(\mathbf{u}=\left(\begin{array}{lll}1 & 2 & 3\end{array}\right)^{T}\) and \(\mathbf{v}=\left(\begin{array}{lll}1 & 2 & 5\end{array}\right)^{T}\).

Admin
Solution

(i) We need to find the vectors in \(\mathbb{R}^{3}\) which arrive at the zero vector under \(T\). Which vectors are transformed (mapped) to the zero vector?

Inspecting the given transformation \(T\left(\left(\begin{array}{l}x \\ y \\ z\end{array}\right)\right)=\left(\begin{array}{l}x \\ y \\ 0\end{array}\right)\), we arrive at the zero vector if \(x=0, y=0\) and \(z=r\), where \(r\) is any real number. Therefore

\(T\left(\left(\begin{array}{l}0 \\0 \\r\end{array}\right)\right)=\left(\begin{array}{l}0 \\0 \\0\end{array}\right)=\mathbf{O} \\\) \(\text { Thus the kernel of } T \text { in this case is }\left.\left\{\left(\begin{array}{l}0 \\0 \\r\end{array}\right)\right| \text { is a real number }\right\} \text {. The kernel of the given }\) \(\text { transformation } T \text { is the } z \text { axis in } \mathbb{R}^{3} \text { as illustrated in Fig. } 5.13 \text {. }\)

The given \(T\) transforms any point in \(\mathbb{R}^{3}\) to the shaded plane shown in Fig. 5.13.

(ii) We have

\(\mathbf{u}-\mathbf{v}=\left(\begin{array}{lll}1 & 2 & 3\end{array}\right)^{T}-\left(\begin{array}{lll}1 & 2 & 5\end{array}\right)^{T}=\left(\begin{array}{lll}0 & 0 & -2\end{array}\right)^{T}\)

Transforming this vector \(\mathbf{u}-\mathbf{v}\) :

\(\left.T(\mathbf{u}-\mathbf{v})=T\left(\begin{array}{lll}(0 & 0 & -2\end{array}\right)^{T}\right)=\left(\begin{array}{lll}0 & 0 & 0\end{array}\right)^{T}=\mathbf{O}\)

Since \(T(\mathbf{u}-\mathbf{v})=\mathbf{O}\) the vector \(\mathbf{u}-\mathbf{v}\) is in \(\operatorname{ker}(T)\). Using \(T\) is linear on \(T(\mathbf{u}-\mathbf{v})=\mathbf{O}\) gives

\(T(\mathbf{u}-\mathbf{v})=T(\mathbf{u})-T(\mathbf{v})=\mathbf{O} \text { implies } T(\mathbf{u})=T(\mathbf{v})\) \(\mathbf{u}-\mathbf{v} \text { is in } \operatorname{ker}(T) \text { means that vectors } \mathbf{u} \text { and } \mathbf{v} \text { arrive at the same point under } T \text { (Fig. 5.14). }\)

Hence the kernel detects when two points are transformed to the same point.
