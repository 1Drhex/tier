Guest Question.

Pile load test at St. Gabriel (Capazzoli, 1968)

Data

Pile diameter, d = 10 in, steel pipe filled with concrete

\(EI =38 \times 10^{8} lb - in ^{2}\)

L = 115 ft

e = 12 in.

c = 600 lb/ft^2

\(\gamma=110 lb / ft ^{3}\)

\(M_{y}=116\) ft-kips

Water table was close to the ground surface.

Required

(a) \(P_{t} \text { vs. } y_{g}\) curve

(b) the ultimate lateral load, \(P_{u}\)

Admin
Solution

We have,

 

\(\text { (a) } n_{h}=\frac{125 c^{1.5} \sqrt{E I \gamma d}}{(1+e / d)^{1.5} P_{e}^{1.5}}\) \(\text { (b) } P_{e}=P_{t} \quad 1+0.67 \frac{e}{T}\) \(\text { (c) } T=\frac{E I}{n_{h}}^{\frac{1}{5}}\)

 

After substituting the known values in Eq. (a) and simplifying, we have

 

\(n_{h}=\frac{16045 \times 10^{3}}{P_{e}^{1.5}}\)

 

(a) Calculation of groundline deflection

1. Let \(P_{e}=P_{t}=500 lbs\)

From Eqs (a) and (c), \(n_{h}=45 lb / in ^{3}, T=38.51 \text { in }\)

From Eq. (d), \(P_{e}=6044 lb .\)

2. For \(P_{e}=6044 lb , n_{h}=34 lb / in ^{3} \text { and } T=41 in\)

3. For T = 41 in, \(P_{e}=5980 lb , \text { and } n_{h}=35 lb / in ^{3}\)

4. For \(n_{h}=35 lb / in ^{3}, T=40.5 in , P_{e}=5988 lb\)

5. \(y_{g}=\frac{2.43 P_{e} T^{3}}{E I}=\frac{2.43 \times 5988 \times(40.5)^{3}}{38 \times 10^{8}}=0.25 in\)

6. Continue steps 1 through 5 for computing \(y_{g}\) for different loads \(P_{t}\). Fig. 16.17 gives a plot of \(P_{t} \text { vs. } y_{g}\) which agrees very well with the measured values.

(b) Ultimate load \(P_{u}\)

A curve of \(M_{\max } \text { vs. } P_{t}\) is given in Fig. 16.17 following the procedure given for the Mustang Island Test. From this curve \(P_{u}=23 k \text { for } M_{y}=116\) ft kips. This agrees well with the values obtained by the methods of Reese (1985) and Broms (1964a).
