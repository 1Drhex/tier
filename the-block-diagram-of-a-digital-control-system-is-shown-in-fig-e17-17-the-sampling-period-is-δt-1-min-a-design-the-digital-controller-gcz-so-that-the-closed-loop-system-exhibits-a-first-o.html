Guest Question.

The block diagram of a digital control system is shown in Fig. E17.17. The sampling period is \(\Delta t=1 min\).

(a) Design the digital controller \(G_{c}(z)\) so that the closed-loop system exhibits a first-order response to a unit step change in the set point (after an appropriate time delay).

(b) Will this controller eliminate offset after a step change in the set point? Justify your answer.

(c) Is the controller physically realizable? Justify your answer.

(d) Design a digital PID controller based on the ITAE (set-point) method in Chapter 12 and examine its performance for a step change in set point. Approximate the sampler and \(ZOH\) by a time delay of \(\theta=\Delta t / 2\).

Admin
Solution

a) The closed-loop response for set-point changes is

\(\frac{Y(s)}{Y_{s p}(s)}=\frac{G_{c} G(s)}{1+G_{c} G(s)} \quad \text { then } \quad G_{c}(z)=\frac{1}{G} \frac{\left(Y / Y_{s p}\right)}{1-\left(Y / Y_{s p}\right)}\)

We want the closed-loop system exhibits a first order plus dead time response,

\(\left(Y / Y_{s p}\right)=\frac{e^{-h s}}{\lambda s+1} \quad \text { or } \quad\left(Y / Y_{s p}\right)=\frac{(1-A) z^{-N-1}}{1-A z^{-1}} \quad \text { where } A= e ^{-\Delta t / \lambda}\)

Moreover,

\(G(s)=\frac{e^{-2 s}}{3 s+1} \quad \text { or } \quad G(z)=\frac{0.284 z^{-3}}{1-0.716 z^{-1}}\)

Thus, the resulting digital controller is the Dahlin’s controller Eq. 17-66.

\(G_{D C}=\frac{(1-A)}{1-A z^{-1}-(1-A) z^{-N-1}} \frac{1-a_{1} z^{-1}}{K\left(1-a_{1}\right)}\) (17-66)

\(G_{c}(z)=\frac{(1-A)}{1-A z^{-1}-(1-A) z^{-N-1}} \frac{1-0.716 z^{-1}}{0.284}\) (1)

If a value of \(\lambda=1\) is considered, then \(A=0.368\) and Eq. 1 is

\(G_{c}(z)=\frac{0.632}{1-0.368 z^{-1}-0.632 z^{-3}} \frac{1-0.716 z^{-1}}{0.284}\) (2)

b) \(\left(1-z^{-1}\right)\) is a factor of the denominator in Eq. 2, indicating the presence integral action. Then no offset occurs.

c) From Eq. 2, the denominator of \(G_{c}(z)\) contains a non-zero \(z^{-0}\) term. Henc the controller is physically realizable.

d) First adjust the process time delay for the zero-order hold by adding \(\Delta t /\) to obtain a time delay of \(2+0.5=2.5 min\). Then obtain the continuos PII controller tuning based on the ITAE (setpoint) tuning relation in Table \(12.3\) with \(K=1, \tau=3, \theta=2.5\). Thus

\(\begin{array}{ll}K K_{c}=0.965(2.5 / 3)^{-0.85} & , K_{c}=1.13 \\\tau / \tau_{I}=0.796+(-0.1465)(2.5 / 3), & , \tau_{I}=4.45 \\\tau_{D} / \tau=0.308(2.5 / 3)^{0.929} & , \tau_{D}=0.78\end{array}\)

Using the position form of the PID control law (Eq. 8-26 or 17-55)

\(p_{k-1}=\bar{p}+K_{c}\left[e_{k-1}+\frac{\Delta t}{\tau_{I}} \sum\limits_{j=1}^{k-1} e_{j}+\frac{\tau_{D}}{\Delta t}\left(e_{k-1}-e_{k-2}\right)\right]\) (8-26)

\(G_{c}(z)=\frac{K_{c}\left(a_{0}+a_{1} z^{-1}+a_{2} z^{-2}\right)}{1-z^{-1}}\) (17-55)

\(\begin{aligned}G_{c}(z) &=1.13\left[1+0.225\left(\frac{1}{1-z^{-1}}\right)+0.78\left(1-z^{-1}\right)\right] \\&=\frac{2.27-2.89 z^{-1}+0.88 z^{-2}}{1-z^{-1}}\end{aligned}\)
