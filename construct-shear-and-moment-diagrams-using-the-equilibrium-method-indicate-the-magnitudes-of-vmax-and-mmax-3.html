Guest Question.

Construct shear and moment diagrams using the equilibrium method. Indicate the magnitudes of Vmax and Mmax.

Admin
Solution

Solve I or Ihe external reactions al B and D.
∑MD =+(10k)(15′)+(20k)(5′)-RB = 0

RB = +25k
∑Fy = -10k – 20k + 25k + RD = 0
RD = +5k

Cut sections E, F and G and write equations of equilibrium ∑Mi = O and ∑Fy = 0 to determine the internal shear and moment developed at each of the respective sections.

Section cut E:
∑Fy = 0: -10k + VE = 0; VE=10k
The shear between A and B remains constant.
∑ME = 0: +10k(x) – ME = 0; ME = 10x
The moment M increases as a function of x, between A and B; x = 0 to x = 5’.

Section F:
∑Fy = -10k + 25k – VF = 0; VF = 15k
The shear remains constant (positive) between B and C.
∑MF = (10k)(x) – (25k)(x – 5’) + MF = 0
MF = 15x -125
The moment varies linearly from x = 5’ to x = 10’.

Section G:
∑Fy = -10k + 25k – 20k + VG = 0;
VG = 5k
Shear is constant between C aqnd D.
∑MG = (10k)(x) – (25k)(x – 5’) – (20k)(x -10’)+ MG = 0; MG = -5x + 75
Moment varies linearly for x = 10’ to x = 15’.
