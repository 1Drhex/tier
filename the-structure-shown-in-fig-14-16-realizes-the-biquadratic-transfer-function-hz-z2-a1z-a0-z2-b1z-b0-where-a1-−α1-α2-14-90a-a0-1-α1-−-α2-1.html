Guest Question.

The structure shown in Fig. 14.16 realizes the biquadratic transfer function

\(H(z)=\frac{z^{2}+a_{1} z+a_{0}}{z^{2}+b_{1} z+b_{0}}\)

where

\(a_{1}=-\left(\alpha_{1}+\alpha_{2}\right)\) (14.90a)

\(a_{0}=1+\alpha_{1}-\alpha_{2}\) (14.90b)

\(b_{1}=-\left(\beta_{1}+\beta_{2}\right)\) (14.90c)

\(b_{0}=1+\beta_{1}-\beta_{2}\) (14.90d)

and is due to Meerkotter [26]. Show that the structure supports the elimination of zero-input limit cycles.

Admin
Solution

Straightforward analysis gives the state-space characterization of the structure as

 

\(q (n+1)= A q ( n )+ b x(n)\)

 

\(y(n)= c ^{T} q ( n )+d x(n)\)

 

where

 

\(A =\left[\begin{array}{ll}a_{11} & a_{12} \\a_{21} & a_{22}\end{array}\right]=\left[\begin{array}{ll}\beta_{1} & \left(\beta_{1}+1\right) \\\left(\beta_{2}-1\right) & \beta_{2}\end{array}\right]\)

 

\(b =\left[\begin{array}{l}\left(\beta_{1}-\alpha_{1}\right) \\\left(\beta_{2}-\alpha_{2}\right)\end{array}\right] \quad c =\left[\begin{array}{l}1 \\1\end{array}\right] \quad \text { and } \quad d=1\)

 

The filter is stable if and only if

 

\(1-b_{0}>0\) (14.91a)

 

\(1+b_{1}+b_{0}>0\) (14.91b)

 

\(1-b_{1}+b_{0}>0\) (14.91c)

 

as can be easily shown by using the Jury-Marden stability criterion (see Sec. 5.3.7. From Eq. (14.90), we can show that

 

\(a_{12} a_{21}=\left(\beta_{1}+1\right)\left(\beta_{2}-1\right)=\frac{1}{4}\left[b_{1}^{2}-\left(1+b_{0}\right)^{2}\right]\)

 

and since \(1+b_{0}>b_{1}\), according to Eq. (14.91c), we conclude that \(a_{12} a_{21}<0\). Hence, zero-input limit cycles can be eliminated by using magnitude truncation only if the condition in Eq. (14.89b) is satisfied. Simple manipulation now yields

 

\(a_{12} a_{21}<0 \quad \text { and } \quad\left|a_{11}-a_{22}\right|+\operatorname{det}( A ) \leq 1\) (14.89b)

 

\(\left|a_{11}-a_{22}\right|+\operatorname{det}( A )=\left|b_{0}-1\right|+b_{0}-1+1=1\)

 

since \(b_{0}-1\) is negative according to Eq. (14.91a); that is, Eq. (14.89b) is satisfied with the equal sign and, therefore, the structure supports the elimination of zero-input limit cycles.
