Guest Question.

\(A 220\) V DC series motor runs drawing a current of \(30\) A from the supply. Armature and field circuit resistances are \(0.4 Ω\) and \(0.1 Ω\), respectively. The load torque varies as the square of the speed.

The flux in the motor may be taken as being proportional to the armature current. To reduce the speed of the motor by \(50\)%, the resistance in ohms the should be added in series with the armature is _____. (Give the answer up to two decimal places).

Admin
Solution

Back \(E.m.f.{ E }_{ 1 }=220-30(0.1+0.4)=205V\)

Torque,\(\quad \quad \tau =\phi { I }_{ a }\)

{Where, \(\quad \phi\) = flux, \({ I }_{ a } = \)armature current}

\(\quad \quad \quad \quad \tau ={ { I }_{ a } }^{ 2 }\quad \quad \quad \quad (1)\)

As in series motor \(\quad \quad \quad\phi \propto { I }_{ a }\)

Also,\(\quad \quad \quad \tau \propto { N }^{ 2 }\quad \quad \quad \quad (2)\)

Where \(N\) is the speed of motor Using Eqs. \((1)\) and \((2)\)

\(\quad \quad \quad \quad { { I }_{ a } }^{ 2 }\propto { N }^{ 2 }\) or \({ { I }_{ a } }\propto { N }\quad \quad \quad \quad (3)\)

Therefore, to reduce speed by \(50\)%, \({ I }_{ a }\)will reduce to \(50\)%, \(i.e., 15 A\). Now back emf will change to

\(\quad \quad \quad \quad { E }_{ 2 }=220-15(R+0.1+0.4)\)

Where \(R\) is the external resistance added in series with armature.

Since,\(\quad \quad E\propto \phi N\)

So,\(\quad \quad \quad { E }_{ 1 }\propto { \phi }_{ 1 }{ .N }_{ 1 }\)

\(\quad \quad \quad \quad { E }_{ 2}\propto { \phi }_{ 2}{ .N }_{ 2}\)

 

\(\quad \quad \quad \quad { \phi }_{ 2 }=\frac { { \phi }_{ 1 } }{ 2 }\)

 

\(\quad \quad \quad \quad { N }_{ 2 }=\frac { { N }_{ 1 } }{ 2 }\)

Thus,

\(\quad \quad \quad \quad 205={ \phi }_{ 1 }{ N }_{ 1 }\quad \quad \quad \quad (4)\)

 

\(\quad \quad \quad \quad 220-15(R+0.5)={ \phi }_{ 2}{ .N }_{ 2}\quad \quad \quad \quad (5)\)

Dividing \(\quad \quad { E}_{ q}. (5)\) by \({ E}_{ q}. (4),\)

\(\quad \quad \quad \quad \frac { 220-15(R+0.5) }{ 205 } =\frac { { \phi }_{ 2 }{ .N }_{ 2 } }{ { \phi }_{ 1 }{ .N }_{ 1 } } =\frac { \left( { { \phi }_{ 1 } }/{ 2 } \right) .\left( { { N }_{ 1 } }/{ 2 } \right) }{ { \phi }_{ 1 }{ .N }_{ 1 } }\)

 

\(\quad \quad \quad \quad \frac { 220-15(R+0.5) }{ 205 } =\frac { 1 }{ 4 }\)

On solving,

\(\quad \quad \quad \quad R = 10.75 \Omega\)

Hence, the correct answer is\( (10.75 \Omega ).\)
