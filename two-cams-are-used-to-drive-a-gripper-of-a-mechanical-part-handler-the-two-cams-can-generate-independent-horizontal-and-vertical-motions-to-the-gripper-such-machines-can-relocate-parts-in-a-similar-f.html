Guest Question.

Two cams are used to drive a gripper of a mechanical part handler. The two cams can generate independent horizontal and vertical motions to the gripper. Such machines can relocate parts in a similar fashion to a robot at a fraction of the cost. The part handler is shown in Figure 9.31.

The prescribed motion for one of the cam followers is as follows:

1. Rise 1.5 in. in 1.5 s using the harmonic motion scheme.
2. Dwell for 2 s.
3. Return in 1.5 s using the harmonic motion scheme.
4. Dwell for 2 s.
5. Repeat the sequence.

An in-line roller follower with a radius of 0.5 in. is used on a cam with a base circle radius of 3.5 in. Tabulate the
follower motion. 1.Calculate the Time for a Full Cycle 2.Calculate the Required Rotational Speed of the Cam 3.Determine the Cam Rotation for Each Follower Motion Interval 4.Calculate the Displacements during Each Follower Motion Interval 5.Calculate the Coordinates of the Cam Profile 6.Summarize the Profile Coordinates for Several Cam Angles 7.Plot the Profile Coordinates.

Admin
Solution

1: The total time to complete the full cycle is needed to determine the required speed of the cam.

\(ΣT_i = T_1 + T_2 + T_3 + T_4\)

= 1.5 + 2.0 + 1.5 + 2.0 = 7.0 s

2: From equation \(\omega _{cam}=\frac{1\ rev}{\Sigma\ T_i}\),

\(\omega _{cam}=\frac{1\ rev}{\Sigma \ T_i}=\frac{1\ rev}{7\ s}=0.143\ rev/s=0.899rad/s=8.57\ rpm\)

3: The angular increment of the cam consumed by each follower motion sequence is determined by equation \(\beta _i=(\omega _{cam})(T_i)\).

\(β_1 = (ω_cam)(T_1)\) = (0.143 rev/s)(1.5 s) = 0.214 rev = 77.2°

\(β_2\) = (0.143 rev/s)(2.0 s) = 0.286 rev = 102.8°

\(β_3\) = (0.143 rev/s)(1.5 s) = 0.214 rev = 77.2°

\(β_4\) = (0.143 rev/s)(2.0 s) = 0.286 rev = 102.8°

4: The equations for the harmonic rise and fall were given in Table 9.3. Substituting into the harmonic rise equations gives

TABLE 9.3 Cam Follower Kinematics for Harmonic Motion
Rise Fall
Displacement: \(\Delta R_i=H_0+\frac{H_i}{2}[1-cos(\frac{\pi \ t_i}{T_i} )]\)

\(=H_0+\frac{H_i}{2}[1-cos(\frac{\pi \phi _i}{\beta _i} )]\)

 \(\Delta R_j=H_F+\frac{H_j}{2}[1+cos(\frac{\pi \ t_j}{T_j} )]\)

\(=H_F+\frac{H_j}{2}[1-cos(\frac{\pi \phi _j}{\beta _j} )]\)
Velocity: \(v_i=\frac{\pi \ H_i}{2\ T_i}[sin(\frac{\pi \ t_i}{T_i} )]\)

\(=\frac{\pi H_i\omega }{2\beta _i}[sin(\frac{\pi \phi _i}{\beta _i} )]\)

 \(v_j=\frac{-\pi \ H_j}{2T_j}[sin(\frac{\pi \ t_j}{T_j} )]\)

\(=\frac{-\pi H_j\omega }{2\beta _j}[sin(\frac{\pi \phi _j}{\beta _j} )]\)
Acceleration: \(a_i=\frac{\pi ^2H_i}{2\ T^2_i}[cos(\frac{\pi \ t_i}{T_i} )]\)

\(=\frac{\pi ^2H_i\omega ^2}{2\beta ^2_i}[cos(\frac{\pi \ \phi _i}{\beta _i} )]\)

 \(a_j=\frac{-\pi \ H_i}{2\ T^2_i}[cos(\frac{\pi\ t_j}{T_j} )]\)

\(=\frac{-\pi ^2H_j\omega ^2}{2\beta ^2_j}[cos(\frac{\pi \phi _j}{\beta _j} )]\)

\(\Delta R_1=\frac{H_1}{2}[1-cos(\frac{\pi\ t_1}{T_1} )]=\frac{(1.5\ in.)}{2}[1-cos(\frac{\pi t_1}{1.5\ s} )]\)

 

\(v_1=\frac{\pi H_1}{2T_1}[sin(\frac{\pi t_1}{T_1} )]=\frac{\pi (1.5\ in.)}{2(1.5s)}[sin(\frac{\pi t_1}{1.5s} )]\)

Substituting into the harmonic fall equations gives

\(\Delta R_2=\frac{H_2}{2}[1+cos(\frac{\pi t_2}{T_2} )]=\frac{(1.5\ in.)}{2}[1+cos(\frac{\pi t_2}{1.5\ s} )]\)

 

\(v_2=\frac{-\pi H_2}{2T_2}[sin(\frac{\pi t_2}{T_2} )]=\frac{-\pi (1.5\ in.)}{2(1.5\ s)}[sin(\frac{\pi t_2}{1.5\ s} )]\)

 

5: Substitution into equations \(\alpha =tan^{-1}[\frac{v}{\omega _{cam}}\frac{(R_f+R_b+\Delta R)}{(R_f+R_b+\Delta R)^2} ]=\delta\) , \(R_x = -[R_f + R_b + ΔR] sinΦ + R_f sin(Φ – a)\), and \(R_y = [R_f + R_b + ΔR] cosΦ + R_f cos(Φ – a)\) gives

 

\(a=tan^{-1}[\frac{v}{\omega _{cam}}\frac{[R_f+R_b+s]}{[R_f+R_b+s]^2} ]=tan^{-1}[\frac{v}{(0.899rad/s)}\frac{[(0.5\ in.)+(3.5\ in.)+s]}{[(0.5\ in.)+(3.5\ in.)+s]^2} ]\)

 

\(R_x = -[R_f + R_b + ΔR] sinΦ + R_f sin(Φ – a) = -[0.5 + 3.5 + ΔR] sinΦ + 0.5 sin(Φ – a)\)

 

\(R_y = -[R_f + R_b + ΔR] cosΦ + R_f cos(Φ – a) = -[0.5 + 3.5 + ΔR] cosΦ – 0.5 cos(Φ – a)\)
