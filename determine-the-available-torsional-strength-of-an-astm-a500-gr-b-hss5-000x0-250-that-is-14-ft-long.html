Guest Question.

Determine the available torsional strength of an ASTM A500 Gr. B HSS5.000×0.250 that is 14 ft long.

Admin
Solution

Material Properties:

ASTM A500 Gr. B \(F_{y}=42 ksi \quad F_{u}=58 ksi\)

Manual
Table 2-3

Geometric properties:

HSS5.000×0.250 \(\frac{D}{t}=21.5 \quad t=0.233 \text { in. } \quad D=5.00 \text { in }\)

Manual
Table 1-13

Calculate the critical stress as the larger of

\(F_{c r}=\frac{1.23 E}{\sqrt{\frac{L}{D}}\left(\frac{D}{t}\right)^{\frac{5}{4}}}=\frac{1.23(29,000 ksi )}{\sqrt{\frac{14.0 ft (12 in / ft )}{5.00 in .}(21.5)^{\frac{5}{4}}}}=133 ksi\)

Eqn. H3-2a

and

\(F_{c r}=\frac{0.60 E}{\left(\frac{D}{t}\right)^{\frac{3}{2}}}=\frac{0.60(29,000 ksi )}{(21.5)^{\frac{3}{2}}}=175 ksi\)

Eqn. H3-2b

However, \(F_{c r}\) shall not exceed \(0.60 F_{y}=0.60(42 ksi )=25.2 ksi\), therefore,

Section H3.1

\(F_{c r}\) = 25.2 ksi

Calculate the torsional shear constant

\(C=7.95 in ^{3}\)

Manual
Table 1-13

Calculate the nominal torsional strength

\(T_{n}=F_{c r} C=25.2 ksi \left(7.95 in .^{3}\right)=200 kip \text { -in. }\)

Eqn. H3-1

Calculate the available torsional strength

LRFD ASD
\(\begin{array}{l}\phi_{T}=0.90 \\\phi_{T} T_{n}=0.90(200 \text { kip-in. })=180 \text { kip-in. }\end{array}\) \(\begin{array}{l}\Omega_{T}=1.67 \\T_{n} / \Omega_{T}=200 \text { kip }- in / 1.67=120 \text { kip-in. }\end{array}\)

Section H3.1

Note: For more complete guidance on design for torsion, see AISC Design Guide 9, Torsional Analysis of Structural Steel Members.
