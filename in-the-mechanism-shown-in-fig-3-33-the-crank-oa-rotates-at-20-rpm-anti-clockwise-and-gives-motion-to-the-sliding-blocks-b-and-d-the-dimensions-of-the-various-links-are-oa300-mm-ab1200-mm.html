Guest Question.

In the mechanism shown in Fig.3.33, the crank OA rotates at 20 rpm anti-clockwise and gives motion to the sliding blocks B and D. The dimensions of the various links are: OA=300 mm, AB=1200 mm, BC=450 mm, and CD=450 mm. Determine (a) velocities of sliding at B and D, (b) angular velocity of CD, (c) linear acceleration of D, and (d) angular acceleration of CD.

Admin
Solution

\( \text { Angular velocity of } O A, \omega=\frac{2 \pi N}{60}=\frac{2 \pi \times 20}{60}=2.1 rad / s \).

\( \text { Linear velocity of } A, v_{a}=\omega \times O A=2.1 \times 0.3=0.63 m / s \).

Velocity diagram
1. Draw configuration diagram as shown in Fig.3.33(a).
2. Draw velocity diagram as shown in Fig.3.33(b).

\( \text { 3. Draw oa } \perp O A \text { such that } o a=6.3 cm \).

\( a b \perp A B \text { and } o b \| O B \text { to intersect at } b \).

\( \text { 4. Measure } a b=5.5 cm , v_{b a}=0.55 m / s \).

\( b c=\frac{B C}{A B} \times a b=\frac{450}{1200} \times 5.5=2.06 cm \).

\( \text { 5. Join } O C \text {. Draw } c d \perp C D \text { and } o d \| \text { line of stroke of slider } D \text { to meet at } d \).

\( \text { 6. } v_{b}=o b=3.8 cm =0.38 m / s ; v_{d}=o d=2.3 cm =0.23 m / s \).

\( \text { 7. } v_{c d}=c d=3.6 cm =0.36 m / s \).

\( \omega_{c d}=\frac{v_{c d}}{C D}=\frac{0.36}{0.45}=0.8 rad / s ( ccw ) \).

Acceleration diagram

\( f_{a}^{n}=\frac{v_{a}^{2}}{O A}=\frac{(0.63)^{2}}{0.3}=1.323 m / s ^{2} \).

\( f_{b a}^{n}=\frac{v_{b a}^{2}}{A B}=\frac{(0.55)^{2}}{1.2}=0.252 m / s ^{2} \).

\( f_{c d}^{n}=\frac{v_{d c}^{2}}{C D}=\frac{(0.36)^{2}}{0.45}=0.288 m / s ^{2} \).

1. Draw acceleration diagram as shown in Fig.3.33(c).

\( \text { 2. Draw oa } \| O A \text { such that } o a=f_{a}^{n}=6.6 cm =1.323 m / s ^{2} \).

\( \text { 3. Draw } a b^{\prime} \| A B, a b^{\prime}=f_{b a}^{n}=0.252 m / s ^{2}=1.26 cm \).

\( \text { 4. Draw } b^{\prime} b \perp A B \text { and } o b \| O B \text { to intersect at } b \).

Joint ab. Measure ab=3.3 cm.

\( \text { 5. } b c=\frac{B C}{A B} \times a b=\frac{450}{1200} \times 3.3=1.237 cm . \text { Join } oc \).

\( \text { 6. Draw } c d^{\prime \prime} \| C D, c d^{\prime}=f_{d c}^{n}=0.288 m / s ^{2}=1.44 cm \).

\( \text { 7. Draw } d^{\prime} d \perp C D \text { and } o d \| \text { the line of stroke of slider } D \text { to intersect of } d \).

Join cd.

\( \text { 8. Linear acceleration of } D, f_{d}=o d=1 cm =0.2 m / s ^{2}\).

\( \text { 9. Linear tangential acceleration of } C D, f_{d c}^{t}=d^{\prime} d=6.5 cm =1.3 m / s ^{2} \).

\( \text { Angular acceleration of } C D, \propto_{c d}=\frac{f_{d c}^{t}}{C D}=\frac{1.3}{0.45}=2.9 rad / s ^{2}( cw ) \).
