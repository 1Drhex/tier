Guest Question.

The reduction of NO by \(H_{2}\) on \(La_{2}O_{3}\) and Sr-promoted \(La_{2}O_{3}\) was examined by Huang et al. [59]. Both \(N_{2}\) and\(N_{2}O\) were observed as products. The sequence of elementary steps proposed for the catalytic cycle was the following, which invoked one type of site (S) to adsorb and activate \(H_{2}\) while the second type (*) interacted with the oxygen-containing reaction intermediates. Stoichiometric numbers for an elementary step are included when needed

(1) \(2 \left[NO_{\left(g\right)} + * \xrightleftharpoons[]{K_{NO}} NO* \right]\)

 

(2) \(2 \left[H_{2_{\left(g\right)}} + 2S \xrightleftharpoons[]{K_{H_{2}}} 2 H-S \right] \)

 

(3) \(NO*+ H-S \xrightarrow[]{K_{o}} HNO* + S\)

 

(4) \(HNO*+ NO* \xrightarrow[]{K_{1}} N_{2}O * + OH*\)

 

(5) \(N_{2}O* \xrightarrow[]{K_{2}} N_{2_{\left(g\right)}} + O*\)

 

(6) \(N_{2}O* \xrightleftharpoons[K_{-3}]{K_{3}} N_{2}O_{\left(g\right)} +*\)

 

(7) \(OH* + H-S \xrightleftharpoons[]{K_{4}} H_{2}O_{\left(g\right)} + * + S\)

 

(8) \(2 H-S + O* \xrightleftharpoons[]{K_{5}} H_{2}O_{\left(g\right)}+ * + 2S\)

——————————————————————

(9) \(2 NO + 2 H_{2} \Longrightarrow N_{2} + 2 H_{2}O\)

Note that step 6 is just a reversible adsorption/desorption step that allows \(N_{2}O\) to leave the surface as a product as well as to react further, and this step is not part of the catalytic cycle to form \(N_{2}\). Derive rate laws for the areal rates of a) NO disappearance, b) \(N_{2}\) formation, and 3) \(N_{2}O\) formation. Youmay assume that NO* is the MARI on the * sites and that H coverage of the S sites is very low.

 

Admin
Solution

(a) \(r_{m_{NO} }= \frac{1}{m } \frac{dN_{i} }{v_{i} dt} = \frac{-1 }{2m} \frac{d N_{NO}}{dt} \)

\(r_{m_{NO} }= L_{*} L_{s}K_{o} \theta _{NO}\theta + L_{*} K_{o} \theta _{NO}\theta_{HNO}\) where \(L_{*}\) and \(L_{s}\) are site densities of * and S sites , respectively.

SSA on HNO* gives : \(\frac{d\left[HNO *\right] }{dt}= L_{*} L_{s}K_{o} \theta _{NO}\theta_{H} – L_{*} K_{1} \theta _{NO}\theta_{HNO}=0\) and \(r_{3} = r_{4}\)

 

so \(r_{m_{NO} }= – \frac{d N_{NO}}{dt}= 2 \ L_{*} L_{s}K_{o} \theta _{NO}\theta_{H}\)

 

Site balance on * sites: \(L_{*} = \left[*\right]+ \left[NO *\right]\) and \( K_{NO}= \frac{\left[NO *\right]}{P_{NO} \left[*\right]}\) so

 

\(\left[*\right] = \frac{L_{*} }{ \left(1+ K_{NO} P_{NO} \right) } , \left[NO *\right] = \frac{L_{*} K_{NO} P_{NO} }{ \left(1+ K_{NO} P_{NO}\right) }\) and \( \theta _{NO}=\frac{\left[NO *\right]}{L_{*}}= \frac{K_{NO} P_{NO}}{1+K_{NO} P_{NO}} \)

 

Site balance on S sites: \(L_{s} = \left[S\right]+ \left[H-S\right]\) and \( K_{H_{2} } = \frac{\left[H-S\right]^{2}}{P_{H_{2} } \left[S\right]^{2} } \) so

 

\(\left[S\right]= \frac{L_{s}}{\left(1+K^{1/2}_{H_{2} } P^{1/2}_{H_{2} } \right) } , \theta _{H} = \frac{\left[H-S\right]}{L_{s} }= \frac{K^{1/2}_{H_{2} } P^{1/2}_{H_{2} } }{1+ K^{1/2}_{H_{2} } P^{1/2}_{H_{2} } } = K^{1/2}_{H_{2} } P^{1/2}_{H_{2} } \ if \quad \theta _{H} <<1\)

 

Then \(r_{m_{NO}}= 2 L_{*} L_{s} K_{o} K_{NO} K^{1/2}_{H_{2} } P_{NO} P^{1/2}_{H_{2} }/\left(1+ K_{NO} P_{NO}\right) \)

 

(b) \(r_{m_{N_{2}}} = \frac{1}{m} \frac{dN_{N_{2} } }{dt} = L_{*} K_{2} \theta _{N_{2}o } \)

 

SSA on \(N_{2}O * gives : \frac{d\left[N_{2}O \right] }{dt}= L_{*} L_{s} K_{1} \theta _{HNO} \theta _{NO}- L_{*} K_{2} \theta _{N_{2}O } – L_{*} K_{3} \theta _{N_{2}O } + L_{*} K_{-3} P_{N_{2}O }\theta _{*} =0\)

 

so \(\left( L_{*} K_{2} + L_{*} K_{3}\right) \theta _{N_{2}O } = L_{*} L_{s} K_{1} \theta _{HNO} \theta _{NO} + L_{*} K_{-3} P_{N_{2}O } \theta _{*} \) and

\(\theta _{N_{2}O }= \frac{L_{s} K_{o} \theta _{NO} \theta _{H} + K_{-3} P_{N_{2}O } \theta _{*}}{ \left(K_{2}+K_{3} \right) } = \frac{L_{s} K_{o} K_{NO} K^{1/2}_{H_{2} } P_{NO } K^{1/2}_{H_{2} } }{ \left(K_{2}+K_{3} \right) \left(1+ K_{NO} P_{NO }\right) } + \frac{K_{-3} P_{N_{2}O } }{ \left(K_{2}+K_{3} \right) \left(1+ K_{NO} P_{NO }\right) }\)

 

thus \(r_{m_{N_2}}=\frac{L_* K_2}{K_2+K_3}\left[\frac{L_sK_o K_{NO}K_{H_2}^{1/2}P_{NO}P_{H_2}^{1/2}+K_{-3}P_{N_2O}}{(1+K_{NO}P_{NO})} \right] \)

 

(c) \(r_{m_{N_{2}O } }= \frac{1}{m} \frac{d N_{N_{2}O } }{dt} = L_{*} K_{3} \theta _{N_{2}O}-L_{*} K_{-3} P_{N_{2}O} \theta _{*}\)

\(= \frac{L_{*} K_{3}}{ K_{2}+ K_{3}} \left[\frac{L_{s} K_{o} K_{NO} K^{1/2}_{H_{2}} P_{NO} P^{1/2}_{H_{2}} + K_{-3} P_{N_{2}O} }{\left(1+ K_{NO} P_{NO}\right) } \right] – \frac{L_{*} K_{-3} P_{N_{2}O} }{\left(1+K_{NO} P_{NO} \right) }\)

 

\(= \frac{L_{*}}{K_{2} + K_{3}} \left[\frac{L_{s} K_{3} K_{o} K_{NO} K^{1/2}_{H_{2} } P_{NO} P^{1/2}_{H_{2} } + K_{3} K_{-3} P_{N_{2}O} – K_{2} K_{-3} P_{N_{2}O} – K_{3} K_{-3} P_{N_{2}O} }{ \left(1+ K_{NO} P_{NO} \right) } \right] \)

 

\(= \frac{L_{*}}{ K_{2}+K_{3}} \left[\frac{L_{s} K_{o}K_{3} K_{NO} K^{1/2}_{H_{2}} P_{NO} P^{1/2}_{H_{2}} – K_{2} K_{-3} P_{N_{2}O}}{\left(1+K_{NO} P_{NO} \right) } \right] \)
