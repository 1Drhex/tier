Neutrinos from Bananas!

An average banana has approximately 450 mg of potassium. Naturally occurring potassium consists of three isotopes: \({ }_{19}^{39} K\) \((93.3 \%),{ }_{19}^{40} K (0.012 \%) \text {, and }{ }_{19}^{41} K (6.73 \%) . \text { Both }{ }_{19}^{39} K \text { and }{ }_{19}^{41} K\) are stable, and \({ }_{19}^{40} K\) undergoes both \(\beta^{-} \text {and } \beta^{+}\) decays with a halflife of \(1.248 \times 10^{9} yr\):

 

\({ }_{19}^{40} K \rightarrow{ }_{18}^{40} Ar +{ }_{+1}^{0} e +v(11.2 \%)\)

 

\({ }_{19}^{40} K \rightarrow{ }_{20}^{40} Ca +{ }_{-1}^{0} e +\bar{v}(88.8 \%)\)

 

How many neutrinos and antineutrinos are emitted from a typical banana each second?

Admin
Solution

To determine how many neutrinos and antineutrinos are emitted from a banana, we need to know the number of \({ }_{19}^{40} K\) atoms in a banana. We first calculate the average atomic mass of potassium. From Table 34-5, we find that the atomic masses of the potassium isotopes are as follows:

 

\({ }_{19}^{39} K : 38.963708 u ;{ }_{19}^{40} K : 39.964000 u ;{ }_{19}^{49} K : 40.961827 u\)

 

The average atomic mass of naturally occurring potassium is

 

\(\begin{aligned}m_{ K }=&(38.963708 u \times 0.933)+(39.964000 u \times 0.00012) \\&+(40.961827 u \times 0.0673)=39.1147 u\end{aligned}\)

 

Therefore, 39.111 47 g of potassium contains \(6.02 \times 10^{23}\) atoms, and the number of potassium atoms in 450 mg is

 

\(\frac{6.02 \times 10^{23} \text { atoms }}{39.11147 g } \times 450 \times 10^{-3} g =6.93 \times 10^{21} \text { atoms }\)

 

Since 0.012% of naturally occurring potassium is \({ }_{19}^{40} K\), the number of \({ }_{19}^{40} K\) atoms in a typical banana is

 

\(6.93 \times 10^{21} \text { atoms } \times \frac{0.012}{100}=8.32 \times 10^{17} \text { atoms }\)

 

The half-life of \({ }_{19}^{40} K\) is the same for \(\beta^{-} \text {and } \beta^{+}\) decays. The activity of the potassium in a typical banana is

 

\(\begin{aligned}R &=\text { number of atoms } \times \text { decay rate } \\&=\text { number of atoms } \times \frac{\ln 2}{\text { half }-\operatorname{life}} \\&=8.32 \times 10^{17} \text { atoms } \times \frac{\ln 2}{1.248 \times 10^{9} yr \times 31557600 s / yr } \\&=14.6 \text { atoms } / s\end{aligned}\)

 

On average, 15 atoms of \({ }_{19}^{40} K\) decay each second. Since 88.8% of these decays are \(\beta^{-}\) decays producing an antineutrino, and 11.2% are \(\beta^{+}\) decays producing a neutrino, an average banana emits approximately 13 antineutrinos and 2 neutrinos each second.

 

Making sense of the result

 

Since naturally occurring potassium contains a small fraction of potassium isotopes that undergo \(\beta^{-} \text {and } \beta^{+}\) decays, it is obvious that a fruit that is rich in potassium would emit neutrinos. Our calculation confirms this.
