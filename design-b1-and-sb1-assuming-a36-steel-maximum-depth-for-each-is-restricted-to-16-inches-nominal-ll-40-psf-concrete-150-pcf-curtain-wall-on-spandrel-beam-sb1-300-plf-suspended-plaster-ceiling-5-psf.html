Guest Question.

Design B1 and SB1 assuming A36 steel. Maximum depth for each is restricted to 16 inches” nominal.

LL 40 psf

Concrete 150 pcf

Curtain wall on spandrel beam SB1 300 plf

Suspended plaster ceiling 5 psf

Metal deck 4 psf

ΔLL < L/360 for B1; ΔDL+LL < L/240 for SB1

Admin
Solution

Loads:

Conc. = (4/12ft.) × (150lb./ft.3) = 50 lb./ft.2

Metal deck = 4 lb./ft.2

Plaster ceiling = 5 lb./ft.2

Dead Load = 59 psf ωDL = (59 lb./ft.2)(8′) = 472 lb./ft.
Live Load = 40 psf ωLL = (40 lb./ft.2)(8′) = 320 lb/ft.
DL + LL = 99 psf
ω = 99 lb./ft.2 × 8’= 792 lb./ft.

 

\(M_{max} = \frac{\omega L^{2}}{8} = \frac{(792lb./ft.)(22^{'})^{2}}{8} = 48,000lb.-ft.\)

 

\(S_{req'd} = \frac{M}{F_{b}} = \frac{(48k-ft.)(12in./ft.)}{22k/in.^{2}} = 26.2in.^{3}\)

 

Try: W14 × 22 (Sx = 29 in.3, A = 6.49 in.2, Ix = 199 in.4)

Deflection check:

 

\(\Delta_{\underset{(LL)}{allow}} = \frac{L}{360} = \frac{(22^{'})(12in./ft.)}{360} = 0.73^{"}\)

 

\(\Delta_{act.} = \frac{5\omega _{LL}L^{4}}{384EI} = \frac{5(320lb.ft.)(22^{'})^{4}(1728)}{384(29 \times 10^{3})(199)} = 0.29^{"} \lt 0.73^{"}\) ∴ OK

 

Beam SB1:

Mmax = 95.6k-ft.

 

\(S_{req'd} = \frac{(95.6\times 12)}{22} = 52.1in.^{3}\)

 

Try: W16 × 36

(Sx = 56.5in.3, A=10.6in.2, Ix = 448in.4)

 

\(\Delta_{\underset{(D+L)}{allow}} = \frac{L}{240} = \frac{(24^{'})(12in./ft.)}{240} = 1.2^{"}\)

 

\(\Delta_{act.} = \frac{5\omega L^{4}}{384EI} + \frac{PL^{3}}{28.2EI}\)

 

\(\Delta_{act.} = \frac{5(.336)(24^{'})^{4}(1728)}{384(29\times 10^{3})(448)} + \frac{(8.95)(24^{'})^{3}(1728)}{28.2(29\times 10^{3})(448)}\)

 

Δact. = 0.193″ + 0.584″ = 0.78″ < 1.20″ ∴ OK

 

\(\underset{(ave)}{f_{v}} = \frac{V}{t_{w}d} = \frac{(12.55k)}{(0.295^{"})(15.86^{"})} = 2.7ksi \lt F_{v} = 14.5ksi\)

 

∴ OK in shear
Use: W16×36 for SB1
