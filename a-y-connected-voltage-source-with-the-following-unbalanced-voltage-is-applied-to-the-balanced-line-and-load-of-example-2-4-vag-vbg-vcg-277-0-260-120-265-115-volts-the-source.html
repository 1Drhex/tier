Solving unbalanced three-phase networks using sequence components

A Y-connected voltage source with the following unbalanced voltage is applied to the balanced line and load of .

\(\left [ \begin{matrix} V_{ag} \\ V_{bg} \\ V_{cg} \end{matrix} \right ] =\left [ \begin{matrix} 277\angle 0° \\ 260\angle -120°\\ 295\angle +115° \end{matrix} \right ]\) volts

The source neutral is solidly grounded. Using the method of symmetrical components, calculate the source currents \(I_{a}\), \(I_{b}\), and \(I_{c}\).

Admin
Solution

Using (8.1.13) through (8.1.15), the sequence components of the source voltages are

\(V_{0} =\frac{1}{3}\left( V_{a} +V_{b}+V_{c}\right)\) (8.1.13)

\(V_{1} =\frac{1}{3}\left( V_{a} +aV_{b}+a^{2} V_{c}\right) \) (8.1.14)

\(V_{2} =\frac{1}{3}\left( V_{a} +a^{2} V_{b}+a V_{c}\right) \) (8.1.15)

\(V_{0} =\frac{1}{3} \left(277\angle 0°+260\angle -120°+295\angle 115°\right)=7.4425+j14.065=15.912\angle 62.11° \) volts

\(V_{1} =\frac{1}{3}\left(277\angle 0°+260\angle -120°+120°+295\angle 115°+240°\right) =\frac{1}{3}\left(277\angle 0°+260\angle 0°+295\angle -5°\right) =276.96-j8.5703=277.1\angle -1.772°\) volts

\(V_{2} =\frac{1}{3}\left(277\angle 0°+260\angle -120°+240°+295\angle 115°+120°\right) =\frac{1}{3}\left(277\angle 0°+260\angle 120°+295\angle 235°\right)=-7.4017-j5.4944=9.218\angle 216.59°\) volts

These sequence voltages are applied to the sequence networks of the line and load, as shown in Figure 8.17. The sequence networks of this figure are uncoupled, and the sequence components of the source currents are easily calculated as follows:

\(I_{0}=\) 0

\(I_{1}=\frac{V_{1}}{Z_{L1}+\frac{Z_{\Delta } }{3} }=\frac{277.1\angle -1.772°}{10.73\angle 43.78°} =25.82\angle -45.55° \) A

\(I_{2}=\frac{V_{2}}{Z_{L2}+\frac{Z_{\Delta } }{3} }=\frac{9.218\angle 216.59°}{10.73\angle 43.78°} =0.8591\angle 172.81° \) A

Using (8.1.20) through (8.1.22), the source currents are

\(I_{a}=I_{0}+I_{1}+I_{2}\) (8.1.20)

\(I_{b}=I_{0}+a^{2}I_{1}+aI_{2}\) (8.1.21)

\(I_{c}=I_{0}+aI_{1}+a^{2}I_{2}\) (8.1.22)

\(I_{a}=\left(0+25.82\angle -45.55°+0.8591\angle 172.81°\right) =17.23-j18.32=25.15\angle -46.76°\) A

\(I_{b}=\left(0+25.82\angle -45.55°+240°+0.8591\angle 172.81°+120°\right)=\left(25.82\angle 194.45°+0.8591\angle 292.18°\right) =-24.67-j7.232=25.71\angle 196.34°\) A

\(I_{c}=\left(0+25.82\angle -45.55°+120°+0.8591\angle 172.81°+240°\right)=\left(25.82\angle 74.45°+0.8591\angle 52.81°\right) =7.441+j25.56=26.62\angle 73.77°\) A

You should calculate the line currents for this example without using symmetrical components in order to verify this result and to compare the two solution methods (see Problem 8.33). Without symmetrical components, coupled KVL equations must be solved. With symmetrical components, the conversion from phase to sequence components decouples the networks as well as the resulting KVL equations, as shown above.
