Guest Question.

Designing an astable multivibrator Design an astable multivibrator as in Fig. 16.49(a) so that k = 75% and \( f_{ o }=2.5 kHz \text {. Assume } V_{ CC }=12 V \). Use PSpice to verify by simulating, EX16-13.SCH.

Admin
Solution

\( k=75 \%=0.75, \text { and } T=1 / f_{ o }=1 / 2.5 kHz =400 \mu s . \) The steps used to design an astable multivibrator are as follows:

Step 1. Find the charging time \( t_{ c } \) and the discharging time \( t_{ d } \):

\(\begin{aligned}t_{ c } &=k T \\&=0.75 \times 400 \mu s =300 \mu s\end{aligned}\) (16.77)

 

\(\begin{aligned}t_{ d } &=(1-k) T \\&=(1-0.75) \times 400 \mu s =100 \mu s\end{aligned}\) (16.78)

 

Step 2. Choose a suitable value of C: Let C = 0.1 ÂµF.
Step 3. Find the value of \( R_{ B } \). From Eq. (16.73),

\( t_{ d }=C R_{ B } \ln (2)=0.69 C R_{ B } \) (16.73)

 

\(\begin{aligned}R_{ B } &=\frac{t_{ d }}{0.69 C} \\&=\frac{100 \mu s }{0.69 \times 0.1 \mu F }=1449 \Omega\end{aligned}\) (16.79)

 

Step 4. Find the value of \( R_{ A } \). From Eq. (16.70),

\( t_{ c }=C\left(R_{ A }+R_{ B }\right) \ln (2)=0.69 C\left(R_{ A }+R_{ B }\right) \) (16.70)

 

\(\begin{aligned}R_{ A } &=\frac{t_{ c }}{0.69 C}-R_{ B } \\&=\frac{300 \mu s }{0.69 \times 0.1 \mu F }-1449=2899 \Omega\end{aligned}\) (16.80)
