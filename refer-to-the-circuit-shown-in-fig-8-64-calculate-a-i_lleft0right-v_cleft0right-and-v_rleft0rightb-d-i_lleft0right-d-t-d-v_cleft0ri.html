Guest Question.

Refer to the circuit shown in Fig. 8.64. Calculate:

(a) \(i_{L}\left(0^{+}\right), v_{c}\left(0^{+}\right)\) and \(v_{R}\left(0^{+}\right)\)

(b) \(d i_{L}\left(0^{+}\right) / d t, d v_{c}\left(0^{+}\right) / d t\), and \(d v_{R}\left(0^{+}\right) / d t\)

(c) \(i_{L}(\infty), v_{c}(\infty) \)and \(v_{R}(\infty)\)

Admin
Solution

At \(\mathrm{t}=0^{-}, \mathrm{u}(\mathrm{t})=0 .\) Consider the circuit shown in Figure \((\mathrm{a}) \cdot \mathrm{i}_{\mathrm{L}}\left(0^{-}\right)=0,\) and \(\mathrm{v}_{\mathrm{R}}\left(0^{-}\right)=\) 0. But, \(-\mathrm{v}_{\mathrm{R}}\left(0^{-}\right)+\mathrm{v}_{\mathrm{C}}\left(0^{-}\right)+10=0,\) or \(\mathrm{v}_{\mathrm{C}}\left(0^{-}\right)=-10 \mathrm{V}\)

(a) At \(t=0^{+}\), since the inductor current and capacitor voltage cannot change abruptly, the inductor current must still be equal to \({{0}} {A},\) the capacitor has a voltage equal to
-10V. since it is in series with the \(+10 \mathrm{V}\) source, together they represent a direct short at \(t=0^{+}\). This means that the entire 2 A from the current source flows through the capacitor and not the resistor. Therefore, \(\mathrm{v}_{\mathrm{R}}\left(0^{+}\right)={\mathbf{0} \mathrm{V}}\)

\(\begin{aligned}&\text { (b) } \quad \text { At } \mathrm{t}=0^{+}, \mathrm{v}_{\mathrm{L}}(0+)=0, \text { therefore } \operatorname{Ldi}_{\mathrm{L}}(0+) / \mathrm{dt}=\mathrm{v}_{\mathrm{L}}\left(0^{+}\right)=0, \text { thus, } \mathrm{di}_{\mathrm{L}} / \mathrm{dt}={\mathbf{0} \mathbf{A} / \mathbf{s}}\\&\mathrm{i}_{\mathrm{C}}\left(0^{+}\right)=2 \mathrm{A}, \text { this means that } \mathrm{d} \mathrm{v}_{\mathrm{C}}\left(0^{+}\right) / \mathrm{dt}=2 / \mathrm{C}={\mathbf{8}} \mathrm{V} / \mathrm{s} . \text { Now for the value of }\\&\mathrm{d} v_{\mathrm{R}}\left(0^{+}\right) / \mathrm{dt} . \text { since } \mathrm{v}_{\mathrm{R}}=\mathrm{v}_{\mathrm{C}}+10, \text { then } \mathrm{d} \mathrm{v}_{\mathrm{R}}\left(0^{+}\right) / \mathrm{dt}=\mathrm{d} \mathrm{v}_{\mathrm{C}}\left(0^{+}\right) / \mathrm{dt}+0={\mathbf{8} \mathbf{V} / \mathbf{s}}\end{aligned}\)

(c) As t approaches infinity, we end up with the equivalent circuit shown in Figure (b).

\(\begin{array}{c}\mathrm{i}_{\mathrm{L}}(\infty)=10(2) /(40+10)={400 \mathrm{mA}} \\\mathrm{v}_{\mathrm{C}}(\infty)=2[10 \| 40]-10=16-10={6 \mathrm{V}} \\\mathrm{V}_{\mathrm{R}}(\infty)=2[10 \| 40]={16 \mathrm{V}}\end{array}\)
