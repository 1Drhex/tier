Guest Question.

Answer the following questions about:
1. Can multiple measurements improve the estimated mobile system position (the mobile system does not move between the measurements)?
2. What is the probability distribution of the mobile system position if the mobile system detects a tile as dark twice in a row?
3. What is the probability distribution of the mobile system position if the mobile system detects a tile as dark and then as bright?
4. What is the probability distribution of the mobile system position if the mobile system detects a tile as dark, then as bright, and then again as dark?

Admin
Solution

1. Multiple measurements can improve the estimate about the mobile
system position if the probability of the correct measurement is higher
than the probability of the measurement mistake.
2. The probability distribution if the sensor detects cell as dark twice in a row is the following:

\(\begin{aligned}& p \left(X_{2} \mid Z_{1}=\text { dark }, Z_{2}=\text { dark }\right)=p\left(X_{2} \mid z_{1}, z_{2}\right) \\&\quad=\frac{ p \left(z_{2} \mid X_{2}\right) * p \left(X_{2} \mid z_{1}\right)}{P\left(z_{2} \mid z_{1}\right)} \\&\quad=\frac{[0.2,0.2,0.6,0.6,0.2]^{T} *[0.11,0.11,0.33,0.33,0.11]^{T}}{P\left(z_{2} \mid z_{1}\right)}\end{aligned}\)

where jth element in the probability distribution \(p\left(X_{2} \mid z_{1}\right)\) is given by \(P\left(X_{2}=x_{j} \mid z_{1}\right)= p ^{T}\left(X_{2}=x_{j} \mid X_{1}\right) p \left(X_{1} \mid z_{1}\right)=P\left(X_{1}=x_{j} \mid z_{1}\right)\), since we have no influence on the states (see ); we only
observe the states with measurements. The conditional probability in
the denominator is

\(\begin{aligned}P\left(z_{2} \mid z_{1}\right) &=\sum_{x_{i}} P\left(z_{2} \mid X_{2}=x_{i}\right) P\left(X_{2}=x_{i} \mid z_{1}\right) \\&=p^{T}\left(z_{2} \mid X_{2}\right) p\left(X_{2} \mid z_{1}\right) \\&=[0.2,0.2,0.6,0.6,0.2][0.11,0.11,0.33,0.33,0.11]^{T} \\&=0.4667\end{aligned}\)

Finally, the solution is

\(\begin{aligned}p\left(X_{2} \mid z_{1}, z_{2}\right) &=\frac{[0.2,0.2,0.6,0.6,0.2]^{T} *[0.11,0.11,0.33,0.33,0.11]^{T}}{[0.2,0.2,0.6,0.6,0.2][0.11,0.11,0.33,0.33,0.11]^{T}} \\&=[0.0476,0.0476,0.4286,0.4286,0.0476]^{T}\end{aligned}\)

3. The bright cell is detected correctly with probability \(p(Z=\text { bright } \mid X=\) \(\text { bright })=1-p(Z=\text { dark } \mid X=\text { bright })=0.8\) and incorrectly with probability \(p(Z=\text { bright } \mid X=\text { dark })=1-p(Z=\text { dark } \mid X=\text { dark })=\) = 0.4. The second measurement can be made based on the probability distribution \(p\left(X_{2} \mid Z_{1}=\text { dark }\right)\)

\(\begin{gathered}p\left(X_{2} \mid Z_{1}=\text { dark }, Z_{2}=\text { bright }\right)=p\left(X_{2} \mid z_{1}, z_{2}\right) \\=\frac{[0.8,0.8,0.4,0.4,0.8]^{T} *[0.11,0.11,0.33,0.33,0.11]^{T}}{P\left(Z_{2}=\text { bright } \mid Z_{1}=\text { dark }\right)} \\P\left(Z_{2}=\text { bright } \mid Z_{1}=\text { dark }\right) \\=\sum_{x_{i}} P\left(Z_{2}=\text { bright } \mid X_{2}=x_{i}\right) P\left(X_{2}=x_{i} \mid Z_{1}=\text { dark }\right) \\=p^{T}\left(Z_{2}=\text { bright } \mid X_{2}\right) p\left(X_{2} \mid Z_{1}=\text { dark }\right) \\=[0.8,0.8,0.4,0.4,0.8][0.11,0.11,0.33,0.33,0.11]^{T}=0.533 \\p\left(X_{2} \mid Z_{1}=\text { dark, } Z_{2}=\text { bright }\right)=[0.167,0.167,0.25,0.25,0.167]^{T}\end{gathered}\)

4. The state probability distribution after three measurements were made is

\(\begin{aligned}&p\left(X_{3} \mid Z_{1}=\text { dark }, Z_{2}=\text { bright }, Z_{3}=\text { dark }\right) \\&\quad=[0.083,0.083,0.375,0.375,0.083]^{T}\end{aligned}\)
