Guest Question.

Find \(v\left(0^{+}\right) \text {and } i_{1}\left(0^{+}\right)\) for the circuit shown in Fig. 8.22a if \(v\left( 0 ^{-}\right)=V_{0}\).

Admin
Solution

We first simplify the circuit of Fig. 8.22a to that of Fig. 8.22b, enabling us to write

 

\(v=V_{0} e^{-t / R_{ eq } C}\)

 

where

 

\(v\left(0^{+}\right)=v\left(0^{-}\right)=V_{0} \quad \text { and } \quad R_{ eq }=R_{2}+\frac{R_{1} R_{3}}{R_{1}+R_{3}}\)

 

Every current and voltage in the resistive portion of the network must have the form \(A e^{-t / R_{ eq } C}\), where A is the initial value of that current or voltage. Thus, the current in \(R_{1}\), for example, may be expressed as

 

\(i_{1}=i_{1}\left(0^{+}\right) e^{-t / \tau}\)

 

where

 

\(\tau=\left(R_{2}+\frac{R_{1} R_{3}}{R_{1}+R_{3}}\right) C\)

 

and \(i_{1}\left(0^{+}\right)\) remains to be determined from the initial condition.Any current flowing in the circuit at \(t=0^{+}\) must come from the capacitor. Therefore, since v cannot change instantaneously, \(v\left(0^{+}\right)=v\left(0^{-}\right)=V_{0}\) and

 

\(i_{1}\left(0^{+}\right)=\frac{V_{0}}{R_{2}+R_{1} R_{3} /\left(R_{1}+R_{3}\right)} \frac{R_{3}}{R_{1}+R_{3}}\)
