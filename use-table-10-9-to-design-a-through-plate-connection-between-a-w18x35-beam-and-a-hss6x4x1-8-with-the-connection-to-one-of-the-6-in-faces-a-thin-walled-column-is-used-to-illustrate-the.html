Guest Question.

Use Table 10-9 to design a through-plate connection between a W18×35 beam and a HSS6×4×1/8 with the connection to one of the 6 in. faces. A thin-walled column is used to illustrate the design of a through-plate connection.
\(\begin{array}{l}P_{D}=6.5 kips \\P_{L}=19.5 kips\end{array}\)
Use 3/4-in. diameter ASTM A325-N bolts in standard holes and 70 ksi weld electrode.

 

Admin
Solution

Material Properties:

Beam ASTM A992 \(F_{y}=50 ksi \quad F_{.1}=65 ksi\)

Column ASTM A500 Gr. B \(F_{y}=46 ksi \quad F_{u}=58 ksi\)

Plate ASTM A36 \(F_{y}=36 \text { ksi } \quad F_{u}=58 ksi\)

Manual Tables
2-3 and 2-4

 

Geometric Properties:

Beam W18×35 \(d=17.7 \text { in. } \quad t_{w}=0.300 \text { in. } \quad T=151 / 2 in .\)

Column HSS6×4×1/8 B = 4.00 in. H = 6.00 in. t = 0.116 in. h/t = 48.7

Manual
Tables 1-1
and 1-11

Determine applicability of Specification Section K1

Strength: \(F_{y}=46 ksi <52 ksi\) o.k.

Ductility: \(\frac{F_{y}}{F_{u}}=\frac{46 ksi }{58 ksi }=0.79<0.80\) o.k.

Section K1.2

Determine if a single-plate connection is suitable (HSS wall is not slender)

Slenderness: \(\lambda=1.40 \sqrt{\frac{E}{F_{y}}}=1.40 \sqrt{\frac{29,000 ksi }{46 ksi }}=35.2<48.7\) n.g.

Since the HSS6×4×1/8 is slender, a through-plate connection should be used instead of a single-plate connection. Through plate connections are typically very expensive. When a single-plate connection is not adequate, another type of connection, such as a double-angle connection may be preferable to a through-plate connection.

Calculate the required strength

LRFD ASD
\(\begin{aligned}R_{u} &=1.2(3.30 kips )+1.6(9.90 kips ) \\&=19.8 kips\end{aligned}\) \(\begin{aligned}R_{a} &=3.30 \text { kips }+9.90 \text { kips } \\&=13.2 \text { kips }\end{aligned}\)

Design the portion of the through-plate connection that resembles a single-plate

Try 3 rows of bolts (L=8 1/2) and a 1/4 in. plate thickness with 3/16-in. fillet welds.

Obtain the available single plate connection strength from Manual Table 10-9

LRFD ASD
\(\phi R_{n}=38.3 \text { kips }>19.8 \text { kips }\) o.k. \(R_{n} / \Omega=25.6 \text { kips }>13.2 \text { kips }\) o.k.

Manual
Table 10-9a

Check weld size for through-plate connection

e = 3.00 in.

\(t_{\min }=\frac{6.19 D}{F_{u}}=\frac{6.19(3)}{58 ksi }=0.320 \text { in. }>0.116 \text { in. }\) (proration required)

Calculate the required weld strength

LRFD ASD
\(\begin{aligned}V_{f i u} &=\frac{R_{u}(B+e)}{B} \\&=\frac{(19.8 kips )(4.00 in .+3.00 in .)}{4.00 in .} \\&=34.7 kips\end{aligned}\)

\(v_{f u}=\frac{V_{f u}}{L}=\frac{34.7 kips }{8.50 in .}=4.08 kips / in\)

 \(\begin{aligned}V_{f a} &=\frac{R_{a}(B+e)}{B} \\&=\frac{(13.2 kips )(4.00 in .+3.00 in. )}{4.00 in. } \\&=23.1 kips\end{aligned}\)

\(v_{f a}=\frac{V_{f a}}{L}=\frac{23.1 kips }{8.50 in .}=2.72 kips / in\)

Calculate the available weld strength

LRFD ASD
\(\begin{aligned}\phi r_{n} &=1.392 D n_{w}\left(\frac{t}{t_{\min }}\right) \\&=1.392(3)(2 \text { welds })\left(\frac{0.116 \text { in. }}{0.320 \text { in. }}\right) \\&=3.03 \text { kips/in. }<4.08 \text { kips/in. }\end{aligned}\)

 

A deeper plate is required.

 

\(L_{r e q}=\frac{V_{f u}}{\phi r_{n}}=\frac{34.7 kips }{3.03 kips / in .}=11.5 in\)

 \(\begin{aligned}\phi r_{n} &=0.928 D n_{w}\left(\frac{t}{t_{\min }}\right) \\&=0.928(3)(2 \text { welds })\left(\frac{0.116 in }{0.320 in }\right) \\&=2.02 kips / in .<2.72 kips / in .\end{aligned}\)

 

A deeper plate is required.

 

\(L_{r e q}=\frac{V_{f a}}{r_{n} / \Omega}=\frac{23.1 kips }{2.02 kips / in .}=11.5 in\)

For three 3/4 in. diameter bolts
