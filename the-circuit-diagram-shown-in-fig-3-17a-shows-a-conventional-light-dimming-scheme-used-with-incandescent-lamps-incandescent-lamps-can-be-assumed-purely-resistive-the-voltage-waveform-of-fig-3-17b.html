Guest Question.

The circuit diagram shown in Fig. 3.17a shows a conventional light-dimming scheme used with incandescent lamps (incandescent lamps can be assumed purely resistive). The voltage waveform of Fig. 3.17b shows the input and output voltages for the dimming circuit shown at an instant the dimmer is reducing light intensity. For this lighting control application, assuming that the incandescent lamp acts like a pure resister, determine the following:
(a) The Fourier series coefﬁcients for the fundamental component of the source current \(\ i_{s} \left(t\right) \)

(b) The \(\ THD_{i} \) of the source current \(\ i_{s} \left(t\right) \)

(c) The power factor as seen by the source

Assume the \(\ V_{s} \left(t\right) = V_{s ,peak} \sin \omega t =120\sqrt{2} \sin \left(2.\pi .60t\right) V \), and the rated lamp power is 100 W at 120Vrms.

Admin
Solution

(a) Since it is assumed that the incandescent lamp acts like a pure resister, the current \(\ i_{s} \left(t\right) \) will be the same shape as the output voltage as shown in Fig. 3.17c and given by:

\(\ i_{s} \left(t\right) = I_{s ,peak} \sin \omega t \)

with its peak \(\ I_{s ,peak} \) is given by:

\(\ I_{s ,peak}=\frac{ V_{s ,peak}}{R_{lamp} } =\frac{120\sqrt{2} }{144} = 1.2 A \)

where \(\ R_{lamp} \) is equivalent lamp resistance at the full rated power, which is obtained from:

\(\ R_{lamp}=\frac{V^{2}_{rms} }{P_{o} } =\frac{120^{2} }{100} = 144\Omega \)

By inspection, the current \(\ i_{s} \left(t\right) \) is half-wave symmetric with:

\(\ i_{s} \left(t\right) =i_{s} \left(t – \frac{T}{2} \right) \)

Hence, its dc value is zero and coefﬁcients \(\ a_{n} \) & \(\ b_{n} \) =0 for n even and for n odd:

\(\ a_{n} =\frac{4}{T} \int_{0}^{\frac{T}{2} }{f\left(t\right) \cos n\omega t dt } \)

 

\(\ b_{n} =\frac{4}{T} \int_{0}^{\frac{T}{2} }{f\left(t\right) \sin n\omega t dt } \)

For n= 1, the Fourier series coefﬁcients for the fundamental component of \(\ i_{s} \left(t\right) \) are obtained, where θ=ωt:

\(\ a_{1} =\frac{2}{\pi } \int_{\frac{\pi }{2} }^{\pi }{I_{s ,peak} \sin \theta \cos \theta d\theta } \)

 

\(\ b_{1} =\frac{2}{\pi } \int_{\frac{\pi }{2} }^{\pi }{I_{s ,peak} \sin^{2}\theta d\theta } \)

Using the trigonometric relation \(\ \sin \theta \cos \theta = \frac{1}{2} \sin 2\theta and \sin^{2}\theta = \frac{1}{2} \left(1 – \cos 2\theta \right) \) and evaluating the above two integrals, it can be shown that the coefﬁcients a1 and b1 are given by:

\(\ a_{1} =\frac{I_{s ,peak}}{2} = -0.38 \)

 

\(\ b_{1} = \frac{I_{s ,peak}}{2} = 0.59 \)

Therefore

\(\ {I_{s1 ,peak}} = \sqrt{a^{2}_{1} + b^{2}_{1} } = 0.7 A \)

 

\(\ {I_{s1 ,rms}} = \frac{I_{s ,peak}}{\sqrt{2} } =0.49 A \)

and the fundamental source current, \(\ i_{s1} \left(t\right) \), may be expressed by:

\(\ i_{s1} \left(t\right) = I_{s1 ,peak} \sin \left(\omega t +\varphi ^{\circ } \right) \)

 

\(\ i_{s1} \left(t\right) = 0.7 \sin \left(2\pi 60 t +32.78^{\circ } \right) \)

(b) \(\ THD_{i} \) for \(\ i_{s} \left(t\right) \) is given by:

\(\ THD_{i} = \sqrt{\left(\frac{I_{s ,rms}}{I_{s1 ,rms}} \right) ^{2} -1 } \times 100 \) %

The rms value for the current \(\ i_{s} \left(t\right) ,I_{s ,rms} \) , is obtained from:

\(\ I_{s ,rms} =\frac{V_{o,rms} }{R_{lamp} } =\frac{84.85}{144} =0.59 A \)

Therefore:

\(\ THD_{1}=\sqrt{\left(\frac{0.59}{0.5} \right) ^{2} -1 } \times 100 \) % =62.6 %

(c) The power factor is given by:

\(\ PF = \frac{I_{s1 ,rms}}{I_{s ,rms}} \cos \left(\theta _{v} -\theta _{i} \right) =\frac{0.49}{0.59} \cos \left(32.78^{\circ } \right) =0.7 \)

Therefore, current (fundamental) lags voltage by \(\ 32.78^{\circ } \) .
