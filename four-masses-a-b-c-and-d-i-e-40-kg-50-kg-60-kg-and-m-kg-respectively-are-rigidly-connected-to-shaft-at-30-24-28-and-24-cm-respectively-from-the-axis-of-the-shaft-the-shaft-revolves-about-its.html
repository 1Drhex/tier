Guest Question.

Four masses, A, B, C and D, i.e. 40 kg, 50 kg, 60 kg and M kg respectively are rigidly connected to shaft at 30, 24, 28 and 24 cm, respectively from the axis of the shaft. The shaft revolves about its axis and the planes of revolution of masses are at equal intervals apart. Determine M and the angular positions of B, C and D in relation to that of A in order that masses may completely balance one another.

Reference plane D (Fig.12.56)

Couple Polygon: Assume the position of mass A in horizontal direction to the right. Draw the couple polygon shown in Fig.12.56(c) as described below from the data in column 6 of Table 12.39.

\( \text { 1. Draw } o^{\prime} a^{\prime}=36 d \text { horizontally to the left, being negative, to a scale of } 1 cm =5 d \text { units. } \)

\( \text { 2. From points } o^{\prime} \text { and } a^{\prime} \text {, draw arcs } o^{\prime} b^{\prime}=24 d \text { and } a^{\prime} b^{\prime}=16.8 d \).

\( \text { 3. By measurement, we find that } \theta_{B}=238^{\circ} \text { and } \theta c=33^{\circ} \).

Force polygon: To find the magnitude of mass M at D and its angular location, draw the force polygon as described below from the data in column 4 of Table 12.39.

\( \text { 1. Draw } o a=12 \text { units to the right to a scale of } 1 cm =2 \text { units. } \)

\( \text { 2. Draw } a b=12 \text { units and parallel to } a^{\prime} b^{\prime}\).

\( \text { 3. Draw } b c=16.8 \text { units and parallel to } o^{\prime} b^{\prime}\).

4. Join oc. Then oc = 0.24 M. By measurement oc = 7.8 cm = 15.6 units.

0.24 M = 15.6
M = 65 kg

\( \text { and } \theta_{D}=200^{\circ} \).

The angular position of the masses B, C and D in relation to A have been shown in Fig.12.56(b).
