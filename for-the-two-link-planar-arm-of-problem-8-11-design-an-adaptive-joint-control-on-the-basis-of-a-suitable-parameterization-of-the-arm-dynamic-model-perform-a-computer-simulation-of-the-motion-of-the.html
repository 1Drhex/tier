Guest Question.

For the two-link planar arm of , design an adaptive joint control, on the basis of a suitable parameterization of the arm dynamic model. Perform a computer simulation of the motion of the controlled arm along the trajectory specified in . Implement the control in discrete–time with a sampling time of 1ms.

Admin
Solution

A minimum parameterization of the dynamic model is obtained as in the solution to , with suitable modification of augmented Link 2 parameters as in the solution to . The generation of joint trajectories is also accomplished as in the solution to . The control scheme in Fig. 8.26 is used. In terms of the new reference velocity and acceleration vectors in (8.92), the regressor in (S7.3) becomes

\( y^{\prime }_{11} \) = \( a_{1}\ddot{\vartheta _{1} } + g c_{1}\)

\( y^{\prime }_{12} \) = \(\ddot{\vartheta _{1} } \)

\( y^{\prime }_{13} \) = \(\left(2a_{1}c_{2} + a_{2}\right) \ddot{\vartheta _{1} } + \left(a_{1}c_{2} + a_{2}\right) \ddot{\vartheta _{2} } – 2 a_{1} s_{2}\dot{\vartheta _{1} }\dot{\vartheta _{2} } – a_{1} s_{2}\dot{\vartheta }^{2}_{2} + g_{ c_{12}}\)

\( y^{\prime }_{14} \) = \( \ddot{\vartheta _{1} } + \ddot{\vartheta _{2} }\)

\( y^{\prime }_{15} \) = \(\ddot{\vartheta _{1} } + k_{r_{2}} \ddot{\vartheta _{2} } \)

\( y^{\prime }_{21} \) = 0

\( y^{\prime }_{22} \) = 0

\( y^{\prime }_{23} \) = \(a_{2}\ddot{\vartheta _{2} } + \left(a_{1} c_{2} + a_{2}\right)\ddot{\vartheta _{1} } + a_{1}s_{2}\dot{\vartheta }^{2}_{1} + gc_{12}\)

\( y^{\prime }_{24} \) = \(\ddot{\vartheta _{2} } + \ddot{\vartheta _{1} }\)

\( y^{\prime }_{25} \) = \( k^{2}_{r2}\ddot{\vartheta _{2} } + k_{r2} \ddot{\vartheta _{1} } \) (S7.3)

\( \dot{q}_{r} \) = \(\dot{q}_{d} + \Lambda \tilde{q}\) \( \ddot{q}_{r} \) = \(\ddot{q}_{d} + \Lambda \dot{\tilde{q} }\) (8.92)

\( y^{\prime }_{11}\) = \( a_{1} \ddot{\vartheta } _{r1} + g c_{1} \)

\( y^{\prime }_{12}\) = \( \ddot{\vartheta } _{r1} \)

\( y^{\prime }_{13}\) = \(\left(2 a_{1} c_{2} + a_{2} \right) \ddot{\vartheta }_{r1} + \left( a_{1} c_{2} + a_{2} \right) \ddot{\vartheta }_{r2} – a_{1} s_{2}\dot{\vartheta }_{r1} \dot{\vartheta }_{2} – a_{1} s_{2}\dot{\vartheta }_{1} \dot{\vartheta }_{r2} – a_{1} s_{2}\dot{\vartheta }_{2} \dot{\vartheta }_{r2} + g c_{12}\)

\( y^{\prime }_{14}\) = \(\ddot{\vartheta }_{r1} + \ddot{\vartheta }_{r2} \)

\( y^{\prime }_{15}\) = \( \ddot{\vartheta }_{r1} + k_{r2} \ddot{\vartheta }_{r2}\)

\( y^{\prime }_{21}\) = 0

\( y^{\prime }_{22}\) = 0

\( y^{\prime }_{23}\) = \(\left( a_{1} c_{2} + a_{2} \right) \ddot{\vartheta }_{r1} + a_{2} \ddot{\vartheta }_{r2} + a_{1} s_{2}\dot{\vartheta }_{1} \dot{\vartheta }_{r1} + g c_{12} \)

\( y^{\prime }_{24}\) = \(\ddot{\vartheta }_{r1} + \ddot{\vartheta }_{r2} \)

\( y^{\prime }_{25}\) = \( k_{r2} \ddot{\vartheta }_{r1} + k^{2}_{r2} \ddot{\vartheta }_{r2} \)

In view of the presence of a concentrated tip load mass, the only variations occurring on the parameters in (S7.2) are:

\(\pi^{\prime }_{1}\) = \(a_{1} \pi _{1} + \pi _{2} + a_{1} \pi _{5} \) =\( a_{1}m_{1} + m_{1} l_{C_{1}} + a_{1}m_{2}\)

\(\pi^{\prime }_{2}\) = \(a_{1} \pi _{2} + \pi _{3} + k^{2}_{r1}\pi _{4}\) = \( a_{1}m_{1} l_{C_{1}} + \bar{I_{1} } + k^{2}_{r1} l_{m_{1}}\)

\(\pi^{\prime }_{3}\) = \(a_{2} \pi _{5} + \pi _{6} \) = \( a_{2}m_{2} + m_{2} l_{C_{2}}\)

\(\pi^{\prime }_{4}\) = \(a_{2} \pi _{6} + \pi _{7} \) = \( a_{2} m_{2} l_{C_{2}} + \bar{I_{2} } \)

\(\pi^{\prime}_{5}\) = \(\pi_{8}\) = \( l_{m_{2}}\) (S7.2)

\(\Delta \pi^{\prime }_{1} \) = \( a_{1} m_{L}\) \(\Delta \pi^{\prime }_{3} \) = \( a_{2} m_{L}\)

Hence, the control law in (8.98) can be formally written as

u = \(\hat{B} \left(q\right)\ddot{q}_{r} + \hat{C} \left(q,\dot{q} \right) \dot{q}_{r} + \hat{F} \dot{q}_{r} + \hat{g} + K_{D} \sigma \) = \( Y\left(q,\dot{q},\dot{q}_{r},\ddot{q}_{r}\right) \hat{\pi } + K_{D} \sigma\) (8.98)

u = \( Y^{\prime }\left(q,\dot{q} ,\dot{q}_{r} , \ddot{q}_{r}\right)\pi + F_{\vartheta }\dot{q}_{r} + m_{L} y_{L} + K_{D}\sigma\)

where \( Y^{\prime } \) is computed as above and

\(y_{L} \) = \(\left[\begin{matrix} a_{1}y^{\prime }_{11} + a_{2} y^{\prime }_{13}\\ a_{1} y^{\prime }_{21} + a_{2} y^{\prime }_{23}\end{matrix} \right] \)

being \(\pi \) = \(m_{L} \) the only unknown parameter.

The gains are chosen as

Λ = 10 I \(K_{D} \) = 10000 I \(k_{\pi} \) = 0.005;

the initial estimate of π is set to 0, and the true value of \(m_{L} \) is utilized only to update the simulated arm model.

The resulting Simulink block diagram is shown in Fig. S8.17. The arm is simulated as a continuous-time system using a variable-step integration method with a maximum step size of 1 ms. All the blocks of the controller are simulated as discrete-time subsystems with the given sampling time of 1 ms.

The files with the solution can be found in Folder 81-4.

The resulting joint position errors and load mass estimate are shown in Fig. S 8.18. For the joint position errors, the solid line refers to Joint 1, while the dashed line refers to Joint 2. It can be seen that the tracking performance is satisfactory, thanks to the adaptive action on the load mass; in this case, the parameter estimate converges to the true value.
