Guest Question.

The member shown in Figure 6.14 is a W12 × 65 of A242 steel. First-order analyses were performed with reduced member stiffnesses. The approximate secondorder analysis method of AISC Appendix 8 can be used, making this a direct analysis method. For LRFD, the analysis results for the controlling factored load combination are \(P_{nt} = 300\) kips, \(M_{ntx} = 135\) kips, and \(M_{nty} = 30\) ft-kips. For ASD, the analysis results for the controlling load combination are \(P_{nt} = 200\) kips, \(M_{ntx} = 90\) ft-kips, and \(M_{nty} = 20\) ft-kips. Use \(K_{y} = 1.0\), and investigate this member for compliance with the AISC Specification.

Admin
Solution

First, determine the yield stress \(F_{y}\). From Table 2-4 in Part 2 of the Manual, we see that A242 steel is available in three different versions. From the dimensions and properties table in Part 1 of the Manual, a W12 × 65 has a flange thickness of \(t_{f} = 0.605\) in. This matches the thickness range corresponding to footnote l in Table 2-4; therefore, \(F_{y} = 50\) ksi.

Compute the strong axis bending moments.

\(C_{mx}=0.6-0.4(\frac{M_{1}}{M_{2}})=0.6-0.4(0)=0.6\)

Since a modified flexural rigidity, \(EI^{*}\), was used in the frame analysis, it must also be used in the computation of \(P_{e1}\). From Equation 6.8,

\(E I^{*} = 0.8\tau_{b} E I = 0.8(1.0)E I = 0.8 E I\)

(In this book, we assume that the requirements for using \(\tau_{b} = 1.0\) have been met.)

\(P_{e1x}=\frac{^{2}E I^{*}}{(K_{1}L)^{2}}=\frac{^{2}(0.8 EI_{x})}{(K_{x}L)^{2}}=\frac{^{2}(0.8) (29,000)(533)}{(1.0×15×12)^{2}}=3767\) kips

 

\(B_{1x}=\frac{C_{mx}}{1-(αP_{r}/P_{e1x})}=\frac{C_{mx}}{1-(1.00P_{u}/P_{e1x})}=\frac{0.6}{1-(300/3767)}\)

\(=0.652<1.0\) \( ∴\) Use \(B_{1x}=1.0\).

The required moment strength is

\(M_{r }= M_{ux} = B_{1x}M_{ntx} + B_{2x}M_{\ell tx} = 1.0(135) + 0 = 135.0\) ft-kips

From the Beam Design Charts with \(C_{b} = 1.0\) and \(L_{b} = 15\) feet, the moment strength is

\(\phi_{b}M_{nx }= 340\) ft-kips and \(\phi_{b}M_{px} = 356\) ft-kips

From Figure 5.15g, \(C_{b} = 1.67\) and

\(C_{b }× (\phi_{b}M_{nx }\) for \(C_{b} = 1.0) = 1.67(340) = 567.8\) ft-kips

This result is larger than \(\phi_{b}M_{px}\); therefore, use \(\phi_{b}M_{nx} = \phi_{b}M_{px} = 356\) ft-kips. Compute the weak axis bending moments.

\(C_{my}=0.6-0.4(\frac{M_{1}}{M_{2}})=0.6-0.4(0)=0.6\)

 

\(P_{e1y}=\frac{^{2}E I^{*}}{(K_{1}L)^{2}}=\frac{^{2}(0.8 EI_{y})}{(K_{y}L)^{2}}=\frac{^{2}(0.8) (29,000)(174)}{(1.0×15×12)^{2}}=1230\) kips

 

\(B_{1y}=\frac{C_{my}}{1-(αP_{r}/P_{e1y})}=\frac{C_{my}}{1-(1.00P_{u}/P_{e1y})}=\frac{0.6}{1-(300/1230)}\)

\(=0.794<1.0\) \( ∴\) Use \(B_{1y}=1.0\).

The required moment strength is

\(M_{r }= M_{uy} = B_{1y}M_{nty} + B_{2y}M_{\ell ty} = 1.0(30) + 0 = 30\) ft-kips

Because the flange of this shape is noncompact (see footnote in the dimensions and properties table), the weak axis bending strength is limited by FLB (see Section 5.15 of this book and Chapter F of the AISC Specification).

\(\lambda=\frac{b_{f}}{2t_{f}}=9.92\)

 

\(\lambda_{p}=0.38\sqrt{\frac{E}{F_{y}}}=0.38\sqrt{\frac{29,000}{50}}=9.152\)

 

\(\lambda_{r}=1.0\sqrt{\frac{E}{F_{y}}}=1.0\sqrt{\frac{29,000}{50}}=24.08\)

Since \(\lambda_{p}<\lambda<\lambda_{r}\)

\(M_{n}=M_{p}-(M_{p}-0.7 F_{y}S_{y})(\frac{\lambda-\lambda_{p}}{\lambda_{r}-\lambda_{p}})\) (AISC Equation F6-2)

 

\(M_{p}=M_{py}=F_{y}Z_{y}=\frac{50(44.1)}{12}=183.8\) ft-kips

 

\(M_{n}=M_{ny}=183.8-(183.8-0.7×50×29.1/12)(\frac{9.92-9.152}{24.08-9.152})=178.7\) ft-kips

 

\(\phi_{b}M_{ny}=0.90(178.7)=160.8\) ft-kips

The value of \(\phi_{b}M_{ny}\) is also given in the \(Z_{y}\) table, listed as \(\phi_{b}M_{py}\). For a W12 × 65, it is given as 161 ft-kips.

The required axial compressive strength is

\(P_{r} = P_{u} = P_{nt} + B_{2}P_{\ell t} = 300 + 0 = 300\) kips

and the axial compressive design strength from the column load tables is

\(\phi_{c}P_{n} = 662\) kips

Determine which interaction formula to use:

\(\frac{P_{u}}{\phi_{c}P_{n}}=\frac{300}{662}=0.4532>0.2\) \(∴\) Use Equation 6.3 (AISC Equation H1-1a).

 

\(\frac{P_{u}}{\phi_{c}P_{n}}+\frac{8}{9}(\frac{M_{ux}}{\phi_{b}M_{nx}}+\frac{M_{uy}}{\phi_{b}M_{ny}})=0.4532+\frac{8}{9}(\frac{135}{356}+\frac{30}{160.8})\)

\(=0.956<1.0\) (OK)

The W12 × 65 is satisfactory.
Compute the strong axis bending moments:

\(C_{mx}=0.6-0.4(\frac{M_{1}}{M_{2}})=0.6-0.4(0)=0.6\)

Since a modified flexural rigidity, \(EI^{*}\), was used in the frame analysis, it must also be used in the computation of \(P_{e1}\). From Equation 6.8,

\(E I^{*} = 0.8\tau_{b} E I = 0.8(1.0)E I = 0.8 E I\)

 

\(P_{e1x}=\frac{^{2}E I^{*}}{(K_{1}L)^{2}}=\frac{^{2}(0.8 EI_{x})}{(K_{x}L)^{2}}=\frac{^{2}(0.8) (29,000)(533)}{(1.0×15×12)^{2}}=3767\) kips

 

\(B_{1x}=\frac{C_{mx}}{1-(αP_{r}/P_{e1x})}=\frac{C_{mx}}{1-(1.60P_{a}/P_{e1x})}=\frac{0.6}{1-(1.60×200/3767)}\)

\(=0.656<1.0\) \( ∴\) Use \(B_{1x}=1.0\).

\(M_{r }= M_{ax} = B_{1x}M_{ntx} + B_{2x}M_{\ell tx} = 1.0(90) + 0 = 90\) ft-kips

From the Beam Design Charts with \(C_{b} = 1.0\) and \(L_{b} = 15\) feet, the moment strength is

\(\frac{M_{nx}}{Ω_{b}}=226\) ft-kips and \(\frac{M_{px}}{Ω_{b}}=237\) ft-kips

From Figure 5.15g, \(C_{b} = 1.67\) and

\(C_{b}×\)(\(\frac{M_{nx}}{Ω_{b}}\) for \(C_{b} = 1.0\))=1.67(226) 377.4 ft-kips

This result is larger than \(\frac{M_{px}} {Ω_{b}}\); therefore, use \(\frac{M_{nx}}{Ω_{b}}=\frac{M_{px}} {Ω_{b}}=237\) ft-kips

Compute the weak axis bending moments.

\(C_{my}=0.6-0.4(\frac{M_{1}}{M_{2}})=0.6-0.4(0)=0.6\)

 

\(P_{e1y}=\frac{^{2}E I^{*}}{(K_{1}L)^{2}}=\frac{^{2}(0.8 EI_{y})}{(K_{y}L)^{2}}=\frac{^{2}(0.8) (29,000)(174)}{(1.0×15×12)^{2}}=1230\) kips

 

\(B_{1y}=\frac{C_{my}}{1-(αP_{r}/P_{e1y})}=\frac{C_{my}}{1-(1.60P_{u}/P_{e1y})}=\frac{0.6}{1-(1.60×200/1230)}\)

\(=0.811<1.0\) \( ∴\) Use \(B_{1y}=1.0\).

\(M_{r }= M_{ay} = B_{1y}M_{nty} + B_{2y}M_{\ell ty} = 1.0(20) + 0 = 20\) ft-kips

Because the flange of this shape is noncompact, the weak axis bending strength is limited by FLB (see Section 5.15 of this book and Chapter F of the AISC Specification).

\(\lambda=\frac{b_{f}}{2t_{f}}=9.92\)

 

\(\lambda_{p}=0.38\sqrt{\frac{E}{F_{y}}}=0.38\sqrt{\frac{29,000}{50}}=9.152\)

\(\lambda_{r}=1.0\sqrt{\frac{E}{F_{y}}}=1.0\sqrt{\frac{29,000}{50}}=24.08\)

 

Since \(\lambda_{p}<\lambda<\lambda_{r}\)

\(M_{n}=M_{p}-(M_{p}-0.7 F_{y}S_{y})(\frac{\lambda-\lambda_{p}}{\lambda_{r}-\lambda_{p}})\) (AISC Equation F6-2)

 

\(M_{p}=M_{py}=F_{y}Z_{y}=\frac{50(44.1)}{12}=183.8\) ft-kips

 

\(M_{n}=M_{ny}=183.8-(183.8-0.7×50×29.1/12)(\frac{9.92-9.152}{24.08-9.152})=178.7\) ft-kips

 

\(\frac{M_{ny}}{Ω_{b}}=\frac{178.7}{1.67}=107.0\) ft-kips

The value of \(M_{ny}/Ω_{b}\) is also given in the \(Z_{y}\) table, listed as \(M_{py}/Ω_{b}\). For a W12 × 65, it is given as 107 ft-kips.
The required axial compressive strength is

\(P_{a} = P_{nt} + B_{2}P_{\ell t} = 200 + 0 = 200\) kips

For KL = 1.0(15) = 15 feet, the allowable compressive strength from the column load tables is

\(\frac{P_{n}}{Ω_{c}}=441\) kips

Check the interaction formula:

\(\frac{P_{a}}{P_{n}/Ω_{c}}=\frac{200}{441}=0.4535>0.2\) \(∴\) Use Equation 6.5 (AISC Equation H1-1a).

 

\(\frac{P_{a}}{P_{n}/Ω_{c}}+\frac{8}{9}(\frac{M_{ax}}{M_{nx}/Ω_{b}}+\frac{M_{ay}}{M_{ny}/Ω_{b}})=0.4535+\frac{8}{9}(\frac{90}{237}+\frac{20}{107})\)

 

\(=0.957<1.0\) (OK)

The W12 × 65 is satisfactory.
