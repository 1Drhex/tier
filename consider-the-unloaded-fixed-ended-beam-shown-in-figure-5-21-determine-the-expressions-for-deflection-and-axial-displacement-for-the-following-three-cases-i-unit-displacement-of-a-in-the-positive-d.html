Guest Question.

Consider the unloaded fixed-ended beam shown in Figure 5.21. Determine the expressions for deflection and axial displacement for the following three cases:
(i) unit displacement of A in the positive direction of x
(ii) unit displacement of A in the positive direction of y
(iii) unit anti-clockwise rotation at A
For each case, calculate the support reactions induced by the enforced displacement. Assume constant cross-sectional and material properties with area A, second moment of area I, and elastic modulus E.

Admin
Solution

The three induced displacements are considered separately in the following.

(i) Unit displacement at A in the positive direction of x
(1) Boundary conditions:

u(x = 0) = 1 u(x = L) = 0.

v(x = 0) = v(x = L) = 0.

\( v^{\prime }(x = 0) = v ^{\prime }(x = L) = 0 \).

(2) Determine constants of integration and expressions for displacements:

\( v^{IV} = 0 \); \( v ^{\prime \prime \prime} = C_{v1} \); \( v ^{\prime \prime } = C_{v1}x + C_{v2} \).

\( v ^{\prime }= C_{v1} \frac{x^2}{2} C_{v2}x + C_{v3} \).

\( v = C_{v1} \frac{x^3}{6} C_{v2}\frac{x^2}{2} + C_{v3}x + C_{v4} \).

\( u ^{\prime \prime } = 0 \); \( u^{\prime } = C_{u1} \); \( u = C_{u1}x + C_{u2} \).

Applying relevant boundary conditions:

v (x = 0) = 0 ∴ \( C_{v4} = 0 \).

\( v ^{\prime } (x = 0) = 0 \) ∴ \( C_{v3} = 0 \).

\( v (x =L) = C_{v1} \frac{L^3}{6} + C_{v2} \frac{L^2}{2} = 0 \).

\( v ^{\prime } (x = L) = C_{v1} \frac{L^2}{2} C_{v2} L = 0 \) ∴ \( C_{v1} = 0 \) \( C_{v2} = 0 \).

u(x = 0) = 1 ∴ \( C_{u2} = 1 \).

\( u (x= L) = C_{u1} L + C_{u2} = 0 \) ∴ \( C_{u1} = -\frac{1}{L} \).

The expressions for u and v become: \( u = – \frac{1}{L} x + 1 \) and v = 0.

(3) The reactions are calculated from equilibrium at the two end nodes (Figure 5.22a and b) and are summarised in Figure 5.22c (remember that the direction of the reaction vector is arbitrary).

\( H_A = – N (x=0) = – EA u ^{\prime } (x=0) = \frac{EA}{L} \); \( V_A = −S(x = 0) = EIv ^{\prime \prime \prime}(x = 0) = 0 \).

\( M_A = -M(x=0) = -EIv ^{\prime \prime }(x = 0) = 0 \); \( H_B = N (x=L) = EA u ^{\prime } (x=L) = – \frac{EA}{L} \).

\( V_B = S(x = L) = – EIv ^{\prime \prime \prime}(x = L) = 0 \).

\( M_B = M(x=L) = EIv ^{\prime \prime }(x = 0) = 0 \).

(ii) Unit displacement at A in the positive direction of y

(1) Boundary conditions:

u(x = 0) = 0 u(x = L) = 0.

v(x = 0) = 1 v(x = L) = 0.

\( v ^{\prime }(x = 0) = 0 \) \( v ^{\prime }(x = L) = 0 \).

(2) Determine constants of integration and expressions for displacements:

v(x = 0) = 1 ∴ \( C_{v4} = 1 \).

\( v ^{\prime }(x = 0) = 0 \) ∴ \( C_{v3} = 0 \).

\( v (x= L) = C_{v1} \frac{L^3}{6} + C_{v2} \frac{L^2}{2} = 0 \) ∴ \( C_{v1} = \frac{12}{L^3} \) , \( C_{v2} = \frac{6}{L^2} \).

u(x = 0) = 0 ∴ \( C_{u2} = 0 \).

\( u(x = L) = C_{u1}L + C_{u2} = 0 \) ∴ \( C_{u1} = 0 \).

The expressions for u and v become: u = 0 and \( v = \frac{2}{L^3}x^3 -\frac{3}{L^2}x^2 +1 \) .

(3) The reactions are calculated from equilibrium considerations (see Figure 5.22a and b) and are summarised in Figure 5.23a.

\( H_A = −N(x = 0) = −EAu^{\prime }(x = 0) = 0 \);

\( V_A = −S(x = 0) = EIv^{\prime \prime \prime}(x = 0) = \frac{12EI}{L^3} \) ;

\( M_A = – M(x=0) = – EI v^{\prime \prime}(x = 0) = \frac{6EI}{L^2} \);

\( H_B = N(x =L) = EAu^{\prime }(x = L) = 0 \);

\( V_B = S(x = L) = -EIv^{\prime \prime \prime}(x = L) = -\frac{12EI}{L^3} \) ;

\( M_B = M(x=L) = EI v^{\prime \prime}(x = 0) = \frac{6EI}{L^2} \);

(iii) Unit anti-clockwise rotation at A
(1) Boundary conditions:

u(x = 0) = 0; u(x = L) = 0.

v(x = 0) = 0; v(x = L) = 0.

\( v^{\prime }(x = 0) = 1 \); \( v^{\prime }(x = L) = 0 \).

(2) Determine constants of integration and expressions for displacements:

v(x = 0) ∴ \( C_{v4} = 0 \); \( v^{\prime }(x = 0) = 1 \) ∴ \( C_{v3} = 1 \).

\( v(x=L) = C_{v1} \frac{L^3}{6} + C_{v2} \frac{L^2}{2} + L = 0 \);

\( v^{\prime }(x = L) = C_{v1} \frac{L^2}{2} + C_{v2} L +10 \); ∴ \( C_{v1} = \frac{6}{L^2} \); \( C_{v2} = -\frac{4}{L} \)

u(x = 0) = 0 ∴ \( C_{u2} = 0 \) \( u(x = L) = C_{u1}L + C_{u2} = 0 \) ∴ \( C_{u1} = 0 \).

The expressions for u and v become: u = 0 and \( v = \frac{1}{L^2}x^3 – \frac{2}{L}x^2 + x \).

(3) The reactions are calculated from equilibrium considerations (see Figure 5.22a and b) and are summarised in Figure 5.23b.

\( H_A = −N(x = 0) = −EAu^{\prime }(x = 0) = 0 \).

\( V_A = -S (x=0) = EI v^{\prime \prime \prime}(x = 0) = \frac{6EI}{L^2} \).

\( M_A = – M (x=0) = – EI v^{\prime \prime}(x = 0) = \frac{4EI}{L} \).

\( H_B = N(x = L) = EAu^{\prime }(x = L) = 0 \).

\( V_B= S (x=L) = -EI v^{\prime \prime \prime}(x = L) = -\frac{6EI}{L^2} \).

\( M_B = M (x=L) = EI v^{\prime \prime}(x = 0) = \frac{2EI}{L} \).
