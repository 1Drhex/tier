Guest Question.

A cylinder/piston contains a gas mixture, 50% \(CO _{2}\) and 50% \(C _{2} H _{6}\) (mole basis) at 700 kPa, 35째C, at which point the cylinder volume is 5 L. The mixture is now compressed to 5.5 MPa in a reversible isothermal process. Calculate the heat transfer and work for the process, using the following model for the gas mixture:
a. Ideal gas mixture. b. The van der Waals equation of state.

Admin
Solution

a) Ideal gas mixture

\(\begin{aligned}& U _{2}- U _{1}= mC _{ v \text { mix }}\left( T _{2}- T _{1}\right)=0 \\& Q _{12}= W _{12}=\int P d V = P _{1} V _{1} \ln \left( V _{2} / V _{1}\right)=- P _{1} V _{1} \ln \left( P _{2} / P _{1}\right) \\&=-700 \times 0.005 \ln (5500 / 700)=-7.71 kJ\end{aligned}\)

b) van der waal’s equation

\(\begin{aligned}\text { For } CO _{2}: b &=\overline{ R } T _{ c } / 8 P _{ c }=8.3145 \times 304.1 / 8 \times 7380=0.04282 \\a &=27 P _{ c } b ^{2}=27 \times 7380 \times 0.04282^{2}=365.45\end{aligned}\)

 

\(\begin{aligned}&\text { For } C _{2} H _{6}: \quad b =\overline{ R } T _{ c } / 8 P _{ c }=8.3145 \times 305.4 / 8 \times 4880=0.06504 \\& a =27 P _{ c } b ^{2}=27 \times 4880 \times 0.06504^{2}=557.41 \\& a _{\operatorname{mix}}=(0.5 \sqrt{365.45}+0.5 \sqrt{557.41})^{2}=456.384 \\& b _{ mix }=0.5 \times 0.04282+0.5 \times 0.06504=0.05393 \\&\frac{8.3145 ^{* } 308.2}{\overline{ v }_{1}-0.05393}-\frac{456.384}{{\overline{ v }_{1} }^{2}}-700=0\end{aligned}\)

 

By trial and error: \(\overline{ v }_{1}=3.5329 m ^{3} / kmol\)

\(\frac{8.3145 ^{*} 308.2}{\overline{ v }_{2}-0.05393}-\frac{456.384}{{\overline{ v }_{2}}^{2}}-5500=0\)

By trial and error: \(\overline{ v }_{2}=0.2815 m ^{3} / kmol\)

\(n = V _{1} / \overline{ v }_{1}=0.005 / 3.5329=0.00142\)

 

\(\begin{aligned}Q _{12}=& n T \left(\overline{ s }_{2}-\overline{ s }_{1}\right)_{ T }= n \overline{ R } T \ln \frac{\overline{ v }_{2}- b }{\overline{ v }_{1}- b } \\&=0.00142 \times 8.3145 \times 308.2 \ln \frac{0.2815-0.05392}{3.5329-0.05392}=-9.93 kJ\end{aligned}\)

 

\(\begin{aligned}& U _{2}- U _{1}=0.00142 \times 456.39\left(3.5329^{-1}-0.2815^{-1}\right)=-2.12 kJ \\& Q _{12}= U _{2}- U _{1}+ W _{12} \Rightarrow W _{12}=-9.93-(-2.12)=-7.81 kJ\end{aligned}\)
