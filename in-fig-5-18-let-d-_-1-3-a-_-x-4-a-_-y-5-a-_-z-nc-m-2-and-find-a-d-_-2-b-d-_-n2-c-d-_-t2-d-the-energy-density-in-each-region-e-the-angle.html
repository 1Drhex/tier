Guest Question.

In fig 5.18 Let \({ D }_{ 1 } = 3{ a }_{ x }-4{ a }_{ y }+5{ a }_{ z } nC/{ m }^{ 2 }\) and find: (a) \({ D }_{ 2 }\); (b) \({ D }_{ N2 }\); (c) \({ D }_{ t2 }\); (d) the energy density in each region; (e) the angle that \({ D }_{ 2 }\) makes with \({ a }_{ z }\); (f) \({ D }_{ 2 }/{ D }_{ 1 }\); (g) \({ P }_{ 2 }/{ P }_{ 1 }\).

Admin
Solution

a) First, the electric ﬁeld in region 1 is
\({ E }_{ 1 }=\left[ \frac { 3 }{ 2{ \epsilon }_{ 0 } } { a }_{ x }-\frac { 4 }{ 2{ \epsilon }_{ 0 } } { { a }_{ y }+\frac { 5 }{ 2{ \epsilon }_{ 0 } } { a }_{ z } } \right] \times { 10 }^{ -9 }\)V/m
Since, at the dielectric interface, tangential electric ﬁeld and normal electric ﬂux density are continuous, we may write
\({ D }_{ 2 }={ \epsilon }_{ R2 }{ \epsilon }_{ 0 }{ E }_{ T1 }+{ D }_{ N1 }=\left( \frac { 5 }{ 2 } \right) 3{ a }_{ x }-\left( \frac { 5 }{ 2 } \right)4 { a }_{ y }+5{ a }_{ z }=7.5{ a }_{ x }-10{ a }_{ y }+5{ a }_{ z }nC/{ m }^{ 2 }\)
b) \({ D }_{ N2 } = 5{ a }_{ z }\), as explained above.
c) \({ D }_{ T2 } = { \epsilon }_{ R2 }{ \epsilon }_{ 0 }{ E }_{ T2 } = { \epsilon }_{ R2 }{ \epsilon }_{ 0 }{ E }_{ T1 } = 7.5{ a }_{ x } – 10{ a }_{ y } nC/{ m }^{ 2 }\).
d) the energy density in each region:
\({ \omega }_{ e1 }=\frac { 1 }{ 2 } { \epsilon }_{ R1 }{ \epsilon }_{ 0 }{ E }_{ 1 }.{ E }_{ 1 }=\frac { 1 }{ 2 } (2){ \epsilon }_{ 0 }\left[ { \left( \frac { 3 }{ 2{ \epsilon }_{ 0 } } \right) }^{ 2 }+{ \left( \frac { 4 }{ 2{ \epsilon }_{ 0 } } \right) }^{ 2 }+{ \left( \frac { 5 }{ 2{ \epsilon }_{ 0 } } \right) }^{ 2 } \right] \times { 10 }^{ -18 }=1.41\mu J/{ m }^{ 3 }\)
\({ \omega }_{ e2 }=\frac { 1 }{ 2 } { \epsilon }_{ R2 }{ \epsilon }_{ 0 }{ E }_{ 2 }.{ E }_{ 2 }=\frac { 1 }{ 2 } (5){ \epsilon }_{ 0 }\left[ { \left( \frac { 3 }{ 2{ \epsilon }_{ 0 } } \right) }^{ 2 }+{ \left( \frac { 4 }{ 2{ \epsilon }_{ 0 } } \right) }^{ 2 }+{ \left( \frac { 5 }{ 2{ \epsilon }_{ 0 } } \right) }^{ 2 } \right] \times { 10 }^{ -18 }=2.04\mu J/{ m }^{ 3 }\)
e) Use \({ D }_{ 2 } . { a }_{ z } = \left| { D }_{ 2 }\right| \cos { \theta } = { D }_{ z } = 5, \)where \(\left| { D }_{ 2 }\right| = { \left[ { \left( 7.5 \right) }^{ 2 }+{ \left( 10 \right) }^{ 2 }+{ \left( 5 \right) }^{ 2 } \right] }^{ 1/2 }=13.5. So \theta = \cos ^{ -1 }{ (5/13.5) } =\mathring { 68 }\)
f) \({ D }_{ 2 }/{ D }_{ 1 } = { \left[ { \left( 7.5 \right) }^{ 2 }+{ \left( 10 \right) }^{ 2 }+{ \left( 5 \right) }^{ 2 } \right] }^{ 1/2 }/{ \left[ { \left( 3 \right) }^{ 2 }+{ \left( 4 \right) }^{ 2 }+{ \left( 5 \right) }^{ 2 } \right] }^{ 1/2 } = 1.91.\)
g) \({ P }_{ 2 }/{P }_{ 1 }: First {P }_{ 1 } = { \epsilon }_{ 0 }{ E }_{ 1 }({ \epsilon }_{ R1 }-1) = 1.5{ a }_{ x } – 2{ a }_{ y } + 2.5{ a }_{ z } nC/{ m}^{ 2 }\)
Then \({ P }_{ 2 } = { \epsilon }_{ 0 }{ E }_{ 2 }({ \epsilon }_{ R2 }-1) = 6{ a }_{ x } – 8{ a }_{ y } + 4{ a }_{ z } nC/{ m}^{ 2 }\). So
\(\frac { { P }_{ 2 } }{{ P }_{ 1 } } = \frac {{ \left[ { \left( 6 \right) }^{ 2 }+{ \left( 8 \right) }^{ 2 }+{ \left( 4 \right) }^{ 2 } \right] }^{ 1/2 } }{{ \left[ { \left( 1.5 \right) }^{ 2 }+{ \left( 2 \right) }^{ 2 }+{ \left( 2.5 \right) }^{ 2 } \right] }^{ 1/2 } } = 3.04\)
