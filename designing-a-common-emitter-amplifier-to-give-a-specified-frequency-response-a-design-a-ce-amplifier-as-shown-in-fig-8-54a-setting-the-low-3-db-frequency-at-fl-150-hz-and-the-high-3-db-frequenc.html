Guest Question.

Designing a common-emitter amplifier to give a specified frequency response

(a) Design a CE amplifier as shown in Fig. 8.54(a), setting the low 3-dB frequency at \( f_{ L }=150 Hz \) and the high 3-dB frequency at \( f_{ H }=250 kHz \). The circuit parameters are \( \beta_{ f }=80, g_{ m }=57.14 \mu A / V , r_{\pi}=1.4 k \Omega, C_{\pi}= 15 pF , C_{\mu}=1 pF , R_{ s }=200 \Omega, R_{1}=7 k \Omega, R_{2}=4.3 k \Omega, R_{ E }=330 \Omega, R_{ C }=5 k \Omega, \text { and } R_{ L }=5 k \Omega\).

(b) Determine the passband gain APB.
(c) Use Miller’s capacitor method to check the high-frequency design.
(d) Use PSpice/SPICE to plot the frequency response from 100 Hz to 1 MHz with decade increments and 100 points per decade.

Admin
Solution

Let \( R_{ B }=R_{1}\left\|R_{2}=7 k \Omega\right\| 4.3 k \Omega=2.66 k \Omega \). Then

\( R_{ i }^{\prime}=r_{\pi}\left\|R_{ B }\right\| R_{ s }=R_{ C \pi}=164 \Omega \)

 

(a) The design will have two parts: one in which we set the low 3-dB frequency at \( f_{ L }=150 Hz \) and one in which we set the high 3-dB frequency at \( f_{ H }=250 kHz \).

The steps to set \( f_{ L }=150 Hz \) are as follows:

Step 1. Calculate the equivalent resistances \( R_{ C 1}, R_{ C 2}, \text { and } R_{ CE } \). From Eq. (8.171),

\(R_{ C 1}=R_{ s }+R_{ B } \| r_{\pi}\) (8.171)

 

\(R_{ Cl }=R_{ s }+R_{ B }\left\|r_{\pi}=200 \Omega+2.66 k \Omega\right\| 1.4 k \Omega=1.12 k \Omega\)

 

From Eq. (8.173),

\(R_{ C 2}=R_{ C }+R_{ L }\) (8.173)

 

\(R_{ C 2}=R_{ C }+R_{ L }=5 k \Omega+5 k \Omega=10 k \Omega\)

 

From Eq. (8.175),

\(R_{ CE }=R_{ E } \| \frac{r_{\pi}+\left(R_{ s } \| R_{ B }\right)}{1+\beta_{ f }}\) (8.175)

 

\(R_{ CE }=330 \|\left[\frac{1.4 k \Omega+200 \Omega \| 2.66 k \Omega}{1+80}\right]=18.48 \Omega\)

 

Step 2. Assume that the frequency corresponding to the lowest resistance is the dominant cutoff frequency \( f_{ L } \). The low 3-dB frequency can be assigned to any resistance. However, assigning the low 3-dB frequency to the lowest resistance will reduce the values of coupling capacitors. Since \( R_{ CE } \) has the lowest value, let \(f_{ CE }=f_{ L }\). That is, \( f_{ CE }=f_{ L 1}=f_{ L }=150 Hz \).

Step 3. Calculate the required value of \(C_{ E }\) from Eq. (8.177):

\(f_{ CE }=\frac{1}{2 \pi R_{ CE } C_{ E }}\) (8.177)

 

\(f_{ CE }=\frac{1}{2 \pi R_{ CE } C_{ E }}=\frac{1}{2 \pi \times 18.48 \times C_{ E }}=150 H _{ Z } \quad \text { or } \quad C_{ E }=57.4 \mu F\)

 

Step 4. Set the frequency corresponding to the next higher resistance. Let \( f_{ L 2}=f_{ C 1}=f_{ L } / 10=150 / 10=15 Hz \). From Eq. (8.172),

 

\(f_{ C 1}=\frac{1}{2 \pi R_{ C 1} C_{1}}\) (8.172)

 

\(f_{ C 1}=\frac{1}{2 \pi R_{ C 1} C_{1}}=\frac{1}{2 \pi \times 1.12 k \Omega \times C_{1}}=15 H _{ z } \quad \text { or } \quad C_{1}=9.5 \mu F\)

 

Step 5. Set the frequency corresponding to the highest resistance. Let \( f_{ L 3}=f_{ C 2}=f_{ L } / 20=150 / 20=7.5 Hz \). From Eq. (8.174),

 

\(f_{ C 2}=\frac{1}{2 \pi R_{ C 2} C_{2}}\) (8.174)

 

\(f_{ C 2}=\frac{1}{2 \pi R_{ C 2} C_{2}}=\frac{1}{2 \pi \times 10 k \Omega \times C_{2}}=7.5 Hz \quad \text { or } \quad C_{2}=2.1 \mu F\)

 

The steps to set \(f_{ H }=250 kHz\) are as follows:

Step 1. From Eq. (8.178),

 

\(R_{ C \pi}=r_{\pi} \|\left(R_{ s } \| R_{ B }\right)\) (8.178)

 

\(R_{ C \pi}=r_{\pi}\left\|\left(R_{ S } \| R_{ B }\right)=1.4 k \Omega\right\|(200 \| 2.66 k \Omega)=164 \Omega\)

 

From Eq. (8.179),

 

\(R_{ C \mu}=\frac{v_{ X }}{i_{ X }}=R_{ L } \| R_{ C }+R_{ i }^{\prime}\left[1+g_{ m }\left(R_{ L } \| R_{ C }\right)\right]\) (8.179)

 

\( R_{ C \mu}=(5 k \Omega \| 5 k \Omega)+164 \Omega \times[1+57.14 m \mho \times(5 k \Omega \| 5 k \Omega)]=26.1 k \Omega \)

 

Step 2. From Eq. (8.180), the high 3-dB frequency \( f_{ H } \) is

 

\(f_{ H }=\frac{1}{2 \pi\left[R_{ C \pi} C_{\pi}+R_{ C \mu}\left(C_{\mu}+C_{ x }\right)\right]}\) (8.180)

 

\( f_{ H }=\frac{1}{2 \pi\left[164 \Omega \times 15 pF +(26.1 k \Omega)\left(C_{\mu}+C_{ x }\right)\right]} 250 kHz \quad \text { or } \quad C_{\mu}+C_{ x }=C_{ eff }=24.3 pF \)

 

which gives \(C_{ X }=24.3-1=23.3 pF \). This is the value of the additional capacitor \(C _{ x }\) that is to be connected between the collector and base terminals of the transistor.

(b) From Eq. (2.114), the passband voltage gain is

\(f_{ H }=\frac{1}{2 \pi\left(R_{C i } C_{ i }+R_{ Co } C_{ o }+R_{ Cc } C\right)}\) (2.114)

 

\(A_{ PB }=\left(-57.14 \times 10^{-3}\right)(5 k \Omega \| 5 k \Omega) \times \frac{2.66 k \Omega \| 1.4 k \Omega}{200+(2.66 k \Omega \| 1.4 k \Omega)}=-117.3 V / V\)

 

(c) Applying Eq. (2.98), we have for the effective capacitance between the base and the emitter terminals

 

\(C_{ m }=C\left[1+g_{ m }\left(R_{ o } \| R_{ L }\right)\right]\) (2.98)

 

\(\begin{aligned}C_{ eq } &=\left(C_{\mu}+C_{ x }\right)\left[1+g_{ m }\left(R_{ L } \| R_{ C }\right)\right]+C_{\pi} \\&=24.3 pF \times[1+57.14 m \mho \times(5 k \Omega \| 5 k \Omega)]+15 pF =3.51 nF\end{aligned}\)

 

The equivalent resistance faced by \( C_{ eq } \text { is } R_{ eq }=R_{1}^{\prime}=r_{\pi}\left\|R_{ B }\right\| R_{ s }=164 \Omega \), so

 

\(f_{ H }=\frac{1}{2 \pi C_{ eq } R_{ eq }}=\frac{1}{2 \pi \times 3.51 nF \times 164 \Omega}=276.5 kHz\)

 

Miller’s capacitor method gives a higher frequency than the zero-value method does. The actual frequency is higher than the one obtained by the zero-value method but lower than the one obtained by Miller’s capacitor method. Thus, design by the zero-value method provides a more conservative estimate, although the SPICE results match the zero-value prediction.

(d) The AC equivalent circuit for PSpice simulation is shown in Fig. 8.59, where \(r_{ o }\) has been added to include the output resistance of the transistor and also to make the circuit more general.

The PSpice plot of the frequency response is shown in Fig. 8.60, which gives the passband gain as \( \left|A_{ PB }\right| = 117.2 \) (expected value is 117.3). The low 3-dB frequency is approximately \( f_{ L }=158.5 Hz \) (expected value is 150 Hz), and the high 3-dB frequency is approximately \( f_{ H }=249.5 kHz \) (expected value is 250 kHz). The design value of \( f_{ H } \) is 250 kHz, and that of \( f_{ L } \) is 150 Hz. Thus, the results are very close.

NOTE: The main objective of this PSpice simulation was to verify the design methods used to set the low and high cutoff frequencies. Thus, the small-signal model rather than the actual PSpice transistor model was used in the simulation. If we designed an amplifier with the small-signal model and then ran the simulation with an actual PSpice transistor, we would expect to get an error. But, the results should be close.
