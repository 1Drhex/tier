Guest Question.

Analytically determine the result J = A –> B +> C +> D for the velocity vectors shown in Figure 3.28. 1.Sketch a Rough Vector Diagram 2.Use x-axis, Angle Method to Determine Components 3.Determine the Components of the Solution 4.Combine the Components of the Solution.

Admin
Solution

1: The horizontal and vertical components of each velocity are determined by trigonometry using equations \(A_{h}=A\ cos\ \theta _{x}\) ,
\(A_{v}=A\ sin\ \theta _{x}\) and shown in Figure 3.29. Also shown are the vectors rearranged in a series: tip-to-tail for addition and tail-to-tip for subtraction.

 

2: The values of the component are listed in Table 3.2.

Vector Reference
Angle \(\theta_{x}\)		 h component (ft/s)

\(V_{h}=V\ cos\ \theta _{x}\)

 v component (ft/s)

\(V_{v}=V\ sin\ \theta _{x}\)
A 300° +3.00 -5.19
B 195° -11.59 -3.11
C 45° +5.66 +5.66
D 330° +8.66 -5.00

3: Algebraic manipulation of the individual horizontal and vertical components gives the components of the resultant:

\(J_{h}=A_{h}-B_{h}+ C_{h}+D_{h}\)

\(= (+3.0) – (-11.59) + (+5.66) + (+8.66) = + 28.91 ft/s\)

\(J_{v}=A_{v}-B_{v}+C_{v}+D_{v}\)

\(= (-5.19) – (-3.11) + (+5.66) + (-5.00) = – 1.42 ft /s\)

4: The magnitude and direction of the resultant may be determined by vectorally adding the components (Figure 3.30).

The magnitude of the solution can be determined from equation \(R=\sqrt{R^{2}_{h}+R^{2}_{v} } \) :

\(J=\sqrt{J^{2}_{h}+J^{2}_{v} }\)

\(=\sqrt{(28.91\ ft/s)^{2}+(-1.42\ ft/s)^{2}}=28.94\ ft/s\)

The angle of the solution can be found from the tangent function:

\(\theta _{x}=tan^{-1}(\frac{J_{v}}{J_{h}} )=tan^{-1}(\frac{-1.42\ ft/s}{28.91\ ft/s} )=-2.8^{\circ }\)

Thus, the solution can be formally stated as

\(J=28.94\ ft/\ s2.8^{\circ }\angle\)
