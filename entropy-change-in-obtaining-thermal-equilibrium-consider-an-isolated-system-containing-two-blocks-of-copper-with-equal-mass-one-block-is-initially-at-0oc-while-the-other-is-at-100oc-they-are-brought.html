Entropy Change in Obtaining Thermal Equilibrium

Consider an isolated system containing two blocks of copper with equal mass. One block is initially at 0ºC while the other is at 100ºC. They are brought into contact with each other and allowed to thermally equilibrate. What is the entropy change for the system during this process? Take the heat capacity for copper to be:

\(c_{P}=24.5[ J /( mol K )]\)

 

Admin
Solution

We label the cooler block “block I” and the hotter block “block II” as illustrated in Figure E3.3. The blocks go from state 1, in which they are at different temperatures, to state 2, in which their temperatures are equal. The pressure remains constant during the process.

We must first find the final temperature, \(T_{2}\). From the fi rst law for an isolated system at constant pressure, we get:

 

\(\Delta H=n_{ I } \Delta h _{ I }+ n _{ II } \Delta h_{ II }=0\) (E3.3A)

 

where the subscripts “I” and “II” refer to the appropriate block. For constant heat capacity, Equation (E3.3A) can be written:

 

\(n_{ I C _{ P }}\left(T_{2}-T_{1, I }\right)+n_{ II } c_{ P }\left(T_{2}-T_{1, II }\right)=0\) (E3.3B)

 

Since the blocks have equal mass,

 

\(n_{ I }= n _{ II }=n\) (E3.3C)

 

Solving Equation (E3.3B) subject to (E3.3C) gives:

 

\(T_{2}=\frac{T_{1, I }+T_{1, II }}{2}=323[ K ]\)

 

To get the total entropy change for the system, we can add the changes of each block individually:

 

\(\Delta S=n_{ I } \Delta s_{ I }+n_{ II } \Delta s_{ II }\) (E3.3D)

 

We can apply Equation (3.1) to solve for \(\Delta s_{ I } \text { and } \Delta s_{ II }\). We first need to find \(\delta q_{ rev }\). We do that by constructing a reversible path at constant pressure from \(T_{1} \text { to } T_{2}\). Such a process can be conceived by placing each block in thermal contact with a series of reservoirs that are each just differentially hotter (or cooler) than the block. Then \(q_{\text {rev }}\) is given by:

 

\(d s \equiv \frac{\delta q_{ rev }}{T}\) (3.1)

 

\(q_{ rev }=\Delta h=\int_{T_{1}}^{T_{2}} c_{p} d T\)

 

or,

 

\(\delta q_{ rev }=c_{P} d T\)

 

Thus, the entropy change is described by:

 

\(\Delta s=\int \frac{\delta q_{ rev }}{T}=\int_{T_{1}}^{T_{2}} \frac{c_{p} d T}{T}\) (E3.3E)

 

Applying Equation (E3.3E) to the system of two blocks with constant heat capacity, we get:

 

\(\Delta S=n_{1} c_{P} \ln \left(\frac{T_{2}}{T_{1, I}}\right)+n_{ II } c_{P} \ln \left(\frac{T_{2}}{T_{1, II }}\right)\)

 

Rearranging and plugging in numerical values yields:

 

\(\Delta s=\frac{\Delta S}{2 n}=\frac{c_{P}}{2}\left[\ln \left(\frac{T_{2}}{T_{1, I }}\right)+\ln \left(\frac{T_{2}}{T_{1, II }}\right)\right]=\frac{c_{P}}{2} \ln \left(\frac{T_{2}^{2}}{T_{1, I } \times T_{1, II }}\right)\)

 

\(=\left(\frac{24.5}{2}\left[\frac{ J }{ mol K }\right]\right) \ln \frac{323^{2}}{(273)(373)}=0.30\left[\frac{ J }{ mol K }\right]\)

 

Since this is an isolated system, the entropy change to the surroundings is zero. The positive value we obtained confi rms that energy spontaneously flows from a hot block to a cold block.
