Guest Question.

For the circuit shown below, draw the time variations of i, \(V _{ L }\), and \(v _{ C }\) when the thyristor is triggered at t = 0.

Admin
Solution

Solution

At t = 0, as soon as the thyristor Q is switched into the conduction state, the diode D is reverse biased and, consequently, the equivalent circuit during this mode is:

Using the equivalent circuit the following equation is obtained:

 

\(100-v_{L}-v_{C}+75=0\)

 

\(\text { or } L \frac{d i}{d t}+\frac{1}{C} \int_{0}^{t} i \cdot dt =175\)

 

\(\text { or } \quad L \frac{ d ^{2} i }{ dt ^{2}}+\frac{ i }{ C }=0\) (1)

 

Therefore, using Eq. (1) the circuit characteristic equation will be:

 

\(LS ^{2}+\frac{1}{ C }=0\) (2)

 

The roots of the above equation are given by:

 

\(S _{1,2}=\pm j \sqrt{\frac{1}{ LC }}=\pm j \omega_{ R }\)

 

Therefore, the equivalent circuit current is given by:

\(i = A _{1} e ^{ s _{1} t }+ A _{2} e ^{ s _{2} t }= A _{1} e ^{ j \omega_{ R } t }+ A _{2} e ^{- j \omega_{ R } t}\) (3)

 

It is well known that:

 

\(\begin{aligned}& e ^{ j \omega_{ R } t }=\cos \omega_{ R } t + j \sin \omega_{ R } t \\& e ^{- j \omega_{ R } t }=\cos \omega_{ R } t – j \sin \omega_{ R } t\end{aligned}\)

 

Therefore, Eq. (3) becomes:

 

\(\begin{aligned}i &= A _{1}\left(\cos \omega_{ R } t + j \sin \omega_{ R } t \right)+ A _{2}\left(\cos \omega_{ R } t – j \sin \omega_{ R } t \right) \\&= A _{1} \cos \omega_{ R } t + j A _{1} \sin \omega_{ R } t + A _{2} \cos \omega_{ R } t – j A _{2} \sin \omega_{ R } t \\&=\left( A _{1}+ A _{2}\right) \cos \omega_{ R } t +\left( j A _{1}- j A _{2}\right) \sin \omega_{ R } t = B _{1} \cos \omega_{ R } t + B _{2} \sin \omega_{ R } t\end{aligned}\) (4)

 

Using the circuit initial condition the constants \(\text { B }_{1}\) and \(\text { B }_{2}\) can be calculated as follows:

 

At t = 0, current i = 0 and, therefore i = 0 = \(B _{1} \cos 0+ B _{2} \sin 0 \text { or } B _{1}=0\) Moreover, differentiating the current expression, the following equation results:

 

\(\frac{ di }{ dt }=- B _{1} \omega_{ R } \sin \omega_{ R } t + B _{2} \omega_{ R } \cos \omega_{ R } t =\frac{ v _{ L }}{ L }\)

 

where \(v _{ L }= L \frac{ di }{ dt }=100- v _{ C }+ v _{ C }(0)\) (5)

 

At t = 0 the following equation holds:

\(v _{ L }(0)=100+ v _{ C }(0)=100+75=175 V\)

 

Therefore

 

\(\left.\frac{ di }{ dt }\right|_{ t =0}=- B _{1} \omega_{ R } \sin 0+ B _{2} \omega_{ R } \cos 0=\frac{175}{ L } \quad \text { or } \quad B _{2}=\frac{175}{\omega_{ R } L }=\frac{175}{\sqrt{ L / C }}\)

 

Substituting the values of B1 and B2 into Eq. (4) yields:

 

\(i =\frac{175}{\sqrt{ L / C }} \sin \omega_{ R } t =350 \sin \frac{10^{6}}{60} t\) (6)

 

Moreover, using Eq. (6) the inductor voltage is:

 

\(v _{ L }= L \frac{ di }{ dt }=30 \cdot 10^{-6} \frac{10^{6}}{60} 350 \cos \frac{10^{6}}{60} t =175 \cos \frac{10^{6}}{60} t =175 \cos \omega_{ R } t\) (7)

 

The period of the current i and voltage \(V _{ L }\) is:

 

\(T =\frac{1}{ f }=\frac{1}{\omega_{ R } / 2 \pi}=\frac{2 \pi}{\omega_{ R }}=\frac{2 \pi}{10^{6} / 60}=\frac{120 \pi}{10^{6}}=377 \mu s\)

 

As can be seen from the given circuit at the instant the inductor voltage
becomes equal to \(v _{ L }=-75 V\), the diode D becomes forward biased entering the conduction state. Also, at that instant the voltage across the capacitor becomes \(v _{ C }= V – v _{ L }=100-(-75)=175 V\) making the thyristor reverse biased and, consequently, turns off. Therefore, the equivalent circuit during this mode becomes:

The time instant at which the diode is turned on can be calculated using the following equation:

 

\(v _{ L }=175 \cos \omega_{ R } t =-75 V \Rightarrow \omega_{ R } t =\cos ^{-1}\left(\frac{-75}{175}\right)=115^{\circ}\)

 

Therefore; \(t =\frac{115}{\omega_{ R }} \frac{\pi}{180}=\frac{115}{10^{6} / 60} \frac{\pi}{180}=120.8 \mu s\)

 

Thus, the current i at \(t =120.8 \mu s\) \(\text { (i.e., at angle } 115^{\circ} \text { ) }\) will be:

 

\(i =350 \sin 115^{\circ}=316 A\)

 

Furthermore, for the circuit path L-D-75 V, the following equation holds:

 

\(v _{ L }= L \frac{ d i _{ D }}{ dt } \quad \text { or } \quad \frac{ di _{ D }}{ dt }=\frac{-75}{ L }=\frac{-75}{30 \times 10^{-6}}=-2.5 \times 10^{6} \frac{ A }{ s }\)

 

Therefore the time required for the current \(i _{ D }\) to be reduced to zero is given by:

 

\(\frac{316}{t_{ x }}=2.5 \times 10^{6} \quad \text { or } \quad t _{ x }=126.5 \mu s\)

 

Using the above results, the time variations of \(\text { i, } v_{L}, v_{C} \text {, and } i_{D}\) are presented in Fig. 3.24.
