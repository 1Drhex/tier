Guest Question.

A single-row, deep-groove ball bearing number 6208 is subjected to the following set of loads for the given times:

Condition \(F_i\) Time
1 650 lb 30 min
2 750 lb 10 min
3 250 lb 20 min
Admin
Solution

Using Equation (14-10), we have
\(F_{m}=\left(\frac{\Sigma_{i}\left(F_{i}\right)^{p} N_{i}}{N}\right)^{1 / p}\) (14–10a)

\(F_{m}=\left(\frac{30(650)^{3}+10(750)^{3}+20(250)^{3}}{30+10+20}\right)^{1 / 3}=597 \mathrm{lb}\)
Now use Equation (14-11):
\(L=\left(\frac{C}{F_{m}}\right)^{p}\) (14–11a)
From Table 14-3, for the 6208 bearing, we find that \(C=6902 \mathrm{lb}\). Then
\(L=\left(\frac{6902}{597}\right)^{3}=1545 \text { million rev }\)
At a rotational speed of \(600 \mathrm{rpm}\), the number of hours of life would be
\(L=\frac{1545 \times 10^{6} \mathrm{rev}}{1} \cdot \frac{\min }{600 \mathrm{rev}} \cdot \frac{\mathrm{h}}{60 \mathrm{~min}}=42917 \mathrm{~h}\)

 

Note that this is the same bearing used in which was selected to operate at least 30 000 hr when carrying a steady load of 650 lb.
