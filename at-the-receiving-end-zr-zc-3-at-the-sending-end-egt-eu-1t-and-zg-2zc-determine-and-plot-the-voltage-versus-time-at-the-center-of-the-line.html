Single-phase lossless-line transients: step-voltage source with unmatched source resistance at sending end, unmatched resistive load at receiving end

At the receiving end, \(Z_{R} =Z_{c}/3 \). At the sending end, \(e_{G}\left(t\right) =Eu_{-1}\left(t\right) \) and \(Z_{G} =2Z_{c} \). Determine and plot the voltage versus time at the center of the line.

Admin
Solution

From (13.2.12),

\(\Gamma _{S} \left(s\right)=\frac{\frac{Z_{G}\left(s\right) }{Z_{c} } -1}{\frac{Z_{G}\left(s\right) }{Z_{c}} +1}\) per unit (13.2.12)

\(\Gamma _{R}=\frac{\frac{1}{3}-1 }{\frac{1}{3}+1 } =-\frac{1}{2}\)

 

\(\Gamma _{s}=\frac{2-1}{2+1} =\frac{1}{3}\)

From (13.2.10), with \(E_G(s) = E/s\),

\(V(x,s)=E_G(s)\left[\frac{Z_c}{Z_G(s)+Z_c} \right] \left[\frac{e^{-sx/v}+\Gamma _R(s)e^{s[(x/v)-2\tau ]}}{1-\Gamma _R(s)\Gamma _s(s)e^{-2s\tau }} \right] \) (13.2.10)

\(V(x,s)=\frac{E}{s}\left(\frac{1}{3} \right) \frac{[e^{-xs/v}-\frac{1}{2}e^{s[(x/v)-2\tau ]} ]}{1+\left(\frac{1}{6}e^{-2s\tau } \right) }\)

The preceding equation can be rewritten using the following geometric series:

\(\frac{1}{1+y} =1-y+y^{2} -y^{3}+y^{4}-…\)

with \(y=\frac{1}{6} e^{-2s\tau } \),

\(V\left(x,s\right)=\frac{E}{3s} \left[e^{^{-sx/\nu } }-\frac{1}{2} e^{s\left[\left(x/\nu \right)-2\tau \right] } \right] \times \left[1-\frac{1}{6}e^{-2s\tau } +\frac{1}{36}e^{-4s\tau }-\frac{1}{216}e^{-6s\tau } +… \right] \)

Multiplying the terms within the brackets,

\(V\left(x,s\right)=\frac{E}{3s} \left[e^{-sx/\nu } -\frac{1}{2}e^{s\left[\left(x/\nu \right)-2\tau \right] } -\frac{1}{6}e^{-s\left[\left(x/\nu \right)+2\tau \right] } +\frac{1}{12}e^{s\left[\left(x/\nu \right)-4\tau \right] } +\frac{1}{36}e^{-s\left[\left(x/\nu \right)+4\tau \right] } -\frac{1}{72}e^{s\left[\left(x/\nu \right)-6\tau \right] } +…\right]\)

Taking the inverse Laplace transform,

\(\upsilon \left(x,t\right)=\frac{E}{3}\left[u_{-1}\left(t-\frac{x}{\nu }\right)-\frac{1}{2}u_{-1}\left(t+\frac{x}{\nu }-2\tau\right)-\frac{1}{6}u_{-1}\left(t-\frac{x}{\nu }-2\tau\right)+\frac{1}{12}u_{-1}\left(t+\frac{x}{\nu }-4\tau \right)+\frac{1}{36}u_{-1}\left(t-\frac{x}{\nu }-4\tau \right)-\frac{1}{72}u_{-1}\left(t+\frac{x}{\nu }-6\tau\right)…\right]\)

At the center of the line, where x = l/2,

\(\upsilon \left(\frac{l}{2^{,} } t\right)=\frac{E}{3} \left[u_{-1}\left(t-\frac{\tau }{2} \right)-\frac{1}{2}u_{-1}\left(t-\frac{3\tau }{2} \right)-\frac{1}{6}u_{-1}\left(t-\frac{5\tau }{2} \right)+\frac{1}{12}u_{-1}\left(t-\frac{7\tau }{2} \right) +\frac{1}{36}u_{-1}\left(t-\frac{9\tau }{2} \right)-\frac{1}{72} u_{-1}\left(t-\frac{11\tau }{2} \right) … \right]\)

υ(l∙2, t) is plotted in Figure 13.7(a). Since neither the source nor the load is matched to the line, the voltage at any point along the line consists of an infinite series of forward and backward traveling waves. At the center of the line, the first forward traveling wave arrives at \(t=\tau /2\) ; then a backward traveling wave arrives at \(3\tau /2\), another forward traveling wave arrives at \(5\tau /2\), another backward traveling wave at \(7\tau/2\), and so on.

The steady-state voltage can be evaluated from the final value theorem. That is,

\(\upsilon _{ss}\left(x\right) =\underset{t\rightarrow \infty }{\lim }\upsilon \left(x,t\right) =\underset{s\rightarrow 0 }{\lim }sV\left(x,s\right)= \underset{s\rightarrow 0 }{\lim }\left\{s\left(\frac{E}{s} \right)\left(\frac{1}{3} \right) \frac{\left[e^{-sx/\nu }-\frac{1}{2}e^{s\left[\left(x/\nu \right)-2\pi \right] } \right] }{1+\frac{1}{6}e^{-2s\tau } } \right\} =E\left(\frac{1}{3} \right)\left(\frac{1-\frac{1}{2} }{1+\frac{1}{6} } \right) =\frac{E}{7}\)

The steady-state solution can also be evaluated from the circuit in Figure 13.7(b). Since there is no steady-state voltage drop across the lossless line when a dc source is applied, the line can be eliminated, leaving only the source and load. The steady-state voltage is then, by voltage division,

\(\upsilon _{ss}\left(x\right)=E\left(\frac{Z_{R} }{Z_{R} +Z_{G} } \right)=E\left(\frac{\frac{1}{3} }{\frac{1}{3}+2 } \right) =\frac{E}{7}\)
