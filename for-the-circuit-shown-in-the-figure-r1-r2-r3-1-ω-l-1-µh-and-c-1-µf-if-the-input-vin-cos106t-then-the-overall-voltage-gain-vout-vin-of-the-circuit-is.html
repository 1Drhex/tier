Guest Question.

For the circuit shown in the figure, \(R_{1}=R_{2}=R_{3}=1 \Omega\), L = 1 µH and C = 1 µF. If the input \(V_{\text {in }}=\cos \left(10^{6} t \right)\), then the overall voltage gain \(\left(V_{\text {out }} / V_{\text {in }}\right)\) of the circuit is

Admin
Solution

Resistance \(R_{1}=R_{2}=R_{3}=1 \Omega\) ,

Inductance L =1μH

Capacitance C = 1μF

The input voltage \(\text { is } V_{\text {in }}=\cos 10^{6} t V\)

Angular frequency \(\omega=10^{6} rad / sec\)

After considering the virtual ground effect, we get (let V is denoted by \(V_{x}\)) then

\(\begin{aligned}&\frac{V_{x}}{V_{i n}}=\left(1+\frac{R_{1}}{s L}\right) \\&\frac{V_{o}}{V_{x}}=\frac{-R_{3}}{\left(R_{2}+\frac{1}{s C}\right)} \\&V_{o}=\frac{-R_{3}}{\left(R_{2}+\frac{1}{s C}\right)} \times\left(1+\frac{R_{1}}{s L}\right) \times V_{i n}\end{aligned}\)

Sub given values, we get

\(\frac{V_{o}}{V_{i n}}=-1\)

Hence, the correct Answer is (–1).
