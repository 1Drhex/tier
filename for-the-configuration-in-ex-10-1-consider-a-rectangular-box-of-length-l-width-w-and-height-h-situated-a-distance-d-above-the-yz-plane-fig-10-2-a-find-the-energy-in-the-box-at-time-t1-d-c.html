Guest Question.

For the configuration in , consider a rectangular box of length l, width w, and height h, situated a distance d above the yz plane (Fig. 10.2).

(a) Find the energy in the box at time \(t_{1}=d / c, \text { and at } t_{2}=(d+h) / c\) .

(b) Find the Poynting vector, and determine the energy per unit time flowing into the box during the interval \(t_{1}<t<t_{2}\) .

(c) Integrate the result in (b) from \( t_{1} \text { to } t_{2}\), and confirm that the increase in energy (part (a)) equals the net influx.

Admin
Solution

\(\text { (a) } W=\frac{1}{2} \int\left(\epsilon_{0} E^{2}+\frac{1}{\mu_{0}} B^{2}\right) d \tau . \text { At } t_{1}=d / c, x \geq d=c t_{1}, \text { so } E =0, B =0, \text { and hence } W\left(t_{1}\right)=0\).

\(\text { At } T_{2}=(d+h) / c, c t_{2}=d+h\) :

\(E =-\frac{\mu_{0} k}{2}(d+h-x) \hat{ z }, \quad B =\frac{1}{c} \frac{\mu_{0} k}{2}(d+h-x) \hat{ y }\),

\(\text { so } B^{2}=\frac{1}{c^{2}} E^{2}\) , and

\(\left(\epsilon_{0} E^{2}+\frac{1}{\mu_{0}} B^{2}\right)=\epsilon_{0}\left(E^{2}+\frac{1}{\mu_{0} \epsilon_{0}} \frac{1}{c^{2}} E^{2}\right)=2 \epsilon_{0} E^{2}\).

Therefore

\(W\left(t_{2}\right)=\frac{1}{2}\left(2 \epsilon_{0}\right) \frac{\mu_{0}^{2} k^{2}}{4} \int_{d}^{(d+h)}(d+h-x)^{2} d x(l w)=\frac{\epsilon_{0} \mu_{0}^{2} k^{2} l w}{4}\left[-\frac{(d+h-x)^{3}}{3}\right]_{d}^{d+h}=\frac{\epsilon_{0} \mu_{0}^{2} k^{2} l w h^{3}}{12}\).

\(\text { (b) } S (x)=\frac{1}{\mu_{0}}( E \times B )=\frac{1}{\mu_{0} c} E^{2}[-\hat{ z } \times(\pm \hat{ y })]=\pm \frac{1}{\mu_{0} c} E^{2} \hat{ x }=\pm \frac{\mu_{0} k^{2}}{4 c}(c t-|x|)^{2} \hat{ x }\)

(plus sign for x > 0, as here). For \(|x|>c t, S =0\).

So the energy per unit time entering the box in this time interval is

\(\frac{d W}{d t}=P=\int S (d) \cdot d a =\frac{\mu_{0} k^{2} l w}{4 c}(c t-d)^{2}\).

Note that no energy flows out the top, since S(d + h) = 0.

\(\text { (c) } W=\int_{t_{1}}^{t_{2}} P d t=\frac{\mu_{0} k^{2} l w}{4 c} \int_{d / c}^{(d+h) / c}(c t-d)^{2} d t=\frac{\mu_{0} k^{2} l w}{4 c}\left[\frac{(c t-d)^{3}}{3 c}\right]_{d / c}^{(d+h) / c}=\frac{\mu_{0} k^{2} l w h^{3}}{12 c^{2}}\) .

Since \(1 / c^{2}=\mu_{0} \epsilon_{0}\) , this agrees with the answer to (a).
