Guest Question.

Determine the flow directions in regions 1 and 3 of Figure P8.6 if the reservoir pressure were increased to 2 MPa.

Admin
Solution

Because the nozzle is designed for an exit Mach number, \(M _{ e }\), of 2.2, it follows that the static to total pressure ratio at the nozzle exit is

 

\(\frac{p_{e}}{p_{o}}=0.0935\)

 

The back pressure is 101 kPa and the reservoir pressure is 2,000 kPa, therefore,

 

\(\frac{ p _{ b }}{ p _{ o }}=\frac{101}{2000}=0.0505\)

 

Since \(p _{ b } / p _{ o }< p _{ e } / p _{ o }\), the nozzle is underexpanded for this back pressure-reservoir pressure combination. The following provides nomenclature and a sketch of the flow field.

 

From the Prandtl-Meyer relation at \(M _{ e }=2.2 \text {, we find that } v _{ e }=31.7325^{\circ} \text {. Since } p _{ b }= p _{1}\) in region 1. Then, from the static to total pressure ratio, \(p _{1} / p _{o}=0.0505\), we find that \(M _{1}=2.5950 \text { and therefore } v_{1}=41.3044^{\circ}\). So the exit flow is turned through the following angle as it passes into region 1

 

\(\alpha_{1}-\alpha_{e}=v_{1}-v_{e}=41.3044-31.7325=9.5719^{\circ}\)

 

Since \(\alpha_{ e }=0^{\circ}, \text { then it follows that } \alpha_{1}=9.5719^{\circ}\). The flow in region 2 must be horizontal, i.e., \(\alpha_{2}=0^{\circ}\), and since we must pass through another expansion fan, we may write that

 

\(-\left(\alpha_{2}-\alpha_{1}\right)=v_{2}-v_{1}\)

 

Hence,

 

\(v_{2}=v_{1}+\alpha_{1}=41.3044+9.5719=50.8763^{\circ}\)

 

From this we find \(M _{2}=3.0587 \text { and therefore } p _{2} / p _{ o 2}=0.0249\). Since the flow is isentropic across both expansion fans, \(p _{ o 2}= p _{ o 1}= p _{ o }=2,000\) kPa. This enables us to determi \(p _{2}=(0.0249)(2,000)=49.8000 kPa \text {. Now since } p _{3}= p _{ b }=101 kPa\), we have the pressure ratio across the oblique shock, i.e., \(p _{3} / p _{2}=101 / 49.8=2.0281\). With this pressure ratio and \(M _{2}=3.0587\) we can first determine the shock angle from Eq.(6.10)

 

\(\frac{p_{2}}{p_{1}}=\frac{2 \gamma M_{1}^{2} \sin ^{2} \theta}{\gamma+1}-\frac{\gamma-1}{\gamma+1}\) (6.10)

 

Hence

 

\(\sin \theta=\pm \sqrt{\frac{(2.0281)(2.4)+0.4}{(2.8)(3.0587)^{2}}}=\pm 0.4484\)

 

From which we find that \(\theta=\pm 26.6423^{\circ}\). With this angle and the Mach number we enter the oblique shock solver to find \(\delta=\pm 9.6362^{\circ} . \text { From this we see that } \alpha_{3}=-9.6362^{\circ}\).
