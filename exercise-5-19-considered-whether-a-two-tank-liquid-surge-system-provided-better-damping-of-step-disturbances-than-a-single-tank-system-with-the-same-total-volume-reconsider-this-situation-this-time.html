Guest Question.

considered whether a two-tank liquid surge system provided better damping of step disturbances than a single-tank system with the same total volume. Reconsider this situation, this time with respect to sinusoidal disturbances; that is, determine which system better damps sinusoidal inputs of frequency \(\omega\). Does your answer depend on the value of \(\omega\) ?

Admin
Solution

From the solution to , for the two-tank system

\(\begin{aligned}&\frac{H_{1}^{\prime}(s) / h_{1 \max }^{\prime}}{Q_{1 i}^{\prime}(s)}=\frac{0.01}{1.32 s+1}=\frac{K}{\tau s+1} \\&\frac{H_{2}^{\prime}(s) / h_{2 \max }^{\prime}}{Q_{1 i}^{\prime}(s)}=\frac{0.01}{(1.32 s+1)^{2}}=\frac{K}{(\tau s+1)^{2}} \\&\frac{Q_{2}^{\prime}(s)}{Q_{1 i}^{\prime}(s)}=\frac{0.1337}{(1.32 s+1)^{2}}=\frac{0.1337}{(\tau s+1)^{2}}\end{aligned}\)

and for the one-tank system

\(\begin{aligned}&\frac{H^{\prime}(s) / h_{\max }^{\prime}}{Q_{1 i}^{\prime}(s)}=\frac{0.01}{2.64 s+1}=\frac{K}{2 \tau s+1} \\&\frac{Q^{\prime}(s)}{Q_{1 i}^{\prime}(s)}=\frac{0.1337}{2.64 s+1}=\frac{0.1337}{2 \tau s+1}\end{aligned}\)

For a sinusoidal input \(q_{1 i}^{\prime}(t)=A \sin \omega t\), the amplitudes of the heights and flow rates are

\(\begin{aligned}&\hat{A}\left[h^{\prime} / h_{\max }^{\prime}\right]=K A / \sqrt{4 \omega^{2} \tau^{2}+1} (1)\\&\hat{A}\left[q^{\prime}\right]=0.1337 A / \sqrt{4 \omega^{2} \tau^{2}+1} (2)\end{aligned}\)

for the one-tank system, and

\(\begin{aligned}&\hat{A}\left[h_{1}^{\prime} / h_{1_{\max }}^{\prime}\right]=K A / \sqrt{\omega^{2} \tau^{2}+1} (3)\\&\hat{A}\left[h_{2}^{\prime} / h_{2_{\max }^{\prime}}^{\prime}\right]=K A / \sqrt{\left(\omega^{2} \tau^{2}+1\right)^{2}} (4)\\&\hat{A}\left[q_{2}^{\prime}\right]=0.1337 A / \sqrt{\left(\omega^{2} \tau^{2}+1\right)^{2}} (5)\end{aligned}\)

for the two-tank system.

Comparing (1) and (3), for all \(\omega\)

\(\hat{A}\left[h_{1}^{\prime} / h_{1 \max }^{\prime}\right] \geq \hat{A}\left[h^{\prime} / h_{\max }^{\prime}\right]\)

Hence, for all \(\omega\), the first tank of the two-tank system will overflow for a smaller value of \(A\) than will the one-tank system. Thus, from the overflow consideration, the one-tank system is better for all \(\omega\). However, if \(A\) is small enough so that overflow is not a concern, the two-tank system will provide a smaller amplitude in the output flow for those values of \(\omega\) that satisfy

\(\hat{A}\left[q_{2}^{\prime}\right] \leq \hat{A}\left[q^{\prime}\right]\)

or \(\frac{0.1337 A}{\sqrt{\left(\omega^{2} \tau^{2}+1\right)^{2}}} \leq \frac{0.1337 A}{\sqrt{4 \omega^{2} \tau^{2}+1}}\)

or \(\omega \geq \sqrt{2} / \tau=1.07\)

Therefore, the two-tank system provides better damping of a sinusoidal disturbance for \(\omega \geq 1.07\) if and only if

\(\hat{A}\left[h_{1}^{\prime} / h_{1_{\max }^{\prime}}^{\prime}\right] \leq 1, \text { that is, } A \leq \frac{\sqrt{1.32^{2} \omega^{2}+1}}{0.01}\)
