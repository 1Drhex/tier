Guest Question.

(a) Using the commutator \(\left[\hat{X},\hat{P}\right] =i\hbar\), show that \(\left[\hat{X}^{m} ,\hat{P}\right]=im\hbar\hat{X}^{m-1}\), with m > 1. Can you think of a direct way to get to the same result?

(b) Use the result of (a) to show the general relation \(\left[F(\hat{X}),\hat{P}\right] =i\hbar dF(\hat{X})/ d\hat{X}\), where \(F(\hat{X})\) is a differentiable operator function of \(\hat{X}\) .

Admin
Solution

(a) Let us attempt a proof by induction. Assuming that \(\left [\hat{X}^{m} ,\hat{P}\right]=im\hbar\hat{X}^{m-1}\) is valid for
m = k (note that it holds for \(n=1;i.e.,\left[\hat{X},\hat{P}\right] =i\hbar\).

\(\left[\hat{X}^{k},\hat{P}\right]=ik\hbar\hat{X}^{k-1}\),

let us show that it holds for m = k + 1:

\(\left[\hat{X}^{k+1},\hat{P}\right]=\left[\hat{X}^{k}\hat{X},\hat{P}\right]=\hat{X}^{k}\left[\hat{X},\hat{P}\right]+\left[\hat{X}^{k},\hat{P}\right]\hat{X}\),

where we have used the relation \(\left[\hat{A}\hat{B},\hat{C}\right] \hat{A}\left[\hat{B}, \hat {C}\right] +\left[\hat{A},\hat{C}\right] \hat{B}\). Now, since \(\left[\hat{X},\hat{P}\right]=i\hbar \) and \( \left [\hat{X}^{k},\hat{P}\right]=ik\hbar\hat{X}^{k-1}\), we rewrite (2.397)

\(\left[\hat{X}^{k+1},\hat{P}\right]=\left[\hat{X}^{k}\hat{X},\hat{P}\right]=\hat{X}^{k}\left[\hat{X},\hat{P}\right]+\left[\hat{X}^{k},\hat{P}\right]\hat{X}\)

as

\(\left[\hat{X}^{k+1},\hat{P}\right]=i\hbar \hat{X}^{k}+(ik\hbar \hat{X}^{k-1})\hat{X}=i\hbar (k+1)\hat{X}^{k}\).

So this relation is valid for any value of k, notably for k = m – 1:

\(\left[\hat{X}^{m},\hat{P}\right]=im\hbar\hat{X}^{m-1}\).

In fact, it is easy to arrive at this result directly through brute force as follows. Using the relation \( \left[\hat{A}^{n}+\hat{B}\right] \hat{A}^{n-1}\left[\hat{A},\hat{B}\right] +\left[\hat{A}^{n-1},\hat {B}\right]\hat{A}\) along with \(\left[\hat{X},\hat{P}_{x} \right] =i\hbar \), we can obtain

\(\left[\hat{X}^{2},\hat{P}_{x}\right] =\hat{X}\left[\hat{X},\hat {P}_{x} \right]+\left[\hat{X},\hat{P}_{x} \right] \hat{X}=2i\hbar \hat{X}\),

which leads to

\(\left[\hat{X}^{3},\hat{P}_{x} \right]=\hat{X}^{2}\left [\hat{X},\hat{P}_{x} \right] +\left[\hat{X}^{2},\hat{P}_{x} \right] \hat{X}=3i\hat{X}^{2}\hbar ;\)

this in turn leads to

\(\left[\hat{X}^{4},\hat{P}_{x} \right]=\hat{X}^{3}\left [\hat {X},\hat{P}_{x} \right] +\left[\hat{X}^{3},\hat{P}_{x} \right]\hat{X} =4i\hat{X}^{3}\hbar\).

Continuing in this way, we can get to any power of \(\hat{X}: \left[\hat{X}^{m},\hat{P}\right] =im\hbar \hat{X}^{m-1}\).

A more direct and simpler method is to apply the commutator \( \left[\hat{X}^{m},\hat{P}\right]\) on some wave function Ψ (x):

\(\left[\hat{X}^{m},\hat{P}_{x}\right]\psi(x)=\left(\hat{X}^{m}\hat{P}_{x}-\hat{P}_{x}\hat{X}^{m}\right) \psi (x)\)

 

\(=x^{m}\left(-i\hbar \frac{d\psi (x)}{dx} \right)+i\hbar \frac{d}{dx}(x^{m}\psi (x))\)

 

\(=x^{m}\left(-i\hbar \frac{d\psi (x)}{dx} \right)+im\hbar x^{m-1}\psi (x)-x^{m}\left(-i\hbar \frac{d\psi (x)}{dx} \right)\)

 

\(=im\hbar x^{m-1}\psi (x)\).

Since \(\left[\hat{X}^{m},\hat{P}_{x}\right]\psi (x)=im\hbar x^{m-1} \psi (x)\) we see that \(\left[\hat{X}^{m} ,\hat {P}\right]=im\hbar\hat{X}^{m-1}\).

(b) Let us Taylor expand \(F(\hat{X})\) in powers of \( \hat{X},F(\hat{X})=\sum_{k}a_{k}\hat{X}^{k}\), and insert this expression into \(\left[F(\hat{X}),\hat{P}\right]\) :

\(\left[F(\hat{X},\hat{P})\right]=\left[\sum\limits_{k}{a_{k} \hat{X}^{k}},\hat{P} \right] =\sum\limits_{k}{a_{k} } \left[\hat {X}^{k},\hat{P}\right]\),

where the commutator \(\left[\hat{X}^{k},\hat{P}\right]\) is given by (2.396).

\(\left[\hat{X}^{k},\hat{P}\right]=ik\hbar\hat{X}^{k-1}\),

Thus, we have

\(\left[F(\hat{X}),\hat{P}\right]=i\hbar \sum\limits_{k}{ka_{k}\hat{X}^{k-1} } =i\hbar \frac{d(\sum_{k}a_{k}\hat{X}^{k} )}{d\hat{X}}=i\hbar \frac{dF(\hat{X})}{d\hat{X}}\).

A much simpler method again consists in applying the commutator \(\left[F(\hat{X}),\hat{P}\right]\) on some wave function Ψ(x). Since \(F(\hat{X})\psi (x)=F(x)\psi (x)\), we have

\(\left[F(\hat{X}),\hat{P}\right]\psi (x)=F(\hat{X})\hat{P}\psi (x)+i\hbar \frac{d}{dx}(F(x)\psi (x))\)

 

\(=F(\hat{X})\hat{P}\psi (x)-\left(-i\hbar \frac{d\psi (x)}{dx} \right)F(x)+i\hbar \frac{dF(x)}{dx}\psi (x)\)

 

\(=F(\hat{X})\hat{P}\psi (x)-F(\hat{X})\hat{P}\psi (x)+i\hbar \frac{dF(x)}{dx}\psi (x)\)

 

\(=i\hbar \frac{dF(x)}{dx}\psi (x)\).

Since \(\left[F(\hat{X}),\hat{P}\right]\psi (x)=i\hbar \frac{dF(x)}{dx}\psi (x)\) we see that \( \left[F(\hat{X}),\hat{P}\right] =i\hbar \frac{dF(\hat{X})}{d\hat{X}}\).
