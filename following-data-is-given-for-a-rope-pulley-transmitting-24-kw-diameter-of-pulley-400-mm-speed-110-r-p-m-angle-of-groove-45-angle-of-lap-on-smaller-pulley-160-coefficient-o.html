Guest Question.

Following data is given for a rope pulley transmitting 24 kW :
Diameter of pulley = 400 mm ; Speed = 110 r.p.m.; angle of groove = \(45^{\circ}\) ; Angle of lap on smaller pulley = \(160^{\circ}\) ; Coefficient of friction = 0.28 ; Number of ropes = 10 ; Mass in kg/m length of ropes = \(53 C^{2}\) ; and working tension is limited to 122 \(C^{2}\) kN, where C is girth of rope in metres.
Find initial tension and diameter of each rope.

Admin
Solution

Given : \(P_{ T }\)= 24 kW ; d = 400 mm = 0.4 m ; N = 110 r.p.m. ; \(2 \beta=45^{\circ} \text { or } \beta=22.5^{\circ}\) ; \(\theta=160^{\circ}=160 \times \pi / 180=2.8 rad ; n=0.28 ; n=10 ; m=53 C ^{2} kg / m ; T=122 C^{2} kN\)\(=122 \times 10^{3} C^{2} N\)

 

Initial tension

We know that power transmitted per rope,

\(P=\frac{\text { Total power transmitted }}{\text { No. of ropes }}=\frac{P_{ T }}{n}=\frac{24}{10}=2.4 kW =2400 W\)

 

and velocity of the rope \(v=\frac{\pi d . N}{60}=\frac{\pi \times 0.4 \times 110}{60}=2.3 m / s\)

Let T = Tension in the tight side of the rope, and

\(T_{2}\) = Tension in the slack side of the rope.

We know that power transmitted per rope ( P )

\(2400=\left(T_{1}-T_{2}\right) v=\left(T_{1}-T_{2}\right) 2.3\)

 

\(\therefore \) \(T_{1}-T_{2}=2400 / 2.3=1043.5 N\) …(i)

We know that

\(2.3 \log \left(\frac{T_{1}}{T_{2}}\right)=\mu . \theta \operatorname{cosec} \beta=0.28 \times 2.8 \times \operatorname{cosec} 22.5^{\circ}=2.05\)

 

\(\log \left(\frac{T_{1}}{T_{2}}\right)=\frac{2.05}{2.3}=0.8913 \text { or } \frac{T_{1}}{T_{2}}=7.786\) …(ii) …(Taking antilog of 0.8913)

From equations (i) and (ii),

\(T_{1}=1197.3 N , \text { and } T_{2}=153.8 N\)

We know that initial tension in each rope,

\(T_{0}=\frac{T_{1}+T_{2}}{2}=\frac{1197.3+153.8}{2}=675.55 N\)

 

Diameter of each rope

Let \(d_{1}\) = Diameter of each rope,

We know that centrifugal tension,

\(T_{ C }=m \cdot v^{2}=53 C^{2}(2.3)^{2}=280.4 C^{2} N\)

and working tension (T),

\(122 \times 10^{3} C^{2}=T_{1}+T_{ C }=1197.3+280.4 C^{2}\)

 

\(122 \times 10^{3} C^{2}-280.4 C^{2}=1197.3\)

 

\(\therefore \) \(C^{2}=9.836 \times 10^{-3} \text {or } C=0.0992 m =99.2 mm\)

We know that girth (i.e. circumference) of rope (C),

\(99.2=\pi d_{1} \text { or } d_{1}=99.2 / \pi=31.57 mm\)
