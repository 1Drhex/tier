Solution

In view of the results of Example 9.4, the hybrid control law can be designed by choosing the control inputs \( \alpha_{\nu } \) and \(f_{\lambda }\) according to (9.80) and (9.84), respectively, with suitable control gains.

\( \alpha _{\nu }\) = \( \dot{\nu }_{d} + K_{P\nu } \left(\nu _{d} – \nu \right) + K_{I \nu } \int_{0}^{t}{\left(\nu _{d} \left(\varsigma \right) – \nu\left(\varsigma \right) \right) } d\varsigma\) (9.80)

\( f_{\lambda } \) = \( – k_{D\lambda } \dot{\hat{\lambda } } + K_{P\lambda } \left(\lambda _{d} – \lambda \right) \) (9.84)

In detail, the control gains in (9.80) can be set to

\( k_{P\nu } \) = 16 \( k_{I\nu } \) = 100

corresponding to \(\omega_{n\nu } \) = 10 rad/s and \(\zeta_{\nu } \) = 0.8 for the dynamics of the velocity error \(\nu_{d} – \nu\)
