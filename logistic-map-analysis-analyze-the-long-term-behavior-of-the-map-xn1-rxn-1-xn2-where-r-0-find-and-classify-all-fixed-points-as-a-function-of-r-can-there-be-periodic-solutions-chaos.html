Guest Question.

Logistic Map: Analysis

Analyze the long-term behavior of the map \(x_{n+1}=r x_{n} /\left(1+x_{n}^{2}\right)\), where \(r>0\). Find and classify all fixed points as a function of \(r\). Can there be periodic solutions? Chaos?

Admin
Solution

\(x_{n+1}=\frac{r x_{n}}{1+x_{n}^{2}}\)

The fixed points occur when

\(\begin{aligned}&x=\frac{r x}{1+x^{2}} \\&x\left(1+x^{2}\right)=r x \\&x\left(1+x^{2}\right)-r x=x\left(1-r+x^{2}\right)=0 \\&x=0, \pm \sqrt{1-r}\end{aligned}\)

So there are three fixed points if \(r \leq 1\) and one fixed point if \(1<r\).

\(\begin{aligned}&f(x)=\frac{r x}{1+x^{2}} \\&f^{\prime}(x)=\frac{r\left(1+x^{2}\right)-2 r x^{2}}{\left(1+x^{2}\right)^{2}}=\frac{r\left(1-x^{2}\right)}{\left(1+x^{2}\right)^{2}} \\&f^{\prime}(0)=r \quad f^{\prime}(\pm \sqrt{1-r})=\frac{2}{r}-1 \\&r<1 \Rightarrow x^{*}=0 \text { is stable, and } x^{*}=\pm \sqrt{1-r} \text { are unstable } \\&r=1 \Rightarrow f^{\prime}(0)=1 \text { is inconclusive } \\&\left|x_{n+1}\right|=\left|\frac{x_{n}}{1+x_{n}^{2}}\right|<\left|\frac{x_{n}}{1}\right|=\left|x_{n}\right| \Rightarrow x^{*}=0 \text { is stable } \\&1<r \Rightarrow x^{*}=0 \text { is stable }\end{aligned}\)

As for the existence of periodic orbits, we already proved that the sequence \(\left|x_{n}\right|\) is monotonically decreasing when \(r \leq 1\) and \(x_{n} \neq 0\), which rules out any periodic orbits.

It turns out that there are no periodic orbits for \(1<r\) either, and never any chaos. We can prove this by showing that the sequence \(\left|x_{n}\right|\) is monotonically increasing, monotonically decreasing, or constant for any positive \(r\) value.

\(\begin{aligned}&|x|<\left|\frac{r x}{1+x^{2}}\right|=\frac{r|x|}{1+x^{2}} \\&|x|\left(1+x^{2}\right)<r|x| \\&|x|\left(1-r+x^{2}\right)<0 \Rightarrow x^{2}<r-1 \Rightarrow|x|<\sqrt{r-1} \\&\left|\frac{r x}{1+x^{2}}\right|=\frac{r|x|}{1+x^{2}}<|x| \\&r|x|<|x|\left(1+x^{2}\right) \\&0<|x|\left(1-r+x^{2}\right) \Rightarrow r-1<x^{2} \Rightarrow \sqrt{r-1}<|x|\end{aligned}\)

So the last bunch of equations shows that the sequence \(\left|x_{n}\right|\) is monotonically increasing or monotonically decreasing, and constant only if the sequence starts at a fixed point. Essentially the iterations always travel away from the origin, always travel towards the origin, or stay at a fixed point.
