Guest Question.

Let us determine the current 10 in the network in Fig. 7.17a using nodal analysis, loop analysis, superposition, source exchange, Thevenin’s theorem, and Norton’s theorem.

Admin
Solution

(1) Nodal Analysis We begin with a nodal analysis of the network. The KCL equation for the supernode that includes the voltage source is

\(\frac{V_{1}}{1+j}- 2\angle 0° + \frac{V_{2}}{1}+\frac{V_{2}}{1-j} = 0\)

and the associated KVL constraint equation is

\(V_{1}+ 6 \angle 0° = V_{2}\)

Solving for \(V_{1}\) in the second equation and using this value in the first equation yields

\(\frac{V_{2}- 6 \angle 0°}{1+j}- 2\angle 0° + V_{2}+ \frac{V_{2}}{1-j} = 0\)

or

\(V_{2}\left[\frac{1}{1+j}+1+\frac{1}{1-j} \right] = \frac{6+2+2j}{1+j}\)

Solving for \(V_{2}\), we obtain

\(V_{2}= \left(\frac{4+j}{1+j} \right)V\)

Therefore,

\(I_{0}= \frac{4+j}{1+j} = \left(\frac{5}{2}- \frac{3}{2}j \right)A\)

(2) Loop Analysis The network in Fig. 7. 17b is used to perform a loop analysis. Note that one loop current is selected that passes through the independent current source. The three loop equations are

\(I_{1}=-2\angle 0°\)

 

\(1(I_{1}+I_{2})+ j1(I_{1}+I_{2})- 6 \angle 0°+1 (I_{2}+I_{3}) – j1(I_{2}+I_{3})=0\)

 

\(1I_{3}+ 1(I_{2}+I_{3})- j1(I_{2}+I_{3})=0\)

Combining the first two equations yields

\(I_{2}(2)+ I_{3}(1-j)=8(I_{1}+I_{2})+2j\)

The third loop equation can be simplified to the form

\(I_{2}(1-j)+I_{3}(2-j)=0\)

Solving this last equation for \(I_{2}\) and substituting the value into the previous equation yields

\(I_{3}\left[\frac{-4+2j}{1-j} +1 -j\right]= 8+2j\)

or

\(I_{3}=\frac{-10+6j}{4}\)

and finally

\(I_{o}= -I_{3}= \left(\frac{5}{2}- \frac{3}{2}j \right)A\)

(3) Superposition In using superposition, we apply one independent source at a time. The network in which the current source acts alone is shown in Fig. 7.18a. By combining the two parallel impedances on each end of the network, we obtain the circuit in Fig. 7.18b, where

\(Z^{'}= \frac{(1 + j)(l – j)}{( 1 + j) + ( 1 – j) } = 1 \ \Omega \)

Therefore, using current division

\(I^{'}_{o}= 1\angle 0°A\)

The circuit in which the voltage source acts alone is shown in Fig. 7. 18c. The voltage \(V^{''}_{1}\) obtained using voltage division is

\(V^{''}_{1}= \frac{(6\angle 0°) \left[\frac{1(1-j)}{1+1-j} \right] }{1+j+\left[\frac{1(1-j)}{1+1-j} \right]}\)

 

\(\ \ \ \ \ =\frac{6(1-j)}{4}V\)

and hence,

\(I^{''}_{o}=\frac{6}{4}(1-j)A\)

Then

\(I_{o}= I^{'}_{o}+ I^{''}_{o}= 1+\frac{6}{4}(1-j) = \left(\frac{5}{2}- \frac{3}{2}j \right) A \)

(4) Source Exchange As a first step in the source exchange
approach, we exchange the current source and parallel impedance for a voltage source in series with the impedance, as shown in Fig. 7.19a. Adding the two voltage sources and transforming them and the series impedance into a current source in parallel with that impedance are shown in Fig. 7.19b. Combining the two impedances that are in parallel with the 1-Ω resistor produces the network in Fig. 7.19c, where

\(Z= \frac{(1 + j)(l – j)}{l+j+l-j } = 1 \ \Omega\)

Therefore, using current division,

\(I_{o}= \left(\frac{8+2j}{1+j} \right)\left(\frac{1}{2} \right)= \frac{4+j}{1+j}\)

 

\(\ \ \ =\left(\frac{5}{2}- \frac{3}{2}j \right)A\)

(5) Thevenin Analysis In applying Thevenin’s theorem to the circuit in Fig. 7.17a, we first find the open-circuit voltage, \(V_{oc}\), as shown in Fig. 7.20a. To simplify the analysis, we perform a source exchange on the left end of the network, which results in the circuit in Fig. 7.20b. Now using voltage division

\(V_{oc}= [6+2(1+j)]\left[\frac{1-j}{1-j+1+j} \right]\)

or

\(V_{oc}= (5- 3j)V\)

The Thevenin equivalent impedance, \(Z_{Th}\), obtained at the open circuit terminals when the current source is replaced with an open circuit and the voltage source is replaced with a short circuit is shown in Fig. 7.20c and calculated to be

\(Z_{Th}= \frac{(1 + j)(l – j)}{l+j+l-j } = 1 \ \Omega \)

Connecting the Thevenin equivalent circuit to the 1-Ω resistor containing \(I_{o}\) in the original network yields the circuit in Fig. 7.20d. The current \(I_{o}\) is then

\(I_{o}= \left(\frac{5}{2}- \frac{3}{2}j \right)A \)

(6) Norton Analysis Finally, in applying Norton’s theorem to the circuit in Fig. 7.17a, we calculate the short-circuit current, \(I_{sc}\) using the network in Fig. 7.21a. Note that because of the short circuit, the voltage source is directly across the impedance in the left-most branch. Therefore,

\(I_{1}= \frac{6\angle 0°}{1+j}\)

Then using KCL,

\(I_{sc}= I_{1}+2\angle 0°= 2+\frac{6}{1+j}\)

 

\(\ \ \ \ \ =\left(\frac{8+2j}{1+j} \right)A\)

The Thevenin equivalent impedance, \(Z_{Th}\), is known to be 1-Ω, and, therefore, connecting the Norton equivalent to the 1-Ω resistor containing \(I_{o}\) yields the network in Fig. 7 .21 b. Using current division, we find that

\(I_{o}=\frac{1}{2} \left(\frac{8+2j}{1+j} \right)\)

 

\(\ \ \ =\left(\frac{5}{2}- \frac{3}{2}j \right)A\)
