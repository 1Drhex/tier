Guest Question.

Consider a particle of mass m that is free to move in a one-dimensional region of length L that closes on itself (for instance, a bead that slides frictionlessly on a circular wire of circumference L, as in ). (a) Show that the stationary states can be written in the form

\( \psi_{n}(x)=\frac{1}{\sqrt{L}} e^{2 \pi i n x / L}, \quad(-L / 2<x<L / 2) \),

where \( n=0, \pm 1, \pm 2, \ldots \) and the allowed energies are

\( E_{n}=\frac{2}{m}\left(\frac{n \pi \hbar}{L}\right)^{2} \).

Notice that—with the exception of the ground state (n = 0 ) – are all doubly degenerate.

(b) Now suppose we introduce the perturbation

\( H^{\prime}=-V_{0} e^{-x^{2} / a^{2}} \),

where \( a \ll L \). (This puts a little “dimple” in the potential at x = 0, as though we bent the wire slightly to make a “trap”.) Find the first-order correction to \( E_n \) , using Equation 7.33. Hint: To evaluate the integrals, exploit the fact that \( a \ll L \) to extend the limits from \( \pm L / 2 \text { to } \pm \infty \) after all, H′ is essentially zero outside \( -a<x<a \).

\( E_{\pm}^{1}=\frac{1}{2}\left[W_{a a}+W_{b b} \pm \sqrt{\left(W_{a a}-W_{b b}\right)^{2}+4\left|W_{a b}\right|^{2}}\right] \) (7.33).

(c) What are the “good” linear combinations of \( \psi_{n} \text { and } \psi_{-n} \), for this problem? (Hint: use Eq. 7.27.) Show that with these states you get the first-order correction using Equation 7.9.

\( \alpha W_{a a}+\beta W_{a b}=\alpha E^{1} \) (7.27).

\( E_{n}^{\prime}=\left\langle\psi_{n}^{0}\left|H^{\prime}\right| \psi_{n}^{0}\right\rangle \) (7.9).

(d) Find a hermitian operator A that fits the requirements of the theorem, and show that the simultaneous eigenstates of \( H^{0} \) and A are precisely the ones you used in (c).

Solution

(a) See .

(b) With \( a \rightarrow n, b \rightarrow-n \text {, we have } \):

\( W_{a a}=W_{b b}=-\frac{V_{0}}{L} \int_{-L / 2}^{L / 2} e^{-x^{2} / a^{2}} d x \approx-\frac{V_{0}}{L} \int_{-\infty}^{\infty} e^{-x^{2} / a^{2}} d x=-\frac{V_{0}}{L} a \sqrt{\pi} \).

\( W_{a b}=-\frac{V_{0}}{L} \int_{-L / 2}^{L / 2} e^{-x^{2} / a^{2}} e^{-4 \pi n i x / L} d x \approx-\frac{V_{0}}{L} \int_{-\infty}^{\infty} e^{-\left(x^{2} / a^{2}+4 \pi n i x / L\right)} d x=-\frac{V_{0}}{L} a \sqrt{\pi} e^{-(2 \pi n a / L)^{2}} \).

(We did this integral in .) In this case \( W_{a a}=W_{b b}, \text { and } W_{a b} \) is real, so Eq. 7.33 ⇒

\( E_{\pm}^{1}=W_{a a} \pm\left|W_{a b}\right|, \quad \text { or } \quad E_{\pm}^{1}= -\sqrt{\pi} \frac{V_{0} a}{L}\left(1 \mp e^{-(2 \pi n a / L)^{2}}\right) \).

\( E_{\pm}^{1}=\frac{1}{2}\left[W_{a a}+W_{b b} \pm \sqrt{\left(W_{a a}-W_{b b}\right)^{2}+4\left|W_{a b}\right|^{2}}\right] \) (7.33).

(c) Equation 7.27 ⇒ \( \beta=\alpha \frac{\left(E^{1}-W_{a a}\right)}{W_{a b}}=\alpha\left[\frac{\pm \sqrt{\pi}\left(V_{0} a / L\right) e^{-(2 \pi n a / L)^{2}}}{-\sqrt{\pi}\left(V_{0} a / L\right) e^{-(2 \pi n a / L)^{2}}}\right]=\mp \alpha \). Evidently, the good linear combinations are:

\( \alpha W_{a a}+\beta W_{a b}=\alpha E^{1} \) (7.27).

\( \psi_{+}=\alpha \psi_{n}-\alpha \psi_{-n}=\frac{1}{\sqrt{2}} \frac{1}{\sqrt{L}}\left[e^{i 2 \pi n x / L}-e^{-i 2 \pi n x / L}\right]= i \sqrt{\frac{2}{L}} \sin \left(\frac{2 \pi n x}{L}\right) \) and

\( \psi_{-}=\alpha \psi_{n}+\alpha \psi_{-n}= \sqrt{\frac{2}{L}} \cos \left(\frac{2 \pi n x}{L}\right) \).

Using Eq: 7.9; we have :

\( E_{n}^{\prime}=\left\langle\psi_{n}^{0}\left|H^{\prime}\right| \psi_{n}^{0}\right\rangle \) (7.9).

\( E_{+}^{1}=\left\langle\psi_{+}\left|H^{\prime}\right| \psi_{+}\right\rangle=\frac{2}{L}\left(-V_{0}\right) \int_{-L / 2}^{L / 2} e^{-x^{2} / a^{2}} \sin ^{2}\left(\frac{2 \pi n x}{L}\right) d x \).

\( E_{-}^{1}=\left\langle\psi_{-}\left|H^{\prime}\right| \psi_{-}\right\rangle=\frac{2}{L}\left(-V_{0}\right) \int_{-L / 2}^{L / 2} e^{-x^{2} / a^{2}} \cos ^{2}\left(\frac{2 \pi n x}{L}\right) d x \).

But \( \sin ^{2} \theta=(1-\cos 2 \theta) / 2, \quad \text { and } \quad \cos ^{2} \theta=(1+\cos 2 \theta) / 2, \text { so } \).

\( E_{\pm}^{1} \approx-\frac{V_{0}}{L} \int_{-\infty}^{\infty} e^{-x^{2} / a^{2}}\left[1 \mp \cos \left(\frac{4 \pi n x}{L}\right)\right] d x=-\frac{V_{0}}{L}\left[\int_{-\infty}^{\infty} e^{-x^{2} / a^{2}} d x \mp \int_{-\infty}^{\infty} e^{-x^{2} / a^{2}} \cos \left(\frac{4 \pi n x}{L}\right) d x\right] \).

\( =-\frac{V_{0}}{L}\left[\sqrt{\pi} a \mp a \sqrt{\pi} e^{-(2 \pi n a / L)^{2}}\right]=-\sqrt{\pi} \frac{V_{0} a}{L}\left[1 \mp e^{-(2 \pi n a / L)^{2}}\right] \), same as (b).

(d) Af(x) = f(-x) (the parity operator). The eigenstates are even functions (with eigenvalue +1) and odd functions (with eigenvalue -1). The linear combinations we found in (c) are precisely the odd and even linear combinations of \( \psi_{n} \text { and } \psi_{-n} \).
