WAVE ON A CLOTHESLINE

Cousin Throckmorton holds one end of the clothesline taut and wiggles it up and down sinusoidally with frequency 2.00 Hz and amplitude 0.075 m. The wave speed on the clothesline is v = 12.0 m/s. At t = 0 Throcky’s end has maximum positive displacement and is instantaneously at rest. Assume that no wave bounces back from the far end. (a) Find the wave amplitude A, angular frequency ω, period T, wavelength λ, and wave number k. (b) Write a wave function describing the wave. (c) Write equations for the displacement, as a function of time, of Throcky’s end of the clothesline and of a point 3.00 m from that end.

Admin
Solution

IDENTIFY and SET UP:

This is a kinematics problem about the clothesline’s wave motion. Throcky produces a sinusoidal wave that propagates along the clothesline, so we can use all of the expressions of this section. In part (a) our target variables are A, ω, T, λ, and k. We use the relationships v = 2πf, f = 1/T, v = λf, and k = 2π/λ. In parts (b) and (c) our target “variables” are expressions for displacement, which we’ll obtain from an appropriate equation for the wave function. We take the positive x-direction to be the direction in which the wave propagates, so either Eq. (15.4) or (15.7) will yield the desired expression. A photograph of the clothesline at time t = 0 would look like Fig. 15.9a, with the maximum displacement at x = 0 (the end that Throcky holds).

\(y(x, t)=A \cos \left[2 \pi\left(\frac{x}{\lambda}-\frac{t}{T}\right)\right]\) (15.4)

\(y(x, t)=A \cos (k x-\omega t)\) (15.7)

EXECUTE:

(a) The wave amplitude and frequency are the same as for the oscillations of Throcky’s end of the clothesline, A = 0.075 m and f = 2.00 Hz. Hence

\(\begin{aligned}\omega &=2 \pi f=\left(2 \pi \frac{\mathrm{rad}}{\mathrm{cycle}}\right)\left(2.00 \frac{\mathrm{cycles}}{\mathrm{s}}\right) \\&=4.00 \pi \mathrm{rad} /\mathrm{s}=12.6 \mathrm{rad} / \mathrm{s}\end{aligned}\)

The period is T = 1/f = 0.500 s, and from Eq. (15.1) ,

\(v=\lambda f\) (15.1)

\(\lambda=\frac{v}{f}=\frac{12.0 \mathrm{~m} / \mathrm{s}}{2.00 \mathrm{~s}^{-1}}=6.00 \mathrm{~m}\)

We find the wave number from Eq. (15.5) or (15.6):

\(k=\frac{2 \pi}{\lambda}\) (15.5)

\(\omega=v k\) (15.6)

\(k=\frac{2 \pi}{\lambda}=\frac{2 \pi \mathrm{rad}}{6.00 \mathrm{~m}}=1.05 \mathrm{rad} / \mathrm{m}\)

or \(k=\frac{\omega}{v}=\frac{4.00 \pi \mathrm{rad} / \mathrm{s}}{12.0 \mathrm{~m} / \mathrm{s}}=1.05 \mathrm{rad} / \mathrm{m}\)

(b) We write the wave function using Eq. (15.4) and the values of A, T, and l from part (a):

\(\begin{aligned}y(x, t) &=A \cos 2 \pi\left(\frac{x}{\lambda}-\frac{t}{T}\right) \\&=(0.075\mathrm{~m}) \cos 2 \pi\left(\frac{x}{6.00 \mathrm{~m}}-\frac{t}{0.500\mathrm{~s}}\right) \\&=(0.075 \mathrm{~m}) \cos [(1.05 \mathrm{rad} / \mathrm{m}) x-(12.6 \mathrm{rad} / \mathrm{s}) t]\end{aligned}\)

We can also get this same expression from Eq. (15.7) by using the values of v and k from part (a).
(c) We can find the displacement as a function of time at x = 0 and x = +3.00 m by substituting these values into the wave function from part (b):

\(\begin{aligned}y(x=0, t) &=(0.075 \mathrm{~m}) \cos 2 \pi\left(\frac{0}{6.00 \mathrm{~m}}-\frac{t}{0.500 \mathrm{~s}}\right) \\&=(0.075 \mathrm{~m}) \cos (12.6 \mathrm{rad} /\mathrm{s}) t \\y(x=+3.00 \mathrm{~m}, t) &=(0.075 \mathrm{~m}) \cos 2 \pi\left(\frac{3.00\mathrm{~m}}{6.00 \mathrm{~m}}-\frac{t}{0.500 \mathrm{~s}}\right) \\&=(0.075 \mathrm{~m}) \cos [\pi-(12.6 \mathrm{rad} / \mathrm{s}) t] \\&=-(0.075\mathrm{~m})\cos (12.6 \mathrm{rad} / \mathrm{s}) t\end{aligned}\)

 

EVALUATE: : In part (b), the quantity (1.05 rad/m)x – (12.6 rad/s)t is the phase of a point x on the string at time t. The two points in part (c) oscillate in SHM with the same frequency and amplitude, but their oscillations differ in phase by (1.05 rad/m)(3.00 m) = 3.15 rad = π radians—that is, one half-cycle—because the points are separated by one half-wavelength: λ/2 = (6.00 m)/2 = 3.00 m.
Thus, while a graph of y versus t for the point at x = 0 is a cosine curve (like Fig. 15.9b), a graph of y versus t for the point x = 3.00 m is a negative cosine curve (the same as a cosine curve shifted by one half-cycle).

Using the expression for y(x = 0, t) in part (c), can you show that the end of the string at x = 0 is instantaneously at rest at t = 0, as stated at the beginning of this example? (Hint: Calculate the y-velocity at this point by taking the derivative of y with respect to t.)
