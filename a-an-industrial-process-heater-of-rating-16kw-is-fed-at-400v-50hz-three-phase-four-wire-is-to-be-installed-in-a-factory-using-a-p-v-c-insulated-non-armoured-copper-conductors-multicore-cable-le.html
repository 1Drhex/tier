Guest Question.

(a) An industrial process heater of rating 16kW is fed at 400V 50Hz. Three-phase four-wire is to be installed in a factory using a p.v.c.-insulated, non-armoured, copper conductors multicore cable. Length of run is 25m clipped direct to a wall; assume a maximum ambient temperature of 35C and protection by BS 3036 fuses.
(b) If the BS 3036 fuses were replaced by BS 88 (Gg) fuses what would be the effect on cable current rating?
Circuit details:
As it is a heater p.f. is unity.
Ambient temperature 35C so Ca is 0.94.
Using BS 3036 fuses so Cr is 0.725.
Current demand \(I_{b} =\frac{16000}{\sqrt{3}\times 400}\)= 23.1A

Admin
Solution

(a) Select as\( I_{n} \)30A BS 3036 fuses. Thus minimum current rating is
\( I_{t} =\frac{30}{0.94\times 0.725}\)
= 44A
Using BS 7671 Table 4D2A or IEE On-Site Guide Table 6E1, from column 7 select 10\(mm^{2}\) cables (57A) and using BS 7671 Table 4D2B or IEE On-Site Guide Table 6E2, column 4, read mV/A/m value for 10\(mm^{2}\) cables as 3.8mV/A/m.
Volts drop in 25m =\(\frac{23.1\times 25\times 3.8}{1000}\) = 2.19V
Thus\(10mm^{2}\) cables will be suitable.
(b) Select as\( I_{n}\) 25A BS 88 fuses. In this case Cr is 1. Thus
minimum current rating is \(I_{t} = \frac{25}{0.94\times 1}\)
= 26.6A
Using BS 7671 Table 4D2A or IEE On-Site Guide Table 6E1, from column 7 select 4\(mm^{2}\) cables (32A) and using BS 7671 Table 4D2B or IEEE On-Site Guide Table 6E2, column 4, read mV/A/m value for 4\(mm^{2}\) cables as 9.5mV/A/m.
Volts drop in 25m = \(\frac{23.1\times 25\times 9.5\times 9.5}{1000}\)
= 5.49V
Thus mm\(^{2}\)cables will be suitable.
