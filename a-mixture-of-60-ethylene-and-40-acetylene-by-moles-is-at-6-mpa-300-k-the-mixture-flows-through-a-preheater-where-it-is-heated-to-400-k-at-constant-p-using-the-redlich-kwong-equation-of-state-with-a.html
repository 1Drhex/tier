Guest Question.

A mixture of 60% ethylene and 40% acetylene by moles is at 6 MPa, 300 K. The mixture flows through a preheater where it is heated to 400 K at constant P. Using the Redlich Kwong equation of state with a, b for a mixture find the inlet specific volume. Repeat using Kays rule and the generalized charts.

Admin
Solution

To do the EOS we need the gas constant, so from Eq.11.5 we get

\(\begin{aligned}& M _{\operatorname{mix}}=\sum y _{ i } M _{ i }=0.6 \times 28.054+0.4 \times 26.068=27.26 \\& R _{\operatorname{mix}}=8.3145 / 27.26=0.305 kJ / kg K\end{aligned}\)

Redlich Kwong EOS the individual component parameters, Eq.12.54, 12.55.

\(\begin{aligned}& a _{ C 2 H 4}=0.42748 \frac{ R ^{2} T _{ c }^{5 / 2}}{ P _{ c }}=0.42748 \frac{0.2964^{2} \times 282.4^{2.5}}{5040}=9.9863 \\& a _{ C 2 H 2}=0.42748 \frac{ R ^{2} T _{ c }^{5 / 2}}{ P _{ c }}=0.42748 \frac{0.3193^{2} \times 308.3^{2.5}}{6140}=11.8462\end{aligned}\)

 

\(\begin{aligned}& b _{ C 2 H 4}=0.08664 \frac{ RT _{ c }}{ P _{ c }}=0.08664 \frac{0.2964 \times 282.4}{5040}=0.001439 \\& b _{ C 2 H 2}=0.08664 \frac{ RT _{ c }}{ P _{ c }}=0.08664 \frac{0.3193 \times 308.3}{6140}=0.001389\end{aligned}\)

Now the mixture parameters are from Eq.12.84 so we need the mass fractions

\(c _{ C 2 H 4}=\frac{ y M }{ M _{ mix }}=\frac{0.6 \times 28.054}{27.26}=0.6175, \quad c _{ C 2 H 4}=1- c _{ C 2 H 4}=0.3825\) \(\begin{gathered}a _{\operatorname{mix}}=\left(\sum c _{ i } a _{ i }^{1 / 2}\right)^{2}=(0.6175 \times \sqrt{9.9863}+0.3825 \times \sqrt{11.8462})^{2}=10.679 \\b _{ mix }=\sum c _{ i } b _{ i }=0.6175 \times 0.001439+0.3825 \times 0.001389=0.00142\end{gathered}\)

Using now Eq.12.53:

\(\begin{gathered}P =\frac{ RT }{ v – b }-\frac{ a }{ v ( v + b ) T ^{1 / 2}} \\6000=\frac{0.305 \times 300}{ v -0.00142}-\frac{10.679}{ v ( v +0.00142) 300^{1 / 2}}\end{gathered}\)

By trial and error we find the specific volume, v = 0.006683 \(m ^{ 3 } / k g\)
Kay’s rule Eq.12.84

\(\begin{aligned}& P _{ c mix }=0.6 \times 5.04+0.4 \times 6.14=5.48 MPa \\& T _{ c mix }=0.6 \times 282.4+0.4 \times 308.3=292.8 K\end{aligned}\)

Reduced properties: \(P _{ r }=\frac{6}{5.48}=1.095, \quad T _{ r }=\frac{300}{292.8}=1.025\)

Fig. D.1: Z = 0.4 (difficult to read)

v = ZRT/P = 0.4 × 0.305 × 300 / 6000 = 0.0061 \(m ^{ 3 } / k g\)

Eq.11.5: \(M_{\operatorname{mix}}=\frac{m_{ tot }}{n_{ tot }}=\frac{\sum n_{i} M_{i}}{n_{ tot }}=\sum y_{i} M_{i}\)

Eq.12.53 : \(Z^{3}-\left(\frac{P_{r}}{8 T_{r}}+1\right) Z^{2}+\left(\frac{27 P_{r}}{64 T_{r}^{2}}\right) Z-\frac{27 P_{r}^{2}}{512 T_{r}^{3}}=0\)

Eq.12.54 : \(P=\frac{R T}{v-b}-\frac{a}{v(v+b) T^{1 / 2}}\)

Eq.12.55 :
\(\begin{aligned}a &=0.42748 \frac{R^{2} T_{c}^{5 / 2}}{P_{c}} \\b &=0.08664 \frac{R T_{c}}{P_{c}}\end{aligned}\)

Eq.12.84 : \(\left(P_{c}\right)_{\operatorname{mix}}=\sum_{i} y_{i} P_{c i}, \quad\left(T_{c}\right)_{\operatorname{mix}}=\sum_{i} y_{i} T_{c i}\)
