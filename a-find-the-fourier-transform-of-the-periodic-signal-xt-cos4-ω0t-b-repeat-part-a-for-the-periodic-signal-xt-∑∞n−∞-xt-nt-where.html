Guest Question.

(a) Find the Fourier transform of the periodic signal

\(x(t)=\cos ^{4} \omega_{0} t\)

(b) Repeat part (a) for the periodic signal

\(\tilde{x}(t)=\sum_{n=-\infty}^{\infty} x(t+n T)\)

where

\(x(t)= \begin{cases}\sin \omega_{0} t & \text { for } 0 \leq t \leq \tau_{0} / 2 \\ 0 & \text { for }-\tau_{0} / 2 \leq t \leq 0\end{cases}\)

where \(\omega_{0}=2 \pi / \tau_{0}\).

Admin
Solution

(a) We can write

 

\(\begin{aligned}x(t) &=\left(\cos ^{2} \omega_{0} t\right)\left(\cos ^{2} \omega_{0} t\right) \\&=\frac{1}{4}\left(\cos 2 \omega_{0} t+1\right)\left(\cos 2 \omega_{0} t+1\right) \\&=\frac{1}{4}\left(\cos ^{2} 2 \omega_{0} t+2 \cos 2 \omega_{0} t+1\right) \\&=\frac{1}{4}\left[\frac{1}{2}\left(\cos 4 \omega_{0} t+1\right)+2 \cos 2 \omega_{0} t+1\right] \\&=\frac{1}{8} \cos 4 \omega_{0} t+\frac{1}{2} \cos 2 \omega_{0} t+\frac{3}{8}\end{aligned}\)

 

Now from Table 6.2, we get

 

\(X(j \omega)=\pi\left\{\frac{1}{8}\left[\delta\left(\omega+4 \omega_{0}\right)+\delta\left(\omega-4 \omega_{0}\right)\right]+\frac{1}{2}\left[\delta\left(\omega+2 \omega_{0}\right)+\delta\left(\omega-2 \omega_{0}\right)\right]+\frac{3}{4} \delta(\omega)\right\}\)

 

(b) The Fourier series of periodic signal \(\tilde{x}(t)\) is given by Eqs. (2.3) and (2.5) where

 

\(\tilde{x}(t)=\sum_{k=-\infty}^{\infty} X_{k} e^{j k \omega_{0} t} \quad \text { for }-\tau_{0} / 2 \leq t \leq \tau_{0} / 2\) (2.3)

 

\(X_{k}=\frac{1}{\tau_{0}} \int_{-\tau_{0} / 2}^{\tau_{0} / 2} x(t) e^{-j k \omega_{0} t} d t\) (2.5)

 

\(\begin{aligned}X_{n} &=\frac{1}{\tau_{0}} \int_{-\tau_{0} / 2}^{\tau_{0} / 2}\left[u(t) \sin \omega_{0} t\right] e^{-j n \omega_{0} t} d t \\&=\frac{1}{\tau_{0}} \int_{0}^{\tau_{0} / 2} \sin \omega_{0} t\left[\cos n \omega_{0} t-j \sin n \omega_{0} t\right] d t \\&=\frac{1}{\tau_{0}} \int_{0}^{\tau_{0} / 2}\left[\cos n \omega_{0} t \sin \omega_{0} t-j \sin n \omega_{0} t \sin \omega_{0} t\right] d t\end{aligned}\)

 

\(\begin{aligned}&=\frac{1}{2 \tau_{0}} \int_{0}^{\tau_{0} / 2}\left\{\left[\sin (n+1) \omega_{0} t-\sin (n-1) \omega_{0} t\right]-j\left[\cos (n-1) \omega_{0} t+\cos (n+1) \omega_{0} t\right]\right\} d t \\&=\frac{1}{2 \tau_{0}}\left[\frac{-\cos (n+1) \omega_{0} t}{(n+1) \omega_{0}}+\frac{\cos (n-1) \omega_{0} t}{(n-1) \omega_{0}}-j \frac{\sin (n-1) \omega_{0} t}{(n-1) \omega_{0}}+j \frac{\sin (n+1) \omega_{0} t}{(n+1) \omega_{0}}\right]_{0}^{\tau_{0} / 2} \\&=\frac{1}{4 \pi}\left[\frac{\cos (n-1) \pi-1}{n-1}-\frac{\cos (n+1) \pi-1}{n+1}-j \frac{\sin (n-1) \pi}{n-1}+j \frac{\sin (n+1) \pi}{n+1}\right] \\&=\frac{1}{4 \pi}\left[\frac{-\cos n \pi-1}{n-1}-\frac{-\cos n \pi-1}{n+1}+j \frac{\sin n \pi}{n-1}-j \frac{\sin n \pi}{n+1}\right] \\&=-\left[\frac{\cos n \pi+1-j \sin n \pi}{2 \pi\left(n^{2}-1\right)}\right]\end{aligned}\)

 

Evaluating \(X_{n}\) and noting that l’Hopital’s rule is required for the cases n = ±1, the following values of \(X_{n}\) can be obtained:

 

\(X_{0}=\frac{1}{\pi} \quad X_{1}=-X_{-1}=-\frac{j}{4} \quad X_{2}=X_{-2}=-\frac{1}{3 \pi}\)

 

\(X_{3}=X_{-3}=0 \quad X_{4}=X_{-4}=-\frac{1}{15 \pi} \quad X_{5}=X_{-5}=0\)

 

\(X_{6}=X_{-6}=-\frac{1}{35 \pi}\), …

 

On using Eqs. (2.9) and (2.10), the Fourier series of \(\tilde{x}(t)\) can be deduced as

 

\(\tilde{x}(t)=\frac{1}{2} a_{0}+\sum_{k=1}^{\infty}\left(a_{k} \cos k \omega_{0} t+b_{k} \sin k \omega_{0} t\right)\) (2.9)

 

\(a_{0}=2 X_{0}=\frac{2}{\tau_{0}} \int_{-\tau_{0} / 2}^{\tau_{0} / 2} x(t) d t\) (2.10a)

 

\(a_{k}=X_{k}+X_{-k}=\frac{2}{\tau_{0}} \int_{-\tau_{0} / 2}^{\tau_{0} / 2} x(t) \cos k \omega_{0} t d t\) (2.10b)

 

\(b_{k}=j\left(X_{k}-X_{-k}\right)=\frac{2}{\tau_{0}} \int_{-\tau_{0} / 2}^{\tau_{0} / 2} x(t) \sin k \omega_{0} t d t\) (2.10c)

 

\(\tilde{x}(t)=\frac{1}{2} a_{0}+\sum_{n=1}^{\infty} a_{n} \cos n \omega_{0} t+\sum_{n=1}^{\infty} b_{n} \sin n \omega_{0} t\) (6.24)

 

where

 

\(a_{0}=2 X_{0} \quad a_{n}=X_{n}+X_{-n} \quad b_{n}=j\left(X_{n}-X_{-n}\right)\)

 

or

 

\(a_{0}=\frac{2}{\pi} \quad a_{1}=0 \quad a_{2}=-\frac{2}{3 \pi} \quad a_{3}=0 \quad a_{4}=-\frac{2}{15 \pi}\)

 

\(a_{5}=0 \quad a_{6}=-\frac{2}{35 \pi} \quad a_{7}=0, \cdots\)

 

\(b_{1}=\frac{1}{2} \quad b_{2}=0 \quad b_{3}=0 \quad b_{4}=0 \quad b_{5}=0 \quad b_{6}=0, \cdots\)

 

Now from Table 6.2, we get

 

\(F \tilde{x}(t)=a_{0} \pi \delta(\omega)+\sum_{k=1}^{\infty} a_{n} \pi\left[\delta\left(\omega+n \omega_{0}\right)+\delta\left(\omega-n \omega_{0}\right)\right]\)

 

\(+\sum_{k=1}^{\infty} j b_{n} \pi\left[\delta\left(\omega+n \omega_{0}\right)-\delta\left(\omega-n \omega_{0}\right)\right]\) (6.25)
