Guest Question.

A hollow structural steel flexural member (Figure P15.20b) is subjected to a load of P = 27 kips as shown in Figure P15.20a. The cross-sectional dimensions of the shape as shown in Figure P15.20b are d = 12.00 in., b = 8.00 in., t = 0.25 in., \(x_{H}\) = 4.0 in., and \(x_{K}\) = 3.0 in. Using a = 19 in., determine the principal stresses and the maximum shear stress acting at points H and K, as shown Figure P15.20b. Show these stresses on an appropriate sketch for each point.

Admin
Solution

Moment of inertia about the z axis:

\(\begin{array}{ccccc}\text { Shape } & I_{C} & d=y_{i}-\bar{y} & d^{2} A & I_{C}+d^{2} A \\& \left( in .{ }^{4}\right) & (\text { in. }) & \left( in .{ }^{4}\right) & \left( in .{ }^{4}\right) \\\hline \text { outer rectangle } & 1,152.000 & 0.000 & 0.000 & 1,152.000 \\\text { inner rectangle } & -950.547 & 0.000 & 0.000 & -950.547 \\\hline \text { Moment of inertia about the } z \text { axis }\left(\text { in. }^{4}\right)= 201.453\end{array}\)

Bending stress at point H:

\(\sigma_{y}=\frac{(513 kip – in .)(4.00 in .)}{201.453 in .^{4}}=10.186 ksi\) (tension, by inspection)

Transverse and horizontal shear stress at point H:

\(\begin{aligned}&Q=(8 \text { in. })(0.25 in .)(5.875 in .)+2(0.25 in .)(1.750 in .)(4.875 in .)=16.016 in .{ }^{3} \\&\tau=\frac{V Q}{I t}=\frac{(27 kips )\left(16.016 in .^{3}\right)}{\left(201.453 in. ^{4}\right)(2)(0.250 in. )}=4.293 ksi \quad \text { (Note: Sense to be determined by inspection) }\end{aligned}\)

Stress element at H:

 

Principal stress calculations:
The principal stress magnitudes can be computed from Eq. (12.12):

\(\begin{aligned}\sigma_{p 1, p 2} &=\frac{\sigma_{x}+\sigma_{y}}{2} \pm \sqrt{\left(\frac{\sigma_{x}-\sigma_{y}}{2}\right)^{2}+\tau_{x y}^{2}} \\&=\frac{(0)+(10.186)}{2} \pm \sqrt{\left(\frac{(0)-(10.186)}{2}\right)^{2}+(4.293)^{2}} \\&=5.093 \pm 6.661\end{aligned}\)

 

\(\begin{aligned}\sigma_{p 1} &=11.75 ksi \text { and } \sigma_{p 2}=-1.568 ksi \\\tau_{\max } &=6.66 ksi \text { (maximum in-plane shear stress) } \\\sigma_{\text {avg }} &=5.09 ksi ( T ) \text { (normal stress on planes of maximum in-plane shear stress) }\end{aligned}\)

 

\(\begin{aligned}\tan 2 \theta_{p} &=\frac{\tau_{x y}}{\left(\sigma_{x}-\sigma_{y}\right) / 2}=\frac{4.293}{[(0)-(10.186)] / 2}=\frac{4.293}{-5.093}=-0.8429 \\&\left.\therefore \theta_{p}=-20.06^{\circ} \quad \text { (clockwise from the } x \text { axis to the direction of } \sigma_{p 2}\right)\end{aligned}\)

 

Bending stress at point K:

\(\sigma_{y}=\frac{(513 kip – in .)(3.00 in .)}{201.453 in .^{4}}=7.639 ksi\) (compression, by inspection)

Transverse and horizontal shear stress at point K:

\(\begin{aligned}&Q=(8 in. )(0.25 in .)(4.875 in .)+2(0.25 in .)(2.750 in .)(4.375 in .)=17.766 in .{ }^{3} \\&\tau=\frac{V Q}{I t}=\frac{(27 kips )\left(17.766 in. ^{3}\right)}{\left(201.453 in. { }^{4}\right)(2)(0.250 in .)}=4.762 ksi \quad \text { (Note: Sense to be determined by inspection) }\end{aligned}\)

Stress element at K:

Principal stress calculations:
The principal stress magnitudes can be computed from Eq. (12.12):

\(\begin{aligned}\sigma_{p 1, p 2} &=\frac{\sigma_{x}+\sigma_{y}}{2} \pm \sqrt{\left(\frac{\sigma_{x}-\sigma_{y}}{2}\right)^{2}+\tau_{x y}^{2}} \\&=\frac{(0)+(-7.639)}{2} \pm \sqrt{\left(\frac{(0)-(-7.639)}{2}\right)^{2}+(4.762)^{2}} \\&=-3.820 \pm 6.105\end{aligned}\)

 

\(\begin{aligned}&\sigma_{p 1}=2.29 ksi \text { and } \quad \sigma_{p 2}=-9.93 ksi \\&\tau_{\max }=6.11 ksi \text { (maximum in-plane shear stress) } \\&\sigma_{\text {avg }}=3.82 ksi ( C ) \text { (normal stress on planes of maximum in-plane shear stress) }\end{aligned}\)

 

\(\begin{aligned}\tan 2 \theta_{p} &=\frac{\tau_{x y}}{\left(\sigma_{x}-\sigma_{y}\right) / 2}=\frac{4.762}{[(0)-(-7.639)] / 2}=\frac{4.762}{3.820}=1.2467 \\&\left.\therefore \theta_{p}=25.63^{\circ} \quad \text { (counterclockwise from the } x \text { axis to the direction of } \sigma_{p 1}\right)\end{aligned}\)
