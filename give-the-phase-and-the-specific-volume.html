Guest Question.

Give the phase and the specific volume.

Admin
Solution

a. R-410A T = -25 F, P = 30 \(lbf / in .^{2}\) Table F.9.2 \(T > T _{ sat }\) = -33.24 F

⇒ sup.vap. \(v \cong 1.9534+\frac{-25+33.24}{-20+33.24}(2.0347-1.9534)= 2 . 0 0 4 ft ^{3} / lbm\)

 

b. R-410A T = -25 F , P = 40 \(lbf / in ^{2}\) Table F.9.2 \(T < T _{\text {sat }}\) = -21.0 F

\(P > P _{\text {sat }} \Rightarrow \text { compresssed Liquid } \quad \text { F.9.1: } v \cong v _{ f }=0.01246 ft ^{3} / lbm\)

 

c. \(H _{2} O\) T = 280 F, P = 35 \(lbf / in .^{2}\) Table F.7.1 \(P < P _{\text {sat }}\) = 49.2 psia

⇒ superheated vapor

\(v \cong 21.734+(10.711-21.734) \times(15 / 20)= 1 . 0 6 6 9 f t ^{3} / l b m\)

 

d. \(NH _{3}\) T = 60 F, P = 15 \(lbf / in ^{2}\) Table F.8.1 \(P _{\text {sat }}\) = 107.6 psia

\(P < P _{\text {sat }} \Rightarrow \text { superheated vapor } \quad v \cong 21.564 ft ^{3} / lbm\)

 

States shown are placed relative to the two-phase region, not to each other.
