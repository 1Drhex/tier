Guest Question.

A structural member in a braced frame must support the following service loads and moments: an axial compressive dead load of 25 kips and a live load of 75 kips; a dead load moment of 12.5 ft-kips about the strong axis and a live load moment of 37.5 ft-kips about the strong axis; a dead load moment of 5 ft-kips about the weak axis and a live load moment of 15 ft-kips about the weak axis. The moments occur at one end; the other end is pinned. The member length is 15 feet. The frame analysis was a first-order analysis with reduced member stiffnesses, so the moment amplification method with \(K_{x} = 1.0\) can be used. In the direction perpendicular to the frame, \(K_{y} = 1.0\). There are no transverse loads on the member. Use A992 steel and select a W10 shape.

Admin
Solution

The factored axial load is

\(P_{nt} = P_{u} = 1.2P_{D} + 1.6P_{L} = 1.2(25) + 1.6(75) = 150\) kips

(There is no amplification of axial loads in members braced against sidesway.) The factored moments are

\(M_{ntx} = 1.2(12.5) + 1.6(37.5) = 75.0\) ft-kips
\(M_{nty} = 1.2(5) + 1.6(15) = 30.0\) ft-kips

The amplification factor \(B_{1}\) can be estimated as 1.0 for purposes of making a trial selection. For the two axes,

\(M_{ux} = B_{1x}M_{ntx} = 1.0(75) = 75\) ft-kips
\(M_{uy} = B_{1y}M_{nty} = 1.0(30) = 30\) ft-kips

Try a W10 ×49. From Table 6-1, \(p = 2.23 × 10^{−3}, b_{x} = 4.35 × 10^{−3}, by = 8.38 × 10^{−3}.\)
Determine which interaction equation to use:

\(\frac{P_{u}}{\phi _{c}P_{n}}=pP_{u}=(2.23 ×10^{-3})(150)=0.335 > 0.2\) ∴ Equation 6.9 controls.

As a rough check (remember that \(B_{1}\) has not yet been computed and \(C_{b}\) has not been accounted for),

\(pP_{u} + b_{x}M_{ux} + b_{y}M_{uy} = (2.23 × 10^{−3})(150) + (4.35 × 10^{−3})(75) + (8.38 × 10^{−3}) (30)\)

\(= 0.912 < 1.0\) (OK)

It is likely that \(B_{1x}\) and \(B_{1y}\) will be equal to 1.0. In addition, the inclusion of \(C_{b}\) will reduce the value of \(b_{x}\) and the result of the interaction equation, so this shape is probably satisfactory.

Calculate \(B_{1}\) for each axis:

\(C_{m} =0.6-0.4\left(\frac{M_{1} }{M_{2}} \right) =0.6-0.4\left(\frac{0}{M_{2}} \right) =0.6 \) (forboth axes)

From Section 6.5, for each axis,

\(P_{e1}=\frac{\pi ^{2}EI^{*} }{(K_{1} L)^{2} }\)

and the modified stiffness is

\(EI^{*} =0.8\tau _{b} EI=0.8(1.0)EI=0.8EI\)

Therefore,

\(P_{e1x}=\frac{\pi ^{2} (0.8EI_{x}) }{(K_{x} L)^{2} } =\frac{\pi ^{2} (0.8)(29,000)(272)}{(15\times 12)^{2} } =1922\) kips

 

\(B_{1x}=\frac{C_{mx}}{1-\frac{P_{u} }{P_{e1x}} } =\frac{0.6}{1-\frac{150}{1922} } =0.651 \lt 1.0\) ∴ \(B_{1x} = 1.0\) as assumed.

 

\(P_{e1y}=\frac{\pi ^{2} (0.8EI_{y}) }{(K_{y} L)^{2} } =\frac{\pi ^{2} (0.8)(29,000)(93.4)}{(15\times 12)^{2} } =660\) kips

 

\(B_{1y}=\frac{C_{my}}{1-\frac{P_{u} }{P_{e1y}} } =\frac{0.6}{1-\frac{150}{660} } =0.776 \lt 1.0\) ∴ \(B_{1y} = 1.0\) as assumed.

From Figure 5.15g, \(C_{b} = 1.67\). Modify \(b_{x}\) to account for \(C_{b}.\)

\(\frac{b_{x}}{C_{b}}=\frac{4.35 ×10^{-3}}{1.67}=2.60 ×10^{-3}\)

For \(L_{b} = 0, b_{x} = 3.92 × 10^{–3}.\)

Since \(3.92 × 10^{–3} > 2.60 × 10^{–3},\) use \(b_{x} = 3.92 ×10^{–3}.\)
Check Equation 6.9: \(p = 2.22 × 10^{–3}, b_{x} = 3.92 × 10^{–3}, by = 8.38 ×10^{–3}\)

\(pP_{u} + b_{x}M_{ux} + b_{y}M_{uy} = (2.22 ×10^{–3})(150) + (3.92 × 10^{–3})(75) + (8.38 × 10^{–3})(30)\)

\(= 0.878 < 1.0\) (OK)

Try the next smaller shape. Try a W10 × 45, with \(p = 3.00 × 10^{–3}, b_{x} = 5.07 ×10^{–3}, b_{y} = 11.7 × 10^{–3}.\)

 

\(P_{e1x}=\frac{\pi ^{2} EI^{*}_{x} }{(K_{x} L)^{2} }=\frac{\pi ^{2} (0.8EI_{x} )}{(K_{x} L)^{2} } =\frac{\pi ^{2} (0.8)(29,000)(248)}{(15\times 12)^{2} } =1753\) kips

 

\(B_{1x}=\frac{C_{mx}}{1-\frac{P_{u} }{P_{e1x}} } =\frac{0.6}{1-\frac{150}{1753} } =0.656 \lt 1.0\) ∴ \(B_{1x} = 1.0\) as assumed.

 

\(P_{e1y}=\frac{\pi ^{2}( 0.8EI_{y}) }{(K_{y} L)^{2} } =\frac{\pi ^{2} (0.8)(29,000)(53.4)}{(15\times 12)^{2} } =377\) kips

 

\(B_{1y}=\frac{C_{my}}{1-\frac{P_{u} }{P_{e1y}} } =\frac{0.6}{1-\frac{150}{377} } =0.996\lt 1.0\) ∴ \(B_{1y} = 1.0\) as assumed.

Modify \(b_{x}\) to account for \(C_{b}.\)

\(\frac{b_{x}}{C_{b}}=\frac{5.07×10^{-3}}{1.67}=3.04×10^{-3}\)

For \(L_{b} = 0, b_{x} = 4.32× 10^{–3}.\)

Since \(4.32× 10^{–3} > 3.04× 10^{–3}\), use \(b_{x} = 4.32×10^{–3}.\)
Check Equation 6.9: with \( p = 3.00× 10^{–3}, b_{x} = 4.32× 10^{–3}, by = 11.7×10^{–3}\)

\(pP_{u} + b_{x}M_{ux} + b_{y}M_{uy} = (3.00×10^{–3})(150) + (4.32× 10^{–3})(75) + (11.7× 10^{–3})(30)\)

\(= 1.13 > 1.0\) (N.G.)

Use a W10 × 49.

The required axial load strength is

\(P_{a} = P_{D} + P_{L} = 25 + 75 = 100\) kips

The moments at each end are

\(M_{ntx} = M_{D} + M_{L} = 12.5 + 37.5 = 50.0\) ft-kips
\(M_{nty} = M_{D} + M_{L} = 5 + 15 = 20.0\) ft-kips

If we assume that \(B_{1} = 1.0\), The required bending moment strengths are

\(M_{ax} = B_{1}M_{ntx} = 1.0(50) = 50\) ft-kips
\(M_{ay} = B_{1}M_{nty} = 1.0(20) = 20\) ft-kips

The effective length for compression and the unbraced length for bending are the same:

\(KL = L_{b} = 15 ft\)

From Table 6-1, try a W10 × 49, with \(p = 3.35 × 10^{−3}, b_{x} = 6.54 × 10^{−3}, b_{y} = 12.6 × 10^{−6}.\)

Determine which interaction equation to use:

\(\frac{P_{a}}{P_{n}/Ω_{c}}=pP_{a}=(3.35×10^{-3})(100)=0.335 > 0.2\) ∴ Equation 6.9

As a rough check (remember that \(B_{1}\) has not yet been computed and \(C_{b}\) has not been accounted for),

\(pP_{a} + b_{x}M_{ax} + b_{y}M_{ay} = (3.35 × 10^{−3})(100) + (6.54 × 10^{−3})(50) + (12.6 × 10^{−3})(20)\)

\(= 0.914 < 1.0\) (OK)

It is likely that \(B_{1x}\) and \(B_{1y}\) will be equal to 1.0. In addition, the inclusion of \(C_{b}\) will reduce the value of \(b_{x}\) and the result of the interaction equation, so this shape is probably satisfactory.

Calculate \(B_{1}\) for each axis:

\(C_{m} =0.6-0.4\left(\frac{M_{1} }{M_{2}} \right) =0.6-0.4\left(\frac{0}{M_{2}} \right) =0.6\) (for both axes)

From Section 6.5, for each axis,

\(P_{e1}=\frac{\pi ^{2}EI^{*} }{(K_{1} L)^{2} }\)

and the modified stiffness is

\(EI^{*} =0.8\tau _{b} EI=0.8(1.0)EI=0.8EI\)

Therefore,

\(P_{e1x}=\frac{\pi ^{2} (0.8EI_{x}) }{(K_{x} L)^{2} } =\frac{\pi ^{2} (0.8)(29,000)(272)}{(15\times 12)^{2} } =1922\) kips

 

\(B_{1x}=\frac{C_{mx}}{1-\frac{1.60 P_{a} }{P_{e1x}} } =\frac{0.6}{1-\frac{1.60(100)}{1922} } =0.654 \lt 1.0 \) ∴ \(B_{1x} = 1.0\) as assumed.

 

\(P_{e1y}=\frac{\pi ^{2} (0.8EI_{y}) }{(K_{y} L)^{2} } =\frac{\pi ^{2} (0.8)(29,000)(93.4)}{(15\times 12)^{2} } =660\) kips

 

\(B_{1y}=\frac{C_{my}}{1-\frac{1.60P_{a} }{P_{e1y}} } =\frac{0.6}{1-\frac{1.60(100)}{660} } =0.792\lt 1.0\) ∴ \(B_{1y} = 1.0\) as assumed.

From Figure 5.15g, \(C_{b} = 1.67\). Modify \(b_{x}\) to account for \(C_{b}.\)

\(\frac{b_{x}}{C_{b}}=\frac{6.54×10^{-3}}{1.67}=3.92×10^{-3}\)

For \(L_{b} = 0, b_{x} = 5.90× 10^{–3}.\)

Since \(5.90× 10^{–3} > 3.92× 10^{–3}\), use \(b_{x} = 5.90×10^{–3}.\)

Check Equation 6.9 with \(p = 3.35 × 10^{−3}, b_{x} = 5.90 × 10^{−3}, b_{y} = 12.6 × 10^{−3}.\)

\(pP_{a} + b_{x}M_{ax} + b_{y}M_{ay} = (3.35 × 10^{−3})(100) + (5.90 × 10^{−3})(50) + (12.6 × 10^{−3})(20)\)

\(= 0.882 < 1.0\) (OK)

Try the next smaller shape. Try W10 × 45, with \(p = 4.51 × 10^{−3}, b_{x} = 7.63 × 10^{−3}, b_{y} = 17.6 ×10^{−3}.\)

\(P_{e1x}=\frac{\pi ^{2} EI^{*}_{x} }{(K_{x} L)^{2} } =\frac{\pi ^{2}(0.8 EI_{x}) }{(K_{x} L)^{2} } =\frac{\pi ^{2} (0.8)(29,000)(248)}{(15\times 12)^{2} } =1753\) kips

 

\(B_{1x}=\frac{C_{mx}}{1-\frac{1.60P_{a} }{P_{e1x}} } =\frac{0.6}{1-\frac{1.60(100)}{1753} } =0.660 \lt 1.0 \) \(∴ B_{1x} = 1.0\) as assumed.

 

\(P_{e1y}=\frac{\pi ^{2} (0.8EI_{y}) }{(K_{y} L)^{2} } =\frac{\pi ^{2} (0.8)(29,000)(53.4)}{(15\times 12)^{2} } =377.4\) kips

 

\(B_{1y}=\frac{C_{my}}{1-\frac{1.60P_{a} }{P_{e1y}} } =\frac{0.6}{1-\frac{1.60(100)}{377.4} } =1.042\)

 

\(M_{ay} = B_{1y}M_{nty} = 1.042(20) =20.8\) ft-kips

 

\(\frac{b_{x}}{C_{b}}=\frac{7.63×10^{-3}}{1.67}=4.57×10^{-3}\)

 

For \(L_{b} = 0, b_{x} = 6.49× 10^{–3}.\)

Since \(6.49× 10^{–3} > 4.57× 10^{–3}\), use \(b_{x} = 6.49×10^{–3}.\)

Check Equation 6.9 with \(p = 4.51× 10^{−3}, b_{x} = 6.49× 10^{−3}, b_{y} = 17.6× 10^{−3}.\)

\(pP_{a} + b_{x}M_{ax} + b_{y}M_{ay} = (4.51× 10^{−3})(100) + (6.49× 10^{−3})(50) + (17.6 × 10^{−3})(20.8)\)

\(= 1.14 > 1.0 \) (N.G.)

Use a W10 × 49.
