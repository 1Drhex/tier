Guest Question.

Determine the number of bolts required to connect, 2L3×3×1/4 tension member to a plate under a beam as shown. The angles have standard holes and the plate has oversized holes per Manual Table J3.3.
\(\begin{array}{l}R_{D}=15 kips \\R_{L}=45 kips\end{array}\)
Use 3/4-in. diameter ASTM A325-SC class A bolts.
Assume that the strength of the beam, angles, and plate have been checked.

Material Properties:
\(\begin{array}{llll} L 3 \times 3 \times 1 / 4 & \text { ASTM A36 } & F_{y}=36 ksi & F_{u}=58 ksi \\\text { Plate Material } & \text { ASTM A36 } & F_{y}=36 ksi & F_{u}=58 ksi\end{array}\)

Manual
Table 2-3
and 2-4

Solution
LRFD ASD
\(R_{u}=(1.2)(15 kips )+(1.6)(45 kips )=90 kips\)

 

Design of bolts

 

Because of oversized holes the slip is a serviceability limit state, Class A faying surface, 3/4 in. diameter ASTM A325-SC bolts in double shear

 

\(\phi r_{n}=18.8 \text { kips } / \text { bolt }\)

 

\( \begin{aligned}n &=\frac{\phi R_{n}}{\phi r_{n}}=\frac{90 kips }{18.8 kips / bolt } \\ &=4.78 \rightarrow 5 bolts\end{aligned}\)

 

Try (5) 3/4-in. dia. ASTM A325-SC bolts

 

Check bolt shear strength

 

\(\phi r_{n}=\phi F_{v} A_{b}=31.8 \text { kips } / \text { bolt }\)

 

 

\(\phi R_{n}=\phi r_{n} n=(31.8 \text { kips } / \text { bolt })(5 \text { bolts })\)

 

= 159 kips > 90 kips o.k.

 

Try, PL 3/8 in.

 

Check bolt bearing strength on plate

 

\(\phi r_{n}=\phi\left(1.2 L_{c} t F_{u}\right) \leq \phi\left(2.4 d t F_{u}\right)\)

 

 

\(\phi=0.75\)

 

Oversized holes, bolt spacing = 3 in.

 

\(\begin{aligned}\phi r_{n} &=(78.3 kips / in .)(3 / 8 in .) \\&=29.4 kips / bolt\end{aligned}\)

 

Oversized holes, edge distance = 1 1/4 in.

 

\(\begin{aligned}\phi r_{n} &=(40.8 \text { kips } / \text { in. })(3 / 8 \text { in. }) \\&=15.3 \text { kips } / \text { bolt }\end{aligned}\)

 

15.3 kips/bolt < 29.4 kips/bolt

 

\(\begin{aligned}\phi R_{n} &=\phi r_{n} n \\&=(15.3 \text { kips } / \text { bolt })(5 \text { bolts }) \\&=76.5 \text { kips }<80 \text { kips n.g.}\end{aligned}\)

 

Try 6 bolts

 

\(\begin{aligned}\phi R_{n} &=\phi r_{n} n \\&=(15.3 \text { kips } / \text { bolt })(6 \text { bolts }) \\&=91.8 \text { kips }>80 \text { kips o.k. }\end{aligned}\)

 

Use 6 bolts

 \(R_{a}=15 \text { kips }+45 \text { kips }=60 \text { kips } \)

Because of oversized holes the slip is a serviceability limit state, Class A faying surface, 3/4 in. diameter ASTM A325-SC bolts in double shear
