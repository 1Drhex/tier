Guest Question.

Joule–Thomson Coefficient from the Virial Equation

Develop an expression for the Joule–Thomson coeffi cient using the pressure-based expansion of the virial equation truncated at the second virial coeffi cient. Use the corresponding state relationships presented for the temperature dependence of B to develop a generalized correlation for \(\mu_{ JT }\).

 

 

Admin
Solution

To use Equation (5.57), we need to write the virial equation in a form that is explicit in volume. This can be done by rearranging Equation (4.27):

 

\(\mu_{ JT }=\left(\frac{\partial T}{\partial P}\right)_{h}=\frac{\left[T\left(\frac{\partial v}{\partial T}\right)_{P}-v\right]}{c_{p}}=\frac{\left[T\left(\frac{\partial v}{\partial T}\right)_{P}-v\right]}{c_{P}^{\text {ideal gas }}-\int_{P_{\text {ideal gas }}}^{P_{\text {real }}}\left[T\left(\frac{\partial^{2} v}{\partial T^{2}}\right)_{P}\right] d P}\) (5.57)

 

\(z=\frac{P v}{R T}=1+B^{\prime} P+C^{\prime} P^{2}+D^{\prime} P^{3}+\ldots\) (4.27)

 

\(v=\frac{R T}{P}\left(1+B^{\prime} P\right)=\frac{R T}{P}+B^{\prime} R T\) (E5.9A)

 

We then take the first and second derivatives:

 

\(\left(\frac{\partial v}{\partial T}\right)_{P}=\frac{R}{P}+R B^{\prime}+R T\left(\frac{ d B^{\prime}}{ d T}\right)\) (E5.9B)

 

and, \(\left(\frac{\partial^{2} v}{\partial T^{2}}\right)_{P}=R\left(\frac{ d B^{\prime}}{ d T}\right)+R T\left(\frac{ d ^{2} B^{\prime}}{ d T^{2}}\right)\) (E5.9C)

 

Using Equations (E5.9A), (E5.9B), and (E5.9C) in Equation (5.57) gives:

 

\(\mu_{ JT }=\left(\frac{\partial T}{\partial P}\right)_{h}=\frac{\left[T\left(\frac{\partial v}{\partial T}\right)_{P}-v\right]}{c_{p}}=\frac{\left[T\left(\frac{\partial v}{\partial T}\right)_{P}-v\right]}{c_{P}^{\text {ideal gas }}-\int_{P_{\text {ideal gas }}}^{P_{\text {real }}}\left[T\left(\frac{\partial^{2} v}{\partial T^{2}}\right)_{P}\right] d P}\)

 

\(=\frac{R T^{2}\left(\frac{ d B^{\prime}}{ d T}\right)}{c_{P}^{\text {ideal gas }}-\int_{P_{\text {ideal gs }}}^{P_{\text {real }}}\left\{T\left[R\left(\frac{ d B^{\prime}}{ d T}\right)+R T\left(\frac{ d ^{2} B^{\prime}}{ d T^{2}}\right)\right]\right\} d P}\) (E5.9D)

 

To evaluate this expression for \(\mu_{ JT }\), we can use the corresponding state relations. Using Equation (4.28) and the equations that follow Equation (4.29), we get:

 

\(C^{\prime}=\frac{C-B^{2}}{(R T)^{2}} \text { and so on }\) (4.28)

 

\(B=2 \pi N_{ A } \int_{0}^{\infty}\left(1-e^{-\Gamma(r) /(k T)}\right) r^{2} d r\) (4.29)

 

\(B^{\prime}=\frac{B}{R T}=\frac{B_{r} T_{c}}{P_{c} T}=\frac{B_{r}}{P_{c} T_{r}}=\frac{B^{(0)}+\omega B^{(1)}}{P_{c} T_{r}}\) (E5.9E)

 

Substituting the generalized relations on page 241 into Equation (E5.9E) gives:

 

\(B^{\prime}=\frac{1}{P_{c}}\left[\left(\frac{0.083}{T_{r}}-\frac{0.422}{T_{r}^{2.6}}\right)+\omega\left(\frac{0.139}{T_{r}}-\frac{0.172}{T_{r}^{5.2}}\right)\right]\) (E5.9F)

 

We then take the first and second derivatives:

 

\(\left(\frac{ d B^{\prime}}{ d T}\right)=\frac{1}{P_{c}}\left[\left(-\frac{0.083}{T T_{r}}+\frac{1.097}{T T_{r}^{2.6}}\right)+\omega\left(-\frac{0.139}{T T_{r}}+\frac{0.894}{T T_{r}^{5.2}}\right)\right]\) (E5.9G)

 

and, \(\left(\frac{ d ^{2} B^{\prime}}{ d T^{2}}\right)=\frac{1}{P_{c}}\left[\left(\frac{0.166}{T^{2} T_{r}}-\frac{3.950}{T^{2} T_{r}^{2.6}}\right)+\omega\left(\frac{0.278}{T^{2} T_{r}}-\frac{5.545}{T^{2} T_{r}^{5.2}}\right)\right]\) (E5.9H)

 

Substituting Equations (E5.9G) and (E5.9H) into (E5.9D) and simplifying gives:

 

\(\mu_{ JT }=\frac{-\frac{R T}{P_{c}}\left[\left(-\frac{0.083}{T_{r}}+\frac{1.097}{T_{r}^{2.6}}\right)+\omega\left(-\frac{0.139}{T_{r}}+\frac{0.994}{T_{r}^{5.2}}\right)\right]}{c_{P}^{\text {ideal gas }}-\int_{0}^{P}\left[\frac{R}{P_{c}}\left[\left(\frac{0.083}{T_{r}}-\frac{2.853}{T_{r}^{2.6}}\right)+\omega\left(\frac{0.139}{T_{r}}-\frac{4.651}{T_{r}^{5.2}}\right)\right]\right] d P}\) (E5.9I)

 

where we have set \(P_{\text {ideal }}\) to zero. Finally, integrating and substituting in \(T=T_{r} T_{c}\), we get:

 

\(\mu_{ JT }=\frac{-\frac{T_{c}}{P_{c}}\left[\left(-0.083+\frac{1.097}{T_{r}^{1.6}}\right)+\omega\left(-0.139+\frac{0.994}{T_{r}^{4.2}}\right)\right]}{\frac{c_{P}^{\text {ideal gas }}}{R}-P_{r}\left[\left(\frac{0.083}{T_{r}}-\frac{2.853}{T_{r}^{2.6}}\right)+\omega\left(\frac{0.139}{T_{r}}-\frac{4.651}{T_{r}^{5.2}}\right)\right]}\) (E5.9J)

 

Equation (E5.9J) presents a generalized relation for \(\mu_{ JT }\). If the critical properties and acentric factor of a species are known, we can use Equation (E5.9J) to calculate the Joule–Thomson coefficient at a specifi ed state at temperature T and pressure P.
