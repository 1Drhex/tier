Guest Question.

The friction hook is made from a fixed frame which is shown colored and a cylinder of negligible weight. A piece of paper is placed between the smooth wall and the cylinder. Determine the smallest coefficient of static friction \(\mu\) at all points of contact so that any weight \(W\) of paper \(p\) can be held.

Given:

\(\theta\) = \(20\) \(deg\)

Admin
Solution

Paper

\(+\uparrow \sum { { F }_{ y } } =0;\)

 

\(\quad F+F-W=0\quad \quad \quad F=\frac { W }{ 2 }\)

 

\(\quad F=\mu N\quad\quad \quad \quad\quad \quad \quad N=\frac { W }{ 2\mu }\)

 

Cylinder:

\(\quad { F }^{ \backprime }r-\left( \frac { W }{ 2 } \right) r=0 \quad\quad \quad\quad \quad \quad\quad \quad \quad { F }^{ \backprime }=\frac { W }{ 2 } \)

 

\( \quad { N }^{ \backprime }-(\frac { W }{ 2 } )\sin { \left( \theta \right) } -\frac { W }{ 2\mu } \cos { \left( \theta \right) } =0\quad \quad { N }^{ \backprime }=\frac { W }{ 2 } \left[ \sin { \left( \theta \right) } +\frac { 1 }{ \mu } \cos { \left( \theta \right) } \right]\)

 

\(\quad { F }^{ \backprime }=\mu { N }^{ \backprime }\quad \quad \frac { W }{ 2 } =\mu \left( \frac { W }{ 2 } \right) \left[ \sin { \left( \theta \right) } +\frac { 1 }{ \mu } \cos { \left( \theta \right) } \right]\)

 

\(\quad \quad \quad\quad \quad \quad \quad 1=\mu \sin { \left( \theta \right) } +\cos { \left( \theta \right) }\)

 

\(\quad \quad \quad\quad \quad \quad \quad \mu =\frac { 1-\cos { \left( \theta \right) } }{ \sin { \left( \theta \right) } } \)

 

\(\quad \quad \quad\quad \quad \quad \quad \mu = 0.176 \)
