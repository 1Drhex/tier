Guest Question.

For the “half-harmonic oscillator” (Example 9.3), make a plot comparing the normalized WKB wave function for n = 3 to the exact solution. You’ll have to experiment to determine how wide to make the patching region. Note: You can do the integrals of p(x) by hand, but feel free to do them numerically. You’ll need to do the integral of \( \left|\psi_{W K B}\right|^{2} \) numerically to normalize the wave function.

Admin
Solution

In the plot below the blue and red curves make up the piecewise WKB solution, and the black dashed line is the exact result. I chose the width of the patching region to be \( 0.3 x_{2} \text {, centered at } x_{2} \).

\( \text { \$Assumptions }=m>0 \text { \&\& } E n>0 \text { \&\& } \omega>0 \);

Analytical Integral

\( p\left[x_{-}\right]:=\operatorname{Sqrt}[2 m(E n-V[x])] \);

\( V\left[x_{-}\right]:=\frac{1}{2} m \omega^{2} x^{2} \);

\( x 2=\frac{\sqrt{2} \sqrt{E n}}{\sqrt{m} \omega} \);

Numerical Method

n = 3;

\( En =\left(2 n -\frac{ 1 }{2}\right) \hbar \omega \);

\( \hbar= m =\omega=1 \);

\( \psi 1\left[ x _{-}\right]:=\frac{2}{\sqrt{ p [ x ]}} \operatorname{Sin}\left[\frac{1}{\hbar} NIn t e g r a t e[p[ xp ],\{ xp , x , x 2\}]+\frac{\pi}{4}\right] \).

\( \psi 3\left[x_{-}\right]:=\frac{1}{\sqrt{A b s[p[x]]}} \operatorname{Exp}\left[-\frac{1}{n} \text { NIntegrate }[A b s[p[x p]],\{x p, x 2, x\}]\right] \).

\( \alpha=\left(\frac{2 m }{\hbar^{2}} V ^{\prime}[ x 2]\right)^{1 / 3} \);

\( \psi 2\left[ x _{-}\right]:=\sqrt{\frac{4 \pi}{\alpha \hbar}} \text { AiryAi }[\alpha( x – x 2)] \).

\( \left.\psi WKB \left[ x _{-}\right]:=\operatorname{If}[ x <.85 \times 2, \psi][ x ], \quad \operatorname{If}[ x <1.15 \times 2, \psi 2[ x ], \psi 3[ X ]]\right] \);

\( \text { norm }=\frac{1}{\sqrt{\text { NIntegrate }\left[\psi WKB [ x ]^{\wedge} 2,\{ x , 0,2 \times 2\}\right]}} \);

\( \psi \operatorname{exact}\left[n_{-}, x_{-}\right]:=\sqrt{2}\left(\frac{m \omega}{\pi \hbar}\right)^{1 / 4} \frac{1}{\sqrt{2^{n} n !}} \text { HermiteH }\left[n, \sqrt{\frac{m \omega}{\hbar}} x\right] \operatorname{Exp}\left[-\frac{1}{2}\left(\sqrt{\frac{m \omega}{\hbar}} x\right)^{2}\right] \).
