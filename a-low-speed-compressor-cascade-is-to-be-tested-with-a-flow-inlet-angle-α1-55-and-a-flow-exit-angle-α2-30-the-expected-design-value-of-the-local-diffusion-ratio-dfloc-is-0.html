Guest Question.

A low-speed compressor cascade is to be tested with a flow inlet angle, \(\alpha _{1}=55° \) , and a flow exit angle, \(\alpha _{2}=30° \). The expected design value of the local diffusion ratio, \(DF_{loc} \), is 0.4. Find a safe value for the pitch–chord ratio if the maximum value of the diffusion factor, DF, is assumed to be 0.6. Using Eqs (3.26), (3.39), and (3.40), determine values for \(\zeta , C_{D} \) , and \(C_{L} \).

\(C_{L}=2\frac{s}{l} \cos \alpha _{m} \left(\tan \alpha _{1} -\tan \alpha _{2} \right) -C_{D}\tan \alpha _{m} \) (3.26a)

\(\frac{L}{D}=\frac{C_{L} }{C_{D}} =\frac{2}{\zeta } \left(\tan \alpha _{1} -\tan \alpha _{2} \right) \) (3.26b)

\(C_{D} =\zeta \left(s/1\right) \cos \alpha _{m} =2\left(\theta _{2} /l\right) \cos ^{2} \alpha _{m}/\cos ^{2} \alpha _{2} \) (3.39)

\(\frac{\theta _{2} }{l} =0.004/\left[1+1.17 \ln \left(1-DF_{loc} \right) \right] \) (3.40)

Admin
Solution

Using Eq. (3.33) with DF = 0.6, the maximum allowable pitch–chord ratio is

 

\( c_{2} / c_{1}\geq 0.72 \) (3.33)

 

\(\frac{s}{l} \leq \frac{2 \cos \alpha _{1} /\cos \alpha _{2}-0.8 }{\cos \alpha _{1} \left(\tan \alpha _{1} -\tan \alpha _{2} \right) } =\frac{2\times 0.5736/0.866-0.8}{0.5736\times \left(1.4281-0.5774\right) } =\underline{1.075} \)

 

From Eq. (3.40), \(\theta _{2}/l =0.004/\left[1+1.17 \ln \left(1-0.4 \right) \right] =0.01. \)

From Eq. (3.39), \( C_{D}=2\left(\theta _{2}/l \right) \cos ^{2} \alpha _{m}/\cos ^{2} \alpha _{2} \), where \(\tan \alpha _{m}=\left(1/2\right) \left(\tan \alpha _{1} +\tan \alpha _{2} \right) =1.00275 \)

Therefore, \( \alpha _{m}=45.08° \), and

\( C_{D}=2\times 0.01\times \cos ^{2} 45.08/\cos ^{2} 30 =\underline{0.013} \)

From Eq. (3.25),the loss coefficient,

 

\( C_{D}=\frac{s\Delta p_{0} \cos \alpha _{m} }{1/2\rho c^{2}_{m} l} =\frac{s \zeta 1/2\rho c^{2}_{m} \cos \alpha _{m} }{1/2\rho c^{2}_{m} l} =\frac{s}{l}\zeta \cos \alpha _{m} \) (3.25)

 

\( \zeta = C_{D}/\frac{s}{l} \cos \alpha _{m} =0.013/\left(1.075\times \cos 45\right) =\underline{0.017} \)

From Eq. (3.26a), \( C_{L}=2 s/l \cos \alpha _{m} \left(\tan \alpha _{1} -\tan \alpha _{2} \right) -C_{D}\tan \alpha _{m} \) ; therefore,

\( C_{L}=2 \times 1.075 \times \cos 45.08 \times \left(\tan 55 -\tan30 \right) -0.013\times \tan45.08 =\underline{1.28} \)
