Designing a second-order high-pass Butterworth filter for |H(j∞)| = 1

(a) Design a second-order Butterworth high-pass filter as in Fig. 12.25(c) to yield |H(j∞)| = 1 (or 0 dB), a cutoff frequency of \( f_{ o }=1 kHz \text {, and } Q=0.707 \).

(b) Use PSpice/SPICE to plot the frequency response of the output voltage of the filter designed in part (a) from 10 Hz to 100 kHz.

Admin
Solution

(a) For Q = 0.707, Example 12.6 gives C = 0.01 µF and R = 15,916Ω. From Eq. (12.52),

\(K=3-\frac{1}{Q}\) (12.52)

 

\( K=\frac{3-1}{Q}=\frac{3-1}{0.707}=1.586 \)

 

and \( R_{ F }=(K-1) R_{1}=(1.586-1) \times 15,916=9327 \Omega \)

From Eq. (12.79),

\( C_{ a }=\frac{C_{2}}{K} \quad[\text { for }|H(j \infty)|=1(\text { or } 0 dB )] \) (12.79)

 

\( C_{ a }=\frac{C}{K}=\frac{0.01 \mu F }{1.586}=6.305 nF \)

 

From Eq. (12.80),

\( C_{ b }=C_{2} \frac{K-1}{K} \quad\left[\text { for }\left|H\left(j^{\infty}\right)\right|=1(\text { or } 0 dB )\right] \) (12.80)

 

\( C_{ b }=\frac{C(K-1)}{K}=\frac{0.01 \mu F \times(1.586-1)}{1.586}=3.695 nF \)

 

NOTE: The exact numbers of the resistances and capacitances are used to demonstrate the validity of the calculated values with the PSpice simulation results. In a practical design, the commercially available standard values will be used (\( \text { i.e., } 9.3 k \Omega \text { instead of } 9327 \Omega \text { and } C_{ b }=4 nF \) instead of 3.695 nF; see Appendix E).

(b) For PSpice simulation, the circuit of Fig. 12.18 can be transformed into the circuit of Fig. 12.26 by removing \( R_{4} \text { and } R_{5} \), interchanging the locations of R and C, replacing \( C_{2} \text { by } C_{ a }, \text { and adding } C_{ b } \) between nodes a and b. The PSpice statements for R and C read as follows:

\(\begin{array}{llllllll}\text { CA } & 1 & 2 & 6.305 N F & \text {; For CA connected between nodes } & 1 \text { and } 2 \\\text { CB } & 2 & 0 & 3.695 N F & \text {; For CB connected between nodes } & 2 \text { and } 0 \\\text { R2 } & 2 & 6 & 15916 & \text {; For R2 connected between nodes } & 2 \text { and } 6 \\\text { C3 } & 2 & 3 & 0.01 U F & \text {; For C3 connected between nodes } & 2 \text { and } 3 \\\text { R3 } & 3 & 0 & 15916 & \text {; For R3 connected between nodes }& 3 \text { and } 0\end{array}\)

The PSpice plot of the gain is shown in Fig. 12.27, which gives \( \left|H\left(j {\infty}\right)\right|=1.0 \text { at } \omega=\infty \).
