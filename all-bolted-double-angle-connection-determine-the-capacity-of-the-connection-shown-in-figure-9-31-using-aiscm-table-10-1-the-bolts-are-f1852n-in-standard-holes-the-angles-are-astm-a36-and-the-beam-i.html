All-Bolted Double-Angle Connection

Determine the capacity of the connection shown in Figure 9-31 using AISCM, Table 10-1. The bolts are F1852N in standard holes. The angles are ASTM A36 and the beam is ASTM A992, grade 50.

Solution

From AISCM, Table 10-1, with N = 4 for 3⁄4-in.-diameter bolts and an angle thickness of 3⁄8 in., the design strength is 127 kips. The capacity of the bolts bearing on the beam, and the block shear on the beam also must be checked.
With reference to Figure 9-32, the \(L_{eh} \) term is reduced by 1⁄4 in. for fabrication tolerances. The capacity of the bolts bearing on the beam web is

\(L_{c1}= 1.75 in. – 0.5\left(\frac{3}{4}+\frac{1}{8} \right)= 1.31\) in.

\(L_{c2}=L_{c3}=L_{c4}= 3 in. – \left(\frac{3}{4}+\frac{1}{8} \right)= 2.12\) in.

For bolt 1,

\(\phi R_{n}=\phi 1.2L_{c}tF_{u}\leq \phi 2.4dtF_{u}\)(AISC eq., J3.6a)

=(0.75)(1.2)(1.31 in.)(0.3 in.)(65 ksi)

<(0.75)(2.4)(0.75 in.)(0.3 in.)(65 ksi)

=22.9 kips < 26.3 kips

=22.9 kips.

For bolts 2, 3, and 4,

\(\phi R_{n}\)=(0.75)(1.2)(2.12 in.)(0.3 in.)(65 ksi)

< (0.75)(2.4)(0.75 in.)(0.3 in.)(65 ksi)

=37.2 kips > 26.3 kips

= 26.3 kips.

The total strength, considering all of the bolts in bearing, is

\(\phi R_{n}\)= 22.9 kips + (3)(26.3 kips) = 101 kips.

Note that since the web thickness for the W24 × 55 is 0.395 in., the bolt bearing on the W18 × 35 will control.

Block shear in the W18 × 35:

\(A_{gv}= [2 in. + (3)(3 in.)]0.3 in. = 3.3 in.^{2}\)

 

\(A_{nv}=\left[2 in. + (3)(3 in.)- 3.5\left(\frac{3}{4}+\frac{1}{8} \right) \right]0.3 in. =2.38 in.^{2}\)

 

\(A_{gv} = (1.75 in. – 0.25 in.)(0.3 in.) = 0.45 in.^{2}\)

 

\(A_{nt}=\left[(1.75 in. – 0.25 in.) – 0.5\left(\frac{3}{4}+\frac{1}{8} \right) \right]0.3 in. =0.318 in.^{2}\)

Using AISCM, equation J4-5, to determine the block shear strength,

\(\phi R_{n} =\phi (0.6 F_{u}A_{nv}+U_{bs}F_{u}A_{nt})\leq \phi (0.6 F_{y}A_{gv}+U_{bs}F_{u}A_{nt})\)

= 0.75[(0.6)(65)(2.38) + (1.0)(65)(0.318)]

≤ 0.75[(0.6)(50)(3.3) + (1.0)(65)(0.31)]

= 85.2 kips ≤ 89.3 kips

= 85.2 kips < 101 k (block shear controls over bolt bearing).

Alternatively, the bottom of Table 10-1 could be used to determine the bolt bearing and block shear strength of the beam web. Using \(L_{eh}=1.75\) in. and \(L_{ev}=2\) in., the design capacity for a beam web coped at the top flange is 284 kips/in. Since the web thickness of the W18 × 35 is \(t_{w} = 0.3\) in., the capacity is

(0.3 in.)(284 kips/in.) = 85.2 kips, which agrees with the value calculated above.
