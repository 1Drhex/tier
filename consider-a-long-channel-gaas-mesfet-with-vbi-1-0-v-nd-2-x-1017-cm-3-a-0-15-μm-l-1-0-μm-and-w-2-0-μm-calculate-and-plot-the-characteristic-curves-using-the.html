Guest Question.

Consider a long-channel GaAs MESFET with \(V_{bi} = 1.0\) V, \(N_d = 2 × 10^{17} cm^{–3}\), a = 0.15 μm, L = 1.0 μm, and W = 2.0 μm. Calculate and plot the characteristic curves using the analytical model. Repeat using the Curtice model assuming \(α = 1.5 V^{–1}\) and \(λ = 0.01 V^{–1}\).

Admin
Solution

As calculated in , \(V_p = –2.1\) V. Assuming long-channel behavior and constant mobility (\(\mu_n = 3000 cm^2V^{–1}s^{–1}\)),

\(I_p=\frac{Wq^2N^2_da^3\mu_n}{6L \varepsilon _s}=\frac{(2\times10^{-4}cm)(1.6\times 10^{-19})^2(2\times10^{17}cm^{-3})^2(0.15\times10^{-4} cm)^3(3000 cm^2V^{-1}s^{-1})}{6(10^{-4}cm)(13.1)(8.85\times10^{-14}F/cm)} =3.0 mA\)

Using the analytical model, the device characteristics are given by

\(I_D=\begin{cases}0, &V_{GS}\leq V_P \\I_{p}\left[\frac{3V_{DS}}{V_{bi}-V_P}-\frac{2(V_{DS}+V_{bi}-V_{GS})^{3/2}-2(V_{bi}-V_{ GS} )^{3/2}}{(V_{bi}-V_P)^{3/2}} \right],&V_{DS}\leq V_{GS}-V_P; and, \\I_p\left[\frac{3(V_{GS}-V_P)}{V_{bi}-V_P}-\frac{2(V_{bi}-V_{p})^{3/2}-2(V_{bi}-V_{GS})^{3/2}}{(V_{bi}-V_P)^{3/2}} \right],& V_{DS}\geq V_{GS}-V_P\end{cases}\)

For the example MESFET,

\(I_D=\begin{cases}0, &V_{GS}\leq -2.1V \\3.0 mA \left[\frac{3V_{DS}}{3.1V}-\frac{2( V_{DS}+ 1.0V-V_{GS})^{3/2}-2(1.0V-V_{GS} )^{3/2}}{(3.1V )^{ 3/2}} \right],&V_{DS}\leq V_{GS}+2.1V; and, \\3.0 mA \left[\frac{3 (V_{GS}+2.1V)}{3.1V}-\frac{2 ( 3.1V )^{3/2}-2(1.0V-V_{GS})^{3/2}}{(3.1V)^{3/2}} \right],&V_{DS}\geq V_{GS}+2.1V\end{cases}\)

For application of the Curtice model, the device transconductance parameter is

\(\beta=\frac{\mu_n\varepsilon _s}{2a} \frac{W}{L}=\frac{(3000 cm^2V^{-1}s^{-1})(13.1)(8.85\times 10^{-14}F/cm)}{2(0.15\times 10^{-4} cm)}\frac{2\times 10^{-4}cm}{10^{-4}cm} =0.23mA /V^2 \)

Using the Curtice model, the characteristics are given by

\(I_D=\begin{cases}0; &V_{GS}\leq -2.1 V \\ 0.23 mA/V^2(V_{GS}+2.1 V)^2(1+0.01 V_{DS}) \tanh(1.5V_{DS}); & V_{GS}\geq -2.1V\end{cases}\)

Both sets of characteristics are plotted in Figure 12.9. The maximum departure between the analytical characteristics and the Curtice model is less than 20%.
