Guest Question.

For vertical flexural member BD, determine the principal stresses and the maximum shear stress acting at point H, as shown on Figures P15.29a and P15.29b. Show these stresses on an appropriate sketch.

Admin
Solution

Moment of inertia about the z axis:

\(\begin{array}{lcccccc}{\text { Shape }} & \begin{array}{c}\text { Width } b \\\text { (in.) }\end{array} & \begin{array}{c}\text { Height } h \\( in .)\end{array} & \begin{array}{c}I_{C} \\\left( in .^{4}\right)\end{array} & \begin{array}{c}d=y_{i}-\bar{y} \\(\text { in. })\end{array} & \begin{array}{c}d^{2} A \\\left( in .{ }^{4}\right)\end{array} & \begin{array}{c}I_{C}+d^{2} A \\\left( in .^{4}\right)\end{array} \\\hline \text { flange } & 10 & 0.75 & 0.3516 & 5.3750 & 216.6797 & 217.0313 \\\text { web } & 0.5 & 10 & 41.6667 & 0.0000 & 0.0000 & 41.6667 \\\text { flange } & 10 & 0.75 & 0.3516 & -5.3750 & 216.6797 & 217.0313 \\\hline \text { Moment of inertia about the } z \text { axis }\left( in. ^{4}\right)= 475.7292\end{array}\)

 

Equilibrium:

\(\begin{aligned}\qquad{\Sigma} F_{x} &=B_{x}-F_{D E} \cos \left(55.008^{\circ}\right)+(35 kips ) \cos \left(50^{\circ}\right)=0 \\\Sigma F_{y} &=B_{y}-F_{D E} \sin \left(55.008^{\circ}\right)-(35 kips ) \sin \left(50^{\circ}\right)=0 \\\Sigma M_{B} &=F_{D E}(100 \text { in. }) \cos \left(55.008^{\circ}\right) \\&+F_{D E}(12 \text { in. }) \sin \left(55.008^{\circ}\right) \\&-(35 kips )(74 \text { in. }) \cos \left(50^{\circ}\right) \\&-(35 kips )(76 \text { in. }) \sin \left(50^{\circ}\right)=0\end{aligned}\)

 

\(\begin{aligned}\therefore F_{D E} &=55.116 \text { kips } \\B_{x} &=9.109 kips \\B_{y} &=71.964 kips\end{aligned}\)

 

Internal forces at H:

\(\begin{array}{ll}\Sigma F_{x}=B_{x}-V_{H}=0 & \therefore V_{H}=9.109 kips \\\Sigma F_{y}=F_{H}+B_{y}=0 & \therefore F_{H}=-71.964 kips \\\Sigma M_{H}=M_{H}+B_{x}(20 in .)=0 & \therefore M_{H}=-182.182 kip \text {-in. }\end{array}\)

Axial stress at H:

\(\begin{aligned}&A=2(10 in .)(0.75 in .)+(0.5 in .)(10 in .)=20 in .^{2} \\&\sigma_{\text {axial }}=\frac{-71.964 kips }{20 in .^{2}}=3.598 ksi ( C )\end{aligned}\)

Shear stress at H:

\(\begin{aligned}&Q_{H}=(10 \text { in. })(0.75 in .)(5 in .+0.375 in .)+(0.5 in .)(2.25 in. )(2.75 in .+1.125 in .)=44.6719 in .^{3} \\&\tau_{H}=\frac{(9.109 kips )\left(44.6719 in.^{3}\right)}{\left(475.7292 in. ^{4}\right)(0.5 in .)}=1.711 ksi\end{aligned}\)

 

Bending stress at H:

\(\sigma_{\text {bend }}=\frac{(182.182 \text { kip-in. })(2.75 \text { in. })}{475.7292 \text { in. }^{4}}=1.053 ksi ( C ) \text { (by inspection) }\)

Summary of stresses at H:

\(\begin{aligned}\sigma_{x} &=0 ksi \\\sigma_{y} &=-3.598 ksi -1.053 ksi =-4.651 ksi \\\tau_{x y} &=1.711 ksi\end{aligned}\)

Principal stress calculations:
The principal stress magnitudes can be computed from Eq. (12.12):

\(\begin{aligned}\sigma_{p 1, p 2} &=\frac{\sigma_{x}+\sigma_{y}}{2} \pm \sqrt{\left(\frac{\sigma_{x}-\sigma_{y}}{2}\right)^{2}+\tau_{x y}^{2}}=\frac{(0)+(-4.651)}{2} \pm \sqrt{\left(\frac{(0)-(-4.651)}{2}\right)^{2}+(1.711)^{2}} \\&=-2.326 \pm 2.887\end{aligned}\)

 

\(\begin{aligned}\sigma_{p 1} &=0.561 ksi \text { and } \quad \sigma_{p 2}=-5.21 ksi \\\tau_{\max } &=2.89 ksi \quad \text { (maximum in-plane shear stress) } \\\sigma_{\text {avg }} &=2.33 ksi ( C ) \quad \text { (normal stress on planes of maximum in-plane shear stress) }\end{aligned}\)

 

\(\begin{aligned}\tan 2 \theta_{p}=& \frac{\tau_{x y}}{\left(\sigma_{x}-\sigma_{y}\right) / 2}=\frac{1.711}{[(0)-(-4.651)] / 2}=\frac{1.711}{2.326}=0.735588 \\&\left.\therefore \theta_{p}=18.17^{\circ} \quad \text { (counterclockwise from the } x \text { axis to the direction of } \sigma_{p 1}\right)\end{aligned}\)
