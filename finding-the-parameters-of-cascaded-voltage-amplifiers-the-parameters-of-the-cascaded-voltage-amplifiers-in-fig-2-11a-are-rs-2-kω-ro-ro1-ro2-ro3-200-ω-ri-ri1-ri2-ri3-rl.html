Guest Question.

Finding the parameters of cascaded voltage amplifiers The parameters of the cascaded voltage amplifiers in Fig. 2.11(a) are \( R_{ s }=2 k \Omega, R_{ o }=R_{ ol }=R_{ o 2}=R_{ o3 }=200 \Omega, R_{ i }=R_{ i 1}=R_{ i 2}=R_{ i 3}=R_{ L }=1.5 k \Omega \text {, } \), and \( A_{ vol }=A_{ vo 2}=A_{ vo3 }=80 \text {. } \) Calculate (a) the overall open-circuit voltage gain \( A_{ vo }=v_{ o } / v_{ i } \) , (b) the effective voltage gain \( A_{ v }=v_{ o } / v_{ s } \) , (c) the overall current gain \( A_{ i }=i_{ o } / i_{ s } \), and (d) the power gain \( A_{ p }=P_{ L } / P_{ i } \).

Admin
Solution

(a) Using Eq. (2.19), we can calculate the voltage gain of stage 1 and stage 2 as follows:

\( v_{ o }=i_{ o } R_{ L }=A_{ vo } v_{ i } \frac{R_{ L }}{R_{ L }+R_{ o }} \) (2.19)

 

\( A_{ v 1}=A_{ v 2}=\frac{A_{ vol } R_{ i 2}}{\left(R_{ i 2}+R_{ ol }\right)}=\frac{80 \times 1.5 k }{(1.5 k +200)}=70.588 V / V \)

 

From Eq. (2.43), the overall open-circuit voltage gain of the cascaded amplifiers is

\( A_{ vo }=A_{ vol } A_{ vo 2} A_{ vo3 } \) (2.43)

 

\(\begin{aligned}A_{ vo } &=\frac{v_{ o }}{v_{ i 1}}=A_{ v 1} A_{ v 2} A_{ vo 3}=(70.588)^{2} \times 80 \\&=398,616 V / V \quad[\text { or } 20 \times \log (398,616)=112.01 dB ]\end{aligned}\)

 

(b) To find the effective voltage gain \( A_{ v } \) from the source to the load, we need to include the source and load resistances. From Eq. (2.21), we get

\( A_{ v }=\frac{v_{ o }}{v_{ s }}=\frac{v_{ o }}{v_{ i }} \times \frac{v_{ i }}{v_{ s }}=\frac{A_{ vo } R_{ i } R_{ L }}{\left(R_{ i }+R_{ s }\right)\left(R_{ L }+R_{ o }\right)}=\frac{A_{ vo }}{\left(1+R_{ s } / R_{i}\right)\left(1+R_{ o } / R_{ L }\right)} \) (2.21)

 

\(\begin{aligned}A_{ v } &=\frac{v_{ o }}{v_{ s }}=\frac{A_{ vo } R_{ i } R_{ L }}{\left(R_{ i }+R_{ s }\right)\left(R_{ L }+R_{ o }\right)}=\frac{398,616 \times 1.5 k \times 1.5 k }{(1.5 k +2 k )(1.5 k +200)} \\&=150,737 V / V \quad(\text { or } 103.56 dB )\end{aligned}\)

 

(c) The overall current gain \( A_{ i } \) of the cascaded amplifiers is

 

\(\begin{aligned}A_{ i } &=\frac{i_{ o }}{ i _{ s }}=A_{ v } \frac{R_{ s }+R_{ i }}{R_{ L }}=150,737 V / V \times \frac{3.5 k }{1.5 k } \\&=351,720 A / A \quad[\text { or } 20 \log (351,720)=110.9 dB ]\end{aligned}\)

 

(d) The power gain \( A_{ p } \) becomes

 

\( \left.A_{ p }=\frac{P_{ L }}{P_{ i }}=A_{ v } A_{ i }=5.30 \times 10^{10} \quad \text { [or }(103.56+110.9)=214.46 dB \right] \)
