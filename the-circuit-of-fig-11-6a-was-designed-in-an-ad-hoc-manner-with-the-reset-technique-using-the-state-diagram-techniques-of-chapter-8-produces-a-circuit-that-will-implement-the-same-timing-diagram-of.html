Guest Question.

The circuit of Fig. 11.6(a) was designed in an ad hoc manner with the reset technique. Using the state diagram techniques of Chapter 8 produces a circuit that will implement the same timing diagram of Fig. 11.6(b).

Admin
Solution

The first task is to use the timing diagram of Fig. 11.6(b) to produce a state diagram. At the bottom of Fig. 11.6(b) are the states A and B at each rising 2-clock edge. Remember, that the interrupt (I) line is generated by 2-clock (i.e.synchronised) and thus changes after the 2-clock rising edge. Hence the state diagram, shown in Fig. 11.8(a), can be drawn. The corresponding state transition table is shown in Fig. 11.8(b) and since there are only two states then only one flipflop is needed. Assigning A=0 and B=1 results in Fig. 11.8(c). From this we produce the K-maps for the next state \(Q^{+}\) and present output LOAD(L). These produce the functions \(Q^{+}=I\) and \(L=IÂ·\overline{Q}\).
