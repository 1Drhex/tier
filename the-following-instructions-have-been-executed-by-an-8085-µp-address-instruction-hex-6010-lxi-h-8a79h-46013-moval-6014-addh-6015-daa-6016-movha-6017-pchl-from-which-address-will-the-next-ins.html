Guest Question.

The following instructions have been executed by an 8085 µP:

ADDRESS INSTRUCTION

(HEX)

6010 LXI H, 8A79H

46013 MOVA,L

6014 ADDH

6015 DAA

6016 MOVH,A

6017 PCHL

From which address will the next instruction be fetched?

(a) 6019

(b) 0379

(c) 6979

(d) None of the above

Admin
Solution

6010 LXI H, 8A79H → HL ← 8A79H
6013 MOV A, L → A = 79
6014 ADD H → A ← A + H A ←
8A + 79

\(\begin{array}{r}10001010 \\01111001 \\\hline 1>00000011 \\\hline\end{array}\)

A = 03H CY = 1 AC = 1

6015 DAA Decimal adjust after accumulator
Since CY = 1, AC = 1
A = 03 + 66 = 69H
6016 MOV H, A → H = 69H
6017 PCHL → Exchange contents of PC &
HL
HL = 6979H
Thus, PC becomes 6979H.
Thus, next instruction would be fetched from 6979H.
Hence, the correct option is (c).
