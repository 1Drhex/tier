Guest Question.

A 70\(\Omega\) resistor is wired in series with a capacitor of an unknown value to a 230 volt 50 Hz supply.Calculate the value of the capacitor in microfarads if a current of 1.3 A flows.

Admin
Solution

First find impedance of circuit (Z)
Step 1
\(U = I \times Z\)

230 = 1.3\( \times\) Z

Z = \(\frac{230}{1.3}\)

Z = 176.92 \(\Omega \)
Step 2
Next find capacitive reactance\(X_{C}\)
\(Z^{2}= R^{2}+X_{C}^{2}\)

176.92=\(\sqrt{70^{2} + X_{C}^{2}}\)
Transpose for \(X_{C}\)

\(X_{C } = \sqrt{176.92^{2} – 70^{2}}\)
\(X_{C} = 162.48 \Omega\)

Now find capacitance
Step 3

\(X_{C} = \frac{10^{6}}{2\pi fC}\)
Transpose for C

C = \(\frac{10^{6}}{2\pi fX_{l}}\)

C =\( \frac{10^{6}}{2\times 3.142\times 50\times 162.48}\)
19.59\(\mu F \)is the capacitor value

On calculator enter EXP 6 ÷(2×3.142×50×162.48) = (answer)
