Guest Question.

Evaluating overlap and Coulomb integrals
Evaluate (a) the overlap integral S and (b) the integral \(j^{\prime } \) for two H1s orbitals, a basis set for \(H^{+}_{2}\).
Method To evaluate integrals of this kind, it is natural to use ellipsoidal coordinates, eqn 8.10, for which the volume element is

\(\mu =\frac{r_{A}+r_{B} }{R}\) \(\nu =\frac{r_{A}-r_{B} }{R}\) (8.10)

\(d\tau =\frac{1}{8} R^{3} (\mu ^{2}-\nu ^{2} )d\mu d\nu d\varphi \)

with\(1\leq \mu \leq \infty ,-1\leq \nu \leq 1,\) and \(0\leq \varphi \leq 2\pi \). The atomic wavefunctions we use are \(\chi (r)=(1/\pi a^{3}_{0} )^{1/2} e^{-r/a_{0} } \), where \(a_{0}\) is the Bohr radius; each orbital is centred on its nucleus. All the integrations are straightforward in ellipsoidal coordinates.

Admin
Solution

(a) The product of the two wavefunctions is

\(\chi _{A} (r_{A} )\chi _{B} (r_{B} )=\left(\frac{1}{\pi a^{3}_{0} } \right) e^{-(r_{A}+r_{B} )/a_{0} } =\left(\frac{1}{\pi a^{3}_{0} } \right) e^{-\mu R/a_{0} } \)

Therefore, the overlap integral is

\(S=\left\langle A\mid B\right\rangle =\frac{R^{3} }{8\pi a^{3}_{0} } \int_{0}^{2\pi }{\int_{1}^{\infty }{\int_{-1}^{1}{(\mu ^{2}-\nu ^{2} )e^{-\mu R/a_{0} }d\mu } d\nu } d\varphi } \)

 

\( =\frac{R^{3} }{8\pi a^{3}_{0} } \times 2\pi \times \int_{1}^{\infty }{\int_{-1}^{1}{(\mu ^{2}-\nu ^{2} )e^{-\mu R/a_{0} }d\mu } d\nu } \) (after integration over \(\varphi \))

\( =\frac{R^{3} }{8\pi a^{3}_{0} } \times 2\pi \times \int_{1}^{\infty }{(2\mu ^{2}-\frac{2}{3} )e^{-\mu R/a_{0} }d\mu } \) (after integration over \(\nu \))

\(=\left\{1+\frac{R}{a_{0} } +\frac{1}{3}\left(\frac{R}{a_{0} } \right)^{2} \right\} e^{-R/a_{0} } \)

(b) The contribution \(j^{\prime }\) is similarly, using \(\mu ^{2} -\nu ^{2} =(\mu +\nu )(\mu -\nu )\) :

\(j^{\prime } =\frac{j_{0} }{\pi a^{3}_{0} } \int_{0}^{2\pi }{\int_{1}^{\infty }{\int_{-1}^{1}{\frac{\frac{1}{8}R^{3}(\mu ^{2}-\nu ^{2} )e^{-(\mu +\nu )R/a_{0} } }{½R(\mu -\nu )} d\mu } d\nu }d\varphi } \)

 

\( =\frac{1 }{2 }j_{0}\left(\frac{1}{a_{0} } \right)^{3} R^{2} \int_{1}^{\infty }{\int_{-1}^{1}{(\mu +\nu )e^{-(\mu +\nu )R/a_{0} }} d\mu d\nu }=\frac{1}{R}j_{0} \left\{1-\left(1+\frac{R}{a_{0} } \right)e^{-2R/a_{0} } \right\}\)

These two functions are plotted in Figs 8.8 and 8.9.
