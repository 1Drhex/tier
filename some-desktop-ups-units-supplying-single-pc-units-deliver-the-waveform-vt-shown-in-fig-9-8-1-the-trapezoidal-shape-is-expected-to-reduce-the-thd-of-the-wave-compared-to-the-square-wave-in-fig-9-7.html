Guest Question.

Some desktop UPS units supplying single PC units deliver the waveform \(v(t)\) shown in Fig. 9.8-1. The trapezoidal shape is expected to reduce the \(THD\) of the wave compared to the square wave in Fig. 9.7-2. (a) Obtain an expression for \(rms\) value of this voltage in terms of \(V\) and \(α\). (b) Find the Fourier series coefficients for v(t). (c) Find \(V\) and \(α\) such that the third harmonic content is zero and the \(rms\) value is \(220 V\). (d) With these values for \(V\) and \(α\), find the \(THD\) of \(v(t).\)

Admin
Solution

The waveform \(v(t)\) exhibits odd symmetry and half-wave symmetry. Its Fourier series will contain only odd harmonics of sine format.

(a) \(rms\) value of \(v(t)=\sqrt{\frac{4}{T} \int_{0}^{0.5 \alpha T}\left(\frac{2 V t}{\alpha T}\right)^{2} d t+V^{2}(0.25 T-0.5 \alpha T)}=V \sqrt{1-\frac{4}{3} \alpha} \mathrm{V}\)

 

(b) Fourier series coefficients can be found by a straight application of the analysis equation of Fourier series (Eqn. 9.2-2). However, a more elegant method is to consider a waveform which will produce \(v(t)\) on integration. Find its Fourier series coefficients and obtain the Fourier series coefficients of \(v(t)\) by dividing those coefficients by \(\operatorname{jn} \omega_{\mathrm{o}}\) The waveform shown in Fig. 9.8-2 is the required one.

(c) Exponential Fourier series coefficients for this waveform is found by splitting the waveform into two – the upper half-cycles alone will constitute a periodic rectangular pulse train with period \(T\) and the lower half-cycles alone will constitute the same rectangular pulse train delayed by \(0.5T\) and negated.

 

\(\tilde{v}_{n}=\frac{1}{T} \int_{-T / 2}^{T / 2} v(t) e^{-j n \omega_{\mathrm{o}} t} d t-\text { analysis equation } \quad\quad (9-2-2)\)

 

\(v_{\mathrm{n}}=\frac{2 V}{\alpha T} \frac{1}{j n \omega_{0} T}\left(e^{j 0.5 n \omega_{0} \alpha T} e^{-j 0.5 n \omega_{0} \alpha T}\right)-\frac{2 V}{\alpha T} \frac{e^{-j 0.5 n \omega_{0} T}}{j n \omega_{0} T}\left(e^{j 0.5 n \omega_{0} \alpha T}-e^{-j 0.5 n \omega_{0} \alpha T}\right)\)

 

\(e^{-j 0.5 n \omega_{0} T}=e^{-j n \pi}=1 \text { for even } n \text { and }-1 \text { for odd } n\)

 

\(\begin{aligned} \therefore \tilde{v}_{n}^{\prime} &=\left\{\begin{array}{l} 0, \text { for even } n \\ \frac{2 V}{\alpha T} \frac{2}{\operatorname{jn} \omega_{o} T}\left(e^{j 0.5 n \omega_{o} \alpha T}-e^{-j 0.5 n \omega_{o} \alpha T}\right), \text { for odd } n \end{array}\right.\\ =&\left\{\begin{array}{l} 0, \text { for even } n \\ \frac{4 V}{T} \frac{\sin \pi n \alpha}{\pi n \alpha}, \text { for odd } n \end{array}\right. \end{aligned}\)

 

Coefficients of exponential Fourier series of \(v(t)\) is obtained by dividing these coefficients by \(\operatorname{jn} \omega_{\mathrm{o}} \)

\(\therefore \tilde{v}_{n}=\left\{\begin{array}{l} 0, \text { for even } n \\ -j \frac{2 V}{\pi n} \frac{\sin \pi n \alpha}{\pi n \alpha}, \text { for odd } n \end{array}\right. \quad\quad (9-8-1)\)

 

The rate of decay of harmonic amplitude must be \(\propto \frac{1}{n^{2}}\) and it is seen to be so. Notice that the waveform becomes a square waveform when \(α = 0\). The Fourier series coefficients given in Eqn. 9.8-1 become the same as that of a square wave, i.e \(\frac{2 V}{\pi n}\) for even and when \(α = 0.5\), \(v(t)\) becomes a triangle waveform with \(\frac{4 V}{\pi^{2} n^{2}}\) for odd as magnitude of exponential Fourier series coefficient magnitude.

(a) Third harmonic content will go to zero when \(sin\) \(3\pi α = 0\). Therefore, \(α = 1/3\) is the required value. The \(rms\) value with this value of a is \(0.7453V\) and for \(220 V\) \(rms\) value \(V\) must be \(≈ 295 V\).

(b) The \(THD\) is evaluated as

\(\mathrm{THD}=100 \times \sqrt{\sum_{n=3 \atop \mathrm{odd} n}^{\infty} \frac{1}{n^{2}}\left(\frac{\sin ^{n \pi / 3}}{n \pi / 3}\right)^{2}} \div \frac{\sin \pi / 3}{\pi / 3} \approx 4.6 \%\)
