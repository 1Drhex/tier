Guest Question.

A point mass M with current position P moves, as shown in Fig. 1.9, along the x-axis with acceleration \(\ a=k\sqrt{v} \), where the constant \(\ k = 2\left(m/s^{3} \right)^{1/2} \). At time \(\ t=0 \), M passes the location \(\ x_{0}=1/3m \) with a velocity \(\ v_{0}=1 m/s \).
Find the location \(x_{1}\) of M at time\(\ t_{1}=2s \). What are the velocity and acceleration at this time?

Admin
Solution

The acceleration is given as a function of velocity. According to (1.13), we have

\(\ \int\limits_{t_{0}}^{t}{d\overline{t} } =\int_{v_{0}}^{v}{\frac{d\overline{v} }{a\left(\overline{v} \right) } } \rightarrow t=t_{0}+\int_{v_{0}}^{v}{\frac{d\overline{v} }{a\left(\overline{v} \right) } }=f\left(v\right) . \) (1.13)

\(\ t=\int_{v_{0}}^{v}{\frac{d\overline{v} }{k\sqrt{ \overline{v}} } }=\frac{2}{k} \left(\sqrt{v}-\sqrt{v_{0}} \right) \rightarrow v=v\left(t\right)=\left(\frac{kt}{2}+\sqrt{v_{0}} \right)^{2} \).

Indeterminate integration of v gives

\(\ x=\int{v dt}=\frac{1}{3}\frac{2}{k} \left(\frac{kt}{2}+\sqrt{v_{0}} \right)^{3} +C .\)

The integration constant C is found using the initial condition \(\ x\left(0\right)=x_{0} \):
\(\ \frac{1}{3}=\frac{1}{3} \frac{2}{2}\left(\sqrt{1 }\right)^{3}+C \rightarrow C=0\).

Thus, we arrive at the result

\(\ x=\frac{2}{3k} \left(\frac{kt}{2}+\sqrt{v_{0}} \right) ^{3} \),

 

\(\ v=\dot{x}=\left(\frac{kt}{2}+\sqrt{v_{0}} \right)^{2} \),

 

\(\ a=\dot{v}=\ddot{x} =k\left(\frac{kt}{2}+\sqrt{v_{0}} \right)\).

At time \(\ t=t_{1}\) we find

\(\ \underline{\underline{x_{1}} } =x\left(t_{1}\right)=\frac{2}{3.2} \left(\frac{2.2}{2}+\sqrt{1} \right)^{3}=\underline{\underline{9m} } \),

 

\(\ \underline{\underline{v_{1}} }=v\left(t_{1}\right)= \left(\frac{2.2}{2}+\sqrt{1} \right)^{2}=\underline{\underline{9\frac{m}{s} } } \),

 

\(\ \underline{\underline{a_{1}} } =a\left(t_{1}\right)=2 \left(\frac{2.2}{2}+\sqrt{1} \right)=\underline{\underline{6\frac{m}{s^{2}}} } \).

As a check, we can easily see that the result is compatible with the initially given acceleration formula: \(\ a=k\sqrt{v } \) .
