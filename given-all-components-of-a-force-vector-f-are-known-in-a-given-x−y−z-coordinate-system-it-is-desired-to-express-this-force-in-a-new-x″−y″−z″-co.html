Guest Question.

Given: All components of a force vector \(\overline{F} \) are known in a given x−y−z coordinate system. It is desired to express this force in a new x″−y″−z″ coordinate system, where the x″−y″−z″ system is generated from the original x−y−z system by the following two rotations (see Figure 2.2):

• A rotation of θ-degrees about the original z-axis (which defines an intermediate x′−y′−z′ coordinate system), followed by

• A rotation of β-degrees about the x′-axis (which defines the final x″−y″−z″ coordinate system)

PROBLEM

a. Determine the direction cosines \(c_{i^{\prime \prime } j}\) relating the original x−y−z coordinate system to the new x″−y″−z″ coordinate system.

b. Obtain a general expression for the force vector \(\overline{F} \) in the x″−y″−z″ coordinate system.

c. Calculate numerical values of the force vector \(\overline{F} \) in the x″−y″−z″ coordinate system if θ = −20°, β = 60°, and \(F_x = 1000 N, F_y = 200 N, F_z = 600 N.\)

Admin
Solution

a. One way to determine direction cosines \(c_{i^{\prime \prime } j}\) is to rotate unit vectors. In this approach unit vectors are first rotated from the original x−y−z coordinate system to the intermediate x′−y′−z′ coordinate system, and then from the x′−y′−z′ system to the final x″−y″−z″ coordinate system.

Define a unit vector \(\overline{I} \) that is aligned with the x-axis:

\(\overline{I}\equiv (1)\hat{i}\)

That is, vector \(\overline{I}\) is a vector for which \(I_x = 1, I_y = 0\), and \(I_z = 0\). The vector I can be rotated to the intermediate x′−y′−z′ coordinate system using Equation 2.6:

\(F_{x^\prime }=c_{x^\prime x}F_{x}+c_{x^\prime y}F_{y}+c_{x^\prime z}F_{z} \\ F_{y^\prime }=c_{y^\prime x}F_{x}+c_{y^\prime y}F_{y}+c_{y^\prime z}F_{z} \\ F_{z^\prime }=c_{z^\prime x}F_{x}+c_{z^\prime y}F_{y}+c_{z^\prime z}F_{z}\) (2.6a)

\(F_{i^\prime }=c_{i^\prime j}F_{j}\) (2.6b)

\(\begin{Bmatrix} F_{x^\prime } \\ F_{y^\prime } \\ F_{z^\prime } \end{Bmatrix} = \begin{bmatrix} c_{x^\prime x} & c_{x^\prime y} & c_{x^\prime z} \\ c_{y^\prime x} & c_{y^\prime y} & c_{y^\prime z} \\ c_{z^\prime x} & c_{z^\prime y} & c_{z^\prime z} \end{bmatrix} \begin{Bmatrix} F_{x} \\ F_{y} \\ F_{z} \end{Bmatrix}\) (2.6c)

\(I_{x^\prime }=c_{x^\prime x }I_x +c_{x^\prime y}I_y +c_{x^\prime z}I_z\)

 

\(I_{y^\prime }=c_{y^\prime x }I_x +c_{y^\prime y}I_y +c_{y^\prime z}I_z\)

 

\(I_{z^\prime }=c_{z^\prime x }I_x +c_{z^\prime y}I_y +c_{z^\prime z}I_z\)

 

The direction cosines associated with a transformation from the x−y−z coordinate system to the intermediate x′−y′−z′ coordinate system can be determined by inspection (see Figure 2.2a), and are given by

\(c_{x^\prime x }\)=cosine(angle between \(x^\prime\)– and x-axes) =cas θ

\(c_{x^\prime y}\)=cosine(angle between \(x^\prime\)– and y-axes)= cos (90°- θ) = sin θ

\(c_{x^\prime z}\)=cosine(angle between \(x^\prime\)– and z-axes)= cos (90°) = 0

\(c_{y^\prime x}\)=cosine(angle between the \(y^\prime\)– and x-axes)= cos (90°+θ) = – sin θ

\(c_{y^\prime y}\)=cosine(angle between the \(y^\prime\)– and y-axes)= cos θ

\(c_{y^\prime z}\)=cosine(angle between the \(y^\prime\)– and z-axes)= cos (90°) = 0

\(c_{z^\prime x}\)=cosine(angle between the \(z^\prime\)– and x-axes)= cos (90°) = 0

\(c_{z^\prime y}\)=cosine(angle between the \(z^\prime\)– and y-axes)= cos (90°) = 0

\(c_{z^\prime z}\)=cosine(angle between the \(z^\prime\)– and z-axes)= cos (0°) = 1

Using these direction cosines:

\(I_{x^\prime }=c_{x^\prime x }I_x +c_{x^\prime x}I_y +c_{x^\prime z}I_z\)= (cos θ)(1) + (sinθ )(0) + (0)(0) = cos θ

\(I_{y^\prime }=c_{y^\prime x }I_x +c_{y^\prime x}I_y +c_{y^\prime z}I_z\)= (−sinθ )(1) + (cos θ)(0) + (0)(0) = −sinθ

\(I_{z^\prime }=c_{z^\prime x }I_x +c_{z^\prime y}I_y +c_{z^\prime z}I_z\) =(0)(1) + (0)(0) + (1)(0) = 0

Therefore, in the x′−y′−z′ coordinate system, the vector \(\overline{I}\) is written:

\(\overline{I}= (\cos \theta )\hat{i} ^{\prime } + ( -\sin \theta )\hat{j}\)

Now define two additional unit vectors, one aligned with the original y-axis (vector \(\overline{J}\) ) and one aligned with the original z-axis (vector \(\overline{K}\) ); that is, let \(\overline{J} = (1)\hat{j}\) and \(\overline{K} ) = (1) \hat{k}\) . Transforming these vectors to the x′−y′−z′ coordinate system, again using the direction cosines listed above, results in

\(\overline{J}= (\sin \theta )\hat{i} ^{\prime } + ( \cos \theta )\hat{j}^\prime\)

 

\(\overline{K}= (1)\hat{k}^\prime\)

We now rotate vectors \(\overline{I} , \overline{J}\) and \(\overline{K}\) from the intermediate x′−y′−z′ coordinate system to the final x″−y″−z″ coordinate system. The direction cosines associated with a transformation from the x′−y′−z′ coordinate system to the final x″−y″−z″ coordinate system are easily determined by inspection (see Figure 2.2b) and are given by

\(c_{x^{\prime \prime} x^\prime }=1\) \(c_{x^{\prime \prime} y^\prime }=0\) \(c_{x^{\prime \prime} z^\prime }=0\)

\(c_{y^{\prime \prime} x^\prime }=0\) \(c_{y^{\prime \prime} y^\prime }=\cos β\) \(c_{y^{\prime \prime} z^\prime }=\sin β \)

\(c_{z^{\prime \prime} x^\prime }=0 \) \(c_{z^{\prime \prime} y^\prime }= – \sin β \) \(c_{z^{\prime \prime} z^\prime }=\cos β \)

These direction cosines together with Equation 2.6 can be used to rotate the vector \(\overline{I}\) from the intermediate x′−y′−z′ coordinate

system to the final x″−y″−z″ coordinate system:

\(I_{x^{\prime\prime } }=c_{x^{\prime\prime } x^\prime } I_{x^\prime } +c_{x^{\prime\prime} y^\prime }I_{y^\prime } +c_{x^{\prime\prime } z^\prime }I_{z^\prime }\) = (1)(cos θ) + (0)(−sin θ) + (0)(0)

 

\(I_{x^{\prime\prime } }=\cos θ\)

 

\(I_{y^{\prime\prime } }=c_{y^{\prime\prime } x^\prime } I_{x^\prime } +c_{y^{\prime\prime} y^\prime }I_{y^\prime } +c_{y^{\prime\prime } z^\prime }I_{z^\prime }\)=(0)(cos θ) + (cos β)(−sin θ) + (sin β)( 0)

\(I_{y^{\prime\prime } }\) = -cos β sin θ

\(I_{z^{\prime\prime } }=c_{z^{\prime\prime } x^\prime } I_{x^\prime } +c_{z^{\prime\prime} y^\prime }I_{y^\prime } +c_{z^{\prime\prime } z^\prime }I_{z^\prime }\)=(0)(cos θ) + (- sin β)(−sin θ) + (cos β)( 0)

\(I_{z^{\prime\prime } }\) = sin β sin θ

Therefore, in the final x″−y″−z″ coordinate system, the vector \(\overline{I}\) is written:

\(\overline{I}= (\cos \theta )\hat{i}^{\prime \prime } +(- \cos \beta \sin\theta )\hat{j}^{\prime \prime } + ( \sin \beta \sin \theta )\hat{k}^{\prime \prime }\) (a)

Recall that in the original x−y−z coordinate system \(\overline{I}\) is simply a unit vector aligned with the original x-axis: \(\overline{I} ≡ (1) \hat{i}\) . Therefore, result (a) defines the direction cosines associated with the angle between the original x-axis and the final x″-, y″-, and z″-axes. That is

\(c_{x^{\prime\prime} x }\) = cos θ

\(c_{y^{\prime\prime} x }\)= – cos β sin θ

\(c_{z^{\prime\prime} x }\)= sin β sin θ

A similar procedure is used to rotate the unit vectors \(\overline{J}\)and \(\overline{K}\) from the intermediate x′−y′−z′ coordinate system to the final x″−y″−z″ coordinate system. These rotations result in

\(\overline{J}= (\cos \theta )\hat{i}^{\prime \prime } +( \cos \beta \cos\theta )\hat{j}^{\prime \prime } + (- \sin \beta \cos \theta )\hat{k}^{\prime \prime }\) (b)

 

\(\overline{K}=(0)\hat{i}^{\prime \prime } + (\sin \beta )\hat{j}^{\prime \prime } + (\cos \beta )\hat{k}^{\prime \prime } \) (c)

As vector \(\overline{J}\) is a unit vector aligned with the original y-axis, \(\overline\){J} = (1)\hat{j} , result (b) defines the direction cosines associated with the angle between the original y-axis and the final x″-, y″-, and z″-axes:

\(c_{x^{\prime \prime} y}= \sin \theta\)

 

\(c_{y^{\prime \prime} y}= \cos β \cos \theta\)

 

\(c_{z^{\prime \prime} y}= \sin β \cos \theta\)

Finally, result (c) defines the direction cosines associated with the angle between the original z-axis and the final x″-, y″-, and z″-axes:

\(c_{x^{\prime \prime} z}= 0 \)

 

\(c_{y^{\prime \prime} y}= \sin β \)

 

\(c_{z^{\prime \prime} y}= \cos β \)

Assembling the preceding results, the set of direction cosines relating the original x−y−z coordinate system to the final x″−y″−z″ coordinate system can be written:

\(\left [ \begin{matrix} c_{x^{\prime \prime }x} & c_{x^{\prime \prime }y} & c_{x^{\prime \prime }z} \\c_{y^{\prime \prime }x} & c_{y^{\prime \prime }y} & c_{y^{\prime \prime }z} \\ c_{z^{\prime \prime }x} & c_{z^{\prime \prime }y} & c_{z^{\prime \prime }z} \end{matrix} \right ]= \left [ \begin{matrix}\cos\theta & \sin\theta & 0 \\ - \cos \beta \sin\theta & \cos \beta \cos\theta & \sin\beta \\ \sin\beta \sin\theta & - \sin\beta \cos\theta & \cos\beta \end{matrix} \right ]\)

b. As direction cosines have been determined, transformation of force vector \(\overline{F} \) can be accomplished using any version of Equation 2.6. For example, using matrix notation, Equation 2.6c:

\(\left \{ \begin{matrix} F_x^{\prime \prime } \\ F_y^{\prime \prime } \\ F_z^{\prime \prime } \end{matrix} \right \}=\left [ \begin{matrix} c_{x^{\prime \prime }x} & c_{x^{\prime \prime }y} & c_{x^{\prime \prime }z} \\ c_{y^{\prime \prime }x} & c_{y^{\prime \prime }y} &c_{y^{\prime \prime }z} \\ c_{z^{\prime \prime }x} & c_{z^{\prime \prime }y} & c_{z^{\prime \prime }z} \end{matrix} \right ]\left \{ \begin{matrix}F_x \\ F_y \\F_z \end{matrix} \right \}\)

 

\(= \left [ \begin{matrix} \cos\theta & \sin\theta & 0 \\ - \cos\beta \sin\theta & \cos \beta \cos\theta & \sin\beta \\ \sin\beta \sin \theta &- \sin \beta \cos \theta & \cos \beta \end{matrix} \right ]\left \{ \begin{matrix} F_x \\ F_y \\ F_z \end{matrix} \right \}\)

 

\(\left \{ \begin{matrix} F_x″\\ F_y″ \\ F_z″ \end{matrix} \right \}= \left \{ \begin{matrix} (\cos\theta )F_x + (\sin\theta )F_y \\ ( - \cos\beta \sin\theta )F_x + (\cos\beta cos\theta )F_y + (\sin\beta )F_z \\ (\sin \beta \sin\theta )F_x + ( - \sin\beta \cos \theta )F_y + (\cos\beta )F_z \end{matrix} \right \}\)

c. Using the specified numerical values and the results of part (b):

\(\left \{ \begin{matrix} F_x″\\ F_y″ \\ F_z″ \end{matrix} \right \}=\left \{ \begin{matrix}[\cos(-20°)](1000N) + [\sin( -20°)]200 N \\ [- \cos(60°)(\sin -20°)](1000N) + [\cos(60°)\cos(20°)](200 N) +[\sin(60°)](600 N) \\ [\sin(60°)\sin(- 20°)](1000 N) + [-\sin(60°)\cos( -20°)](200 N)+ (\cos60°)(600 N) \end{matrix} \right \}\)

 

\(\left \{ \begin{matrix} F_x″\\ F_y″ \\ F_z″ \end{matrix} \right \}=\left \{ \begin{matrix} 873.1 N \\ 784.6 N \\ - 159.0 N \end{matrix} \right \}\)

Using vector notation, \(\overline{F}\) can now be expressed in the two different coordinate systems as

\(\overline{F}= (1000 N)\hat{i} + (200 N) \hat{j} + (600 N)\hat{k}\)

or, equivalently

\(\overline{F}= (873.1 N)\hat{i}″ + (784.6 N) \hat{j}″ + (- 159 N)\hat{k}″\)

Where \(\hat{i}, \hat{j}, \hat{k}\) and \(\hat{i}″, \hat{j}″,\hat{k}″\) are unit vectors in the x−y−z and x″−y″−z″ coordinate systems, respectively. Force vector \(\overline{F}\) drawn in the x−y−z and x″−y″−z″ coordinate systems is shown in Figure 2.3a and b, respectively. The two descriptions of \(\overline{F}\) are entirely equivalent. A convenient way of (partially) verifying this equivalence is to calculate the magnitude of the original and transformed force vectors. As the magnitude is an invariant, it is independent of the coordinate system used to describe
the force vector. Using Equation 2.2, the magnitude of the force vector in the x−y−z coordinate system is

\(|\overline{F}|= \sqrt{F_x^2 +F_y^2 + F_z^2}\) (2.2)

\(|\overline{F}|= \sqrt{F_x^2 +F_y^2 + F_z^2}= \sqrt{(1000 N)^2 +(200 N)^2 + (600 N)^2}=1183 N\)

The magnitude of the force vector in the x″−y″−z″ coordinate
system is
\(|\overline{F}|= \sqrt{(F_x″)^2 +(F_y″)^2 + (F_z″)^2}= \sqrt{871.3 N)^2 +(784.6 N)^2 + (- 159.0 N)^2}=1183 N\)(agrees)
