Guest Question.

Table 2.6 shows the results of standard compaction test on a sand/cement mixture having G = 2.70.

a. Plot these results together with the zero air void line, and determine what percentage of air voids exists in the sample at OMC \(\left(i.e.A_{OMC} \right)\) .

Table 2.6 Compaction data.

20 16 12.5 10 8 5 Water content
\(\left[\%\right] \)
1.73 1.84 1.89 1.85 1.78 1.64 Dry density \(\left[Mg/ m^{3} \right] \)
Admin
Solution

a. For plotting the ZAV, we can use the following relationship:

\(\rho _{d} =\frac{G_{s}\rho _{w} }{1+e} \Leftrightarrow \rho _{d} =\frac{G_{s}\rho _{w}}{1+\frac{wG_{s} }{S_{r} } }\)

 

\(\therefore \rho _{d\left(ZAV,S_{r}=1 \right) } =\frac{G_{s}\rho _{w} }{1+wG_{s}}\)

Table 2.7 shows the results obtained using the previous relationship, and these are shown in Figure 2.8.

Using the relationship derived earlier \(\rho _{d} =\frac{G_{s}\rho _{w} \left(1-A_{\nu } \right) }{\left(G_{s}w+1 \right) } \) then, for OMC, we have

\(\rho _{d} =1.89 Mg/ m^{3}\)

 

\(w=12.5\%\)

 

\(1.89=\frac{2.7\times 1\times \left(1-A_{\nu } \right) }{\left(2.7\times 0.125+ 1\right) }\)

 

\(\left(1-A_{\nu } \right)=0.94\) or \(A_{\nu }=0.064\) or \(A_{\nu }=6.4\%\)

b. For the condition \(A_{OMC}\pm 2.5\%; \) then \(A_{\nu }=3.9\%\) and \(A_{\nu }=8.9\%\) (Fig. 2.9 ).
