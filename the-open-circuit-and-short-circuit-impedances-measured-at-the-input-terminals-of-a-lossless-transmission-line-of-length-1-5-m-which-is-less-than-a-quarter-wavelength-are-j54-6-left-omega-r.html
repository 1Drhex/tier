Guest Question.

The open-circuit and short-circuit impedances measured at the input terminals of a lossless transmission line of length \(1.5 (m)\), which is less than a quarter wavelength, are \(-j54.6 \left( \Omega \right)\) and \(j 103 \left( \Omega \right)\). respectively.

\(a)\) Find \({ Z }_{ 0 }\) and \(\gamma\) of the line.

\(b)\) Without changing the operating frequency, find the input impedance of a short-circuited line that is twice the given length.

\(c)\) How long should the short-circuited line be in order for it to appear as an open circuit at the input terminals?

Admin
Solution

The given quantities are

 

\({ Z }_{ io }=-j54.6,\quad \quad { Z }_{ is }=j103,\quad \quad \ell =1.5\)

 

\(a)\) Using Eqs (8-85) and (8-86), we find

 

\({ Z }_{ 0 }=\sqrt { -j54.6\left( j103 \right) } =75\quad \left( \Omega \right)\)

 

\(\gamma =\frac { 1 }{ 1.5 } \tanh ^{ -1 }{ \sqrt { \frac { j103 }{ -j54.6 } } } =\frac { j }{ 1.5 } \tanh ^{ -1 }{ 1.373 } =j0.628\quad \left( rad/m \right) \)

 

\(b)\) For a short-circuited line twice as long, \(\ell = 3.0 (m)\),

 

\(\gamma \ell =j0.628\times 3.0=j1.884\quad \left( rad \right)\)

 

The input impedance is, from Eq. (8-84b),

 

\({ Z }_{ is }=75\tanh { \left( j1.884 \right) } =j75\tan { { 108 }^{ o } }\)

 

\(\quad =j75\left( -3.08 \right) =-j231\quad \left( \Omega \right)\)

 

Note that \({ Z }_{ is }\) for the \(3-(m)\) shorted line is now a capacitive reactance, whereas that for the given \(1.5-(m) \) shorted line is an inductive reactance.

 

\(c)\) In order for a short-circuited line to appear a an open circuit at the input terminals, it should be an odd multiple of a quarter-wavelength . long, which will make tan \(\beta \ell \rightarrow \infty \), in Eq. (8-82).

 

\(\lambda =\frac { 2\pi }{ \beta } =\frac { 2\pi }{ 0.628 } =10\quad \left( m \right)\)

 

Hence the required line length is

 

\(\ell =\frac { \lambda }{ 4 } +\left( n-1 \right) \frac { \lambda }{ 2 }\)

 

\(\quad =2.5+5\left( n-1 \right) \quad \left( m \right) ,\quad \quad n=1,2,3…\)

 

\({ Z }_{ is }=j{ X }_{ is }=j{ R }_{ 0 }\tan { \beta \ell } \quad \quad \quad \left( 8-82 \right)\)

 

Shore-circuited line, \({ Z }_{ L }=0:\quad \quad { Z }_{ is }={ Z }_{ 0 }\tan { \gamma \ell } \quad \quad \quad \left( 8-84b \right) \)

 

\(\gamma =\frac { 1 }{ \ell } \tanh ^{ -1 }{ \sqrt { \frac { { Z }_{ is } }{ { Z }_{ io } } } } \quad \left( { m }^{ -1 } \right) \quad \quad \left( 8-86 \right)\)
