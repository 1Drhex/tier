The Vector Space of All Polynomials of Degree 2 or Less

Let \(p_{2} \) be the set of all polynomials of the form \(q(x) = b_{0} + b_{1} x + b_{2} x^{2}\), where \(a_0, a_1, and a_2\) are real numbers. The sum of two polynomials \(p(x) = a_{0} + a_{1} x + a_{2} x^{2} \) and \(q(x) = b_{0} + b_{1} x + b_{2} x^{2}\) is defined in the usual way, \(p\left(x\right) + q\left(x\right) = (a_{0} + b_{0}) + (a_{1}+ b_{1})x + (a_{2}+ b_{2})x^{2} \) and the scalar multiple of p(x) by the scalar c is defined by \(cp(x) = ca_{0} + ca_{1} x + ca_{2} x^{2}\).

Show that \(p_{2} \) is a vector space.

Admin
Solution

Verification of each of the ten vector space axioms is a straightforward application of the properties of real numbers. For example, the set of real numbers is closed under addition, so it follows that a0+ b0, a1+ b1, and a2+ b2 are real numbers, and
\(p\left(x\right) + q\left(x\right) = (a_{0} + b_{0}) + (a_{1}+ b_{1})x + (a_{2}+ b_{2})x^{2} \)
is in the set \( P_{2} \)latex]because it is a polynomial of degree 2 or less. So, \( P_{2} \)is closed under addition. To verify the commutative property of addition, write \(p\left(x\right) + q\left(x\right) = \left(a_{0} + a_{1} x + a_{2} x^{2}\right) +\left(b_{0} + b_{1} x + b_{2} x^{2}\right) =(a_{0} + b_{0}) + (a_{1}+ b_{1})x + (a_{2}+ b_{2})x^{2} =(b_{0} + a_{0}) + ( b_{1}+a_{1})x + (b_{2}+ a_{2})x^{2} =q\left(x\right) + p\left(x\right)\)

Can you see where the commutative property of addition of real numbers was used? The zero vector in this space is the zero polynomial\(0(x) = 0 + 0 x + 0 x^{2}\).
Verify the other vector space axioms to show that \( P_{2} \) is a vector space.
