Guest Question.

Suppose a system obeys the linear differential equation

\(c \dot{y}+\alpha y=x\)

where the overdot implies differentiation with respect to t . If \(x(t)=\cos \omega_{k} t\) what is the response y(t)?

Admin
Solution

We will first derive the complex response of the system to complex input, then take the real part for the solution. The complex response of the system to the

 

frequency ωk is obtained by setting the input \(x_{c}(t)\) and output \(y_{c}(t)\) as follows:

 

\(\begin{aligned}&x_{c}(t)=e^{i \omega_{k} t} \\&y_{c}(t)=H\left(\omega_{k}\right) x_{c}(t)=H\left(\omega_{k}\right) e^{i \omega_{k} t}\end{aligned}\)

 

Substitution of these into the system differential equation gives

 

\(c \frac{d}{d t} H\left(\omega_{k}\right) e^{i \omega_{k} t}+\alpha H\left(\omega_{k}\right) e^{i \omega_{k} t}=e^{i \omega_{k} t}\)

 

or

 

\(\left(i c \omega_{k}+\alpha\right) H\left(\omega_{k}\right) e^{i \omega_{k} t}=e^{i \omega_{k} t}\)

 

which can be solved for the transfer function to give

 

\(\begin{aligned}H\left(\omega_{k}\right) &=\frac{1}{i c \omega_{k}+\alpha} \\&=\frac{\alpha-i c \omega_{k}}{\alpha^{2}+c^{2} \omega_{k}^{2}} \\&=\left(\frac{\alpha}{\alpha^{2}+c^{2} \omega_{k}^{2}}\right)-i\left(\frac{c \omega_{k}}{\alpha^{2}+c^{2} \omega_{k}^{2}}\right)\end{aligned}\)

 

The magnitude of the transfer function tells us how much the input signal is amplified,

 

\(\left|H\left(\omega_{k}\right)\right|=\frac{\sqrt{\alpha^{2}+c^{2} \omega_{k}^{2}}}{\alpha^{2}+c^{2} \omega_{k}^{2}}=\frac{1}{\sqrt{\alpha^{2}+c^{2} \omega_{k}^{2}}}\)

 

Recalling that H(ω) \(=A_{\omega}+i B_{\omega}\) , we must have

 

\(A_{\omega_{k}}=\frac{\alpha}{\alpha^{2}+c^{2} \omega_{k}^{2}}, \quad B_{\omega_{k}}=-\frac{c \omega_{k}}{\alpha^{2}+c^{2} \omega_{k}^{2}}\)

 

The complex response \(y_{c}(t)\) to the complex input \(y_{c}(t)\) \(=\) \(e^{i \omega_{k} t}\) is thus \(y_{c}(t)=H\left(\omega_{k}\right) e^{i \omega_{k} t}\), which expands into

 

\(\begin{aligned}y_{c}(t)=& \frac{1}{\alpha^{2}+c^{2} \omega_{k}^{2}}\left[\alpha \cos \omega_{k} t+c \omega_{k} \sin \omega_{k} t\right.\\&\left.+i\left(\alpha \sin \omega_{k} t-c \omega_{k} \cos \omega_{k} t\right)\right]\end{aligned}\)

 

The real response to the real input x(t ) \(=\cos \omega_{k} t\) is therefore

 

\(\begin{aligned}y(t) &=\operatorname{Re}\left(y_{c}(t)\right) \\&=\frac{1}{\alpha^{2}+c^{2} \omega_{k}^{2}}\left[\alpha \cos \omega_{k} t+c \omega_{k} \sin \omega_{k} t\right] \\&=\left(\frac{\sqrt{\alpha^{2}+c^{2} \omega_{k}^{2}}}{\alpha^{2}+c^{2} \omega_{k}^{2}}\right) \cos \left(\omega_{k} t+\tan ^{-1}\left(\frac{c \omega_{k}}{\alpha}\right)\right) \\&=\frac{1}{\sqrt{\alpha^{2}+c^{2} \omega_{k}^{2}}} \cos \left(\omega_{k} t+\phi_{k}\right)\end{aligned}\)

 

\(=\left|H\left(\omega_{k}\right)\right| \cos \left(\omega_{k} t+\phi_{k}\right)\) (3.32)

 

where \(\phi_{k}=\tan ^{-1}\left(c \omega_{k} / \alpha\right)\) is the phase shift

 

The transfer function H(ω) gives the steady-state response of a linear system to a sinusoidal input at frequency ω. If we make use of the superposition principle of linear systems, then we could compute a series of transfer functions \(H\left(\omega_{1}\right), H\left(\omega_{2}\right)\), . . . corresponding to sinusoidal excitations at frequencies \(\omega_{1}, \omega_{2}\), . . . . The overall system response would be the sum of all the individual responses. To determine the spectral density function \(S_{Y}(\omega)\) of the system response Y (t ), we start by assuming that the input X(t ) is equal to the sinusoidal component given by Eq. 3.15,

 

\(X_{k}(t)=C_{k} \cos \left(\omega_{k} t+\Phi_{k}\right)\) (3.15)

 

\(X_{k}(t)=C_{k} \cos \left(\omega_{k} t+\Phi_{k}\right)\) (3.33)

 

where \(\Phi_{k}\) is uniformly distributed between 0 and 2π and independent of \(C_{k}\) . Assuming that the spectral density function of \(X(t), S_{X}(\omega)\), is known, we select \(C_{k}\) to be random with

 

\(E \left[C_{k}^{2}\right]=2 S_{X}\left(\omega_{k}\right) \Delta \omega\)

 

so that Eq. 3.18 holds. Equation 3.32 tells us that the random response \(Y_{k}(t) \text { will be amplified by }\left|H\left(\omega_{k}\right)\right|\) and phase shifted by \(\phi_{k}\) from the random input \(X_{k}(t)\),

 

\(S\left(\omega_{k}\right) \Delta \omega=\operatorname{Var}\left[X_{k}(t)\right]= E \left[X_{k}^{2}(t)\right]=\frac{1}{2} E \left[C_{k}^{2}\right]\) (3.18)

 

\(Y_{k}(t)=\left|H\left(\omega_{k}\right)\right| C_{k} \cos \left(\omega_{k} t+\Phi_{k}+\phi_{k}\right)\)

 

The spectral density of \(Y_{k}(t)\) is obtained in exactly the same way as the spectral density of \(X_{k}(t)\) was found in Eq. 3.18,

 

\(\begin{aligned}S_{Y}\left(\omega_{k}\right) \Delta \omega &=\operatorname{Var}\left[Y_{k}(t)\right]= E \left[Y_{k}^{2}(t)\right] \\&= E \left[\left|H\left(\omega_{k}\right)\right|^{2} C_{k}^{2} \cos ^{2}\left(\omega_{k} t+\Phi_{k}+\phi_{k}\right)\right] \\&=\left|H\left(\omega_{k}\right)\right|^{2} E \left[C_{k}^{2}\right] E \left[\cos ^{2}\left(\omega_{k} t+\Phi_{k}+\phi_{k}\right)\right] \\&=\left|H\left(\omega_{k}\right)\right|^{2}\left(2 S_{X}\left(\omega_{k}\right) \Delta \omega\right)\left(\frac{1}{2}\right) \\&=\left|H\left(\omega_{k}\right)\right|^{2} S_{X}\left(\omega_{k}\right) \Delta \omega\end{aligned}\)

 

Generalizing this to any input frequency leads to one of the most important results in random vibration theory, namely that the response spectrum is a simple function of the input spectrum,

 

\(S_{Y}(\omega)=|H(\omega)|^{2} S_{X}(\omega)\)
