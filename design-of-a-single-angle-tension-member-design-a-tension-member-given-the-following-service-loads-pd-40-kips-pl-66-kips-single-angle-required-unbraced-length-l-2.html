Guest Question.

Design of a Single-Angle Tension Member Design a tension member given the following:

• Service loads: \(P_{D} = 40 kips\), \(P_{L} = 66 kips;\)

• Single angle required;

• Unbraced length, \(L =20 ft.;\)

• ASTM \(A36\) steel; and

• Two lines of four \(3⁄4-in\).-diameter bolts.

Admin
Solution
\(\begin{array}{l} P_{u}=1.4 P_{D}=(1.4)(40)=56 \mathrm{kips} \\ P_{u}=1.2 P_{D L}+1.6 P_{L L}=[(1.2)(40)]+[(1.6)(66)]=154 \mathrm{kips} \end{array}\)

 

\(\text { 1. } A_{g} \geq \frac{P_{u}}{0.9 F_{y}}\) (assume slenderness ratio \(<L/300\))

\(A_{g} \geq \frac{(154)}{(0.9)(36)}=4.75 \text { in. }^{2}\)

 

2. Shear lag factor, \(U\), is \(0.80\) for single angles (from Table 4-1). Alternatively, \(U\) may be calculated using \(\ell=(3)(3 \text { in. })=9 \text { in. }\) (three spaces at \(3 in\).), but an angle size would have to be assumed.

\(\begin{aligned} A_{g} & \geq \frac{P_{u}}{0.75 F_{u} U}+\Sigma A_{\text {holes }} \\ A_{g} & \geq \frac{(154)}{(0.75)(58)(0.80)}+(2 \text { holes })\left(\frac{3}{4}+\frac{1}{8}\right)(t) \end{aligned}\)

 

\(A_{g \text { required }}=4.43+1.75 t\) where \(t\) is the thickness of the angle

\(r_{\min }=\frac{L}{300}=\frac{(20)(12)}{300}=0.80 \text { in. }\)

3. Select \(L8× 4 × 1⁄2\) because of its lighter weight; also, it would have greater block shear capacity than the \(L6 × 6 × 1/2\) (same weight).

4. Slenderness ratio checked in step 2.

5. Check block shear capacity. The spacing of the bolts will have to be assumed. See Figure 4-19 for the assumed bolt layout

Mode 1 Block Shear:

\(\begin{aligned} A_{g v} &=(2)(10.5)(0.5)=10.5 \mathrm{in} .^{2} \\ A_{n v} &=A_{g v}-\Sigma A_{\text {holes }} \\ &=10.5-(2)(3.5 \text { holes })\left(\frac{3}{4}+\frac{1}{8}\right)(0.5)=7.43 \mathrm{in}^{2} \\ A_{g t} &=(3)(1 / 2)=1.5 \mathrm{in}^{2} \\ A_{n t} &=A_{g t}-\Sigma A_{\mathrm{holes}} \\ &=1.5-(2)(0.5 \mathrm{holes})\left(\frac{3}{4}+\frac{1}{8}\right)(0.5)=1.06 \mathrm{in} .^{2} \end{aligned}\).

\(\begin{aligned} \phi P_{n}=& \phi\left(0.60 F_{u} A_{n v}+U_{b s} F_{u} A_{n t}\right) \leq \phi\left(0.60 F_{y} A_{g v}+U_{b s} F_{u} A_{n t}\right) \\ \phi P_{n}=& 0.75[(0.60)(58)(7.43)+(1.0)(58)(1.06)] \\ \leq & 0.75[(0.60)(36)(10.5)+(1.0)(58)(1.06)] \\ & 240 \text { kips }>216 \text { kips } \end{aligned}\).

\(\phi P_{n}=216 kips\) (mode 1 block shear capacity) \(> T_{u}= 154 kips\) \(OK\)

Mode 2 Block Shear:

\(\begin{array}{l} A_{g v}=(10.5)(0.5)=5.25 \mathrm{in} .^{2} \\ A_{n v}=A_{g v}-\Sigma A_{\mathrm{holes}} \end{array}\).

\(\begin{aligned} &=5.25-(3.5 \text { holes })\left(\frac{3}{4}+\frac{1}{8}\right)(0.5)=3.72 \mathrm{in} .^{2} \\ A_{g t} &=(3+2.5)(1 / 2)=2.75 \mathrm{in} .^{2} \\ A_{n t} &=A_{g t}-\Sigma A_{\mathrm{holes}} \\ &=2.75-(1.5 \text { holes })\left(\frac{3}{4}+\frac{1}{8}\right)(0.5)=2.09 \mathrm{in} .^{2} \end{aligned}\).

\(\begin{aligned} \phi P_{n}=& \phi\left(0.60 F_{u} A_{n v}+U_{b s} F_{u} A_{n t}\right) \leq \phi\left(0.60 F_{y} A_{g v}+U_{b s} F_{u} A_{n t}\right) \\ \phi P_{n}=& 0.75[(0.60)(58)(3.72)+(1.0)(58)(2.09)] \\ \leq & 0.75[(0.60)(36)(5.25)+(1.0)(58)(2.09)] \\ & 188 \text { kips }>175 \text { kips } \end{aligned}\).

\(\phi P_{n}=175 kips\) (mode 2 block shear capacity) \(> Tu = 154 kips\) \(OK\)

Select an \(L8× 4 × 1⁄2\) with two lines of four \(3⁄4-in\).-diameter bolts
