Guest Question.

Compute the z-transform and the region of convergence for each of the following functions: \(x[n]=-(\frac{5}{4} )^{n}u[-n-1] \) and \(y[n]=-(\frac{5}{4} )^{n-3} u[-n+2].\)

Admin
Solution

The function y[n] arises from the time shift of x[n] by 3 time units to the right, y[n] = x[n – 3], hence, as shown in the figures of Table 3.4, there are some positive values of the variable n for which y[n] ≠ 0. The function y[n] is non-causal.

From the transform of (5), given in the Transform Lookup Table 3.1, it is true that \(X(z)=\frac{z}{z-\frac{5}{4} }\) with a region of convergence \(|z|<\frac{5}{4}\) .
The transform of y[n] can be easily computed by using the time-shift property (Property 2 – Table 3.2). The function is given as

\(Y(z)=\frac{z^{-2}}{z-\frac{5}{4} }=\frac{1}{z^{2}(z-\frac{5}{4})}\) (1)

The poles and regions of convergence of X(z) and Y(z) functions are presented in the following Table 3.4.

The computation of the z-transform of the function \(y[n]=-(\frac{5}{4} )^{n-3}u[-n+2]\) without using the time-shift property is given next.
Based on the definition of the z-transform, we get:

\(Y(z)=\sum\limits_{n=-\infty }^{n=+\infty }{y[n]z^{-n}}==- \sum\limits_{n=-\infty }^{n=+\infty }{(\frac{5}{4} )^{n-3} u[-n+2]z^{-n}}=-\sum\limits_{n=-\infty }^{n=2}{(\frac{5}{4} )^{n-3}z^{-n}}\)

\(=-(\frac{5}{4} )^{-3}\sum\limits_{n=-\infty }^{n=2}{(\frac{5}{4} )^{n}z^{-n}}=-(\frac{5}{4} )^{-3}\sum\limits_{n=-\infty }^{n=2}{(\frac{5}{4}z^{-1} )^{n}}\)
0r \(Y(z)=\sum\limits_{n=-\infty }^{n=+\infty }{y[n]z^{-n}}=-(\frac{5}{4} )^{-3}\sum\limits_{n=-\infty }^{n=2}{(\frac{5}{4}z^{-1} )^{n}}\) (2)
The sum \(\sum\limits_{n=-\infty }^{n=2}{(\frac{5}{4}z^{-1} )^{n}}\) is written as \(\sum\limits_{n=-\infty }^{n=2}{(\frac{5}{4}z^{-1} )^{n}}\)
\(=…+(\frac{5}{4}z^{-1} )^{-k}+…(\frac{5}{4}z^{-1} )^{-2}+(\frac{5}{4}z^{-1} )^{-1}+(\frac{5}{4}z^{-1} )^{0}+(\frac{5}{4}z^{-1} )^{1}+(\frac{5}{4}z^{-1} )^{2}\) (3)

A more sophisticated way to build in the relation is to set \((\frac{5}{4}z^{-1} )^{-1}=W.\)

Then, \(\sum\limits_{n=-\infty }^{n=2}{(\frac{5}{4}z^{-1})^{n}}=…+W^{k}+…W^{2}+W^{1}+W^{0}+W^{-1}+W^{-2} or \sum\limits_{n=-2}^{n=\infty }{W^{n}}=….+ W^{k}+…W^{2}+W^{1}+W^{0}+W^{-1}+W^{-2}\) or \(\sum\limits_{n=-2}^{n=\infty }{W^{n}}=W^{-2}+W^{-1}+W^{0}+W^{1}+W^{2}+…+W^{k}+…\)

which is a series with the first term \(W^{-2}=(\frac{5}{4}z^{-1} )^{2}\) and ratio \(W=\frac{4}{5}z.\)

And for decreasing geometric progression:|W|<1 or \(|\frac{4}{5}z|<1\) or \(|z|<\frac{5}{4}\) .

Substituting the first term \(a_{1}\) for \(W^{-2}=(\frac{5}{4}z^{-1} )^{2}\) and then \(W=\frac{4}{5}\) into \(\frac{a_{1}}{1-W}\) yields

\(\frac{a_{1}}{1-W} =\frac{(\frac{5}{4}z^{-1} )^{2}}{1-\frac{4}{5}z }=-\frac{(\frac{5}{4}z^{-1} )^{2}}{\frac{4}{5}z-1 }=-\frac{5}{4}\frac{(\frac{5}{4}z^{-1} )^{2}}{z-\frac{5}{4}}=-(\frac{5}{4})^{3}\frac{z^{-2}}{z-\frac{5}{4}}\) (4)

Then, from relation (2), Y(z) is computed:

\(Y(z)=\sum\limits_{n=-\infty }^{n=+\infty }{y[n]z^{-n}}=-(\frac{5}{4})^{-3} \sum\limits_{n=-\infty }^{n=2}{(\frac{5}{4}z^{-1} )^{n}}=\frac{z^{-2}}{z-\frac{5}{4}} =\frac{1}{z^{2}(z-\frac{5}{4})}\) (5)
