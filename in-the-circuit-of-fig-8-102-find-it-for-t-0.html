Guest Question.

In the circuit of Fig. 8.102, find i(t) for t > 0

Admin
Solution

For t < 0, i(0) = 0 and v(0) = 0.

For t > 0, the circuit is as shown below.

Applying KVL to the larger loop ,

\(-20+6 \mathrm{i}_{0}+0.25 \mathrm{di}_{0} / \mathrm{dt}+25 \int\left(\mathrm{i}_{\mathrm{o}}+\mathrm{i}\right) \mathrm{dt}=0 (1)\)

For the smaller loop, \(4 \mathrm{i}+25 \int\left(\mathrm{i}+\mathrm{i}_{\mathrm{o}}\right) \mathrm{dt}=0\) or \(\int\left(\mathrm{i}+\mathrm{i}_{\mathrm{o}}\right) \mathrm{dt}=-0.16 \mathrm{i} (2)\)

Taking the derivative, \(4 \mathrm{di} / \mathrm{dt}+25\left(\mathrm{i}+\mathrm{i}_{\mathrm{o}}\right)=0 \text { or } \mathrm{i}_{\mathrm{o}}=-0.16 \mathrm{di} / \mathrm{dt}-\mathrm{i} (3)\)

and \(\mathrm{di}_{0} / \mathrm{dt}=-0.16 \mathrm{d}^{2} \mathrm{i} / \mathrm{dt}^{2}-\mathrm{di} / \mathrm{dt} (4)\)

From (1),(2),(3), and (4), \(\quad-20-0.96 \mathrm{di} / \mathrm{dt}-6 \mathrm{i}-0.04 \mathrm{d}^{2} \mathrm{i} / \mathrm{dt}^{2}-0.25 \mathrm{di} / \mathrm{dt}-4 \mathrm{i}=0\)
Which becomes, \(\mathrm{d}^{2} \mathrm{i} / \mathrm{dt}^{2}+30.25 \mathrm{di} / \mathrm{dt}+250 \mathrm{i}=-500\)
This leads to, \(\mathrm{s}^{2}+30.25 \mathrm{s}+250=0\)

\(\text { or } \mathrm{s}_{1,2}=\frac{-30.25 \pm \sqrt{(30.25)^{2}-1000}}{2}=-15.125 \pm \mathrm{j} 4.608\)

This is clearly an underdamped response.

Thus, \(\mathrm{i}(\mathrm{t})=\mathrm{I}_{\mathrm{s}}+\mathrm{e}^{-15.125 \mathrm{t}}\left(\mathrm{A}_{1} \cos (4.608 \mathrm{t})+\mathrm{A}_{2} \sin (4.608 \mathrm{t})\right) \mathrm{A}\)

At t=0, \(i_{0}(0)=0\) and \(i(0)=0=I_{s}+A_{1}\) or \(A_{1}=-I_{s} .\) As t approaches infinity \(, i_{0}(\infty)= 20 / 10=2 \mathrm{A}=-\mathrm{i}(\infty)\) or \(\mathrm{i}(\infty)=-2 \mathrm{A}=\mathrm{I}_{\mathrm{s}}\) and \(\mathrm{A}_{1}=2\)

In addition, from (3), we \(\operatorname{get} \operatorname{di}(0) / \mathrm{dt}=-6.25 \mathrm{i}_{0}(0)-6.25 \mathrm{i}(0)=0\)

\(\mathrm{di} / \mathrm{dt}=0-15.125 \mathrm{e}^{-15.125 \mathrm{t}}\left(\mathrm{A}_{1} \cos (4.608 \mathrm{t})+\mathrm{A}_{2} \sin (4.608 \mathrm{t})\right)+\mathrm{e}^{-15.125 \mathrm{t}}\left(-\mathrm{A}_{1} 4.608 \sin (4.608 \mathrm{t})\right.\left.+\mathrm{A}_{2} 4.608 \cos (4.608 \mathrm{t})\right) .\) At \(\mathrm{t}=0, \operatorname{di}(0) / \mathrm{dt}=0=-15.125 \mathrm{A}_{1}+4.608 \mathrm{A}_{2}=-30.25+4.608 \mathrm{A}_{2}\) or \(\mathrm{A}_{2}=30.25 / 4.608=6.565\).

This leads to,
\(\mathrm{i}(\mathrm{t})={\left(-2+\mathrm{e}^{-15.125 t}\right.}(2 \cos (4.608 \mathrm{t})+6.565 \sin (4.608 \mathrm{t}))) \mathrm{A}\)
