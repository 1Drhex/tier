Guest Question.

A large sheet of stainless steel is removed from an annealing bath and allowed to cool in air (see Fig. 3.8). If the initial temperature of the sheet is 500°C, determine how long it will take for the center of the sheet to cool to 250°C. The sheet is 2 cm thick and has a density of 8500 kg/\(m^3\), a specific heat of 460 J/kg K, and a thermal conductivity of 20 W/m K. The heat transfer coefficient to the air is 80 W/\(m^2\) K, and the ambient air temperature is 20°C. Use an explicit method and compare your results with those from a chart solution.

Admin
Solution

The transient cooling of stainless steel sheet can be modeled as a semi-infinite slab (one dimension in space) problem, because the thickness of the sheet is much smaller than its width and length. Also, before carrying out the numerical analysis it is instructive first to find the chart solution. The Biot number is

\(Bi = \frac{\overline{h} L}{k}= \frac{\left(80\frac{W}{m ^{2}K } \right)(0.01 m) }{\left(20\frac{W}{mK} \right) } = 0.04\)

 

or 1/Bi = 25. Note that Bi < 0.1 and hence the sheet can be treated as a lumped capacitance. For the ordinate in Fig. 2.42(a) we have

\(\frac{T(0,t) – T_{\infty } }{T(x,0) – T_{\infty }}=\frac{250 -20}{500 -20}= 0.479\)

 

which gives

\(\frac{\alpha t}{L^2} = 19\)

 

from which

\( t = \frac{(19)(0.01 m^2)}{\left(20 \frac{W}{mK} \right) }\left(8500\frac{kg}{m^3} \right)\left(460 \frac{ W s}{kg K} \right) = 371 s \)

 

To implement the explicit numerical method, consider half the sheet thickness, with x = 0 being the exposed left face and x =L = 0.01 m being the sheet centerline. Eq. (3.14) can be used for all interior nodes:

\(T_{i,m+1} = T_{i,m} +\frac{\Delta t }{\rho c\Delta x}\left\{\frac{k}{\Delta x} (T_{i+1,m} – 2T_{i,m} +T_{i-1,m}) + \dot{q}_{G,i,m} \Delta x\right\} \) (3.14)

\(T_{i,m+1} = T_{i,m} +\frac{\Delta t\alpha }{\Delta x^2}(T_{i+1,m} – 2T_{i,m} +T_{i-1,m})\)

 

For the left face Eq. (3.18) simplifies to

\(T^{\prime }_{1,m+1} = T_{1,m} + \frac{2\Delta t}{\rho c\Delta x}\left\{\overline{h}(T_{\infty } -T_{1,m} ) + \dot{q}_{G,i,m} \frac{\Delta x}{2} +k\frac{T_{2,m} -T_{1,m}}{\Delta x }\right\} \) (3.18)

\(T_{1,m+1} = T_{1,m} + \frac{2\Delta t}{\rho c\Delta x}\left(\overline{h}(T_{\infty } -T_{1,m} ) +k\frac{T_{2,m} -T_{1,m}}{\Delta x } \right)\)

 

Since the right boundary corresponds to the plane of symmetry, there can be no heat flow across the x = L plane, and Eq. (3.19) therefore simplifies to

\(T_{N,m+1}= T_{N,m}+ \frac{2\pmb{\Delta}t }{\rho c\Delta x}\left\{q^{\prime \prime }_{N,m}+ \frac{\Delta x}{2} \dot{q}_{G,N,m}+ \frac{k}{\Delta x} (T_{N-1,m} – T_{N,m} )\right\} \) (3.19)

\(T_{N,m+1}= T_{N,m}+ \frac{2\pmb{\Delta}t\alpha }{\Delta x^2}(T_{N-1,m} – T_{N,m} )\)

 

To simplify the notation, let us define

\(C_1 =\frac{\alpha{\Delta}t }{\Delta x^2}\) \(C_{2}= \frac{2\overline{h }{\Delta}t }{pc\Delta x }\) \(C_3=\frac{2\alpha{\Delta}t }{\Delta x^2}\)

 

Note that \(C_1 = C_3/2 = Fo\).

As before, let \(T_{1,old}\equiv T_{i,m}\) and \(T_{i,new}\equiv T_{i,m+1}\). Then our equations become

\(T_{i,new} = T_{1,old}+C_{2}(T_\infty -T_{1,old})+C_{3}(T_{2,old} -T_{1,old})\) (a)

\(T_{i,new} = T_{i,old}+C_{1}(T_{i+1,old}-2T_{i,old}+T_{i-1,old})\) i = 2, 3, . . . , N – 1 (b)

\(T_{N,new}= T_{N,old} + C_{3}(T_{N-1,old}- T_{N,old} )\) (c)

The initial condition is

\(T_{i,m=0} = 500°C\) for all i

The solution procedure is shown in Fig. 3.9. Using 20 control volumes ( N = 21) gives . Since we are using an explicit scheme, Δx = L/20 = 0.01/20 = 0.0005 m. Since we are using an explicit scheme, we must have

\(\Delta t \lt \frac{\Delta x^2}{2\alpha } =\frac{(0.0005 m)^2}{2\left(20\frac{W}{mK} \right) }\left(8500\frac{kg}{m^3} \right)\left(460\frac{W s}{kg K} \right) =0.0244 s \)

 

Since \(\overline{h}\Delta x /k\ll 1\), the stability criterion for the boundary control volume will give a similar value for \(\Delta t\). To provide a margin, we will use \(\Delta t\) = 0.02 s.

A computer program that carries out the procedure just described gives a result of t = 367.5 s, which is about 1.5% less than the chart solution. Considering that the charts cannot be read precisely, this agreement is quite good. Running the program with half the number of control volumes, ( N = 11) and a time step of 0.08 s gives t = 367.6 s , indicating that the solution for \(\Delta t\)=0.02 s is sufficiently accurate.
