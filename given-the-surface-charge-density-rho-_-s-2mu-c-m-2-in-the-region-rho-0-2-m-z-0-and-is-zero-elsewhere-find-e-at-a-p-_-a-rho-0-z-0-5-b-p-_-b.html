Guest Question.

Given the surface charge density, \({ \rho }_{ s }\) = \(2\mu C/{ m }^{ 2 }\) in the region \(\rho \)< 0.2 m, z = 0, and is zero elsewhere, find E at: (a) \({ P }_{ A } (\rho = 0, z = 0.5)\); (b) \({ P }_{ B } (\rho = 0, z = -0.5)\)

Admin
Solution

a) First we recognize from symmetry that only a z component of E will be present. Considering a general point z on the z axis, we have r = \(z{ a}_{z }\). then with \(\acute { r }\) =\(\rho { a }_{ \rho }\), we obtain \(r – \acute { r }\) = \(z{ a}_{z } – \rho { a }_{ \rho }\). the superposition integral for the z component of E will be :
\({E}_{z ,PA }\) =\( \frac { { \rho }_{ s } }{ 4\pi { \epsilon }_{ 0 } } \int _{ 0 }^{ 2\pi } \int _{ 0 }^{ 0.2 }{ \frac { z\rho d\rho d\phi }{ { ({ \rho }^{ 2 }+{ z }^{ 2 }) }^{ 1.5 } } }\) = \(-\frac { { 2\pi \rho }_{ s } }{ 4\pi { \epsilon }_{ 0 } } z { \left[ \frac { 1 }{ \sqrt { { z }^{ 2 }+{ \rho }^{ 2 } } } \right] }_{ 0 }^{ 0.2 } \)= \(\frac { { \rho }_{ s } }{ 2{ \epsilon }_{ 0 } } z\left[ \frac { 1 }{ \sqrt { { z }^{ 2 } } } -\frac { 1 }{ \sqrt { { z }^{ 2 }+0.4 } } \right]\)
with z = 0.5m, the above evaluates as \({E}_{z ,PA }\) = 8.1 kV/m.
b) With z at -0.5m we evaluate the expression for \({E}_{z }\) to obtain \({E}_{z ,PB }\) = -8.1 kV/m.
