Guest Question.

Reconsider the , but let the tanks have a small amount of heat transfer so the final mixture is at 400 K. Find the final pressure, the heat transfer and the entropy change for the process.

Admin
Solution

C.V. Both tanks. Control mass with mixing and heating of two ideal gases.

\(\begin{aligned}& n _{ Ar }= P _{ A1 } V _{ A } / \overline{ R } T _{ A1 }=\frac{300 \times 1}{8.3145 \times 283.2}=0.1274 kmol \\& n _{ C _{2} H _{6}}= P _{ B 1} V _{ B } / \overline{ R } T _{ B 1}=\frac{200 \times 2}{8.3145 \times 323.2}=0.1489 kmol\end{aligned}\)

Continuity Eq.: \(n _{2}= n _{ Ar }+ n _{ C _{2} H _{6}}=0.2763 kmol\)

Energy Eq.: \(U _{2}- U _{1}= n _{ Ar } \overline{ C }_{ V 0}\left( T _{2}- T _{ A1 }\right)+ n _{ C _{2} H _{6}} \overline{ C }_{ VO }\left( T _{2}- T _{ B 1}\right)={ }_{1} Q _{2}\)

\(\begin{aligned}& P _{2}= n _{2} \overline{ R } T _{2} /\left( V _{ A }+ V _{ B }\right)=0.2763 \times 8.3145 \times 400 / 3= 3 0 6 . 3 kPa \\&{ }_{1} Q _{2}=0.1274 \times 39.948 \times 0.312(400-283.15) \\&\quad+0.1489 \times 30.07 \times 1.49(400-323.15)= 6 9 8 . 3 kJ\end{aligned}\)

 

\(\begin{aligned}&\Delta S _{ SURR }=-{ }_{1} Q _{2} / T _{ SURR } ; \quad \Delta S _{ SYS }= n _{ Ar } \Delta \overline{ S }_{ Ar }+ n _{ C _{2} H _{6}} \Delta \overline{ S }_{ C _{2} H _{6}} \\& y _{ Ar }=0.1274 / 0.2763=0.4611\end{aligned}\)

 

\(\begin{aligned}\Delta \overline{ S }_{ Ar } &=\overline{ C }_{ P Ar } \ln \frac{ T _{2}}{ T _{ A 1}}-\overline{ R } \ln \frac{ y _{ Ar } P _{2}}{ P _{ A 1}} \\&=39.948 \times 0.520 \ln \frac{400}{283.15}-8.3145 \ln \frac{0.4611 \times 306.3}{300} \\&=13.445 kJ / kmol K\end{aligned}\)

 

\(\begin{aligned}\Delta \overline{ S }_{ C _{2} H _{6}} &=\overline{ C }_{ C _{2} H _{6}} \ln \frac{ T _{2}}{ T _{ B 1}}-\overline{ R } \ln \frac{ y _{ C _{2} H _{6}} P _{2}}{ P _{ B 1}} \\&=30.07 \times 1.766 \ln \frac{400}{323.15}-8.3145 \ln \frac{0.5389 \times 306.3}{200} \\&=12.9270 kJ / kmol K\end{aligned}\)

 

Assume the surroundings are at 400 K (it heats the gas)

\(\begin{aligned}\Delta S _{ NET } &= n _{ Ar } \Delta \overline{ S }_{ Ar }+ n _{ C _{2} H _{6}} \Delta \overline{ S }_{ C _{2} H _{6}}+\Delta S _{ SURR } \\&=0.1274 \times 13.445+0.1489 \times 12.9270-698.3 / 400 \\&= 1 . 8 9 2 k J / K\end{aligned}\)
