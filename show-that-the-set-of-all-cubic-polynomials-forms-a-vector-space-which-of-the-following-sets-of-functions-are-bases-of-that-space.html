Guest Question.

Show that the set of all cubic polynomials forms a vector space. Which of the following sets of functions are bases of that space?

Admin
Solution
  1. \(\left \{ 1,x,x^{2},x^{3} \right \}\)

By checking axioms (a)–(h) on p. 10 it is readily shown that all cubics \(ax^{2}+bx^{2}+cx+d\) form a vector space. Note that the space is four dimensional.

(a) All cubics can be written in the form

\(ax^{3}+bx+cx+d\)

and \(\left \{ 1,x,x^{2},x^{3} \right \}\) are a linearly independent set spanning four-dimensional space. Thus, it is an appropriate basis.

2.\(\left \{ 1-x,1+x,1-x^{3},1+x^{3} \right \}\)

(b) No, does not span the required four-dimensional space. Thus a general

cubic cannot be written as a linear combination of

\(\left ( 1-x \right )\), \(\left ( 1+x \right )\),\(\left ( 1-x^{3} \right )\),\(\left ( 1+x^{3} \right )\)

as no term in\( x^{2}\) is present.

(C)Yes as linearly independent set spanning the four-dimensional space

\(a\left ( 1+x \right )+b\left ( 1+x \right )+c\left ( x^{2}-x^{3} \right )+d\left ( x^{2}+x_{3} \right )\) =\(\left ( a+b \right )+\left ( b-a \right )x+\left ( c+a \right )x^{2}+\left ( d-c \right )x^{3}\)=\(\alpha +\beta x+\gamma x^{2}+\delta x^{3}\).

(D) Yes as a linear independent set spanning the four-dimensional space

\(a\left ( x-x^{2} \right )+b\left ( x+x^{2} \right )+c\left ( 1-x^{3} \right )+\left ( d-c \right )x^{3}\) =\(\left ( a+b \right )+\left ( b-a \right )x+\left ( c+d \right )x^{2}+\left ( d-c \right )x^{3}\) =\(\alpha +\beta x+\gamma x^{2}+\delta x^{3}\).

(E) No not linearly independent set as

\(\left ( 4x^{3}+1 \right )\)=\(\left ( 3x^{2}+4x^{3} \right )-\left ( 3x^{2}+2x \right )+\left ( 1+2x \right )\).
