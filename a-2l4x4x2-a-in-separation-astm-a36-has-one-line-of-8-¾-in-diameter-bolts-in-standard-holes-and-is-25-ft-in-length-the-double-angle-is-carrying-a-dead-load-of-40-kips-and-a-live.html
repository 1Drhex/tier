Guest Question.

A \(2 L 4 \times 4 \times 1 / 2(3 / 8- in . \text { separation })\), ASTM A36, has one line of \(\text { (8) } 3 / 4 \text { -in }\). diameter bolts in standard holes and is 25 ft in length. The double angle is carrying a dead load of 40 kips and a live load of 120 kips in tension. Verify the strength by both LRFD and ASD.

Admin
Solution

Material Properties:

\(\begin{array}{llll}2 L 4 \times 4 \times 1 / 2 & \text { ASTM A36 } & F_{y}=36 ksi & F_{u}=58 ksi\end{array}\) Manual Table 2-3

Geometric Properties:

\(\begin{array}{lll}\text { For a single } L 4 \times 4 \times 1 / 2 & A_{g}=3.75 in .{ }^{2} & r_{x}=1.21 in \text { . }\end{array}\)

 

\(r_{y}=1.83 \text { in. } \quad \bar{x}=1.18 \text { in } .\) Manual Table 1-7

Calculate the required tensile strength

LRFD ASD
\(P_{u}=1.2(40.0 kips) + 1.6(120 kips)\)

= 240 kips

 \(P_{a}=40.0 \text { kips }+120 \text { kips }\)

= 160 kips

Calculate the available tensile yield strength

\(P_{n}=F_{y} A_{g}=(36 ksi )(2)\left(3.75 in .^{2}\right)=270 kips\) Eqn. D2-1

LRFD ASD
\(\phi_{t}=0.90\) \(\Omega_{t}=1.67\)
\(\phi_{t} P_{n}=0.90(270 kips )=243 \text { kips }\) \(P_{n} / \Omega_{t}=(270 kips ) / 1.67=162 \text { kips }\)

Section D2

Calculate the available tensile rupture strength

Calculate U

\(U=1-\frac{\bar{x}}{l}=1-\frac{1.18 \text { in. }}{21.0 \text { in. }}=0.944\) Table D3-1 Case 2

 

\(\text { Calculate } A_{n}\)

Section D3.2

\(A_{n}=A_{g}-2\left(d_{h}+1 / 16 \text { in. }\right) t\)

 

\(=2\left(3.75 \operatorname{in} .^{2}\right)-2(13 / 16 in .+1 / 16 in .)(1 / 2 in .)=6.63 in .^{2}\)

 

\(\text { Calculate } A_{e}\)

 

\(A_{e}=A_{n} U=6.63 in .^{2}(0.944)=6.26 in .^{2}\) Eqn. D3-1

 

\(P_{n}=F_{u} A_{e}=(58 ksi )\left(6.26 in .^{2}\right)=363 kips\) Eqn. D2-2

LRFD ASD
\(\phi_{t}=0.75\) \(\Omega_{t}=2.00\)
\(\phi_{t} P_{n}=0.75(363 kips )=272 kips\) \(P_{n} / \Omega_{t}=(363 kips ) / 2.00=182 \text { kips }\)

Section D2

The available strength is governed by the tensile yielding limit state.

LRFD ASD
243 kips > 240 kips o.k. 162 kips > 160 kips o.k.

Check the non-mandatory slenderness limit Section D1

\(L / r=\left(\frac{25.0 ft }{1.21 in }\right)\left(\frac{12.0 in .}{ ft }\right)=248<300\) o.k.
