Solution

(a) Using Equations 16.9,16.10,16.12, and 16.14 , we find the following:

\(k=\frac{2 \pi}{\lambda}=\frac{2 \pi \mathrm{rad}}{40.0 \mathrm{~cm}}=0.157 \mathrm{rad} / \mathrm{cm}\) \(\omega=2 \pi f=2 \pi\left(8.00 \mathrm{~s}^{-1}\right)=50.3 \mathrm{rad} / \mathrm{s} \) \(T=\frac{1}{f}=\frac{1}{8.00 \mathrm{~s}^{-1}}=0.125 \mathrm{~s} \) \(v=\lambda f=(40.0 \mathrm{~cm})\left(8.00 \mathrm{~s}^{-1}\right)=320 \mathrm{~cm} / \mathrm{s}\)

 

(b) Determine the phase constant \(\phi, \) and write a general expression for the wave function.

Because \(A=15.0 \mathrm{~cm}\) and because \(y=15.0 \mathrm{~cm}\) at x=0 and t=0 substitution into Equation 16.15 gives

\(150=(15.0) \sin \phi\)

or \(\sin \phi=1\)

We may take the principal value \(\phi=\pi / 2\) rad (or \(\left.90^{\circ}\right) \). Hence, the wave function is of the form \(y=A \sin \left(k_{x x}-\omega t+\frac{\pi}{2}\right)=A \cos (k x-w t) \)

By inspection, we can see that the wave function must have this form, noting that the casine function has the same shape as the sine function displaced by \(90^{\circ}\). Substituting the values for A, k, and \(\omega\) into this expression, we obtain

\(y=(150 \mathrm{~cm}) \cos (0.157 x-50.3 t) \)
