Walking Vibrations with Open-web Steel Joist Framing
The floor framing system shown in Figure 12-11 is to be used for an office building. The building has full-height partitions. Determine whether the floor is adequate for walking vibrations. Assume lightweight concrete with a density of 110 pcf and a 28-day compressive strength of 3500 psi. The floor loads are as follows:

Floor dead load = 55 psf (includes weight of slab, deck, finishes, mechanical and electrical fixtures, and partitions) Sustained live load = 11 psf (see Table 12-1)

Table 12-1 Recommended sustained live load values

Occupancy Sustained Live Load
Office floors 11 psf
Residential floors 6 psf
Footbridges, gymnasiums, shopping center floors 0 psf
Admin
Solution

Joist Section Properties and Deflection

For a 24K5 open-web steel joist,

d = 24 in.

A = 2.0 \(in.^2\)(1.0 \(in.^2\) per chord)

I = 210 \(in.^4\) (Note: Exact section properties for the joist would be provided by the joist supplier)

Effective slab width, \(b_{eff}\leq 0.4\times\) Beam span = (0.4)(30 ft.) = 12 ft.

≤Beam trib width = 3 ft. = 36 in. → Governs

\(t_{c}\)= Concrete slab thickness above deck ribs = 2.5 in.

\(h_{r}\) =Depth of metal deck ribs = 1 in.

Note: Only concrete above the deck ribs is used to compute the \(I_{comp}\) for the joist (see Figure 12-12)

\(E_{s}\)= 29,000 ksi

\(E_{c} =33w^{1.5} _{c}\sqrt{f′_{c} }\)

 

\(=(33)(110)^{1.5}\sqrt{3500} = 2252\) ksi

\(n= \frac{E_{s}}{1.35E_{c}}\)

 

\(= \frac{29,000}{(1.35)(2252)}= 9.5\)

 

\(b_{e}\)= Transformed effective width of the slab

= \(\frac{b_{eff} }{n}\)

=\(\frac{36 in. }{9.5}= 3.8\) in.

The centroid (see Figure 12-9) is found by summing the moments of areas about the top of the slab. From statics,

\(\overline{y} = \frac{\sum{Ay} }{\sum{A} }\)

 

\(= \frac{\left[(2.5 in.)(3.8 in.)\left(\frac{2.5 in.}{2} \right) \right]+\left[(2.0 in.^{2} )\left(\frac{24 in}{2} +3.5 in.\right) \right] }{[(2.5 in.)(3.8 in.)] + 2.0 in.^2} = 3.73 in.\)

 

The composite moment of inertia is

\(I_{comp}=\sum{(I +Ad^{2} )}\)

 

\(=\left[(3.8 in.)\left(\frac{(2.5)^{3} }{12} \right) \right]+\left[(3.8 in.)(2.5 in.)\left(3.73 in – \frac{2.5 in.}{2} \right)^2 \right]\)

 

\(+ 210 in.^{4} +\left[(2.0 in.^{2} )\left(\frac{24 in.}{2}+ 2.5 in. +3.5 in. – 3.73 in. \right) ^{2} \right]\)

 

\(= 550 in.^4\)

 

Since \(6 < L_j/D = (30 × 12)/24 in. = 15 < 24\), equation (12-10) is used to calculate \(C_r\):

\(C_{r}= 0.9\left(1 – e^{- 0.28\left(\frac{L_{j} }{D} \right) } \right) ^{2.8}\) for \(6 \leq frac{L_j}{D} \leq 24\) (12-10)

\(C_{r}= 0.9\left(1 – e^{- 0.28\left(\frac{L_{j} }{D} \right) } \right) ^{2.8}\)

 

\(= 0.9\left(1 – e^{- 0.28\left(\frac{30\times 12 }{24} \right) } \right) ^{2.8}= 0.862\)

 

\(\gamma =\frac{1}{C_{r} } – 1\)

 

\( =\frac{1}{0.862 } – 1= 0.159\)

 

The effective moment of inertia of the joists is modified as follows for this case:

\(I_{jeff}=\frac{1}{\frac{\gamma }{I_{chords} }+\frac{1}{I_{comp} } }\)

 

\( =\frac{1}{\frac{0.159 }{210 }+\frac{1}{550 } } =388 in.^{4}\)

 

w= (55 psf + 11 psf)(3 ft.) = 198 lb.ft.

= 16.5 lb.in.

\(\Delta _{j} =\frac{5wL^{4} _{j} }{384EI_{j} }\)

 

\(\Delta _{j} =\frac{(5)(16.5)(30 ft.\times 12)^{4} }{(384)(29 \times 10^{6} )(388) } =0.321 in.\)

 

Girder Section Properties and Deflection

From Part 1 of the AISCM for a W21 × 44,

d = 20.7 in.

A = 13.0\( in.^2\)

I = 843 \(in.^4\)

Effective slab width, \(b_{eff}\)≤ 0.4 × Beam span = 0.4 × 21 ft.= 8.4 ft. = 100.8 in. → Governs

≤ Girder tributary width = 30 ft.

\(t_{c}\)= Concrete slab thickness above deck ribs = 2.5 in.

\(h_{r}\)= Depth of metal deck ribs = 1 in.

\(d_{e}= t_{c}+\frac{h_{r} }{2}\)

 

\( = 2.5 in.+\frac{1 in. }{2}= 3\) in.

Note: Concrete within and above the deck ribs is used to compute the \(I_{comp}\) for the girder.

\(b_{e}\)= Transformed effective width of the slab

\( =\frac{b_{eff} }{n}\)

 

\( =\frac{100.8 in. }{9.5} = 10.6\) in. (see Figure 12-13)

The centroid (see Figure 12-13) is found by summing the moments of areas about the top of the slab. From statics,

\(\overline{y}= \frac{\sum{Ay} }{\sum{A} }\)

 

\( = \frac{\left[(3 in.)(10.6 in.)\left(\frac{3 in.}{2} \right) \right]+\left[(13.0 in.^2)\left(\frac{20.7 in.}{2}+ 2.5 in. + 3.5 in \right) \right] }{[(3 in.)(10.6 in.)] + 13.0 in.^2} = 5.78 in.\)

 

The composite moment of inertia is

\(I_{comp}= \sum{(I +Ad^2)}\)

 

\( =\left[(10.6 in.)\left(\frac{(3)^{3} }{12} \right) \right] +\left[(10.6 in.)(3 in.)\left(5.78 in -\frac{3 in.}{2} \right)^2 \right]\)

 

\( + 843 in.^4 +\left[(13.0 in.^2)\left(\frac{20.7 in.}{2 in.}+3.5 in. -5.78 in. \right) ^{2} \right]\)

 

= 2901 \(in.^4\)

The composite moment of inertia is reduced since the joist seats prevent the slab from acting fully compositely with the girder:

\(I_{geff}= I_{nc}+\frac{(I_{c} – I_{nc} )}{4}\)

 

\( = 843+\frac{(2901 – 843 )}{4} = 1357 in.^4\)

 

Girder deflection:

w = (55 psf + 11 psf)(30 ft.) = 1980 lb./ft.

= 165 lb./in.

\(\Delta _{g}= \frac{5wL^{4} _{g} }{384EL_{g} }\)

 

\( = \frac{(5)(165)(21 ft. \times 12)^{4}}{(384)(29 \times 10^6)(1356) } =0.22 in\)

 

Joist effective weight, \(W_j\):

\(W_{j}=\gamma w_{t}B_{j}L_{j}\)

 

Since joists are used, γ = 1.0.

\(D_{s} = \frac{d^{3} _{e} }{n}\)

 

\( = \frac{(3 in.)^{3} }{9.5} = 2.84in.^4/ft.\)

 

\(D_{j} = \frac{I_{j}}{S}\)

 

\( = \frac{388}{3 ft.}=129 in.^4/ft.\)

 

\(C_{j}\)=2.0 (interior joist is being considered)

\(B_{j}= C_{j}\left(\frac{D_{s} }{D_{j} } \right) ^{0.25}\) \(L_j\)≤ 2⁄3 floor width

\(= (2.0)\left(\frac{2.84 }{129 } \right) ^{0.25}\) (30 ft.)≤ 2⁄3 (90 ft.)= 60 ft.

\(W_j\) = (1.0)(55 psf + 11 psf)(23.1 ft.)(30 ft.)= 45,738 lb.

Girder effective weight, \(W_g\):

\(W_{g} = w_{t}B_{g}L_{g}\)

 

Since the span of the girder is less than the width of the beam panel (\(L_g = 21\) ft. < \(B_j = 23.1\) ft.), the girder deflection has to be modified using equation (12-25):

\(\Delta ^′_{g} = \frac{L_{g} }{B_{j} }\Delta _{g}\geq 0.5\Delta _{g}\)

 

\(= \left(\frac{21 ft.}{23.1 ft.}\right) (0.22 in.)\geq (0.5)(0.22 in.)\)

 

= 0.20 in. > 0.11 in.

\(\Delta ^′_{g}\)= 0.20 in.

 

\(D_{g}=\frac{I_g}{L_j}\)

 

\( =\frac{1357}{30 ft.}= 45.2 in.^4/ft.\)

 

\(C_g\) = 1.8 (girders supporting rolled steel beams)

\(B_{g}=C_{g} \left(\frac{D_{j} }{D_{g} } \right)^{0.25} L_{g}\leq\) 2/3 floor length → Ignore since floor length is not given

\( =(1.8) \left(\frac{129 }{45.2 } \right)^{0.25} (21 ft.) \)

 

= 49.1 ft.

\(W_g\) = (55 psf + 11 psf)(49.1 ft.)(21 ft.) = 68,095 lb.

Effective weight of the floor system, W:

From equation (12-13),

\(W=\left(\frac{\Delta _{j} }{\Delta _{j}+\Delta _{g}} \right)W_{b}+\left(\frac{\Delta _{g}}{\Delta _{j}+\Delta _{g}} \right)W_{g}\) (12-13)

\( =\left(\frac{0.321 in. }{0.321 in. + 0.20 in.} \right)(45,738 lb.)+\left(\frac{0.20 in.}{0.321 in. + 0.20 in.} \right)(68,095 lb.)\)

= 54320 lb.

The natural frequency of the floor system is determined from equation (12-4):

\(f_{n}=0.18\sqrt{\frac{g}{\Delta _{j}+\Delta _{g} } }\) (12-4)

\( =0.18\sqrt{\frac{386}{0.321 in. + 0.20 in } }= 4.90 Hz. Since \gt 3 \)Hz, initially OK

From Table 12-2,

Table 12-2 Recommended vibration values for variables in equation (12-12)

Occupancy Constant Force,\( P_{o}\) Damping Ratio,β Acceleration Limit,\(\frac{a_{o} }{g}\times\) 100%
Offices, residences, churches 65 lb. \(0.02 – \times 0.05^{1}\) 0.50%
Shopping malls 65 lb. 0.02 1.50%
Footbridges—Indoor 92 lb. 0.01 1.50%
Footbridges—Outdoor 92 lb. 0.01 5.00%

\(P_o\) = 65 lb., and

β=0.05 (floor with full-height partitions)

The acceleration of the floor system due to walking vibrations is found by using equation (12-12):

\(\frac{a_{p} }{g}=\frac{P_{o}e^{(- 0.35f_{n} )} }{\beta W}\leq \frac{a_{o} }{g}\) (12-12)

\(=\frac{(65)\left(e^{(- 35)(4.90) }\right) }{(0.05)(54320)}= 0.0043\) ,or 0.43%.

From Table 12-2, the recommended acceleration limit is \(a_o\)/g = 0.5%, so the floor is adequate for walking vibrations. Note that if partial-height or no partitions were used, then it can be seen by inspection that the floor would not be adequate for walking vibrations (\(a_p\)/g = 0.72% and 1.07%, respectively).
