Guest Question.

In Figure P17.23, the 12-kg mass is falling at a velocity of v = 1.5 m/s at the instant it is h = 300 mm above the spring and post assembly. The solid bronze post has a length of L = 450 mm, a diameter of 60 mm, and a modulus of elasticity of E = 105 GPa. Compute the maximum stress in the bronze post and the impact factor:
(a) if the spring has a stiffness of k = 5,000 N/mm.
(b) if the spring has a stiffness of k = 500 N/mm.

Admin
Solution

(a) Spring stiffness k = 5,000 N/mm

\(\begin{aligned}&A=\frac{\pi}{4}(60 mm )^{2}=2,827.4334 mm ^{2} \\&W=(12 kg )\left(9.806650 m / s ^{2}\right)=117.6798 N\end{aligned}\)

Conservation of energy:

\(\frac{1}{2} m v^{2}+W\left(h+\Delta_{\max }\right)=\frac{1}{2} k_{\text {spring }} \Delta_{\text {spring }}^{2}+\frac{1}{2} k_{\text {post }} \delta_{\text {post }}^{2}\) (1)

where

\(\Delta_{\max }=\Delta_{\text {spring }}+\delta_{\text {post }}\) (2)

From equilibrium,

\(\begin{gathered}P_{\text {spring }}=P_{\text {post }} \\k_{\text {spring }} \Delta_{\text {spring }}=k_{\text {post }} \delta_{\text {post }}\end{gathered}\)

Note that \(\delta_{\text {post }}\) is the dynamic deformation of the post. The spring constant of the post is:

\(k_{\text {post }}=\frac{A E}{L}=\frac{\left(2,827.4334 mm ^{2}\right)\left(105,000 N / mm ^{2}\right)}{450 mm }=659,734.4573 N / mm\)

therefore, the dynamic deflection \(\Delta_{\text {spring }}\) can be expressed as:

\(\begin{gathered}(5,000 N / mm ) \Delta_{\text {spring }}=(659,734.4573 N / mm ) \delta_{\text {post }} \\\therefore \Delta_{\text {spring }}=\frac{659,734.4573 N / mm }{5,000 N / mm } \delta_{\text {post }}=131.9469 \delta_{\text {post }} (3)\end{gathered}\)

Substitute Equation (3) into Equation (2):

\(\Delta_{\max }=\Delta_{\text {spring }}+\delta_{\text {post }}=131.9469 \delta_{\text {post }}+\delta_{\text {post }}=132.9469 \delta_{\text {post }}\) (4)

Next, substitute Equations (3) and (4) into Equation (1):

\(\frac{1}{2} m v^{2}+W\left(h+132.9469 \delta_{\text {post }}\right)=\frac{1}{2} k_{\text {spring }}\left(131.9469 \delta_{\text {post }}\right)^{2}+\frac{1}{2} k_{\text {post }} \delta_{\text {post }}^{2}\)

Substitute values for \(m, v, W, k_{\text {spring }}, \text { and } k_{\text {post }}:\)

\(\begin{aligned}&\frac{1}{2}(12 kg )(1.5 m / s )^{2}+(117.6798 N )\left(300 mm +132.9469 \delta_{\text {post }}\right) \\&=\frac{1}{2}(5,000 N / mm )\left(131.9469 \delta_{\text {post }}\right)^{2}+\frac{1}{2}(659,734.4573 N / mm ) \delta_{\text {post }}^{2}\end{aligned}\)

Use quadratic formula to solve for \(\delta_{\text {post }} \text { : }\)

\(\delta_{\text {post }}=0.033538 mm\)

The dynamic force in the post can be computed as:

\(F_{\text {post }}=k_{\text {post }} \delta_{\text {post }}=(659,734.4573 N / mm )(0.033538 mm )=22,126.3670 N\)

and thus, the maximum dynamic stress is

\(\sigma_{\max }=\frac{F_{\text {post }}}{A}=\frac{22,126.3670 N }{2,827.4334 mm ^{2}}=7.8256 MPa =7.83 MPa\)

 

(b) Spring stiffness k = 500 N/mm
The dynamic deflection \(\Delta_{\text {spring }}\) can be expressed as:

\(\begin{aligned}(500 N / mm ) \Delta_{\text {spring }} &=(659,734.4573 N / mm ) \delta_{\text {post }} \\& \therefore \Delta_{\text {spring }}=\frac{659,734.4573 N / mm }{500 N / mm } \delta_{\text {post }}=1,319.4689 \delta_{\text {post }} (3)\end{aligned}\)

Substitute Equation (3) into Equation (2):

\(\Delta_{\max }=\Delta_{\text {spring }}+\delta_{\text {post }}=1,319.469 \delta_{\text {post }}+\delta_{\text {post }}=1,320.469 \delta_{\text {post }}\) (4)

Next, substitute Equations (3) and (4) into Equation (1):

\(\frac{1}{2} m v^{2}+W\left(h+1,320.469 \delta_{\text {post }}\right)=\frac{1}{2} k_{\text {spring }}\left(1,320.469 \delta_{\text {post }}\right)^{2}+\frac{1}{2} k_{\text {post }} \delta_{\text {post }}^{2}\)

Substitute values for \(m, v, W, k_{\text {spring }}, \text { and } k_{\text {post }}:\)

\(\begin{aligned}&\frac{1}{2}(12 kg )(1.5 m / s )^{2}+(117.6798 N )\left(300 mm +1,320.469 \delta_{\text {post }}\right) \\&\quad=\frac{1}{2}(5,000 N / mm )\left(1,320.469 \delta_{\text {post }}\right)^{2}+\frac{1}{2}(659,734.4573 N / mm ) \delta_{\text {post }}^{2}\end{aligned}\)

Use quadratic formula to solve for \(\delta_{\text {post }}\):

\(\delta_{\text {post }}=0.010765 mm\)

The dynamic force in the post can be computed as:

\(F_{\text {post }}=k_{\text {post }} \delta_{\text {post }}=(659,734.4573 N / mm )(0.010765 mm )=7,102.0072 N\)

and thus, the maximum dynamic stress is

\(\sigma_{\max }=\frac{F_{\text {post }}}{A}=\frac{7,102.0072 N }{2,827.4334 mm ^{2}}=2.5118 MPa =2.51 MPa\)
