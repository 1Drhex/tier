Guest Question.

A 250 V shunt motor has an armature current of 20 A when running at 1000 rpm against full-load torque. The armature resistance is \( 0.5 \Omega . \) What resistance must be inserted in series with the armature to reduce the speed to 800 rpm at the same torque, and what will be the speed if the load torque is halved with this resistance in the circuit. Assume the flux to remain constant throughout and neglect brush contact drop

 

 

Admin
Solution
\(\begin{array}{l}\text { At normal conditions; } I_{a 1}=20 A ; N_{1}=1000 rpm\\\text { Back emf, } E_{b 1}=V-I_{a 1} R_{a}=250-20 \times 0.5=240 V\end{array}\)

 

\(\begin{array}{l}\text { Let } R \text { be the resistance connected in series with the armature circuit to reduce the speed to } 800 rpm \text { , }\\\text { 1.e., } N_{2}=800 rpm\end{array}\)

 

\(\text { Load torque, } T_{2}=T_{1} \quad \text { (given) }\)

 

\(\text { or } \quad I_{a 2} \phi_{2}=I_{a 1} \phi_{1} \quad\left(\text { since } T \propto \phi I_{\alpha}\right)\)

 

\(\text { or } \quad I_{a 2}=I_{a 1}=20 A \quad \text { (flux } \phi \text { being constant } )\)

 

\(\text { Back emf } E_{b 2}=V-I_{a 2}\left(R_{a}-R\right)=250-20(0.5+R)=240-20 R\)

 

\(\text { Now, } \quad \frac{E_{b 2}}{E_{b 1}}=\frac{N_{2}}{N_{1}}(\phi \text { being constant })\)

 

\(\begin{array}{l}\text { or }\\\frac{240-20 R}{240}=\frac{800}{1000}\end{array}\)

 

or \( R=2.4 \Omega\)

 

When load torque is reduced to half

\(\begin{array}{ll}\text { i.e., } & T_{3}=0.5 T_{1} \text { or } I_{a 3} \phi_{3}=0.5 I_{a 1} \phi_{1} \\\text { or } & I_{a 3}=0.5 I_{a 1}=0.5 \times 20=10 A \text { (since } \phi \text { is constant) }\end{array}\)

 

\(\text { Back emf, } E_{b 3}=V-I_{a 3}\left(R+R_{a}\right)=250-10(2.4+0.5)=221 V\)

 

\(\frac{N_{3}}{N_{1}}=\frac{E_{b 3}}{E_{b 1}}(\phi \text { being constant })\)

 

\(\begin{array}{l}\text { or }\\N_{3}=N_{1} \times \frac{E_{b 3}}{E_{b 1}}=1000 \times \frac{221}{240}=920.8 rpm \end{array}\)
