Guest Question.

Consider operators \( \hat{A} \text { and } \hat{B} \) that do not commute with each other \( (\hat{ C }=[\hat{A}, \hat{B}]) \) but do commute with their commutator: \( [\hat{A}, \hat{C}]=[\hat{B}, \hat{C}] = 0 \), (for instance, \( \hat{x} \text { and } \hat{p} \)).

(a) Show that

\( \left[\hat{A}^{n}, \hat{B}\right]=n \hat{A}^{n-1} \hat{C} \).

Hint: You can prove this by induction on n, using Equation 3.65.

\( [\hat{A} \hat{B}, \hat{C}]=\hat{A}[\hat{B}, \hat{C}]+[\hat{A}, \hat{C}] \hat{B} \) (3.65).

(b) Show that

\( \left[e^{\lambda \hat{A}}, \hat{B}\right]=\lambda e^{\lambda \hat{A}} \hat{C} \),

where λ is any complex number. Hint: Express \( e^{\lambda \hat{A}} \) as a power series.

(c) Derive the Baker–Campbell–Hausdorff formula:

\( e^{\hat{A}+\hat{B}}=e^{\hat{A}} e^{\hat{B}} e^{-\hat{C} / 2} \).

Hint: Define the functions

\( \hat{f}(\lambda)=e^{\lambda(\hat{A}+\hat{B})}, \quad \hat{g}(\lambda)=e^{\lambda \hat{A}} e^{\lambda \hat{B}} e^{-\lambda^{2} \hat{C} / 2} \).

Note that these functions are equal at λ = 0 , and show that they satisfy the same differential equation: \( d \hat{f} / d \lambda=(\hat{A}+\hat{B}) \hat{f} \) and \( d \hat{g} / d \lambda=(\hat{A}+\hat{B}) \hat{g} \). Therefore, the functions are themselves equal for all λ.

Admin
Solution

(a) It’s trivial for n = 1. Assume it is true for (n – 1) (i.e. \( \left[\hat{A}^{n-1}, \hat{B}\right]=(n-1) \hat{A}^{n-2} \hat{C} \) , and prove it then holds for n:

\( \left[\hat{A}^{n}, \hat{B}\right]=\left[\hat{A}\left(\hat{A}^{n-1}\right), \hat{B}\right]=\hat{A}\left[\hat{A}^{n-1}, \hat{B}\right]+[\hat{A}, \hat{B}] \hat{A}^{n-1}=\hat{A}(n-1) \hat{A}^{n-2} \hat{C}+\hat{C} \hat{A}^{n-1}=n \hat{A}^{n-1} \hat{C} \).

(b)

\( \left[e^{\lambda \hat{A}}, \hat{B}\right]=\left[\sum_{n=0}^{\infty} \frac{1}{n !} \lambda^{n} \hat{A}^{n}, \hat{B}\right]=\sum_{n=0}^{\infty} \frac{1}{n !} \lambda^{n}\left[\hat{A}^{n}, \hat{B}\right]=\sum_{n=0}^{\infty} \frac{1}{n !} \lambda^{n} n \hat{A}^{n-1} \hat{C} \)

\( =\lambda \sum_{n=1}^{\infty} \frac{1}{(n-1) !} \lambda^{n-1} \hat{A}^{n-1} \hat{C}=\lambda \sum_{m=0}^{\infty} \frac{1}{m !} \lambda^{m} \hat{A}^{m} \hat{C}=\lambda e^{\lambda \hat{A}} \hat{C} \). QED

(c) Note first that \( \frac{d}{d \lambda}\left(e^{\lambda \hat{Q}}\right)=\frac{d}{d \lambda}\left(1+\lambda \hat{Q}+\frac{1}{2} \lambda^{2} \hat{Q}^{2}+\cdots\right)=\hat{Q}+\lambda \hat{Q}^{2}+\frac{1}{2} \lambda^{2} \hat{Q}^{3}+\cdots=\hat{Q} e^{\lambda \hat{Q}} \), so

\( \frac{d \hat{f}}{d \lambda}=(\hat{A}+\hat{B}) e^{\lambda(\hat{A}+\hat{B})}=(\hat{A}+\hat{B}) \hat{f} \).

\( \frac{d \hat{g}}{d \lambda}=\hat{A} e^{\lambda \hat{A}} e^{\lambda \hat{B}} e^{-\lambda^{2} \hat{C} / 2}+e^{\lambda \hat{A}} \hat{B} e^{\lambda \hat{B}} e^{-\lambda^{2} \hat{C} / 2}+e^{\lambda \hat{A}} e^{\lambda \hat{B}}(-\lambda \hat{C}) e^{-\lambda^{2} \hat{C} / 2} \).

In the second term on the right, we can move \( \hat{B} \) to the left, using (b): \( e^{\lambda \hat{A}} \hat{B}=\hat{B} e^{\lambda \hat{A}}+\lambda e^{\lambda \hat{A}} \hat{C} \). (Of course, \( \hat{C} \) moves freely through the other terms, because it commutes with \( \hat{A} \text { and } \hat{B} \)). Thus

\( \frac{d \hat{g}}{d \lambda}=(\hat{A}+\hat{B}+\lambda \hat{C}-\lambda \hat{C}) e^{\lambda \hat{A}} e^{\lambda \hat{B}} e^{-\lambda^{2} \hat{C} / 2}=(\hat{A}+\hat{B}) \hat{g} \).

Since \( \hat{f}(\lambda) \text { and } \hat{g}(\lambda) \) are equal for all λ, they are equal in particular for λ = 1:

\(e^{\hat{A}+\hat{B}}=e^{\hat{A}} e^{\hat{B}} e^{-\hat{C} / 2} \).

This generalizes the old rule for numbers (as opposed to operators): \( e^{x+y}=e^{x} e^{y} \).
