Guest Question.

Determining the design specifications of a voltage amplifier A voltage amplifier is required to amplify the output signal from a communication receiver that produces a voltage signal of \( v_{ s }=20 mV \) with an internal resistance of \( R_{ s }=1.5 k \Omega \). The load resistance is \( R_{ L }=15 k \Omega \). The desired output voltage is \( v_{ O } \geq 10 V \). The amplifier must not draw more than 1 µA from the receiver. The variation in output voltage when the load is disconnected should be less than 0.5%. Determine the design specifications of the voltage amplifier.

Admin
Solution

Since the input current is \( i_{ s } \leq 1 \mu A \), the input resistance of the amplifier can be found from

\( R_{ s }+R_{ i }=\frac{v_{ s }}{i_{ s }} \geq \frac{20 mV }{1 \mu A }=20 k \Omega \)

which gives

\( R_{ i } \geq 20 k \Omega-R_{ s }=20 k \Omega-1.5 k \Omega=18.5 k \Omega \)

The variation in output voltage, which depends on the ratio \( R_{ o } / R_{ L } \), can be found from

\( \frac{\Delta v_{ o }}{v_{ o }}=\frac{R_{ o }}{R_{ L }+R_{ o }} \) (2.25)

which, for \( \Delta v_{0} / v_{ o } \leq 0.5 \% \text { and } R_{ L }=15 k \Omega \text {, gives } R_{ o } \leq 75 \Omega \). The desired effective voltage gain is \( A_{ v }= \left.v_{ o } / v_{ s } \geq 10 V / 20 mV =500 V / V \text { (or } 53.98 dB \right) \). The open-circuit voltage gain can be found from Eq. (2.21):

\( A_{ v }=\frac{v_{ o }}{v_{ s }}=\frac{v_{ o }}{v_{ i }} \times \frac{v_{ i }}{v_{ s }}=\frac{A_{ vo } R_{ i } R_{ L }}{\left(R_{ i }+R_{ s }\right)\left(R_{ L }+R_{ o }\right)}=\frac{A_{ vo }}{\left(1+R_{ s } / R_{i}\right)\left(1+R_{ o } / R_{ L }\right)} \) (2.21)

 

\( 500 \leq \frac{A_{ vo }}{\left(1+R_{ s } / R_{ i }\right)\left(1+R_{ o } / R_{ L }\right)}=\frac{A_{ vo }}{(1+1.5 k / 18.5 k )(1+75 / 15 k )} \)

 

or \( A_{ vo } \geq 543 V / V \quad(\text { or } 54.7 dB ) \)

The amplifier specifications are \( \left.R_{ i } \geq 18.5 k \Omega, R_{ o } \leq 75 \Omega, \text { and } A_{ vo } \geq 543 V / V \text { (or } 54.7 dB \right) .\)
