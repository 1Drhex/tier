Guest Question.

Consider again and determine the probability that the floor is clean if the sensor in time step k = 2 again detects the floor to be clean.

Admin
Solution

In time step k = 1 of the following situation was considered:
the floor was clean and the sensor also detected the floor to be clean (z1 =
clean). The following conditional probability was calculated:

 

\(P\left(x_{1} \mid z_{1}\right)=\frac{0.8 \cdot 0.4}{0.38}=0.8421\)

In the next time step k = 2 the sensor returns a reading \(z_{2}\) = clean with probability \(P\left(z_{2} \mid x_{2}\right)\) = 0.8 of a correct sensor reading and with probability \(P\left(z_{2} \mid \bar{x}_{2}\right)\) = 0.1 of an incorrect sensor reading (the sensor characteristic was assumed to be time invariant).

Let us first evaluate the predicted probability \(P\left(x_{2} \mid z_{1}\right)\), that is, the floor is dirty based on the previous measurement. Considering Eq. (6.24), where the operation of integration is substituted for the operation of summation, yields

\(p\left(x_{k} \mid z_{1: k-1}\right)=\int p\left(x_{k} \mid x_{k-1}\right) p\left(x_{k-1} \mid z_{1: k-1}\right) d x_{k-1}\)

 

\(P\left(x_{k} \mid z_{1: k-1}\right)=\sum_{x_{k-1} \in X} P\left(x_{k} \mid x_{k-1}\right) P\left(x_{k-1} \mid z_{1: k-1}\right)\)

For now assume that the mobile system only detects the state of the floor, but it cannot influence the state of the floor (i.e., the mobile system does not perform any floor cleaning and also cannot make the floor dirty). Therefore, the state transition probability is simply \(P\left(x_{2} \mid x_{1}\right)\) = 1 and \(P\left(x_{2} \mid \bar{x}_{1}\right)\) = 0. Hence,

 

\(\begin{aligned}P\left(x_{2} \mid z_{1: 1}\right) &=P\left(x_{2} \mid x_{1}\right) P\left(x_{1} \mid z_{1}\right)+P\left(x_{2} \mid \bar{x}_{1}\right) P\left(\bar{x}_{1} \mid z_{1}\right) \\&=1 \cdot 0.8421+0 \cdot 0.1579 \\&=0.8421\end{aligned}\)

 

which is a logical result since if the measurement \(z_{2}\) is not considered, no new information about the system has been obtained. Therefore, the state of the floor has the same probability in the time step k = 2 as it was the state probability in the previous time step k = 1.
The measurement can be fused with the current estimate in the correction step, using relation (6.25):

 

\(p\left(x_{k} \mid z_{1: k}\right)=\frac{p\left(z_{k} \mid x_{k}\right) p\left(x_{k} \mid z_{1: k-1}\right)}{p\left(z_{k} \mid z_{1: k-1}\right)}\)

 

\(\begin{aligned}P\left(x_{2} \mid z_{1: 2}\right) &=\frac{P\left(z_{2} \mid x_{2}\right) P\left(x_{2} \mid z_{1: 1}\right)}{P\left(z_{2} \mid z_{1: 1}\right)} \\&=\frac{0.8 \cdot 0.8421}{P\left(z_{2} \mid z_{1: 1}\right)}\end{aligned}\)

where the value of the normalization factor needs to be calculated. The
missing factor is the probability of a clean floor in time step k = 2, and it can be determined with the summation of all the possible state combinations that result in the current measurement \(z_{k}\), considering the outcomes of the previous measurements \(z_{1: k-1}\):

 

\(P\left(z_{k} \mid z_{1: k-1}\right)=\sum_{x_{k} \in X} P\left(z_{k} \mid x_{k}\right) P\left(x_{k} \mid z_{1: k-1}\right)\)

 

In our case,

 

\(\begin{aligned}P\left(z_{2} \mid z_{1}\right) &=P\left(z_{2} \mid x_{2}\right) P\left(x_{2} \mid z_{1}\right)+P\left(z_{2} \mid \bar{x}_{2}\right) P\left(\bar{x}_{2} \mid z_{1}\right) \\&=0.8 \cdot 0.8421+0.1 \cdot 0.1579 \\&=0.6895\end{aligned}\)

The final result, the probability that the floor is clean if the sensor has
detected the floor to be clean twice in a row, is

 

\(P\left(x_{2} \mid z_{1: 2}\right)=0.9771\)
