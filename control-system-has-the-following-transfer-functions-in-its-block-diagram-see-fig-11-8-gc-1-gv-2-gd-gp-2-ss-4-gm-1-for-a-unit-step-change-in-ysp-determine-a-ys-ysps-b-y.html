Guest Question.

A control system has the following transfer functions in its block diagram (see Fig. 11.8): \(G_{c}=1, G_{v}=2, G_{d}=G_{p}=\) \(\frac{2}{s(s+4)}, G_{m}=1\). For a unit step change in \(Y_{s p}\), determine

(a) \(Y(s) / Y_{s p}(s)\)

(b) \(y(\infty)\)

(c) Offset (note proportional control is being used)

(d) \(y(0.5)\)

(e) if the closed-loop response is oscillatory

Admin
Solution

(a)

From Eq. 11-26 we get the closed loop transfer function for set point changes

\(\frac{Y}{Y_{s p}}=\frac{K_{m} G_{c} G_{v} G_{p}}{1+G_{c} G_{v} G_{p} G_{m}}\) (11-26)

\(\frac{Y}{Y_{s p}}=\frac{K_{m} G_{c} G_{v} G_{p}}{1+G_{c} G_{v} G_{p} G_{m}}\)

Substituting the information from the problem gives

\(\frac{Y}{Y_{s p}}=\frac{\frac{4}{s(s+4)}}{1+\frac{4}{s(s+4)}}=\frac{4}{s(s+4)+4}=\frac{4}{s^{2}+4 s+4}\)

Or in standard form (Eq. 5-40), with \(\tau=\frac{1}{2}\) and \(_{\zeta}=1\)

\(\tau^{2}=\tau_{1} \tau_{2}\) (5-40)

\(\frac{Y}{Y_{s p}}=\frac{1}{\frac{1}{4} s^{2}+s+1}\)

(b)

Given a unit step change in set point we obtain

\(Y(s)=\frac{4}{s\left(s^{2}+4 s+4\right)}\)

Using the Final Value Theorem we get

\(\underset{s\rightarrow 0}{\lim }sY(s)=\frac{4}{s^{2}+4 s+4}=\frac{4}{4}=1\)

Therefore \(y(\infty)=1\)

(c) As the step change is a unit step change, and we have shown that \(y(\infty)=1\), we can say that offset \(=0\). This is consistent with the fact that the gain of the overall transfer function is 1 , so no offset will occur. Normally proportional control does not eliminate offset, but it does for this integrating process.

(d) Using Eq. \(5-50\) or taking the inverse Laplace transform of the response given above we get

\(\text { Time to first peak: }\quad t_{p}=\pi \tau / \sqrt{1-\zeta^{2}}\) (5-50)

\(y(t)=1-(1+2 t) e^{-2 t}\)

Substituting the value of \(0.5\) for \(t\) gives

\(y(t)=0.264\)

(e) We can tell from the response derived above that the response will not be oscillatory, since \(\zeta=1\).
