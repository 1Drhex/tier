Guest Question.

The truss shown in Fig. 6.17 carries the forces \(F_{1} = 10 KN\) and \(F_{2}=20 KN\).
Calculate the forces in all the members.

Admin
Solution

In this example the method of joints can be applied without having to determine the support reactions. We draw the free-body diagrams for each joint, assuming that all the forces in the members are tensile forces.

Beginning with the loaded joint I we write down the equilibrium conditions:

I \(\uparrow : S_{1}\sin 60^{\circ } -F_{1} =0\) ,

\(\rightarrow :S_{2} + S_{1} \cos 60^{\circ } =0\) ,

\(\rightarrow \underline{\underline{S_{1}} } =-\frac{2}{\sqrt{3} }F_{1} =\underline{\underline{11.6KN} } , \underline{\underline{S_{2}} } =-\frac{1}{2} S_{1} =\underline{\underline{-5.8 KN} }\) .

II \(\downarrow : S_{1} \sin 60^{\circ } + S_{3} \sin 60^{\circ } =0\) ,

\(\rightarrow : S_{4} -S_{1} \cos 60^{\circ } + S_{3} \cos 60^{\circ } =0\) ,

\(\rightarrow \underline{\underline{S_{3}} } =-S_{1} =\underline{\underline{-11.6KN} } , \underline{\underline{S_{4}} } = S_{1} =\underline{\underline{11.6KN} }\) .

III \(\uparrow : (S_{3}+ S_{5}) \sin 60^{\circ } -F_{2} =0\) ,

\(\rightarrow :-S_{2}+(S_{5}-S_{3})\cos 60^{\circ } +S_{6}=0\) ,

\(\rightarrow \underline{\underline{S_{5}} } =\frac{2}{\sqrt{3} } (F_{1}+F_{2})=\underline{\underline{34.6KN} } , \underline{\underline{S_{6}} } =\underline{\underline{-28.9 KN} }\) .

IV \(\downarrow : S_{5} \sin 60^{\circ } + S_{7} \sin 60^{\circ } =0\) ,

\(\rightarrow : -S_{4} +(S_{7}-S_{5})\cos 60^{\circ } +S_{8} =0\) ,

\(\rightarrow \underline{\underline{S_{7}} } =-S_{5} =\underline{\underline{-34.6 KN} } , S_{8}=\underline{\underline{46.2KN} }\) .

It is useful to present the results in dimensionless form in a table, including the negative signs:

i 1 2 3 4 5 6 7 8
\(S_{i} /KN\) 11.6 -5.8 -11.6 11.6 34.6 -28.9 -34.6 46.2

As a check we determine the forces in the members 6, 7, 8 with the method of sections:

\(\overset{\curvearrowleft }{IV} : \frac{3}{2} a F_{1} + \frac{a}{2} F_{2} +a \sin 60^{\circ } S_{6} =0 \rightarrow \underline{\underline{S_{6} =-28.9 KN} }\) ,

\(\overset{\curvearrowleft }{V} : 2aF_{1} +a F_{2} -a \sin 60^{\circ } S_{8} =0 \rightarrow \underline{\underline{S_{8}=46.2 KN} }\) ,

\(\downarrow : F_{1} +F_{2} +S_{7} \cos 30^{\circ }=0 \rightarrow \underline{\underline{S_{7} =-34.6 KN} }\).
