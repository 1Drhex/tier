Guest Question.

The electric field of a f = 10 GHz time-harmonic uniform plane wave traveling in a perfect dielectric medium is given by

\(\textbf{E} = (\hat{\textbf{a}}_x + j 2\hat{\textbf{a}}_y) e^{ −j 600\pi z }\)

where z is in meters. Determine, assuming the permeability of the medium is the same as that of free space, the:

(a) Wavelength of the wave (in meters).

(b) Velocity of the wave (in meters/sec).

(c) Dielectric constant (relative permittivity) of the medium (dimensionless).

(d) Intrinsic impedance of the medium (in ohms).

(e) Wave impedance of the medium (in ohms).

(f) Vector magnetic field of the wave.

(g) Polarization of the wave (linear, circular, elliptical; AR; and sense of rotation).

Admin
Solution
\(\underline E = (\hat a_x + j 2\hat a_y) e^{ -j600\pi z } = (\hat a_x + j2\hat a_y) e^{ -j\beta z }\)

(a)

\(\beta = \frac { 2\pi } \lambda = 600\pi \rightarrow \lambda = \frac { 2\pi } { 600\pi } = \frac 1 { 300 } = 3.333 \times 10^{ -3 } \text{ meters}\)

(b)

\(\upsilon = 9f = \frac 1 { 300 } (10 \times 10^9) = 3.333 \times 10^7 \text{ m/sec.}\)

(c)

\(\\ \beta = \omega\sqrt { \mu \varepsilon } = 2\pi f \sqrt{\mu_0 \varepsilon_r \varepsilon_0} = 2\pi f \sqrt{ \mu_0 \varepsilon_0 } \sqrt{ \varepsilon_r } = \frac { 2\pi f } { \upsilon_0 } \sqrt{ \varepsilon_r } = \beta \\ \sqrt{ \varepsilon_r } = \frac { \beta \upsilon_0 } { 2\pi f } = \frac { 600\pi (3\times 10^8) } { 2\pi (10 \times 10^9) } = 9 \\ \varepsilon_r = (9)^2 = 81\)

(d)

\(\eta = \sqrt{ \frac \mu \varepsilon } = \sqrt{ \frac { \mu_0 } { \varepsilon_r \varepsilon_0 } } = \frac 1 { \sqrt{ \varepsilon_r } } \sqrt{ \frac { \mu_0 } { \varepsilon_0 } } = \frac { 377 } { \sqrt { 81 } } = \frac { 377 } 9 = 41.89\)

(e) For a uniform plane wave

\(Z_{ kl } = \eta = \sqrt { \frac \mu \varepsilon } = 41.89\)

(f)

\(\underline H = \frac 1 \eta (\hat a_y – j2 \hat a_x) e^{ -j 600\pi \beta z } = \frac 1 { 41.89 } (-j2\hat a_x + \hat a_y) e^{ -j 600\pi \beta z } \\ \underline H = (-j 47.74 \hat a_x + 23.87 \hat a_y) e^{ -j600\pi z }\)

(using Right Rule between E and H fields)

Alternate: You can also use Maxwell’s equation oto find the H-field.

\(\underline H = +j \frac 1 { \omega \mu } \nabla \times E = j \frac 1 { \omega \mu } \begin{vmatrix} \hat a_x & \hat a_y & \hat a_z \\ \frac \partial { \partial x } & \frac \partial { \partial y } & \frac \partial { \partial z } \\ E_x & E_y & 0 \end{vmatrix} = j \frac 1 { \omega\mu } [\hat a_x (-\frac { \partial E_y } { \partial z }) + \hat a_y (\frac { \partial E_x } { \partial z }) + \hat a_z (0)] \\ = j \frac 1 { \omega\mu } [\hat a_x (-\frac { \partial E_y } { \partial z }) + \hat a_y (\frac { \partial E_x } { \partial z })] = j\frac 1 { \omega\mu } [-\hat a_x (2j)(-j 600\pi\beta) + \hat a_y (-j600\pi)] e^{ -j600\pi z } \\ \underline H = j\frac { (-j\beta) } { \omega \mu } [-\hat a_x (2j) + \hat a_y] e^{ -j600\pi z } = \sqrt{ \frac \varepsilon \mu } (-j2\hat a_x + \hat a_y) e^{ -j600\pi } \\ \underline H = \frac 1 \eta (-j2 \hat a_x + \hat a_y) e^{ -j600\pi } = (-j47.74 \hat a_x + 23.87 \hat a_y) e^{ -j 600 \pi z }\)

which is the same as above (and as it should be).
