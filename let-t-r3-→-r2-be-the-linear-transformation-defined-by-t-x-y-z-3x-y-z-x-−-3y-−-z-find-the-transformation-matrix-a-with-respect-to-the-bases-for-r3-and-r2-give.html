Guest Question.

Let \(T: \mathbb{R}^{3} \rightarrow \mathbb{R}^{2}\) be the linear transformation defined by

\( T\left(\left(\begin{array}{l} x \\ y \\ z \end{array}\right)\right)=\left(\begin{array}{c} 3 x+y+z \\ x-3 y-z \end{array}\right) \)

Find the transformation matrix A with respect to the bases for \(\mathbb{R}^{3}\) and \(\mathbb{R}^{2}\) given by:

\(B=\left\{\mathbf{v}_{1}=\left(\begin{array}{l}1 \\1 \\1\end{array}\right), \mathbf{v}_{2}=\left(\begin{array}{r}-1 \\0 \\1\end{array}\right), \mathbf{v}_{3}=\left(\begin{array}{l}0 \\0 \\1\end{array}\right)\right\} \text { and } C=\left\{\mathbf{w}_{1}=\left(\begin{array}{l}1 \\2\end{array}\right), \mathrm{w}_{2}=\left(\begin{array}{r}-1 \\1\end{array}\right)\right\} \text { respectively }\)

Note that \(B\) is a set of basis (axes) vectors for the start vector space \(\mathbb{R}^{3}\) while \(C\) is a set of basis (axes) vectors for the arrival vector space \(\mathbb{R}^{2}\).

Admin
Solution

Applying the given linear transformation \(T\) to each of the basis (axes) vectors \(B=\left\{\mathbf{v}_{1}, \mathbf{v}_{2}, \mathbf{v}_{3}\right\}\) :

\(T\left(\mathbf{v}_{1}\right)=T\left(\left(\begin{array}{l}1 \\1 \\1\end{array}\right)\right)=\left(\begin{array}{l}3(1)+1+1 \\1-3(1)-1\end{array}\right)=\left(\begin{array}{r}5 \\-3\end{array}\right) \quad\left[\text { because } T\left(\left(\begin{array}{l}x \\y \\z\end{array}\right)\right)=\left(\begin{array}{c}3 x+y+z \\x-3 y-z\end{array}\right)\right]\)

Similarly (verify this) we have

\(T\left(\mathbf{v}_{2}\right)=\left(\begin{array}{l}-2 \\-2\end{array}\right) \text { and } T\left(\mathbf{v}_{3}\right)=\left(\begin{array}{r}1 \\-1\end{array}\right)\)

What else do we need to find?

We need to write each of these arriving vectors, \(T\left(\mathrm{v}_{1}\right), T\left(\mathrm{v}_{2}\right)\) and \(T\left(\mathrm{v}_{3}\right)\), as the coordinates of the basis (axes) \(C=\left\{\mathbf{w}_{1}=\left(\begin{array}{l}1 \\ 2\end{array}\right), \mathbf{w}_{2}=\left(\begin{array}{r}-1 \\ 1\end{array}\right)\right\}:\)

\(\begin{aligned}&T\left(\mathbf{v}_{1}\right)=\left(\begin{array}{r}5 \\-3\end{array}\right)=a \mathrm{w}_{1}+b \mathbf{w}_{2}=a\left(\begin{array}{l}1 \\2\end{array}\right)+b\left(\begin{array}{r}-1 \\1\end{array}\right) (1) \\&T\left(\mathbf{v}_{2}\right)=\left(\begin{array}{r}-2 \\-2\end{array}\right)=c \mathbf{w}_{1}+d \mathbf{w}_{2}=c\left(\begin{array}{l}1 \\2\end{array}\right)+d\left(\begin{array}{r}-1 \\1\end{array}\right) (2) \\&T\left(\mathbf{v}_{3}\right)=\left(\begin{array}{r}1 \\-1\end{array}\right)=e \mathbf{w}_{1}+f \mathbf{w}_{2}=e\left(\begin{array}{l}1 \\2\end{array}\right)+f\left(\begin{array}{r}-1 \\1\end{array}\right) (3)\end{aligned}\)

How can we find the matrix \(A\) ?

By the above Proposition \((5.31)\) we have \(\mathrm{A}=\left(\left[T\left(\mathbf{v}_{1}\right)\right]_{C} \quad\left[T\left(\mathbf{v}_{2}\right)\right]_{C}\left[T\left(\mathbf{v}_{3}\right)\right]_{C}\right)\) which in this case is

\(\mathbf{A}=\left(\begin{array}{lll}a & c & e \\b & d & f\end{array}\right)\)

Why?

Because by \((1),(2)\) and \((3)\) we have

\(\left[T\left(\mathbf{v}_{1}\right)\right]_{C}=\left(\begin{array}{l}a \\b\end{array}\right),\left[T\left(\mathbf{v}_{2}\right)\right]_{C}=\left(\begin{array}{l}c \\d\end{array}\right) \text { and }\left[T\left(\mathbf{v}_{3}\right)\right]_{C}=\left(\begin{array}{l}e \\f\end{array}\right)\)

Remember, the scalars in the linear combination of \(\mathrm{w}_{1}\) and \(\mathrm{w}_{2}\) give the coordinates of vectors \(T\left(\mathrm{v}_{1}\right), T\left(\mathrm{v}_{2}\right)\) and \(T\left(\mathrm{v}_{3}\right)\) with respect to basis (axes) \(C\).

How can we determine the unknowns \(a, b, c, \ldots, f ?\)

We need to solve the three pairs of simultaneous equations ( 1 ), ( 2 ) and (3) in the above. Consider the first pair (1):

\(\left(\begin{array}{r}5 \\-3\end{array}\right)=a\left(\begin{array}{l}1 \\2\end{array}\right)+b\left(\begin{array}{r}-1 \\1\end{array}\right)=\left(\begin{array}{c}a \\2 a\end{array}\right)+\left(\begin{array}{r}-b \\b\end{array}\right)=\left(\begin{array}{c}a-b \\2 a+b\end{array}\right)\)

What is the solution to this pair of simultaneous equations?

\(\left.\begin{array}{r}a-b=5 \\2 a+b=-3\end{array}\right\} \text { gives } a=\frac{2}{3} \text { and } b=-\frac{13}{3}\)

Similarly, we can find the solutions of the other two simultaneous equations. They are

(2) \(\left.\begin{array}{r}c-d=-2 \\ 2 c+d=-2\end{array}\right\}\) gives \(c=-\frac{4}{3}\) and \(d=\frac{2}{3}\)

\(\left.\begin{array}{l}\text { (3) } \quad \begin{array}{r}e-f=1 \\ 2 e+f=-1\end{array}\end{array}\right\}\) gives \(e=0\) and \(f=-1\)

What is the matrix \(A\) equal to?

\(\mathrm{A}=\left(\begin{array}{lll}a & c & e \\b & d & f\end{array}\right)=\left(\begin{array}{rrr}2 / 3 & -4 / 3 & 0 \\-13 / 3 & 2 / 3 & -1\end{array}\right)\)

This means that \([T(\mathbf{u})]_{C}=\mathbf{A}[\mathbf{u}]_{B}\) which is

\([T(\mathbf{u})]_{C}=\left(\begin{array}{rrr}2 / 3 & -4 / 3 & 0 \\-13 / 3 & 2 / 3 & -1\end{array}\right)[\mathbf{u}]_{B}\)

Proposition (5.31). Let \(T: V \rightarrow W\) be a linear transformation and \(V\) and \(W\) be finite-dimensional vector spaces with basis \(B=\left\{\mathbf{v}_{1}, \mathbf{v}_{2}, \ldots, \mathbf{v}_{n}\right\}\) and \(C=\left\{\mathbf{w}_{1}, \mathbf{w}_{2}, \ldots, \mathbf{w}_{m}\right\}\) respectively. Then we can write the transformation as

\([T(\mathbf{u})]_{C}=\mathbf{A}[\mathbf{u}]_{B}\)

where \([T(\mathbf{u})]_{C}\) are the coordinates of \(T(\mathbf{u})\) with respect to the basis (axes) \(C,[\mathbf{u}]_{B}\) are the coordinates of \(\mathbf{u}\) with respect to the basis (axes) \(B\) and \(\mathbf{A}\) is a matrix given by

\(\mathbf{A}=\left(\left[\begin{array}{ll}\left.T\left(\mathbf{v}_{1}\right)\right]_{C} & {\left[T\left(\mathbf{v}_{2}\right)\right]_{C}} & \cdots & \left.\left[T\left(\mathbf{v}_{n}\right)\right]_{C}\right)\end{array}\right.\right.\)
