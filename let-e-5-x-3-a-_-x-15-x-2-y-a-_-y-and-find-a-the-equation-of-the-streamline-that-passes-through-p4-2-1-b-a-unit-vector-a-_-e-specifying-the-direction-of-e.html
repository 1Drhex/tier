Guest Question.

Let E = 5\({ x }^{ 3 }{ a }_{ x } - 15{ x }^{ 2 }y{ a }_{ y }\) , and find : (a) the equation of the streamline that passes through P(4, 2, 1); (b) a unit vector \({ a }_{ E }\) specifying the direction of E at Q(3, -2, 5); (C) a unit vector \({ a }_{ N }\) = (1, m, 0) that is perpendicular to \({ a }_{ E }\) at Q.

Admin
Solution

a) write
\(\frac { dy }{ dx }\) =\(\frac { { E }_{ y } }{ { E }_{ x } } \)=\(\frac { -15{ x }^{ 2 }y }{ 5{ x }^{ 3 } }\) =\( \frac { -3y }{ x }\)
so
\(\frac { dy }{ y }\) = \(-3 \frac { dx }{ x } \Longrightarrow \ln { y }\) =\(-3\ln { x } + \ln { C } \)
thus
y =\( { e }^{ -3\ln { x } }{ e }^{ -3\ln { C } }\) = \(\frac { C }{ { x }^{ 3 } }\)
At P , have 2 = \( c/{ (4) }^{ 3 } \Longrightarrow C \)= 128. finally, at P.
y = \(\frac { 128 }{ { x }^{ 3 } } \)
b) At Q , \({ E }_{ Q }\) =\( 135{ a }_{ x } + 270{ a }_{ y }\) , and \(\left| { E }_{ Q } \right| \) = 301.9. \(thus { a }_{ E }\) = \(0.45{ a }_{ x } + 0.89{ a }_{ y }\).
c) Since this vector is to have no z component, we can find it through \({ a }_{ N } \)= \(\pm ({ a }_{ E }\times { a }_{ z })\). Performing this , we find \({ a }_{ N }\) = \(\pm(0.89{ a }_{ x } – 0.45{ a }_{ y })\).
