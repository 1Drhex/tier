Guest Question.

Consider a sliding-block linkage in the posture illustrated in Fig. 2.23. Perform the posture
analysis; that is, find θ4 and distance \(R_{A O_{4}}\).

Admin
Solution

Using the vector diagram shown in Fig. 2.23 we recognize this as a case 1 problem, and the vector loop equation can be written as

\(\overset{??}{R_{AO_{4}}}= \overset{\sqrt{}\sqrt{}}{R_{O_{2}O_{4}}} + \overset{\sqrt{}I}{R_{AO_{2}}},\) (1)

 

where the “I” symbol, for the direction of vector \(R _{A O_{2}}\) , denotes that the orientation of link 2 is the given input.

(a) Graphic approach When Fig. 2.23 is drawn to a suitable scale (for example, 6 in/in), then, by direct measurement of this figure, we find that

θ4 = 105.3º

\(R_{A O_{4}}=2.08 \text { in }(6 in / in )=12.48 in\)

 

(b) Analytic approach For case 1, we employ Eqs. (2.43) and (2.44); therefore,

 

\(C=\sqrt{A^{2}+B^{2}+2 A B \cos \left(\theta_{B}-\theta_{A}\right)}. \) (2.43)

 

\(\theta_{C}=\tan ^{-1} \frac{A \sin \theta_{A}+B \sin \theta_{B}}{A \cos \theta_{A}+B \cos \theta_{B}},\) (2.44)

 

\(R_{A O_{4}}=\sqrt{R_{O_{2} O_{4}}^{2} R_{A O_{2}}^{2}+2 R_{O_{2} O_{4}} R_{A O_{2}} \cos \left(\theta_{2}-90^{\circ}\right)}\)

 

\(=\sqrt{(9 in )^{2}+(4.5 in )^{2}+2(9 in )(4.5 in ) \cos \left(135^{\circ}-90^{\circ}\right)} = 12.59 in, \)

 

\( \theta_{4}=\tan ^{-1} \frac{R_{O_{2} O_{4}} \sin 90^{\circ}+R_{A O_{2}} \sin \theta_{2}}{R_{O_{2} O_{4}} \cos 90^{\circ}+R_{A O_{2}} \cos \theta_{2}} \)

 

\(=\tan ^{-1} \frac{(9 \text { in }) \sin 90^{\circ}+(4.5 \text { in }) \sin 135^{\circ}}{(9 \text { in }) \cos 90^{\circ}+(4.5 \text { in }) \cos 135^{\circ}}=\tan ^{-1}\left(\frac{12.182 \text { in }}{-3.182 in }\right)=-75.36^{\circ} .\)

 

However, note that the calculator used here did not recognize for the arctangent function that the negative sign in the denominator indicates an angle in the second or third quadrant.
Therefore, the proper value is

 

\(\theta_{4}=-75.36^{\circ} \pm 180^{\circ}=104.64^{\circ}\)

 

(c) Complex-algebraic approach The terms for Eq. (1) are

 

\(R _{A O_{4}}=R_{A O_{4}} \angle \theta_{4}=R_{A O_{4}} \cos \theta_{4}+j R_{A O_{4}} \sin \theta_{4}\)

 

\(R _{O_{2} A O_{4}}=9 \text { in } \angle 90^{\circ}=(9 \text { in }) \cos 90^{\circ}+j(9 \text { in }) \sin 90^{\circ}=0+j 9 \text { in, }\)

 

\(R _{A O_{2}}=4.5 \text { in } \angle 135^{\circ}=(4.5 \text { in }) \cos 135^{\circ}+j(4.5 \text { in }) \sin 135^{\circ}\)

 

\(=-3.182 in +j 3.182 in\)

Substituting these into Eq. (1) yields

 

\(R _{A O_{4}}=(0+j 9 in )+(-3.182 in +j 3.182 in )=-3.182 in +j 12.182 in.\)

 

Thus,

 

\(R_{A O_{4}}=\sqrt{(-3.182 in )^{2}+(12.182 in )^{2}}=12.59 in\)

 

and

\(\theta_{4}=\tan ^{-1} \frac{R_{A O_{4}}^{y}}{R_{A O_{4}}^{x}}=\tan ^{-1} \frac{12.182 in }{-3.182 in }=104.64^{\circ}.\)

Again, note the comment in (b) regarding arctangent results from the calculator.

(d) Vector-algebraic approach Here we employ a scientific hand calculator that will add and subtract complex numbers in rectangular notation and will convert from rectangular to polar notation or vice versa. Thus,

\(R _{O_{2} O_{4}}=9 \text { in }\angle 90^{\circ}=0+j 9 \text { in }\)

 

\(R _{A O_{2}}=4.5 \text { in } \angle 135^{\circ}=-3.182 \text { in }+j 3.182 \text { in, }\)

 

\(R _{A O_{4}}= R _{O_{2} O_{4}}+ R _{A O_{2}}=(0+j 9 \text { in })+(-3.182 \text { in }+j 3.182)\)

 

\(=-3.182 in +j 12.182 in =12.59 in \angle 104.64^{\circ}.\)

 

(e) Numeric approach Some scientific hand calculators permit equations containing complex numbers to be solved in either rectangular or polar form or both, with the results displayed in either mode. Using such a calculator, we enter

 

\(R _{A O_{4}}=9 \text { in } \angle 90^{\circ}+4.5 \text { in } \angle 135^{\circ}=12.59 \text { in } \angle 104.64^{\circ}.\)

 

Note that all algebraic methods give results that are in exact agreement and are more accurate than graphic results obtained manually.
