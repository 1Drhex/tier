Guest Question.

Prove that the first three examples in Table A.7 predict the same solution for x at y = 2 given the diferential equation \(\frac{dx }{dy}= - \frac{2x}{y}\), where x = 2 at y = 1.

TABLE A.7 Solutions to selected diferential equations
Diferential equation Solution
\(\frac{dx}{dy} = \frac{g(y)}{f(x)} \) \(\int{f(x)dx } = \int{g(y)dy } + C \)
\(\frac{dx}{dy} = h(w)\), w = x/y ln y = ∫[h(w)-w]-¹ dw + C, h(w)≠  w
ln x = ln y + C, h(w) = w
\(\frac{dx}{dy} + g_{1}(y)x = g_{2}(y)\) \(x\exp (\int{g_{1} (y)dy })= \int{g_{2} (y) }\exp (\int{g_{1} (y)dy })dy +C\)
\(\frac{d^{2}Y }{dx^{2} }= a^{2}y\) \(y= C_{1}e^{ax}+ C_{2}e^{bx}\)

Solution

Our equation is in the proper form to apply the irst solution in Table A.7, with g(y) = -2/y and f(x) = 1/x. We rewrite “x” in the integrand as a dummy variable x¿ to avoid confusing it with x:

\( \int_{2}^{x}{f(\acute{x} )d\acute{x} }\;= \int_{1}^{2}{g(y )dy }\;\)

 

\( \int_{2}^{x}{\frac{d\acute{x} }{\acute{x} } }\;= -2 \int_{1}^{2}{\frac{dy}{y} }\;\)

 

\( \ln (\frac{x}{2} )= -2 \ln 2\)

 

\( x= \frac{1}{2}\).

To apply the second solution in Table A.7, we would set h(w) = -2w and solve:

\(\ln y\mid ^{2}_{1} = \int_{2}^{x/2}{[h(w)-w]^{-1} dw } \)

 

\(\ln 2 = -\frac{1}{3} \int_{2}^{x/2}{\frac{dw}{w} }\)

 

\(\ln 2 = -\frac{1}{3} \int_{2}^{x/2}{\frac{dw}{w} }\)

 

\(\ln 2 = -\frac{1}{3}\ln(\frac{x}{4} )\)

 

\(\ln(\frac{1}{8} )= \ln(\frac{x}{4} )\)

 

\( x= \frac{1}{2}\).

Note that the limits for the integral over w are the limits of w = x/y: 2/1 at x = 2 and y = 1, and x/2 at y = 2.

For the third solution in Table A.7, set g1(y) = 2/y and g2(y) = 0:

\(x\exp (\int_{1}^{y}{g_{1} (y)dy } )= \int_{1}^{2}{g_{2}(y) \exp (\int_{1}^{2}{g_{1} (y)dy } )}+ C\)

 

\(x\exp (\int_{1}^{y}{\frac{2dy}{y} } )= C\)

 

x exp( ln y) = C

 

xy² = C for x = 2, y = 1 we find C = 2

 

x(2)² = C now set y = 2

 

\(x= \frac{1}{2}\).
