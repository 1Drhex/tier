Guest Question.

The beam shown in Figure 5.11 is a W16 × 31 of A992 steel. It supports a reinforced concrete floor slab that provides continuous lateral support of the compression flange. The service dead load is 450 lbft. This load is superimposed on the beam; it does not include the weight of the beam itself. The service live load is 550 lbft. Does this beam have adequate moment strength?

Admin
Solution

First, determine the nominal flexural strength. Check for compactness.

\(\frac{b_{f} }{2t_{f}}=6.28\) (from Part 1 of the Manual)

\(0.38\sqrt{\frac{E}{F_{y} } } =0.38\sqrt{\frac{29,000}{50} } =9.15\gt 6.28 \) \(\therefore\) The flange is compact.

 

\(\frac{h}{t_{w} } \lt 3.76\sqrt{{\frac{E}{F_{y} } }} \) \(\therefore\) The web is compact.

(The web is compact for all shapes in the Manual for \(F_{y} ≤ 65\) ksi.)

This shape can also be identified as compact because there is no footnote in the dimensions and properties tables to indicate otherwise. Because the beam is compact and laterally supported, the nominal flexural strength is

\(M_{n} = M_{p} = F_{y}Z_{x} = 50(54.0) = 2700\) \(in.-kips\) = 225.0 ft kips.

Compute the maximum bending moment. The total service dead load, including the weight of the beam, is
\(w_{D} = 450 + 31 = 481\) lb/ft
For a simply supported, uniformly loaded beam, the maximum bending moment occurs at midspan and is equal to

\(M_{max }=\frac{1}{8} wL^{2} \)

where w is the load in units of force per unit length, and L is the span length. Then

\(M_{D} =\frac{1}{8} w_{D}L^{2} =\frac{1}{8}(0.481)(30)^{2} =54.11\) ft-kips

\(M_{L}=\frac{1}{8}(0.550)(30)^{2} =61.88\) ft-kips

 

The dead load is less than 8 times the live load, so load combination 2 controls:

\(M_{u} = 1.2M_{D} + 1.6M_{L} = 1.2(54.11) + 1.6(61.88) = 164\) ft-kips.

Alternatively, the loads can be factored at the outset:

\(w_{u} = 1.2w_{D} + 1.6w_{L} = 1.2(0.481) + 1.6(0.550) = 1.457\) kips ft

 

\(M_{u} =\frac{1}{8} w_{u}L^{2} =\frac{1}{8}(1.457)(30)^{2} =164\) ft-kips

The design strength is

\(\phi _{b} M_{n} = 0.90(225.0) = 203\) ft-kips \(> 164\) ft-kips (OK)

 

The design moment is greater than the factored load moment, so the W16 × 31 is satisfactory.

ASD load combination 2 controls.

\(M_{a} = M_{D} + M_{L} = 54.11 + 61.88 = 116.0\) ft-kips

Alternatively, the loads can be added before the moment is computed:

\(w_{a} =w_{D} +w_{L} =0.481+0.550=1.031\) kips/ft

 

\(M_{a}=\frac{1}{8} w_{a}L^{2} =\frac{1}{8}(1.031)(30)^{2} =116.0\) kips/ft

The allowable moment is

\(\frac{M_{n}}{\Omega _{b} } =\frac{M_{n}}{1.67 } =0.6M_{n}=0.6(225.0)=135\) kips-ft\(\gt 116\) kips-ft (OK)

Allowable stress solution:
The applied stress is

\(f_{b} =\frac{M_{a}}{S_{x} } =\frac{116.0(12)}{47.2} =29.5\) ksi

The allowable stress is

\(f_{b} =\frac{0.6M_{n}}{S_{x} } =\frac{0.6(225.0)(12)}{47.2} =34.3\) ksi

Since \(f_{b} < F_{b}\), the beam has enough strength.

The W16 × 31 is satisfactory.
