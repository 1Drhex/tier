Estimating the Project Volatility for a Simple Deferral Option

If a project under consideration is deferred for two years, the required investment would be $35 million, and it would be expected to last three years with the following cash flows in thousand dollars.

\(A_{n}\)

Estimate		 Pessimistic
Estimate		 Period (n)

$30,000 $16,000 $2,000 3
$40,000 $20,000 $4,500 4
$38,000 $18,000 $3,000 5
Admin
Solution

Given: Project cash flow estimates for each period over the project life, risk-free rate = 6%, and T = 2.

Find: \(\mu _{T} , \sigma ^{2}_{T}\) , project volatility (\(\sigma\)), and option premium.

Analytical Approach:

n = 3: Using Eqs. (13.15) and (13.16),

\(E\left[A_{n} \right] = \frac{L+4M_{o}+H }{6}\) (13.15)

and

\(Var\left[A_{n} \right] = \left(\frac{H – L}{6} \right) ^{2}\) (13.16)

we find that

\(E\left[A_{3} \right] = \frac{\$2,000 + 4 \left(\$16,000\right)+ \$30,000 }{6} = \$16,000\)

\(Var \left[A_{3} \right] = \left(\frac{30,000 – 2,000}{6} \right) ^{2} = 21,777,778\)

Similarly, we can calculate the means and variances for other periods as

Estimate		 Most-Likely
Estimate		 Optimistic
Estimate		 \(E\left[A_{n} \right]\) \(Var\left[A_{n} \right]\)

Using Eqs. (12.11) and (12.12)

\(E\left[PW\left(i\right) \right] = \sum\limits_{n = 0}^{N}{\frac{E\left(A_{n} \right) }{\left(1 + i\right) ^{n} } }\) (12.11)

and

\(Var\left[PW\left(i\right) \right] = \sum\limits_{n = 0}^{N}{\frac{Var\left(A_{n} \right) }{\left(1 + i\right) ^{2n} } }\) (12.12)

with the assumption of a statistical \(independence^{7}\) among the cash flows, we compute the project value distribution at period 1 (\(V_{2}\)) at a risk-free discount \(rate^{8}\) of 6% as

\(E\left[V_{2} \right] =\mu _{1} =\frac{\$16,000}{\left(1 + 0.06\right) } +\frac{\$20,750}{\left(1 + 0.06\right) ^{2} } + \frac{\$18,833}{\left(1 + 0.06\right) ^{3}} = \$49,374\)

\(Var\left[V_{2} \right] = \sigma ^{2}_{1} = \frac{21,777,778}{\left(1 + 0.06\right) ^{2} } +\frac{35,006,944}{\left(1 + 0.06\right) ^{4} } + \frac{34,027,778}{\left(1 + 0.06\right) ^{6}} =\left(8,432\right) ^{2}\)

Using Eq. (13.14),

\(\sigma = \sqrt{\frac{\ln \left(\left(\frac{\sigma _{T} }{\mu _{T} } \right) ^{2}+1 \right) }{T} }\) (13.14)

we estimate the project’s volatility (\(\sigma\)) as

\(\sigma = \sqrt{\frac{\ln \left(\left(\frac{\sigma _{T} }{\mu _{T} } \right) ^{2}+1 \right) }{T} }\) = \( \sqrt{\frac{\ln \left(\left(\frac{8,432 }{49,374 } \right) ^{2}+1 \right) }{2} }\)

= 0.1199

Then, we may use 11.99% as the project volatility in real-options analysis.

Simulation Approach: If the cash flow estimates (\(A_{n}\)) are given in terms of individual probability distribution, we may use Crystal Ball to develop the \(V_{2}\) distribution. Once we obtain \(E\left[V_{2} \right]\) and \(Var\left[V_{2} \right]\), the remaining procedure will be the same as before.

Option Value Calculation: Finally, we may calculate the option premium by either binomial lattice (discrete process assumption) or Black and Scholes (continuousprocess assumption).

First, we need to determine the value of \(V_{0}\). Here it is not logical to use the value \(E\left[V_{2} \right]\), as it was based on a risk-free rate. Recall that we are not dealing with a \(V_{0}\) distribution directly—instead we are using a point estimate, ignoring the variance figure. In that case, we need to use a risk-adjusted discount rate to determine the \(V_{0}\) value.

\(V_{0} = \left(\frac{\$16,000}{\left(1 + 0.10\right) }+ \frac{\$20,750}{\left(1 + 0.10\right) ^{2} }+ \frac{\$18,833}{\left(1 + 0.10\right) ^{3}} \right) \left(P/F, 10\%, 2\right)\)

= $37,888

Or, the conventional NPW is

Conventional NPW = \(V_{0}\) – \(I\) = $37,888 – $35,000

= $2,888

Option Value Calculation by Black–Scholes Model: Substituting the estimated parameters into Eq. (13.10),

\(C_{call} = S_{0} N\left(d_{1} \right) – Ke^{-rT} N\left(d_{2} \right)\) (13.10)

we find that the option value is $8,954:

\(V_{0}\) = $37,888, K = $35,000, r = 0.06, T = 2, \(\sigma ^{2} = \left(0.1199\right) ^{2}\) = 0.0144

\(d_{1} =\frac{\ln \left(37,888/35,000\right)+\left(0.06 +^{0.0144} /2 \right)\left(2\right) }{0.1199\sqrt{2} }\)

= 1.26008

\(d_{2} = 1.26008 – 0.1199 \sqrt{2}\)

= 1.090504

\(N\left(d_{1} \right) = 0.896178\)

\(N\left(d_{2} \right) = 0.862254\)

\(C_{call} = 37,888 \left(0.896178\right) – 35,000e^{-0.06\left(2\right) } \left(0.862254\right)\)

= $7,188

  • Option Value Calculation by Binomial Model: If we use the discrete version of the binomial lattice model, we need to create an event tree that models the potential values of the underlying risky asset. This tree contains no decision nodes and simply models the evolution of the underlying asset. Defining T as the number of years per upward movement and \(\sigma ^{2}\) as the annualized volatility of the underlying project value, we can determine the up and down movements with Eq. (13.10):

\(C_{call} = S_{0} N\left(d_{1} \right) – Ke^{-rT} N\left(d_{2} \right)\) (13.10)

Up movement = \(u = e^{\sigma \sqrt{\Delta t} }\)

Down movement = \(d = \frac{1}{u}\)

Substituting numerical values into these formulas with \(\Delta t = 1\) yields

\(u = e^{\sigma \sqrt{\Delta t} } =e^{0.1199\sqrt{1} } = 1.1274\)

\(d = \frac{1}{u} = \frac{1}{1.1274} = 0.8870\)

When we add decision points to an event tree, it becomes a decision tree (Figure 13.36).

To determine the value of the project with the flexibility to defer it, we work backward through the decision tree (Figure 13.37), using the risk-neutral probability approach at each node:

R = 1 + r = 1 + 0.06 = 1.06

\(q = \frac{R – d}{u – d} = \frac{1.06 – 0.8870}{1.1274 – 0.8870} = 0.7196\)

1 – q = 1 – 0.7196 = 0.2804
C = $7,100

Note that this amount ($7,100) is calculated under the binomial model, which is different from the value ($7,188) obtained under the Black–Scholes model earlier. This is not surprising, as the discrete model with T = 2 is a rough approximation of the continuous version. Since the option itself is worth $7,188 million, the value of real-option is

Value of real-option = $7,188 – $2,888 = $4,300

Clearly, it makes sense to defer the investment project for one year in this case.
