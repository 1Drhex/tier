Guest Question.

Evaluate the partial derivatives of the shape functions at \( \zeta=1 / 2, \eta=1 / 2\) ofa quadrilateral element given in , assuming that the temperature is approximated by (a) bilinear and (b) quadratic interpolating polynomials.

 

 

 

 

 

Admin
Solution

(a) Bilinear The shape function derivatives in local coordinates are (Equation (3.91):

\(\begin{array}{l}N_{1}=\frac{(x-b)(y-a)}{(-b-b)(-a-a)}=\frac{1}{4 a b}(b-x)(a-y)=\frac{1}{4}(1-\zeta)(1-\eta) \\N_{2}=\frac{(x-(-b))(y-a)}{(b-(-b))(-a-a)}=\frac{1}{4 a b}(b+x)(a-y)=\frac{1}{4}(1+\zeta)(1-\eta)\\N_{3}=\frac{(x-(-b))(y-(-a))}{(b-(-b))(-a-a)}=\frac{1}{4 ab}(b+x)(a+y)=\frac{1}{4}(1+\zeta)(1+\eta) \\N_{4}=\frac{(x-b)(y-(-a))}{(-b-b)(a-(-a))}=\frac{1}{4 a b}(b-x)(a+y)=\frac{1}{4}(1-\zeta)(1+\eta)\end{array}\)

 

\(\begin{array}{c}\frac{\partial N_{1}}{\partial \zeta}=-\frac{1-\eta}{4} ; \frac{\partial N_{1}}{\partial \eta}=\frac{1-\zeta}{4} \\\frac{\partial N_{2}}{\partial \zeta}=\frac{1-\eta}{4} ; \frac{\partial N_{2}}{\partial \eta}=\frac{1+\zeta}{4} \\\frac{\partial N_{3}}{\partial \zeta}=\frac{1+\eta}{4} ; \frac{\partial N_{3}}{\partial\eta}=\frac{1+\zeta}{4} \\\frac{\partial N_{4}}{\partial \zeta}=-\frac{1+\eta}{4} ; \frac{\partial N_{4}}{\partial\eta}=\frac{1-\zeta}{4}\end{array}\) (3.120)

The Jacobian matrix and its inverse are calculated from Equations (3.113) and (3.116)

\(\left\{\begin{array}{l}\frac{\partial N_{i}}{\partial \zeta} \\\frac{\partial N_{i}}{\partial \eta}\end{array}\right\}=\left[\begin{array}{ll}\frac{\partial x}{\partial \zeta} & \frac{\partial y}{\partial \zeta} \\\frac{\partial x}{\partial \eta} & \frac{\partial y}{\partial \eta}\end{array}\right]\left\{\begin{array}{l}\frac{\partial N_{i}}{\partial x} \\\frac{\partial N_{i}}{\partial y}\end{array}\right\}=[J]\left\{\begin{array}{l}\frac{\partial N_{i}}{\partial x} \\\frac{\partial N_{i}}{\partial y}\end{array}\right\}\)

 

\([ J ]^{-1}=\frac{1}{\operatorname{det}[ J ]}\left[\begin{array}{cc}\frac{\partial y}{\partial \eta} & -\frac{\partial y}{\partial \zeta} \\-\frac{\partial x}{\partial \eta} & \frac{\partial x}{\partial \zeta}\end{array}\right]\)

that is,

\([ J ]=\left[\begin{array}{cc}\sum_{i=1}^{4} \frac{\partial N_{i}}{\partial \zeta} x_{i} & \sum_{i=1}^{4} \frac{\partial N_{i}}{\partial \zeta} y_{i} \\\sum_{i=1}^{4} \frac{\partial N_{i}}{\partial \eta} x_{i} & \sum_{i=1}^{4} \frac{\partial N_{i}}{\partial \eta} y_{i}\end{array}\right]=\frac{1}{8}\left[\begin{array}{cc}20 & 0 \\0 & 20\end{array}\right]\) (3.121)

\(\text { The determinant of the Jacobian matrix is }\)

\(\operatorname{det}[ J ]=\frac{400}{8}\) (3.122)

\(\text { Employing Equation (3.116) }\)

\([ J ]^{-1}=\frac{8}{400}\left[\begin{array}{cc}20 & 0 \\0 & 20\end{array}\right]\) (3.123)

\(\text { Substituting } \zeta=1 / 2 \text { and } \eta=1 / 2 \text { into Equation }(3.120)\)

\(\frac{\partial N_{1}}{\partial \zeta}=-\frac{1}{8} \quad \text { and } \quad \frac{\partial N_{1}}{\partial \eta}=-\frac{1}{8}\) (3.124)

\(\text { Substituting into Equation }(3.115)\)

 

\(\left\{\begin{array}{l}\frac{\partial N_{i}}{\partial x} \\\frac{\partial N_{i}}{\partial y}\end{array}\right\}=[.]^{-1}\left\{\begin{array}{l}\frac{\partial N_{i}}{\partial \zeta} \\\frac{\partial N_{i}}{\partial \eta}\end{array}\right\}\)

 

\(\left\{\begin{array}{l}\frac{\partial N_{1}}{\partial x} \\\frac{\partial N_{1}}{\partial y}\end{array}\right\}=-\frac{1}{20}\left\{\begin{array}{l}1 \\1\end{array}\right\}\) (3.125)

\(\text { In a similar fashion all other nodal derivatives can be calculated. }\)

(b) Quadratic variation The shape function at node I is

\(N_{1}=-\frac{1}{4}(1-\zeta)(1-\eta)(\zeta+\eta+1)\) (3.126)

\(\text { The derivatives with respect to transformed coordinates are }\)

\(\frac{\partial N_{1}}{\partial \zeta}=\frac{3}{16} \quad \text { and } \quad \frac{\partial N_{1}}{\partial \eta}=\frac{3}{16}\) (3.127)

\(\text { The derivatives with respect to global coordinates are }\)

\(\left\{\begin{array}{l}\frac{\partial N_{1}}{\partial x} \\\frac{\partial N_{1}}{\partial y}\end{array}\right\}=\frac{1}{1120}\left\{\begin{array}{l}3 \\6\end{array}\right\}\) (3.128)

\(\text { Other derivatives can be established in a similar manner. }\)
