Guest Question.

A point mass is thrown from a tower (Fig. 1.19a) with an initial velocity \(v_{0}\) at an angle α with respect to the horizontal. It lands at a distance L from the base of the tower.
a) What is the height H of the tower?
b) How long is the mass in the air?
c) What is the speed of the mass at impact?

Admin
Solution

a) To start, let us introduce a coordinate system with origin at the top of the tower (Fig. 1.19b). The coordinates of the impact point are then x = L and z = −H. Substituting into the equation for the trajectory (1.41)
\(\ z\left(x\right)=-\frac{g}{2v^{2}_{0}\cos^{2}\alpha }x^{2}+\tan \alpha · x\).
yields

\(\ \underline{\underline{H=\frac{g}{2v^{2}_{0}\cos^{2}\alpha }L^{2}-L\tan\alpha .} } \)

b) Since the initial and final locations of the mass do not have the same elevation, we can not determine the time of travel from (1.43)
\(\ t_{d}=\frac{x_{d}}{v_{0}\cos \alpha }=2\frac{v_{0}}{g} \sin \alpha . \)
We need to use the parametric representation (1.40)

\(\ \dot{x} =v_{0} \cos \alpha ,\dot{y}=0,\dot{z}=-gt+v_{0}\sin \alpha, \\ x=v_{0} \cos\alpha.t,y=0, z=-g\frac{t^{2}}{2}+v_{0}\sin \alpha · t \)

with x = L and t = T, which gives

\(\ \underline{\underline{T=\frac{L}{v_{0}\cos \alpha } } } . \)

c) The velocity at impact time t = T has, according to (1.40), two components

\(\ \dot{x} =v_{0}\cos \alpha ,\dot{z}=-gT+v_{0} \sin \alpha \)
and thus the magnitude
\(\ \underline{\underline{v} } =\sqrt{\dot{x}^{2}+\dot{z}^{2} } =\sqrt{v^{2}_{0}\cos ^{2}\alpha +\left(v_{0}\sin \alpha -g\frac{L}{v_{0}\cos \alpha } \right)^{2} } \)

\(\ =\sqrt{v^{2}_{0}-2gL\tan \alpha +g^{2}\frac{L^{2}}{v^{2}_{0}\cos^{2}\alpha } } =\underline{\underline{\sqrt{v^{2}_{0}+2gH} } } .\)

The value of the impact speed is seen to be independent of the initial angle α.
