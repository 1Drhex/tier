Guest Question.

The voltage \(v(t)\) across the terminal a and b as shown in the figure, is a sinusoidal voltage having a frequency \( ω = 100 radian/s\). When the inductor current \(i(t)\) is in phase with the voltage \( v(t)\), the magnitude of the impedance \(Z (in Ω)\) seen between the terminals \(a\) and \(b\) is ______ (up to \(2\) decimal places)

Admin
Solution

Consider the circuit given below

at Resonance
\(\quad \quad \quad \quad { Z }_{ ab }={ \left( { Z }_{ ab } \right) }_{ real }\)

Imag \({{ Z }_{ ab }} = 0\)

\(\quad \quad \quad \quad { Z }_{ ab }={ jwL+\left( 100\parallel \frac { 1 }{ SC } \right) }\)

 

\(\quad \quad \quad \quad { Z }_{ ab }={ jwL+\frac { 100\times \frac { 1 }{ SC } }{ 100+\frac { 1 }{ SC } } }\)

 

\(\quad \quad \quad \quad { Z }_{ ab }={ jwL+\frac { 100 }{ 1+100SC } }\)

 

\(\quad \quad \quad \quad \quad\quad ={ jwL+\frac { 100 }{ 1+j100\times 100\times { 10 }^{ -6 }\times 100 } }\)

 

\(\quad \quad \quad \quad { Z }_{ ab }={ jwL+\frac { 100 }{ 1+j1 } }\)

 

\(\quad \quad \quad \quad { Z }_{ ab }={ jwL+\frac { 100\left\{ 1-j \right\} }{ 1+{ 1 }^{ 2 } } }\)

At resonance,

\(\quad \quad \quad \quad { Z }_{ ab }={ Z }_{ real }=\frac { 100 }{ 2 } =50 \Omega \)

Hence, the correct answer is \(50.\)
