Guest Question.

A process, including the sensor and control valve, can be modeled by the transfer function:

\(G(s)=\frac{4 e^{-3 s}}{s}\)

(a) If a proportional-only controller is used, what is the maximum value of controller gain \(K_{c}\) that will result in a stable closed-loop system? (Determine the exact value of \(K_{c}\), not an approximate value.)

(b) Specify a PI controller using the AMIGO controller settings.

Admin
Solution

The process model is,

\(G(s)=\frac{4 e^{-3 s}}{s} \quad\) (assume the time delay has units of minutes) (1)

(a) Proportional only control, \(G_{c}(s)=K_{c}\). The characteristic equation is:

\(1+K_{c} G(s)=0\)

Substitute and rearrange,

\(s+4 K_{c} e^{-3 s}=0\)

Substitute the stability limit conditions from Section 11.4.3: \(s=j \omega, \omega=\omega_{ u }\), and \(K_{c} .=K_{c u}\)

\(j \omega_{u}+4 K_{c u} e^{-3 \omega_{u} j}=0\) (2)

Apply Euler’s identity, \(e ^{-\theta j}=\cos (\theta)-j \sin (\theta):\)

\(e ^{-3 \omega_{u} j}=\cos \left(3 \omega_{u}\right)-j \sin \left(3 \omega_{u}\right)\)

Substitute into (2),

\(j \omega_{u}+4 K_{c u}\left[\cos \left(3 \omega_{u}\right)-j \sin \left(3 \omega_{u}\right)\right]=0\)

Collect terms for the real and imaginary parts:

\(\begin{aligned}&4 K_{c u} \cos \left(3 \omega_{u}\right)=0 (3) \\&\omega_{u}-4 K_{c u} \sin \left(3 \omega_{u}\right)=0 (4)\end{aligned}\)

For (3), because \(K_{c u} \neq 0\), it follows that:

\(\begin{aligned}&\cos \left(3 \omega_{u}\right)=0 \Rightarrow 3 \omega_{u}=\frac{\pi}{2} (5) \\&\Rightarrow \omega_{u}=\frac{\pi}{6}=0.5236 rad / min (6)\end{aligned}\)

From (4) – (6),

\(0.5236-4 K_{c u} \sin \left(\frac{\pi}{2}\right)=0 \Rightarrow K_{c u}=\frac{0.5236}{4}=0.130\)

(b) Controller settings using AMIGO method

The model parameters are: \(K=4, \theta=3\)

For this model, use the right-hand column of Table \(12.5\).

\(\begin{aligned}&K_{c}=\frac{0.35}{K \theta}=\frac{0.35}{12}=0.029 \\&\tau_{I}=13.4 \theta=13.4(3)=40.2\end{aligned}\)

Table 12.5 AMIGO Tuning Rules for PI Controllers (Åström and Hägglund, 2006)∗
\(\text { Model: } G(s)=\frac{K e^{-\theta s}}{\tau s+1}\) \(\text { Model: } G(s)=\frac{K e^{-\theta s}}{s}\)
\(K_{c}=\frac{0.15}{K}+\left(0.35-\frac{\theta \tau}{(\theta+\tau)^{2}}\right) \frac{\tau}{K \theta}\) \(K_{c}=\frac{0.35}{K \theta}\)
\(\tau_{I}=0.35 \theta+\frac{13 \theta \tau^{2}}{\tau^{2}+12 \theta \tau+7 \theta^{2}}\) \(\tau_{I}=13.4 \theta\)
\({ }^{*} \text { Only valid for } \theta>0\)
