Guest Question.

Two steel [E= 30,000 ksi] pipes (1) and (2) are connected at flange B, as shown in Figure P5.31. Pipe (1) has an outside diameter of 6.625 in. and a wall thickness of 0.28 in. Pipe (2) has an outside diameter of 4.00 in. and a wall thickness of 0.226 in. If the normal stress in each steel pipe must be limited to 18 ksi, determine:(a) the maximum downward load Pthat may be applied at flange B.(b) the deflection of flange Bat the load determined in part (a).

Admin
Solution
(a) Pipe section properties: The pipe cross-sectional areas are:
\({ D }_{ 1 }=6.625\quad ㏌\quad \quad \quad \quad \quad \quad \quad { d }_{ 1 }=6.625㏌-2\left( 0.28㏌ \right) =6.0650㏌\)
\({ A }_{ 1 }\frac { \pi }{ 4 } \left[ { \left( 6.625\quad ㏌ \right) }^{ 2 }-{ \left( 6.0625\quad ㏌ \right) }^{ 2 } \right] =5.5814{ ㏌ }^{ 2 }\)
\({ D }_{ 2 }=4.00㏌\quad \quad \quad \quad \quad { d }_{ 2 }=4.00㏌-2\left( 6.625\quad ㏌ \right) =3.5480㏌\)
\({ A }_{ 2 }\frac { \pi }{ 4 } \left[ { \left( 4.00\quad ㏌ \right) }^{ 2 }-{ \left( 3.5480\quad ㏌ \right) }^{ 2 } \right] =2.6795{ ㏌ }^{ 2 }\)
Equilibrium: Consider a FBD of flange B. Sum forces in the vertical direction to obtain:
\(\Sigma { F }_{ y }={ F }_{ 2 }-{ F }_{ 1 }-P=0\)
Geometry of Deformations:
\({ \delta }_{ 1 }+{ \delta }_{ 2 }=0\)
Force-Deformation Relationships:
\({ \delta }_{ 1 }=\frac { { F }_{ 1 }{ L }_{ 1 } }{ { A }_{ 1 }{ E }_{ 1 } } \quad \quad \quad { \delta }_{ 2 }=\frac { { F }_{ 2 }{ L }_{ 2 } }{ { A }_{ 2 }{ E }_{ 2 } } \)
Compatibility Equation: Substitute Eqs. (c) into Eq. (b) to derive the compatibility equation:
\(\frac { { F }_{ 1 }{ L }_{ 1 } }{ { A }_{ 1 }{ E }_{ 1 } } =\frac { { F }_{ 2 }{ L }_{ 2 } }{ { A }_{ 2 }{ E }_{ 2 } } =0\)
For this problem, it is convenient to express the compatibility equation in terms of the normal stress \({ \sigma }_{ 1 }\)and\({ \sigma }_{ 2 }\):
\(\frac { { \sigma }_{ 1 }{ L }_{ 1 } }{ { E }_{ 1 } } +\frac { { \sigma }_{ 2 }{ L }_{ 2 } }{ { E }_{ 2 } } =0\)
Solve the Equations: Solve Eq. (e) for\({ \sigma }_{ 1 }\):
\({ \sigma }_{ 1 }={ \sigma }_{ 2 }\frac { { L }_{ 2 } }{ { L }_{ 1 } } \frac { { E }_{ 1 } }{ { E }_{ 2 } } \)
The elastic moduli of the two pipes are equal; therefore,\({ E }_{ 1 }/{ E }_{ 2 }\quad =1\quad \). The normal stresses in pipes (1) and (2) are limited to 18 ksi. Assume that the normal stress in pipe (2) controls, meaning that\({ \sigma }_{ 2 }\)= 18 ksi. Calculate the corresponding stress in pipe (1):
\({ \sigma }_{ 1 }={ \sigma }_{ 2 }\frac { { L }_{ 2 } }{ { L }_{ 1 } } \frac { { E }_{ 1 } }{ { E }_{ 2 } } =-\left( 18ksi \right) \frac { 16ft }{ 10ft } =-28.80\quad ksi\)
Since this stress magnitude is greater than 18 ksi, our assumption is proven incorrect. We now know that the normal stress in pipe (1) actually controls; thus,\({ \sigma }_{ 1 }\)= −18 ksi (negative by inspection). The normal stress in pipe (1) can be found from Eq. (f)
\({ \sigma }_{ 2 }={ \sigma }_{ 1 }\frac { { L }_{ 1 } }{ { L }_{ 2 } } \frac { { E }_{ 2 } }{ { E }_{ 1 } } =-\left( -18ksi \right) \frac { 10ft }{ 16ft } =-11.250\quad ksi\)
Now that the stresses are known, the allowable forces\({ F }_{ 1 }\)and\({ F }_{ 2 }\)can be computed:
\({ F }_{ 1 }={ \sigma }_{ 1 }{ A }_{ 1 }=\left( -18ksi \right) \left( 5.5814{ ㏌ }^{ 2 } \right) =-100.4652\quad kips\)
\({ F }_{ 2 }={ \sigma }_{ 2 }{ A }_{ 2 }=\left( 11.250ksi \right) \left( 2.6795{ ㏌ }^{ 2 } \right) =-30.1444\quad kips\)
Substitute these values into Eq. (a) to obtain the allowable load P:
\({ P }_{ max }={ F }_{ 2 }-{ F }_{ 1 }=30.1444\quad kips-\left( -100.4652kips \right) =130.6kips\)
(b) The deflection of flange Bis equal to the deformation of pipe (2):
\({ u }_{ B }={ \delta }_{ 2 }=\frac { { \sigma }_{ 2 }{ L }_{ 2 } }{ { E }_{ 2 } } =\frac { \left( 11.250ksi \right) \left( 16ft \right) \left( 12㏌/ft \right) }{ 30,000\quad ksi } =0.0720㏌\downarrow \)
