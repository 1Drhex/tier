Guest Question.

Find the effective area of a Hertzian dipole, and determine the general relation between the directivity and effective area of any antenna.

Admin
Solution

With the Hertzian dipole as the receiving antenna, and having length d, its load voltage, \(V_{L} \) , will depend on the electric field that it intercepts from antenna 1. Specifically, we find the projection of the transmitting antenna field along the length of receiving antenna. This projected field, when multiplied by the length of antenna 2, gives the input voltage to the receiving antenna equivalent circuit. Referring to Figure 14.18, the projection angle is \( \alpha \) , and thus the voltage that drives the Hertzian dipole will be

\(V_{in} = E_{i } \cos \alpha \times d= E_{i } \space d \space \sin \theta_{2} \)

The equivalent circuit for the Hertzian dipole is now the same as that of the receiving antenna as shown in Figure 14.17, except that we replace the source voltage, \(I_{1}Z_{21} \), with \(V_{in} \) as given above. Assuming a conjugate-matched load (\(Z_{L} =Z^{*}_{22} \)), the current through the load is now

\(I_{L}=\frac{E_{i} \space d \space \sin \theta _{2} }{Z_{22}+Z_{L} } =\frac{E_{i} \space d \space \sin \theta _{2} }{2R_{22} } \)

The power delivered to the matched load is then

\(P_{L2}=\frac{1}{2} Re\left\{ V_{L} I^{*}_{L} \right\} =\frac{1}{2}R_{22} |I_{L} |^{2} =\frac{(E_{i}d )^{2} \sin^{2} \theta_{2} } {8R_{22} } \) (99)

For the Hertzian dipole, \(R_{22} \) is the radiation resistance. This was previously found to be (Eq. (30) \(R_{rad} =\frac{2P_{r} }{I^{2}_{0} } =80 \pi^{2} \left( \frac{d}{\lambda } \right) ^{2}\) )

\(R_{22}=R_{rad} =80 \pi^{2} \left( \frac{d}{\lambda } \right) ^{2} \)

Substituting this into (99) \(P_{L2}=\frac{1}{2} Re\left\{ V_{L} I^{*}_{L} \right\} =\frac{1}{2}R_{22} |I_{L} |^{2} =\frac{(E_{i}d )^{2} \sin^{2} \theta_{2} } {8R_{22} }\) , we find

\(P_{L2}=\frac{1}{640} \left( \frac{E_{i} \space \lambda \space \sin \theta_{2} }{\pi } \right) ^{2} \space \space [Watts]\) (100)

The average power density that is incident on the receiving antenna is now

\(S_{r1}(r,\theta_{1},\phi_{1} )=\frac{E_{i}(r,\theta_{1},\phi_{1})^{2} }{2\eta _{0} } =\frac{E^{2}_{i} }{240\pi } \space \space [Watts/m^{2} ]\) (101)

Using (100) \(P_{L2}=\frac{1}{640} \left( \frac{E_{i} \space \lambda \space \sin \theta_{2} }{\pi } \right) ^{2} \space \space [Watts]\) and (101) \(S_{r1}(r,\theta_{1},\phi_{1} )=\frac{E_{i}(r,\theta_{1},\phi_{1})^{2} }{2\eta _{0} } =\frac{E^{2}_{i} }{240\pi } \space \space [Watts/m^{2} ]\) , the effective area of the Hertzian is

\(A_{e2} (\theta_{2} )=\frac{P_{L2} }{S_{r1} } =\frac{3}{8\pi } \lambda^{2} \sin^{2} (\theta_{2} ) \space \space \space [m^{2} ]\) (102)

The directivity for the Hertzian dipole, derived in , is

\(D_{2} (\theta_{2} ) = \frac{3}{2} \sin^{2} (\theta_{2} ) \) (103)

Comparing Eqs. (102) \(A_{e2} (\theta_{2} )=\frac{P_{L2} }{S_{r1} } =\frac{3}{8\pi } \lambda^{2} \sin^{2} (\theta_{2} ) \space \space \space [m^{2} ]\) and (103) \(D_{2} (\theta_{2} ) = \frac{3}{2} \sin^{2} (\theta_{2} ) \) , we find the relation that we are looking for: the effective area and directivity for any antenna are related through

\(D (\theta ,\phi ) =\frac{4\pi }{\lambda ^{2} } A_{e} (\theta ,\phi ) \) (104)
