Guest Question.

A solid aluminum [E= 70 GPa] rod (1) is connected to a solid bronze [E= 100 GPa] rod at flange B, as shown in Fig P5.32. Aluminum rod (1) has an outside diameter of 35mm and bronze rod (2) has an outside diameter of 20 mm. The normal stress in the aluminum rod must be limited to 160 MPa, and the normal stress in the bronze rod must be limited to 110 MPa. Determine:(a) the maximum downward load Pthat may be applied at flange B.(b) the deflection of flange Bat the load determined in part (a).

Admin
Solution
(a) Rod section properties: The rod cross-sectional areas are:
\({ A }_{ 1 }\frac { \pi }{ 4 } ={ \left( 35mm \right) }^{ 2 }=962.1128\quad { mm }^{ 2 }\quad \quad \quad \quad \quad{ A }_{ 2 }\frac { \pi }{ 4 } ={ \left( 20\quad mm \right) }^{ 2 }=314.1593\quad { mm }^{ 2 }\quad \)
Equilibrium: Consider a FBD of flange B. Sum forces in the vertical direction to obtain:
\(\Sigma { F }_{ y }={ F }_{ 2 }-{ F }_{ 1 }-P=0\)
Geometry of Deformations:
\({ \delta }_{ 1 }+{ \delta }_{ 2 }=0\)
Force-Deformation Relationships:
\({ \delta }_{ 1 }=\frac { { F }_{ 1 }{ L }_{ 1 } }{ { A }_{ 1 }{ E }_{ 1 } } \quad \quad \quad { \delta }_{ 2 }=\frac { { F }_{ 2 }{ L }_{ 2 } }{ { A }_{ 2 }{ E }_{ 2 } } \)
Compatibility Equation: Substitute Eqs. (c) into Eq. (b) to derive the compatibility equation:
\(\frac { { F }_{ 1 }{ L }_{ 1 } }{ { A }_{ 1 }{ E }_{ 1 } } =\frac { { F }_{ 2 }{ L }_{ 2 } }{ { A }_{ 2 }{ E }_{ 2 } } =0\)
For this problem, it is convenient to express the compatibility equation in terms of the normal stress\({ \sigma }_{ 1 }\)and\({ \sigma }_{ 2 }\):
\(\frac { { \sigma }_{ 1 }{ L }_{ 1 } }{ { E }_{ 1 } } +\frac { { \sigma }_{ 2 }{ L }_{ 2 } }{ { E }_{ 2 } } =0\)
Solve the Equations:Solve Eq. (e) for\({ \sigma }_{ 1 }\):
\({ \sigma }_{ 1 }={ \sigma }_{ 2 }\frac { { L }_{ 2 } }{ { L }_{ 1 } } \frac { { E }_{ 1 } }{ { E }_{ 2 } } \)
The normal stress in aluminum rod (1) is limited to 160 MPa. We will assume that the aluminum rod controls, and then, we will calculate the corresponding stress in bronze rod (2):
\({ \sigma }_{ 2 }={ \sigma }_{ 1 }\frac { { L }_{ 2 } }{ { L }_{ 1 } } \frac { { E }_{ 1 } }{ { E }_{ 2 } } =-\left( 160MPa \right) \frac { 175mm }{ 340mm } \frac { 100GPa }{ 70GPa } =-117.647MPa\)
This stress magnitude is greater than the 110 MPa allowable stress for the bronze rod; therefore, this calculation shows that the stress in the bronze rod actually controls. If the stress in the bronze rod is limited to 110 MPa, the normal stress in aluminum rod (1) can be found from Eq. (f)
\({ \sigma }_{ 1 }={ \sigma }_{ 2 }\frac { { L }_{ 2 } }{ { L }_{ 1 } } \frac { { E }_{ 1 } }{ { E }_{ 2 } } =-\left( 110MPa \right) \frac { 340mm }{ 175mm } \frac { 70GPa }{ 100\quad GPa } =-149.600\quad MPa\)
Now that the stresses are known, the allowable forces\({ F }_{ 1 }\)and \({ F }_{ 2 }\) can be computed:
\({ F }_{ 1 }={ \sigma }_{ 1 }{ A }_{ 1 }=\left( -149.600\quad MPa \right) \left( 962.1128{ mm }^{ 2 } \right) =-143,932.1N=-143.9321KN\)
\({ F }_{ 2 }={ \sigma }_{ 2 }{ A }_{ 2 }=\left( 110MPa \right) \left( 314.1593{ mm }^{ 2 } \right) =34,557.5N=34.5575KN\)
Substitute these values into Eq. (a) to obtain the allowable load P:
\({ P }_{ max }={ F }_{ 2 }-{ F }_{ 1 }=34.5575KN-\left( -143.9321KN \right) =178.5KN\)
(b) The deflection of flange Bis equal to the deformation of rod (2):
\({ u }_{ B }={ \delta }_{ 2 }=\frac { { \sigma }_{ 2 }{ L }_{ 2 } }{ { E }_{ 2 } } =\frac { \left( 110MPa \right) \left( 340mm \right) }{ 100,000\quad MPa } =0.374mm\downarrow \)
