Guest Question.

Consider the transfer function model in . For an initial condition of y(0) = 4 and a step change in u of magnitude 2 (at t = 0), calculate the response, y(t). Hint: First determine the corresponding differential equation model.

Admin
Solution

From ,

\(\frac{Y(s)}{U(s)}=\frac{3 e^{-s}}{10 s+1}\)

Rearrange,

\(Y(s)[10 s+1]=3 e^{-s} U(s)\) (2)

\(\text { Take } L ^{-1} \text { of }(2) \text {, }\)

\(10 \frac{d y}{d t}+y=3 u(t-1)\) (3)

Take L of (3) for y(0)=4,

\(10[s Y(s)-4]+Y(s)=3 e^{-s} U(s)\)

\(\text { Substitute } U(s)=2 / s \text { and rearrange to give, }\)

\(10 s Y -40+ Y =\frac{6 e^{-s}}{s}\)

\(Y(10 s+1)=\frac{6 e^{-s}}{s}+40\)

Partial fraction expansion:

\(Y(s)=e^{-s} \frac{6}{s(10 s+1)}+\frac{40}{(10 s+1)}\)

\(\frac{6}{s(10 s+1)}=\frac{a_{1}}{s}+\frac{a_{2}}{10 s+1}\)

 

\(\text { Find } \alpha_{1} \text { : Multiply by } s \text { and set } s=0 \Rightarrow \alpha_{1}=6\)

 

\(\text { Find } \alpha_{2}: \text { Multiply by } 10 s+1 \text { and set } s=-0.1 \Rightarrow \alpha_{2}=60\)

 

\(Y(s)=e^{-s}\left(\frac{6}{s}+\frac{6}{s+0.1}\right)+\frac{4}{(s+0.1)}\)

Take \(L_{-1}\),

\(y(t)=6 S(t-1)\left(1+e^{-(t-1) / 10}\right)+4 e^{-t / 10}\)

 

\(\text { Check: At } t=0, \quad y(0)=4 \text {. }\)
