Guest Question.

The Hamiltonian for a certain three-level system is represented by the matrix

\( H=\left(\begin{array}{lll} a & 0 & b \\ 0 & c & 0 \\ b & 0 & a \end{array}\right) \) , where a, b, and c are real numbers.

(a) If the system starts out in the state

\( | S (0)\rangle=\left(\begin{array}{l} 0 \\ 1 \\ 0 \end{array}\right) \)

what is |S(0) 〉 ?

(b) If the system starts out in the state

\( | S (0)\rangle=\left(\begin{array}{l} 1 \\ 0 \\ 0 \end{array}\right) \),

what is |S(t) 〉 ?

Admin
Solution

First find the eigenvalues and eigenvectors of the Hamiltonian. The characteristic equation says

\( \left|\begin{array}{ccc} (a-E) & 0 & b \\ 0 & (c-E) & 0 \\ b & 0 & (a-E) \end{array}\right|=(a-E)(c-E)(a-E)-b^{2}(c-E)=(c-E)\left[(a-E)^{2}-b^{2}\right]=0 \).

Either E = c, or else (a – E)² = b² ⇒ E = a ± b. So the eigenvalues are

\( E_{1}=c, \quad E_{2}=a+b, \quad E_{3}=a-b \).

To find the corresponding eigenvectors, write

\( \left(\begin{array}{lll} a & 0 & b \\ 0 & c & 0 \\ b & 0 & a \end{array}\right)\left(\begin{array}{l} \alpha \\ \beta \\ \gamma \end{array}\right)=E_{n}\left(\begin{array}{l} \alpha \\ \beta \\ \gamma \end{array}\right) \).

(1)

\( \left.\begin{array}{r} a \alpha+b \gamma=c \alpha \Rightarrow(a-c) \alpha+b \gamma=0 \\ c \beta=c \beta \\ (\text { redundant }) \\ b \alpha+a \gamma=c \gamma \Rightarrow(a-c) \gamma+b \alpha=0 \end{array}\right\} \Rightarrow\left[(a-c)^{2}-b^{2}\right] \alpha=0 \).

So (excluding the degenerate case a – c = ±b) α = 0, and hence also γ= 0.

(2)

\( a \alpha+b \gamma=(a+b) \alpha \Rightarrow \quad \alpha-\gamma=0 \);

\( c \beta=(a+b) \beta \Rightarrow \quad \beta=0 \);

\( b \alpha+a \gamma=(a+b) \gamma \quad \text { (redundant) } \)

\( \text { So } \alpha=\gamma \text { and } \beta=0 \).

(3)

\( a \alpha+b \gamma=(a-b) \alpha \Rightarrow \quad \alpha+\gamma=0 \);

\( c \beta=(a-b) \beta \Rightarrow \quad \beta=0 \);

\( b \alpha+a \gamma=(a-b) \gamma \quad \text { (redundant) } \)

\( \text { So } \alpha=-\gamma \text { and } \beta=0 \).

Conclusion: The (normalized) eigenvectors of H are

\( \left|s_{1}\right\rangle=\left(\begin{array}{l} 0 \\ 1 \\ 0 \end{array}\right), \quad\left|s_{2}\right\rangle=\frac{1}{\sqrt{2}}\left(\begin{array}{l} 1 \\ 0 \\ 1 \end{array}\right), \quad\left|s_{3}\right\rangle=\frac{1}{\sqrt{2}}\left(\begin{array}{c} 1 \\ 0 \\ -1 \end{array}\right) \).

(a) Here \( | S (0)\rangle=\left|s_{1}\right\rangle \), so

\( | S (t)\rangle=e^{-i E_{1} t / \hbar}\left|s_{1}\right\rangle =e^{-i c t / \hbar}\left(\begin{array}{l} 0 \\ 1 \\ 0 \end{array}\right) \).

(b)

\( | S (0)\rangle=\frac{1}{\sqrt{2}}\left(\left|s_{2}\right\rangle+\left|s_{3}\right\rangle\right) \).

\( | S (t)\rangle=\frac{1}{\sqrt{2}}\left(e^{-i E_{2} t / \hbar}\left|s_{2}\right\rangle+e^{-i E_{3} t / \hbar}\left|s_{3}\right\rangle\right)=\frac{1}{\sqrt{2}}\left[e^{-i(a+b) t / \hbar} \frac{1}{\sqrt{2}}\left(\begin{array}{l} 1 \\ 0 \\ 1 \end{array}\right)+e^{-i(a-b) t / \hbar} \frac{1}{\sqrt{2}}\left(\begin{array}{c} 1 \\ 0 \\ -1 \end{array}\right)\right] \).

\( =\frac{1}{2} e^{-i a t / \hbar}\left(\begin{array}{c}e^{-i b t / \hbar}+e^{i b t / \hbar} \\ 0 \\ e^{-i b t / \hbar}-e^{i b t / \hbar} \end{array}\right)= e^{-i a t / \hbar}\left(\begin{array}{c} \cos (b t / \hbar) \\ 0 \\ -i \sin (b t / \hbar) \end{array}\right) \).
