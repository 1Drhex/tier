Guest Question.

Use the Tables in Part 3 of the Manual and select a W-shape of A992 steel and steel headed stud anchors for the following conditions: The beam spacing is 5 feet 6 inches, and the span length is 30 feet. The slab has a total thickness of 4 1⁄2 inches and is supported by a formed steel deck, the cross section of which is shown in Figure 9.24. The 28-day compressive strength of the concrete is \( f'_c = 4 ksi\). The loads consist of a 20-psf construction load, a 20-psf partition load, a ceiling weighing 5 psf, and a live load of 150 psf. The maximum live-load deflection cannot exceed L/240.

Admin
Solution

Loads to be supported before the concrete cures:
Slab wt. =\( \frac{4.5}{12}(150)\) = 56.25 psf (conservatively)
\(w_{slab} = 56.25(5.5) = 309.4 lb/ft\)
Construction load = 20(5.5) = 110.0 lb/ft
We will account for the beam weight later. After the concrete cures,
Partition load = 20(5.5) = 110.0 lb/ft
Live load: 150(5.5) = 825.0 lb/ft
Ceiling: 5(5.5) = 27.5 lb/ft
Loads to be carried by the composite section:
\(w_D = w_{slab} + w_{ceil} = 309.4 + 27.5 = 336.9 lb/ft\)
\(w_L = 110 + 825 = 935 lb/ft\)
\(w_u = 1.2w_D + 1.6w_L = 1.2(0.3369) + 1.6(0.935) = 1.900 kips/ft\)
\(M_u = \frac{1}{8}w_{u}L^{2} = \frac{1}{8}(1.900)(30)^{2} = 214 ft-kips\)

Assume a = 2 in.:
\(Y2 = t – \frac{a}{2} = 4.5 – \frac{2}{2} = 3.5 in.\)
From Table 3-19 in the Manual, any combination of steel shape, \(\sum Q_n,\) and Y2 that furnishes a design strength of more than 214 ft-kips will be an acceptable trial beam. Two possibilities are summarized in Table 9.8. The W14 \(\times\) 22 is the lighter shape, but because \(\sum Q_n\) is larger, it will require more steel anchors. For this reason, try the W14 \(\times \)26.

Compute the design strength:
b = beam spacing or span length / 4
= 5.5(12) = 66 in. or 30(12)/4 = 90 in.
Since 66 in. < 90 in., b = 66 in. Next, refine the value for Y2.
From \(C = \sum Q_n,\)
\(a =\frac {\sum Q_n}{0.85f'_c}= \frac {135}{0.85(4)(66)}= 0.6016 in\).
\(Y2 = t – \frac{a}{2}= 4.5 – \frac{0.6016}{2} = 4.199 in.\)
From Table 3-19 in the Manual, by interpolation,
\(\phi _{b}M_n = 237 ft-kips > 214 ft-kips (OK)\)
Adjust \(M_u \)for the beam weight:
\(w_u = 1.900 + 1.2(0.026) = 1.931 kips/ft\)
\(M_u =\frac{1}{8}w_{u}L^{2} = \frac{1}{8}(1.931)(30)^{2} = 217 ft-kips < 237 ft-kips (OK)\)
Check shear: From the \(Z_x \)table,\( \phi _{v}V_n = 106 kips.\)
\(V_u = \frac{w_uL}{2} = \frac{1.931(30)}{2} = 29.0 kips < 106 kips (OK)\)

Before the concrete cures,
\(w_D = 309.4 + 26 = 335.4 lb/ft\)
\(w_L = 110.0 lb/ft\)
\(w_u = 1.2w_D + 1.6wL = 1.2(0.3354) + 1.6(0.110) = 0.5785 kips/ft\)
\(M_u =\frac{1}{8}w_{u}L^{2} = \frac{1}{8}(0.5785)(30)^{2} = 65.1 ft-kips\)
\(\phi _{b}M_n = \phi _{b}M_p = 151 ft-kips > 65.1 ft-kips (OK)\)
The live-load deflection will be checked after the stud anchors have been selected, because the plastic neutral axis location may change, and this would affect the lower-bound moment of inertia.
The maximum stud diameter is \(2.5t_f = 2.5(0.420) = 1.05 in.\), but the maximum
diameter with formed steel deck = 3⁄4 in.
The minimum length is \(4d = 4(3⁄4) = 3 in. or h_r + 1.5 = 1.5 + 1.5 = 3 in.\)
For a length of 3 in., the cover above the top of the stud is
4.5 – 3 = 1.5 in. > 0.5 in. (OK)
Try 3⁄4 \(\times\) 3 studs:
\(A_{sa} = \pi(0.75)^{2}/4 = 0.4418 in.^{2}\)

\(Q_n = 0.5A_{sa}\sqrt{f'_cEc} \leq R_{g}R_{p}A_{sa}F_u\)
For one stud per rib, \(R_g = 1.0.\) In this book, we always conservatively use \(R_p = 0.6\) with steel deck.
\(Q_n = 0.5(0.4418)sqrt{4(3492)} = 26.11 kips\)
\(R_{g}R_{p}A_{sa}F_u = 1.0(0.6)(0.4418)(65) = 17.23 kips < 26.11 kips\)
\(\therefore\) Use \(Q_n = 17.23 kips.\)
(Alternatively, the stud shear strength can be found from Table 3-21 in the Manual. We consider the stud to be a “weak” stud, in keeping with our practice of always using \(R_p \) 0.6 with formed steel deck. The value of \(Q_n\) is found to be 17.2 kips.)
Number of studs:
\(N_1 =\frac {V'}{Q_n}= \frac {135}{17.23} = 7.84 \therefore \)Use 8.
Total no. required = 2(8) = 16

The approximate spacing is
\(\frac {30(12)}{16} = 22.5 in.\)
Minimum longitudinal spacing = 4d = \(4\left ( \frac{3}{4} \right ) = 3 in.\)
Maximum longitudinal spacing =\( 8t \leq 36 in.\)
8t = 8(4.5) = 36 in.
Try one stud every 3rd rib: Spacing = 3(6) = 18 in. < 36 in. (OK)
Total number \(= \frac {30(12)}{18} = 20\)
For \(N_1 = 20/2 = 10, \sum Q_n = 10(17.23) = 172.3 kips.\) From \(C = \sum Q_n,\)
\(a =\frac {\sum Q_n}{0.85f'_c}= \frac { 172.3}{0.85(4)(66)}= 0.7678\)
\(Y2 = t – \frac{a}{2}= 4.5 – \frac{0.7678}{2} = 4.116 in.\)
From Table 3-19 in the Manual, for \(\sum Q_n\) = 172.3 kips and Y2 = 4.116 in.,
\(\phi _{b}M_n = 250 ft-kips\) (by interpolation) . 217 ft-kips (OK)
Check the live load deflection: For \(\sum Q_n = 172.3\) kips and Y2 = 4.116 in., the lower bound moment of inertia from Table 3-20 in the Manual is
\(I_{LB} = 538 in.^{4} \) (by interpolation)
and the live load deflection is
\(\Delta_L =\frac {5w_LL^{4}}{384EI_{LB}}\)
\(= \frac {5(0.935/12)(30 \times 12)^{4}}{384(29,000)538} = 1.09 in.\)
The maximum permissible live load deflection is
\(\frac{L}{240} = \frac{30(12)}{240} = 1.5 in. > 1.09 in. (OK)\)
Ductility: Because the span length is only 30 ft, the ductility is adequate (condition 1 from the Commentary to AISC I3.2d).

Use a W14 \(\times \)26 and 20 studs, 3⁄4 \(\times \)3, one every third rib.

Loads to be carried by the composite section:
\(w_D = w_{slab} + w_{ceil} = 309.4 + 27.5 = 336.9 lb/ft\)
\(w_L = 110 + 825 = 935 lb/ft\)
\(w_a = w_D + w_L = 0.3369 + 0.935 = 1.272 kips/ft\)
\(M_a = \frac{1}{8}w_{a}L^{2} = \frac{1}{8}(1.272)(30)^{2} = 143 ft-kips\)
Assume a = 2 in.:
\(Y2 = t – \frac{a}{2}= 4.5 – \frac{ 2}{2}= 3.5 in.\)
From Table 3-19 in the Manual, any combination of steel shape, \(\sum Q_n,\) and Y2 that furnishes an allowable strength of more than 143 ft-kips will be an acceptable trial beam. Two possibilities are summarized in Table 9.9.

The W14 \(\times\) 22 is the lighter shape, but because \(\sum Q_n \)is larger, it will require more studs. For this reason, try the W14\( \times \)26. Compute the allowable strength:
b = beam spacing or span length / 4
= 5.5(12) = 66 in. or 30(12)/4 = 90 in.
Since 66 in. < 90 in., b = 66 in. Next, refine the value of Y2. From \(C = \sum Q_n ,\)
\(a =\frac {\sum Q_n}{0.85f'_c}= \frac {135}{0.85(4)(66)}= 0.6016 in.\)
\(Y2 = t – \frac{a}{2}= 4.5 – \frac{ 0.6016}{2} = 4.199 in\).
From Table 3-19 in the Manual, by interpolation,
\(\frac{M_n}{\Omega_b}= 158 ft-kips > 143 ft-kips (OK)\)

Adjust \(M_a\) for the beam weight:
\(w_a = 1.272 + 0.026 = 1.298 kips/ft\)
\(M_a = \frac{1}{8}w_{a}L^{2} = \frac{1}{8}(1.298)(30)^{2} = 146 ft-kips < 158 ft-kips (OK)\)
Check shear: From the \(Z_x \)table,\( V_n/\Omega _{v} = 70.9 kips\)
\(V_a = \frac{w_aL}{2} = \frac{1.298(30)}{2} = 19.5 kips < 70.9 kips (OK)\)
Before the concrete cures,
\(w_D = 309.4 + 26 = 335.4 lb/ft\)
\(w_L = 110.0 lb/ft\)
\(w_a = w_D + w_L = 0.3354 + 0.1100 = 0.4454 kips/ft\)
\(M_a = \frac{1}{8}w_{a}L^{2} = \frac{1}{8}(0.4454)(30)^{2} = 50.1 ft-kips\)
From Table 3-19,
\( \frac{M_n}{\Omega_b}= \frac{M_p}{\Omega_b}= 100 ft-kips > 50.1 ft-kips (OK)\)
For the design of the studs and the deflection check, see the LRFD solution. One 3⁄4 \(\times \)3 stud will be used every third rib. From Table 3-19, the allowable strength for
this composite beam is
\(\frac{M_n}{\Omega_b}= 166 ft-kips > 146 ft-kips (OK)\)
Ductility: Because the span length is only 30 ft, the ductility is adequate (condition 1 from the Commentary to AISC I3.2d).
Use a W14 \(\times\) 26 and 20 studs, 3⁄4 \(\times\) 3, one every third rib.
