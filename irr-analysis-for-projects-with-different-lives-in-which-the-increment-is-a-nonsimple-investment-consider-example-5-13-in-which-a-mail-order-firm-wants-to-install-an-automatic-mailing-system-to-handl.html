IRR Analysis for Projects with Different Lives in Which the Increment Is a Nonsimple Investment

Consider , in which a mail-order firm wants to install an automatic mailing system to handle product announcements and invoices. The firm has a choice between two different types of machines. Using the IRR as a decision criterion, select the best machine. Assume a MARR of 15%, as before.

Admin
Solution

Given: Cash flows for two projects with unequal lives, as shown in , and MARR = 15%.
Find: The alternative that is preferable.

Since the analysis period is equal to the least common multiple of 12 years, we may compute the incremental cash flow over this 12-year period. As shown in Figure 7.15, we subtract the cash flows of model A from those of model B to form the increment of investment. (Recall that we want the first cash-flow difference to be a negative value.) We can then compute the IRR on this incremental cash flow.

  • Five sign changes occur in the incremental cash flows, indicating a nonsimple incremental investment.
  • Even though there are five sign changes in the cash flow, there is only one positive i* for this problem: 63.32%.
  • Unfortunately, however, the net investment test shows that it is a mixed investment, so we need to calculate either RIC or use the MIRR on incremental investment.

RIC: We need to employ an external rate to compute the RIC in order to make a proper accept-reject decision. Assuming that the firmâ€™s MARR is 15%, we find that the RIC is 50.68%. Since \(IRR_{B-A} \gt MARR\) , model B would be selected, which is consistent with the NPW analysis. In fact, we can use the Goal Seek function in Excel to find the RIC, as shown in Table 7.9.

MIRR: We can also calculate the MIRR on this incremental investment at i = 15%.

\(\frac{\$1,000\left(F/P,15\%,11\right)+\$1,000\left(F/P,15\%,10\right)+… \$500 }{\$2,500 +\$12,500\left(F/P,15\%,4\right)+\$12,500\left(F/P,15\%,8\right) }\)

\(= \frac{\$100,887.62}{\$13,733.18}\)

= 7.3463

\(=\left( 1 + MIRR\right) ^{12}\)

\(MIRR = 18.08\% \gt 15\%\)

which indicates that model B be preferred.

Comments: Given the complications inherent in IRR analysis in comparing mutually exclusive alternatives, it is usually more desirable to employ one of the other equivalence techniques for this purpose. As an engineering manager, you should keep in mind the intuitive appeal of the rate-of-return measure. Once you have selected a project on the basis of NPW or AE analysis, you may also wish to express its worth as a rate of return for the benefit of your associates.
