Guest Question.

Use Gauss quadrature to evaluate the integrals
\(K_{11}=\int_{x_{a}}^{x_{b}}\left ( \frac{\mathrm{d^{2}} \phi _{1}}{\mathrm{d} x^{2}} \right )^{2}dx\) , \(G_{11}=\int_{x_{a}}^{x_{b}}(\phi _{1})^{2}dx\)
for the case in which the interpolation functions \(\phi _{i}\) are the fifth-order Hermite polynomials .

Admin
Solution

First note that \([\bar{x}=(1+\zeta )h/2]\)

\(\phi _{1}=1-10\frac{\bar{x}^{3}}{h^{3}}+15\frac{\bar{x}^{4}}{h^{4}}-6\frac{\bar{x}^{5}}{h^{5}}\) = \(1-\frac{5}{4}(1+\zeta )^{3}+\frac{15}{16}(1+\zeta )^{4}-\frac{3}{6}(1+\zeta )^{5}\)

\(\frac{\mathrm{d^{2}} \phi _{i}}{\mathrm{d} x^{2}}=\frac{4}{h_{e}^{2}}\frac{\mathrm{d^{2}} \phi _{i}}{\mathrm{d} \zeta ^{2}}\) , \(\frac{\mathrm{d^{2}} \phi _{1}}{\mathrm{d} \zeta ^{2}}=-\frac{15}{2}(1+\zeta )+\frac{45}{4}(1+\zeta )^{2}-\frac{15}{4}(1+\zeta )^{3}\)

We have

\(K_{11}=\frac{2}{h_{e}^{3}}\int_{-1}^{+1}\left ( \frac{\mathrm{d^{2}} \phi _{1}}{\mathrm{d} \zeta ^{2}} \right )^{2}d\zeta \)

= \(\frac{2}{h_{e}^{3}}\int_{-1}^{+1}\left [ -\frac{15}{2}(1+\zeta ) +\frac{45}{4}(1+\zeta )^{2}-\frac{15}{4}(1+\zeta )^{3}\right ]^{2}d\zeta \)

Thus, \(K_{11}\) is a sixth-order polynomial of \(\zeta \). Hence, it can be evaluated exactly by using N = [(6 + 1)/2] = 4:

\(K_{12}=\frac{2}{h_{e}^{3}}\sum_{I=1}^{4}W_{I}\left [ -1\frac{15}{2}(1+\zeta _{I}) +\frac{45}{4}(1+\zeta _{I})^{2}-\frac{15}{4}(1+\zeta _{I})^{3}\right ]^{2}\)

Similarly,

\(G_{11}=\int_{0}^{h_{e}}(\phi _{1})^{2}d\bar{x}\) = \(\frac{h_{e}}{2}\int_{-1}^{+1}\left [ 1-\frac{5}{4}(1+\zeta )^{3}+\frac{15}{16}(1+\zeta )^{4}-\frac{3}{16}(1+\zeta )^{5} \right ]^{2}d\zeta \)

which is a tenth degree polynomial in \(\zeta \); hence, N = [(10 + 1)/2] = 6. We have

\(G_{11}=\frac{h_{e}}{2}\sum_{I=1}^{6}\left [ 1-\frac{5}{4}(1+\zeta _{I})^{3}+\frac{15}{16} (1+\zeta _{I})^{4}-\frac{3}{16}(1+\zeta _{I})^{5}\right ]^{2}\)

The values obtained (with the help of Maple or Matlab programs) using the Gauss quadrature are (exact)

\(K_{11}=\frac{120}{7h_{e}^{3}}\) , \(G_{11}=\frac{181he}{462}\)
