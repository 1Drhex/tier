Guest Question.

For the system shown in Fig. 1, compute the constants \(K_{1}\) to \(K_{6}\) for the two operating points (a) \(P_{g}=1.0\) (b) \(P_{g}=1.1\). For both cases, assume that \(V_{t}=E_{b}=1.0\). The system data are \(R_{E}=1.0,x_{E}=5.0\),

the machine data: \(x_{d}=1.6,x_{q}=1.55,x_{d}^{'}=0.32,T_{do}^{'}=6.0,H=5,D=0,F_{B}=60\) Hz

Admin
Solution

From the discussion given in section 7.1.4, the external network is a special case of a two-port network for which the parameters are calculated as

\(y_{11}=\frac{1}{R_{E}}=\frac{1}{jx_{E}}=\frac{1}{R_{e}+jx_{e}},h_{12}=\frac{R_{E}}{R_{E}+jx_{E}}\)

The system shown in Fig.1 is equivalent to the system shown in Fig.2 with infinite bus voltage replaced by

\(E_{b}^{'}\angle \delta _{b}=h_{12}E_{b}\)

(a) The initial conditions are calculated as

\(\hat{V_{t}}=1.0\angle 0,I_{a}=1.0,E_{qo}=1.8446,\delta _{o}=57.17^{\circ},i_{do}=-0.8403,i_{qo}=0.5421,v_{do}=-0.8403,v_{qo}=0.5421\)

The constants are obtained from using the expressions given in section 7.1.3. They are

\(K_{1}=0.1971,K_{2}=2.1575,K_{3}=0.4490,K_{4}=0.2629,K_{5}=0.0068,K_{6}=1.0650\)

(b) The initial conditions are

\(\hat{V_{t}}=1.0\angle 30,I_{a}=1.1003,E_{qo}=1.998,\delta _{o}=88.58^{\circ},i_{do}=-0.9527,i_{qo}=0.5506,v_{do}=-0.8534,v_{qo}=0.5213\)

The constants are

\(K_{1}=0.0345,K_{2}=2.2571,K_{3}=0.4490,K_{4}=0.1826,K_{5}=-0.0649,K_{6}=1.0613\)
