Guest Question.

A specially rolled A36 structural steel section for the attachment has a cross section as shown in Fig. 9–19 and has yield and ultimate tensile strengths of 36 and 58 kpsi,respectively.
It is statically loaded through the attachment centroid by a load of \(F =24 \ kip\). Unsymmetrical weld tracks can compensate for eccentricity such that there is no moment to be resisted by the welds. Specify the weld track lengths \(l_1 \ and \ l_2\) for a \(\frac{5}{16}\)-in fillet weld using an E70XX electrode. This is part of a design problem in which the design variables include weld lengths and the fillet leg size.

Admin
Solution

The y coordinate of the section centroid of the attachment is

\(\bar{y} =\frac{\sum{y_i A_i } }{\sum{A_i} } =\frac{1\left(0.75\right)2+3\left(0.375\right)2 }{0.75\left(2\right)+0.375\left(2\right) } =1.67 \ in \)

Summing moments about point B to zero gives

\(\sum{M_B=0+-F_1+F\bar{y} } =-F_1\left(4\right) +24\left(1.67\right) \)

from which

\(F_1= 10 \ kip \)

It follows that

\(F_2= 24 − 10.0 = 14.0 kip\)

The weld throat areas have to be in the ratio \({14}/{10} = 1.4 \ , \ that \ is \ , l_2= 1.4l_1\).
The weld length design variables are coupled by this relation, so\( l_{1}\)is the weld length design variable. The other design variable is the fillet weld leg size h, which has been decided by the problem statement. From Table 9–4 , the allowable shear stress on the throat \(\tau_{all}\) is

\( \tau_{all}= 0.3\left(70\right) = 21 \ kpsi\)
Table 9–4
Stresses Permitted by the AISC Code for Weld Metal
Type of Loading Type of Weld Permissible Stress n*
tension butt \(0.60 s_y\) 1.67
Bearing Butt \(0.90 s_y\) 1.11
Bending Butt \(0.60 – 0.66 S_y\) 0.52-1.67
Simple compression Butt \(0.60 S_y\) 1.67
Shear Butt or filter \(0.30S^†_{ut}\)
*The factor of safety n has been computed by using the distortion-energy theory.
\(†\)Shear stress on base metal should not exceed \(0.40 S_y\) of base metal.

The shear stress \(\tau\) on the 45° throat is

\(\tau=\frac{F}{\left(0.707\right) h \left(l_1+l_2\right) } =\frac{F}{\left(0.707\right) h \left(l_1+1.4 l_1\right) } \\ =\frac{F}{\left(0.707\right) h \left(2.4l_1\right) }=\tau _{all}=21 \ kpsi \)
from which the weld length \(l_1\) is
\(l_1=\frac{24}{21\left(0.707\right)0.3125\left(2.4\right) } =2.16 \ in \)

and

\(l_2=1.4l_1=1.4\left(2.16\right) =3.02 \ in \)
These are the weld-bead lengths required by weld metal strength. The attachment shear stress allowable in the base metal, from Table 9–4, is
\(\tau _{all} =0.4 S_y=0.4\left(36\right) =14.4 \ kpsi\)

The shear stress \(\tau\) in the base metal adjacent to the weld is
\(\tau =\frac{F}{h\left(l_1+l_2\right) } =\frac{F}{h\left(l_1+1.4 l_1\right) } =\frac{F}{h \left(2.4 l_1\right) } =\tau _{all}=14.1 \ kpsi \)
from which
\(l_1=\frac{F}{14.4 h \left(2.4\right)}=\frac{24}{14.4\left(0.3125\right)2.4 } =2.22 \ in \\ l_2 = 1.4 l_1=1.4\left(2.22\right) =3.11 \ in \)
These are the weld-bead lengths required by base metal (attachment) strength.
The base metal controls the weld lengths. For the allowable tensile stress \(\sigma_{all}\) in the shank of the attachment, the AISC allowable for tension members is \(0.6S_y\); therefore,
\(\sigma _{all}=0.6 S_y=0.6\left(36\right) =21.6 \ kpsi \)
The nominal tensile stress σ is uniform across the attachment cross section because of the load application at the centroid. The stress \(\sigma\) is
\(\sigma=\frac{F}{A}=\frac{24}{0.75 \left(2\right)+2\left(0.375\right)}=10.7 \ kpsi \)

Since \(\sigma \leq \sigma _{all}\) , the shank section is satisfactory. With \(l_1\) set to a nominal \(2 \frac{1}{4} \ in \ , \ l_2 \ should \ be \ 1.4\left(2.25\right) = 3.15 \ in\).

Set \(l_1= 2\frac{1}{4} \ in \ , l_2= 3\frac{1}{4} \ in\). The small magnitude of the departure from \({l_2}/{l_1}= 1.4\) is not serious. The joint is essentially moment-free.
