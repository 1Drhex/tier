Guest Question.

A 2.3 kV, \(3-\phi \) star-connected synchronous motor has \(Z_{S} = \left(0.2 + j 2.2\right) ohm\) per phase. The motor is operating at 0.5 power factor leading with a line current of 200 A. Determine the generated emf per phase.

Admin
Solution

Solution by using phasor algebra

Supply voltage (phase value), \(V =\frac{2.3\times 1000}{\sqrt{3} }= 1328 V\)

Taking V as reference vector, \(\overline{V} = V\angle 0^{\circ }=V\pm j0\)

p.f., \(\cos \phi =0.5; \sin \cos ^{-1}0.5=0.866; \phi =\cos ^{-1}0.5=60^{\circ } leading.\)

Current, \(\overline{I}=I\angle +\phi ^{\prime }=200\angle 60^{\circ }\)

Impedance, \(\overline{Z}=R + jX_{S}=\left(0.2+j2.2\right) ohm \\[0.5cm]\hspace{30 pt} \quad \quad \enspace \> =\sqrt{\left(0.2\right)^{2}+\left(2.2\right)^{2} }\angle \tan ^{-1}\frac{2.2}{0.2}=2.21\angle 84.8^{\circ } \)

Now, \(\overline{E}=\overline{V}-\overline{I} \overline{Z}_{S}\\[0.5cm] \hspace{30 pt} = \left(V\pm j0\right)-\left[\left(200\angle 60^{\circ }\right)\left(2.21\angle 84.8^{\circ }\right) \right] \\[0.5cm] \hspace{30 pt} = 1328-\left[442\angle 144.8^{\circ }\right] \\[0.5cm] \hspace{30 pt} = 1328-\left[442\cos 144.8^{\circ }+j\sin 144.8^{\circ }\right] \\[0.5cm] \hspace{30 pt} = 1328-\left[442\times \left(-0.8171\right) +j0.5764\right] \\[0.5cm] \hspace{30 pt} = 1328-\left[-361.16+j254.77\right] \\[0.5cm] \hspace{30 pt}= 1689.16-j254.77=1708\angle -8.58^{\circ } \)

Generated emf/phase, \(E = \mathbf{1708 V} \)

Torque angle, \(\delta = \mathbf{8.58^{\circ } lagging}\)

Solution by vector method

Supply voltage (phase value), \(V=\frac{2.3\times 1000}{\sqrt{3} } =1328 V\)

Input current, \(I = 200 A\)

Power factor, \(\cos \phi =0.5; \phi =\cos ^{-1}0.5=60^{\circ }; \sin \phi =0.866\)

Angle between \(E_{r}\) and I, \(\theta =\tan ^{-1}\frac{X_{S}}{R}= \tan ^{-1}\frac{2.2}{0.2}=84.8^{\circ }\)

Synchronous impedance/phase, \( Z_{S}=\sqrt{R^{2}+X^{2}_{S} } =\sqrt{\left(0.2\right)^{2}+\left(2.2\right)^{2} } =2.21 \Omega \)

Voltage drop in impedance, i.e., \(E_{r} = IZ_{S} = 200 \times 2.21 = 442 V \)

From phasor diagram shown in Fig. 8.14.

Induced emf, \( E=\sqrt{V^{2}+E^{2}_{r} -2 VE_{r}\cos \left(\theta +\phi \right) } \\[0.5cm] \hspace{30 pt} \hspace{30 pt} \; =\sqrt{\left(1328\right)^{2}+\left(442\right)^{2}-2\times 1328\times 442\times \cos \left(84.8^{\circ }+60^{\circ }\right)}\\[0.5cm] \hspace{30 pt} \hspace{30 pt} \; = \mathbf{1708 V} \)

New, from phasor diagram,

\(\frac{E}{\sin \left(\theta +\phi \right) }=\frac{E_{r}}{\sin \delta }\) or \(\sin \delta =\frac{E_{r}}{E}\times \sin \left(\theta +\phi \right) \)

or \(\sin \delta =\frac{442}{1708}\times \sin \left(84.8^{\circ } +60^{\circ } \right) \\[0.5cm] \hspace{30 pt} =\frac{442}{1708}\times0.5764=0.1492\)

Torque angle, \(\delta =\sin ^{-1}0.1492=\mathbf{8.58^{\circ } lagging} \)

Note: All the numerical problems can be solved by either of the two methods.
