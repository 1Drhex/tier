Guest Question.

We can apply the surface source density distribution to describe the flow around a cylinder in a uniform stream, where the free-stream velocity is \( U_{\infty }\) .For simplicity, we will assume the radius of the cylinder is unity. The cylinder is represented by eight equal-length linear segments, as shown in Fig. 3.28 .The panels are arranged such that panel 1 is perpendicular to the undisturbed stream

Admin
Solution

– We can calculate the contribution of the source distribution on panel 2 to the normal velocity at the control point of panel 3. A detailed sketch of the two panels involved in this sample calculation is presented in Fig. 3.29 . Referring to equation (3.67), we are to evaluate the integral:

\(\int \frac{\partial }{\partial n_{i}}\left ( ln r_{ij} \right )ds_{j}\)

where i = 3 and j = 2. We will call this integral I32. Note that

\(\frac{\partial }{\partial n_{3}}\left ( ln r_{32} \right )=\frac{1}{r_{32}}\frac{\partial r_{32}}{\partial n_{3}}\) \(=\frac{\left ( x_{3}-x_{2} \right )\frac{\partial x_{3}}{\partial n_{3}}+\left ( z_{3}-z_{2} \right )\frac{\partial z_{3}}{\partial n_{3}}}{\left ( x_{3}-x_{2} \right )^{2}+\left ( z_{3}-z_{2} \right )^{2}}\)

where \( x_{3} = 0.00 and z_{3} = 0.92388 \) are the coordinates of the control point of panel 3. Note also that

\(\frac{\partial x_{3}}{\partial n_{3}}=0.00,\) \(\frac{\partial z_{3}}{\partial n_{3}}=1.00\)

Furthermore, for the source line represented by panel 2 we know:

\(x_{2}=-0.92388+0.70711s_{2}\) \(z_{2}=+0.38268+0.70711s_{2}\)

and the length of the panel is:

\(l_{2}=0.76537\)

Combining these expressions, we can obtain:

\(I_{32}=\int_{0}^{0.76537}\frac{\left ( 0.92388-0.38268-0.70711s_{2} \right )ds_{2}}{\left ( 0.92388-0.70711s_{2} \right )+\left ( 0.92388-0.38268-0.70711s_{2} \right )^{2}}\)

This equation can be rewritten as:

\( I_{32}=0.54120\int_{0}^{0.76537}\frac{ds_{2}}{1.14645 -2.07195s_{2}+1.00002s_{2}^{2}}-0.70711\int_{0}^{0.76537}\frac{s_{2}ds_{2}}{1.14645-2.07195s_{2}+1.00002s_{2}^{2}}\)

Using the integral tables to evaluate these expressions, we can now obtain:

\(I^{32}=-0.70711\left [ \frac{1}{2}ln\left ( s_{2}^{2}-2.07195s_{2}+1.14645 \right ) \right ]_{s_{2}=0}^{s_{2}=0.76537}-0.70711\left [ tan^{-1}\left ( \frac{2s_{2}-2.07195}{\sqrt{0.29291}} \right ) \right ]_{s_{2}=0}^{s_{2}=0.76537}\)

Therefore:

\(I_{32}=0.3528\)

In a similar manner, we could calculate the contributions of source panels 1, 4, 5, 6, 7, and 8 to the normal velocity at the control point of panel 3. Substituting the values of these integrals into equation (3.67), we obtain a linear equation of the form:

\(I_{31}k_{1}+I_{32}k_{2}+\pi k_{3}+I_{34}k_{4}+I_{35}k_{5}+I_{36}k_{6}+I_{37}k_{7}+I_{38}k_{8}=0.00\)

The right-hand side is zero since \(\alpha =0 and\delta _{3}=0\) . Repeating the process for all eight control points, we would obtain a set of eight linear equations involving the eight unknown source strengths. Solving the system of equations, we would find that

\(k_{1}=2\pi U_{\infty }\left ( +0.3765 \right )\) \(k_{2}=2\pi U_{\infty }=2\pi U_{\infty }\left ( +0.2662 \right )\) \(k_{3}=0.00\) \(k_{4}=2\pi U_{\infty }\left ( -0.2662 \right )\) \(k_{5}=2\pi U_{\infty }\left ( -0.3765 \right )\) \(k_{6}=2\pi U_{\infty }\left ( -0.2662 \right )\) \(k_{7}=0.00\) \(k_{8}=2\pi U_{\infty }\left ( +0.2662 \right )\)

Note there is a symmetrical pattern in the source distribution, as should be expected due to the symmetrical flow pattern. Also,

\(\sum k_{i}=0\)

as must be true since the sum of the strengths of the sources and sinks (negative sources) must be zero if we are to have a closed configuration
