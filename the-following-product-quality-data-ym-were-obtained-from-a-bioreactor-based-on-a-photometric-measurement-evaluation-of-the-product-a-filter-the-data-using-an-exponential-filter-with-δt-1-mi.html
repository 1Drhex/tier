(a) Filter the data using an exponential filter with \(\Delta t=1 min\). Consider \(\alpha=0.2\) and \(\alpha=0.5\).

Admin
Solution

Using Eq. 17-9 for \(\alpha=0.2\) and \(\alpha=0.5\), Eq. \(17-18\) for \(N^{*}=4\), and Eq. \(17-19\) for \(\Delta y=0.5\), the results are tabulated and plotted below.

\(y_{F}(k)=\alpha y_{m}(k)+(1-\alpha) y_{F}(k-1)\) (17-9)

\(y_{F}(k)=y_{F}(k-1)+\frac{1}{N^{*}}\left(y_{m}(k)-y_{m}\left(k-N^{*}\right)\right)\) (17-18)

\(y_{F}(k)= \begin{cases}y_{m}(k) & \text { if }\left|y_{m}(k)-y(k-1)\right| \leq \Delta y \\ y_{F}(k-1)-\Delta y & \text { if } y_{F}(k-1)-y_{m}(k)>\Delta y \\ y_{F}(k-1)+\Delta y & \text { if } y_{m}(k)-y_{F}(k-1)>\Delta y\end{cases}\) (17-19)
