Guest Question.

Designing a triangular-wave generator

(a) Design the triangular-wave generator shown in Fig. 16.37(a) so that \( f_{o}=4 kHz \text { and } V_{\text {th }}=\left|-V_{\text {th }}\right|=5 V \).

Assume \( +V_{ sat }=\left|-V_{ sat }\right|=14 V \).

(b) Use PSpice/SPICE to check your design.

Admin
Solution

(a) The steps used to design the triangular-wave generator are as follows:
Step 1. Find the values of \( R_{1} \text { and } R_{ F } \). Equation (16.34) gives

\( V_{ th }=\frac{R_{1}}{R_{ F }}\left(+V_{ sat }\right) \) (16.34)

 

\( \frac{R_{1}}{R_{ F }}=\frac{V_{ th }}{V_{ sat }}=\frac{5}{14}=0.36 \)

Let \( R_{1}=10 k \Omega \); then

\( R_{ F }=\frac{R_{1}}{0.36}=\frac{10 k \Omega}{0.36}=28 k \Omega \quad \text { (use a } 30- k \Omega \text { potentiometer) } \)

Step 2. Choose a suitable value of C: Let C = 0.01 µF.
Step 3. Find the value of R. Equation (16.39) gives

\( f_{ o }=\frac{R_{ F }}{4 R C R_{1}} \) (16.39)

 

\( R=\frac{R_{ F }}{4 f_{ o } C R_{1}} \) (16.40)

 

\( =\frac{28 k \Omega}{4 \times 4 kHz \times 0.01 \mu F \times 10 k \Omega}=17.5 k \Omega \quad \text { (use a 20-k } \Omega \text { potentiometer) } \)

(b) The circuit for PSpice simulation is shown in Fig. 16.38. The op-amp is simulated by PSpice macromodel LM111.

The PSpice plots of the voltages at the output of amplifiers \( A _{1} \text { and } A _{2} \) (using EX16-9.SCH) are shown in Fig. 16.39. The results are \( V_{ sat }=14.4 V \) (expected value is 14 V), \( -V_{\text {sat }}=-14.8 V \) (expected value is -14 V), \( V_{\text {th }}=5.3 V \) (expected value is 5 V), \( -V_{\text {th }}=-5.2 V \) (expected value is -5 V), and T =312.1 – 62.6 = 249.5 µs (expected value is 250 µs). If the integrator time constant τ = RC is made much smaller than the period of the output waveform, the triangular wave can be made very close to a sine wave.
