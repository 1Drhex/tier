Guest Question.

Use the variational method to estimate the ground state energy of the hydrogen atom.

Admin
Solution

The ground state wave function has no nodes and vanishes at infinity. Let us try

\(\psi (r,\theta ,\phi )=e^{-r/a},\) (9.152)

where α is a scale parameter; there is no angular dependence of ψ(r) since the ground state function is spherically symmetric. The energy is given by

\( E(\alpha )=\frac{〈\psi |\hat{H}|\psi 〉 }{〈\psi |\psi 〉}=-\frac{\left\langle\psi |(\hbar ^2/2m)\triangledown ^2+e^2/r|\psi \right\rangle }{〈\psi |\psi 〉} ,\) (9.153)

where

\( 〈\psi |\psi 〉=\int_{0}^{+\infty }{r^2e^{-2r/\alpha }dr} \int_{0}^{\pi }{\sin \theta d\theta }\int_{0}^{2\pi }{d\phi =\pi \alpha ^3} \) (9.154)

and

\(-\left\langle\psi \left|\frac{e^2}{r} \right|\psi \right\rangle=-4\pi e^2\int_{0}^{+\infty }{re^{-2r/\alpha }dr}= -\pi e^2\alpha ^2. \) (9.155)

To calculate the kinetic energy term, we may use (9.128) \(-\left\langle\psi \left|\frac{\hbar ^2}{2m}\Delta \right|\psi \right\rangle=\frac{\hbar ^2}{2m}\int{\left(\vec{\triangledown }\psi ^*(\vec{r} ) \right) } .\left(\vec{\triangledown }\psi (\vec{r} ) \right)d^3r. \)

\(-\left\langle\psi \left|\frac{\hbar ^2}{2m}\triangledown ^2 \right|\psi \right\rangle=\frac{\hbar ^2}{2m}\int{\left(\vec{\triangledown }\psi ^*(r ) \right) } .\left(\vec{\triangledown }\psi (r) \right)d^3r, \) (9.156)

where

\(\vec{\triangledown} \psi ^*(r ) =\vec{\triangledown }\psi (r)=\frac{d\psi (r)}{dr}\hat{r}=-\frac{1}{\alpha }e^{-r/\alpha }\hat{r} ; \) (9.157)

hence

\(-\left\langle\psi \left|\frac{\hbar ^2}{2m}\triangledown^2 \right|\psi \right\rangle=\frac{4\pi }{\alpha ^2}\frac{\hbar ^2}{2m}\int_{0}^{+\infty }{r^2e^{-2r/\alpha }}dr=\frac{\hbar ^2\pi }{2m}\alpha . \) (9.158)

Inserting (9.154), (9.155), and (9.158) into (9.153), we obtain

\(E(\alpha )=\frac{\hbar ^2}{2m\alpha ^2}-\frac{e^2}{\alpha }. \) (9.159)

Minimizing this relation with respect to \(\alpha ,dE(\alpha )/d\alpha =-\hbar ^2/(m\alpha ^3_0)+e^2/\alpha ^2_0=0, \) we obtain \(\alpha _0=\hbar ^2 /(me^2) \) which, when inserted into (9.159), leads to the ground state energy

\(E(\alpha _0)=-\frac{me^4}{2\hbar ^2}. \) (9.160)

This is the correct ground state energy for the hydrogen atom. The variational method has given back the correct energy because the trial function (9.152) happens to be identical with the exact ground state wave function. Note that the scale parameter \(\alpha _0=\hbar ^2 /(me^2) \) has the dimensions of length; it is equal to the Bohr radius.
