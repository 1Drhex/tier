Guest Question.

The block diagram of a conventional feedback control system contains the following transfer functions:

\(\begin{aligned}G_{c} &=K_{c}\left(1+\frac{1}{5 s}\right) \quad G_{v}=1 \\G_{m} &=\frac{1}{s+1} \\G_{p} &=G_{d}=\frac{5 e^{-2 s}}{10 s+1}\end{aligned}\)

(a) Plot the Bode diagram for the open-loop transfer function.

(b) For what values of \(K_{c}\) is the system stable?

(c) If \(K_{c}=0.2\), what is the phase margin?

(d) What value of \(K_{c}\) will result in a gain margin of \(1.7\) ?

Admin
Solution

a) Using Eqs. 14-56 and 14-57

\(\begin{aligned}A R_{O L}\left(\omega_{c}\right) &=\left|G_{O L}\left(j \omega_{c}\right)\right|=1 (14-56)\\\phi_{O L}\left(\omega_{c}\right) &=\angle G_{O L}\left(j \omega_{c}\right)=-180^{\circ} (14-57)\end{aligned}\)

\(\begin{aligned}& AR _{O L}=\left(K_{c} \sqrt{\frac{1}{25 \omega^{2}}+1}\right)\left(\frac{5}{\sqrt{100 \omega^{2}+1}}\right)\left(\frac{1}{\sqrt{\omega^{2}+1}}\right)(1.0) \\&\varphi=\tan ^{-1}(-1 / 5 \omega)+0+\left(-2 \omega-\tan ^{-1}(10 \omega)\right)+\left(-\tan ^{-1}(\omega)\right)\end{aligned}\)

b) Set \(\varphi=180^{\circ}\) and solve for \(\omega\) to obtain \(\omega_{c}=0.4695\)

Then \(\left. AR _{o L}\right|_{\omega-\omega_{c}}=1=K_{c u}(1.025)\)

Therefore, \(K_{c u}=1 / 1.025=0.976\)

System is stable for \(K_{c} \leq 0.976\)

c) For \(K_{c}=0.2\), set \(AR _{O L}=1\) and solve for \(\omega\) to obtain \(\omega_{g}=0.1404\)

Then \(\varphi_{g}=\left.\varphi\right|_{\omega=\omega_{g}}=-133.6^{\circ}\)

From Eq. 14-61, \(P M=180^{\circ}+\varphi_{g}=46.4^{\circ}\)

d) From Eq. 14-60

\(G M \triangleq \frac{1}{A R_{c}}\) (14-60)

\(G M=1.7=\frac{1}{A_{c}}=\frac{1}{\left. AR _{O L}\right|_{\omega=\omega_{c}}}\)

From part b), \(\left.\quad A R_{O L}\right|_{\omega=\omega_{c}}=1.025 K_{c}\)

Therefore \(\quad 1.025 K_{c}=1 / 1.7 \quad\) or \(\quad K_{c}=0.574\)
