Guest Question.

For the vertical flexural member shown, determine the principal stresses and the maximum shear stress acting at points H and K, as shown on Figures P15.27a and P15.27b. Show these stresses on an appropriate sketch for each point.

Admin
Solution

Section properties:

\(\begin{aligned}&A=(12 \text { in. })(6 in .)=72 in .{ }^{2} \\&I_{z}=\frac{(6 \text { in. })(12 in .)^{3}}{12}=864 in .^{4}\end{aligned}\) \(\begin{aligned}&Q_{H}=(6 \text { in. })(3.25 in .)\left(2.75 in .+\frac{3.25 in. }{2}\right)=85.3125 in .{ }^{3} \\&Q_{K}=Q_{H}=85.3125 in .^{3}\end{aligned}\)

Internal forces at H:

\(\begin{aligned}\Sigma F_{x} &=V_{H}-6 \text { kips }=0 \\& \therefore V_{H}=6 \text { kips } \\\Sigma F_{y} &=-F_{H}-16 \text { kips }-8 \text { kips }=0 \\& \therefore F_{H}=-24 \text { kips } \\\Sigma M &=M_{H}+(6 kips )(18 \text { in. })-(8 kips )(34 \text { in. })=0 \\& \therefore M_{H}=164 \text { kip-in. }\end{aligned}\)

Axial stress at H:

\(\sigma_{\text {axial }}=\frac{24 kips }{72 in. ^{2}}=0.333333 ksi =333.333 psi ( C ) \quad \text { (by inspection) }\)

Shear stress at H:

\(\tau_{H}=\frac{(6 kips )\left(85.3125 in .^{3}\right)}{\left(864 in .^{4}\right)(6 \text { in. })}=0.098741 ksi =98.741 psi\)

Bending stress at H:

\(\begin{aligned}\sigma_{\text {bend }} &=\frac{(164 \text { kip-in. })(2.75 \text { in. })}{864 \text { in. }^{4}}=0.521991 ksi \\&=521.991 psi ( T ) \quad \text { (by inspection) }\end{aligned}\)

Summary of stresses at H:

\(\begin{aligned}\sigma_{x} &=0 psi \\\sigma_{y} &=-333.333 psi +521.991 psi =188.658 psi \\\tau_{x y} &=-98.741 psi\end{aligned}\)

Principal stress calculations:
The principal stress magnitudes can be computed from Eq. (12.12):

\(\begin{aligned}\sigma_{p 1, p 2} &=\frac{\sigma_{x}+\sigma_{y}}{2} \pm \sqrt{\left(\frac{\sigma_{x}-\sigma_{y}}{2}\right)^{2}+\tau_{x y}^{2}}=\frac{(0)+(188.548)}{2} \pm \sqrt{\left(\frac{(0)-(188.548)}{2}\right)^{2}+(-98.741)^{2}} \\&=94.329 \pm 136.557\end{aligned}\)

 

\(\begin{aligned}&\sigma_{p 1}=231 psi \text { and } \quad \sigma_{p 2}=-42.2 psi \\&\tau_{\max }=136.6 psi (\text { maximum in-plane shear stress }) \\&\sigma_{\text {avg }}=94.3 psi ( T ) \quad \text { (normal stress on planes of maximum in-plane shear stress) }\end{aligned}\)

 

\(\begin{aligned}\tan 2 \theta_{p} &=\frac{\tau_{x y}}{\left(\sigma_{x}-\sigma_{y}\right) / 2}=\frac{-98.741}{[(0)-(188.657)] / 2}=\frac{-98.741}{-94.329}=1.046779 \\&\left.\therefore \theta_{p}=23.16^{\circ} \quad \text { (counterclockwise from the } x \text { axis to the direction of } \sigma_{p 2}\right)\end{aligned}\)

 

 

 

Internal forces at K:

\(\begin{aligned}\Sigma F_{x} &=V_{K}-6 \text { kips }=0 \\& \therefore V_{K}=6 \text { kips } \\\Sigma F_{y} &=-F_{K}-16 \text { kips }-8 \text { kips }=0 \\& \therefore F_{K}=-24 \text { kips } \\\Sigma M &=M_{K}+(6 kips )(18 in .+56 \text { in. })-(8 kips )(34 \text { in. })=0 \\& \therefore M_{K}=-172 \text { kip-in. }\end{aligned}\)

 

Axial stress at K:

\(\sigma_{\text {axial }}=\frac{24 \text { kips }}{72 \text { in. }^{2}}=0.333333 ksi =333.333 psi ( C ) \text { (by inspection) }\)

Shear stress at K:

\(\tau_{H}=\frac{(6 \text { kips })\left(85.3125 in.^{3}\right)}{\left(864 in .^{4}\right)(6 \text { in. })}=0.098741 ksi =98.741 psi\)

Bending stress at K:

\(\sigma_{\text {bend }}=\frac{(172 \text { kip-in.) }(2.75 in .)}{864 in .{ }^{4}}=0.547453 ksi =547.453 psi ( C ) \quad \text { (by inspection) }\)

 

Summary of stresses at K:

\(\begin{aligned}\sigma_{x} &=0 psi \\\sigma_{y} &=-333.333 psi -547.453 psi =-880.786 psi \\\tau_{x y} &=-98.741 psi\end{aligned}\)

 

Principal stress calculations:
The principal stress magnitudes can be computed from Eq. (12.12):

\(\begin{aligned}\sigma_{p 1, p 2} &=\frac{\sigma_{x}+\sigma_{y}}{2} \pm \sqrt{\left(\frac{\sigma_{x}-\sigma_{y}}{2}\right)^{2}+\tau_{x y}^{2}}=\frac{(0)+(-880.786)}{2} \pm \sqrt{\left(\frac{(0)-(-880.786)}{2}\right)^{2}+(-98.741)^{2}} \\&=-440.393 \pm 451.327\end{aligned}\)

 

\(\begin{aligned}&\sigma_{p 1}=10.93 psi \quad \text { and } \quad \sigma_{p 2}=-892 psi \\&\tau_{\max }=451 psi \quad \text { (maximum in-plane shear stress) }\\&\sigma_{\text {avg }}=440 psi ( C ) \text{(normal stress on planes of maximum in-plane shear str s)} \\&\end{aligned}\)

 

\(\begin{aligned}\tan 2 \theta_{p} &=\frac{\tau_{x y}}{\left(\sigma_{x}-\sigma_{y}\right) / 2}=\frac{-98.741}{[(0)-(-880.786)] / 2}=\frac{-98.741}{440.393}=-0.224212 \\&\left.\therefore \theta_{p}=-6.32^{\circ} \quad \text { (clockwise from the } x \text { axis to the direction of } \sigma_{p 1}\right)\end{aligned}\)
