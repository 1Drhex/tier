Guest Question.

For the three-phase half-wave controlled rectifier, the following parameters are give: rms input phase voltage 220 Vm 50 Hz, load resistance 1 Ω, load inductance 100 mH, and firing angle \(α=4.5^{\circ}\) a) Calculate the average output voltage and current. b) If the input source has a per phase inductance of \(L _{ S }\) = 1 mH per phase, calculate the overlap angle and the average output voltage when the output current is 158 A.

Admin
Solution

Solution

a) The average output voltage is:

 

\(\begin{aligned}\overline{ V }_{ o } &=\frac{3 \sqrt{6} \widetilde{ V }_{ i }}{2 \pi} \cos \alpha \\&=\frac{3 \sqrt{6} \times 220}{2 \pi} \cos 45^{\circ}=181.94 V\end{aligned}\)

 

Therefore, the average output current is:

 

\(\overline{ I }_{ o }=\frac{\overline{ V }_{ o }}{ R }=\frac{181.94}{1}=181.94 A\)

 

Since ωL > R (31.4 Ω > 1 Ω), the output current is considered to be pure dc with a value of 181.94 A. The following figure presents the power simulation (PSIM) results of the rectifier without taking into consideration the input source inductance. At this point, it should be mentioned,

that when simulating a three-phase thyristor rectifier the simulation thyristor firing angleα should take into account the \(30^{\circ}\) phase delay from the origin of \(v_{an}\).

b) The overlap angle can be calculated using Eq. (5.89) as follows:

 

\(\begin{aligned}&\frac{ di _{1}}{ dt }+\frac{ di _{2}}{ dt }=\frac{ d \overline{ I }_{ o }}{ dt }=0 \\&\text { or } \frac{ di _{1}}{ dt }=-\frac{ di _{2}}{ dt }\end{aligned}\)

 

\(\overline{ I }_{ o }=\frac{\sqrt{6} \tilde{ V }_{ i }}{2 \omega L _{ s }}(\cos \alpha-\cos (\alpha+\mu))\)

 

or

 

\(158=\frac{\sqrt{6} \times 220}{2 \pi \times 50 \times 0.001}\left(\cos 45^{\circ}-\cos \left(45^{\circ}+\mu\right)\right)\)

 

or

 

\(\mu=13.32^{\circ}\)

 

Since for the three-phase half-wave rectifier there are three commutation overlaps per period, then according to Eq. (5.97) the average output voltage will be reduced by:

 

\(\begin{aligned}\overline{ V }_{\mu} &=\frac{1}{2 \pi} \int_{\alpha}^{\alpha+\mu} \frac{ v _{ bn }- v _{ an }}{2} d (\omega t ) \\&=\frac{1}{2 \pi} \int_{\alpha}^{\alpha+\mu} \frac{ v _{ ba }}{2} d (\omega t ) \\&=\frac{1}{4 \pi} \int_{\alpha}^{\alpha+\mu} \sqrt{6} \widetilde{ V }_{ i } \sin (\omega t ) d (\omega t ) \\&=\frac{\sqrt{6} \widetilde{V}_{ i }}{4 \pi}[\cos \alpha-\cos (\alpha+\mu)]\end{aligned}\) (5.97)

 

\(\begin{aligned}\bar{V}_{o(\text { loss })} &=3 \frac{\sqrt{6 V}_{i}}{4 \pi}[\cos \alpha-\cos (\alpha+\mu)] \\&=3 \frac{\sqrt{6} \times 220}{4 \pi}\left(\cos 45^{\circ}-\cos \left(45^{\circ}+13.32^{\circ}\right)\right)=23.4 V\end{aligned}\)

 

Therefore, the average output voltage in this case will be:

 

\(\begin{aligned}\bar{V}_{o(n e t)} &=\text { average output voltage }-\text { voltage loss due to the } 3 \text { overlaps } \\&=181.94-23.4=158.54 V\end{aligned}\)
