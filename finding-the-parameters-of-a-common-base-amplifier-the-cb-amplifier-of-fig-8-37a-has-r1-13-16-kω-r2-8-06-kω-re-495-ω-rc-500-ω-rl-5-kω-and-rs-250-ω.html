Guest Question.

Finding the parameters of a common-base amplifier The CB amplifier of Fig. 8.37(a) has \( R_{1}=13.16 k \Omega, R_{2}=8.06 k \Omega, R_{ E }=495 \Omega, R_{ C }=500 \Omega, R_{ L }=5 k \Omega \text {, and } R_{ s }=250 \Omega \). The parameters of the transistor are \( r_{\pi}=258 \Omega, \beta_{ f }=100 \text {, and } r_{ o 1}=\infty \text {. Assume that } C_{1}=C_{2}=\infty \text {. } \)

(a) Calculate the input resistance \( R_{\text {in }}\left(=v_{ S } / i_{ S }\right) \), the no-load voltage gain \( A_{ vo }\left(=v_{ o } / v_{ e }\right) \), the output resistance \( R_{ o } \), the overall voltage gain \( A_{ v }\left(=v_{ L } / v_{ S }\right) \), and the maximum permissible voltage gain \( A_{ vo (\max )}\).

(b) Use PSpice/SPICE to verify your results in part (a).

Admin
Solution

(a) We first find \( R_{ B } \text { and } R_{ X } \).

 

\(\begin{aligned}&R_{ B }=R_{1}\left\|R_{2}=13.16 k \right\| 8.06 k =5 k \Omega \\&R_{ x }=\frac{r_{\pi}+R_{ B }}{1+\beta_{ f }}=\frac{258+5 k }{101}=52.1 \Omega\end{aligned}\)

 

From Eq. (8.106),

 

\( R_{ i }=\frac{v_{ e }}{i_{ s }}=R_{ E }\left\|R_{ x }=R_{ E }\right\| \frac{r_{\pi}+R_{ B }}{1+\beta_{ f }} \) (8.106)

 

\( R_{ i }=R_{ E }\left\|R_{ x }=495\right\| 52.1=47.1 \Omega \)

 

Thus

 

\( R_{\text {in }}=R_{ i }+R_{ s }=47.1+250=297.1 \Omega \)

 

From Eq. (8.107),

 

\(A_{ vo }=\frac{v_{ o }}{v_{ e }}=\frac{g_{ m } r_{\pi} R_{ C }}{r_{\pi}+R_{ B }}=\frac{\beta_{ f } R_{ C }}{r_{\pi}+R_{ B }}\) (8.107)

 

\( A_{ vo }=\frac{\beta_{ f } R_{ C }}{r_{\pi}+R_{ B }}=\frac{100 \times 500}{258+5 k }=9.5 \)

 

and \( R_{ o }=R_{ C }=500 \Omega\)

 

Using Fig. 8.38(c), we find that the overall voltage gain \( A_{ V } \) is

 

\(A_{ v }=\frac{v_{ L }}{v_{ s }}=\frac{A_{ vo } R_{ i } R_{ L }}{\left(R_{ i }+R_{ s }\right)\left(R_{ L }+R_{ 0 }\right)}=\frac{9.5 \times 47.1 \times 5 k }{(47.1+250)(5 k +500)}=1.37\)

 

The maximum permissible voltage gain from Eq. (8.108) is given by

 

\(A_{ vo (\max )}=\frac{\beta_{ f } R_{ C }}{r_{\pi}}=g_{ m } R_{ C }\) (8.108)

 

\(A_{ vo (\max )}=\frac{\beta_{ f } R_{ C }}{r_{\pi}}=\frac{100 \times 500}{258}=194\)

 

(b) The emitter-follower circuit for PSpice simulation is shown in Fig. 8.40.

 

The PSpice plots, which are shown in Fig. 8.41 for \( v_{ s }=1 mV , \text { give } v_{ o }=1.44 mV \text { and } R_{ in }=V_{ s ( rms )} / I_{ s ( rms )}= 424 \Omega \) (expected value is 297 Ω), and the voltage gain is \( A_{ V }=v_{ o } / v_{ s }=1.43 \) (expected value is 1.37). Thus, the results are close to the expected values.
