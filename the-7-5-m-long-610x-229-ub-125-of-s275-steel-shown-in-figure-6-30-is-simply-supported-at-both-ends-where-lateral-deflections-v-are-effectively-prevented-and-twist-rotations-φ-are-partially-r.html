Guest Question.

The 7.5 m long 610× 229 UB 125 of S275 steel shown in Figure 6.30 is simply supported at both ends where lateral deflections v are effectively prevented and twist rotations Φ are partially restrained. Check the adequacy of the beam for a central concentrated top flange load caused by an unfactored dead load of 60 kN (which includes an allowance for self-weight) and an unfactored imposed load of 100 kN.

Admin
Solution

Design bending moment.

\(M_{Ed} = \left\{(1.35 × 60) + (1.5 × 100)\right\} × 7.5/4 = 433 kNm.\)

Section resistance.
As in ,

\(f_{y} = 265 N/mm^2\), the section is Class 1, and the section resistance is \(M_{c,Rd} = 974 kNm > 433 kNm = M_{Ed}\) and the section resistance is adequate.

Elastic buckling moment.

Using equation 6.56 \(k_{cr} = 1 + (d_{f} /6L)(t_{f} /t_{w})^3 (1 + b_{f}/d_{f})/2,\)

to allow for the partial torsional end restraints

\(K_{cr} = 1 + {\left\{(612.2 − 19.6)/(6 × 7500)\right\} }(19.6/11.9)^3 × (\left\{1 + 229.0/(612.2 − 19.6\right ) ) /2 = 1.041\)

so that \(L_{cr} = 1.041 × 7500 = 7806 mm.\)

Adapting equation 6.36

\(M_{cr} =\sqrt{}\left\{(\frac{π^2EI_{z}}{L^2_{cr}} )(GI_{t} +\frac{π^2EI_{w}}{L^2_{cr}} )\right\} ,\)

 

\(M_{zx0} =\sqrt{\left\{\frac{π^2 × 210 000 × 3932 × 10^4}{7806^2} ×(81 000 × 154 × 10^4 +\frac{π^2 × 210 000 × 3.45 × 10^{12}}{7806^2} ) \right\} }Nmm = 569 kNm.\)

 

Using equation 6.12, \(N_{cr,z} = π^2EI_{z}/L^2.\)

 

\(N_{cr,z} = π^2 × 210 000 × 3932 × 10^4/7500^2 N = 1449 kN.\)

 

Using Figure 6.7, \(α_{m} = 1.35\)

 

\(\frac{0.4α_{m}z_{Q}N_{cr,z}}{M_{zx0}}=\frac{0.4 × 1.35 × (− 612.2/2) × 1449 × 10^3}{569 × 10^6}= −0.421\)

 

Adapting equation 6.11,

\(\frac{M_{cr}}{M_{zx}}= α_{m}\left\{\sqrt{} [1 +(\frac{0.4α_{m}z_{Q}N_{cr,z}}{M_{zx}} )^2]+\frac{0.4α_{m}z_{Q}N_{cr,z}}{M_{zx}} \right\}\)

 

\(M_{cr} = 1.35 × 569( √[1 + (−0.421)^2] + (−0.421)) Nmm = 510 kNm.\)

 

(Using the computer program PRFELB [18] with L = 7806 mm leads to \(M_{cr} = 522 kNm.)\)

 

Member resistance.

Using equation 6.25, \(\bar{\lambda }_{LT}=\sqrt{(W_{y}f_{y}/M_{cr}) } \)

 

\(\bar{\lambda }_{LT}=\sqrt{(974/510) } = 1.382\)

 

\(h/b = 612.2/229.0 = 2.67 > 2\)

 

Using the EC3 simple general method with \(β = 1.0, \bar{\lambda }_{LT,0} = 0.2\) (Clause 6.3.2.2) and \(α_{LT} = 0.34\) (Tables 6.4 and 6.3), and equation 6.27,

\(Φ_{LT} = 0.5\left\{1 + α_{LT} (\bar{\lambda } _{LT} − \bar{\lambda } _{LT,0}) + β\bar{\lambda }^2_{LT} \right\} \)

 

\(Φ_{LT} = 0.5\left\{1 + 0.34(1.382 − 0.2) + 1.0 × 1.382^2\right\} = 1.656\)

 

Using equation 6.26,

 

\(\frac{M_{b,Rd}γ_{M1}}{W_{y}f_{y}} =\frac{1}{Φ_{LT} +\sqrt{Φ^2_{LT} − β\bar{\lambda }^2_{LT}} }\)

 

\(M_{b,Rd} = 974/(1.656 + √(1.656^2 − 1.382^2)) kNm = 379 kNm < 433 kNm = M_{Ed}\)

 

and the beam appears to be inadequate.

Using the EC3 less conservative method with \(β = 0.75, \bar{\lambda }_{LT,0} = 0.4\) (Clause 6.3.2.3) and \(α_{LT} = 0.49\) (Tables 6.5 and 6.3), and equation 6.27,

\(Φ_{LT} = 0.5\left\{1 + α_{LT} (\bar{\lambda } _{LT} − \bar{\lambda } _{LT,0}) + β\bar{\lambda }^2_{LT} \right\} \)

 

\(Φ_{LT} = 0.5{\left\{1 + 0.49(1.382 − 0.4) + 0.75 × 1.382^2\right\} } = 1.457\)

 

Using equation 6.26,

\(\frac{M_{b,Rd}γ_{M1}}{W_{y}f_{y}} =\frac{1}{Φ_{LT} +\sqrt{Φ^2_{LT} − β\bar{\lambda }^2_{LT}} }\)

 

\(M_{b,Rd} = 974/(1.457 + √(1.457^2 − 0.75 × 1.382^2)) kNm = 426 kNm < 433 kNm = M_{Ed}\)

 

and the design moment resistance still appears to be inadequate.

The design moment resistance is further increased by using equations 6.29, 6.30, and 6.28

\(6.29, f = 1 − 0.5(1 − k_{c})\left\{1 − 2(\bar{\lambda }_{LT} − 0.8)^2\right\} \)

 

\(6.30, k_{c} = 1/√C_{1} = 1/√α_{m}\)

 

\(6.28, M_{b,Rd,mod} = M_{b,Rd} /f ≤ M_{b,Rd}\)

 

\(f = 1 − 0.5 × (1 − 1/ √1.35)(1 − 2 × (1.382 − 0.8)^2) = 0.978\), and

 

\(M_{b,Rd,mod} = 426/0.978 = 436 kNm > 433 kNm = M_{Ed}\)

 

and the design moment resistance is adequate after all.
