Guest Question.

A BPSK scheme operating over an AWGN channel with noise power spectral density of \(N_{0} / 2\), uses equiprobable signals \(s_{1}(t)=\sqrt{\frac{2 E}{T}} \sin \left(\omega_{o} t\right)\) and \(s_{2}(t)=-\sqrt{\frac{2 E}{T}} \sin \left(\omega_{o} t\right)\) over the symbol internal (0, T). If the local oscillator in a coherent receiver is ahead in phase 45° with respect to the received signals, the probability of error in the resulting system is

(a) \(Q \left(\sqrt{\frac{2 E }{ N _{ o }}}\right)\) (b) \(Q \left(\sqrt{\frac{ E }{ N _{ o }}}\right)\)

(c) \(Q \left(\sqrt{\frac{ E }{2 N _{ o }}}\right)\) (d) \(Q \left(\sqrt{\frac{ E }{4 N _{ o }}}\right)\)

Admin
Solution

BPSK receiver is shown as

Given BPSK signals are \(S_{1}(t)=\sqrt{\frac{2 E}{T}} \sin \left(\omega_{c} t\right)\) and \(S_{2}(t)=-\sqrt{\frac{2 E}{T}} \sin \left(\omega_{c} t\right)\) for the interval of 0 ≤ t ≤ t duration. Let \(\phi\) (t) represent the local oscillator generated signal which is 45° leading with respect to the received signal. \(\text { Then } \phi(t)=\sqrt{\frac{2}{T}} \sin \left(\omega_{c} t+45\right) \quad 0 \leq t \leq T\) assuming \(\phi\)(t) is of Unit energy. Let \(R_{1}(t) \text { and } R_{2}(t)\) be the received signal output, given as

\(\begin{aligned}&=R_{1}(t)=\int_{0}^{T} S_{1}(t) \phi(t) d t \\&=\int_{0}^{T} \sqrt{\frac{2 E}{T}} \sin \left(\omega_{c} t\right) \sqrt{\frac{2}{T}} \sin \left(\omega_{c} t+45^{\circ}\right) d t=\sqrt{\frac{E}{2}}\end{aligned}\)

Similarly,

\(R_{2}(t)=-\sqrt{\frac{E}{2}} .\)

Let decision threshold be given as \(\gamma=\sqrt{\frac{E}{2}}+N\). Then probability of error while transmitting \(S_{1}\) and receiving it as \(S_{2}\) is

\(\begin{aligned}&P\left(S_{2} / S_{1}\right)=P(\gamma<0)=P\left(\sqrt{\frac{E}{2}}+N<0\right) \\&=P\left(N<-\sqrt{\frac{E}{2}}\right)\end{aligned}\)

And

\(\begin{aligned}&P\left(S_{2} / S_{1}\right)=P(\gamma<0)=P\left(\sqrt{\frac{E}{2}}+N<0\right) \\&=P\left(N<-\sqrt{\frac{E}{2}}\right)\end{aligned}\)

 

\(\begin{gathered}P\left(N<\sqrt{\frac{E}{2}}\right)=\int_{-\infty}^{0} \frac{1}{\sqrt{2 \pi \frac{N_{0}}{2}}} \exp \left(\frac{\left(x+\sqrt{\frac{E}{2}}\right)^{2}}{2 \frac{N_{0}}{2}}\right) d x \\=\int_{-\infty}^{0} \frac{1}{\sqrt{\pi N_{0}}} \exp \frac{\left(x+\sqrt{\frac{E}{2}}\right)^{2}}{N_{0}} d x \\\text { Let } \frac{\left(x+\sqrt{\frac{E}{2}}\right)^{2}}{N_{0}}=Z . \text { Then } d x=\sqrt{\frac{N_{0}}{2} d z} \\P\left(N<\sqrt{\frac{E}{2}}\right)=\int_{\sqrt{\frac{E}{N_{0}}}}^{\infty} \frac{1}{2 \pi} \exp \left(-\frac{Z^{2}}{2}\right) d z\end{gathered}\)

So,

\(=Q\left(\sqrt{\frac{E}{N_{0}}}\right)\)

As symbols are equiprobable

\(\text { so } P(e)=\frac{1}{2}\left(P\left(S_{2} / S_{1}\right)+P\left(S_{1} / S_{2}\right)\right)\)

so

\(P(e)=\frac{1}{2} Q\left(\sqrt{\frac{E}{N_{0}}}\right)+\frac{1}{2} Q\left(\sqrt{\frac{E}{N_{0}}}\right)=Q\left(\sqrt{\frac{E}{N_{0}}}\right)\)

Hence, the correct option is (b)
