ANALYSIS OF A FOUR-BAR MECHANISM

Figure 4.14(a) depicts a four-bar mechanism having

r\(_1\)=8.0 cm; r\(_2\)=3.5 cm; r\(_3\)=10.0 cm ; r\(_4\)=9.0cm

\(\theta_2\)=15° ; \(\dot\theta_2\)=36.7° rad/sec(constant), therefore \(\ddot\theta_2\)=0

Determine the angular displacements, angular velocities, and angular accelerations of links 3 and 4.

Admin
Solution

Using Equations (4.3-54)\([h_1=\frac{r_1}{r_2};h_3=\frac{r_1}{r_4};h_5=\frac{r_1^2+r_2^2-r_3^2+r_4^2}{2r_2r_4} ]\)and (4.3-63)\([h_2=\frac{r_1}{r_3};h_4=\frac{-r_1^2-r_2^2-r_3^2+r_4^2}{2r_2r_3} ]\), we first calculate

\(h_1=\frac{r_1}{r_2}=\frac{8.0}{3.5}=2.286\) ; h\(_2\)=0.800; h\(_4\)=-1.361

Then, using Equations (4.3-57)\([b=-2\sin\theta_2]\), (4.3-61)\([a=-h_1+(1+h_2)\cos\theta_2+h_4]\), and (4.3-62)\([c=h_1+(1-h_2)\cos\theta_2+h_4]\), we obtain

\(a=-h_1+(1+h_2)\cos\theta_2+h_4\)

\(=-2.286+(1+0.800)\cos15°-1.361=-1.908\)

b=-0.518; c=0.732

Using Equations (4.3-64)\([(\theta_3)_2=2tan^{-1}\left[\frac{-b-(-b^2-4ac)^{1/2}}{2a}\right],(\theta_4)_1=2tan^{-1}\left[\frac{-b-(-b^2-4de)^{1/2}}{2d}\right]]\) and (4.3-65)\([(\theta_3)_2=2tan^{-1} \left[\frac{-b+(b^2-4ac)^{1/2}}{2a}\right],(\theta_4)_1=2tan^{-1}\left[ \frac{-b+(b^2-4de)^{1/2}}{2d}\right]]\), we obtain

\((\theta_3)_1=2tan^{-1}\left[\frac{-b-(b^2-4ac)^{1/2}}{2a}\right]\)

\(=2tan^{-1}\left[\frac{-(-0.518)-\left((-0.518)^2-4(-1.908)\times 0.732)\right)^{1/2}}{2\times(-1.908)}\right]=53.0\)°

\((\theta_3)_2=-75.2°\)

from which we select by viewing the given geometry of the mechanism

\(\theta_3=(\theta_3)_1=53.0\)°

Similarly, the angular displacement of link 4 is

\(\theta_4=(\theta_4)_1=81.0\)°

Using Equation (4.3-69), the angular velocity of link 3 is

\(\dot{\theta}_3=\frac{r_2\dot{\theta}_2}{r_3}\frac{\sin(\theta_2-\theta_4)}{\sin(\theta_4-\theta_3)}\)

\(=\left[\frac{3.5\times36.7}{10.0}\frac{\sin(15.0°-81.0°)}{\sin(81.0°-53.0°)}\right]rad/sec=-25.0\) rad/sec

Similarly, using Equation (4.3-70)\([\dot{\theta}_4=\frac{r_2\dot{\theta}_2}{r_4}\frac{\sin(\theta_2-\theta_4)}{\sin(\theta_4-\theta_3)}]\), we find

\(\dot{\theta}_4=-18.7\) rad/sec

Using Equations (4.3-71)\([\ddot{\theta }_3=\frac{-r_2\dot{\theta }^2_2\cos(\theta _2-\theta _4)-r_2\ddot{\theta }_2\sin(\theta _2-\theta _4)-r_3\dot{\theta}^2 _3 \cos(\theta _3-\theta _4)-r_4\dot{\theta}^2 _4}{r_3\sin(\theta _3-\theta _4)} ]\) and (4.3-72)\([\ddot{\theta }_4=\frac{-r_2\dot{\theta }^2_2\cos(\theta _2-\theta _2)-r_2\ddot{\theta }_2\sin(\theta _2-\theta _3)-r_3\dot{\theta}^2 _3+r_4\dot{\theta}_4^2 \cos (\theta _3-\theta _4)}{r_4\sin(\theta _3-\theta _4)}]\), the angular accelerations of links 3 and 4 are 9\(_3\) =

\(\ddot{\theta}_3=908\) rad/sec²; \(\ddot{\theta}_4=1700\) rad/sec²

A positive value of either rotational speed or angular acceleration is in the counterclockwise direction, whereas a negative value corresponds to the clockwise direction.
