Guest Question.

Torsion in a Spandrel Beam Supporting Brick Veneer

Determine whether the beam shown in Figure 6-20 is adequate for combined bending loads and torsion. The floor-to-floor height is 12 ft. and the beam span is 17 ft. The loads shown are service loads. Use a unit weight of 40 psf for the brick veneer, and assume that the top flange has continuous lateral support.

Admin
Solution

The loading shown is equivalent to Case 3. We will use the both methods presented for Case 3 loading to compare the results.

Veneer weight,\(w_{v} =(0.040 ksf)(12 ft.)=0.48 kips/ft.\).

The critical load combination for these loads is \(1.2D + 1.6L\).

Vertical Load

\(1.2(1.0 + 0.48) + 1.6(1.2) = 3.70 kips/ft.; M_{ux} = \frac{w_{ux}L^{2} }{8}=\frac{(3.70)(17)^{2} }{8}=134 ft.-kips\)

 

Horizontal Load

\(M = Pe = (1.2)(0.48)(8 in./12) = 0.384 ft.-kips/ft\)

 

\(h_{o}=d – t_{f} = 15.9 in. – 0.43 in. = 15.47 in=1.29 ft.\)

 

\(\frac{Pe}{h_{o} }=\frac{0.384}{1.29}=0.298 kips/ft.\rightarrow M_{uy}=\frac{w_{uy}L^{2} }{8}=\frac{(0.298)(17)^{2} }{8 }= 10.8 ft.-kips\)

\(\phi _{b}M_{nx}= 240 ft.-kips(L_{b}=0 ft. )\)

 

\(M_{ny}=M_{py}=F_{y}Z_{y}\leq 1.6F_{y}S_{y}\)

 

\(= (50)(10.8) \lt (1.6)(50)(7) = 540 in-k \lt 560 in.-kips\)

 

\(\phi _{b}M_{ny}=\frac{(0.9)(540)}{12}=40.5 ft.-kips\)

 

Checking the interaction equation for Case 2:

\(\frac{M_{ux} }{\phi _{b}M_{nx}}+\frac{M_{uy} }{0.5(\phi _{b}M_{ny})}\leq1.0; \frac{134}{240}+\frac{10.8}{0.5(40.5)}=1.09\gt 1.0\)

 

The W16 × 36 is not adequate for biaxial bending.

\(C_{W}=1460 in.^{6}\)

 

\(J=0.545 in.^{4}\)

 

\(a=\sqrt{\frac{EC_{w} }{GJ} }=\sqrt{\frac{(29,000)(1460)}{(11,200)(0.545)} }=83.3 in\).

 

\(\frac{L}{a}=\frac{(17)(12)}{83.3}=2.45\)

 

From Table 6-7,

 

\(w = (1.2)(0.48) = 0.576 kips/ft. = 0.048 kips/in.\) \(e = 8 in.\)

 

\(L = (17)(12) = 204 in.\)

 

\(M_{r}=\frac{weLa}{h_{0} }\left[0.097 + 0.094\left(\frac{L}{2a}-0.0255\left(\frac{L}{2a} \right) ^{2} \right) \right]\)

 

\(M_{f}=\frac{(0.048)(8)(204)(83.3)}{15.47}\left[0.097 + 0.094\left(\frac{204}{(2)(83.3)} \right)-0.0255\left(\frac{204}{(2)(83.3)} \right) ^{2} \right]\)

 

\(M_{f}=73.3 in.-kips = 6.11 ft. kips\)

 

This value is less than the value previously calculated (10.8 ft.-kips). Checking the interaction equation for the new value gives

\(\frac{M_{ux} }{\phi _{b}M_{nx} }+\frac{M_{uy}}{0.5(\phi _{b}M_{ny})}\leq 1.0;\) \(\frac{134}{240}+\frac{6.11}{0.5(40.5)} =0.86\lt 1.0\)

 

The W16 × 36 is adequate for biaxial bending.

Check Deflection:

For deflections, the combined weight of the veneer and the live load will be used and will be compared with a deflection limit of the smaller of L/600 and 0.3″. (A 0.3″ deflection controls only for L > 15′, 0″). Further, the Brick Industry Association [5] recommends a maximum torsional rotation of 1/16 in., which will also be checked, Thus,

\(w = 1.2 + 0.48 = 1.68 kips/ft.\)

 

\(\Delta =\frac{5 wL^{4} }{384El} =\frac{5(1.68/12)(17 \times 12)^{4} }{(384)(29\times 10^{6} )(448)}= 0.243 in.\lt \frac{L}{600} =\frac{(17)(12)}{600} =0.34 in. or 0.3 in., OK\)

 

Check torsional rotation:

\(w= 0.480 kips/ft = 0.04 kips/in.\)

 

From Table 6-7,

\(\theta =\frac{weLa}{GJ}=\left[-0.032 + 0.062\left(\frac{L}{2a} \right) + 0.052\left(\frac{L}{2a} \right) ^{2} \right]\)

 

\(\theta =\frac{(0.04)(8)(204)(83.3)}{(11,200)(0.545)} \left[-0.032+0.062\left(\frac{204}{(2)(83.3)} \right) +0.052\left(\frac{204}{(2)(83.3)} \right) ^{2} \right]\)

\(\theta =0.109\) radians

\((0.109)\left(\frac{180}{\pi } \right)=6.22^{\circ }\)

With reference to Figure 6-20b, the vertical displacement for this rotation is

\(\Delta _{v}=(tan 6.22)(8″)=0.872\) in.

This amount of twist far exceeds the deflection limit and would not be adequate. The reader should confirm that adding lateral support at 1⁄3 points would decrease the torsional rotation to about 0.11° with \(\Delta _{v}\)=0.015in) and would be recommended here.
