Guest Question.

Designing a shunt-series feedback circuit An amplifier with shunt-series feedback is shown in Fig. 10.48. The amplifier has \( R_{1}=6.6 k \Omega, R_{2}=2 k \Omega, R_{ Cl }=5 k \Omega, R_{3}=5 k \Omega, R_{4}=10 k \Omega, R_{ E }=500 \Omega \), and \( R_{ C 2}=5 k \Omega \). The transistor parameters are \( h_{ fe }=150, r_{\pi}=r_{\pi 1}=r_{\pi 2}=2.58 k \Omega \text {, and } r_{ o }=\infty \text {. } \)

(a) Determine the value of feedback resistor \( R _{ F } \)so that the closed-loop current gain \( A_{ f } \) is 10% of the open-loop current gain A.
(b) Use PSpice/SPICE to check your results.

NOTE: The following analysis can also be applied to MOSFET amplifiers by substituting a very large value of the B-E resistance \(r_{\pi}\), tending to infinity, \( 1 / r_{\pi} \simeq 0 \).

Admin
Solution

The feedback current \( i_{ f } \) is proportional to the emitter voltage \( v_{ e }=v_{ f } \), which in turn is proportional to the output current \( i_{ o } \approx-i_{ c }\). The feedback mechanism is shown in Fig. 10.49(a) and the feedback network in Fig. 10.49(b). Replacing the transistors by their small-signal model gives the small-signal AC equivalent circuit of the amplifier shown in Fig. 10.49(c),

(a) We have

\(\begin{aligned}g_{ m } &=\frac{h_{ fe }}{r_{\pi}}=\frac{150}{2.58 k }=5.814 mA / V \\R_{ B 1} &=R_{1}\left\|R_{2}=6.6 k \right\| 2 k =1.53 k \Omega \\R_{ B 2} &=R_{3}\left\|R_{4}\right\| R_{ Cl }=5 k \|10 k \| 5 k =2 k \Omega\end{aligned}\)

Step 1. Assume that there is no feedback—that is, \( R_{ F }=\infty \).

Step 2. Find the open-loop parameters A, \( R_{ i } \text {, and } R_{ o } \). The input resistance of the amplifier is

\( R_{ i }=\frac{v_{ s }}{i_{ s }}=R_{ B 1}\left\|r_{\pi 1}=1.53 k \right\| 2.58 k =962.4 \Omega \)

The output resistance of the amplifier is

\( R_{ o }=r_{ o }=\infty \quad \text { and } \quad R_{ oe }=R_{ o } \)

 

From Fig. 10.49(c), we get

\(\begin{aligned}v_{ b 1} &=v_{ bel }=R_{ i } i_{ s }=962.4 i_{ s } \\v_{ b 2} &=-\left\{R_{ B 2} \|\left[r_{\pi 2}+\left(1+g_{ m 2} r_{\pi 2}\right) R_{ E }\right]\right\} g_{ m 1} v_{ be 1} \\&=-\{2 k \|[2.58 k +(1+58.14 m \times 2.58 k ) \times 500]\} \times 58.14 m \times\left(962.4 i_{ s }\right)=-109.1 \times 10^{3} i_{ s }\end{aligned}\)

 

\(\begin{aligned}v_{ be 2} &=\frac{v_{ b 2} \times r_{\pi 2}}{\left[r_{\pi 2}+\left(1+g_{ m 2} r_{\pi 2}\right) R_{ E }\right]} \\&=\frac{\left(-109.1 \times 10^{3} i_{ s }\right) \times 2.58 k }{[2.58 k +(1+58.14 m \times 2.58 k ) \times 500]}=-3.605 i_{ s }\end{aligned}\)

 

\( i_{ o }=g_{ m 2} v_{ be 2}=58.14 m \times\left(-3.605 i_{ s }\right)=-209.2 i_{ s } \)

Therefore, the open-loop current gain A is given by \( A=i_{ o } / i_{ s }=-209.2 A / A \).

Step 3. Find the values of feedback factor \( \beta \text { and resistance } R_{ F } \). Since the closed-loop current gain \( A_{f} \) is 10% of the open-loop current gain A, we can write

\( A_{ f }=\frac{|A|}{1+\beta|A|}=0.1 A \)

That is,

\( 1+\left.\beta\right|_{A} \mid=10 \quad \text { or }\left.\quad \beta\right|_{A} \mid=9 \quad \text { and } \quad \beta=\frac{9}{|A|}=\frac{9}{209.2}=0.043 \)

Since the feedback network in Fig. 10.49(b) must be represented by a current amplifier, the feedback factor β is given by

\( \beta=\left.\frac{i_{ f }}{i_{ y }}\right|_{v_{ f }=0}=\frac{R_{ E }}{R_{ E }+R_{ F }}=\frac{500}{500+R_{ F }} \)

which, for \( \beta=0.043, \text { gives } R_{ F }=11.13 k \Omega .\)

 

Step 4. Find the new open-loop parameters A, \( R_{ i } \text {, and } R_{ o } . R_{ F } \) is included by short-circuiting the shunt feedback side and severing the series feedback side. This arrangement is shown in Fig. 10.49(c) by the gray lines.
The input resistance of the amplifier is

\( R_{ i }=\frac{v_{ s }}{i_{ s }}=\left(R_{ F }+R_{ E }\right)\left\|R_{ B 1}\right\| r_{\pi 1}=(11.13 k +500)\|1.53 k \| 2.58 k =887.2 \Omega \)

The output resistance of the amplifier is

\( R_{ o }=r_{ o }=\infty \text { and } R_{ oe }=R_{ o }=\infty \)

From Fig. 10.49(c), we get

\(\begin{aligned}v_{ bl } &=v_{ bel }=R_{ i } i_{ s }=887.2 i_{ s } \\v_{ b 2} &=-\left\{R_{ B 2} \|\left[r_{\pi 2}+\left(1+g_{ m 2} r_{\pi 2}\right)\left(R_{ E } \| R_{ F }\right)\right]\right\} g_{ m 1} v_{ be 1} \\&=-\{2 k \|[2.58 k +(1+58.14 m \times 2.58 k ) \times(11.13 k \| 500)]\} \times 58.14 m \times\left(887.2 i_{ s }\right) \\&=-100.7 \times 10^{3} i_{ s }\end{aligned}\)

 

\(\begin{aligned}v_{ be 2} &=\frac{v_{ b 2} \times r_{\pi 2}}{\left[r_{\pi 2}+\left(1+g_{ m 2} r_{\pi 2}\right)\left(R_{ E } \| R_{ F }\right)\right]} \\&=\frac{\left(-100.7 \times 10^{3} i_{ s }\right) \times 2.58 k }{[2.58 k +(1+58.14 m \times 2.58 k ) \times(11.13 k \| 500)]}=-3.471 \times 10^{3} i_{ s }\end{aligned}\)

 

\( i_{ o }=g_{ m 2} v_{ be 2}=58.14 m \times\left(-3.471 \times 10^{3} i_{ s }\right)=-201.4 i_{ s } \)

 

Therefore, the open-loop current gain A is given by \( A=i_{ o } / i_{ s }=-201.4 A / A \).
The closed-loop parameters are as follows:

\( R_{ if }=\frac{R_{ i }}{1+\beta A}=\frac{887.2}{1+0.043 A / A \times 201.4 A / A }=91.8 \Omega \)

The feedback will not affect the output resistance; that is, \( R_{\text {of }}=\infty \) and

\( A_{ f }=\frac{201.4}{1+0.043 A / A \times 201.4 A / A }=20.85 A / A \)

Step 5. Repeating steps 3 to 5 for the second iteration with A = 201.4 A ⁄A, we get the following values:

\(\begin{aligned}\beta &=\frac{9}{|A|}=\frac{9}{201.4}=0.0447 \\R_{ F } &=10.69 k \Omega \\R_{ i } &=\frac{v_{ s }}{i_{ s }}=\left(R_{ F }+R_{ E }\right)\left\|R_{ B 1}\right\| r_{\pi 1}=(10.69 k +500)\|1.53 k \| 2.58 k =886.2 \Omega \\R_{ o } &=r_{ o }=\infty \\R_{ oe } &=R_{ o }=\infty \\v_{ b 1} &=v_{ bel }=R_{ i } i_{ s }=886.2 i_{ s } \\v_{ b 2} &=-\left\{R_{ B 2} \|\left[r_{\pi 2}+\left(1+g_{ m 2} r_{\pi 2}\right)\left(R_{ E } \| R_{ F }\right)\right]\right\} g_{ m 1} v_{ be 1} \\&=-\{2 k \|[2.58 k +(1+58.14 m \times 2.58 k ) \times(11.13 k \| 500)]\} \times 58.14 m \times\left(886.2 i_{ s }\right) \\&=-100.4 \times 10^{3} i_{ s }\end{aligned}\)

 

\(\begin{aligned}v_{ be 2} &=\frac{v_{ b 2} \times r_{\pi 2}}{\left[r_{\pi 2}+\left(1+g_{ m 2} r_{\pi 2}\right)\left(R_{ F } \| R_{ F }\right)\right]} \\&=\frac{\left(-100.4 \times 10^{3} i_{ s }\right) \times 2.58 k }{[2.58 k +(1+58.14 m \times 2.58 k ) \times(11.13 k \| 500)]}=-3.466 \times 10^{3} i_{ s } \\i_{ o } &=g_{ m 2} v_{ be 2}=58.14 m \times\left(-3.466 \times 10^{3} i_{ s }\right)=-201.1 i_{ s }\end{aligned}\)

 

Therefore, the open-loop current gain A is given by \( A=i_{ o } / i_{ s }=-201.1 A / A.\)

\( R_{ if }=\frac{R_{ i }}{1+\beta A }=\frac{886.2}{1+0.0447 A / A \times 201.1 A / A }=88.5 \Omega \)

 

The output resistance with feedback is

\( R_{ of }=R_{ o }(1+\beta A)=\infty \)

Since the output is taken across \( R_{ C } \), the output resistance of the amplifier is

\(\begin{aligned}&\qquad R_{ out }=R_{ of } \| R_{ C }=R_{ C }=5 k \Omega \\&\text { and } \quad A_{ f }=\frac{201.1}{1+0.0447 A / A \times 201.1 A / A }=20.13 A / A\end{aligned}\)

which is 10% of A = 201.1. Thus, there is no need for further iterations.

(b) The shunt-series feedback amplifier for PSpice simulation is shown in Fig. 10.50. The results of PSpice simulation (.TF analysis) are shown below, with hand calculations to the right:

\(\begin{array}{ll}I ( VZ ) / IS =2.002 E +01=20.02 & A_{ f }=20.13 A / A \\\text { INPUT RESISTANCE AT IS=8.803E+01=88.03 } \Omega & R_{\text {if }}=88.5 \Omega \\\text { OUTPUT RESISTANCE AT } I ( VZ )=7.991 E +13=79.91 T \Omega & R_{\text {of }}=\infty\end{array}\)

 

NOTE: The output current is measured through a dummy voltage source \( V_{ Z } \text { connected across } R_{ C 2} \text {. } \) If \( V_{Z} \)were connected in series with \( R_{ C 2} \), PSpice would give a very large output resistance as a result of the ideal current source of transistor \( Q _{2} \).
