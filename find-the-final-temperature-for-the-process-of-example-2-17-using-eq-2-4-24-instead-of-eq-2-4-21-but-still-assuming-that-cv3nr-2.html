Guest Question.

Find the final temperature for the process of , using Eq. (2.4-24):\(\Bigl(\frac{T_{2}}{T_{1}}\Bigr)=\Bigl(\frac{V_{1}-nb}{V_{2}-nb} \Bigr)^{nR/C_{V}}\) instead of Eq. (2.4-21):\(\Bigl(\frac{T_{2}}{T_{1}}\Bigr)=\Bigl(\frac{V_{1}}{V_{2}} \Bigr)^{nR/C_{V}}=\Bigl(\frac{V_{1}}{V_{2}} \Bigr)^{R/C_{V,m}}\), but still assuming that \(C_{V}=3nR/2\).

Admin
Solution

From Table A.3, the van der Waals parameter b is equal to \(3.219×10^{-5} m^{3} mol^{-1}\) for argon.

\(T_{2}=(373.15K)\Bigl(\frac{5.000×10^{-3}m^{3}mol^{-1}-3.22×10^{-5}m^{3}mol^{-1}}{20.00×10^{-3}m^{3}mol^{-1}-3.22×10^{-5}m^{3}mol^{-1}} \Bigr)^{2/3}=147.6K\)

This value differs from the ideal value by 0.5 K.
