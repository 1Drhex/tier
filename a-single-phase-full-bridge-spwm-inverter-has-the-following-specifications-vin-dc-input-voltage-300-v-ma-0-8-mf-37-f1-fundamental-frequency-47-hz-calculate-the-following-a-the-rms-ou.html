Guest Question.

A single-phase full-bridge SPWM inverter has the following specifications:

\(V_{in}\) = dc input voltage = 300 V, \(m_a = 0.8, m_f = 37\).

\(f_1\) = fundamental frequency = 47 Hz. Calculate the following:

a) The rms output line-to-line voltage of the fundamental component.

b) The rms value and frequency of the dominant harmonic component.

Admin
Solution

a) Using Eq. (\(\widehat{V}_{ab,1}\)=amplitude of inverter output line-to-line voltage fundamental component

=\(\widehat{V}_{ao,1} -\widehat{V}_{bo,1} =\widehat{V}_{ao,1}-(-\widehat{V}_{ao,1}) = 2\widehat{V}_{ao,1}=V_{in} m_a\)

where \(V_{in}\) = dc input voltage and \(m_a\) = amplitude modulation factor.):

\(\widetilde{V}_{ab,1} =\frac{300 \times 0.8}{\sqrt{2}}=169.7 V\)

b) Using Eq. (d = order of dominant harmonic =\( 2m_f – 3\)) and the results of Table 6.4(b) for \(m_a = 0.8\) and \(m_f = 38\), the following results are obtained:

\(d = 2m_f – 3 = 2 \times 37 – 3 = 71th\) with frequency \(71 \times 47 = 3337 Hz\)

 

\(\widetilde{V}_{ab,71} =\frac{0.139 \times 300}{\sqrt{2}}=29.5 V\)
