Guest Question.

Consider a linear system governed by the first-order differential equation

\( c \dot{x} (t)+kx(t)=f(t) \)

The accompanying sketch shows one physical system that is governed by this differential equation, with \( k \) being a spring stiffness, \( c \) a dashpot value, and \( f(t) \) an applied force. Find the impulse response function \( h_{x}(t) \) such that Eq. 5.2

\( x(t)=\int_{-\infty }^{\infty }{f(s)h_{x}(t-s)ds} \equiv \int_{-\infty }^{\infty }{f(t-r)h_{x}(r)dr} \)

describes the solution of the problem.

Admin
Solution

Rewriting Eqs. 5.9

\(\sum\limits_{j=0}^{n}{a_{j}\frac{d^{j}h_{x}(t)}{dt^{j}} } =0\)

and 5.11

\(\left(\frac{d^{n-1}h_{x}(t)}{dt^{n-1}} \right) _{t=0^{+}}=a_{n}^{-1}\)

gives

\( c\frac{dh_{x}(t)}{dt} +kh_{x}(t)=0 \) for \( t\gt 0 \)

With \( h_{x}(0^{+})=c^{-1} \) . Taking the usual approach of trying an exponential form for the solution of a linear differential equation, we try \( h_{x}(t)= ae^{bt} \) . This has the proper initial condition if \( a=c^{-1} \) and it satisfies the equation if \( cb+k = 0 \) , or \( b=-k/c \) , so the answer is

\( h_{x}(t)=c^{-1}e^{-kt/c}U(t) \)

Taking the derivative of this function gives

\( h^{'}_{x}(t)=-kc^{-2}e^{-kt/c}U(t)+c^{-1}e^{-kt/c}\delta (t) =-kc^{-2}e^{-kt/c}U(t)+c^{-1}\delta (t) \)

in which the last term has been simplified by taking advantage of the fact that \( e^{-kt/c}=1 \) at the one value for which \( \delta (t) \) is not zero. Substitution of \( h_{t} \) and \( h^{'}_{x}(t) \) confirms that we have, indeed, found the solution of the differential equation \( c\acute{h} _{x}(t)+kh_{x}(t)=\delta (t) \).

Note also that

\( h_{x,staic}\equiv \int_{-\infty }^{\infty }{h_{x}(r)dr=k^{-1}\lt \infty } \)

so the response for this system has a bounded static value. For the physical model shown, it is fairly obvious that the static response to force \( f_{0} \) should be \( f_{0}/k \) , so it should be no surprise that \( h_{x,static}= k^{-1} \)
