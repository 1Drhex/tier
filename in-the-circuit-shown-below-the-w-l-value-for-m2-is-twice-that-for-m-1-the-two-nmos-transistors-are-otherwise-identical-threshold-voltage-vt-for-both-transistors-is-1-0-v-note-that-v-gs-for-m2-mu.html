Guest Question.

In the circuit shown below, the (W/L) value for \(M_{2}\) is twice that for \(M_{1}\). The two nMOS transistors are otherwise identical. Threshold voltage \(V_{ T }\) for both transistors is 1.0 V. Note that \(V_{ GS }\) for \(M_{2}\) must be > 1.0V.

Current through the nMOS transistors can be modelled as:

\(\begin{aligned}&I_{ DS }=\mu C_{ ox }\left(\frac{W}{L}\right)\left(\left(V_{ GS }-V_{ T }\right) V_{ DS }-\frac{1}{2} V_{ DS }^{2}\right) \text { for } V_{ DS } \leq \\&V_{ GS }-V_{ T } \\&I_{ DS }=\mu C_{ ox }\left(\frac{W}{L}\right)\left(V_{ GS }-V_{ T }\right)^{2} / 2 \text { for } V_{ DS } \geq V_{ GS }-V_{ T }\end{aligned}\)

The voltage (in volts, accurate to two decimal places) at \(V_{ x }\) is _______.

Admin
Solution

Consider the figure given below

From the above figure we have

\(\begin{aligned}M_{2}: V_{ GS 2} &=2-V_{ x } \\V_{ DS 2} &=3.3-V_{ x } \\V_{ DS 2} & \geq V_{ GS 2}-V_{ t } \\3.3-V_{ x } & \geq 1-V_{ x }\end{aligned}\)

Therefore \(M_{2}\) operates in saturation region.

\(\begin{aligned}M_{1}: V_{ GS 1} &=2 \\V_{ DS 1} &=V_{ x } \\V_{ x } & \leq 1\end{aligned}\)

We know that \(M_{1}\) operates in linear region

\(\begin{gathered}I_{ D 1}=I_{ D 2} \\\mu_{ n } c_{ ox }\left(\frac{\omega}{L}\right)\left[\left(V_{ GS 1}-V_{ T }\right)\right] V_{ DS }-\frac{V_{ DS }^{2}}{2} \\=\frac{1}{2} \mu_{ n } c_{ ox }\left(\frac{\omega}{L}\right)\left[\left(V_{ GS 2}-V_{ T }\right)\right]^{2} \\\mu_{ n } c_{ ox }\left(\frac{\omega}{L}\right)[(2-1)] V_{x}-\frac{V_{x}^{2}}{2} \\=\frac{1}{2} \mu_{ n } c_{ ox }\left(\frac{2 \omega}{L}\right)\left[2-V_{x}-1\right]^{2} \\3 V_{x}^{2}-6 V_{x}+2=0 \\V_{ x }=1 \pm \frac{1}{\sqrt{3}} V\end{gathered}\)

 

\(V_{ x } \text { cannot be } 1+\frac{1}{\sqrt{3}} \text { V. So, } V_{ x } \text { should be } 1-\frac{1}{\sqrt{3}}=0.42\) V

Hence, the correct answer is 0.41 to 0.435.
