Guest Question.

The crankshaft is subjected to torque \(M\). Determine the horizontal compressive force \(F\) applied to the piston for equilibrium when \(\theta = { \theta }_{ 0 }\)

Given:

\(M\) = \(50 N m \)

\({ \theta }_{ 0 }\) = \(60 deg \)

\(a\) = \(100 mm \)

\(b\) = \(400 mm\)

Admin
Solution
\(\theta = { \theta }_{ 0 }\)

 

\({ b }^{ 2 }={ a }^{ 2 }+{ x }^{ 2 }-2ax\cos { \left( \theta \right) }\)

 

\(0=2x\delta x-2a\cos { \left( \theta \right) } \delta x+2ax\sin { \left( \theta \right) } \delta \theta\)

 

\(\delta x=\left( \frac { ax\sin { \left( \theta \right) } }{ x-a\cos { \left( \theta \right) } } \right) \delta \theta\)

 

\(\delta U=-F\delta x+M\delta \theta =\left[ -F\left( \frac { ax\sin { \left( \theta \right) } }{ x-a\cos { \left( \theta \right) } } \right) +M \right] \delta \theta =0\)

 

Guesses \(\quad x=1m\quad\quad\quad \quad F=1N\)

 

Given \(\quad \quad { b }^{ 2 }={ a }^{ 2 }+{ x }^{ 2 }-2ax\cos { \left( \theta \right) } \quad \quad\quad -F\left( \frac { ax\sin { \left( \theta \right) } }{ x-a\cos { \left( \theta \right) } } \right) +M=0\)

 

\(\left( \begin{matrix} F \\ x \end{matrix} \right) =Find\left( F,x \right) \quad \quad x=440.512mm\quad \quad F=512N\)
