*PDF refers to Probability Density Function (see Section 1–12).
†To compare discrete frequency data with continuous density functions, \( f_{i}\) must be divided by \( N_{w}\) . Here, N = sample size = 1000, and w = width of range interval = 1 kpsi

From Equation (1–6),

\( \bar{x}=\frac{1}{N} \sum\limits_{i=1}^{k}{f_{i}x_{i}}=\frac{1}{1000}(63 625) = 63.625 kpsi \)

From Equation (1–7),

\( s_{x}= \sqrt{\frac{\sum\limits_{i=1}^{k}{f_{i}x_{i}^{2}-N\bar{x}^{2} } }{N-1} } =\sqrt{\frac{4 054 864 − 1000(63.625^{2})}{1000-1} } = 2.594 245 = 2.594 kpsi\)

From Equation (1–4), with \( μ_{x} = \bar{x} and \hat{\sigma }_{x} = s_{x}\) , the PDF for a normal density function is

\( f(x)= \frac{1}{\hat{\sigma }_{x}\sqrt{2\pi } } exp \left[-\frac{1}{2} (\frac{x-\mu _{x}}{\hat{\sigma }_{x}} )^{2} \right] \)

 

\( =\frac{1}{2.594 245 \sqrt{2\pi } } exp \left[-\frac{1}{2}(\frac{x − 63.625}{2.594 245} )^{2} \right] \)

For example, f (63.5) = 0.1536.
