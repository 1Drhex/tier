Guest Question.

In the mechanism shown in Fig. 8.7, the slider C is moving to the right with a velocity of 1 m/s and an acceleration of 2.5 m\(/ s^{2}\).
The dimensions of various links are AB = 3 m inclined at \(45^{\circ}\) with the vertical and BC = 1.5 m inclined at \(45^{\circ}\) with the horizontal. Determine: 1. the magnitude of vertical and horizontal component of the acceleration of the point B, and 2. the angular acceleration of the links AB and BC.

Admin
Solution

Given : \(v_{ C }=1 m / s ; a_{ C }=2.5 m / s ^{2}\) ; A B = 3 m ; BC = 1.5 m

First of all, draw the space diagram, as shown in Fig. (a), to some suitable scale. Now the velocity diagram, as shown in Fig.(b), is drawn as discussed below:

1. Since the points A and D are fixed points, therefore they lie at one place in the velocity diagram. Draw vector dc parallel to DC, to some suitable scale, which represents the velocity of slider C with respect to D or simply velocity of C, such that

\(\text { vector } d c=v_{ CD }=v_{ C }=1 m / s\)

2. Since point B has two motions, one with respect to A and the other with respect to C, therefore from point a, draw vector ab perpendicular to A B to represent the velocity of B with respect to A, i.e. \(v_{ BA }\) and from point c draw vector cb perpendicular to CB to represent the velocity of B with respect to C i.e. \(v_{ BC }\) .The vectors ab and cb intersect at b.

By measurement, we find that velocity of B with respect to A,

\(v_{ BA }=\text { vector } a b=0.72 m / s\)

and velocity of B with respect to C,

\(v_{ BC }=\text { vector } c b=0.72 m / s\)

We know that radial component of acceleration of B with respect to C,

\(a_{ BC }^{r}=\frac{v_{ BC }^{2}}{C B}=\frac{(0.72)^{2}}{1.5}=0.346 m / s ^{2}\)

and radial component of acceleration of B with respect to A,

\(a_{ BA }^{r}=\frac{v_{ BA }^{2}}{A B}=\frac{(0.72)^{2}}{3}=0.173 m / s ^{2}\)

Now the acceleration diagram, as shown in Fig.(c), is drawn as discussed below:

1. Since the points A and D are fixed points, therefore they lie at one place in the acceleration diagram. Draw vector \(d^{\prime} c^{\prime}\) parallel to DC, to some suitable scale, to represent the acceleration of C with respect to D or simply acceleration of C i.e. \(a_{ CD } \text { or } a_{ C }\) such that

\(\text { vector } d^{\prime} c^{\prime}=a_{ CD }=a_{ C }=2.5 m / s ^{2}\)

2. The acceleration of B with respect to C will have two components, i.e. one radial component of B with respect to \(C\left(a_{ BC }^{r}\right)\) and the other tangential component of B with respect to\(C\left(a_{ BC }^{t}\right)\) Therefore from point \(c^{\prime}\), draw vector \(c^{\prime}\) x parallel to CB to represent \(a_{ BC }^{r}\) such that

\(\text { vector } c^{\prime} x=a_{ BC }^{r}=0.346 m / s ^{2}\)

3. Now from point x, draw vector \(x b^{\prime}\) perpendicular to vector \(c^{\prime} x\) or CB to represent \(a_{ BC }^{t}\) whose magnitude is yet unknown.

4. The acceleration of B with respect to A will also have two components, i.e. one radial component of B with respect to \(A\left(a_{ BA }^{r}\right)\) and other tangential component of B with respect to \(A\left(a^{t}{ }_{ BA }\right)\) Therefore from point a’ draw vector \(a^{\prime} y\) parallel to A B to represent \(a_{ BA }^{r}\) , such that

\(\text { vector } a^{\prime} y=a_{ BA }^{r}=0.173 m / s ^{2}\)

5. From point y, draw vector \(y b^{\prime}\) perpendicular to vector \(a^{\prime} y\) or AB to represent \(a_{ BA }^{t}\). The vector \(y b^{\prime}\) intersect the vector \(x b^{\prime} \text { at } b^{\prime}\). Join \(a^{\prime} b^{\prime} \text { and } c^{\prime} b^{\prime}\). The vector \(a^{\prime} b^{\prime}\) represents the acceleration of point \(B\left(a_{ B }\right)\) and the vector \(c^{\prime} b^{\prime}\) represents the acceleration of B with respect to C.

 

1. Magnitude of vertical and horizontal component of the acceleration of the point B

Draw \(b^{\prime} b^{\prime \prime}\) perpendicular to \(a^{\prime} c^{\prime}\). The vector \(b^{\prime} b^{\prime \prime}\) is the vertical component of the acceleration of the point B and \(a^{\prime} b^{\prime \prime}\) is the horizontal component of the acceleration of the point B. By measurement,

\(\text { vector } b^{\prime} b^{\prime \prime}=1.13 m / s ^{2} \text { and vector } a^{\prime} b^{\prime \prime}=0.9 m / s ^{2}\)

 

2. Angular acceleration of AB and BC

By measurement from acceleration diagram, we find that tangential component of acceleration of the point B with respect to A,

\(a_{ BA }^{t}=\text { vector } y b^{\prime}=1.41 m / s ^{2}\)

and tangential component of acceleration of the point B with respect to C,

\(a_{ BC }^{t}=\text { vector } x b^{\prime}=1.94 m / s ^{2}\)

We know that angular acceleration of A B,

\(\alpha_{ AB }=\frac{a_{ BA }^{t}}{A B}=\frac{1.41}{3}=0.47 rad / s ^{2}\)

and angular acceleration of BC,

\(\alpha_{ BC }=\frac{a_{ BA }^{t}}{C B}=\frac{1.94}{1.5}=1.3 rad / s ^{2}\)
