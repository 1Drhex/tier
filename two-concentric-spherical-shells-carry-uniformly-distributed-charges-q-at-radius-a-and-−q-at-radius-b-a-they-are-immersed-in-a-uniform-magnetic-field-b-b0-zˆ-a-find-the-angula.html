Guest Question.

Two concentric spherical shells carry uniformly distributed charges +Q (at radius a) and −Q (at radius b > a). They are immersed in a uniform magnetic field \( B =B_{0} \hat{ z }\).

(a) Find the angular momentum of the fields (with respect to the center)

(b) Now the magnetic field is gradually turned off. Find the torque on each sphere, and the resulting angular momentum of the system

Admin
Solution

(a) Between the shells, \(E =\frac{Q}{4 \pi \epsilon_{0}} \frac{1}{r^{2}} \hat{ r }, \quad g =\epsilon_{0}( E \times B )=\frac{Q B_{0}}{4 \pi r^{2}}(\hat{ r } \times \hat{ z })\) .

\(L =\int( r \times g ) d \tau=\frac{Q B_{0}}{4 \pi} \int \frac{1}{r^{2}}[ r \times(\hat{ r } \times \hat{ z })] r^{2} \sin \theta d r d \theta d \phi=\frac{Q B_{0}}{4 \pi} \int r[\hat{ r } \times(\hat{ r } \times \hat{ z })] \sin \theta d r d \theta d \phi\) .

\(\text { Now } \hat{ r } \times(\hat{ r } \times \hat{ z })=\hat{ r }(\hat{ r } \cdot \hat{ z })-\hat{ z }(\hat{ r } \cdot \hat{ r })=\hat{ r } \cos \theta=\hat{ z }\) , but L has to be along the z direction, so we pick off the z component of \(\hat{ r }:[\hat{ r } \times(\hat{ r } \times \hat{ z })]_{z}=\cos ^{2} \theta-1=-\sin ^{2} \theta\).

\(L_{z}=-\frac{Q B_{0}}{4 \pi} \int r \sin ^{3} \theta d r d \theta d \phi=-\frac{Q B_{0}}{4 \pi} 2 \pi \int_{0}^{\pi} \sin ^{3} \theta d \theta \int_{a}^{b} r d r=-\frac{Q B_{0}}{2}\left(\frac{4}{3}\right)\left(\frac{b^{2}-a^{2}}{2}\right)=-\frac{1}{3} Q B_{0}\left(b^{2}-a^{2}\right)\).

\(L =-\frac{1}{3} Q B_{0}\left(b^{2}-a^{2}\right) \hat{ z }\) .

(b) The changing magnetic field induces an electric field (). Assuming symmetry about the z axis, \(E =-\frac{s}{2} \frac{d B}{d t} \hat{ \phi }\) . The force on a patch of area da is \(d F = E \sigma d a\), and the torque on this patch is \(N = s \times d F\). The net torque on the sphere at radius a is (using \(s =a \sin \theta \hat{ s } \text { and } d a=a^{2} \sin \theta d \theta d \phi \) ):

\(N _{a}=-\left(\frac{Q}{4 \pi a^{2}}\right)\left(\frac{1}{2} \frac{d B}{d t} \hat{ z }\right) \int s^{2} d a=-\frac{Q a^{2}}{8 \pi} \frac{d B}{d t} \hat{ z } 2 \pi \int \sin ^{3} \theta d \theta=\frac{Q a^{2}}{3} \frac{d B}{d t} \hat{ z }\).

Similarly, \(N _{b}=\frac{Q b^{2}}{3} \frac{d B}{d t} \hat{ z }, \text { so the total torque is } N =\frac{Q}{3} \frac{d B}{d t}\left(b^{2}-a^{2}\right) \hat{ z }\) , and the angular momentum delivered to the spheres is

\(L =\int N d t=\frac{Q}{3}\left(b^{2}-a^{2}\right) \hat{ z } \int_{B_{0}}^{0} \frac{d B}{d t} d t=-\frac{Q}{3}\left(b^{2}-a^{2}\right) B_{0} \hat{ z }\)
