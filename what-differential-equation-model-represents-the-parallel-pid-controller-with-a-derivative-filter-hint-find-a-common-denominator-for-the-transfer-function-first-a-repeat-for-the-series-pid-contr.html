Guest Question.

What differential equation model represents the parallel PID controller with a derivative filter? (Hint: Find a common denominator for the transfer function first.)

(a) Repeat for the series PID controller with a derivative filter.

(b) Simulate the time response of each controller for a unit step change in \(e(t)\) and the following parameter values:

\(K_{c}=2, \quad \tau_{I}=3 \min , \quad \tau_{D}=5 \min , \quad \alpha=0.1, \quad M=1\)

Admin
Solution

a)

\(\begin{aligned}\frac{P^{\prime}(s)}{E(s)} &=K_{c}\left(1+\frac{1}{\tau_{I} s}+\frac{\tau_{D} s}{\alpha \tau_{D} s+1}\right) \\&=K_{c} \frac{\left(\tau_{I} s\left(\alpha \tau_{D} s+1\right)+\alpha \tau_{D} s+1+\tau_{D} s \tau_{I} s\right)}{\tau_{I} s\left(\alpha \tau_{D} s+1\right)} \\&=K_{c}\left[\frac{1+\left(\tau_{I}+\alpha \tau_{D}\right) s+(1+\alpha) \tau_{I} \tau_{D} s^{2}}{\tau_{I} s\left(\alpha \tau_{D} s+1\right)}\right]\end{aligned}\)

Cross- multiplying

\(\left(\alpha \tau_{I} \tau_{D} s^{2}+\tau_{I} s\right) P^{\prime}(s)=K_{c}\left(1+\left(\tau_{I}+\alpha \tau_{D}\right) s+(1+\alpha) \tau_{I} \tau_{D} s^{2}\right) E(s)\)

Taking inverse Laplace transforms gives,

\(\alpha \tau_{I} \tau_{D} \frac{d^{2} p^{\prime}(t)}{d t^{2}}+\tau_{I} \frac{d p^{\prime}(t)}{d t}=K_{c}\left(e(t)+\left(\tau_{I}+\alpha \tau_{D}\right) \frac{d e(t)}{d t}+(1+\alpha) \tau_{I} \tau_{D} \frac{d^{2} e(t)}{d t^{2}}\right)\)

b) \(\quad \frac{P^{\prime}(s)}{E(s)}=K_{c}\left(\frac{\tau_{I} s+1}{\tau_{I} s}\right)\left(\frac{\tau_{D} s}{\alpha \tau_{D} s+1}\right)\)

Cross-multiplying

\(\tau_{I} s^{2}\left(\alpha \tau_{D} s+1\right) P^{\prime}(s)=K_{c}\left(\left(\tau_{I} s+1\right)\left(\tau_{D} s+1\right)\right) E(s)\)

\(\alpha \tau_{I} \tau_{D} \frac{d^{2} p^{\prime}(t)}{d t^{2}}+\tau_{I} \frac{d p^{\prime}(t)}{d t}=K_{c}\left(e(t)+\left(\tau_{I}+\tau_{D}\right) \frac{d e(t)}{d t}+\tau_{I} \tau_{D} \frac{d^{2} e(t)}{d t^{2}}\right)\)

c) The simulation is performed for the following parameter values:

\(K_{c}=2, \quad \tau_{I}=3, \quad \tau_{D}=0.5 \quad, \quad \alpha=0.1 \quad, \quad M=1\)
