Guest Question.

Reactant gas \(\left(u_{0}=0.3 \mathrm{~m} / \mathrm{s}, v_{0}=0.3 \pi \mathrm{m}^{3} / \mathrm{s}\right.\) passes upward through a \(2-\mathrm{m}\) diameter fluidized bed \(\left(u_{m f}=0.03 \mathrm{~m} / \mathrm{s}, \epsilon_{m f}=0.5\right)\) containing 7 tons of catalyst \((W=7000\) \(\left.\mathrm{kg}, \rho_{s}=2000 \mathrm{~kg} / \mathrm{m}^{3}\right) .\) Reaction proceeds as follows:

\(\mathrm{A} \rightarrow \mathrm{R}, \quad-r_{\mathrm{A}}^{\prime \prime \prime}=k^{\prime \prime \prime} C_{\mathrm{A}} \ldots . \text { with } k^{\prime \prime \prime}=0.8 \frac{\mathrm{m}^{3}}{\mathrm{~m}^{3} \mathrm{~s} \cdot \mathrm{s}}\)

(a) Calculate the conversion of reactant.

(b) Find the proper mean concentration of A seen by the solids.

(c) If gas were made to flow downward through the solids we would have a packed bed. Assuming plug flow of gas find the conversion of reactant for this situation.

Additional Data

\(C_{\mathrm{A} 0}=100 \mathrm{~mol} / \mathrm{m}^{3}, \mathscr{D}=20 \times 10^{-6} \mathrm{~m}^{2} / \mathrm{s}, \alpha=0.33\)

Estimated bubble size in the bed: \(d_{b}=0.32 \mathrm{~m}\) See Fig. E20.1 which represents this system.

Admin
Solution

Preliminary Determine the rise velocity of bubbles

\(\begin{aligned}u_{b r} &=0.711(9.8 \times 0.32)^{1 / 2}=1.26 \mathrm{~m} / \mathrm{s} \\u_{b} &=0.30-0.03+1.26=1.53 \mathrm{~m} / \mathrm{s}\end{aligned}\)

Now see whether the fast bubble model of this chapter applies.

– Check for slugging: The bubble size \((32 \mathrm{~cm})\) is small compared to the bed size \((200 \mathrm{~cm})\), hence no slugging.

– Check the fast bubble assumption: Take the velocity ratio

\(\frac{u_{b}}{u_{f}}=\frac{u_{b}}{u_{m f} / \varepsilon_{m f}}=\frac{1.53}{0.03 / 0.5}=25.5\)

Since the bubble rises \(\sim 25\) times as fast as the emulsion gas we have a fast bubble with thin cloud-less than \(1 \mathrm{~cm}\) thick.

Thus we can safely use the bubbling bed model of this chapter.

(a) Calculate \(X_{\mathrm{A}}\). Replacing numbers into the expressions for this model gives, in turn,

\(\begin{aligned}\delta &=\frac{0.30}{1.53}=0.196 \\\epsilon_{f} &=1-\left(1-\epsilon_{m f}\right)(1-\delta)=1-0.5(1-0.196)=0.60 \\K_{b c} &=4.50\left(\frac{0.03}{0.32}\right)+5.85\left(\frac{\left(20 \times 10^{-6}\right)^{1 / 2}(9.8)^{1 / 4}}{(0.32)^{5 / 4}}\right)=0.614 \mathrm{~s}^{-1} \\K_{c e} &=6.77\left(\frac{0.5\left(20 \times 10^{-6}\right) 1.26}{(0.32)^{3}}\right)^{1 / 2}=0.133 \mathrm{~s}^{-1}\end{aligned}\)

 

\(\begin{aligned}f_{b} &=0.001 \sim 0.01, \text { choose it to be } 0.001 \\f_{c} &=0.196(1-0.5)\left(\frac{3 \times 0.03 / 0.5}{1.26-0.03 / 0.5}+0.33\right)=0.047 \\f_{e} &=(1-0.6)-0.047-0.001=0.352 \\H_{\mathrm{BFB}} &=\frac{7000}{2000(\pi)(1-0.6)}=2.785 \mathrm{~m}\end{aligned}f_{b}+f_{c}+f_{e}=\left(1-\epsilon_{f}\right)=0.4\)

 

\(\ln \frac{C_{\mathrm{A} 0}}{C_{\mathrm{A}}}=\) \(\frac{\left[0.001(0.8)+\frac{1}{\frac{1}{0.196(0.614)}+\frac{1}{0.047(0.8)+\frac{1}{\frac{1}{0.196(0.133)}+\frac{1}{0.354(0.8)}}}}\right]}{(0.001+0.047+0.352)}\) \(.\frac{2.785(0.4)}{0.3}\) \(=[0.0415] \cdot(9.284)=0.385\)

Therefore

\(\frac{C_{\mathrm{A}}}{C_{\mathrm{A} 0}}=0.68 . . . \text { or } \underline{\underline{X}_{\mathrm{A}}=32 \%}(\text { see Fig. } 20.10)\)

(b) Find \(\bar{C}_{\mathrm{A}}\) seen by the solids. Since every particle samples all the gas in the bed, \(\tau^{\prime \prime \prime}=\left(C_{\mathrm{A} 0}-C_{\mathrm{A}}\right) /-r_{\mathrm{A}}^{\prime \prime},\) or

\(\bar{C}_{\mathrm{A}}=\frac{\left(C_{\mathrm{A} 0}-C_{\mathrm{A}}\right) v_{0}}{k^{\prime \prime \prime} V_{s}}=\frac{(100-68) 0.3 \pi}{(0.8)(3.5)}=\underline{\underline{11 \mathrm{~mol} / \mathrm{m}^{3}}}\)

(c) Calculate \(X_{\mathrm{A}}\) for a fixed bed. for plug flow,

\(\ln \frac{C_{\mathrm{A} 0}}{C_{\mathrm{A} p}}=k^{\prime \prime \prime} \tau^{\prime \prime \prime}=k^{\prime \prime \prime} \frac{f_{\text {total }} H_{\mathrm{BFB}}}{u_{0}}=k^{\prime \prime \prime} \frac{V_{s}}{v_{0}}=(0.8)\left(\frac{3.5}{0.3 \pi}\right)=2.97\)

Therefore

\(\frac{C_{\mathrm{A} p}}{C_{\mathrm{A} 0}}=0.05 \quad \ldots \text { or } \quad \underline{X_{\mathrm{A}}=95 \%}\)

This point is shown in Fig. 20.10 .
