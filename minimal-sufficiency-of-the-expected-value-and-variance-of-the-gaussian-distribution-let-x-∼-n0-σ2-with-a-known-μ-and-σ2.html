Guest Question.

Minimal Sufficiency of the Expected Value and Variance of the Gaussian Distribution

Let \(x\sim N(0,\sigma ^{2})\) with a known μ and \(σ^{2}\). Find a minimal sufficient statistic for \(\theta =[\mu ,\sigma ^{2}]^{T}\) and for the IID samples \(x=[x_{1},...,x_{n}]^{T}.\)
.

Admin
Solution

\(f_{\theta }(x)=(\frac{1}{\sqrt{2\pi \sigma ^{2}} } )^{n}e^{-\frac{1}{2\sigma ^{2}}\sum\limits_{i=1}^{n}{(x_{i}-\mu )^{2}} }\)
\(=(\frac{1}{\sqrt{2\pi \sigma ^{2}} } )^{n}e^{-\frac{1}{2\sigma ^{2}}(\sum\limits_{i=1}^{n}{x_{i}^{2}-2\mu \sum\limits_{i=1}^{n}{x_{i}+n\mu ^{2}}})}\)
\(=\underbrace{(\frac{1}{\sqrt{2\pi \sigma ^{2}} } )^{n}e^{-\frac{n\mu ^{2}}{2\sigma ^{2}} }e^{[\frac{\mu }{\sigma ^{2}},-\frac{1}{2\sigma ^{2}} ]\overbrace{[\sum\limits_{i=1}^{n}{x_{i}},\sum\limits_{i=1}^{n}{x_{i}^{2}}]^{T}}^{T(x)} }}_{g(T,\theta )}\underbrace{1}_{h(x)}.\)

It is true that

\(T(x)=[\underbrace{ \sum\limits_{i=1}^{n}{x_{i}}}_{T_{1}},\underbrace{ \sum\limits_{i=1}^{n}{x_{i}^{2}}}_{T_{2}} ]^{T}\)

is a (jointly) sufficient statistic for \(μ, σ^{2}\). Also, since \(q(T)=[n^{-1}T_{1},(n-1)^{-1}(T_{2}-T_{1}^{2})]\) is a 1-1 function of T, then \(s=[\bar{x},\sigma ^{2} ]\)expresses sufficiency.

As in , the minimal sufficiency of this statistical quantity results from the trajectory of the likelihood ratio\( \left\{\Lambda _{\theta }(x)\right\}_{\theta \in \Theta }\) with \(θ = [μ, σ^{2}], Θ = ℝ × ℝ^{+}\). Choosing arbitrarily a reference point \(\theta _{0}=[\mu _{0},\sigma _{0}^{2}]=[0,1]\), we obtain:

\(\Lambda _{\theta }=\frac{f_{\theta }(x)}{f_{\theta _{0}}(x)}=(\frac{\sigma _{0}}{\sigma } )^{n} e^{(-\frac{n\mu ^{2}}{2\sigma ^{2}} )}e^{[\frac{\mu }{\sigma ^{2}},-\frac{\delta }{2} ][\sum\limits_{i=1}^{n}{x_{i}},\sum\limits_{i=1}^{n}{x_{i}^{2}}]^{T}}\)

where \(\delta =\frac{\sigma _{0}^{2}-\sigma ^{2}}{\sigma ^{2}\sigma _{0}^{2}}\) .Setting \(\tau = [\sum\limits_{i=1}^{n}{x_{i}},\sum\limits_{i=1}^{n}{x_{i}^{2}}] \)Condition 1 is satisfied again. Condition 2 needs further analysis. Although \(Λ_{θ}(τ)\) is no longer an invertible function to τ for each value of \(θ = [μ, σ^{2}]\), we can obtain two values θ ∈ [ \({θ_{1}, θ_{2}}\) ] in Θ, for which the function of vectors \([\Lambda _{\theta _{1}}(\tau ),\Lambda _{\theta _{2}}(\tau )]\) is invertible to τ. Since this vector is defined by Λ(τ), vector τ ‘follows’ the likelihood ratios.
In order to formulate the invertibility relation, let \(λ = [λ_{1}, λ_{2}]^{T}\) represent a pair of observed samples, resulting from the vector \([\Lambda _{\theta _{1}}(\tau ),\Lambda _{\theta _{2}}(\tau )]^{T}\) at Λ(τ) surface. Now the problem is to define τ from equation \(λ =[\Lambda _{\theta _{1}}(\tau ),\Lambda _{\theta _{2}}(\tau )]^{T}\). By talking the logarithm of each side of the equation, we find that it can equally be expressed as a 2 × 2 linear system of equations of the form λ’ = Aτ, where A is a matrix of the elements \(θ_{0}, θ_{1}, θ_{2}\) and λ’ is a linear function of ln λ. As a typical example, we verify the above if we choose \(θ_{0} =[0, 1], θ_{1} = [1, 1], θ_{2} = [0, 1/2]\). Then we have, δ = 0 or 1 for \(θ = θ_{1}\) or \(θ_{2},\) respectively, and A = diag(1, –1/2) is an invertible matrix. Therefore, we conclude that the vector [expected value, variance] of samples indexes the trajectories, thus expresses minimal sufficiency.
