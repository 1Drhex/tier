Guest Question.

Derive the shape function derivatives for a one-dimensional quadratic element that has nodal coordinates \( x_{i}=2, x_{j}=4\) and \(x_{k}=6 . \)

 

Admin
Solution

The Jacobian is written as

\([J]=\frac{d x}{d \zeta}\)

\(\begin{array}{l}=\frac{d N_{i}}{d \zeta} x_{i}+\frac{d N_{j}}{d \zeta} x_{j}+\frac{d N_{k}}{d \zeta} x_{k} \\=\left(-\frac{1}{2}+\zeta\right) 2+(-2 \zeta) 4+\left(\frac{1}{2}+\zeta\right) 6=2.0 ;\end{array}\) (3.108)

thus,

\([J]^{-1}=\frac{1}{2}\) (3.109)

\(\text { The shape function derivatives are written as follows: }\)\(\left\{\begin{array}{l}\frac{d N_{i}}{d x} \\\frac{d N_{j}}{d x} \\\frac{d N_{k}}{d x}\end{array}\right\}=[J]^{-1}\left\{\begin{array}{c}\frac{d N_{i}}{d \zeta} \\\frac{d N_{j}}{d \zeta} \\\frac{d N_{k}}{d \zeta}\end{array}\right\}=\frac{1}{2}\left\{\begin{array}{c}-\frac{1}{2}+\zeta \\-2 \zeta \\\frac{1}{2}+\zeta\end{array}\right\}\) (3.110)

\(\text { For two-dimensional cases, we may express } x \text { and } y \text { as functions of } \zeta \text { and } \eta \text { , that is, }\)

\(x=x(\zeta, \eta) \quad \text { and } \quad y=y(\zeta, \eta)\) (3.111)

Since we deal with Cartesian derivatives for the calculation of the stiffness matrix, we transform the derivatives of the shape functions using the chain rule as

\(\begin{array}{l}\frac{\partial N_{i}}{\partial \zeta}(x, y)=\frac{\partial N_{i}}{\partial x} \frac{\partial x}{\partial \zeta}+\frac{\partial N_{i}}{\partial y} \frac{\partial y}{\partial \zeta} \\\frac{\partial N_{i}}{\partial \eta}(x, y)=\frac{\partial N_{i}}{\partial x} \frac{\partial x}{\partial \eta}+\frac{\partial N_{i}}{\partial y} \frac{\partial y}{\partial \eta}\end{array}\) (3.112)

which can be written as

\(\left\{\begin{array}{l}\frac{\partial N_{i}}{\partial \zeta} \\\frac{\partial N_{i}}{\partial \eta}\end{array}\right\}=\left[\begin{array}{ll}\frac{\partial x}{\partial \zeta} & \frac{\partial y}{\partial \zeta} \\\frac{\partial x}{\partial \eta} & \frac{\partial y}{\partial \eta}\end{array}\right]\left\{\begin{array}{l}\frac{\partial N_{i}}{\partial x} \\\frac{\partial N_{i}}{\partial y}\end{array}\right\}=[ J ]\left\{\begin{array}{l}\frac{\partial N_{i}}{\partial x} \\\frac{\partial N_{i}}{\partial y}\end{array}\right\}\) (3.113)

Where

\([ J ]=\left[\begin{array}{ll}\frac{\partial x}{\partial \zeta} & \frac{\partial y}{\partial \zeta} \\\frac{\partial x}{\partial \eta} & \frac{\partial y}{\partial \eta}\end{array}\right]\) (3.114)

Therefore, we can write

\(\left\{\begin{array}{l}\frac{\partial N_{i}}{\partial x} \\\frac{\partial N_{i}}{\partial y}\end{array}\right\}=[ J ]^{-1}\left\{\begin{array}{l}\frac{\partial N_{i}}{\partial \zeta} \\\frac{\partial N_{i}}{\partial \eta}\end{array}\right\}\) (3.115)

\(\text { Note that the inverse of the Jacobian matrix }[ J ]^{-1} \text { is calculated as }\)

\([ J ]^{-1}=\frac{1}{\operatorname{det}[ J ]}\left[\begin{array}{cc}\frac{\partial y}{\partial \eta} & -\frac{\partial y}{\partial \zeta} \\-\frac{\partial x}{\partial \eta} & \frac{\partial x}{\partial \zeta}\end{array}\right]\) (3.116)

The derivatives have to be numerically evaluated at each integration point, as a closed form solution does not exist.For an eight-node isoparametric element (Figure 3.18), the values of the temperature T at any point are given by

\(T=\sum_{i=1}^{8} N_{i} T_{i}\) (3.117)

The coordinate values of x and y at any point within an element are given by the following expressions:

\(\begin{aligned}x(\zeta, \eta) &=\sum_{i=1}^{8} N_{i}(\zeta, \eta) x_{i} \\y(\zeta, \eta) &=\sum_{i=1}^{8} N_{i}(\zeta, \eta) y_{i}\end{aligned}\) (3.118)

\(\text { where }\left(x_{i}, y_{i}\right) \text { are the coordinates of the node } i \text { and the quadratic shape functions are given by }\)

\(\begin{array}{l}N_{1}=-\frac{1}{4}(1-\zeta)(1-\eta)(1+\zeta+\eta) \\N_{2}=\frac{1}{2}\left(1-\zeta^{2}\right)(1-\eta) \\N_{3}=\frac{1}{4}(1+\zeta)(1-\eta)(\zeta-\eta-1) \\N_{4}=\frac{1}{2}(1+\zeta)\left(1-\eta^{2}\right) \\N_{5}=\frac{1}{4}(1+\zeta)(1+\eta)(\zeta+\eta-1) \\N_{6}=\frac{1}{2}\left(1-\zeta^{2}\right)(1+\eta) \\N_{7}=\frac{1}{4}(1-\zeta)(1+\eta)(-\zeta+\eta-1) \\N_{8}=\frac{1}{2}(1-\zeta)\left(1-\eta^{2}\right)\end{array}\) (3.119)

The \( \zeta\) and \( \eta\) variables are curvilinear coordinates and as such their direction will vary with position. The nodes of the element are input in an anti-clockwise sequence starting from any corner node. The directions of \( \zeta\) and \( \eta\) are indicated on Figure (3.18) that is, positive \( \zeta\) in the direction from nodes 1 to 3 and positive \( \eta\) in the direction from nodes 3 to 5 .
