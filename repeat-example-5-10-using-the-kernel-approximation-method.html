Guest Question.

Repeat using the kernel approximation method.

Admin
Solution

We again need to solve equation (5.49),

\(f(\xi )=\epsilon +\frac{1}{2 }(1-\epsilon )\int_{0}^{W}f(\xi ^{\prime } ) \frac{d\xi ^{\prime }}{[1+(\xi ^{\prime }-\xi )^{2} ]^{3/2} }\)

this time by approximating the kernel. For convenience we shall choose a simple exponential form,

\(K(\xi ,\xi ^{\prime })=\frac{1}{\left[1+(\xi ^{\prime }-\xi )^{2} \right] ^{3/2} } \simeq a e^{-b\left|\xi ^{\prime }-\xi\right| }\).

We shall determine “optimum” parameters a and b by letting the approximation satisfy the 0th and 1st moments. This implies multiplying the expression by \(\left|\xi ^{\prime }-\xi\right|\) raised to the 0th and 1st powers, followed by integration over the entire domain for \(\left|\xi ^{\prime }-\xi\right|\), i.e., from 0 to âˆž (since W could be arbitrarily large)\(^{3}\). Thus,

0th moment: \(\int_{0}^{\infty }\frac{dx}{(1+x^{2} )^{3/2} } =1=\int_{0}^{\infty }a e^{-bx} dx=\frac{a}{b}\),

1st moment: \(\int_{0}^{\infty }\frac{xdx}{(1+x^{2} )^{3/2} } =1=\int_{0}^{x }a e^{-bx} x dx=\frac{a}{b^{2} }\),

leading to a=b= 1 and

\(K(\xi ,\xi ^{\prime })\simeq e^{-\left|\xi ^{\prime }-\xi\right|}\).

Substituting this expression into equation (5.49)

\(f(\xi )=\epsilon +\frac{1}{2 }(1-\epsilon )\int_{0}^{W}f(\xi ^{\prime } ) \frac{d\xi ^{\prime }}{[1+(\xi ^{\prime }-\xi )^{2} ]^{3/2} }\)

leads to

\(f(\xi )\simeq \epsilon +\frac{1}{2}(1-\epsilon )\left[\int_{0}^{\xi }f(\xi ^{\prime })e^{-(\xi -\xi ^{\prime })} d\xi ^{\prime }+\int_{\xi }^{W}f(\xi ^{\prime })e^{-(\xi ^{\prime }-\xi ) }d\xi ^{\prime } \right]\).

We shall now differentiate this expression twice with respect to \(\xi\), for which we need to employ Leibnitz’ rule, equation (3.106).

\(\frac{d}{dx} \int_{a(x)}^{b(x)}f(x,y)dy=\frac{db}{dx}f(x,b)-\frac{da}{dx}f(x,a)+\int_{a}^{b}\frac{df}{dx}(x,y)dy\),

Therefore,

\(\frac{df}{f\xi } =\frac{1}{2} (1-\epsilon )\left[f(\xi )-\int_{0}^{\xi }f(\xi ^{\prime } )e^{-(\xi -\xi ^{\prime } )} d\xi ^{\prime } -f(\xi )+\int_{\xi }^{W}f(\xi ^{\prime } )e^{-(\xi ^{\prime } -\xi )}d\xi ^{\prime } \right]\),

\(\frac{d^{2}f }{d\xi ^{2} } =\frac{1}{2} (1-\epsilon )\left[-f(\xi )+\int_{0}^{\xi }f(\xi ^{\prime } )e^{-(\xi -\xi ^{\prime } )} d\xi ^{\prime } -f(\xi )+\int_{\xi }^{W}f(\xi ^{\prime } )e^{-(\xi ^{\prime } -\xi )}d\xi ^{\prime } \right]\),

or, by comparison with the expression for \(f(\xi )\),

\(\frac{d^{2}f }{d\xi ^{2} } =f-\epsilon -(1-\epsilon )f=\epsilon (f-1)\).

Thus, the governing integral equation has been converted into a second-order ordinary differential equation, which is readily solved as

\(f(\xi )=1+C_{1} e^{-\sqrt{\epsilon }\xi } +C_{2}e^{+\sqrt{\epsilon }\xi }\).

While an integral equation does not require any boundary conditions, we have converted the governing equation into a differential equation that requires two boundary conditions in order to determine \(C_{1}\) and \(C_{2}\). The dilemma is overcome by substituting the general solution back into the governing integral equation (with approximated kernel). This calculation can be done for variable values of \(\xi\) by comparing coefficients of independent functions of \(\xi\), or simply for two arbitrarily selected values for \(\xi\). The first method gives the engineer proof that his analysis is without mistake, but is usually considerably more tedious. Often it is also possible to employ symmetry, as is the case here, since \(f(\xi)=f(W-\xi)\) or

\(C_{1}\left[e^{-\sqrt{\epsilon }\xi }-e^{-\sqrt{\epsilon }(W-\xi ) }\right] =-C_{2} \left[e^{\sqrt{\epsilon }\xi }-e^{\sqrt{\epsilon }(W-\xi ) }\right]=C_{2}e^{\sqrt{\epsilon }W }\left[e^{-\sqrt{\epsilon }\xi }-e^{-\sqrt{\epsilon }(W-\xi ) }\right]\),

or

\(C_{1}=C_{2}e^{\sqrt{\epsilon }W }\).

Consequently,

\(f(\xi )=1+C_{1}\left[e^{-\sqrt{\epsilon }\xi }+e^{-\sqrt{\epsilon }(W-\xi ) }\right]\),

\(^{3}\)Using the actual W at hand will result in a better approximation, but new values for a and h must be determined if W is changed; in addition, the mathematics become considerably more involved.

and substituting this expression into the governing equation at \(\xi=0\) gives

\(f(0)=1+C_{1}(1+e^{-\sqrt{\epsilon }\xi })\)

\(=\epsilon +\frac{1}{2} (1-\epsilon )\int_{0}^{W}\left\{1+C_{1}\left[e^{-\sqrt{\epsilon }\xi ^{\prime } }+e^{-\sqrt{\epsilon }(W-\xi ^{\prime } ) }\right] \right\} e^{-\xi ^{\prime }} d\xi ^{\prime }\)

\(=\epsilon +\frac{1}{2} (1-\epsilon )\int_{0}^{W}\left\{e^{-\xi ^{\prime }}+C_{1}\left[e^{-(1+\sqrt{\epsilon })\xi ^{\prime }}+e^{-\xi ^{\prime }-\sqrt{\epsilon }(W-\xi ^{\prime })} \right] \right\} d\xi ^{\prime }\)

\(=\epsilon -\frac{1}{2} (1-\epsilon )\left\{e^{-\xi ^{\prime }}+C_{1}\left[\frac{e^{-(1+\sqrt{\epsilon })\xi ^{\prime }}}{1+\sqrt{\epsilon }}+\frac{e^{-\xi-\sqrt{\epsilon }(W-\xi ^{\prime })}}{1-\sqrt{\epsilon }} \right] \right\} \mid ^{W}_{0}\)

\(=\epsilon +\frac{1}{2} (1-\epsilon )\left\{1-e^{-W}+C_{1}\left[\frac{1-e^{-(1+\sqrt{\epsilon })W}}{1+\sqrt{\epsilon }}+\frac{e^{-\sqrt{\epsilon }W} -e^{-W}}{1-\sqrt{\epsilon }} \right] \right\}\).

Solving this for \(C_{1}\) gives

\(1-\epsilon -\frac{1}{2} (1-\epsilon )(1-e^{-W})=C_{1}\left[\frac{1}{2} (1-\sqrt{\epsilon } )(1-e^{-(1+\sqrt{\epsilon })W})+\frac{1}{2} (1+\sqrt{\epsilon } )(e^{-\sqrt{\epsilon }W}-e^{-W} )-(1-e^{-\sqrt{\epsilon }W})\right]\)

\(\frac{1}{2} (1-\epsilon )(1+e^{-W})=C_{1}\left\{\frac{1}{2}(1-\sqrt{\epsilon } )+\frac{1}{2}(1+\sqrt{\epsilon } )e^{-\sqrt{\epsilon }W}-1-e^{-\sqrt{\epsilon }W}-\left[\frac{1}{2}(1-\sqrt{\epsilon } )e^{-\sqrt{\epsilon }W}+\frac{1}{2}(1+\sqrt{\epsilon } )\right] e^{-W}\right\}\),

or

\(C_{1}=-\frac{1-\epsilon }{(1+\sqrt{\epsilon } )+(1-\sqrt{\epsilon } )e^{-\sqrt{\epsilon }W}}\)

and

\(f(\xi )=1-(1-\epsilon )\frac{e^{-\sqrt{\epsilon }\xi }+e^{-\sqrt{\epsilon }(W-\xi )} }{(1+\sqrt{\epsilon } )+(1-\sqrt{\epsilon } )e^{-\sqrt{\epsilon }W}}\).

Finally, the nondimensional heat flux follows as

\(\Psi (\xi )=\frac{\epsilon }{1-\epsilon } [1-f(\xi )]=\frac{\epsilon \left[e^{-\sqrt{\epsilon }\xi }+e^{-\sqrt{\epsilon }(W-\xi )} \right] }{(1+\sqrt{\epsilon } )+(1-\sqrt{\epsilon } )e^{-\sqrt{\epsilon }W}}\),

which is also included in Fig. 5-14.

Note that \(e^{-\left|\xi ^{\prime }-\xi \right| }\) is not a particularly good approximation for the kernel, since the actual kernel has a zero first derivative at \(\xi ^{\prime }= \xi\). A better approximation can be obtained by using

\(K(\xi ,\xi ^{\prime })\simeq a_{1} e^{-b_{1} \left|\xi ^{\prime }-\xi \right|} + a_{2} e^{-b_{2} \left|\xi ^{\prime }-\xi \right|}\)

(with \(a_{1}\gt 1\) and \(a_{2}\lt 0\)). If W is relatively small, say \(\lt \frac{1}{2}\), a good approximation may be obtained using

\(K(\xi ,\xi ^{\prime })\simeq\cos a (\xi ^{\prime }-\xi)\)

(since the kernel has an inflection point at \(\left|\xi ^{\prime }-\xi \right|=\frac{1}{2}\)).

We shall conclude this chapter with two examples that demonstrate that exact analytical solutions are possible for a few simple geometries for which the view factors between area elements attain certain special forms.
